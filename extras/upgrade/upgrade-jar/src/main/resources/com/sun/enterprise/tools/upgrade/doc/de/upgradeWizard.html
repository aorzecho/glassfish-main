Aktualisieren mithilfe des Upgrade-Assistenten

Geben Sie mithilfe des Upgrade-Assistenten die Installationsverzeichnisse der Quell- und Zielserver und ggf. das Master-Passwort des Zielservers an.

  1. Geben Sie im Feld für das Quelldomänenverzeichnis den Speicherort der aktuellen Installation an, aus welcher die Konfiguration importiert werden soll.  Sie müssen das Domänenverzeichnis eingeben. Beispiel: <install-root>/domains/domain1.

  2. Geben Sie im Feld "Ziel-Domänen-Root-Verzeichnis" den Speicherort der Application Server-Installation ein, auf welche die Konfiguration übertragen werden soll. Sie müssen in dieses Feld das Domänen-Root eingeben.
  3. Geben Sie das Master-Passwort des Quellanwendungsservers an.
  4. Klicken Sie anschließend auf "Weiter".
  5. Das Fenster "Upgrade-Ergebnisse" wird geöffnet und zeigt den Upgrade-Status an.
  6. Klicken Sie auf "Fertig stellen", um das Upgrade-Tool zu schließen, wenn der Upgrade-Prozess abgeschlossen ist.
