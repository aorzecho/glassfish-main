Esecuzione dell'aggiornamento utilizzando Upgrade Wizard (Aggiornamento guidato)

Utilizzare Upgrade Wizard (Aggiornamento guidato) per specificare le directory di installazione di origine e di destinazione e la password principale del server di destinazione, se necessaria.

  1. Nel campo Source Domain Directory (Directory dominio di origine) immettere il percorso dell'installazione esistente dalla quale importare la configurazione.  È necessario immettere la directory del dominio. Ad esempio, <radice-installazione>/domains/domain1.

  2. Nel campo Target Domains Root Directory (Directory radice dominio di destinazione), immettere il percorso dell'installazione del server di applicazioni in cui trasferire la configurazione. È necessario fornire la directory radice del dominio.
  3. Fornire la password principale del server delle applicazioni di origine.
  4. Al termine, premere il pulsante Next (Avanti).
  5. Viene visualizzata la schermata Upgrade Results (Risultati aggiornamento) indicante lo stato dell'operazione di aggiornamento.
  6. Al termine della procedura di aggiornamento, fare clic sul pulsante Finish (Fine) per chiudere lo strumento di aggiornamento.
