Atualizando com o Assistente de atualização

Use a tela do Assistente de atualização para especificar os diretórios de origem e de destino da instalação e a senha mestre do servidor de destino, se necessário.

  1. No campo Diretório de origem do domínio, insira a localização da instalação existente a partir da qual importar a configuração.  É necessário inserir o diretório de domínio. Por exemplo, <install-root>/domains/domain1.

  2. No campo Diretório raiz de destino dos domínios, insira a localização da instalação do Application Server para o qual transferir a configuração. É necessário fornecer a raiz dos domínios como a entrada deste campo.
  3. Forneça a senha mestre do servidor de aplicativos de destino.
  4. Clique no botão Avançar ao terminar.
  5. O painel Upgrade Results é exibido, mostrando o status da operação de atualização.
  6. Clique no botão Finish para fechar a Upgrade Tool quando o processo de atualização tiver terminado.
