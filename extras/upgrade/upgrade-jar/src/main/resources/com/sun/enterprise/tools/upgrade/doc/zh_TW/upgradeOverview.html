升級工具：概述

使用 [升級工具] 可從 Application Server 9.1 或 GlassFish Prelude 安裝將配置資訊傳送到 GlassFish Server Open Source Edition 3.1 安裝。傳送的資訊包括，有關已部署應用程式、檔案範圍、安全性憑證以及其他資源和伺服器配置設定的資料。

此工具會自動偵測指定來源伺服器安裝的版本。

注意：在開始升級程序之前，請確定來源伺服器 (要升級的伺服器) 和目標伺服器 (要升級至的伺服器) 均已停止。

升級術語

以下是與升級程序相關的重要術語：

  • 來源網域目錄：將升級至新伺服器的舊網域。原始網域不會變更；只會複製它，並升級副本。

  • 目標網域根目錄：要升級至的安裝網域目錄。這可以是任何目錄，但是如果來源網域將會從一個 SERVER_HOME/domains 目錄移至另一個目錄，則可以將協力廠商 jar 檔案從來源伺服器複製到目標伺服器。

  • 主密碼：目標伺服器的主密碼。保留空白即使用預設值。
