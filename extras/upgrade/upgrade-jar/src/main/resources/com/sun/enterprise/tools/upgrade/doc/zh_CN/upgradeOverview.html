升级工具：概述

使用升级工具将配置信息从 Application Server 9.1 或 GlassFish Prelude 安装转移到 GlassFish Server Open Source Edition 3.1 安装。传输的信息包括有关部署的应用程序、文件领域、安全证书和其他资源及服务器配置设置的数据。

该工具自动检测指定源服务器安装的版本。

注：在开始升级过程之前，确保源服务器（从中升级的服务器）和目标服务器（升级到的服务器）已停止。

升级术语

以下是与升级过程相关的重要术语：

  • 源域目录：将升级到新服务器的旧域。未更改原始域；该域已复制，副本已升级。

  • 目标域根目录：升级到的安装域目录。这可以是任何目录，但如果源域从一个 SERVER_HOME/域目录移动到其他位置，则第三方 jar 文件可从源服务器复制到目标服务器。

  • 主密码：目标服务器的主密码。默认情况下保留为空。
