업그레이드 마법사를 사용한 업그레이드

업그레이드 마법사 화면을 사용하여 소스 및 대상 설치 디렉토리와 필요한 경우 대상 서버의 마스터 비밀번호를 지정합니다.

  1. 소스 도메인 디렉토리 필드에 구성을 가져올 기존 설치 위치를 입력합니다.  도메인 디렉토리를 입력해야 합니다(예: <install-root>/domains/domain1).

  2. 대상 도메인 루트 디렉토리 필드에 구성을 전송할 Application Server 설치 위치를 입력합니다. 이 필드에 도메인 루트를 입력해야 합니다.
  3. 소스 응용 프로그램 서버의 마스터 비밀번호를 입력합니다.
  4. 완료되면 다음 버튼을 누릅니다.
  5. 업그레이드 작업의 상태를 나타내는 업그레이드 결과 패널이 표시됩니다.
  6. 업그레이드 프로세스가 완료되면 마침 버튼을 눌러 업그레이드 도구를 닫습니다.
