업그레이드 결과 화면

업그레이드 결과 화면에는 업그레이드 상태가 표시됩니다.

  1. 결과 패널에는 성공적으로 완료된 단계와 실패한 단계를 보여주는 메시지가 표시됩니다.

    표시된 메시지는 <AS_HOME>/domains 디렉토리에 있는 upgrade.log 파일에도 기록됩니다.

  2. 화면 아래쪽의 상태 표시줄은 진행 상태를 나타냅니다.
  3. 완료되면 마침 버튼을 누릅니다.

업그레이드 도구가 대상 서버에서 asadmin을 시작하여 업그레이드를 수행합니다. 이 프로세스는 개별 프로세스이며 업그레이드 중에 문제가 발생할 경우 서버 로그 파일을 검사해야 할 수 있습니다.
