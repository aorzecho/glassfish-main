Recursos externos

Un recurso JNDI externo accede a un depósito JNDI externo, como, por ejemplo, un servidor LDAP.

Utilice la página Recursos externos para configurar recursos JNDI externos.

Para cada recurso externo, se proporciona la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso externo.

Habilitado

"True" si el recurso externo está habilitado y "False" si está deshabilitado.

Tipo de recurso

El tipo de recurso externo.

Descripción

Una descripción del recurso externo.

La tabla de recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso externo.

Borrar

Botón para eliminar una o más recursos externos seleccionados.

Habilitar

Para habilitar uno o más recursos externos seleccionados.

Deshabilitar

Botón para deshabilitar uno o más recursos externos seleccionados.
