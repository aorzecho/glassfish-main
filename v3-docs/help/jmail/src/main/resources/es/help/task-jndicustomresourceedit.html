Editar un Recurso JNDI Personalizado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JNDI.

  3. En el nodo JNDI, seleccione Recursos Personalizados.

    Se abrirá la página Recursos Personalizados.

  4. En la página Recursos Personalizados, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar Recurso Personalizado.

    Nota:

    El nombre JNDI es un campo de sólo lectura. Únicamente puede especificar el nombre JNDI al crear un nuevo recurso personalizado.

  5. Para modificar el tipo de recurso, realice una de las acciones siguientes.

    • En la lista desplegable, seleccione una definición de tipo.

    • Seleccione el segundo botón de radio y escriba una definición de tipo completamente cualificada que siga el formato xxx.xxx.

  6. En el campo Clase de Fábrica, escriba un nombre de clase de fábrica para el recurso personalizado.

    La Clase de Fábrica es el nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.ObjectFactory.

  7. En el campo Descripción, escriba una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  8. Seleccione la casilla de verificación de activación de Estado para activar el recurso.

    Por defecto, el recurso está activado, lo cual significa que está disponible en cuanto se crea.

  9. En la sección Propiedades Adicionales, especifique propiedades adicionales del recurso personalizado.

    GlassFish Server no define ninguna propiedad adicional para los recursos personalizados.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, seleccione la casilla de verificación a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  10. Haga clic en Guardar.
