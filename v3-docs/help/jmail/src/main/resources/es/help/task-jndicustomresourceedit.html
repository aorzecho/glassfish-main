Para editar un recurso JNDI personalizado

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. En el nodo Recursos, expanda el nodo JNDI.
  3. En el nodo JNDI, seleccione Recursos personalizados.

    Se abrirá la página Recursos personalizados.

  4. En la página Recursos personalizados, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar recurso personalizado.

    Nota - El nombre JNDI es un campo de sólo lectura. Únicamente puede especificar el nombre JNDI al crear un nuevo recurso personalizado.

  5. Para modificar el tipo de recurso, realice una de las acciones siguientes.
    • En la lista desplegable, seleccione una definición de tipo.
    • Seleccione el segundo botón de opción y escriba una definición de tipo completamente cualificada que siga el formato xxx.xxx.
  6. En el campo Clase de fábrica, escriba un nombre de clase de fábrica para el recurso personalizado.

    La Clase de fábrica es el nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.ObjectFactory.

  7. En el campo Descripción, escriba una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  8. Seleccione la casilla de verificación Estado habilitado para habilitar el recurso.

    De forma predeterminada, el recurso está habilitado, lo cual significa que está disponible en cuanto se crea.

  9. (Opcional) En la sección Propiedades adicionales, especifique propiedades adicionales del recurso personalizado.

    Servidor GlassFish no define ninguna propiedad adicional para los recursos personalizados.

    • Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.
    • Para modificar una propiedad, edite su campo Valor.
    • Para eliminar una propiedad, marque la casilla de verificación situada a la izquierda del campo Nombre de la propiedad que quiera eliminar y, a continuación, haga clic en el botón Eliminar propiedades.
  10. Haga clic en Guardar.
