Para editar un recurso JNDI externo

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. En el nodo Recursos, expanda el nodo JNDI.
  3. En el nodo JNDI, seleccione Recursos externos.

    Se abrirá la página Recursos externos.

  4. En la página Recursos externos, haga clic en el nombre del recurso que va a editar.

    Se abrirá la página Editar recurso externo.

  5. Para modificar el tipo de recurso, realice una de las acciones siguientes.
    • En la lista desplegable, seleccione una definición de tipo.
    • Seleccione el segundo botón de opción y escriba una definición de tipo completamente cualificada que siga el formato xxx.xxx.
  6. En el campo Clase de fábrica, escriba un depósito externo de clase de fábrica JNDI para el recurso externo.

    La Clase de fábrica es el nombre especificado por el usuario para la clase de fábrica, como com.sun.jndi.ldap. Esta clase implementa la interfaz javax.naming.spi.InitialContextFactory.

  7. En el campo Búsqueda JNDI, escriba el valor JNDI que buscar en el depósito externo.

    Por ejemplo, cuando se crea un recurso externo para conectar a un depósito externo, para probar una clase de bean, la búsqueda JNDI puede seguir el formato: cn=testmybean.

  8. En el campo Descripción, escriba una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  9. Seleccione la casilla de verificación Estado habilitado para habilitar el recurso.

    De forma predeterminada, el recurso está habilitado, lo cual significa que está disponible en cuanto se crea.

  10. (Opcional) En la sección Propiedades adicionales, especifique propiedades adicionales del recurso externo.

    GlassFish Server no define ninguna propiedad adicional para los recursos externos.

    • Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.
    • Para modificar una propiedad, edite su campo Valor.
    • Para eliminar una propiedad, marque la casilla de verificación situada a la izquierda del campo Nombre de la propiedad que quiera eliminar y, a continuación, haga clic en el botón Eliminar propiedades.
  11. Haga clic en Guardar.
