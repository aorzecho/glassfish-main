Para crear un recurso JNDI personalizado

Un recurso JNDI personalizado especifica una fábrica de objetos de recursos para todo el servidor.

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. En el nodo Recursos, expanda el nodo JNDI.
  3. En el nodo JNDI, seleccione Recursos personalizados.

    Se abrirá la página Recursos personalizados.

  4. En la página Recursos personalizados, haga clic en Nuevo.

    Se abrirá la página Nuevo recurso personalizado.

  5. En el campo Nombre JNDI, introduzca un nombre exclusivo.

    Este nombre se registrará en el servicio de nombres de JNDI. El nombre JNDI organiza y ubica componentes dentro de un entorno informático distribuido del mismo modo que los ficheros organizan y representan ubicaciones de libros en una biblioteca. Por consiguiente, el nombre JNDI se convierte en un importante método de acceso al recurso personalizado.

  6. En el campo Tipo de recurso, realice una de las acciones siguientes.
    • En la lista desplegable, seleccione una definición de tipo.
    • Seleccione el segundo botón de opción y escriba una definición de tipo completamente cualificada que siga el formato xxx.xxx.
  7. En el campo Clase de fábrica, escriba un nombre de clase de fábrica para el recurso personalizado que se va a crear.

    La Clase de fábrica es el nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.ObjectFactory.

  8. En el campo Descripción, escriba una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  9. Seleccione la casilla de verificación Estado habilitado para habilitar el recurso.

    De forma predeterminada, el recurso está habilitado, lo cual significa que está disponible en cuanto se crea.

  10. (Opcional) En la sección Propiedades adicionales, especifique propiedades adicionales del recurso personalizado.

    GlassFish Server no define ninguna propiedad adicional para los recursos personalizados.

    Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.

  11. Haga clic en Aceptar.
