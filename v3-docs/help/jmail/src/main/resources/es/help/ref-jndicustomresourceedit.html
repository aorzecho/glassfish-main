Editar recurso personalizado

Utilice la página Editar recurso personalizado para editar un recurso JNDI personalizado.

La página Editar recurso personalizado contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso. El nombre JNDI es un campo de sólo lectura. Únicamente puede especificar el nombre JNDI al crear un nuevo recurso personalizado.

Tipo de recurso

La definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

El nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.ObjectFactory.

Descripción

Una descripción de este recurso personalizado.

Estado

El estado del recurso personalizado. El recurso personalizado puede habilitarse o deshabilitarse. El valor predeterminado es Habilitado.

Propiedades adicionales

Propiedades adicionales para el recurso personalizado. Servidor GlassFish no define ninguna propiedad adicional de los recursos personalizados.
