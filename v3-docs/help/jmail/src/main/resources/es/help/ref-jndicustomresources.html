Recursos Personalizados

Un recurso personalizado especifica una fábrica de objetos de recursos personalizada para todo el servidor.

Utilice la página Recursos personalizados para configurar los recursos JNDI personalizados.

Para cada recurso personalizado, se proporciona la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso personalizado.

Habilitado

"True" si el recurso personalizado está habilitado y "False" si está deshabilitado.

Tipo de recurso

El tipo de recurso personalizado.

Descripción

Una descripción del recurso personalizado.

La tabla de recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso personalizado.

Borrar

Botón para eliminar uno o varios recursos personalizados seleccionados.

Habilitar

Botón para habilitar uno o varios recursos personalizados seleccionados.

Deshabilitar

Botón para deshabilitar uno o varios recursos personalizados seleccionados.
