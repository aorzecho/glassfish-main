Nuevo recurso externo

Utilice la página Nuevo recurso externo para crear un recurso JNDI externo.

La página Nuevo recurso externo contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso.

Tipo de recurso

La definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

El nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.InitialContextFactory.

Búsqueda JNDI

El valor JNDI para buscar en el depósito externo.

Por ejemplo, cuando se crea un recurso externo para conectar a un depósito externo, para probar una clase de bean, la búsqueda JNDI puede seguir el formato: cn=testmybean.

Descripción

Una descripción de este recurso externo.

Estado

El estado del recurso externo. El recurso externo puede habilitarse o deshabilitarse. El valor predeterminado es Habilitado.

Propiedades adicionales

Propiedades adicionales del recurso externo. GlassFish Server no define ninguna propiedad adicional para los recursos externos.

Destinos

Clústeres e instancias independientes del recurso externo. Mueva determinados destinos a la columna Destinos seleccionados mediante los botones Agregar y Añadir todos. Mueva los destinos que no necesite a la columna Destinos disponibles mediante los botones Eliminar y Eliminar todos.

Esta opción sólo se muestra si en el dominio se han creado instancias de clúster o independientes.
