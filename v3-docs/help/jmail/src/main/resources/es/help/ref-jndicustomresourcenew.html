Nuevo Recurso Personalizado

Utilice la página Nuevo recurso personalizado para crear un nuevo recurso JNDI personalizado.

La página Nuevo recurso personalizado contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso.

Tipo de recurso

La definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

El nombre especificado por el usuario para la clase de fábrica. Esta clase implementa la interfaz javax.naming.spi.ObjectFactory.

Descripción

Una descripción de este recurso personalizado.

Estado

El estado del recurso personalizado. El recurso personalizado puede habilitarse o deshabilitarse. El valor predeterminado es Habilitado.

Propiedades adicionales

Propiedades adicionales para el recurso personalizado. GlassFish Server no define ninguna propiedad adicional de los recursos personalizados.

Destinos

Clústeres e instancias independientes del recurso personalizado. Mueva los destinos deseados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.

Esta opción sólo se muestra si se han creado clusters o instancias independientes en el dominio.
