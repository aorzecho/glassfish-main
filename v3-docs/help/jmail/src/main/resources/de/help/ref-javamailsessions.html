JavaMail-Sitzungen

GlassFish-Server beinhaltet die JavaMail-API. JavaMail bietet Zugriff von Java-Anwendungen auf Mailserver in Ihrem Netzwerk oder im Internet, die Internet Message Access Protocol (IMAP) oder Simple Mail Transfer Protocol (SMTP) unterstützen.

Auf der Seite "JavaMail-Sitzung" können Sie JavaMail-Sitzungen konfigurieren.

Für jede JavaMail-Sitzung werden die folgenden Informationen zur Verfügung gestellt.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die JavaMail-Sitzung identifiziert.

Enabled (Aktiviert)

"True", wenn die Sitzung aktiviert ist, bzw. "false", wenn sie deaktiviert ist.

Beschreibung

Eine Beschreibung der JavaMail-Sitzung.

Die Tabelle "Sitzungen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen JavaMail-Sitzung.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter JavaMail-Sitzungen.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter JavaMail-Sitzungen.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter JavaMail-Sitzungen.
