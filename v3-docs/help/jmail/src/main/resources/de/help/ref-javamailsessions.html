JavaMail-Sessions

GlassFish Server beinhaltet die JavaMail-API. JavaMail bietet Zugriff von Java-Anwendungen auf Mailserver in Ihrem Netzwerk oder im Internet, die Internet Message Access Protocol (IMAP) oder Simple Mail Transfer Protocol (SMTP) unterstützen.

Auf der Seite "JavaMail-Session" können Sie JavaMail-Sessions konfigurieren.

Für jede JavaMail-Session werden die folgenden Informationen zur Verfügung gestellt.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die JavaMail-Session identifiziert.

Enabled (Aktiviert)

"True", wenn die Session aktiviert ist, bzw. "False", wenn sie deaktiviert ist.

Beschreibung

Eine Beschreibung der JavaMail-Session.

Die Tabelle "Sessions" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen JavaMail-Session.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter JavaMail-Sessions.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter JavaMail-Sessions.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter JavaMail-Sessions.

Verwandte Aufgaben

  • So wird eine JavaMail-Session erstellt

  • So wird eine JavaMail-Session bearbeitet

  • So wird eine JavaMail-Session gelöscht

Zugehörige asadmin-Befehle

  • create-javamail-resource(1)

  • list-javamail-resources(1)

  • delete-javamail-resource(1)
