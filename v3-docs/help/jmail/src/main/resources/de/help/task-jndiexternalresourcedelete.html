So wird eine externe JNDI-Ressource gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JNDI-Knoten ein.

  3. Wählen Sie unter dem JNDI-Knoten den Eintrag "Externe Ressourcen" aus.

    Die Seite "Externe Ressourcen" wird geöffnet.

  4. Aktivieren Sie in der Tabelle "Ressourcen" das Kontrollkästchen neben dem Namen der zu löschenden Ressource.

  5. Klicken Sie auf "Löschen".
