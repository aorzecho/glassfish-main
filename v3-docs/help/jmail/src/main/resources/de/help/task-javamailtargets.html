So verwalten Sie JavaMail-Sessionziele

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "JavaMail-Sessions" aus.

    Die Seite "JavaMail-Sessions" wird geöffnet.

  3. Klicken Sie in der Tabelle "Sessions" auf den Namen der Session, deren Ziele Sie verwalten möchten.

    Die Seite "JavaMail-Session bearbeiten" wird geöffnet.

  4. Klicken Sie auf die Registerkarte "Ziel".

    Die Seite "JavaMail-Sessionziele" wird geöffnet.

  5. Wählen Sie in der Tabelle "Ziele" die Aktion, die Sie durchführen möchten:

    • Um Ziele hinzuzufügen oder zu entfernen, klicken Sie auf "Ziele verwalten".

    • Um ein Ziel zu aktivieren, aktivieren Sie das Kontrollkästchen neben der gewünschten Serverinstanz oder dem gewünschten Cluster und klicken Sie auf "Aktivieren".

    • Um ein Ziel zu deaktivieren, aktivieren Sie das Kontrollkästchen neben der gewünschten Serverinstanz oder dem gewünschten Cluster und klicken Sie auf "Deaktivieren".
