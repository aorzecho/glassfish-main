Nuova risorsa personalizzata

Nella pagina Nuova risorsa personalizzata è possibile creare una nuova risorsa JNDI personalizzata.

La pagina Nuova risorsa personalizzata contiene le opzioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa.

Tipo di risorsa

Definizione completamente qualificata del tipo di risorsa.

Classe factory

Nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

Descrizione

Descrizione della risorsa personalizzata corrente.

Stato

Stato della risorsa personalizzata. La risorsa personalizzata può essere abilitata o disabilitata. Il valore predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa personalizzata. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

Destinazioni

Cluster e istanze standalone per la risorsa personalizzata. Per spostare le destinazioni desiderate nella colonna Destinazioni selezionate, utilizzare i pulsanti Aggiungi e Aggiungi tutto. Per spostare le destinazioni non necessarie nella colonna Destinazioni disponibili, utilizzare i pulsanti Rimuovi e Rimuovi tutto.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
