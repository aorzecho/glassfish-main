New Custom Resource

Nella pagina New Custom Resource è possibile creare una nuova risorsa JNDI personalizzata.

La pagina New Custom Resource contiene le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa.

Resource Type

Definizione completa del tipo per la risorsa.

Factory Class

Nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

Description

Descrizione della risorsa personalizzata.

Status

Stato della risorsa personalizzata. La risorsa personalizzata può essere abilitata o disabilitata. Il valore predefinito è Enabled.

Additional Properties

Proprietà aggiuntive per la risorsa personalizzata. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

Targets

Cluster e istanze standalone per la risorsa personalizzata. Per spostare le destinazioni desiderate nella colonna Selected Targets, utilizzare i pulsanti Add e Add All. Per spostare le destinazioni non necessarie nella colonna Available Targets, utilizzare i pulsanti Remove e Remove All.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
