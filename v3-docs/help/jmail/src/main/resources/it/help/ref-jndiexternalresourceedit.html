Modifica risorsa esterna

Nella pagina Modifica risorsa esterna è possibile modificare una risorsa JNDI esterna.

La pagina Modifica risorsa esterna contiene le opzioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa esterna.

Tipo di risorsa

Definizione completamente qualificata del tipo di risorsa.

Classe factory

Nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

Ricerca JNDI

Valore JNDI per eseguire ricerche nel repository esterno.

Ad esempio, quando si crea una risorsa esterna da connettere a un repository esterno, per eseguire il test di una classe di bean la ricerca JNDI può avere il seguente aspetto: cn=testmybean.

Descrizione

Descrizione della risorsa esterna corrente.

Stato

Stato della risorsa esterna. La risorsa esterna può essere abilitata o disabilitata. Il valore predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa esterna. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse esterne.
