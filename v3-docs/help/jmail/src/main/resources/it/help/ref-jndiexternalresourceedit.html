Edit External Resource

Nella pagina Edit External Resource è possibile modificare una risorsa JNDI esterna.

La pagina Edit External Resource contiene le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa. Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa esterna.

Resource Type

Definizione completa del tipo per la risorsa.

Factory Class

Nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

JNDI Lookup

Valore JNDI per eseguire ricerche nel repository esterno.

Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per testare una classe di bean, la funzione JNDI Lookup può avere l'aspetto seguente: cn= testmybean.

Description

Descrizione della risorsa esterna.

Status

Stato della risorsa esterna. La risorsa esterna può essere abilitata o disabilitata. Il valore predefinito è Enabled.

Additional Properties

Proprietà aggiuntive per la risorsa esterna. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse esterne.
