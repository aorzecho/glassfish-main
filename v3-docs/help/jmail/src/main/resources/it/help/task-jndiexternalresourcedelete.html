Eliminare una risorsa JNDI esterna

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JNDI.
  3. Nel nodo JNDI, selezionare External Resources.

    Viene visualizzata la pagina External Resources.

  4. Nella tabella Resources, selezionare la casella di controllo in corrispondenza del nome della risorsa da eliminare.
  5. Fare clic su Delete.
