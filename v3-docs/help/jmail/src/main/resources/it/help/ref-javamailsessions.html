JavaMail Sessions

GlassFish Server include l'API JavaMail. JavaMail offre accesso da applicazioni Java a server di posta elettronica compatibili con IMAP (Internet Message Access Protocol) e SMTP (Simple Mail Transfer Protocol) nella rete o in Internet.

Nella pagina JavaMail Sessions è possibile configurare sessioni JavaMail.

Per ogni sessione JavaMail vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la sessione JavaMail.

Enabled

Opzione attivata se la sessione è abilitata e disattivata se la sessione è disabilitata.

Description

Descrizione della sessione JavaMail.

La tabella Sessions contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova sessione JavaMail.

Delete

Pulsante che consente di eliminare una o più sessioni JavaMail selezionate.

Enable

Pulsante che consente di abilitare una o più sessioni JavaMail selezionate.

Disable

Pulsante che consente di disabilitare una o più sessioni JavaMail selezionate.
