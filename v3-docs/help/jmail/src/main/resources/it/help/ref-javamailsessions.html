Sessioni JavaMail

GlassFish Server include l'API JavaMail. JavaMail fornisce l'accesso da applicazioni Java a server di posta che supportano i protocolli IMAP (Internet Message Access Protocol) e SMTP (Simple Mail Transfer Protocol) nella rete o in Internet.

Nella pagina Sessioni JavaMail è possibile configurare sessioni JavaMail.

Per ogni sessione JavaMail vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la sessione JavaMail.

Abilitato

Opzione attivata se la sessione è abilitata; disattivata se la sessione è disabilitata.

Descrizione

Descrizione della sessione JavaMail.

La tabella Sessioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova sessione JavaMail.

Elimina

Pulsante che consente di eliminare una o più sessioni JavaMail selezionate.

Abilita

Pulsante che consente di abilitare una o più sessioni JavaMail selezionate.

Disabilita

Pulsante che consente di disabilitare una o più sessioni JavaMail selezionate.
