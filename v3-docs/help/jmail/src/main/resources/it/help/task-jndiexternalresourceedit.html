Modificare una risorsa JNDI esterna

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JNDI.
  3. Nel nodo JNDI, selezionare External Resources.

    Viene visualizzata la pagina External Resources.

  4. Nella pagina External Resources fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Edit External Resource.

  5. Per modificare il valore relativo a Resource Type, procedere in uno dei seguenti modi.
    • Dall'elenco a discesa, selezionare una definizione di tipo.
    • Selezionare il secondo pulsate di scelta e digitare una definizione di tipo completa, con il formato xxx. xxx.
  6. Nel campo Factory Class, digitare un repository esterno di classe di factory JNDI per la risorsa esterna.

    Factory Class è il nome specificato per la classe di factory, ad esempio com.sun.jndi.ldap. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

  7. Nel campo JNDI Lookup, digitare il valore JNDI per eseguire ricerche nel repository esterno.

    Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per testare una classe di bean, la funzione JNDI Lookup può avere l'aspetto seguente: cn= testmybean.

  8. Nel campo Description, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  9. Selezionare la casella di controllo Status Enabled per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  10. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa esterna.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse esterne.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  11. Fare clic su Save.
