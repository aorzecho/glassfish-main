Gestire le destinazioni di una sessione JavaMail

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, selezionare JavaMail Sessions.

    Viene visualizzata la pagina JavaMail Sessions.

  3. Nella tabella Sessions, fare clic sul nome della sessione di cui si desidera gestire le destinazioni.

    Viene visualizzata la pagina Edit JavaMail Session.

  4. Fare clic sulla scheda Target.

    Viene visualizzata la pagina JavaMail Session Targets.

  5. Nella tabella Targets, scegliere l'azione da eseguire:
    • Per aggiungere o rimuovere destinazioni, fare clic su Manage Targets.
    • Per abilitare una destinazione, fare clic sulla casella di controllo accanto all'istanza server o al cluster desiderato e fare clic su Enable.
    • Per disabilitare una destinazione, fare clic sulla casella di controllo accanto all'istanza server o al cluster desiderato e fare clic su Disable.
