Creare una risorsa JNDI esterna

Le risorse JNDI esterne accedono a repository JNDI esterni, ad esempio server LDAP.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JNDI.
  3. Nel nodo JNDI, selezionare External Resources.

    Viene visualizzata la pagina External Resources.

  4. Nella pagina External Resources, fare clic su New.

    Viene visualizzata la pagina New External Resource.

  5. Nel campo JNDI Name, digitare un nome univoco.

    Il nome verrà registrato nel servizio di denominazione JNDI. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza il nome JNDI diventa un metodo importante per l'accesso alla risorsa.

  6. Nel campo Resource Type procedere in uno dei seguenti modi.
    • Dall'elenco a discesa, selezionare una definizione di tipo.
    • Selezionare il secondo pulsate di scelta e digitare una definizione di tipo completa, con il formato xxx. xxx.
  7. Nel campo Factory Class, digitare un repository esterno di classe di factory JNDI per la risorsa esterna da creare.

    Factory Class è il nome specificato dall'utente per la classe di factory, ad esempio com.sun.jndi.ldap . Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

  8. Nel campo JNDI Lookup, digitare il valore JNDI per eseguire ricerche nel repository esterno.

    Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per testare una classe di bean, la funzione JNDI Lookup può avere l'aspetto seguente: cn= testmybean.

  9. Nel campo Description, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  10. Selezionare la casella di controllo Status Enabled per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  11. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa esterna.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse esterne.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  12. Fare clic su OK.
