Creare una risorsa JNDI esterna

Le risorse JNDI esterne accedono a repository JNDI esterni, ad esempio server LDAP.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JNDI.

  3. Nel nodo JNDI, selezionare Risorse esterne.

    Viene visualizzata la pagina Risorse esterne.

  4. Nella pagina Risorse esterne, fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa esterna.

  5. Nel campo Nome JNDI, digitare un nome univoco.

    Il nome verrà registrato nel servizio di denominazione JNDI. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza il nome JNDI diventa un metodo importante per l'accesso alla risorsa.

  6. Nel campo Tipo di risorsa procedere in uno dei seguenti modi.

    • Dall'elenco a discesa, selezionare una definizione di tipo.

    • Selezionare il secondo pulsante di scelta e digitare una definizione di tipo completamente qualificata, nel formato .xxx.

  7. Nel campo Classe factory, digitare un repository esterno di classe factory JNDI per la risorsa esterna da creare.

    Classe factory è il nome specificato per la classe di factory, ad esempio com.sun.jndi.ldap. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

  8. Nel campo Ricerca JNDI, digitare il valore JNDI per eseguire ricerche nel repository esterno.

    Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per eseguire test su una classe di bean, la funzione Ricerca JNDI può avere l'aspetto seguente: cn=testmybean.

  9. Nel campo Descrizione, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  10. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  11. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa esterna.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse esterne.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  12. Fare clic su OK.
