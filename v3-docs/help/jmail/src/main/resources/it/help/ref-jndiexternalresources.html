External Resources

Le risorse JNDI esterne accedono a repository JNDI esterni, ad esempio server LDAP.

Nella pagina External Resources è possibile configurare le risorse JNDI esterne.

Per ogni risorsa esterna, vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa esterna.

Enabled

Opzione attivata se la risorsa esterna è abilitata e disattivata se la risorsa esterna è disabilitata.

Resource Type

Tipo di risorsa esterna.

Description

Descrizione della risorsa esterna.

La tabella Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa esterna.

Delete

Pulsante che consente di eliminare una o più risorse esterne selezionate.

Enable

Pulsante che consente di abilitare una o più risorse esterne selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse esterne selezionate.

  • Eliminare una risorsa JNDI esterna
