Risorse esterne

Una risorsa JNDI esterna accede a un repository JNDI esterno, ad esempio un server LDAP.

Nella pagina Risorse esterne è possibile configurare le risorse JNDI esterne.

Per ogni risorsa esterna vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa esterna.

Abilitato

Opzione attivata se la risorsa esterna è abilitata; disattivata se la risorsa esterna è disabilitata.

Tipo di risorsa

Tipo di risorsa esterna.

Descrizione

Descrizione della risorsa esterna.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa esterna.

Elimina

Pulsante che consente di eliminare una o più risorse esterne selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse esterne selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse esterne selezionate.
