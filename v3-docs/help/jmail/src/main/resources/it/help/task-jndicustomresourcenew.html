Creare una risorsa JNDI personalizzata

Una risorsa personalizzata JNDI specifica una object factory personalizzata per la risorsa a livello di server.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JNDI.

  3. Nel nodo JNDI, selezionare Risorse personalizzate.

    Viene visualizzata la pagina Risorse personalizzate.

  4. Nella pagina Risorse personalizzate, fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa personalizzata.

  5. Nel campo Nome JNDI, digitare un nome univoco.

    Il nome verrà registrato nel servizio di denominazione JNDI. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza il nome JNDI diventa un metodo importante per l'accesso alla risorsa personalizzata.

  6. Nel campo Tipo di risorsa procedere in uno dei seguenti modi.

    • Dall'elenco a discesa, selezionare una definizione di tipo.

    • Selezionare il secondo pulsante di scelta e digitare una definizione di tipo completamente qualificata, nel formato . xxx.

  7. Nel campo Classe factory, digitare un nome di classe di factory per la risorsa personalizzata da creare.

    Classe factory è il nome specificato dall'utente per la classe di factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

  8. Nel campo Descrizione, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  9. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  10. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa personalizzata.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  11. Fare clic su OK.
