Creare una risorsa JNDI personalizzata

Una risorsa personalizzata JNDI specifica una object factory personalizzata per la risorsa a livello di server.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JNDI.
  3. Nel nodo JNDI, selezionare Custom Resources.

    Viene visualizzata la pagina Custom Resources.

  4. Nella pagina Custom Resources, fare clic su New.

    Viene visualizzata la pagina New Custom Resource.

  5. Nel campo JNDI Name, digitare un nome univoco.

    Il nome verrà registrato nel servizio di denominazione JNDI. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza il nome JNDI diventa un metodo importante per l'accesso alla risorsa personalizzata.

  6. Nel campo Resource Type procedere in uno dei seguenti modi.
    • Dall'elenco a discesa, selezionare una definizione di tipo.
    • Selezionare il secondo pulsate di scelta e digitare una definizione di tipo completa, con il formato xxx. xxx.
  7. Nel campo Factory Class, digitare un nome di classe di factory per la risorsa personalizzata da creare.

    Factory Class è il nome specificato dall'utente per la classe di factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory .

  8. Nel campo Description, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  9. Selezionare la casella di controllo Status Enabled per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  10. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa personalizzata.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  11. Fare clic su OK.
