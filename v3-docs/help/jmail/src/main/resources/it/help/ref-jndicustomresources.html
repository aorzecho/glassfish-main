Risorse personalizzate

Una risorsa personalizzata specifica un object factory personalizzato per la risorsa a livello di server.

Nella pagina Risorse personalizzate è possibile configurare le risorse JNDI personalizzate.

Per ogni risorsa personalizzata vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa personalizzata.

Abilitato

Opzione attivata se la risorsa personalizzata è abilitata; disattivata se la risorsa personalizzata è disabilitata.

Tipo di risorsa

Tipo di risorsa personalizzata.

Descrizione

Descrizione della risorsa personalizzata.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa personalizzata.

Elimina

Pulsante che consente di eliminare una o più risorse personalizzate selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse personalizzate selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse personalizzate selezionate.
