Custom Resources

Una risorsa personalizzata specifica una object factory personalizzata per la risorsa a livello di server.

Nella pagina Custom Resources è possibile configurare le risorse JNDI personalizzate.

Per ogni risorsa personalizzata vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa personalizzata.

Enabled

Opzione attivata se la risorsa personalizzata è abilitata e disattivata se la risorsa personalizzata è disabilitata.

Resource Type

Tipo di risorsa personalizzata.

Description

Descrizione della risorsa personalizzata.

La tabella Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa personalizzata.

Delete

Pulsante che consente di eliminare una o più risorse personalizzate selezionate.

Enable

Pulsante che consente di abilitare una o più risorse personalizzate selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse personalizzate selezionate.
