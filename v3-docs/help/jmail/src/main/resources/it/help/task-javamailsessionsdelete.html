Eliminare una sessione JavaMail

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, selezionare Sessioni JavaMail.

    Viene visualizzata la pagina Sessioni JavaMail.

  3. Nella pagina Sessioni JavaMail, selezionare la casella di controllo in corrispondenza del nome di sessione da eliminare.

  4. Fare clic su Elimina.
