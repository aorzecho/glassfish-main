Eliminare una sessione JavaMail

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, selezionare JavaMail Sessions.

    Viene visualizzata la pagina JavaMail Sessions.

  3. Nella pagina JavaMail Sessions, selezionare la casella di controllo in corrispondenza del nome di sessione da eliminare.
  4. Fare clic su Delete.
