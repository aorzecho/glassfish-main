Modificare una risorsa JNDI personalizzata

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JNDI.
  3. Nel nodo JNDI, selezionare Custom Resources.

    Viene visualizzata la pagina Custom Resources.

  4. Nella pagina Custom Resources, fare clic sul nome della risorsa che verrà modificata.

    Viene aperta la pagina New Custom Resource.

    Nota - Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa personalizzata.

  5. Per modificare il valore relativo a Resource Type, procedere in uno dei seguenti modi.
    • Dall'elenco a discesa, selezionare una definizione di tipo.
    • Selezionare il secondo pulsate di scelta e digitare una definizione di tipo completa, con il formato xxx. xxx.
  6. Nel campo Factory Class, digitare un nome di classe di factory per la risorsa personalizzata.

    Factory Class è il nome specificato dall'utente per la classe di factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory .

  7. Nel campo Description, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  8. Selezionare la casella di controllo Status Enabled per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  9. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa personalizzata.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  10. Fare clic su Save.
