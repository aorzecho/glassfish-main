Modificare una risorsa JNDI personalizzata

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JNDI.

  3. Nel nodo JNDI, selezionare Risorse personalizzate.

    Viene visualizzata la pagina Risorse personalizzate.

  4. Nella pagina Risorse personalizzate, fare clic sul nome della risorsa che verrà modificata.

    Viene aperta la pagina Modifica risorsa personalizzata.

    Nota:

    Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa personalizzata.

  5. Per modificare il valore relativo a Tipo di risorsa, procedere in uno dei seguenti modi.

    • Dall'elenco a discesa, selezionare una definizione di tipo.

    • Selezionare il secondo pulsante di scelta e digitare una definizione di tipo completamente qualificata, nel formato . xxx.

  6. Nel campo Classe factory, digitare un nome di classe di factory per la risorsa personalizzata.

    Classe factory è il nome specificato dall'utente per la classe di factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

  7. Nel campo Descrizione, digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa e non può superare i 250 caratteri di lunghezza.

  8. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ossia disponibile dal momento in cui viene creata.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa personalizzata.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse personalizzate.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.
