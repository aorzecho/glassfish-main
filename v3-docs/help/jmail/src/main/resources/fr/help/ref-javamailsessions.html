Sessions JavaMail

GlassFish Server inclut l’API JavaMail. JavaMail permet, à partir d’applications Java, d’accéder à des serveurs de courrier IMAP (Internet Message Access Protocol) et SMTP (Simple Mail Transfer Protocol) via votre réseau ou Internet.

Utilisez la page Sessions JavaMail pour configurer les sessions JavaMail.

Pour chaque session JavaMail, les informations suivantes sont fournies :

Nom JNDI

Nom unique identifiant la session JavaMail.

Activée

True si la session est activée ou False si la session est désactivée.

Description

Description de la session JavaMail.

Le tableau Sessions contient également les options suivantes :

Nouveau

Bouton permettant de créer une nouvelle session JavaMail.

Supprimer

Bouton permettant de supprimer une ou plusieurs sessions JavaMail sélectionnées.

Activer

Bouton permettant d’activer une ou plusieurs sessions JavaMail sélectionnées.

Désactiver

Bouton permettant de désactiver une ou plusieurs sessions JavaMail sélectionnées.
