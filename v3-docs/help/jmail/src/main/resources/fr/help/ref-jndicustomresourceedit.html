Modification des ressources personnalisées

La page Modification des ressources personnalisées permet de modifier une ressource JNDI personnalisée.

La page Modification des ressources personnalisées contient les options suivantes :

Nom JNDI

Nom unique identifiant la ressource. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu’à la création d’une ressource personnalisée.

Type de ressource

Définition complète du type de ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l’utilisateur. Cette classe implémente l’interface javax.naming.spi.ObjectFactory.

Description

Description de cette ressource personnalisée.

Statut

Statut de la ressource personnalisée. La ressource personnalisée peut être activée ou désactivée. La valeur par défaut est Activée.

Autres propriétés

Propriétés supplémentaires de la ressource personnalisée. GlassFish Server ne définit aucune autre propriété pour les ressources personnalisées.
