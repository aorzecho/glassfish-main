Supprimer une ressource JNDI externe

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JNDI.

  3. Sous le noeud JNDI, sélectionnez Ressources externes.

    La page Ressources externes apparaît.

  4. Dans la table Ressources, sélectionnez la case située en regard du nom de la ressource à supprimer.

  5. Cliquez sur Supprimer.
