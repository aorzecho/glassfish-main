Nouvelle ressource externe

La page Nouvelle ressource externe permet de créer une ressource JNDI externe.

La page Nouvelle ressource externe contient les options suivantes :

Nom JNDI

Nom unique identifiant la ressource.

Type de ressource

Définition complète du type de ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l’utilisateur. Cette classe implémente l’interface javax.naming.spi.InitialContextFactory.

Recherche JNDI

Valeur JNDI à rechercher dans le référentiel externe.

Par exemple, si vous créez une ressource externe pour vous connecter à un référentiel externe, pour tester une classe bean, la recherche JNDI peut prendre la forme suivante : cn=testbean.

Description

Description de cette ressource externe.

Statut

Statut de la ressource externe. La ressource externe peut être activée ou désactivée. La valeur par défaut est Activée.

Autres propriétés

Propriétés supplémentaires de la ressource externe. GlassFish Server ne définit aucune autre propriété pour les ressources externes.

Cibles

Clusters et instances autonomes pour la ressource externe. Déplacez les cibles voulues dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout Ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Supprimer et Tout Supprimer.

Cette option s'affiche uniquement si les clusters ou les instances autonomes ont été créés dans le domaine.
