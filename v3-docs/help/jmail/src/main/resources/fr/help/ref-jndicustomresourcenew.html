Nouvelle ressource personnalisée

La page Nouvelle ressource personnalisée permet de créer une ressource JNDI personnalisée.

Elle contient les options suivantes.

Nom JNDI

Nom unique identifiant la ressource.

Type de ressource

Définition complète du type de ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l'utilisateur. Cette classe implémente l'interface javax.naming.spi.ObjectFactory.

Description

Description de cette ressource personnalisée.

Statut

Statut de la ressource personnalisée. La ressource personnalisée peut être activée ou désactivée. La valeur par défaut est Activé.

Autres propriétés

Propriétés supplémentaires de la ressource personnalisée. GlassFish Server ne définit aucune autre propriété pour les ressources personnalisées.

Cibles

Clusters et instances autonomes de la ressource personnalisée. Déplacez les cibles souhaitées dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Enlever et Tout enlever.

Cette option apparaît uniquement si des clusters ou des instances autonomes ont été créés dans le domaine.
