Nouvelle ressource personnalisée

La page Nouvelle ressource personnalisée permet de créer une ressource JNDI personnalisée.

La page Nouvelle ressource personnalisée contient les options suivantes :

Nom JNDI

Nom unique identifiant la ressource.

Type de ressource

Définition complète du type de ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l’utilisateur. Cette classe implémente l’interface javax.naming.spi.ObjectFactory.

Description

Description de cette ressource personnalisée.

Statut

Statut de la ressource personnalisée. La ressource personnalisée peut être activée ou désactivée. La valeur par défaut est Activée.

Autres propriétés

Propriétés supplémentaires de la ressource personnalisée. GlassFish Server ne définit aucune autre propriété pour les ressources personnalisées.

Cibles

Clusters et instances autonomes pour la ressource personnalisée. Déplacez les cibles voulues dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout Ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Supprimer et Tout Supprimer.

Cette option s'affiche uniquement si les clusters ou les instances autonomes ont été créés dans le domaine.
