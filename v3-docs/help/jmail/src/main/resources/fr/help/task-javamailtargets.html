Pour gérer les cibles de session JavaMail

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, sélectionnez Sessions JavaMail.

    La page Sessions JavaMail s’affiche.

  3. Dans la table Sessions, cliquez sur le nom de session dont vous voulez gérer les cibles.

    La page Modification d’une session JavaMail s’affiche.

  4. Cliquez sur l’onglet Cible.

    La page Cibles de sessions JavaMail s'ouvre.

  5. Dans la table Cibles, sélectionnez l'action que vous souhaitez effectuer :
    • Pour ajouter ou supprimer des cibles, cliquez sur Gestion des cibles.
    • Pour activer une cible, cochez la case située en regard de l'instance de serveur ou du cluster de votre choix et cliquez sur Activer.
    • Pour désactiver une cible, cochez la case située en regard de l'instance de serveur ou du cluster de votre choix et cliquez sur Désactiver.
