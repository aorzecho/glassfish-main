Créer une ressource JNDI externe

Une ressource JNDI externe accède à un référentiel JNDI externe, tel un serveur LDAP.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JNDI.

  3. Sous le noeud JNDI, sélectionnez Ressources externes.

    La page Ressources externes apparaît.

  4. Sur la page Ressources externes, cliquez sur Nouveau.

    La page Nouvelle ressource externe apparaît.

  5. Dans le champ Nom JNDI, entrez un nom unique.

    Il sera inscrit auprès du service de dénomination JNDI. Le nom JNDI organise et localise les composants dans un environnement informatique distribué de la même manière que des catalogues sur fiches organisent et représentent des emplacements de livres dans une bibliothèque. Par conséquent, le nom JNDI est une méthode importante d'accès à la ressource

  6. Dans le champ Type de ressource, effectuez l'une des opérations suivantes.

    • Sélectionnez une définition de type dans la liste déroulante.

    • Sélectionnez le deuxième bouton radio et entrez une définition de type qualifiée complète au format suivant : xxx.xxx.

  7. Dans le champ Classe de fabrique, entrez un nom de référentiel externe de classe de fabrique JNDI pour la ressource externe à créer.

    La classe de fabrique correspond au nom spécifié par l'utilisateur pour la classe de fabrique, tel que com.sun.jndi.ldap. Cette classe implémente l'interface javax.naming.spi.InitialContextFactory.

  8. Dans le champ Recherche JNDI, entrez la valeur JNDI à rechercher dans le référentiel externe.

    Par exemple, si vous créez une ressource externe pour vous connecter à un référentiel externe, pour tester une classe de bean, la recherche JNDI peut prendre la forme suivante : cn=testbean.

  9. Dans le champ Description, entrez une brève description de la ressource.

    Cette description doit se présenter sous la forme d'une chaîne pouvant inclure un maximum de 250 caractères.

  10. Cochez la case Activé du champ Statut pour activer la ressource.

    Par défaut, la ressource est activée, ce qui signifie qu'elle est disponible dès sa création.

  11. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires de la ressource externe.

    GlassFish Server ne définit aucune autre propriété pour les ressources externes.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  12. Cliquez sur OK.
