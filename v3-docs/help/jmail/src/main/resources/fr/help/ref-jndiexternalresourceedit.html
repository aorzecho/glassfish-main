Modifier une ressource externe

La page Modifier une ressource externe permet de modifier une ressource JNDI externe.

Elle contient les options suivantes.

Nom JNDI

Nom unique identifiant la ressource. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource externe.

Type de ressource

Définition complète du type de ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l'utilisateur. Cette classe implémente l'interface javax.naming.spi.InitialContextFactory.

Recherche JNDI

Valeur JNDI à rechercher dans le référentiel externe.

Par exemple, si vous créez une ressource externe pour vous connecter à un référentiel externe, pour tester une classe de bean, la recherche JNDI peut prendre la forme suivante : cn=testbean.

Description

Description de cette ressource externe.

Statut

Statut de la ressource externe. La ressource externe peut être activée ou désactivée. La valeur par défaut est Activé.

Autres propriétés

Propriétés supplémentaires de la ressource externe. GlassFish Server ne définit aucune autre propriété pour les ressources externes.
