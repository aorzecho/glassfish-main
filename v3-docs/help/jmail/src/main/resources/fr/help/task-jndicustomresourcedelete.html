Suppression d’une ressource JNDI personnalisée

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, développez le nœud JNDI.
  3. Sous le nœud JNDI, sélectionnez Ressources personnalisées.

    La page Ressources personnalisées s’affiche.

  4. Sur la page Ressources personnalisées, cochez la case située en regard du nom de la ressource à supprimer.
  5. Cliquez sur Supprimer.
