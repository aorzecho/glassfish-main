Ressources externes

Une ressource JNDI externe accède à un référentiel JNDI externe, tel un serveur LDAP.

Utilisez la page Ressources externes pour configurer les ressources JNDI externes.

Pour chaque ressource externe, les informations suivantes sont fournies :

Nom JNDI

Nom unique identifiant la ressource externe.

Activée

True si la ressource externe est activée ou False si la ressource externe est désactivée.

Type de ressource

Type de ressource externe.

Description

Description de la ressource externe.

Le tableau Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource externe.

Supprimer

Bouton permettant de supprimer une ou plusieurs ressources externes sélectionnées.

Activer

Bouton permettant d’activer une ou plusieurs ressources externes sélectionnées.

Désactiver

Bouton permettant de désactiver une ou plusieurs ressources externes sélectionnées.
