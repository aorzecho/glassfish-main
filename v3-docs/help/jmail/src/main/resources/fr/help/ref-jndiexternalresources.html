Ressources externes

Une ressource JNDI externe accède à un référentiel JNDI externe, tel un serveur LDAP.

La page Ressources externes permet de configurer les ressources JNDI externes.

Pour chaque ressource externe, les informations suivantes sont fournies :

Nom JNDI

Nom unique identifiant la ressource externe.

Activé

True si la ressource externe est activée ou False si la ressource externe est désactivée.

Type de ressource

Type de la ressource externe.

Description

Description de la ressource externe.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource externe.

Supprimer

Bouton permettant de supprimer les ressources externes sélectionnées.

Activer

Bouton permettant d'activer les ressources externes sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources externes sélectionnées.
