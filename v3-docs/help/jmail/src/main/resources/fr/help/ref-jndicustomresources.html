Ressources personnalisées

Une ressource personnalisée spécifie une fabrique personnalisée d’objets de ressource au niveau du serveur.

Utilisez la page Ressources personnalisées pour configurer les ressources JNDI.

Pour chaque ressource personnalisée, les informations suivantes sont fournies :

Nom JNDI

Nom unique identifiant la ressource personnalisée.

Activée

True si la ressource personnalisée est activée ou False si la ressource personnalisée est désactivée.

Type de ressource

Type de ressource personnalisée.

Description

Description de la ressource personnalisée.

Le tableau Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource personnalisée.

Supprimer

Bouton permettant de supprimer une ou plusieurs ressources personnalisées sélectionnées.

Activer

Bouton permettant d’activer une ou plusieurs ressources personnalisées sélectionnées.

Désactiver

Bouton permettant de désactiver une ou plusieurs ressources personnalisées sélectionnées.

  • Suppression d'une ressource JNDI personnalisée
