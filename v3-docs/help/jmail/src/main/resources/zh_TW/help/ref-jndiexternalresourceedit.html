編輯外部資源

使用 [編輯外部資源] 頁面可修改外部 JNDI 資源。

[編輯外部資源] 頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。[JNDI 名稱] 是唯讀欄位。您僅能在建立新的外部資源時指定 JNDI 名稱。

資源類型

完全合格的資源類型定義。

工廠類別

使用者指定的工廠類別名稱。此類別會實作 javax.naming.spi.InitialContextFactory 介面。

JNDI 查找

要在外部儲存庫中查找的 JNDI 值。

例如，建立連線至外部儲存庫的外部資源之後，為了測試 Bean 類別，可進行如下 JNDI 查找︰cn=testmybean。

說明

此外部資源的說明。

狀態

外部資源的狀態。您可啟用或停用外部資源。預設值為 [啟用]。

其他特性

外部資源的其他特性。GlassFish Server 未定義外部資源的任何其他特性。
