編輯自訂的 JNDI 資源

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 在 [資源] 節點下，展開 [JNDI] 節點。
  3. 在 [JNDI] 節點下，選取 [自訂資源]。

    [自訂資源] 頁面會隨即開啟。

  4. 在 [自訂資源] 頁面中，按一下要編輯的資源名稱。

    [編輯自訂資源] 頁面會隨即開啟。

    備註 - [JNDI 名稱] 是唯讀欄位。您僅能在建立新的自訂資源時指定 JNDI 名稱。

  5. 要修改資源類型，請執行以下作業之一。
    • 從下拉式清單中，選取類型定義。
    • 選取第二個單選按鈕，並以 xxx.xxx 格式鍵入完全合格的類型定義。
  6. 在 [工廠類別] 欄位中，鍵入自訂資源的工廠類別名稱。

    [工廠類別] 是使用者為工廠類別指定的名稱。該類別實作 javax.naming.spi.ObjectFactory 介面。

  7. 在 [說明] 欄位中，鍵入資源的簡短說明。

    該說明是一個字串值，最多可以包含 250 個字元。

  8. 選取 [啟用狀態] 核取方塊，以啟用資源。

    依預設會啟用資源，表示資源於建立後就可以使用。

  9. (可選擇) 在 [其他特性] 區段中，指定自訂資源的其他特性。

    GlassFish Server 未定義自訂資源的任何其他特性。

    • 若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。
    • 若要修改特性，請編輯特性的 [值] 欄位。
    • 若要刪除特性，請選取您要刪除之特性 [名稱] 欄位左側的核取方塊，然後按一下 [刪除特性] 按鈕。
  10. 按一下 [儲存]。
