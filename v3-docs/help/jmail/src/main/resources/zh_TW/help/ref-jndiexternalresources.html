外部資源

外部 JNDI 資源會存取外部 JNDI 儲存庫 (例如 LDAP 伺服器)。

使用 「外部資源」 頁面可組態外部 JNDI 資源。

對於每個外部資源，都將提供以下資訊。

JNDI 名稱

可識別外部資源的唯一名稱。

已啟用

如果外部資源已啟用，則為 True；如果外部資源已停用，則為 False。

資源類型

外部資源的類型。

說明

外部資源的說明。

資源表也包含下列選項。

新建

建立新外部資源的按鈕。

刪除

刪除一或多個所選取外部資源的按鈕。

啟用

啟用一或多個所選取外部資源的按鈕。

停用

停用一或多個所選取外部資源的按鈕。
