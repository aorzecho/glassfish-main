自訂資源

自訂資源會指定整個伺服器範圍的自訂資源物件工廠。

使用 [自訂資源] 頁面可配置自訂 JNDI 資源。

對於每個自訂資源，都將提供以下資訊。

JNDI 名稱

可識別自訂資源的唯一名稱。

已啟用

如果自訂資源已啟用，則為 True；如果自訂資源已停用，則為 False。

資源類型

自訂資源的類型。

說明

自訂資源的說明。

資源表也包含下列選項。

新建

建立新自訂資源的按鈕。

刪除

刪除一或多個所選取自訂資源的按鈕。

啟用

啟用一或多個所選取自訂資源的按鈕。

停用

停用一或多個所選取自訂資源的按鈕。
