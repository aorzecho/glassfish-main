刪除外部 JNDI 資源

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 在 [資源] 節點下，展開 [JNDI] 節點。
  3. 在 [JNDI] 節點下，選取 [外部資源]。

    [外部資源] 頁面會隨即開啟。

  4. 在資源表上，選取要刪除的資源名稱旁邊的核取方塊。
  5. 按一下 [刪除]。
