新建自訂資源

使用 「新建自訂資源」 頁面可建立自訂 JNDI 資源。

「新建自訂資源」 頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。

資源類型

完全合格的資源類型定義。

工廠類別

使用者指定的工廠類別名稱。該類別實作 javax.naming.spi.ObjectFactory 介面。

說明

此自訂資源的說明。

狀態

自訂資源的狀態。您可啟用或停用自訂資源。預設值為 「啟用」。

其他特性

自訂資源的其他特性。GlassFish Server 未定義自訂資源的任何其他特性。

目標

自訂資源的叢集與獨立實例。可使用 「加入」 與 「全部加入」 按鈕，將所需的目標移至 「選取的目標」 欄。可使用 「移除」 與 「全部移除」 按鈕，將任何不需要的目標移至 「可用的目標」 欄。

唯有網域中已建立叢集或獨立實例時，才會顯示此選項。
