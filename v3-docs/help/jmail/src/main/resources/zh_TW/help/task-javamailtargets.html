管理 JavaMail 階段作業目標

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 在 [資源] 節點下，選取 [JavaMail 階段作業]。

    [JavaMail 階段作業] 頁面會隨即開啟。

  3. 在 [階段作業] 表格上，按一下要管理其目標的階段作業名稱。

    [編輯 JavaMail 階段作業] 頁面會隨即開啟。

  4. 按一下 [目標] 標籤。

    [JavaMail 階段作業目標] 頁面會隨即開啟。

  5. 在 [目標] 表格中，選擇要採取的動作：
    • 若要增加或移除目標，請按一下 [管理目標]。
    • 若要啟用目標，請按一下所需伺服器實例或叢集旁邊的核取方塊，然後按一下 [啟用]。
    • 若要停用目標，請按一下所需伺服器實例或叢集旁邊的核取方塊，然後按一下 [停用]。
