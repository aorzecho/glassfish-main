外部资源

外部 JNDI 资源可以访问外部 JNDI 系统信息库，如 LDAP 服务器。

使用“外部资源”页面可以配置外部 JNDI 资源。

对于每个外部资源，将提供以下信息。

JNDI 名称

标识外部资源的唯一名称。

已启用

如果外部资源已启用，则为 true；如果外部资源已禁用，则为 false。

资源类型

外部资源的类型。

说明

外部资源的说明。

“资源”表还包含以下选项。

新建

用于创建新外部资源的按钮。

删除

用于删除一个或多个选定外部资源的按钮。

启用

用于启用一个或多个选定外部资源的按钮。

禁用

用于禁用一个或多个选定外部资源的按钮。
