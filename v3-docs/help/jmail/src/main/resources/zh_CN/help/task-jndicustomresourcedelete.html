删除自定义 JNDI 资源

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开 "JNDI" 节点。
  3. 在 "JNDI" 节点下，选择“自定义资源”。

    此时将打开“自定义资源”页面。

  4. 在“自定义资源”页面上，选中您要删除的资源名称旁边的复选框。
  5. 单击“删除”。
