管理 JavaMail 会话目标

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，选择“JavaMail 会话”。

    此时将打开“JavaMail 会话”页面。

  3. 在“会话”表上，单击要管理其目标的会话名称。

    此时将打开“编辑 JavaMail 会话”页面。

  4. 单击“目标”选项卡。

    此时将打开“JavaMail 会话目标”页面。

  5. 在“目标”表上，选择要采取的操作：
    • 要添加或删除目标，请单击“管理目标”。
    • 要启用目标，请单击所需服务器实例或群集旁的复选框，并单击“启用”。
    • 要禁用目标，请单击所需服务器实例或群集旁的复选框，并单击“禁用”。
