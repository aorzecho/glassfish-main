编辑定制 JNDI 资源

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JNDI”节点。

  3. 在“JNDI”节点下，选择“定制资源”。

    此时将打开“定制资源”页。

  4. 在“定制资源”页上，单击您要编辑的资源的名称。

    此时将打开“编辑定制资源”页。

    注:

    JNDI 名称是只读字段。只能在创建新的定制资源时指定 JNDI 名称。

  5. 要修改“资源类型”，请执行以下操作之一。

    • 从下拉列表中，选择类型定义。

    • 选择第二个单选按钮并键入 xxx.xxx 格式的全限定类型定义。

  6. 在“工厂类”字段中，键入定制资源的工厂类名。

    该字段中的值是用户指定的工厂类的名称。此类实现了 javax.naming.spi.ObjectFactory 接口。

  7. 在“说明”字段中，键入资源的简短说明。

    此描述是字符串值，最多可以包含 250 个字符。

  8. 选择“启用状态”复选框以启用资源。

    默认情况下，该资源已启用，也就是说该资源一经创建即可使用。

  9. 在“其他属性”部分中，指定定制资源的其他属性。

    GlassFish Server 不定义定制资源的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  10. 单击“保存”。
