创建自定义 JNDI 资源

自定义 JNDI 资源可以指定自定义服务器范围资源对象工厂。

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开 "JNDI" 节点。
  3. 在 "JNDI" 节点下，选择“自定义资源”。

    此时将打开“自定义资源”页面。

  4. 在“自定义资源”页面上，单击“新建”。

    此时将打开“新建自定义资源”页面。

  5. 在“JNDI 名称”字段中，键入一个唯一的名称。

    此名称将被注册到 JNDI 命名服务中。JNDI 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，JNDI 名称是访问自定义资源的重要方法之一。

  6. 在“资源类型”字段中，执行以下操作之一。
    • 从下拉式列表中，选择类型定义。
    • 选择第二个单选按钮并键入 xxx.xxx 格式的全限定类型定义。
  7. 在“工厂类”字段中，为要创建的自定义资源键入工厂类名。

    该字段中的值是用户指定的工厂类的名称。此类实现了 javax.naming.spi.ObjectFactory 接口。

  8. 在“说明”字段中，键入资源的简短说明。

    此描述是字符串值，最多可以包含 250 个字符。

  9. 选中“启用状态”复选框以启用资源。

    默认情况下，该资源已启用，也就是说该资源一经创建即可使用。

  10. （可选的）在“其他属性”部分中，指定自定义资源的其他属性。

    GlassFish Server 不定义自定义资源的任何其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  11. 单击“确定”。
