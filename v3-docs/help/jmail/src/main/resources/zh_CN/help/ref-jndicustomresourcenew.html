新建定制资源

使用“新建定制资源”页可以创建定制 JNDI 资源。

“新建定制资源”页包含以下选项：

JNDI 名称

标识资源的唯一名称。

资源类型

资源的全限定类型定义。

工厂类

用户指定的工厂类的名称。此类实现了 javax.naming.spi.ObjectFactory 接口。

说明

此定制资源的说明。

状态

定制资源的状态。可以启用或禁用定制资源。默认值为“已启用”。

其他属性

定制资源的其他属性。GlassFish Server 不定义定制资源的任何其他属性。

目标

定制资源的集群和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“所选目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。

仅当在域中已创建集群或独立实例时才会显示此选项。
