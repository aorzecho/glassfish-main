新建外部资源

使用“新建外部资源”页面可以创建外部 JNDI 资源。

“新建外部资源”页面包含以下选项。

JNDI 名称

标识资源的唯一名称。

资源类型

资源的全限定类型定义。

工厂类

用户指定的工厂类的名称。此类实现了 javax.naming.spi.InitialContextFactory 接口。

JNDI 查找

要在外部系统信息库中查找的 JNDI 值。

例如，在创建一个与外部系统信息库连接的外部资源时，为了测试某个 Bean 类，“JNDI 查找”可能会如下所示：cn=testmybean。

说明

此外部资源的说明。

状态

外部资源的状态。可以启用或禁用外部资源。默认值为“已启用”。

其他属性

外部资源的其他属性。GlassFish Server 不定义外部资源的任何其他属性。

目标

外部资源的群集和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“选定目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。

仅当已在域中创建群集或独立实例时才会显示此选项。
