自定义资源

自定义资源可以指定自定义服务器范围资源对象工厂。

使用“自定义资源”页面可以配置自定义 JNDI 资源。

对于每个自定义资源，将提供以下信息。

JNDI 名称

标识自定义资源的唯一名称。

已启用

如果自定义资源已启用，则为 true；如果自定义资源已禁用，则为 false。

资源类型

自定义资源的类型。

说明

自定义资源的说明。

“资源”表还包含以下选项。

新建

用于创建新自定义资源的按钮。

删除

用于删除一个或多个选定自定义资源的按钮。

启用

用于启用一个或多个选定自定义资源的按钮。

禁用

用于禁用一个或多个选定自定义资源的按钮。
