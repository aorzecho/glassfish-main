사용자 정의 자원

사용자 정의 자원은 사용자 정의 서버 차원의 자원 객체 팩토리를 지정합니다.

사용자 정의 자원 페이지에서는 사용자 정의 JNDI 자원을 구성할 수 있습니다.

각 사용자 정의 자원에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

사용자 정의 자원을 식별하는 고유한 이름입니다.

사용 가능

사용자 정의 자원이 사용 가능한 경우 true, 사용 가능하지 않은 경우 false입니다.

자원 유형

사용자 정의 자원의 유형입니다.

설명

사용자 정의 자원에 대한 설명입니다.

자원 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 사용자 정의 자원을 만드는 버튼입니다.

삭제

하나 이상의 선택된 사용자 정의 자원을 삭제하는 버튼입니다.

활성화

하나 이상의 선택된 사용자 정의 자원을 활성화하는 버튼입니다.

비활성화

하나 이상의 선택된 사용자 정의 자원을 비활성화하는 버튼입니다.

관련 작업

  • 사용자 정의 JNDI 자원을 만드는 방법

  • 사용자 정의 JNDI 자원을 편집하는 방법

  • 사용자 정의 JNDI 자원을 삭제하는 방법

관련 asadmin 명령

  • create-custom-resource(1)

  • list-custom-resources(1)

  • delete-custom-resource(1)
