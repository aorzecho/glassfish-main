JavaMail 세션

GlassFish Server에는 JavaMail API가 포함되어 있습니다. JavaMail을 사용하면 Java 응용 프로그램이 네트워크 또는 인터넷상의 IMAP(Internet Message Access Protocol) 및 SMTP(Simple Mail Transfer Protocol) 가능 메일 서버에 액세스할 수 있습니다.

JavaMail 세션 페이지에서는 JavaMail 세션을 구성할 수 있습니다.

각 JavaMail 세션에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

JavaMail 세션을 식별하는 고유한 이름입니다.

사용 가능

세션이 사용 가능한 경우 true, 사용 가능하지 않은 경우 false입니다.

설명

JavaMail 세션에 대한 설명입니다.

세션 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 JavaMail 세션을 만드는 버튼입니다.

삭제

하나 이상의 선택된 JavaMail 세션을 삭제하는 버튼입니다.

활성화

하나 이상의 선택된 JavaMail 세션을 활성화하는 버튼입니다.

비활성화

하나 이상의 선택된 JavaMail 세션을 비활성화하는 버튼입니다.

관련 작업

  • JavaMail 세션을 만드는 방법

  • JavaMail 세션을 편집하는 방법

  • JavaMail 세션을 삭제하는 방법

  • list-javamail-resources(1)

  • delete-javamail-resource(1)
