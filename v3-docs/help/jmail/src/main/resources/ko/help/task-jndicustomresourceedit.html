사용자 정의 JNDI 자원을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드 아래에서 JNDI 노드를 확장합니다.
  3. JNDI 노드 아래에서 사용자 정의 자원을 선택합니다.

    사용자 정의 자원 페이지가 열립니다.

  4. 사용자 정의 자원 페이지에서 편집할 자원의 이름을 누릅니다.

    사용자 정의 자원 편집 페이지가 열립니다.

    주 - JNDI 이름은 읽기 전용 필드입니다. 새 사용자 정의 자원을 만들 때만 JNDI 이름을 지정할 수 있습니다.

  5. 자원 유형을 수정하려면 다음 중 하나를 수행합니다.
    • 드롭다운 목록에서 유형 정의를 선택합니다.
    • 두 번째 라디오 버튼을 선택하고 xxx.xxx 형식의 정규화된 유형 정의를 입력합니다.
  6. 팩토리 클래스 필드에 사용자 정의 자원에 대한 팩토리 클래스 이름을 입력합니다.

    팩토리 클래스는 팩토리 클래스에 대한 사용자 지정 이름입니다. 이 클래스는 javax.naming.spi.ObjectFactory 인터페이스를 구현합니다.

  7. 설명 필드에 자원에 대한 간단한 설명을 입력합니다.

    이 설명은 최대 250자가 포함될 수 있는 문자열 값입니다.

  8. 자원을 활성화하려면 상태 사용 가능 확인란을 선택합니다.

    기본적으로 자원은 활성화됩니다. 즉, 만드는 즉시 사용할 수 있습니다.

  9. (옵션) 추가 등록 정보 섹션에서 사용자 정의 자원에 대한 추가 등록 정보를 지정합니다.

    GlassFish 서버에서는 사용자 정의 자원에 대한 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  10. 저장을 누릅니다.
