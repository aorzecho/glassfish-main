외부 JNDI 리소스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드 아래에서 JNDI 노드를 확장합니다.

  3. JNDI 노드 아래에서 [외부 리소스]를 선택합니다.

    [외부 리소스] 페이지가 열립니다.

  4. [외부 리소스] 페이지에서 편집할 리소스의 이름을 누릅니다.

    외부 리소스 편집 페이지가 열립니다.

  5. 리소스 유형을 수정하려면 다음 중 하나를 수행합니다.

    • 드롭다운 목록에서 유형 정의를 선택합니다.

    • 두번째 라디오 단추를 선택하고 xxx.xxx 형식의 정규화된 유형 정의를 입력합니다.

  6. 팩토리 클래스 필드에 외부 리소스에 대한 JNDI 팩토리 클래스 외부 저장소를 입력합니다.

    팩토리 클래스는 팩토리 클래스에 대한 사용자 지정 이름(예: com.sun.jndi.ldap )입니다. 이 클래스는 javax.naming.spi.InitialContextFactory 인터페이스를 구현합니다.

  7. [JNDI 조회] 필드에 외부 저장소에서 조회할 JNDI 값을 입력합니다.

    예를 들어 Bean 클래스를 테스트하기 위해 외부 저장소에 접속할 외부 리소스를 생성할 때 JNDI 조회는 cn=testmyBean과 같을 수 있습니다.

  8. [설명] 필드에 리소스에 대한 간단한 설명을 입력합니다.

    이 설명은 최대 250자가 포함될 수 있는 문자열 값입니다.

  9. 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    기본적으로 리소스는 사용으로 설정되어 있습니다. 즉, 생성하는 즉시 사용할 수 있습니다.

  10. [추가 속성] 섹션에서 외부 리소스에 대한 추가 속성을 지정합니다.

    GlassFish Server에서는 외부 리소스에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  11. [저장]을 누릅니다.
