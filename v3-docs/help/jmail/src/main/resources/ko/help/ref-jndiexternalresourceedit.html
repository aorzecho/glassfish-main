외부 리소스 편집

외부 리소스 편집 페이지에서는 외부 JNDI 리소스를 수정할 수 있습니다.

외부 리소스 편집 페이지에는 다음 옵션이 포함되어 있습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다. JNDI 이름은 읽기 전용 필드입니다. 새 외부 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

리소스 유형

리소스에 대한 전체 유형 정의입니다.

팩토리 클래스

팩토리 클래스에 대한 사용자 지정 이름입니다. 이 클래스는 javax.naming.spi.InitialContextFactory 인터페이스를 구현합니다.

JNDI 조회

외부 저장소에서 조회할 JNDI 값입니다.

예를 들어 Bean 클래스를 테스트하기 위해 외부 저장소에 접속할 외부 리소스를 생성할 때 JNDI 조회는 cn=testmyBean과 같을 수 있습니다.

설명

이 외부 리소스에 대한 설명입니다.

상태

외부 리소스의 상태입니다. 외부 리소스는 사용 또는 사용 안함으로 설정할 수 있습니다. 기본값은 사용입니다.

추가 속성

외부 리소스에 대한 추가 속성입니다. GlassFish Server에서는 외부 리소스에 대한 추가 속성을 정의하지 않습니다.

관련 작업

  • 외부 JNDI 리소스를 생성하는 방법

  • 외부 JNDI 리소스를 편집하는 방법

  • 외부 JNDI 리소스를 삭제하는 방법

관련 asadmin 명령

  • create-jndi-resource(1)

  • list-jndi-resources(1)

  • delete-jndi-resource(1)
