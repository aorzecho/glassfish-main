JNDI 사용자 정의 리소스를 삭제하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드 아래에서 JNDI 노드를 확장합니다.

  3. JNDI 노드 아래에서 [사용자 정의 리소스]를 선택합니다.

    [사용자 정의 리소스] 페이지가 열립니다.

  4. [사용자 정의 리소스] 페이지에서 삭제하려는 리소스 이름 옆의 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
