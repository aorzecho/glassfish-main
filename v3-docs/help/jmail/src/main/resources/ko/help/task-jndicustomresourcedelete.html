사용자 정의 JNDI 자원을 삭제하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드 아래에서 JNDI 노드를 확장합니다.
  3. JNDI 노드 아래에서 사용자 정의 자원을 선택합니다.

    사용자 정의 자원 페이지가 열립니다.

  4. 사용자 정의 자원 페이지에서 삭제하려는 자원 이름 옆의 확인란을 선택합니다.
  5. 삭제를 누릅니다.
