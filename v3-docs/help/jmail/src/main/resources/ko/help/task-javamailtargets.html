JavaMail 세션 대상을 관리하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드 아래에서 [JavaMail 세션]을 선택합니다.

    [JavaMail 세션] 페이지가 열립니다.

  3. 세션 테이블에서 대상을 관리할 세션 이름을 누릅니다.

    [JavaMail 세션 편집] 페이지가 열립니다.

  4. [대상] 탭을 누릅니다.

    [JavaMail 세션 대상] 페이지가 열립니다.

  5. 대상 테이블에서 수행할 작업을 선택합니다.

    • 대상을 추가하거나 제거하려면 대상 관리를 누릅니다.

    • 대상을 사용으로 설정하려면 원하는 서버 인스턴스 또는 클러스터 옆의 체크 박스를 누르고 사용을 누릅니다.

    • 대상을 사용 안함으로 설정하려면 원하는 서버 인스턴스 또는 클러스터 옆의 체크 박스를 누르고 사용 안함을 누릅니다.
