Novo recurso externo

Utilize a página Novo recurso externo para criar um recurso JNDI externo.

A página Novo recurso externo contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso.

Tipo de recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de fábrica

O nome especificado de usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.InitialContextFactory.

Pesquisa JNDI

O valor de JNDI para pesquisar no repositório externo.

Por exemplo, ao criar um recurso externo para conectar-se a um repositório externo, para testar uma classe de bean a Pesquisa JNDI pode ter esse aspecto: cn=tesmybean.

Descrição

Uma descrição do recurso externo.

Status

O status do recurso externo. O recurso externo pode ser ativado ou desativado. O valor padrão é Ativo.

Propriedades adicionais

Propriedades adicionais para o recurso externo. O Servidor GlassFish não define quaisquer propriedades adicionais para recursos externos.

Destinos

Clusters e instâncias independentes do recurso externo. Mova os destinos desejados para a coluna Destinos selecionados utilizando os botões Adicionar e Adicionar tudo. Mova os destinos desnecessários para a coluna Destinos disponíveis utilizando os botões Remover e Remover tudo.

Esta opção é exibida somente se os clusters ou instâncias independentes tiverem sido criados no domínio.
