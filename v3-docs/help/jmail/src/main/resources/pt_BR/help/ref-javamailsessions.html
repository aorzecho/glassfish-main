Sessões de JavaMail

O GlassFish Server inclui a JavaMail API. JavaMail oferece acesso, a partir de aplicações Java, a servidores de correio capazes de Internet Message Access Protocol (IMAP) e Simple Mail Transfer Protocol (SMTP) em sua rede ou na Internet.

Utilize a página Sessões JavaMail para configurar sessões JavaMail.

Para cada usuário, as informações a seguir são fornecidas.

Nome JNDI

Um nome exclusivo que identifica a sessão JavaMail.

Ativado

Verdadeiro se a sessão estiver ativada ou falso se a sessão estiver desativada.

Descrição

Uma descrição da sessão JavaMail.

A tabela Sessões também contém as opções a seguir.

Novo

Botão para criar uma nova sessão JavaMail.

Deletar

Botão para deletar uma ou mais sessões JavaMail selecionadas.

Ativar

Botão para ativar uma ou mais sessões JavaMail selecionadas.

Desativar

Botão para desativar uma ou mais sessões JavaMail selecionadas.
