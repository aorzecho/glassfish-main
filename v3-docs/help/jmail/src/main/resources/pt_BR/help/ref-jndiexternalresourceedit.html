Editar Recurso Externo

Utilize a página Editar recurso externo para modificar um recurso JNDI externo.

A página Editar recurso externo contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso. O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI ao criar um novo recurso externo.

Tipo de recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de fábrica

O nome especificado de usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.InitialContextFactory.

Pesquisa JNDI

O valor de JNDI para pesquisar no repositório externo.

Por exemplo, ao criar um recurso externo para conectá-lo a um repositório externo, para testar uma classe de bean, a Pesquisa de JNDI terá a seguinte aparênciacn=testmybean.

Descrição

Uma descrição do recurso externo.

Status

O status do recurso externo. O recurso externo pode ser ativado ou desativado. O valor default é Ativo.

Propriedades adicionais

Propriedades adicionais para o recurso externo. O GlassFish Server não define quaisquer propriedades adicionais para recursos externos.
