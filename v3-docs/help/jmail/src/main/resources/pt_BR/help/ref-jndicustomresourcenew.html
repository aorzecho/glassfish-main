Novo Recurso Personalizado

Utilize a página Novo recurso personalizado para criar um recuso JNDI personalizado.

A página Novo recurso personalizado contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso.

Tipo de recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de fábrica

O nome especificado de usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.ObjectFactory.

Descrição

Uma descrição do recurso personalizado.

Status

O status do recurso personalizado. O recurso personalizado pode ser ativado ou desativado. O valor default é Ativo.

Propriedades adicionais

Propriedades adicionais para o recurso personalizado. O GlassFish Server não define quaisquer propriedades adicionais para recursos personalizados.

Destinos

Clusters e instâncias standalone do recurso personalizado. Mova os destinos desejados para a coluna Destinos selecionados utilizando os botões Adicionar e Adicionar tudo. Mova os destinos desnecessários para a coluna Destinos disponíveis utilizando os botões Remover e Remover tudo.

Esta opção é exibida somente se os clusters ou instâncias standalone tiverem sido criados no domínio.
