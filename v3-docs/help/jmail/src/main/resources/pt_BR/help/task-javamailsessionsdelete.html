Para excluir uma sessão JavaMail

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, selecione Sessões JavaMail.

    A página Sessões JavaMail é aberta.

  3. Na página Sessões JavaMail, marque a caixa de seleção ao lado do nome da sessão a ser excluída.
  4. Clique em Excluir.
