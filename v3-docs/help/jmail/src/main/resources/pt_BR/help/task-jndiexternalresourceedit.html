Para Editar um Recurso Externo da JNDI

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JNDI.

  3. No nó JNDI, selecione Recursos externos.

    A página Recursos externos é aberta.

  4. Na página Recursos externos, clique no nome do recurso a ser editado.

    A página Editar recurso externo é aberta.

  5. Para modificar o Tipo de recurso, efetue um dos procedimentos a seguir.

    • Na lista drop-down, selecione uma definição de tipo.

    • Selecione o segundo botão de rádio e digite uma definição de tipo totalmente qualificada que siga o formato xxx.xxx.

  6. No campo Classe de fábrica, digite um repositório externo de classe de fábrica JNDI para o recurso externo.

    A Classe de fábrica é o nome de usuário especificado para a classe de fábrica, como com.sun.jndi.ldap. Essa classe implementa a interface javax.naming.spi.InitialContextFactory.

  7. No campo Pesquisa JNDI, digite o valor de JNDI a ser pesquisado no repositório externo.

    Por exemplo, ao criar um recurso externo para conectá-lo a um repositório externo, para testar uma classe de bean, a Pesquisa de JNDI terá a seguinte aparênciacn=testmybean.

  8. No campo Descrição, digite uma descrição breve do recurso.

    Essa descrição é um valor de string e pode incluir um máximo de 250 caracteres.

  9. Marque a caixa de seleção Status ativo para ativar o recurso.

    Por default, o recurso está ativado, significando que está disponível assim que é criado.

  10. Na seção Propriedades adicionais, especifique propriedades adicionais para o recurso externo.

    O GlassFish Server não define propriedades adicionais para recursos externos.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  11. Clique em Salvar.
