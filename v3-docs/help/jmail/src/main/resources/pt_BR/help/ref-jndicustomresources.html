Recursos Personalizados

Um recurso personalizado especifica uma fábrica de objetos de recursos personalizados para todos os servidores.

Utilize a página Recursos personalizados para configurar recursos JNDI personalizados.

Para cada recurso personalizado, as informações a seguir são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso personalizado.

Ativado

Verdadeiro se o recurso personalizado estiver ativo ou falso se o recurso personalizado estiver desativado.

Tipo de recurso

O tipo do recurso personalizado.

Descrição

Uma descrição do recurso personalizado.

A tabela Recursos também contém as opções a seguir.

Novo

Botão para criar um novo recurso personalizado.

Deletar

Botão para deletar um ou mais recursos personalizados selecionados.

Ativar

Botão para ativar um ou mais recursos personalizados selecionados.

Desativar

Botão para desativar um ou mais recursos personalizados selecionados.
