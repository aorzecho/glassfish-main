新しい外部リソース

新しいJNDIリソースを作成するには、「新しい外部リソース」ページを使用します。

「新しい外部リソース」ページには、次のオプションが含まれます。

JNDI名

リソースを識別する一意の名前。

リソース・タイプ

リソースの完全修飾タイプ定義。

ファクトリ・クラス

ファクトリ・クラスのユーザー指定の名前。このクラスはjavax.naming.spi.InitialContextFactoryインタフェースを実装します。

JNDIルックアップ

外部リポジトリの検索に使用するJNDI値。

たとえば、外部リポジトリに接続してBeanクラスをテストする外部リソースを作成する場合、「JNDIルックアップ」はcn=testmybeanとなります。

説明

外部リソースの説明。

ステータス

外部リソースのステータス。外部リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

追加プロパティ

外部リソースの追加プロパティ。GlassFish Serverは外部リソースの追加プロパティを定義しません。

ターゲット

外部リソースのクラスタおよびスタンドアロン・インスタンス。必要なターゲットを「選択したターゲット」列に移動するには、「追加」ボタンまたは「すべて追加」ボタンを使用します。不要なターゲットを「使用可能なターゲット」列に移動するには、「削除」ボタンまたは「すべて削除」ボタンを使用します。

このオプションは、ドメイン内にクラスタまたはスタンドアロン・インスタンスが作成されている場合にのみ表示されます。
