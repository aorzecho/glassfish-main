JavaMail セッションターゲットを管理する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JavaMail セッション」を選択します。

    「JavaMail セッション」ページが開きます。

  3. 「セッション」の表で、ターゲットを管理するセッションの名前をクリックします。

    「JavaMail セッションを編集」ページが開きます。

  4. 「ターゲット」タブをクリックします。

    「JavaMail セッションターゲット」ページが開きます。

  5. 「ターゲット」の表で、必要なアクションを選択します。
    • ターゲットを追加または削除するには、「ターゲットを管理」をクリックします。
    • ターゲットを有効にするには、目的のサーバーインスタンスまたはクラスタの隣のチェックボックスをクリックし、「有効」をクリックします。
    • ターゲットを無効にするには、目的のサーバーインスタンスまたはクラスタの隣のチェックボックスをクリックし、「無効」をクリックします。
