新しいカスタムリソース

新しいカスタムリソースを作成するには、「新しいカスタムリソース」ページを使用します。

「新しいカスタムリソース」ページには、次のオプションが含まれます。

JNDI 名

リソースを識別する一意の名前。

リソースタイプ

リソースの完全修飾タイプ定義。

ファクトリクラス

ファクトリクラスのユーザー指定の名前。このクラスは javax.naming.spi.ObjectFactory インタフェースを実装する必要があります。

説明

カスタムリソースの説明。

ステータス

カスタムリソースのステータス。カスタムリソースは、有効または無効の状態になります。デフォルト値は「有効」です。

追加プロパティー

カスタムリソースの追加プロパティー。GlassFish Server はカスタムリソースの追加プロパティーを定義しません。

ターゲット

カスタムリソースのクラスタおよびスタンドアロンインスタンス。必要なターゲットを「選択したターゲット」列に移動するには、「追加」ボタンまたは「すべて追加」ボタンを使用します。不要なターゲットを「利用可能なターゲット」列に移動するには、「削除」ボタンまたは「すべて削除」ボタンを使用します。

このオプションは、ドメイン内にクラスタまたはスタンドアロンインスタンスが作成されている場合にのみ表示されます。
