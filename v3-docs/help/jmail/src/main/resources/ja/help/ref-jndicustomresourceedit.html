カスタムリソースを編集

カスタム JNDI リソースを編集するには、「カスタムリソースを編集」ページを使用します。

「カスタムリソースを編集」ページには、次のオプションが含まれます。

JNDI 名

リソースを識別する一意の名前。JNDI 名は読み取り専用フィールドです。JNDI 名を指定できるのは、新しいカスタムリソースの作成時のみです。

リソースタイプ

リソースの完全修飾タイプ定義。

ファクトリクラス

ファクトリクラスのユーザー指定の名前。このクラスは javax.naming.spi.ObjectFactory インタフェースを実装する必要があります。

説明

カスタムリソースの説明。

ステータス

カスタムリソースのステータス。カスタムリソースは、有効または無効の状態になります。デフォルト値は「有効」です。

追加プロパティー

カスタムリソースの追加プロパティー。GlassFish Server はカスタムリソースの追加プロパティーを定義しません。
