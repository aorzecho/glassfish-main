JavaMail セッション

GlassFish サーバー には JavaMail API が含まれています。JavaMail は Java アプリケーションから、ネットワークまたはインターネット上の IMAP (Internet Message Access Protocol) および SMTP (メール転送プロトコル) 対応のメールサーバーにアクセスできます。

JavaMail セッションを設定するには、「JavaMail セッション」ページを使用します。

JavaMail セッションごとに、次の情報が表示されます。

JNDI 名

JavaMail セッションを識別する一意の名前。

使用可能

セッションが有効な場合は true、無効な場合は false です。

説明

JavaMail セッションの説明。

「セッション」の表には、次のオプションも表示されます。

新規

新しい JavaMail セッションを作成するボタン。

削除

選択した 1 つまたは複数の JavaMail セッションを削除するボタン。

有効

選択した 1 つまたは複数の JavaMail セッションを有効にするボタン。

無効

選択した 1 つまたは複数の JavaMail セッションを無効にするボタン。

関連操作

  • 「JavaMail セッションを作成する」

  • 「JavaMail セッションを編集する」

  • 「JavaMail セッションを削除する」

  • list-javamail-resources(1)
