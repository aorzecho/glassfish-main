JavaMailセッション

GlassFish ServerにはJavaMail APIが含まれています。JavaMailはJavaアプリケーションから、ネットワークまたはインターネット上のIMAP(Internet Message Access Protocol)およびSMTP(メール転送プロトコル)対応のメール・サーバーにアクセスできます。

JavaMailセッションを構成するには、「JavaMailセッション」ページを使用します。

JavaMailセッションごとに、次の情報が表示されます。

JNDI名

JavaMailセッションを識別する一意の名前。

有効

セッションが有効な場合はtrue、無効な場合はfalseです。

説明

JavaMailセッションの説明。

「セッション」の表には、次のオプションも表示されます。

新規

新しいJavaMailセッションを作成するボタン。

削除

選択した1つまたは複数のJavaMailセッションを削除するボタン。

有効

選択した1つまたは複数のJavaMailセッションを有効にするボタン。

無効

選択した1つまたは複数のJavaMailセッションを無効にするボタン。
