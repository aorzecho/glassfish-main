外部リソース

外部JNDIリソースは、LDAPサーバーなどの外部JNDIリポジトリにアクセスします。

外部JNDIリソースを構成するには、「外部リソース」ページを使用します。

外部リソースごとに、次の情報が表示されます。

JNDI名

外部リソースを識別する一意の名前。

有効

外部リソースが有効な場合はtrue、無効の場合はfalseです。

リソース・タイプ

外部リソースのタイプ。

説明

外部リソースの説明。

「リソース」の表には、次のオプションも表示されます。

新規

新しい外部リソースを作成するボタン。

削除

選択した1つまたは複数の外部リソースを削除するボタン。

有効

選択した1つまたは複数の外部リソースを有効にするボタン。

無効

選択した1つまたは複数の外部リソースを無効にするボタン。
