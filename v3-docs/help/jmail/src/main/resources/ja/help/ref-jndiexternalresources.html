外部リソース

外部 JNDI リソースは、LDAP サーバーなどの外部 JNDI リポジトリにアクセスします。

外部 JNDI リソースを設定するには、「外部リソース」ページを使用します。

外部リソースごとに、次の情報が表示されます。

JNDI 名

外部リソースを識別する一意の名前。

使用可能

外部リソースが有効な場合は true、無効の場合は false です。

リソースタイプ

外部リソースのタイプ。

説明

外部リソースの説明。

「リソース」の表には、次のオプションも表示されます。

新規

新しい外部リソースを作成するボタン。

削除

選択した 1 つまたは複数の外部リソースを削除するボタン。

有効

選択した 1 つまたは複数の外部リソースを有効にするボタン。

無効

選択した 1 つまたは複数の外部リソースを無効にするボタン。

関連操作

  • 「外部 JNDI リソースを作成する」

  • 「外部 JNDI リソースを編集する」

  • 「外部 JNDI リソースを削除する」

  • create-jndi-resource(1)

  • list-jndi-resources(1)
