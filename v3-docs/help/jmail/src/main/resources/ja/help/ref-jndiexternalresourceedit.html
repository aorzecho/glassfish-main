外部リソースを編集

外部JNDIリソースを変更するには、「外部リソースを編集」ページを使用します。

「外部リソースを編集」ページには、次のオプションが含まれます。

JNDI名

リソースを識別する一意の名前。JNDI名は読取り専用フィールドです。JNDI名を指定できるのは、新しい外部リソースの作成時のみです。

リソース・タイプ

リソースの完全修飾タイプ定義。

ファクトリ・クラス

ファクトリ・クラスのユーザー指定の名前。このクラスはjavax.naming.spi.InitialContextFactoryインタフェースを実装します。

JNDIルックアップ

外部リポジトリの検索に使用するJNDI値。

たとえば、外部リポジトリに接続してBeanクラスをテストする外部リソースを作成する場合、「JNDIルックアップ」はcn=testmybeanとなります。

説明

外部リソースの説明。

ステータス

外部リソースのステータス。外部リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

追加プロパティ

外部リソースの追加プロパティ。GlassFish Serverは外部リソースの追加プロパティを定義しません。
