カスタムリソース

カスタムリソースは、サーバー全体のカスタムリソースオブジェクトファクトリを指定します。

カスタム JNDI リソースを設定するには、「カスタムリソース」ページを使用します。

カスタムリソースごとに、次の情報が表示されます。

JNDI 名

カスタムリソースを識別する一意の名前。

使用可能

カスタムリソースが有効な場合は true、無効な場合は false です。

リソースタイプ

カスタムリソースのタイプ。

説明

カスタムリソースの説明。

「リソース」の表には、次のオプションも表示されます。

新規

新しいカスタムリソースを作成するボタン。

削除

選択した 1 つまたは複数のカスタムリソースを削除するボタン。

有効

選択した 1 つまたは複数のカスタムリソースを有効にするボタン。

無効

選択した 1 つまたは複数のカスタムリソースを無効にするボタン。

関連操作

  • 「カスタム JNDI リソースを作成する」

  • 「カスタム JNDI リソースを編集する」

  • 「カスタム JNDI リソースを削除する」
