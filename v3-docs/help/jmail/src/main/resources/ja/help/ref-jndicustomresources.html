カスタム・リソース

カスタム・リソースは、サーバー全体のカスタム・リソース・オブジェクト・ファクトリを指定します。

カスタムJNDIリソースを構成するには、「カスタム・リソース」ページを使用します。

カスタム・リソースごとに、次の情報が表示されます。

JNDI名

カスタム・リソースを識別する一意の名前。

有効

カスタム・リソースが有効な場合はtrue、無効な場合はfalseです。

リソース・タイプ

カスタム・リソースのタイプ。

説明

カスタム・リソースの説明。

「リソース」の表には、次のオプションも表示されます。

新規

新しいカスタム・リソースを作成するボタン。

削除

選択した1つまたは複数のカスタム・リソースを削除するボタン。

有効

選択した1つまたは複数のカスタム・リソースを有効にするボタン。

無効

選択した1つまたは複数のカスタム・リソースを無効にするボタン。
