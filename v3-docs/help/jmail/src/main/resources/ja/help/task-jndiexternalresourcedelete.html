外部JNDIリソースを削除する

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「JNDI」ノードを展開します。

  3. 「JNDI」ノードで、「外部リソース」を選択します。

    「外部リソース」ページが開きます。

  4. 「リソース」の表で、削除するリソースの名前の隣にあるチェックボックスを選択します。

  5. 「削除」をクリックします。
