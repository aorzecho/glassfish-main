JavaMail セッションを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JavaMail セッション」を選択します。

    「JavaMail セッション」ページが開きます。

  3. 「JavaMail セッション」ページで、削除するセッションの名前の隣にあるチェックボックスを選択します。
  4. 「削除」をクリックします。
