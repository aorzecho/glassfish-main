事务服务

使用“事务服务”页面可以配置事务恢复、超时和日志记录。

“事务服务”页面包含以下选项：

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

重新启动时

如果选中“已启用”复选框，GlassFish Server 将在服务器重新启动时尝试恢复未完成的事务。默认情况下禁用此选项。

事务超时

回滚未完成事务之前，服务器等待的秒数。默认值为 0，表示服务器无限期地等待事务完成。

重试超时

GlassFish Server 尝试连接到不可访问的服务器的秒数。默认值为 600（10 分钟）。

事务日志位置

保存服务器日志的目录。事务日志保存在此字段所指定目录的 tx 子目录中。默认值为“域属性”页面（位于 "GlassFish Server" 节点的“高级”选项卡下）的“日志根”字段指定的目录。

启发式决策

对于涉及不可访问服务器的事务，是要提交还是回滚。默认值为“回滚”。提交不确定的事务会破坏应用程序的数据完整性。

关键点间隔

关键点操作（压缩事务日志文件）之间的事务数。默认值为 65,536。

其他属性

事务服务的其他属性。

有关可用属性的说明，请参见特定于事务服务的属性。
