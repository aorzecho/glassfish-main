设置事务超时值

默认情况下，服务器不会使事务超时。即，服务器无限期地等待事务完成。如果为事务设置了超时值，而事务在配置的时间内未完成，则 GlassFish Server 将回滚此事务。

  1. 在导航树中，展开“配置”节点。
  2. 选择要配置的实例：
    • 要配置特定的实例，请选择该实例的配置节点。例如，对于默认实例 server，请选择 server-config 节点。
    • 要配置所有实例的默认设置，请选择 default-config 节点。
  3. 选择“事务服务”节点。

    此时将打开“事务服务”页面。

  4. 在“事务超时”字段中，键入事务超时之前经过的秒数。

    默认值为 0。此值将禁用事务超时。

  5. 单击“保存”。
  6. 重新启动 GlassFish Server。
