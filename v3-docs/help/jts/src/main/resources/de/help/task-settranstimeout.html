So legen Sie einen Zeitüberschreitungswert für Transaktionen fest

Standardmäßig ist auf dem Server keine Zeitüberschreitung für Transaktionen festgelegt. Das heißt, dass der Server für eine unbegrenzte Zeit wartet, bis eine Transaktion abgeschlossen ist. Wenn Sie einen Zeitüberschreitungswert für Transaktionen festlegen, setzt GlassFish-Server die Transaktion zurück, wenn sie nicht innerhalb des konfigurierten Zeitraums abgeschlossen werden kann.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Konfiguration".
  2. Wählen Sie die zu konfigurierende Instanz:
    • Um eine bestimmte Instanz zu konfigurieren, wählen Sie den Konfigurationsknoten der Instanz aus. Für die Standardinstanz server wählen Sie z. B. den Knoten server-config.
    • Um die Standardeinstellungen für alle Instanzen zu konfigurieren, wählen Sie den Knoten default-config.
  3. Wählen Sie den Knoten "Transaktionsdienst".

    Die Seite "Transaktionsdienst" wird geöffnet.

  4. Geben Sie im Feld "Transaktionszeitüberschreitung" den Zeitraum (in Sekunden) bis zur Zeitüberschreitung für die Transaktion an.

    Der Standardwert ist 0. Mit diesem Wert ist die Zeitüberschreitung bei Transaktionen deaktiviert.

  5. Klicken Sie auf "Speichern".
  6. Starten Sie GlassFish-Server neu.
