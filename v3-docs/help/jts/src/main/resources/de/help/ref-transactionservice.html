Transaktionsdienst

Auf der Seite "Transaktionsdienst" können Sie die Transaktionswiederherstellung, Zeitüberschreitungen und die Protokollierung konfigurieren.

Die Seite "Transaktionsdienst" umfasst die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Beim Neustart

Wenn das Kontrollkästchen "Aktiviert" gewählt ist, versucht GlassFish-Server unvollständige Transaktionen wiederherzustellen, sobald der Server neu gestartet wird. Diese Option ist standardmäßig deaktiviert.

Transaktionszeitüberschreitung

Der Zeitraum (in Sekunden), der verstreicht, bevor der Server eine unvollständige Transaktion zurücksetzt. Der Standardwert ist 0; das bedeutet, dass es von Seiten des Servers keine Zeitbeschränkung für den Abschluss einer Transaktion gibt.

Wiederholungszeitüberschreitung

Der Zeitraum (in Sekunden), während dessen GlassFish-Server versucht, eine Verbindung zu einem nicht erreichbaren Server herzustellen. Der Standardwert beträgt 600 (10 Minuten).

Speicherort des Transaktionsprotokolls

Das Verzeichnis, in dem die Serverprotokolle gespeichert sind. Transaktionsprotokolle werden im Unterverzeichnis tx des Verzeichnisses gespeichert, das in diesem Feld angegeben ist. Der Standardwert ist das Verzeichnis, das auf der Seite "Domänenattribute" im Feld "Protokollstamm" angegeben ist, das sich im Knoten GlassFish-Server auf der Registerkarte "Erweitert" befindet.

Heuristische Entscheidung

Die Entscheidung, ob Transaktionen, die nicht erreichbare Server betreffen, durchgeführt oder zurückgesetzt werden. Der Standardwert ist Zurücksetzen. Das Durchführen unbestimmter Transaktionen kann die Datenintegrität Ihrer Anwendung gefährden.

Schlüsselpunktintervall

Die Anzahl der Transaktionen zwischen Schlüsselpunktvorgängen, die das Transaktionsprotokoll komprimieren. Der Standardwert beträgt 65.536.

Weitere Eigenschaften

Weitere Eigenschaften für den Transaktionsdienst.

Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Für den Transaktionsdienst spezifische Eigenschaften.

Verwandte Aufgaben

  • So konfigurieren Sie den Transaktionsdienst

  • So konfigurieren Sie die Wiederherstellung von Transaktionen in GlassFish-Server

  • So legen Sie einen Zeitüberschreitungswert für Transaktionen fest

  • So legen Sie den Speicherort für Transaktionsprotokolle fest

  • So legen Sie das Schlüsselpunktintervall fest
