Transaktionsservice

Auf der Seite "Transaktionsservice" können Sie Transaktions-Recovery, Timeouts und Logging konfigurieren.

Die Seite "Transaktionsservice" umfasst die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Beim Neustart

Wenn das Kontrollkästchen "Aktiviert" gewählt ist, versucht GlassFish Server unvollständige Transaktionen wiederherzustellen, sobald der Server neu gestartet wird. Diese Option ist standardmäßig deaktiviert.

Transaktionstimeout

Der Zeitraum (in Sekunden), der verstreicht, bevor der Server eine unvollständige Transaktion zurücksetzt. Der Standardwert ist 0; das bedeutet, dass es von Seiten des Servers keine Zeitbeschränkung für den Abschluss einer Transaktion gibt.

Wiederholungstimeout

Der Zeitraum (in Sekunden), während dessen GlassFish Server versucht, eine Verbindung zu einem nicht erreichbaren Server herzustellen. Der Standardwert beträgt 600 (10 Minuten).

Verzeichnis des Transaktionslogs

Das Verzeichnis, in dem die Serverprotokolle gespeichert sind. Transaktionslogs werden im Unterverzeichnis tx des Verzeichnisses gespeichert, das in diesem Feld angegeben ist. Der Standardwert ist das Verzeichnis, das auf der Seite "Domainattribute" im Feld "Log-Root" angegeben ist, das sich im Knoten GlassFish Server in der Registerkarte "Erweitert" befindet.

Heuristische Entscheidung

Die Entscheidung, ob Transaktionen, die nicht erreichbare Server betreffen, festgeschrieben oder zurückgesetzt werden. Der Standardwert ist Zurücksetzen. Das Festschreiben unbestimmter Transaktionen kann die Datenintegrität Ihrer Anwendung gefährden.

Schlüsselpunktintervall

Die Anzahl der Transaktionen zwischen Schlüsselpunktvorgängen, die das Transaktionslog komprimieren. Der Standardwert beträgt 65.536.

Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Für Transaktionsservice spezifische Eigenschaften.

Verwandte Aufgaben

  • So wird der Transaktionsservice konfiguriert

  • So wird konfiguriert, wie GlassFish Server Transaktionen wiederherstellt

  • So wird ein Timeoutwert für Transaktionen festgelegt

  • So wird das Verzeichnis der Transaktionslogs festgelegt

  • So wird das Schlüsselpunktintervall festgelegt
