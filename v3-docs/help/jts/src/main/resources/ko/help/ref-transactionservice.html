트랜잭션 서비스

트랜잭션 서비스 페이지에서는 트랜잭션 복구, 시간 초과 및 로깅을 구성할 수 있습니다.

트랜잭션 서비스 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

다시 시작 시

사용 가능 확인란이 선택되어 있으면 GlassFish Server에서 서버가 다시 시작할 때 완료되지 않은 트랜잭션을 복구하려 합니다. 이 옵션은 기본적으로 비활성화됩니다.

트랜잭션 시간 초과

서버가 완료되지 않은 트랜잭션을 롤백하기 전에 대기하는 시간(초)입니다. 기본값은 0이며, 이때 서버는 트랜잭션이 완료될 때까지 무기한 기다립니다.

재시도 시간 초과

GlassFish Server에서 연결할 수 없는 서버에 연결을 시도하는 시간(초)입니다. 기본값은 600(10분)입니다.

트랜잭션 로그 위치

서버 로그가 저장되는 디렉토리입니다. 트랜잭션 로그는 이 필드에서 지정한 디렉토리의 tx 하위 디렉토리에 저장됩니다. 기본값은 GlassFish Server 노드의 고급 탭 아래에 있는 도메인 속성 페이지의 로그 루트 필드에 지정된 디렉토리입니다.

발견적 판단

연결할 수 없는 서버와 관련된 트랜잭션을 완결할지 또는 롤백할지를 결정합니다. 기본값은 롤백입니다. 확정되지 않은 트랜잭션을 완결하면 응용 프로그램의 데이터 무결성이 손상될 수 있습니다.

키포인트 간격

트랜잭션 로그 파일을 압축하는 키포인트 작업 간의 트랜잭션 수입니다. 기본값은 65,536입니다.

추가 등록 정보

트랜잭션 서비스의 추가 등록 정보입니다.

사용할 수 있는 등록 정보에 대한 설명은 트랜잭션 서비스와 관련된 등록 정보를 참조하십시오.
