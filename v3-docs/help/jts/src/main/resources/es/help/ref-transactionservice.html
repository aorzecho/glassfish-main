Servicio de transacción

Utilice la página Servicio de transacción para configurar la recuperación, tiempos de espera y registro de la transacción.

La página Servicio de transacción contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre de la configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

En el reinicio

Si se selecciona la casilla de verificación Habilitado, GlassFish Server intenta recuperar las transacciones incompletas cuando se reinicia el servidor. Esta opción se deshabilita por defecto.

Tiempo de espera de la transacción

Número se segundos que espera un servidor antes de anular una transacción que no se haya completado. El valor predeterminado es cero, lo que supone que el servidor espera indefinidamente para que se complete la transacción.

Volver a intentar tiempo de espera

Número de segundos durante los cuales GlassFish Server intenta conectarse a un servidor inaccesible. El valor predeterminado es 600 (10 minutos).

Ubicación del registro de transacción

El directorio en el que se mantienen los registros del servidor. Los registros de transacciones se guardan en el subdirectorio tx del directorio especificado en el campo. El valor predeterminado es el directorio especificado en el campo Raíz de registro de la página Atributos de dominio, que está ubicada en la ficha Avanzado del nodo GlassFish Server.

Decisión heurística

Si las transacciones que implican servidores inaccesibles se confirman o se anulan. El valor predeterminado es Anulación. La confirmación de transacciones indeterminadas puede poner en peligro la integridad de los datos de la aplicación.

Intervalo de puntos clave

El número de transacciones entre las operaciones de puntos clave, que comprimen la transacción del archivo de registro. El valor predeterminado es 65536.
