Servicio de Transacción

Utilice la página Servicio de Transacción para configurar la recuperación, timeouts y registro de la transacción.

La página Servicio de Transacción contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Al Reiniciar

Si se selecciona la casilla de verificación Activado, Servidor GlassFish intenta recuperar las transacciones incompletas cuando se reinicia el servidor. Esta opción se encuentra desactivada por defecto.

Timeout de Transacción

Número se segundos que espera un servidor antes de anular una transacción que no se haya completado. El valor por defecto es cero, lo que supone que el servidor espera indefinidamente para que se complete la transacción.

Timeout de Reintentos

Número de segundos durante los cuales Servidor GlassFish intenta conectarse a un servidor inaccesible. El valor por defecto es 600 (10 minutos).

Ubicación de Log de Transacciones

El directorio en el que se mantienen los logs del servidor. Los logs de transacciones se guardan en el subdirectorio tx del directorio especificado en el campo. El valor por defecto es el directorio especificado en el campo Raíz de Log de la página Atributos de Dominio, que está ubicada en el separador Avanzado del nodo Servidor GlassFish.

Decisión Heurística

Si las transacciones que implican servidores inaccesibles se confirman o se anulan. El valor por defecto es Realizar Rollback. La confirmación de transacciones indeterminadas puede poner en peligro la integridad de los datos de la aplicación.

Intervalo de Punto Clave

El número de transacciones entre las operaciones de puntos clave, que comprimen la transacción del archivo log. El valor por defecto es 65536.

Propiedades Adicionales

Propiedades adicionales del Servicio de Transacción.

Para obtener una descripción de las propiedades disponibles, consulte Propiedades Específicas del Servicio de Transacción.
