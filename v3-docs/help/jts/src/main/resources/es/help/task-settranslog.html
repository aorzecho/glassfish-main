Cómo establecer la ubicación de los registros de transacciones

El registro de transacciones registra la información sobre cada transacción para poder mantener la integridad de datos de los recursos implicados y recuperar el sistema de los fallos. Los registros de transacciones se guardan en el subdirectorio tx del directorio especificado en el campo Ubicación del registro de transacciones. Estos registros no están orientados para la lectura por parte de seres humanos.

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo para la instancia predeterminada server, seleccione el nodo server-config.
    • Para configurar los ajustes predeterminados para todas las instancias, seleccione el nodo default-config.
  3. Seleccione el nodo Servicio de transacciones.

    Se abrirá la página Servicio de transacciones.

  4. En el campo Registro de transacciones, introduzca la ubicación de los registros de transacción.

    Se creará un subdirectorio tx y los registros de las transacciones se guardarán en ese directorio.

  5. Haga clic en Guardar.
