Cómo establecer un valor de tiempo de espera para las transacciones

Por defecto el servidor no pone un límite de tiempo a las transacciones. Es decir, el servidor se esperará de forma indefinida a que finalice la transacción. Si establece un valor de tiempo de espera para las transacciones, entonces en el caso de que una transacción no se realice dentro del tiempo establecido GlassFish Server deshará la transacción.

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo para la instancia predeterminada server, seleccione el nodo server-config.
    • Para configurar los ajustes predeterminados para todas las instancias, seleccione el nodo default-config.
  3. Seleccione el nodo Servicio de transacciones.

    Se abrirá la página Servicio de transacciones.

  4. En el campo Tiempo de espera de transacción, introduzca el número de segundos que deben transcurrir antes de que se agote el tiempo de espera de la transacción.

    El valor predeterminado es 0. Este valor deshabilita los tiempos de espera de transacciones.

  5. Haga clic en Guardar.
  6. Reinicie GlassFish Server.
