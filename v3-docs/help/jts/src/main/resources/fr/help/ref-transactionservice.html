Service de transaction

La page Service de transaction permet de configurer la récupération, les temporisations et la journalisation des transactions.

Elle contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Au redémarrage

Si la case Activé est cochée, GlassFish Server tente de récupérer les transactions incomplètes au redémarrage du serveur. Cette option est désactivée par défaut.

Temporisation de transaction

Délai d'attente, en secondes, du serveur au-delà duquel ce dernier annule une transaction qui n'est pas terminée. La valeur par défaut est 0. Elle signifie que le serveur attend la fin d'une transaction pendant une durée indéterminée.

Temporisation de tentative

Durée, en secondes, pendant laquelle GlassFish Server tente de se connecter à un serveur inaccessible. La valeur par défaut est 600 (10 minutes).

Emplacement des journaux des transactions

Répertoire dans lequel sont conservés les journaux du serveur. Les journaux de transaction sont conservés dans le sous-répertoire tx du répertoire spécifié dans ce champ. La valeur par défaut correspond au répertoire spécifié dans le champ Racine journal de la page Attributs de domaine, qui se trouve dans l'onglet Avancé du noeud GlassFish Server.

Décision heuristique

Indique si les transactions qui impliquent des serveurs inaccessibles doivent être validées ou annulées. La valeur par défaut est Annuler. Valider les transactions indéterminées risque de compromettre l'intégrité des données dans votre application.

Intervalle entre points clés

Nombre de transactions effectuées entre des opérations de point clé qui compriment le fichier journal de transactions. La valeur par défaut est 65 536.

Autres propriétés

Propriétés supplémentaires du service de transaction.

Pour obtenir une description des propriétés disponibles, reportez-vous à Propriétés propres au service de transaction.
