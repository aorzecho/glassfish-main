Définir l'emplacement des journaux des transactions

Les journaux des transactions enregistrent les informations sur chaque transaction afin de garantir l'intégrité des données des ressources impliquées et de récupérer les données en cas de panne. Les journaux de transaction sont conservés dans le sous-répertoire tx du répertoire spécifié dans le champ Emplacement du journal des transactions. Ces journaux ne sont pas lisibles par l'homme.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sélectionnez l'instance à configurer :

    • Pour configurer une instance spécifique, sélectionnez le noeud de configuration de l'instance. Par exemple, pour l'instance par défaut, server, sélectionnez le noeud server-config.

    • Pour configurer les paramètres par défaut pour toutes les instances, sélectionnez le noeud default-config.

  3. Sélectionnez le noeud Service de transaction.

    La page Service de transaction apparaît.

  4. Dans le champ Emplacement du journal des transactions, saisissez l'emplacement des journaux des transactions.

    Un sous-répertoire tx est créé pour y conserver les journaux des transactions.

  5. Cliquez sur Enregistrer.
