Définition de l’emplacement des journaux des transactions

Les journaux de transaction consignent les informations sur chaque transaction afin de garantir l’intégrité des données des ressources impliquées et de récupérer les données en cas de panne. Les journaux de transaction sont conservés dans le sous-répertoire tx du répertoire spécifié dans le champ Emplacement du journal des transactions. Ces journaux ne sont pas lisibles par l’homme.

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez l’instance à configurer :
    • Pour configurer une instance en particulier, sélectionnez le nœud de configuration de l’instance. Par exemple, pour l’instance par défaut, server, sélectionnez le nœud server-config.
    • Pour configurer les réglages par défaut pour toutes les instances, sélectionnez le nœud default-config.
  3. Sélectionnez le nœud Service de transaction.

    La page Service de transaction s’affiche.

  4. Dans le champ Emplacement du journal des transactions, saisissez l’emplacement des journaux des transactions.

    Un sous-répertoire tx est créé pour y conserver les journaux des transactions.

  5. Cliquez sur Enregistrer.
