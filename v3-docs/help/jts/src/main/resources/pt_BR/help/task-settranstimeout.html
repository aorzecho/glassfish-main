Para Definir um Valor de Time-out da Transação

Por default, o servidor não limita o tempo de uma transação. Ou seja, o servidor espera indefinidamente pela conclusão de uma transação. Caso você defina um valor de time-out para transações, se uma transação não for concluída dentro do tempo configurado, o GlassFish Server fará roll back da transação.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância em particular, selecione o nó de configuração da instância. Por exemplo, na instância default, server, selecione o nó server-config.

    • Para definir as definições default de todas as instâncias, selecione o nó default-config.

  3. Selecione o nó Serviço de transação.

    A página Serviço de transação é aberta.

  4. No campo Time-out para Transação, digite o número de segundos até que se esgote o tempo da transação.

    O valor default é 0. Esse valor desativa os time-outs das transações.

  5. Clique em Salvar.

  6. Reinicie o GlassFish Server.
