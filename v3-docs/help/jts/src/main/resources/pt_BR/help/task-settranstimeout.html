Para definir um valor de tempo limite de transação

Por padrão, o servidor não limita o tempo de uma transação. Ou seja, o servidor espera indefinidamente pela conclusão de uma transação. Se você definir um valor de tempo limite de transações, caso a transação não se complete no tempo configurado, o Servidor GlassFish reverte a transação.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a instância que será configurada:
    • Para configurar uma determinada instância, selecione o nó de configuração da instância. Por exemplo, na instância padrão, server, selecione o nó server-config.
    • Para definir as configurações padrão de todas as instâncias, selecione o nó default-config.
  3. Selecione o nó Serviço de transação.

    A página Serviço de transação é aberta.

  4. No campo Tempo limite da transação, digite o número de segundos até que se esgote o tempo da transação.

    O valor padrão é 0. Esse valor desabilita os tempos limite da transação.

  5. Clique em Salvar.
  6. Reinicie o Servidor GlassFish.
