Serviço de transação

Utilize a página Serviço de transação para configurar recuperação, tempo limite e log de transação.

A página Serviço de transação contém estas opções:

Carregar padrões

Botão para restaurar com os valores padrão as configurações que apresentam valores padrão. As configurações que não apresentam valores padrão não são alteradas.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Ao reinicializar

Se a caixa de seleção Ativo estiver marcada, o Servidor GlassFish tentará recuperar as transações incompletas quando o servidor for reiniciado. Por padrão, a opção está desativada.

Tempo limite de transação

Número de segundos que o servidor aguarda antes de reverter uma transação não concluída. O valor padrão é 0, significando que o servidor aguarda indefinidamente que uma transação seja concluída.

Tempo limite da nova tentativa

Número de segundos em que o Servidor GlassFish tenta se conectar a um servidor inacessível. O valor padrão é 600 (10 minutos).

Local do log de transação

O diretório onde os logs do servidor são mantidos. Os logs de transação são mantidos no subdiretório tx do diretório especificado neste campo. O valor padrão é o diretório especificado pelo campo Raiz de log da página Atributos de domínio, que está localizada na guia Avançado do nó do Servidor GlassFish.

Decisão heurística

Define se as transações envolvendo servidores inacessíveis devem ser confirmadas ou revertidas. O valor padrão Reverter. A confirmação de transações indeterminadas pode comprometer a integridade dos dados do aplicativo.

Intervalo de pontos principais

O número de transações entre as operações de ponto-chave, que compacta o arquivo de log de transação. O valor padrão é 65536.

Propriedades adicionais

As propriedades adicionais do serviço de transação.

Para obter uma descrição das propriedades disponíveis, consulte Propriedades específicas do serviço de transação.
