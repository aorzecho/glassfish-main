Serviço da Transação

Use a página Serviço de Transação para configurar recuperação, time-out e log de transação.

A página Serviço de transação contém estas opções:

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Ao reinicializar

Se a caixa de seleção Ativo estiver marcada, o GlassFish Server tentará recuperar as transações incompletas quando o servidor for reiniciado. Por default, essa opção está desativada.

Time-out para transação

Número de segundos que o servidor aguarda antes de reverter uma transação não concluída. O valor default é 0, significando que o servidor aguarda indefinidamente que uma transação seja concluída.

Time-out para nova tentativa

Número de segundos em que o GlassFish Server tenta se conectar a um servidor inacessível. O valor default é 600 (10 minutos).

Local do log de transação

O diretório onde os logs do servidor são mantidos. Os logs de transação são mantidos no subdiretório tx do diretório especificado neste campo. O valor default é o diretório especificado pelo campo Raiz de log da página Atributos de domínio, que está localizada na guia Avançado do nó do GlassFish Server.

Decisão heurística

Define se as transações envolvendo servidores inacessíveis devem ser confirmadas ou revertidas. O valor default Reverter. A confirmação de transações indeterminadas pode comprometer a integridade dos dados da aplicação.

Intervalo de pontos principais

O número de transações entre as operações de ponto-chave, que compacta o arquivo de log de transação. O valor default é 65536.

Propriedades adicionais

As propriedades adicionais do serviço de transação.

Para obter uma descrição das propriedades disponíveis, Propriedades Específicas ao Serviço da Transação consulte.
