Para Configurar o Serviço da Transação

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância em particular, selecione o nó de configuração da instância. Por exemplo, na instância default, server, selecione o nó server-config.

    • Para definir as definições default de todas as instâncias, selecione o nó default-config.

  3. Selecione o nó Serviço de transação.

    A página Serviço de transação é aberta.

  4. Na página Serviço de transação, marque a caixa de seleção Ativado ao reiniciar para ativar a recuperação de transações incompletas quando o servidor é reiniciado.

    Por default, essa opção está desativada.

  5. No campo Time-out da Transação, digite o número de segundos que o servidor aguarda antes de reverter uma transação que não foi concluída.

    O valor 0 desativa os time-outs de transação. O valor default é 0, significando que o servidor aguarda indefinidamente que uma transação seja concluída.

  6. No campo Time-out para Tentativa, digite o número de segundos que o GlassFish Server tenta se conectar a um servidor inacessível.

    O valor default é 600 (10 minutos).

  7. No campo Local do log de transação, digite o diretório em que os logs do servidor são mantidos.

    Os logs de transação são mantidos no subdiretório tx do diretório especificado neste campo. O valor default é o diretório especificado pelo campo Raiz de log da página Atributos do domínio, que está localizado na guia Avançado do nó do GlassFish Server.

  8. Na lista drop-down Decisão heurística, selecione a política a ser seguida se o resultado de uma transação distribuída não puder ser determinado pelo fato de outros participantes estarem inacessíveis.

    A menos que haja um bom motivo para selecionar Confirmar, deixe a Decisão Heurística definida como Reversão (o default). A confirmação de transações indeterminadas pode comprometer a integridade dos dados da aplicação.

  9. No campo Intervalo de ponto-chave, digite o número de transações entre as operações de ponto-chave.

    As operações de ponto-chave compactam o arquivo de log da transação. O valor default é 65536.

  10. Na seção Propriedades adicionais, especifique essas propriedades.

    Para obter uma descrição das propriedades disponíveis, Propriedades Específicas ao Serviço da Transação consulte.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  11. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.

  12. Reinicie o GlassFish Server.
