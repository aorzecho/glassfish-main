Para configurar o serviço de transação

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a instância que será configurada:
    • Para configurar uma determinada instância, selecione o nó de configuração da instância. Por exemplo, na instância padrão, server, selecione o nó server-config.
    • Para definir as configurações padrão de todas as instâncias, selecione o nó default-config.
  3. Selecione o nó Serviço de transação.

    A página Serviço de transação é aberta.

  4. Na página Serviço de transação, marque a caixa de seleção Habilitado ao reiniciar para ativar a recuperação de transações incompletas quando o servidor é reiniciado.

    Por padrão, a opção está desativada.

  5. No campo Tempo limite de transação, digite o número de segundos que o servidor aguarda antes de reverter uma transação que não foi concluída.

    Um valor de 0 desativa os tempo limite da transação. O valor padrão é 0, significando que o servidor aguarda indefinidamente que uma transação seja concluída.

  6. No campo Tempo limite de tentativa, digite o número de segundos que o Servidor GlassFish tenta se conectar a um servidor inacessível.

    O valor padrão é 600 (10 minutos).

  7. No campo Local do log de transação, digite o diretório em que os logs do servidor são mantidos.

    Os logs de transação são mantidos no subdiretório tx do diretório especificado neste campo. O valor padrão é o diretório especificado pelo campo Raiz de log da página Atributos do domínio, que está localizado na guia Avançado do nó do Servidor GlassFish.

  8. Na lista suspensa Decisão heurística, selecione a diretriz a ser seguida se o resultado de uma transação distribuída não puder ser determinado pelo fato de outros participantes estarem inacessíveis.

    A menos que haja um bom motivo para selecionar Confirmar, deixe a Decisão heurística definida como Reversão (o padrão). A confirmação de transações indeterminadas pode comprometer a integridade dos dados do aplicativo.

  9. No campo Intervalo de ponto-chave, digite o número de transações entre as operações de ponto-chave.

    As operações de ponto-chave compactam o arquivo de log da transação. O valor padrão é 65536.

  10. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    Para obter uma descrição das propriedades disponíveis, consulte Propriedades específicas do serviço de transação.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em brando que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  11. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e depois em Salvar.

  12. Reinicie o Servidor GlassFish.
