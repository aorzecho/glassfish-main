Para definir o intervalo do ponto-chave

As operações de ponto-chave compactam o arquivo de log da transação. O intervalo de ponto-chave é o número de transações entre as operações de ponto-chave no log. As operações de ponto-chave podem reduzir o tamanho dos arquivos de log de transação. Um número maior de intervalos de ponto-chave resulta em arquivos maiores de log de transação, mas em menos operações de ponto-chave e desempenho potencialmente melhor. Um intervalo menor de ponto-chave (por exemplo, 256) resulta em arquivos de log menores, mas em desempenho ligeiramente reduzido devido à frequência maior de operações de ponto-chave.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a instância que será configurada:
    • Para configurar uma determinada instância, selecione o nó de configuração da instância. Por exemplo, na instância padrão, server, selecione o nó server-config.
    • Para definir as configurações padrão de todas as instâncias, selecione o nó default-config.
  3. Selecione o nó Serviço de transação.

    A página Serviço de transação é aberta.

  4. No campo Intervalo de ponto-chave, digite o número de transações entre as operações de ponto-chave.

    O valor padrão é 65536.

  5. Clique em Salvar.
