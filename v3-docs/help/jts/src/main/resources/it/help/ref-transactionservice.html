Servizio transazioni

Nella pagina Servizio transazioni è possibile configurare il recupero, il timeout e le operazioni di registrazione delle transazioni.

La pagina Servizio transazioni contiene le seguenti opzioni.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Campo di sola lettura.

Al riavvio

Se la casella di controllo Abilitato è selezionata, al riavvio del server viene eseguito un tentativo di recupero delle transazioni incomplete. Questa opzione è disabilitata per impostazione predefinita.

Timeout transazione

Numero di secondi di attesa del server prima di eseguire il rollback di una transazione incompleta. Il valore predefinito è 0, che indica un tempo di attesa illimitato del server per il completamento di una transazione.

Timeout tentativi

Numero di secondi durante i quali viene eseguito un tentativo di connessione tra GlassFish Server e un server irraggiungibile. Il valore predefinito è 600 (10 minuti).

Posizione log delle transazioni

Directory in cui vengono memorizzati i log del server. I log delle transazioni vengono memorizzati nella sottodirectory tx della directory specificata in questo campo. Il valore predefinito è la directory specificata nel campo Radice log disponibile nella pagina Attributi dominio all'interno della scheda Advanced del nodo GlassFish Server.

Decisione euristica

Consente di specificare se eseguire il commit o il rollback delle transazioni per cui vengono utilizzati server irraggiungibili. Il valore predefinito è Rollback. L'esecuzione del commit di un numero indeterminato di transazioni potrebbe avere ripercussioni sull'integrità dei dati dell'applicazione.

Intervallo punti chiave

Numero di transazioni tra un'operazione punto chiave e l'altra, che determinano il livello di compressione del file di log delle transazioni. Il valore predefinito è 65.536.
