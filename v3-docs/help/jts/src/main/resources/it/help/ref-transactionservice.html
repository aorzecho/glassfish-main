Transaction Service

Nella pagina Transaction Service è possibile configurare il ripristino, il timeout e le operazioni di registrazione delle transazioni.

La pagina Transaction Service contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

On Restart

Se la casella di controllo Enabled è selezionata, al riavvio del server viene eseguito un tentativo di ripristino delle transazioni incomplete. Questa opzione è disabilitata per impostazione predefinita.

Transaction Timeout

Numero di secondi di attesa del server prima di eseguire il rollback di una transazione incompleta. Il valore predefinito è 0, che indica un tempo di attesa illimitato del server per il completamento di una transazione.

Retry Timeout

Numero di secondi durante i quali viene eseguito un tentativo di connessione tra GlassFish Server e un server irraggiungibile. Il valore predefinito è 600 (10 minuti).

Transaction Log Location

Directory in cui vengono memorizzati i log del server. I log delle transazioni vengono memorizzati nella sottodirectory tx della directory specificata in questo campo. Il valore predefinito è la directory specificata nel campo Log Root disponibile nella pagina Domain Attributes all'interno della scheda Advanced del nodo GlassFish Server.

Heuristic Decision

Consente di specificare se eseguire il commit o il rollback delle transazioni per cui vengono utilizzati server irraggiungibili. Il valore predefinito è Rollback. L'esecuzione del commit di un numero indeterminato di transazioni potrebbe avere ripercussioni sull'integrità dei dati dell'applicazione.

Keypoint Interval

Numero di transazioni tra un'operazione keypoint e l'altra, che determinano il livello di compressione del file di log delle transazioni. Il valore predefinito è 65.536.

Additional Properties

Proprietà aggiuntive per Transaction Service.

Per una descrizione delle proprietà disponibili, vedere Proprietà specifiche di Transaction Service.

Attività correlate

  • Configurare Transaction Service

  • Configurare la modalità di ripristino delle transazioni in GlassFish Server

  • Impostare un valore di timeout per le transazioni

  • Impostare la posizione dei log delle transazioni

  • Impostare l'intervallo keypoint
