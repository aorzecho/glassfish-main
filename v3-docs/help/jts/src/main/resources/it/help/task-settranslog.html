Impostare la posizione dei log delle transazioni

Nei log delle transazioni vengono registrate informazioni relative alle singole transazioni allo scopo di mantenere l'integrità dei dati delle risorse utilizzate e di eseguire operazioni di recupero. I log delle transazioni vengono memorizzati nella sottodirectory tx della directory specificata nel campo Posizione log delle transazioni. I log non sono scritti in forma leggibile.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza specifica, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite per tutte le istanze, selezionare il nodo default-config.

  3. Selezionare il nodo Servizio transazioni.

    Viene visualizzata la pagina Servizio transazioni.

  4. Nel campo Posizione log delle transazioni, digitare la posizione dei log delle transazioni.

    Viene creata una sottodirectory tx in cui verranno memorizzati i log delle transazioni.

  5. Fare clic su Salva.
