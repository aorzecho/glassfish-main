Impostare la posizione dei log delle transazioni

Nei log delle transazioni vengono registrate informazioni relative alle singole transazioni allo scopo di mantenere l'integrità dei dati delle risorse utilizzate e di eseguire operazioni di ripristino. I log delle transazioni vengono memorizzati nella sottodirectory tx della directory specificata nel campo Transaction Log Location. I log non sono scritti in forma leggibile.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare l'istanza da configurare:
    • Per configurare una particolare istanza, selezionare il nodo config dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.
    • Per configurare le impostazioni predefinite per tutte le istanze, selezionare il nodo default-config.
  3. Selezionare il nodo Transaction Service.

    Viene visualizzata la pagina Transaction Service.

  4. Nel campo Transaction Log Location, digitare la posizione dei log delle transazioni.

    Viene creata una sottodirectory tx in cui verranno memorizzati i log delle transazioni.

  5. Fare clic su Save.
