設定作業事件記錄的位置

為了保持被呼叫資源的資料完整性，同時為了能夠從故障中恢復，作業事件記錄將記錄有關每個作業事件的資訊。作業事件記錄儲存在 [作業事件記錄位置] 欄位所指定目錄的 tx 子目錄中。這些不是一般人可解讀的記錄。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 選取要配置的實例：
    • 若要配置特定的實例，請選取該實例的配置節點。例如，對於預設實例 server，請選取 [server-config] 節點。
    • 若要配置所有實例的預設設定，請選取 [default-config] 節點。
  3. 選取 [作業事件服務] 節點。

    [作業事件服務] 頁面會隨即開啟。

  4. 在 [作業事件記錄位置] 欄位中，鍵入作業事件記錄的位置。

    tx 子目錄便會建立，並且會在該目錄下保存作業事件記錄。

  5. 按一下 [儲存]。
