配置 MDB 池设置

消息驱动 Bean 的池与会话 Bean 的池类似。默认情况下，容器会维护一个由消息驱动 Bean 组成的池。

要调整此池的配置，请执行以下步骤。

  1. 在导航树中，展开“配置”节点。

  2. 选择“EJB 容器”节点。

    此时将打开“企业 Java Bean (EJB)”页。

  3. 单击“MDB 设置”选项卡。

    此时将打开“MDB 默认池设置”页。

  4. 在“初始和最小池大小”字段中，键入要在池中维护的消息 Bean 的最小数目。

    默认值为 0。

  5. 在“最大池大小”字段中，键入容器可以同时在池中维护的 Bean 的最大数目。

    默认值为 32。

  6. 在“池调整大小数量”字段中，键入当 Bean 处于空闲状态的时间超过“池空闲超时”字段中指定的时间时，要从池中删除的 Bean 的数目。

    该值必须至少为 0，并小于“最大池大小”值。默认值为 8。

  7. 在“池空闲超时”字段中，键入在将池中的 Bean 从池中删除之前，Bean 可以保持空闲状态的时间 (秒)。

    超过此时间后，Bean 将被销毁。如果值为 0，则表示 Bean 可以无限期保持空闲状态。默认值为 600。

  8. 在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义 MDB 池的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  9. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。

  10. 重新启动 GlassFish Server。
