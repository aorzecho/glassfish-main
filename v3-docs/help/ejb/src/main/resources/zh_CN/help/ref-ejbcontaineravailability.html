EJB 容器可用性

使用“EJB 容器可用性”页可配置 EJB 容器的可用性设置。

“EJB 容器可用性”页包含以下选项：

加载默认值

使具有默认值的设置还原到其默认值的按钮。不具有默认值的设置不会发生更改。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

可用性服务

如果选中“已启用”复选框，将为 EJB 容器启用可用性。

HA 持久类型

选中“已启用可用性服务”时所使用的持久性机制。默认值为 replicated，采用该值时将基于集群中的故障转移提供持久性。请勿更改此值。

SFSB 持久类型

未选中“已启用可用性服务”时所使用的持久性机制。默认值为 file，采用该值时将基于文件系统提供持久性。请勿更改此值。

“会话存储位置”字段确定持久保存 SFSB 检查点状态的位置。要设置“会话存储位置”，请参阅配置 EJB 设置。

其他属性

EJB 容器可用性的其他属性。

GlassFish Server 不定义任何其他 EJB 容器可用性属性。
