EJB 容器可用性

使用“EJB 容器可用性”页面可配置 EJB 容器的可用性设置。

“EJB 容器可用性”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

可用性服务

如果选中“已启用”复选框，将为 EJB 容器启用可用性。

HA 持久性类型

选中“已启用可用性服务”时使用的持久性机制。默认值为 replicated，可提供基于群集中故障转移的持久性。不要更改此值。

SFSB 持久性类型

未选中“已启用可用性服务”时使用的持久性机制。默认值为 file，可提供基于文件系统的持久性。不要更改此值。

“会话存储位置”字段确定持久保存 SFSB 检查点状态的位置。要设置“会话存储位置”，请参见配置 EJB 设置。

其他属性

EJB 容器可用性的其他属性。

GlassFish Server 不定义任何其他 EJB 容器可用性属性。
