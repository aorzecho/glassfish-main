MDB 默认池设置

使用“MDB 默认池设置”页面可以配置 MDB 池。

“MDB 默认池设置”页面包含以下选项。

装入默认值

使具有默认值的设置恢复其默认值的按钮。不具有默认值的设置不会发生更改。

配置名称

要应用该页面上设置的配置名称。此字段为只读。

初始和最小池大小

要在池中维护的消息 Bean 的最小数目。默认值为 0。

最大池大小

容器可以同时在池中维护的 Bean 的最大数目。默认值为 32。

池大小调整数量

当 Bean 处于空闲状态的时间超过“池空闲超时”字段中指定的时间时，要从池中删除的 Bean 的数目。该值必须至少为 0，并小于“最大池大小”值。默认值为 8。

池空闲超时

在将池中的 Bean 从池中删除之前，Bean 可以保持空闲状态的时间（以秒为单位）。超过此时间后，Bean 将被销毁。如果值为 0，则表示 Bean 可以无限期保持空闲状态。默认值为 600。

其他属性

MDB 池的其他属性。GlassFish Server 不定义 MDB 池的任何其他属性。
