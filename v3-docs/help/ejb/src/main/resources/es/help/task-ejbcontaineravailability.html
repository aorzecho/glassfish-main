Definir la Configuración de Disponibilidad para el Contenedor EJB

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración que esté cambiando.

  3. En el nodo de configuración, seleccione el nodo Servicio de disponibilidad.

    Se abrirá la página Servicio de disponibilidad.

  4. En la página Servicio de Disponibilidad, haga clic en el separador Disponibilidad de Contenedor EJB.

    La página Disponibilidad de Contenedor EJB se abre.

  5. Seleccione la casilla Servicio de disponibilidad habilitado para habilitar la disponibilidad del contenedor EJB.

    Para deshabilitar la disponibilidad, quite la marca de selección de la casilla Servicio de disponibilidad.

  6. El tipo de persistencia HA es replicated.

    Éste es el mecanismo de persistencia utilizado cuando el Servicio de disponibilidad habilitado está seleccionado. La persistencia se basa en caso de conmutación de error en un clúster. No cambie este valor.

  7. El tipo de persistencia SFSB es file.

    Éste es el mecanismo de persistencia utilizado cuando el Servicio de disponibilidad habilitado no está seleccionado. La persistencia se basa en el sistema de archivos. No cambie este valor.

    El campo Ubicación del almacenamiento de sesión determina dónde continúan activos los estados del punto de control SFSB. Para definir la ubicación del almacén de sesión, consulte Definir la Configuración de EJB.

  8. Para restablecer los valores predeterminados, haga clic en el botón Cargar predeterminados.

  9. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para la disponibilidad del contenedor EJB.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  10. Haga clic en Guardar.
