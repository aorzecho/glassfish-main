Disponibilidad de Contenedor EJB

Utilice la página Disponibilidad de Contenedor EJB para configurar las opciones de disponibilidad para el contenedor EJB.

La página Disponibilidad de Contenedor EJB contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restaurar la configuración que tiene valores por defecto a estos valores. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Servicio de Disponibilidad

Si se activa la casilla de control Activado, se ha activado la disponibilidad para el contenedor EJB.

Tipo de Persistencia de HA

El mecanismo de persistencia utilizado cuando la casilla de control Servicio de Disponibilidad Activado está activada. El valor por defecto es replicated, que proporciona persistencia basándose en el failover en un cluster. No cambie este valor.

Tipo de Persistencia de SFSB

El mecanismo de persistencia utilizado cuando la casilla de control Servicio de Disponibilidad Activado no está activada. El valor por defecto es file, que proporciona persistencia basada en el sistema de archivos. No cambie este valor.

El campo Ubicación del almacenamiento de sesión determina dónde continúan activos los estados del punto de control SFSB. Para definir la ubicación del almacén de sesión, consulte Definir la Configuración de EJB.

Propiedades Adicionales

Propiedades adicionales de la disponibilidad de contenedor EJB.

GlassFish Server no define propiedades adicionales para la disponibilidad de contenedor EBJ.
