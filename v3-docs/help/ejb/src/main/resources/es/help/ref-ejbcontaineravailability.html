Disponibilidad de contenedor EJB

Utilice la página de Disponibilidad de contenedor EJB para configurar las opciones del contenedor EJB.

La página de Disponibilidad de contenedor EJB contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre de configuración

El nombre de configuración al que se aplican los valores de esta página. Este campo es de sólo lectura.

Servicio de disponibilidad

Si selecciona la casilla de verificación Habilitado, activará la disponibilidad para el contenedor EJB.

Tipo de persistencia HA

El mecanismo de persistencia utilizado cuando el Servicio de Disponibilidad Habilitado está seleccionado. El valor predeterminado es replicated, que proporciona persistencia basada en conmutación por error en un clúster. No cambie este valor.

Tipo de persistencia SFSB

El mecanismo de persistencia utilizado cuando el Servicio de Disponibilidad Habilitado no está seleccionado. El valor predeterminado es file, que proporciona persistencia basada en el sistema de archivos. No cambie este valor.

El campo Ubicación del almacenamiento de sesión determina dónde continúan activos los estados del punto de control SFSB. Para establecer la ubicación del almacenamiento de sesión, consulte To Configure EJB Settings.

Propiedades adicionales

Propiedades adicionales para la Disponibilidad de contenedor EJB.

GlassFish Server no define propiedades adicionales para la Disponibilidad de contenedor EJB.
