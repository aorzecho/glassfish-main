Configurer les paramètres de disponibilité du conteneur EJB

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sélectionnez la configuration à modifier.

  3. Sous le noeud Configuration, sélectionnez le noeud Service de disponibilité.

    La page Service de disponibilité apparaît.

  4. Sur la page Service de disponibilité, cliquez sur l'onglet Disponibilité du conteneur EJB.

    La page Disponibilité du conteneur EJB apparaît.

  5. Cochez la case Activé du champ Service de disponibilité pour activer la disponibilité du conteneur EJB.

    Pour désactiver la disponibilité, désélectionnez la case Activé du champ Service de disponibilité.

  6. Le type de persistance HA est replicated.

    Ce mécanisme de persistance est utilisé lorsque le service de disponibilité est activé. La persistance est basée sur le basculement dans un cluster. Ne modifiez pas cette valeur.

  7. Le type de persistance SFSB est file.

    Ce mécanisme de persistance est utilisé lorsque le service de disponibilité n'est pas activé. La persistance est basée sur le système de fichiers. Ne modifiez pas cette valeur.

    Le champ Emplacement de stockage des sessions détermine l'endroit où les états de point de reprise SFSB sont conservés. Pour plus d'informations sur la définition de l'emplacement de stockage des sessions, reportez-vous à Configurer les paramètres EJB.

  8. Pour restaurer les valeurs par défaut, cliquez sur le bouton Charger les valeurs par défaut.

  9. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server ne définit aucune autre propriété pour la disponibilité du conteneur EJB.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  10. Cliquez sur Enregistrer.
