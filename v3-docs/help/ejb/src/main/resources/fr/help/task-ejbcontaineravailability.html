Pour configurer les paramètres de disponibilité du conteneur EJB

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez la configuration que vous êtes en train de modifier.
  3. Sous le nœud Configuration, sélectionnez le nœud Service de disponibilité.

    La page Service de disponibilité s'ouvre.

  4. Sur la page Service de disponibilité, cliquez sur l'onglet Disponibilité du conteneur EJB.

    La page correspondante s'ouvre.

  5. Cochez la case Activé pour activer la disponibilité du conteneur EJB.

    Pour désactiver la disponibilité, décochez la case Service de disponibilité activé.

  6. Le type de persistance HA est replicated.

    Ce mécanisme de persistance est utilisé lorsque le service de disponibilité est activé. La persistance est basée sur le basculement sur un cluster. Ne modifiez pas cette valeur.

  7. Le type de persistance SFSB est file.

    Ce mécanisme de persistance est utilisé lorsque le service de disponibilité n'est pas activé. La persistance est basée sur le système de fichiers. Ne modifiez pas cette valeur.

    Le champ Emplacement de stockage des sessions détermine où les états de point de contrôle SFSB sont maintenus. Pour définir l'emplacement de stockage des sessions, voir To Configure EJB Settings.

  8. Pour restaurer les valeurs par défaut, cliquez sur le bouton Charger les valeurs par défaut.
  9. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    GlassFish Server ne définit aucune autre propriété pour la disponibilité du conteneur EJB.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  10. Cliquez sur Enregistrer.
