Disponibilité du conteneur EJB

La page Disponibilité du conteneur EJB vous permet de configurer les paramètres de disponibilité du conteneur EJB.

Elle contient les options suivantes.

Restaurer les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut de tous les paramètres.

Nom de configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Service de disponibilité

Si la case Activé est cochée, la disponibilité est activée pour le conteneur EJB.

Type de persistance HA

Mécanisme de persistance utilisé lorsque le service de disponibilité est activé. La valeur par défaut est replicated, qui fournit la persistance basée sur le basculement dans un cluster. Ne modifiez pas cette valeur.

Type de persistance SFSB

Mécanisme de persistance utilisé lorsque le service de disponibilité n'est pas activé. La valeur par défaut est file, qui fournit la persistance en fonction du système de fichiers. Ne modifiez pas cette valeur.

Le champ Emplacement de stockage des sessions détermine où les états de point de contrôle SFSB sont maintenus. Pour définir l'emplacement de stockage des sessions, voir To Configure EJB Settings.

Autres propriétés

Propriétés supplémentaires de la disponibilité du conteneur EJB.

GlassFish Server ne définit aucune autre propriété de disponibilité du conteneur EJB.
