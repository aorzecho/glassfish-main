Configurare le impostazioni di disponibilità per il contenitore EJB

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione, selezionare il nodo Servizio disponibilità.

    Viene visualizzata la pagina Servizio disponibilità.

  4. Nella pagina Servizio disponibilità, fare clic sulla scheda Disponibilità contenitore EJB.

    Viene visualizzata la pagina Disponibilità contenitore EJB.

  5. Selezionare la casella Servizio disponibilità abilitato per abilitare la disponibilità per il contenitore EJB.

    Per disabilitare la disponibilità, rimuovere il segno di spunta dalla casella Servizio disponibilità abilitato.

  6. L'opzione Tipo di persistenza HA è impostata su replicated.

    Si tratta del meccanismo di persistenza utilizzato quando la casella Servizio disponibilità abilitato è selezionata. La persistenza si basa su failover in un cluster. Non modificare questo valore.

  7. L'opzione Tipo di persistenza SFSB è impostata su file.

    Si tratta del meccanismo di persistenza utilizzato quando la casella Servizio disponibilità abilitato non è selezionata. La persistenza si basa sul file system. Non modificare questo valore.

    Il campo Posizione di memorizzazione sessione determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare Posizione di memorizzazione sessione, vedere To Configure EJB Settings.

  8. Per ripristinare valori predefiniti, fare clic sul pulsante Carica impostazioni predefinite.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per la disponibilità del contenitore EJB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.
