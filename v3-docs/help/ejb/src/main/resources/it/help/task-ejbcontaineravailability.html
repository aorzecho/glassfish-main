Configurare le impostazioni di disponibilità per il contenitore EJB

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare la configurazione da modificare.
  3. Nel nodo Configuration, selezionare il nodo Availability Service.

    Viene visualizzata la pagina Availability Service.

  4. Nella pagina Availability Service, fare clic sulla scheda EJB Container Availability.

    Viene visualizzata la pagina EJB Container Availability.

  5. Selezionare la casella Availability Service Enabled per abilitare la disponibilità per il contenitore EJB.

    Per disabilitare la disponibilità, rimuovere il segno di spunta dalla casella Availability Service Enabled.

  6. L'opzione HA Persistence Type è impostata su replicated.

    Si tratta del meccanismo di persistenza utilizzato quando la casella Availability Service Enabled è selezionata. La persistenza si basa su failover in un cluster. Non modificare questo valore.

  7. L'opzione SFSB Persistence Type è impostata su file.

    Si tratta del meccanismo di persistenza utilizzato quando la casella Availability Service Enabled non è selezionata. La persistenza si basa sul file system. Non modificare questo valore.

    Il campo Session Store Location determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare Session Store Location, vedere To Configure EJB Settings.

  8. Per ripristinare valori predefiniti, fare clic sul pulsante Load Defaults.
  9. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per la disponibilità del contenitore EJB.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario modificarne il campo Value.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  10. Fare clic su Save.
