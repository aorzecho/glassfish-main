Configurare le impostazioni del pool MDB

Il pool per i bean basati su messaggi è simile al pool di bean di sessione. Per impostazione predefinita, il contenitore include un pool di bean basati su messaggi.

Per regolare la configurazione del pool, attenersi alla procedura seguente.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare il nodo Contenitore EJB.

    Viene aperta la pagina EJB (Enterprise Java Beans).

  3. Fare clic sulla scheda Impostazioni MDB.

    Viene visualizzata la pagina Impostazioni del pool predefinito MDB.

  4. Nel campo Dimensione pool iniziale e minima, digitare il numero minimo di MDB da conservare nel pool.

    Il valore predefinito è 0.

  5. Nel campo Dimensione massima pool, digitare il numero massimo di bean che il contenitore può conservare nel pool in una volta sola.

    Il valore predefinito è 32.

  6. Nel campo Quantità di ridimensionamento pool, digitare il numero di bean che verranno rimossi dal pool se rimangono inattivi per un periodo di tempo superiore a quello specificato nel campo Timeout inattività pool.

    Il valore deve essere compreso tra 0 e il valore specificato nel campo Dimensione massima pool. Il valore predefinito è 8.

  7. Nel campo Timeout inattività pool, digitare il tempo, in secondi, durante il quale un bean può rimanere inattivo prima che venga rimosso dal pool.

    Dopo questo periodo di tempo, il bean viene eliminato. Un valore pari a 0 indica che un bean può rimanere inattivo per un periodo di tempo indeterminato. Il valore predefinito è 600.

  8. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per il pool di MDB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  10. Riavviare GlassFish Server.
