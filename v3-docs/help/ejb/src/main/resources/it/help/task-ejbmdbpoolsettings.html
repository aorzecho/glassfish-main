Configurare le impostazioni del pool di MDB

Il pool di MDB è simile al pool di bean di sessione. Per impostazione predefinita, il contenitore include un pool di MDB.

Per impostare la configurazione del pool, attenersi alla procedura seguente.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare il nodo EJB Container.

    Viene aperta la pagina EJB (Enterprise Java Beans).

  3. Fare clic sulla scheda MDB Settings.

    Viene visualizzata la pagina MDB Default Pool Settings.

  4. Nel campo Initial and Minimum Pool Size, digitare il numero minimo di MDB da conservare nel pool.

    Il valore predefinito è 0.

  5. Nel campo Maximum Pool Size, digitare il numero massimo di bean che il contenitore può conservare nel pool in una volta sola.

    Il valore predefinito è 32.

  6. Nel campo Pool Resize Quantity, digitare il numero di bean che verranno rimossi dal pool se rimangono inattivi per un periodo di tempo superiore a quello specificato nel campo Pool Idle Timeout.

    Il valore deve essere compreso tra 0 e il valore specificato nel campo Maximum Pool Size. Il valore predefinito è 8.

  7. Nel campo Pool Idle Timeout, digitare il tempo, in secondi, durante il quale un bean può rimanere inattivo prima che venga rimosso dal pool.

    Dopo questo periodo di tempo, il bean viene eliminato. Un valore pari a 0 indica che un bean può rimanere inattivo per un periodo di tempo indeterminato. Il valore predefinito è 600.

  8. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per il pool di MDB.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario modificarne il campo Value.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  9. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.

  10. Riavviare GlassFish Server.
