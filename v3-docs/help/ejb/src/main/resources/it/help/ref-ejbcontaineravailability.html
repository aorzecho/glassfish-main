Disponibilità contenitore EJB

Nella pagina Disponibilità contenitore EJB è possibile configurare le impostazioni di disponibilità per il contenitore EJB.

La pagina Disponibilità contenitore EJB contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Servizio disponibilità

Se la casella di controllo Abilitato è selezionata, la disponibilità è abilitata per il contenitore EJB.

Tipo di persistenza HA

Il meccanismo di persistenza utilizzato quando la casella di controllo Servizio disponibilità abilitato è selezionata. Il valore predefinito è replicated, che fornisce la persistenza in base al failover in un cluster. Non modificare questo valore.

Tipo di persistenza SFSB

Il meccanismo di persistenza utilizzato quando la casella di controllo Servizio disponibilità abilitato non è selezionata. Il valore predefinito è file, che fornisce la persistenza in base al file system. Non modificare questo valore.

Il campo Posizione area di memorizzazione sessione determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare il campo Posizione area di memorizzazione sessione, vedere To Configure EJB Settings.

Proprietà aggiuntive

Proprietà aggiuntive per la disponibilità del contenitore EJB.

In GlassFish Server non è definita alcuna proprietà aggiuntiva per la disponibilità del contenitore EJB.
