Disponibilità contenitore EJB

Nella pagina EJB Container Availability è possibile configurare le impostazioni di disponibilità del contenitore EJB.

La pagina EJB Container Availability contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Availability Service

Se la casella di controllo Enabled è selezionata, la disponibilità è abilitata per il contenitore EJB.

HA Persistence Type

Meccanismo di persistenza utilizzato quando la casella Availability Service Enabled è selezionata. Il valore predefinito è replicated, che fornisce persistenza basata su failover in un cluster. Non modificare questo valore.

SFSB Persistence Type

Meccanismo di persistenza utilizzato quando la casella Availability Service Enabled non è selezionata. Il valore predefinito è file, che fornisce persistenza basata sul file system. Non modificare questo valore.

Il campo Session Store Location determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare Session Store Location, vedere To Configure EJB Settings.

Additional Properties

Proprietà aggiuntive per EJB Container Availability.

In GlassFish Server non è definita alcuna proprietà aggiuntiva per EJB Container Availability.
