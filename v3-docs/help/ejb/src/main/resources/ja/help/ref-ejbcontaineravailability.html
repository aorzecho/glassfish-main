EJBコンテナの可用性

EJBコンテナの可用性設定を構成するには、「EJBコンテナの可用性」ページを使用します。

「EJBコンテナの可用性」ページには、次のオプションが含まれます。

デフォルトをロード

デフォルト値を持つ設定をデフォルト値に戻すボタン。デフォルト値を持たない設定は変更されません。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

可用性サービス

「有効」チェックボックスを選択すると、EJBコンテナで可用性が有効になります。

HA永続性のタイプ

「可用性サービス」で「有効」が選択された場合に使用される永続性メカニズム。デフォルトはreplicatedで、クラスタのフェイルオーバーに基づいて永続性が提供されます。この値は変更しないでください。

SFSB永続性のタイプ

「可用性サービス」で「有効」が選択されていない場合に使用される永続性メカニズム。デフォルトはfileで、ファイル・システムに基づいて永続性が提供されます。この値は変更しないでください。

SFSBチェックポイントの状態が保持される場所は、「セッション格納位置」フィールドによって決まります。「セッション格納位置」の設定方法は、「EJB設定を構成する」を参照してください。

追加プロパティ

EJBコンテナの可用性の追加プロパティ。

GlassFish ServerはEJBコンテナの可用性の追加プロパティを定義しません。
