EJB 컨테이너에 대한 가용성 설정을 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 변경 중인 구성을 선택합니다.
  3. 구성 노드 아래에서 가용성 서비스 노드를 선택합니다.

    가용성 서비스 페이지가 열립니다.

  4. 가용성 서비스 페이지에서 EJB 컨테이너 가용성 탭을 누릅니다.

    EJB 컨테이너 가용성 페이지가 열립니다.

  5. EJB 컨테이너에 대한 가용성을 활성화하려면 가용성 서비스 사용 가능 상자를 선택합니다.

    가용성을 비활성화하려면 가용성 서비스 사용 가능 상자에서 선택 표시를 제거합니다.

  6. HA 지속성 유형은 replicated입니다.

    이 지속성 메커니즘은 가용성 서비스 사용 가능을 선택한 경우 사용됩니다. 지속성은 클러스터의 페일오버를 기준으로 합니다. 이 값을 변경하지 마십시오.

  7. SFSB 지속성 유형은 file입니다.

    이 지속성 메커니즘은 가용성 서비스 사용 가능을 선택하지 않은 경우 사용됩니다. 지속성은 파일 시스템을 기준으로 합니다. 이 값을 변경하지 마십시오.

    세션 저장 위치 필드는 SFSB 검사점 상태가 지속되는 위치를 결정합니다. 세션 저장 위치를 설정하려면 EJB 설정을 구성하는 방법을 참조하십시오.

  8. 기본값을 복원하려면 기본값 로드 버튼을 누릅니다.
  9. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish 서버에서는 EJB 컨테이너 가용성에 대한 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  10. 저장을 누릅니다.
