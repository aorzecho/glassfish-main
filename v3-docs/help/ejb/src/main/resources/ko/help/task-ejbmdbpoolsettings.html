MDB 풀 설정을 구성하는 방법

Message-Driven Bean의 풀은 Session Bean의 풀과 비슷합니다. 기본적으로 컨테이너는 Message-Driven Bean의 풀을 관리합니다.

이 풀의 구성을 조정하려면 다음 단계를 수행합니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. EJB 컨테이너 노드를 선택합니다.

    EJB(Enterprise Java Bean) 페이지가 열립니다.

  3. MDB 설정 탭을 누릅니다.

    MDB 기본 풀 설정 페이지가 열립니다.

  4. 초기 및 최소 풀 크기 필드에 풀에서 유지할 최소 Message Bean 수를 입력합니다.

    기본값은 0입니다.

  5. 최대 풀 크기 필드에 컨테이너가 한 번에 풀에서 유지할 수 있는 최대 Bean 수를 입력합니다.

    기본값은 32입니다.

  6. 풀 크기 조정 개수 필드에 Bean이 풀 유휴 시간 초과 필드에 지정된 시간 이상 동안 유휴 상태인 경우 풀에서 제거되는 Bean의 개수를 입력합니다.

    이 값은 0보다 크거나 같고 최대 풀 크기 값보다 작아야 합니다. 기본값은 8입니다.

  7. 풀 유휴 시간 초과 필드에 풀의 Bean이 풀에서 제거되기 전까지 유휴 상태를 지속할 수 있는 시간을 초 단위로 입력합니다.

    이 시간이 지나면 Bean은 삭제됩니다. 이 값을 0으로 지정하면 Bean은 무기한 유휴 상태를 유지할 수 있습니다. 기본값은 600입니다.

  8. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish 서버에서는 MDB 풀에 대한 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  9. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.

  10. GlassFish 서버를 다시 시작합니다.
