EJB 컨테이너 가용성

EJB 컨테이너에 대한 가용성 설정을 구성하려면 EJB 컨테이너 가용성 페이지를 사용합니다.

EJB 컨테이너 가용성 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

가용성 서비스

사용 가능 확인란을 선택하면 EJB 컨테이너에 대해 가용성이 활성화됩니다.

HA 지속성 유형

가용성 서비스 사용 가능을 선택한 경우 이 지속성 메커니즘이 사용됩니다. 기본값은 클러스터의 페일오버를 기준으로 하여 지속성을 제공하는 replicated입니다. 이 값을 변경하지 마십시오.

SFSB 지속성 유형

가용성 서비스 사용 가능을 선택하지 않은 경우 이 지속성 메커니즘이 사용됩니다. 기본값은 파일 시스템을 기준으로 하여 지속성을 제공하는 file입니다. 이 값을 변경하지 마십시오.

세션 저장 위치 필드는 SFSB 검사점 상태가 지속되는 위치를 결정합니다. 세션 저장 위치를 설정하려면 EJB 설정을 구성하는 방법을 참조하십시오.

추가 등록 정보

EJB 컨테이너 가용성에 대한 추가 등록 정보입니다.

GlassFish Server에서는 추가 EJB 컨테이너 가용성 등록 정보를 정의하지 않습니다.
