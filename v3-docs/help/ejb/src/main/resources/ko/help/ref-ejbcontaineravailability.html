EJB 컨테이너 가용성

EJB 컨테이너 가용성 페이지에서는 EJB 컨테이너에 대한 가용성 설정을 구성할 수 있습니다.

EJB 컨테이너 가용성 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 단추입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

가용성 서비스

사용 체크 박스를 선택하면 EJB 컨테이너에 대해 가용성이 사용으로 설정됩니다.

HA 지속성 유형

이 지속성 방식은 [가용성 서비스 사용]을 선택한 경우에 사용됩니다. 기본값은 replicated이며, 이 값은 클러스터의 복구를 기준으로 지속성을 제공합니다. 이 값을 변경하지 마십시오.

SFSB 지속성 유형

이 지속성 방식은 [가용성 서비스 사용]을 선택하지 않은 경우에 사용됩니다. 기본값은 file이며, 이 값은 파일 시스템을 기준으로 지속성을 제공합니다. 이 값을 변경하지 마십시오.

[세션 저장 위치] 필드는 SFSB 체크포인트 상태가 지속되는 위치를 결정합니다. 세션 저장 위치를 설정하려면 EJB 설정을 구성하는 방법을 참조하십시오.

추가 속성

EJB 컨테이너 가용성에 대한 추가 속성입니다.

GlassFish Server에서는 추가 EJB 컨테이너 가용성 속성을 정의하지 않습니다.
