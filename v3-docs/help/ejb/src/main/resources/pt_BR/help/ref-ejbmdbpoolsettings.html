Configurações do Conjunto MDB padrão

Utilize a página Configurações do conjunto MDB padrão, para configurar o conjunto MDB.

A página Configurações do conjunto MDB padrão contém as opções a seguir.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Tamanho inicial e mínimo do conjunto

O número mínimo de beans de mensagem a serem mantidos no conjunto. O valor padrão é 0.

Tamanho máximo do conjunto

O número máximo de beans que o recipiente pode manter no conjunto ao mesmo tempo. O valor padrão é 32.

Quantidade de redimensionamento do conjunto

O número de beans que são removidos do conjunto, se estiverem ociosos por mais tempo do que o especificado no campo Tempo limite de conjunto ocioso. O valor precisa ser ao menos 0 e menor do que o valor de Tamanho máximo do conjunto. O valor padrão é 8.

Tempo limite de conjunto ocioso

O tempo, em segundos, que um conjunto pode permanecer ocioso antes que seja removido do conjunto. Após esta quantidade de tempo, o bean é destruído. Um valor de 0 significa que um bean pode permanecer ocioso indefinidamente. O valor padrão é 600.

Propriedades adicionais

Propriedades adicionais do conjunto MDB. O Servidor GlassFish não define quaisquer propriedades adicionais para o conjunto MDB.
