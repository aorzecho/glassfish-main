Disponibilidade do Contêiner EJB

Utilize a página Disponibilidade do Contêiner EJB para configurar as definições de disponibilidade do contêiner EJB.

A página Disponibilidade do Contêiner EJB contém as opções a seguir.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é somente para leitura.

Serviço de Disponibilidade

Se a caixa de seleção Ativado estiver selecionada, a disponibilidade está ativada para o contêiner EJB.

Tipo de Persistência de HA

Trata-se do mecanismo de persistência utilizado quando a opção Serviço de Disponibilidade Ativado estiver marcada. O default é replicado, o que gera persistência com base no failover de um cluster. Não altere este valor.

Tipo de Persistência de SFSB

Trata-se do mecanismo de persistência utilizado quando a opção Serviço de Disponibilidade Ativado não estiver marcada. O default é arquivo, o que gera persistência com base no sistema de arquivos. Não altere este valor.

O campo Local de Armazenamento da Sessão determina onde os estados do checkpoint SFSB são persistentes. Para definir o Local de Armazenamento da Sessão, consulte Para Configurar Definições do EJB.

Propriedades Adicionais

Propriedades adicionais da Disponibilidade do Contêiner EJB.

O GlassFish Server não define propriedades adicionais da Disponibilidade do Contêiner EJB.
