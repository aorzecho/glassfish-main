Disponibilidade de recipiente EJB

Utilize a página Disponibilidade de recipiente EJB para configurar as definições do recipiente EJB.

A página Disponibilidade de recipiente EJB contém as opções a seguir.

Carregar padrões

Botão para restaurar com os valores padrão as configurações que apresentam valores padrão. As configurações que não apresentam valores padrão não são alteradas.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Serviço de disponibilidade

Se a caixa de seleção Ativo estiver marcada, a disponibilidade está ativada para o recipiente EJB.

Tipo de persistência de alta disponibilidade

Mecanismo de persistência utilizado quando a opção Serviço de disponibilidade ativado estiver marcada. O padrão é replicated, que oferece persistência com base no failover em um cluster. Não altere este valor.

Tipo de persistência de SFSB

Mecanismo de persistência utilizado quando a opção Serviço de disponibilidade ativado não estiver marcada. O padrão é file, que oferece persistência com base no sistema de arquivos. Não altere este valor.

O campo Local de armazenamento da sessão determina onde os estados do ponto de verificação SFSB são persistentes. Para definir o Local de armazenamento da sessão, consulte Para configurar as definições do EJB.

Propriedades adicionais

Propriedades adicionais da disponibilidade do recipiente EJB.

O Servidor GlassFish não define propriedades adicionais da disponibilidade do recipiente EJB.
