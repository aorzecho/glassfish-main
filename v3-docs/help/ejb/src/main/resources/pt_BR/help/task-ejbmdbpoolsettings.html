Para configurar as definições do conjunto MDB

O conjunto para beans conduzidos por mensagem é semelhante ao conjunto para os beans de sessão. Por padrão, o recipiente mantém um conjunto de beans conduzidos por mensagem.

Para ajustar a configuração desse conjunto, execute as etapas a seguir.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione o nó Recipiente EJB.

    A página Enterprise Java Beans (EJB) é aberta.

  3. Clique na guia Configurações MDB.

    A página Configurações do conjunto MDB padrão é aberta.

  4. No campo Tamanho inicial e Mínimo do conjunto, digite o número mínimo de beans de mensagem a serem mantidos no conjunto.

    O valor padrão é 0.

  5. No campo Tamanho máximo do conjunto, digite o número máximo de beans que o recipiente pode manter no conjunto ao mesmo tempo.

    O valor padrão é 32.

  6. No campo Quantidade de redimensionamento do conjunto, digite o número de beans que são removidos do conjunto, se estiverem ociosos por mais tempo do que o especificado no campo Tempo limite do conjunto ocioso.

    O valor precisa ser ao menos 0 e menor do que o valor de Tamanho máximo do conjunto. O valor padrão é 8.

  7. No campo Tempo limite do conjunto ocioso, digite o tempo, em segundos, que um bean do conjunto pode permanecer ocioso antes que seja removido.

    Após essa quantidade de tempo, o bean é destruído. Um valor de 0 significa que um bean pode permanecer ocioso indefinidamente. O valor padrão é 600.

  8. (Opcional) Na seção Propriedades adicionais, especifique as propriedades adicionais.

    O Servidor GlassFish não define quaisquer propriedades adicionais para o conjunto MDB.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  9. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e, em seguida, em Salvar.

  10. Reinicie o Servidor GlassFish.
