MDB 預設池設定

使用 [MDB 預設池設定] 頁面可配置 MDB 池。

[MDB 預設池設定] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

池的初始大小和最小大小

保留在池中的最小訊息 Bean 數目。預設值為 0。

池的最大大小

容器一次可保留在池中的最大 Bean 數目。預設值為 32。

池設定大小數量

閒置時間超過 [池閒置逾時] 欄位中指定的時間時，要從池中移除的 Bean 數目。此值必須至少為 0 且小於 [池的最大大小] 值。預設值為 8。

池閒置逾時

Bean 可在池中維持閒置而不加以刪除的時間 (以秒為單位)。超過這段時間後，將會銷毀 Bean。0 值表示 Bean 可以無限期保持閒置。預設值為 600。

其他特性

MDB 池的其他特性。GlassFish Server 未定義 MDB 池的任何其他特性。
