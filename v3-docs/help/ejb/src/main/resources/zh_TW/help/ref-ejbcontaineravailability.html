EJB 容器可用性

使用 [EJB 容器可用性] 頁面可配置 EJB 容器的可用性設定。

[EJB 容器可用性] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

可用性服務

如果選取 [已啟用] 核取方塊，則會為 EJB 容器啟用可用性。

HA 持續性類型

核取 [啟用可用服務] 時使用的持續性機制。預設值為 replicated，其會根據叢集中的容錯移轉來提供持續性。請勿變更此值。

SFSB 持續性類型

未核取 [啟用可用服務] 時使用的持續性機制。預設值為 file，其會根據檔案系統來提供持續性。請勿變更此值。

[階段作業儲存位置] 欄位可決定儲存 SFSB 檢查點狀態的位置。若要設定 [階段作業儲存位置]，請參閱配置 EJB 設定。

其他特性

EJB 容器可用性的其他特性。

GlassFish Server 未定義任何其他 EJB 容器可用性特性。
