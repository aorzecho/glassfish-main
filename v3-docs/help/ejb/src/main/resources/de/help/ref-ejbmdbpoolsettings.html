MDB-Standardpooleinstellungen

Auf der Seite "MDB-Standardpooleinstellungen" können Sie den MDB-Pool konfigurieren.

Die Seite "MDB-Standardpooleinstellungen" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Initial and Minimum Pool Size (Ursprüngliche und minimale Poolgröße)

Die Mindestanzahl Nachrichten-Beans, die im Pool vorhanden sein müssen. Der Standardwert ist 0.

Maximum Pool Size (Maximale Poolgröße)

Die maximale Anzahl an Beans, die der Container parallel im Pool verwalten kann. Der Standardwert lautet 32.

Pool Resize Quantity (Anzahl der Größenänderungen des Pools)

Die Anzahl an Beans, die aus dem Pool entfernt werden, wenn sie sich länger als im Feld "Pool-Leerlaufzeit" angegeben im Leerlauf befinden. Der Wert muss größer oder gleich 0 und kleiner als der Wert für die maximale Poolgröße sein. Der Standardwert ist 8.

Pool-Leerlaufzeit

Die Anzahl Sekunden, die sich eine Bean im Pool im Leerlauf befinden kann, bevor sie aus dem Pool entfernt wird. Nach diesem Zeitraum wird die Bean zerstört. Der Wert 0 bedeutet, dass sich eine Bean zeitlich unbegrenzt im Leerlauf befinden kann. Der Standardwert ist 600.

Weitere Eigenschaften

Weitere Eigenschaften für den MDB-Pool. GlassFish Server definiert keine weiteren Eigenschaften für den MDB-Pool.
