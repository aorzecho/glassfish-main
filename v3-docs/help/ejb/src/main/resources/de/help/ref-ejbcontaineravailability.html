EJB-Container-Verfügbarkeit

Auf der Seite "EJB-Container-Verfügbarkeit" können Sie Verfügbarkeitseinstellungen für den EJB-Container konfigurieren.

Die Seite "EJB-Container-Verfügbarkeit" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Verfügbarkeitsdienst

Wenn das Kontrollkästchen "Aktiviert" aktiviert ist, wird die Verfügbarkeit für den EJB-Container aktiviert.

HA-Fortdauertyp

Der Fortdauermechanismums, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsdienst aktiviert ist. Der Standardwert lautet replicated, woraus die Fortdauer resultiert, die auf einem Failover in einem Cluster basiert. Ändern Sie diesen Wert nicht.

SFSB-Fortdauertyp

Der Fortdauermechanismums, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsdienst nicht aktiviert ist. Der Standardwert lautet file, woraus die Fortdauer resultiert, die auf dem Dateisystem basiert. Ändern Sie diesen Wert nicht.

Das Feld "Sitzungsspeicherort" bestimmt, wo SFSB-Fixpunktzustände beibehalten werden. Informationen zum Sitzungsspeicherort finden Sie unter To Configure EJB Settings.

Weitere Eigenschaften

Weitere Eigenschaften für die EJB-Container-Verfügbarkeit.

Bei GlassFish-Server sind keine weiteren Eigenschaften für die Eigenschaften der EJB-Container-Verfügbarkeit definiert.
