So werden Verfügbarkeitseinstellungen für den EJB-Container konfiguriert

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Wählen Sie die Konfiguration aus, die Sie ändern.

  3. Wählen Sie unter dem Konfigurationsknoten den Verfügbarkeitsservice-Knoten aus.

    Die Seite "Verfügbarkeitsservice" wird geöffnet.

  4. Klicken Sie auf der Seite "Verfügbarkeitsservice" auf die Registerkarte "EJB-Container-Verfügbarkeit".

    Die Seite "EJB-Container-Verfügbarkeit" wird geöffnet.

  5. Aktivieren Sie das Kontrollkästchen "Aktiviert" für den Verfügbarkeitsservice, um die Verfügbarkeit für den EJB-Container zu aktivieren.

    Um die Verfügbarkeit aufzuheben, deaktivieren Sie das Kontrollkästchen "Aktiviert" für den Verfügbarkeitsservice.

  6. Der HA-Persistenztyp lautet replicated.

    Dies ist der Persistenzmechanismus, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsservice aktiviert ist. Die Persistenz basiert auf einem Failover in einem Cluster. Ändern Sie diesen Wert nicht.

  7. Der SFSB-Persistenztyp lautet file.

    Dies ist der Persistenzmechanismus, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsservice nicht aktiviert ist. Die Persistenz basiert auf dem Dateisystem. Ändern Sie diesen Wert nicht.

    Das Feld "Sessionspeicherort" bestimmt, wo SFSB-Fixpunktzustände beibehalten werden. Informationen zum Verzeichnis des Sessionspeichers finden Sie unter So werden EJB-Einstellungen konfiguriert.

  8. Klicken Sie zum Wiederherstellen der Standardwerte auf die Schaltfläche "Standardeinstellungen laden".

  9. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    GlassFish Server definiert keine weiteren Eigenschaften für die EJB-Container-Verfügbarkeit.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".

  10. Klicken Sie auf "Speichern".
