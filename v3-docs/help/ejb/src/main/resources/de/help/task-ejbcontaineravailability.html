So konfigurieren Sie die Verfügbarkeitseinstellungen für den EJB-Container

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Konfiguration".
  2. Wählen Sie die Konfiguration aus, die Sie ändern.
  3. Wählen Sie unter dem Konfigurationsknoten den Verfügbarkeitsdienst-Knoten aus.

    Die Seite "Verfügbarkeitsdienst" wird geöffnet.

  4. Klicken Sie auf der Seite "Verfügbarkeitsdienst" auf die Registerkarte "EJB-Container-Verfügbarkeit".

    Die Seite "EJB-Container-Verfügbarkeit" wird geöffnet.

  5. Aktivieren Sie das Kontrollkästchen "Aktiviert" für den Verfügbarkeitsdienst, um die Verfügbarkeit für den EJB-Container zu aktivieren.

    Um die Verfügbarkeit aufzuheben, deaktivieren Sie das Kontrollkästchen "Aktiviert" für den Verfügbarkeitsdienst.

  6. Der HA-Fortdauertyp lautet replicated.

    Dies ist der Fortdauermechanismums, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsdienst aktiviert ist. Die Fortdauer basiert auf einem Failover in einem Cluster. Ändern Sie diesen Wert nicht.

  7. Der SFSB-Fortdauertyp lautet file.

    Dies ist der Fortdauermechanismums, der verwendet wird, wenn "Aktiviert" für den Verfügbarkeitsdienst nicht aktiviert ist. Die Fortdauer basiert auf dem Dateisystem. Ändern Sie diesen Wert nicht.

    Das Feld "Sitzungsspeicherort" bestimmt, wo SFSB-Fixpunktzustände beibehalten werden. Informationen zum Sitzungsspeicherort finden Sie unter To Configure EJB Settings.

  8. Klicken Sie zum Wiederherstellen der Standardwerte auf die Schaltfläche "Standardeinstellungen laden".
  9. (Optional) Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    GlassFish-Server definiert keine weiteren Eigenschaften für die EJB-Container-Verfügbarkeit.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.
    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.
    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".
  10. Klicken Sie auf "Speichern".
