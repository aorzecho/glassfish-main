Manager-Eigenschaften

Mit dem Sitzungsmanager können Sie konfigurieren, wie Sitzungen erstellt und aufgelöst werden, wo der Sitzungsstatus gespeichert wird und wie viele Sitzungen maximal zulässig sind.

Auf der Seite "Manager-Eigenschaften" können Sie Sitzungsmanager-Eigenschaften für den Webcontainer hinzufügen oder ändern.

Die Seite "Manager Properties" (Manager-Eigenschaften) umfasst die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Reap Interval (Reap-Intervall)

Die Anzahl der Sekunden, bis die Daten zur inaktiven Sitzung aus dem Speicher gelöscht werden. Der Standardwert ist 60.

Legen Sie hierfür einen niedrigeren Wert fest als für die Häufigkeit der Änderung von Sitzungsdaten. Beispiel: Für ein Servlet, das die Aufrufe einer stark frequentierten Website zählt, sollte ein möglichst niedriger Wert festgelegt werden (1 Sekunde), da anderenfalls bei jedem Neustart des Servers die letzten Aufrufe verloren gehen könnten.

Max Sessions (Maximale Sitzungen)

Die maximale Anzahl an Sitzungen, die sich im Cache befinden können. Beim Wert –1 ist die Anzahl der Sitzungen unbegrenzt. Der Standardwert ist –1.

Nach Erreichen des Grenzwerts wird bei dem Versuch, eine neue Sitzung zu erstellen, die Ausnahme IllegalStateException ausgelöst.

Session File Name (Sitzungsdateiname)

Der absolute oder relative Pfad zu dem Verzeichnis, in dem der Sitzungsstatus zwischen Anwendungsneustarts gespeichert wird, sofern die Speicherung des Status möglich ist. Wenn kein Pfad festgelegt wird, wird der Sitzungsstatus nicht gespeichert. Standardmäßig ist kein Pfad festgelegt.

Ein relativer Pfad wird relativ zum temporären Verzeichnis für diese Webanwendung angegeben.

Weitere Eigenschaften

Weitere Eigenschaften für den Sitzungsmanager. Der GlassFish-Server definiert keine zusätzlichen Eigenschaften für den Sitzungsmanager.
