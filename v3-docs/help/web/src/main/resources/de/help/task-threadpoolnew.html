So erstellen Sie einen Threadpool

Threadpools werden von Netzwerk-Zielgeräten und vom ORB verwendet. In der Regel nutzt ein Netzwerk-Zielgerät einen der beiden Threadpools, die bei der Installation von GlassFish-Server erstellt werden, und der ORB nutzt den anderen. Sie können beide standardmäßigen Threadpools bearbeiten und Einstellungen entsprechend Ihren jeweiligen Anwendungen oder Hardwaresystemen festlegen, oder Sie können einen Threadpool mit den betreffenden Einstellungen erstellen und diesen verwenden.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den Knoten "Threadpools".

    Die Seite "Threadpools" wird geöffnet.

  3. Klicken Sie auf der Seite "Threadpools" auf "Neu".

    Die Seite "Neuer Threadpool" wird geöffnet.

  4. Geben Sie in das Feld "Name" einen Namen für den Threadpool ein.
  5. Geben Sie im Feld "Klassenname" den vollständig qualifizierten Namen der Klasse ein, die den Threadpool implementiert.

    Der Standardwert ist com.sun.grizzly.http.StatsThreadPool .

  6. Geben Sie im Feld "Maximale Zahl in der Warteschlange" die maximale Anzahl von Meldungen ein, die in die Warteschlange gestellt werden können, bis Threads für ihre Verarbeitung zur Verfügung stehen.

    Der Wert –1 zeigt an, dass die Warteschlangengröße unbegrenzt ist. Der Standardwert ist 4096.

  7. Geben Sie im Feld "Maximale Größe des Threadpools" die Höchstzahl an Threads ein, die ein Threadpool enthalten soll.

    Der Standardwert ist 5. Dies ist der empfohlene Wert für Threadpools von Netzwerkzielgeräten.

  8. Geben Sie im Feld "Minimale Größe des Threadpools" die Mindestzahl an Threads ein, die ein Threadpool enthalten soll.

    Diese Threads werden bei der Instanziierung des Threadpools erstellt. Der Standardwert ist 2.

  9. Geben Sie im Feld "Leerlaufzeitüberschreitung" die maximale Dauer (in Sekunden) an, die ein Thread im Pool im Leerlauf bleiben kann.

    Nach Ablauf dieser Zeitspanne wird der Thread aus dem Pool entfernt. Der Standardwert ist 900.

  10. Klicken Sie auf "OK".
