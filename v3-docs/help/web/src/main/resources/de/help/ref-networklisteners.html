Netzwerk-Zielgeräte

Auf der Seite "Netzwerk-Zielgeräte" können Sie Netzwerk-Zielgeräte konfigurieren.

Wenn Sie GlassFish-Server erstmalig installieren, sind standardmäßig drei Netzwerk-Zielgeräte vorhanden: admin-listener, http-listener1 und http-listener2. Das Netzwerk-Zielgerät http-listener2 ist mit SSL konfiguriert und standardmäßig deaktiviert.

Für jedes Netzwerk-Zielgerät werden die folgenden Informationen zur Verfügung gestellt.

Name (Name)

Der Name des Netzwerk-Zielgeräts.

Port # (Port-Nr.)

Die Nummer des Ports, den das Netzwerk-Zielgerät überwacht.

Protocol (Protokoll)

Das vom Netzwerk-Zielgerät verwendete Protokoll.

Thread Pool (Thread-Pool)

Der vom Netzwerk-Zielgerät verwendete Threadpool.

Enabled (Aktiviert)

"True", wenn das Netzwerk-Zielgerät aktiviert ist, bzw. "false", wenn es nicht aktiviert ist.

Die Tabelle "Netzwerk-Zielgeräte" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Netzwerk-Zielgeräts.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Netzwerk-Zielgeräte.

Verwandte Aufgaben

  • So bearbeiten Sie ein Netzwerk-Zielgerät

  • So löschen Sie ein Netzwerk-Zielgerät

  • So erstellen Sie einen virtuellen Server
