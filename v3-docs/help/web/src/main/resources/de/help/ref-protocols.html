Protokolle

Auf der Seite "Protokolle" können Sie Netzwerkprotokolle konfigurieren.

Für jedes Protokoll stehen die folgenden Informationen zur Verfügung.

Name

Der Name des Protokolls.

Security Enabled (Sicherheit aktiviert)

Wahr, wenn die Sicherheit für das Protokoll aktiviert ist. Falsch, wenn die Sicherheit für das Protokoll nicht aktiviert ist.

Die Tabelle "Protokolle" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Protokolls.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Protokolle.

Verwandte Aufgaben

  • So wird ein Protokoll erstellt

  • So wird ein Protokoll bearbeitet

  • So werden SSL-Einstellungen für ein Protokoll bearbeitet

  • So werden HTTP-Einstellungen für ein Protokoll bearbeitet

  • So werden Dateicacheeinstellungen für ein Protokoll bearbeitet

  • So wird ein Protokoll gelöscht

Zugehörige asadmin-Befehle

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
