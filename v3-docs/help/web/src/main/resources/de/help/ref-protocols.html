Protokolle

Auf der Seite "Protokolle" können Sie Netzwerk-Zielgeräte konfigurieren.

Für jedes Protokoll stehen die folgenden Informationen zur Verfügung.

Name

Der Name des Protokolls.

Security Enabled (Sicherheit aktiviert)

Wahr, wenn die Sicherheit für das Protokoll aktiviert ist. Falsch, wenn die Sicherheit für das Protokoll nicht aktiviert ist.

Die Tabelle "Protokolle" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Protokolls.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Protokolle.

Verwandte Aufgaben

  • So erstellen Sie ein Protokoll

  • So bearbeiten Sie ein Protokoll

  • So bearbeiten Sie SSL-Einstellungen für ein Protokoll

  • So bearbeiten Sie HTTP-Einstellungen für ein Protokoll

  • So bearbeiten Sie Dateicache-Einstellungen für ein Protokoll

  • So löschen Sie ein Protokoll

Verwandte asadmin-Befehle

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
