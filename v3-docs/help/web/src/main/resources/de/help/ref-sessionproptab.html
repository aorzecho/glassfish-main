Sessioneigenschaften

Verwenden Sie die Seite "Sessioneigenschaften", um Eigenschaften für den Webcontainer hinzuzufügen oder zu ändern.

Die Seite "Sessioneigenschaften" umfasst die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Sessiontimeout

Gibt die maximale Anzahl Sekunden an, die eine inaktive Session gültig bleibt. Bei einem Wert kleiner oder gleich 0 laufen Sessions niemals ab. Der Standardwert ist 1800.

Weitere Eigenschaften

Weitere Eigenschaften für Webcontainer-Sessions. Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Für Webcontainersessions spezifische Eigenschaften.

Verwandte Aufgaben

  • So werden allgemeine Eigenschaften für den Webcontainer konfiguriert

  • So werden Sessioneigenschaften für den Webcontainer konfiguriert

  • So werden Managereigenschaften für den Webcontainer konfiguriert

  • So werden Speichereigenschaften für den Webcontainer konfiguriert

  • So werden Verfügbarkeitseinstellungen für den Webcontainer konfiguriert
