Allgemeine Eigenschaften

Verwenden Sie die Seite "General Properties" (Allgemeine Eigenschaften), um Eigenschaften für den Webcontainer hinzuzufügen oder zu ändern.

Der Webcontainer hostet Webanwendungen. Er stellt die Umgebung bereit, in der Servlets und JavaServer Pages (JSP-Dateien) ausgeführt werden. Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Spezifische Eigenschaften von Webcontainern.

Für jede Eigenschaft enthält die Seite "Allgemeine Eigenschaften" die folgenden Informationen.

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft.

Description (Beschreibung)

Eine Beschreibung der Eigenschaft.

Verwandte Aufgaben

  • So konfigurieren Sie die allgemeinen Eigenschaften des Webcontainers

  • So konfigurieren Sie die Sitzungseigenschaften des Webcontainers

  • So konfigurieren Sie die Manager-Eigenschaften des Webcontainers

  • So konfigurieren Sie die Speichereigenschaften des Webcontainers

  • So konfigurieren Sie Verfügbarkeitseinstellungen für den Webcontainer
