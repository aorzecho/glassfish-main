Für den Webcontainer spezifische Eigenschaften

Die folgende weitere allgemeine Eigenschaft steht für den Webcontainer zur Verfügung.

dispatcher-max-depth

Verhindert, das rekursive Einschließen- oder Weiterleiten-Anweisungen durch Festlegen einer maximalen Dispatch-Verschachtelungsebene eine Endlosschleife bilden. Wird diese Ebene überschritten, wird folgende Meldung in das Serverlog geschrieben:

Exceeded maximum depth for nested request dispatches

Der Standardwert lautet 20.
