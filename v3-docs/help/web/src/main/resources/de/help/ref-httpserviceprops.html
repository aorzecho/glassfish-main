Spezifische Eigenschaften des HTTP-Diensts

Folgende weitere Eigenschaften stehen für den HTTP-Dienst zur Verfügung. Wenn Sie diese Eigenschaften angeben, gelten sie für alle Netzwerk-Zielgeräte.

ssl-cache-entriese

Die Anzahl der zwischenzuspeichernden SSL-Sitzungen. Der Standardwert beträgt 10.000.

ssl3-session-timeout

Das Intervall, in dem SSL3-Sitzungen zwischengespeichert werden. Der Standardwert beträgt 86.400.

ssl-session-timeout

Das Intervall, in dem SSL2-Sitzungen zwischengespeichert werden. Der Standardwert beträgt 100.

proxyHandler

Der vollständig qualifizierte Klassenname einer benutzerdefinierten Implementierung der abstrakten Klasse com.sun.appserv.ProxyHandler, der es einer Backend-Instanz von GlassFish-Server ermöglicht, Informationen über die ursprüngliche Clientanforderung abzurufen, die von einem SSL-beendenden Proxyserver (z. B. einem Lastenausgleichssystem) abgefangen wurde. Eine Implementierung dieser abstrakten Klasse überprüft eine bestimmte Anforderung auf die benutzerdefinierten Anforderungsheader, über die der Proxyserver die Informationen über die ursprüngliche Clientanforderung an die Enterprise Server-Instanz kommunziert, und gibt diese Informationen anschließend an den Aufrufer zurück.

Die Standardimplementierung liest die IP-Adresse des Clients aus einem HTTP-Anforderungsheader namens Proxy-ip, die SSL keysize aus einem HTTP-Anforderungsheader namens Proxy-keysize und die SSL-Clientzertifikatskette aus einem HTTP-Anforderungsheader namens Proxy-auth-cert. Der Wert von Proxy-auth-cert muss die BASE-64-codierte Clientzertifikatskette ohne die Begrenzungen BEGIN CERTIFICATE und END CERTIFICATE enthalten; außerdem muss \n durch % d% a ersetzt werden.

Diese Einstellung wird nur verwendet, wenn das Kontrollkästchen "Autorisierungs-Durchlauf" auf der Seite "HTTP" des Netzwerk-Zielgeräteprotokolls aktiviert ist.

connectionTimeout

Gibt die Anzahl Sekunden an, die Netzwerk-Zielgeräte nach dem Akzeptieren einer Verbindung auf das Senden der Anforderungs-URI-Zeile warten. Der Standardwert ist 30.
