Spezifische Eigenschaften von Webcontainer-Sitzungen

Für eine Webcontainer-Sitzung stehen die folgenden Eigenschaften zur Verfügung.

enableCookies

Wenn der Wert auf true gesetzt ist, werden Cookies für die Sitzungsverfolgung verwendet. Der Standardwert ist true.

enableURLRewriting

Erlaubt das erneute Schreiben des URL. Ermöglicht die Sitzungsverfolgung anhand des erneuten Schreibens des URL, wenn der Browser keine Cookies akzeptiert. Sie müssen auch einen Aufruf encodeURL oder encodeRedirectURL im Servlet oder die Seite JavaServer Pages (JSP). Der Standardwert ist true.

idLengthBytes

Gibt die Zahl der Bytes in der Sitzungs-ID in diesem Webmodul an. Der Standardwert ist 128.
