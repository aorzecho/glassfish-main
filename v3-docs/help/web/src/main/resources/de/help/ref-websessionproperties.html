Für Webcontainersessions spezifische Eigenschaften

Für eine Webcontainer-Session stehen die folgenden Eigenschaften zur Verfügung.

enableCookies

Wenn der Wert auf true gesetzt ist, werden Cookies für das Session-Tracking verwendet. Der Standardwert ist true.

enableURLRewriting

Erlaubt das erneute Schreiben des URL. Ermöglicht das Session Tracking anhand des erneuten Schreibens des URL, wenn der Browser keine Cookies akzeptiert. Sie müssen auch einen Aufruf encodeURL oder encodeRedirectURL im Servlet oder der JavaServer Pages-(JSP-)Seite verwenden. Der Standardwert ist true.

idLengthBytes

Gibt die Zahl der Bytes in der Session-ID in diesem Webmodul an. Der Standardwert ist 128.
