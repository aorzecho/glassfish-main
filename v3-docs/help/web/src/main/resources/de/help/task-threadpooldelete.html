So löschen Sie einen Threadpool

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den Knoten "Threadpools".

    Die Seite "Threadpools" wird geöffnet.

  3. Aktivieren Sie auf der Seite "Threadpools" das Kontrollkästchen links neben dem zu löschenden Threadpool.
  4. Klicken Sie auf "Löschen".
