So löschen Sie ein Protokoll

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "Netzwerk konfigurieren".
  3. Wählen Sie unter dem Knoten "Netzwerkkonfiguration" den Knoten "Protokolle".

    Die Seite "Protokolle" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Protokolle" das Kontrollkästchen links neben dem Protokoll, das Sie löschen möchten.
  5. Klicken Sie auf "Löschen".
