Protokoll bearbeiten

Auf der Seite "Protokoll bearbeiten" können Sie die Einstellungen eines vorhandenen Netzwerkprotokolls ändern.

Die Seite "Protokoll bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Name

Der Name des Protokolls. Das Feld Name ist ein schreibgeschütztes Feld. Sie können den Namen nur festlegen, wenn Sie ein neues Protokoll erstellen.

Security

Wenn diese Option gewählt wird, wird die Sicherheit für das Protokoll aktiviert. Bei aktivierter Sicherheit können Sie die Sicherheitseinstellungen auf der Registerkarte "SSL" ändern.

Verwandte Aufgaben

  • So erstellen Sie ein Protokoll

  • So bearbeiten Sie ein Protokoll

  • So bearbeiten Sie SSL-Einstellungen für ein Protokoll

  • So bearbeiten Sie HTTP-Einstellungen für ein Protokoll

  • So bearbeiten Sie Dateicache-Einstellungen für ein Protokoll

  • So löschen Sie ein Protokoll

Verwandte asadmin-Befehle

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
