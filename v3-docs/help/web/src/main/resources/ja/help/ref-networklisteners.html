ネットワーク・リスナー

ネットワーク・リスナーを構成するには、「ネットワーク・リスナー」ページを使用します。

GlassFish Serverを初めてインストールする場合は、デフォルトでadmin-listener、http-listener1、およびhttp-listener2という名前の3つのネットワーク・リスナーが存在します。SSLで構成されるhttp-listener2は、デフォルトでは無効になっています。

ネットワーク・リスナーごとに、次の情報が表示されます。

名前

ネットワーク・リスナーの名前。

ポート#

ネットワーク・リスナーがリスニングしているポート番号。

プロトコル

ネットワーク・リスナーによって使用されるプロトコル。

スレッド・プール

ネットワーク・リスナーによって使用されるスレッド・プール。

有効

ネットワーク・リスナーが有効な場合はtrue、ネットワーク・リスナーが有効でない場合はfalseです。

「ネットワーク・リスナー」の表には、次のオプションも表示されます。

新規

新しいネットワーク・リスナーを作成するボタン。

削除

選択した1つまたは複数のネットワーク・リスナーを削除するボタン。
