仮想サーバー

仮想サーバーは、複数のインターネット・ドメイン名を同一の物理サーバーでホスティングするためのオブジェクトで、仮想ホストとも呼ばれます。同一物理サーバーにホスティングされるすべての仮想サーバーは、その物理サーバーのIP(Internet Protocol)アドレスを共有します。仮想サーバーは、サーバーのドメイン名(www.sun.comなど)と、GlassFish Serverが稼働するサーバーを関連付けます。

仮想サーバーを構成するには、「仮想サーバー」ページを使用します。

仮想サーバーごとに、次の情報が表示されます。

名前

仮想サーバーの名前。

状態

on、off、disabledのいずれか

デフォルトWebモジュール

仮想サーバーにデプロイされているその他のWebモジュールにマッピングできないすべてのリクエストに応答するデプロイ済Webモジュール(デプロイされている場合)。

「仮想サーバー」の表には、次のオプションも表示されます。

新規

新しい仮想サーバーを作成するボタン。

削除

選択した1つまたは複数の仮想サーバーを削除するボタン。
