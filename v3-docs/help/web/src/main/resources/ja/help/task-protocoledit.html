プロトコルを編集する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「ネットワーク構成」ノードを展開します。
  3. 「ネットワーク構成」ノードの下で、「プロトコル」ノードを選択します。

    「プロトコル」ページが開きます。

  4. 「プロトコル」ページで、編集するプロトコルの名前をクリックします。

    「プロトコルを編集」ページが開きます。

  5. 「セキュリティを有効」チェックボックスをオンにし、プロトコルに対してセキュリティーを有効にします。

    セキュリティーが有効な場合は、「SSL」タブを使用してセキュリティー設定を変更できます。

  6. 「保存」をクリックします。

    ヒント - デフォルト設定に戻すには、「デフォルトを読込み」ボタンをクリックし、「保存」を保存します。

  7. 「セキュリティを有効」チェックボックスを選択した場合は、「プロトコルの SSL 設定を編集する」の次の手順を実行します。
