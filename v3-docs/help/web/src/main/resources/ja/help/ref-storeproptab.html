ストアプロパティ

「ストアプロパティー」ページを使用して、セッション持続 (ストレージ) プロパティーを追加または変更します。

「ストアプロパティー」ページには、次のオプションが含まれます。

設定名

このページの設定を適用する設定の名前。 このフィールドは読み取り専用です。

Directory

各セッションファイルが書き込まれるディレクトリの絶対パス名または相対パス名。デフォルト値は、domain-dir/generated/jsp/j2ee-apps/appname/appname_war です。相対パスは、この Web アプリケーションの一時作業ディレクトリからの相対で指定します。

追加プロパティー

ストアの追加プロパティー。GlassFish Server はストアの追加プロパティーを定義しません。
