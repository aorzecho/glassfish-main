仮想サーバーを削除する

  1. ナビゲーション・ツリーで、「構成」ノードを展開します。

  2. Select the configuration that you are changing.

  3. 「構成」ノードの下で、「仮想サーバー」ノードを選択します。

    「仮想サーバー」ページが開きます。

  4. 「仮想サーバー」ページで、削除する仮想サーバーの名前の左側にあるチェックボックスを選択します。

  5. 「削除」をクリックします。

    デフォルトの仮想サーバー、つまり、__asadminとserverを削除することは推奨されません。
