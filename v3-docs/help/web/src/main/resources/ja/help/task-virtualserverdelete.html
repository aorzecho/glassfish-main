仮想サーバーを削除する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「仮想サーバー」ノードを選択します。

    「仮想サーバー」ページが開きます。

  3. 「仮想サーバー」ページで、削除する仮想サーバーの名前の左側にあるチェックボックスを選択します。
  4. 「削除」をクリックします。

    デフォルトの仮想サーバー、つまり、__asadmin と server を削除することは推奨されません。
