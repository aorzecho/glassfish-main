プロトコルの編集

既存のネットワークプロトコルの設定を変更するには、「プロトコルを編集」ページを使用します。

「プロトコルを編集」ページには、次のオプションが含まれます。

デフォルトを読込み

デフォルト値を持つ設定をデフォルト値に戻すボタン。デフォルト値を持たない設定は変更されません。

設定名

このページの設定を適用する設定の名前。 このフィールドは読み取り専用です。

名前

プロトコルの名前。「名前」フィールドは読み取り専用です。「名前」を指定できるのは、新しいプロトコルの作成時のみです。

Security

このオプションが選択されている場合、プロトコルに対してセキュリティーは有効です。セキュリティーが有効な場合は、「SSL」タブを使用してセキュリティー設定を変更できます。

  • 「プロトコルを削除する」
