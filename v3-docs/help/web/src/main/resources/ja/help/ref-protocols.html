プロトコル

ネットワークプロトコルを設定するには、「プロトコル」ページを使用します。

プロトコルごとに、次の情報が表示されます。

名前

プロトコルの名前。

セキュリティを有効

プロトコルに対してセキュリティーが有効な場合は True、無効な場合は False です。

「プロトコル」の表には、次のオプションも表示されます。

新規

新しいプロトコルを作成するボタン。

削除

選択した 1 つまたは複数のプロトコルを削除するボタン。

関連操作

  • 「プロトコルを作成する」

  • 「プロトコルを編集する」

  • 「プロトコルの SSL 設定を編集する」

  • 「プロトコルの HTTP 設定を編集する」

  • 「プロトコルのファイルキャッシュ設定を編集する」

  • 「プロトコルを削除する」

関連する asadmin コマンド

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
