トランスポート

トランスポートを設定するには、「トランスポート」ページを使用します。各ネットワークリスナーは、トランスポートに関連付けられます。

GlassFish サーバー をはじめてインストールする場合は、tcp という名前のトランスポートがデフォルトで存在します。TCP または UDP トランスポートを変更または作成するか、カスタムトランスポート実装を指定できます。

トランスポートごとに、次の情報が表示されます。

トランスポート名

トランスポートの名前。

クラス名

トランスポート実装のクラス名。

バイトバッファーのタイプ

トランスポートで使用する ByteBuffer のタイプ。

「トランスポート」の表には、次のオプションも表示されます。

新規

新しいトランスポートを作成するボタン。

削除

選択した 1 つまたは複数のトランスポートを削除するボタン。

関連操作

  • 「トランスポートを作成する」

  • 「トランスポートを編集する」

  • 「トランスポートを削除する」

関連する asadmin コマンド

  • create-transport(1)

  • list-transports(1)

  • delete-transport(1)
