一般プロパティ

「一般プロパティ」ページを使用して、Webコンテナのプロパティを追加または変更します。

Webコンテナは、Webアプリケーションをホストします。Webコンテナは、サーブレットとJavaServer Pages(JSP)ファイルの実行環境を提供します。使用可能なプロパティについては、Webコンテナに固有のプロパティを参照してください。

「一般プロパティ」ページには、プロパティごとに次の情報が表示されます。
