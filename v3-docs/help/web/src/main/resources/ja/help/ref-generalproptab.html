一般プロパティー

「一般プロパティー」ページを使用して、Web コンテナのプロパティーを追加または変更します。

Web コンテナは、Web アプリケーションをホストします。Web コンテナは、サーブレットと JavaServer Pages (JSP) ファイルの実行環境を提供します。利用可能なプロパティーの説明は、「Web コンテナに固有のプロパティー」を参照してください。

「一般プロパティー」ページには、プロパティーごとに次の情報が表示されます。

名前

プロパティーの名前。

値

プロパティーの値

説明

プロパティーの説明。
