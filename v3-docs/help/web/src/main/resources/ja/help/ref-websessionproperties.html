Webコンテナ・セッションに固有のプロパティ

Webコンテナ・セッションで使用できるプロパティは次のとおりです。

enableCookies

trueに設定されている場合、セッション・トラッキングにクッキーを使用します。デフォルト値はtrueです。

enableURLRewriting

URLの書換えを有効にします。これにより、ブラウザがクッキーを受け入れない場合にURLを書き換える方法でセッション・トラッキングを行います。また、encodeURLまたはencodeRedirectURL呼出しをサーブレットやJavaServer Pages(JSP)ページで使用する必要があります。デフォルト値はtrueです。

idLengthBytes

このWebモジュールのセッションIDのバイト数を指定します。デフォルト値は128です。
