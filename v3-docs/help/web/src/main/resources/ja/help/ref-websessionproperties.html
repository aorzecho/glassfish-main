Web コンテナセッションに固有のプロパティー

Web コンテナセッションで使用できるプロパティーは次のとおりです。

enableCookies

true に設定されている場合、セッショントラッキングにクッキーを使用します。デフォルト値は true です。

enableURLRewriting

URL の書き換えを有効にします。これにより、ブラウザがクッキーを受け入れない場合に URL を書き換える方法でセッショントラッキングを行います。また、encodeURL または encodeRedirectURL 呼び出しをサーブレットや JavaServer Pages (JSP) ページで使用する必要があります。デフォルト値は true です。

idLengthBytes

この Web モジュールのセッション ID のバイト数を指定します。デフォルト値は 128 です。
