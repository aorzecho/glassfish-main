セッションプロパティ

「セッションプロパティー」ページを使用して、Web コンテナセッションのプロパティーを追加または変更します。

「セッションプロパティー」ページには、次のオプションが含まれます。

デフォルトを読込み

デフォルト値を持つ設定をデフォルト値に戻すボタン。デフォルト値を持たない設定は変更されません。

設定名

このページの設定を適用する設定の名前。 このフィールドは読み取り専用です。

セッションタイムアウト

アクティブでないセッションが有効な状態を維持できる最大時間を秒単位で指定します。0 以下の値を設定すると、セッションは有効期限切れになりません。デフォルト値は 1800 です。

追加プロパティー

Web コンテナセッションの追加プロパティー。利用可能なプロパティーの説明は、「Web コンテナセッションに固有のプロパティー」を参照してください。
