Web コンテナに固有のプロパティー

Web コンテナで使用できる追加のプロパティーは次のとおりです。

dispatcher-max-depth

入れ子の最大ディスパッチレベルを設定し、再帰的インクルード文、またはフォワード文で無限ループが作成されないようにします。このレベルを超過すると、次のメッセージがサーバーログに書き込まれます。 

Exceeded maximum depth for nested request dispatches

デフォルト値は 20 です。
