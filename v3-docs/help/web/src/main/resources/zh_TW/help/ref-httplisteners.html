HTTP 偵聽程式

使用「HTTP 偵聽程式」頁面可組態 HTTP 偵聽程式。

HTTP 偵聽程式是一種網路偵聽程式。如果要存取進階功能，您可以使用「網路偵聽程式」頁面來編輯 HTTP 偵聽程式。

當您第一次安裝 GlassFish Server 時，預設已存在三個 HTTP 偵聽程式，名稱為 admin-listener、http-listener1 和 http-listener2。使用 SSL 組態的 http-listener2 預設會停用。

每個 HTTP 偵聽程式均提供以下資訊。

名稱

HTTP 偵聽程式的名稱。

連接埠號碼

HTTP 偵聽程式偵聽的連接埠號碼。

位址

HTTP 偵聽程式使用的 IP 位址。

已啟用

HTTP 偵聽程式啟用時為 True，HTTP 偵聽程式未啟用時為 False。

HTTP 偵聽程式表也包含下列選項。

新建

建立新 HTTP 偵聽程式的按鈕。

刪除

刪除一或多個所選 HTTP 偵聽程式的按鈕。
