編輯通訊協定

使用 [編輯通訊協定] 頁面可修改現有網路通訊協定的設定。

[編輯通訊協定] 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

配置名稱

此頁面上的設定要套用到的配置名稱。此為唯讀欄位。

名稱

通訊協定的名稱。[名稱] 欄位是唯讀欄位。您僅能在建立新的通訊協定時指定名稱。

安全性

如果選取此選項，則啟用通訊協定的安全性。如果已啟用安全性，您可以使用 [SSL] 標籤修改安全性設定。
