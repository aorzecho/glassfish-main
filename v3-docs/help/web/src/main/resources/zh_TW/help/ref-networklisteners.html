網路偵聽程式

使用 「網路偵聽程式」 頁面可組態網路偵聽程式。

當您第一次安裝 GlassFish Server 時，預設已存在三個網路偵聽程式，名稱為 admin-listener、http-listener1 和 http-listener2。使用 SSL 組態的 http-listener2 預設會停用。

對於每個網路偵聽程式，均提供以下資訊。

名稱

網路偵聽程式的名稱。

連接埠號碼

網路偵聽程式偵聽的連接埠號碼。

協定

網路偵聽程式所使用的協定。

執行緒集區

網路偵聽程式所使用的執行緒集區。

已啟用

網路偵聽程式啟用時為 True，網路偵聽程式未啟用時為 False。

網路偵聽程式表也包含下列選項。

新建

建立新網路偵聽程式的按鈕。

刪除

刪除一或多個選取的網路偵聽程式的按鈕。
