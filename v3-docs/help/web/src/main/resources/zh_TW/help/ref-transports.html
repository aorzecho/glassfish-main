傳輸

使用 [傳輸] 頁面可配置傳輸。每個網路偵聽程式都會和一種傳輸關聯。

當您第一次安裝 GlassFish Server 時，預設已存在一個 TCP 傳輸，名稱為 tcp。您可以修改或建立 TCP 或 UDP 傳輸，或提供自訂的傳輸實作。

對於每個傳輸，均提供以下資訊。

傳輸名稱

傳輸的名稱。

類別名稱

傳輸實作的類別名稱。

ByteBuffer 類型

要與傳輸共同使用的 ByteBuffer 類型。

傳輸表也包含下列選項。

新建

建立新傳輸的按鈕。

刪除

刪除一或多個選取傳輸的按鈕。

相關作業

  • 建立傳輸

  • 編輯傳輸

  • 刪除傳輸

相關的 asadmin 指令
