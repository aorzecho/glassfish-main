新建虛擬伺服器

使用 「新建虛擬伺服器」 頁面可建立虛擬伺服器。

「新建虛擬伺服器」 頁面包含下列選項。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

ID

內部顯示的虛擬伺服器識別碼。而不會提供給 HTTP 用戶端。提供給 HTTP 用戶端的主機名稱必須在 「主機」 欄位中指定。

主機

一或多個執行伺服器的機器之主機名稱。使用已在網路的 DNS 伺服器註冊 (對於 UNIX 系統，則是已在 /etc/hosts 檔案中註冊) 的實際主機名稱或虛擬主機名稱。預設值為系統特性值 ${com.sun.aas.hostName}。

狀態

虛擬伺服器所需的狀態。值可能為下列其中之一：

開

虛擬伺服器處於使用中狀態。此為預設值。

關

虛擬伺服器處於非使用中狀態。若嘗試存取伺服器則傳回錯誤碼 404 (資源無法使用)。

已停用

虛擬伺服器已停用。若嘗試存取伺服器則傳回錯誤碼 403 (拒絕執行請求)。

SSO

指定單次登入運作方式是否繼承自 HTTP 服務 (已啟用或已停用)。依預設，會繼承自 HTTP 服務。若啟用此選項，會針對此虛擬伺服器上為相同範圍組態的 Web 應用程式啟用單次登入。若停用此選項，則會針對此虛擬伺服器停用單次登入，且使用者必須在使用虛擬伺服器上的每個應用程式時進行分別認證。

網路偵聽程式

與此伺服器關聯的一或多個網路偵聽程式 (如果有)。

預設 Web 模組

已建置的 Web 模組 (如果有)，此模組將回應所有無法對映至其他 Web 模組 (已建置到虛擬伺服器) 的請求。如果未指定預設 Web 模組，則使用具有空環境根目錄的 Web 模組。如果沒有環境根目錄為空的 Web 模組，則會建立和使用系統預設的 Web 模組。

記錄檔

將顯示此虛擬伺服器記錄訊息所在的檔案之路徑名稱。依預設，記錄訊息會顯示在 domain-dir/logs/server.log。

文件根

伺服器根文件目錄的絕對路徑。預設值為 domain-dir/docroot。

存取記錄

指定存取記錄是否繼承自 HTTP 服務 (已啟用或已停用)。依預設，會繼承自 HTTP 服務。

目錄

伺服器存取記錄的絕對目錄路徑。預設值為 domain-dir/logs/access。

其他特性

虛擬伺服器的其他特性。

如需可用特性的相關資訊，請參閱虛擬伺服器特定的特性。
