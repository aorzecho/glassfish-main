通訊協定

使用 [通訊協定] 頁面可配置網路通訊協定。

對於每個通訊協定，均提供以下資訊。

名稱

通訊協定的名稱。

啟用安全性

如果已啟用通訊協定的安全性，則為 True；如果未啟用通訊協定的安全性，則為 False。

通訊協定表也包含下列選項。

新建

建立新通訊協定的按鈕。

刪除

刪除一或多個選取通訊協定的按鈕。
