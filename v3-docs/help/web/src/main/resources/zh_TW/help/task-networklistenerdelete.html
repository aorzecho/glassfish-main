刪除網路偵聽程式

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [網路配置] 節點。
  3. 在 [網路配置] 節點下，選取 [網路偵聽程式] 節點。

    [網路偵聽程式] 頁面會隨即開啟。

  4. 在 [網路偵聽程式] 頁面上，選取您要刪除的網路偵聽程式左側的核取方塊。
  5. 按一下 [刪除]。
