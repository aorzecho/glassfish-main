一般特性

使用 「一般特性」 頁面可增加或修改 Web 容器的特性。

Web 容器託管 Web 應用程式。它提供用於執行 Servlet 和 JavaServer Pages (JSP) 檔案的環境。如需可用特性的說明，請參閱Web 容器特定的特性。

對於每個特性，「一般特性」 頁面均提供下列資訊。

名稱

特性的名稱。

值

特性的值。

說明

特性的說明。
