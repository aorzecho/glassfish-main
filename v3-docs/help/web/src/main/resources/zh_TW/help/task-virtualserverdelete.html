刪除虛擬伺服器

  1. 在瀏覽樹狀結構中，展開 「組態」 節點。

  2. Select the configuration that you are changing.

  3. 在 「組態」 節點下，選取 「虛擬伺服器」 節點。

    「虛擬伺服器」 頁面會隨即開啟。

  4. 在 「虛擬伺服器」 頁面上，選取要刪除的虛擬伺服器名稱左側之核取方塊。

  5. 按一下 「刪除」。

    不建議刪除預設虛擬伺服器 __asadmin 和 server。
