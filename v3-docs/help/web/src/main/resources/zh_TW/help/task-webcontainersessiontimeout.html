配置 Web 容器的階段作業特性

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 選取您要變更的配置。
  3. 在配置的節點下，選取 [Web 容器] 節點。

    [一般特性] 頁面會隨即開啟。

  4. 在 [一般特性] 頁面上，按一下 [階段作業特性] 標籤。

    [階段作業特性] 頁面會隨即開啟。

  5. 在 [階段作業逾時] 欄位中，鍵入非使用中階段作業保持有效的最大秒數。

    預設值為 1800。若此值設為 0 或負值，則階段作業永遠不會過期。

  6. (可選擇) 在 [其他特性] 區段中，指定其他特性。

    如需可用特性的說明，請參閱Web 容器階段作業特有的特性。

    • 若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。
    • 若要修改特性，請編輯特性的 [值] 欄位。
    • 若要刪除特性，請選取您要刪除之特性 [名稱] 欄位左側的核取方塊，然後按一下 [刪除特性] 按鈕。
  7. 按一下 [儲存]。

    提示 - 若要回到預設設定，請按一下 [載入預設值] 按鈕，然後按一下 [儲存]。
