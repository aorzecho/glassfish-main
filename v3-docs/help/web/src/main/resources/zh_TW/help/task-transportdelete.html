刪除傳輸

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [網路配置] 節點。
  3. 在 [網路配置] 節點下，選取 [傳輸] 節點。

    此時將會開啟 [傳輸] 頁面。

  4. 在 [傳輸] 頁面上，選取您要刪除的傳輸左側的核取方塊。
  5. 按一下 [刪除]。
