編輯執行緒集區

使用 「編輯執行緒集區」 頁面可修改現有的執行緒集區。

「編輯執行緒集區」 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

名稱

執行緒集區的名稱。「名稱」 欄位是唯讀欄位。您僅能在建立新的執行緒集區時指定名稱。

類別名稱

實作此執行緒集區的類別之完全合格名稱。預設值為 com.sun.grizzly.http.StatsThreadPool。

佇列最大大小

在有執行緒可處理訊息之前，可排入佇列的最大訊息數目。-1 值表示佇列大小沒有限制。預設值為 4096。

最大執行緒集區大小

執行緒集區中執行緒的最大數目。預設值為 5。

最小執行緒集區大小

執行緒集區中執行緒的最小數目。預設值為 2。

閒置執行緒逾時

執行緒可以在集區中閒置的最長時間 (以秒為單位)。一旦超過此時間，即從集區中移除該執行緒。預設值為 900。
