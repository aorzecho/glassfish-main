編輯執行緒池

使用 [編輯執行緒池] 頁面可修改現有的執行緒池。

[編輯執行緒池] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

名稱

執行緒池的名稱。[名稱] 欄位是唯讀欄位。您僅能在建立新的執行緒池時指定名稱。

類別名稱

實作此執行緒池的類別之完全合格名稱。預設值為 com.sun.grizzly.http.StatsThreadPool。

佇列最大大小

在有執行緒可處理訊息之前，可排入佇列的最大訊息數目。–1 值表示佇列大小沒有限制。預設值為 4096。

最大執行緒池大小

執行緒池中執行緒的最大數目。預設值為 5。

最小執行緒池大小

執行緒池中執行緒的最小數目。預設值為 2。

閒置執行緒逾時

執行緒可以在池中閒置的最長時間 (以秒為單位)。一旦超過此時間，即從池中移除該執行緒。預設值為 900。
