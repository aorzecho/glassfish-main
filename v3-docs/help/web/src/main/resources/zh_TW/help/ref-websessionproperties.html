Web 容器階段作業特有的特性

以下是適用於 Web 容器階段作業的特性。

enableCookies

若設定為 true，則會使用 Cookie 追蹤階段作業。預設值為 true。

enableURLRewriting

啟用 URL 重新寫入。當瀏覽器不接受 Cookie 時，此功能可以 URL 重新寫入的方式追蹤階段作業。您還必須在 Servlet 中使用 encodeURL 或 encodeRedirectURL 呼叫，或是使用 JavaServer Pages (JSP) 頁面。預設值為 true。

idLengthBytes

指定此 Web 模組的階段作業 ID 中的位元組數目。預設值為 128。
