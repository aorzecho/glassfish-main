Suprimir un Transporte

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, amplíe el nodo Configuración de Red.

  3. En el nodo Configuración de Red, seleccione el nodo Transportes.

    Se abrirá la página Transportes.

  4. En la página Transportes, marque la casilla de verificación situada a la izquierda del transporte que desee suprimir.

  5. Haga clic en Suprimir.
