Propiedades generales

Utilice la página Propiedades generales para agregar o modificar propiedades del contenedor web.

Contenedor web que aloja aplicaciones web. Proporciona el entorno en el que se ejecutan los servlets y los archivos de JavaServer Pages (JSP). Para obtener una descripción de las propiedades disponibles, consulte Las propiedades específicas del contenedor web .

La página Propiedades generales contiene la siguiente información para cada propiedad.

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad.

Descripción

Una descripción de la propiedad.
