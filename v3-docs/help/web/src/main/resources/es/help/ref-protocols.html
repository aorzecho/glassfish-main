Protocolos

Utilice la página Protocolos para configurar los protocolos de red.

Para cada protocolo, aparece la siguiente información.

Nombre

El nombre del protocolo.

Seguridad habilitada

True si la seguridad está habilitada para el protocolo, o false si no está habilitada.

La tabla Protocolos contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo protocolo.

Borrar

Botón para eliminar uno o varios protocolos seleccionados.
