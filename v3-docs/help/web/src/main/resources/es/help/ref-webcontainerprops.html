Las propiedades específicas del contenedor web

La siguiente propiedad general adicional está disponible para el contenedor web.

dispatcher-max-depth

Impide que la inclusión de recursivos o las instrucciones de reenvío creen un bucle infinito configurando el nivel de distribución de anidado máximo. Si se supera este límite, se escribe el siguiente mensaje en el registro del servidor: 

Exceeded maximum depth for nested request dispatches

El valor predeterminado es 20.
