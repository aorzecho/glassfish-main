Editar Protocolo

Utilice la página Editar Protocolo para modificar la configuración de un protocolo de red existente.

La página Editar Protocolo contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

El nombre del protocolo. El campo Nombre es un campo de sólo lectura. Únicamente puede especificar el nombre al crear un nuevo protocolo.

Seguridad

Si se selecciona esta opción, se activa la seguridad para el protocolo. Si la seguridad se encuentra activada, puede utilizar el separador SSL para modificar la configuración de seguridad.
