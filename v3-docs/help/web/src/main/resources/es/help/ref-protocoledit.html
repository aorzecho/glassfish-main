Editar protocolo

Utilice la página Editar protocolo para modificar la configuración de un protocolo de red existente.

La página Editar protocolo contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer las opciones que tienen valores por defecto a estos. Las opciones que no tienen valores por defecto no se modifican.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre

El nombre del protocolo. El campo Nombre es un campo de sólo lectura. Únicamente puede especificar el nombre al crear un nuevo protocolo.

Security

Si se selecciona esta opción, se habilita la seguridad para el protocolo. Si la seguridad se encuentra habilitada, puede utilizar la ficha SSL para modificar la configuración de seguridad.
