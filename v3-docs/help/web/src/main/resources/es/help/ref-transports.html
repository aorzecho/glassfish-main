Transportes

Utilice la página Transportes para configurar los transportes. Cada escucha de red se encuentra asociada a un transporte.

Al instalar por primera vez GlassFish Server, un transporte TCP existe de forma predeterminada con el nombre tcp. Puede modificar o crear un transporte TCP o UDP, o bien proporcionar una implementación de transporte personalizada.

Para cada transporte, aparece la siguiente información.

Nombre de transporte

El nombre del transporte.

Nombre de clase

El nombre de clase de la implementación del transporte.

Tipo de búfer en byte

El tipo de búfer que se debe usar con el transporte.

La tabla Transportes contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo transporte.

Borrar

Botón para eliminar uno o varios transportes seleccionados.
