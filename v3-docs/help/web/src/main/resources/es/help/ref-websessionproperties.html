Propiedades específicas de las sesiones del contenedor web

Las siguientes propiedades adicionales están disponibles para una sesión del contenedor web.

enableCookies

Si se establece entrue, utiliza cookies para el seguimiento de sesión. El valor predeterminado es true.

enableURLRewriting

Permite reescribir la URL. Esto proporciona el seguimiento de la sesión por medio de la reescritura de la URL cuando el navegador no acepta cookies. También puede utilizar una llamada a encodeURL o a encodeRedirectURL en el servlet o en JavaServer Pages (JSP). El valor predeterminado es true.

idLengthBytes

Especifica el número de bytes en este Id. de sesión del módulo web. El valor predeterminado es 128.
