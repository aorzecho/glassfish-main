Para eliminar un servidor virtual

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Servidores virtuales.

    Se abrirá la página Servidores virtuales.

  3. En la página Servidores virtuales, marque la casilla de verificación situada a la izquierda del nombre del servidor virtual que va a eliminar.
  4. Haga clic en Eliminar.

    No se recomienda eliminar ninguno de los servidores predeterminados __asadmin ni server.
