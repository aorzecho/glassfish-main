Propiedades de Almacén

Utilice la página Propiedades de Almacén para agregar o modificar propiedades de persistencia de sesión (almacén) para el contenedor web.

La página Propiedades de Almacén contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Directorio

El nombre de la ruta de acceso absoluta o relativa del directorio en el que se escriben los archivos de sesión individuales. El valor por defecto es domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Una ruta de acceso relativa se relaciona con el directorio de trabajo temporal para esta aplicación web.

Propiedades Adicionales

Propiedades adicionales del almacén. Servidor GlassFish no define ninguna propiedad adicional para el almacén.
