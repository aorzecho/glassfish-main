Propiedades de almacén

Utilice la página Propiedades de almacén para agregar o modificar propiedades de persistencia de sesión (almacén) para el contenedor web.

La página Almacenar propiedades contiene las siguientes opciones.

Nombre de la configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Directorio

El nombre de la ruta absoluta o relativa del directorio en el que se escriben los archivos de sesión individuales. El valor predeterminado es domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Una ruta relativa se relaciona con el directorio de trabajo temporal para esta aplicación web.

Propiedades adicionales

Propiedades adicionales del almacén. GlassFish Server no define ninguna propiedad adicional para el almacén.
