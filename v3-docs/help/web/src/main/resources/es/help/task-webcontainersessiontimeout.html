Configurar Propiedades de Sesión para el Contenedor Web

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración que esté cambiando.

  3. En el nodo de configuración, seleccione el nodo Contenedor Web.

    Se abre la página Propiedades Generales.

  4. En la página Propiedades Generales, haga clic en el separador Propiedades de Sesión.

    Se abrirá la página Propiedades de Sesión.

  5. En el campo Timeout de Sesión, escriba el número máximo de segundos que una sesión inactiva sigue siendo válida.

    El valor por defecto es 1800. Si el valor se establece en 0 o menos, las sesiones no caducan nunca.

  6. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    Para obtener una descripción de las propiedades disponibles, consulte Propiedades Específicas de las Sesiones del Contenedor Web.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, seleccione la casilla de verificación a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  7. Haga clic en Guardar.

    Sugerencia:

    Para volver a la configuración por defecto, haga clic en el botón Cargar por Defecto y, a continuación, en Guardar.
