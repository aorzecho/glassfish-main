Propiedades de Gestor

El gestor de sesiones proporciona los medios para configurar el modo de crear y destruir las sesiones, el lugar en el que se ubica el estado de sesión y el número máximo de sesiones disponibles.

Utilice la página Propiedades de Gestor para agregar o modificar propiedades de gestor de sesiones para el contenedor web.

La página Propiedades de Gestor contiene las opciones siguientes.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Intervalo de Recolección

Número de segundos que transcurren antes de que los datos de sesión inactiva se suprimen del almacén. El valor por defecto es 60.

Establezca este valor de forma que sea menor que la frecuencia con la que cambian los datos de la sesión. Por ejemplo, este valor debe ser lo más bajo posible (1 segundo) para un servlet con contador de visitas en un sitio web con visitas frecuentes; de lo contrario, podría perder las últimas visitas cada vez que reinicie el servidor.

Número Máximo de Sesiones

El número máximo de sesiones que puede haber en la caché. Un valor de-1 indica que no hay un límite para el número de sesiones. El valor por defecto es -1.

Una vez alcanzado el límite, un intento de crear una nueva sesión provoca la emisión de una excepción IllegalStateException.

Nombre de Archivo de Sesión

La ruta de acceso absoluta o relativa al directorio en el que se conserva el estado de la sesión entre reinicios de la aplicación, en caso de que sea posible conservar el estado. Si no se define esta ruta de acceso, el estado de la sesión no se conserva. Por defecto, esta de acceso no se encuentra definida.

Una ruta de acceso relativa se relaciona con el directorio temporal para esta aplicación web.

Propiedades Adicionales

Propiedades adicionales del gestor de sesiones. Servidor GlassFish no define ninguna propiedad adicional para el gestor de sesiones.
