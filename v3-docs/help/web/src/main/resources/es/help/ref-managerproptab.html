Propiedades de administrador

El administrador de sesiones proporciona los medios para configurar el modo de crear y destruir las sesiones, dónde se ubica el estado de sesión y el número máximo de sesiones disponibles.

Utilice la página Propiedades de administrador para agregar o modificar propiedades del administrador de sesión para el contenedor web.

La página Propiedades de administrador contiene las opciones siguientes.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre de la configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Intervalo de cierre

Número de segundos que transcurren antes de que los datos de sesión inactiva se eliminen del almacén. El valor predeterminado es 60.

Establezca este valor de forma que sea menor que la frecuencia con la que cambian los datos de la sesión. Por ejemplo, este valor debe ser lo más bajo posible (1 segundo) para un servlet con contador de visitas en un sitio web con visitas frecuentes; de lo contrario, podría perder las últimas visitas cada vez que reinicie el servidor.

Número máximo de sesiones

El número máximo de sesiones que puede haber en la caché. Un valor de -1 indica que no hay un límite para el número de sesiones. El valor predeterminado es -1.

Una vez alcanzado el límite, un intento de crear una nueva sesión provoca la emisión de una excepción IllegalStateException.

Nombre de archivo de sesión

La ruta absoluta o relativa al directorio en el que el se conserva el estado de la sesión entre reinicios de la aplicación, en caso de que sea posible conservar el estado. Si no se define esta ruta, el estado de la sesión no se conserva. De forma predeterminada, esta ruta no se encuentra definida.

Una ruta relativa se relaciona con el directorio temporal para esta aplicación web.

Propiedades adicionales

Propiedades adicionales del administrador de sesión. GlassFish Server no define ninguna propiedad adicional para el administrador de sesión.
