Listeners HTTP

Utilice la página Listeners HTTP para configurar los listeners HTTP.

Un listener HTTP es un tipo de listener de red. Para acceder a las funciones avanzadas, puede editar un listener HTTP mediante las páginas del listener de red.

Al instalar por primera vez Servidor GlassFish, existen por defecto tres listeners HTTP denominados admin-listener, http-listener1 y http-listener2. El listener de red http-listener2, que está configurado con SSL, está desactivado por defecto.

Para cada listener HTTP se ofrece la siguiente información.

Nombre

El nombre del listener HTTP.

Número de Puerto

El número de puerto en el que se el listener HTTP está efectuando la escucha.

Dirección

La dirección IP utilizada por el listener HTTP.

Activado

El valor true indica que el listener HTTP está activado y el valor false indica que no lo está.

La tabla de listeners HTTP contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo listener HTTP.

Suprimir

Botón para suprimir uno o varios listeners HTTP seleccionados.
