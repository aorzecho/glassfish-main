Propiedades Específicas del Servicio HTTP

Las siguientes propiedades adicionales están disponibles para el servicio HTTP. Si especifica estas propiedades, estas se aplicarán a todas las Listeners de Red.

ssl-cache-entriese

El número de sesiones SSL que desea almacenar en caché. El valor por defecto es 10000.

ssl3-session-timeout

El intervalo en el que las sesiones SSL3 se almacenan en caché. El valor por defecto es 86,400.

ssl-session-timeout

El intervalo en el que las sesiones SSL2 se almacenan en caché. El valor por defecto es 100.

proxyHandler

El nombre de clase completo de una implementación personalizada de la clase abstracta com.sun.appserv.ProxyHandler que permite a una instancia back-end de Servidor GlassFish recuperar información relativa a la solicitud original del cliente interceptada por un servidor proxy con terminación SSL (por ejemplo, un equilibrador de carga). Una implementación de esta clase abstracta inspecciona una solicitud específica en lo que concierne a las cabeceras de solicitud mediante las cuales el servidor proxy comunica la información acerca de la solicitud original del cliente a la instancia de Enterprise Server y devuelve esa información al emisor de la llamada.

La implementación por defecto lee la dirección de IP del cliente desde una cabecera de solicitud HTTP denominada Proxy-ip, los keysize de SSL desde una cabecera de solicitud HTTP denominada Proxy-keysize y la cadena del certificado de cliente SSL desde una cabecera de solicitud HTTP denominada Proxy-auth-cert . El valor Proxy-auth-cert debe contener la cadena de certificado de cliente con codificación BASE-64 sin los límites BEGIN CERTIFICATE y END CERTIFICATE y con \n sustituido por % d% a.

Esta configuración se utiliza únicamente si la casilla de verificación Traspaso de Autenticación de la página HTTP de los protocolos de listener de red se ha definido como true.

connectionTimeout

Especifica el número de segundos que esperan los listeners de red después de aceptar una conexión para que se presente la línea URI de solicitud. El valor por defecto es 30.
