Servidores Virtuales

Un servidor virtual, a veces denominado host virtual, es un objeto que permite al mismo servidor físico alojar varios nombres de dominio de Internet. Todos los servidores virtuales alojados en el mismo servidor físico comparten la dirección de Protocolo de Internet (IP) de dicho servidor físico. Un servidor virtual asocia el nombre de un dominio de un servidor (como www.sun.com) con el servidor particular en el que se ejecuta Servidor GlassFish.

Utilice la página Servidores Virtuales para configurar los servidores virtuales.

Para cada servidor virtual, se muestra la siguiente información.

Nombre

El nombre del servidor virtual.

Estado

Bien activado, desactivado o desactivado.

Módulo Web por Defecto

El módulo web desplegado (si lo hay) que debe responder a todas las solicitudes que no pueden asignarse a otros módulos web desplegados en el servidor virtual.

La tabla Servidores Virtuales contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo servidor virtual.

Suprimir

Botón para suprimir uno o varios servidores virtuales seleccionados.
