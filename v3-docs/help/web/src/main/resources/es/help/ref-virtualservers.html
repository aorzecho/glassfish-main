Servidores virtuales

Un servidor virtual, a veces denominado host virtual, es un objeto que permite al mismo servidor físico alojar varios nombres de dominio de Internet. Todos los servidores virtuales alojados en el mismo servidor físico comparten la dirección de Protocolo de Internet (IP) de dicho servidor físico. Un servidor virtual asocia el nombre de un dominio de un servidor (como www.sun.com) con el servidor particular en el que se ejecuta GlassFish Server.

Utilice la página Servidores virtuales para configurar los servidores virtuales.

Para cada servidor virtual, aparece la siguiente información.

Nombre

El nombre del servidor virtual.

Estado

Bien on, off o disabled.

Módulo web predeterminado

El módulo web implementado (si lo hay) que debe responder a todas las solicitudes que no pueden asignarse a otros módulos web implementados en el servidor virtual.

La tabla Servidores virtuales contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo servidor virtual.

Borrar

Botón para eliminar uno o varios servidores virtuales seleccionados.
