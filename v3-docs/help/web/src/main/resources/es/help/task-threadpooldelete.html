Cómo eliminar un conjunto de subprocesos

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Conjuntos de subprocesos.

    Se abrirá la página Conjuntos de subprocesos.

  3. En la página Conjuntos de subprocesos, marque la casilla de verificación situada a la izquierda del conjunto de subprocesos que se va a eliminar.
  4. Haga clic en Delete.
