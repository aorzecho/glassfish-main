Cómo editar un protocolo

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. En el nodo Configuración, expanda el nodo Configuración de red.
  3. En el nodo Configuración de red, seleccione el nodo Protocolos.

    Se abrirá la página Protocolos.

  4. En la página Protocolos, haga clic en el nombre del protocolo que está editando.

    Se abrirá la página Editar protocolo.

  5. Seleccione la casilla de verificación Seguridad habilitada para habilitar la seguridad del protocolo.

    Si la seguridad se encuentra habilitada, puede utilizar la ficha SSL para modificar la configuración de seguridad.

  6. Haga clic en Guardar.

    Consejo - Para volver a los valores de configuración predeterminados, haga clic en el botón Cargar predeterminados y, a continuación, en Guardar.

  7. Si la casilla de verificación Seguridad habilitada se encuentra seleccionada, siga los pasos que se indican en Cómo editar la configuración SSL para un protocolo.
