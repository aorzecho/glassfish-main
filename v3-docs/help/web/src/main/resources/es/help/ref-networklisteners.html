Escuchas de red

Utilice la página Escuchas de red para configurar las escuchas de red.

Al instalar por primera vez GlassFish Server, existen de forma predeterminada tres escuchas de red denominadas admin-listener, http-listener1 y http-listener2. La escucha de red http-listener2, que está configurada con SSL, se encuentra deshabilitada de forma predeterminada.

En cada escucha de red aparece la siguiente información.

Nombre

El nombre de la escucha de red.

Puerto #

El número de puerto en el que se está efectuando la escucha de red.

Protocolo

El protocolo que utiliza la escucha de red.

Conjunto de subprocesos

El conjunto de subprocesos que utiliza la escucha de red.

Habilitado

El valor true indica que la escucha de red se encuentra habilitada y el valor false indica que no se encuentra habilitada.

La tabla de escuchas de red contiene además las siguientes opciones.

Nueva

Botón para crear una nueva escucha de red.

Borrar

Botón para eliminar una o varias escuchas de red seleccionadas.
