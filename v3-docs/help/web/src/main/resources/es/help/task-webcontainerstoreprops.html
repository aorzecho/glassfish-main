Configurar Propiedades de Almacén para el Contenedor Web

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración que esté cambiando.

  3. En el nodo de configuración, seleccione el nodo Contenedor Web.

    Se abre la página Propiedades Generales.

  4. En la página Propiedades Generales, haga clic en el separador Propiedades de Almacén.

    Se abrirá la página Propiedades de Almacén.

  5. En el campo Directorio, escriba el nombre de la ruta de acceso absoluta o relativa del directorio en el que se escriben los archivos de sesión individuales.

    El valor por defecto es domain-dir/generated/jsp/j2ee-apps/appname/appname_war.

    Una ruta de acceso relativa se relaciona con el directorio de trabajo temporal para esta aplicación web.

    Este valor únicamente se aplica si el Tipo de Persistencia es file. Para definir el Tipo de Persistencia, consulte Configurar Opciones de Disponibilidad para el Contenedor Web.

  6. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    Servidor GlassFish no define ninguna propiedad adicional para el almacén.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, seleccione la casilla de verificación a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  7. Haga clic en Guardar.
