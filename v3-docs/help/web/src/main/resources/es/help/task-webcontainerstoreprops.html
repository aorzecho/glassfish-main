Cómo configurar propiedades de almacenamiento para el contenedor web

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. Seleccione la configuración que esté cambiando.
  3. En el nodo de configuración, seleccione el nodo Contenedor web.

    Se abre la página Propiedades generales.

  4. En la página Propiedades generales, haga clic en la ficha Propiedades del almacén.

    Se abrirá la página Propiedades del almacén.

  5. En el campo Directorio, escriba el nombre de la ruta absoluta o relativa del directorio en el que se escriben los archivos de sesión individuales.

    El valor predeterminado es dir_dominio/generated/jsp/j2ee-apps/appname/appname_war.

    Una ruta relativa se relaciona con el directorio de trabajo temporal para esta aplicación web.

    Este valor sólo se aplica si el Tipo de persistencia es file. Para definir el Tipo de persistencia, consulte Cómo configurar las opciones de disponibilidad para el contenedor web.

  6. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Servidor GlassFish no define ninguna propiedad adicional para el almacén.

    • Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.
    • Para modificar una propiedad, edite su campo Valor.
    • Para eliminar una propiedad, seleccione la casilla de verificación a la izquierda del campo Nombre de la propiedad que está eliminando y, a continuación, haga clic en el botón Eliminar propiedades.
  7. Haga clic en Guardar.
