Cómo eliminar una escucha de red

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. En el nodo Configuración, expanda el nodo Configuración de red.
  3. En el nodo Configuración de red, seleccione el nodo Escuchas de red.

    Se abrirá la página Escuchas de red.

  4. En la página Escuchas de red, marque la casilla de verificación situada a la izquierda de la escucha de red que desee eliminar.
  5. Haga clic en Eliminar.
