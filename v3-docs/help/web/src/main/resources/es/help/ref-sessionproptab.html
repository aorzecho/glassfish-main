Propiedades de Sesión

Utilice la página Propiedades de Sesión para agregar o modificar las propiedades de la sesión del contenedor web.

La página Propiedades de Sesión contiene las opciones siguientes.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Timeout de Sesión

Especifica el número máximo de segundos que una sesión inactiva sigue siendo válida. Si el valor se establece en 0 o menos, las sesiones no caducan nunca. El valor por defecto es 1800.

Propiedades Adicionales

Propiedades adicionales de las sesiones del contenedor web. Para obtener una descripción de las propiedades disponibles, consulte Propiedades Específicas de las Sesiones del Contenedor Web.
