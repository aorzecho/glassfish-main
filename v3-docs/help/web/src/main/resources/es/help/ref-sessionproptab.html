Propiedades de sesión

Utilice la página Propiedades de sesión para agregar o modificar las propiedades de la sesión del contenedor web.

La página Propiedades de sesión contiene las opciones siguientes.

Cargar valores predeterminados

Botón para restablecer las opciones que tienen valores por defecto a estos. Las opciones que no tienen valores por defecto no se modifican.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Tiempo de espera de sesión

Especifica el número máximo de segundos que una sesión inactiva sigue siendo válida. Si el valor se establece en 0 o menos, las sesiones no caducan nunca. El valor predeterminado es 1800.

Propiedades adicionales

Propiedades adicionales de las sesiones del contenedor web. Para obtener una descripción de las propiedades disponibles, consulte Propiedades específicas de las sesiones del contenedor web.
