网络侦听器

使用“网络侦听器”页面可以配置网络侦听器。

默认情况下，当您第一次安装 GlassFish Server 时，存在三个网络侦听器，名称为 admin-listener、http-listener1 和 http-listener2。默认情况下，通过 SSL 配置的 http-listener2 被禁用。

对于每个网络侦听器，将提供以下信息。

名称

网络侦听器名称。

端口号

网络侦听器侦听的端口号。

协议

网络侦听器使用的协议。

线程池

网络侦听器使用的线程池。

已启用

如果启用网络侦听器，则为 true；如果未启用网络侦听器，则为 false。

“网络侦听器”表还包含以下选项。

新建

用于创建新网络侦听器的按钮。

删除

用于删除一个或多个选定网络侦听器的按钮。
