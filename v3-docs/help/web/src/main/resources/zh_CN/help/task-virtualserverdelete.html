删除虚拟服务器

  1. 在导航树中，展开“配置”节点。

  2. Select the configuration that you are changing.

  3. 在“配置”节点下，选择“虚拟服务器”节点。

    此时将打开“虚拟服务器”页。

  4. 在“虚拟服务器”页上，选中要删除的虚拟服务器名称左侧的复选框。

  5. 单击“删除”。

    建议不要删除以下任一默认虚拟服务器：__asadmin 和 server。
