管理器属性

会话管理器提供了用于配置会话的创建和销毁方式、存储会话状态的位置以及可用会话的最大数量的方法。

使用“管理器属性”页面可以为 Web 容器添加或修改会话管理器属性。

“管理器属性”页面包含以下选项：

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

清除时间间隔

从库中删除非活动会话数据之前经过的秒数。默认值为 60。

将此值设置为低于会话数据更改频率的值。例如，对于一个频繁访问的网站的命中计数器 Servlet，此值应该尽可能低（1 秒）；否则，在每次重新启动服务器时可能丢失最近几次命中。

最大会话数

高速缓存中可以存储的最大会话数。如果值为 -1，则表示会话数不受限制。默认值为 –1。

到达限制后，如果尝试创建新会话，则会导致系统抛出 IllegalStateException。

会话文件名

应用程序重新启动之间保留会话状态的目录的绝对或相对路径（如果可以保留状态的话）。如果未设置此路径，则会话状态不会保留。默认情况下，未设置此路径。

相对路径是相对于此 Web 应用程序的临时目录的路径。

其他属性

会话管理器的其他属性。GlassFish Server 不定义会话管理器的任何其他属性。
