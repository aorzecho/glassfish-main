传输

使用“传输”页可以配置传输。每个网络监听程序与一个传输关联。

默认情况下，当您第一次安装 GlassFish Server 时，存在一个 TCP 传输，名称为 tcp。您可以修改或创建 TCP 或 UDP 传输，或提供定制传输实现。

对于每个传输，将提供以下信息：

传输名称

传输名称。

类名

传输实现的类名。

字节缓冲区类型

要与传输搭配使用的 ByteBuffer 的类型。

“传输”表还包含以下选项：

新建

用于创建新传输的按钮。

删除

用于删除一个或多个所选传输的按钮。

相关任务

  • 创建传输

  • 编辑传输

  • 删除传输

相关的 asadmin 命令

  • create-transport(1)

  • list-transports(1)

  • delete-transport(1)
