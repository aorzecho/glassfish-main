编辑协议

使用“编辑协议”页面可以修改现有网络协议的设置。

“编辑协议”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

名称

协议名称。“名称”字段是只读字段。只能在创建新协议时指定“名称”。

安全性

如果选中此选项，则将为协议启用安全性。如果启用安全性，则可以使用 "SSL" 选项卡来修改安全性设置。
