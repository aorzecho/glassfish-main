删除协议

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“网络配置”节点。
  3. 在“网络配置”节点下，选择“协议”节点。

    此时将打开“协议”页面。

  4. 在“协议”页面上，选中要删除的协议左侧的复选框。
  5. 单击“删除”。
