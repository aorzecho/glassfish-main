编辑协议

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“网络配置”节点。
  3. 在“网络配置”节点下，选择“协议”节点。

    此时将打开“协议”页面。

  4. 在“协议”页面上，单击要编辑的协议的名称。

    此时将打开“编辑协议”页面。

  5. 选择“启用安全性”复选框以为协议启用安全性。

    如果启用安全性，则可以使用 "SSL" 选项卡来修改安全性设置。

  6. 单击“保存”。

    提示 - 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。

  7. 如果选中“启用安全性”复选框，请按照编辑协议的 SSL 设置中的步骤操作。
