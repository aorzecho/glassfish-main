删除 HTTP 监听程序

  1. 在导航树中，展开“配置”节点。

  2. 在“配置”节点下，展开“HTTP 服务”节点。

  3. 在“网络配置”节点下，选择“HTTP 监听程序”节点。

    此时将打开“HTTP 监听程序”页。

  4. 在“HTTP 监听程序”页上，选中要删除的 HTTP 监听程序左侧的复选框。

  5. 单击“删除”。
