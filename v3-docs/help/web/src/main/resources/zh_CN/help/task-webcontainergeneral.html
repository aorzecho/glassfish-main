为 Web 容器配置常规属性

  1. 在导航树中，展开“配置”节点。

  2. 选择您要更改的配置。

  3. 在配置的节点下，选择“Web 容器”节点。

    此时将打开“常规属性”页。

  4. 在“其他属性”部分中，指定其他属性。

    有关可用属性的说明，请参阅特定于 Web 容器的属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  5. 单击“保存”。
