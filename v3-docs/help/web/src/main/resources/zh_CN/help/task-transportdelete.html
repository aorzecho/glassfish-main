删除传输

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“网络配置”节点。
  3. 在“网络配置”节点下，选择“传输”节点。

    此时将打开“传输”页面。

  4. 在“传输”页面上，选中要删除的传输左侧的复选框。
  5. 单击“删除”。
