特定于 HTTP 服务的属性

以下附加属性可用于 HTTP 服务。如果指定这些属性，它们将应用到所有网络监听程序。

ssl-cache-entriese

要高速缓存的 SSL 会话的数量。默认值为 10,000。

ssl3-session-timeout

高速缓存 SSL3 会话的时间间隔。默认值为 86,400。

ssl-session-timeout

高速缓存 SSL2 会话的时间间隔。默认值为 100。

proxyHandler

com.sun.appserv.ProxyHandler 抽象类的定制实现的全限定类名，后端 GlassFish Server 实例可以通过它检索关于 SSL 终端代理服务器 (例如负载平衡器) 拦截的原始客户机请求的信息。此抽象类的实现将在给定请求中检查定制请求标头，这样，代理服务器可将有关原始客户机请求的信息传送至 Enterprise Server 实例，并将该信息返回给其调用方。

默认实现将从名为 Proxy-ip 的 HTTP 请求标头中读取客户机 IP 地址，从名为 Proxy-keysize 的 HTTP 请求标头中读取 SSL keysize，并从名为 Proxy-auth-cert 的 HTTP 请求标头中读取 SSL 客户机证书链。Proxy-auth-cert 值必须包含 BASE-64 编码的客户机证书链，且无 BEGIN CERTIFICATE 和 END CERTIFICATE 边界，并将 \n 替换为 % d% a。

仅当网络监听程序协议的“HTTP”页的“传递验证”复选框设置为“真”时，才使用此设置。

connectionTimeout

指定在接受一个连接后，请求 URI 行被提交之前网络监听程序需等待的秒数。默认值为 30。
