为 Web 容器配置会话属性

  1. 在导航树中，展开“配置”节点。

  2. 选择您要更改的配置。

  3. 在配置的节点下，选择“Web 容器”节点。

    此时将打开“常规属性”页。

  4. 在“常规属性”页上，单击“会话属性”选项卡。

    此时将打开“会话属性”页。

  5. 在“会话超时”字段中，键入非活动会话保持有效的最大秒数。

    默认值为 1800。如果该值设置为 0 或更小，则会话永不过期。

  6. 在“其他属性”部分中，指定其他属性。

    有关可用属性的说明，请参阅特定于 Web 容器会话的属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  7. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。
