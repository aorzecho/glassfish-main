Supprimer un transport

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Config réseau.
  3. Sous le nœud Configuration de réseau, sélectionnez le nœud Transports.

    La page Transports s’ouvre.

  4. Sur la page Transports, sélectionnez la case à gauche du transport que vous supprimez
  5. Cliquez sur Supprimer.
