Supprimer un serveur virtuel

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Serveurs virtuels.

    La page Serveurs virtuels s’ouvre.

  3. Sur la page Serveurs virtuels, sélectionnez la case à gauche du nom du serveur virtuel que vous supprimez.
  4. Cliquez sur Supprimer.

    La suppression de l’un des serveurs virtuels par défaut, __asadmin et server, est déconseillée.
