Supprimer un processus d'écoute HTTP

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Service HTTP.

  3. Sous le noeud Configuration réseau, sélectionnez le noeud Processus d'écoute réseau.

    La page Processus d'écoute HTTP apparaît.

  4. Sur la page Processus d'écoute HTTP, cochez la case à gauche du processus d'écoute HTTP que vous supprimez.

  5. Cliquez sur Supprimer.
