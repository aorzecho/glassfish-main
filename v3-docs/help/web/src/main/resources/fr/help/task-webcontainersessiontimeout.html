Configurer les propriétés de session pour le conteneur Web

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez la configuration à modifier.
  3. Sous le nœud Configuration, sélectionnez le nœud Conteneur Web.

    La page Propriétés générales s’affiche.

  4. Dans la page Propriétés générales, cliquez sur l’onglet Propriétés des sessions.

    La page Propriétés des sessions s’affiche.

  5. Dans le champ Délai d’expiration de session, tapez le nombre maximum de secondes pendant lesquelles une session inactive restera valide.

    La valeur par défaut est 1800. Si la valeur est définie sur 0 ou moins, les sessions n’expirent jamais.

  6. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    Pour obtenir une description des propriétés disponibles, consultez Propriétés spécifiques aux sessions de conteneur Web.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  7. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
