Propriétés générales

La page Propriétés générales vous permet d’ajouter ou de modifier les propriétés du conteneur Web.

Le conteneur Web héberge des applications Web. Il fournit l’environnement dans lequel sont executés les servlets et les fichiers JavaServer Pages (JSP). Pour une description des propriétés disponibles, consultez la rubrique Propriétés spécifiques au conteneur Web .

Pour chaque propriété, la page Propriétés générales contient les informations suivantes.

Nom

Nom de la propriété.

Valeur

Valeur de la propriété.

Description

Une description de la propriété.
