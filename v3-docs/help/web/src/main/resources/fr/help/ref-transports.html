Transports

Utilisez la page Transports pour configurer les transports. Chaque auditeur de réseau est associé à un transport.

Lorsque vous installez pour la première fois GlassFish Server, un transport TCP existe par défaut avec le nom tcp. Vous pouvez modifier ou créer un transport TCP ou UDP ou fournir une implémentation de transport personnalisée.

Pour chaque transport, les informations suivantes sont fournies.

Nom du transport

Le nom du transport.

Nom de classe

Le nom de classe de l’implémentation de transport.

Type de tampon de byte

Le type de tampon d’octet qui sera utilisé avec le transport.

Le tableau Transports contient également les options suivantes.

Nouveau

Bouton pour créer un nouveau transport.

Supprimer

Bouton pour supprimer un ou plusieurs transports sélectionnés.
