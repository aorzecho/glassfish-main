Protocoles

Utilisez la page Protocoles pour configurer les protocoles réseau.

Pour chaque protocole, les informations suivantes sont fournies.

Nom

Nom du protocole.

Sécurité activée

True si la sécurité est activée pour le protocole ou False si la sécurité n’est pas activée pour le protocole.

Le tableau Protocoles contient également les options suivantes.

Nouveau

Bouton pour créer un nouveau protocole.

Supprimer

Bouton pour supprimer un ou plusieurs protocoles sélectionnés.
