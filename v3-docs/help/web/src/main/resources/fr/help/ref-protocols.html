Protocoles

Utilisez la page Protocoles pour configurer les protocoles réseau.

Pour chaque protocole, les informations suivantes sont fournies.

Nom

Nom du protocole.

Sécurité activée

Valeur True si la sécurité est activée pour le protocole ou False si elle ne l'est pas.

La table Protocoles contient également les options suivantes :

Nouveau

Bouton permettant de créer un protocole.

Supprimer

Bouton permettant de supprimer les protocoles sélectionnés.
