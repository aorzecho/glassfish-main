Serveurs virtuels

Un serveur virtuel, parfois appelé hôte virtuel, est un objet qui permet au même serveur physique d’héberger plusieurs noms de domaines Internet. Tous les serveurs virtuels hébergés sur le même serveur physique partagent la même adresse IP (Internet Protocol) de ce serveur. Un serveur virtuel associe le nom de domaine d’un serveur (tel que www.sun.com) au serveur sur lequel GlassFish Server est exécuté.

La page Serveurs virtuels vous permet de configurer ceux-ci.

Pour chaque serveur virtuel, les informations suivantes sont fournies.

Nom

Nom du serveur virtuel.

État

on, off ou disabled.

Module Web par défaut

Module Web déployé (le cas échéant) qui doit répondre à toutes les requêtes ne pouvant pas être mappées à d’autres modules Web déployés sur le serveur virtuel.

Le tableau Serveurs virtuels contient également les options suivantes.

Nouveau

Bouton pour créer un nouveau serveur virtuel.

Supprimer

Bouton pour supprimer un ou plusieurs serveurs virtuels sélectionnés.
