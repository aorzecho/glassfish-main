Processus d'écoute réseau

La page Processus d'écoute réseau permet de configurer les processus d'écoute réseau.

Lorsque vous installez GlassFish Server pour la première fois, il existe trois processus d'écoute réseau par défaut portant les noms suivants : admin-listener, http-listener1 et http-listener2. http-listener2, qui est configuré avec SSL, est désactivé par défaut.

Pour chaque processus d'écoute réseau, les informations suivantes sont fournies.

Nom

Nom du processus d'écoute réseau.

Port #

Numéro de port sur lequel le processus d'écoute réseau écoute.

Protocole

Protocole utilisé par le processus d'écoute réseau.

Pool de threads

Pool de threads utilisé par le processus d'écoute réseau.

Activé

Valeur True si le processus d'écoute réseau est activé ou False s'il ne l'est pas.

La table Processus d'écoute réseau contient également les options ci-dessous.

Nouveau

Bouton permettant de créer un processus d'écoute réseau.

Supprimer

Bouton permettant de supprimer les processus d'écoute réseau sélectionnés.
