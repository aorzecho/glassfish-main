Réseau de listeners

La page Listeners réseau vous permet de configurer ceux-ci.

Lorsque vous installez pour la première fois GlassFish Server, trois auditeurs de réseau existent par défaut, avec les noms admin-listener, http-listener1 et http-listener2. http-listener2, qui est configuré avec SSL, est désactivé par défaut.

Pour chaque auditeur de réseau, les informations suivantes sont fournies.

Nom

Nom du listener réseau.

Port #

Numéro de port sur lequel le listener réseau écoute.

Protocole

Protocole utilisé par le listener réseau.

Pool de threads

Pool de threads utilisé par le listener réseau.

Activée

True si l’auditeur de réseau est activé, ou false si l’auditeur de réseau n’est pas activé.

Le tableau Auditeurs de réseau contient également les options ci-dessous.

Nouveau

Bouton pour créer un auditeur de réseau.

Supprimer

Bouton pour supprimer une ou plusieurs auditeurs de réseau sélectionnés.
