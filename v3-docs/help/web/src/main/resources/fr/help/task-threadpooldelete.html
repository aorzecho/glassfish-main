Suppression d’un pool de threads

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Pools de threads.

    La page Pools de threads s’affiche.

  3. Dans la page Pools de threads, cochez la case située à gauche du pool de threads à supprimer.
  4. Cliquez sur Supprimer.
