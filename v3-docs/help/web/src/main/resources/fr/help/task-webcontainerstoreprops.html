Configurer les propriétés de stockage pour le conteneur Web

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez la configuration à modifier.
  3. Sous le nœud Configuration, sélectionnez le nœud Conteneur Web.

    La page Propriétés générales s’affiche.

  4. Dans la page Propriétés générales, cliquez sur l’onglet Propriétés du magasin.

    La page Propriétés du magasin s’affiche.

  5. Dans le champ Répertoire, saisissez le chemin d’accès, absolu ou relatif, du répertoire dans lequel les fichiers de sessions individuelles sont enregistrés.

    La valeur par défaut est domain-dir/generated/jsp/j2ee-apps/appname/appname_war.

    Un chemin d’accès relatif est relatif au répertoire de travail temporaire pour cette application Web.

    Ce paramètre ne s'applique que si le type de persistance est file. Pour définir le type de persistance, voir Pour configurer les paramètres de disponibilité du conteneur Web.

  6. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    Le GlassFish Server ne définit aucune autre propriété pour la zone de stockage.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  7. Cliquez sur Enregistrer.
