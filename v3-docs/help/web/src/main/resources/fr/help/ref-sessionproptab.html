Propriétés des sessions

La page Propriétés des sessions vous permet d’ajouter ou de modifier des propriétés pour la session de conteneur Web.

La page Propriétés des sessions contient les options suivantes :

Charger les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut de tous les paramètres.

Nom de configuration

Le nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Délai d’expiration de la session

Spécifie le nombre maximal de secondes pendant lesquelles une session inactive restera valide. Si la valeur est définie sur 0 ou moins, les sessions n’expirent jamais. La valeur par défaut est 1800.

Autres propriétés

Autres propriétés pour les sessions de conteneur Web. Pour obtenir une description des propriétés disponibles, consultez Propriétés spécifiques aux sessions de conteneur Web.
