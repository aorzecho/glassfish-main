Propriétés de la session

La page Propriétés des sessions permet d'ajouter ou de modifier les propriétés de la session de conteneur Web.

La page Propriétés des sessions contient les options suivantes :

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Temporisation de session

Spécifie le nombre maximal de secondes pendant lesquelles une session inactive reste valide. Si la valeur est définie sur 0 ou moins, les sessions n'expirent jamais. La valeur par défaut est 1 800.

Autres propriétés

Autres propriétés pour les sessions de conteneur Web. Pour obtenir une description des propriétés disponibles, reportez-vous à Propriétés propres aux sessions de conteneur Web.
