Propriétés de stockage

Utilisez la page Propriétés de stockage pour ajouter ou modifier les propriétés de persistance (stockage) de la session pour le conteneur Web.

La page Propriétés de stockage contient les options suivantes :

Nom de configuration

Le nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Répertoire

Le chemin d’accès absolu ou relatif du répertoire dans lequel les fichiers de sessions individuelles sont enregistrés. La valeur par défaut est domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Un chemin d’accès relatif est relatif au répertoire de travail temporaire pour cette application Web.

Autres propriétés

Autres propriétés de la zone de stockage. Le GlassFish Server ne définit aucune autre propriété pour la zone de stockage.
