Propriétés de stockage

La page Propriétés de stockage permet d'ajouter ou de modifier les propriétés de persistance (stockage) de la session pour le conteneur Web.

La page Propriétés de stockage contient les options suivantes :

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Répertoire

Chemin absolu ou relatif du répertoire dans lequel les fichiers des sessions individuelles sont écrits. La valeur par défaut est domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Un chemin d'accès relatif est relatif au répertoire de travail temporaire pour cette application Web.

Autres propriétés

Autres propriétés de la zone de stockage. GlassFish Server ne définit aucune autre propriété pour la zone de stockage.
