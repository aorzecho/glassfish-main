Propriétés spécifiques au conteneur Web

La propriété générale supplémentaire suivante est disponible pour le conteneur web.

dispatcher-max-depth

Empêche les instructions d’inclusion ou de transmission récursives de créer une boucle infinie en définissant un niveau de répartition imbriquée maximum. Si le niveau est dépassé, le message suivant est écrit dans le journal du serveur : 

Exceeded maximum depth for nested request dispatches

La valeur par défaut est 20.
