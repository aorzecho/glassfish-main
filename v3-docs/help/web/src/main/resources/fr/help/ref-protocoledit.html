Modifier un protocole

La page Modifier un protocole permet de modifier les paramètres d'un protocole réseau existant.

La page Modifier un protocole contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom

Nom du protocole. Le champ Nom est en lecture seule. Vous ne pouvez spécifier le nom qu'à la création du protocole.

Sécurité

Si l'option est sélectionnée, la sécurité est activée pour le protocole. Si la sécurité est activée, vous pouvez utiliser l'onglet SSL pour modifier les paramètres de sécurité.
