Modification du protocole

Utilisez la page Modification de protocole pour modifier les paramètres d’un protocole réseau existant.

La page Modification de protocole contient les options suivantes.

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom

Nom du protocole. Le champ Nom est en lecture seule. Vous pouvez spécifier le nom uniquement lorsque vous créez un nouveau protocole.

Sécurité

Si l’option est sélectionnée, la sécurité est activée pour le protocole. Si la sécurité est activée, vous pouvez utiliser l’onglet SSL pour modifier les paramètres de sécurité.
