Propriétés spécifiques aux sessions de conteneur Web

Les propriétés suivantes sont disponibles pour une session de conteneur Web.

enableCookies

Si défini sur true, utilise des cookies pour le suivi de session. La valeur par défaut est true.

enableURLRewriting

Permet la réécriture d’URL. Cela permet le suivi de session au moyen de la réécriture d’URL lorsque le navigateur n’accepte pas les cookies. Vous devez également utiliser un appel encodeURL ou encodeRedirectURL dans le servlet ou dans la page JavaServer Pages (JSP). La valeur par défaut est true.

idLengthBytes

Spécifie le nombre d’octets dans cet ID de session du module Web. La valeur par défaut est 128.
