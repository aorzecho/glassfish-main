Configurare le proprietà manager per il contenitore Web

La gestione delle sessioni consente di configurare la modalità di creazione ed eliminazione delle sessioni, la posizione di memorizzazione dello stato della sessione e il numero massimo di sessioni disponibili.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo della configurazione, selezionare il nodo Contenitore Web.

    Viene visualizzata la pagina Proprietà generali.

  4. Nella pagina Proprietà generali, fare clic sulla scheda Proprietà manager.

    Viene visualizzata la pagina Proprietà manager.

  5. Nel campo Intervallo di recupero, digitare il numero di secondi al termine dei quali i dati della sessione inattiva vengono eliminati dall'archivio.

    Il valore predefinito è 60.

    Specificare un valore inferiore alla frequenza con cui i dati della sessione vengono modificati. Nel caso di un sito Web utilizzato di frequente, si consiglia ad esempio di specificare il valore più basso possibile (1 secondo) per un servlet del contatore di accessi; in caso contrario, potrebbero andare perse le ultime visite ogni volta che si riavvia il server.

  6. Nel campo Numero massimo sessioni, digitare il numero massimo di sessioni memorizzabili nella cache.

    Un valore pari a -1 indica un numero di sessioni illimitato. Il valore predefinito è -1.

    Una volta raggiunto il limite, ogni tentativo di creare una nuova sessione genera l'evento IllegalStateException.

  7. Nel campo Nome file sessione, digitare il percorso completo o relativo della directory in cui viene salvato lo stato della sessione a ogni avvio dell'applicazione, se consentito.

    Se il percorso non è stato impostato, lo stato della sessione non viene salvato. Per impostazione predefinita, tale percorso non è impostato.

    I percorsi relativi fanno riferimento alla directory temporanea di questa applicazione Web.

    Questa impostazione è valida solo se l'opzione Tipo di persistenza è impostata su memory. Per impostare il tipo di persistenza, vedere Configurare le impostazioni di disponibilità per il contenitore Web.

  8. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per la gestione delle sessioni.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
