Transports

Nella pagina Transports è possibile configurare i trasporti. Ogni listener di rete viene associato a un trasporto.

La prima volta che si installa GlassFish Server, per impostazione predefinita è disponibile un trasporto TCP con il nome tcp. È possibile modificare o creare un trasporto TCP o UDP oppure specificare un'implementazione personalizzata per un trasporto.

Per ciascun trasporto vengono fornite le seguenti informazioni.

Transport Name

Nome del trasporto.

Classname

Nome della classe di implementazione del trasporto.

Byte Buffer Type

Tipo di ByteBuffer da utilizzare con il trasporto.

La tabella Transports contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo trasporto.

Delete

Pulsante che consente di eliminare uno o più trasporti selezionati.
