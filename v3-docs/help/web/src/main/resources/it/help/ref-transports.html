Trasporti

Nella pagina Trasporti è possibile configurare i trasporti. Ogni listener di rete viene associato a un trasporto.

La prima volta che si installa GlassFish Server, per impostazione predefinita è disponibile un trasporto TCP con il nome tcp. È possibile modificare o creare un trasporto TCP o UDP oppure specificare un'implementazione di trasporto personalizzata.

Per ogni trasporto vengono fornite le informazioni riportate di seguito.

Nome trasporto

Nome del trasporto.

Nome classe

Nome della classe di implementazione del trasporto.

Tipo di buffer di byte

Tipo di ByteBuffer da utilizzare con il trasporto.

La tabella Trasporti contiene le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo trasporto.

Elimina

Pulsante che consente di eliminare uno o più trasporti selezionati.
