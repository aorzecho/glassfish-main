Listener HTTP

Nella pagina Listener HTTP è possibile configurare i listener HTTP.

Un listener HTTP è un tipo di listener di rete. Per accedere alle funzioni avanzate, è possibile modificare un listener HTTP utilizzando le pagine Listener di rete.

Quando si esegue la prima installazione di GlassFish Server, per impostazione predefinita sono disponibili tre listener di rete denominati admin-listener, http-listener1 e http-listener2. Per impostazione predefinita, http-listener2, configurato tramite SSL, è disabilitato.

Per ogni listener HTTP vengono fornite le informazioni riportate di seguito.

Nome

Nome del listener HTTP.

Porta #.

Numero di porta di ascolto del listener HTTP.

Indirizzo

Indirizzo IP usato dal listener HTTP.

Abilitato

Opzione attivata se il listener HTTP è abilitato; disattivata se il listener HTTP è disabilitato.

La tabella Listener HTTP contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo listener HTTP.

Elimina

Pulsante che consente di eliminare uno o più listener HTTP selezionati.
