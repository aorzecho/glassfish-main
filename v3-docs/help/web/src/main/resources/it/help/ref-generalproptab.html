Proprietà generali

Nella pagina Proprietà generali è possibile aggiungere o modificare le proprietà di un contenitore Web.

Il contenitore Web ospita applicazioni Web. Fornisce l'ambiente in cui servlet e file JSP (JavaServer Pages) vengono eseguiti. Per una descrizione delle proprietà disponibili, vedere Proprietà specifica del contenitore Web.

Per ogni proprietà, nella pagina Proprietà generali vengono fornite le informazioni riportate di seguito.

Descrizione
