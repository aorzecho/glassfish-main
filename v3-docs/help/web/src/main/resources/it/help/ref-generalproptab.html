General Properties

Nella pagina General Properties è possibile aggiungere o modificare le proprietà di un contenitore Web.

Il contenitore Web ospita applicazioni Web. Fornisce l'ambiente in cui servlet e file JSP (JavaServer Pages) vengono eseguiti. Per una descrizione delle proprietà disponibili, vedere Proprietà specifiche del contenitore Web.

Per ogni proprietà, nella pagina General Properties vengono fornite le seguenti informazioni.

Name

Nome della proprietà.

Value

Valore della proprietà.

Description

Descrizione della proprietà.
