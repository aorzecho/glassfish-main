Proprietà specifiche del contenitore Web

Per il contenitore Web sono disponibili le seguenti proprietà generali aggiuntive.

dispatcher-max-depth

Evita che le istruzioni di inoltro o inclusione ricorsiva creino un ciclo infinito mediante l'impostazione di un livello massimo di dispatch nidificato. Se tale livello viene superato, nel log del server viene scritto il seguente messaggio: 

Exceeded maximum depth for nested request dispatches

Il valore predefinito è 20.
