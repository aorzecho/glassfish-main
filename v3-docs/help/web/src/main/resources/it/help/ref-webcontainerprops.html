Proprietà specifica del contenitore Web

Per il contenitore Web sono disponibili le proprietà generali aggiuntive riportate di seguito.

dispatcher-max-depth

Evita che le istruzioni include o forward ricorsive creino un loop infinito mediante l'impostazione di un livello massimo di dispatch nidificati. Se tale livello viene superato, nel log del server viene scritto il seguente messaggio:

Exceeded maximum depth for nested request dispatches

Il valore predefinito è 20.
