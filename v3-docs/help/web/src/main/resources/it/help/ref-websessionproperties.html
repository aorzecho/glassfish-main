Proprietà specifiche delle sessioni del contenitore Web

Per una sessione del contenitore Web sono disponibili le proprietà riportate di seguito.

enableCookies

Se l'impostazione è true, utilizza i cookie per la registrazione delle sessioni. Il valore predefinito è true.

enableURLRewriting

Abilita la riscrittura dell'URL. Questa proprietà consente la registrazione delle sessioni mediante la riscrittura dell'URL quando il browser non accetta cookie. È inoltre necessario utilizzare una chiamata encodeURL o encodeRedirectURL nel servlet o nella pagina JSP (JavaServer Pages). Il valore predefinito è true.

idLengthBytes

Specifica il numero di byte nell'ID di sessione del modulo Web. Il valore predefinito è 128.
