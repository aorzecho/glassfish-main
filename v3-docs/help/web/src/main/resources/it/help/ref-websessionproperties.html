Proprietà specifiche delle sessioni di contenitori Web

Per una sessione di contenitori Web sono disponibili le seguenti proprietà.

enableCookies

Se impostato su true, per tenere traccia delle sessioni verranno utilizzati i cookie. Il valore predefinito è true.

enableURLRewriting

Abilita la riscrittura dell'URL. Se abilitata, la riscrittura dell'URL consente di tenere traccia delle sessioni quando il browser non accetta cookie. È inoltre necessario utilizzare una chiamata encodeURL o encodeRedirectURL nel servlet o nella pagina JSP (JavaServer Pages). Il valore predefinito è true.

idLengthBytes

Specifica il numero di byte in questo ID di sessione del modulo Web. Il valore predefinito è 128.
