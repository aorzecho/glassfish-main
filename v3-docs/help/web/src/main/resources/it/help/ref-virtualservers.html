Virtual Servers

Un server virtuale, anche denominato host virtuale, è un oggetto che consente allo stesso server fisico di ospitare più nomi di domini Internet. Tutti i server virtuali ospitati sullo stesso server fisico condividono l'indirizzo IP di tale server fisico. Un server virtuale associa un nome di dominio per un server (quale www.sun.com) al server specifico in cui è in esecuzione GlassFish Server.

Nella pagina Virtual Servers è possibile configurare server virtuali.

Per ciascun server virtuale vengono fornite le seguenti informazioni.

Name

Nome del server virtuale.

State

on, off o disabled.

Default Web Module

Modulo Web distribuito (se disponibile) che deve rispondere a tutte le richieste che non possono essere mappate su altri moduli Web distribuiti nel server virtuale.

La tabella Virtual Servers contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo server virtuale.

Delete

Pulsante che consente di eliminare uno o più server virtuali selezionati.
