Proprietà specifiche del servizio HTTP

Per il servizio HTTP sono disponibili le proprietà aggiuntive riportate di seguito. Se specificate, queste proprietà vengono applicate a tutti i Listener di rete.

ssl-cache-entriese

Numero di sessioni SSL da inserire nella cache. Il valore predefinito è 10.000.

ssl3-session-timeout

Intervallo durante il quale le sessioni SSL3 vengono inserite nella cache. Il valore predefinito è 86.400.

ssl-session-timeout

Intervallo durante il quale le sessioni SSL2 vengono inserite nella cache. Il valore predefinito è 100.

proxyHandler

Nome completamente qualificato di un'implementazione personalizzata della classe astratta com.sun.appserv.ProxyHandler, che consente a un'istanza backend di GlassFish Server di recuperare informazioni relative alla richiesta client originale intercettata da un server proxy di terminazione SSL, ad esempio un load balancer. L'implementazione di questa classe astratta consente di ispezionare una richiesta specifica per individuare le intestazioni di richiesta personalizzate tramite le quali il server proxy comunica le informazioni sulla richiesta client originale all'istanza di Enterprise Server, quindi di restituire tali informazioni al chiamante.

L'implementazione predefinita legge l'indirizzo IP del client da un'intestazione di richiesta HTTP denominata Proxy-ip, il valore keysize di SSL da un'intestazione di richiesta HTTP denominata Proxy-keysize e la catena di certificati client SSL da un'intestazione di richiesta HTTP denominata Proxy-auth-cert . Il valore Proxy-auth-cert deve contenere la catena di certificati client con codifica BASE-64 senza i limiti BEGIN CERTIFICATE ed END CERTIFICATE e con \n sostituito da % d% a.

Questa impostazione viene utilizzata soltanto se la casella di controllo Autenticazione pass-through relativa alla pagina HTTP del protocollo dei listener di rete è impostata su true.

connectionTimeout

Specifica il numero di secondi che il listener di rete deve attendere, dopo l'accettazione di una connessione, per la visualizzazione della riga URI di richiesta. Il valore predefinito è 30.
