Network Listeners

Nella pagina Network Listeners è possibile configurare i listener di rete.

Quando si esegue la prima installazione di GlassFish Server, sono disponibili tre listener di rete per impostazione predefinita, con i nomi admin-listener, http-listener1 e http-listener2. Il listener di rete http-listener2, configurato tramite SSL, è disabilitato per impostazione predefinita.

Per ogni listener di rete vengono fornite le seguenti informazioni.

Name

Nome del listener di rete.

Port #

Numero della porta su cui il listener di rete è in attesa.

Protocol

Protocollo utilizzato dal listener di rete.

Thread Pool

Pool di thread utilizzato dal listener di rete.

Enabled

Opzione attivata se il listener di rete è abilitato e disattivata se il listener è disabilitato.

La tabella Network Listeners contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un nuovo listener di rete.

Delete

Pulsante che consente di eliminare uno o più listener di rete selezionati.
