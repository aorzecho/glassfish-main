Listener di rete

Nella pagina Listener di rete è possibile configurare i listener di rete.

Quando si esegue la prima installazione di GlassFish Server, per impostazione predefinita, sono disponibili tre listener di rete denominati admin-listener, http-listener1 e http-listener2. Per impostazione predefinita, http-listener2, configurato tramite SSL, è disabilitato.

Per ogni listener di rete vengono fornite le informazioni riportate di seguito.

Nome

Nome del listener di rete.

Porta #.

Numero di porta di ascolto del il listener di rete.

Protocollo

Protocollo utilizzato dal listener di rete.

Pool di thread

Pool di thread utilizzato dal listener di rete.

Abilitato

Opzione disattivata se il listener di rete è abilitato; disattivata se il listener è disabilitato.

La tabella Listener di rete contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo listener di rete.

Elimina

Pulsante che consente di eliminare uno o più listener di rete selezionati.
