Manager Properties

La gestione delle sessioni consente di configurare la modalità di creazione ed eliminazione delle sessioni, la posizione di memorizzazione dello stato della sessione e il numero massimo di sessioni disponibili.

Nella pagina Manager Properties è possibile aggiungere o modificare le proprietà di gestione delle sessioni per il contenitore Web.

La pagina Manager Properties contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Reap Interval

Numero di secondi precedenti all'eliminazione dei dati di una sessione inattiva dall'archivio. Il valore predefinito è 60.

Specificare un valore inferiore alla frequenza con cui i dati della sessione vengono modificati. Nel caso di un sito Web utilizzato di frequente, si consiglia ad esempio di specificare il valore più basso possibile (1 secondo) per un servlet del contatore di accessi; in caso contrario, potrebbero andare perse le ultime visite ogni volta che si riavvia il server.

Max Sessions

Numero massimo di sessioni memorizzabili nella cache. Un valore pari a -1 indica un numero di sessioni illimitato. Il valore predefinito è -1.

Una volta raggiunto il limite, ogni tentativo di creare una nuova sessione genera l'evento IllegalStateException.

Session File Name

Percorso completo o relativo della directory in cui viene salvato lo stato della sessione a ogni avvio dell'applicazione, se consentito. Se il percorso non è stato impostato, lo stato della sessione non viene salvato. Per impostazione predefinita, tale percorso non è impostato.

I percorsi relativi fanno riferimento alla directory temporanea di questa applicazione Web.

Additional Properties

Proprietà aggiuntive per la gestione delle sessioni. In GlassFish Server non è definita alcuna proprietà aggiuntiva per la gestione delle sessioni.
