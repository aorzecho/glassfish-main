Proprietà manager

Session Manager consente di configurare la modalità di creazione ed eliminazione delle sessioni, la posizione di memorizzazione dello stato delle sessioni e il numero massimo di sessioni disponibili.

Nella pagina Proprietà manager è possibile aggiungere o modificare le proprietà del manager per il contenitore Web.

La pagina Proprietà manager contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Intervallo di recupero

Numero di secondi prima dell'eliminazione dei dati di una sessione inattiva dall'area di memorizzazione. Il valore predefinito è 60.

Impostare un valore inferiore alla frequenza con cui i dati della sessione vengono modificati. Ad esempio, nel caso di un sito Web utilizzato di frequente, si consiglia di specificare il valore più basso possibile (1 secondo) per un servlet del contatore di accessi; in caso contrario, potrebbero andare persi gli ultimi accessi ogni volta che si riavvia il server.

Numero massimo sessioni

Numero massimo di sessioni che possono essere inserite nella cache. Un valore pari a -1 indica un numero di sessioni illimitato. Il valore predefinito è -1.

Una volta raggiunto il limite, un tentativo di creare una nuova sessione restituisce l'evento IllegalStateException.

Nome file sessione

Percorso assoluto o relativo della directory in cui viene conservato lo stato della sessione a ogni avvio dell'applicazione, se la conservazione dello stato è possibile. Se questo percorso non viene impostato, lo stato della sessione non viene conservato. Per impostazione predefinita, questo percorso non è impostato.

Un percorso relativo indica la directory temporanea di questa applicazione Web.

Proprietà aggiuntive

Proprietà aggiuntive per il Session Manager. In GlassFish Server non è definita alcuna proprietà aggiuntiva per il Session Manager.
