Configurare le proprietà dell'archivio per il contenitore Web

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare la configurazione da modificare.
  3. Nel nodo della configurazione, selezionare il nodo Web Container.

    Viene visualizzata la pagina General Properties.

  4. Nella pagina General Properties, fare clic sulla scheda Store Properties.

    Viene visualizzata la pagina Store Properties.

  5. Nel campo Directory, digitare il nome del percorso assoluto o relativo della directory in cui vengono scritti i file delle singole sessioni.

    Il valore predefinito è domain-dir/generated/jsp/j2ee-apps/ appname/appname_war.

    I percorsi relativi fanno riferimento alla directory di lavoro temporanea di questa applicazione Web.

    Questa impostazione è valida solo se l'opzione Persistence Type è impostata su file. Per impostare l'opzione Persistence Type, vedere Configurare le impostazioni di disponibilità per il contenitore Web.

  6. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per l'archivio.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  7. Fare clic su Save.
