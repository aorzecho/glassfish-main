Configurare le proprietà dell'area di memorizzazione per il contenitore Web

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo della configurazione, selezionare il nodo Contenitore Web.

    Viene visualizzata la pagina Proprietà generali.

  4. Nella pagina Proprietà generali, fare clic sulla scheda Proprietà area di memorizzazione.

    Viene visualizzata la pagina Proprietà area di memorizzazione.

  5. Nel campo Directory, digitare il nome del percorso assoluto o relativo della directory in cui vengono scritti i file delle singole sessioni.

    Il valore predefinito è domain-dir/generated/jsp/j2ee-apps/appname/appname_war.

    I percorsi relativi fanno riferimento alla directory di lavoro temporanea di questa applicazione Web.

    Questa impostazione è valida solo se l'opzione Tipo di persistenza è impostata su file. Per impostare il tipo di persistenza, vedere Configurare le impostazioni di disponibilità per il contenitore Web.

  6. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per l'archivio.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  7. Fare clic su Salva.
