Proprietà area di memorizzazione

Nella pagina Proprietà area di memorizzazione è possibile aggiungere o modificare le proprietà di persistenza (memorizzazione) delle sessioni per il contenitore Web.

La pagina Proprietà area di memorizzazione contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Directory

Percorso assoluto o relativo della directory in cui vengono scritti i singoli file di sessione. Il valore predefinito è domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Un percorso relativo indica la directory di lavoro temporanea di questa applicazione Web.

Proprietà aggiuntive

Proprietà aggiuntive per l'area di memorizzazione. In GlassFish Server non è definita alcuna proprietà aggiuntiva per l'area di memorizzazione.
