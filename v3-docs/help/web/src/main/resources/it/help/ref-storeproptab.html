Store Properties

Nella pagina Store Properties è possibile aggiungere o modificare le proprietà di persistenza (memorizzazione) delle sessioni per il contenitore Web.

La pagina Store Properties contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Directory

Nome del percorso completo o relativo della directory in cui vengono scritti i file di sessioni singole. Il valore predefinito è domain-dir/generated/jsp/j2ee-apps/ appname/appname_war. I percorsi relativi fanno riferimento alla directory di lavoro temporanea di questa applicazione Web.

Additional Properties

Proprietà aggiuntive per l'archivio. In GlassFish Server non è definita alcuna proprietà aggiuntiva per l'archivio.
