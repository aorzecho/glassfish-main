New Thread Pool

Nella pagina New Thread Pool è possibile creare un pool di thread.

La pagina New Thread Pool contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Name

Nome del pool di thread.

Class Name

Nome completo della classe che implementa il pool di thread. Il valore predefinito è com.sun.grizzly.http.StatsThreadPool.

Max Queue Size

Numero massimo di messaggi che è possibile accodare fino a quando non sono disponibili i thread per elaborarli. Un valore pari a -1 indica una dimensione illimitata della coda. Il valore predefinito è 4096.

Max Thread Pool Size

Numero massimo di thread nel pool di thread. Il valore predefinito è 5. Questo è il valore consigliato per i pool di thread di listener di rete.

Min Thread Pool Size

Numero minimo di thread nel pool di thread. Il valore predefinito è 2.

Idle Thread Timeout

Numero massimo di secondi durante i quali un thread può rimanere inattivo nel pool. Una volta trascorso tale numero di secondi, il thread viene rimosso dal pool. Il valore predefinito è 900.
