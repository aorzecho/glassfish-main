Proprietà di sessione

Nella pagina Proprietà di sessione è possibile aggiungere o modificare le proprietà per la sessione di un contenitore Web.

La pagina Proprietà di sessione contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Timeout sessione

Specifica il numero massimo di secondi durante i quali una sessione inattiva rimane valida. Se il valore è impostato su 0 o su un valore inferiore, le sessioni non scadono mai. Il valore predefinito è 1800.

Proprietà aggiuntive

Proprietà aggiuntive per le sessioni del contenitore Web. Per una descrizione delle proprietà disponibili, vedere Proprietà specifiche delle sessioni del contenitore Web.
