Session Properties

Nella pagina Session Properties è possibile aggiungere o modificare le proprietà della sessione di un contenitore Web.

La pagina Session Properties contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Session Timeout

Specifica il numero massimo di secondi di validità di una sessione inattiva. Se il valore è impostato su 0, le sessioni non scadono mai. Il valore predefinito è 1800.

Additional Properties

Proprietà aggiuntive per le sessioni di contenitori Web. Per una descrizione delle proprietà disponibili, vedere Proprietà specifiche delle sessioni di contenitori Web.
