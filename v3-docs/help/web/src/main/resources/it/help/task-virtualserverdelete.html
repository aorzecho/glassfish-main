Eliminare un server virtuale

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Select the configuration that you are changing.

  3. Nel nodo Configurazione, selezionare il nodo Server virtuali.

    Viene visualizzata la pagina Server virtuali.

  4. Nella pagina Server virtuali, selezionare la casella di controllo a sinistra del nome del server virtuale da eliminare.

  5. Fare clic su Elimina.

    L'eliminazione di uno dei server predefiniti, __asadmin e server, è sconsigliata.
