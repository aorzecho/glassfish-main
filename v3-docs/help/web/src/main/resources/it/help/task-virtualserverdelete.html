Eliminare un server virtuale

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Virtual Servers.

    Viene visualizzata la pagina Virtual Servers.

  3. Nella pagina Virtual Servers, selezionare la casella di controllo a sinistra del nome del server virtuale da eliminare.
  4. Fare clic su Delete.

    L'eliminazione di uno dei server predefiniti, __asadmin e server, è sconsigliata.
