Protocols

Nella pagina Protocols è possibile configurare protocolli di rete.

Per ciascun protocollo vengono fornite le seguenti informazioni.

Name

Nome del protocollo.

Security Enabled

Opzione attivata se la protezione per il protocollo è abilitata e disattivata se la protezione per il protocollo non è abilitata.

La tabella Protocols contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo protocollo.

Delete

Pulsante che consente di eliminare una o più protocolli selezionati.

Attività correlate

  • Creare un protocollo

  • Modificare un protocollo

  • Modificare le impostazioni SSL di un protocollo

  • Modificare le impostazioni HTTP di un protocollo

  • Modificare le impostazioni della cache di file per un protocollo

  • Eliminare un protocollo
