Protocolli

Nella pagina Protocolli è possibile configurare protocolli di rete.

Per ogni protocollo vengono fornite le informazioni riportate di seguito.

Nome

Nome del protocollo.

Sicurezza abilitata

Opzione attivata se la sicurezza per il protocollo è abilitata; disattivata se la sicurezza per il protocollo non è abilitata.

La tabella Protocolli contiene le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo protocollo.

Elimina

Pulsante che consente di eliminare una o più protocolli selezionati.
