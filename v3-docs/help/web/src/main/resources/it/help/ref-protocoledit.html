Modifica protocollo

Nella pagina Modifica protocollo è possibile modificare le impostazioni di un protocollo di rete esistente.

La pagina Modifica protocollo contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome del protocollo. Il campo Nome è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo protocollo.

Sicurezza

Se questa opzione è selezionata, la sicurezza viene abilitata per il protocollo. Se la sicurezza è abilitata, è possibile utilizzare la scheda SSL per modificare le impostazioni di sicurezza.
