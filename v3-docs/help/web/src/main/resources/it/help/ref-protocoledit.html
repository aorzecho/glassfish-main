Modifica di un protocollo

Nella pagina Edit Protocol è possibile modificare le impostazioni di un protocollo di rete esistente.

La pagina Edit Protocol contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Name

Nome del protocollo. Il campo Name è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo protocollo.

Security

Se l'opzione è selezionata, per il protocollo viene abilitata la protezione. Se la protezione è abilitata, è possibile utilizzare la scheda SSL per modificare le impostazioni di protezione.
