프로토콜 편집

프로토콜 편집 페이지에서는 기존 네트워크 프로토콜의 설정을 수정할 수 있습니다.

프로토콜 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 데 사용하는 버튼입니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

프로토콜의 이름입니다. 이름 필드는 읽기 전용 필드입니다. 새 프로토콜을 만들 때만 이름을 지정할 수 있습니다.

Security

이 옵션을 선택하면 프로토콜에 대해 보안이 활성화됩니다. 보안이 활성화되면 SSL 탭을 사용하여 보안 설정을 수정할 수 있습니다.

관련 작업

  • 프로토콜을 만드는 방법

  • 프로토콜을 편집하는 방법

  • 프로토콜에 대한 SSL 설정을 편집하는 방법

  • 프로토콜에 대한 HTTP 설정을 편집하는 방법

  • 프로토콜에 대한 파일 캐시 설정을 편집하는 방법

  • 프로토콜을 삭제하는 방법

관련 asadmin 명령

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
