저장소 속성

저장소 속성 페이지에서는 웹 컨테이너의 세션 지속성(저장소) 속성을 추가하거나 수정할 수 있습니다.

저장소 속성 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

디렉토리

개별 세션 파일이 기록되는 디렉토리의 절대 또는 상대 경로 이름입니다. 기본값은 domain-dir/generated/jsp/j2ee-apps/appname/appname_war입니다. 상대 경로는 이 웹 응용 프로그램의 임시 작업 디렉토리에 상대적입니다.

추가 속성

저장소의 추가 속성입니다. GlassFish Server에서는 저장소에 대한 추가 속성을 정의하지 않습니다.

관련 작업

  • 웹 컨테이너에 대한 일반 속성을 구성하는 방법

  • 웹 컨테이너 세션 속성을 구성하는 방법

  • 웹 컨테이너 관리자 속성을 구성하는 방법

  • 웹 컨테이너 저장소 속성을 구성하는 방법

  • 웹 컨테이너 가용성 설정을 구성하는 방법
