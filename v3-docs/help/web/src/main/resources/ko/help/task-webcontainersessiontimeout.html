웹 컨테이너의 세션 등록 정보를 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 변경 중인 구성을 선택합니다.
  3. 구성 노드에서 웹 컨테이너 노드를 선택합니다.

    일반 등록 정보 페이지가 열립니다.

  4. 일반 등록 정보 페이지에서 세션 등록 정보 탭을 누릅니다.

    세션 등록 정보 페이지가 열립니다.

  5. 세션 시간 초과 필드에 비활성 세션이 유효 상태로 유지되는 최대 시간(초)을 입력합니다.

    기본값은 1800입니다. 값을 0 이하로 설정하면 세션이 만료되지 않습니다.

  6. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    사용할 수 있는 등록 정보에 대한 설명은 웹 컨테이너 세션과 관련된 등록 정보를 참조하십시오.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  7. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.
