스레드 풀을 삭제하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 [스레드 풀] 노드를 선택합니다.

    [스레드 풀] 페이지가 열립니다.

  3. [스레드 풀] 페이지에서 삭제할 스레드 풀의 왼쪽에 있는 체크 박스를 선택합니다.

  4. [삭제]를 누릅니다.
