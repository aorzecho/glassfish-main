일반 등록 정보

일반 등록 정보 페이지에서는 웹 컨테이너의 등록 정보를 추가하거나 수정할 수 있습니다.

웹 컨테이너는 웹 응용 프로그램을 호스트합니다. 웹 컨테이너는 서블릿 및 JavaServer Pages(JSP) 파일이 실행되는 환경을 제공합니다. 사용할 수 있는 등록 정보에 대한 설명은 웹 컨테이너와 관련된 등록 정보를 참조하십시오.

각 등록 정보에 대해 일반 등록 정보 페이지에는 다음 정보가 포함됩니다.

이름

등록 정보의 이름입니다.

값

등록 정보의 값입니다.

설명

등록 정보에 대한 설명입니다.
