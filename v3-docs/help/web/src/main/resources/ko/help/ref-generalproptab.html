일반 속성

[일반 속성] 페이지에서는 웹 컨테이너의 속성을 추가하거나 수정할 수 있습니다.

웹 컨테이너는 웹 응용 프로그램을 호스트합니다. 또한 서블릿 및 JSP(JavaServer Pages) 파일이 실행되는 환경을 제공합니다. 사용 가능한 속성에 대한 설명은 웹 컨테이너 관련 속성을(를) 참조하십시오.

각 속성에 대해 [일반 속성] 페이지에는 다음 정보가 포함됩니다.

이름

속성의 이름입니다.

값

속성의 값입니다.

설명

속성에 대한 설명입니다.

관련 작업

  • 웹 컨테이너에 대한 일반 속성을 구성하는 방법

  • 웹 컨테이너 세션 속성을 구성하는 방법

  • 웹 컨테이너 관리자 속성을 구성하는 방법

  • 웹 컨테이너 저장소 속성을 구성하는 방법

  • 웹 컨테이너 가용성 설정을 구성하는 방법
