스레드 풀 편집

스레드 풀 편집 페이지에서는 기존 스레드 풀을 수정할 수 있습니다.

스레드 풀 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

스레드 풀의 이름입니다. 이름 필드는 읽기 전용 필드입니다. 새 스레드 풀을 만들 때만 이름을 지정할 수 있습니다.

클래스 이름

스레드 풀을 구현하는 클래스의 정규화된 이름입니다. 기본값은 com.sun.grizzly.http.StatsThreadPool입니다.

최대 대기열 크기

스레드에서 메시지를 처리할 수 있을 때까지 대기열에 있을 수 있는 최대 메시지 수입니다. 값이 –1이면 대기열 크기에 제한이 없습니다. 기본값은 4096입니다.

최대 스레드 풀 크기

스레드 풀에 있는 스레드의 최대 수입니다. 기본값은 5입니다.

최소 스레드 풀 크기

스레드 풀에 있는 스레드의 최소 수입니다. 기본값은 2입니다.

유휴 스레드 시간 초과

풀에서 스레드가 유휴 상태로 있을 수 있는 최대 시간(초)입니다. 이 시간이 만료되면 스레드가 풀에서 제거됩니다. 기본값은 900입니다.
