프로토콜

프로토콜 페이지에서는 네트워크 프로토콜을 구성할 수 있습니다.

각 프로토콜에 대해 제공되는 정보는 다음과 같습니다.

이름

프로토콜의 이름입니다.

보안 활성화

프로토콜에 대해 보안이 활성화된 경우 true, 프로토콜에 대해 보안이 활성화되지 않은 경우 false입니다.

프로토콜 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 프로토콜을 만드는 버튼입니다.

삭제

하나 이상의 선택된 프로토콜을 삭제하는 버튼입니다.

관련 작업

  • 프로토콜을 만드는 방법

  • 프로토콜을 편집하는 방법

  • 프로토콜에 대한 SSL 설정을 편집하는 방법

  • 프로토콜에 대한 HTTP 설정을 편집하는 방법

  • 프로토콜에 대한 파일 캐시 설정을 편집하는 방법

  • 프로토콜을 삭제하는 방법

관련 asadmin 명령

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
