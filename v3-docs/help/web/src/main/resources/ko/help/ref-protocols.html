프로토콜

[프로토콜] 페이지에서는 네트워크 프로토콜을 구성할 수 있습니다.

각 프로토콜에 대해 제공되는 정보는 다음과 같습니다.

이름

프로토콜의 이름입니다.

보안 사용

프로토콜에 대해 보안이 사용으로 설정된 경우 true, 프로토콜에 대해 보안이 사용으로 설정되지 않은 경우 false입니다.

프로토콜 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 프로토콜을 생성하는 단추입니다.

삭제

선택한 프로토콜을 하나 이상 삭제하는 단추입니다.

관련 작업

  • 프로토콜을 생성하는 방법

  • 프로토콜을 편집하는 방법

  • 프로토콜 SSL 설정을 편집하는 방법

  • 프로토콜 HTTP 설정을 편집하는 방법

  • 프로토콜 파일 캐시 설정을 편집하는 방법

  • 프로토콜을 삭제하는 방법

관련 asadmin 명령

  • create-protocol(1)

  • list-protocols(1)

  • delete-protocol(1)
