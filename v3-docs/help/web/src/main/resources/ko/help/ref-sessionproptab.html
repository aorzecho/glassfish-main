세션 속성

세션 속성 페이지에서는 웹 컨테이너 세션의 속성을 추가 또는 수정할 수 있습니다.

세션 속성 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 단추입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

세션 시간 초과

비활성 세션이 적합한 상태로 유지되는 최대 시간(초)을 지정합니다. 값을 0 이하로 설정하면 세션이 만료되지 않습니다. 기본값은 1800입니다.

추가 속성

웹 컨테이너 세션의 추가 속성입니다. 사용 가능한 속성에 대한 설명은 웹 컨테이너 세션 관련 속성을(를) 참조하십시오.

관련 작업

  • 웹 컨테이너에 대한 일반 속성을 구성하는 방법

  • 웹 컨테이너 세션 속성을 구성하는 방법

  • 웹 컨테이너 관리자 속성을 구성하는 방법

  • 웹 컨테이너 저장소 속성을 구성하는 방법

  • 웹 컨테이너 가용성 설정을 구성하는 방법
