세션 등록 정보

세션 등록 정보 페이지에서는 웹 컨테이너 세션의 등록 정보를 추가 또는 수정할 수 있습니다.

세션 등록 정보 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 데 사용하는 버튼입니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

세션 시간 초과

비활성 세션이 유효한 상태로 유지되는 최대 시간(초)을 지정합니다. 값을 0 이하로 설정하면 세션이 만료되지 않습니다. 기본값은 1800입니다.

추가 등록 정보

웹 컨테이너 세션의 추가 등록 정보입니다. 사용할 수 있는 등록 정보에 대한 설명은 웹 컨테이너 세션과 관련된 등록 정보를 참조하십시오.

관련 작업

  • 웹 컨테이너의 일반 등록 정보를 구성하는 방법

  • 웹 컨테이너의 세션 등록 정보를 구성하는 방법

  • 웹 컨테이너의 관리자 등록 정보를 구성하는 방법

  • 웹 컨테이너의 저장소 등록 정보를 구성하는 방법

  • 웹 컨테이너에 대한 가용성 설정을 구성하는 방법
