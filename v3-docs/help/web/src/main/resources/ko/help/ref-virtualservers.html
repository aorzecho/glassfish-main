가상 서버

가상 호스트라고도 하는 가상 서버는 동일한 물리적 서버에서 여러 개의 인터넷 도메인 이름을 호스트할 때 사용할 수 있는 객체입니다. 동일한 물리적 서버에서 호스트되는 모든 가상 서버는 해당 서버의 인터넷 프로토콜(IP) 주소를 공유합니다. 가상 서버는 서버의 도메인 이름(예: www.sun.com)을 GlassFish Server 가 실행되는 특정 서버에 연결합니다.

가상 서버 페이지에서는 가상 서버를 구성할 수 있습니다.

각 가상 서버에 대해 제공되는 정보는 다음과 같습니다.

이름

가상 서버의 이름입니다.

상태

켜기, 끄기 또는 사용 불가입니다.

기본 웹 모듈

가상 서버에 배포된 다른 웹 모듈에 매핑될 수 없는 모든 요청에 응답할 배포된 웹 모듈(있는 경우)입니다.

가상 서버 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 가상 서버를 만드는 버튼입니다.

삭제

하나 이상의 선택된 가상 서버를 삭제하는 버튼입니다.

관련 작업

  • 가상 서버를 만드는 방법

  • 가상 서버를 편집하는 방법

  • 가상 서버를 삭제하는 방법

관련 asadmin 명령

  • create-virtual-server(1)

  • list-virtual-servers(1)

  • delete-virtual-server(1)
