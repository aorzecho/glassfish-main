새 스레드 풀

[새 스레드 풀] 페이지에서는 스레드 풀을 생성할 수 있습니다.

[새 스레드 풀] 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

스레드 풀의 이름입니다.

클래스 이름

스레드 풀을 구현하는 클래스의 전체 이름입니다. 기본값은 com.sun.grizzly.http.StatsThreadPool입니다.

최대 대기열 크기

스레드에서 메시지를 처리할 수 있을 때까지 대기열에 있을 수 있는 최대 메시지 수입니다. 값이 –1이면 대기열 크기에 제한이 없습니다. 기본값은 4096입니다.

최대 스레드 풀 크기

스레드 풀에 있는 스레드의 최대 수입니다. 기본값은 5입니다. 네트워크 리스너 스레드 풀에는 이 값을 사용하는 것이 좋습니다.

최소 스레드 풀 크기

스레드 풀에 있는 스레드의 최소 수입니다. 기본값은 2입니다.

유휴 스레드 시간 초과

풀에서 스레드가 유휴 상태로 있을 수 있는 최대 시간(초)입니다. 이 시간이 만료되면 스레드가 풀에서 제거됩니다. 기본값은 900입니다.

관련 작업

  • 스레드 풀을 생성하는 방법

  • 스레드 풀을 편집하는 방법

  • 스레드 풀을 삭제하는 방법

관련 asadmin 명령

  • list-threadpools(1)
