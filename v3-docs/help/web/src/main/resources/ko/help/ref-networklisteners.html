네트워크 수신기

네트워크 수신기 페이지에서는 네트워크 수신기를 구성할 수 있습니다.

GlassFish 서버를 처음 설치하는 경우 기본적으로 admin-listener, http-listener1 및 http-listener2라는 세 개의 네트워크 수신기가 있습니다. SSL로 구성된 http-listener2는 기본적으로 비활성화됩니다.

각 네트워크 수신기에 대해 제공되는 정보는 다음과 같습니다.

이름

네트워크 수신기의 이름입니다.

포트 번호

네트워크 수신기에서 수신하는 포트 번호입니다.

프로토콜

네트워크 수신기에서 사용되는 프로토콜입니다.

스레드 풀

네트워크 수신기에서 사용되는 스레드 풀입니다.

사용 가능

네트워크 수신기를 사용하는 경우 true이고 네트워크 수신기를 사용하지 않는 경우 false입니다.

네트워크 수신기 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 네트워크 수신기를 만드는 버튼입니다.

삭제

하나 이상의 선택된 네트워크 수신기를 삭제하는 버튼입니다.

관련 작업

  • 네트워크 수신기를 만드는 방법

  • 네트워크 수신기를 편집하는 방법

  • 네트워크 수신기를 삭제하는 방법

  • 가상 서버를 만드는 방법

관련 asadmin 명령

  • create-network-listener(1)

  • list-network-listeners(1)

  • delete-network-listener(1)
