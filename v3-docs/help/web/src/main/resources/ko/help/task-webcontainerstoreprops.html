웹 컨테이너의 저장소 등록 정보를 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 변경 중인 구성을 선택합니다.
  3. 구성 노드에서 웹 컨테이너 노드를 선택합니다.

    일반 등록 정보 페이지가 열립니다.

  4. 일반 등록 정보 페이지에서 저장소 등록 정보 탭을 누릅니다.

    저장소 등록 정보 페이지가 열립니다.

  5. 디렉토리 필드에 개별 세션 파일이 기록되는 디렉토리의 절대 또는 상대 경로 이름을 입력합니다.

    기본값은 domain-dir/generated/jsp/j2ee-apps/appname/appname_war입니다.

    상대 경로는 이 웹 응용 프로그램의 임시 작업 디렉토리에 상대적입니다.

    이 설정은 지속성 유형이 파일인 경우에만 적용할 수 있습니다. 지속성 유형을 설정하려면 웹 컨테이너에 대한 가용성 설정을 구성하는 방법을 참조하십시오.

  6. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish 서버에서는 저장소에 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  7. 저장을 누릅니다.
참조

  • 웹 컨테이너의 일반 등록 정보를 구성하는 방법

  • 웹 컨테이너의 세션 등록 정보를 구성하는 방법

  • 웹 컨테이너의 관리자 등록 정보를 구성하는 방법

  • 웹 컨테이너에 대한 가용성 설정을 구성하는 방법
