프로토콜을 편집하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 [네트워크 구성] 노드를 확장합니다.

  3. [네트워크 구성] 노드에서 [프로토콜] 노드를 선택합니다.

    [프로토콜] 페이지가 열립니다.

  4. [프로토콜] 페이지에서 편집할 프로토콜의 이름을 누릅니다.

    [프로토콜 편집] 페이지가 열립니다.

  5. 프로토콜에 대해 보안을 사용하려면 [보안 사용] 체크 박스를 선택합니다.

    보안이 사용으로 설정되면 SSL 탭을 사용하여 보안 설정을 수정할 수 있습니다.

  6. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.

  7. [보안 사용] 체크 박스를 선택한 경우 프로토콜 SSL 설정을 편집하는 방법의 단계를 따릅니다.

  • 네트워크 리스너를 삭제하는 방법
