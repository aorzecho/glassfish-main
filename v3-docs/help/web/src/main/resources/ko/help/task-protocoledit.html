프로토콜을 편집하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 네트워크 구성 노드를 확장합니다.
  3. 네트워크 구성 노드에서 프로토콜 노드를 선택합니다.

    프로토콜 페이지가 열립니다.

  4. 프로토콜 페이지에서 편집할 프로토콜의 이름을 누릅니다.

    프로토콜 편집 페이지가 열립니다.

  5. 프로토콜에 대해 보안을 사용하려면 보안 사용 가능 확인란을 선택합니다.

    보안이 활성화되면 SSL 탭을 사용하여 보안 설정을 수정할 수 있습니다.

  6. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.

  7. 보안 활성화 확인란을 선택한 경우 프로토콜에 대한 SSL 설정을 편집하는 방법의 단계를 따릅니다.

참조

  • 네트워크 수신기를 만드는 방법

  • 네트워크 수신기를 편집하는 방법

  • 네트워크 수신기를 삭제하는 방법

  • 프로토콜을 만드는 방법

  • 프로토콜에 대한 SSL 설정을 편집하는 방법

  • 프로토콜에 대한 HTTP 설정을 편집하는 방법

  • 프로토콜에 대한 파일 캐시 설정을 편집하는 방법

  • 프로토콜을 삭제하는 방법

  • 전송을 만드는 방법

  • 전송을 편집하는 방법

  • 전송을 삭제하는 방법
