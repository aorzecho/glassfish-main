Servidores virtuais

Um servidor virtual, às vezes chamado de host virtual, é um objeto que permite que o mesmo servidor físico hospede vários nomes de domínio da Internet. Todos os servidores virtuais hospedados no mesmo servidor físico compartilham o endereço IP do servidor físico. Um servidor virtual associa um nome de domínio (por exemplo, www.sun.com) ao servidor em que o Servidor GlassFish está em execução.

Use a página Servidores virtuais para configurá-los.

Para cada servidor virtual, as seguintes informações são fornecidas.

Nome

O nome do servidor virtual.

Estado

Ligado, desligado ou desativado.

Módulo da Web padrão

O módulo da Web implantado (se existente), designado para responder todas as solicitações que não puderem ser mapeadas por outros módulos Web implantados no servidor virtual.

A tabela Servidores virtuais também contém as seguintes opções.

Novo

Botão para criar um novo servidor virtual

Excluir

Botão para excluir um ou mais servidores virtuais selecionados.
