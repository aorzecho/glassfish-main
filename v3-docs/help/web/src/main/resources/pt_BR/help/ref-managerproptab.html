Propriedades do gerenciador

O gerenciador de sessão fornece os meios para configurar o modo como as sessões são criadas e destruídas, onde o estado de sessão é armazenado e o número máximo de sessões disponíveis.

Use a página Propriedades do Gerenciador para adicionar ou modificar propriedades do gerenciador para o recipiente da Web.

A página Gerenciador de Sessão contém as seguintes opções.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Intervalo de Reaplic.

O número de segundos antes que a sessão inativa seja excluída do armazenamento. O valor padrão é 60.

Defina este valor menor do que a frequência na qual os dados da sessão são modificados. Por exemplo, este valor deveria ser o menor possível (1 segundo) para um servlet de contagem de acessos em um website com acesso frequente, do contrário você perderá alguns acessos cada vez que reiniciar o servidor.

Máx. de Sessões

O número máximo de sessões que podem ser armazenadas no cache. Um valor de -1 indica nenhum limite para o número de sessões. O valor padrão é -1.

Após o limite ser alcançado, uma tentativa para criar uma nova sessão causa a ocorrência de uma IllegalStateException.

Nome de Arquivo de Sessão

O caminho absoluto ou relativo do diretório no qual o estado da sessão é preservado entre as reinicializações do aplicativo, se for possível preservar o estado. Se este caminho não é definido, o estado da sessão não é preservado. Por padrão, este caminho não é definido.

Um caminho relativo para o diretório temporário do aplicativo Web.

Propriedades Adicionais

Propriedades adicionais do gerenciador de sessão. O Servidor GlassFish não define propriedades adicionais para o gerenciador de sessão.
