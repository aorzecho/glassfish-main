Listeners HTTP

Use a página Listeners HTTP para configurar listeners HTTP.

Um listener HTTP é um tipo de listener de rede. Para acessar recursos avançados, você pode editar o listener HTTP usando as páginas do Listener de Rede.

Quando o GlassFish Server é instalado pela primeira vez, três listeners HTTP existem por default, com os nomes admin-listener, http-listener1 e http-listener2. O http-listener2,↓que é configurado com o SSL, é desativado por default.

Para cada listener HTTP, as seguintes informações são fornecidas.

Nome

O nome do listener HTTP.

Porta #

O número da porta na qual o listener HTTP está fazendo listening.

Endereço

O endereço de UP usado pelo listener HTTP.

Ativo

Verdadeiro se o listener HTTP estiver ativado ou falso se o listener HTTP não estiver ativado.

A tabela de Listeners HTTP também contém as seguintes opções.

Novo

Botão para criar um novo listener HTTP.

Deletar

Botão para deletar um ou mais listeners HTTP selecionados.
