Editar conjunto de segmentos

Use a página Editar conjunto de segmentos para modificar um conjunto de segmentos existente.

A página Editar conjunto de segmentos contém estas opções.

Carregar padrões

Botão para restaurar com os valores padrão as configurações que apresentam valores padrão. As configurações que não apresentam valores padrão não são alteradas.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Nome

O nome do conjunto de segmentos. O campo Nome é de somente leitura. Você só pode especificar o nome quando cria um novo conjunto de segmentos.

Nome da classe

O nome totalmente qualificado da classe que implementa o conjunto de segmentos. O valor padrão é com.sun.grizzly.http.StatsThreadPool.

Tamanho máx. da fila

O número máximo de mensagens que podem ser colocadas na fila até que segmentos estejam disponíveis para processá-las. Um valor de -1 indica que não há limite para o tamanho da fila. O valor padrão é 4096.

Tamanho máx. do conjunto de segmentos

O número máximo de segmentos no conjunto do segmentos. O valor padrão é 5.

Tamanho mín. do conjunto de segmentos

O número mínimo de segmentos no conjunto do segmentos. O valor padrão é 2.

Tempo limite de segmento ocioso

A quantidade máxima de tempo, em segundos, que um segmento pode permanecer inativo no conjunto. Após esse período, o segmento será removido do conjunto. O valor padrão é 900.
