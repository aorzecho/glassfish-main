Para editar um conjunto de segmentos

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, selecione o nó conjunto de segmentos.

    É aberta a página conjunto de segmentos.

  3. Na página conjunto de segmentos, clique no nome do conjunto de segmentos a ser editado.

    É aberta a página Editar conjunto de segmentos.

  4. Na página Editar conjunto de segmentos, No campo Nome da classe, digite o nome totalmente qualificado da classe de implementação do conjunto de segmentos.

    O valor padrão é com.sun.grizzly.http.StatsThreadPool.

  5. No campo Tamanho máximo da fila, digite o número máximo de mensagens que podem ser colocadas na fila até que os segmentos estejam disponíveis para processá-las.

    Um valor de -1 indica que não há limite para o tamanho da fila. O valor padrão é 4096.

  6. No campo Tamanho Máximo do conjunto de segmentos, digite a quantidade máxima de segmentos que pode haver no conjunto de segmentos.

    O valor padrão é 5. Esse é o valor recomendado para conjunto de segmentos de escuta de rede.

  7. No campo Tamanho Mínimo do conjunto de segmentos, digite a quantidade mínima de segmentos que pode haver no conjunto de segmentos.

    Esses segmentos são criados quando o conjunto do segmentos é instanciado. O valor padrão é 2.

  8. No campo Tempo limite de segmentos inativos, digite a quantidade máxima de tempo, em segundos, que um segmento pode permanecer ocioso no conjunto.

    Após esse período, o segmento será removido do conjunto. O valor padrão é 900.

  9. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
