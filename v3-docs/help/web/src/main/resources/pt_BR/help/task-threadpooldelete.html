Para excluir um conjunto de segmentos.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, selecione o nó conjunto de segmentos.

    É aberta a página conjunto de segmentos.

  3. Na página conjunto de segmentos, marque a caixa de seleção à esquerda do conjunto de segmentos a ser excluído.
  4. Clique em Excluir.
