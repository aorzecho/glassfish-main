Para Deletar um Pool de Threads

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, selecione o nó Pool de Threads.

    É aberta a página Pool de Threads.

  3. Na página Pool de Threads, marque a caixa de seleção à esquerda do Pool de Threads a ser deletado.

  4. Clique em Deletar.
