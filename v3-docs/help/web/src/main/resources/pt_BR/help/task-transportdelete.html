Para excluir um transporte

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó Configuração da Rede.
  3. No nó Config da rede, selecione o nó Transportes.

    É aberta a página Transportes.

  4. Na página Transportes, selecione a caixa de seleção à esquerda do transporte que estiver excluindo.
  5. Clique em Excluir.
