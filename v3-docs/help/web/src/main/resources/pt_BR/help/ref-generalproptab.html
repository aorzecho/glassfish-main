Propriedades gerais

Use a página Propriedades gerais para adicionar ou modificar propriedades do recipiente da Web.

O recipiente da Web hospeda aplicativos da Web. Fornece o ambiente para execução de servlets e arquivos do JavaServer Pages (JSP). Para obter uma descrição das propriedades disponíveis, consulte Propriedades específicas para sessões do recipiente da Web.

Para cada propriedade, a página Propriedades gerais contém as seguintes informações.

Nome

O nome da propriedade.

Valor

O valor da propriedade.

Descrição

A descrição da propriedade.
