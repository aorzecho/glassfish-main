Propriedades Gerais

Use a página Propriedades Gerais para adicionar ou modificar propriedades do contêiner Web.

O contêiner Web hospeda aplicações Web. Ele fornece o ambiente no qual arquivos de servlets e JavaServer Pages (JSP) são executados. Para obter uma descrição das propriedades disponíveis, Propriedades Específicas do Contêiner Web consulte.

Para cada propriedade, a página Propriedades gerais contém as seguintes informações.

Nome

O nome da propriedade.

Valor

O valor da propriedade.

Descrição

A descrição da propriedade.
