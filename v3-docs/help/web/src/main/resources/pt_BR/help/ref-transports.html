Transportes

Use a página Transportes para configurar os transportes. Cada listener de rede é associado com um transporte.

Na sua primeira instalação do GlassFish Server, existe um transporte por default, com o nome tcp. Você pode modificar ou criar um transporte TCP ou UDP ou fornecer uma implementação personalizada de transporte.

Para cada transporte, as seguintes informações são fornecidas.

Nome do transporte

O nome do transporte.

Nome da classe

O nome da classe da implementação do transporte.

Tipo do buffer de bytes

O tipo de ByteBuffer a ser usado com o transporte.

A tabela Transportes também contém as seguintes opções.

Novo

Botão para criar um novo transporte.

Deletar

Botão para deletar um ou mais transportes selecionados.
