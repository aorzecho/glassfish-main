Editar protocolo

Utilize a página Editar protocolo para modificar as configurações de um protocolo de rede existente.

A página Editar protocolo contém as opções a seguir.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Nome

O nome do protocolo. O campo Nome é de somente leitura. Você só pode especificar o nome quando cria um novo protocolo.

Segurança

Se essa opção for selecionada, a segurança será ativada para o protocolo. Se a segurança está ativada, você pode usar a guia SSL para modificar as configurações de segurança.
