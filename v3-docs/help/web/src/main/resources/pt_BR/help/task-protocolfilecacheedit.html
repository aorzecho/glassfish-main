Para editar as configurações de um arquivo de cache para um protocolo

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó Configuração da Rede.
  3. No nó Config da rede, selecione o nó Protocolos.

    É aberta a página Protocolos.

  4. Na página Protocolos, clique no nome do protocolo a ser editado.

    É aberta a página Editar Protocolo.

  5. Na página Editar Protocolo, clique na guia Arquivo de cache.

    É aberta a página Cache de Arquivo.

  6. Marque a caixa de seleção Status Ativado para habilitar o cache do arquivo.

    Por padrão, essa opção está selecionada.

  7. No campo Tempo Máximo, digite o tempo máximo, em segundos, de uma entrada de cache válida.

    Esse parâmetro controla a quantidade de informações armazenadas em cache que será utilizada após o armazenamento de um arquivo no cache. Uma entrada mais antiga do que a duração máxima será substituída pela nova entrada do mesmo arquivo. Se o conteúdo não for alterado com frequência, aumente esse valor para melhorar o desempenho.

    A duração máxima mais eficiente depende da regularidade com que os arquivos existentes são modificados. Por exemplo, se os arquivos forem modificados quatro vezes ao dia, em intervalos regulares, considere a possibilidade de definir a duração máxima em 21600 segundos (6 horas). Ou então, defina a duração máxima como sendo o tempo mais longo desejado para a exibição da versão anterior de um arquivo após a modificação.

    O valor padrão é 30.

  8. No campo Tamanho Máximo de Cache, digite o tamanho total máximo (em bytes) do cache do arquivo em disco.

    O valor padrão é 10485760.

  9. No campo Contagem Máxima de Arquivos, digite o número máximo de arquivos que pode ser armazenado no cache do arquivo.

    Se o valor for muito alto, o servidor armazenará em cache os arquivos pouco necessários, o que consome memória. Se o valor for muito baixo, os benefícios do armazenamento em cache não será obtidos. Tente diferentes valores desse atributo para encontrar a solução mais eficiente para aplicativos específicos.

    O valor padrão é 1024.

  10. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
