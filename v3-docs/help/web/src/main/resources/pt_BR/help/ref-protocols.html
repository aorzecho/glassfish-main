Protocolos

Utilize a página Protocolos para configurar os protocolos de rede.

Para cada protocolo, as informações a seguir são fornecidas.

Nome

O nome do protocolo.

Segurança ativa

Verdadeiro se a segurança estiver ativa para o protocolo ou falso se a segurança não estiver ativa para o protocolo.

A tabela Protocolos também contém as opções a seguir.

Novo

Botão utilizado para criar um novo protocolo.

Deletar

Botão utilizado para deletar um ou mais protocolos selecionados.
