Propriedades de armazenamento

Use a página Propriedades de Armazenamento para adicionar ou modificar propriedades da persistência da sessão (armazenamento) para o recipiente da Web.

A página Propriedades de Armazenamento contém as seguintes opções.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Diretório

O nome do caminho absoluto ou relativo do diretório no qual os arquivos individuais da sessão são gravados. O valor padrão é domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Um caminho relativo para o diretório temporário de trabalho para este aplicativo Web.

Propriedades Adicionais

Propriedades adicionais do armazenamento. O Servidor GlassFish não define propriedades adicionais para o armazenamento.
