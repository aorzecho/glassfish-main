Propriedades de Armazenamento

Use a página Propriedades de Armazenamento para adicionar ou modificar propriedades da persistência da sessão (armazenamento) para o contêiner Web.

A página Propriedades de Armazenamento contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Diretório

O nome do caminho absoluto ou relativo do diretório no qual os arquivos individuais da sessão são gravados. O valor default é domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Um caminho relativo para o diretório temporário de trabalho para esta aplicação Web.

Propriedades Adicionais

Propriedades adicionais do armazenamento. O GlassFish Server não define propriedades adicionais para o armazenamento.
