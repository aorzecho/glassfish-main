Propriedades específicas para sessões de recipiente da Web

As seguintes propriedades estão disponíveis para uma sessão de recipiente da Web.

enableCookies

Se definida como verdadeiro, usa cookies para rastreamento de sessão. O valor padrão é verdadeiro.

enableURLRewriting

Ativa regravação de URL. Isso fornece um rastreamento de sessão por meio de regravação de URL quando o navegador não aceita cookies. Você também deve usar uma chamada encodeURL ou encodeRedirectURL no servlet ou na página JavaServer Pages (JSP). O valor padrão é verdadeiro.

idLengthBytes

Especifica o número de bytes da ID de sessão do módulo da Web. O valor padrão é 128.
