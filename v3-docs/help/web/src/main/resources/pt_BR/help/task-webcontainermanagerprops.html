Para configurar as propriedades do gerenciador do recipiente da Web

O gerenciador de sessão fornece os meios para configurar o modo como as sessões são criadas e destruídas, onde o estado de sessão é armazenado e o número máximo de sessões disponíveis.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a configuração que está sendo alterada.
  3. No nó da configuração, selecione o nó Recipiente da Web.

    A página Propriedades Gerais é exibida.

  4. Na página Propriedades Gerais, clique na guia Propriedades do Gerenciador.

    A página Propriedades do Gerenciador é exibida.

  5. No campo Intervalo de duplicação, digite o número de segundos que antecede a exclusão dos dados de sessão inativa do armazenamento.

    O valor padrão é 60.

    Defina este valor menor do que a frequência na qual os dados da sessão são modificados. Por exemplo, este valor deveria ser o menor possível (1 segundo) para um servlet de contagem de acessos em um website com acesso frequente, do contrário você perderá alguns acessos cada vez que reiniciar o servidor.

  6. No campo Máximo de sessões, digite o número máximo de sessões que pode ser armazenado no cache.

    Um valor de -1 indica nenhum limite para o número de sessões. O valor padrão é -1.

    Após o limite ser alcançado, uma tentativa para criar uma nova sessão causa a ocorrência de uma IllegalStateException.

  7. No campo Nome do arquivo da sessão, digite o caminho absoluto ou relativo do diretório no qual o estado da sessão é preservado entre as reinicializações do aplicativo, se for possível preservar o estado.

    Se este caminho não é definido, o estado da sessão não é preservado. Por padrão, este caminho não é definido.

    Um caminho relativo para o diretório temporário do aplicativo Web.

    Esta configuração se aplica somente se o tipo de persistência for memory. Para definir o tipo de persistência, consulte Para configurar as definições de disponibilidade de um recipiente da Web.

  8. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    O Servidor GlassFish não define propriedades adicionais para o gerenciador de sessão.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir Propriedades.
  9. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
