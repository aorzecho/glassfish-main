Excluir um servidor virtual

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, selecione o nó Servidores Virtuais.

    A página Servidores Virtuais é aberta.

  3. NA página Servidores Virtuais, selecione a caixa de seleção à esquerda do nome do servidor virtual a ser excluído.
  4. Clique em Excluir.

    Não é recomendável excluir nenhum dos servidores virtuais padrão, __asadmin ou server.
