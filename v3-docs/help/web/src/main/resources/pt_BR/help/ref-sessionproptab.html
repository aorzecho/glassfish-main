Propriedades da sessão

Use a página Propriedades de Sessão para adicionar ou modificar propriedades da sessão de recipiente da Web.

A página Propriedades de Sessão contém as seguintes opções.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Tempo-limite da sessão

Especifica o número máximo de segundos durante o qual uma sessão inativa é válida. Se o valor é definido como 0, a sessão nunca expira. O valor padrão é 1.800.

Propriedades Adicionais

Propriedades adicionais das sessões de recipiente da Web. Para obter uma descrição das propriedades disponíveis, consulte Propriedades específicas para sessões de recipiente da Web.
