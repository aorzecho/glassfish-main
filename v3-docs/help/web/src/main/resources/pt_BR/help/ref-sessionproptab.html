Propriedades da Sessão

Use a página Propriedades de Sessão para adicionar ou modificar propriedades da sessão de contêiner Web.

A página Propriedades de Sessão contém as opções a seguir.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Time-out para Sessão

Especifica o número máximo de segundos durante o qual uma sessão inativa é válida. Se o valor é definido como 0, a sessão nunca expira. O valor default é 1.800.

Propriedades Adicionais

Propriedades adicionais das sessões de contêiner Web. Para obter uma descrição das propriedades disponíveis, Propriedades Específicas de Sessões do Contêiner Web consulte.
