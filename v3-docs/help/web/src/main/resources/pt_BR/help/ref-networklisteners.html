Listeners de Rede

Utilize a página Listeners de rede para configurá-las.

Na sua primeira instalação do GlassFish Server, existem três listeners de rede por default, com os nomes admin-listener, http-listener1↓ e http-listener2. O http-listener2,↓que é configurado com o SSL, é desativado por default.

Para cada listener de rede, são fornecidas as informações a seguir.

Nome

O nome do listener de rede.

Porta #

O número da porta utilizada pelo listener de rede.

Protocolo

O protocolo utilizado pelo listener de rede.

Pool de Threads

O pool de threads utilizado pelo listener de rede.

Ativado

Verdadeiro se o listener de rede estiver ativado ou falso se não estiver ativado.

A tabela Listeners de rede também contém as opções a seguir.

Novo

Botão utilizado para criar um novo listener de rede.

Deletar

Botão utilizado para deletar uma ou mais listeners de rede selecionados.
