Escutas de rede

Utilize a página Escutas de rede para configurá-las.

Na sua primeira instalação do Servidor GlassFish, existem três escutas de rede por padrão, com os nomes admin-listener, http-listener1 e http-listener2. O http-listener2, que é configurado com o SSL, é desativado por padrão.

Para cada escuta de rede, são fornecidas as seguintes informações.

Nome

O nome da escuta de rede.

Nº da porta

O número da porta em que a escuta de rede atua.

Protocolo

O protocolo utilizado pelo escuta de rede.

Conjunto de segmentos

O conjunto de segmentos utilizado pela escuta de rede.

Ativado

Verdadeiro se a escuta de rede estiver ativa ou falso se não estiver ativo.

A tabela Escutas de rede também contém as opções a seguir.

Novo

Botão utilizado para criar uma nova escuta de rede.

Excluir

Botão utilizado para excluir uma ou mais escutas de rede selecionados.
