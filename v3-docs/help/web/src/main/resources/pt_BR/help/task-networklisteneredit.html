Para editar uma escuta de rede

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó Configuração da Rede.
  3. No nó Config da rede, selecione o nó Escutas de rede.

    É exibida a página Escutas de rede.

  4. Na página Escutas de rede, clique no nome do escuta de rede que está editando.

    É aberta a página Editar Escuta de rede.

  5. Marque ou desmarque a caixa de seleção Status Ativado para alterar o status da escuta de rede.

    A escuta pode ser ativada ou desativada. Por padrão, a opção está ativada. Se o interpretador estiver desativado, qualquer tentativa de conectar ao interpretador resulta em uma exceção de soquete (java.net.ConnectException).

  6. Selecione a caixa de seleção Segurança ativada para ativar a segurança do protocolo usado pelo escuta de rede.

    Por padrão, a opção está desativada.

  7. Selecione a caixa de seleção Interpretador JK ativado para especificar que o interpretador suporta o protocolo mod-jk do Apache.

    O protocolo mod-jk é um protocolo proprietário de comunicação/rede entre o httpd (servidor da Web) da Apache e uma instância back-end do Servidor GlassFish. Se o interpretador suporta este protocolo, o httpd deveria manusear todo o conteúdo estático, enquanto quaisquer solicitações por recursos dinâmicos (ou seja, Servlets e JSPs) são roteadas para o interpretador mod-jk ativado para a rede do Servidor GlassFish, usando o protocolo mod-jk. Por padrão, a opção está desativada.

  8. No campo Porta, digite o número da porta pela qual a escuta de rede vai ouvir.

    Os valores válidos são 1 a 65.535. Em sistemas UNIX, a criação de soquetes que escutam nas portas 1 a 1024 requer privilégios de superusuário.

  9. No campo Endereço, digite o endereço IP no qual a escuta de rede vai ouvir.

    O endereço pode estar na notação de par de pontos ou IPv6. Ele pode ser qualquer (para INADDR_ANY) para escutar todos s endereços de IP. Ele pode ser um nome de host.

  10. Na lista suspensa Conjunto de segmentos, selecione o conjunto de segmentos a ser associado à escuta de rede.
  11. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e depois em Salvar.

Passos Seguintes

Você pode usar as abas SSL, HTTP e Arquivo para editar as configurações para o protocolo associado à escuta de rede. A guia SSL somente é significativa se você ativou a segurança para a escuta de rede.
