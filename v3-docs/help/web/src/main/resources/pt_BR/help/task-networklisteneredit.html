Para Editar um Listener de Rede

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó Configuração da Rede.

  3. No nó Config da rede, selecione o nó Listeners de rede.

    É exibida a página Listeners de Rede.

  4. Na página Listeners de Rede, clique no nome do listener de rede que está editando.

    É aberta a página Editar Listener de Rede.

  5. Marque ou desmarque a caixa de seleção Status Ativado para alterar o status do listener de rede.

    O listener pode ser ativado ou desativado. Por default, a opção está ativada. Se o interpretador estiver desativado, qualquer tentativa de conectar ao listener resulta em uma exceção de soquete (java.net.ConnectException).

  6. Marque a caixa de seleção Segurança Ativada para ativar a segurança do protocolo usado pelo listener de rede.

    Por default, a opção está desativada.

  7. Marque a caixa de seleção Interpretador JK ativado para especificar que o interpretador suporta o protocolo mod-jk do Apache.

    O protocolo mod-jk é um protocolo proprietário de communication/network entre o httpd do Apache (servidor da web) e uma instância de back-end do GlassFish Server. Se o listener oferecer suporte a esse protocolo, o httpd deverá tratar todo o conteúdo estático, enquanto as solicitações por recursos dinâmicos (ou seja, Servlets e JSPs) serão roteadas para o interpretador mod-jk ativado para a rede do GlassFish Server, utilizando o protocolo mod-jk. Por default, a opção está desativada.

  8. No campo Porta, digite o número da porta do listener de rede.

    Os valores válidos são 1 a 65.535. Em sistemas UNIX, criar soquetes que fazem listening nas portas 1 - 1024 exige privilégios de superusuário.

  9. No campo Endereço, digite o endereço IP do listener de rede.

    O endereço pode estar na notação de par de pontos ou IPv6. Pode ser qualquer (para INADDR_ANY) para fazer listening em todos os endereços IP. Ele pode ser um nome de host.

  10. Na lista drop-down Pool de Threads, selecione o pool de threads a ser associado ao listener de rede.

  11. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.
Próximas Etapas

Você pode usar as guias SSL, HTTP e Cache Arquivo para editar as definições para o protocolo associado ao listener de rede. A guia SSL somente é significativa se você ativou a segurança para o listener de rede.
