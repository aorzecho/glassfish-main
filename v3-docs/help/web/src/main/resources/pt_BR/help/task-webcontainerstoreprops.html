Para configurar as propriedades de armazenamento do recipiente da Web

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a configuração que está sendo alterada.
  3. No nó da configuração, selecione o nó Recipiente da Web.

    A página Propriedades Gerais é exibida.

  4. Na página Propriedades Gerais, clique na guia Propriedades de Armazenamento.

    A página Propriedades de Armazenamento é exibida.

  5. No campo Diretório, digite o nome do caminho absoluto ou relativo do diretório no qual os arquivos individuais da sessão são gravados.

    O valor padrão é domain-dir/generated/jsp/j2ee-apps/appname/appname_war.

    Um caminho relativo para o diretório temporário de trabalho para este aplicativo Web.

    Esta configuração se aplica somente se o tipo de persistência for file. Para definir o tipo de persistência, consulte Para configurar as definições de disponibilidade de um recipiente da Web.

  6. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    O Servidor GlassFish não define propriedades adicionais para o armazenamento.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir Propriedades.
  7. Clique em Salvar.
