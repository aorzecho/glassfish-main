So zeigen Sie die Server-Überwachungsdaten an

Bevor Sie beginnen

Bevor Sie Überwachungsdaten anzeigen können, müssen Sie die Überwachung konfigurieren. Einzelheiten finden Sie unter So konfigurieren Sie den Überwachungsdienst.

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  3. Klicken Sie auf die Registerkarte "Server".

    Die Seite "Serverüberwachung" wird geöffnet.

  4. Wählen Sie aus der Dropdown-Liste "Ansicht" die Komponente, für die Sie Überwachungsdaten anzeigen möchten.
  5. Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.
