So wird der JMX-Connector bearbeitet

Verwenden Sie den Admin-Service, um einen mit JSR-160 kompatiblen Remote-JMX-Connector zu konfigurieren.

Der mit JSR-160 kompatible Remote-JMX-Connector regelt den Datenaustausch zwischen JMX-Clients, z.B. JConsole, und dem JMX-Service zur Domainverwaltung.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Wählen Sie unter dem Konfigurationsknoten den Knoten "Admin-Service".

    Die Seite "JMX-Connector bearbeiten" wird geöffnet.

  3. Aktivieren Sie das Kontrollkästchen "Sicherheit aktiviert", damit der JMX-Datenaustausch verschlüsselt werden kann.

    Diese Option ist standardmäßig deaktiviert.

  4. Geben Sie im Feld Adresse die IP-Adresse des Naming Service an, bei dem der JMX-Connector-Server-Stub registriert ist.

    Dies ist nicht der Port des Server-Sockets, der die eigentliche JMX-Datenübertragung übernimmt. Dies ist die Adresse der Netzwerk-Schnittstelle, in der die RMI-Registry gestartet wird. Falls Ihr System mehrere Netzwerk-Schnittstellen besitzt, ändern Sie diesen Wert so, dass nur eine bestimmte Schnittstelle ausgewählt wird. Der Standardwert ist 0.0.0.0.

  5. Geben Sie im Feld "Port" die Portnummer ein, auf der der Naming Service (RMI-Registry) auf RMI-Client-Verbindungen horcht.

    Die einzige Verwendung dieses Naming Service besteht darin, die RMI-Stubs herunterzuladen. Falls der Standard-Port besetzt ist, wird ein freier Port verwendet. Gültige Werte sind 1 - 65535. Bei UNIX-Systemen erfordert das Erstellen von Sockets, die auf den Ports 1 - 1024 horchen, Superuser-Berechtigungen. Der Standardwert beträgt 8686.

  6. Geben Sie im Feld "Realm-Name" den Namen ein, der die gesonderte administrative Realm darstellt.

    Alle Authentifizierungsvorgänge werden von dieser Realm abgewickelt. Der Standardwert ist admin-realm .

  7. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    Der GlassFish Server legt verschiedene Eigenschaften standardmäßig fest.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".

  8. Klicken Sie auf "Speichern".

  9. Wenn Sie das Kontrollkästchen "Sicherheit aktiviert" gewählt haben, befolgen Sie die Schritte in So werden SSL-Einstellungen für den JMX-Connector bearbeitet
