So werden Logdateien für den DAS angezeigt

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Schaltfläche "Logdateien anzeigen".

    Ein Log Viewer-Fenster wird geöffnet.
