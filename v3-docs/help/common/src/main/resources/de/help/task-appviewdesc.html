So werden Deskriptoren einer Anwendung angezeigt

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie auf der Seite "Anwendungen" auf den Namen der Anwendung, deren Beschreibung Sie anzeigen möchten.

    Der Name kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  3. Klicken Sie auf der Seite "Anwendung bearbeiten" auf die Registerkarte "Beschreibung".

    Die Seite "Modulbeschreibungen" wird geöffnet.

  4. Klicken Sie auf der Registerkarte "Deployment-Deskriptoren" auf den Namen der XML-Datei, die Sie anzeigen möchten.

    Die Seite "Deployment-Deskriptor" wird geöffnet und zeigt die XML-Datei an.
