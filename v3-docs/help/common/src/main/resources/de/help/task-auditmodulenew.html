So erstellen Sie ein Überwachungsmodul

Mit diesen Modulen wird überwacht, wie GlassFish-Server wichtige Ereignisse, z. B. Fehler oder Sicherheitsverletzungen, aufzeichnet. GlassFish-Server zeichnet sämtliche Authentifizierungsereignisse in Protokolldateien auf. Ein vollständiges Zugriffsprotokoll umfasst eine Reihe von GlassFish-Server-Zugriffsereignissen.

GlassFish-Server bietet ein einfaches standardmäßiges Überwachungsmodul; weitere Informationen finden Sie unter So verwenden Sie das standardmäßige Überwachungsmodul.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Sicherheitsknoten.
  3. Wählen Sie unter dem Sicherheitsknoten den Knoten "Überwachungsmodule".

    Die Seite "Überwachungsmodule" wird geöffnet.

  4. Klicken Sie auf der Seite "Überwachungsmodule" auf "Neu".

    Die Seite "Neues Überwachungsmodul" wird geöffnet.

  5. Geben Sie in das Feld "Name" auf der Seite "Neues Überwachungsmodul" den Namen des Überwachungsmoduls ein.
  6. Geben Sie in das Feld "Klassenname" den vollqualifizierten Namen der Klasse ein, die dieses Modul implementiert.

    Der Klassenname für das standardmäßige Überwachungsmodul lautet com.sun.enterprise.security.Audit.

  7. (Optional) Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    Eine Beschreibung der zusätzlichen Eigenschaft, die für Überwachungsmodule verfügbar ist, finden Sie unter Besondere Eigenschaften von Überwachungsmodulen.

  8. Klicken Sie auf "OK".
