So werden Domainlogs gesammelt und heruntergeladen

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Domain".

    Die Seite "Domainattribute" wird geöffnet.

  2. Wählen Sie die Registerkarte "Domainlogs".

    Die Seite "Domänenlogs" wird geöffnet.

    Wenn Cluster vorhanden sind, werden Dropdown-Listen für Cluster und Instanzen angezeigt. Wenn keine Cluster vorhanden sind, wird nur eine Dropdown-Liste für Instanzen angezeigt.

  3. Wählen Sie "Cluster" oder "Instanz".

  4. Wählen Sie das Cluster oder die Instanz, für die Sie Logdateien sammeln und herunterladen.

  5. Klicken Sie auf "Logs sammeln".

    Das Herunterladenverfahren Ihres Browsers fordert Sie auf, die Datei log-files-name.zip herunterzuladen, wobei name der Name des Clusters oder der Instanz ist.
