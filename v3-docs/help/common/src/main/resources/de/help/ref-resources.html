Ressourcen

Verwenden Sie die Seite "Ressourcen", um die folgenden Ressourcen für die ausgewählte GlassFish Server-Instanz oder das -Cluster zu konfigurieren:

  • JDBC

  • Connectors

  • Ressourcenadapter-Konfigurationen

  • JMS Resources

  • JavaMail Sessions

  • JNDI

Die Seite "Ressourcen" enthält darüber hinaus die folgenden Optionen:

Ressourcen hinzufügen

Schaltfläche, mit der Sie mindestens eine Ressource aus einer glassfish-resources.xml-Datei erstellen können.
