Modul-Deskriptoren

Verwenden Sie die Seite "Modulbeschreibungen", um Modulbeschreibungen für eine Anwendung anzuzeigen.

Die Seite Module Descriptors (Modul-Deskriptoren) umfasst die folgenden Optionen.

Deployment Descriptors (Bereitstellungs-Deskriptoren)

Eine Tabelle, die den Anwendungsnamen, den Namen der untergeordneten Komponente (sofern zutreffend) und den Namen der Deskriptor-Datei angibt.

Verwandte Aufgaben

  • So zeigen Sie die Beschreibung einer Anwendung an

  • So zeigen Sie eine Liste eingesetzter Anwendungen an

  • So stellen Sie eine Webanwendung bereit

  • So stellen Sie eine Unternehmensanwendung bereit

  • So stellen Sie einen Anwendungs-Client bereit

  • So stellen Sie ein Verbindungsmodul bereit

  • So stellen Sie ein EJB Jar bereit

  • So bearbeiten Sie eine bereitgestellte Anwendung

  • So zeigen Sie Anwendungsziele an

  • So zeigen Sie Endpunkt-Informationen für einen Webdienst an

  • So starten Sie eine Anwendung

  • So starten Sie einen Anwendungs-Client

  • Eine bereitgestellte Anwendung erneut bereitstellen

  • So laden Sie eine bereitgestellte Anwendung erneut

Verwandter asadmin-Befehl

  • list-applications(1)
