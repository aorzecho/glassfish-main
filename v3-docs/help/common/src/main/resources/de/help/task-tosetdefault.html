So legen Sie den Standardbereich fest

Der Bereich default realm ist der Bereich, den GlassFish-Server zur Authentifizierung und Autorisierung verwendet, wenn in der Bereitstellungsbeschreibung einer Anwendung kein Bereich angegeben ist.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie den Knoten "Sicherheit".

    Die Seite "Sicherheit" wird geöffnet.

  3. Wählen Sie aus der Dropdown-Liste "Standardbereich" den aktiven Bereich aus, den der Server zur Authentifizierung verwenden soll.

    Der Standardwert ist file.

  4. Klicken Sie auf "Speichern".
