Domainlogs

Verwenden Sie die Seite "Domainlogs", um Logdateien für GlassFish Server zu sammeln und herunterzuladen.

Die Seite "Domainlogs" umfasst die folgenden Optionen. Sie können ein Cluster oder eine Serverinstanz wählen, für die Logs gesammelt werden sollen.

Cluster

Eine Dropdown-Liste der Cluster, sofern Cluster vorhanden sind.

Instanz

Eine Dropdown-Liste der Serverinstanzen.

Logs sammeln

Schaltfläche, mit der Logs in einer log-files-name.zip-Datei gesammelt werden, wobei name der Name des Clusters oder der Instanz ist.

Die log-files-name.zip-Datei wird mit dem Herunterladenverfahren Ihres Browsers heruntergeladen.

Verwandte Aufgaben

  • So werden Domainlogs gesammelt und heruntergeladen

  • So werden Logdateien für den DAS angezeigt

Entsprechende asadmin-Unterbefehle

collect-log-files(1)
