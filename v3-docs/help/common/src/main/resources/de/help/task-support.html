So zeigen Sie die Seite "GlassFish News" für GlassFish-Server an

Die Seite "Glass Fish News" umfasst eine Reihe von Links mit zusätzlichen Informationen zu Oracle GlassFish-Server, z. B. Newsletter, Artikel, technische Tipps, Diskussionsforen und relevante Blogs.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Allgemeine Aufgaben".

    Die Seite "Allgemeine Aufgaben" wird geöffnet.

  2. Klicken Sie unter "GlassFish News" auf den Link "GlassFish News".

    Die Webseite "GlassFish News" wird in einer separaten Registerkarte oder in einem Fenster Ihres Browsers geöffnet.
