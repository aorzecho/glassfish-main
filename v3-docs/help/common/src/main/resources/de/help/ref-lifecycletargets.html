Lebenszyklusmodul-Ziele

Verwenden Sie die Seite "Lebenszyklusmodulziele", um Ziel-Cluster und eigenständige Serverinstanzen anzuzeigen, auf denen das Lebenszyklusmodul aktiviert werden kann.

Die Seite "Lebenszyklusmodul" enthält die folgenden Informationen.

Zielname

Der Name des Clusters oder der eigenständigen Serverinstanz.

Aktiviert

Zeigt true an, wenn das Lebenszyklusmodul auf dem Ziel aktiviert ist, und false, wenn das Lebenszyklusmodul deaktiviert ist.

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Ziele verwalten

Schaltfläche zum Verwalten von Lebenszyklusmodulzielen.

Weitere Aktionen

Dropdown-Liste mit folgenden Aktionen.

Aktivieren

Aktion zum Aktivieren des Lebenszyklusmoduls auf dem ausgewählten Ziel.

Deaktivieren

Aktion zum Deaktivieren des Lebenszyklusmoduls auf dem ausgewählten Ziel.
