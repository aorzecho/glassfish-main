Neues Administratorpasswort

Auf der Seite "Neues Administratorpasswort" können Sie das Passwort für das aktuelle Administratorkonto ändern.

Die Seite "Neues Administratorpasswort" enthält die folgenden Optionen.

Benutzer-ID

Die Benutzer-ID des aktuellen Administrators. Dies ist ein schreibgeschütztes Feld.

Gruppenliste

Die Gruppe, zu der der Administrator gehört. Dies ist ein schreibgeschütztes Feld. Der Wert ist asadmin.

Neues Passwort

Das neue Passwort für den Administrator.

Neues Passwort bestätigen

Das neue Passwort für den Administrator.

Verbundene Aufgabe

  • So ändern Sie das Administrator-Passwort

Verwandter asadmin-Befehl

  • change-admin-password(1)
