Für JACC-Provider spezifische Eigenschaften

Die folgende Eigenschaft ist für einen JACC-Provider verfügbar.

Repository

Das Verzeichnis, das die JACC-Policy-Datei enthält. Der Standard-GlassFish Server-JACC-Provider verwendet das Standardverzeichnis ${com.sun.aas.instanceRoot}/generated/policy. Diese Eigenschaft ist nicht standardmäßig für den einfachenJACC-Provider von GlassFish Server definiert.
