Webdienst-Endpunkt-Informationen

Auf der Seite "Webdienst-Endpunkt-Informationen" finden Sie Informationen über einen Webdienst-Endpunkt.

Die Seite "Webdienst-Endpunkt-Informationen" enthält die folgenden Informationen.

Application Name (Anwendungsname)

Der Name der bereitgestellten Anwendung. Klicken Sie auf diesen Link oder auf die Schaltfläche "Zurück", um zur Seite "Anwendung bearbeiten" zurückzukehren.

Der Name kann eine optionale Versionskennung enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide .

Tester

Der Link "Tester" wird nur angezeigt, wenn der Bereitstellungstyp der Anwendung ein anderer Typ als RI ist (z. B. 109). Klicken Sie auf diesen Link, um eine Seite anzuzeigen, auf der Sie Parameter zum Testen der Methoden des Webdienstes bereitstellen können.

WSDL

Die Web Services Definition Language (WSDL) für den Endpunkt. Klicken Sie auf diesen Link, um die WSDL-Datei in einer neuen Browser-Registerkarte oder einem neuen Fenster anzuzeigen.

Beachten Sie, dass einige Webbrowser versuchen, die WSDL-Quelle zu interpretieren, wenn Sie auf den WSDL-Link klicken. Aus diesem Grunde wird dann eine leere Seite angezeigt. Bei diesen Browsern verwenden Sie für die leere Seite den Befehl "View Source" (Quelle anzeigen), um die WSDL-Quelle anzuzeigen.

Name des Endpunkts

Der Name des Endpunkts.

Dienstname

Der Name des Webdienstes.

Portname

Der Name des Webdienstports.

Bereitstellungstyp

Der Bereitstellungstyp des Webdienstes (z. B. RI oder 109).

Implementierungstyp

Der Implementierungstyp des Webdienstes (z. B. SERVLET oder EJB).

Implementierungsklassenname

Der Implementierungsklassenname des Webdienstes.

Endpunkt-Adressen-URI

Der URI der Endpunkt-Adresse.

Namespace

Der Ziel-Namespace, der dem Webdienst zugeordnet ist.

Beschreibung

Eine Beschreibung des Endpunkts (sofern vorhanden).
