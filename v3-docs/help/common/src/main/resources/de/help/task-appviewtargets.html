So zeigen Sie Anwendungsziele an

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Wählen Sie eine Anwendung, für die Sie die Ziele anzeigen möchten.

    Die Anwendung kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite für die Anwendung wird geöffnet.

  3. Wählen Sie die Registerkarte "Ziel".

    Die Seite "Anwendungsziele" wird geöffnet, und die aktuell für die Anwendung definierten Ziele werden angezeigt.

  4. Um eine Anwendung auf einer bestimmten Zielinstanz oder einem Cluster zu aktivieren oder zu deaktivieren, klicken Sie auf das Kontrollkästchen neben dem Ziel und wählen aus dem Dropdown-Menü "Weitere Aktionen" den Eintrag "Aktivieren" oder "Deaktivieren" aus.

  5. Um das Load Balancing für ein Ziel auf einer bestimmten Zielinstanz oder einem Cluster zu aktivieren, klicken Sie auf das Kontrollkästchen neben dem Ziel und wählen aus dem Dropdown-Menü "Weitere Aktionen" den Eintrag "Load Balancer aktivieren" oder "Load Balancer deaktivieren".

  6. Um eine Anwendung mit bestimmten virtuellen Servern zu verknüpfen, wählen Sie für die Zielinstanz oder das Cluster die Option "Virtuelle Server verwalten".

    Die Seite "Virtuelle Serverziele" wird geöffnet.

    Um virtuelle Zielserver hinzuzufügen, wählen Sie diese aus der Liste "Verfügbare Ziele" aus und verwenden die Schaltfläche "Hinzufügen", um sie in die Liste "Ausgewählte Ziele" zu verschieben.

    Das Modul oder die Anwendung wird standardmäßig mit dem standardmäßigen virtuellen Server (server) verknüpft. Verwenden Sie die Schaltfläche "Alle hinzufügen", um alle Ziele hinzuzufügen. Verwenden Sie die Schaltflächen "Entfernen" und "Alle entfernen", um Ziele aus der Liste "Ausgewählte Ziele" zu entfernen.

    Klicken Sie auf "Speichern", um zur Seite "Anwendungsziele" zurückzukehren.

  7. Um Zielinstanzen oder Cluster für eine Anwendung hinzuzufügen oder zu löschen, wählen Sie "Ziele verwalten".

    Die Seite "Ziel verwalten" wird geöffnet.

    Um Ziele hinzuzufügen, wählen Sie diese aus der Liste "Verfügbare Ziele" aus und verwenden die Schaltfläche "Hinzufügen", um sie in die Liste "Ausgewählte Ziele" zu verschieben.

    Verwenden Sie die Schaltfläche "Alle hinzufügen", um alle Ziele hinzuzufügen. Verwenden Sie die Schaltflächen "Entfernen" und "Alle entfernen", um Ziele aus der Liste "Ausgewählte Ziele" zu entfernen.

    Klicken Sie auf "Speichern", um zur Seite "Anwendungsziele" zurückzukehren.

  8. Klicken Sie auf "Speichern".

    Die Anwendung ist nun für die bearbeitete Liste der Ziele verfügbar.
