Serverüberwachung

Verwenden Sie die Seite "Serverüberwachung", um Überwachungsdaten für die Serverinstanz anzuzeigen.

Die Seite "Serverüberwachung" enthält die folgenden Informationen.

Instanzname

Der Name der Serverinstanz.

Anzeigen

Der virtuelle Server, HTTP-Listener, Garbage Collector oder das Threadsystem, für die Überwachungsstatistiken angezeigt werden.

Eine Beschreibung der überwachten Eigenschaften finden Sie in Chapter 8, Administering the Monitoring Service, im Oracle GlassFish Server Administration Guide.

Verwandte Aufgaben

  • So wird der Überwachungsservice konfiguriert

  • So werden Anwendungsüberwachungsdaten angezeigt

  • So werden Serverüberwachungsdaten angezeigt

  • So werden Ressourcenüberwachungsdaten angezeigt

Zugehörige asadmin-Befehle

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
