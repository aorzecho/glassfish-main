So werden Anwendungsüberwachungsdaten angezeigt

Bevor Sie beginnen

Bevor Sie Überwachungsdaten anzeigen können, müssen Sie die Überwachung konfigurieren. Weitere Einzelheiten finden Sie in So wird der Überwachungsservice konfiguriert.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Überwachungsdaten".

    Die Seite "Überwachung" wird geöffnet.

  2. Klicken Sie in der Spalte "Überwachungsdaten anzeigen" auf den Link "Anwendung", der der Serverinstanz entspricht, die Sie überwachen.

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  3. Wählen Sie aus der Dropdown-Liste "Anwendung" die Anwendung, für die Sie Überwachungsdaten anzeigen möchten.

  4. Wählen Sie aus der Dropdown-Liste "Komponente" die Anwendungskomponente, für die Sie Überwachungsdaten anzeigen möchten.

  5. Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.

Alternative Schritte:

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  3. Wählen Sie aus der Dropdown-Liste "Anwendung" die Anwendung, für die Sie Überwachungsdaten anzeigen möchten.

  4. Wählen Sie aus der Dropdown-Liste "Komponente" die Anwendungskomponente, für die Sie Überwachungsdaten anzeigen möchten.

  5. Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.
