Ressourcenziele verwalten

Verwenden Sie die Seite "Ressourcenziele verwalten", um Ziel-Cluster und eigenständige Serverinstanzen zu ändern, auf denen eine Ressource aktiviert werden kann. Die Ressource kann nur auf Zielen in der Spalte "Ausgewählte Ziele" aktiviert werden.

Die Seite "Ressourcenziele" enthält die folgenden Informationen.

Verfügbare Ziele

Die Cluster und Instanzen, auf denen die Ressource nicht aktiviert werden kann.

Ausgewählte Ziele

Die Cluster und Instanzen, auf denen die Ressource aktiviert werden kann.

Hinzufügen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Alle hinzufügen

Schaltfläche zum Verschieben aller "Verfügbaren Ziele" nach "Ausgewählte Ziele".

Entfernen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Ausgewählte Ziele" nach "Verfügbare Ziele".

Alle entfernen

Schaltfläche zum Verschieben aller "Ausgewählten Ziele" nach "Verfügbare Ziele".
