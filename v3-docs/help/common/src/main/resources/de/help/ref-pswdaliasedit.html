Passwort-Alias bearbeiten

Verwenden Sie die Seite "Passwort-Alias bearbeiten", um das Passwort zu ändern, das zu einem vorhandenen Passwort-Alias gehört.

Die Seite "Passwort-Alias bearbeiten" enthält die folgenden Optionen:

Aliasname

Der Name des Alias. Diese Option kann nicht bearbeitet werden.

Passwort

Das Passwort, das das Passwort-Alias in verschlüsselter Form speichert.

Passwort bestätigen

Das Passwort, das das Passwort-Alias in verschlüsselter Form speichert. Die Werte "Passwort" und "Passwort bestätigen" werden verglichen, um sicherzustellen, dass das Passwort fehlerfrei eingegeben wurde.
