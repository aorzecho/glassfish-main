Dateibenutzer

Auf der Seite "Dateibenutzer" können Sie die Benutzer des file-Bereichs verwalten. Benutzer und Gruppen im file-Bereich sind in der Schlüsseldatei aufgeführt, deren Speicherort durch die Eigenschaft file angegeben wird.

Ein Benutzer im Bereich file kann zu einer Java-Gruppe gehören, einer Benutzerkategorie, die durch gemeinsame Merkmale klassifiziert ist. So können Kunden einer eCommerce-Anwendung z. B. zur Gruppe CUSTOMER gehören, die Großkunden würden jedoch der Gruppe PREFERRED angehören. Das Kategorisieren von Benutzern in Gruppen vereinfacht die Zugangssteuerung bei einer großen Benutzeranzahl.

Nach der Installation von GlassFish-Server ist der während der Installation eingegebene Administrator zunächst der einzige Benutzer. Dieser Benutzer ist standardmäßig Mitglied der Gruppe asadmin im Bereich admin-realm und damit berechtigt, Änderungen an GlassFish-Server vorzunehmen. Benutzer, die dieser Gruppe zugewiesen sind, verfügen über Administratorrechte und können damit auf das asadmin-Tool und die Administration Console zugreifen.

Die Seite Dateibenutzer enthält eine Liste mit Benutzern für den ausgewählten Bereich. Für jeden Benutzer werden die folgenden Informationen zur Verfügung gestellt:

Benutzer-ID

Der Name des Benutzers.

Gruppenliste

Die Gruppen, denen der Benutzer angehört.

Die Tabelle "Dateibenutzer" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Benutzers.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Benutzer.

Verwandte Aufgaben

  • So fügen Sie einen Benutzer für den Dateibereich hinzu

  • So bearbeiten Sie Informationen zu Dateibereichsbenutzern

  • So löschen Sie einen Benutzer aus dem Dateibereich

Verwandte asadmin-Befehle

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
