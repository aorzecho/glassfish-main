Dateibenutzer

Auf der Seite "Dateibenutzer" können Sie die Benutzer der Datei-Realm verwalten. Benutzer und Gruppen in der Datei-Realm sind in der Schlüsseldatei aufgeführt, deren Verzeichnis durch die Eigenschaft file angegeben wird.

Ein Benutzer in der Realm file kann zu einer Java-Gruppe gehören, einer Benutzerkategorie, die durch gemeinsame Merkmale klassifiziert ist. So können Kunden einer eCommerce-Anwendung z B. zur Gruppe CUSTOMER gehören, die Großkunden würden jedoch der Gruppe PREFERRED angehören. Das Kategorisieren von Benutzern in Gruppen vereinfacht die Zugangssteuerung bei einer großen Benutzeranzahl.

Nach der Installation von GlassFish Server ist der während der Installation eingegebene Administrator zunächst der einzige Benutzer. Dieser Benutzer ist standardmäßig Mitglied der Gruppe asadmin in der Realm admin-realm und damit berechtigt, Änderungen an GlassFish Server vorzunehmen. Benutzer, die dieser Gruppe zugewiesen sind, verfügen über Administratorberechtigungen und können damit auf das asadmin-Tool und die Administrationskonsole zugreifen.

Die Seite "Dateibenutzer" enthält eine Liste mit Benutzern für die ausgewählte Realm. Für jeden Benutzer werden die folgenden Informationen zur Verfügung gestellt:

Benutzer-ID

Der Name des Benutzers.

Gruppenliste

Die Gruppen, denen der Benutzer angehört.

Die Tabelle "Dateibenutzer" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Benutzers.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Benutzer.

Verwandte Aufgaben

  • So werden Benutzer der Datei-Realm hinzugefügt

  • So werden Benutzerinformationen der Datei-Realm bearbeitet

  • So werden Benutzer aus der Datei-Realm gelöscht
