Verfügbare Updates

Die Seite Available Updates (Verfügbare Updates) liefert Informationen zu neueren Versionen von Komponenten des Installation-Image, die installiert sind.

Halten Sie zum Installieren der verfügbaren Aktualisierungen zunächst die GlassFish-Server-Domain an. Verwenden Sie dann einen der folgenden Befehle:

  • glassfish3/bin/updatetool (zum Starten des grafischen Aktualisierungstools)

  • glassfish3/bin/pkg (Befehlszeilenversion des Aktualisierungstools)

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Kapitel 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Das Aktualisierungstool verfügt über eine eigene Online-Hilfe.

Starten Sie den Server neu, wenn die Installation der Aktualisierungen abgeschlossen ist.

Die Seite Available Updates (Verfügbare Updates) umfasst die folgenden Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Available Updates

Für jede Komponente, für die eine aktuellere Version verfügbar ist, stellt diese Tabelle die folgenden Informationen bereit.

Komponente

Der Name der Komponente.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, zum Beispiel Anwendungsserver, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Installierte Version

Die Versionsnummer des Pakets, in dem die aktuelle Komponente bereitgestellt wird.

Neue Version

Die Versionsnummer der verfügbaren Aktualisierung.

Ändern

Unterschied in der Größe zwischen der installierten Version und der verfügbaren Aktualisierung.

Quelle

Der Name des Netzwerk-Repository, aus dem die Komponenten abgerufen werden.
