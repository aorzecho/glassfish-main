So zeigen Sie verfügbare Updates für Komponenten an

Die Seite Available Updates (Verfügbare Updates) liefert Informationen zu neueren Versionen von Komponenten des Installation-Image, die installiert sind.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Aktualisierungstool" aus.

    Die Seite "Verfügbare Add-Ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-Ons" auf die Registerkarte "Verfügbare Updates".

    Die Seite "Verfügbare Updates" wird geöffnet. Wenn keine Updates verfügbar sind, ist die Tabelle leer.

Nächste Schritte

Halten Sie zum Installieren der verfügbaren Aktualisierungen zunächst die GlassFish-Server-Domain an. Verwenden Sie dann einen der folgenden Befehle:

  • as-install-parent/bin/updatetool (zum Starten des grafischen Aktualisierungstools)

  • as-install-parent/bin/pkg (Befehlszeilenversion des Aktualisierungstools)

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Kapitel 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Das Aktualisierungstool verfügt über eine eigene Online-Hilfe.

Starten Sie den Server neu, wenn die Installation der Aktualisierungen abgeschlossen ist.
