So werden verfügbare Updates für Komponenten angezeigt

Die Seite "Verfügbare Updates" liefert Informationen zu neueren Versionen von Komponenten des Installationsimages, die installiert sind.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Updatetool" aus.

    Die Seite "Verfügbare Add-ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-ons" auf die Registerkarte "Verfügbare Updates".

    Die Seite "Verfügbare Updates" wird geöffnet. Wenn keine Updates verfügbar sind, ist die Tabelle leer.

Nächste Schritte

Halten Sie zum Installieren der verfügbaren Aktualisierungen zunächst die GlassFish Server-Domain an. Verwenden Sie dann einen der folgenden Befehle:

  • as-install-parent/bin/updatetool, das das grafische Updatetoolutility startet

  • as-install-parent/bin/pkg, eine Befehlszeilenversion des Updatetools

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Chapter 11, Extending and Updating GlassFish Server, im Oracle GlassFish Server Administration Guide. Das Aktualisierungstool verfügt über eine eigene Onlinehilfe.

Starten Sie den Server neu, wenn die Installation der Aktualisierungen abgeschlossen ist.
