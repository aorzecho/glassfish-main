Ressourcenziele

Verwenden Sie die Seite "Ressourcenziele", um Ziel-Cluster und eigenständige Serverinstanzen anzuzeigen, auf denen die Ressource aktiviert werden kann.

Die Seite "Ressourcenziele" enthält die folgenden Informationen für die einzelnen Ziele.

Zielname

Der Name des Clusters oder der eigenständigen Serverinstanz.

Aktiviert

Zeigt ein Häkchen an, wenn die Ressource auf dem Ziel aktiviert ist, oder ein X, wenn die Ressource deaktiviert ist.

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Ziele verwalten

Schaltfläche zum Verwalten von Ressourcenzielen.

Aktivieren

Schaltfläche zum Aktivieren der Ressource auf dem ausgewählten Ziel.

Deaktivieren

Aktion zum Deaktivieren der Ressource auf dem ausgewählten Ziel.
