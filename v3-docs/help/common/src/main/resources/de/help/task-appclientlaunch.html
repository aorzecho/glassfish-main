So werden Anwendungsclients gestartet

Bevor Sie beginnen

Stellen Sie sicher, dass der zu startende Anwendungsclient aktiviert ist. Der Client muss ein Web-Client sein, der mit Java Web Start ausgeführt werden kann.

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" auf den Namen des zu startenden Anwendungsclients.

    Der Name kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  3. Stellen Sie sicher, dass das Kontrollkästchen "Java Web Start" aktiviert ist.

    Java Web Start muss aktiviert sein, bevor Sie einen Anwendungsclient starten können. Wenn das Kontrollkästchen "Java Web Start" nicht aktiviert ist, aktivieren Sie es und klicken dann auf "Speichern".

  4. Klicken Sie in der Tabelle "Module und Komponenten" unter der Spalte "Aktion" auf "Starten".

    Die Seite "Anwendungsclient starten" wird angezeigt.

  5. Geben Sie auf der Seite "Anwendungsclient starten", die Argumente ein, die an den Anwendungsclient übergeben werden sollen.

  6. Klicken Sie auf "Starten".
