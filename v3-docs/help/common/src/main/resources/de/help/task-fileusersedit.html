So bearbeiten Sie Informationen zu Dateibereichsbenutzern

  1. Erweitern Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Cluster-Konfiguration, für die Sie Admin-Bereichsbenutzer verwalten möchten.
  2. Erweitern Sie den Knoten "Sicherheit".
  3. Erweitern Sie den Knoten "Bereiche".
  4. Wählen Sie den Knoten file.

    Die Seite "Bereich bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Bereich bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Dateibenutzer" wird geöffnet.

  6. Klicken Sie auf der Seite "Dateibenutzer" in der Spalte "Benutzer-ID" der Tabelle auf den Namen des zu ändernden Benutzers.

    Die Seite "Dateibereichsbenutzer bearbeiten" wird geöffnet.

  7. Bearbeiten Sie das Feld "Gruppenliste", um die Gruppen, zu denen der Benutzer gehört, zu ändern.

    Verwenden Sie Kommata zum Trennen der Gruppennamen. Diese Gruppen müssen an keiner weiteren Stelle definiert werden.

  8. Geben Sie in die Felder "Neues Passwort" und "Neues Passwort bestätigen" ein neues Passwort ein, um das Passwort des Benutzers zu ändern.
  9. Klicken Sie auf "Speichern".
