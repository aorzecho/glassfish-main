Anwendungsziele

Verwenden Sie die Seite "Anwendungsziele", um Ziel-Cluster und eigenständige Serverinstanzen anzuzeigen, auf denen die Anwendung aktiviert werden kann.

Die Seite "Anwendungsziele" enthält die folgenden Informationen.

Zielname

Der Name des Clusters oder der eigenständigen Serverinstanz.

Aktiviert

Zeigt true an, wenn die Anwendung auf dem Ziel aktiviert ist, und false, wenn die Anwendung deaktiviert ist.

Lastenausgleich aktiviert

Zeigt true an, wenn die Anwendung für den Lastenausgleich auf dem Ziel aktiviert ist, und false, wenn die Anwendung für den Lastenausgleich deaktiviert ist.

Virtual Servers (Virtuelle Server)

Um die Anwendung mit bestimmten virtuellen Servern auf dem Ziel zu verknüpfen, wählen Sie für das Ziel die Option "Virtuelle Server verwalten".

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Ziele verwalten

Schaltfläche zum Verwalten von Anwendungszielen.

Weitere Aktionen

Dropdown-Liste mit folgenden Aktionen.

Aktivieren

Aktion zum Aktivieren der Anwendung auf dem ausgewählten Ziel.

Deaktivieren

Aktion zum Deaktivieren der Anwendung auf dem ausgewählten Ziel.

Lastenausgleich aktivieren

Aktion zum Aktivieren der Anwendung für den Lastenausgleich auf dem ausgewählten Ziel.

Lastenausgleich deaktivieren

Aktion zum Deaktivieren der Anwendung für den Lastenausgleich auf dem ausgewählten Ziel.
