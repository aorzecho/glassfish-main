Allgemeine Aufgaben

Die Seite Common Tasks (Allgemeine Aufgaben) enthält Tastenkombinationen für allgemeine Administration Console-Aufgaben.

Führen Sie über die Seite Common Tasks (Allgemeine Aufgaben) die folgenden Operationen aus:

  • So zeigen Sie die Seite "GlassFish News" für GlassFish-Server an

  • So zeigen Sie die Seite "Abonnements" für GlassFish-Server an

  • So zeigen Sie eine Liste eingesetzter Anwendungen an

  • So stellen Sie eine Webanwendung bereit

  • So stellen Sie eine Unternehmensanwendung bereit

  • So stellen Sie einen Anwendungs-Client bereit

  • So stellen Sie ein Verbindungsmodul bereit

  • So stellen Sie ein EJB Jar bereit

  • So zeigen Sie installierte Komponenten an

  • So installieren Sie Add-On-Komponenten

  • So zeigen Sie verfügbare Updates für Komponenten an

  • So erstellen Sie einen JDBC-Verbindungspool

Darüber hinaus finden Sie auf der Seite Common Tasks (Allgemeine Aufgaben) Verknüpfungen zur GlassFish-Server-Dokumentation.
