Updatetool-Konfiguration

Die Seite "Konfiguration" liefert Konfigurationsinformationen zu Updatetool und ermöglicht Ihnen, die Art und Weise zu ändern, in der Updatetool eine Verbindung mit dem Internet herstellt. Um andere Aspekte der Updatetool-Konfiguration zu ändern, verwenden Sie die Standalone-Anwendung updatetool im Verzeichnis as-install-parent/bin.

Die Updatetool-Konfigurationsseite enthält folgende Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Bevorzugtes Repository

Der Name des bevorzugten Netzwerk-Repositorys, das mit diesem Installationsimage verknüpft ist, Beispiel: glassfish.org.

Internet Connection (Internetverbindung)

Wie GlassFish Server eine Verbindung zum Internet herstellt:

Direct Connection (Direktverbindung)

Diese Option gibt an, dass GlassFish Server direkt mit dem Internet verbunden ist.

Through a Proxy (Über einen Proxy)

Diese Option gibt an, dass GlassFish Server eine Verbindung zum Internet über einen Proxyserver herstellt. Wenn Sie diese Option auswählen, werden dieser Seite die Felder Proxy Host (Proxyhost) und Proxy Port (Proxyport) hinzugefügt.

Proxyhost

Der Hostname des Proxyservers für den Computer, auf dem GlassFish Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem GlassFish Server ausgeführt wird, über einen Proxyserver mit dem Internet verbunden wird.

Proxy-Port

Die Portnummer des Proxyservers für den Computer, auf dem GlassFish Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem GlassFish Server ausgeführt wird, über einen Proxyserver mit dem Internet verbunden wird.

Repositorys (Datensammlungen)

Diese Tabelle liefert die folgenden Informationen zu allen Netzwerk-Repositorys, die mit diesem Installationsimage verknüpft sind:

Repository

Der Name des Repositorys, zum Beispiel dev.glassfish.org .

Verwandte Aufgaben

So werden Netzwerkproxyeinstellungen für den Zugriff auf Netzwerk-Repositorys konfiguriert
