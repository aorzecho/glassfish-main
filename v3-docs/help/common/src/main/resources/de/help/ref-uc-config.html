Aktualisierungstool-Konfiguration

Die Seite "Konfiguration" liefert Konfigurationsinformationen zu Aktualisierungstool und ermöglicht Ihnen, die Art und Weise zu ändern, in der Aktualisierungstool eine Verbindung mit dem Internet herstellt. Um andere Aspekte der Aktualisierungstool-Konfiguration zu ändern, verwenden Sie die eigenständige Anwendung updatetool (Aktualisierungstool) im Verzeichnis as-install-parent/bin .

Die Aktualisierungstool-Konfigurationsseite enthält folgende Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Bevorzugtes Repository

Der Name des bevorzugten Netzwerk-Repositorys, das mit diesem Installations-Image verbunden ist, zum Beispiel glassfish.org.

Internet Connection (Internetverbindung)

Wie GlassFish-Server eine Verbindung zum Internet herstellt:

Direct Connection (Direktverbindung)

Diese Option gibt an, dass GlassFish-Server direkt mit dem Internet verbunden ist.

Through a Proxy (Über einen Proxy)

Diese Option gibt an, dass GlassFish-Server eine Verbindung zum Internet über einen Proxyserver herstellt. Wenn Sie diese Option auswählen, werden dieser Seite die Felder Proxy Host (Proxyhost) und Proxy Port (Proxyport) hinzugefügt.

Proxy-Host

Der Hostname des Proxy-Servers für den Computer, auf dem GlassFish-Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem GlassFish-Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Proxy-Port

Die Port-Nummer des Proxy-Servers für den Computer, auf dem GlassFish-Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem GlassFish-Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Repositories (Datensammlungen)

Diese Tabelle liefert die folgenden Informationen zu allen Netzwerk-Repositorys, die mit diesem Installations-Image verbunden sind:

Repository

Der Name des Repositorys, zum Beispiel dev.glassfish.org.

Verwandte Aufgaben

So konfigurieren Sie Netzwerk-Proxy-Einstellungen für den Zugriff auf Netzwerk-Repositorys

Verwandte asadmin-Befehle
