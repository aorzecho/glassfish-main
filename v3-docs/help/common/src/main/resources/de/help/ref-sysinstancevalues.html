Instanzwerte

Verwenden Sie die Seite "Instanzenwerte", um die Werte einer Konfigurationsystemseigenschaft für die GlassFish Server-Instanzen zu bearbeiten, die auf die ausgewählte benannte Konfiguration verweisen.

Die Seite "Instanzenwerte" zeigt eine Liste von Instanzen an, die auf die ausgewählte Konfiguration verweisen, in der die Eigenschaft festgelegt ist.

Instanzname

Der Name, der der Instanz bei der Erstellung zugewiesen wurde. Wenn Sie auf diesen Namen klicken, wird die Seite "Allgemeine Informationen" für die Instanz geöffnet.

Clustername

Der Name des Clusters, zu dem die Instanz gegebenenfalls gehört. Dieses Feld ist schreibgeschützt.

Standardwert

Der Wert, der für die Eigenschaft in der benannten Konfiguration festgelegt ist, auf den die Instanz verweist. Dieses Feld ist schreibgeschützt.

Wert übergehen

Der Wert der Eigenschaft, der für die ausgewählte Instanz festgelegt ist. Durch diesen Wert wird der Standardwert überschrieben.

Verwandte Aufgaben

So legen Sie instanzenspezifische Eigenschaften fest
