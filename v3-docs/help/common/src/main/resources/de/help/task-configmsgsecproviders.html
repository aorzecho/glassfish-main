So konfigurieren Sie GlassFish-Server-Funktionen für die Verwendung durch Meldungssicherheitsanbieter

  1. Wenn Sie ein Benutzernamen-Token verwenden, konfigurieren Sie eine Benutzerdatenbank (sofern erforderlich).

    Wenn Sie ein Benutzernamen/Paswort-Token verwenden, konfigurieren Sie zunächst einen geeigneten Bereich und eine geeignete Benutzerdatenbank für den Bereich.

    Die Konfiguration einer Benutzerdatenbank ist unter So bearbeiten Sie einen Bereich beschrieben.

  2. Verwalten Sie Zertifikate und private Schlüssel, sofern erforderlich.
Nächste Schritte

Nach der Konfiguration der GlassFish-Server-Funktionen für die Verwendung durch Meldungssicherheitsanbieter können die mit GlassFish-Server installierten Anbieter aktiviert werden, wie unter So bearbeiten Sie eine Meldungssicherheitskonfiguration beschrieben.
