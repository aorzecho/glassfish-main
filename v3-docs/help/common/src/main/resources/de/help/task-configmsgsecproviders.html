So werden GlassFish Server-Funktionalitäten zur Verwendung durch Nachrichtensicherheitsprovider konfiguriert

  1. Wenn Sie ein Benutzernamen-Token verwenden, konfigurieren Sie eine Benutzerdatenbank (sofern erforderlich).

    Wenn Sie ein Benutzername-/Kennworttoken verwenden, konfigurieren Sie zunächst eine geeignete Realm und eine geeignete Benutzerdatenbank für die Realm.

    Das Konfigurieren einer Benutzerdatenbank wird in So wird eine Realm bearbeitet beschrieben.

  2. Verwalten Sie Zertifikate und Private Keys, sofern erforderlich.

Nächste Schritte

Nach der Konfiguration der GlassFish Server-Funktionalität für die Verwendung durch Nachrichtensicherheitsprovider können die mit GlassFish Server installierten Provider aktiviert werden, wie unter So wird eine Nachrichtensicherheitskonfiguration bearbeitet beschrieben .
