So wird das Audit-Logging aktiviert oder deaktiviert

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. Wählen Sie unter dem Konfigurationsknoten den Sicherheitsknoten aus.

    Die Seite "Sicherheit" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Sicherheit" das Kontrollkästchen "Audit-Logging aktiviert", um das Audit-Logging zu aktivieren.

    Zum Aktivieren der Auditprotokollierung müssen Sie auch die Eigenschaft auditOn für das aktive Auditmodul auf True festlegen.

  5. Klicken Sie auf "Speichern".
