So löschen Sie einen Benutzer aus dem Admin-Bereich

  1. Erweitern Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Cluster-Konfiguration, für die Sie Admin-Bereichsbenutzer verwalten möchten.
  2. Erweitern Sie den Knoten "Sicherheit".
  3. Erweitern Sie den Knoten "Bereiche".
  4. Wählen Sie den Knoten admin-realm.

    Die Seite "Bereich bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Bereich bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Admin-Benutzer" wird geöffnet.

  6. Aktivieren Sie auf der Seite "Admin-Benutzer" das Kontrollkästchen links neben dem Namen des zu löschenden Benutzers.
  7. Klicken Sie auf "Löschen".

    Sie werden aufgefordert, den Löschvorgang zu bestätigen.

  8. Klicken Sie auf "OK", um den Benutzer zu löschen.
