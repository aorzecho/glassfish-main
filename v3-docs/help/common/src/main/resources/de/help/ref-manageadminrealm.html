Admin-Benutzer

Auf der Seite "Admin-Benutzer" können Sie Benutzer des admin-realm verwalten. Der admin-realm ist ein Dateibereich. Benutzer und Gruppen im file-Bereich sind in der Schlüsseldatei aufgeführt, deren Speicherort durch die Eigenschaft file angegeben wird.

Nach der Installation von GlassFish-Server ist der während der Installation eingegebene Administrator zunächst der einzige Benutzer. Dieser Benutzer ist standardmäßig Mitglied der Gruppe asadmin im Bereich admin-realm und damit berechtigt, Änderungen an GlassFish-Server vorzunehmen. Benutzer, die dieser Gruppe zugewiesen sind, verfügen über Administratorrechte und können damit auf das asadmin-Tool und die Administration Console zugreifen.

Die Seite "Admin-Benutzer" enthält eine Liste mit Benutzern für den ausgewählten Bereich. Für jeden Benutzer werden die folgenden Informationen zur Verfügung gestellt:

Benutzer-ID

Der Name des Benutzers.

Gruppenliste

Die Gruppen, denen der Benutzer angehört.

Die Tabelle "Admin-Benutzer" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Benutzers.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Benutzer.

Verwandte Aufgaben

  • So fügen Sie dem Admin-Bereich einen Benutzer hinzu

  • So bearbeiten Sie Informationen zu Admin-Bereichsbenutzern

  • So löschen Sie einen Benutzer aus dem Admin-Bereich

Verwandte asadmin-Befehle

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
