Neues Passwort-Alias

Verwenden Sie die Seite "Neues Passwort-Alias", um ein neues Passwort-Alias zu erstellen, das in einer Passwortdatei oder in der Domänenkonfigurationsdatei verwendet wird.

Die Seite "Neues Passwort-Alias" enthält die folgenden Optionen:

Aliasname

Der Name des Alias. Dieser Name muss für alle Passwort-Aliase in der Domäne eindeutig sein.

Passwort

Das Passwort, das das Passwort-Alias in verschlüsselter Form speichert.

Passwort bestätigen

Das Passwort, das das Passwort-Alias in verschlüsselter Form speichert. Die Werte "Passwort" und "Passwort bestätigen" werden verglichen, um sicherzustellen, dass das Passwort fehlerfrei eingegeben wurde.
