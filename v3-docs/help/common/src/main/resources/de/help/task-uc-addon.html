So werden Add-on-Komponenten installiert

Bei der Installation einer Add-on-Komponente sind folgende Vorgänge erforderlich:

  • Übertragung des Contents der Komponente und der Komponentenabhängigkeiten aus einem Netzwerk-Repository

  • Installation der Komponente unter dem GlassFish Server Installationsimage auf Ihrem System

Falls eine Komponente andere Komponenten erfordert, die noch nicht installiert sind, werden die erforderlichen Komponenten ebenfalls installiert.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Updatetool" aus.

    Die Seite "Verfügbare Add-ons" wird geöffnet.

  2. Wählen Sie auf der Seite "Verfügbare Add-ons" in der Tabelle mit den verfügbaren Add-ons die zu installierenden Komponenten aus.

    Sind keine Add-on-Komponenten verfügbar, ist die Tabelle leer.

    • Um eine einzelne Komponente auszuwählen, aktivieren Sie das Kontrollkästchen neben dem Namen der Komponente.

    • Um alle Komponenten auszuwählen, klicken Sie im Header der Tabelle auf das Symbol "Alle auswählen".

    • Um die Auswahl aller Komponenten aufzuheben, klicken Sie im Header der Tabelle auf das Symbol "Auswahl für alle aufheben".

  3. Klicken Sie auf "Installieren".

    Sie werden in einem Dialogfeld aufgefordert zu bestätigen, dass die ausgewählten Komponenten und alle dafür erforderlichen Komponenten installiert werden sollen.

  4. Klicken Sie in diesem Dialogfeld auf "OK".

    Sie werden gefragt, ob Sie die Lizenzvereinbarungen für die zu installierenden Komponenten akzeptieren.

  5. Akzeptieren Sie die Lizenzvereinbarungen für die zu installierenden Komponenten.

    GlassFish Server bestätigt den Abschluss der Installation. Die Komponenten werden aus der Tabelle der Add-on-Komponenten gelöscht und zur Tabelle mit den installierten Komponenten auf der Seite "Installierte Komponenten" hinzugefügt.

  6. Starten Sie den Server neu.

    Weitere Informationen finden Sie in der folgenden Dokumentation:

    • To Stop a Domain im Oracle GlassFish Server Administration Guide

    • To Start a Domain im Oracle GlassFish Server Administration Guide
