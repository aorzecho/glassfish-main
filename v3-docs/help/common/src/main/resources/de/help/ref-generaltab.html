Allgemeine Informationen

Verwenden Sie die Seite Allgemeine Informationen, um GlassFish-Server-Einstellungen zu überprüfen und Java Virtual Machine-Daten anzuzeigen.

Die Seite Allgemeine Informationen enthält die folgenden Informationen.

Schaltfläche Stopp

Klicken Sie auf die Schaltfläche Stopp, um GlassFish-Server anzuhalten.

Schaltfläche Neustart

Klicken Sie auf die Schaltfläche Neustart, um GlassFish-Server neu zu starten.

Schaltfläche Protokolldateien anzeigen

Klicken Sie auf die Schaltfläche "Protokolldateien anzeigen", um die Protokolldateien für eine GlassFish Server-Instanz oder einen Cluster anzuzeigen.

Name

Der Name des Servers.

JVM

Wenn Sie auf einen JVM-Bericht klicken, wird ein separates Fenster geöffnet, das Berichte über die Java Virtual Machine anzeigt, einschließlich eines zusammenfassenden Berichts, Speichermanagement- und Aufräumbericht, Klassenladungsbericht und einem aktuellen Thread-Abbild.

Configuration Directory (Konfigurationsverzeichnis)

Das Verzeichnis auf dem Hostcomputer, das die Konfigurationsdateien enthält, z. B. domain.xml.

Installierte Version

Die Produktversionskennung.

Debug (Fehlersuche)

Gibt an, ob die Fehlersuche aktiviert ist. Ist die Fehlersuche aktiviert, wird die Port-Nummer ebenfalls angezeigt.

HTTP-Port(s)

Die aktuell konfigurierten Ports für HTTP-Anforderungen und -Antworten.

IIOP-Port(s)

Die aktuell konfigurierten Ports für IIOP-Anforderungen und -Antworten.

  • So zeigen Sie Protokolldateien für den DAS an
