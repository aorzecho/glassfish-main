Komponentendetails

Die Seite "Komponentendetails" bietet detaillierte Informationen zu einer Komponente.

Die Seite Component Details (Komponentendetails) umfasst die folgenden Optionen.

Back (Zurück)

Diese Schaltfläche wechselt zu der Seite zurück, auf der Sie die Komponente ausgewählt haben, deren Details Sie gerade anzeigen.

Package Name (Paketname)

Der Name des Pakets, in dem die Komponente geliefert wird.

Unique Identifier (Eindeutige ID)

Die vollständige ID dieser Komponente wie im Repository vorliegend.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, zum Beispiel Anwendungsserver, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Version

Die Versionsnummer des Pakets, in dem die Komponente geliefert wird.

Date Published (Veröffentlichungsdatum)

Datum und Uhrzeit der Veröffentlichung dieser Komponentenversion im Repository. Dieses Datum ist nicht das Datum und die Uhrzeit der Installation der Komponente innerhalb des Installations-Image.

Installationsgröße

Der Festplattenspeicherplatz, den die Installation der Komponente beansprucht.

Quelle

Für installierte Komponenten ist dies der Name des Netzwerk-Repository, von dem die Komponente abgerufen wurde.

Für eine Add-On-Komponente oder ein Update ist dies der Name des Netzwerk-Repositorys, von dem die Komponente abzurufen ist.

Description (Beschreibung)

Eine Übersicht der Komponente.

Verwandte Aufgaben

So zeigen Sie detaillierte Informationen zu einer Komponente an
