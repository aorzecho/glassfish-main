Systemeigenschaften

Auf der Seite "Systemeigenschaften" werden die Java-Systemeigenschaften aufgelistet, die für die ausgewählte benannte Konfiguration definiert sind. Java-Systemeigenschaften werden an das Java-Anwendungsstarttool über die Option -D des Java-Anwendungsstarttools weitergeleitet, wenn GlassFish-Server gestartet wird.

Diese Eigenschaften definieren die Standardwerte der Porteinstellungen für alle GlassFish-Server-Instanzen, die auf die Konfiguration verweisen.

Weitere Informationen zu vordefinierten Porteinstellungen in einer Konfiguration finden Sie unter Vordefinierte Systemeigenschaften .

Für jede Instanz werden folgende Informationen angezeigt:

Name der Instanzenvariablen

Der Name der Systemeigenschaft.

Standardwert

Der Wert, der für die Eigenschaft in der benannten Konfiguration festgelegt ist, auf den die Instanz verweist. Dieses Feld ist schreibgeschützt.

Instanzenwerte

Ein Link zur Seite Instanzenwerte für die Eigenschaft.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Eigenschaften. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.

Die Seite "Systemeigenschaften" enthält darüber hinaus die folgenden Optionen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Dynamische Neukonfiguration

Wenn diese Option aktiviert ist, werden Änderungen an der Konfiguration auf die Instanzen übertragen, die auf die Konfiguration verweisen, ohne dass die Instanzen neu gestartet werden müssen.

Clusters und/oder Instanzen, die diese Konfiguration verwenden

Eine Liste der Cluster und Instanzen, die auf die Konfiguration verweisen, mit einer Angabe, ob die einzelnen Cluster oder Instanzen ausgeführt werden:

  • Wenn der Cluster oder die Instanz ausgeführt wird, wird ein Häkchen (✓) angezeigt.

  • Wenn der Cluster oder die Instanz angehalten ist, wird ein Ausrufezeichen (!) angezeigt.

Verwandte Aufgaben

  • So legen Sie Standardwerte für Konfigurationseigenschaften fest
