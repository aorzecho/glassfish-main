So konfigurieren Sie Protokollebenen

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Konfigurationen".
  2. Klicken Sie unter dem Knoten "Konfiguration" auf die Serverinstanz oder die Cluster-Konfiguration, für die Sie Einstellungen der Protokollfunktion konfigurieren möchten.

    Die Seite "Konfiguration" wird geöffnet.

  3. Klicken Sie auf der Seite "Konfiguration" auf "Einstellungen Protokollfunktion".

    Die Seite "Einstellungen Protokollfunktion" für das ausgewählte Konfigurationsziel wird geöffnet.

  4. Klicken Sie auf der Seite "Einstellungen Protokollfunktion" auf das Register Protokollebenen.

    Die Seite "Modul-Protokollebenen" wird geöffnet.

  5. Ändern Sie auf der Seite "Modul-Protokollebenen" die Protokollebenen für ein Modul oder mehrere Module wie folgt.
    • Wählen Sie zum Ändern der Protokollebene für ein einzelnes Modul einen neuen Wert aus der Dropdown-Liste gegenüber dem Modul aus, dessen Protokollebene Sie ändern wollen.
    • So ändern Sie die Protokollebene für eine Gruppe von Modulen:
      1. Aktivieren Sie die Kontrollkästchen neben den Namen der Module, deren Protokollebenen Sie ändern wollen.
      2. Wählen Sie einen Wert aus der Dropdown-Liste oben in der Tabelle aus.
      3. Klicken Sie auf die Schaltfläche "Ebene ändern".

    Meldungen der ausgewählten oder einer höheren Ebene werden im Protokoll angezeigt. Wählen Sie einen der folgenden Werte (sortiert von der höchsten zur niedrigsten Ebene):

    • SCHWERWIEGEND

    • WARNUNG

    • INFO

    • KONFIGURATION

    • GENAU

    • GENAUER

    • SEHR GENAU

    • DEAKTIVIERT

    Einzelheiten zu den Bedeutungen der Protokollebenen finden Sie unter Modul-Protokollebenen. Die Standardebene lautet INFO. Diese Protokollebene umfasst Meldungen der Stufen INFO, WARNUNG und SCHWERWIEGEND.

  6. Klicken Sie auf "Speichern".

    Änderungen an der Protokollebene treten sofort in Kraft. Sie werden auch zur Verwendung gespeichert, wenn der Server neu gestartet wird.
