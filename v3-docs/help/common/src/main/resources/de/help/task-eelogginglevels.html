So werden Logebenen konfiguriert

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfigurationen" ein.

  2. Klicken Sie unter dem Knoten "Konfiguration" auf die Serverinstanz oder die Clusterkonfiguration, für die Sie Einstellungen der Protokollfunktion konfigurieren möchten.

    Die Seite "Konfiguration" wird geöffnet.

  3. Klicken Sie auf der Seite "Konfiguration" auf "Einstellungen Protokollfunktion".

    Die Seite "Logger-Einstellungen" für das ausgewählte Konfigurationsziel wird geöffnet.

  4. Klicken Sie auf der Seite "Logger-Einstellungen" auf die Registerkarte "Logebenen".

    Die Seite "Modullogebenen" wird geöffnet.

  5. Ändern Sie auf der Seite "Modullogebenen" die Logebenen für ein Modul oder mehrere Module wie folgt.

    • Wählen Sie zum Ändern der Logebene für ein einzelnes Modul einen neuen Wert aus der Dropdown-Liste gegenüber dem Modul, dessen Logebene Sie ändern wollen.

    • So ändern Sie die Logebene für eine Gruppe von Modulen:

      1. Aktivieren Sie die Kontrollkästchen neben den Namen der Module, deren Logebenen Sie ändern wollen.

      2. Wählen Sie einen Wert aus der Dropdown-Liste oben in der Tabelle.

      3. Klicken Sie auf die Schaltfläche "Ebene ändern".

    Meldungen der ausgewählten oder einer höheren Ebene werden im Log angezeigt. Wählen Sie einen der folgenden Werte (sortiert von der höchsten zur niedrigsten Ebene):

    • Schwerwiegend

    • Warnung

    • Info

    • CONFIG

    • Genau

    • Genauer

    • Sehr genau

    • Deaktiviert

    Einzelheiten zur Bedeutung der Logebenen finden Sie unter Modullogebenen. Die Standardebene lautet INFO (Info). Dieses Log umfasst standardmäßig Meldungen der Ebenen INFO, WARNING (Warnung) und SEVERE (Schwerwiegend).

  6. Klicken Sie auf "Speichern".

    Änderungen an der Logebene treten sofort in Kraft. Sie werden auch zur Verwendung gespeichert, wenn der Server neu gestartet wird.
