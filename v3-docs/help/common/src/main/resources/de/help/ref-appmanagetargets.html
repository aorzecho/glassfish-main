Ziele verwalten

Verwenden Sie die Seite "Ziele verwalten", um Ziel-Cluster und eigenständige Serverinstanzen zu ändern, auf denen eine Anwendung aktiviert werden kann. Die Anwendung kann nur auf Zielen in der Spalte "Ausgewählte Ziele" aktiviert werden.

Die Seite "Ziele verwalten" enthält die folgenden Informationen.

Verfügbare Ziele

Die Cluster und Instanzen, auf denen die Anwendung nicht bereitgestellt wird.

Ausgewählte Ziele

Die Cluster und Instanzen, auf denen die Anwendung bereitgestellt wird.

Hinzufügen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Alle hinzufügen

Schaltfläche zum Verschieben aller Ziele von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Entfernen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Ausgewählte Ziele" nach "Verfügbare Ziele".

Alle entfernen

Schaltfläche zum Verschieben aller Ziele von "Ausgewählte Ziele" nach "Verfügbare Ziele".
