Konfigurationen

Verwenden Sie die Seite "Konfigurationen", um Konfigurationen zu verwalten und die Standalone-Serverinstanzen und Cluster anzuzeigen, die die Konfigurationen verwenden.

Die default-config-Konfiguration wird von GlassFish Server vordefiniert. Die server-config-Konfiguration wird automatisch für den Domain Administration Server (DAS) erstellt, wenn die Domain erstellt wird. Weitere Informationen finden Sie in About Named Configurations im Oracle GlassFish Server High Availability Administration Guide.

Für jede Konfiguration werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Konfiguration.

Instanz

Die Liste der Standalone-Instanzen und Cluster, die die Konfiguration verwenden.

Die Tabelle "Konfigurationen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Konfiguration.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Konfigurationen.
