Konfigurationen

Verwenden Sie die Seite "Konfigurationen", um Konfigurationen zu verwalten, und zeigen Sie mithilfe der Konfigurationen die eigenständigen Serverinstanzen und Cluster an.

Die Konfiguration default-config ist durch GlassFish-Server vordefiniert. Die Konfiguration server-config wird für den Domain Administration Server (DAS) automatisch bei Erstellung der Domäne erstellt. Weitere Informationen finden Sie unter About Named Configurations in Oracle GlassFish Server 3.1 High Availability Administration Guide.

Für jede Konfiguration werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Konfiguration.

Instanz

Die Liste der eigenständigen Instanzen und Cluster, die diese Konfiguration verwenden.

Die Tabelle "Konfigurationen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Konfiguration.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Konfigurationen.
