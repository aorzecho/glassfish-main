Datei-Realm-Benutzer bearbeiten

Verwenden Sie die Seite "Datei-Realm-Benutzer bearbeiten" zum Ändern des Kennworts eines Benutzers für die aktuelle Realm.

Hinweis:

Wenn die sichere Administration aktiviert ist, können Sie das Kennwort eines Admin-Realm-Benutzers nicht in einen leeren Wert ändern.

Die Seite "Datei-Realm-Benutzer bearbeiten" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Realm-Name

Der Name der aktuell ausgewählten Realm. Dies ist ein schreibgeschütztes Feld.

Benutzer-ID

Der Name des Accounts, den Sie bearbeiten. Dies ist ein schreibgeschütztes Feld.

Gruppenliste

Gruppen, zu denen dieser Benutzer gehört. Trennen Sie Gruppen durch Kommata.

Für Benutzer von admin-realm ist dies ein schreibgeschütztes Feld, und die Gruppe ist asadmin.

Neues Kennwort

Neues Kennwort für diesen Benutzer.

Neues Kennwort bestätigen

Neues Kennwort für diesen Benutzer.

Verwandte Aufgaben

  • So werden Benutzer der Datei-Realm hinzugefügt

  • So werden Benutzerinformationen der Datei-Realm bearbeitet

  • So werden Benutzer aus der Datei-Realm gelöscht
