Dateibereichsbenutzer bearbeiten

Verwenden Sie die Seite "Dateibereichsbenutzer bearbeiten" zum Ändern des Passworts eines Benutzers für den aktuellen Bereich.

Die Seite "Dateibereichsbenutzer bearbeiten" umfasst die folgenden Optionen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Bereichsname

Der Name des aktuell ausgewählten Bereichs. Dies ist ein schreibgeschütztes Feld.

Benutzer-ID

Der Name des Zugangs, den Sie bearbeiten. Dies ist ein schreibgeschütztes Feld.

Gruppenliste

Gruppen, zu denen dieser Benutzer gehört. Trennen Sie Gruppen durch Kommata.

Für Benutzer von admin-realm ist dies ein schreibgeschütztes Feld, und die Gruppe ist asadmin.

Neues Passwort

Neues Passwort für diesen Benutzer.

Neues Passwort bestätigen

Neues Passwort für diesen Benutzer.

Verwandte Aufgaben

  • So fügen Sie einen Benutzer für den Dateibereich hinzu

  • So bearbeiten Sie Informationen zu Dateibereichsbenutzern

  • So löschen Sie einen Benutzer aus dem Dateibereich

Verwandte asadmin-Befehle

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
