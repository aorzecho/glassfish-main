So bearbeiten Sie eine Meldungssicherheitskonfiguration

Zum Aktivieren der Meldungssicherheit für in GlassFish-Server bereitgestellte Webdienstendpunkte muss ein Anbieter als Standardanbieter für die Serverseite angegeben werden. Wenn Sie einen Standardanbieter für die Meldungssicherheit aktivieren, müssen ebenfalls Anbieter für die Clients der in GlassFish-Server bereitgestellten Webdienste aktiviert werden. Informationen zum Aktivieren der Anbieter, die von Clients verwendet werden, werden in So aktivieren Sie die Meldungssicherheit für Anwendungs-Clients diskutiert.

Zum Aktivieren der Meldungssicherheit für Webdienstaufrufe von bereitgestellten Endpunkten muss ein standardmäßiger Clientanbieter angegeben werden. Wenn ein standardmäßiger Clientanbieter für GlassFish-Server aktiviert wurde, muss sichergestellt werden, dass die von in GlassFish-Server bereitgestellten Endpunkten aufgerufenen Dienste für Sicherheit auf Meldungsebene konfiguriert sind.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie den Knoten "Sicherheit".
  3. Wählen Sie den Knoten "Meldungssicherheit".

    Die Seite "Nachrichtensicherheitskonfigurationen" wird geöffnet.

  4. Klicken Sie auf der Seite "Nachrichtensicherheitskonfigurationen" auf den Namen der Authentifizierungslayer der Konfiguration, die Sie bearbeiten.

    Standardmäßig ist die einzige konfigurierte Nachrichtensicherheitskonfiguration SOAP.

    Die Seite "Neue Meldungssicherheitskonfiguration bearbeiten" wird geöffnet.

  5. Wählen Sie auf der Seite "Meldungssicherheitskonfiguration bearbeiten" aus der Dropdown-Liste "Standardanbieter" einen Server-seitigen Anbieter aus.

    Der standardmäßige Serveranbieter, auch Standardanbieter genannt, gibt den Serveranbieter an, der für Anwendungen aufgerufen wird, für die kein spezifischer Serveranbieter festgelegt ist. Zur Auswahl stehen Serveranbieter, XWS_ServerProvider oder keine. Standardmäßig wird kein Anbieter ausgewählt, so dass GlassFish-Server Server-seitig keinen Meldungssicherheitsanbieter aufruft. Wählen Sie im Allgemeinen ServerProvider als Standardanbieter aus.

  6. Wählen Sie aus der Dropdown-Liste "Standard-Client-Anbieter" einen Client-Anbieter aus.

    Der standardmäßige Clientanbieter gibt den Clientanbieter an, der für Anwendungen aufgerufen wird, für die kein spezifischer Clientanbieter festgelegt ist. Zur Auswahl stehen ClientProvider, XWS_ClientProvider oder keine. Standardmäßig ist kein Anbieter ausgewählt, so dass GlassFish-Server Client-seitig keinen Meldungssicherheitsanbieter aufruft. Wählen Sie im Allgemeinen für den Standard-Clientanbieter "keine" aus. Wählen Sie ClientProvider, um einen Standardanbieter und eine Meldungssicherheitsrichtilinie zu aktivieren, die für Webdienstaufrufe von in GlassFish-Server bereitgestellten Webdienstendpunkten angewendet werden.

  7. Klicken Sie auf "Speichern".
  8. Anweisungen zum Ändern von Meldungssicherheitsrichtlinien für aktivierte Client- oder Serveranbieter finden Sie unter So bearbeiten Sie eine Konfiguration von Meldungssicherheitsanbietern.
