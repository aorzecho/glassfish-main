So werden Anwendungsclient-Stubs heruntergeladen

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" auf den Namen des Anwendungsclients, für den Sie Stubs herunterladen.

    Der Name kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  3. Klicken Sie in der Tabelle "Module und Komponenten" unter der Spalte "Aktion" auf "Client-Stubs herunterladen".

    Die Datei client-stubs.zip wird mit dem Herunterladenverfahren Ihres Browsers heruntergeladen.
