Domainattribute festlegen

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Domain".

    Die Seite "Domainattribute" wird geöffnet.

  2. Geben Sie im Feld "Anwendungs-Root" den vollständigen Pfad zu dem Verzeichnis ein, in dem sich die bereitgestellten Anwendungen befinden.

    Der Standardwert ist domain-dir/applications.

  3. Geben Sie im Feld "Log-Root" den vollständigen Pfad des Verzeichnisses ein, in dem die Serverlogdateien gespeichert werden.

    Der Standardwert ist domain-dir/logs.

  4. Lassen Sie das Feld "Gebietsschema" leer, um das Standardgebietsschema des Hosts zu verwenden.

    Ein Gebietsschema ist eine ID, die eine bestimmte Kombination aus Sprache und Region angibt. Beispiel: Das Gebietsschema für Englisch (USA) ist en_US, das Gebietsschema für Japanisch ist ja_JP. Um ein nicht englisches Gebietsschema zu verwenden, muss GlassFish Server lokalisiert, d.h. in eine andere Sprache übersetzt werden.

  5. Klicken Sie auf "Speichern".

  6. Starten Sie den Server neu.
