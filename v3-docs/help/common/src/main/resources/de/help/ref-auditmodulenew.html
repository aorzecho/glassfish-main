Neues Überwachungsmodul

Auf der Seite "Neues Überwachungsmodul" können Sie ein neues Überwachungsmodul definieren.

Die Seite "Neues Überwachungsmodul" umfasst die folgenden Optionen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Name

Name des neuen Überwachungsmoduls.

Klassenname

Der vollqualifizierte Name der Klasse, welche dieses Modul implementiert. Der Klassenname für das standardmäßige Überwachungsmodul lautet com.sun.enterprise.security.Audit.

Weitere Eigenschaften

Weitere Eigenschaften für das Überwachungsmodul. Eine Beschreibung der zusätzlichen Eigenschaft, die für Überwachungsmodule verfügbar ist, finden Sie unter Besondere Eigenschaften von Überwachungsmodulen.

Verwandte Aufgaben

  • So erstellen Sie ein Überwachungsmodul

  • So bearbeiten Sie ein Überwachungsmodul

  • So löschen Sie ein Überwachungsmodul

  • So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung

  • So legen Sie das aktive Überwachungsmodul fest

  • So verwenden Sie das standardmäßige Überwachungsmodul

Verwandte asadmin-Befehle

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
