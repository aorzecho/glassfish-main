So zeigen Sie die Seite "Abonnements" für GlassFish-Server an

  1. Wählen Sie in der Navigationsstruktur den Knoten "Allgemeine Aufgaben".

    Die Seite "Allgemeine Aufgaben" wird geöffnet.

  2. Klicken Sie unter "GlassFish News" auf den Link "Abonnements".

    Die Webseite "GlassFish-Abonnements" wird in einem separaten Tab oder Fenster Ihres Browsers geöffnet.
