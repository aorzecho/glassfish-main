Passwort-Aliase

Verwenden Sie die Seite "Passwort-Aliase", um Passwort-Aliase zu erstellen und zu verwalten.

Auf der Seite "Passwort-Aliase" wird eine Liste der Passwort-Aliase in der Domäne angezeigt. Für jedes Alias werden folgende Informationen angezeigt:

Name

Der Name, der dem Passwort-Alias bei seiner Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Passwort-Alias bearbeiten für das Alias geöffnet.

Die Tabelle "Passwort-Aliase" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines Passwort-Alias.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Passwort-Aliase.
