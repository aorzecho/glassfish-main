Modul-Protokollebenen

Verwenden Sie die Seite Modul-Protokollebenen, um die Protokollebenen für einzelne Module zu konfigurieren.

Folgende Informationen werden für jedes Modul zur Verfügung gestellt.

Name der Protokollfunktion

Der Name der Protokollfunktion für das Modul (z. B. javax.enterprise.system.tools.admin).

Protokollfunktionsebene

Die aktuelle Protokollierungsebene für das Modul.

Die Tabelle "Einstellungen für Protokollierfunktion" enthält darüber hinaus die folgenden Optionen.

Protokollfunktion hinzufügen

Schaltfläche zum Hinzufügen eines Protokollfunktionsmoduls. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Einstellungen Protokollfunktion" hinzugefügt. Geben Sie Informationen über das Protokollfunktionsmodul in die neue Zeile ein.

Protokollfunktion löschen

Schaltfläche zum Löschen eines oder mehrerer Module.

Ebene

Dropdown-Liste, aus der Sie eine Protokollierungsebene auswählen können, die auf alle ausgewählten Module angewendet werden soll.

Ebene ändern

Schaltfläche zum Ändern der Protokollierungsebene für eines oder mehrere ausgewählte Module.

Die folgenden Protokollebenen sind verfügbar. Sie sind in der Reihenfolge vom höchsten zum niedrigsten aufgelistet.

SCHWERWIEGEND

Ereignisse, die die normale Programmausführung stören.

WARNUNG

Warnungen, einschließlich Ausnahmen.

INFO

Nachrichten, die mit Serverkonfiguration oder Serverstatus in Zusammenhang stehen, ausgenommen Fehler.

KONFIGURATION

Nachrichten, die mit der Serverkonfiguration in Zusammenhang stehen.

GENAU

Minimale Überlänge

GENAUER

Moderate Überlänge.

SEHR GENAU

Maximale Überlänge.

DEAKTIVIERT

Keine Protokollnachrichten.
