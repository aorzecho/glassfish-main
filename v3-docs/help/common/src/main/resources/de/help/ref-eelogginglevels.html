Modullogebenen

Verwenden Sie die Seite "Modul-Logebenen", um die Logebenen für einzelne Module zu konfigurieren.

Folgende Informationen werden für jedes Modul zur Verfügung gestellt.

Logger-Name

Der Name der Protokollfunktion für das Modul (z.B. javax.enterprise.system.tools.admin).

Logger-Ebene

Die aktuelle Protokollierungsebene für das Modul.

Die Tabelle "Logger-Einstellungen" enthält darüber hinaus die folgenden Optionen.

Logger hinzufügen

Schaltfläche zum Hinzufügen eines Logger-Moduls. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Einstellungen Protokollfunktion" hinzugefügt. Geben Sie Informationen über das Protokollfunktionsmodul in die neue Zeile ein.

Protokollfunktion löschen

Schaltfläche zum Löschen eines oder mehrerer Module.

Ebene

Dropdown-Liste, aus der Sie eine Protokollierungsebene auswählen können, die auf alle ausgewählten Module angewendet werden soll.

Ebene ändern

Schaltfläche zum Ändern der Protokollierungsebene für eines oder mehrere ausgewählte Module.

Die folgenden Logebenen sind verfügbar. Sie sind in der Reihenfolge vom höchsten zum niedrigsten aufgelistet.

Schwerwiegend

Ereignisse, die die normale Programmausführung stören.

Warnung

Warnungen, einschließlich Ausnahmen.

Info

Nachrichten, die mit Serverkonfiguration oder Serverstatus in Zusammenhang stehen, ausgenommen Fehler.

CONFIG

Nachrichten, die mit der Serverkonfiguration in Zusammenhang stehen.

Genau

Minimale Überlänge

Genauer

Moderate Überlänge.

Sehr genau

Maximale Überlänge.

Deaktiviert

Keine Protokollnachrichten.

Verwandte Aufgaben

  • So werden allgemeine Logging-Einstellungen konfiguriert

  • So werden Logebenen konfiguriert

  • So werden Logdateien für den DAS angezeigt
