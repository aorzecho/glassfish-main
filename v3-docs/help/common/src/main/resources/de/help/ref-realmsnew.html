Neuer Bereich

Auf der Seite "Neuer Bereich" können Sie einen neuen Bereich erstellen.

Die Seite "Neuer Bereich" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Name

Der Name des neuen Bereichs.

Klassenname

Der Klassenname des neuen Bereichs.

Zulässige Optionen sind in der folgenden Tabelle aufgeführt.

Bereichsname

Klassenname

file

com.sun.enterprise.security.auth.realm.file.FileRealm

certificate

com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

ldap

com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

JDBC

com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

solaris

com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

PAM

com.sun.enterprise.security.auth.realm.solaris.PamRealm

Um einen benutzerdefinierten Bereich zu erstellen, wählen Sie das untere Optionsfeld und geben Sie den Klassennamen für den zu erstellenden benutzerdefinierten Bereich ein.

Weitere Eigenschaften

Weitere Eigenschaften für den Bereich. Gültige Eigenschaften sind abhängig vom Typ des Bereichs.

  • Für die FileRealm-Klasse spezifische Eigenschaften

  • Für die CertificateRealm-Klasse spezifische Eigenschaften

  • Für die JDBCRealm-Klasse spezifische Eigenschaften

  • Für die LDAPRealm-Klasse spezifische Eigenschaften

  • Für die SolarisRealm-Klasse spezifische Eigenschaften

  • Für die PamRealm-Klasse spezifische Eigenschaften
