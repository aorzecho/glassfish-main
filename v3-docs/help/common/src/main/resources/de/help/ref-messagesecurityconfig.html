Nachrichtensicherheitskonfigurationen

Auf der Seite "Nachrichtensicherheitskonfigurationen" können Sie die verfügbaren Konfigurationen für Nachrichtensicherheitsanbieter anzeigen und ändern.

Um Nachrichtensicherheit zu konfigurieren, benötigen Sie ein grundlegendes Verständnis von Sicherheits- und Webservice-Konzepten. Weitere Informationen finden Sie in Kapitel 3, Administering Message Security in Oracle GlassFish Server 3.1 Security Guide und Kapitel 6, Administering Web Applications in Oracle GlassFish Server 3.1 Administration Guide.

GlassFish-Server implementiert die Meldungssicherheit mithilfe von Meldungssicherheitsanbietern, die auf SOAP-Verarbeitungsebene integriert sind. Die Meldungssicherheitsanbieter hängen von den Sicherheitsfunktionen von GlassFish-Server ab.

Für jede Konfiguration werden die folgenden Informationen zur Verfügung gestellt.

Authentifizierungs-Layer

Die Nachrichtenlayer, in der Authentifizierung durchgeführt wird. Standardmäßig wird eine SOAP-Konfiguration bereitgestellt. Es ist darüber hinaus möglich, eine HttpServlet-Konfiguration bereitzustellen.

Standardanbieter

Der Standard-Serveranbieter für diese Konfiguration. Standardmäßig ist kein Serveranbieter konfiguriert.

Standard-Clientanbieter

Der Standard-Clientanbieter für diese Konfiguration. Standardmäßig ist kein Standard-Clientanbieter konfiguriert.

Die Tabelle "Nachrichtensicherheitskonfiguration" enthält auch die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Konfiguration.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Konfigurationen.

Verwandte Aufgaben

  • So konfigurieren Sie GlassFish-Server-Funktionen für die Verwendung durch Meldungssicherheitsanbieter

  • So erstellen Sie eine Meldungssicherheitskonfiguration

  • So bearbeiten Sie eine Meldungssicherheitskonfiguration

  • So löschen Sie eine Meldungssicherheitskonfiguration

  • So erstellen Sie eine Meldungssicherheitsanbieter-Konfiguration

  • So bearbeiten Sie eine Konfiguration von Meldungssicherheitsanbietern

  • So löschen Sie eine Konfiguration des Meldungssicherheitsanbieters

  • So aktivieren Sie die Meldungssicherheit für Anwendungs-Clients
