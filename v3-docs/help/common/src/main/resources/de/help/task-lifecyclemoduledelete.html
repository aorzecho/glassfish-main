So löschen Sie ein Lebenszyklusmodul

  1. Wählen Sie in der Navigationsstruktur den Knoten "Lebenszyklusmodul".

    Die Seite "Lebenszyklusmodule" wird geöffnet.

  2. Aktivieren Sie auf der Seite "Lebenszyklusmodule" das Kontrollkästchen links neben dem Überwachungsmodul, das Sie löschen möchten.
  3. Klicken Sie auf "Löschen".
