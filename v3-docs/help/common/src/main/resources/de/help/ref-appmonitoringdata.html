Anwendungsüberwachung

Verwenden Sie die Seite Anwendungsüberwachung, um Überwachungsdaten für eine ausgewählte Anwendung anzuzeigen. Eine Beschreibung der überwachten Eigenschaften finden Sie in Kapitel 8, Administering the Monitoring Service in Oracle GlassFish Server 3.1 Administration Guide.

Verwandte Aufgaben

  • So konfigurieren Sie den Überwachungsdienst

  • So zeigen Sie Überwachungsdaten der Anwendung an

  • So zeigen Sie die Server-Überwachungsdaten an

  • So zeigen Sie Ressourcen-Überwachungsdaten an

Verwandte asadmin-Befehle

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
