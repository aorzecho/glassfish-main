So werden JVM-Profiler-Einstellungen konfiguriert

Ein Profiler generiert Daten, die zur Analyse der Performance sowie zur Ermittlung von potenziellen Engpässen verwendet werden können.

  1. Wählen Sie in der Navigationsstruktur den Konfigurationsknoten.

  2. Select the configuration that you are changing.

  3. Wählen Sie unter dem Konfigurationsknoten den JVM-Einstellungsknoten aus.

    Die Seite "Allgemeine JVM-Einstellungen" wird geöffnet.

  4. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "Profiler".

    Die Seite "JVM-Profilereinstellungen" wird geöffnet.

  5. Geben Sie die Profilereinstellungen an.

    Die eingegebenen Informationen richten sich danach, welchen Profiler Sie verwenden. Beispiele und Anweisungen finden Sie in Chapter 3, Debugging Applications, im Oracle GlassFish Server Application Development Guide.

  6. Klicken Sie auf "Speichern".

  7. Starten Sie den Server neu.
