Realms

GlassFish Server verfügt über drei vorkonfigurierte Realms: file, certificate und admin-realm. Darüber hinaus können die Realms ldap, JDBC, solaris sowie benutzerdefinierte Realms erstellt werden. Im Allgemeinen ist auf jedem Server eine Realm jeden Typs vorhanden, GlassFish Server verfügt jedoch über zwei Datei-Realms: file und admin-realm. Dies sind zwei Realms desselben Typs, die für unterschiedliche Zwecke verwendet werden. Darüber hinaus kann für jeden virtuellen Server auf dem System eine separate Zertifikatdatenbank bereitgestellt werden.

Auf der Seite "Realms" können Sie Realms konfigurieren.

Für jede Realm werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Realm.

Klassenname

Der Klassenname der Realm

Die Tabelle "Realms" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Realm.

Löschen

Schaltfläche zum Löschen von ausgewählten Realms.

Verwandte Aufgaben

  • So wird eine Realm erstellt

  • So wird eine Realm bearbeitet

  • So wird eine Realm gelöscht

  • So werden benutzerdefinierte Realms erstellt

Zugehörige asadmin-Befehle

  • create-auth-realm(1)

  • delete-auth-realm(1)

  • list-auth-realms(1)
