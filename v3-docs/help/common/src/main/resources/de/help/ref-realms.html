Bereiche

GlassFish-Server verfügt über drei vorkonfigurierte Bereiche: file, certificate und admin-realm. Darüber hinaus können die Bereiche ldap, JDBC, solaris sowie benutzerdefinierte Bereiche erstellt werden. Im Allgemeinen ist auf jedem Server ein Bereich jeden Typs vorhanden, GlassFish-Server verfügt jedoch über zwei Dateibereiche: file and admin-realm. Dies sind zwei Bereiche desselben Typs, die für unterschiedliche Zwecke verwendet werden. Darüber hinaus kann für jeden virtuellen Server auf dem System eine separate Zertifikatdatenbank bereitgestellt werden.

Auf der Seite "Bereiche" können Sie Bereiche konfigurieren.

Für jeden Bereich werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des Bereichs.

Klassenname

Der Klassenname des Bereichs

Die Tabelle "Bereiche" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Bereichs.

Löschen

Schaltfläche zum Löschen eines oder mehrerer Bereiche.

Verwandte Aufgaben

  • So erstellen Sie einen Bereich

  • So bearbeiten Sie einen Bereich

  • So löschen Sie einen Bereich

  • So erstellen Sie einen benutzerdefinierten Bereich

Verwandte asadmin-Befehle

  • create-auth-realm(1)

  • delete-auth-realm(1)

  • list-auth-realms(1)
