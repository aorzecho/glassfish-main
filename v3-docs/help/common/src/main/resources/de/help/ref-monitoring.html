Überwachung

Verwenden Sie die Seite "Überwachung", um die Überwachung zu konfigurieren und Überwachungsdaten für geclusterte und nicht geclusterte Serverinstanzen anzuzeigen.

Für jede Serverinstanz werden die folgenden Informationen zur Verfügung gestellt.

Instanzname

Der Name der Serverinstanz. Klicken Sie auf den Instanznamen, um zu der Seite "Allgemeine Informationen" für die Instanz zu gehen.

Clustername

Der Name des Clusters, zu dem die Serverinstanz gehört, wenn es sich um eine geclusterte Instanz handelt. Klicken Sie auf den Clusternamen, um zu der Seite "Allgemeine Informationen" für das Cluster zu gehen.

Aktion

Links, mit denen Sie die folgenden Vorgänge auf den Serverinstanzen ausführen können.

Überwachung konfigurieren

Zeigt die Seite "Überwachungsservice" für die Konfiguration des Clusters oder der Standalone-Instanz an.

Überwachungsdaten anzeigen

Links, mit denen Sie die folgenden Typen von Überwachungsdaten für die Serverinstanzen anzeigen können.

Anwendung

Zeigt die Seite "Anwendungsüberwachung" für die Serverinstanz an.

Server

Zeigt die Seite "Serverüberwachung" für die Serverinstanz an.

Ressourcen

Zeigt die Seite "Ressourcenüberwachung" für die Serverinstanz an.

Verwandte Aufgaben

  • So wird der Überwachungsservice konfiguriert

  • So werden Anwendungsüberwachungsdaten angezeigt

  • So werden Serverüberwachungsdaten angezeigt

  • So werden Ressourcenüberwachungsdaten angezeigt

Zugehörige asadmin-Befehle

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
