JVM-Pfadeinstellungen

Konfigurieren Sie auf der Seite "JVM-Pfadeinstellungen" den Pfad für die native Bibliothek. Der Pfad zur nativen Bibliothek ist eine Verkettung folgender Pfade:

  • Das Präfix für den Pfad der nativen Bibliothek

  • Der Serverpfad für die nativen Bibliotheken (d. h. as-install/lib)

  • Die 64-Bit-Verzeichnisse der nativen Bibliotheken (falls zutreffend)

  • Der Wert der Variablen java.library.path für die JVM, die zum Starten von GlassFish-Server verwendet wird(je nach Betriebssystem verschieden)

  • Alle Pfade, die auf der Seite JVM-Profiler-Einstellungen angegeben sind, wenn der Profiler aktiviert ist

  • Das Suffix für den Pfad der nativen Bibliothek

Andere Einstellungen werden für Domains, die von einer älteren Version von GlassFish-Server aktualisiert wurden, als schreibgeschützte Werte angezeigt. Sie werden in der Version v3 nicht unterstützt.

Die Seite "JVM-Pfadeinstellungen" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Environment Classpath (Umgebungsklassenpfad)

Wenn Sie das Kontrollkästchen Ignore (Ignorieren) aktivieren, wird die Umgebungsvariable CLASSPATH ignoriert. Die Umgebungsvariable CLASSPATH ist für grundlegende Lernprogramme in der Programmierung geeignet, wird für Unternehmensumgebungen jedoch nicht empfohlen. Dies ist ein schreibgeschütztes Feld.

System Classpath (Systemklassenpfad)

Der Pfad zu den Klassen auf Ihrem Computer. Dies ist ein schreibgeschütztes Feld.

Classpath Prefix (Klassenpfadpräfix)

Vollständige Pfade zu den JAR-Dateien, die dem Klassenpfad des Servers - jeweils einer pro Zeile - vorangestellt werden müssen. Dies ist ein schreibgeschütztes Feld.

Classpath Suffix (Klassenpfadsuffix)

Vollständige Pfade zu den JAR-Dateien, die an den Klassenpfad - jeweils einer pro Zeile - angehängt werden müssen. Dies ist ein schreibgeschütztes Feld.

Native Library Path Prefix (Präfix für Pfad zur nativen Bibliothek)

Vollständige Pfade zu den JAR-Dateien, die dem Pfad zur nativen Bibliothek - jeweils einer pro Zeile - vorangestellt werden müssen.

Native Library Path Suffix (Suffix für Pfad zur nativen Bibliothek)

Vollständige Pfade zu den JAR-Dateien, die an den Pfad zur nativen Bibliothek - jeweils einer pro Zeile - angehängt werden müssen.
