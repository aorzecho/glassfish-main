Überwachungsmodule

Auf der Seite "Überwachungsmodule" können Sie Überwachungsmodule konfigurieren.

Mit diesen Modulen wird überwacht, wie GlassFish-Server wichtige Ereignisse, z. B. Fehler oder Sicherheitsverletzungen, aufzeichnet. GlassFish-Server zeichnet sämtliche Authentifizierungsereignisse in Protokolldateien auf. Ein vollständiges Zugriffsprotokoll umfasst eine Reihe von GlassFish-Server-Zugriffsereignissen.

Folgende Informationen werden für jedes Überwachungsmodul zur Verfügung gestellt.

Name

Der Name des Überwachungsmoduls.

Klassenname

Der vollqualifizierte Name der Klasse, welche dieses Modul implementiert. Der Klassenname für das standardmäßige Überwachungsmodul lautet com.sun.enterprise.security.Audit.

Die Tabelle "Module" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Überwachungsmoduls.

Löschen

Schaltfläche zum Löschen eines oder mehrerer Überwachungsmodule.

Verwandte asadmin-Befehle

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
