So werden Netzwerkproxyeinstellungen für den Zugriff auf Netzwerk-Repositorys konfiguriert

Netzwerk-Proxy-Einstellungen legen fest, wie GlassFish Server eine Verbindung mit dem Internet herstellt. Damit Updatetool über das Internet auf Netzwerk-Repositorys zugreifen kann, müssen Sie Ihre Netzwerkproxyeinstellungen konfigurieren.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Updatetool" aus.

    Die Seite "Verfügbare Add-ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-ons" auf die Registerkarte "Konfiguration".

    Die Seite Updatetool-Konfiguration wird angezeigt.

  3. Geben Sie auf der Seite "Updatetool-Konfiguration" an, wie der Rechner, auf dem GlassFish Server ausgeführt wird, eine Verbindung zum Internet herstellt.

    • Stellt der Rechner eine Direktverbindung mit dem Internet her, so wählen Sie die Option "Direktverbindung".

    • Stellt der Rechner eine Internetverbindung über einen Proxy her, wählen Sie die Option "Über einen Proxy".

  4. Wenn der Computer, auf dem GlassFish Server ausgeführt wird, über einen Proxy eine Verbindung zum Internet herstellt, geben Sie den Hostnamen und die Port-Nummer des Proxy an.

    Wenn der Computer, auf dem GlassFish Server ausgeführt wird, eine direkte Verbindung zum Internet herstellt, lassen Sie diesen Schritt aus.

    Möglicherweise wird ein Webbrowser auf dem Computer ausgeführt, auf dem GlassFish Server ausgeführt wird. In diesem Fall können Sie den Hostnamen und die Port-Nummer des Proxy über den Webbrowser beziehen. Andernfalls erhalten Sie die benötigten Informationen von Ihrem Systemadministrator.

  5. Klicken Sie auf "Speichern".
