So konfigurieren Sie den Überwachungsdienst

  1. Erweitern Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Cluster-Konfiguration, für die Sie die Überwachung konfigurieren möchten.
  2. Erweitern Sie den Knoten "Überwachung".

    Die Seite "Überwachungsdienst" wird geöffnet.

  3. Aktivieren Sie das Kontrollkästchen "Überwachungsdienst", um die Überwachung für GlassFish-Server zu aktivieren, oder deaktivieren Sie das Kontrollkästchen, um die Überwachung zu deaktivieren.

    Diese Option ist standardmäßig aktiviert.

  4. Aktivieren Sie das Kontrollkästchen "Überwachungs-MBeans", um alle für die Überwachung benötigten MBeans bereitzustellen; deaktivieren Sie das Kontrollkästchen, wenn die MBeans nicht bereitgestellt werden sollen.

    Diese Option ist standardmäßig aktiviert.

    Wenn das Kontrollkästchen nicht aktiviert ist, können Sie keine Überwachungsdaten in der Administration Console anzeigen, auch dann nicht, wenn das Kontrollkästchen "Überwachungsdienst" aktiviert ist. Sie müssen die Überwachungsdaten stattdessen über die Befehlszeile aufrufen. Wenn Sie die Auswahl des Kontrollkästchens aufheben, wird in GlassFish-Server weniger Speicher belegt.

  5. Aktivieren Sie das Kontrollkäschen "DTrace", um die Verwendung der Solaris DTrace-Funktion zu ermöglichen.

    Zur Verwendung der Solaris DTrace-Funktion ist außerdem Oracle Enterprise Manager DTrace-Überwachung erforderlich. Diese Option ist standardmäßig deaktiviert.

  6. Führen Sie in der Tabelle eine der folgenden Aktionen durch.
    • Wählen Sie in der Spalte "Überwachungsebene" LOW oder HIGH aus der Dropdown-Liste für mindestens eine Komponente in der Spalte Komponente aus, um die Überwachung für diese Komponenten zu aktivieren.
    • Aktivieren Sie die Kontrollkästchen neben mindestens einer Komponente, wählen Sie anschließend LOW oder HIGH aus der Dropdown-Liste oben in der Tabelle aus und klicken Sie auf die Schaltfläche "Ebene ändern", um die Überwachungsebene für alle diese Komponenten auf diesen Wert einzustellen.

    In dieser Version besteht kein Unterschied zwischen den Überwachungsebenen der Optionen "NIEDRIG" und "HOCH". Um die Überwachung für mindestens eine Komponente zu deaktivieren, wählen Sie OFF aus. Der Standardwert ist "AUS".

    Sie müssen eine Anwendung, eine Komponente oder einen Dienst bereitstellen, bevor Sie die Überwachung hierfür konfigurieren können.

  7. Klicken Sie auf "Speichern".
