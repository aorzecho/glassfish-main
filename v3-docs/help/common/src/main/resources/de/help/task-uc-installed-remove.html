So werden installierte Komponenten angezeigt

Die Seite "Installierte Komponenten" liefert Informationen zu Komponenten des Installationsimages, die installiert sind.

Zum Deinstallieren einer Komponente müssen folgende Elemente entfernt werden:

  • Alle Dateien und Verzeichnisse aus der ursprünglichen Installation der Komponente

  • Alle Dateien und Verzeichnisse aus Updates der Komponente nach der ursprünglichen Installation

  1. Wählen Sie in der Navigationsstruktur den Knoten "Updatetool" aus.

    Die Seite "Verfügbare Add-ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-ons" auf die Registerkarte "Installierte Komponenten".

    Die Seite "Installierte Komponenten" wird angezeigt.

Nächste Schritte

Um installierte Komponenten zu deinstallieren, stoppen Sie die GlassFish Server-Domain zunächst. Verwenden Sie dann einen der folgenden Befehle:

  • as-install-parent/bin/updatetool, das das grafische Updatetoolutility startet

  • as-install-parent/bin/pkg, eine Befehlszeilenversion des Updatetools

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Chapter 11, Extending and Updating GlassFish Server, im Oracle GlassFish Server Administration Guide. Das Aktualisierungstool verfügt über eine eigene Onlinehilfe.

Starten Sie den Server neu, wenn die Deinstallation der Komponenten abgeschlossen ist.
