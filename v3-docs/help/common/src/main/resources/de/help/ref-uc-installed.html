Installierte Komponenten

Die Seite Installed Components (Installierte Komponenten) liefert Informationen zu Komponenten des Installation-Image, die installiert sind.

Um installierte Komponenten zu deinstallieren, halten Sie die GlassFish-Server-Domain zunächst an. Verwenden Sie dann einen der folgenden Befehle:

  • glassfish3/bin/updatetool (zum Starten des grafischen Aktualisierungstools)

  • glassfish3/bin/pkg (Befehlszeilenversion des Aktualisierungstools)

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Kapitel 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Das Aktualisierungstool verfügt über eine eigene Online-Hilfe.

Starten Sie den Server neu, wenn die Deinstallation der Komponenten abgeschlossen ist.

Die Seite Installed Components (Installierte Komponenten) umfasst die folgenden Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Installed Components

In dieser Tabelle werden für die einzelnen installierten Komponenten die folgenden Informationen angezeigt.

Komponente

Der Name der Komponente.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, zum Beispiel Anwendungsserver, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Version

Die Versionsnummer des Pakets, in dem die Komponente geliefert wird.

Installationsgröße

Der Festplattenspeicherplatz, den die Installation der Komponente beansprucht.

Quelle

Der Name des Netzwerk-Repository, aus dem die Komponente abgerufen wurde.
