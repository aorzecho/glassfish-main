Installierte Komponenten

Die Seite "Installierte Komponenten" liefert Informationen zu Komponenten des Installationsimages, die installiert sind.

Um installierte Komponenten zu deinstallieren, stoppen Sie die GlassFish Server-Domain zunächst. Verwenden Sie dann einen der folgenden Befehle:

  • glassfish3/bin/updatetool, zum Starten des grafischen Updatetoolutilitys

  • glassfish3/bin/pkg, eine Befehlszeilenversion des Updatetools

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Chapter 11, Extending and Updating GlassFish Server, im Oracle GlassFish Server Administration Guide. Das Aktualisierungstool verfügt über eine eigene Onlinehilfe.

Starten Sie den Server neu, wenn die Deinstallation der Komponenten abgeschlossen ist.

Die Seite "Installierte Komponenten" umfasst die folgenden Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Installed Components

In dieser Tabelle werden für die einzelnen installierten Komponenten die folgenden Informationen angezeigt.

Komponente

Der Name der Komponente.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, zum Beispiel Application Server, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Version

Die Versionsnummer des Packages, in dem die Komponente geliefert wird.

Installationsgröße

Der Festplattenspeicherplatz, den die Installation der Komponente beansprucht.

Quelle

Der Name desNetzwerk-Repository, aus dem die Komponente abgerufen wurde.

Verwandte Aufgaben
