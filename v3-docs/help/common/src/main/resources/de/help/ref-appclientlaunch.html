Anwendungsclient starten

Verwenden Sie die Seite "Anwendungs-Client starten", um einen bereitgestellten Anwendungs-Client zu starten. Der Client muss ein Web-Client sein, der mit Java Web Start ausgeführt werden kann.

Die Seite "Anwendungs-Client starten" enthält die folgenden Optionen.

Anwendung

Name der Anwendung, die den Anwendungs-Client enthält.

Der Name kann eine optionale Versionskennung enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide .

Modul

Name des Moduls, das den Anwendungs-Client enthält.

Verbindungen

URL, mit der der Client bereitgestellt wird.

Argumente

Die Argumente, die an die Anwendung übergeben werden sollen.

  • So zeigen Sie eine Liste eingesetzter Anwendungen an

  • So stellen Sie eine Webanwendung bereit

  • So stellen Sie eine Unternehmensanwendung bereit

  • So stellen Sie einen Anwendungs-Client bereit

  • So stellen Sie ein Verbindungsmodul bereit

  • So stellen Sie ein EJB Jar bereit

  • So bearbeiten Sie eine bereitgestellte Anwendung

  • So zeigen Sie die Beschreibung einer Anwendung an

  • So zeigen Sie Endpunkt-Informationen für einen Webdienst an

  • So starten Sie eine Anwendung

  • Eine bereitgestellte Anwendung erneut bereitstellen

  • So laden Sie eine bereitgestellte Anwendung erneut
