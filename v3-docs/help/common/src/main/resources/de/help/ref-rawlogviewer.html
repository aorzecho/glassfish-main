Raw Log Viewer

Verwenden Sie die Seite "Raw Log Viewer", um die Raw-Logdateien für den Admin-Server (namens server) anzuzeigen.

Die Seite "Raw Log Viewer" umfasst die folgenden Optionen.

Instanz

Die Serverinstanz oder das Cluster, für die/den Datensätze angezeigt werden. Nur server kann gewählt werden.

Für jede Meldung zeigt der Raw Log Viewer die folgenden Informationen an.

Zeitstempel

Datum und Uhrzeit der Erstellung der Logmeldung.

Datensatznummer

Die Nummer der Meldung im Serverlog.

Logebene

Die Logebene der Nachricht.

Version

Die Version von GlassFish Server.

Logger

Das Logger-Modul, das die Logmeldung generiert hat.

Name-/Wertpaare

Der Name und die ID des Threads der Lognachricht.

Nachricht

Der kurze Text der Logmeldung. Klicken Sie auf den Link (Details), um eine formatierte Version einer Nachricht anzuzeigen.

Verwandte Aufgaben

  • So werden Raw-Logdateien für den DAS angezeigt

  • So werden allgemeine Logging-Einstellungen konfiguriert

  • So werden Logebenen konfiguriert

  • So werden Logdateien für den DAS angezeigt

Zugehörige asadmin-Befehle

  • set-log-level

  • rotate-log(1)
