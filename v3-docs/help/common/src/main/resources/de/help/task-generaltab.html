So werden allgemeine Informationen für den DAS angezeigt

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Führen Sie auf der Seite "Allgemeine Informationen" einen der folgenden Schritte aus.

    • Prüfen Sie die schreibgeschützte Anzeige "Status", um zu sehen, ob der Server gestartet ist.

    • Prüfen Sie die schreibgeschützte Anzeige "Produktive Zeit", um zu sehen, wie viele Stunden und Minuten der Server kontinuierlich ausgeführt wurde.

    • Klicken Sie auf die Schaltfläche Stop, um GlassFish Server anzuhalten.

    • Klicken Sie auf die Schaltfläche "Neustart", um GlassFish Server anzuhalten und neu zu starten.

    • Klicken Sie auf die Schaltfläche "Logdateien anzeigen", um die Logdateien für GlassFish Server-Instanzen oder -Cluster anzuzeigen.

      Weitere Einzelheiten finden Sie in So werden Logdateien für den DAS angezeigt.

    • Klicken Sie auf die Schaltfläche "Raw-Log anzeigen", um die Raw-Logdateien für den DAS anzuzeigen.

      Weitere Einzelheiten finden Sie in So werden Raw-Logdateien für den DAS angezeigt.

    • Klicken Sie auf "JVM-Bericht", um ein separates Fenster zu öffnen, das es Ihnen ermöglicht, Berichte auf der Java Virtual Machine anzuzeigen, u.a. einen zusammenfassenden Bericht, Speichermanagement-Bericht, Class Loading-Bericht und einen aktuellen Thread-Dump.
