So zeigen Sie allgemeine Informationen für den DAS an

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Führen Sie auf der Seite "Allgemeine Informationen" einen der folgenden Schritte aus.
    • Klicken Sie auf die Schaltfläche Stop, um GlassFish-Server anzuhalten.
    • Klicken Sie auf die Schaltfläche "Neustart", um GlassFish-Server anzuhalten und neu zu starten.
    • Klicken Sie auf die Schaltfläche "Protokolldateien anzeigen", um die Protokolldateien für GlassFish Server-Instanzen oder Cluster anzuzeigen.

      Einzelheiten finden Sie unter So zeigen Sie Protokolldateien für den DAS an.

    • Klicken Sie auf JVM-Bericht, um ein separates Fenster zu öffnen, das es Ihnen ermöglicht, Berichte auf der Java Virtual Machine anzuzeigen, u. a. einen zusammenfassenden Bericht, Speichermanagement-Bericht, Klassenladebericht und einen aktuellen Thread-Pool.
