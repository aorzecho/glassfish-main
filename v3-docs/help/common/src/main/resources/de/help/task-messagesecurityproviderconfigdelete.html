So löschen Sie eine Konfiguration des Meldungssicherheitsanbieters

Bevor Sie beginnen

Es ist nicht möglich, Meldungssicherheitsanbieter für die standardmäßige Konfiguration server-config zu erstellen oder zu löschen. Die einzigen Meldungssicherheitsanbieter, die nicht für die standardmäßige Konfiguration server-config unterstützt werden, sind SOAP und HttpServlet. Da diese beiden Meldungssicherheitsanbieter bereits standardmäßig erstellt sind, wenn GlassFish-Server installiert wird, wird die Schaltfläche "Löschen" nicht auf der Seite "Meldungssicherheitskonfigurationen" für die standardmäßige Konfiguration server-config angezeigt.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Konfiguration".
  2. Erweitern Sie den Knoten "Sicherheit".
  3. Wählen Sie den Knoten "Meldungssicherheit".

    Die Seite "Nachrichtensicherheitskonfigurationen" wird geöffnet.

  4. Wählen Sie auf der Seite "Nachrichtensicherheitskonfigurationen" SOAP aus.

    Die Seite "Meldungssicherheitskonfiguration bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Meldungssicherheitskonfiguration bearbeiten" auf den Register "Anbieter".

    Die Seite "Anbieterkonfiguration bearbeiten" wird geöffnet.

  6. Aktivieren Sie auf der Seite "Nachrichtensicherheitskonfigurationen" das Kontrollkästchen links von der Anbieterkonfiguration, die Sie löschen.
  7. Klicken Sie auf "Löschen".
