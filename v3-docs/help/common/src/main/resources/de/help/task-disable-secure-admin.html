So deaktivieren Sie die sichere Administration

  1. Wählen Sie in der Navigationsstruktur den Serverknoten (Admin-Serverknoten).

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Schaltfläche "Sichere Administration".

    Die Seite "Sichere Administration" wird geöffnet.

  3. Klicken Sie auf "Sichere Administration deaktivieren".

    Daraufhin wird mit einer Meldung angegeben, dass die Deaktivierung der sicheren Administration den DAS sofort neu startet.

  4. Klicken Sie auf "OK".

    Die sichere Administration wird deaktiviert, und die Seite "Serverneustart" wird geöffnet.
