JVM-Profiler-Einstellungen

Ein Profiler generiert Daten, die zur Analyse der Performance sowie zur Ermittlung von potenziellen Engpässen verwendet werden können.

Geben Sie die Profilereinstellungen an. Die eingegebenen Informationen richten sich danach, welchen Profiler Sie verwenden. Beispiele und Anweisungen finden Sie in Chapter 3, Debugging Applications, im Oracle GlassFish Server Application Development Guide.

Verwandte Aufgaben

  • So werden allgemeine JVM-Einstellungen konfiguriert

  • So werden JVM-Pfadeinstellungen konfiguriert

  • So werden JVM-Optionen konfiguriert

  • So wird der Sicherheitsmanager deaktiviert

  • So werden JVM-Profiler-Einstellungen konfiguriert

Zugehörige asadmin-Befehle

  • create-profiler(1)

  • delete-profiler(1)
