查看组件的可用更新

“可用更新”页面提供了有关已安装的安装映像组件的更新版本的信息。

  1. 在导航树中，选择“更新工具”节点。

    此时将打开“可用的附加软件”页面。

  2. 在“可用的附加软件”页面上，单击“可用更新”选项卡。

    此时将打开“可用更新”页面。如果没有可用的更新，则此表为空。

接下来的操作

要安装可用更新，请首先停止 GlassFish Server 域。然后使用以下命令之一：

  • as-install-parent/bin/updatetool，可以启动更新工具图形实用程序

  • as-install-parent/bin/pkg，更新工具的命令行版本

有关使用 pkg 命令的信息，请参见 《Oracle GlassFish Server 3.1 Administration Guide》中的第 11  章 "Extending and Updating GlassFish Server"。更新工具实用程序有自己的联机帮助。

在安装完更新后，请重新启动服务器。
