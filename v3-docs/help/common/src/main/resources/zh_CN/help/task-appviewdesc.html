查看应用程序描述符

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 在“应用程序”页面上，单击要查看其描述符的应用程序的名称。

    名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号(:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

    此时将打开“编辑应用程序”页面。

  3. 在“编辑应用程序”页面上，单击“描述符”选项卡。

    此时将打开“模块描述符”页面。

  4. 在“部署描述符”表中，单击要查看的 XML 文件的名称。

    此时将打开“部署描述符”页面，其中显示该 XML 文件。
