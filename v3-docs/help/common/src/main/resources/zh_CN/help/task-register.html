注册 Oracle GlassFish Server

作为 Oracle GlassFish Server 的注册用户，您可以享受以下好处：

  • 修补程序信息和错误更新

  • 截屏视频 (screencast) 和教程

  • 新闻和事件

  • 支持和培训服务

要注册 Oracle GlassFish Server 安装，必须拥有 Sun 联机帐户。如果没有 Sun 联机帐户，则在注册过程中 GlassFish Server 允许您创建一个帐户。

注 - 如果在注册 GlassFish Server 时创建一个Sun 联机帐户，则系统将仅提示您提供必需的信息。注册 GlassFish Server 后，可以通过更新 Sun 联机帐户来提供其他可选的信息。

开始之前

如果运行 GlassFish Server 的计算机通过代理连接到 Internet，请获取代理的主机名和端口号。如果运行 GlassFish Server 的计算机上正在运行 Web 浏览器，则可以通过 Web 浏览器获取这些信息。否则，请您的系统管理员提供这些信息。

  1. 在导航树中，选择“注册”节点。

    此时将打开“产品注册”页面。

  2. 在“产品注册”页面中提供必要的信息。
    • 如果您已经拥有 Sun 联机帐户，请按照以下步骤提供必需的信息：
      1. 选择“我已经有了 Sun 联机帐户或 Sun Developer Network (SDN)”选项。
      2. 在“用户名”字段中，键入注册 Sun 联机帐户时指定的用户名。
      3. 在“密码”字段中，键入注册 Sun 联机帐户时指定的密码。
    • 如果没有 Sun 联机帐户，请按照以下步骤提供必需的信息：
      1. 选择“我没有 Sun 联机帐户”选项。
      2. 在“电子邮件地址”字段中，键入您的电子邮件地址。

        电子邮件地址将是您的 Sun 联机帐户用户名。

      3. 在“密码”字段中，键入 Sun 联机帐户使用的密码。

        密码区分大小写，长度必须为 6 至 10 个字符。

      4. 在“重新键入密码”字段中，重新键入在上一步骤中键入的密码。
      5. 在“名字”字段中，键入您的名字。
      6. 在“姓”字段中，键入您的姓。
      7. 在“公司”字段中，键入您所在公司的名称（若适用）。
      8. 从下拉式菜单中选择您居住的国家或地区。
  3. 如果运行 GlassFish Server 的计算机通过代理连接到 Internet，请提供代理的主机名和端口号。

    如果运行 GlassFish Server 的计算机直接连接到 Internet，则忽略此步骤。

    运行 GlassFish Server 的计算机上可能正在运行 Web 浏览器。在这种情况下，可以通过 Web 浏览器获取代理的主机名和端口号。否则，请您的系统管理员提供这些信息。

  4. 单击“注册”。

    如果“支持”页面打开，则可确认注册已经完成。
