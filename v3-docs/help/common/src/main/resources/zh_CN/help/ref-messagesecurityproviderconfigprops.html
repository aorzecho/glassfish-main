特定于消息安全性提供者配置的属性

以下属性可用于消息安全性提供者配置。

server.config

包含服务器配置信息的 XML 文件的目录和文件名。默认值为 domain-dir/config/wss-server-config-1.0.xml。

debug

如果为 true，则将服务器提供者调试消息转储到服务器日志。默认值为 false。

dynamic.username.password

如果为 true，则通知提供者运行时从每个请求的 CallbackHandler 收集用户名和密码。如果为 false，则在模块初始化过程中收集一次 wsse:UsernameToken(s) 的用户名和密码。此属性仅适用于 ClientAuthModule。默认值为 false。

encryption.key.alias

指定提供者使用的加密密钥。可通过密钥的 keystore 别名来指定密钥。默认值为 s1as。

signature.key.alias

指定提供者使用的签名密钥。可通过密钥的 keystore 别名来指定密钥。默认值为 s1as。
