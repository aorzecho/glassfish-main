域日志

使用“域日志”页可为 GlassFish Server 收集和下载日志文件。

“域日志”页包含以下选项：您可以选择要为其收集日志的一个集群或一个服务器实例。

集群

集群 (如果存在任何集群) 的下拉列表。

实例

服务器实例的下拉列表。

收集日志

用于将日志收集到 log-files-name.zip 文件中的按钮，其中 name 为集群或实例的名称。

log-files-name.zip 文件是使用浏览器的下载机制下载的。
