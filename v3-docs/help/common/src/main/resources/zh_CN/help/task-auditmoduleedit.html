编辑审计模块

默认情况下，未打开审计模块。有关如何激活审计模块的详细信息，请参阅启用或禁用审计日志记录。

  1. 在导航树中，展开“配置”节点。

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. 在“配置”节点下，展开“安全性”节点。

  4. 在“安全性”节点下，选择“审计模块”节点。

    此时将打开“审计模块”页。

  5. 在“审计模块”页上，单击要编辑的审计模块的名称。

    将打开“编辑审计模块”页。

  6. 在“编辑审计模块”页的“类名”字段中，键入新的值以修改类名。

  7. 在“其他属性”部分中，指定其他属性。

    有关可用于审计模块的其他属性的说明，请参阅特定于审计模块的属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  8. 单击“保存”。
