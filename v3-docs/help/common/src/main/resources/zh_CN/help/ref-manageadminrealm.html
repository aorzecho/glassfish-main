管理员用户

使用“管理员用户”页可以管理 admin-realm 用户。admin-realm 是文件领域。文件领域中的用户和组在密钥文件中列出，该文件的位置由文件属性指定。

初次安装 GlassFish Server 之后，唯一的用户是在安装过程中输入的管理员用户。默认情况下，此用户属于 admin-realm 领域中的 asadmin 组，该组可以授予修改 GlassFish Server 的权限。指定给该组的任何用户都将具有管理员权限，也就是说，这些用户具有访问 asadmin 工具和管理控制台的权限。

“管理员用户”页显示所选领域的用户列表。对于每个用户，将提供以下信息：

用户 ID

用户名。

组列表

用户所属的组。

“管理员用户”表还包含以下选项：

新建

用于创建新用户的按钮。

删除

用于删除一个或多个所选用户的按钮。

相关任务

  • 向管理领域添加用户

  • 编辑管理领域用户信息

  • 从管理领域中删除用户
