编缉密码别名

使用“编缉密码别名”页面可更改与现有密码别名关联的密码。

“编辑密码别名”页面包含以下选项：

别名名称

别名的名称。此选项不可编辑。

密码

密码别名以加密形式存储的密码。

确认密码

密码别名以加密形式存储的密码。系统将比较“密码”和“确认密码”值，确保输入的密码没有键入错误。

等效的 asadmin 子命令

  • update-password-alias(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • list-password-aliases(1)
