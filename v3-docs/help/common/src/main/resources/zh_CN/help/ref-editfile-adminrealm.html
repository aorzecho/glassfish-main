特定于 FileRealm 类的属性

文件领域需要以下属性：

JAAS 上下文

JAAS (Java 验证和授权服务) 上下文 (用于此领域的登录模块标识符)。唯一的有效值是 fileRealm。

密钥文件

文件 (服务器将在其中存储此领域的所有用户、组和口令信息) 的完整路径和名称。默认值为 domain-dir/config/keyfile (用于文件领域) 和 domain-dir/config/admin-keyfile (用于 admin-realm 领域)。

文件领域的密钥文件最初为空，因此在使用文件领域之前，必须在“文件用户”页上添加用户。

admin-realm 领域的密钥文件最初包含管理员用户名、加密格式的管理员口令，以及此用户所属的组 (默认情况下为 asadmin)。有关将用户添加到 admin-realm 的详细信息，请参阅授予对管理工具的访问权限。

注:

admin-realm 的 asadmin 组中的用户已被授权，可以使用管理控制台和 asadmin 工具。只能将具有服务器管理权限的用户添加到该组中。

以下可选属性可用于文件领域。

分配组

以逗号分隔的组名称列表。出具有效证书的所有客户机都分配到这些组中，例如 employee、manager (这些是用户组的名称)。
