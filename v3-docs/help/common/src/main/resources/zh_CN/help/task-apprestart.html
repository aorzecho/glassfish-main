重新装入已部署的应用程序

重新装入应用程序只会重新启动该应用程序，而不会重新部署该应用程序，同时会保留该应用程序部署时的设置和其他选项。

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 在“已部署应用程序”表中，单击要重新部署的应用程序行中的“重新装入”。

    应用程序名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

    此时将再次打开“应用程序”页面，并显示消息指明重新装入是否成功。
