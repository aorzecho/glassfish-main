产品注册

要注册 Oracle GlassFish Server 安装，必须拥有 Sun 联机帐户。如果没有 Sun 联机帐户，则在注册过程中 GlassFish Server 允许您创建一个帐户。

“产品注册”页面包含以下选项：

我已经有了 Sun 联机帐户或 Sun Developer Network (SDN) 帐户

如果您已经有了帐户，则选择该选项。如果选择此选项，将会看到以下字段：

用户名

您在注册 Sun 联机帐户时指定的用户名。

Password

您在注册 Sun 联机帐户时指定的密码。

代理主机

运行 GlassFish Server 的计算机的代理服务器的主机名。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

代理端口

运行 GlassFish Server 的计算机的代理服务器的端口号。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

我没有 Sun 联机帐户。我要注册。

如果您没有帐户，并且要注册 GlassFish Server 安装，请选择该选项。如果选择此选项，将会看到以下字段：

电子邮件地址

您的电子邮件地址，它将是您的 Sun 联机帐户用户名。

Password

用于您 Sun 联机帐户的密码。密码区分大小写，长度必须为 6 至 10 个字符。

重新键入密码

您必须重新键入在“密码”字段中指定的密码。

名字

您的名字。

姓氏

您的姓氏。

公司

您所在的公司（若适用的话）。

国家/地区

您居住的国家或地区。

代理主机

运行 GlassFish Server 的计算机的代理服务器的主机名。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

代理端口

运行 GlassFish Server 的计算机的代理服务器的端口号。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。
