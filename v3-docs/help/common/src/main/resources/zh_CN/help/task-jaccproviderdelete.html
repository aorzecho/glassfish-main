删除 JACC 提供者

  1. 在导航树中的“配置”节点下，展开要为其创建 JACC 提供者的 GlassFish 服务器 实例或群集。

    要创建可用于所有实例和群集的提供者，请选择 default-config 节点。

  2. 在展开的实例或群集配置节点下，展开“安全性”节点。
  3. 在“安全性”节点下，选择“JACC 提供者”节点。

    此时将打开“JACC 提供者”页面。

  4. 选中要删除的 JACC 提供者左侧的复选框。
  5. 单击“删除”。
