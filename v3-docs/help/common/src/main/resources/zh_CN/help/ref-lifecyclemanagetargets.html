管理目标

使用“管理资源目标”页面可更改可在其上启用生命周期模块的目标群集和独立服务器实例。只能在“选定目标”列中的目标上启用生命周期模块。

“管理资源目标”页面包含以下信息。

可用目标

无法启用生命周期模块的群集和实例。

选定目标

可以启用生命周期模块的群集和实例。

添加

将选定目标从“可用目标”移动到“选定目标”的按钮。

全部添加

将所有“可用目标”移动到“选定目标”的按钮。

删除

将选定目标从“选定目标”移动到“可用目标”的按钮。

全部删除

将所有“选定目标”移动到“可用目标”的按钮。
