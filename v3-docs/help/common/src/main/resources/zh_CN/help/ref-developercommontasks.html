日常任务

“日常任务”页面提供日常管理控制台任务的快捷方式。

可使用“日常任务”页面进行以下操作：

  • 查看 GlassFish Server 的"GlassFish 新闻"页面

  • 查看 GlassFish Server 的"订阅"页面

  • 查看已部署应用程序列表

  • 部署 Web 应用程序

  • 部署企业应用程序

  • 部署应用程序客户机

  • 部署连接器模块

  • 部署 EJB Jar

  • 查看安装的组件

  • 安装附加组件

  • 查看组件的可用更新

  • 创建 JDBC 连接池

“日常任务”页面还提供了某些 GlassFish Server 文档的链接。
