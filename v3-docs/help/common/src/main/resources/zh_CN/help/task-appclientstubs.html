下载应用程序客户机存根

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页。

  2. 从“已部署的应用程序”表中，单击要为其下载存根的应用程序客户机的名称。

    名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的详细信息，请参阅 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

    此时将打开“编辑应用程序”页。

  3. 在“模块和组件”表中，单击“操作”列下的“下载客户机存根”。

    使用浏览器的下载机制下载 client-stubs.zip 文件。
