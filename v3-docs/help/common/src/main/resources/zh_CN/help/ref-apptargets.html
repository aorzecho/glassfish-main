应用程序目标

使用“应用程序目标”页面可查看可在其上启用应用程序的目标群集和独立服务器实例。

“应用程序目标”页面包含以下信息。

目标名称

群集或独立服务器实例的名称。

已启用

如果应用程序在目标上已启用，则显示 true，如果应用程序已禁用，则显示 false。

已启用 LB

如果应用程序在目标上已启用负载平衡，则显示 true，如果应用程序已禁用负载平衡，则显示 false。

虚拟服务器

要将应用程序与目标上的特定虚拟服务器相关联，请为该目标选择“管理虚拟服务器”。

“目标”表还包含以下选项。

管理目标

管理应用程序目标的按钮

更多操作

以下操作的下拉式列表。

启用

在选定目标上启用应用程序的操作。

禁用

在选定目标上禁用应用程序的操作。

负载平衡器启用

在选定目标上为应用程序启用负载平衡的操作。

负载平衡器禁用

在选定目标上为应用程序禁用负载平衡的操作。
