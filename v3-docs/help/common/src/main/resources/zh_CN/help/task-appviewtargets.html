查看应用程序目标

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 选择您要查看目标的应用程序。

    应用程序名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

    此时将打开应用程序的页面。

  3. 选择“目标”选项卡。

    此时将打开“应用程序目标”页面，显示当前为应用程序指定的目标。

  4. 要在特定目标实例或群集上启用或禁用应用程序，请单击目标旁的复选框，并从“更多操作”下拉菜单中选择“启用”或“禁用”。
  5. 要在特定目标实例或群集上为目标启用负载平衡，请单击目标旁的复选框，并从“更多操作”下拉菜单中选择“启用负载平衡器”或“禁用负载平衡器”。
  6. 要将应用程序与特定虚拟服务器相关联，请为所需目标实例或群集选择“管理虚拟服务器”。

    此时将打开“虚拟服务器目标”页面。

    要添加目标虚拟服务器，请从“可用目标”列表中选择它们，然后使用“添加”按钮将它们移动到“选定目标”列表。

    默认情况下，模块或应用程序与名为 server 的默认虚拟服务器相关联。使用“全部添加”按钮可添加所有目标。使用“删除”和“全部删除”按钮可将目标从“选定目标”列表中删除。

    选择“保存”将返回到“应用程序目标”页面。

  7. 要为应用程序添加目标实例或群集，请选择“管理目标”。

    此时将打开“管理目标”页面。

    要添加目标，请从“可用目标”列表中选择它们，然后使用“添加”按钮将它们移动到“选定目标”列表。

    使用“全部添加”按钮可添加所有目标。使用“删除”和“全部删除”按钮可将目标从“选定目标”列表中删除。

    选择“保存”将返回到“应用程序目标”页面。

  8. 单击“保存”。

    应用程序此时将出现在修订后的目标列表上。
