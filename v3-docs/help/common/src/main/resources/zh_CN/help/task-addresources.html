添加在 XML 文件中配置的资源

  1. 在导航树中，选择“资源”节点。

    此时将打开“资源”页。

  2. 单击“添加资源”按钮。

    此时将打开“添加资源”页。

  3. 选择以下其中一个选项以指定 glassfish-resources.xml 文件的位置：

    要上载到服务器的 XML 文件

    键入该文件的路径 (如果该文件位于远程计算机上)。

    可以从 GlassFish Server 访问的本地 XML 文件

    键入该文件的路径 (如果该文件位于域管理服务器 (DAS) 计算机上)。

  4. 从“目标”下拉列表中，选择要在其上创建资源的集群或独立服务器实例。

  5. 单击“确定”。
