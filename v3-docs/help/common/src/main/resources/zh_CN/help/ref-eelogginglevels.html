模块日志级别

使用“模块日志级别”页面可以为各个模块配置日志记录级别。

对于每个模块，将提供以下信息。

日志程序名称

模块日志程序的名称（例如，javax.enterprise.system.tools.admin）。

日志程序级别

模块的当前日志记录级别。

“日志程序设置”表还包含以下选项。

更改级别

用于更改一个或多个选定模块的日志记录级别的按钮。

级别

一个下拉式列表，您可从该列表中选择要应用到所有选定模块的日志记录级别。

以下为可用的日志级别。这些级别以从最高到最低的顺序列出。

SEVERE

干扰正常程序执行的事件。

WARNING

警告（包括异常）。

INFO

与服务器配置或服务器状态有关的消息（不包括错误）。

CONFIG

与服务器配置有关的消息。

FINE

最小详尽程度。

FINER

中等详尽程度。

FINEST

最大详尽程度。

关闭

无日志记录消息。
