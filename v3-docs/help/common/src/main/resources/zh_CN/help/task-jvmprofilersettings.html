配置 JVM 分析器设置

分析器工具可生成用于分析性能和识别潜在瓶颈的数据。

  1. 在导航树中，选择“配置”节点。

  2. Select the configuration that you are changing.

  3. 在“配置”节点下，选择“JVM 设置”节点。

    此时将打开“JVM 常规设置”页。

  4. 在“JVM 常规设置”页上，单击“分析器”选项卡。

    此时将打开“JVM 分析器设置”页。

  5. 指定分析器设置。

    指定的信息取决于所使用的分析器产品。有关示例和说明，请参阅 Oracle GlassFish Server Application Development Guide 中的第 3 章：Debugging Applications。

  6. 单击“保存”。

  7. 重新启动服务器。
