安装的组件

“安装的组件”页面提供了有关已安装的安装映像组件的信息。

要卸载任何已安装组件，请首先停止 GlassFish Server 域。然后使用以下命令之一：

  • glassfish3/bin/updatetool，可以启动更新工具图形实用程序

  • glassfish3/bin/pkg，更新工具的命令行版本

有关使用 pkg 命令的信息，请参见《Oracle GlassFish Server 3.1 Administration Guide》中的第 11  章 "Extending and Updating GlassFish Server"。更新工具实用程序有自己的联机帮助。

在卸载完组件后，请重新启动服务器。

“安装的组件”页面包含以下选项：

安装路径

安装映像的根目录或顶层目录的文件系统路径。

已安装组件

对于每个已安装组件，此表提供以下信息。

组件

组件名称。

类别

与组件相关联的软件类别，例如 "Application Server"、“系统工具”和“数据库”。类别是在创建组件时由组件作者分配到组件中的。

版本

提供组件的软件包的版本号。

安装大小

安装组件所消耗的磁盘空间大小。

源

从其获取组件的网络系统信息库的名称。

相关任务

查看安装的组件
