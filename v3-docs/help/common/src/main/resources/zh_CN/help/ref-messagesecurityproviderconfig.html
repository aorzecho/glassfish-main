提供者配置

使用“提供者配置”页面可以为 SOAP 或 HttpServlet 验证层配置消息安全性提供者配置。

默认情况下，已为 SOAP 验证层配置了四个提供者。

对于每个提供者配置，将提供以下信息。

提供者 ID

提供者配置名称。

提供者类型

提供者的类型，可以为以下任意一种：

client

指定提供者为客户机验证提供者。

server

指定提供者为服务器验证提供者。

client-server

指定提供者既为客户机验证提供者也为服务器验证提供者。

默认提供者

如果设置为 true，则提供者将为默认提供者。

类名

提供者的 Java 实现类的名称。

“提供者配置”表还包含以下选项。

新建

用于创建新的提供者配置的按钮。

删除

用于删除一个或多个选定提供者配置的按钮。
