修改应用程序范围的资源

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页。

  2. 选择要查看其应用程序范围的资源的应用程序。

    应用程序名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的详细信息，请参阅 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

    此时将打开应用程序的页。

  3. 选择“资源”选项卡。

    此时将打开“应用程序范围的资源”页，当前为该应用程序指定的资源将在包含以下列的表中显示：

    应用程序名称

    应用程序名称。

    模块名称

    应用程序中的模块的名称。

    资源名称

    应用程序或模块的应用程序范围的资源的名称。

    资源类型

    资源类型，如 JDBC 连接池或连接器资源。

  4. 单击要修改的资源的名称。

    打开的页面取决于资源类型。例如，如果资源为 JDBC 连接池，则会打开“编辑 JDBC 连接池”页。
