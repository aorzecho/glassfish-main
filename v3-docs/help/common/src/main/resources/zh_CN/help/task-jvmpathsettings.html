配置 JVM 路径设置

  1. 在导航树中，选择“配置”节点。
  2. 在“配置”节点下，选择“JVM 设置”节点。

    此时将打开“JVM 常规设置”页面。

  3. 在“JVM 常规设置”页面上，单击“路径设置”选项卡。

    此时将打开“JVM 路径设置”页面。

  4. 在“本地库路径前缀”字段中，键入要前置到本地库路径的 JAR 文件的完整路径。

    要前置多个文件，请在每行中键入一个路径。

  5. 在“本地库路径后缀”字段中，键入要附加到本地库路径的 JAR 文件的完整路径。

    要附加多个文件，请在每行中键入一个路径。

  6. 单击“保存”。
  7. 重新启动服务器。
