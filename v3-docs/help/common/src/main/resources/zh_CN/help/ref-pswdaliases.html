密码别名

使用“密码别名”页面可创建和管理密码别名。

“密码别名”页面显示域中密码别名的列表。对于每个别名，都将显示以下信息：

名称

创建密码别名时分配给它的名称。单击名称将打开别名的编缉密码别名页面。

“密码别名”表还包含以下选项。

新建

创建密码别名的按钮。

删除

删除一个或多个选定密码别名的按钮。

等效的 asadmin 子命令

  • list-password-aliases(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • update-password-alias(1)
