应用程序

“应用程序”页面显示在 GlassFish Server 上部署的应用程序列表。您可以查看和管理已部署的应用程序，并且可以部署更多的应用程序。

对于每个应用程序，均提供以下信息：

名称

应用程序名称。

名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号(:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

已启用或状态

只要存在存在默认服务器实例 server，则当应用程序已启用时，此为复选标记；应用程序已禁用时，则为 X。

如果存在群集或其他独立服务器实例，选择“在目标上已启用”可查看已部署应用程序的目标。

引擎

应用程序使用的容器类型。容器类型可为以下任意一种：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld（Java EE 平台应用程序的 Contexts 和 Dependency Injection 的容器）

操作

部署组件以后，可对组件执行的操作的链接：“重新部署”和“重新装入”（适用于所有组件）；“启动”（适用于 Web 应用程序）。

已部署应用程序表还包含以下选项。

部署

用于部署应用程序的按钮。

取消部署

用于取消部署一个或多个选定应用程序的按钮。

启用

为用于启用一个或多个选定应用程序的按钮，只要存在默认服务器实例 server。

禁用

为用于禁用一个或多个选定应用程序的按钮，只要存在默认服务器实例 server。

过滤

按引擎过滤应用程序的下拉式列表。
