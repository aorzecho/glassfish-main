禁用安全管理器

禁用 GlassFish Server 安全管理器可能会提高某些类型应用程序的性能。即使已将安全管理器禁用，Java 平台授权和验证功能仍将起作用。可以在开发环境中禁用安全管理器，但不要在生产环境中禁用安全管理器。

  1. 在导航树中，选择“配置”节点。

  2. Select the configuration that you are changing.

  3. 在“配置”节点下，选择“JVM 设置”节点。

    此时将打开“JVM 常规设置”页。

  4. 在“JVM 常规设置”页上，单击“JVM 选项”选项卡。

    此时将打开“JVM 选项”页。

  5. 选中包含 -Djava.security.policy 选项的“值”字段左侧的复选框。

  6. 单击“删除”。

  7. 单击“保存”。

  8. 重新启动服务器。
