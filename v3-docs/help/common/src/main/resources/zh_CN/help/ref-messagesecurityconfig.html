消息安全性配置

使用“消息安全性配置”页面可以查看和修改消息安全性提供者的可用配置。

要配置消息安全性，需要基本了解安全性和 Web 服务概念。有关更多信息，请参见《Oracle GlassFish Server 3.1 Security Guide》中的第 3  章 "Administering Message Security"和《Oracle GlassFish Server 3.1 Administration Guide》中的第 6  章 "Administering Web Applications"。

GlassFish Server 使用在其 SOAP 处理层中集成的消息安全性提供者来实现消息安全性。消息安全性提供者取决于 GlassFish Server 的其他安全性工具。

对于每个配置，将提供以下信息。

验证层

执行验证的消息层。默认情况下，已提供 SOAP 配置。还可以提供 HttpServlet 配置。

默认提供者

此配置的默认服务器提供者。默认情况下，不配置任何默认服务器提供者。

默认客户机提供者

此配置的默认客户机提供者。默认情况下，不配置任何默认客户机提供者。

“消息安全性配置”表还包含以下选项。

新建

用于创建新配置的按钮。

删除

用于删除一个或多个选定配置的按钮。
