消息安全配置

使用“消息安全配置”页可以查看和修改消息安全提供方的可用配置。

要配置消息安全，需要基本了解安全性和 Web 服务概念。有关详细信息，请参阅 Oracle GlassFish Server Security Guide 中的第 3 章：Administering Message Security 和 Oracle GlassFish Server Administration Guide 中的第 6 章：Administering Web Applications。

GlassFish Server 使用在其 SOAP 处理层中集成的消息安全提供方来实现消息安全。消息安全提供方取决于 GlassFish Server 的其他安全性工具。

对于每个配置，将提供以下信息：

验证层

执行验证的消息层。默认情况下，已提供 SOAP 配置。还可以提供 HttpServlet 配置。

默认提供方

此配置的默认服务器提供方。默认情况下，不配置任何默认服务器提供方。

默认的客户机提供方

此配置的默认的客户机提供方。默认情况下，不配置任何默认的客户机提供方。

“消息安全配置”表还包含以下选项：

新建

用于创建新配置的按钮。

删除

用于删除一个或多个所选配置的按钮。
