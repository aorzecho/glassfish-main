应用程序客户机启动

使用“应用程序客户机启动”页面可启动已部署的应用程序客户机。客户机必须是可以通过 Java Web Start 运行的 Web 客户机。

“启动应用程序客户机”页面包含以下选项：

应用程序

包含应用程序客户机的应用程序名称。

名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号(:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

模块

包含应用程序客户机的模块名称。

链接

客户机部署位置的 URL。

参数

要传递给应用程序的参数。
