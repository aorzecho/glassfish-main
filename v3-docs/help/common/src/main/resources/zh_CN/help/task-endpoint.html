查看 Web 服务的端点信息

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 在“应用程序”页面上，单击您要查看其端点信息的 Web 服务的名称。

    名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号(:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

    此时将打开“编辑应用程序”页面。

  3. 在“模块和组件”表中，单击子组件的“查看端点”链接。

    此时将打开“Web 服务端点信息”页面。

  4. 如果可以看到“测试器”链接，则单击此链接可以查看一个页面，通过该页面，可以提供参数来测试 Web 服务的方法。

    仅当应用程序的部署类型不是 RI（例如为 109）时，才可以看到“测试器”链接。“测试器”页面在新浏览器选项卡或窗口中打开。

  5. 单击 "WSDL" 链接可以查看端点的 Web 服务定义语言 (Web Services Definition Language, WSDL) 文件。

    WSDL 文件在新浏览器选项卡或窗口中打开。

    请注意，单击 WSDL 链接时，某些 Web 浏览器会尝试解释 WSDL 源代码，导致显示空白页面。对于这类浏览器，请在空白页面上使用浏览器的“查看源代码”命令来查看 WSDL 源代码。

  6. 单击“应用程序名称”链接或“上一页”按钮可以返回“编辑应用程序”页面。
