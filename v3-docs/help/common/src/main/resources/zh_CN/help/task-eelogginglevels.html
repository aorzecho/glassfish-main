配置日志级别

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，单击您要为其配置日志程序设置的服务器实例或群集配置。

    此时将打开“配置”页面。

  3. 在“配置”页面上，单击“日志程序设置”。

    此时将打开选定配置目标的“日志程序设置”页面。

  4. 在“日志程序设置”页面上，单击“日志级别”选项卡。

    此时将打开“模块日志级别”页面。

  5. 在“模块日志级别”页面上，更改一个或多个模块的日志级别，如下所示：
    • 要更改单个模块的日志级别，请从要更改其日志级别的模块对应的下拉式列表中选择新的值。
    • 更改一组模块的日志级别：
      1. 选中要更改其日志级别的模块名称旁边的复选框。
      2. 从表顶部的下拉式列表中，选择一个值。
      3. 单击“更改级别”按钮。

    所选级别或更高级别的消息将显示在日志中。可以选择以下任一值（以最高级别至最低级别的顺序列出）：

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • 关闭

    有关日志级别含义的详细信息，请参见模块日志级别。默认级别为 INFO，默认情况下，日志将包含 INFO、WARNING 或 SEVERE 级别的消息。

  6. 单击“保存”。

    对日志级别设置所做的更改将立即生效。这些更改也会保存下来，以便在服务器重新启动时使用。
