配置 JVM 选项

使用“JVM 选项”页可以指定运行 GlassFish Server 的 Java 应用程序启动器 (java) 的选项。-D 选项用于指定特定于 GlassFish Server 的属性。

  1. 在导航树中，选择“配置”节点。

  2. Select the configuration that you are changing.

  3. 在“配置”节点下，选择“JVM 设置”节点。

    此时将打开“JVM 常规设置”页。

  4. 在“JVM 常规设置”页上，单击“JVM 选项”选项卡。

    此时将打开“JVM 选项”页。

  5. 要修改选项，请编辑该选项的“值”字段。

  6. 要添加选项，请执行以下步骤：

    1. 单击“添加 JVM 选项”按钮。

    2. 在显示的空白行中，在“值”字段中键入选项信息。

      如果选项信息包含一个或多个空格，请将整个字符串括在双引号中。

  7. 要删除选项，请执行以下步骤：

    1. 选中要删除的选项的“值”字段左侧的复选框。

    2. 单击“删除”。

  8. 单击“保存”。

  9. 重新启动服务器。
