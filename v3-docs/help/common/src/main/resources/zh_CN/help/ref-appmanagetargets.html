管理目标

使用“管理目标”页可更改可在其上启用应用程序的目标集群和独立服务器实例。只能在“所选目标”列中的目标上启用应用程序。

“管理目标”页包含以下信息。

可用目标

未部署应用程序的集群和实例。

所选目标

已部署的应用程序的集群和实例。

添加

将所选目标从“可用目标”移动到“所选目标”的按钮。

全部添加

将所有“可用目标”移动到“所选目标”的按钮。

删除

将所选目标从“所选目标”移动到“可用目标”的按钮。

全部删除

将所有“所选目标”移动到“可用目标”的按钮。
