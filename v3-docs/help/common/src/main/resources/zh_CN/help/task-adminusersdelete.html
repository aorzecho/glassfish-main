从管理领域中删除用户

  1. 在“配置”节点下，展开您要为其管理“管理领域”用户的服务器实例或群集配置。
  2. 展开“安全性”节点。
  3. 展开“领域”节点。
  4. 选择 "admin-realm" 节点。

    将打开“编辑领域”页面。

  5. 在“编辑领域”页面上，单击“管理用户”按钮。

    此时将打开“管理员用户”页面。

  6. 在“管理员用户”页面上，选中要删除的用户名左侧的复选框。
  7. 单击“删除”。

    此时将提示您确认删除。

  8. 单击“确定”删除用户。
