新建口令别名

使用“新建口令别名”页可创建在口令文件或域配置文件中使用的新口令别名。

“新建口令别名”页包含以下选项：

别名名称

别名的名称。在域中的所有口令别名中，此名称必须是唯一的。

口令

口令别名以加密形式存储的口令。

确认口令

口令别名以加密形式存储的口令。系统将比较“口令”和“确认口令”值，确保输入的口令没有键入错误。
