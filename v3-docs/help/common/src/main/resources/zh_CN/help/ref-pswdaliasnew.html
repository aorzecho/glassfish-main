新建密码别名

使用“新建密码别名”页面可创建在密码文件或域配置文件中使用的新密码别名。

“新建密码别名”页面包含以下选项：

别名名称

别名的名称。在域中的所有密码别名中，此名称必须是唯一的。

密码

密码别名以加密形式存储的密码。

确认密码

密码别名以加密形式存储的密码。系统将比较“密码”和“确认密码”值，确保输入的密码没有键入错误。
