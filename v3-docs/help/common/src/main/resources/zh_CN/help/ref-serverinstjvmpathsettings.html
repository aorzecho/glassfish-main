JVM 路径设置

使用“JVM 路径设置”页面可以配置本地库路径。以下所有路径连接起来组成本地库路径：

  • 本地库路径前缀

  • 服务器的本地库路径（即 as-install/lib）

  • 64 位本地库目录（如果适用）

  • 用于启动 GlassFish Server 的 JVM 的 java.library.path 变量值（该值在各种操作系统中有所不同）

  • 如果启用了事件探查器，则为在 JVM 事件探查器设置页面上指定的任何路径

  • 本地库路径后缀

对于已从早期发行版的 GlassFish Server 升级的域，其他设置将显示为只读值。v3 发行版不支持这些设置。

“JVM 路径设置”页面包含以下选项。

配置名称

要应用该页面上设置的配置名称。此字段为只读。

环境类路径

如果选中“忽略”复选框，将忽略 CLASSPATH 环境变量。对于编程基本教程而言，CLASSPATH 环境变量使用起来很方便，但建议不要将此变量用于企业环境。这是只读字段。

系统类路径

系统中的类的路径。这是只读字段。

类路径前缀

要前置到服务器类路径的 JAR 文件完整路径，每行一个。这是只读字段。

类路径后缀

要附加到服务器类路径的 JAR 文件完整路径，每行一个。这是只读字段。

本地库路径前缀

要前置到本地库路径的 JAR 文件完整路径，每行一个。

本地库路径后缀

要附加到本地库路径的 JAR 文件完整路径，每行一个。
