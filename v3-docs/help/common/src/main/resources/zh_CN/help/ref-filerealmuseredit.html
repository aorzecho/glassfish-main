编辑文件领域用户

使用“编辑文件领域用户”页可以更改用户用于当前领域的口令。

注:

如果启用了安全管理，则无法将 admin-realm 用户的口令更改为空值。

“编辑文件领域用户”页包含以下选项：

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

领域名称

当前选中领域的名称。这是只读字段。

用户 ID

您要编辑的帐户的名称。这是只读字段。

组列表

此用户所属的组。用逗号将组名分隔开。

对于 admin-realm 用户，这是只读字段，并且组为 asadmin。

新口令

此用户的新口令。

确认新口令

此用户的新口令。
