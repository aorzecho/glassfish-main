编辑 File 领域用户

使用“编辑 File 领域用户”页面可以更改用户用于当前领域的密码。

“编辑 File 领域用户”页面包含以下选项：

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

领域名

当前选中领域的名称。这是只读字段。

用户 ID

您要编辑的帐户的名称。这是只读字段。

组列表

此用户所属的组。用逗号将组名分隔开。

对于 admin-realm 用户，这是只读字段，并且组为 asadmin。

新密码

此用户的新密码。

确认新密码

此用户的新密码。
