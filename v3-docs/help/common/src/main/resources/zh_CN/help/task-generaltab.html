查看 DAS 的常规信息

  1. 在导航树中，选择 "server"（管理服务器）节点。

    将打开“常规信息”页面。

  2. 在“常规信息”页面上，执行以下任一步骤：
    • 单击“停止”按钮以停止 GlassFish Server。
    • 单击“重新启动”按钮以停止并重新启动 GlassFish Server。
    • 单击“查看日志文件”按钮可查看 GlassFish 服务器 实例或群集的日志文件。

      有关详细信息，请参见查看 DAS 的日志文件。

    • 单击“JVM 报告”以打开一个单独的窗口，此窗口可让您查看有关 Java 虚拟机的报告，包括摘要报告、内存管理报告、类装入报告和当前线程转储。
