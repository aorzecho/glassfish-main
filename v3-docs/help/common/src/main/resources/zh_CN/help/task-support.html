查看 GlassFish Server 的“GlassFish 新闻”页

“GlassFish 新闻”页提供关于 Oracle GlassFish Server 的附加信息 (如时事通讯、文章、技术提示、论坛以及相关博客) 的链接集合。

  1. 在导航树中，选择“常见任务”节点。

    此时将打开“常见任务”页。

  2. 在“GlassFish 新闻”下，单击“GlassFish 新闻”链接。

    此时将在单独的浏览器选项卡或窗口中打开“GlassFish 新闻”页。
