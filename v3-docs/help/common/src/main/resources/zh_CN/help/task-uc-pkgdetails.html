查看有关组件的详细信息

UpdateTool提供了有关已安装组件、已安装组件的可用更新，以及附加组件的详细信息。

  1. 在导航树中，选择“UpdateTool”节点。

    此时将显示“可用的附加软件”页。

  2. 单击列出此组件的页的选项卡。

    此组件列在以下某个页上：

    • 可用的附加软件

    • 可用更新

    • 安装的组件

  3. 在组件表中，选择此组件。

    将显示所选组件的“组件详细信息”页。
