应用程序范围的资源

使用“应用程序范围的资源”页可查看仅对特定应用程序可用的资源。

“应用程序范围的资源”页包含一个提供以下信息的表：

应用程序名称

应用程序名称。

模块名称

应用程序中的模块的名称。

资源名称

应用程序或模块的应用程序范围的资源的名称。单击该名称可修改资源设置。

资源类型

资源类型，如 JDBC 连接池或连接器资源。
