许可证协议

“许可证协议”页面显示了选择进行安装或更新的所有组件的许可证协议。组件的许可证协议由组件编写人员随此组件一起提供。网络系统信息库中的不同组件可能在不同的许可证下发布，例如:

  • 通用开发和发布许可 (Common Development and Distribution License, CDDL)

  • GNU 通用公共许可证 (General Public License, GPL)

  • Apache 许可证

  • wxWindows 库许可证

“许可证协议”页面包含以下选项：

接受

此按钮接受所有选定组件的许可证条款，并继续进行安装或升级。

拒绝

此按钮拒绝所有选定组件的许可证条款，并使您返回到从中选择组件的页面。
