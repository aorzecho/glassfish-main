原始日志查看器

使用“原始日志查看器”页可查看管理服务器 (名为 server) 的原始日志文件。

“原始日志查看器”页包含以下选项：

实例

要显示日志记录的服务器实例或集群。唯一的选项为 server。

对于每个消息，原始日志查看器将显示以下信息：

时间戳

发布日志消息的日期和时间。

记录编号

服务器日志中的消息编号。

日志级别

消息的日志级别。

版本

GlassFish Server 的版本。

日志记录程序

生成日志消息的日志记录程序模块。

名称/值对

日志消息的线程名称和 ID。

消息

日志消息的简短文本。单击 (详细信息) 链接可以查看消息的格式化版本。
