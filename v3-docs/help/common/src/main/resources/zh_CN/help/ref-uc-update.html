可用更新

“可用更新”页提供了已安装的安装映像组件的更新版本相关信息。

要安装可用更新，请首先停止 GlassFish Server 域。然后使用以下命令之一：

  • glassfish3/bin/updatetool，可以启动 UpdateTool 图形实用程序

  • glassfish3/bin/pkg，UpdateTool 的命令行版本

有关使用 pkg 命令的信息，请参阅 Oracle GlassFish Server Administration Guide 中的第 11 章：Extending and Updating GlassFish Server。UpdateTool 实用程序有自己的联机帮助。

在安装完更新后，请重新启动服务器。

“可用更新”页包含以下选项：

安装路径

安装映像的根目录或顶层目录的文件系统路径。

可用更新

对于有新版本可用的每个组件，此表提供以下信息。

组件

组件名称。

类别

与组件相关联的软件类别，例如“应用程序服务器”、“系统工具”和“数据库”。类别是在创建组件时由组件作者分配到组件中的。

已安装的版本

提供当前组件的程序包的版本号。

新版本

可用更新的版本号。

更改

已安装的版本与可用更新之间的大小差异。

源

将从其获取组件的网络资料档案库的名称。

相关任务

查看组件的可用更新
