领域

GlassFish Server 预先配置了三个领域：文件、证书和 admin-realm。还可以创建 ldap、JDBC、solaris 和定制领域。通常，一个服务器上每种类型的领域都会有一个，但在 GlassFish Server 上有两个文件领域：文件和 admin-realm。这两个领域的类型相同但用于两种不同的目的。还可以在系统中为每台虚拟服务器创建不同的证书数据库。

使用“领域”页可以配置领域。

对于每个领域，将提供以下信息：

名称

领域名称。

类名

领域的类名。

“领域”表还包含以下选项：

新建

用于创建新领域的按钮。

删除

用于删除一个或多个所选领域的按钮。
