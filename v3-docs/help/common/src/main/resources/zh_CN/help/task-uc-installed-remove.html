查看安装的组件

“安装的组件”页面提供了有关已安装的安装映像组件的信息。

卸载组件涉及到删除以下项：

  • 在组件的初始安装时引入的所有文件和目录

  • 在初始安装后更新组件时引入的所有文件和目录

  1. 在导航树中，选择“更新工具”节点。

    此时将打开“可用的附加软件”页面。

  2. 在“可用的附加软件”页面上，单击“安装的组件”选项卡。

    此时将打开“安装的组件”页面。

接下来的操作

要卸载任何已安装组件，请首先停止 GlassFish Server 域。然后使用以下命令之一：

  • as-install-parent/bin/updatetool，可以启动更新工具图形实用程序

  • as-install-parent/bin/pkg，更新工具的命令行版本

有关使用 pkg 命令的信息，请参见 《Oracle GlassFish Server 3.1 Administration Guide》中的第 11  章 "Extending and Updating GlassFish Server"。更新工具实用程序有自己的联机帮助。

在卸载完组件后，请重新启动服务器。
