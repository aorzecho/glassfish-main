编辑领域

  1. 在“配置”节点下，展开您要为其配置领域的服务器实例或群集配置。
  2. 展开“安全性”节点。
  3. 展开“领域”节点。

    此时将打开“领域”页面。

  4. 选择一个现有领域的名称。

    将打开“编辑领域”页面。

  5. 要修改领域的类名，请执行以下操作之一：
    • 从“类名”下拉式列表中，为正在创建的领域选择类名。
    • 选择第二个单选按钮，然后为正在创建的自定义领域键入类名。
  6. （可选的）在“其他属性”部分中，指定其他属性。

    有效属性取决于领域类型。

    • 特定于 FileRealm 类的属性

    • 特定于 CertificateRealm 类的属性

    • 特定于 JDBCRealm 类的属性

    • 特定于 LDAPRealm 类的属性

    • 特定于 SolarisRealm 类的属性

    • 特定于 PamRealm 类的属性

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。
    • 要修改属性，请编辑该属性的“值”字段。
    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。
  7. 单击“保存”。
