查看资源监视数据

开始之前

查看监视数据之前，必须配置监视。有关详细信息，请参阅配置监视服务。

  1. 在导航树中，选择“监视数据”节点。

    此时将打开“监视”页。

  2. 在“查看监视数据”列中，单击与所监视的服务器实例对应的“资源”链接。

    此时将打开“资源监视”页。

  3. 在“连接池”下拉列表中，选择要查看其监视数据的连接池。

  4. 单击“刷新”按钮更新显示的监视数据。

备选步骤：

  1. 在导航树中，选择服务器 (管理服务器) 节点。

    将打开“一般信息”页。

  2. 在“一般信息”页中，单击“监视”选项卡。

    此时将打开“应用程序监视”页。

  3. 单击“资源”选项卡。

    此时将打开“资源监视”页。

  4. 在“连接池”下拉列表中，选择要查看其监视数据的连接池。

  5. 单击“刷新”按钮更新显示的监视数据。
