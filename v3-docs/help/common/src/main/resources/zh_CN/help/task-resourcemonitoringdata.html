查看资源监视数据

开始之前

查看监视数据之前，必须配置监视。有关详细信息，请参见配置监视服务。

  1. 在导航树中，选择 "server"（管理服务器）节点。

    将打开“常规信息”页面。

  2. 在“常规信息”页面中，单击“监视”选项卡。

    此时将打开“应用程序监视”页面。

  3. 单击“资源”选项卡。

    此时将打开“资源监视”页面。

  4. 在“查看”下拉式列表中，选择要查看监视数据的资源。
  5. （可选的）单击“刷新”按钮更新显示的监视数据。
