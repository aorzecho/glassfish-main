编辑领域

使用“编辑领域”页面可以修改领域。

“编辑领域”页面包含以下选项：

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

领域名

领域名称。这是只读字段。只能在创建新领域时指定名称。

类名

您要编辑的领域的类名。

其他属性

领域的其他属性。有效属性取决于领域类型。

  • 特定于 FileRealm 类的属性

  • 特定于 CertificateRealm 类的属性

  • 特定于 JDBCRealm 类的属性

  • 特定于 LDAPRealm 类的属性

  • 特定于 SolarisRealm 类的属性

  • 特定于 PamRealm 类的属性
