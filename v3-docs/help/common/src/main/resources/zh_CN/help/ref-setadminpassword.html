新建管理员口令

使用“新建管理员口令”页可以更改当前管理员帐户的口令。

注:

如果启用了安全管理，则您不能将口令更改为空值。

“新建管理员口令”页包含以下选项：

用户 ID

当前管理员的用户 ID。这是只读字段。

组列表

管理员所属的组。这是只读字段。值为 asadmin。

新口令

管理员的新口令。

确认新口令

管理员的新口令。
