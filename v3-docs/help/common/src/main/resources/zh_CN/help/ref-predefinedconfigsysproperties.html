预定义系统属性

GlassFish Server 定义以下配置系统属性：

ASADMIN_LISTENER_PORT

该属性指定 DAS 通过其连接到实例以管理实例的 HTTP 端口或 HTTPS 端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

HTTP_LISTENER_PORT

此属性指定用于监听 HTTP 请求的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

HTTP_SSL_LISTENER_PORT

此属性指定用于监听 HTTPS 请求的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

IIOP_LISTENER_PORT

此属性指定用于 IIOP 连接的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

IIOP_SSL_LISTENER_PORT

此属性指定用于安全 IIOP 连接的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

IIOP_SSL_MUTUALAUTH_PORT

此属性指定用于安全 IIOP 连接 (带有客户机验证功能) 的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

JAVA_DEBUGGER_PORT

此属性指定用于连接到 Java 平台调试器体系结构 (JPDA) (http://java.sun.com/javase/technologies/core/toolsapis/jpda/) 调试器的端口的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

JMS_PROVIDER_PORT

此属性指定 Java 消息服务提供方的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

JMX_SYSTEM_CONNECTOR_PORT

此属性指定 JMX 连接器对其进行监听的端口号。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。

OSGI_SHELL_TELNET_PORT

此属性指定用于连接到 Apache Felix 远程 Shell (http://felix.apache.org/site/apache-felix-remote-shell.html) 的端口的端口号。此 shell 使用 Felix shell 服务与 OSGi 模块管理子系统进行交互。有效值为 1–65535。在 UNIX 上，创建用于监听 1 到 1024 端口的套接字需要超级用户权限。
