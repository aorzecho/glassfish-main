预定义系统属性

GlassFish Server 定义以下配置系统属性：

ASADMIN_LISTENER_PORT

该属性指定 DAS 通过其连接到实例以管理实例的 HTTP 端口或 HTTPS 端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

HTTP_LISTENER_PORT

此属性可指定用于侦听 HTTP 请求的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

HTTP_SSL_LISTENER_PORT

此属性可指定用于侦听 HTTPS 请求的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

IIOP_LISTENER_PORT

此属性可指定用于 IIOP 连接的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

IIOP_SSL_LISTENER_PORT

此属性可指定用于安全 IIOP 连接的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

IIOP_SSL_MUTUALAUTH_PORT

此属性可指定用于带有客户机验证功能的安全 IIOP 连接的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

JAVA_DEBUGGER_PORT

此属性指定用于连接到 Java 平台调试器体系结构 (Java Platform Debugger Architecture, JPDA) 调试器的端口的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

JMS_PROVIDER_PORT

此属性指定 Java 消息服务提供商的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

JMX_SYSTEM_CONNECTOR_PORT

该属性指定 JMX 连接器对其进行侦听的端口号。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。

OSGI_SHELL_TELNET_PORT

This property specifies the port number of the port that is used for connections to the Apache Felix Remote Shell. 此 shell 使用 Felix shell 服务与 OSGi 模块管理子系统进行交互。有效值为 1–65535。在 UNIX 上，创建在端口 1–1024 上进行侦听的套接字需要超级用户权限。
