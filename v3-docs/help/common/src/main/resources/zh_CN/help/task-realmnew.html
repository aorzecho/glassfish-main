创建领域

GlassFish Server 预先配置了三个领域：文件、证书和 admin-realm。还可以创建 ldap、JDBC、solaris、PAM 和定制领域。通常，一个服务器上每种类型的领域都会有一个，但在 GlassFish Server 上有两个文件领域：文件和 admin-realm。这两个领域的类型相同但用于两种不同的目的。还可以在系统中为每台虚拟服务器创建不同的证书数据库。

  1. 在“配置”节点下，展开您要为其配置领域的服务器实例或集群配置。

  2. 展开“安全性”节点。

  3. 展开“领域”节点。

    此时将打开“领域”页。

  4. 在“领域”页中，单击“新建”。

    此时将打开“新建领域”页。

  5. 在“名称”字段中，键入领域的名称。

  6. 要为领域指定类名，请执行以下操作之一：

    • 从“类名”下拉列表中，为正在创建的领域选择类名。

      下表显示了有效选项。

      领域名称
      		 类名
      file com.sun.enterprise.security.auth.realm.file.FileRealm
      certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • 选择下方的单选按钮，然后为正在创建的定制领域键入类名。

  7. 在“其他属性”部分中，指定其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    有效属性取决于领域类型。

    • 特定于 FileRealm 类的属性

    • 特定于 CertificateRealm 类的属性

    • 特定于 JDBCRealm 类的属性

    • 特定于 LDAPRealm 类的属性

    • 特定于 SolarisRealm 类的属性

    • 特定于 PamRealm 类的属性

  8. 单击“确定”。
