配置 GlassFish Server 工具以供消息安全性提供者使用

  1. 如果使用了用户名令牌，请在必要时配置用户数据库。

    使用用户名/密码令牌时，请先配置适当的领域并为该领域配置适当的用户数据库。

    编辑领域中说明了如何配置用户数据库。
