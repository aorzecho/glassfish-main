启动应用程序客户机

开始之前

确保要启动的应用程序客户机已启用。客户机必须是可以通过 Java Web Start 运行的 Web 客户机。

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 从“已部署的应用程序”表中，单击要启动的应用程序客户机的名称。

    名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号(:) 分隔。有关模块和应用程序版本的更多信息，请参见《Oracle GlassFish Server 3.1 Application Deployment Guide》中的"Module and Application Versions"。

    此时将打开“编辑应用程序”页面。

  3. 确认是否已选中 "Java Web Start" 复选框。

    启动应用程序客户机之前，必须启用 Java Web Start。如果未选中 "Java Web Start" 复选框，请将其选中，然后单击“保存”。

  4. 在“模块和组件”表中，单击“操作”列下的“启动”。

    此时将打开“启动应用程序客户机”页面。

  5. 在“启动应用程序客户机”页面上，键入要传递给应用程序客户机的参数。
  6. 单击“启动”。
