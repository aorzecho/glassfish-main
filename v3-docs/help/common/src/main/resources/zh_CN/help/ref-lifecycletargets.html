生命周期模块目标

使用“生命周期模块目标”页可查看可在其上启用生命周期模块的目标集群和独立服务器实例。

“生命周期模块目标”页包含以下信息。

目标名称

集群或独立服务器实例的名称。

已启用

如果生命周期模块在目标上已启用，则显示真，如果生命周期模块已禁用，则显示假。

“目标”表还包含以下选项：

管理目标

管理生命周期模块目标的按钮。

更多操作

以下操作的下拉列表。

启用

在所选目标上启用生命周期模块的操作。

禁用

在所选目标上禁用生命周期模块的操作。
