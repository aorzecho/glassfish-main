配置

使用“配置”页面可管理配置以及查看使用配置的独立服务器实例和群集。

default-config 配置由 GlassFish Server 预定义。在创建域时自动为域管理服务器 (Domain Administration Server, DAS) 创建 server-config 配置。有关更多信息，请参见《Oracle GlassFish Server 3.1 High Availability Administration Guide》中的"About Named Configurations"。

对于每个配置，将提供以下信息。

名称

配置的名称。

实例

使用该配置的独立实例和群集的列表。

“配置”表还包含以下选项。

新建

用于创建新配置的按钮。

删除

用于删除一个或多个选定配置的按钮。
