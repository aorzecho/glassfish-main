编辑 JMX 连接器

使用“管理服务”可以配置符合 JSR-160 标准的远程 JMX 连接器。

符合 JSR-160 标准的远程 JMX 连接器处理 JMX 客户机（例如 JConsole）和 JMX 服务之间的通信，以实现域管理目的。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“管理服务”节点。

    此时将打开“编辑 JMX 连接器”页面。

  3. 选中“启用安全性”复选框以对 JMX 通信进行加密。

    默认情况下禁用此选项。

  4. 在“地址”字段中，键入 JMX 连接器服务器存根在其中注册的命名服务的 IP 地址。

    这不是进行实际 JMX 通信的服务器套接字的端口。这是启动 RMI 注册表的网络接口的地址。如果系统有多个网络接口，请修改此值，以便仅选择某个特定的接口。默认值为 0.0.0.0。

  5. 在“端口”字段中，键入命名服务（RMI 注册表）侦听 RMI 客户机连接的端口号。

    此命名服务的唯一用途是下载 RMI 存根。如果默认端口已占用，则使用空闲端口。合法值为 1 - 65535。在 UNIX 系统上，创建侦听端口 1 - 1024 的套接字需要超级用户权限。默认值为 8686。

  6. 在“领域名”字段中，键入代表特殊管理领域的名称。

    所有验证均由此领域处理。默认值为 admin-realm。

  7. （可选的）在“其他属性”部分中，指定其他属性。

    默认情况下，GlassFish Server 会指定若干属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。
    • 要修改属性，请编辑该属性的“值”字段。
    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。
  8. 单击“保存”。
  9. 如果已选中“启用安全性”复选框，请按照编辑 JMX 连接器的 SSL 设置中的步骤操作。
