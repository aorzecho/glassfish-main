JVM 选项

使用“JVM 选项”页可以指定运行 GlassFish Server 的 Java 应用程序启动器 (java) 的选项。-D 选项用于指定特定于 GlassFish Server 的属性。

  • 要修改选项，请编辑该选项的“值”字段。

  • 要添加选项，请单击“添加 JVM 选项”按钮。在显示的空白行中，在“值”字段中键入选项信息。

    如果选项信息包含一个或多个空格，请将整个字符串括在双引号中。

  • 要删除选项，请选中要删除的选项的“值”字段左侧的复选框，然后再单击“删除”。
