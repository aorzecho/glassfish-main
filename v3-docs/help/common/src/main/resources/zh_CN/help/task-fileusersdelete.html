从文件领域中删除用户

  1. 在“配置”节点下，展开您要为其管理“管理领域”用户的服务器实例或集群配置。

  2. 展开“安全性”节点。

  3. 展开“领域”节点。

  4. 选择 file 节点。

    将打开“编辑领域”页。

  5. 在“编辑领域”页上，单击“管理用户”按钮。

    将打开“文件用户”页。

  6. 在“文件用户”页上，选择要删除的用户名左侧的复选框。

  7. 单击“删除”。

    此时将出现一个对话框，要求确认是否应删除该用户。

  8. 在此对话框中，单击“确定”。
