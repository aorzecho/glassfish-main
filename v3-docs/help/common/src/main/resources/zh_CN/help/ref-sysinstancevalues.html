实例值

使用“实例值”页可为引用所选的命名配置的 GlassFish Server 实例编辑配置系统属性的值。

“实例值”页显示引用所选配置的实例的列表，在此配置中设置了该属性。

实例名称

创建实例时为其指定的名称。单击此名称可打开实例的“一般信息”页。

集群名称

实例所属的集群的名称 (如果有)。此字段为只读。

默认值

为实例引用的命名配置中的属性所设置的值。此字段为只读。

覆盖值

为所选的实例设置的属性的值。此值会覆盖默认值。
