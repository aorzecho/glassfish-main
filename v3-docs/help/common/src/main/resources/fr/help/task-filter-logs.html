Filtrer les enregistrements de journal affichés

Avant de commencer

Assurez-vous que vous êtes en train d'afficher les fichiers journaux pour le serveur d'administration de domaine (DAS), une instance GlassFish Server clusterisée ou une instance autonome. Pour plus d’informations, consultez les rubriques suivantes :

  • Afficher les journaux du DAS

  • To View Log Files for a Clustered GlassFish Server Instance

  • To View Log Files for a Standalone GlassFish Server Instance

  1. Dans la zone Critères de recherche, filtrez les enregistrements par contenu, période et niveau du journal.
    1. Pour afficher uniquement les enregistrements contenant un texte spécifique, saisissez le texte dans le champ Recherche de texte.

      La recherche est sensible à la casse.

    2. Pour sélectionner la période pour laquelle les enregistrements de journal sont affichés, utilisez l'option Horodatage.
      • Pour afficher les messages les plus récents, sélectionnez Le plus récent (valeur par défaut).
      • Pour afficher les messages uniquement sur une période donnée, sélectionnez Plage spécifique et entrez une date et une heure dans les champs De et À qui apparaissent.

        La valeur Date doit prendre la forme suivante : 

        MM/DD/YY
        MM

        Nombre entier compris entre 1 et 12 pour désigner le numéro du mois, par exemple 1 pour le mois de janvier ou 10 pour le mois d'octobre.

        DD

        Nombre entier compris entre 1 et 31 pour désigner le jour du mois.

        YY

        Deux derniers chiffres de l'année, par exemple, 11 pour l'année 2011.

        Par exemple, pour spécifier la date du 1er février 2011, saisissez 2/1/11 dans le champ Date.

        La valeur Heure doit prendre la forme suivante : 

        hh:mm:ss.SSS
        hh

        Nombre entier compris entre 0 et 23 pour indiquer l'heure. Pour les heures de l'après-midi, utilisez des nombres entiers compris entre 12 et 23, par exemple 14 pour indiquer 2 PM.

        mm

        Nombre entier compris entre 0 et 59 pour indiquer le nombre de minutes après l'heure.

        ss

        Nombre entier compris entre 0 et 59 pour indiquer le nombre de secondes.

        SSS

        Nombre entier à trois chiffres, précédé de zéros non significatifs si nécessaire, compris entre 000 et 999 pour indiquer le nombre de millisecondes.

        Par exemple, pour indiquer l'heure 5.10 PM, saisissez 17:10:00.000 dans le champ Heure.

        Si la valeur de début est postérieure à la valeur de fin, un message d’erreur apparaît.

    3. Filtrer les messages par niveau de journalisation

      Pour être sûr que les messages que vous voulez afficher apparaissent dans le journal du serveur, commencez par définir les niveaux de journalisation appropriés sur la page Niveaux de journalisation de module.

      Si vous choisissez de filtrer les messages du journal en fonction du niveau de journalisation, seuls les messages correspondant aux critères de filtre spécifiés s’affichent. Toutefois, ce filtrage n’affecte pas les messages consignés dans le journal du serveur.

      1. Dans la liste déroulante Niveau de journalisation, sélectionnez le niveau de journalisation.
      2. Pour afficher des messages au niveau sélectionné uniquement, sélectionnez l'option Ne pas inclure des messages plus graves.

        Par défaut, tous les messages figurant dans le journal du serveur à partir du niveau de journalisation choisi sont affichés.

  2. Dans la zone Recherche avancée, filtrez les enregistrements par espace de noms du journal et sélectionnez la quantité d'informations qui s'affiche.
    1. Cliquez sur le lien Recherche avancée.

      La zone Recherche avancée est ajoutée à la fenêtre de la visionneuse du journal.

    2. Pour filtrer par module, sélectionnez un ou plusieurs espaces de noms dans la liste déroulante Journal.

      Utilisez Maj + clic ou Ctrl + clic pour sélectionner plusieurs espaces de noms.

      Si vous sélectionnez un espace de noms à un niveau supérieur, tous les espaces de noms situés en dessous seront également sélectionnés. Par exemple, si vous sélectionnez javax.enterprise.system, seront également sélectionnés les journaux de tous les modules situés sous cet espace de noms, tels que javax.enterprise.system.core, javax.enterprise.system.tools.admin , etc.

    3. Pour afficher les messages de journaux d'une application particulière, saisissez le nom de chaque journal sur une ligne distincte dans le champ Journal personnalisé.

      Si l’application comprend plusieurs modules, vous pouvez les afficher tous ou seulement une partie. Par exemple, une application peut disposer des journaux suivants :

      • com.example.myapp.module1

      • com.example.myapp.module2

      • com.example.myapp.module3

      Pour afficher des messages de tous les modules dans l’application, tapez com.mycompany.myapp . Pour afficher les messages du module2 uniquement, entrez com.example.myapp.module2.

      Si vous spécifiez un ou plusieurs enregistreurs personnalisés, les messages des modules du GlassFish Server n’apparaissent que si vous les avez explicitement spécifiés dans la zone Enregistreur.

    4. Dans la liste déroulante Afficher, sélectionnez le nombre de messages à afficher.

      Le nombre de messages affichés par défaut est de 40. Les autres choix sont 100, 250 et 1000.

    5. Pour afficher les suivis de pile, désélectionnez l'option Tronquer les messages trop longs.

      Par défaut, les suivis de pile n’apparaissent pas dans la visionneuse. Pour les afficher, cliquez sur le lien (détails) pour un message.

    6. Pour masquer la zone Recherche avancée, cliquez sur le lien Recherche élémentaire.
  3. Pour indiquer l'instance et le fichier journal associé à afficher, utilisez la zone Modifier la recherche.
    • Dans la liste déroulante Instance, sélectionnez l'instance.

      La valeur par défaut est l'instance sélectionnée initialement pour afficher les fichiers journaux. Si vous avez choisi d'afficher des fichiers journaux pour le serveur d'administration de domaine (DAS), la valeur par défaut est server, ce qui désigne le DAS.

    • Dans la liste déroulante Fichier journal, sélectionnez le fichier journal.

      La valeur par défaut est server.log. Les autres choix dépendent des paramètres du journal pour la cible sélectionnée.

  4. Dans la zone Résultats de la recherche, vous pouvez personnaliser l’affichage des messages.

    La zone Résultats de la recherche affiche les 40 dernières entrées dans le journal du serveur, en fonction des paramètres définis aux pages Paramètres de journalisation et Niveaux de journalisation.

    • Pour trier les messages différemment, cliquez sur les flèches situées dans les colonnes.
    • Pour afficher la version formatée d'un message, cliquez sur le lien (détails) pour le message.

      La fenêtre Détails de l'entrée de journal s'ouvre, contenant une version formatée du message.

    • Pour afficher les entrées antérieures ou postérieures du fichier journal, cliquez sur Enregistrements avant et Enregistrements après en haut de la table.
