Descripteur de déploiement

Utilisez la page Descripteur de déploiement pour afficher le descripteur de déploiement pour un module.

Cette page contient les informations suivantes.

Nom de l’application

Le nom de l’application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

Nom du module

Le nom du module qui contient le descripteur de déploiement.

Nom du fichier du descripteur

Le nom du fichier du descripteur de déploiement.
