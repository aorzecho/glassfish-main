Configurations

Utilisez la page Configurations pour gérer les configurations et afficher les instances de serveur autonomes et les clusters les utilisant.

Pour chaque configuration, les informations suivantes sont fournies.

Nom

Nom de la configuration.

Instance

Liste des instances autonomes et des clusters utilisant la configuration.

Le tableau Configurations contient également les options suivantes.

Nouveau

Bouton permettant de créer une configuration.

Supprimer

Bouton permettant de supprimer une ou plusieurs configurations sélectionnées.
