Enregistrement d'Oracle GlassFish Server

Les utilisateurs enregistrés d'Oracle GlassFish Server bénéficient des avantages suivants :

  • informations de patch et mises à jour de bogue ;

  • screencasts et didacticiels ;

  • actualités et événements ;

  • support et offres de formation.

Pour enregistrer votre installation d'Oracle GlassFish Server, vous devez posséder un compte Sun en ligne. Si vous n’avez pas encore de compte en ligne Sun, GlassFish Server vous permet d’en créer un lors de l’enregistrement.

Remarque - Si vous optez pour cette solution, vous êtes invitée à saisir uniquement les informations obligatoires. Après l’enregistrement de GlassFish Server, vous pouvez fournir des informations supplémentaires facultatives en mettant à jour votre compte en ligne Sun.

Avant de commencer

Si la machine sur laquelle GlassFish Server s’exécute se connecte à Internet par l’intermédiaire d’un proxy, procurez-vous le nom de l’hôte et son numéro de port. Si un navigateur Web s’exécute sur la même machine que GlassFish Server, vous pouvez obtenir ces informations dans ce navigateur. Sinon, contactez votre administrateur système.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Enregistrement.

    La page Enregistrement du produit s’ouvre.

  2. Dans la page d’enregistrement du produit, saisissez les informations requises.
    • Si vous disposez déjà d’un compte en ligne Sun, procédez comme suit :
      1. Sélectionnez l’option J’ai déjà un compte en ligne Sun ou un compte SDN (Sun Developer Network).
      2. Dans le champ Nom d’utilisateur, entrez le nom spécifié à la création de votre compte en ligne.
      3. Faites de même pour le champ Mot de passe.
    • Si vous ne disposez pas d’un compte en ligne Sun, procédez comme suit :
      1. Sélectionnez l’option Je ne possède pas de compte en ligne Sun.
      2. Dans le champ Adresse électronique, entrez votre adresse e-mail.

        Cette dernière sera le nom d’utilisateur de votre compte en ligne Sun.

      3. Dans le champ Mot de passe, entrez le mot de passe choisi pour accéder à votre compte.

        Celui-ci est sensible à la casse et doit comporter entre 6 et 10 caractères.

      4. Dans le champ Confirmation du mot de passe, entrez à nouveau le mot de passe choisi précédemment.
      5. Dans le champ Prénom, entrez votre prénom.
      6. Dans le champ Nom, entrez votre nom.
      7. Dans le champ Société, saisissez le nom de votre société, le cas échéant.
      8. À partir de la liste déroulante, sélectionnez votre pays ou territoire de résidence.
  3. Si la machine sur laquelle GlassFish Server s’exécute se connecte à Internet par l’intermédiaire d’un proxy, entrez le nom de l’hôte et son numéro de port.

    Si la machine sur laquelle GlassFish Server s’exécute se connecte directement à Internet, ignorez cette étape.

    Il est possible qu’un navigateur Web s’exécute sur la même machine que GlassFish Server. Dans ce cas, vous pouvez obtenir le nom d’hôte et le numéro de port du proxy dans ce navigateur. Sinon, contactez votre administrateur système.

  4. Cliquez sur S’enregistrer.

    Pour confirmer la fin de l’enregistrement, la page Support s’ouvre.
