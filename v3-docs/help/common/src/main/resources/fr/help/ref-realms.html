Domaines

Le GlassFish Server est préconfiguré avec trois domaines : file, certificate et admin-realm. Il est également possible de créer d’autres domaines ldap, JDBC, solaris et des domaines personnalisés. Un serveur comprend généralement un domaine de chaque type, tandis que le GlassFish Server comprend deux domaines de fichiers : file et admin-realm. Il s’agit de deux domaines du même type, mais ayant deux utilisations différentes. Il est également possible d’avoir une base de données de certificats différente pour chaque serveur virtuel du système.

Utilisez la page Domaines pour configurer les domaines.

Pour chaque domaine, les informations suivantes sont fournies.

Nom

Nom du domaine.

Nom de classe

Nom de classe du domaine.

Le tableau Domaines contient également les options suivantes.

Nouveau

Bouton permettant de créer un nouveau domaine.

Supprimer

Bouton permettant de supprimer un ou plusieurs domaines sélectionnés.
