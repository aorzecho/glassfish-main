Cibles de l'application

La page Cibles de l'application permet d'afficher les clusters et les instances de serveur autonomes cible sur lesquels l'application peut être activée.

Elle contient les informations suivantes :

Nom de cible

Nom du cluster ou de l'instance de serveur autonome.

Activé

Affiche True si l'application est activée sur la cible ou False si l'application est désactivée.

EC activé

Affiche True si l'application est activée pour l'équilibrage de charge sur la cible, ou False si l'application est désactivée pour l'équilibrage de charge.

Serveurs virtuels

Pour associer l'application à des serveurs virtuels spécifiques sur une cible, sélectionnez Gérer les serveurs virtuels pour cette cible.

La table Cibles contient également les options suivantes :

Gérer les cibles

Bouton permettant de gérer les cibles de l'application.

Plus d'actions

Liste déroulante des actions suivantes.

Activer

Action permettant d'activer l'application sur la cible sélectionnée.

Désactiver

Action permettant de désactiver l'application sur la cible sélectionnée.

Equilibreur de charge activé

Action permettant d'activer l'application pour l'équilibrage de charge sur la cible sélectionnée.

Equilibreur de charge désactivé

Action permettant de désactiver l'application pour l'équilibrage de charge sur la cible sélectionnée.
