Installer des composants d'extension

L'installation d'un composant d'extension inclut les opérations suivantes :

  • Transférer le contenu du composant et de ses dépendances depuis un référentiel réseau

  • Installer le composant sous l'image d'installation GlassFish Server dans votre système

Si un composant nécessite un autre composant qui n'est pas encore installé, ce dernier est également installé.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Outil de mise à jour.

    La page Extensions disponibles apparaît.

  2. Dans la table de la page Extensions disponibles, sélectionnez les composants à installer.

    Si aucun composant d'extension n'est disponible, la table est vide.

    • Pour sélectionner un composant, cochez la case située en regard de son nom.

    • Pour sélectionner tous les composants, cliquez sur l'icône Sélectionner tout dans l'en-tête de la table.

    • Pour désélectionner tous les composants, cliquez sur l'icône Désélectionner tout dans l'en-tête de la table.

  3. Cliquez sur Installer.

    Une boîte de dialogue vous demande de confirmer que les composants sélectionnés, ainsi que tout autre composant requis, doivent être installés.

  4. Dans la boîte de dialogue, cliquez sur OK.

    Vous devez accepter les contrats de licence des composants que vous installez.

  5. Acceptez les contrats de licence des composants que vous installez.

    GlassFish Server confirme que l'installation est terminée. Les composants sont enlevés de la table des composants d'extension et ajoutés à la table de la page Composants installés.

  6. Redémarrez le serveur.

    Pour obtenir des instructions détaillées, reportez-vous à la documentation ci-dessous :

    • Arrêter un domaine - manuel Oracle GlassFish Server Administration Guide

    • Démarrer un domaine - manuel Oracle GlassFish Server Administration Guide
