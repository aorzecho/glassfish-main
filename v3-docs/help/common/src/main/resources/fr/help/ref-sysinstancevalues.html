Valeurs de l’instance

La page Valeurs de l'instance permet de modifier les valeurs d'une propriété système de configuration pour les instances GlassFish Server faisant référence à la configuration nommée sélectionnée.

La page Valeurs de l'instance affiche une liste d'instances qui font référence à la configuration sélectionnée dans laquelle la propriété est définie.

Nom d’instance

Nom assigné à l'instance lors de sa création. Le fait de cliquer sur le nom ouvre la page Informations générales de l'instance.

Nom du cluster

Nom du cluster dont l'instance est membre, le cas échéant. Ce champ est en lecture seule.

Valeur par défaut

Valeur définie pour la propriété dans la configuration nommée à laquelle l'instance fait référence. Ce champ est en lecture seule.

Remplacer la valeur

Valeur de la propriété définie pour l'instance sélectionnée. Cette valeur remplace la valeur par défaut.
