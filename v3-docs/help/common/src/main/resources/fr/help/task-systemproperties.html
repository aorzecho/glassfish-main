Pour gérer les propriétés système

  1. Dans l’arborescence de navigation, sélectionnez le nœud GlassFish Server.

    La page Informations générales s’affiche.

  2. Sur la page Informations générales, cliquez sur l’onglet Propriétés système.

    La page Propriétés système s’affiche.

  3. Dans la section Autres propriétés, spécifiez les autres propriétés système pour la configuration.

    Un certain nombre de propriétés système sont prédéfinies mais ne sont pas indiquées sur cette page. Pour en savoir plus, reportez-vous à la section Propriétés système prédéfinies.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  4. Cliquez sur Enregistrer.
