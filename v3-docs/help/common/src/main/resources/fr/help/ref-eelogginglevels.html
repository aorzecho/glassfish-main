Niveaux de journalisation des modules

Utilisez la page Niveaux de journalisation des modules pour configurer les niveaux de journalisation des modules individuels.

Pour chaque module, les informations suivantes sont fournies.

Nom du journal

Nom du journal pour le module (par exemple, javax.enterprise.system.tools.admin).

Niveau de journalisation

Niveau de journalisation actuel pour le module.

Le tableau Paramètres de l’enregistreur contient également les options suivantes.

Add Logger (Ajouter un journal)

Bouton permettant d'ajouter un module de journalisation. Le fait de cliquer sur ce bouton permet d'ajouter une ligne au tableau Paramètres de journalisation. Entrez les informations sur le module de journalisation dans la nouvelle ligne.

Delete Logger (Supprimer le journal)

Bouton permettant de supprimer un ou plusieurs modules sélectionnés.

Niveau

Liste déroulante dans laquelle vous pouvez sélectionner un niveau de journalisation à appliquer à tous les modules sélectionnés.

Modifier le niveau

Bouton permettant de modifier le niveau de journalisation d’un ou plusieurs modules sélectionnés.

Les niveaux de journalisation suivants sont disponibles. Ils sont répertoriés en ordre descendant, du plus élevé au plus bas.

GRAVE

Evénements qui interfèrent avec l’exécution normale des programmes.

WARNING

Avertissements, notamment les exceptions.

INFO

Messages liés à la configuration du serveur ou au statut du serveur, à l’exclusion des erreurs.

CONFIG

Messages liés à la configuration du serveur.

FINE

Verbosité minimale

FINER

Verbosité modérée.

FINEST

Verbosité maximale.

OFF

Aucun message de journalisation.
