Affichage des données de contrôle des ressources

Avant de commencer

Avant d’afficher les données de contrôle, vous devez configurer le contrôle. Consultez Configurer le contrôle du service pour en savoir plus.

  1. Dans l’arborescence de navigation, sélectionnez le nœud du serveur (Admin Server).

    La page Informations générales s’affiche.

  2. Dans la page Informations générales, cliquez sur l’onglet Contrôle.

    La page Contrôle des applications s’ouvre.

  3. Cliquez sur l’onglet Ressources.

    La page Contrôle des ressources s’ouvre.

  4. Dans la liste déroulante Afficher, sélectionnez la ressource pour laquelle vous souhaitez afficher les données de contrôle.
  5. (Facultatif) Cliquez sur le bouton Actualiser pour mettre à jour les données de contrôle affichées.
