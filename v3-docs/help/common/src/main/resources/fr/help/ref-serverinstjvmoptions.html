Options JVM

La page Options JVM vous permet de spécifier les options du lanceur de l’application Java (outil java) exécutant GlassFish Server. Les options -D désignent les propriétés spécifiques à GlassFish Server.

  • Pour modifier une option, modifiez le contenu du champ Valeur.

  • Pour ajouter une option, cliquez sur le bouton Ajouter une option JVM. Sur la ligne vide qui apparaît, saisissez les informations d’option dans le champ Valeur.

  • Pour supprimer une option, cochez la case accolée au champ Valeur de l’option à supprimer, puis cliquez sur Supprimer.
