Options JVM

La page Options JVM permet de spécifier les options du lanceur de l'application Java (java) exécutant GlassFish Server. Les options -D désignent les propriétés propres à GlassFish Server.

  • Pour modifier une option, modifiez le contenu du champ Valeur.

  • Pour ajouter une option, cliquez sur le bouton Ajouter une option JVM. Sur la ligne vide qui apparaît, saisissez les informations d'option dans le champ Valeur.

    Si les informations d'option contiennent des espaces, mettez la totalité de la chaîne entre guillemets.

  • Pour supprimer une option, cochez la case en regard du champ Valeur de l'option à supprimer, puis cliquez sur Supprimer.
