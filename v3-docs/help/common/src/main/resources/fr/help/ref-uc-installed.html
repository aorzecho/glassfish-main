Composants installés

La page Composants installés contient des informations sur les composants de l’image d’installation déjà installés.

Pour désinstaller des composants installés, commencez par arrêter le domaine GlassFish Server. Utilisez ensuite l’une des commandes suivantes :

  • glassfish3/bin/updatetool, qui démarre l’utilitaire graphique de l’outil de mise à jour

  • glassfish3/bin/pkg, version ligne de commande de l’outil de mise à jour

Pour de plus amples informations sur l'utilisation de la commande pkg, reportez-vous au Chapitre 11, Extending and Updating GlassFish Server du Oracle GlassFish Server 3.1 Administration Guide. L’utilitaire Outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé la désinstallation des composants.

La page Composants installés contient les options suivantes :

Chemin d’installation

Chemin du système de fichiers du répertoire root ou du répertoire supérieur de l’image d’installation.

Installed Components

Pour chacun des composants installés, le tableau fournit les informations suivantes.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel à laquelle le composant est associé, comme par exemple serveurs d’application, outils système et bases de données. La catégorie est affectée au composant par le fabricant du composant au moment de sa création.

Version

Numéro de version du package comprenant le composant.

Taille de l’installation

Espace disque utilisé pour l’installation du composant.

Source

Nom du Référentiel réseau à partir duquel le composant a été obtenu.
