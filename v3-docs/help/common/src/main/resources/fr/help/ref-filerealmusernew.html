Nouvel utilisateur de domaine de fichier

Pour autoriser un nouvel utilisateur à accéder au domaine actuel, utilisez la page Nouvel utilisateur de domaine de fichier.

La page Nouvel utilisateur de domaine de fichier contient les options suivantes :

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom de domaine

Nom du domaine actuellement sélectionné. Ce champ est en lecture seule.

ID utilisateur

ID utilisateur du compte du nouvel utilisateur.

Liste des groupes

Groupes auxquels le nouvel utilisateur appartient. Séparez les noms de groupes par des virgules.

Pour les utilisateurs du domaine d’administration, ce champ est en lecture seule, et le groupe est asadmin.

Nouveau mot de passe

Mot de passe du nouvel utilisateur.

Confirmer le mot de passe

Mot de passe du nouvel utilisateur.

  • Suppression d'un utilisateur du domaine de fichiers
