Nouvel utilisateur du domaine de fichiers

Pour autoriser un nouvel utilisateur à accéder au domaine en cours, utilisez la page Nouvel utilisateur du domaine de fichiers.

Remarque:

Si l'administration sécurisée est activée, vous ne pouvez pas créer un utilisateur admin-realm avec un mot de passe vide.

La page Nouvel utilisateur du domaine de fichiers contient les options suivantes :

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom de domaine

Nom du domaine actuellement sélectionné. Ce champ est en lecture seule.

ID utilisateur

ID utilisateur du compte du nouvel utilisateur.

Liste des groupes

Groupes auxquels le nouvel utilisateur appartient. Séparez les noms de groupes par des virgules.

Pour les utilisateurs admin-realm, ce champ est en lecture seule, et le groupe est asadmin.

Nouveau mot de passe

Mot de passe du nouvel utilisateur.

Confirmer le nouveau mot de passe

Mot de passe du nouvel utilisateur.
