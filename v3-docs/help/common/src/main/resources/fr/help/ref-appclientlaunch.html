Lancement de client d’application

Utilisez la page de lancement de client d'application pour lancer un client d'application déployé. Le client doit être un client Web qui peut être exécuté avec Java Web Start.

La page de lancement de client d'application contient les options suivantes :

Application

Le nom de l'application contenant le client de l'application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

Module

Le nom du module contenant le client de l'application.

Lien

L'URL à laquelle le client est déployé.

Arguments

Les arguments à transmettre à l'application.
