Lancement du client d'application

Utilisez la page de lancement de client d'application pour lancer un client d'application déployé. Il doit s'agir d'un client Web qui peut être exécuté avec Java Web Start.

La page de lancement de client d'application contient les options suivantes :

Application

Nom de l'application contenant le client de l'application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à "Module and Application Versions" dans le manuel Oracle GlassFish Server Application Deployment Guide.

Module

Nom du module contenant le client de l'application.

Liens

URL à laquelle le client est déployé.

Arguments

Arguments à transmettre à l'application.
