Configurer les paramètres du profileur JVM

Un outil de profilage génère des données servant à analyser les performances et à identifier les éventuels goulots d’étranglement.

  1. Dans l’arborescence de navigation, sélectionnez le noeud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Paramètres JVM.

    La page Paramètres JVM s’ouvre.

  3. Dans la page Paramètres généraux JVM, cliquez sur l’onglet Profileur.

    La page Paramètres du profileur JVM s’ouvre.

  4. Définissez les paramètres du profileur.

    Les informations saisies dépendent du profileur utilisé. Pour obtenir des exemples et des instructions, reportez-vous au Chapitre 3, Debugging Applications du Oracle GlassFish Server 3.1 Application Development Guide.

  5. Cliquez sur Enregistrer.
  6. Redémarrez le serveur.
