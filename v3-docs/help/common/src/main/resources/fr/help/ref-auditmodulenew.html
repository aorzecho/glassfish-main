Nouveau module d’audit

La page Nouveau module d’audit vous permet de définir un nouveau module d’audit.

La page Nouveau module d’audit contient les options suivantes :

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom

Nom du nouveau module d’audit.

Nom de classe

Nom complet de la classe qui implémente ce module. Le nom de classe du module d’audit par défaut est com.sun.enterprise.security.Audit.

Autres propriétés

Autres propriétés du module d’audit. Pour obtenir une description de la propriété supplémentaire disponible pour les modules d’audit, consultez Propriétés propres aux modules d'audit.
