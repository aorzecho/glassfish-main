Descripteurs de modules

La page Descripteurs de modules permet d'afficher les descripteurs de modules d'une application.

Elle contient les options suivantes :

Descripteurs de déploiement

Table affichant le nom de l'application, le nom du sous-composant (le cas échéant) et le nom de fichier descripteur.
