Propriétés propres aux modules d’audit

La propriété suivante est disponible pour un module d’audit.

auditOn

Une valeur true entraîne le chargement du module d’audit et assure qu’il est appelé par la bibliothèque d’audit du GlassFish Server aux points d’audit. Une valeur false désactive la journalisation d’audit pour le module.
