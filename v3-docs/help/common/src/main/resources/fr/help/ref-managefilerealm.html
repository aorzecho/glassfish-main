Utilisateurs de fichiers

Utilisez la page Utilisateurs de fichiers pour gérer les utilisateurs de domaine file. Les utilisateurs et les groupes du domaine file sont répertoriés dans le fichier clé, dont l’emplacement est spécifié par la propriété file.

Un utilisateur du domaine fichier peut appartenir à un a groupe Java, c’est-à-dire une catégorie d’utilisateurs classés par caractéristiques communes. Par exemple, les clients d’une application d’e-commerce appartiendront au groupe CUSTOMER, tandis que les plus gros clients appartiendront au groupe PREFERRED. La classification des utilisateurs dans des groupes facilite le contrôle de l’accès d’un grand nombre d’utilisateurs.

Une fois le GlassFish Server installé, le seul utilisateur est l’administrateur configuré lors de l’installation. Par défaut, cet utilisateur appartient au groupe asadmin, du domaine administrateur, qui autorise la modification du GlassFish Server. Les utilisateurs rattachés à ce groupe ont des privilèges d’administrateur, c’est-à-dire qu’ils ont accès à l’outil asadmin et à la Console d'administration .

La page Utilisateurs de fichiers affiche une liste des utilisateurs pour le domaine sélectionné. Pour chaque utilisateur, les informations suivantes sont fournies :

ID utilisateur

Nom d’utilisateur.

Liste des groupes

Groupes auxquels l’utilisateur appartient.

Le tableau Utilisateurs de fichiers contient également des options suivantes.

Nouveau

Bouton permettant de créer un nouvel utilisateur.

Supprimer

Bouton permettant de supprimer un ou plusieurs utilisateurs sélectionnés.
