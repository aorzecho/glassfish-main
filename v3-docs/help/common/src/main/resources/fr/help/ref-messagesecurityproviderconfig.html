Configurations de fournisseur

La page Configurations de fournisseur permet de définir les configurations de fournisseurs de sécurité des messages pour la couche d'authentification SOAP ou HttpServlet.

Par défaut, quatre fournisseurs sont configurés pour la couche d'authentification SOAP.

Pour chaque configuration de fournisseur, les informations suivantes sont fournies.

ID fournisseur

Nom de la configuration du fournisseur.

Type de fournisseur

Type du fournisseur, notamment :

client

Spécifie que le fournisseur est le fournisseur d'authentification client.

server

Spécifie que le fournisseur est le fournisseur d'authentification du serveur.

client-server

Spécifie que le fournisseur est à la fois le fournisseur d'authentification client et du serveur.

Fournisseur par défaut

Si la valeur est True, le fournisseur est le fournisseur par défaut.

Nom de classe

Nom de la classe d'implémentation Java du fournisseur.

La table Configurations de fournisseur contient également les options suivantes :

Nouveau

Bouton permettant de créer une configuration de fournisseur.

Supprimer

Bouton permettant de supprimer les configurations de fournisseur sélectionnées.
