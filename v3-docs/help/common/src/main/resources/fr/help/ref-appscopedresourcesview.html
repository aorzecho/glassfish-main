Afficher les ressources de niveau application

  1. Dans l'arborescence de navigation, sélectionnez le noeud Applications.

    La page Applications apparaît.

  2. Sélectionnez l'application dont vous souhaitez afficher les ressources.

    Le nom de l'application peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à "Module and Application Versions" dans le manuel Oracle GlassFish Server Application Deployment Guide.

    La page de l'application apparaît.

  3. Sélectionnez l'onglet Ressources.

    La page Ressources de niveau application apparaît et les ressources actuellement spécifiées pour l'application s'affichent dans une table comportant les colonnes suivantes :

    Nom de l'application

    Nom de l'application.

    Nom du module

    Nom du module dans l'application.

    Nom de ressource

    Nom de la ressource de niveau application pour le module ou l'application.

    Type de ressource

    Type de ressource (par exemple, des pools de connexions JDBC ou des ressources de connecteur).
