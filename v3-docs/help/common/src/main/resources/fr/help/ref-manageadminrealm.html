Administrateurs

La page Administrateurs permet de gérer les utilisateurs admin-realm. admin-realm est un domaine de fichiers (file). Les utilisateurs et les groupes du domaine file sont répertoriés dans le fichier de clés, dont l'emplacement est spécifié par la propriété file.

Juste après l'installation de GlassFish Server, le seul utilisateur est l'administrateur configuré lors de l'installation. Par défaut, cet utilisateur appartient au groupe asadmin, du domaine administrateur, qui autorise la modification de GlassFish Server. Les utilisateurs rattachés à ce groupe ont des privilèges d'administrateur, c'est-à-dire qu'ils ont accès à l'outil asadmin et à la Console d'administration .

La page Administrateurs affiche une liste d'utilisateurs pour le domaine sélectionné. Pour chaque utilisateur, les informations suivantes sont fournies :

ID utilisateur

Nom utilisateur.

Liste des groupes

Groupes auxquels l'utilisateur appartient.

La page Administrateurs contient également les options suivantes.

Nouveau

Bouton permettant de créer un utilisateur.

Supprimer

Bouton permettant de supprimer les utilisateurs sélectionnés.
