Recharger une application déployée

Le rechargement d'une application redémarre simplement l’application sans la redéployer tout en conservant les paramètres et les options avec lesquels l’application a été déployée.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s’affiche.

  2. Dans le tableau Applications déployées, cliquez sur Recharger sur la ligne de l’application que vous redéployez.

    Le nom de l'application peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

    La page Applications s’affiche à nouveau, avec un message indiquant si le rechargement a réussi.
