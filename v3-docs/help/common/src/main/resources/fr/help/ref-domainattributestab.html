Attributs de domaine

La page Attributs de domaine permet de définir les attributs de domaine avancés de GlassFish Server.

Elle contient les options suivantes :

Racine application

Chemin complet du répertoire où résident les applications déployées. La valeur par défaut est domain-dir/applications.

Racine journal

Identifie l'emplacement de stockage des fichiers journaux du serveur. La valeur par défaut est domain-dir/logs.

Environnement local

Si ce champ n'est pas renseigné, l'environnement local par défaut de l'hôte est utilisé.

Un environnement local est un identifiant qui spécifie une combinaison particulière de langue et de région. Par exemple, l'environnement local pour l'anglais des Etats-Unis est en_US et pour le japonais, ja_JP. Pour utiliser un environnement local non anglophone, GlassFish Server doit être localisé, c'est-à-dire adapté de l'anglais en une autre langue.
