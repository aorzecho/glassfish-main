Pour changer le mot de passe administrateur

  1. Dans l’arborescence de navigation, sélectionnez le nœud GlassFish Server.

    La page Informations générales s’affiche.

  2. Sélectionnez l’onglet Mot de passe de l’administrateur.

    La page Nouveau mot de passe administrateur s’ouvre.

    La page affiche l’ID utilisateur et le nom du groupe du compte d’administrateur actuel. Par défaut, l’ID utilisateur est admin et le nom du groupe est asadmin.

  3. Dans le champ Nouveau mot de passe, saisissez le nouveau mot de passe pour le compte administrateur actuel.
  4. Dans le champ Confirmer le nouveau mot de passe, saisissez à nouveau le nouveau mot de passe.
  5. Cliquez sur Enregistrer.
