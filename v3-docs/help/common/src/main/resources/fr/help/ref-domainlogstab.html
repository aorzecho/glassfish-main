Journaux de domaine

La page Journaux de domaine permet de collecter et de télécharger les fichiers journaux de GlassFish Server.

Elle contient les options suivantes. Vous pouvez sélectionner un cluster ou une instance de serveur dont vous voulez collecter les journaux.

Cluster

Liste déroulante des clusters, s'il en existe.

Instance

Liste déroulante des instances de serveur.

Collecter les journaux

Bouton permettant de collecter les journaux dans un fichier log-files-name.zip, où name est le nom du cluster ou de l'instance.

Le fichier log-files-name.zip est téléchargé à l'aide du mécanisme de téléchargement de votre navigateur.

Tâches connexes

  • Collecter et télécharger des journaux de domaine

  • Afficher les fichiers journaux du serveur DAS

Sous-commandes asadmin équivalentes

collect-log-files(1)
