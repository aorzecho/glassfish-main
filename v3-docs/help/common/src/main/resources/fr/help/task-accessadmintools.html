Autoriser l'accès aux outils d'administration

Seuls les utilisateurs du groupe asadmin ont accès à la Console d'administration et à l'utilitaire de ligne de commande asadmin. Pour autoriser l'accès à ces outils d'administration, ajoutez les utilisateurs au groupe asadmin dans le domaine admin-realm.

  1. Sous le noeud Configurations, développez la configuration de d'instance de serveur ou de cluster pour laquelle vous souhaitez configurer les privilèges d'accès d'administration.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud admin-realm.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs apparaît.

    Après l'installation, le nom utilisateur et le mot de passe de l'administrateur entrés lors de l'installation figurent dans un fichier nommé admin-keyfile. Par défaut, cet utilisateur appartient au groupe asadmin, ce qui lui confère le droit de modifier GlassFish Server. N'affectez des utilisateurs à ce groupe que si vous voulez leur accorder des privilèges d'administrateur pour l'instance ou le cluster GlassFish Server sélectionné.

    Si vous ajoutez des utilisateurs au domaine admin-realm, mais que vous les affectez à un autre groupe que asadmin, les informations utilisateur seront toujours écrites dans le fichier nommé admin-keyfile, mais l'utilisateur n'aura pas accès aux outils d'administration ni aux applications du domaine file.

  6. Sur la page Administrateurs, cliquez sur Nouveau pour ajouter un utilisateur au domaine admin-realm.

    La page Nouvel utilisateur du domaine de fichiers apparaît.

  7. Dans le champ ID utilisateur, saisissez le nom du nouvel utilisateur.

  8. Dans le champ Liste des groupes, saisissez le nom des groupes auxquels l'utilisateur appartient.

    Pour autoriser un utilisateur à modifier GlassFish Server, ajoutez le groupe asadmin à la liste des groupes.

  9. Dans les champs Nouveau mot de passe et Confirmer le mot de passe, saisissez le mot de passe du nouvel utilisateur.

  10. Cliquez sur OK pour ajouter cet utilisateur au domaine admin-realm ou sur Annuler pour quitter sans enregistrer.
