Mises à jour disponibles

La page Mises à jour disponibles contient des informations sur les dernières versions des composants de l'image d'installation déjà installés.

Pour installer les mises à jour disponibles, commencez par arrêter le domaine GlassFish Server. Utilisez ensuite l'une des commandes suivantes :

  • glassfish3/bin/updatetool, qui démarre l'utilitaire graphique de l'outil de mise à jour

  • glassfish3/bin/pkg, version ligne de commande de l'outil de mise à jour

Pour plus d'informations sur l'utilisation de la commande pkg, reportez-vous au chapitre 11, "Extending and Updating GlassFish Server", dans le manuel Oracle GlassFish Server Administration Guide. L'utilitaire de l'outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé l'installation des mises à jour.

La page Mises à jour disponibles contient les options suivantes :

Chemin d'installation

Chemin (dans le système de fichiers) du répertoire racine ou du répertoire supérieur de l'image d'installation.

Mises à jour disponibles

Pour chaque composant pour lequel une version plus récente est disponible, cette table fournit les informations suivantes.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel auquel le composant est associé (par exemple, serveurs d'applications, outils système et bases de données). La catégorie est affectée au composant par le fabricant au moment de la création du composant.

Version installée

Numéro de version du package contenant le composant en cours.

Nouvelle version

Numéro de version de la mise à jour disponible.

Modifier

Différence de taille entre la version installée et la mise à jour disponible.

Source

Nom du Référentiel réseau à partir duquel le composant peut être obtenu.

Tâches connexes

Afficher les mises à jour disponibles pour les composants
