Mises à jour disponibles

La page Mises à jour disponibles contient des informations sur les dernières versions des composants de l’image d’installation déjà installés.

Pour installer les mises à jour disponibles, commencez par le domaine GlassFish Server. Utilisez ensuite l’une des commandes suivantes :

  • glassfish3/bin/updatetool, qui démarre l’utilitaire graphique de l’outil de mise à jour

  • glassfish3/bin/pkg, version ligne de commande de l’outil de mise à jour

Pour de plus amples informations sur l'utilisation de la commande pkg, reportez-vous au Chapitre 11, Extending and Updating GlassFish Server du Oracle GlassFish Server 3.1 Administration Guide. L’utilitaire Outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé l’installation des mises à jour.

La page Mises à jour disponibles contient les options suivantes :

Chemin d’installation

Chemin du système de fichiers du répertoire root ou du répertoire supérieur de l’image d’installation.

Available Updates

Pour chaque composant pour lequel une version plus récente est disponible, le tableau fournit les informations suivantes.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel à laquelle le composant est associé, comme par exemple serveurs d’application, outils système et bases de données. La catégorie est affectée au composant par le fabricant du composant au moment de sa création.

Version installée

Numéro de version du package dans lequel le composant actuel est fourni.

Nouvelle version

Numéro de version de la mise à jour disponible.

Modifier

Différence de taille entre la version installée et la mise à jour disponible.

Source

Nom du Référentiel réseau à partir duquel le composant peut être obtenu.

Tâches connexes

Affichage des mises à jour disponibles pour les composants

Commandes asadmin connexes

Aucune.
