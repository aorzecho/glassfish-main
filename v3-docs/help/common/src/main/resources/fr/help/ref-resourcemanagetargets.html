Gérer les cibles des ressources

La page Gérer les cibles des ressources permet de modifier les clusters et les instances de serveur autonomes cible sur lesquels une ressource peut être activée. Cela n'est possible que sur les cibles dans la colonne Cibles sélectionnées.

La page Gérer les cibles des ressources contient les informations suivantes :

Cibles disponibles

Clusters et instances sur lesquels la ressource ne peut pas être activée.

Cibles sélectionnées

Clusters et instances sur lesquels la ressource peut être activée.

Ajouter

Bouton permettant de déplacer la cible sélectionnée de la colonne Cibles disponibles vers la liste Cibles sélectionnées.

Tout ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers la colonne Cibles sélectionnées.

Enlever

Bouton permettant de déplacer la cible sélectionnée de la colonne Cibles sélectionnées vers la liste Cibles disponibles.

Tout enlever

Bouton permettant de déplacer toutes les cibles sélectionnées vers la colonne Cibles disponibles.
