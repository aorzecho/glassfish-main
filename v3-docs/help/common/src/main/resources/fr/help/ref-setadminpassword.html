Nouveau mot de passe de l’administrateur

Utilisez la page Nouveau mot de passe administrateur pour modifier le mot de passe du compte administrateur actuel.

La page Nouveau mot de passe administrateur contient les options suivantes.

ID utilisateur

ID utilisateur de l’administrateur actuel. Ce champ est en lecture seule.

Liste des groupes

Groupe auquel l’administrateur appartient. Ce champ est en lecture seule. La valeur est asadmin.

Nouveau mot de passe

Nouveau mot de passe pour l’administrateur.

Confirmer le mot de passe

Nouveau mot de passe pour l’administrateur.

  • Pour changer le mot de passe administrateur
