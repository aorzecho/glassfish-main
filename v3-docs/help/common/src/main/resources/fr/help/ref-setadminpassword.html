Nouveau mot de passe de l'administrateur

Utilisez la page Nouveau mot de passe de l'administrateur pour modifier le mot de passe du compte administrateur en cours.

Remarque:

Si l'administration sécurisée est activée, vous ne pouvez pas changer le mot de passe pour une valeur nulle.

La page Nouveau mot de passe de l'administrateur contient les options suivantes :

ID utilisateur

ID utilisateur de l'administrateur en cours. Ce champ est en lecture seule.

Liste des groupes

Groupe auquel l'administrateur appartient. Ce champ est en lecture seule. La valeur est asadmin.

Nouveau mot de passe

Nouveau mot de passe de l'administrateur.

Confirmer le nouveau mot de passe

Nouveau mot de passe de l'administrateur.

Tâche connexe

Modifier le mot de passe administrateur

Commande asadmin connexe
