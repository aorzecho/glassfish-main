Modifier les informations relatives aux utilisateurs du domaine admin-realm

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez gérer les utilisateurs du domaine admin-realm.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud admin-realm.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs apparaît.

  6. Sur la page Administrateurs, dans la colonne ID utilisateur de la table, cliquez sur le nom de l'utilisateur à modifier.

    La page Modifier l'utilisateur du domaine de fichiers apparaît.

  7. Dans le champ Liste de groupes, modifiez les groupes auxquels l'utilisateur appartient.

    Utilisez des virgules pour séparer les noms des groupes. Vous n'avez pas à définir ces groupes ailleurs.

  8. Saisissez un nouveau mot de passe dans les champs Nouveau mot de passe et Confirmer nouveau mot de passe pour modifier le mot de passe de l'utilisateur.

  9. Cliquez sur Enregistrer.
