Supprimer une configuration de sécurité des messages

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Développez le noeud Sécurité.

  3. Sélectionnez le noeud Sécurité des messages.

    La page Configurations de sécurité des messages apparaît.

  4. Sur la page Configurations de sécurité des messages, cochez la case située sur la gauche de la configuration de sécurité des messages à supprimer.

    Remarque:

    Il n'est pas possible de créer ou supprimer les fournisseurs de sécurité des messages pour la configuration par défaut server-config. Les seuls fournisseurs de sécurité des messages pris en charge pour la configuration par défaut server-config sont SOAP et HttpServlet. Comme ces deux fournisseurs de message sont déjà créés par défaut lors de l'installation de GlassFish Server, le bouton Supprimer ne figure pas sur la page Configuration de la sécurité des messages pour la configuration par défaut server-config.

  5. Cliquez sur Supprimer.
