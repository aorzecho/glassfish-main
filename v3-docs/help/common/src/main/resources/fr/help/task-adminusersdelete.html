Supprimer un utilisateur du domaine admin-realm

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez gérer les utilisateurs du domaine admin-realm.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud admin-realm.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs apparaît.

  6. Sur la page Administrateurs, cochez la case à gauche du nom de l'utilisateur à supprimer.

  7. Cliquez sur Supprimer.

    Vous êtes invité à confirmer la suppression.

  8. Cliquez sur OK pour supprimer l'utilisateur.
