Suppression d’un utilisateur du domaine Admin

  1. Sous le nœud Configurations, développez la configuration de l'instance de serveur ou de cluster pour laquelle vous souhaitez gérer les utilisateurs du domaine Admin.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.
  4. Sélectionnez le nœud admin-realm.

    La page Modification du domaine s’ouvre.

  5. Sur la page Modification du domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs s’ouvre.

  6. Sur la page Administrateurs, cochez la case à gauche du nom de l’utilisateur à supprimer.
  7. Cliquez sur Supprimer.

    Vous êtes invité à confirmer la suppression.

  8. Cliquez sur OK pour supprimer l'utilisateur.
