Afficher les cibles de l’application

  1. Dans l’arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s’affiche.

  2. Sélectionnez l'application pour laquelle vous souhaitez afficher les cibles.

    Le nom de l'application peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

    La page de l'application s'ouvre.

  3. Sélectionnez l’onglet Cible.

    La page Cibles de l'application s'ouvre et les cibles actuellement spécifiées pour l’application s'affichent.

  4. Pour activer ou désactiver une application sur une instance ou un cluster cible spécifique, cochez la case située à côté de la cible et sélectionnez Activer ou Désactiver dans le menu déroulant Plus d'actions.
  5. Pour permettre l’équilibrage de charge d’une cible sur une instance ou un cluster cible spécifique, cochez la case située à côté de la cible et sélectionnez Equilibreur de charge activé ou Equilibreur de charge désactivé dans le menu déroulant Plus d'actions.
  6. Pour associer une application à des serveurs virtuels spécifiques, sélectionnez Gestion des serveurs virtuels pour l'instance ou le cluster cible de votre choix.

    La page Cibles du serveur virtuel s'ouvre.

    Pour ajouter des serveurs virtuels cibles, sélectionnez-les dans la liste Cibles disponibles et déplacez-les vers la liste Cibles sélectionnées à l'aide du bouton Ajouter.

    Par défaut, le module ou l'application est associé au serveur virtuel par défaut, nommé server. Utilisez le bouton Tout Ajouter pour ajouter toutes les cibles. Utilisez les boutons Supprimer et Tout Supprimer pour supprimer des cibles de la liste Cibles sélectionnées.

    Cliquez sur Enregistrer pour revenir à la page Cibles de l'application.

  7. Pour ajouter ou supprimer des instances ou clusters cibles pour l'application, sélectionnez Gestion des cibles.

    La page Gestion des cibles s’ouvre.

    Pour ajouter des cibles, sélectionnez-les dans la liste Cibles disponibles et déplacez-les vers la liste Cibles sélectionnées à l'aide du bouton Ajouter.

    Utilisez le bouton Tout Ajouter pour ajouter toutes les cibles. Utilisez les boutons Supprimer et Tout Supprimer pour supprimer des cibles de la liste Cibles sélectionnées.

    Cliquez sur Enregistrer pour revenir à la page Cibles de l'application.

  8. Cliquez sur Enregistrer.

    L'application est désormais disponible dans la liste révisée des cibles.
