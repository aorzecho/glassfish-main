Lancer un client d'application

Avant de commencer

Assurez-vous que le client d'application à lancer est activé. Il doit s'agir d'un client Web qui peut être exécuté avec Java Web Start.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Applications.

    La page Applications apparaît.

  2. Dans la table Applications déployées, cliquez sur le nom du client d'application à lancer.

    Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à "Module and Application Versions" dans le manuel Oracle GlassFish Server Application Deployment Guide.

    La page Modifier une application apparaît.

  3. Vérifiez que la case Java Web Start est cochée.

    Java Web Start doit être activé pour pouvoir lancer un client d'application. Si la case Java Web Start n'est pas cochée, sélectionnez-la, puis cliquez sur Enregistrer.

  4. Dans la table Modules et composants, cliquez sur Lancer sous la colonne Action.

    La page de lancement de client d'application apparaît.

  5. Sur cette page, saisissez les arguments à transmettre au client d'application.

  6. Cliquez sur Lancer.
