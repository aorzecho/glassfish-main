Lancer un client d'application

Avant de commencer

Assurez-vous que le client d’application que vous voulez lancer est activé. Le client doit être un client Web qui peut être exécuté avec Java Web Start.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s’affiche.

  2. Dans le tableau Applications déployées, cliquez sur le nom du client d’application que vous souhaitez lancer.

    Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

    La page Modification de l’application s’affiche.

  3. Vérifiez que la case à cocher Java Web Start est activée.

    Java Web Start doit être activé pour pouvoir lancer un client d'application. Si la case à cocher Java Web Start n'est pas activée, sélectionnez-la, puis cliquez sur Enregistrer.

  4. Dans le tableau Modules et composants, cliquez sur Lancer sous la colonne Action.

    La page de lancement de client d'application s'ouvre.

  5. Sur cette page, saisissez les arguments à transmettre au client d'application.
  6. Cliquez sur Lancer.
