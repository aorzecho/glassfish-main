Modifier l'alias du mot de passe

Utilisez la page Modifier l'alias du mot de passe pour modifier le mot de passe associé à un alias de mot de passe existant.

La page Modifier l'alias du mot de passe contient les options suivantes :

Nom d'alias

Nom de l'alias. Cette option n'est pas modifiable.

Mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme chiffrée.

Confirmez le mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme chiffrée. Les valeurs Mot de passe et Confirmer le mot de passe sont comparées pour s'assurer que le mot de passe a été saisi sans erreur.
