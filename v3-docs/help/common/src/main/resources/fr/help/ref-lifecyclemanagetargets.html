Gestion des cibles

Utilisez la page Gérer les cibles des ressources pour modifier les clusters et instances de serveur autonomes cibles sur lesquels un module de cycle de vie peut être activé. Le module de cycle de vie peut être activé uniquement sur les cibles dans la colonne Cibles sélectionnées.

La page Gérer les cibles des ressources contient les informations suivantes.

Cibles disponibles

Clusters et instances sur lesquels le module de cycle de vie ne peut pas être activé.

Cibles sélectionnées

Clusters et instances sur lesquels le module de cycle de vie peut être activé.

Ajouter

Bouton permettant de déplacer la cible sélectionnée des cibles disponibles vers les cibles sélectionnées.

Tout Ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers les cibles sélectionnées.

Supprimer

Bouton permettant de déplacer la cible sélectionnée des cibles sélectionnées vers les cibles disponibles.

Tout Supprimer

Bouton permettant de déplacer toutes les cibles sélectionnées vers les cibles disponibles.
