Informations relatives à un point d’extrémité de service Web

Utilisez la page Informations relatives aux extrémités des services Web pour afficher les informations sur les extrémités des services Web.

La page Informations relatives aux extrémité des services Web contient les informations suivantes.

Nom de l’application

Nom de l’application déployée. Cliquez sur ce lien ou sur le bouton Précédent pour revenir à la page Modification d’une application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

Testeur

Le lien Testeur s’affiche uniquement si le type de déploiement de l’application n'est pas RI (par exemple, s’il est 109). Cliquez sur ce lien pour afficher une page qui vous permet de fournir des paramètres pour tester les méthodes du service Web.

WSDL

Langage Web Services Definition Language (WSDL) pour l’extrémité. Cliquez sur ce lien pour afficher le fichier WSDL dans un nouvel onglet ou dans une nouvelle fenêtre du navigateur.

Notez que certains navigateurs Web tentent d'interpréter la source WSDL lorsque vous cliquez sur le lien WSDL, affichant ainsi une page vierge. Pour de tels navigateurs, utilisez la commande d'affichage de la source du navigateur sur la page vierge pour afficher la source WSDL.
