Définir les attributs de domaine

  1. Dans l'arborescence de navigation, sélectionnez le noeud Domaine.

    La page Attributs de domaine apparaît.

  2. Dans le champ Racine application, saisissez le chemin complet du répertoire dans lequel les applications déployées résident.

    La valeur par défaut est domain-dir/applications.

  3. Dans le champ Racine journal, saisissez le chemin complet du répertoire dans lequel les fichiers journaux du serveur sont conservés.

    La valeur par défaut est domain-dir/logs.

  4. Pour utiliser l'environnement local par défaut de l'hôte, ne renseignez pas le champ Environnement local.

    Un environnement local est un identifiant qui spécifie une combinaison particulière de langue et de région. Par exemple, l'environnement local pour l'anglais des Etats-Unis est en_US et pour le japonais, ja_JP. Pour utiliser un environnement local non anglophone, GlassFish Server doit être localisé, c'est-à-dire adapté de l'anglais en une autre langue.

  5. Cliquez sur Enregistrer.

  6. Redémarrez le serveur.
