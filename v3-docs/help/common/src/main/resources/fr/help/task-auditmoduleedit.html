Pour modifier un module d’audit

Les modules d’audit ne sont pas activés par défaut. Pour plus d’informations sur l’activation des modules d’audit, consultez Activer ou désactiver la journalisation d'audit.

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Sécurité.
  3. Sous le nœud Sécurité, sélectionnez le nœud Modules d’audit.

    La page Modules d’audit s’affiche.

  4. Sur la page Modules d’audit, cliquez sur le nom du module d’audit que vous modifiez.

    La page Modification du module d’audit s’ouvre.

  5. Sur la page Modification du module d’audit, saisissez une nouvelle valeur dans le champ Nom de classe pour modifier le nom de classe.
  6. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    Pour obtenir une description de la propriété supplémentaire disponible pour les modules d’audit, consultez Propriétés propres aux modules d'audit.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  7. Cliquez sur Enregistrer.
