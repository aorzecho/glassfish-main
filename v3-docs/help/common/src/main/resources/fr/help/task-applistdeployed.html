Afficher la liste des applications déployées

  1. Dans l'arborescence de navigation, sélectionnez le noeud Applications.

    La page Applications apparaît. La table répertorie les applications déployées.

  2. Sélectionnez un type de module dans la liste déroulante Filtres pour filtrer la liste des applications par type de module.
