Pour afficher la liste des applications déployées

  1. Dans l’arborescence de navigation, sélectionnez le nœud Applications.

    La page des applications s’affiche. Le tableau répertorie les applications déployées.

  2. Sélectionnez un type de module dans la liste déroulante des Filtres pour filtrer la liste des applications par type de module.
