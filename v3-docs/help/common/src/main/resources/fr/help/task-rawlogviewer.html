Afficher les fichiers journaux raw du serveur DAS

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, cliquez sur le bouton Afficher le journal raw.

    La fenêtre Afficher le journal raw apparaît.
