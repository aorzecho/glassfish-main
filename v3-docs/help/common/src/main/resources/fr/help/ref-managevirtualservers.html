Cibles du serveur virtuel

Utilisez la page Cibles du serveur virtuel pour modifier les serveurs virtuels cibles auxquels une application ou un module est associé.

La page Cibles du serveur virtuel contient les informations suivantes.

Cibles disponibles

Serveurs virtuels auxquels aucune application ou aucun module n'est associé.

Cibles sélectionnées

Serveurs virtuels auxquels une application ou un module est associé.

Ajouter

Bouton permettant de déplacer la cible sélectionnée des cibles disponibles vers les cibles sélectionnées.

Tout Ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers les cibles sélectionnées.

Supprimer

Bouton permettant de déplacer la cible sélectionnée des cibles sélectionnées vers les cibles disponibles.

Tout Supprimer

Bouton permettant de déplacer toutes les cibles sélectionnées vers les cibles disponibles.
