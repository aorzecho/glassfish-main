Modification d’un fournisseur JACC

La page Modification d'un fournisseur JACC vous permet de modifier un fournisseur JACC pour le cluster ou l'instance GlassFish Server sélectionné.

La page Modifier le fournisseur JACC contient les options suivantes.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom

Nom du fournisseur JACC. Ce champ est en lecture seule. Vous ne pouvez spécifier le nom qu’à la création d’un nouveau fournisseur JACC.

Configuration de stratégies

Nom complet du javax.security.jacc.PolicyConfigurationFactory.provider mettant en œuvre le javax.security.jacc.PolicyConfigurationFactory spécifique au fournisseur.

Fournisseur de stratégies

Nom complet du javax.security.jacc.policy.provider mettant en œuvre le java.security.Policy.

Autres propriétés

Autres propriétés du fournisseur JACC. Pour obtenir une description des autres propriétés disponibles pour les fournisseurs JACC, consultez la rubrique Propriétés propres aux fournisseurs JACC.
