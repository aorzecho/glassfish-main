Modifier un fournisseur JACC

La page Modifier un fournisseur JACC permet de modifier un fournisseur JACC pour le cluster ou l'instance GlassFish Server sélectionné.

Elle contient les options suivantes :

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom

Nom du fournisseur JACC. Ce champ est en lecture seule. Vous ne pouvez spécifier le nom qu'à la création du fournisseur JACC.

Configuration de règles

Nom qualifié complet de l'élément javax.security.jacc.PolicyConfigurationFactory.provider qui implémente l'élément javax.security.jacc.PolicyConfigurationFactory propre au fournisseur.

Fournisseur de règles

Nom qualifié complet de l'élément javax.security.jacc.policy.provider qui implémente l'élément java.security.Policy.

Autres propriétés

Autres propriétés du fournisseur JACC. Pour une description des propriétés supplémentaires disponibles pour les fournisseurs JACC, reportez-vous à Propriétés propres aux fournisseurs JACC.
