Paramètres du profileur JVM

Un outil de profilage génère des données servant à analyser les performances et à identifier les éventuels goulots d’étranglement.

Définissez les paramètres du profileur. Les informations saisies dépendent du profileur utilisé. Pour obtenir des exemples et des instructions, reportez-vous au Chapitre 3, Debugging Applications du Oracle GlassFish Server 3.1 Application Development Guide.
