SSL

Utilisez la page SSL pour modifier les paramètres SSL du connecteur JMX.

Les paramètres de cette page n’ont d’importance que si la sécurité est activée pour le connecteur JMX sur la page Modifier le connecteur JMX.

La page SSL contient les options suivantes.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

SSL3

Si cette case est cochée, le protocole SSL3 est activé pour le connecteur JMX. Cette option est activée par défaut.

TLS

Si cette case est cochée, le protocole TLS est activé pour le connecteur JMX. Cette option est activée par défaut.

Authentification client

Si cette case est cochée, les clients doivent s’identifier sur le serveur à chaque requête. Cette option est désactivée par défaut.

Surnom du certificat

Surnom du certificat du serveur dans la base de données des certificats ou jeton PKCS n° 11. Dans le certificat, le format du nom est nom du jeton:surnom. Même la partie nom du jeton: du nom de cet attribut est facultatif.

Key Store

Nom du fichier keystore (par exemple, keystore.jks ).

Algorithme de confiance

Nom de l’algorithme de gestion de confiance (par exemple, PKIX) à utiliser pour la validation du chemin de certification.

Longueur max. du certificat

Nombre maximum des certificats intermédiaires non auto-émis pouvant exister dans un chemin de certification. Ce champ est utilisé uniquement le champ Algorithme de confiance est défini à PKIX. Zéro (0) implique que le chemin peut contenir un seul certificat. -1 implique que la longueur du chemin est illimitée (pas de maximum). La définition d’une valeur à moins de -1 entraîne l’émission d’une exception.

Trust Store

Nom du fichier truststore (par exemple, cacerts.jks ).

Suites de chiffrement

Zone où vous pouvez ajouter ou supprimer des suites de chiffrement. Si vous n’ajoutez pas de suite de chiffrement, toutes les suites seront utilisées.
