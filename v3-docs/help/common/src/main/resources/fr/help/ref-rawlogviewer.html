Visualiseur de journal raw

La page Visualiseur de journal raw permet de visualiser les fichiers journaux raw du serveur d'administration (nommé server).

Elle contient les options suivantes :

Instance

Instance de serveur ou cluster dont les enregistrements de journal seront affichés. La seule possibilité est server.

Pour chaque message, le Visualiseur de journal raw affiche les informations suivantes.

Horodatage

Date et heure d'émission du message du journal.

Numéro d'enregistrement

Numéro du message dans le journal du serveur.

Niveau de journalisation

Niveau de journalisation du message.

Version

Version de GlassFish Server

Journaliseur

Module de journalisation qui a généré le message du journal.

Paires nom-valeur

Nom du thread et ID du message du journal.

Message

Texte bref du message du journal. Cliquez sur le lien (détails) pour afficher la version formatée du message.
