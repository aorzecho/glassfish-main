Ajouter des ressources

La page Ajouter des ressources permet de créer des ressources à partir d'un fichier glassfish-resources.xml sur les cibles indiquées.

Elle contient les informations suivantes.

Emplacement

Emplacement du fichier glassfish-resources.xml :

Fichier XML à télécharger sur le serveur

Chemin d'accès au fichier situé sur un ordinateur distant.

Fichier XML local accessible à partir de GlassFish Server

Chemin d'accès au fichier situé sur l'ordinateur du serveur d'administration de domaine (DAS).

Cible

Liste déroulante des clusters et des instances de serveur autonomes.

Tâches connexes

Ajouter des ressources configurées dans un fichier XML

Commandes asadmin connexes

add-resources(1)
