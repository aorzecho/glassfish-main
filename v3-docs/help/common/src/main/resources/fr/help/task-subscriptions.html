Afficher la page Abonnements pour GlassFish Server

  1. Dans l’arborescence de navigation, sélectionnez le nœud Tâches courantes.

    La page Tâches courantes s’affiche.

  2. Sous Forums GlassFish, cliquez sur le lien Abonnements.

    La page Web Abonnements GlassFish s’ouvre dans un onglet ou une fenêtre distincte du navigateur.
