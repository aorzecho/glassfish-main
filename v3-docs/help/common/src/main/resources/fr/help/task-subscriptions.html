Afficher la page Abonnements de GlassFish Server

  1. Dans l'arborescence de navigation, sélectionnez le noeud Tâches courantes.

    La page Tâches courantes apparaît.

  2. Sous Actualités de GlassFish, cliquez sur le lien Abonnements.

    La page Web Abonnements GlassFish apparaît dans un autre onglet ou une fenêtre distincte du navigateur.
