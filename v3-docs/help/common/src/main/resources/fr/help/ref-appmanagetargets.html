Gérer les cibles

La page Gérer les cibles permet de modifier les clusters et les instances de serveur autonomes cible sur lesquels une application peut être activée. L'application peut être activée uniquement sur les cibles figurant dans la colonne Cibles sélectionnées.

La page Gérer les cibles contient les informations suivantes :

Cibles disponibles

Clusters et instances sur lesquels l'application n'est pas déployée.

Cibles sélectionnées

Clusters et instances sur lesquels l'application est déployée.

Ajouter

Bouton permettant de déplacer la cible sélectionnée de la colonne Cibles disponibles vers la liste Cibles sélectionnées.

Tout ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers la colonne Cibles sélectionnées.

Enlever

Bouton permettant de déplacer la cible sélectionnée de la colonne Cibles sélectionnées vers la liste Cibles disponibles.

Tout enlever

Bouton permettant de déplacer toutes les cibles sélectionnées vers la colonne Cibles disponibles.
