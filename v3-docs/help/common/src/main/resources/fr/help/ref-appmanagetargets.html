Gestion des cibles

Utilisez la page Gestion des cibles pour modifier les clusters et les instances de serveur autonomes cibles sur lesquels une application peut être activée. L'application peut être activée uniquement sur les cibles dans la colonne Cibles sélectionnées.

La page Gestion des cibles contient les informations suivantes.

Cibles disponibles

Clusters et instances sur lesquels l'application n'est pas déployée.

Cibles sélectionnées

Clusters et instances sur lesquels l'application est déployée.

Ajouter

Bouton permettant de déplacer la cible sélectionnée des cibles disponibles vers les cibles sélectionnées.

Tout Ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers les cibles sélectionnées.

Supprimer

Bouton permettant de déplacer la cible sélectionnée des cibles sélectionnées vers les cibles disponibles.

Tout Supprimer

Bouton permettant de déplacer toutes les cibles sélectionnées vers les cibles disponibles.
