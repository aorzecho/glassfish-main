Ajouter des ressources configurées dans un fichier XML

  1. Dans l'arborescence de navigation, sélectionnez le noeud Ressources.

    La page Ressources apparaît.

  2. Cliquez sur le bouton Ajouter des ressources.

    La page Ajouter des ressources apparaît.

  3. Sélectionnez l'une des options suivantes pour spécifier l'emplacement du fichier glassfish-resources.xml :

    Fichier XML à télécharger sur le serveur

    Saisissez le chemin d'accès au fichier s'il est situé sur un ordinateur distant.

    Fichier XML local accessible à partir de GlassFish Server

    Saisissez le chemin d'accès au fichier s'il est situé sur l'ordinateur du serveur d'administration de domaine (DAS).

  4. Dans la liste déroulante Cible, sélectionnez le cluster ou l'instance de serveur autonome sur lequel créer les ressources.

  5. Cliquez sur OK.
