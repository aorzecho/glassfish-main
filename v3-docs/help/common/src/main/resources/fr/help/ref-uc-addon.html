Add-ons disponibles

La page Add-ons disponibles contient des informations sur les composants de l’image d’installation non encore installés.

La page Add-ons disponibles contient les options suivantes :

Chemin d’installation

Chemin du système de fichiers du répertoire root ou du répertoire supérieur de l’image d’installation.

Installer

Bouton permettant d’installer un ou plusieurs composants add-on sélectionnés.

Add-ons disponibles

Pour chaque composant de l’image d’installation qui n’est pas encore installé, ce tableau fournit les informations ci-dessous.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel à laquelle le composant est associé, comme par exemple serveurs d’application, outils système et bases de données. La catégorie est affectée au composant par le fabricant du composant au moment de sa création.

Version

Numéro de version du package comprenant le composant.

Taille de l’installation

Espace disque utilisé pour l’installation du composant.

Source

Nom du Référentiel réseau à partir duquel le composant peut être obtenu.

Tâches connexes

Installer des composants add-ons

Commandes asadmin connexes

Aucune.
