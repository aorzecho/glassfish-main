Modifier l'utilisateur du domaine de fichiers

La page Modifier l'utilisateur du domaine de fichiers permet de modifier le mot de passe de l'utilisateur pour le domaine en cours.

Remarque:

Si l'administration sécurisée est activée, vous ne pouvez pas changer le mot de passe d'un utilisateur admin-realm en une valeur vide.

La page Modifier l'utilisateur du domaine de fichiers contient les options suivantes :

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom de domaine

Nom du domaine actuellement sélectionné. Ce champ est en lecture seule.

ID utilisateur

Nom du compte que vous modifiez. Ce champ est en lecture seule.

Liste des groupes

Groupes auxquels cet utilisateur appartient. Séparez les noms de groupe par une virgule.

Pour les utilisateurs admin-realm, ce champ est en lecture seule, et le groupe est asadmin.

Nouveau mot de passe

Nouveau mot de passe de cet utilisateur.

Confirmer le nouveau mot de passe

Nouveau mot de passe de cet utilisateur.
