Afficher la page À la une de GlassFish pour GlassFish Server

La page Forums GlassFish fournit un ensemble de liens permettant d’accéder à des informations supplémentaires sur Oracle GlassFish Server, telles que des bulletins d’information, des articles, des conseils techniques, des forums de discussion et des blogs sur le sujet.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Tâches courantes.

    La page Tâches courantes s’affiche.

  2. Sous À la une de GlassFish, cliquez sur le lien Forums GlassFish.

    La page Forums GlassFish s’ouvre dans un onglet ou une fenêtre distincte du navigateur.
