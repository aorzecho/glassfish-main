Afficher la page Actualités de GlassFish pour GlassFish Server

La page Actualités de GlassFish fournit un ensemble de liens vers des informations supplémentaires concernant Oracle GlassFish Server, telles que des bulletins d'information, des articles, des conseils techniques, des forums de discussion et des blogs sur le sujet.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Tâches courantes.

    La page Tâches courantes apparaît.

  2. Sous Actualités de GlassFish, cliquez sur Actualités de GlassFish.

    La page Actualités de GlassFish apparaît dans un autre onglet ou une fenêtre distincte du navigateur.
