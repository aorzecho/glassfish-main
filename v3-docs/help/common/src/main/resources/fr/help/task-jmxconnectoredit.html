Pour modifier le connecteur JMX

Utilisez le Service d’administration pour configurer un connecteur JMX distant conforme au JSR-160.

Le connecteur JMX distant conforme au JSR-160 gère la communication entre les clients JMX, comme par exemple JConsole et le service JMX, à des fins de gestion de domaine.

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Service d’administration.

    La page Modifier le connecteur JMX s’ouvre.

  3. Cochez la case Sécurité activée pour que la communication JMX soit cryptée.

    Cette option est désactivée par défaut.

  4. Dans le champ Adresse, saisissez l’adresse IP du service de nommage où le stub du serveur du connecteur JMX est enregistré.

    Ce n’est pas le port du socket de serveur qui effectue la véritable communication JMX. C’est l’adresse de l’interface réseau depuis laquelle le registre RMI est lancé. Si votre système est doté de plusieurs interfaces réseau, modifiez cette valeur pour qu’une seule interface soit sélectionnée. La valeur par défaut est 0.0.0.0.

  5. Dans le champ Port, saisissez le numéro de port sur lequel le service de nommage (registre RMI) écoute les connexions client RMI.

    La seule utilisation de ce service de nommage consiste à télécharger les stubs RMI. Si le port par défaut est occupé, on utilise un port libre. Les valeurs légales sont de 1 à 65535. Sous les systèmes UNIX, la création de sockets qui écoutent sur les ports 1 à 1024 nécessite des privilèges de superutilisateur. La valeur par défaut est 8686.

  6. Dans le champ Nom de domaine, saisissez le nom qui représente le domaine administratif particulier.

    Toutes les authentifications sont gérées par ce domaine. La valeur par défaut est admin-realm .

  7. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    Le GlassFish Server spécifie plusieurs propriétés par défaut.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  8. Cliquez sur Enregistrer.
  9. Si vous avez coché la case Sécurité activée, suivez les étapes de Pour modifier les paramètres SSL du connecteur JMX.
