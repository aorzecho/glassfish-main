Nouvel alias de mot de passe

Utilisez la page Nouvel alias de mot de passe pour créer un nouvel alias de mot de passe à utiliser dans un fichier de mot de passe ou dans le fichier de configuration de domaine.

La page Nouvel alias de mot de passe contient les options suivantes :

Nom d'alias

Nom de l'alias. Ce nom doit être unique parmi tous les alias de mot de passe dans le domaine.

Mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme chiffrée.

Confirmez le mot de passe

Mot de passe stocké par l'alias de mot de passe sous forme chiffrée. Les valeurs Mot de passe et Confirmer le mot de passe sont comparées pour s'assurer que le mot de passe a été saisi sans erreur.
