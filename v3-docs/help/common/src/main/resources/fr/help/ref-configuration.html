Configuration

Utilisez le noeud de configuration d'une instance ou d'un cluster GlassFish Server pour configurer les composants suivants :

  • Paramètres généraux JVM

  • Paramètres du journaliseur

  • Conteneur Web

  • Conteneur EJB

  • Java Message Service

  • Sécurité

  • Service de disponibilité

  • Service de transaction

  • Service HTTP

  • Serveurs virtuels

  • Configuration réseau

  • Pools de threads

  • ORB

  • Service d'administration

  • Surveillance

  • Service de gestion de groupes

  • Propriétés système
