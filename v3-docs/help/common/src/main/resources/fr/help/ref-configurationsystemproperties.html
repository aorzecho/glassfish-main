Propriétés système

La page Propriétés système affiche la liste des propriétés système Java définies pour la configuration nommée sélectionnée. Les propriétés système Java sont transmises au lanceur d'application Java via son option -D au démarrage de GlassFish Server.

Ces propriétés définissent les valeurs par défaut des paramètres de port pour toutes les instances GlassFish Server faisant référence à la configuration.

Pour plus d'informations sur les paramètres de port prédéfinis dans une configuration, reportez-vous à Propriétés système prédéfinies.

Pour chaque propriété, les informations suivantes sont affichées :

Nom de la variable d'instance

Nom de la propriété système.

Valeur par défaut

Valeur définie pour la propriété dans la configuration nommée à laquelle l'instance fait référence. Ce champ est en lecture seule.

Valeurs de l'instance

Lien vers la page Valeurs de l'instance de la propriété.

La table Autres propriétés contient également les options suivantes :

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Le fait de cliquer sur ce bouton permet d'ajouter une ligne à la table Autres propriétés.

Supprimer des propriétés

Bouton permettant de supprimer les propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est configurée, n'est pas définie.

La page Propriétés système contient également les options suivantes :

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Reconfiguration dynamique

Si cette option est activée, les modifications apportées à la configuration sont appliquées aux instances faisant référence à la configuration sans nécessiter le redémarrage des instances.

Clusters et/ou instances utilisant cette configuration

Liste des clusters et instances faisant référence à la configuration en indiquant si chaque cluster ou instance est en cours d'exécution :

  • Si le cluster ou l'instance est en cours d'exécution, une coche (✓) apparaît.

  • Si le cluster ou l'instance est arrêté, un point d'exclamation (!) apparaît.
