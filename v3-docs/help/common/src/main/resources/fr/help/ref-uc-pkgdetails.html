Détails du composant

La page Détails du composant fournit des informations détaillées à propos d'un composant.

Elle contient les options suivantes :

Précédent

Ce bouton permet de revenir à la page sur laquelle vous avez sélectionné le composant dont vous consultez les détails.

Nom du package

Nom du package comprenant le composant.

Identificateur unique

Identificateur qualifié complet du composant tel qu'il existe dans le référentiel.

Catégorie

Catégorie du logiciel auquel le composant est associé (par exemple, serveurs d'applications, outils système et bases de données). La catégorie est affectée au composant par le fabricant au moment de la création du composant.

Version

Numéro de version du package comprenant le composant.

Date de publication

Date et heure auxquelles cette version du composant a été publiée dans le référentiel. Cette date ne correspond pas à la date et à l'heure auxquelles le composant a été installé dans l'image d'installation.

Taille après installation

Espace disque utilisé pour l'installation du composant.

Source

Pour un composant installé, nom du Référentiel réseau à partir duquel le composant a été obtenu.

Pour un composant d'extension ou une mise à jour, nom du référentiel réseau à partir duquel le composant a été obtenu.

Description

Présentation du composant.

Tâches connexes

Afficher les informations détaillées d'un composant

Commandes asadmin connexes

Aucune.
