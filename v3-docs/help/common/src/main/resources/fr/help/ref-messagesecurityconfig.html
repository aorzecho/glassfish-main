Configuration de la sécurité des messages

Utilisez la page Configurations de la sécurité des messages pour afficher et modifier les configurations disponibles pour les fournisseurs de sécurité des messages.

Pour configurer la sécurité des messages, vous devez comprendre les bases des concepts de sécurité et des services Web. Pour plus d'informations, reportez-vous au Chapitre 3, Administering Message Security du Oracle GlassFish Server 3.1 Security Guide et au Chapitre 6, Administering Web Applications du Oracle GlassFish Server 3.1 Administration Guide.

Le GlassFish Server implémente la sécurité des messages via les fournisseurs de sécurité des messages intégrés à sa couche de traitement SOAP. Les fournisseurs de sécurité des messages dépendent des autres utilitaires de sécurité du GlassFish Server.

Pour chaque configuration, les informations suivantes sont fournies.

Couche d’authentification

Couche message à laquelle l’authentification est effectuée. Par défaut, une configuration SOAP est fournie. Il est également possible de fournir une configuration HttpServlet.

Fournisseur par défaut

Fournisseur de serveur par défaut pour cette configuration. Par défaut, aucun fournisseur de serveur par défaut n’est configuré.

Fournisseur client par défaut

Fournisseur client par défaut pour cette configuration. Par défaut, aucun fournisseur client par défaut n’est configuré.

Le tableau Configurations de la sécurité des messages contient également les options suivantes.

Nouveau

Bouton permettant de créer une configuration.

Supprimer

Bouton permettant de supprimer une ou plusieurs configurations sélectionnées.
