Configurations de la sécurité des messages

La page Configurations de la sécurité des messages permet d'afficher et de modifier les configurations disponibles pour les fournisseurs de sécurité des messages.

Pour configurer la sécurité des messages, vous devez comprendre les bases des concepts de sécurité et des services Web. Pour plus d'informations, reportez-vous au chapitre 3, "Administering Message Security", du manuel Oracle GlassFish Server Security Guide et au chapitre 6, "Administering Web Applications", du manuel Oracle GlassFish Server Administration Guide.

GlassFish Server implémente la sécurité des messages via les fournisseurs de sécurité des messages intégrés à sa couche de traitement SOAP. Les fournisseurs de sécurité des messages dépendent des autres utilitaires de sécurité de GlassFish Server.

Pour chaque configuration, les informations suivantes sont fournies :

Couche d'authentification

Couche de messages au niveau de laquelle l'authentification est effectuée. Par défaut, une configuration SOAP est fournie. Il est également possible de fournir une configuration HttpServlet.

Fournisseur par défaut

Fournisseur de serveur par défaut pour cette configuration. Aucun fournisseur de serveur par défaut n'est configuré.

Fournisseur client par défaut

Fournisseur client par défaut pour cette configuration. Aucun fournisseur client par défaut n'est configuré.

La table Configuration de la sécurité des messages contient également les options suivantes :

Nouveau

Bouton permettant de créer une configuration.

Supprimer

Bouton permettant de supprimer les configurations sélectionnées.
