Ajouter un utilisateur au domaine admin-realm

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez gérer les utilisateurs du domaine admin-realm.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud admin-realm.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs apparaît.

  6. Sur la page Administrateurs, cliquez sur Nouveau.

    La page Nouvel utilisateur du domaine de fichiers apparaît.

  7. Dans le champ ID utilisateur, saisissez le nom de l'utilisateur.

  8. Dans le champ Liste des groupes, saisissez une liste des groupes auxquels l'utilisateur appartient.

    Utilisez des virgules pour séparer les noms des groupes. Vous n'avez pas à définir ces groupes ailleurs.

  9. Dans le champ Mot de passe, saisissez un mot de passe unique.

  10. Dans le champ Confirmer le nouveau mot de passe, saisissez à nouveau le mot de passe utilisateur.

  11. Cliquez sur OK.
