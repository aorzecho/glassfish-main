Cibles de ressource

Utilisez la page Cibles de ressources pour afficher les clusters et instances de serveur autonomes cibles sur lesquels la ressource peut être activée.

La page Cibles de ressource contient les informations suivantes pour chaque cible.

Nom de cible

Nom du cluster ou de l'instance de serveur autonome.

Activée

Affiche une coche si la ressource est activée sur la cible ou un X si la ressource est désactivée.

Le tableau Cibles contient également les options suivantes.

Gestion des cibles

Bouton permettant de gérer les cibles de ressource.

Activer

Bouton permettant d'activer la ressource sur la cible sélectionnée.

Désactiver

Action permettant de désactiver la ressource sur la cible sélectionnée.
