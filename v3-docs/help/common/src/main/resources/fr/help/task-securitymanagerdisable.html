Désactiver le gestionnaire de sécurité

La désactivation du gestionnaire de sécurité d’GlassFish Server peut améliorer les performances de certains types d’applications. Les fonctions d’autorisation et d’authentification de la plate-forme Java pourront toujours être utilisées, même en cas de désactivation du gestionnaire de sécurité. Vous pouvez désactiver le gestionnaire de sécurité dans un environnement de développement. Toutefois, ne le désactivez pas dans un environnement de production.

  1. Dans l’arborescence de navigation, sélectionnez le noeud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Paramètres JVM.

    La page Paramètres JVM s’ouvre.

  3. Dans la page Paramètres généraux JVM, cliquez sur l’onglet Options JVM.

    La page Options JVM s’ouvre.

  4. Cochez la case située à gauche du champ Valeur contenant l’option -Djava.security.policy.
  5. Cliquez sur Supprimer.
  6. Cliquez sur Enregistrer.
  7. Redémarrez le serveur.
