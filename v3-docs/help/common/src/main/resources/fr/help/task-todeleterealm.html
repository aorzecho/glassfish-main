Supprimer un domaine

  1. Sous le nœud Configurations, développez la configuration de l'instance de serveur ou du cluster pour laquelle vous souhaitez configurer des domaines.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.

    La page Domaines s’affiche.

  4. Sur la page Domaines, cochez la case située à gauche du nom du domaine à supprimer.
  5. Cliquez sur Supprimer.
