Supprimer un domaine

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez configurer des domaines.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

    La page Domaines apparaît.

  4. Sur la page Domaines, cochez la case située à gauche du nom du domaine à supprimer.

  5. Cliquez sur Supprimer.
