Créer un domaine

Le GlassFish Server est préconfiguré avec trois domaines : file, certificate et admin-realm. Il est également possible de créer d’autres domaines ldap, JDBC, solaris, PAM et des domaines personnalisés. Un serveur comprend généralement un domaine de chaque type, tandis que le GlassFish Server comprend deux domaines de fichiers : file et admin-realm. Il s’agit de deux domaines du même type, mais ayant deux utilisations différentes. Il est également possible d’avoir une base de données de certificats différente pour chaque serveur virtuel du système.

  1. Sous le nœud Configurations, développez la configuration de l'instance de serveur ou du cluster pour laquelle vous souhaitez configurer des domaines.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.

    La page Domaines s’affiche.

  4. Sur la page Domaines, cliquez sur Nouveau.

    La page Nouveau domaine s’affiche.

  5. Dans le champ Nom, attribuez un nom au domaine.
  6. Pour spécifier le nom de classe pour le domaine, procédez de l’une des manières suivantes :
    • Dans la liste déroulante Nom de la classe, sélectionnez le nom de classe du domaine à créer.

      Les choix possibles sont affichés dans le tableau suivant.

      Nom de domaine

      Nom de la classe

      fichier

      com.sun.enterprise.security.auth.realm.file.FileRealm

      certificat

      com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

      ldap

      com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

      JDBC

      com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

      solaris

      com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

      PAM

      com.sun.enterprise.security.auth.realm.solaris.PamRealm
    • Sélectionnez le bouton radio inférieur et saisissez le nom de classe du domaine personnalisé à créer.
  7. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    Les propriétés valides dépendent du type de domaine.

    • Propriétés spécifiques de la classe FileRealm

    • Propriétés spécifiques de la classe CertificateRealm

    • Propriétés spécifiques de la classe JDBCRealm

    • Propriétés spécifiques à la classe LDAPRealm

    • Propriétés spécifiques à la classe SolarisRealm

    • Propriétés spécifiques de la classe PamRealm

  8. Cliquez sur OK.
