Alias de mot de passe

Utilisez la page Alias de mot de passe pour créer et gérer des alias de mot de passe.

La page Alias de mot de passe affiche une liste des alias de mot de passe dans le domaine. Pour chaque alias, les informations suivantes sont affichées :

Nom

Nom assigné à l'alias de mot de passe lors de sa création. Le fait de cliquer sur le nom ouvre la page Modifier l'alias du mot de passe pour l'alias.

Le tableau Alias de mot de passe contient également les options suivantes.

Nouveau

Bouton permettant de créer un alias de mot de passe.

Supprimer

Bouton permettant de supprimer un ou plusieurs alias de mot de passe sélectionnés.

Sous-commandes asadmin équivalentes
