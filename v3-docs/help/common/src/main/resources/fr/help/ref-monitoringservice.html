Service de contrôle

La page Service de contrôle permet de configurer les options de contrôle pour les composants ou services des serveurs individuels.

Vous devez déployer une application, un composant ou un service avant d’en configurer le contrôle.

La page Service de contrôle contient les options suivantes :

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Service de contrôle

Si la case Service de contrôle activé est sélectionnée, la fonction de contrôle est activée pour GlassFish Server. Cette option est activée par défaut.

MBeans de contrôle

Si la case MBeans de contrôle activés est sélectionnée, tous les MBeans nécessaires au contrôle sont déployés sur GlassFish Server. Cette option est activée par défaut.

Si la case n’est pas cochée, vous ne pouvez pas afficher les données de contrôle sous la Console d'administration , même si la case Service de contrôle est sélectionnée. Au lieu de cela, vous devez utiliser la ligne de commande pour afficher les données de contrôle. Si vous désactivez la case à cocher, l’empreinte de mémoire de GlassFish Server est réduite.

DTrace

Si la case DTrace activé est sélectionnée et si vous avez le contrôle DTrace de Oracle Enterprise Manager, vous pouvez utiliser l’outil DTrace Solaris. Cette option est désactivée par défaut.

Pour chaque composant disponible, les informations disponibles sont fournies.

Composant

Nom du composant du GlassFish Server.

Niveau de contrôle

Niveau de contrôle défini pour le composant. Pour chaque composant, les niveaux disponibles sont BAS, HAUT et DÉSACTIVÉ. Le niveau par défaut est OFF. Dans cette version, il n’existe aucune différence entre les options BAS et HAUT.

Le tableau Paramètres du niveau de composant contient également les options suivantes.

Modifier le niveau

Bouton permettant de modifier le niveau de contrôle d’un ou plusieurs composants sélectionnés.

Niveau

Liste déroulante à partir de laquelle vous pouvez sélectionner un niveau de contrôle à appliquer à tous les composants sélectionnés.
