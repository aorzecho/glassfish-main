Afficher des informations générales sur le DAS

  1. Dans l’arborescence de navigation, sélectionnez le nœud du serveur (Admin Server).

    La page Informations générales s’affiche.

  2. Sur la page Informations générales, procédez à l’une des étapes suivantes.
    • Cliquez sur le bouton Stop pour arrêter le GlassFish Server.
    • Cliquez sur le bouton Redémarrer pour redémarrer le GlassFish Server.
    • Cliquez sur le bouton Afficher les journaux pour visualiser les fichiers journaux d'un cluster ou d'une instance GlassFish Server.

      Reportez-vous à la section Afficher les journaux du DAS pour plus de détails.

    • Cliquez sur Rapport JVM pour ouvrir une fenêtre distincte qui vous permet d’afficher des rapports sur la Java Virtual Machine, notamment un rapport de synthèse, un rapport de gestion de la mémoire, un rapport de chargement de classe et un vidage des threads en cours.
