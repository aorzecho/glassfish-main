Propriétés spécifiques de la classe PamRealm

La propriété suivante est requise pour un domaine PAM.

Contexte JAAS

Contexte JAAS (Java Authentication and Authorization Service), identificateur du module de connexion à utiliser pour ce domaine. La seule valeur valide est pamRealm.

La propriété facultative suivante est disponible pour un domaine PAM.

Assignation de groupes

Liste de noms de groupes séparés par une virgule. Tous les clients présentant un certificat valide sont affectés à ces groupes, comme par exemple employee, manager qui sont des noms de groupes d’utilisateurs.
