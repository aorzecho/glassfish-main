Concepts clés de l'Outil de mise à jour

L'Outil de mise à jour gère les installations des composants du logiciel GlassFish Server contenus dans un même répertoire du système de fichiers.

La version de la Console d'administration de l'Outil de mise à jour permet d'effectuer les opérations suivantes :

  • Installer des composants d'extension

  • Afficher des mises à jour pour les composants installés

  • Afficher les composants installés

Pour mettre à jour ou enlever des composants installés, arrêtez le domaine de GlassFish Server, puis utilisez l'une des commandes suivantes :

  • as-install-parent/bin/updatetool, qui démarre l'utilitaire graphique de l'outil de mise à jour

  • as-install-parent/bin/pkg, version ligne de commande de l'outil de mise à jour

Les concepts-clés de l'Outil de mise à jour sont décrits dans les sections suivantes.

Composant

Un composant correspond à l'unité installable que gère l'Outil de mise à jour. Un composant regroupe des fichiers qui sont installés ensemble sur le système dans une image d'installation.

Image d'installation

Une image d'installation correspond au répertoire contenant les composants du logiciel, y compris le contenu du répertoire. L'image d'installation est ajoutée à l'Outil de mise à jour suite au téléchargement et à l'installation du logiciel GlassFish Server. Pour vous permettre d'ajouter et de mettre à jour les composants du logiciel GlassFish Server, chaque image d'installation contient des informations sur l'installation des composants.

Référentiel réseau

Un référentiel réseau contient les éléments suivants :

  • Composants pouvant être ajoutés à une image d'installation

  • Mises à jour de composants existants dans une image d'installation

Une image d'installation est associée à un ou plusieurs référentiels réseaux. Chaque référentiel réseau est désigné par un nom et une URL. Le nom de référentiel est propre à chaque image d'installation. Il représente à la fois l'organisation ou le projet qui gère le référentiel et la portée bien définie du contenu. Pour en savoir plus sur les référentiels réseau préconfigurés de GlassFish Server, reportez-vous à "Preconfigured Repositories for GlassFish Server" dans le manuel Oracle GlassFish Server Administration Guide.

Composant d'extension

Un composant d'extension est un composant de l'image d'installation qui n'est pas encore installé.
