Concepts-clés de Outil de mise à jour

Outil de mise à jour gère les installations des composants du logiciel GlassFish Server contenus dans un seul répertoire système de fichiers.

La version Console d'administration de Outil de mise à jour vous permet d’effectuer les opérations suivantes :

  • Installer des composants add-on

  • Afficher des mises à jour pour les composants installés

  • Afficher les composants installés

Pour mettre à jour ou supprimer des composants installés, arrêtez le domaine de GlassFish Server, puis utilisez l’une des commandes suivantes :

  • as-install-parent/bin/updatetool, qui démarre l’utilitaire graphique de l’outil de mise à jour

  • as-install-parent/bin/pkg, version ligne de commande de l’outil de mise à jour

Les concepts-clés de Outil de mise à jour sont décrits dans les sections suivantes.

Composant

Un composant correspond à l’unité installable que gère Outil de mise à jour. Un composant regroupe des fichiers qui sont installés ensemble dans votre système dans une image d’installation.

Image d’installation

Une image d’installation correspond au répertoire contenant les composants du logiciel, y compris le contenu du répertoire. L’image d’installation est ajoutée à Outil de mise à jour suite au téléchargement et à l’installation du logiciel GlassFish Server. Pour vous aider à ajouter et mettre à jour les composants du logiciel GlassFish Server, chaque image d’installation contient des informations sur l’installation des composants.

Référentiel réseau

Un référentiel réseau contient les éléments suivants :

  • Composants pouvant être ajoutés à une image d’installation

  • Mises à jour de composants existants dans une image d’installation

Une image d’installation est associée à un ou plusieurs référentiels réseaux. Chaque référentiel réseau est désigné par un nom et une URL. Un nom de référentiel est propre à chaque image d’installation et représente à la fois l’organisation ou le projet qui gère le référentiel et le périmètre bien défini du contenu. Pour en savoir plus sur les référentiels réseau préconfigurés pour GlassFish Server, reportez-vous à Preconfigured Repositories for GlassFish Server du Oracle GlassFish Server 3.1 Administration Guide.

Composant add-on

Un composant add-on est un composant de l’image d’installation qui n’est pas encore installé.
