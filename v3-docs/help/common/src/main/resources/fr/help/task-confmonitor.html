Configurer le contrôle du service

  1. Sous le nœud Configurations, développez la configuration de l'instance de serveur ou de cluster pour lequel vous souhaitez configurer le contrôle.
  2. Développez le nœud Contrôle.

    La page Service de contrôle s’affiche.

  3. Sélectionnez la case Service de contrôle activé pour activer la fonction de contrôle pour GlassFish Server, ou désélectionnez cette case pour désactiver le contrôle.

    Cette option est activée par défaut.

  4. Sélectionnez la case MBeans activés pour déployer tous les MBeans nécessaires pour le contrôle, ou désélectionnez cette case pour annuler le déploiement de ces MBeans.

    Cette option est activée par défaut.

    Si la case n’est pas cochée, vous ne pouvez pas afficher les données de contrôle sous la Console d'administration , même si la case Service de contrôle est sélectionnée. Au lieu de cela, vous devez utiliser la ligne de commande pour afficher les données de contrôle. Si vous désactivez la case à cocher, l’empreinte de mémoire de GlassFish Server est réduite.

  5. Sélectionnez la case DTrace activé pour activer l’utilisation de l’outil Solaris DTrace.

    Pour pouvoir utiliser l’outil Solaris DTrace, vous devez également disposer du contrôle DTrace de Oracle Enterprise Manager. Cette option est désactivée par défaut.

  6. Dans le tableau, effectuez l’une des actions suivantes.
    • Dans la colonne Niveau de contrôle, sélectionnez HAUT ou BAS dans la liste déroulante pour un ou plusieurs composants de la colonne Composant pour activer le contrôle de ces composants.
    • Cochez les cases en regard d’un ou de plusieurs composants, puis sélectionnez HAUT ou BAS dans la liste déroulante Niveau au sommet du tableau et cliquez sur le bouton Modifier le niveau pour définir le niveau de contrôle à cette valeur pour tous ces composants.

    Dans cette version, il n’existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle d’un ou de plusieurs composants, sélectionnez DÉSACTIVÉ. La valeur par défaut est OFF.

    Vous devez déployer une application, un composant ou un service avant d’en configurer le contrôle.

  7. Cliquez sur Enregistrer.
