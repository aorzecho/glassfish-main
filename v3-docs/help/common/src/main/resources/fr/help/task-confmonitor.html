Configurer le service de surveillance

  1. Pour ouvrir la page Service de surveillance, effectuez l'une des opérations suivantes :

    • Sélectionnez le noeud Données de surveillance et cliquez sur le lien Configurer la surveillance de l'instance de serveur pour laquelle vous voulez configurer la surveillance. Pour configurer la surveillance d'un cluster, sélectionnez une instance dans le cluster.

    • Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez configurer la surveillance, puis développez le noeud Surveillance.

    La page Service de surveillance apparaît.

  2. Sélectionnez la case Service de surveillance activé pour activer la surveillance sur GlassFish Server, ou désélectionnez cette case pour la désactiver.

    Cette option est activée par défaut.

  3. Sélectionnez la case MBeans activés pour déployer tous les MBeans nécessaires à la surveillance, ou désélectionnez cette case pour annuler le déploiement de ces MBeans.

    Cette option est activée par défaut.

    Si la case n'est pas cochée, vous ne pouvez pas afficher les données de surveillance dans la Console d'administration , même si la case Service de surveillance est cochée. Au lieu de cela, vous devez utiliser la ligne de commande pour afficher les données de surveillance. Si vous désactivez la case à cocher, l'empreinte de mémoire de GlassFish Server est réduite.

  4. Sélectionnez la case DTrace activé pour activer l'utilisation de l'outil Solaris DTrace.

    Pour pouvoir utiliser l'outil Solaris DTrace, vous devez également disposer de la surveillance DTrace d'Oracle Enterprise Manager. Cette option est désactivée par défaut.

  5. Dans la table, effectuez l'une des actions suivantes.

    • Dans la colonne Niveau de surveillance, sélectionnez HIGH ou LOW dans la liste déroulante pour les composants de la colonne Composant, afin d'activer la surveillance sur ces derniers.

    • Cochez les cases en regard des composants, puis sélectionnez HIGH ou LOW dans la liste déroulante Niveau en haut de la table et cliquez sur le bouton Modifier le niveau afin de définir le niveau de surveillance sur cette valeur pour tous ces composants.

    Dans cette version, il n'existe aucune différence entre les options LOW et HIGH. Pour désactiver la surveillance des composants, sélectionnez OFF. La valeur par défaut est OFF.

    Vous devez déployer une application, un composant ou un service avant d'en configurer la surveillance.

  6. Cliquez sur Enregistrer.
