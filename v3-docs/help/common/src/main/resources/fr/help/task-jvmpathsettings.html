Configurer les paramètres du chemin d’accès à la JVM

  1. Dans l’arborescence de navigation, sélectionnez le noeud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Paramètres JVM.

    La page Paramètres JVM s’ouvre.

  3. Dans la page Paramètres généraux JVM, cliquez sur l’onglet Paramètres du chemin.

    La page Paramètres du chemin JVM s’ouvre.

  4. Dans le champ Préfixe du chemin de bibliothèque native, saisissez les chemins complets des fichiers JAR pour lesquels vous voulez ajouter un préfixe au chemin de bibliothèque native.

    Pour ajouter un préfixe à plusieurs fichiers, saisissez un chemin par ligne.

  5. Dans le champ Suffixe du chemin de bibliothèque native, saisissez les chemins complets des fichiers JAR pour lesquels vous voulez ajouter un suffixe au chemin de bibliothèque native.

    Pour ajouter un suffixe à plusieurs fichiers, saisissez un chemin par ligne.

  6. Cliquez sur Enregistrer.
  7. Redémarrez le serveur.
