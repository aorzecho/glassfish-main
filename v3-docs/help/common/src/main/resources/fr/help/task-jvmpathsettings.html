Configurer les paramètres du chemin JVM

  1. Dans l'arborescence de navigation, sélectionnez le noeud Configuration.

  2. Select the configuration that you are changing.

  3. Sous le noeud Configuration, sélectionnez le noeud Paramètres JVM.

    La page Paramètres JVM apparaît.

  4. Sur la page Paramètres généraux JVM, cliquez sur l'onglet Paramètres du chemin.

    La page Paramètres du chemin JVM apparaît.

  5. Dans le champ Préfixe du chemin de bibliothèque native, saisissez les chemins complets des fichiers JAR à ajouter comme préfixe au chemin de bibliothèque native.

    Pour ajouter un préfixe à plusieurs fichiers, saisissez un chemin par ligne.

  6. Dans le champ Suffixe du chemin de bibliothèque native, saisissez les chemins complets des fichiers JAR à ajouter comme suffixe au chemin de bibliothèque native.

    Pour ajouter un suffixe à plusieurs fichiers, saisissez un chemin par ligne.

  7. Cliquez sur Enregistrer.

  8. Redémarrez le serveur.
