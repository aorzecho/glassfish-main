도메인 속성을 설정하는 방법

  1. 탐색 트리에서 [도메인] 노드를 선택합니다.

    [도메인 속성] 페이지가 열립니다.

  2. 응용 프로그램 루트 필드에 배치된 응용 프로그램이 있는 디렉토리의 전체 경로를 입력합니다.

    기본값은 domain-dir/applications입니다.

  3. [로그 루트] 필드에 서버 로그 파일을 저장할 디렉토리의 전체 경로를 입력합니다.

    기본값은 domain-dir/logs입니다.

  4. 호스트의 기본 로케일을 사용하려면 로케일 필드를 비워 둡니다.

    로케일은 언어 및 지역의 특정 조합을 지정하는 식별자입니다. 예를 들어 미국 영어의 로케일은 en_US이고 일본어의 로케일은 ja_JP입니다. 비영어 로케일을 사용하려면 GlassFish Server이(가) 현지화(영어를 다른 언어로 번역하는 작업이 포함된 프로세스)되어야 합니다.

  5. [저장]을 누릅니다.

  6. 서버를 재시작합니다.

관련 항목

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 로그를 수집 및 다운로드하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 시스템 속성을 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 리소스 모니터링 데이터를 보는 방법
