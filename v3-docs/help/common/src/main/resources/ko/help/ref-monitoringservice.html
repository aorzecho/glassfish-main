모니터링 서비스

모니터링 서비스 페이지에서는 개별 서버 구성 요소 또는 서비스의 모니터링 옵션을 구성할 수 있습니다.

응용 프로그램, 구성 요소 또는 서비스에 대한 모니터링을 구성하려면 먼저 배포해야 합니다.

모니터링 서비스 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

모니터링 서비스

모니터링 서비스 사용 확인란이 선택되어 있으면 GlassFish Server에 대한 모니터링이 활성화됩니다. 이 옵션은 기본적으로 활성화됩니다.

MBean 모니터링

MBean 모니터링 사용 확인란이 선택되어 있으면 모니터링에 필요한 모든 MBean이 GlassFish Server에 배포됩니다. 이 옵션은 기본적으로 활성화됩니다.

이 확인란이 선택되어 있지 않으면 모니터링 서비스 확인란이 선택되어 있어도 관리 콘솔의 모니터링 데이터를 볼 수 없습니다. 모니터링 데이터를 보려면 명령줄을 사용해야 합니다. 확인란 선택을 취소하면 GlassFish Server의 메모리 범위가 줄어듭니다.

DTrace

DTrace 사용 확인란이 선택되어 있고 Oracle Enterprise Manager DTrace Monitoring이 있는 경우 Solaris DTrace 기능을 사용할 수 있습니다. 이 옵션은 기본적으로 비활성화됩니다.

사용 가능한 각 구성 요소에 대해 제공되는 정보는 다음과 같습니다.

구성 요소

GlassFish Server 구성 요소의 이름입니다.

모니터링 수준

구성 요소에 대해 설정된 모니터링의 수준입니다. 각 구성 요소에 대해 사용 가능한 수준은 LOW, HIGH 및 OFF입니다. 기본 수준은 끄기입니다. 이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다.

구성 요소 수준 설정 테이블에는 다음 옵션도 포함되어 있습니다.

수준 변경

하나 이상의 선택된 구성 요소에 대해 모니터링 수준을 변경하는 버튼입니다.

수준

모든 선택된 구성 요소에 적용되는 모니터링 수준을 선택할 수 있는 드롭다운 목록입니다.

관련 작업

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 자원 모니터링 데이터 보기
