배포된 응용 프로그램의 목록을 보는 방법

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.

    응용 프로그램 페이지가 열립니다. 배포된 응용 프로그램 목록이 테이블에 표시됩니다.

  2. 필터 드롭다운 목록에서 모듈 유형을 선택하여 모듈 유형별로 응용 프로그램 목록을 필터링합니다.
참조

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스에 대한 종단점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 로드하는 방법
