SolarisRealm 클래스 관련 속성

Solaris 영역의 경우 다음 속성이 필요합니다.

JAAS 컨텍스트

JAAS(Java Authentication and Authorization Service) 컨텍스트는 이 영역에서 사용할 로그인 모듈의 식별자입니다. 유일하게 적합한 값은 solarisRealm입니다.

Solaris 영역의 경우 다음과 같은 선택적 속성을 사용할 수 있습니다.

그룹 지정

콤마로 구분된 그룹 이름 목록입니다. 적합한 인증서를 제공하는 모든 클라이언트가 이러한 그룹에 지정됩니다. 예를 들어 employee,manager는 사용자 그룹의 이름입니다.
