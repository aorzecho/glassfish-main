메시지 보안 구성을 삭제하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [메시지 보안] 노드를 선택합니다.

    [메시지 보안 구성] 페이지가 열립니다.

  4. [메시지 보안 구성] 페이지에서 삭제 중인 메시지 보안 구성의 왼쪽에 있는 체크 박스를 선택합니다.

    주:

    기본 server-config 구성에 대한 메시지 보안 제공자는 생성하거나 삭제할 수 없습니다. 기본 server-config에 대해 지원되는 메시지 보안 제공자는 SOAP와 HttpServlet뿐입니다. 이 두 메시지 제공자는 GlassFish Server 설치 시 이미 기본적으로 생성되었기 때문에 삭제 단추가 기본 server-config의 [메시지 보안 구성] 페이지에 표시되지 않습니다.

  5. [삭제]를 누릅니다.

관련 항목

  • GlassFish Server 기능을 메시지 보안 제공자가 사용할 수 있도록 구성하는 방법

  • 메시지 보안 구성을 생성하는 방법

  • 메시지 보안 구성을 편집하는 방법

  • 메시지 보안 제공자 구성을 생성하는 방법

  • 메시지 보안 제공자 구성을 편집하는 방법

  • 메시지 보안 제공자 구성을 삭제하는 방법

  • 응용 프로그램 클라이언트에 대해 메시지 보안을 사용으로 설정하는 방법
