원시 로그 뷰어

원시 로그 뷰어 페이지에서는 관리 서버(이름: server)에 대한 원시 로그 파일을 볼 수 있습니다.

원시 로그 뷰어 페이지에는 다음 옵션이 포함되어 있습니다.

인스턴스

로그 레코드를 표시할 서버 인스턴스나 클러스터입니다. 유일한 선택 항목은 server입니다.

각 메시지에 대한 원시 로그 뷰어에는 다음 정보가 포함됩니다.

타임스탬프

로그 메시지가 발행된 날짜와 시간입니다.

레코드 번호

서버 로그에서 메시지의 번호입니다.

로그 레벨

메시지의 로그 레벨입니다.

버전

GlassFish Server의 버전입니다.

로거

로그 메시지를 생성한 로거 모듈입니다.

이름/값 쌍

로그 메시지의 스레드 이름 및 ID입니다.

메시지

로그 메시지의 간략한 텍스트입니다. (세부 사항) 링크를 누르면 서식이 지정된 버전의 메시지를 볼 수 있습니다.

관련 작업

  • DAS에 대한 원시 로그 파일을 보는 방법

  • 일반 로깅 설정을 구성하는 방법

  • 로그 레벨을 구성하는 방법

  • DAS에 대한 로그 파일을 보는 방법

관련 asadmin 명령

  • set-log-level

  • rotate-log(1)
