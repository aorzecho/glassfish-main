대상 관리

자원 대상 관리 페이지를 사용하여 라이프사이클 모듈을 활성화할 수 있는 대상 클러스터와 독립 실행형 서버 인스턴스를 변경할 수 있습니다. 라이프사이클 모듈은 선택한 대상 열의 대상에서만 활성화할 수 있습니다.

자원 대상 관리 페이지에는 다음 정보가 포함됩니다.

사용 가능한 대상

라이프사이클 모듈을 활성화할 수 없는 클러스터와 인스턴스입니다.

선택한 대상

라이프사이클 모듈을 활성화할 수 있는 클러스터와 인스턴스입니다.

추가

선택한 대상을 사용 가능한 대상에서 선택한 대상으로 이동하는 버튼입니다.

모두 추가

사용 가능한 대상을 모두 선택한 대상으로 이동하는 버튼입니다.

제거

선택한 대상을 선택한 대상에서 사용 가능한 대상으로 이동하는 버튼입니다.

모두 제거

선택한 대상을 모두 사용 가능한 대상으로 이동하는 버튼입니다.
