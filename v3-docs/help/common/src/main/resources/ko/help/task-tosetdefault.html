기본 영역을 설정하는 방법

기본 영역은 응용 프로그램의 배포 설명자에 영역이 지정되어 있지 않은 경우 GlassFish 서버가 인증 및 권한 부여에 사용하는 영역입니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  3. 기본 영역 드롭다운 목록에서 서버가 인증을 위해 사용하는 활성 영역을 선택합니다.

    기본값은 file입니다.

  4. 저장을 누릅니다.
