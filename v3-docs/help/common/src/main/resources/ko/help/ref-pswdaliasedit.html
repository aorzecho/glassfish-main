암호 별칭 편집

암호 별칭 편집 페이지를 사용하여 기존 암호 별칭과 연관된 암호를 변경합니다.

암호 별칭 편집 페이지에는 다음 옵션이 포함되어 있습니다.

별칭 이름

별칭 이름입니다. 이 옵션은 편집할 수 없습니다.

암호

암호 별칭이 암호화된 형식으로 저장되는 암호입니다.

암호 확인

암호 별칭이 암호화된 형식으로 저장되는 암호입니다. 암호를 잘못 입력하지 않았는지 확인하기 위해 암호와 암호 확인 값이 비교됩니다.

동일한 asadmin 하위 명령

  • create-password-alias(1)

  • delete-password-alias(1)
