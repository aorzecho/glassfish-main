응용 프로그램 클라이언트를 실행하는 방법

시작하기 전에

실행할 응용 프로그램 클라이언트가 사용으로 설정되었는지 확인합니다. 클라이언트는 Java Web Start로 실행할 수 있는 웹 클라이언트입니다.

  1. 탐색 트리에서 [응용 프로그램] 노드를 선택합니다.

    [응용 프로그램] 페이지가 열립니다.

  2. 배치된 응용 프로그램 테이블에서 실행할 응용 프로그램 클라이언트의 이름을 누릅니다.

    이름에는 버전 식별자(옵션)가 포함될 수 있으며, 이 식별자는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    [응용 프로그램 편집] 페이지가 열립니다.

  3. Java Web Start 체크 박스가 선택되어 있는지 확인합니다.

    응용 프로그램 클라이언트를 실행하려면 먼저 Java Web Start을 사용으로 설정해야 합니다. Java Web Start 체크 박스가 선택되어 있지 않으면 [저장]을 누릅니다.

  4. 모듈 및 구성 요소 테이블에서 작업 열의 실행을 누릅니다.

    응용 프로그램 클라이언트 실행 페이지가 열립니다.

  5. 응용 프로그램 클라이언트 실행 페이지에서 응용 프로그램 클라이언트에 전달할 인수를 입력합니다.

  6. 실행을 누릅니다.
