가상 서버 대상

가상 서버 대상 페이지를 사용하여 응용 프로그램이나 모듈이 연결된 대상 가상 서버를 변경합니다.

가상 서버 대상 페이지에는 다음 정보가 포함됩니다.

사용 가능한 대상

응용 프로그램이나 모듈이 연결되지 않은 가상 서버입니다.

선택한 대상

응용 프로그램이나 모듈이 연결된 가상 서버입니다.

추가

선택한 대상을 사용 가능한 대상에서 선택한 대상으로 이동하는 버튼입니다.

모두 추가

사용 가능한 대상을 모두 선택한 대상으로 이동하는 버튼입니다.

제거

선택한 대상을 선택한 대상에서 사용 가능한 대상으로 이동하는 버튼입니다.

모두 제거

선택한 대상을 모두 사용 가능한 대상으로 이동하는 버튼입니다.

관련 작업

  • 응용 프로그램의 설명자를 보는 방법

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스에 대한 종단점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 로드하는 방법
