메시지 보안 제공자 구성 관련 속성

메시지 보안 제공자를 구성하는 데 사용할 수 있는 속성은 다음과 같습니다.

server.config

서버 구성 정보가 포함된 XML 파일의 디렉토리 및 파일 이름입니다. 기본값은 domain-dir/config/wss-server-config-1.0.xml입니다.

debug

true이면 서버 제공자 디버그 메시지를 서버 로그에 덤프할 수 있습니다. 기본값은 false입니다.

dynamic.username.password

true이면 제공자 런타임에게 각 요청에 대해 CallbackHandler로부터 사용자 이름과 비밀번호를 수집하도록 신호를 보냅니다. false이면 wsse:UsernameToken(s) 에 대한 사용자 이름과 비밀번호가 모듈 초기화 도중 한 번 수집됩니다. 이 속성은 ClientAuthModule에만 적용 가능합니다. 기본값은 false입니다.

encryption.key.alias

제공자가 사용하는 암호화 키를 지정합니다. 키는 해당 keystore 별칭으로 식별됩니다. 기본값은 s1as입니다.

signature.key.alias

제공자가 사용하는 서명 키를 지정합니다. 키는 해당 keystore 별칭으로 식별됩니다. 기본값은 s1as입니다.
