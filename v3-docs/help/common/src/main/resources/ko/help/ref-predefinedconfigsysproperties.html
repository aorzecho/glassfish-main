사전 정의된 시스템 등록 정보

GlassFish 서버에서는 다음 구성 시스템 등록 정보를 정의합니다.

ASADMIN_LISTENER_PORT

이 등록 정보는 관리용 HTTP 포트 또는 HTTPS 포트의 포트 번호를 지정합니다. 이 포트는 인스턴스를 관리하기 위해 웹 브라우저에 지정하는 URL의 포트입니다(예: http://localhost:4949). 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

HTTP_LISTENER_PORT

이 등록 정보는 HTTP 요청을 수신하는 데 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

HTTP_SSL_LISTENER_PORT

이 등록 정보는 HTTPS 요청을 수신하는 데 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

IIOP_LISTENER_PORT

이 등록 정보는 IIOP 연결에 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

IIOP_SSL_LISTENER_PORT

이 등록 정보는 보안 IIOP 연결에 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

IIOP_SSL_MUTUALAUTH_PORT

이 등록 정보는 클라이언트 인증으로 보안된 IIOP 연결에 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

JAVA_DEBUGGER_PORT

이 등록 정보는 JPDA(Java Platform Debugger Architecture) 디버거에 연결하는 데 사용되는 포트의 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

JMS_PROVIDER_PORT

이 등록 정보는 Java Message Service 공급자에 대한 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

JMX_SYSTEM_CONNECTOR_PORT

이 등록 정보는 JMX 커넥터가 수신하는 포트 번호를 지정합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.

OSGI_SHELL_TELNET_PORT

이 등록 정보는 Apache Felix Remote Shell에 연결하는 데 사용되는 포트의 포트 번호를 지정합니다. 이 셸은 Felix 셸 서비스를 사용하여 OSGi 모듈 관리 하위 시스템과 상호 작용합니다. 유효한 값은 1–65535입니다. UNIX의 경우 1–1024 포트에서 수신하는 소켓을 만들려면 수퍼유저 권한이 필요합니다.
