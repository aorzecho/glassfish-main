관리 사용자

관리 사용자 페이지에서는 admin-realm 사용자를 관리할 수 있습니다. admin-realm은 파일 영역입니다. file 영역의 사용자 및 그룹은 키 파일에 나열되며, 키 파일의 위치는 file 등록 정보에 지정되어 있습니다.

GlassFish Server 설치 후 처음에는 설치 시 입력한 관리자만이 유일한 사용자입니다. 기본적으로 이 사용자는 GlassFish Server를 수정할 수 있는 권한을 부여하는 admin-realm 영역의 asadmin 그룹에 속합니다. 이 그룹에 할당된 사용자는 관리자 권한을 갖습니다. 즉, asadmin 도구 및 관리 콘솔에 액세스할 수 있는 권한을 부여 받습니다.

관리 사용자 페이지에는 선택된 영역의 사용자 목록이 표시됩니다. 각 사용자에 대해 제공되는 정보는 다음과 같습니다.

사용자 ID

사용자 이름입니다.

그룹 목록

사용자가 속한 그룹입니다.

관리 사용자 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 사용자를 만드는 버튼입니다.

삭제

하나 이상의 선택된 사용자를 삭제하는 버튼입니다.
