JVM 경로 설정

JVM 경로 설정 페이지에서는 원시 라이브러리 경로를 구성할 수 있습니다. 원시 라이브러리 경로는 다음 경로가 결합된 것입니다.

  • 원시 라이브러리 경로 접두어

  • 서버의 원시 라이브러리 경로(즉, as-install/lib)

  • 64비트 원시 라이브러리 디렉토리(해당하는 경우)

  • GlassFish 서버를 시작하는 데 사용되는 JVM의 java.library.path 변수 값(운영 체제 간에 차이점이 있음)

  • JVM 프로필러 설정 페이지에 지정된 모든 경로 (프로필러를 사용할 수 있는 경우)

  • 원시 라이브러리 경로 접미어

다른 설정은 GlassFish 서버의 이전 릴리스에서 업그레이드된 도메인에 대한 읽기 전용 값으로 표시됩니다. 이 설정은 v3 릴리스에서는 지원되지 않습니다.

JVM 경로 설정 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

환경 클래스 경로

무시 확인란을 선택하면 CLASSPATH 환경 변수가 무시됩니다. CLASSPATH 환경 변수는 프로그래밍의 기본 자습서에는 편리하지만 엔터프라이즈 환경에서는 사용하지 않는 것이 좋습니다. 읽기 전용 필드입니다.

시스템 클래스 경로

시스템에 있는 클래스의 경로입니다. 읽기 전용 필드입니다.

클래스 경로 접두어

서버의 클래스 경로 앞에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다. 읽기 전용 필드입니다.

클래스 경로 접미어

서버의 클래스 경로 뒤에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다. 읽기 전용 필드입니다.

원시 라이브러리 경로 접두어

원시 라이브러리 경로에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.

원시 라이브러리 경로 접미어

원시 라이브러리 경로에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.
