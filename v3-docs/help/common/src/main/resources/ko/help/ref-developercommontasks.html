일반 작업

[일반 작업] 페이지에는 일반적인 관리 콘솔 작업에 대한 바로 가기가 있습니다.

[일반 작업] 페이지에서는 다음 작업을 수행할 수 있습니다.

  • GlassFish Server의 [GlassFish 뉴스] 페이지를 보는 방법

  • GlassFish Server의 가입 페이지를 보는 방법

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 리소스 모니터링 데이터를 보는 방법

  • 설치된 구성 요소를 확인하는 방법

  • 추가 기능 구성 요소를 설치하는 방법

  • 구성 요소의 사용 가능한 업데이트를 확인하는 방법

  • To Create a JDBC Connection Pool

또한 [일반 작업] 페이지에는 일부 GlassFish Server 설명서에 대한 링크도 제공됩니다.
