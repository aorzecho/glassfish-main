새 감사 모듈

새 감사 모듈 페이지에서는 새 감사 모듈을 정의할 수 있습니다.

새 감사 모듈 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

새 감사 모듈의 이름입니다.

클래스 이름

이 모듈을 구현하는 클래스의 정규화된 이름입니다. 기본 감사 모듈의 클래스 이름은 com.sun.enterprise.security.Audit입니다.

추가 등록 정보

감사 모듈의 추가 등록 정보입니다. 감사 모듈에 사용 가능한 추가 등록 정보의 설명은 감사 모듈과 관련된 등록 정보를 참조하십시오.

관련 작업

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법

관련 asadmin 명령

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
