Oracle GlassFish 서버 등록

Oracle GlassFish 서버의 등록 사용자에게는 다음과 같은 이점이 제공됩니다.

  • 패치 정보 및 버그 업데이트

  • 스크린캐스트 및 자습서

  • 뉴스 및 이벤트

  • 지원 및 교육 오퍼링

설치된 Oracle GlassFish 서버를 등록하려면 Sun 온라인 계정이 있어야 합니다. Sun 온라인 계정이 없는 경우 GlassFish 서버를 등록하는 동안 GlassFish 서버를 사용하여 계정을 만들 수 있습니다.

주 - GlassFish 서버를 등록하는 동안 Sun 온라인 계정을 만들 경우 필요한 정보만 입력하라는 메시지가 표시됩니다. GlassFish 서버를 등록한 후 Sun 온라인 계정 업데이트를 통해 선택적 정보를 추가로 제공할 수 있습니다.

시작하기 전에

GlassFish 서버가 실행 중인 시스템이 프록시를 통해 인터넷에 연결되면 프록시의 호스트 이름과 포트 번호를 가져옵니다. 웹 브라우저와 GlassFish 서버가 동일한 시스템에서 실행 중이면 웹 브라우저에서 이 정보를 가져올 수 있습니다. 그렇지 않으면 시스템 관리자에게 이 정보를 문의하십시오.

  1. 탐색 트리에서 등록 노드를 선택합니다.

    제품 등록 페이지가 열립니다.

  2. 제품 등록 페이지에서 필수 정보를 제공합니다.
    • Sun 온라인 계정이 이미 있는 경우 필수 정보를 다음과 같이 제공합니다.
      1. 이미 Sun 온라인 계정 또는 Sun 개발자 네트워크(SDN) 계정이 있습니다 옵션을 선택합니다.
      2. 사용자 이름 필드에서 Sun 온라인 계정을 등록할 때 지정한 사용자 이름을 입력합니다.
      3. 비밀번호 필드에서 Sun 온라인 계정을 등록할 때 지정한 비밀번호를 입력합니다.
    • Sun 온라인 계정이 없는 경우 필수 정보를 다음과 같이 제공합니다.
      1. Sun 온라인 계정 없음 옵션을 선택합니다.
      2. 전자 메일 주소 필드에 전자 메일 주소를 입력합니다.

        전자 메일 주소가 Sun 온라인 계정 사용자 이름이 됩니다.

      3. 비밀번호 필드에 Sun 온라인 계정에 사용할 비밀번호를 입력합니다.

        비밀번호는 대소문자를 구분하며 6–10자 이내로 입력해야 합니다.

      4. 비밀번호 다시 입력 필드에 이전 단계에서 입력한 비밀번호를 다시 입력합니다.
      5. 이름 필드에 이름을 입력합니다.
      6. 성 필드에 성을 입력합니다.
      7. 회사 필드에 회사 이름을 입력합니다(적용 가능한 경우).
      8. 드롭다운 목록에서 거주하는 국가 또는 지역을 선택합니다.
  3. GlassFish 서버가 실행 중인 시스템이 프록시를 통해 인터넷에 연결되면 프록시의 호스트 이름과 포트 번호를 제공합니다.

    GlassFish 서버가 실행 중인 시스템이 인터넷에 직접 연결될 경우 이 단계를 생략합니다.

    웹 브라우저와 GlassFish 서버가 동일한 시스템에서 실행될 수 있습니다. 그럴 경우 웹 브라우저에서 프록시의 호스트 이름과 포트 번호를 가져올 수 있습니다. 그렇지 않으면 시스템 관리자에게 이 정보를 문의하십시오.

  4. 등록을 누릅니다.

    등록이 완료되었는지 확인하도록 지원 페이지가 열립니다.
