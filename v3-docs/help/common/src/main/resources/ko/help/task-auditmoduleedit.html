감사 모듈을 편집하는 방법

기본적으로 감사 모듈은 활성화되지 않습니다. 감사 모듈을 활성화하는 방법에 대한 자세한 내용은 감사 로깅을 활성화하거나 비활성화하는 방법을 참조하십시오.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 확장합니다.
  3. 보안 노드 아래에서 감사 모듈 노드를 선택합니다.

    감사 모듈 페이지가 열립니다.

  4. 감사 모듈 페이지에서 편집 중인 감사 모듈의 이름을 누릅니다.

    감사 모듈 편집 페이지가 열립니다.

  5. 감사 모듈 편집 페이지에서 클래스 이름 필드에 새 값을 입력하여 클래스 이름을 수정합니다.
  6. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    감사 모듈에 사용 가능한 추가 등록 정보의 설명은 감사 모듈과 관련된 등록 정보를 참조하십시오.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  7. 저장을 누릅니다.
참조

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
