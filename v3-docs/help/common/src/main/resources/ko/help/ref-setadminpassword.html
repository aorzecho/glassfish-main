새 관리자 암호

새 관리자 암호 페이지에서 현재 관리자 계정의 암호를 변경할 수 있습니다.

새 관리자 암호 페이지에는 다음 옵션이 포함되어 있습니다.

사용자 ID

현재 관리자의 사용자 ID입니다. 읽기 전용 필드입니다.

그룹 목록

관리자가 속한 그룹입니다. 읽기 전용 필드입니다. 값은 asadmin입니다.

새 암호

관리자의 새 암호입니다.

새 암호 확인

관리자의 새 암호입니다.

관련 작업

  • 관리자 암호를 변경하는 방법

관련 asadmin 명령

  • change-admin-password(1)
