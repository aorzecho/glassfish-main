새 관리자 비밀번호

[새 관리자 비밀번호] 페이지에서 현재 관리자 계정의 비밀번호를 변경할 수 있습니다.

주:

보안 관리가 사용으로 설정된 경우에는 비밀번호를 빈 값으로 변경할 수 없습니다.

[새 관리자 비밀번호] 페이지에는 다음 옵션이 포함되어 있습니다.

사용자 ID

현재 관리자의 사용자 ID입니다. 읽기 전용 필드입니다.

그룹 목록

관리자가 속한 그룹입니다. 읽기 전용 필드입니다. 값은 asadmin입니다.

새 비밀번호

관리자의 새 비밀번호입니다.

새 비밀번호 확인

관리자의 새 비밀번호입니다.

관련 작업

관리자 비밀번호를 변경하는 방법

관련 asadmin 명령

change-admin-password(1)
