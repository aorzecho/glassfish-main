영역

GlassFish Server는 file, certificate 및 admin-realm의 세 가지 영역으로 미리 구성됩니다. ldap, JDBC, solaris 및 사용자 정의 영역을 만들 수도 있습니다. 일반적으로 한 서버에 각 유형당 하나의 영역이 있지만 GlassFish Server에는 file 및 admin-realm의 두 파일 영역이 있습니다. 이 두 영역의 유형은 동일하지만 사용 목적은 다릅니다. 사용자 시스템의 각 가상 서버가 서로 다른 인증서 데이터베이스를 사용하도록 지정할 수도 있습니다.

영역 페이지에서는 영역을 구성할 수 있습니다.

각 영역에 대해 제공되는 정보는 다음과 같습니다.

이름

영역의 이름입니다.

영역의 클래스 이름입니다.
