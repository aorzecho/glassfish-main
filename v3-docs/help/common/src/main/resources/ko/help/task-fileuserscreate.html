파일 영역에 사용자를 추가하는 방법

  1. [구성] 노드에서 관리 영역 사용자를 관리할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

  4. file 노드를 선택합니다.

    [영역 편집] 페이지가 열립니다.

  5. [영역 편집] 페이지에서 [사용자 관리] 단추를 누릅니다.

    [파일 사용자] 페이지가 열립니다.

  6. [파일 사용자] 페이지에서 [새로 만들기]를 누릅니다.

    [새 파일 영역 사용자] 페이지가 열립니다.

  7. [사용자 ID] 필드에 사용자의 이름을 입력합니다.

  8. [그룹 목록] 필드에 사용자가 속한 그룹의 목록을 입력합니다.

    콤마를 사용하여 그룹 이름을 구분합니다. 이 그룹은 다른 곳에서 정의할 필요가 없습니다.

  9. [비밀번호] 필드에 고유한 비밀번호를 입력합니다.

  10. [새 비밀번호 확인] 필드에 사용자 비밀번호를 다시 입력합니다.

  11. [확인]을 누릅니다.

관련 항목

  • 파일 영역에서 사용자를 삭제하는 방법
