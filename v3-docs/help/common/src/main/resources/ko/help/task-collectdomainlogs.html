도메인 로그를 수집 및 다운로드하는 방법

  1. 탐색 트리에서 [도메인] 노드를 선택합니다.

    [도메인 속성] 페이지가 열립니다.

  2. [도메인 로그] 탭을 선택합니다.

    [도메인 로그] 페이지가 열립니다.

    클러스터가 있는 경우 클러스터 및 인스턴스에 대한 드롭다운 목록이 표시됩니다. 클러스터가 없는 경우 인스턴스에 대한 드롭다운 목록만 표시됩니다.

  3. 클러스터 또는 인스턴스를 선택합니다.

  4. 로그 파일을 수집 및 다운로드할 클러스터 또는 인스턴스를 선택합니다.

  5. 로그 수집을 누릅니다.

    브라우저의 다운로드 방식을 통해 log-files-name.zip 파일을 다운로드할지 묻는 메시지를 표시합니다. 여기서 name은 클러스터 또는 인스턴스의 이름입니다.
