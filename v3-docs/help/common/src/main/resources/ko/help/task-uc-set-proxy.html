네트워크 저장소 액세스를 위해 네트워크 프록시 설정을 구성하는 방법

네트워크 프록시 설정에서 GlassFish Server를 인터넷에 연결하는 방법을 결정합니다. 인터넷을 통해 업데이트 도구를 네트워크 저장소에 액세스하려면 네트워크 프록시 설정을 구성합니다.

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 열립니다.

  2. 사용 가능한 애드온 페이지에서 구성 탭을 누릅니다.

    업데이트 도구 구성 페이지가 표시됩니다.

  3. 업데이트 도구 구성 페이지에서 GlassFish Server를 실행하는 시스템이 인터넷에 연결되는 방법을 지정합니다.
    • 시스템을 인터넷에 직접 연결한 경우 직접 연결 옵션을 선택합니다.
    • 시스템이 프록시를 통해 연결된 경우 프록시를 통해 옵션을 선택합니다.
  4. ProductName;를 실행 중인 시스템에서 프록시를 통해 인터넷에 연결하는 경우에는 프록시 호스트 이름과 프록시 포트 번호를 제공합니다.

    GlassFish Server가 실행 중인 시스템이 인터넷에 직접 연결될 경우 이 단계를 생략합니다.

    웹 브라우저와 GlassFish Server가 동일한 시스템에서 실행될 수 있습니다. 그럴 경우 웹 브라우저에서 프록시의 호스트 이름과 포트 번호를 가져올 수 있습니다. 그렇지 않으면 시스템 관리자에게 이 정보를 문의하십시오.

  5. 저장을 누릅니다.
