관리 영역에서 사용자 비밀번호를 변경하는 방법

현재 관리자 계정의 비밀번호를 변경하려면 관리자 비밀번호를 변경하는 방법을(를) 참조하십시오. admin-realm에 둘 이상의 사용자가 있을 경우 다른 사용자의 비밀번호도 변경할 수 있습니다.

관리자 비밀번호를 암호화하면 응용 프로그램의 보안이 향상됩니다. 보안 절차에 대한 자세한 내용은 Oracle GlassFish Server Security Guide의 1장, Administering System Security를 참조하십시오.

  1. [구성] 노드에서 관리 비밀번호를 수정할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

  4. admin-realm 노드를 선택합니다.

    [영역 편집] 페이지가 열립니다.

  5. [영역 편집] 페이지에서 [사용자 관리] 단추를 누릅니다.

    [관리 사용자] 페이지가 열립니다.

  6. [관리 사용자] 페이지에서 비밀번호를 추가 또는 변경할 사용자를 누릅니다.

    [파일 영역 사용자 편집] 페이지가 열립니다.

  7. [새 비밀번호] 및 [새 비밀번호 확인] 필드에 새 비밀번호를 입력합니다.

  8. [저장]을 누릅니다.
