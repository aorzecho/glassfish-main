응용 프로그램 범위 리소스를 수정하는 방법

  1. 탐색 트리에서 [응용 프로그램] 노드를 선택합니다.

    [응용 프로그램] 페이지가 열립니다.

  2. 응용 프로그램 범위 리소스를 표시할 응용 프로그램을 선택합니다.

    응용 프로그램 이름에는 선택적인 버전 식별자가 포함될 수 있으며 이 식별자는 이름 뒤에 오고 콜론(:)을 통해 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    [응용 프로그램] 페이지가 열립니다.

  3. [리소스] 탭을 선택합니다.

    [응용 프로그램 범위 리소스] 페이지가 열리고, 해당 응용 프로그램에 대해 현재 지정된 리소스가 다음 열을 포함하는 테이블에 표시됩니다.

    응용 프로그램 이름

    응용 프로그램의 이름입니다.

    모듈 이름

    응용 프로그램 내에 있는 모듈의 이름입니다.

    리소스 이름

    응용 프로그램 또는 모듈에 대한 응용 프로그램 범위 리소스의 이름입니다.

    리소스 유형

    리소스 유형(예: JDBC 접속 풀 또는 커넥터 리소스)입니다.

  4. 수정할 리소스의 이름을 누릅니다.

    리소스 유형에 따라 열리는 페이지가 달라집니다. 예를 들어 리소스가 JDBC 접속 풀인 경우 JDBC [접속 풀 편집] 페이지가 열립니다.
