JVM 프로파일러 설정을 구성하는 방법

프로파일러 도구는 성능을 분석하고 잠재적 병목 현상을 식별하는 데 사용되는 데이터를 생성합니다.

  1. 탐색 트리에서 [구성] 노드를 선택합니다.

  2. Select the configuration that you are changing.

  3. [구성] 노드에서 [JVM 설정] 노드를 선택합니다.

    [JVM 일반 설정] 페이지가 열립니다.

  4. [JVM 일반 설정] 페이지에서 [프로파일러] 탭을 누릅니다.

    [JVM 프로파일러 설정] 페이지가 열립니다.

  5. 프로파일러 설정을 지정합니다.

    지정하는 정보는 사용 중인 프로파일러 제품에 따라 다릅니다. 예제 및 지침은 Oracle GlassFish Server Application Development Guide의 3장, Debugging Applications를 참조하십시오.

  6. [저장]을 누릅니다.

  7. 서버를 재시작합니다.
