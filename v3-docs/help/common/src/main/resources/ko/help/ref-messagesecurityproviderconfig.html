제공자 구성

[제공자 구성] 페이지에서는 SOAP 또는 HttpServlet 인증 계층에 대한 메시지 보안 제공자 구성을 구성할 수 있습니다.

기본적으로 SOAP 인증 계층에 대해 네 개의 제공자가 구성됩니다.

각 제공자 구성에 대해 제공되는 정보는 다음과 같습니다.

제공자 ID

제공자 구성의 이름입니다.

제공자 유형

제공자 유형으로, 다음 중 하나가 될 수 있습니다.

client

제공자가 클라이언트 인증 제공자임을 지정합니다.

server

제공자가 서버 인증 제공자임을 지정합니다.

client-server

제공자가 클라이언트 및 서버 인증 제공자임을 지정합니다.

기본 제공자

true로 설정하면 제공자는 기본 제공자입니다.

클래스 이름

제공자의 Java 구현 클래스의 이름입니다.

제공자 구성 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 제공자 구성을 생성하는 단추입니다.

삭제

선택한 제공자 구성을 하나 이상 삭제하는 단추입니다.

관련 작업

  • GlassFish Server 기능을 메시지 보안 제공자가 사용할 수 있도록 구성하는 방법

  • 메시지 보안 구성을 생성하는 방법

  • 메시지 보안 구성을 편집하는 방법

  • 메시지 보안 구성을 삭제하는 방법

  • 메시지 보안 제공자 구성을 생성하는 방법

  • 메시지 보안 제공자 구성을 편집하는 방법

  • 메시지 보안 제공자 구성을 삭제하는 방법

  • 응용 프로그램 클라이언트에 대해 메시지 보안을 사용으로 설정하는 방법

  • create-message-security-provider(1)

  • list-message-security-providers(1)

  • delete-message-security-provider(1)
