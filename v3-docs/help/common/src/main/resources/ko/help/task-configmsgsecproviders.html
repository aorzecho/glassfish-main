GlassFish Server 기능을 메시지 보안 제공자가 사용할 수 있도록 구성하는 방법

  1. 사용자 이름 토큰을 사용할 경우 필요하면 사용자 데이터베이스를 구성합니다.

    사용자 이름/비밀번호 토큰을 사용할 경우 먼저 영역 및 해당 영역의 사용자 데이터베이스를 적절하게 구성합니다.

    사용자 데이터베이스 구성은 영역을 편집하는 방법을(를) 참조하십시오.

  2. 필요한 경우 인증서 및 전용 키를 관리합니다.

다음 단계

GlassFish Server의 기능을 메시지 보안 제공자가 사용할 수 있도록 구성한 후에는 메시지 보안 구성을 편집하는 방법에 설명된 바와 같이 GlassFish Server과(와) 함께 설치된 제공자를 사용으로 설정할 수 있습니다.
