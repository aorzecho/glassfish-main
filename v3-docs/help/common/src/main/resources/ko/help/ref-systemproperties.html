시스템 속성

시스템 속성 페이지에서는 GlassFish Server 속성을 보고 관리할 수 있습니다.

시스템 속성은 서버 레벨에서의 설정에 대한 공통 값입니다. 달러 기호와 굽은 괄호로 묶어 관리 콘솔 텍스트 필드의 시스템 속성을 참조할 수 있습니다. 예를 들어 prop-name이라는 속성을 참조하려면 ${prop-name} 구문을 사용합니다.

각 속성에 대해 시스템 속성 페이지에는 다음 정보가 포함됩니다.

이름

속성의 이름입니다.

값

속성의 값입니다.

설명

속성에 대한 선택적 설명입니다.

여러 개의 시스템 속성이 사전 정의되어 있지만 이 페이지에는 나와 있지 않습니다. 자세한 내용은 미리 정의된 시스템 속성을(를) 참조하십시오.
