응용 프로그램 범위 리소스

[응용 프로그램 범위 리소스] 페이지에서는 특정 응용 프로그램에서만 사용할 수 있는 리소스를 볼 수 있습니다.

[응용 프로그램 범위 리소스] 페이지는 다음 정보를 포함하는 테이블로 구성됩니다.

응용 프로그램 이름

응용 프로그램의 이름입니다.

모듈 이름

응용 프로그램 내에 있는 모듈의 이름입니다.

리소스 이름

응용 프로그램 또는 모듈에 대한 응용 프로그램 범위 리소스의 이름입니다. 이름을 누르면 리소스 설정을 수정할 수 있습니다.

리소스 유형

리소스 유형(예: JDBC 접속 풀 또는 커넥터 리소스)입니다.

관련 작업

  • 응용 프로그램 기술자를 보는 방법

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 배치된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스 끝점 정보를 보는 방법

  • 응용 프로그램을 실행하는 방법

  • 응용 프로그램 클라이언트를 실행하는 방법

  • 응용 프로그램 클라이언트 스텁을 다운로드하는 방법

  • 배치된 응용 프로그램을 재배치하는 방법

  • 배치된 응용 프로그램을 재로드하는 방법
