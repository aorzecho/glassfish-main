응용 프로그램 클라이언트 시작

응용 프로그램 클라이언트 시작 페이지를 사용하여 배포된 응용 프로그램 클라이언트를 시작합니다. 클라이언트는 Java Web Start으로 실행할 수 있는 웹 클라이언트입니다.

응용 프로그램 클라이언트 시작 페이지에는 다음 옵션이 포함됩니다.

응용 프로그램

응용 프로그램 클라이언트를 포함하는 응용 프로그램의 이름입니다.

이름에는 버전 ID(옵션)가 포함될 수 있으며, 이 ID는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server 3.1 Application Deployment Guide의 Module and Application Versions를 참조하십시오.

모듈

응용 프로그램 클라이언트를 포함하는 모듈의 이름입니다.

링크

클라이언트가 배포된 URL입니다.

인수

응용 프로그램에 전달할 인수입니다

관련 작업

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 웹 서비스에 대한 종단점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 로드하는 방법
