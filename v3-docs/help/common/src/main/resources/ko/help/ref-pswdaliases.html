암호 별칭

암호 별칭 페이지를 사용하여 암호 별칭을 만들고 관리합니다.

암호 별칭 페이지에는 도메인의 암호 별칭 목록이 표시됩니다. 각 별칭에 대해 표시되는 정보는 다음과 같습니다.

이름

암호 별칭이 만들어질 때 암호 별칭에 지정된 이름입니다. 이름을 누르면 별칭에 대한 암호 별칭 편집 페이지가 열립니다.

암호 별칭 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

암호 별칭을 만드는 버튼입니다.

삭제

하나 이상의 선택된 암호 별칭을 삭제하는 버튼입니다.

동일한 asadmin 하위 명령

  • list-password-aliases(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • update-password-alias(1)
