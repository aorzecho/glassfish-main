비밀번호 별칭

[비밀번호 별칭] 페이지를 사용하여 비밀번호 별칭을 생성하고 관리합니다.

[비밀번호 별칭] 페이지에는 도메인의 비밀번호 별칭 목록이 표시됩니다. 각 별칭에 대해 표시되는 정보는 다음과 같습니다.

이름

생성 시 비밀번호 별칭에 지정된 이름입니다. 이름을 누르면 해당 별칭에 대한 비밀번호 별칭 편집 페이지가 열립니다.

비밀번호 별칭 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

비밀번호 별칭을 생성하는 단추입니다.

삭제

선택한 비밀번호 별칭을 하나 이상 삭제하는 단추입니다.

동일한 asadmin 하위 명령

  • create-password-alias(1)

  • delete-password-alias(1)
