서버 모니터링

서버 모니터링 페이지에서는 선택된 가상 서버에 대한 모니터링 데이터를 볼 수 있습니다. 모니터링되는 등록 정보에 대한 설명은 Oracle GlassFish Server 3.1 Administration Guide의 8 장, Administering the Monitoring Service를 참조하십시오.

관련 작업

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 자원 모니터링 데이터 보기

관련 asadmin 명령

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
