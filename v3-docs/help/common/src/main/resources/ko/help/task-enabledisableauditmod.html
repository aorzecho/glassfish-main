감사 로깅을 활성화하거나 비활성화하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  3. 보안 페이지에서 감사 로깅 사용 가능 확인란을 선택하여 감사 로깅을 활성화합니다.

    감사 로깅을 활성화하려면 활성 감사 모듈에 대한 auditOn 등록 정보를 true로 설정해야 합니다.

  4. 저장을 누릅니다.

참조

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
