ACC 제공자 관련 속성

JACC 제공자에 사용 가능한 속성은 다음과 같습니다.

repository

JACC 정책 파일이 들어 있는 디렉토리입니다. default GlassFish Server JACC 제공자의 경우, 기본 디렉토리는 ${com.sun.aas.instanceRoot}/generated/policy입니다. 이 속성은 기본적으로 simple GlassFish Server JACC 제공자에 대해 정의되지 않습니다.
