모듈 기술자

모듈 기술자 페이지에서는 응용 프로그램의 모듈 기술자를 볼 수 있습니다.

모듈 기술자 페이지에는 다음 옵션이 포함되어 있습니다.

배치 기술자

응용 프로그램 이름, 하위 구성 요소 이름(적용 가능한 경우), 기술자 파일 이름을 표시하는 테이블입니다.

관련 작업

  • 응용 프로그램 기술자를 보는 방법

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 배치된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스 끝점 정보를 보는 방법

  • 응용 프로그램을 실행하는 방법

  • 응용 프로그램 클라이언트를 실행하는 방법

  • 응용 프로그램 클라이언트 스텁을 다운로드하는 방법

  • 배치된 응용 프로그램을 재배치하는 방법

  • 배치된 응용 프로그램을 재로드하는 방법

  • 응용 프로그램 범위 리소스를 보는 방법

  • 응용 프로그램 범위 리소스를 수정하는 방법

관련 asadmin 명령

list-applications(1)
