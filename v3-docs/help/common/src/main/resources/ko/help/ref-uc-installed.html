설치된 구성 요소

설치된 구성 요소 페이지에서는 설치된 설치 이미지의 구성 요소에 대한 정보를 제공합니다.

설치된 구성 요소를 제거하려면 먼저 GlassFish 서버 도메인을 중지해야 합니다. 그런 후 다음 명령 중 하나를 사용하십시오.

  • glassfish3/bin/updatetool - 업데이트 도구 그래픽 유틸리티를 시작합니다.

  • glassfish3/bin/pkg - 업데이트 도구의 명령줄 버전입니다.

pkg 명령 사용에 대해서는 Oracle GlassFish Server 3.1 Administration Guide의 11 장, Extending and Updating GlassFish Server를 참조하십시오. 업데이트 도구 유틸리티에는 자체 온라인 도움말이 있습니다.

구성 요소 제거가 끝난 후 서버를 다시 시작하십시오.

설치된 구성 요소 페이지에는 다음 옵션이 포함되어 있습니다.

설치 경로

설치 이미지의 루트 또는 최상위 수준 디렉토리에 대한 파일 시스템 경로입니다.

Installed Components

설치된 각 구성 요소의 경우 이 테이블에서 제공되는 정보는 다음과 같습니다.

구성 요소

구성 요소의 이름입니다.

범주

예를 들어 Application Server, 시스템 도구 및 데이터베이스와 같이 구성 요소가 연결된 소프트웨어의 범주입니다. 구성 요소가 만들어지면 범주가 구성 요소의 작성자에 의해 해당 구성 요소에 할당됩니다.

버전

구성 요소를 제공하는 패키지의 버전 번호입니다.

설치된 크기

구성 요소의 설치 시 사용하는 디스크 공간의 크기입니다.

소스

구성 요소를 얻는 네트워크 저장소의 이름입니다.
