모듈 로그 레벨

모듈 로그 레벨 페이지에서는 개별 모듈에 대한 로깅 레벨을 구성할 수 있습니다.

각 모듈에 대해 제공되는 정보는 다음과 같습니다.

로거 이름

모듈에 대한 로거의 이름입니다(예: javax.enterprise.system.tools.admin ).

로거 레벨

모듈에 대한 현재 로깅 레벨입니다.

로깅 설정 테이블에는 다음 옵션도 포함되어 있습니다.

로거 추가

로거 모듈을 추가하는 단추입니다. 이 단추를 누르면 로거 설정 테이블에 행이 추가됩니다. 로거 모듈 정보를 새 행에 입력합니다.

로거 삭제

선택한 모듈을 하나 이상 삭제하는 단추입니다.

레벨

모든 선택된 모듈에 적용되는 로깅 레벨을 선택할 수 있는 드롭다운 목록입니다.

레벨 변경

선택한 하나 이상의 모듈에 대해 로깅 레벨을 변경하는 단추입니다.

다음의 로그 레벨을 사용할 수 있습니다. 높은 것부터 낮은 것 순서로 나열되어 있습니다.

SEVERE

정상적인 프로그램 실행을 방해하는 이벤트입니다.

WARNING

예외 사항을 포함한 경고입니다.

INFO

오류를 제외한 서버 구성 또는 서버 상태에 관련된 메시지입니다.

CONFIG

서버 구성에 관련된 메시지입니다.

FINE

최저 세부 정보 표시.

FINER

중간 세부 정보 표시.

FINEST

최대 세부 정보 표시.

OFF

로깅 메시지 없음

관련 작업

  • 일반 로깅 설정을 구성하는 방법

  • 로그 레벨을 구성하는 방법

  • DAS에 대한 로그 파일을 보는 방법

관련 asadmin 명령

  • collect-log-files(1)

  • list-log-levels(1)

  • rotate-log(1)

  • set-log-levels(1)
