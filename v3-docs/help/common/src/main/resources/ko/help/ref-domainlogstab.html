도메인 로그

[도메인 로그] 페이지에서는 GlassFish Server에 대한 로그 파일을 수집하고 다운로드할 수 있습니다.

[도메인 로그] 페이지에는 다음 옵션이 포함되어 있습니다. 로그를 수집할 클러스터 또는 서버 인스턴스를 한 개 선택할 수 있습니다.

클러스터

클러스터에 대한 드롭다운 목록입니다(클러스터가 있는 경우).

인스턴스

서버 인스턴스에 대한 드롭다운 목록입니다.

로그 수집

로그를 log-files-name.zip 파일에 수집하는 단추입니다. 여기서 name은 클러스터 또는 인스턴스의 이름입니다.

브라우저의 다운로드 방식을 사용하여 log-files-name.zip 파일이 다운로드됩니다.

관련 작업

  • 도메인 로그를 수집 및 다운로드하는 방법

  • DAS에 대한 로그 파일을 보는 방법

동일한 asadmin 하위 명령

collect-log-files(1)
