관리 영역에서 사용자 삭제

  1. 구성 노드에서 관리 영역 사용자를 관리할 서버 인스턴스나 클러스터 구성을 확장합니다.
  2. 보안 노드를 확장합니다.
  3. 영역 노드를 확장합니다.
  4. admin-realm 노드를 선택합니다.

    영역 편집 페이지가 열립니다.

  5. 영역 편집 페이지에서 사용자 관리 버튼을 누릅니다.

    관리 사용자 페이지가 열립니다.

  6. 관리 사용자 페이지에서 삭제할 사용자 이름의 왼쪽에 있는 확인란을 선택합니다.
  7. 삭제를 누릅니다.

    삭제를 확인하는 메시지가 표시됩니다.

  8. 사용자를 삭제하려면 확인을 누릅니다.
