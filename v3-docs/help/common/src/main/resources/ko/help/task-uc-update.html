구성 요소에 사용 가능한 업데이트 보기

사용 가능한 업데이트 페이지에서는 설치된 설치 이미지의 최신 버전 구성 요소에 대한 정보를 제공합니다.

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 열립니다.

  2. 사용 가능한 애드온 페이지에서 사용 가능한 업데이트 탭을 누릅니다.

    사용 가능한 업데이트 페이지가 열립니다. 업데이트를 사용할 수 없는 경우 테이블이 비어 있습니다.

다음 순서

사용 가능한 업데이트를 설치하려면 먼저 GlassFish 서버 도메인을 중지해야 합니다. 그런 후 다음 명령 중 하나를 사용하십시오.

  • as-install-parent/bin/updatetool - 업데이트 도구 그래픽 유틸리티를 시작합니다.

  • as-install-parent/bin/pkg - 업데이트 도구의 명령줄 버전

pkg 명령 사용에 대해서는 Oracle GlassFish Server 3.1 Administration Guide의 11 장, Extending and Updating GlassFish Server를 참조하십시오. 업데이트 도구 유틸리티에는 자체 온라인 도움말이 있습니다.

업데이트 설치가 끝난 후 서버를 다시 시작하십시오.

참조

  • 애드온 구성 요소 설치 방법

  • 설치된 구성 요소를 보는 방법

  • 구성 요소에 대한 자세한 내용을 보는 방법

  • 네트워크 저장소 액세스를 위해 네트워크 프록시 설정을 구성하는 방법
