업데이트 툴 주요 개념

업데이트 툴는 해당 시스템에서 단일 파일 시스템 디렉토리 아래 포함된 GlassFish Server 소프트웨어 구성 요소의 설치를 관리합니다.

업데이트 툴의 관리 콘솔 버전에서는 다음을 수행할 수 있습니다.

  • 추가 기능 구성 요소 설치

  • 설치된 구성 요소에 대한 업데이트 보기

  • 설치된 구성 요소 보기

설치된 구성 요소를 업데이트 또는 제거하려면 GlassFish Server 도메인을 정지한 후 다음 명령 중 하나를 사용해야 합니다.

  • as-install-parent/bin/updatetool - 업데이트 툴 그래픽 유틸리티를 시작합니다.

  • as-install-parent/bin/pkg - 업데이트 툴의 명령행 버전입니다.

다음 섹션에서 업데이트 툴의 주요 개념에 대해 설명합니다.

구성 요소

구성 요소는 업데이트 툴가 관리하는 설치 가능한 단위입니다. 구성 요소는 설치 이미지 내의 시스템에서 단위로 설치되는 파일로 구성됩니다.

설치 이미지

설치 이미지는 디렉토리의 내용을 포함하여 소프트웨어 구성 요소가 포함된 디렉토리입니다. 설치 이미지는 GlassFish Server 소프트웨어를 다운로드하여 설치한 후 업데이트 툴에 추가됩니다. GlassFish Server 소프트웨어 구성 요소의 추가 및 업데이트를 사용으로 설정하기 위해 각 설치 이미지에 구성 요소 설치에 대한 정보가 포함됩니다.

네트워크 저장소

네트워크 저장소에는 다음 항목이 포함되어 있습니다.

  • 설치 이미지에 추가할 수 있는 구성 요소

  • 설치 이미지의 기존 구성 요소에 대한 업데이트

설치 이미지는 하나 이상의 네트워크 저장소와 연결됩니다. 각 네트워크 저장소는 이름 및 연관된 URL에 의해 표시됩니다. 저장소 이름은 각 설치 이미지에 대해 고유하며, 저장소 및 적절하게 정의된 내용 범위를 관리하는 조직 또는 프로젝트 모두를 표시합니다. GlassFish Server에 대해 미리 구성된 네트워크 저장소에 대해 알아보려면 Oracle GlassFish Server Administration Guide의 Preconfigured Repositories for GlassFish Server를 참조하십시오.

추가 기능 구성 요소

추가 기능 구성 요소는 아직 설치되지 않은 설치 이미지의 구성 요소입니다.
