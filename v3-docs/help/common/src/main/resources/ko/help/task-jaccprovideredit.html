JACC 제공자를 편집하는 방법

  1. 탐색 트리의 [구성] 노드에서 JACC 제공자를 생성할 GlassFish Server 인스턴스나 클러스터를 확장합니다.

    모든 인스턴스와 클러스터에서 사용할 수 있는 제공자를 생성하려면 default-config 노드를 선택합니다.

  2. 확장된 인스턴스나 클러스터 구성 노드에서 [보안] 노드를 확장합니다.

  3. [보안] 노드에서 [JACC 제공자] 노드를 선택합니다.

    [JACC 제공자] 페이지가 열립니다.

  4. [JACC 제공자] 페이지에서 편집 중인 JACC 제공자의 이름을 누릅니다.

    [JACC 제공자 편집] 페이지가 열립니다.

  5. [JACC 제공자 편집] 페이지의 [정책 구성] 필드에 정책 구성 팩토리를 구현하는 클래스의 이름을 입력합니다.

  6. [정책 제공자] 필드에 정책 팩토리를 구현하는 클래스 이름을 입력합니다.

  7. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    JACC 제공자에 사용 가능한 추가 속성에 대한 설명은 ACC 제공자 관련 속성을(를) 참조하십시오.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  8. [저장]을 누릅니다.
