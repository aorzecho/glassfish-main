자원 모니터링 데이터 보기

시작하기 전에

모니터링 데이터를 보려면 먼저 모니터링을 구성해야 합니다. 자세한 내용은 모니터링 서비스를 구성하는 방법을 참조하십시오.

  1. 탐색 트리에서 서버(관리 서버) 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 모니터 탭을 누릅니다.

    응용 프로그램 모니터링 페이지가 열립니다.

  3. 자원 탭을 누릅니다.

    자원 모니터링 페이지가 열립니다.

  4. 보기 드롭다운 목록에서 모니터링 데이터를 볼 자원을 선택합니다.
  5. (옵션) 갱신 버튼을 눌러 표시된 모니터링 데이터를 업데이트합니다.
참조

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 관리자 암호를 변경하는 방법

  • 시스템 등록 정보를 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법
