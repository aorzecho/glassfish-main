보안 관리자를 비활성화하는 방법

GlassFish Server의 보안 관리자를 비활성화하면 일부 유형의 응용 프로그램에 대한 성능을 높일 수 있습니다. 보안 관리자를 비활성화하더라도 Java 플랫폼 권한 부여 및 인증 기능은 계속 작동합니다. 개발 환경에서는 보안 관리자를 비활성화할 수 있지만 프로덕션 환경에서는 비활성화하지 말아야 합니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 JVM 설정 노드를 선택합니다.

    JVM 일반 설정 페이지가 열립니다.

  3. JVM 일반 설정 페이지에서 JVM 옵션 탭을 누릅니다.

    JVM 옵션 페이지가 열립니다.

  4. -Djava.security.policy 옵션이 포함된 값 필드 왼쪽의 확인란을 선택합니다.
  5. 삭제를 누릅니다.
  6. 저장을 누릅니다.
  7. 서버를 다시 시작합니다.

참조

  • JVM 일반 설정을 구성하는 방법

  • JVM 경로 설정을 구성하는 방법

  • JVM 옵션을 구성하는 방법

  • JVM 프로필러 설정을 구성하는 방법
