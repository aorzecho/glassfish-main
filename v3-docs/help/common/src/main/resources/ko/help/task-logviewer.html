DAS에 대한 로그 파일을 보는 방법

  1. 탐색 트리에서 서버(관리 서버) 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 로그 파일 보기 버튼을 누릅니다.

    로그 뷰어 창이 열립니다.

참조

  • 일반 로깅 설정을 구성하는 방법

  • 로그 수준을 구성하는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • DAS에 대한 일반 정보를 보는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 관리자 암호를 변경하는 방법

  • 시스템 등록 정보를 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 자원 모니터링 데이터 보기
