새 비밀번호 별칭

새 [비밀번호 별칭] 페이지를 사용하여 비밀번호 파일이나 도메인 구성 파일에 사용할 새 비밀번호 별칭을 생성합니다.

새 [비밀번호 별칭] 페이지에는 다음 옵션이 포함되어 있습니다.

별칭 이름

별칭 이름입니다. 이 이름은 도메인의 모든 비밀번호 별칭에서 고유해야 합니다.

비밀번호

비밀번호 별칭이 암호화된 형식으로 저장되는 비밀번호입니다.

비밀번호 확인

비밀번호 별칭이 암호화된 형식으로 저장되는 비밀번호입니다. 비밀번호를 잘못 입력하지 않았는지 확인하기 위해 비밀번호와 비밀번호 확인 값이 비교됩니다.
