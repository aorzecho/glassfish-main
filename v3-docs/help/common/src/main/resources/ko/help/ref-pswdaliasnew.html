새 암호 별칭

새 암호 별칭 페이지를 사용하여 암호 파일이나 도메인 구성 파일에 사용할 새 암호 별칭을 만듭니다.

새 암호 별칭 페이지에는 다음 옵션이 포함되어 있습니다.

별칭 이름

별칭 이름입니다. 이 이름은 도메인의 모든 암호 별칭에서 고유해야 합니다.

암호

암호 별칭이 암호화된 형식으로 저장되는 암호입니다.

암호 확인

암호 별칭이 암호화된 형식으로 저장되는 암호입니다. 암호를 잘못 입력하지 않았는지 확인하기 위해 암호와 암호 확인 값이 비교됩니다.
