업데이트 도구 구성

구성 페이지는 업데이트 도구에 대한 구성 정보를 제공하므로 이 페이지를 사용하여 업데이트 도구에서 인터넷에 연결하는 방법을 변경할 수 있습니다. 업데이트 도구 구성의 다른 부분을 변경하려면 as-install-parent/bin에 있는 독립 실행형 updatetool 응용 프로그램을 사용합니다.

업데이트 도구 구성 페이지에는 다음 옵션이 포함되어 있습니다.

설치 경로

설치 이미지의 루트 또는 최상위 수준 디렉토리에 대한 파일 시스템 경로입니다.

기본 저장소

이 설치 이미지와 연결된 기본 네트워크 저장소의 이름이며 예를 들면 glassfish.org입니다.

인터넷 연결

인터넷에 GlassFish 서버 연결 방법:

직접 연결

이 옵션을 선택하면 GlassFish 서버가 인터넷에 직접 연결됩니다.

프록시를 통해

이 옵션을 선택하면 GlassFish 서버가 프록시 서버를 통해 인터넷에 연결됩니다. 이 옵션을 선택하면 프록시 호스트 필드 및 프록시 포트 필드가 이 페이지에 추가됩니다.

프록시 호스트

GlassFish 서버가 실행 중인 시스템에 대한 프록시 서버의 호스트 이름입니다.

이 필드는 GlassFish 서버가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

프록시 포트

GlassFish 서버가 실행 중인 시스템에 대한 프록시 서버의 포트 번호입니다.

이 필드는 GlassFish 서버가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

Repository

예를 들어 dev.glassfish.org와 같은 저장소의 이름입니다.
