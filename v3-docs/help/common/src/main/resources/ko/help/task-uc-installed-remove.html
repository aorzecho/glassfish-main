설치된 구성 요소를 보는 방법

설치된 구성 요소 페이지에서는 설치된 설치 이미지의 구성 요소에 대한 정보를 제공합니다.

구성 요소 제거에는 다음 항목의 제거가 포함됩니다.

  • 구성 요소의 원래 설치에 의해 적용된 모든 파일 및 디렉토리

  • 원래 설치 후 구성 요소에 대한 업데이트에 의해 적용된 모든 파일 및 디렉토리

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 열립니다.

  2. 사용 가능한 애드온 페이지에서 설치된 구성 요소 탭을 누릅니다.

    설치된 구성 요소 페이지가 열립니다.

다음 순서

설치된 구성 요소를 제거하려면 먼저 GlassFish Server 도메인을 중지해야 합니다. 그런 후 다음 명령 중 하나를 사용하십시오.

  • as-install-parent/bin/updatetool - 업데이트 도구 그래픽 유틸리티를 시작합니다.

  • as-install-parent/bin/pkg - 업데이트 도구의 명령줄 버전

pkg 명령 사용에 대해서는 Oracle GlassFish Server 3.1 Administration Guide의 11 장, Extending and Updating GlassFish Server를 참조하십시오. 업데이트 도구 유틸리티에는 자체 온라인 도움말이 있습니다.

구성 요소 제거가 끝난 후 서버를 다시 시작하십시오.
