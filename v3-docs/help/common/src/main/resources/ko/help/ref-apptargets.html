응용 프로그램 대상

응용 프로그램 대상 페이지를 사용하여 응용 프로그램을 활성화할 수 있는 대상 클러스터와 독립 실행형 서버 인스턴스를 볼 수 있습니다.

응용 프로그램 대상 페이지에는 다음 정보가 포함됩니다.

대상 이름

클러스터나 독립 실행형 서버 인스턴스의 이름입니다.

사용 가능

응용 프로그램이 대상에서 활성화된 경우 true가 표시되고 비활성화된 경우 false가 표시됩니다.

LB 활성화됨

대상에서 응용 프로그램이 로드 균형 조정에 대해 활성화된 경우 true가 표시되고 비활성화된 경우 false가 표시됩니다.

가상 서버

응용 프로그램을 대상의 특정 가상 서버와 연결하려면 해당 대상에 대해 가상 서버 관리를 선택합니다.

대상 테이블에는 다음 옵션도 포함되어 있습니다.

대상 관리

응용 프로그램 대상을 관리하는 버튼입니다.

추가 작업

다음 작업에 대한 드롭다운 목록입니다.

활성화

선택한 대상에서 응용 프로그램을 활성화하는 작업입니다.

비활성화

선택한 대상에서 응용 프로그램을 비활성화하는 작업입니다.

로드 밸런서 활성화

선택한 대상에서 로드 균형을 조정하기 위해 응용 프로그램을 활성화하는 작업입니다.

로드 밸런서 비활성화

선택한 대상에서 로드 균형을 조정하기 위해 응용 프로그램을 비활성화하는 작업입니다.

관련 작업

  • 응용 프로그램의 설명자를 보는 방법

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스에 대한 종단점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 로드하는 방법
