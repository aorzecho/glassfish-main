수명 주기 모듈

[수명 주기 모듈] 페이지에서는 수명 주기 모듈을 구성할 수 있습니다.

수명 주기 모듈은 서버의 수명 주기에서 한 개 이상의 이벤트로 실행되면 작업을 수행합니다. 발생할 수 있는 트리거 서버 이벤트에는 초기화, 시작, 요청 서비스 준비 및 종료 등이 있습니다. 수명 주기 모듈은 Java EE 사양의 일부가 아니지만 GlassFish Server의 개선 사항입니다.

각 수명 주기 모듈에 대해 제공되는 정보는 다음과 같습니다.

이름

수명 주기 모듈의 이름입니다.

사용 또는 상태

기본 서버 인스턴스(server)만 있는 경우 응용 프로그램이 사용으로 설정되면 체크 표시가 나타나고 사용 안함으로 설정되면 X가 표시됩니다.

클러스터 또는 다른 독립형 서버 인스턴스가 있는 경우 응용 프로그램이 배치되는 대상을 보려면 모든 대상에서 사용을 선택합니다.

로드 순서

수명 주기 모듈을 로드해야 할 상대적 순서를 지정하는 값입니다.

수명 주기 모듈 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 수명 주기 모듈을 생성하는 단추입니다.

삭제

선택한 수명 주기 모듈을 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 수명 주기 모듈을 사용하도록 설정하는 단추입니다. 기본 서버 인스턴스(server)만 있는 경우에 표시됩니다.

사용 안함

선택한 하나 이상의 수명 주기 모듈을 사용 안함으로 설정하는 단추입니다. 기본 서버 인스턴스(server)만 있는 경우에 표시됩니다.
