로그 레벨을 구성하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 로깅 설정을 구성할 서버 인스턴스나 클러스터 구성을 누릅니다.

    [구성] 페이지가 열립니다.

  3. [구성] 페이지에서 [로깅 설정]을 누릅니다.

    선택한 구성 대상에 대한 [로깅 설정] 페이지가 열립니다.

  4. [로깅 설정] 페이지에서 [로그 레벨] 탭을 누릅니다.

    [모듈 로그 레벨] 페이지가 열립니다.

  5. [모듈 로그 레벨] 페이지에서 다음과 같이 하나 이상의 모듈에 대한 로그 레벨을 변경합니다.

    • 개별 모듈에 대한 로그 레벨을 변경하려면 로그 레벨을 변경할 모듈 반대쪽의 드롭다운 목록에서 새 값을 선택합니다.

    • 모듈 그룹의 로그 레벨을 변경하는 방법

      1. 로그 레벨을 변경할 모듈의 이름 옆에 있는 체크 박스를 선택합니다.

      2. 테이블 맨 위의 드롭다운 목록에서 값을 선택합니다.

      3. [레벨 변경] 단추를 누릅니다.

    선택한 레벨 이상의 메시지가 로그에 기록됩니다. 다음 값 중 하나를 선택합니다(높은 레벨에서 낮은 레벨로 나열).

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • OFF

    로그 레벨의 의미에 대한 자세한 내용은 모듈 로그 레벨을(를) 참조하십시오. 기본 레벨은 INFO이므로 기본적으로 로그에는 INFO, WARNING 또는 SEVERE 레벨의 메시지가 포함됩니다.

  6. [저장]을 누릅니다.

    로그 레벨 설정에 대한 변경 사항은 바로 적용됩니다. 또한 서버를 재시작할 때 사용할 수 있도록 저장됩니다.
