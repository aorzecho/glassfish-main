JACC 제공자 편집

[JACC 제공자 편집] 페이지에서는 선택한 GlassFish Server 인스턴스나 클러스터에 대한 JACC 제공자를 수정할 수 있습니다.

[JACC 제공자 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

JACC 제공자의 이름입니다. 읽기 전용 필드입니다. 새 JACC 제공자를 생성할 때만 이름을 지정할 수 있습니다.

정책 구성

제공자별 javax.security.jacc.PolicyConfigurationFactory를 구현하는 javax.security.jacc.PolicyConfigurationFactory.provider에 대한 정규화된 클래스 이름입니다.

정책 제공자

java.security.Policy를 구현하는 javax.security.jacc.policy.provider에 대한 정규화된 클래스 이름입니다.

추가 속성

JACC 제공자의 추가 속성입니다. JACC 제공자에 사용 가능한 추가 속성에 대한 설명은 ACC 제공자 관련 속성을(를) 참조하십시오.

관련 작업

  • JACC 제공자를 생성하는 방법

  • JACC 제공자를 편집하는 방법

  • JACC 제공자를 삭제하는 방법

관련 asadmin 명령

  • create-jacc-provider(1)

  • delete-jacc-provider(1)

  • list-jacc-providers(1)
