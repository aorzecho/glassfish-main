일반 정보

일반 정보 페이지에서는 GlassFish Server 설정을 확인하고 Java Virtual Machine 데이터를 볼 수 있습니다.

일반 정보 페이지에는 다음 정보가 포함됩니다.

중지 버튼

GlassFish Server를 중지하려면 중지 버튼을 누릅니다.

다시 시작 버튼

GlassFish Server를 다시 시작하려면 다시 시작 버튼을 누릅니다.

로그 파일 보기 버튼

GlassFish Server 인스턴스나 클러스터에 대한 로그 파일을 보려면 로그 파일 보기 버튼을 누릅니다.

이름

서버의 이름입니다.

JVM

JVM 보고서를 누르면 별도의 창이 열리고 요약 보고서, 메모리 관리 및 가비지 컬렉션 보고서, 클래스 로딩 보고서 및 현재 스레드 덤프를 포함한 Java Virtual Machine에 대한 보고서가 표시됩니다.

구성 디렉토리

domain.xml과 같은 구성 파일이 포함된 호스트 시스템의 디렉토리입니다.

설치된 버전

제품 버전 식별자입니다.

디버그

디버깅을 사용하도록 설정되어 있는지 여부를 식별합니다. 디버깅을 사용하도록 설정된 경우 포트 번호도 표시됩니다.

HTTP 포트

HTTP 요청 및 응답에 대해 현재 구성된 포트입니다.

IIOP 포트

IIOP 요청 및 응답에 대해 현재 구성된 포트입니다.
