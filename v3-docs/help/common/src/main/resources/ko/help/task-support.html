GlassFish 서버에 대한 GlassFish 뉴스 페이지를 보는 방법

GlassFish 뉴스 페이지에서는 뉴스레터, 기사, 토론 포럼, 관련 블로그 등 Oracle GlassFish 서버와 관련한 추가 정보로 연결되는 링크 집합을 제공합니다.

  1. 탐색 트리에서 일반 작업 노드를 선택합니다.

    일반 작업 페이지가 열립니다.

  2. GlassFish 뉴스에서 GlassFish 뉴스 링크를 누릅니다.

    GlassFish 뉴스 페이지가 별도의 브라우저 탭이나 창에 열립니다.
