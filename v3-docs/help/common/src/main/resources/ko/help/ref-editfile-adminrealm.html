FileRealm 클래스 관련 속성

file 영역의 경우 다음 속성이 필요합니다.

JAAS 컨텍스트

JAAS(Java Authentication and Authorization Service) 컨텍스트는 이 영역에서 사용할 로그인 모듈의 식별자입니다. 유일한 적합한 값은 fileRealm입니다.

키 파일

서버가 이 영역에 대해 사용자, 그룹 및 비밀번호 정보를 저장하는 파일의 전체 경로 및 이름입니다. 기본값은 file 영역의 경우 domain-dir/config/keyfile이고, admin-realm 영역의 경우 domain-dir/config/admin-keyfile입니다.

file 영역의 키 파일은 처음에 비어 있으므로 사용자가 file 영역을 사용하기 전에 파일 사용자 페이지에서 추가해야 합니다.

처음에 admin-realm 영역의 키 파일에는 암호화된 형식의 관리자 이름, 관리자 비밀번호 및 이 사용자가 속한 그룹(기본적으로 asadmin)이 포함되어 있습니다. admin-realm에 사용자를 추가하는 방법은 관리 툴에 대한 액세스 권한 부여을(를) 참조하십시오.

주:

admin-realm의 asadmin 그룹에 속한 사용자는 관리 콘솔 및 asadmin 도구를 사용할 수 있는 권한이 부여됩니다. 이 그룹에는 서버 관리 권한을 가진 사용자만을 추가하십시오.

file 영역의 경우 다음과 같은 선택적 속성을 사용할 수 있습니다.

그룹 지정

콤마로 구분된 그룹 이름 목록입니다. 적합한 인증서를 제공하는 모든 클라이언트가 이러한 그룹에 지정됩니다. 예를 들어 employee,manager는 사용자 그룹의 이름입니다.
