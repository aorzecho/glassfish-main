시스템 등록 정보

시스템 등록 정보 페이지에는 선택한 명명된 구성에 정의된 Java 시스템 등록 정보 목록이 표시됩니다. Java 시스템 등록 정보는 GlassFish Server 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다.

이러한 등록 정보는 구성을 참조하는 모든 GlassFish Server 인스턴스에 대한 기본 포트 설정 값을 정의합니다.

구성의 미리 정의된 포트 설정에 대한 자세한 내용은 사전 정의된 시스템 등록 정보 를 참조하십시오.

각 등록 정보에 대해 표시되는 정보는 다음과 같습니다.

인스턴스 변수 이름

시스템 등록 정보의 이름입니다.

기본값

인스턴스가 참조하는 명명된 구성의 등록 정보에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

인스턴스 값

등록 정보에 대한 인스턴스 값 페이지의 링크입니다.

또한 추가 등록 정보 테이블에는 다음 옵션이 포함되어 있습니다.

등록 정보 추가

등록 정보를 추가하는 버튼입니다. 이 버튼을 누르면 추가 등록 정보 테이블에 행이 추가됩니다.

등록 정보 삭제

하나 이상의 선택된 등록 정보를 삭제하는 버튼입니다. 삭제된 등록 정보는 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

또한 시스템 등록 정보 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

동적 재구성

이 옵션을 사용하도록 설정하면 구성 변경 사항은 인스턴스를 다시 시작하지 않아도 구성을 참조하는 인스턴스에 적용됩니다.

이 구성을 사용하는 클러스터 및/또는 인스턴스

각 클러스터나 인스턴스 실행 여부 표시로 구성을 참조하는 클러스터 및 인스턴스의 목록입니다.

  • 클러스터나 인스턴스가 실행되고 있는 경우 체크 표시(✓)가 나타납니다.

  • 클러스터나 인스턴스가 중지된 경우 느낌표(!)가 표시됩니다.

관련 작업

  • 구성 등록 정보의 기본값을 설정하는 방법

동일한 asadmin 하위 명령

  • create-system-properties(1)

  • delete-system-property(1)

  • list-system-properties(1)
