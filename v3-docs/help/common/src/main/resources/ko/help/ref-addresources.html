리소스 추가

리소스 추가 페이지에서는 glassfish-resources.xml 파일의 리소스를 지정된 대상에 하나 이상 생성할 수 있습니다.

리소스 추가 페이지에는 다음 정보가 포함됩니다.

위치

glassfish-resources.xml 파일의 위치입니다.

서버에 업로드할 XML 파일

원격 시스템에 있는 파일의 경로입니다.

GlassFish Server에서 액세스 가능한 로컬 XML 파일

DAS(도메인 관리 서버) 시스템에 있는 파일의 경로입니다.

대상

클러스터 및 독립형 서버 인스턴스에 대한 드롭다운 목록입니다.

관련 작업

XML 파일에 구성된 리소스를 추가하는 방법

관련 asadmin 명령

add-resources(1)
