라이프사이클 모듈 대상

라이프사이클 모듈 대상 페이지를 사용하여 라이프사이클 모듈을 활성화할 수 있는 대상 클러스터와 독립 실행형 서버 인스턴스를 볼 수 있습니다.

라이프사이클 모듈 대상 페이지에는 다음 정보가 포함됩니다.

대상 이름

클러스터나 독립 실행형 서버 인스턴스의 이름입니다.

사용 가능

라이프사이클 모듈이 대상에서 활성화된 경우 true가 표시되고 비활성화된 경우 false가 표시됩니다.

대상 테이블에는 다음 옵션도 포함되어 있습니다.

대상 관리

라이프사이클 모듈 대상을 관리하는 버튼입니다.

추가 작업

다음 작업에 대한 드롭다운 목록입니다.

활성화

선택한 대상에서 라이프사이클 모듈을 활성화하는 작업입니다.

비활성화

선택한 대상에서 라이프사이클 모듈을 비활성화하는 작업입니다.
