감사 모듈 편집

감사 모듈 편집 페이지에서는 감사 모듈을 수정할 수 있습니다.

감사 모듈 편집 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

감사 모듈의 이름입니다. 읽기 전용 필드입니다. 새 감사 모듈을 생성할 때만 이름을 지정할 수 있습니다.

클래스 이름

이 모듈을 구현하는 클래스의 전체 이름입니다. 기본 감사 모듈의 클래스 이름은 com.sun.enterprise.security.Audit입니다.

추가 속성

감사 모듈의 추가 속성입니다. 감사 모듈에 사용 가능한 추가 속성에 대한 설명은 감사 모듈 관련 속성을(를) 참조하십시오.

관련 작업

  • 감사 모듈을 생성하는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법

관련 asadmin 명령

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
