사용 가능한 업데이트

[사용 가능한 업데이트] 페이지에서는 설치된 설치 이미지의 최신 버전 구성 요소에 대한 정보를 제공합니다.

사용 가능한 업데이트를 설치하려면 먼저 GlassFish Server 도메인을 정지해야 합니다. 그런 후 다음 명령 중 하나를 사용하십시오.

  • glassfish3/bin/updatetool - 업데이트 툴 그래픽 유틸리티를 시작합니다.

  • glassfish3/bin/pkg - 업데이트 툴의 명령행 버전입니다.

pkg 명령 사용에 대한 자세한 내용은 Oracle GlassFish Server Administration Guide의 11 장, Extending and Updating GlassFish Server를 참조하십시오. 업데이트 툴 유틸리티에는 자체 온라인 도움말이 있습니다.

업데이트 설치가 끝난 후 서버를 재시작하십시오.

사용 가능한 업데이트 페이지에는 다음 옵션이 포함되어 있습니다.

설치 경로

설치 이미지의 루트 또는 최상위 레벨 디렉토리에 대한 파일 시스템 경로입니다.
