영역을 삭제하는 방법

  1. 구성 노드에서 영역을 구성할 서버 인스턴스나 클러스터 구성을 확장합니다.
  2. 보안 노드를 확장합니다.
  3. 영역 노드를 확장합니다.

    영역 페이지가 열립니다.

  4. 영역 페이지에서 삭제할 영역 이름의 왼쪽에 있는 확인란을 선택합니다.
  5. 삭제를 누릅니다.
