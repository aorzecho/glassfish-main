대상 관리

대상 관리 페이지를 사용하여 응용 프로그램을 활성화할 수 있는 대상 클러스터와 독립 실행형 서버 인스턴스를 변경할 수 있습니다. 응용 프로그램은 선택한 대상 열의 대상에서만 활성화할 수 있습니다.

대상 관리 페이지에는 다음 정보가 포함됩니다.

사용 가능한 대상

응용 프로그램이 배포되지 않는 클러스터와 인스턴스입니다.

선택한 대상

응용 프로그램이 배포되는 클러스터와 인스턴스입니다.

추가

선택한 대상을 사용 가능한 대상에서 선택한 대상으로 이동하는 버튼입니다.

모두 추가

사용 가능한 대상을 모두 선택한 대상으로 이동하는 버튼입니다.

제거

선택한 대상을 선택한 대상에서 사용 가능한 대상으로 이동하는 버튼입니다.

모두 제거

선택한 대상을 모두 사용 가능한 대상으로 이동하는 버튼입니다.

관련 작업

  • 응용 프로그램의 설명자를 보는 방법

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스에 대한 종단점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 로드하는 방법
