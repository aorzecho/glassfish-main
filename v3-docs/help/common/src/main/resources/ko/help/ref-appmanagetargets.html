대상 관리

[대상 관리] 페이지를 사용하여 응용 프로그램을 사용할 수 있는 대상 클러스터와 독립형 서버 인스턴스를 변경할 수 있습니다. 응용 프로그램은 선택한 대상 열에 있는 대상에서만 사용으로 설정할 수 있습니다.

[대상 관리] 페이지에는 다음 정보가 포함됩니다.

사용 가능한 대상

응용 프로그램이 배치되지 않는 클러스터와 인스턴스입니다.

선택한 대상

응용 프로그램이 배치되는 클러스터와 인스턴스입니다.

추가

선택한 대상을 사용 가능한 대상에서 선택한 대상으로 이동하는 단추입니다.

모두 추가

사용 가능한 대상을 모두 선택한 대상으로 이동하는 단추입니다.

제거

선택한 대상을 선택한 대상에서 사용 가능한 대상으로 이동하는 단추입니다.

모두 제거

선택한 대상을 모두 사용 가능한 대상으로 이동하는 단추입니다.

관련 작업

  • 응용 프로그램 기술자를 보는 방법

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 배치된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 웹 서비스 끝점 정보를 보는 방법

  • 응용 프로그램을 실행하는 방법

  • 응용 프로그램 클라이언트를 실행하는 방법

  • 응용 프로그램 클라이언트 스텁을 다운로드하는 방법

  • 배치된 응용 프로그램을 재배치하는 방법

  • 배치된 응용 프로그램을 재로드하는 방법

  • 응용 프로그램 범위 리소스를 보는 방법

  • 응용 프로그램 범위 리소스를 수정하는 방법
