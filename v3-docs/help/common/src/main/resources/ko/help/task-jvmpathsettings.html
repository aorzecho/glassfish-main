JVM 경로 설정을 구성하는 방법

  1. 탐색 트리에서 [구성] 노드를 선택합니다.

  2. Select the configuration that you are changing.

  3. [구성] 노드에서 [JVM 설정] 노드를 선택합니다.

    [JVM 일반 설정] 페이지가 열립니다.

  4. [JVM 일반 설정] 페이지에서 [경로 설정] 탭을 누릅니다.

    [JVM 경로 설정] 페이지가 열립니다.

  5. 원시 라이브러리 경로 접두어 필드에 원시 라이브러리 경로 앞에 추가할 JAR 파일의 전체 경로를 입력합니다.

    여러 파일을 덧붙이려면 경로를 한 줄에 하나씩 입력합니다.

  6. 원시 라이브러리 경로 접미어 필드에 원시 라이브러리 경로에 추가할 JAR 파일의 전체 경로를 입력합니다.

    여러 파일을 추가하려면 경로를 한 줄에 하나씩 입력합니다.

  7. [저장]을 누릅니다.

  8. 서버를 재시작합니다.
