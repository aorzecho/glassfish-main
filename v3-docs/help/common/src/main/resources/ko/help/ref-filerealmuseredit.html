파일 영역 사용자 편집

파일 영역 사용자 편집 페이지에서는 현재 영역의 사용자 암호를 변경할 수 있습니다.

파일 영역 사용자 편집 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

영역 이름

현재 선택된 영역의 이름입니다. 읽기 전용 필드입니다.

사용자 ID

편집 중인 계정의 이름입니다. 읽기 전용 필드입니다.

그룹 목록

이 사용자가 속한 그룹입니다. 그룹 이름은 쉼표로 구분합니다.

admin-realm 사용자의 경우 이는 읽기 전용 필드이며 그룹은 asadmin입니다.

새 암호

이 사용자의 새 암호입니다.

새 암호 확인

이 사용자의 새 암호입니다.

관련 작업

  • 사용자를 파일 영역에 추가하는 방법

  • 파일 영역 사용자 정보를 편집하는 방법

  • 파일 영역에서 사용자를 삭제하는 방법

관련 asadmin 명령

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
