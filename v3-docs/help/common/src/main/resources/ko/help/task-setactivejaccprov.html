활성 JACC 공급자를 설정하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  3. JACC 드롭다운 목록에서 서버에서 사용할 JACC 공급자를 선택합니다.

    기본적으로 default 및 simple의 두 가지 JACC 공급자를 사용할 수 있습니다. 기본 공급자는 default입니다.

  4. 저장을 누릅니다.
