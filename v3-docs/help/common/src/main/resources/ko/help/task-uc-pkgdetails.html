구성 요소에 대한 자세한 정보를 보는 방법

업데이트 툴은 설치된 구성 요소, 설치된 구성 요소에 사용 가능한 업데이트 및 추가 기능 구성 요소에 대한 자세한 정보를 제공합니다.

  1. 탐색 트리에서 업데이트 툴 노드를 선택합니다.

    [사용 가능한 추가 기능] 페이지가 표시됩니다.

  2. 구성 요소가 나열된 페이지의 탭을 누릅니다.

    구성 요소가 다음 페이지 중 하나에 나열됩니다.

    • 사용 가능한 추가 기능

    • 사용 가능한 업데이트

    • 설치된 구성 요소

  3. 구성 요소 테이블에서 구성 요소를 선택합니다.

    선택한 구성 요소의 구성 요소 세부 정보 페이지가 표시됩니다.

관련 항목

  • 구성 요소의 사용 가능한 업데이트를 확인하는 방법

  • 추가 기능 구성 요소를 설치하는 방법

  • 설치된 구성 요소를 확인하는 방법

  • 네트워크 저장소에 액세스할 수 있도록 네트워크 프록시 설정을 구성하는 방법
