구성 요소에 대한 자세한 내용을 보는 방법

업데이트 도구에서는 설치된 구성 요소, 설치된 구성 요소에 대해 사용 가능한 업데이트 및 애드온 구성 요소에 대한 자세한 내용을 제공합니다.

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 표시됩니다.

  2. 구성 요소가 나열된 페이지의 탭을 누릅니다.

    구성 요소가 다음 페이지 중 하나에 나열됩니다.

    • 사용 가능한 애드온

    • 사용 가능한 업데이트

    • 설치된 구성 요소

  3. 구성 요소 테이블에서 구성 요소를 선택합니다.

    선택한 구성 요소의 구성 요소 세부 정보 페이지가 표시됩니다.

참조

  • 구성 요소에 사용 가능한 업데이트 보기

  • 애드온 구성 요소 설치 방법

  • 설치된 구성 요소를 보는 방법

  • 네트워크 저장소 액세스를 위해 네트워크 프록시 설정을 구성하는 방법
