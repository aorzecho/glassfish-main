시스템 등록 정보를 관리하는 방법

  1. 탐색 트리에서 GlassFish Server 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 시스템 등록 정보 탭을 누릅니다.

    시스템 등록 정보 페이지가 열립니다.

  3. 추가 등록 정보 섹션에서 구성에 대한 시스템 등록 정보를 지정합니다.

    여러 개의 시스템 등록 정보가 사전 정의되어 있지만 이 페이지에는 나와 있지 않습니다. 자세한 내용은 사전 정의된 시스템 등록 정보를 참조하십시오.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  4. 저장을 누릅니다.

참조

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 관리자 암호를 변경하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 자원 모니터링 데이터 보기
