시스템 속성을 관리하는 방법

  1. 탐색 트리에서 GlassFish Server 노드를 선택합니다.

    [일반 정보] 페이지가 열립니다.

  2. [일반 정보] 페이지에서 [시스템 속성] 탭을 누릅니다.

    [시스템 속성] 페이지가 열립니다.

  3. [추가 속성] 섹션에서 구성에 대한 시스템 속성을 지정합니다.

    여러 개의 시스템 속성이 사전 정의되어 있지만 이 페이지에는 나와 있지 않습니다. 자세한 내용은 미리 정의된 시스템 속성을(를) 참조하십시오.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  4. [저장]을 누릅니다.

관련 항목

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 리소스 모니터링 데이터를 보는 방법
