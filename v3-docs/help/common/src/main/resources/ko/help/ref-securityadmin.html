보안 관리

[보안 관리] 페이지에서는 보안 관리를 사용 또는 사용 안함으로 설정할 수 있습니다.

보안 관리가 사용으로 설정된 경우:

  • 도메인의 인스턴스 및 DAS가 SSL 인증서를 사용하여 서로에게 보내는 메시지를 암호화합니다.

  • DAS는 원격 관리 클라이언트(예: asadmin 유틸리티 및 IDE)에서 보내는 관리 메시지를 허용합니다.

[보안 관리] 페이지에는 다음 정보가 포함됩니다.

보안 관리 사용

보안 관리를 사용으로 설정하는 단추입니다. 이 단추는 보안 관리가 사용으로 설정되지 않은 경우에만 나타납니다.

주:

도메인의 관리 사용자가 빈 비밀번호를 사용하는 경우 보안 관리를 사용으로 설정할 수 없습니다.
보안 관리 사용 안함

보안 관리를 사용 안함으로 설정하는 단추입니다. 이 단추는 보안 관리가 사용으로 설정된 경우에만 나타납니다.

관리 별칭

DAS의 SSL/TLS 인증서를 나타내는 별칭입니다. 이 별칭은 인스턴스에 대해 DAS를 식별하는 데 사용됩니다. 기본값은 s1as입니다. 이 옵션은 보안 관리가 사용으로 설정되지 않은 경우에만 나타납니다.

인스턴스 별칭

인스턴스의 SSL/TLS 인증서를 나타내는 별칭입니다. 이 별칭은 DAS에 대해 인스턴스를 식별하는 데 사용됩니다. 기본값은 glassfish-instance입니다. 이 옵션은 보안 관리가 사용으로 설정되지 않은 경우에만 나타납니다.

관련 작업

  • 보안 관리를 사용으로 설정하는 방법

  • 보안 관리를 사용 안함으로 설정하는 방법

관련 asadmin 명령

  • enable-secure-admin(1)

  • disable-secure-admin(1)
