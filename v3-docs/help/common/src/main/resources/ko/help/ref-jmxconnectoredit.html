JMX 커넥터 편집

[JMX 커넥터 편집] 페이지에서는 JSR 360 호환 JMX 커넥터를 편집할 수 있습니다.

[JMX 커넥터 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

JMX 커넥터의 이름입니다. 읽기 전용 필드입니다.

JMX 프로토콜

이 JMX 커넥터가 지원하는 프로토콜의 이름입니다. 읽기 전용 필드입니다.

JMXServiceURL

JMX 서비스 URL입니다. 읽기 전용 필드입니다.

보안

사용 체크 박스를 선택하면 JMX 통신이 암호화됩니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

주소

JMX 커넥터 서버 스텁이 등록된 이름 지정 서비스의 IP 주소입니다. 이는 실제 JMX 통신을 수행하는 서버 소켓의 포트가 아닙니다. RMI 레지스트리가 시작되는 네트워크 인터페이스의 주소입니다. 시스템에 여러 개의 네트워크 인터페이스가 있는 경우 특정 인터페이스만 선택되도록 이 값을 수정합니다. 기본값은 0.0.0.0입니다.

포트

이름 지정 서비스(RMI 레지스트리)가 RMI 클라이언트 접속을 수신하는 포트 번호입니다. 이 이름 지정 서비스는 RMI 스텁을 다운로드하는 데만 사용합니다. 기본 포트가 사용 중인 경우 빈 포트가 사용됩니다. 유효한 값은 1 - 65535입니다. UNIX 시스템에서 1 - 1024 포트를 수신하는 소켓을 생성하려면 수퍼유저 권한이 필요합니다. 기본값은 8686입니다.

영역 이름

특별 관리 영역을 나타내는 이름입니다. 모든 인증이 이 영역에 의해 처리됩니다. 기본값은 admin-realm입니다.

추가 속성

JMX 커넥터의 추가 속성입니다. 기본적으로 몇 개의 속성이 지정됩니다.
