감사 모듈을 생성하는 방법

감사는 GlassFish Server에서 오류나 보안 위반과 같은 중요한 이벤트를 기록하는 방법입니다. GlassFish Server은(는) 모든 인증 이벤트를 로그 파일에 기록합니다. 전체 액세스 로그는 GlassFish Server 액세스 이벤트의 순차적인 추적을 제공합니다.

GlassFish Server에서는 간단한 기본 감사 모듈을 제공합니다. 자세한 내용은 기본 감사 모듈을 사용하는 방법을(를) 참조하십시오.

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. [구성] 노드에서 [보안] 노드를 확장합니다.

  4. [보안] 노드 아래에서 [감사 모듈] 노드를 선택합니다.

    [감사 모듈] 페이지가 열립니다.

  5. [감사 모듈] 페이지에서 [새로 만들기]를 누릅니다.

    [새 감사 모듈] 페이지가 열립니다.

  6. [새 감사 모듈] 페이지의 [이름] 필드에 감사 모듈 이름을 입력합니다.

  7. [클래스 이름] 필드에 이 모듈을 구현하는 클래스의 전체 이름을 입력합니다.

    기본 감사 모듈의 클래스 이름은 com.sun.enterprise.security.Audit입니다.

  8. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    감사 모듈에 사용 가능한 추가 속성에 대한 설명은 감사 모듈 관련 속성을(를) 참조하십시오.

  9. [확인]을 누릅니다.

관련 항목

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
