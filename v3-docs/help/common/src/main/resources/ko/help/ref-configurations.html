구성

[구성] 페이지에서는 구성을 관리하고 해당 구성을 사용하는 독립형 서버 인스턴스 및 클러스터를 볼 수 있습니다.

default-config 구성은 GlassFish Server에 미리 정의되어 있습니다. server-config 구성은 도메인 생성 시 DAS(도메인 관리 서버)용으로 자동 생성됩니다. 자세한 내용은 Oracle GlassFish Server High Availability Administration Guide의 About Named Configurations를 참조하십시오.

각 구성에 대해 제공되는 정보는 다음과 같습니다.

이름

구성 이름입니다.

인스턴스

해당 구성을 사용하는 독립형 인스턴스 및 클러스터 목록입니다.

구성 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 구성을 생성하는 단추입니다.

삭제

선택한 구성을 하나 이상 삭제하는 단추입니다.
