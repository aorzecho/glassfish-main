구성

구성 페이지에서 구성을 관리하고 구성을 사용하는 독립 실행형 서버 인스턴스와 클러스터를 볼 수 있습니다.

각 구성에 대해 제공되는 정보는 다음과 같습니다.

이름

구성 이름입니다.

인스턴스

구성을 사용하는 독립 실행형 인스턴스와 클러스터 목록입니다.

구성 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 구성을 만드는 버튼입니다.

삭제

하나 이상의 선택된 구성을 삭제하는 버튼입니다.
