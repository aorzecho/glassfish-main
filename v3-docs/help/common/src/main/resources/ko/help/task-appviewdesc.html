응용 프로그램의 설명자를 보는 방법

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.

    응용 프로그램 페이지가 열립니다.

  2. 응용 프로그램 페이지에서 설명자를 볼 응용 프로그램의 이름을 누릅니다.

    이름에는 버전 ID(옵션)가 포함될 수 있으며, 이 ID는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server 3.1 Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    응용 프로그램 편집 페이지가 열립니다.

  3. 응용 프로그램 편집 페이지에서 설명자 탭을 누릅니다.

    모듈 설명자 페이지가 열립니다.

  4. 배포 설명자 테이블에서 보려는 XML 파일의 이름을 누릅니다.

    배포 설명자 페이지가 열리고 XML 파일이 표시됩니다.

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 대상을 보는 방법

  • 응용 프로그램을 시작하는 방법
