배포 설명자

배포 설명자 페이지를 사용하여 모듈의 배포 설명자를 볼 수 있습니다.

배포 설명자 페이지에는 다음 정보가 포함됩니다.

응용 프로그램 이름

응용 프로그램의 이름입니다.

모듈 이름

배포 설명자를 포함하는 모듈의 이름입니다.

설명자 파일 이름

배포 설명자 파일의 이름입니다.

관련 작업

  • 응용 프로그램의 설명자를 보는 방법

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • Ruby 응용 프로그램을 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 웹 서비스에 대한 종점 정보 보기

  • 응용 프로그램을 시작하는 방법

  • 응용 프로그램 클라이언트를 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 시작하는 방법

관련 asadmin 명령

  • list-applications(1)
