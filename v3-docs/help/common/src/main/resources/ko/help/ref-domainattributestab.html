도메인 속성

[도메인 속성] 페이지에서는 GlassFish Server에 대한 고급 도메인 속성을 설정할 수 있습니다.

[도메인 속성] 페이지에는 다음 옵션이 포함되어 있습니다.

응용 프로그램 루트

배치된 응용 프로그램이 있는 디렉토리의 전체 경로입니다. 기본값은 domain-dir/applications입니다.

로그 루트

서버 로그 파일을 저장하는 위치를 식별합니다. 기본값은 domain-dir/logs입니다.

로케일

이 필드를 비워 두면 호스트의 기본 로케일이 사용됩니다.

로케일은 언어 및 지역의 특정 조합을 지정하는 식별자입니다. 예를 들어 미국 영어의 로케일은 en_US이고 일본어의 로케일은 ja_JP입니다. 비영어 로케일을 사용하려면 GlassFish Server이(가) 현지화(영어를 다른 언어로 번역하는 작업이 포함된 프로세스)되어야 합니다.

  • 도메인 속성을 설정하는 방법
