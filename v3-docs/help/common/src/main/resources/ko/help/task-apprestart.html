배포된 응용 프로그램을 다시 로드하는 방법

응용 프로그램을 다시 로드하면 응용 프로그램이 재배포되는 것이 아니라 다시 시작되는 것이며, 응용 프로그램이 배포되었을 당시의 설정 및 기타 옵션은 그대로 유지됩니다.

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.

    응용 프로그램 페이지가 열립니다.

  2. 배포된 응용 프로그램 테이블에서 재배포할 응용 프로그램의 행에 있는 다시 로드를 누릅니다.

    응용 프로그램 이름에는 선택적인 버전 식별자가 포함될 수 있으며 이 식별자는 이름 뒤에 오고 콜론(:)을 통해 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server 3.1 Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    응용 프로그램 페이지가 다시 나타나며 성공적으로 다시 로드되었는지 여부를 알리는 메시지가 표시됩니다.

참조

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 응용 프로그램을 시작하는 방법
