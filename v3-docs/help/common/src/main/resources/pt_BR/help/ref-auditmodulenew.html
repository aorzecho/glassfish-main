Novo Módulo de Auditoria

Use a página Novo Módulo de Auditoria para defini-lo.

A página Novo Módulo de Auditoria contém estas opções.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Nome

Nome do novo módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria padrão é com.sun.enterprise.security.Audit .

Propriedades Adicionais

Propriedades adicionais do módulo de auditoria. Para uma descrição da propriedade adicional que está disponível para módulos de auditoria, consulte Propriedades Específicas de Módulos de Auditoria.
