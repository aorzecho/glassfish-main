Propriedades do Sistema

Usar a página Propriedades do Sistema para exibir e gerenciar propriedades do Servidor GlassFish.

Uma propriedade do sistema define um valor comum de uma configuração no nível do servidor. Você pode se referir a uma propriedade do sistema em um campo de texto Console do Admin incluindo um símbolo de dólar e chaves. Por exemplo, para referir-se a uma propriedade chamada prop-name, você usaria a sintaxe ${prop-name}.

Para cada propriedade, a página Propriedades do Sistema contém as seguintes informações.

Nome

O nome da propriedade.

Valor

O valor da propriedade.

Descrição

Uma descrição opcional da propriedade.

Uma série de propriedades do sistema são predefinidas, mas não são mostradas nesta página. Para obter os detalhes, consulte Propriedades Predefinidas do Sistema.
