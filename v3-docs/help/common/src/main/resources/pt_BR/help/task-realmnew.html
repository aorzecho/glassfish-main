Para Criar um Realm

O GlassFish Server vem pré-configurado com estes três realms: arquivo, certificado e admin-realm. Também é possível criar ldap, JDBC, solaris, PAM e realms personalizados. Geralmente, você tem um realm de cada tipo em um servidor, mas, no GlassFish Server há dois realms de arquivo: arquivo e admin-realm. Há dois realms do mesmo tipo, mas usados para finalidades diferentes: É possível também ter um banco de dados de certificado para cada servidor virtual do sistema.

  1. No nó Configurações, expanda a configuração de cluster ou instância de servidor na qual você deseja configurar os realms.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

    É aberta a página Realms.

  4. Na página Realms, clique em Novo.

    É aberta a página Novo Realm.

  5. No campo Nome, digite um nome para o realm.

  6. Para especificar o nome da classe do realm, faça uma das seguintes coisas:

    • Na lista drop-down Nome da Classe, selecione o nome da classe do realm que está sendo criado.

      As opções válidas são mostradas na seguinte tabela.

      Nome do Realm Nome da Classe
      arquivo com.sun.enterprise.security.auth.realm.file.FileRealm
      certificado com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • Selecione o botão de opções inferior e digite o nome da classe do realm personalizado que está sendo criado.

  7. Na seção Propriedades Adicionais, especifique essas propriedades.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em brando que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    As propriedades válidas dependem do tipo de realm.

    • Propriedades Específicas da Classe do FileRealm

    • Propriedades Específicas da Classe CertificateRealm

    • Propriedades Específicas da ClasseJDBCRealm

    • Propriedades Específicas da ClasseLDAPRealm

    • Propriedades Específicas da ClasseSolarisRealm

    • Propriedades Específicas da ClassePamRealm

  8. Clique em OK.
