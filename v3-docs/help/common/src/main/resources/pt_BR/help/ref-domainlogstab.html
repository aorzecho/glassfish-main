Logs de Domínio

Use a página Logs de Domínio para coletar e fazer download de arquivos de log para o GlassFish Server.

A página de Logs do Domínio contém a opções a seguir. Você pode selecionar um cluster ou uma instância do servidor para o qual coletar logs.

Cluster

Uma lista drop-down de clusters se existirem.

Instância

Uma lista drop-down de instâncias do servidor.

Coletar Logs

Botão para coletar os logs em um arquivo log-files-name.zip é o nome do cluster ou instância.

O arquivo log-files-name.zip é submetido a download usando o mecanismo de download do seu browser.
