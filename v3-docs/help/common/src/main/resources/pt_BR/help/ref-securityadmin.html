Administração Segura

Use a página Administração Segura para ativar ou desativar a administração segura.

Quando a administração segura estiver ativada:

  • O DAS e as instâncias no domínio usam certificados de SSL para criptografar as mensagens que trocam entre si.

  • O DAS aceita mensagens de administração de clientes admin remotos, como os recursos do asadmin e IDES.

A página Administração Segura contém as seguintes informações.

Ativar Admin Segura

Botão para ativar administração segura. Este botão aparece somente quando a administração segura não está ativada.

Observação:

Se algum usuário administrativo no domínio tiver uma senha vazia, tentativas de ativar administração segura irão falhar.
Desativar Admin Segura

Botão para desativar administração segura. Este botão aparece somente quando a administração segura está ativada.

Alias de Administração

O alias que se refere ao certificado SSL/TLS no DAS. Esse alias é usado pelo DAS para identificar-se para instâncias. O valor default é s1as. Este botão aparece somente quando a administração segura não está ativada.

Alias da Instância

O alias que se refere ao certificado SSL/TLS nas instâncias. Esse alias é usado pelas instâncias para identificar-se para o DAS. O valor default é glassfish-instance. Este botão aparece somente quando a administração segura não está ativada.
