Para excluir uma configuração de segurança de mensagens

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Expanda o nó Segurança.
  3. Selecione o nó Segurança de Mensagens.

    A página Configurações de Segurança de Mensagens é exibida.

  4. Na página Configurações de Segurança de Mensagens, marque a caixa de seleção à esquerda da Configuração de Segurança de Mensagens que você está excluindo.

    Observação - É possível criar ou excluir os provedores de segurança de mensagens da configuração server-config padrão. Os únicos provedores de segurança de mensagem que são suportados no server-config padrão são SOAP e HttpServlet. Como estes dois provedores de mensagem já são criados por padrão quando o Servidor GlassFish é instalado, o botão Excluir não é exibido na página Configurações de segurança de mensagens no server-config padrão.

  5. Clique em Excluir.
