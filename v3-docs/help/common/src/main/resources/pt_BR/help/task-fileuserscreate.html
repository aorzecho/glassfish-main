realms Para Adicionar um Usuário ao Realm do Arquivo

  1. No nó Configuração, expanda a configuração de cluster ou instância de servidor na qual você deseja gerenciar os usuários de realm administrativos.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

  4. Selecione o nó arquivo.

    A página Editar Realm é exibida.

  5. Na página Editar Realm, clique no botão Gerenciar Usuários.

    A página Usuários do Arquivo é exibida.

  6. Na página Usuários do Arquivo, clique em Novo.

    A página Novo Usuário do Realm Arquivo é exibida.

  7. No campo ID de Usuário, digite o nome do usuário.

  8. No campo Lista de Grupos, digite uma lista dos grupos aos quais o usuário pertence.

    Use vírgulas para separar os nomes de grupos. Esses grupos não precisam estar definidos em outro lugar.

  9. No campo Senha. digite uma senha exclusiva.

  10. No campo Confirmar Nova Senha, digite novamente a senha de usuário.

  11. Clique em OK.
