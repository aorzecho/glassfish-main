Editar Domínio

Use a página Editar Domínio para modificar um domínio.

A página Editar Domínio contém estas opções.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Nome de Domínio

O nome do domínio. Este é um campo somente leitura. Você só pode especificar o nome quando cria um novo domínio.

Nome da Classe

O nome da classe do domínio que está sendo editado.

Propriedades Adicionais

Propriedades adicionais do domínio. As propriedades válidas dependem do tipo de domínio.

  • Propriedades específicas da classe FileRealm

  • Propriedades específicas da classe CertificateRealm

  • Propriedades específicas para a classe JDBCRealm

  • Propriedades específicas da classe LDAPRealm

  • Propriedades específicas da classe SolarisRealm

  • Propriedades específicas da classe PamRealm
