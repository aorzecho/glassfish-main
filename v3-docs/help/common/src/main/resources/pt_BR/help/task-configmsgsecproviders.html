Para configurar os recursos do Servidor GlassFish para uso pelos provedores de segurança de mensagens

  1. Se estiver usando um símbolo de nome de usuário, configure um banco de dados de usuário, se necessário.

    Ao usar um símbolo de nome de usuário/senha, primeiro configure um domínio adequado e um banco de dados de usuário adequado para o domínio.

    A configuração de um banco de dados de usuário é discutida em Para editar um domínio.

  2. Gerencie os certificados e as chaves particulares, se necessário.
Passos Seguintes

Quando os recursos do Servidor GlassFish estiverem configurados para uso pelos provedores de segurança de mensagens, os provedores instalados no Servidor GlassFish poderão ser ativados como descrito em Para editar uma configuração de segurança de mensagens.
