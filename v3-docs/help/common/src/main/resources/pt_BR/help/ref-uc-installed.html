Componentes Instalados

A página Componentes Instalados fornece informações sobre os componentes da imagem de instalação instalados.

Para desinstalar qualquer componente instalado, pare primeiro o domínio do Servidor GlassFish. A seguir, use um destes comandos:

  • glassfish3/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de atualização

  • glassfish3/bin/pkg, a versão de linha de comando da Ferramenta de atualização

Para obter informações sobre como utilizar o comando pkg, consulte Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server 3.1 Administration Guide. O utilitário Ferramenta de atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter terminado de desinstalar os componentes.

A página Componentes Instalados contém estas opções:

Caminho da Instalação

O caminho do sistema de arquivos do diretório raiz ou de nível superior da imagem de instalação.

Componentes Instalados

Para cada componente instalado, esta tabela fornece as seguintes informações.

Componente

O nome do componente.

Categoria

A categoria de software que é associada ao componente; por exemplo, Servidores de Aplicativos, Ferramentas do Sistema e Bancos de Dados. A categoria é atribuída pelo autor do componente quando o componente é criado.

Versão

O número de versão do pacote em que o componente foi fornecido.

Tamanho Instalado

A quantidade de espaço em disco que a instalação do componente consome.

Origem

O nome do Repositório de rede de onde o componente foi obtido.
