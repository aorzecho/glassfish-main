Para visualizar os arquivos de log do DAS

  1. Na árvore de navegação, selecione o nó do servidor (servidor admin).

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique no botão Exibir Arquivos de Log.

    A janela Visualizador de logs é exibida.
