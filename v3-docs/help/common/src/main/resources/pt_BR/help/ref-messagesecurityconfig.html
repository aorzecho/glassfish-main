Segurança de Mensagens

Use a página Configurações de Segurança de Mensagens para exibir e modificar as configurações disponíveis para provedores de segurança de mensagens.

Para configurar a segurança de mensagens, você precisa de uma compreensão básica de conceitos de segurança e de serviços da Web. Para obter mais informações, consulte o Capítulo 3, Administering Message Security, no Oracle GlassFish Server 3.1 Security Guide e o Capítulo 6, Administering Web Applications, no Oracle GlassFish Server 3.1 Administration Guide.

O Servidor GlassFish implementa a segurança de mensagens usando provedores de segurança de mensagens integrados em sua camada de processamento SOAP. Os provedores de segurança de mensagens dependen de outros recursos de segurança de Servidor GlassFish.

Para cada configuração, as seguintes informações são fornecidas.

Camada de Autenticação

A camada de mensagens na qual a autenticação é realizada. Por padrão, uma configuração SOAP é fornecida. Também é possível fornecer uma configuração HttpServlet.

Provedor Padrão

O provedor do servidor padrão desta configuração. Por padrão, nenhum provedor de servidor padrão está configurado.

Provedor de Cliente Padrão

O provedor de cliente padrão desta configuração. Por padrão, nenhum provedor de cliente padrão está configurado.

A tabela Configuração de Segurança de Mensagens também contém as seguintes opções.

Novo

Botão para criar uma nova configuração.

Excluir

Botão para excluir uma ou mais configurações selecionadas.
