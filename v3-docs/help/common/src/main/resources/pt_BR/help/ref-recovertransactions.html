Recuperar Transações

Utilize a página Recuperar transações para recuperar transações de uma instância do GlassFish Server.

Se a instância estiver em execução, a página Recuperar transações contém somente o texto A recuperação é realizada pelo próprio servidor em execução e o botão OK.

Se a instância não estiver em execução, a página Recuperar transações contém as seguintes opções.

Nome da instância

Nome designado à instância quando a instância foi criada. Este campo é somente para leitura.

Servidor de destino

Outra instância que pode recuperar as transações na instância atual.

Diretório do log de transações

O diretório no qual os logs da transação são armazenados, se os logs estiverem armazenados em um sistema de arquivos compartilhados. Esta definição não se aplica a logs de transação armazenados em um banco de dados. O default é /logs/tx no diretório de domínio.

Os logs de transação são mantidos no subdiretório tx do diretório especificado pela definição do Local do Log de Transação do Serviço de transação na configuração do cluster. O valor default do Local do log de transação é o diretório especificado pelo campo Raiz de log da página Atributos do Domínio, que está localizado na guia Avançado do nó do GlassFish Server.
