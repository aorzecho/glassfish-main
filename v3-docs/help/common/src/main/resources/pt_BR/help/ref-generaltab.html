Informações gerais

Use a página Informações Gerais para verificar as configurações do Servidor GlassFish e para exibir os dados do Java Virtual Machine.

A página Informações Gerais contém as seguintes informações.

Botão Parar

Clique no botão Parar para interromper o Servidor GlassFish.

Botão Reiniciar

Clique no botão Reiniciar para iniciar novamente o Servidor GlassFish.

Botão Exibir arquivos de log.

Clique no botão Exibir arquivos de log para exibir os arquivos de log de uma instância ou cluster do GlassFish Server.

Nome

O nome do servidor.

JVM

Se você clicar em JVM Report, será aberta uma janela separada que exibe relatórios sobre o Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória e de coleta de erros,um relatório de carregamento de classe e um despejo de segmentos atual.

Diretório de Configuração

O diretório localizado no computador host, que contém arquivos de configuração; por exemplo, domain.xml.

Versão Instalada

O identificador de versão do produto.

Depuração

Identifica se a depuração está ativada. Se a depuração estiver ativada, o número de porta também será exibido.

Porta(s) HTTP

As portas configuradas atualmente para solicitações e respostas HTTP.

Porta(s) IIOP

As portas configuradas atualmente para solicitações e respostas IIOP.

Tarefa relacionada

  • Para visualizar informações gerais do DAS

  • Para visualizar os arquivos de log do DAS
