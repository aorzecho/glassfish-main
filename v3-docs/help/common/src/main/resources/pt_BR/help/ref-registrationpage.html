Registro de Produto

Para registrar sua instalação do OracleServidor GlassFish, é preciso ter uma Conta Sun On-line. Caso você não tenha uma Conta Sun Online, o Servidor GlassFish permitirá a criação de uma conta durante o registro.

A página Registro de Produto contém estas opções.

Já tenho uma Conta Sun Online ou uma Conta Sun Developer Network (SDN)

Selecione esta opção se já tiver uma conta. Se você selecionar essa opção, os seguintes campos estarão visíveis.

Nome de Usuário

O nome de usuário especificado quando a Conta Sun Online foi registrada.

Senha

A senha especificada quando a Conta Sun Online foi registrada.

Host Proxy

O nome de host do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.

Porta Proxy

O número de porta do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.

Não tenho uma Conta Sun Online. Inscreva-me.

Selecione esta opção se não tiver uma conta e quiser registrar sua instalação do Servidor GlassFish. Se você selecionar essa opção, os seguintes campos estarão visíveis.

Endereço de E-mail

Seu endereço de e-mail, que será o nome de usuário da sua Conta Sun Online.

Senha

A senha a ser usada para a Conta Sun Online. A senha diferencia maiúsculas/minúsculas e deve ter de 6 a 10 caracteres.

Digitar Senha Novamente

Digite novamente a senha especificada no campo Senha.

Nome

Seu nome.

Sobrenome

Seu sobrenome.

Empresa

A empresa em que trabalha, se aplicável.

País/Território

O país ou território onde você reside.

Host Proxy

O nome de host do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.

Porta Proxy

O número de porta do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.
