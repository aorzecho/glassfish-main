Para exibir uma lista de aplicativos implantados

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida. A tabela lista os aplicativos implementados.

  2. Selecione um tipo de módulo na lista suspensa Filtro para filtrar a lista de aplicativos por tipo de módulo.
