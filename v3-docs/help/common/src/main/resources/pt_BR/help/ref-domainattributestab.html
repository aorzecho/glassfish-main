Atributos de Domínio

Use a página Atributos do Domínio para configurar atributos de domínio avançados para o GlassFish Server.

A página Atributos do Domínio contém as opções a seguir.

Raiz da Aplicação

O caminho completo do diretório em que residem as aplicações implantadas. O valor default é domain-dir/applications.

Raiz de Log

Identifica onde o arquivos de log do servidor são mantidos. O valor default é domain-dir/logs.

Configuração Regional

Se este campo for deixado em branco, as configurações regionais do host serão utilizadas.

Uma configuração regional é um identificador que especifica uma determinada combinação de idioma e região. Por exemplo, a área geográfica do Inglês Americano é en_US, e do Japonês é ja_JP. Para usar uma configuração regional que não seja o inglês, o GlassFish Server deve ser localizado, um processo que inclui a tradução de inglês para outro idioma.
