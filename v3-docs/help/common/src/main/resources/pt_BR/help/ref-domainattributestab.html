Atributos de Domínio

Use a página Atributos de Domínio para configurar atributos de domínio avançados para o Servidor GlassFish.

A página Atributos de Domínio contém as seguintes opções.

Raiz do Aplicativo

O caminho completo do diretório onde residem os aplicativos implantados. O valor padrão é domain-dir/applications.

Raiz de Log

Identifica onde o arquivos de log do servidor são mantidos. O valor padrão é domain-dir/logs.

Localidade

Se este campo for deixado em branco, a localidade padrão do host será utilizada.

Uma localidade é um identificador que especifica uma determinada combinação de idioma e região. Por exemplo, a localidade para inglês americano é en_US e para japonês é ja_JP. Para usar uma localidade que não seja o inglês, o Servidor GlassFish deve ser localizado, um processo que inclui a tradução de inglês para outro idioma.
