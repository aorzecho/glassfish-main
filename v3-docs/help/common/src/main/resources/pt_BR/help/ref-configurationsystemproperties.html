Propriedades do Sistema

A página Propriedades do sistema exibe uma lista das propriedades do sistema Java que estão definidas na configuração nomeada selecionada. As propriedades do sistema Java são transmitidas para o iniciador de aplicativo Java através da opção -D do iniciador de aplicativo Java quando o Servidor GlassFish é iniciado

Estas propriedades definem os valores padrão das configurações da porta de todas as instâncias do Servidor GlassFish que fazem referência à configuração.

Para obter informações sobre as configurações de porta predefinidas, consulte Propriedades Predefinidas do Sistema .

São exibidas as seguintes informações para cada propriedade:

Nome da variável da instância

Nome da propriedade do sistema.

Valor padrão

Valor definido para a propriedade na configuração nomeada que a instância faz referência. Este campo é somente leitura.

Valores da instância

Link para a página Valores da instância da propriedade.

A tabela Propriedades adicionais também contém as seguintes opções.

Adicionar propriedade

Botão para adicionar uma propriedade. Se clicar neste botão, uma linha é adicionada à tabela Propriedades adicionais.

Excluir propriedades

Botão para excluir uma ou mais propriedades selecionadas. Qualquer propriedade excluída volta ao seu valor padrão ou, se nenhum valor tiver sido definido, fica com valor indefinido.

A página Propriedades do sistema também contém as seguintes opções:

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Reconfiguração dinâmica

Se esta opção for ativada, são aplicadas alterações na configuração das instâncias que fazem referência à configuração sem a necessidade de reiniciar as instâncias.

Clusters e/ou instâncias que usam esta configuração

Lista dos clusters e das instâncias que fazem referência à configuração com uma indicação de se cada cluster ou instância está em execução:

  • Se o cluster estiver em execução, uma marca de seleção (✓) é exibida.

  • Se a instância ou o cluster estiver interrompido, um ponto de exclamação (!) é exibido.
