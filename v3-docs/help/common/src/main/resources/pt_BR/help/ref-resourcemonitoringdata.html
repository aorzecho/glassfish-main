Monitoramento do Recurso

Usar a página Monitoramento do Recurso para exibir os dados de monitoramento de um recurso selecionado. Para obter uma descrição das propriedades monitoradas, consulte Capítulo 8, Administering the Monitoring Service, no Oracle GlassFish Server 3.1 Administration Guide.

Tarefas relacionadas

  • Para configurar o serviço de monitoramento

  • Para exibir dados de monitoramento do aplicativo

  • Para exibir dados de monitoramento do servidor

  • Para exibir dados de monitoramento do recurso

Comandos asadmin relacionados

  • disable-monitoring(1)

  • monitor(1)
