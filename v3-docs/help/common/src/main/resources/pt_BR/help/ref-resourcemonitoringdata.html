Monitoramento da Recursos

Usar a página Monitoramento do Recurso para exibir os dados de monitoramento de um recurso selecionado.

A página Monitoramento de Recursos contém as seguintes informações.

Nome da instância

O nome da instância do servidor.

Pool de Conexões

O pool de conexões cujas estatísticas de monitoramento são exibidas.

Para a descrição das propriedades monitoradas, consulte Capítulo 8, Administrando o Serviço de Monitoramento em Oracle GlassFish Server Administration Guide.
