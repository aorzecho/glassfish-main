Para exibir as atualizações disponíveis para componentes

A página Atualizações Disponíveis fornece informações sobre as versões mais recentes dos componentes da imagem de instalação instalados.

  1. Na árvore de navegação, selecione o nó Ferramenta de atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, clique na guia Atualizações Disponíveis.

    É aberta a página Atualizações Disponíveis. Se não houver atualizações disponíveis, a tabela estará vazia.

Passos Seguintes

Para instalar as atualizações disponíveis, pare primeiro o domínio do Servidor GlassFish. A seguir, use um destes comandos:

  • as-install-parent/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de atualização

  • as-install-parent/bin/pkg, a versão de linha de comando da Ferramenta de atualização

Para obter informações sobre como utilizar o comando pkg, consulte Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server 3.1 Administration Guide. O utilitário Ferramenta de atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter terminado de instalar as atualizações.
