Destinos de servidor virtual

Utilize a página Destinos de servidor virtual para alterar os servidores virtuais de destino aos quais um aplicativo ou módulo está associado.

A página Destinos de servidor virtual contém as seguintes informações.

Destinos disponíveis

Servidores virtuais aos quais não há nenhum aplicativo ou módulo associado.

Destinos selecionados

Servidores virtuais aos quais um aplicativo ou módulo está associado.

Adicionar

Botão para mover o destino selecionado de Destinos disponíveis para Destinos selecionados.

Adicionar tudo

Botão para mover todos os Destinos disponíveis para Destinos selecionados.

Remover

Botão para remover o destino selecionado de Destinos selecionados para Destinos disponíveis.

Remover tudo

Botão para mover todos os Destinos selecionados para Destinos disponíveis.
