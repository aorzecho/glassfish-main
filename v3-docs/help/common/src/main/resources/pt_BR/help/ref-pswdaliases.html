Aliases de Senha

Utilize a página Aliases de senha para criar e gerenciar aliases de senha.

A página Aliases de senha exibe uma lista de aliases de senha no domínio. São exibidas as seguintes informações para cada alias:

Nome

Nome designado ao alias de senha ao ter sido criado. Clicar no nome abre a Editar Alias de Senha página do alias.

A tabela Aliases de senha também contém as seguintes opções.

Novo

Botão para criar um alias de senha.

Deletar

Botão para deletar um ou mais aliases de senha selecionados.

Subcomandos asadmin equivalentes

  • list-password-aliases(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • update-password-alias(1)
