Editar Conector JMX

Use a página Editar Conector JMX para editar o conector JMX JSR 360-compliant.

A página Editar conector JMX contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do conector JMX. Este campo é somente leitura.

Protocol JMX

O nome do protocolo que esse conector JMX suporta. Este campo é somente leitura.

JMXServiceURL

O URL de serviços JMX. Este campo é somente leitura.

Segurança

Se a caixa de seleção Disponível estiver marcada, a comunicação JMX será criptografada. Por default, essa opção está desativada.

Endereço

O endereço IP do serviço de nomes no qual o stub do servidor do conector JMX está registrado. Esta não é a porta do soquete do servidor que faz a comunicação JMX real. Este é o endereço da interface de rede onde o registro do RMI foi iniciado. Se o seu sistema tem múltiplas interfaces de rede, modifique esse valor para que apenas uma determinada interface seja selecionada. O valor default é 0.0.0.0.

Porta

O número da porta na qual o serviço de nomes (registro RMI) fazem de listening de conexões do cliente RMI. O único uso deste serviço de nomeação é fazer download para os stubs de RMI. Se a porta default estiver ocupada, uma porta livre será usada. Os valores legais são 1- 65535. Em sistemas UNIX, criar sockets que fazem listening nas portas 1 - 1024 exige privilégios de superusuários. O valor default é 8686.

Nome do Realm

O nome que representa o realm administrativo especial. Todas as autenticações são manuseadas por este realm. O valor default é admin-realm.

Propriedades Adicionais

Propriedade adicional para o conector JMX. Diversas propriedades são especificadas por default.
