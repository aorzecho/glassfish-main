Visualizador de Log Bruto

Use a página Visualizador de Log Bruto para visualizar os arquivos de log bruto para o Admin Server (denominado servidor).

A página Visualizador de Log Bruto contém as seguintes opções.

Instância

A instância do servidor ou cluster para o qual os registros de log serão exibidos. A única opção é servidor.

Para cada mensagem, o Visualizador de Log Bruto exibe as seguintes informações.

Timestamp

A data e a hora em que a mensagem de log foi emitida.

Número do Registro

O número da mensagem no log do servidor.

Nível do Log

O nível do log da mensagem.

Versão

A versão do GlassFish Server.

Logger

O módulo do logger que gerou a mensagem de log.

Name/Value Pairs

O nome e o ID do thread da mensagem de log.

Mensagem

O texto breve da mensagem de log. Clique no link (detalhes) para exibir uma versão formatada de uma mensagem.
