Monitoramento da Aplicação

Usar a página Monitoramento da Aplicação para exibir os dados de monitoramento de uma aplicação selecionada.

A página Monitoramento da Aplicação contém as seguintes informações.

Nome da instância

O nome da instância do servidor.

Aplicação

A aplicação para a qual estatísticas de monitoramento são exibidas.

Componente

O componente da aplicação para a qual estatísticas de monitoramento são exibidas.

Para a descrição das propriedades monitoradas, consulteCapítulo 8, Administrando o Serviço de Monitoramento, em Oracle GlassFish Server Administration Guide.
