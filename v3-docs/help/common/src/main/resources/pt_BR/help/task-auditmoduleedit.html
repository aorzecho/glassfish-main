Para editar um módulo de auditoria

Os módulos de auditoria não estão ativados por padrão. Para obter mais informações sobre como ativar módulos de auditoria, consulte Para ativar ou desativar logs de auditoria.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, expanda o nó Segurança.
  3. Sob o nó Segurança, selecione o nó Módulos de Auditoria.

    A página Módulos de Auditoria é exibida.

  4. Na página Módulos de Auditoria, clique no nome do módulo de auditoria que você está editando.

    A página Editar Módulo de Auditoria é exibida.

  5. Na página Editar Módulo de Auditoria, digite um novo valor no campo Nome da Classe para modificar esse nome.
  6. (Opcional) Na seção Propriedades Adicionais, especifique essas propriedades.

    Para uma descrição da propriedade adicional que está disponível para módulos de auditoria, consulte Propriedades Específicas de Módulos de Auditoria.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir Propriedades.
  7. Clique em Salvar.
