Nova Senha do Administrador

Use a página Nova Senha do Administrador para alterar a senha da conta do administrador atual.

A página Nova Senha do Administrador contém as seguintes opções.

ID do Usuário

A ID de usuário do administrador atual. Este é um campo somente leitura.

Lista de Grupos

O grupo ao qual o administrador pertence. Este é um campo somente leitura. O valor é asadmin.

Nova Senha

A nova senha do administrador.

Confirmar Nova Senha

A nova senha do administrador.

Tarefa relacionada

  • Para alterar a Senha do Administrador

Comando asadmin relacionado

  • change-admin-password(1)
