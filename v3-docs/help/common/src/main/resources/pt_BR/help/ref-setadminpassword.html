Nova Senha do Administrador

Use a página Nova Senha do Administrador para alterar a senha da conta do administrador atual.

Observação:

Se a administração segura estiver ativada, você não poderá alterar a senha para um valor vazio.

A página Nova Senha do Administrador contém as seguintes opções.

ID do Usuário

O ID de usuário do administrador atual. Este é um campo somente para leitura.

Lista de Grupos

O grupo ao qual o administrador pertence. Este é um campo somente para leitura. O valor é asadmin.

Nova Senha

A nova senha do administrador.

Confirmar Nova Senha

A nova senha do administrador.

Para Alterar a Senha do Administrador
