Para Exibir a Página Notícias do GlassFish relativa ao GlassFish Server

A página Notícias do GlassFish News oferece um conjunto de links para informações adicionais sobre o Oracle GlassFish Server, como informativos, artigos, dicas técnicas, fóruns de discussão e blogs relevantes.

  1. Na árvore de navegação, selecione o nó Tarefas comuns.

    A página Tarefas comuns é exibida.

  2. Em Notícias do GlassFish, clique no link Notícias do GlassFish.

    A página Notícias do GlassFish é exibida em uma guia ou janela separada do browser.
