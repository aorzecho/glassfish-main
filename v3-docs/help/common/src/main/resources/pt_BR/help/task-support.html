Para exibir a página Notícias do GlassFish do Servidor GlassFish

A página Notícias do GlassFish traz uma coleção de links para informações adicionais sobre o Oracle Servidor GlassFish, como newsletters, artigos, dicas técnicas, fóruns de discussão e blogs relevantes.

  1. Na árvore de navegação, selecione o nó Tarefas comuns.

    A página Tarefas comuns é exibida.

  2. Em Notícias do GlassFish, clique no link Notícias do GlassFish.

    A página Notícias do GlassFish é exibida em uma guia ou janela separada do navegador.
