Configuração de Ferramenta de Atualização

A página Configuração fornece informações de configuração do Ferramenta de Atualização e permite alterar como o &UpdateToolAdminConsoleNode se conecta à Internet. Para alterar outros aspectos da configuração de Ferramenta de Atualização, use a aplicação updatetool standalone localizada em as-install-parent/bin.

A página Configuração do Ferramenta de Atualização contém as seguintes opções.

Caminho da Instalação

O caminho do sistema de arquivos do diretório raiz ou de nível superior da imagem de instalação.

Repositório Preferencial

O nome do repositório da rede preferido que está associado com esta imagem de instalação, por exemplo, glassfish.org.

Conexão com a Internet

Como o GlassFish Server se conecta à Internet:

Conexão Direta

Esta opção especifica que o GlassFish Server se conecta diretamente à Internet.

Via Proxy

Esta opção especifica se o GlassFish Server se conecta à Internet por meio de um servidor proxy. Quando a opção é selecionada, o campo Host Proxy e o campo Porta Proxy são adicionados à página.

Host Proxy

O nome de host do servidor proxy para o computador em que o GlassFish Server está em execução.

Esse campo só será aplicável se o computador em que GlassFish Server estiver em execução se conectar à Internet por meio de um servidor proxy.

Porta Proxy

O número de porta do servidor proxy para o computador em que o GlassFish Server está em execução.

Esse campo só será aplicável se o computador em que GlassFish Server estiver em execução se conectar à Internet por meio de um servidor proxy.

Repositórios

Esta tabela fornece ass seguintes informações sobre cada repositório de rede que é associado com essa imagem da instalação:

Repositório

O nome do repositório; por exemplo, dev.glassfish.org .
