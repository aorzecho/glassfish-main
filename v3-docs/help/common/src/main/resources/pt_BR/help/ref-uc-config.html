Configuração do Ferramenta de atualização

A página Configuração fornece informações de configuração do Ferramenta de atualização e permite alterar como o &UpdateToolAdminConsoleNode se conecta à Internet. Para alterar outros aspectos da configuração do Ferramenta de atualização, utilize a aplicação updatetool independente localizada em as-install-parent/bin .

A página Configuração do Ferramenta de atualização contém as seguintes opções.

Caminho da Instalação

O caminho do sistema de arquivos do diretório raiz ou de nível superior da imagem de instalação.

Repositório Preferencial

O nome do repositório de rede preferencial, que é associado à imagem de instalação; por exemplo, glassfish.org.

Conexão com a Internet

Como o Servidor GlassFish se conecta à Internet:

Conexão Direta

Esta opção especifica que o Servidor GlassFish se conecta diretamente à Internet.

Via Proxy

Esta opção especifica se o Servidor GlassFish se conecta à Internet por meio de um servidor proxy. Quando a opção é selecionada, o campo Host Proxy e o campo Porta Proxy são adicionados à página.

Host Proxy

O nome de host do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.

Porta Proxy

O número de porta do servidor proxy para o computador em que o Servidor GlassFish está em execução.

Esse campo só será aplicável se o computador em que Servidor GlassFish estiver em execução se conectar à Internet por meio de um servidor proxy.

Repositórios

Esta tabela fornece as seguintes informações sobre cada repositório de rede associado à imagem de instalação:

Repositório

O nome do repositório; por exemplo, dev.glassfish.org.

Tarefas relacionadas

Para definir configurações de proxy de rede para acessar repositórios de rede
