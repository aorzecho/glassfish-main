Para Editar o Conector JMX

Use o Serviço de Administração para configurar um conector JMX remoto compatível com JSR-160.

O conector JMX remoto compatível com JSR-160 lida com a comunicação entre clientes JMX, como o JConsole, e o serviço JMX, para fins de gerenciamento de domínio.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Sob o nó Configuração, selecione o nó Serviço de Administração.

    A página Editar Conector JMX é exibida.

  3. Marque a caixa de seleção Segurança Ativada para que a comunicação JMX seja criptografada.

    Por default, essa opção está desativada.

  4. No campo Endereço, digite o endereço IP do serviço de nomes onde o stub do servidor do conector JMX está registrado.

    Esta não é a porta do soquete do servidor que faz a comunicação JMX real. Este é o endereço da interface de rede em que o registro do RMI foi iniciado. Se o seu sistema tem múltiplas interfaces de rede, modifique esse valor para que apenas uma determinada interface seja selecionada. O valor default é 0.0.0.0.

  5. No campo Porta, digite o número da porta no qual o serviço de nomes (registro RMI) faz listening das conexões clientes RMI.

    O único uso deste serviço de nomeação é fazer download para os stubs de RMI. Se a porta default estiver ocupada, uma porta livre será usada. Os valores legais são 1- 65535. Em sistemas UNIX, criar sockets que fazem listening nas portas 1 - 1024 exige privilégios de superusuários. O valor default é 8686.

  6. No campo Nome do Realm, digite o nome que representa o realm administrativo especial.

    Todas as autenticações são manuseadas por este realm. O valor default é admin-realm.

  7. Na seção Propriedades Adicionais, especifique essas propriedades.

    O GlassFish Server especifica diversas propriedades por default.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  8. Clique em Salvar.

  9. Caso você tenha marcado a caixa de seleção Segurança Ativada, siga as etapas contidas em Para Editar as Definições da SSL do Conector JMX.
