Para instalar componentes complementares

A instalação de um componente complementar envolve as seguintes operações:

  • Transferência do conteúdo do componente e das dependências do componente de um repositório de rede

  • Instalação do componente da imagem de instalação do Servidor GlassFish no sistema

Se um componente precisar de qualquer outro componente ainda não instalado, os componentes necessários também serão instalados.

  1. Na árvore de navegação, selecione o nó Ferramenta de atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, na tabela de componentes complementares disponíveis, selecione os componentes que estão sendo instalados.

    Se não houver componentes complementares disponíveis, a tabela estará vazia.

    • Para selecionar um componente individual, marque a caixa de seleção adjacente ao nome do componente.
    • Para selecionar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela.
    • Para desmarcar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela Desmarcar Tudo.
  3. Clique em Instalar.

    Uma caixa de diálogo pede que você confirme se devem ser instalados os componentes selecionados e qualquer outro componente necessário.

  4. Na caixa de diálogo, clique em OK.

    Você é solicitado a aceitar os contratos de licença dos componentes que estiver instalando.

  5. Aceite os contratos de licença dos componentes que está instalando.

    O Servidor GlassFish confirma que a instalação está concluída. Os componentes são removidos da tabela de componentes complementares e adicionados à tabela de componentes instalados na página Componentes Instalados.

  6. Reinicie o servidor.

    Para obter instruções detalhadas, consulte a seguinte documentação:

    • To Stop a Domain no Oracle GlassFish Server 3.1 Administration Guide

    • To Start a Domain no Oracle GlassFish Server 3.1 Administration Guide
