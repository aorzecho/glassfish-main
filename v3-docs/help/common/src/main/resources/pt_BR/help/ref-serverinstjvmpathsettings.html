Configurações do Caminho JVM

Use a página Configurações do Caminho JVM para configurar o caminho da biblioteca nativa. O caminho da biblioteca nativa é uma concatenação dos seguintes caminhos:

  • O prefixo do caminho da biblioteca nativa

  • O caminho do servidor para suas bibliotecas nativas (ou seja, as-install/lib)

  • Os diretórios da biblioteca nativa de 64 bits, se aplicável.

  • O valor da variável java.library.path para o JVM, usada para iniciar o Servidor GlassFish (isso difere entre os sistemas operacionais)

  • Quaisquer caminhos especificados na página Configurações do Gerador de Perfil de JVM, se o perfilador estiver instalado.

  • O sufixo do caminho da biblioteca nativa

Outras configurações são mostradas como valores de somente leitura para domínios que foram atualizados de uma versão anterior do Servidor GlassFish. Elas não são suportadas na versão v3.

A página Configurações do Caminho JVM contém as seguintes opções.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Caminho de Classe de Ambiente

Se a caixa de seleção Ignorar estiver marcada, a variável de ambiente CAMINHO DE CLASSE será ignorada. A variável de ambiente CAMINHO DE CLASSE é conveniente para tutoriais básicos de programação, mas não é recomendada para ambientes empresariais. Este é um campo somente leitura.

Caminho de Classe do Sistema

O caminho para as classes no seu sistema. Este é um campo somente leitura.

Prefixo de Caminho de Classe

Caminhos completos para os arquivos JAR a serem colocados no início do caminho de classe do servidor, um por linha. Este é um campo somente leitura.

Sufixo de Caminho de Classe

Caminhos completos para os arquivos JAR a serem anexados ao caminho de classe do servidor, um por linha. Este é um campo somente leitura.

Prefixo de Caminho da Biblioteca Nativa

Caminhos completos para os arquivos JAR a serem colocados no início do caminho da biblioteca nativa, um por linha.

Sufixo de Caminho da Biblioteca Nativa

Caminhos completos para os arquivos JAR a serem anexados ao caminho da biblioteca nativa, um por linha.
