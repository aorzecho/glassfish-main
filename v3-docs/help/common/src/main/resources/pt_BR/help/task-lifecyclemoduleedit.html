Para Editar um Módulo de Ciclo de Vida

  1. Na árvore de navegação, selecione o nó Módulos do Ciclo de Vida.

    A página Módulos do Ciclo de Vida é exibida.

  2. Na página Módulos do Ciclo de Vida, clique no nome do módulo do ciclo de vida a ser editado.

    A página Editar Módulo do Ciclo de Vida é aberta.

  3. No campo Nome da Classe, digite o nome totalmente qualificado do arquivo de classe do módulo do ciclo de vida.

    A classe deve implementar a interface com.sun.appserv.server.LifecycleListener.

  4. No campo Caminho da classe, digite o caminho da classe do módulo do ciclo de vida.

    O caminho da classe especifica onde o módulo do ciclo de vida está localizado. O valor default é domain-dir/applications. Se o módulo já estiver neste diretório (isto é, no caminho da classe do servidor), esse campo poderá ser deixado em branco.

  5. No campo Ordem de Carregamento, digite um valor especificando a ordem na qual esse módulo do ciclo de vida deve ser carregado na inicialização.

    Os módulos com valores inteiros menores de ordem de carregamento são carregados antes. O valor pode variar de 101 até o MAXINT do sistema operacional. Valores de 1 a 100 são reservados.

  6. No campo Descrição, digite uma descrição do módulo do ciclo de vida.

  7. Marque a caixa de seleção Status Ativado para ativar o módulo do ciclo de vida.

    Por default, a opção está ativada.

  8. Marque a caixa de seleção Inicialização de Instância de Prevenção de Falha de Carregamento para especificar que o servidor seja submetido a shutdown em caso de falha do carregamento do módulo do ciclo de vida.

    Por default, essa opção está desativada.

  9. A guia Destinos aparece se clusters ou instâncias de servidores standalone, além da instância default do servidor (server), existirem. Selecione a guia Destinos para exibir os destinos nos quais o módulo do ciclo de vida foi ativado. Consulte Para visualizar destinos da aplicação para ver instruções sobre como ativar ou desativar um módulo de ciclo de vida em um ou vários alvos.

  10. Clique em Salvar.
