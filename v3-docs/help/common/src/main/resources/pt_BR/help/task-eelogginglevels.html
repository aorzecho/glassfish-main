Para Configurar Níveis de Log

  1. Na árvore de navegação, expanda o nó Configurações.

  2. No nó Configuração, clique na configuração de cluster ou instância de servidor na qual deseja configurar as Definições do Logger.

    A página Configuração é exibida.

  3. Na página Configuração, clique em Definições do Logger.

    A página Definições do Logger da configuração selecionada se abre.

  4. Na página Definições do Logger, clique na guia Níveis de Log.

    A página Níveis de Log do Módulo é exibida.

  5. Na página Níveis de Log do Módulo, altere os níveis de log de um ou mais módulos, como a seguir.

    • Para alterar o nível de log de um módulo individual, selecione um novo valor da lista drop-down, oposta ao módulo cujo nível de log você deseja alterar.

    • Para alterar o nível de log de um grupo de módulos:

      1. Marque as caixas de seleção próximas aos nomes dos módulos cujos níveis de log você deseja alterar.

      2. Selecione um valor na lista drop-down na parte superior da tabela.

      3. Clique no botão Alterar Nível.

    As mensagens do nível selecionado ou superiores aparecerão no log. Escolha qualquer um dos valores a seguir (listados do maior para o menor).

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • DESLIGADO

    Para ver detalhes sobre os significados dos níveis de log, consulte Níveis de Log do Módulo. O nível default é INFO, portanto, por default, o log conterá as mensagens dos níveis INFO, WARNING ou SEVERE.

  6. Clique em Salvar.

    As alterações das definições do Nível de Log têm efeito imediato. Elas também são salvas para uso quando o servidor é reiniciado.
