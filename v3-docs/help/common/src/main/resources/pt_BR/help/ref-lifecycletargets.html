Destinos do módulo de ciclo de vida

Utilize a página Destinos do módulo do ciclo de vida para exibir os clusters e as instâncias de servidores standalone nos quais o módulo do ciclo de vida pode ser ativado.

A página Módulos do ciclo de vida contém as seguintes informações.

Nome do destino

Nome do cluster ou da instância de servidor standalone.

Ativo

Exibe true se o módulo do ciclo de vida estiver ativado no destino ou false se o módulo do ciclo de vida estiver desativado.

A tabela Destinos também contém as seguintes opções.

Gerenciar destinos

Botão para gerenciar os destinos do módulo do ciclo de vida.

Mais ações

Lista drop-down das ações seguintes.

Ativar

Ação para ativar o módulo do ciclo de vida no destino selecionado.

Desativar

Ação para desativar o módulo do ciclo de vida no destino selecionado.
