Para Exibir Componentes Instalados

A página de Componentes Instalados fornece informações sobre componentes da imagem de instalação que ainda não foi instalada.

A desinstalação de um componente envolve a remoção dos seguintes itens:

  • Todos os arquivos e diretórios introduzidos pela instalação original do componente

  • Todos os arquivos e diretórios introduzidos por atualizações do componente, depois da instalação original

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Add-ons Disponíveis.

  2. Na página Add-ons Disponíveis, clique na guia Componentes Instalados.

    A página Componentes Instalado é exibida.

Próximas Etapas

Para desinstalar qualquer componente instalado, pare primeiro o domínio do GlassFish Server. A seguir, use um destes comandos:

  • as-install-parent/bin/updatetool, que começa com o recurso gráfico da Ferramenta de Atualização

  • as-install-parent/bin/pkg, uma versão da linha de comandos da Ferramenta de Atualização

Para obter mais informações sobre o uso do comando pkg, consulte o Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server Administration Guide. O utilitário Ferramenta de Atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter finalizado a desinstalação dos componentes.
