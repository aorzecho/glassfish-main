Para exibir os componentes instalados

A página Componentes Instalados fornece informações sobre os componentes da imagem de instalação instalados.

A desinstalação de um componente envolve a remoção dos seguintes itens:

  • Todos os arquivos e diretórios introduzidos pela instalação original do componente

  • Todos os arquivos e diretórios introduzidos por atualizações do componente, depois da instalação original

  1. Na árvore de navegação, selecione o nó Ferramenta de atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, clique na guia Componentes Instalados.

    A página Componentes Instalado é exibida.

Passos Seguintes

Para desinstalar qualquer componente instalado, pare primeiro o domínio do Servidor GlassFish. A seguir, use um destes comandos:

  • as-install-parent/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de atualização

  • as-install-parent/bin/pkg, a versão de linha de comando da Ferramenta de atualização

Para obter informações sobre como utilizar o comando pkg, consulte Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server 3.1 Administration Guide. O utilitário Ferramenta de atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter terminado de desinstalar os componentes.
