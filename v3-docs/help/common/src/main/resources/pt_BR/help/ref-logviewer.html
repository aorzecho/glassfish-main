Visualizador de Log

Use a página Visualizador de Log para exibir os arquivos de log de instâncias e clusters GlassFish Server e para personalizar a exibição do log.

A página Visualizador de Logs contém três áreas:

  • Critérios de Pesquisa

  • Pesquisa Avançada

  • Modificar Pesquisa

  • Resultados da pesquisa

A área Critérios de Pesquisa contém estas opções.

Pesquisa Advançada

Clicar neste link abre uma área que permite fazer ajustes adicionais ao visualizador de logs.

Timestamp

O botão de opção Mais Recente exibe as mensagens mais recentes. O botão de opção Faixa Específica abre os campos De e Para que permite exibir mensagens apenas de um determinado período de tempo.

Para o valor Tempo, a sintaxe deve considerar que esta forma (SSS significa milissegundos):

hh:mm:ss.SSS

Por exemplo:

17:10:00.000

Se o valor De for posterior ao valor Para, aparecerá uma mensagem de erro.

Nível de Log

O nível de log selecionado para filtragem. Por default, a exibição inclui todas as mensagens que aparecem no log do servidor, no nível de log escolhido e nos níveis mais graves. Marque a caixa de seleção de nome "Não incluir mais mensagens severas" para exibir mensagens somente no nível escolhido.

Para assegurar que as mensagens que você deseja exibir apareçam no log do servidor, primeiro defina os níveis de log apropriados na página Níveis de Log do Módulo. Consulte Para Configurar Níveis de Log.

Se você escolher filtrar as mensagens de log com base no nível de log, apenas as mensagens que correspondem aos critérios de filtro específicos serão mostradas. Contudo, essa filtragem não afeta quais mensagens são registradas no log do servidor.

A área Pesquisa Avançada contém as seguintes opções.

Logger

Espaços para nomes do Logger a serem filtrados por módulo. Use shift-clique ou control-clique para escolher múltiplos espaços para nomes.

Selecionar um espaço para nomes em um nível mais elevado seleciona todos os espaços para nomes abaixo dele. Por exemplo, selecionar javax.enterprise.system também seleciona os loggers de todos os módulos naquele namespace: javax.enterprise.system.core , javax.enterprise.system.tools.admin e assim por diante.

Logger Personalizado

Os Loggers específicos de determinada aplicação, um por linha.

Se a aplicação tiver diversos módulos, você poderá exibir qualquer um deles. Por exemplo, suponha que a aplicação tenha loggers com os seguintes nomes:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Para exibir as mensagens de todos os módulos na aplicação, digite com.mycompany.myapp . Para exibir apenas mensagens do module2, digite com.mycompany.myapp.module2.

Quando você especifica um ou mais loggers, as mensagens dos módulos do GlassFish Server aparecem apenas se você as especificar explicitamente na área do Logger.

Exibir

O número de mensagens a exibir por vez. O número default de mensagens exibidas é 40. Outras opções são 100, 250 e 1.000.

Para visualizar rastreamentos de pilha, desmarque a caixa de seleção "Limitar mensagens muito longas". Por default, os rastreamentos de pilha não aparecem no visualizador; para exibi-las, clique no link (detalhes) para obter uma mensagem.

A área Modificar pesquisa permite especificar o cluster ou a instância do servidor e o nome de uma arquivo de log associado para exibição.

Instância

A instância do servidor ou cluster para o qual os registros de log serão exibidos. O default é servidor. As outras opções dependem dos clusters e instâncias que podem ser definidos e ativados para o domínio.

Arquivo de log

Nome do arquivo de log que deseja exibir. O default é server.log. As outras opções dependem das definições do log no destino selecionado.

A tabela Resultados do visualizador exibe as mensagens de log do servidor. Utilize botões Registros antes de e Registros depois de na parte superior da tabela para selecionar a faixa de registros que deseja exibir.

Em cada mensagem, a tabela Resultados do visualizador de log contém as seguintes informações.

Número do Registro

O número da mensagem no log do servidor.

Nível do Log

O nível do log da mensagem.

Mensagem

O texto breve da mensagem de log. Clique no link (detalhes) para exibir uma versão formatada de uma mensagem.

Logger

O módulo do logger que gerou a mensagem de log.

Timestamp

A data e a hora em que a mensagem de log foi emitida.

Name/Value Pairs

O nome e o ID do thread da mensagem de log.
