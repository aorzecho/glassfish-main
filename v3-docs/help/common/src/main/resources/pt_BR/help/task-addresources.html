Para Adicionar Recursos Configurados em um Arquivo XML

  1. Na árvore de navegação, selecione o nó Recursos.

    A página Recursos será aberta.

  2. Clique no botão Adicionar Recursos.

    A página Adicionar Recursos será aberta.

  3. Selecione uma das seguintes opções para especificar a localização do arquivo glassfish-resources.xml:

    Arquivo XML a Ser Submetido a Upload para o Servidor

    Digite o caminho para o arquivo caso ele se localize em uma máquina remota.

    Arquivo de XML Local que está Acessível de GlassFish Server

    Digite o caminho para o arquivo caso ele se localize na máquina do servidor de administração de domínios (DAS).

  4. Na lista drop-down Alvo, selecione o cluster ou instância do servidor standalone na qual serão criados os recursos.

  5. Clique em OK.
