Para exibir informações detalhadas sobre um componente

O Ferramenta de atualização traz informações detalhadas sobre componentes instalados, atualizações disponíveis para componentes instalados e componentes complementares.

  1. Na árvore de navegação, selecione o nó Ferramenta de atualização.

    É exibida a página Complementos Disponíveis.

  2. Clique na guia da página onde o componente está listado.

    O componente é listado em uma destas páginas:

    • Suplementos Disponíveis

    • Atualizações Disponíveis

    • Componentes Instalados

  3. Na tabela de componentes, selecione o componente.

    É exibida a página Detalhes do Componente do componente selecionado.
