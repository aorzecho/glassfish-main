Para excluir um usuário do realm administrativo

  1. No nó Configuração, expanda a configuração de cluster ou instância de servidor na qual deseja gerenciar os usuários de realm administrativos.
  2. Expanda o nó Segurança.
  3. Expanda o nó Domínios.
  4. Selecione o nó admin-realm.

    A página Editar Domínio é exibida.

  5. Na página Editar Domínio, clique no botão Gerenciar Usuários.

    A página Usuários Administrativos é exibida.

  6. Na página Usuários Administrativos, selecione a caixa de seleção à esquerda do nome do usuário a ser excluído.
  7. Clique em Excluir.

    Você é solicitado a confirmar a exclusão.

  8. Clique em OK para excluir o usuário.
