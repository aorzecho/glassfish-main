Gerenciar destinos

Utilize a página Gerenciar destinos do recurso para exibir os clusters e as instâncias de servidores independentes de destino nos quais um módulo do ciclo de vida pode ser ativado. O módulo do ciclo de vida pode ser ativado somente nos destinos da coluna Destinos selecionados.

A página Gerenciar destinos do recurso contém as seguintes informações.

Destinos disponíveis

Clusters e instâncias nos quais o módulo do ciclo de vida não pode ser ativado.

Destinos selecionados

Clusters e instâncias nos quais o módulo do ciclo de vida pode ser ativado.

Adicionar

Botão para mover o destino selecionado de Destinos disponíveis para Destinos selecionados.

Adicionar tudo

Botão para mover todos os Destinos disponíveis para Destinos selecionados.

Remover

Botão para remover o destino selecionado de Destinos selecionados para Destinos disponíveis.

Remover tudo

Botão para mover todos os Destinos selecionados para Destinos disponíveis.
