Para Deletar uma Configuração do Provedor de Segurança de Mensagens

Antes de Começar

É possível criar ou deletar os provedores de segurança de mensagens da configuração server-config default. Os únicos provedores de segurança de mensagem que são suportados no server-config default são SOAP e HttpServlet. Como estes dois provedores de mensagem já são criados por default quando o GlassFish Server é instalado, o botão Deletar não é exibido na página Configurações de Segurança de mensagens no server-config default.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Expanda o nó Segurança.

  3. Selecione o nó Segurança de Mensagens.

    A página Configurações de Segurança de Mensagens é exibida.

  4. Na página Configurações de Segurança de Mensagens, selecione SOAP.

    A página Editar Configuração de Segurança de Mensagens é exibida.

  5. Na página Editar Configuração de Segurança de Mensagens, clique na guia Provedores.

    A página Editar Configuração do Provedor é exibida.

  6. Na página Editar Configuração do Provedor, marque a caixa de seleção à esquerda da configuração do provedor que você está deletando.

  7. Clique em Deletar.
