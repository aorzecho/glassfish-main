Para visualizar informações gerais do DAS

  1. Na árvore de navegação, selecione o nó do servidor (servidor admin).

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, realize qualquer uma das etapas a seguir.
    • Clique no botão Parar para interromper o Servidor GlassFish.
    • Clique no botão Reiniciar para interromper e reiniciar o Servidor GlassFish.
    • Clique no botão Exibir arquivos de log para exibir os arquivos de log das instâncias ou clusters do Servidor GlassFish da .

      Consulte Para visualizar os arquivos de log do DAS para obter detalhes.

    • Clique no Relatório JVM para abrir uma janela separada que lhe permite exibir relatórios no Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória, um relatório de carregamento de classes e um despejo de segmentos atual.
