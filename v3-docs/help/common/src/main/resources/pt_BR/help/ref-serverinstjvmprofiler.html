Configurações do Gerador de Perfil de JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

Especifique as configurações do gerador de perfil. As informações especificadas dependem do produto gerador de perfis em uso. Para obter exemplos e instruções, consulte o Capítulo 3, Debugging Applications, no Oracle GlassFish Server 3.1 Application Development Guide.
