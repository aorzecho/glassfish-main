Definições de Caminho de JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

Especifique as definições do gerador de perfil. As informações especificadas dependem do produto gerador de perfis em uso. Para ver exemplos e instruções, consulte Capítulo 3, Aplicações de Depuração, em Oracle GlassFish Server Application Development Guide.
