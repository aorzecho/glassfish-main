Destinos do recurso

Utilize a página Destinos do recurso para exibir os clusters e as instâncias de servidores independentes de destino nos quais o recurso pode ser ativado.

A página Gerenciar recursos contém as seguintes informações de cada destino.

Nome do destino

Nome do cluster ou da instância de servidor independente.

Ativo

Exibe uma marca de seleção se o recurso estiver ativado no destino ou um X se o recurso estiver desativado.

A tabela Destinos também contém as seguintes opções.

Gerenciar destinos

Botão para gerenciar os destinos do recurso.

Ativar

Botão para ativar o recurso no destino selecionado.

Desativar

Ação para desativar o recurso no destino selecionado.
