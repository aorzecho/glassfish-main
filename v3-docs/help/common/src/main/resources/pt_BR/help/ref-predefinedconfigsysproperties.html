Propriedades Predefinidas do Sistema

O Servidor GlassFish define as seguintes propriedades do sistema de configuração:

ASADMIN_LISTENER_PORT

Essa propriedade especifica o número das portas HTTP ou HTTPS para administração. Essa é a porta no URL especificada no navegador da Web para gerenciar a instância, por exemplo, http://localhost:4949 . Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

HTTP_LISTENER_PORT

Essa propriedade especifica o número de porta que é utilizada para interpretar pedidos HTTP. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

HTTP_SSL_LISTENER_PORT

Essa propriedade especifica o número da porta que é utilizada para interpretar pedidos HTTPS. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

IIOP_LISTENER_PORT

Essa propriedade especifica o número da porta que é utilizada para conexões IIOP. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

IIOP_SSL_LISTENER_PORT

Essa propriedade especifica o número da porta que é utilizada para conexões IIOP seguras. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

IIOP_SSL_MUTUALAUTH_PORT

Essa propriedade especifica o número da porta que é utilizada para conexões IIOP seguras com autenticação do cliente. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

JAVA_DEBUGGER_PORT

Esta propriedade especifica o número da porta que é utilizada para as conexões com o depurador Java Platform Debugger Architecture (JPDA). Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

JMS_PROVIDER_PORT

Essa propriedade especifica o número da porta para o provedor Java Message Service. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

JMX_SYSTEM_CONNECTOR_PORT

Essa propriedade especifica o número da porta na qual o conector JMX escuta. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.

OSGI_SHELL_TELNET_PORT

Esta propriedade especifica o número da porta que é utilizada para as conexões com o shell remoto Apache Felix. Este shell utiliza o serviço de shell Felix para interagir com o subsistema de gerenciamento do módulo OSGi. Os valores válidos são 1–65535. Em UNIX, a criação de soquetes que escutam nas portas 1–1024 requer privilégios de superusuário.
