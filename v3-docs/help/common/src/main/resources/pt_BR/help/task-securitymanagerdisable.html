Para Desativar o Gerenciador de Segurança

A desativação do gerenciador de segurança do GlassFish Server pode melhorar o desempenho de alguns tipos de aplicações. Os recursos de autorização e autenticação da plataforma Java ainda funcionarão, mesmo que o gerenciador de segurança tenha sido desativado. Você pode desativar o gerenciador de segurança em um ambiente de desenvolvimento, porém, não desative-o em um ambiente de produção.

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Select the configuration that you are changing.

  3. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Definições Gerais de JVM é exibida.

  4. Na página Definições Gerais de JVM, clique na guia Opções de JVM.

    A página Opções de JVM é exibida.

  5. Marque a caixa de seleção à esquerda do campo Valor que contém a opção -Djava.security.policy.

  6. Clique em Deletar.

  7. Clique em Salvar.

  8. Reinicie o servidor.
