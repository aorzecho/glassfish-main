Para Desativar a Administração Segura

  1. Na árvore de navegação, selecione o nó do servidor (servidor admin).

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique no botão Administração Segura.

    A página Administração Segura será aberta.

  3. Clique em Desativar Admin. Segura.

    Uma mensagem será exibida, indicando que a desativação da administração segura reiniciará o DAS imediatamente.

  4. Clique em OK.

    A administração segura é desativada e a página de reinicialização do servidor é aberta.
