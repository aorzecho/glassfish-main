Para definir configurações de proxy de rede para acessar repositórios de rede

As configurações de proxy de rede determinam como o Servidor GlassFish conecta-se com a Internet. Para permitir que o Ferramenta de atualização acesse os repositórios da rede pela Internet, defina as configurações do proxy da rede.

  1. Na árvore de navegação, selecione o nó Ferramenta de atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, clique na guia Configuração.

    É exibida a página Configuração Ferramenta de atualização.

  3. Na página Configuração Ferramenta de atualização, especifique como a máquina na qual Servidor GlassFish está sendo executado se conecta com a Internet.
    • Se a máquina se conecta com a Internet diretamente, selecione a opção Conexão Direta.
    • Se a máquina se conecta com a Internet por meio de um proxy, selecione a opção Por meio de um Proxy.
  4. Se a máquina na qual o Servidor GlassFish está sendo executado se conectar com a Internet por meio de um proxy, forneça o nome do Host do Proxy e o número da Porta do Proxy.

    Se a máquina na qual o Servidor GlassFish está sendo executado se conectar com a Internet diretamente, omita esta etapa.

    Um navegador da web pode estar sendo executado na máquina na qual está sendo executado o Servidor GlassFish. Nesta situação, você pode obter o nome do host e o número da porta do proxy do navegador da web. Você também pode solicitar essa informação ao administrador do sistema.

  5. Clique em Salvar.
