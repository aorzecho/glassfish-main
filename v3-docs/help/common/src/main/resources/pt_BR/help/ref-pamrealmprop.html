Propriedades específicas da classe PamRealm

A seguinte propriedade é necessária para um domínio PAM.

Contexto JAAS

O contexto JAAS (Java Authentication and Authorization Service) (o identificador do módulo de login para usar este domínio). O único valor válido é pamRealm.

A propriedade opcional a seguir está disponível no domínio PAM.

Atribuir Grupos

Uma lista de nomes de grupos, separados por vírgula. Todos os clientes que apresentarem certificados válidos serão atribuídos a esses grupos; por exemplo, funcionário,gerente, onde esses forem os nomes de grupos de usuários.
