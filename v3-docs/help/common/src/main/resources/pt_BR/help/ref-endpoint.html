Informações de Ponto Final de Web Service

Usa a página Informações do Ponto Final do Web Service para exibir informações sobre um ponto final do Web service.

A página Informações do Ponto Final do Web Service contém as iinformações a seguir.

Nome da Aplicação

O nome da aplicação implantada. Clique neste link ou no botão Voltar para retornar à página Editar Aplicação.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Versões Modulares e da Aplicação em Oracle GlassFish Server Application Deployment Guide.

Testador

O link do Testador é visível se o Tipo de Implantação não for RI (por exemplo, se for 109). Clique neste link para visualizar uma página que permite fornecer parâmetros para testar os métodos do Web service.

WSDL

A WSDL (Web Services Definition Language) para o ponto final. Clique neste link para exibir a página WSDL em uma guia ou janela separada do browser.

Observe que alguns Web browsers tentam interpretar o código-fonte WSDL quando você clica no link WSDL e exibem uma página em branco. Nesses browsers, utilize o comando Exibir código-fonte na página em branco para exibir o códig-fonte WSDL.

Nome do ponto final

O nome do ponto final.

Nome do serviço

O nome do Web service.

Nome da porta

O nome da porta do Web service.

Tipo de Implantação

O tipo de implantação para o Web service (por exemplo, RI ou 109).

Tipo de implementação

O tipo de implementação para o Web service (por exemplo, SERVLET ou EJB).

Nome da classe de implementação

O nome da classe de implementação do Web service.

URI do endereço do ponto final

O URI do endereço do ponto final.

Namespace

O namespace de destino associado com o Web service.

Descrição

Descrição do ponto final, se houver um.
