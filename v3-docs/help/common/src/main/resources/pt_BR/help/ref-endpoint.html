Informações do ponto final dos serviços Web

Usa a página Informações do ponto final dos serviços Web para visualizar informações sobre um ponto final dos serviços Web.

A página Informações do ponto final dos serviços Web, contém as seguintes informações.

Nome do Aplicativo

O nome do aplicativo implementado. Clique neste link ou no botão Voltar para retornar à página Editar Aplicativo.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois pontos (:). Para obter mais informações sobre versões de aplicativos e módulos, consulte Module and Application Versions no Oracle GlassFish Server 3.1 Application Deployment Guide .

Testador

O link do Testador é visível se o Tipo de implementação não for RI (por exemplo, se for 109). Clique neste link para visualizar uma página que permite fornecer parâmetros para testar os métodos dos serviços Web.

WSDL

A Web Services Definition Language (WSDL) para o ponto final. Clique neste link para exibir a página WSDL em uma guia ou janela separada do navegador.

Observe que alguns navegadores da Web tentam interpretar a fonte da WSDL ao clicar no link WSDL e exibe uma página em branco. Em tais navegadores, utilize o comando Exibir fonte na página em branco para visualizar a fonte da WSDL.

Nome do ponto final

O nome do ponto final.

Nome do serviço

O nome do serviço Web.

Nome da porta

O nome da porta do serviço Web.

Tipo de implementação

O tipo de implementação para o serviço Web (por exemplo, RI ou 109).

Tipo de implementação

O tipo de implementação para o serviço Web (por exemplo, SERVLET ou EJB).

Nome da classe de implementação

O nome da classe de implementação do serviço Web.

URI do endereço do ponto final

O URI do endereço do ponto final.

Namespace

O namespace de destino associado com o serviço Web.

Descrição

Descrição do ponto final, se houver um.
