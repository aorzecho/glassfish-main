Para visualizar destinos do aplicativo

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida.

  2. Selecione um aplicativo do qual deseja visualizar os destinos.

    O nome do aplicativo pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois pontos (:). Para obter mais informações sobre versões de aplicativos e módulos, consulte Module and Application Versions no Oracle GlassFish Server 3.1 Application Deployment Guide .

    A página do aplicativo se abre.

  3. Selecione a guia Destino.

    A página Destinos do aplicativo se abre e os destinos atualmente especificados para o aplicativo são exibidos.

  4. Para ativar ou desativar um aplicativo em uma instância ou cluster de destino específico, clique na caixa de seleção ao lado do destino e selecione Ativar ou Desativar no menu suspenso Ações.
  5. Para ativar o balanceamento de carga de um destino em uma instância ou cluster de destino específico, clique na caixa de seleção ao lado do destino e selecione Ativar balanceador de carga ou Desativar balanceador de carga no menu suspenso Mais ações.
  6. Para associar um aplicativo a servidores virtuais específicos, selecione Gerenciar servidores virtuais na instância ou cluster de destino.

    A página Destinos de servidor virtual é exibida.

    Para adicionar servidores virtuais de destino, selecione-os na lista Destinos disponíveis e utilize o botão Adicionar para movê-los para a lista Destinos selecionados.

    Por padrão, o módulo ou aplicativo está associado ao servidor virtual padrão, denominado server. Utilize a botão Adicionar tudo para adicionar todos os destinos. Utilize os botões Remover e Remover tudo para remover os destinos da lista Destinos selecionados.

    Selecione Salvar para voltar à página Destinos do aplicativo.

  7. Para adicionar ou excluir instâncias ou clusters de destino do aplicativo, selecione Gerenciar destinos.

    A página Gerenciar destinos é exibida.

    Para adicionar destinos, selecione-os na lista Destinos disponíveis e utilize o botão Adicionar para movê-los para a lista Destinos selecionados.

    Utilize a botão Adicionar tudo para adicionar todos os destinos. Utilize os botões Remover e Remover tudo para remover os destinos da lista Destinos selecionados.

    Selecione Salvar para voltar à página Destinos do aplicativo.

  8. Clique em Salvar.

    O aplicativo está agora disponível na lista de destinos alterada.
