Para visualizar destinos da aplicação

  1. Na árvore de navegação, selecione o nó Aplicações.

    A página Aplicações é exibida.

  2. Selecione uma aplicação do qual deseja visualizar os destinos.

    O nome da aplicação pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Versões Modulares e da Aplicação em Oracle GlassFish Server Application Deployment Guide.

    A página da aplicação se abre.

  3. Selecione a guia Destino.

    A página Destinos da Aplicação se abre e os destinos atualmente especificados para a aplicação são exibidos.

  4. Para ativar ou desativar uma aplicação em uma instância ou cluster de destino específico, clique na caixa de seleção ao lado do destino e selecione Ativar ou Desativar no menu drop-down Ações.

  5. Para ativar o balanceamento de carga de um destino em uma instância ou cluster de destino específico, clique na caixa de seleção ao lado do destino e selecione Ativar balanceador de carga ou Desativar balanceador de carga no menu drop-down Mais ações.

  6. Para associar uma aplicação a servidores virtuais específicos, selecione Gerenciar servidores virtuais na instância ou cluster de destino.

    A página Destinos de servidor virtual é exibida.

    Para adicionar servidores virtuais de destino, selecione-os na lista Destinos disponíveis e utilize o botão Adicionar para movê-los para a lista Destinos selecionados.

    Por default, o módulo ou aplicação está associado ao servidor virtual default, denominado server. Utilize a botão Adicionar tudo para adicionar todos os destinos. Utilize os botões Remover e Remover tudo para remover os destinos da lista Destinos selecionados.

    Selecione Salvar para voltar à página Destinos da Aplicação.

  7. Para adicionar ou deletar instâncias ou clusters de destino da aplicação, selecione Gerenciar Destinos.

    A página Gerenciar destinos é exibida.

    Para adicionar destinos, selecione-os na lista Destinos disponíveis e utilize o botão Adicionar para movê-los para a lista Destinos selecionados.

    Utilize a botão Adicionar tudo para adicionar todos os destinos. Utilize os botões Remover e Remover tudo para remover os destinos da lista Destinos selecionados.

    Selecione Salvar para voltar à página Destinos da Aplicação.

  8. Clique em Salvar.

    A aplicação está agora disponível na lista de destinos alterada.
