Para definir o módulo de auditoria ativo

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Segurança.

    A página Segurança é exibida.

  3. Na lista suspensa Módulos de Auditoria, selecione o nome do módulo de auditoria a ser usado pelo servidor.

    Por padrão, só há um módulo de auditoria disponível, denominado padrão.

  4. Clique em Salvar.
