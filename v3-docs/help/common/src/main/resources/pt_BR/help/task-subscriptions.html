Para Exibir a Página Assinaturas do GlassFish Server

  1. Na árvore de navegação, selecione o nó Tarefas comuns.

    A página Tarefas Comuns é exibida.

  2. Em Notícias do GlassFish, clique no link Inscrições.

    A página Web Inscrições do GlassFish é exibida em uma guia ou janela separada do browser.
