Definições de Log

Utilize a página Definições do Logger para configurar o log do cluster ou instância selecionada do GlassFish Server.

A página Definições do Logger contém as opções a seguir.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Gravar no Log do Sistema

Se esta opção estiver selecionada, a saída do log será enviada ao recurso syslog além do log do servidor. Esse recurso está disponível apenas nos sistemas Solaris e Linux. Por default, essa opção está desativada.

Arquivo de Log

Um nome alternativo ou local do arquivo do log do servidor. A localização do default é domain-dir/logs/server.log.

Log Handler

O nome da classe absoluta de um log handler personalizado. Um log handler personalizado permite que você envie logs para um destino diferente de server.log ou syslog. O handler personalizado deve estender a classe java.util.logging.Handler (uma API compatível com JSR 047). Coloque a classe do handler no caminho da classe do GlassFish Server, para que o handler seja instalado durante a inicialização do servidor.

Limite de Rotação de Arquivo

O tamanho em bytes que o arquivo do log do servidor deve alcançar antes de ser rotacionado. Quando o log do servidor alcançar o tamanho especificado, o GlassFish Server cria um novo arquivo vazio denominadoserver.log e renomeia o arquivo antigo server.log_data, onde data é a data e a hora em que o arquivo foi rotacionado. O valor default é 2 megabytes. O valor mínimo para o limite é de 500 quilobytes; se você especificar um valor menor, o arquivo será rotacionado quando alcançar 500 KB. Para desativar a rotação de arquivos de log, defina o valor para 0.

Limite de Tempo de Rotação de Arquivo

O número de minutos após o qual o log do servidor deve ser rotacionado. O valor default é 0, o que significa que o arquivo é rotacionado quando ele alcança o tamanho especificado no campo Limite de Rotação de Arquivo. Se você especificar um ou mais minutos, o limite de tempo tem precedência sobre o limite de tamanho especificado pelo campo Limite de Rotação de Arquivo.

Frequência de Descarga

O número máximo de mensagens a serem escritas de cada vez da fila para o log do servidor. O valor default é 1.
