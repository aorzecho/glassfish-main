Módulos do Ciclo de Vida

Use a página Módulos do Ciclo de Vida para configurar os módulos do ciclo de vida.

Um módulo do ciclo de vida realiza tarefas quando ele é acionado por um ou mais eventos no ciclo de vida do servidor. Os possíveis eventos do servidor do disparador são: inicialização, pronto para solicitações de serviço e desligamento. Os módulos do ciclo de vida não fazem parte da especificação Java EE, mas são um aprimoramento do Servidor GlassFish.

Para cada módulo do ciclo de vida, as informações a seguir são fornecidas.

Nome

O nome do módulo do ciclo de vida.

Ativado ou status

Uma marca de seleção se o aplicativo estiver ativado ou um X se o aplicativo estiver desativado, somente se a instância padrão do servidor ,server, existir.

Se os clusters ou outras instâncias de servidores independentes existirem, selecione Ativado em Todos os destinos para visualizar os destinos nos quais o aplicativo está implantado.

Ordem de Carregamento

O valor que especifica a ordem relativa na qual o módulo do ciclo de vida deveser carregado.

A tabela Módulos do Ciclo de Vida também contém as seguintes opções.

Novo

Botão para criar um novo módulo do ciclo de vida.

Excluir

Botão para excluir um ou mais módulos do ciclo de vida selecionados.

Ativar

Botão para ativar um ou mais módulos do ciclo de vida selecionados. Presente somente se a instância padrão do servidor, server, existir.

Desativar

Botão para desativar um ou mais módulos do ciclo de vida selecionados. Presente somente se a instância padrão do servidor, server, existir.
