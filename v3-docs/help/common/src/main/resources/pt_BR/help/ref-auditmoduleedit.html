Editar Módulo de Auditoria

Use a página Editar Módulo de Auditoria para modificar um módulo de auditoria.

A página Editar Módulo de Auditoria contém estas opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do módulo de auditoria. Este é um campo somente leitura. Você só pode especificar o nome quando cria um novo módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria default é com.sun.enterprise.security.Audit .

Propriedades Adicionais

Propriedades adicionais do módulo de auditoria. Para obter uma descrição de uma propriedade adicional que está disponível para módulos de auditoria, consulte Propriedades Específicas para Auditar Módulos.
