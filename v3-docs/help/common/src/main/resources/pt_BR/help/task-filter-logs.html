Para filtrar registros de log exibidos

Antes de começar

Certifique-se de estar visualizando os arquivos de log do serviço de administração do domínio (DAS), uma instância do Servidor GlassFish em cluster ou uma instância independente. Para obter mais informações, consulte estes tópicos:

  • Para visualizar os arquivos de log do DAS

  • Para exibir arquivos de log de uma instância do Servidor GlassFish em cluster

  • Para exibir arquivos de log de uma instância do Servidor GlassFish independente

  1. Na área Critério de pesquisa, filtre os registros por conteúdo, período de tempo e nível de log.
    1. Para exibir somente os registros que contêm o texto específico, na caixa Pesquisa do texto, digite o texto.

      A pesquisa faz distinção entre maiúsculas e minúsculas.

    2. Para selecionar o período de tempo cujos registros de log são exibidos, utilize a opção Carimbo de hora.
      • Para exibir as mensagens mais recentes, selecione Mais recente (padrão).
      • Para exibir as mensagens apenas de um determinado período de tempo, selecione Intervalo Específico e digite os valores de data e hora nos campos De e Para que aparecem.

        O valor Data deve ter a seguinte forma:

        MM/DD/YY
        MM

        Inteiro no intervalo de 1 a 12 que corresponde ao mês, por exemplo, 1 para janeiro ou 10 para outubro.

        DD

        Inteiro no intervalo de 1 a 31 que corresponde ao dia do mês.

        YY

        Os últimos dois dígitos do ano, por exemplo, 11 para o ano de 2011.

        Por exemplo, para especificar a data de 1 de fevereiro de 2011, digite 2/1/11 no campo Data.

        O valor Hora deve ter a seguinte forma:

        hh:mm:ss.SSS
        hh

        Inteiro no intervalo de 0 e 23 representa a hora. Para as horas no período da tarde, utilize os inteiros de 12 a 23, por exemplo, 14 para representar 2 PM.

        mm

        Inteiro no intervalo de 0 a 59 para os minutos.

        ss

        Inteiro no intervalo de 0 a 59 para os segundos.

        SSS

        Inteiro de três dígitos, com zero no início se for necessário, no intervalo de 000 a 999 para os milissegundos.

        Por exemplo, para especificar a hora de 5:10 PM, digite 17:10:00.000 no campo Hora.

        se o valor De for posterior ao valor Para, aparecerá uma mensagem de erro.

    3. Filtrar mensagens pelo nível de log.

      Para assegurar que as mensagens que você deseja exibir apareçam no log do servidor, primeiro defina os níveis de log apropriados na página Níveis de log do módulo.

      Se optar por filtrar as mensagens de log pelo nível de log, apenas as mensagens que correspondam aos critérios de filtro especificados serão mostradas. Contudo, essa filtragem não afeta quais mensagens são registradas no log do servidor.

      1. Na lista suspensa Nível de log, selecione o nível de log.
      2. Para exibir mensagens somente no nível selecionado, selecione a opção Não incluir mais mensagens graves.

        Por padrão, são exibidas todas as mensagens no log do servidor no nível de log escolhido e nos níveis mais graves.

  2. Na área Pesquisa avançada, filtre registros pelo espaço para nome do agente de log e selecione a quantidade de informações que será exibida.
    1. Clique no link Pesquisa avançada.

      A área Pesquisa avançada é adicionada à janela Visualizador de log.

    2. Para filtrar por módulo, na lista suspensa Agente de log, selecione um ou mais espaços para nome.

      Utilize shift-clique ou control-clique para escolher vários espaços para nome.

      Selecionar um espaço para nomes em um nível mais elevado seleciona todos os espaços para nomes abaixo dele. Por exemplo, se javax.enterprise.system for selecionado, os agentes de log de todos os módulos naquele espaço para nome são selecionados: javax.enterprise.system.core , javax.enterprise.system.tools.admin e assim por diante.

    3. Para visualizar as mensagens dos agentes de log de um aplicativo em particular, no campo Agente de log personalizado, digite o nome de cada agente de log em outra linha.

      Se o aplicativo tiver diversos módulos, você poderá exibir qualquer módulo ou todos os módulos. Por exemplo, um aplicativo pode ter os seguintes agentes de log:

      • com.example.myapp.module1

      • com.example.myapp.module2

      • com.example.myapp.module3

      Para exibir as mensagens de todos os módulos no aplicativo, digite com.mycompany.myapp . Para exibir apenas mensagens do module2, digite com.example.myapp.module2.

      Quando você especifica um ou mais agentes de logs, as mensagens dos módulos do Servidor GlassFish aparecem apenas se você as especificar explicitamente na área do Agente de Log.

    4. Na lista suspensa Exibir, selecione o número de mensagens que será exibido.

      O número padrão de mensagens exibidas é 40. Outras opções são 100, 250 e 1.000.

    5. Para exibir os rastreamentos de pilha, desmarque a opção Limitar mensagens excessivamente longas.

      Por padrão, os rastreamentos de pilha não aparecem no visualizador. Para exibi-los, clique no link (detalhes) para obter uma mensagem.

    6. Para ocultar a área Pesquisa avançada, clique no link Pesquisa básica.
  3. Para especificar a instância e o arquivo de log associado que deseja visualizar, utilize a área Modificar pesquisa.
    • Na lista suspensa Instância, selecione a instância.

      O padrão é a instância inicialmente selecionada para exibir os arquivos de log. Se tiver optado por exibir os arquivos de log do servidor de administração do domínio (DAS), o padrão é o servidor, o qual representa o DAS.

    • Na lista suspensa Arquivo de log, selecione o arquivo de log.

      O padrão é server.log. As outras opções dependem das configurações do log no destino selecionado.

  4. Na área Resultados da Pesquisa, personalize a exibição das mensagens de log.

    A área Resultados da pesquisa exibe as 40 entradas mais recentes no log do servidor, com as configurações nas páginas Configurações do agente e log e Níveis de log.

    • Para ordenar as mensagens de outro modo, clique nas setas das colunas.
    • Para exibir a versão formatada de uma mensagem, clique no link (detalhes) da mensagem.

      A janela Detalhes da entrada de log se abre, a qual contém uma versão formatada da mensagem.

    • Para exibir entradas anteriores e posteriores no arquivo de log, clique em Registros antes de e Registros depois de na parte superior da tabela.
