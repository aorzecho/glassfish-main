Para Filtrar Registros de Log Exibidos

Antes de Começar

Certifique-se de estar visualizando os arquivos de log do serviço de administração de domínios (DAS), uma instância clusterizada do GlassFish Server ou uma instância standalone. Para obter mais informações, consulte estes tópicos:

  • Para Exibir Arquivos de Log para o DAS

  • Para exibir arquivos de log de uma instância clusterizada do GlassFish Server

  • Para exibir arquivos de log de uma instância do GlassFish Server standalone

  1. Na área Critério de pesquisa, filtre os registros por conteúdo, período de tempo e nível de log.

    1. Para exibir somente os registros que contêm o texto específico, na caixa Pesquisa do texto, digite o texto.

      A pesquisa faz distinção entre maiúsculas e minúsculas.

    2. Para selecionar o período de tempo cujos registros de log são exibidos, utilize a opção Timestamp.

      • Para exibir as mensagens mais recentes, selecione Mais recente (default).

      • Para exibir as mensagens apenas de um determinado período de tempo, selecione Faixa Específica e digite os valores de data e hora nos campos De e Até que aparecem.

        O valor Data deve ter a seguinte forma:

        MM/DD/YY

        MM é um número inteiro na faixa de 1-12 relativo ao número do mês, por exemplo: 1 para janeiro ou 10 para outubro. DD é um número inteiro na faixa de 1-31 relativo ao dia do mês. YY são os últimos dois dígitos do ano, por exemplo: 11 para o ano de 2011.

        Por exemplo, para especificar a data 1 de fevereiro de 2011, digite 2/1/11 no campo Data.

        O valor Hora deve ter a seguinte forma:

        hh:mm:ss.SSS

        hh é um número inteiro na faixa de 0-23 que representa a hora. Para representar as horas do período da tarde, use números inteiros na faixa de 12-23, por exemplo: 14 para representar 2 horas da tarde. mm é um número inteiro na faixa de 0-59 que representa o número de minutos passados em relação à hora. ss é um número inteiro na faixa de 0-59 que representa o número de segundos. SSS é um número inteiro de três dígitos, com zeros à esquerda, se necessário, na faixa de 000-999, que representa o número de milissegundos.

        Por exemplo, para especificar a hora de 5:10 PM, digite 17:10:00.000 no campo Hora.

        se o valor De for posterior ao valor Para, aparecerá uma mensagem de erro.

    3. Filtrar mensagens pelo nível de log.

      Para garantir que as mensagens que você deseja ver apareçam no log do servidor, primeiro defina os níveis apropriados de log na página Níveis de Log do Módulo. Consulte Para Configurar Níveis de Log para obter instruções.

      Se optar por filtrar as mensagens de log pelo nível de log, apenas as mensagens que correspondam aos critérios de filtro especificados serão mostradas. Contudo, essa filtragem não afeta quais mensagens são registradas no log do servidor.

      1. Na lista drop-down Nível de log, selecione o nível de log.

      2. Para exibir mensagens somente no nível selecionado, selecione a opção Não incluir mais mensagens graves.

        Por default, são exibidas todas as mensagens no log do servidor no nível de log escolhido e nos níveis mais graves.

  2. Na área Pesquisa Avançada, filtre registros pelo espaço para nome do logger e selecione a quantidade de informações que será exibida.

    1. Clique no link Pesquisa avançada.

      A área Pesquisa avançada é adicionada à janela Visualizador de log.

    2. Para filtrar por módulo, na lista drop-down Logger, selecione um ou mais espaços para nome.

      Utilize shift-clique ou control-clique para escolher vários espaços para nome.

      Selecionar um espaço para nomes em um nível mais elevado seleciona todos os espaços para nomes abaixo dele. Por exemplo, se javax.enterprise.system for selecionado, os loggers de todos os módulos naquele espaço para nome são selecionados: javax.enterprise.system.core , javax.enterprise.system.tools.admin e assim por diante.

    3. Para exibir mensagens dos loggers de uma aplicação específica, no campo Logger Personalizado, digite o nome de Logger em uma linha separada.

      Se a aplicação tiver diversos módulos, você poderá exibir qualquer módulo ou todos os módulos. Por exemplo, uma aplicação pode ter os seguintes loggers:

      com.example.myapp.module1
com.example.myapp.module2
com.example.myapp.module3

      Para exibir as mensagens de todos os módulos na aplicação, digite com.mycompany.myapp . Para exibir apenas mensagens do module2, digite com.example.myapp.module2.

      Quando você especifica um ou mais loggers, as mensagens dos módulos do GlassFish Server aparecem apenas se você as especificar explicitamente na área do Logger.

    4. Na lista drop-down Exibir, selecione o número de mensagens que será exibido.

      O número default de mensagens exibidas é 40. Outras opções são 100, 250 e 1.000.

    5. Para exibir os rastreamentos de pilha, desmarque a opção Limitar mensagens excessivamente longas.

      Por default, os rastreamentos de pilha não aparecem no visualizador. Para exibi-los, clique no link (detalhes) para obter uma mensagem.

    6. Para ocultar a área Pesquisa avançada, clique no link Pesquisa básica.

  3. Para especificar a instância e o arquivo de log associado que deseja visualizar, utilize a área Modificar pesquisa.

    • Na lista drop-down Instância, selecione a instância.

      O default é a instância inicialmente selecionada para exibir os arquivos de log. Se tiver optado por exibir os arquivos de log do servidor de administração de domínios (DAS), o default é o servidor, o qual representa o DAS.

    • Na lista drop-down Arquivo de log, selecione o arquivo de log.

      O default é server.log. As outras opções dependem das definições do log no destino selecionado.

  4. Na área Resultados da Pesquisa, personalize a exibição das mensagens de log.

    A área Resultados da pesquisa exibe as 40 entradas mais recentes no log do servidor, com as Definições nas páginas Definições do agente e log e Níveis de log.

    • Para ordenar as mensagens de outro modo, clique nas setas das colunas.

    • Para exibir a versão formatada de uma mensagem, clique no link (detalhes) da mensagem.

      A janela Detalhes da entrada de log se abre, a qual contém uma versão formatada da mensagem.

    • Para exibir entradas anteriores e posteriores no arquivo de log, clique em Registros antes de e Registros depois de na parte superior da tabela.
