Para Editar Informações do Usuário do Realm Admin

  1. No nó Configuração, expanda a configuração de cluster ou instância de servidor na qual você deseja gerenciar os usuários de realm administrativos.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

  4. Selecione o nó admin-realm.

    A página Editar Realm é exibida.

  5. Na página Editar Realm, clique no botão Gerenciar Usuários.

    A página Usuários Administrativos é exibida.

  6. Na página Usuários Administrativos, na coluna ID de Usuário da tabela, clique no nome do usuário a ser modificado.

    A página Editar Usuário do Realm Arquivo é exibida.

  7. Edite o campo Lista de Grupos para modificar os grupos aos quais o usuário pertence.

    Use vírgulas para separar os nomes de grupos. Esses grupos não precisam estar definidos em outro lugar.

  8. Digite uma nova senha nos campos Nova Senha e Confirmar Nova Senha para alterar a senha do usuário.

  9. Clique em Salvar.
