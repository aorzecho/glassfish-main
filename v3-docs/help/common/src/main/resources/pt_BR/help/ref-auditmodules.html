Módulos de Auditoria

Use a página Módulos de Auditoria para configurar os módulos de auditoria.

A auditoria é como o Servidor GlassFish grava eventos significativos, como erros ou vulnerabilidades de segurança. O Servidor GlassFish registra todos os eventos de autenticação em arquivos de log. Um log de acesso completo fornece uma pista sequencial dos eventos de acesso do Servidor GlassFish.

Para cada módulo de auditoria, as seguintes informações são fornecidas

Nome

O nome do módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria padrão é com.sun.enterprise.security.Audit .

A tabela Módulos também contém as seguintes opções.

Novo

Botão para criar um novo módulo de auditoria

Excluir

Botão para excluir um ou mais módulos de auditoria selecionados.
