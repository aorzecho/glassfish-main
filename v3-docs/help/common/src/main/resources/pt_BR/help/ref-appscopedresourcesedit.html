Para Modificar Recursos no Escopo da Aplicação

  1. Na árvore de navegação, selecione o nó Aplicações.

    A página Aplicações é exibida.

  2. Selecione uma aplicação cujos recursos com escopo definido pela aplicação você deseja ver.

    O nome da aplicação pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Versões Modulares e da Aplicação em Oracle GlassFish Server Application Deployment Guide.

    A página da aplicação se abre.

  3. Selecione a guia Recursos.

    A página Recursos com Escopo Definido pela Aplicação será aberta, e os recursos especificados no momento para a aplicação aparecerão em uma tabela com as seguintes colunas:

    Nome da Aplicação

    O nome da aplicação.

    Nome do Módulo

    O nome do módulo na aplicação.

    Nome do Recurso

    O nome do recurso no escopo da aplicação para a aplicação ou módulo.

    Tipo de Recurso

    O tipo de recurso, por exemplo Pools de Conexões JDBC ou Recursos do Conector.

  4. Clique no nome do recurso que você deseja modificar.

    A página que abrir vai depender do tipo de recurso. Por exemplo, se o recurso for um pool de conexões JDBC, a página Editar Pool de Conexões JDBC será aberta.
