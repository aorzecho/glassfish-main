Para Configurar o Serviço de Monitoramento

  1. Para abrir a página Serviço de Monitoramento, proceda de uma das formas a seguir:

    • Selecione o nó Dados de Monitoramento e clique no link Configurar Monitoramento relativo à instância do servidor cujo monitoramento você deseja configurar. Para configurar o monitoramento de um cluster, escolha qualquer instância do cluster.

    • Sob o nó Configurações, expanda a configuração da instância ou cluster do servidor cujo monitoramento você deseja configurar e expanda o nó Monitoramento.

    A página Serviço de Monitoramento é exibida.

  2. Marque a caixa de seleção Serviço de Monitoramento Ativado, para ativar o monitoramento para o GlassFish Server, ou cancele a seleção da caixa de seleção para desativar o monitoramento.

    Por default, a opção está ativada.

  3. Marque a caixa de seleção Monitoramento MBeans Ativado, para implantar todos os MBeans necessários para o monitoramento, ou cancele a seleção da caixa de seleção para cancelar a implantação destes MBeans.

    Por default, a opção está ativada.

    Se a caixa de seleção não estiver selecionada, não é possível exibir os dados de monitoramento no Console de Administração, mesmo se a caixa de seleção Monitorando Serviço estiver selecionada. Em vez disso, é preciso usar a linha de comando para exibir os dados do monitoramento. Caso você cancele a seleção da caixa de seleção, o perímetro da memória do GlassFish Server é reduzido.

  4. Marque a caixa DTrace Ativado para ativar o uso do recurso Solaris DTrace.

    Para usar o recurso Solaris DTrace, você também precisa ter o Oracle Enterprise Manager DTrace Monitoring. Por default, essa opção está desativada.

  5. Na tabela, realize uma das seguintes ações:

    • Na coluna Nível de Monitoramento, selecione BAIXO ou ALTO na lista drop-down, para um ou mais componentes na coluna Componente, para ativar o monitoramento para esses componentes.

    • Marque as caixas de seleção ao lado de um ou mais componentes, depois selecione BAIXO ou ALTO na lista drop-down Nível na parte superior da tabela e clique no botão Alterar Nível, para definir o nível de monitoramento para aquele valor , para todos esses componentes.

    Não há diferenças entre os níveis de monitoramento oferecidos pelas opções BAIXO e ALTO, nesta versão. Para desativar o monitoramento de um ou mais componentes, selecione DESLIGADO. O valor default é DESLIGADO.

    É preciso implantar uma aplicação, componente ou serviço, antes de configurar o monitoramento para o mesmo.

  6. Clique em Salvar.
