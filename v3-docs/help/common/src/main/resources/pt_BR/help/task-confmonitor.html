Para configurar o serviço de monitoramento

  1. No nó Configurações, expanda a configuração de cluster ou instância de servidor na qual deseja configurar o monitoramento.
  2. Expanda o nó Monitoramento.

    A página Serviço de Monitoramento é exibida.

  3. Selecione a caixa de seleção Serviço de Monitoramento Ativado, para ativar o monitoramento para o Servidor GlassFish, ou cancele a seleção da caixa de seleção para desativar o monitoramento.

    Por padrão, a opção está ativada.

  4. Selecione a caixa de seleção Monitoramento MBeans Ativado, para implementar todos os MBeans necessários para o monitoramento, ou cancele a seleção da caixa de seleção para desimplementar estes MBeans.

    Por padrão, a opção está ativada.

    Se a caixa de seleção não estiver selecionada, não é possível exibir os dados de monitoramento no Console de Administração, mesmo se a caixa de seleção Monitorando Serviço estiver selecionada. Em vez disso, é preciso usar a linha de comando para exibir os dados do monitoramento. Caso você cancele a seleção da caixa de seleção, o perímetro da memória do Servidor GlassFish é reduzido.

  5. Marque a caixa de seleção DTrace Ativado para habilitar o aplicativo DTrace do Solaris.

    Para poder usar o recurso DTrace do Solaris, é preciso também ter o Oracle Enterprise Manager DTrace Monitoring. Por padrão, a opção está desativada.

  6. Na tabela, realize uma das seguintes ações:
    • Na coluna Nível de Monitoramento, selecione BAIXO ou ALTO na lista suspensa, para um ou mais componentes na coluna Componente, para ativar o monitoramento para esses componentes.
    • Marque as caixas de seleção ao lado de um ou mais componentes, depois selecione BAIXO ou ALTO na lista suspensa Nível na parte superior da tabela e clique no botão Alterar Nível, para definir o nível de monitoramento para aquele valor , para todos esses componentes.

    Não há diferenças entre os níveis de monitoramento oferecidos pelas opções BAIXO e ALTO, nesta versão. Para desativar o monitoramento de um ou mais componentes, selecione DESLIG. O valor padrão é DESLIG.

    É preciso implementar um aplicativo, componente ou serviço, antes de configurar o monitoramento para o mesmo.

  7. Clique em Salvar.
