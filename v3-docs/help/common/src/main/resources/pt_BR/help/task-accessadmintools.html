Para dar acesso a ferramentas de administração

Somente usuários do grupo asadmin podem acessar o Console de Administração e o utilitário da linha de comando asadmin. Para dar aos usuários acesso a essas ferramentas de administração, adicione-as ao grupo asadmin no admin-realm.

  1. No nó Configurações, expanda a configuração de cluster ou instância de servidor na qual deseja configurar os privilégios de acesso de administração.
  2. Expanda o nó Segurança.
  3. Expanda o nó Domínios.
  4. Selecione o nó admin-realm.

    A página Editar Domínio é exibida.

  5. Na página Editar Domínio, clique no botão Gerenciar Usuários.

    A página Usuários Administrativos é exibida.

    Inicialmente, depois da instalação, o nome de usuário e senha do administrador inseridos na instalação são listados em um arquivo chamado admin-keyfile. Por padrão, esse usuário pertence ao grupo asadmin, que dá direitos para modificar o Servidor GlassFish. Atribua usuários a esse grupo somente se quiser dar a eles privilégios de administrador para o cluster ou instância selecionada do &ProductName.

    Se você adicionar usuários ao domínio admin-realm, mas atribuir o usuário a um grupo diferente do asadmin, as informações de usuários ainda serão gravadas no arquivo denominado admin-keyfile, porém os usuários não terão acesso às ferramentas administrativas ou a aplicativos no domínio file.

  6. Na página Usuários Administrativos, clique em Novo, para adicionar um novo usuário ao domínio admin-realm .

    A página Novo Usuário do Domínio Arquivo é exibida.

  7. No campo ID de Usuário, digite o nome do novo usuário.
  8. No campo Lista de Grupos, digite os nomes dos grupos aos quais o usuário vai pertencer.

    Para autorizar um usuário a fazer modificações no Servidor GlassFish, inclua o grupo asadmin na Lista de Grupos.

  9. Nos campos Nova Senha e Confirmar Nova Senha, digite a senha do novo usuário.
  10. Clique em OK para adicionar esse usuário ao domínio admin-realm ou clique em Cancelar para sair sem salvar.
