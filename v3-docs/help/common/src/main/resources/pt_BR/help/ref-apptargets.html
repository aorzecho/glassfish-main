Destinos do aplicativo

Utilize a página Destinos do aplicativo para exibir os clusters e as instâncias de servidores independentes de destino nos quais o aplicativo pode ser ativado.

A página Destinos do aplicativo contém as seguintes informações.

Nome do destino

Nome do cluster ou da instância de servidor independente.

Ativo

Exibe true se o aplicativo estiver ativado no destino ou false se o aplicativo estiver desativado.

Balanceador de carga ativado

Exibe true se o aplicativo estiver ativado para balanceamento de carga no destino ou false se o aplicativo estiver desativado para balanceamento de carga.

Servidores Virtuais

Para associar o aplicativo a servidores virtuais específicos em um destino, selecione Gerenciar servidores virtuais em tal destino.

A tabela Destinos também contém as seguintes opções.

Gerenciar destinos

Botão para gerenciar os destinos do aplicativo.

Mais ações

Lista suspensa das ações seguintes.

Ativar

Ação para ativar o aplicativo no destino selecionado.

Desativar

Ação para desativar o aplicativo no destino selecionado.

Ativar balanceador de carga

Ação para ativar o balanceamento de carga do aplicativo no destino selecionado.

Desativar balanceador de carga

Ação para desativar o balanceamento de carga do aplicativo no destino selecionado.
