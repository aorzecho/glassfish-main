Alvos da Aplicação

Utilize a página Destinos da Aplicação para exibir os clusters e as instâncias de servidores standalone de destino nos quais a aplicação pode ser ativada.

A página Destinos da Aplicação contém as informações a seguir.

Nome do destino

Nome do cluster ou da instância de servidor standalone.

Ativo

Exibirá true se a aplicação estiver ativada no destino ou false se a aplicação estiver desativada.

Balanceador de carga ativado

Exibirá true se a aplicação estiver ativada para balanceamento de carga no destino ou false se a aplicação estiver desativada para balanceamento de carga.

Servidores Virtuais

Para associar a aplicação a servidores virtuais específicos em um destino, selecione Gerenciar Servidores Virtuais no destino em questão.

A tabela Destinos também contém as seguintes opções.

Gerenciar destinos

Botão para gerenciar os destinos da aplicação.

Mais ações

Lista drop-down das ações seguintes.

Ativar

Ação para ativar a aplicação no destino selecionado.

Desativar

Ação para desativar a aplicação no destino selecionado.

Ativar balanceador de carga

Ação para ativar o balanceamento de carga da aplicação no destino selecionado.

Desativar balanceador de carga

Ação para desativar o balanceamento de carga da aplicação no destino selecionado.
