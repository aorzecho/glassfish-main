Editar alias de senha

Utilize a página Editar alias de senha para alterar a senha associada a um alias de senha existente.

A página Editar alias de senha contém as seguintes opções:

Nome do alias

O nome do alias. Esta opção não pode ser editada.

Senha

Senha que o alias de senha armazena na forma criptografada.

Confirmar senha

Senha que o alias de senha armazena na forma criptografada. Os valores de Senha e Confirmar senha são comparados para garantir que a senha tenha sido introduzida sem erros.
