Para configurar as definições de caminho de JVM

  1. Na árvore de navegação, selecione o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Configurações Gerais de JVM é exibida.

  3. Na página Configurações Gerais de JVM, clique na guia Configurações de Caminho.

    A página Configurações do Caminho JVM é exibida.

  4. No campo Prefixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem colocados no início do caminho da biblioteca nativa.

    Para colocar vários arquivos no início, digite um caminho por linha.

  5. No campo Sufixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem anexados ao caminho da biblioteca nativa.

    Para anexar vários arquivos, digite um caminho por linha.

  6. Clique em Salvar.
  7. Reinicie o servidor.
