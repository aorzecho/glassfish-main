Para Configurar as Definições do Caminho da JVM

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Select the configuration that you are changing.

  3. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Definições Gerais de JVM é exibida.

  4. Na página Definições Gerais de JVM, clique na guia Definições de Caminho.

    A página Definições do Caminho JVM é exibida.

  5. No campo Prefixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem colocados no início do caminho da biblioteca nativa.

    Para colocar vários arquivos no início, digite um caminho por linha.

  6. No campo Sufixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem acrescentados ao caminho da biblioteca nativa.

    Para acrescentar vários arquivos, digite um caminho por linha.

  7. Clique em Salvar.

  8. Reinicie o servidor.
