Tarefas Comuns

A página Tarefas Comuns fornece atalhos para as tarefas comuns do Console de Administração.

Use a página Tarefas Comuns para as seguintes operações:

  • Para exibir a página Notícias do GlassFish do Servidor GlassFish

  • Para exibir a página Inscrições do Servidor GlassFish

  • Para exibir uma lista de aplicativos implantados

  • Para implantar um aplicativo Web

  • Para implantar um Aplicativo Empresarial

  • Para implantar um cliente aplicativo

  • Para implantar um módulo conector

  • Para implantar um EJB Jar

  • Para exibir os componentes instalados

  • Para instalar componentes complementares

  • Para exibir as atualizações disponíveis para componentes

  • Para criar um pool de conexão de JDBC

A página Tarefas Comuns também fornece links para alguns dos documentos do Servidor GlassFish
