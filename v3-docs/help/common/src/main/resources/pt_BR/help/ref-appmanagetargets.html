Gerenciar Alvos

Utilize a página Gerenciar Destinos para exibir os clusters de destino e as instâncias de servidores standalone nos quais uma aplicação pode estar ativada. A aplicação só pode ser ativada nos destinos da coluna Destinos Selecionados.

A página Gerenciar destinos contém as seguintes informações.

Destinos disponíveis

Clusters e instâncias nos quais a aplicação não está implantada.

Destinos selecionados

Clusters e instâncias nos quais a aplicação está implantada.

Adicionar

Botão para mover o destino selecionado de Destinos disponíveis para Destinos selecionados.

Adicionar tudo

Botão para mover todos os Destinos disponíveis para Destinos selecionados.

Remover

Botão para remover o destino selecionado de Destinos selecionados para Destinos disponíveis.

Remover tudo

Botão para mover todos os Destinos selecionados para Destinos disponíveis.
