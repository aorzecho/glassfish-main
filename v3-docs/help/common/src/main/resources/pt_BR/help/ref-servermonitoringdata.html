Monitoramento da Aplicação

Use a página de Monitoramento do Servidor para exibir os dados de monitoramento para a instância do servidor.

A página Monitoramento do Servidor contém as seguintes informações.

Nome da Instância

O nome da instância do servidor.

Exibir

O servidor virtual, listener HTTP, lixeira ou sistema de thread para o qual as estatísticas de monitoramento são exibidas.

Para a descrição das propriedades monitoradas, consulte Capítulo 8, Administrando o Serviço de Monitoramento em Oracle GlassFish Server Administration Guide.
