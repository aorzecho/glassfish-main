Para Deletar um Realm

  1. No nó Configurações, expanda a configuração de cluster ou instância de servidor na qual você deseja configurar os realms.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

    É aberta a página Realms.

  4. Na página Realms, marque a caixa de seleção à esquerda do nome do realm a ser deletado.

  5. Clique em Deletar.
