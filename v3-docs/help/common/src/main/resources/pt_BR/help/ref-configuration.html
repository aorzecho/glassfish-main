Configuração

  • Configurações JVM

  • Configurações do Agente de Log

  • Recipiente Web

  • Recipiente EJB

  • Java Message Service

  • Segurança

  • Serviço de disponibilidade

  • Serviço de Transação

  • Serviço HTTP

  • Servidores Virtuais

  • Configuração de Rede

  • Conjuntos de Segmentos

  • ORB

  • Serviços Administrativos

  • Monitoramento

  • Serviço de gerenciamento de grupos

  • Propriedades do Sistema
