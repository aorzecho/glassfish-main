Valores da instância

Utilize a página Valores da instância para editar os valores de uma propriedade do sistema de configuração das instâncias do Servidor GlassFish que fazem referência à configuração nomeada selecionada.

A página Valores da instância exibe uma lista de instâncias que fazem referência à configuração selecionada, na qual a propriedade é definida.

Nome da instância

Nome atribuído à instância quando a instância foi criada. Ao clicar neste nome da instância, a página Informações gerais se abre.

Nome do cluster

O nome do cluster em que a instância é membro, se algum. Este campo é somente leitura.

Valor padrão

Valor definido para a propriedade na configuração nomeada que a instância faz referência. Este campo é somente leitura.

Novo valor

Valor da propriedade definida para a instância selecionada. Este valor substitui o valor padrão.
