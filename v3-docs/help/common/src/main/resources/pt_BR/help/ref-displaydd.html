Descritor de implantação

Use a página do Descritor de deployment para visualizar o descritor de deployment de um módulo.

A página Descritor de deployment contém as seguintes informações.

Nome do Aplicativo

O nome do aplicativo.

Nome do módulo

Nome do módulo que contém o descritor de deployment.

Nome do arquivo do descritor

Nome do arquivo do descritor de deployment.
