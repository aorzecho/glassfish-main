Configurações

Utilize a página Configurações para gerenciar as configurações e visualizar as instâncias independentes do servidor ou os clusters usando as configurações.

Para cada configuração, as seguintes informações são fornecidas.

Nome

O nome da configuração do

Instância

Lista de instâncias independentes e clusters usando a configuração.

A tabela Configurações também contém as seguintes opções.

Novo

Botão para criar uma nova configuração.

Excluir

Botão para excluir uma ou mais configurações selecionadas.
