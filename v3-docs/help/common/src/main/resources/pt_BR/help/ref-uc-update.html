Atualizações Disponíveis

A página Atualizações Disponíveis fornece informações sobre as versões mais recentes dos componentes da imagem de instalação instalados.

Para instalar as atualizações disponíveis, pare primeiro o domínio do Servidor GlassFish. A seguir, use um destes comandos:

  • glassfish3/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de atualização

  • glassfish3/bin/pkg, a versão de linha de comando da Ferramenta de atualização

Para obter informações sobre como utilizar o comando pkg, consulte Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server 3.1 Administration Guide. O utilitário Ferramenta de atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter terminado de instalar as atualizações.

A página Atualizações Disponíveis contém as seguintes opções.

Caminho da Instalação

O caminho do sistema de arquivos do diretório raiz ou de nível superior da imagem de instalação.

Atualizações Disponíveis

Para cada componente para o qual uma nova versão está disponível, esta tabela fornece as seguintes informações:

Componente

O nome do componente.

Categoria

A categoria de software que é associada ao componente; por exemplo, Servidores de Aplicativos, Ferramentas do Sistema e Bancos de Dados. A categoria é atribuída pelo autor do componente quando o componente é criado.

Versão Instalada

O número de versão do pacote em que o componente atual é fornecido.

Nova Versão

O número da versão da atualização disponível.

Alterar

Diferença de tamanho entre a versão instalada e a atualização disponível.

Origem

O nome do Repositório de rede a partir do qual o componente deve ser obtido.

Tarefas relacionadas
