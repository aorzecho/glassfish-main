Para visualizar as informações do ponto final para um serviço Web

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida.

  2. Na página Aplicativos, clique no nome do serviço Web cujas informações do ponto final você está exibindo.

    O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois pontos (:). Para obter mais informações sobre versões de aplicativos e módulos, consulte Module and Application Versions no Oracle GlassFish Server 3.1 Application Deployment Guide .

    A página Editar aplicativo é exibida.

  3. Na tabela Módulos e componentes, clique no link Exibir ponto final para o subcomponente.

    A página Informações do Ponto Final dos Serviços Web é exibida.

  4. Se o link do Testador estiver visível, clique neste link para visualizar uma página que permite fornecer parâmetros para testar os métodos dos serviços Web.

    O link do Testador é visível se o Tipo de implementação não for RI (por exemplo, se for 109). A página Testador é aberta em uma nova janela ou guia do navegador.

  5. Clique no link WSDL para visualizar o arquivo Web Services Definition Language (WSDL) do ponto final.

    O arquivo WSDL é aberto em uma nova janela ou guia do navegador.

    Observe que alguns navegadores da Web tentam interpretar a fonte da WSDL ao clicar no link WSDL e exibe uma página em branco. Em tais navegadores, utilize o comando Exibir fonte na página em branco para visualizar a fonte da WSDL.

  6. Clique no link Nome do Aplicativo ou no botão Voltar para retornar à página Editar Aplicativo.
