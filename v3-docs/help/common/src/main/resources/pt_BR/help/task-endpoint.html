Para Exibir Informações de Ponto Final de um Web Service

  1. Na árvore de navegação, selecione o nó Aplicações.

    A página Aplicações é exibida.

  2. Na página Aplicações, clique no nome do Web service cujas informações do ponto final você está exibindo.

    O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Versões Modulares e da Aplicação em Oracle GlassFish Server Application Deployment Guide.

    A página Editar Aplicação é exibida.

  3. Na tabela Módulos e componentes, clique no link Exibir ponto final para o subcomponente.

    A página Informações do Ponto Final dos Web Services é exibida.

  4. Se o link do Testador estiver visível, clique neste link para visualizar uma página que permite fornecer parâmetros para testar os métodos do Web service.

    O link do Testador ficará visível se o Tipo de Implantação não for RI (por exemplo, se for 109). A página Testador é aberta em uma nova janela ou guia do browser.

  5. Clique no link WSDL para visualizar o arquivo Web Services Definition Language (WSDL) do ponto final.

    O arquivo WSDL é aberto em uma nova janela ou guia do browser.

    Observe que alguns Web browsers tentam interpretar o código-fonte WSDL quando você clica no link WSDL e exibem uma página em branco. Nesses browsers, utilize o comando Exibir código-fonte na página em branco para exibir o códig-fonte WSDL.

  6. Clique no link Nome da Aplicação ou no botão Voltar para retornar à página Editar Aplicação.
