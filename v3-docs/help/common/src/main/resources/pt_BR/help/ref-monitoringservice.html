Serviço de Monitoramento

Use a página Serviço de Monitoramento para configurar as opções de monitoramento para componentes ou serviços individuais.

É preciso implantar uma aplicação, componente ou serviço, antes de configurar o monitoramento para o mesmo.

A página Serviço de Monitoramento contém estas opções:

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Monitoring Service

Se a caixa de seleção Serviço de Monitoramento Ativado estiver selecionada, o monitoramento é ativado para o GlassFish Server. Por default, a opção está ativada.

Monitoramento MBeans

Se a caixa de seleção Monitoramento MBeans Ativado estiver selecionada, todos os MBeans necessários para o monitoramento são implantados no GlassFish Server. Por default, a opção está ativada.

Se a caixa de seleção não estiver selecionada, não é possível exibir os dados de monitoramento no Console de Administração, mesmo se a caixa de seleção Monitorando Serviço estiver selecionada. Em vez disso, é preciso usar a linha de comando para exibir os dados do monitoramento. Caso você cancele a seleção da caixa de seleção, o perímetro da memória do GlassFish Server é reduzido.

DTrace

Se a caixa de seleção DTrace Ativado for selecionada e se você tiver Monitoramento de DTrace do Enterprise Manager do Oracle, você pode usar o recurso Solaris DTrace. Por default, essa opção está desativada.

Para cada componente disponível, as seguintes informações são fornecidas.

Componente

O nome do componente do GlassFish Server.

Nível de Monitoramento

O nível de monitoramento definido pelo componente. Para componente, os níveis disponíveis são BAIXO, ALTO e DESLIGADO. O nível default é DESLIGADO. Não há diferenças entre os níveis de monitoramento oferecidos pelas opções BAIXO e ALTO, nesta versão.

A tabela Definições do Nível do Componente também contém as seguintes opções.

Alterar Nível

Botão para alterar o nível de monitoramento para um ou mais componentes selecionados.

Nível

Lista drop-down a partir da qual é possível selecionar um nível de monitoramento a ser aplicado em todos os componentes selecionados.
