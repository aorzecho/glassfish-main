Usuários Administrativos

Use a página Usuários Administrativos para gerenciar os usuários do realm-admin. O realm-admin é um arquivo realm. Usuários e grupos do domínio arquivo são listados no arquivo principal, cuja localização está especificada na propriedade arquivo.

Inicialmente, após a instalação do Servidor GlassFish, o único usuário é o administrador especificado durante a instalação. Por padrão, esse usuário pertence ao grupo asadmin, no domínio admin, o que lhe direitos para modificar o Servidor GlassFish. Todos os usuários atribuídos a esse grupo terão privilégios de administrador, ou seja, eles terão acesso à ferramenta asadmin e ao Console de Administração.

A página Usuários Administrativos exibe uma lista de usuários do realm selecionado. Para cada usuário, as seguintes informações são fornecidas:

ID do Usuário

O nome de usuário.

Lista de Grupos

Os grupos aos quais o usuário pertence.

A tabela Usuários Administrativos também contém as seguintes opções.

Novo

Botão para criar um novo usuário.

Excluir

Botão para excluir um ou mais usuários selecionados.
