Usuários de Admin

Use a página Usuários Administrativos para gerenciar os usuários do realm-admin. O realm-admin é um realm de arquivo. Usuários e grupos do realm arquivo são listados no arquivo de chave, cuja localização está especificada na propriedade arquivo.

Inicialmente, após a instalação do GlassFish Server, o único usuário é o administrador especificado durante a instalação. Por default, esse usuário pertence ao grupo asadmin, no realm admin, o que lhe direitos para modificar o GlassFish Server. Todos os usuários designados a esse grupo terão privilégios de administrador, ou seja, eles terão acesso à ferramenta asadmin e ao Console de Administração.

A página Usuários Administrativos exibe uma lista de usuários do realm selecionado. Para cada usuário, as seguintes informações são fornecidas:

ID do Usuário

O nome de usuário.

Lista de Grupos

Os grupos aos quais o usuário pertence.

A tabela Usuários Administrativos também contém as seguintes opções.

Novo

Botão para criar um novo usuário.

Deletar

Botão para deletar um ou mais usuários selecionados.
