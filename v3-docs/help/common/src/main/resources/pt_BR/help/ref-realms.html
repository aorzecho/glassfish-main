Domínios

O Servidor GlassFish vem pré-configurado com estes domínios: arquivo, certificado e domínio admin. Também é possível criar ldap, JDBC, solaris e domínios personalizados. Geralmente, você tem um domínio de cada tipo em um servidor, mas, no Servidor GlassFish, há dois domínios de arquivo: arquivo e domínio admin. Há dois domínios do mesmo tipo, mas usados para finalidades diferentes: É possível também ter um banco de dados de certificado para cada servidor virtual do sistema.

Use a página Realms para configurar os realms.

Para cada realm, as seguintes informações são fornecidas.

Nome

O nome do domínio.

Nome da Classe

O nome da classe para o realm.

A tabela Realms também contém as seguintes opções.

Novo

Botão para criar um novo realm.

Excluir

Botão para excluir um ou mais realms selecionados.
