Propriedades específicas de provedores de JACC

A propriedade a seguir está disponível para um provedor de JACC.

repositório

O diretório que contém o arquivo da diretiva do JACC. Para o provedor de JACC default do Servidor GlassFish da , o diretório padrão é ${com.sun.aas.instanceRoot}/generated/policy . Esta propriedade não está definida por padrão para o provedor de JACC simple do Servidor GlassFish da .
