Principais conceitos do Ferramenta de atualização

O Ferramenta de atualização gerencia as instalações de componentes de software do Servidor GlassFish que estão contidos em um único diretório do sistema de arquivos do seu sistema.

A versão Console de Administração do Ferramenta de atualização permite efetuar o seguinte:

  • Instalar componentes adicionais

  • Exibir as atualizações para os componentes instalados

  • Exibir os componentes instalados

Para atualizar ou remover os componentes instalados, é preciso parar o domínio do Servidor GlassFish, e a seguir usar um dos seguintes comandos:

  • as-install-parent/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de atualização

  • as-install-parent/bin/pkg, a versão de linha de comando da Ferramenta de atualização

Os principais conceitos do Ferramenta de atualização estão descritos nas seções a seguir.

Componente

Um componente é a unidade instalável que o Ferramenta de atualização gerencia. Um componente consiste em arquivos que são instalados como uma unidade no sistema, em uma imagem de instalação.

Imagem de instalação

Uma imagem de instalação é o diretório que contém os componentes de software, incluindo o conteúdo do diretório. A imagem de instalação é adicionada ao Ferramenta de atualização como resultado do download e da instalação do software Servidor GlassFish. Para ativar a adição e a atualização dos componentes de software do Servidor GlassFish, cada imagem de instalação contém informações sobre a instalação de componentes.

Repositório de rede

Um repositório de rede contém estes itens:

  • Componentes que podem ser adicionados a uma imagem de instalação

  • Atualizações para componentes existentes em uma imagem de instalação

Uma imagem de instalação é associada a um ou mais repositórios de rede. Cada repositório de rede é representado por um nome e um URL associado. Um nome de repositório é exclusivo para cada imagem de instalação e representa a organização ou o projeto que gerencia o repositório e um escopo bem definido de conteúdo. Para aprender sobre repositórios pré-configurados de rede para o Servidor GlassFish, consulte Preconfigured Repositories for GlassFish Server no Oracle GlassFish Server 3.1 Administration Guide .

Componente de suplemento

Um componente de suplemento é um componente da imagem de instalação que ainda não foi instalado.
