Editar Usuário do Realm do Arquivo

Use a página Editar Usuário do Realm Arquivo para alterar os privilégios de acesso do usuário para o realm atual.

Observação:

Se a administração segura estiver ativada, você não poderá alterar a senha de um usuário admin realm para um valor vazio.

A página Editar Usuário do Realm Arquivo contém as opções a seguir.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome do Realm

O nome do realm selecionado atualmente. Este é um campo somente para leitura.

ID do Usuário

O nome da conta que você está editando. Este é um campo somente para leitura.

Lista de Grupos

Grupos ao qual este usuário pertence. Separe os nomes de grupos por vírgulas.

Para usuários do admin-realm, esse é um campo somente para leitura, e o grupo é asadmin.

Nova Senha

A nova senha do usuário.

Confirmar Nova Senha

A nova senha do usuário.
