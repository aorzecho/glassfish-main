Editar Usuário do Domínio Arquivo

Use a página Editar Usuário do Domínio Arquivo para alterar os privilégios de acesso do usuário para o domínio atual.

A página Editar Usuário do Domínio Arquivo contém estas opções.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Nome de Domínio

O nome do domínio selecionado atualmente. Este é um campo somente leitura.

ID do Usuário

O nome da conta que você está editando. Este é um campo somente leitura.

Lista de Grupos

Grupos ao qual este usuário pertence. Separe os nomes de grupos por vírgulas.

Para usuários do admin-domínio, esse é um campo somente leitura, e o grupo é asadmin.

Nova Senha

A nova senha do usuário.

Confirmar Nova Senha

A nova senha do usuário.
