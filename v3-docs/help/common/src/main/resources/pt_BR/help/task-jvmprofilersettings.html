Para configurar as definições do gerador de perfil do JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

  1. Na árvore de navegação, selecione o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Configurações Gerais de JVM é exibida.

  3. Na página Configurações Gerais de JVM, clique na guia Gerador de Perfil.

    A página Configurações do Gerador de Perfil do JVM é exibida.

  4. Especifique as configurações do gerador de perfil.

    As informações especificadas dependem do produto gerador de perfis em uso. Para obter exemplos e instruções, consulte o Capítulo 3, Debugging Applications, no Oracle GlassFish Server 3.1 Application Development Guide.

  5. Clique em Salvar.
  6. Reinicie o servidor.
