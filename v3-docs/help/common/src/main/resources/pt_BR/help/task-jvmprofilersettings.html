Para Configurar as Definições do Gerador de Perfis da JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Select the configuration that you are changing.

  3. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Definições Gerais de JVM é exibida.

  4. Na página Definições Gerais de JVM, clique na guia Gerador de Perfil.

    A página Definições do Gerador de PerfilJVM é exibida.

  5. Especifique as definições do gerador de perfil.

    As informações especificadas dependem do produto gerador de perfis em uso. Para ver exemplos e instruções, consulte Capítulo 3, Aplicações de Depuração, em Oracle GlassFish Server Application Development Guide.

  6. Clique em Salvar.

  7. Reinicie o servidor.
