Propriedades Predefinidas do Sistema

As propriedades do sistema a seguir são predefinidas para o Servidor GlassFish.

com.sun.aas.installRoot

O diretório onde o Servidor GlassFish está instalado.

com.sun.aas.instanceRoot

O diretório de nível superior de um domínio do Servidor GlassFish.

com.sun.aas.hostName

O nome do host (máquina).

com.sun.aas.javaRoot

O diretório de instalação do Java Runtime.

com.sun.aas.imqLib

O diretório da biblioteca do software Oracle Fila de Mensagens.

com.sun.aas.domainName

O nome do domínio. Esta propriedade não é usada na configuração padrão, mas pode ser usada para personalizar a configuração. O valor padrão é domain1.
