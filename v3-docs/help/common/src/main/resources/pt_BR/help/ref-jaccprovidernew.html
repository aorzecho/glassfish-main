Novo Provedor de JACC

Utilize a página Novo provedor de JACC para criar um novo Java Authorization Contract for Containers (JACC) compatível com o JSR-115/196 que possa ser utilizado por módulos de autorização de terceiros em aplicativos em execução no Servidor GlassFish da . O provedor de JACC é criado como um elemento jacc-provider dentro do elemento security-service no arquivo domain.xml do domínio.

A instalação padrão do Servidor GlassFish da inclui dois provedores de JACC, denominados default e simple . Todos os novos provedores JACC criados são somados a estes dois provedores padrão. Os provedores de JACC padrão do Servidor GlassFish da implementam um mecanismo simples de autorização com base em arquivo que cumpre com a especificação do JACC.

Todos os provedores de JACC do Servidor GlassFish da devem utilizar as APIs do serviço de autenticação e autorização Java (JAAS). O JAAS ativa os serviços para autenticar e fazer cumprir os controles de acesso dos usuários. O JAAS implementa uma versão do padrão da estrutura do módulo de autenticação plugável (PAM) da tecnologia Java.

O JSR 196 permite desenvolver plug-ins em camadas diferentes. É possível definir plug-ins que alterem a forma como o novo mecanismo de autenticação é configurado, como AuthConfigProvider e AuthConfigFactory. Também é possível definir os novos mecanismos de autenticação, como ServerAuthModule e ClientAuthModule.

A página Novo Provedor de JACC contém as seguintes opções.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Nome

O nome do provedor JACC. Este nome é utilizado para fazer referência ao elemento jacc-provider no domain.xml.

Configuração da Diretiva

Nome de classe totalmente qualificado do javax.security.jacc.PolicyConfigurationFactory.provider que implementa o javax.security.jacc.PolicyConfigurationFactory específico do provedor.

Provedor de Diretivas

Nome de classe totalmente qualificado do javax.security.jacc.policy.provider que implementa o java.security.Policy.

Propriedades adicionais

Propriedades adicionais do provedor de JACC. Para obter uma descrição das propriedades adicionais disponíveis para provedores de JACC, consulte Propriedades específicas de provedores de JACC.
