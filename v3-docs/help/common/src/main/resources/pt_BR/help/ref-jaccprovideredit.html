Editar provedor de JACC

Utilize a página Editar provedor de JACC para modificar um provedor de JACC do cluster ou instância selecionada do Servidor GlassFish da .

A página Editar Provedor de JACC contém as seguintes opções.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Nome

O nome do provedor JACC. Este é um campo somente leitura. Você só pode especificar o nome quando cria um novo provedor de JACC.

Configuração da Diretiva

Nome de classe totalmente qualificado do javax.security.jacc.PolicyConfigurationFactory.provider que implementa o javax.security.jacc.PolicyConfigurationFactory específico do provedor.

Provedor de Diretivas

Nome de classe totalmente qualificado do javax.security.jacc.policy.provider que implementa o java.security.Policy.

Propriedades adicionais

Propriedades adicionais do provedor de JACC. Para obter uma descrição das propriedades adicionais disponíveis para provedores de JACC, consulte Propriedades específicas de provedores de JACC.
