Editar Provedor de JACC

Use a página Editar Provedor de JACC para alterar um provedor de JACC para a instância ou cluster GlassFish Server selecionado.

A página Editar Provedor de JACC contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do de JACC. Este é um campo somente leitura. Você só pode especificar o nome quando cria um novo provedor de JACC.

Configuração da Política

Nome de classe totalmente qualificado do javax.security.jacc.PolicyConfigurationFactory.provider que implementa o javax.security.jacc.PolicyConfigurationFactory específico do provedor.

Provedor de Políticas

Nome de classe totalmente qualificado do javax.security.jacc.policy.provider que implementa o java.security.Policy.

Propriedades adicionais

Propriedades adicionais do provedor de JACC. Para ver uma descrição das propriedades adicionais que estão disponíveis para provedores de JACC, consulte Propriedades Específicas de Provedores de JACC.
