Descritores de Módulos

Use a página Descritores de módulos para exibir os descritores de módulos de um aplicativo.

A página Descritores de Módulos contém as seguintes opções.

Descritores da Implantação

Uma tabela que exibe o nome do aplicativo, o nome do subcomponente (se aplicável) e o nome do arquivo descritor.
