Para definir o domínio padrão

O domínio padrão é aquele usado pelo Servidor GlassFish para autenticação e autorização, caso o descritor de desenvolvimento de um aplicativo não especifique um domínio.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione o nó Segurança.

    A página Segurança é exibida.

  3. Na lista suspensa Domínio Padrão, selecione o domínio ativo que o servidor usará para autenticação.

    O valor padrão é arquivo.

  4. Clique em Salvar.
