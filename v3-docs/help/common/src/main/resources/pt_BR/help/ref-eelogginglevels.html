Níveis de Log de Módulo

Use a página Níveis de Log de Módulo para configurar os níveis de log de módulos individuais.

Para cada módulo, as seguintes informações são fornecidas.

Nome do Agente de Log

O nome do agente de log do módulo (por exemplo, javax.enterprise.system.tools.admin ).

Nível do Agente de Log

O nível de registro atual do módulo.

A tabela Configurações do agente de log também contém as seguintes opções.

Adicionar agente de log

Botão para adicionar um módulo de agente de log. Se clicar neste botão, uma linha é adicionada à tabela Configurações do agente de log. Insira as informações sobre o módulo do agente de log na nova linha.

Excluir agente de log

Botão para excluir um ou mais módulos selecionados.

Nível

Lista suspensa a partir da qual é possível selecionar um nível de registro a ser aplicado em todos os módulos selecionados.

Alterar Nível

Botão para alterar o nível de registro para um ou mais módulos selecionados.

Os seguintes níveis de log estão disponíveis. Eles estão listados do mais alto para o mais baixo.

SEVERE

Eventos que interferem com a execução normal do programa.

WARNING

Avisos, inclusive exceções.

INFO

Mensagens relacionadas à configuração do servidor ou status do servidor, excluindo erros.

CONFIG

Mensagens relacionadas à configuração do servidor.

FINE

Verbosidade mínima.

FINER

Verbosidade moderada.

FINEST

Verbosidade máxima.

OFF

Nenhuma mensagem de log.
