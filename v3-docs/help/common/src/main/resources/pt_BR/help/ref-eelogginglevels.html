Níveis de Log do Módulo

Use a página Níveis de Log de Módulo para configurar os níveis de log de módulos individuais.

Para cada módulo, as seguintes informações são fornecidas.

Nome do Logger

O nome do logger do módulo (por exemplo, javax.enterprise.system.tools.admin ).

Nível do Logger

O nível de log atual do módulo.

A tabela Definições do Logger também contém as seguintes opções.

Adicionar logger

Botão para adicionar um módulo de logger. Se clicar neste botão, uma linha é adicionada à tabela Definições do Logger. Insira as informações sobre o módulo do logger na nova linha.

Deletar logger

Botão para deletar um ou mais módulos selecionados.

Nível

Lista drop-down ns qual é possível selecionar um nível de log a ser aplicado em todos os módulos selecionados.

Alterar Nível

Botão para alterar o nível de log para um ou mais módulos selecionados.

Os seguintes níveis de log estão disponíveis. Eles estão listados do mais alto para o mais baixo.

SEVERE

Eventos que interferem com a execução normal do programa.

WARNING

Advertências, inclusive exceções.

INFO

Mensagens relacionadas à configuração do servidor ou status do servidor, excluindo erros.

CONFIG

Mensagens relacionadas à configuração do servidor.

FINE

Verbosidade mínima.

FINER

Verbosidade moderada.

FINEST

Verbosidade máxima.

DESLIGADO

Nenhuma mensagem de log.
