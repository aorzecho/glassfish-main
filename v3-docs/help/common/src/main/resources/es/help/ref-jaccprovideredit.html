Editar Proveedor JACC

Utilice la página Editar proveedor JACC para modificar un proveedor JACC para la instancia o cluster de Servidor GlassFish seleccionado.

La página Editar proveedor JACC contiene las siguientes opciones.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del proveedor JACC. Éste es un campo de sólo lectura. Únicamente puede especificar el nombre al crear un nuevo proveedor JACC.

Configuración de directivas

El nombre de clase completo de javax.security.jacc.PolicyConfigurationFactory.provider que implementa el javax.security.jacc.PolicyConfigurationFactory específico del proveedor.

Proveedor de directivas

El nombre de clase completo de javax.security.jacc.policy.provider que implementa java.security.Policy.

Propiedades adicionales

Propiedades adicionales del proveedor JACC. Para obtener una descripción sobre las propiedades adicionales disponibles para proveedores JACC, consulte Propiedades específicas de los proveedores JACC.
