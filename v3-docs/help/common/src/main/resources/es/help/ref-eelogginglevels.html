Niveles de registro del módulo

Utilice la página Niveles de registro del módulo para configurar los niveles de registro para módulos individuales.

Para cada módulo, se proporciona la siguiente información.

Nombre de registro

El nombre del registrador para el módulo (por ejemplo, javax.enterprise.system.tools.admin).

Nivel de registro

Nivel de registro actual del módulo.

La tabla Configuración de registrador también contiene las siguientes opciones.

Agregar registro

Botón para agregar un módulo de registro. Al hacer clic en este botón se agrega una fila a la tabla Configuración de registro. Especifique información sobre el módulo de registro en la fila nueva.

Eliminar registro

Botón para eliminar uno o más módulos seleccionados.

Nivel

Lista desplegable en la que puede seleccionar un nivel de registro para que se aplique a todos los módulos seleccionados.

Cambiar nivel

Botón para cambiar el nivel de registro para uno o más módulos seleccionados.

Están disponibles los siguientes niveles de registro. En la lista aparecen de mayor a menor.

SEVERE

Eventos que interfieren con la ejecución normal del programa.

WARNING

Advertencias, incluyendo excepciones.

INFO

Mensajes relacionados con la configuración o estado del servidor, excluyendo errores.

CONFIG

Mensajes relacionados con la configuración del servidor.

FINE

Cantidad mínima de detalles.

FINER

Cantidad moderada de detalles.

FINEST

Cantidad máxima de detalles.

DESACTIVADO

No hay mensajes de registro.
