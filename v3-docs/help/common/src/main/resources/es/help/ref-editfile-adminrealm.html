Propiedades específicas de la clase FileRealm

Las siguientes propiedades son necesarias para un dominiofile.

Contexto JAAS

El contexto JAAS (Servicio de autorización y autenticación de Java) (el identificador para el módulo de inicio de sesión que se utiliza para este dominio). El único valor válido es fileRealm

Archivo clave

Ruta completa y nombre del archivo donde el servidor almacenará toda la información sobre usuarios, grupos y contraseñas para este dominio. El valor predeterminado es domain-dir/config/keyfile para el dominio file y domain-dir/config/admin-keyfile para el dominio admin-realm .

El archivo clave para el dominio file está inicialmente vacío, así que los usuarios deben agregarse en la página Usuarios de archivo antes de que se utilice el dominio file.

El archivo clave para el dominio admin-realm inicialmente contiene el nombre de usuario del administrador, la contraseña del administrador en formato cifrado, y el grupo al que pertenece este usuario (asadmin de forma predeterminada). Para obtener más información sobre la adición de usuarios a admin-realm, consulte Cómo otorgar acceso a las herramientas de administración..

Nota - Los usuarios del grupo asadmin en admin-realm están autorizados a utilizar las herramientas Consola de administración y asadmin. Agregue sólo usuarios a este grupo que tiene privilegios administrativos del servidor.

La siguiente propiedad opcional se encuentra disponible para un dominio file.

Asignar grupos

Una lista separada por comas de nombres de grupos. Todos los clientes que presenten certificados válidos se asignan a estos grupos, por ejemplo, employee,manager , donde estos son los nombres de grupos de usuarios.
