Información general

Consulte la página Información general para comprobar la configuración de GlassFish Server y ver los datos de la Máquina virtual de Java.

La página Información general contiene la siguiente información.

Botón Detener

Haga clic en Detener para detener el GlassFish Server.

Botón Reiniciar

Haga clic en Reiniciar para reiniciar el GlassFish Server.

Botón Ver archivos de registro

Haga clic en el botón Visualizar archivos de registro para ver los archivos de registro para una instancia o clúster de GlassFish Server.

Nombre

El nombre del servidor.

JVM

Si hace clic sobre el Informe de JVM, se abrirá una ventana independiente que muestra informes en la máquina virtual de Java, incluyendo un informe resumido, un informe sobre administración de la memoria y recolección de elementos no utilizados, otro informe sobre carga de clases y un volcado de subprocesos actuales.

Directorio de configuración

Directorio del sistema anfitrión que contiene archivos de configuración, como domain.xml.

Versión instalada

identificador de versión del producto.

Depurar

Identifica si la depuración está habilitada. Si la depuración está habilitada, también aparecerá el número de puerto.

Puerto(s) HTTP

Puertos configurados actualmente para solicitudes y respuestas HTTP.

Puerto(s) IIOP

Puertos configurados actualmente para solicitudes y respuestas IIOP.
