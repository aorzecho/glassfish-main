Servicio de supervisión

Utilice la página Servicio de supervisión para configurar opciones de supervisión de servicios o componentes del servidor individual.

Debe implementar una aplicación, componente o servicio antes de configurar su supervisión.

La página Servicio de supervisión contiene las siguientes opciones.

Nombre de la configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Monitoring Service

Si la casilla de control Servicio de Supervisión Activado está seleccionada, la supervisión está activada para Servidor GlassFish. Esta opción se habilita por defecto.

MBeans de supervisión

Si la casilla de control MBeans de Supervisión Habilitados está seleccionada, todos los MBeans necesarios para la supervisión estarán desplegados en Servidor GlassFish. Esta opción se habilita por defecto.

Si la casilla de control no está seleccionada, no podrá ver datos de supervisión en la Consola de Administración incluso si la casilla de control Servicio de Supervisión está seleccionada. En su lugar, debe utilizar la línea de comandos para ver los datos de supervisión. Si anula la selección de la casilla de control, la huella de memoria de Servidor GlassFish se reduce.

DTrace

Si la casilla de control DTrace habilitado está seleccionada y tiene supervisión de Oracle Enterprise Manager DTrace, puede usar la utilidad Solaris DTrace. Esta opción se deshabilita por defecto.

Para cada componente disponible, se proporciona la siguiente información.

Componente

Nombre del componente de Servidor GlassFish.

Nivel de supervisión

Nivel de supervisión determinado por el componente. Para cada componente están disponibles los niveles BAJO, ALTO Y DESACTIVADO. El nivel predeterminado es DESACTIVADO. No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión.

La tabla Configuración de nivel de componentes también contiene las siguientes opciones.

Cambiar nivel

Botón para cambiar el nivel de supervisión de uno o más componentes seleccionados.

Nivel

Lista desplegable en la que puede seleccionar un nivel de supervisión para que se aplique a todos los componentes seleccionados.
