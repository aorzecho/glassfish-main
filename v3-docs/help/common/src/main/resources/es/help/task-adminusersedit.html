Para editar la información de los usuarios del dominio de administración

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para la que desee configurar la Configuración de registro.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.
  4. Seleccione el nodo admin-realm.

    Aparecerá la página Editar dominio.

  5. En la página Editar dominio, haga clic en el botón Administrar usuarios.

    Se abre la página Usuarios de administración.

  6. En la página Usuarios de administración, en la columna Id. de usuario de la tabla, haga clic en el nombre del usuario que desee modificar.

    Se abrirá la página Edición de usuario del dominio file.

  7. Edite el campo Lista de grupo para modificar los grupos a los que pertenece el usuario.

    Utilice comas para separar los nombres de grupo. Estos grupos no hace falta definirlos en otro lugar.

  8. Escriba una nueva contraseña en los campos Nueva contraseña y Confirmar nueva contraseña para modificar la contraseña del usuario.
  9. Haga clic en Guardar.
