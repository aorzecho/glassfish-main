Nuevo módulo de auditoría

Utilice la página Nuevo módulo de auditoría para definir un nuevo módulo de auditoría.

La página Nuevo recurso JDBC contiene las siguientes opciones.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del módulo de auditoría nuevo.

Nombre de clase

El nombre completo de la clase que implementa este módulo. El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.

Propiedades adicionales

Propiedades adicionales del módulo de auditoría. Para ver una descripción de las propiedades adicionales disponibles para los módulos de auditoría, consulte Propiedades específicas de los módulos de auditoría.
