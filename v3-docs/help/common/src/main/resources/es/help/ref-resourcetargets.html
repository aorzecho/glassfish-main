Destinos de recursos

Utilice la página Destinos de recursos para ver los clústeres de destino y las instancias de servidor independientes en los que se puede habilitar el recurso.

La página Destinos de recursos contiene la siguiente información para cada destino.

Nombre de destino

El nombre del clúster o instancia de servidor independiente.

Habilitado

Muestra una marca de verificación si el recurso está habilitado en el destino, o una equis (X) si está inhabilitado.

La tabla Destinos también contiene las siguientes opciones.

Administrar destinos

Botón para gestionar los destinos de recursos.

Habilitar

Acción para habilitar el recurso en el destino seleccionado.

Deshabilitar

Acción para inhabilitar el recurso en el destino seleccionado.
