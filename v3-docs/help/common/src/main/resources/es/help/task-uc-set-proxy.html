Cómo configurar los valores del servidor proxy de red para acceder a los depósitos de red

Los valores de configuración del proxy de red determinan la manera en que GlassFish Server se conecta a Internet. Para habilitar a Herramienta de actualización para que acceda a los depósitos de red a través de Internet, configure los valores de configuración del proxy de red.

  1. En el árbol de navegación, seleccione el nodo Herramienta de actualización.

    Se abrirá la página Complementos disponibles.

  2. En la página Complementos disponibles, haga clic en la ficha Configuración.

    Se muestra la página Configuración de Herramienta de actualización.

  3. En la página Configuración de Herramienta de actualización, especifique cómo se conecta a Internet el equipo en el que se está ejecuntado GlassFish Server.
    • Si el equipo se conecta a Internet directamente, seleccione la opción Conexión directa.
    • Si el equipo se conecta a Internet a través de un proxy, seleccione la opción Mediante un servidor proxy.
  4. Si el equipo en el que se está ejecutando GlassFish Server se conecta a Internet a través de un proxy, obtenga el nombre de host y el número de puerto del proxy.

    Si el equipo en el que se está ejecutando GlassFish Server se conecta directamente a Internet, omita este paso.

    Un navegador web debe estar ejecutándose en el equipo en el que se está ejecutando GlassFish Server. En esta situación, puede obtener el nombre de host y el número de puerto del proxy del navegador web. De lo contrario, póngase en contacto con el administrador del sistema para obtener más información.

  5. Haga clic en Guardar.
