Propiedades específicas de las configuraciones del proveedor de seguridad de los mensajes

Las siguientes propiedades están disponibles para la configuración del proveedor de seguridad de mensajes.

server.config

Directorio y nombre de archivo de un archivo XML que contiene la información de configuración del servidor. El valor por defecto es domain-dir/config/wss-server-config-1.0.xml.

debug

Si está configurado en true, se producirá el volcado de mensajes de depuración del proveedor del servidor en el registro del servidor. El valor predeterminado es falso.

dynamic.username.password

Si está configurado en true, señala el tiempo de ejecución del proveedor para recopilar el nombre de usuario y contraseña de CallbackHander para cada solicitud. Si está configurado en false, el nombre de usuario y la contraseña para wsse:UsernameToken(s) se recopila una vez, durante la inicialización del módulo. Esta propiedad es aplicable únicamente para ClientAuthModule. El valor predeterminado es falso.

encryption.key.alias

Especifica la clave de cifrado utilizada por el proveedor. Esta clave se identifica por su alias keystore. El valor predeterminado es s1as.

signature.key.alias

Especifica la clave de firma utilizada por el proveedor. Esta clave se identifica por su alias keystore. El valor predeterminado es s1as.
