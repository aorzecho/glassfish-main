Configuraciones de seguridad de mensajes

Utilice la página Configuraciones de seguridad de mensajes para ver y modificar las configuraciones disponibles para los proveedores de seguridad de mensajes.

Para configurar la seguridad de mensajes, necesita un conocimiento básico de seguridad y de conceptos de servicios web. Para obtener más información, consulte el Capítulo 3, Administering Message Security de Oracle GlassFish Server 3.1 Security Guide y el Capítulo 6, Administering Web Applications de Oracle GlassFish Server 3.1 Administration Guide.

GlassFish Server implementa la seguridad de los mensajes mediante proveedores de seguridad de mensajes integrados en la capa de procesamiento SOAP. Los proveedores de seguridad de mensajes dependen de otras facilidades de seguridad de GlassFish Server.

Para cada configuración, se proporciona la siguiente información.

Capa de autenticación

Capa del mensaje en la que se lleva a cabo la autenticación. De manera predeterminada, se proporciona una configuración SOAP. También es posible proporcionar una configuración HttpServlet.

Proveedor predeterminado

Proveedor de servidor predeterminado para esta configuración. No hay ningún proveedor de servidor configurado de manera predeterminada.

Proveedor de cliente predeterminado

Proveedor de cliente predeterminado para esta configuración. No hay ningún proveedor de cliente configurado de manera predeterminada.

La tabla Configuración de seguridad de mensajes también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración.

Borrar

Botón para eliminar una o varias configuraciones seleccionadas.
