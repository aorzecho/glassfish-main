Cambio de contraseña del administrador

  1. En el árbol de navegación, seleccione el nodo Servidor GlassFish.

    Aparecerá la página de Información general.

  2. Seleccione la ficha Contraseña del administrador.

    Se abre la página Nueva contraseña de administrador.

    La página mostrará el identificador de usuario y el nombre de grupo de la cuenta actual de administrador. De forma predeterminada, el Id. de usuario es admin, y el nombre del grupo es asadmin.

  3. En el campo Nueva contraseña, introduzca la nueva contraseña para la cuenta actual de administrador.
  4. En el campo Confirmar nueva contraseña, vuelva a escribir la nueva contraseña.
  5. Haga clic en Guardar.
