Recursos

Utilice la página Recursos para configurar los siguientes recursos para la instancia o cluster de Servidor GlassFish seleccionado:

  • JDBC

  • Conectores

  • Configuraciones del Adaptador de Recursos

  • Recursos de JMS

  • Sesiones JavaMail

  • JNDI

La página Recursos también contiene la opción siguiente:

Agregar Recursos

Botón para crear uno o más recursos desde un archivo glassfish-resources.xml.
