Configuración

Utilice el nodo de configuración para una instancia o clúster de GlassFish Server para configurar los siguientes componentes:

  • JVM Settings

  • Configuración de registro

  • Contenedor web

  • Contenedor EJB

  • Java Message Service

  • Seguridad

  • Servicio de disponibilidad

  • Servicio de transacción

  • Servicio HTTP

  • Servidores virtuales

  • Network Config

  • Conjuntos de subprocesos

  • ORB

  • Servicio de administración

  • Supervisión

  • Servicio de administración de grupos

  • Propiedades del sistema
