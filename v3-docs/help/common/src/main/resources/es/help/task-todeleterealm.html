Cómo eliminar un dominio

  1. En el nodo Configuraciones, amplíe la configración de instancia o clúster del servidor para el que desee configurar los dominios.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.

    Se abrirá la página Dominios.

  4. En la página Dominios, marque la casilla de verificación situada a la izquierda del nombre del dominio que se va a eliminar.
  5. Haga clic en Eliminar.
