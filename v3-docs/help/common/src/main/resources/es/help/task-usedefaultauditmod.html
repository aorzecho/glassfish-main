Cómo utilizar el módulo de auditoría predeterminado

El módulo de auditoría default registra solicitudes de autenticación y de autorización en el archivo de registro del servidor. Para obtener información sobre cómo cambiar la ubicación del archivo de registro consulte Para configurar parámetros generales de registros.

Las entradas de registro de autenticación incluyen la información siguiente:

  • Nombres de usuarios que han intentado autenticarse

  • El dominio que ha procesado la solicitud de acceso

  • La dirección del módulo web o componente EJB solicitado

  • Éxito o fallo de la solicitud

Independientemente de si se activa el registro de auditoría, Servidor GlassFish registra todos los sucesos de autenticación denegados.

Las entradas de registro de autorización incluyen la información siguiente:

  • Los nombres de los usuarios autenticados, si hay alguno

  • La dirección web o componente EJB solicitado

  • Éxito o fallo de las solicitudes

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  3. En la lista desplegable Módulos de auditoría, seleccione el módulo de auditoría llamado default.
  4. Haga clic en Guardar.
