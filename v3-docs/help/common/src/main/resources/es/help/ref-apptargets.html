Destinos de aplicaciones

Utilice la página Destinos de aplicaciones para ver los clústeres de destino y las instancias de servidor independientes en los que se puede habilitar la aplicación.

La página Destinos de aplicaciones contiene la siguiente información.

Nombre de destino

El nombre del clúster o instancia de servidor independiente.

Habilitado

Muestra true si se ha habilitado la aplicación en el destino, o false si la aplicación está inhabilitada.

Equilibrador de carga activado

Muestra true si se ha habilitado equilibrado de carga para la aplicación en el destino, o false si no se ha habilitado equilibrado de carga para la aplicación.

Servidores virtuales

Para asociar la aplicación con servidores virtuales específicos en un destino, seleccione Administrar servidores virtuales para ese destino.

La tabla Destinos también contiene las siguientes opciones.

Administrar destinos

Botón para gestionar los destinos de aplicación.

Más acciones

Lista desplegable con las siguientes acciones.

Habilitar

Acción para habilitar la aplicación en el destino seleccionado.

Deshabilitar

Acción para deshabilitar la aplicación en el destino seleccionado.

Habilitar Equilibrador de carga

Acción para habilitar el equilibrado de carga de la aplicación en el destino seleccionado.

Deshabilitar Equilibrador de carga

Acción para deshabilitar el equilibrado de carga de la aplicación en el destino seleccionado.
