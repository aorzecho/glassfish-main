Logs de Dominio

Utilice la página Logs de Dominio para recopilar y descargar archivos log para Servidor GlassFish.

La página Logs de Dominio contiene las siguientes opciones. Puede seleccionar un cluster o una instancia de servidor para el que desee recopilar logs.

Cluster

Lista desplegable de clusters si hay algún cluster.

Instancia

Lista desplegable de instancias de servidor.

Recopilar Logs

Botón para recopilar los logs en un archivo log-files-name.zip, donde name es el nombre del cluster o de la instancia.

El archivo log-files-name.zip se descarga mediante el mecanismo de descarga de su explorador.
