Nuevo usuario de dominio de archivo

Utilice la página Nuevo usuario del dominio de archivo para garantizar el acceso al dominio actual a un usuario nuevo.

La página Nuevo dominio contiene las siguientes opciones.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre de dominio

Nombre del dominio actualmente seleccionado. Éste es un campo de sólo lectura.

Id. de usuario

Id. de usuario para una cuenta de usuario nueva.

Lista de grupos

Grupos al que pertenecerá el usuario nuevo. Si introduce varios grupos, sepárelos con comas.

Para usuarios admin-realm, éste es un campo de sólo lectura, y el grupo es asadmin.

Contraseña nueva

Contraseña del usuario nuevo.

Contraseña nueva (confirmar)

Contraseña del usuario nuevo.
