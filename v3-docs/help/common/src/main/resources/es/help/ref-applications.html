Aplicaciones

La página Aplicaciones muestra una lista de aplicaciones que se despliegan en Servidor GlassFish. Puede ver y gestionar las aplicaciones que ya estén desplegadas y desplegar otras nuevas.

Para cada aplicación aparece la siguiente información.

Nombre

Nombre de la aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

Activado o Estado

Una marca de control si la aplicación está activada o una X si está desactivada, en el caso de que solo exista la instancia de servidor por defecto, server.

Si existen clusters u otras instancias de servidores independientes, seleccione Activado en Destinos para ver los destinos en los que está desplegada la aplicación.

Motores

Tipos de contenedores empleados por la aplicación. Entre los tipos de contenedor se incluyen los siguientes:

  • web

  • webservices

  • ejb

  • conector

  • appclient

  • weld (contenedor para los contextos y la inyección de dependencia para las aplicaciones de la plataforma Java EE)

Acción

Enlaza con las acciones que se pueden efectuar en el componente después de desplegarlo: volver a desplegar y a cargar todos los componentes, iniciar aplicaciones web y clientes de aplicaciones, y descargar stubs de cliente para los clientes de aplicaciones.

La tabla Aplicaciones Desplegadas también contiene las siguientes opciones.

Desplegar

Botón para desplegar una aplicación.

Anular Despliegue

Botón para anular el despliegue de una o más aplicaciones seleccionadas.

Activar

Botón para activar una o más aplicaciones seleccionadas, si solo existe la instancia de servidor por defecto, server.

Desactivar

Botón para desactivar una o más aplicaciones seleccionadas, si solo existe la instancia de servidor por defecto, server.

Filtro

Lista desplegable que filtra las aplicaciones por motor.
