Aplicaciones

La página Aplicaciones muestra una lista de aplicaciones que se implementan en Servidor GlassFish. Puede ver y administrar las aplicaciones que ya estén implementadas e implementar otras nuevas.

En cada aplicación aparece la siguiente información.

Nombre

El nombre de la aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte Module and Application Versions de Oracle GlassFish Server 3.1 Application Deployment Guide.

Activado o Estado

Una marca de verificación si la aplicación está habilitada o una X si está deshabilitada, en el caso de que solo exista la instancia de servidor predeterminada, server.

Si existen clústeres u otras instancias de servidores independientes, seleccione Habilitado en Destinos para ver los destinos en los que está implementada la aplicación.

Motores

Los tipos de contenedores empleados por la aplicación. Entre los tipos de contenedor se incluyen los siguientes:

  • web

  • webservices

  • ejb

  • conector

  • appclient

  • weld (el contenedor para los contextos y la inyección de dependencias para las aplicaciones de la plataforma Java EE)

Action (Acción)

Vínculos a las acciones que se pueden llevar a cabo en el componente después de implementarlo: Reimplementar y Recargar para todos los componentes, e Iniciar para las aplicaciones web.

La tabla Aplicaciones implementadas también contiene las siguientes opciones.

Implementar

Botón para implementar una aplicación.

Anular implementación

Botón para anular la implementación de una o más aplicaciones seleccionadas.

Habilitar

Botón para habilitar una o más aplicaciones seleccionadas, si solo existe la instancia de servidor predeterminada, server.

Deshabilitar

Botón para deshabilitar una o más aplicaciones seleccionadas, si solo existe la instancia de servidor predeterminada, server.

Filtrar

Lista desplegable que filtra las aplicaciones por motor.
