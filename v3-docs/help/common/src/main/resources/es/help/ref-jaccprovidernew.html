Nuevo proveedor JACC

Consulte la página Nuevo proveedor JACC para crear un nuevo proveedor JACC (Java Authorization Contract for Containers) que cumple la especificación JSR-115/196 que pueden utilizar módulos de autorización de terceros para las aplicaciones que se ejecutan en Servidor GlassFish. El proveedor JACC se crea como un elemento jacc-provider dentro del elemento security-service el archivo domain.xml del dominio.

La instalación predeterminada de Servidor GlassFish incluye dos proveedores de JACC denominados default y simple. Los nuevos proveedores JACC que se creen se añaden a estos dos proveedores predeterminados. Estos proveedores JACC de Servidor GlassFish predeterminados implementan un motor de autorización simple, basado en archivos, que cumple la especificación JACC.

Todos los proveedores JACC de Servidor GlassFish deben utilizar la API Java Authentication and Authorization Service (JAAS). JAAS habilita servicios para autenticar e imponer controles de acceso a los usuarios. JAAS implementa una versión de la tecnología Java del marco estándar Pluggable Authentication Module (PAM).

JSR 196 permite desarrollar complementos en diferentes niveles. Puede definir complementos que cambian la forma en que se configura el nuevo mecanismo de autenticación, como AuthConfigProvider y AuthConfigFactory. También puede definir nuevos mecanismos de autenticación, como ServerAuthModule y ClientAuthModule.

La página Nuevo proveedor JACC contiene las siguientes opciones.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del proveedor JACC. Este nombre se utiliza para hacer referencia al elemento jacc-provider elemento en domain.xml.

Configuración de directivas

El nombre de clase completo de javax.security.jacc.PolicyConfigurationFactory.provider que implementa el javax.security.jacc.PolicyConfigurationFactory específico del proveedor.

Proveedor de directivas

El nombre de clase completo de javax.security.jacc.policy.provider que implementa java.security.Policy.

Propiedades adicionales

Propiedades adicionales del proveedor JACC. Para obtener una descripción sobre las propiedades adicionales disponibles para proveedores JACC, consulte Propiedades específicas de los proveedores JACC.
