Tareas Comunes

La página Tareas Comunes cuenta con accesos directos a las tareas comunes de la Consola de Administración.

Utilice la página Tareas Comunes para realizar las siguientes operaciones:

  • Ver la Página Noticias de GlassFish de Servidor GlassFish

  • Ver la Página Suscripciones de Servidor GlassFish

  • Ver una Lista de Aplicaciones Desplegadas

  • Desplegar una Aplicación Web

  • Desplegar una Aplicación de Empresa

  • Desplegar un Cliente de Aplicaciones

  • Desplegar un Módulo del Conector

  • Desplegar un Archivo Jar de EJB

  • Configurar el Servicio de Supervisión

  • Ver Datos de Supervisión de Aplicación

  • Ver Datos de Supervisión de Servidor

  • Ver Datos de Supervisión de Recurso

  • Ver Componentes Instalados

  • Instalar Componentes de Complementos

  • Ver Actualizaciones Disponibles para Componentes

  • Crear un pool de conexiones JDBC

La página Tareas Comunes proporciona además enlaces a algunos documentos de Servidor GlassFish.
