Ver una lista de aplicaciones implementadas

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones. La tabla muestra las aplicaciones implementadas.

  2. Seleccione un tipo de módulo de la lista desplegable Filtro para filtrar la lista de aplicaciones por tipo de módulo.
