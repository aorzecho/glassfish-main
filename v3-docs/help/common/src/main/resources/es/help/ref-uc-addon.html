Complementos disponibles

La página Complementos disponibles ofrece información sobre los componentes de la imagen de instalación que todavía no están instalados.

La página Seguridad contiene las siguientes opciones.

Ruta de instalación

Ruta del sistema de archivos de la raíz o directorio superior de la imagen de instalación.

Instalar

Botón para instalar uno o más complementos seleccionados.

Available Add-Ons

Para cada componente de la imagen de instalación que todavía no está instalado, esta tabla proporciona la siguiente información.

Componente

Nombre del componente.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas de sistemas y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión

Número de versión del paquete en el que se proporciona el componente.

Tamaño de instalación

Cantidad de espacio en disco que consume la instalación del componente.

Origen

El nombre del Depósito de red del que se obtendría el componente.
