Cómo configurar el servicio de supervisión

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para el que desee configurar la supervisión.
  2. Amplíe el nodo Supervisión.

    Se abrirá la página Servicio de supervisión.

  3. Seleccione la casilla de verificación Servicio de supervisión habilitado para habilitar la supervisión para GlassFish Server, o anule la selección de la casilla de verificación para deshabilitar la supervisión.

    Esta opción se habilita por defecto.

  4. Seleccione la casilla de verificación MBean de supervisión habilitados para implementar todos los MBean necesarios para la supervisión, o anule la selección de la casilla de verificación para anular la implementación de estos MBean.

    Esta opción se habilita por defecto.

    Si la casilla de verificación no está seleccionada, no podrá ver datos de supervisión en la Consola de administración incluso si la casilla de verificación Servicio de supervisión está seleccionada. En su lugar, debe utilizar la línea de comandos para ver los datos de supervisión. Si anula la selección de la casilla de verificación, la huella de memoria de GlassFish Server se reduce.

  5. Seleccione la casilla de verificación DTrace activado para habilitar el uso de la utilidad DTrace de Solaris.

    Para poder utilizar la utilidad DTrace deSolaris también debe tener la supervisión DTrace Manager Enterprise de Oracle. Esta opción se deshabilita por defecto.

  6. En la tabla, realice una de las siguientes acciones.
    • En la columna Nivel de supervisión, seleccione BAJO o ALTO en la lista desplegable para uno o más componentes de la columna Componentes para habilitar la supervisión para esos componentes.
    • Seleccione las casillas de verificación junto a uno o más componentes y, a continuación, seleccione BAJO o ALTO en la lista desplegable en la parte superior de la tabla y haga clic en el botón Cambiar nivel para ajustar el nivel de supervisión en ese nivel para todos esos componentes.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para deshabilitar la supervisión de uno o más componentes, seleccione DESACTIVADO. El valor predeterminado es DESACTIVADO.

    Debe implementar una aplicación, componente o servicio antes de configurar su supervisión.

  7. Haga clic en Guardar.
