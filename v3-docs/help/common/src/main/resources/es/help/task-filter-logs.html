Cómo filtrar los mensajes de registro mostrados

Antes de Empezar

Compruebe que está viendo archivos de registro para el servidor de administración de dominio (DAS), una instancia en clúster de Servidor GlassFish o una instancia independiente. Para obtener más información, consulte los temas siguientes:

  • Ver Archivos Log para DAS

  • Cómo ver archivos de registro para una instancia de Servidor GlassFish agrupada en clúster

  • Cómo ver archivos de registro para una instancia de Servidor GlassFish independiente

  1. En el área Criterios de Búsqueda, filtre los registros por contenido, período de tiempo y nivel de registro.

    1. Para mostrar sólo los registros que contienen un texto específico, escriba el texto en el campo Búsqueda de texto.

      La búsqueda distingue entre mayúsculas y minúsculas.

    2. Para seleccionar el periodo de tiempo para el que se muestran los registros, utilice la opción Registro de Hora

      • Para ver los mensajes más recientes, seleccione Más reciente (el valor predeterminado).

      • Para ver mensajes sólo de un determinado periodo de tiempo, seleccione Intervalo específico e introduzca un valor de fecha y hora en los campos Desde y Hasta.

        El valor Fecha debe tener el formato siguiente:

        MM/DD/YY

        MM es un entero en el rango 1-12 para el número de mes; por ejemplo, 1 para enero o 10 para octubre. DD es un entero en el rango 1-31 para el día del mes. YY son los últimos dos dígitos del año; por ejemplo, 11 para el año 2011.

        Por ejemplo, para especificar la fecha 1 de febrero de 2011, escriba 2/1/11 en el campo de fecha.

        El valor Hora debe tener el formato siguiente:

        hh:mm:ss.SSS

        hh es un entero en el rango 0-23 para representar la hora. Para las horas por la tarde, utilice números enteros en el rango 12-23; por ejemplo, 14 para representar las 2 PM. mm es un entero en el rango 0-59 para el número de minutos después de la hora. ss es un entero en el rango 0-59 para el número de segundos. SSS es un entero de tres dígitos, con ceros iniciales si es necesario, en el rango 000-999 para el número de milisegundos.

        Por ejemplo, para especificar la hora 5:10 PM, escriba 17:10:00.000 en el campo Hora.

        Si el valor introducido en el campo De es posterior al valor introducido en el campo A, aparece un mensaje de error.

    3. Filtre los mensajes por nivel de log.

      Para garantizar que los mensajes que quiere visualizar aparecen en el log del servidor, primero defina los niveles de log adecuados en la página Niveles de Log del Módulo. Consulte Configurar los Niveles de Log para obtener instrucciones.

      Si decide filtrar mensajes de log según el nivel de log, solo se mostrarán los mensajes que se ajusten a los criterios de filtro especificados. Sin embargo, este filtro no afecta a qué mensajes están registrados en el log del servidor.

      1. En la lista desplegable Nivel de Log, seleccione el nivel de log.

      2. Para mostrar mensajes solo en el nivel seleccionado, seleccione la opción No incluir más mensajes graves.

        Por defecto, se muestran todos los mensajes del log del servidor en el nivel de log seleccionado y niveles más graves.

  2. En el área Búsqueda Avanzada, filtre los registros por espacio de nombres del registrador y seleccione la cantidad de información que se muestra.

    1. Haga clic en el enlace Búsqueda Avanzada.

      El área Búsqueda Avanzada se agrega a la ventana Visor de Logs.

    2. Para filtrar por módulo, seleccione uno o más espacios de nombres en la lista desplegable Registrador.

      Utilice Mayús-clic o Ctrl-clic para seleccionar varios espacios de nombres.

      La selección de un espacio de nombres en un nivel superior selecciona todos los espacios de nombre existentes por debajo. Por ejemplo, al seleccionar javax.enterprise.system también se seleccionan los registradores de todos los módulos existentes por debajo de ese espacio de nombres, como javax.enterprise.system.core, javax.enterprise.system.tools.admin, etc.

    3. Para ver mensajes de los registradores para una aplicación en concreto, escriba el nombre de cada registrador en una línea separada en el campo Registrador Personalizado.

      Si la aplicación tiene varios módulos, puede visualizar uno o todos ellos. Por ejemplo, una aplicación podría tener los siguientes registradores:

      com.example.myapp.module1
com.example.myapp.module2
com.example.myapp.module3

      Para ver los mensajes de todos los módulos de la aplicación, introduzca com.mycompany.myapp. Para ver solo los mensajes de module2, escriba com.example.myapp.module2.

      Cuando especifica uno o varios registradores personalizados, los mensajes de los módulos de Servidor GlassFish aparecen sólo si los indica explícitamente en el área Registrador.

    4. En la lista desplegable Mostrar, seleccione el número de mensajes que desea mostrar.

      El número por defecto de mensajes mostrados es 40. También puede elegir 100, 250 y 1000.

    5. Para ver rastreos de pila, desactive la casilla de control Limitar mensajes demasiado largos.

      Por defecto, los rastreos de pila no aparecen en el visor. Para ver rastreos de pila, haga clic en el enlace (detalles) de un mensaje.

    6. Para ocultar el área Búsqueda Avanzada, haga clic en el enlace Búsqueda Básica.

  3. Para especificar la instancia y el archivo log asociado que desea ver, utilice el área Modificar Búsqueda.

    • En la lista desplegable Instancia, seleccione la instancia.

      El valor por defecto es la instancia seleccionada inicialmente para ver sus archivos log. Si ha seleccionado ver los archivos log del servidor de administración de dominio (DAS), el valor por defecto es el servidor, lo que indica DAS.

    • En la lista desplegable Archivo Log, seleccione el archivo log.

      El valor por defecto es server.log. Otras opciones dependen de la configuración de log para el destino seleccionado.

  4. En el área Resultados de Búsqueda, personalice la visualización de mensajes de log.

    El área Resultados de Búsqueda muestra las últimas 40 entradas en el log del servidor, con la configuración especificada en las páginas Configuración de Registro y Niveles de Log.

    • Para ordenar los mensajes de una forma distinta, haga clic en las flechas de las columnas.

    • Para ver una versión con formato de un mensaje, haga clic en el enlace (detalles) del mensaje.

      Se abrirá la ventana Detalles de Entrada de Log, que contiene una versión con formato del mensaje.

    • Para ver entradas anteriores o posteriores del archivo log, haga clic en Registros anteriores a y Registros posteriores a, en la parte superior de la tabla.
