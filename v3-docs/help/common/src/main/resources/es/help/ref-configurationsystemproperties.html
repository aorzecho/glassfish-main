Propiedades del sistema

La página Propiedades de sistema muestra una lista de las propiedades del sistema Java definidas en la configuración con nombre seleccionada. Las propiedades del sistema Java se transfieren al iniciador de aplicaciones Java mediante la opción -D del iniciador de aplicaciones Java al iniciar Servidor GlassFish.

Estas propiedades definen la configuración de puertos predeterminada para todas las instancias de Servidor GlassFish que hacen referencia a la configuración.

Para obtener información sobre los parámetros de puertos predefinidos en una configuración, consulte Propiedades del sistema predefinidas .

Para cada propiedad se muestra la información siguiente:

Nombre de variable de instancia

Nombre de la propiedad del sistema.

Valor predeterminado

Valor que se define para la propiedad en la configuración especificada a la que hace referencia la instancia. Este campo es de sólo lectura.

Valores de instancia

Un vínculo a la página Valores de instancia de la propiedad.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se añade una fila a la tabla Propiedades adicionales.

Eliminar propiedades

Botón para eliminar una o varias propiedades seleccionadas. Cualquier propiedad que se elimine vuelve a su valor predeterminado o, si no se ha definido ningún valor predeterminado, se queda sin definir.

La página Propiedades de sistema también contiene las opciones siguientes:

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Reconfiguración dinámica

Si esta opción está habilitada, los cambios realizados en la configuración se aplican a las instancias que hacen referencia a la configuración sin necesidad de reiniciar dichas instancias.

Clústeres o instancias que utilizan esta configuración

Una lista de los clústeres y las instancias que hacen referencia a la configuración, con una indicación de si cada clúster o instancia se está ejecutando:

  • Si el clúster o instancia se está ejecutando, se muestra una marca de verificación (✓).

  • Si el clúster o instancia está detenido, se muestra un signo de exclamación (!).
