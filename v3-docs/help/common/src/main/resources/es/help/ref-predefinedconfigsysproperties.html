Propiedades del sistema predefinidas

Servidor GlassFish define la siguiente configuración de propiedades del sistema:

ASADMIN_LISTENER_PORT

Esta propiedad especifica el número de puerto del puerto HTTP o HTTPS puerto a través del cual el DAS se conecta a la instancia para administrar la instancia. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

HTTP_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para recibir las solicitudes HTTP. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

HTTP_SSL_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para recibir solicitudes HTTPS. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

IIOP_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para las conexiones IIOP. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

IIOP_SSL_LISTENER_PORT

Esta propiedad especifica el número del puerto que se utiliza para conexiones IIOP seguras. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

IIOP_SSL_MUTUALAUTH_PORT

Esta propiedad especifica el número del puerto que se utiliza para conexiones IIOP seguras con autenticación de cliente. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

JAVA_DEBUGGER_PORT

Esta propiedad especifica el número del puerto que se utiliza para las conexiones al depurador Java Platform Debugger Architecture (JPDA). Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

JMS_PROVIDER_PORT

Esta propiedad especifica el número de puerto del proveedor de Java Message Service. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

JMX_SYSTEM_CONNECTOR_PORT

Esta propiedad especifica el número de puerto en el que escucha el conector JMX. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.

OSGI_SHELL_TELNET_PORT

Esta propiedad especifica el número de puerto del puerto que se utiliza para las conexiones conApache Felix Remote Shell. Este shell utiliza el servicio Felix shell para interactuar con el subsistema de administración de módulos OSGi. Los valores válidos son 1–65535. En sistemas UNIX, la creación de sockets de escucha en los puertos 1-1024 requiere privilegios de superusuario.
