Ver Archivos Log No Procesados para DAS

  1. En el árbol de navegación, seleccione el nodo del servidor (Admin Server).

    Aparecerá la página de Información general.

  2. En la página Información General, haga clic en el botón Ver log raw.

    Se abre la ventana del visualizador de logs raw.
