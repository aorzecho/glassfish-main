Propiedades específicas de la clase PAMRealm

Un dominio PAM requiere la siguiente propiedad.

Contexto JAAS

El contexto JAAS (Servicio de autorización y autenticación de Java) (el identificador para el módulo de inicio de sesión que se utiliza para este dominio). El único valor válido es pamRealm.

La siguiente propiedad adicional está disponible para un dominio PAM.

Asignar grupos

Una lista separada por comas de nombres de grupos. Todos los clientes que presenten certificados válidos se asignan a estos grupos, por ejemplo, employee,manager , donde estos son los nombres de grupos de usuarios.
