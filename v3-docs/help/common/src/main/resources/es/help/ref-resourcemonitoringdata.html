Supervisión de recursos

Utilice la página Supervisión de recursos para ver los datos de supervisión de un recurso seleccionado. Para obtener una descripción de las propiedades supervisadas, consulte el Capítulo 8, Administering the Monitoring Service de Oracle GlassFish Server 3.1 Administration Guide.
