Usuarios admin

Utilice la página Usuarios de administrador para administrar los usuarios del dominio de administración. El dominio de administración es un dominio de archivo. Los usuarios y grupos del dominio file aparecen en el archivo de claves, cuya ubicación está especificada por la propiedad file.

Inicialmente, tras la instalación de Servidor GlassFish, el único usuario es el administrador introducido durante la instalación. Por defecto, este usuario pertenece al grupo asadmin, en el dominio admin-realm, lo que le da derecho a modificar Servidor GlassFish. Los usuarios asignados a este grupo tendrán privilegios de administrador, es decir que podrán acceder a la herramienta asadmin y a la Consola de Administración.

La página Usuarios de administración muestra una lista de los usuarios para el dominio seleccionado. Para cada recurso, se proporciona la siguiente información:

ID de Usuario

El nombre de usuario.

Lista de grupos

Los grupos a los que pertenece el usuario.

La tabla Usuarios de administración también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo usuario.

Borrar

Botón para eliminar uno o varios usuarios seleccionados.
