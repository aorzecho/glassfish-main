Usuarios de archivo

Utilice la página Usuarios de archivo para administrar los usuarios de dominios de archivo. Los usuarios y grupos del dominio file aparecen en el archivo de claves, cuya ubicación está especificada por la propiedad file.

Los usuarios del dominio file pueden pertenecer a un grupo Java, una categoría de usuarios clasificados por rasgos en común. Por ejemplo, los clientes de una aplicación de comercio electrónico pueden pertenecer al grupo CLIENTE, pero los que más gastan pertenecerían al grupo PREFERENTE. La clasificación de usuarios en grupos simplifica el control del acceso de grandes números de usuarios.

Inicialmente, tras la instalación de GlassFish Server, el único usuario es el administrador introducido durante la instalación. Por defecto, este usuario pertenece al grupo asadmin, en el dominio admin-realm, lo que le da derecho a modificar GlassFish Server. Los usuarios asignados a este grupo tendrán privilegios de administrador, es decir que podrán acceder a la herramienta asadmin y a la Consola de administración.

La página Usuarios del archivo muestra una lista de usuarios del dominio seleccionado. Para cada recurso, se proporciona la siguiente información:

Id. de usuario

El nombre de usuario.

Lista de grupos

Los grupos a los que pertenece el usuario.

La tabla Usuarios de archivos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo usuario.

Borrar

Botón para eliminar uno o varios usuarios seleccionados.
