Recuperar transacciones

Utilice la página Recuperar transacciones para recuperar transacciones para una instancia de Servidor GlassFish.

Si la instancia se está ejecutando, la página Recuperar transacciones solo contiene el texto Recovery is performed by the running server itself (La recuperación la realiza el propio servidor en ejecución) y el botón Aceptar.

Si la instancia no se está ejecutando, la página Recuperar transacciones contiene las siguientes opciones.

Nombre de instancia

El nombre asignado a la instancia durante su creación. Este campo es de sólo lectura.

Servidor de destino

Otra instancia que puede recuperar las transacciones en la instancia actual.

Directorio de registro de transacciones

Directorio en el cual se almacenan los registros de transacciones. El valor predeterminado es /logs/tx en el directorio de dominio.

Los registros de transacciones se guardan en el subdirectorio tx del directorio especificado en el campo Ubicación del registro de transacción del servicio de transacción para la configuración del clúster. El valor predeterminado de Ubicación del registro de transacción es el directorio especificado en el campo Raíz de registro de la página Atributos de dominio, que está ubicada en la ficha Avanzado del nodo Servidor GlassFish.
