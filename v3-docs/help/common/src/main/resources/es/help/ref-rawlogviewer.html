Visor de Logs No Procesados

Utilice la página Visor de Logs No Procesados para ver los archivos log no procesados del servidor de administración (llamado server).

La página Visor de Logs No Procesados contiene las opciones siguientes.

Instancia

La instancia de servidor o el cluster para el que se mostrarán los registros de log. La única opción es server.

Para cada mensaje, el Visor de Logs No Procesados muestra la información siguiente.

Registro de Hora

Fecha y hora en que se emitió el mensaje de log.

Número de Registro

Número del mensaje en el log del servidor.

Nivel de Log

Nivel de log del mensaje.

Versión

Versión de Servidor GlassFish.

Registrador

Módulo de registrador que generó el mensaje de log.

Pares de Nombre-Valor

Nombre de thread e ID del mensaje de log.

Mensaje

Breve texto del mensaje de log. Haga clic en el enlace (details) para ver una versión con formato de un mensaje.
