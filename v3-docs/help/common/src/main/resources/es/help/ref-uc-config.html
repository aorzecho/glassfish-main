Configuración de Herramienta de actualización

La página Configuración ofrece información de configuración sobre Herramienta de actualización y permite cambiar cómo se conecta a Internet Herramienta de actualización. Para modificar otros aspectos de la configuración de Herramienta de actualización, utilice la aplicación updatetool, que se encuentra en as-install-parent/bin.

La página Configuración de Herramienta de actualización contiene las siguientes opciones.

Ruta de instalación

Ruta del sistema de archivos de la raíz o directorio superior de la imagen de instalación.

Depósito preferido

Nombre del depósito de red preferido que se asocia con esta imagen de instalación, por ejemplo, glassfish.org.

Conexión a Internet

Cómo conectar Servidor GlassFish a Internet:

Conexión directa

Esta opción especifica que Servidor GlassFish se conecta directamente a Internet.

Mediante un servidor proxy

Esta opción especifica que Servidor GlassFish se conecta a Internet mediante un servidor proxy. Cuando esta opción está seleccionada, se incluirán en esta página el campo Host del proxy y el campo Puerto del proxy.

Host del proxy

El nombre de host del servidor proxy del equipo en el que se está ejecutando Servidor GlassFish.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor GlassFish se conecta a Internet a través de un servidor proxy.

Puerto del proxy

El número de puerto del servidor proxy del equipo en el que se está ejecutando Servidor GlassFish.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor GlassFish se conecta a Internet a través de un servidor proxy.

Depósito

El nombre del depósito, por ejemplo, dev.glassfish.org.
