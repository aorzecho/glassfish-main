Suprimir un Módulo de Auditoría

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. En el nodo Configuración, amplíe el nodo Seguridad.

  4. En el nodo Seguridad, seleccione el nodo Módulos de Auditoría.

    Se abrirá la página Módulos de Auditoría.

  5. En la página Módulos de Auditoría, active la casilla de control que aparece a la izquierda del módulo de auditoría que está suprimiendo.

  6. Haga clic en Suprimir.
