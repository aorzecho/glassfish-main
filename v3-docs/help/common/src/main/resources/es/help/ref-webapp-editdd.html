Descriptores del Módulo

Utilice la página Descriptores del Módulo para ver los descriptores del módulo para una aplicación.

La página Descriptores del Módulo contiene las siguientes opciones.

Descriptores de Despliegue

Tabla que muestra el nombre de aplicación, el del subcomponente (si procede) y el del archivo descriptor.
