Atributos de Dominio

Utilice la página Atributos de Dominio para definir atributos de dominio avanzados de Servidor GlassFish.

La página Atributos de Dominio contiene las siguientes opciones.

Raíz de Aplicación

Ruta de acceso completa del directorio donde residen las aplicaciones desplegadas. El valor por defecto es domain-dir/applications.

Raíz de Log

Identifica el lugar en el que se guardan los archivos log del servidor. El valor por defecto es domain-dir/logs.

Configuración Regional

Si se deja este campo en blanco, se utilizará la configuración regional por defecto del host.

La configuración regional es un identificador que especifica una combinación concreta de idioma y región. Por ejemplo, la configuración regional para inglés de Estados Unidos es en_US y para japonés es ja_JP. Para utilizar una configuración regional que no sea del idioma inglés, Servidor GlassFish tiene que estar localizado, un proceso que incluye traducir el inglés a otro idioma.
