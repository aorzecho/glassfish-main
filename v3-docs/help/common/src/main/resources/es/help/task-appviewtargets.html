Cómo ver los destinos de las aplicaciones

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones.

  2. Seleccione la aplicación cuyos destinos desee ver.

    El nombre de la aplicación puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte Module and Application Versions de Oracle GlassFish Server 3.1 Application Deployment Guide.

    Se abrirá la página de la aplicación.

  3. Seleccione la ficha Destino.

    Se abrirá la página Destinos de aplicación, y se mostrarán los destinos actualmente especificados para la aplicación.

  4. Para habilitar o inhabilitar una aplicación en un clúster o instancia de destino específicos, haga clic en la casilla de verificación que aparece junto al destino y seleccione Habilitar o Inhabilitar en el menú desplegable Más acciones.
  5. Para habilitar el equilibrado de carga de un destino en un clúster o instancia de destino específicos, haga clic en la casilla de verificación que aparece junto al destino y seleccione Habilitar equilibrado de carga o Inhabilitar equilibrado de carga en el menú desplegable Más acciones.
  6. Para asociar una aplicación con servidores virtuales específicos, seleccione Gestionar servidores virtuales para la instancia o clúster de destino deseados.

    Se abrirá la página Destinos de servidores virtuales.

    Para agregar servidores virtuales de destino, selecciónelos en la lista Destinos disponibles y utilice el botón Agregar para moverlos a la lista Destinos seleccionados.

    De forma predeterminada, el módulo o aplicación se asocia con el servidor virtual predeterminado, denominado server. Utilice el botón Agregar todos para agregar todos los destinos. Utilice los botones Eliminar y Eliminar todos para eliminar destinos de la lista Destinos seleccionados.

    Seleccione Guardar para volver a la página Destinos de aplicación.

  7. Para agregar o eliminar instancias o clústeres de destino para la aplicación, seleccione Administrar destinos.

    Se abrirá la página Administrar destinos.

    Para agregar destinos, selecciónelos en la lista Destinos disponibles y utilice el botón Agregar para moverlos a la lista Destinos seleccionados.

    Utilice el botón Agregar todos para agregar todos los destinos. Utilice los botones Eliminar y Eliminar todos para eliminar destinos de la lista Destinos seleccionados.

    Seleccione Guardar para volver a la página Destinos de aplicación.

  8. Haga clic en Guardar.

    La aplicación pasa a estar disponible en la lista de destinos modificada.
