Ver los Destinos de Aplicaciones

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones.

  2. Seleccione la aplicación cuyos destinos desee ver.

    El nombre de la aplicación puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

    Se abrirá la página de la aplicación.

  3. Seleccione el separador Destino.

    Se abrirá la página Destinos de Aplicaciones y se mostrarán los destinos actualmente especificados para la aplicación.

  4. Para activar o desactivar una aplicación en un cluster o instancia de destino específicos, active la casilla de control que aparece junto al destino y seleccione Activar Desactivar en el menú desplegable Más Acciones.

  5. Para activar el equilibraio de carga de un destino en un cluster o instancia de destino específicos, active la casilla de control que aparece junto al destino y seleccione Activación de Equilibrador de Carga o Desactivación de Equilibrador de Carga en el menú desplegable Más Acciones.

  6. Para asociar una aplicación a servidores virtuales específicos, seleccione Gestionar Servidores Virtuales para la instancia o cluster de destino deseados.

    Se abrirá la página Destinos de Servidor Virtuales.

    Para agregar servidores virtuales de destino, selecciónelos en la lista Destinos Disponibles y utilice el botón Agregar para moverlos a la lista Destinos Seleccionados.

    Por defecto, el módulo o aplicación se asocia al servidor virtual por defecto, denominado server. Utilice el botón Agregar Todos para agregar todos los destinos. Utilice los botones Eliminar y Eliminar Todos para eliminar destinos de la lista Destinos Seleccionados.

    Seleccione Guardar para volver a la página Destinos de Aplicaciones.

  7. Para agregar o suprimir instancias o clusters de destino para la aplicación, seleccione Gestionar Destinos.

    Se abrirá la página Gestionar Destinos.

    Para agregar destinos, selecciónelos en la lista Destinos Disponibles y utilice el botón Agregar para moverlos a la lista Destinos Seleccionados.

    Utilice el botón Agregar Todos para agregar todos los destinos. Utilice los botones Eliminar y Eliminar Todos para eliminar destinos de la lista Destinos Seleccionados.

    Seleccione Guardar para volver a la página Destinos de Aplicaciones.

  8. Haga clic en Guardar.

    La aplicación pasa a estar disponible en la lista de destinos modificada.
