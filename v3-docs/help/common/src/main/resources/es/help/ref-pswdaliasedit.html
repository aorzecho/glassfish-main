Editar alias de contraseña

Utilice la página Editar alias de contraseña para cambiar la contraseña asociada a un alias existente.

La página Editar alias de contraseña contiene las siguientes opciones:

Nombre de alias

Nombre del alias. Esta opción no se puede editar.

Contraseña

La contraseña que el alias almacena en un formato cifrado.

Confirmar contraseña

La contraseña que el alias almacena en un formato cifrado. Los valores Contraseña y Confirmar contraseña se comparan entre sí para garantizar que la contraseña se ha escrito sin errores.
