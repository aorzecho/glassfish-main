Configuración de analizador JVM

Una herramienta de analizador genera datos que se utilizan para analizar el rendimiento e identificar cuellos de botella potenciales.

Especifique la configuración del analizador. La información que especifica depende del producto de analizador que esté utilizando. Para obtener ejemplos e instrucciones, consulte el Capítulo 3, Debugging Applications de Oracle GlassFish Server 3.1 Application Development Guide.
