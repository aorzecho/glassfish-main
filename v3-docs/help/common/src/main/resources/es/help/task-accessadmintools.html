Otorgar Acceso a las Herramientas de Administración

Sólo los usuarios del grupo asadmin pueden acceder a la Consola de Administración y a la utilidad de línea de comandos asadmin. Para otorgar acceso a los usuarios a estas herramientas de administración, agregue los usuarios al grupo asadmin de admin-realm.

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee configurar los privilegios de acceso de administración.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

  4. Seleccione el nodo admin-realm.

    Se abrirá la página Editar Dominio.

  5. En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

    Se abrirá la página Usuarios de Administración.

    Inicialmente tras la instalación, el nombre de usuario y la contraseña del administrador introducidos durante la instalación aparecen en un archivo llamado admin-keyfile. Por defecto este usuario pertenece al grupo asadmin, que da derecho a modificar Servidor GlassFish. Asigne usuarios a este grupo únicamente si les quiere otorgar privilegios de administrador para la instancia o cluster de Servidor GlassFish seleccionada.

    Si agrega usuarios al dominio admin-realm, pero los asigna a un grupo diferente de asadmin, la información de los usuarios permanecerá escrita en el archivo admin-keyfile, pero el usuario no tendrá acceso a las herramientas administrativas ni a las aplicaciones del dominio file.

  6. En la página Usuarios de Administración, haga clic en Nuevo para agregar un nuevo usuario al dominio admin-realm.

    Se abrirá la página Nuevo Usuario de Dominio de Archivo.

  7. En el campo ID de Usuario, introduzca el nombre del nuevo usuario.

  8. En el campo Lista de Grupos, introduzca los nombres de los grupos a los que pertenecerá el usuario.

    Para autorizar a un usuario la realización de cambios a Servidor GlassFish, incluya el grupo asadmin en la lista de grupos.

  9. En los campos Contraseña Nueva y Confirmar Contraseña Nueva, escriba la contraseña del nuevo usuario.

  10. Haga clic en Aceptar para agregar este usuario al dominio admin-realm o haga clic en Cancelar para salir sin guardar.
