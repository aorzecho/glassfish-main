Cómo otorgar acceso a las herramientas de administración.

Sólo los usuarios del grupo asadmin pueden acceder a Consola de administración y a la utilidad de línea de comandos asadmin. Para otorgar acceso a los usuarios de estas herramientas de administración, agregue los usuarios al grupo asadmin de admin-realm.

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para el que desee configurar los privilegios de acceso de administración.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.
  4. Seleccione el nodo admin-realm.

    Aparecerá la página Editar dominio.

  5. En la página Editar dominio, haga clic en el botón Administrar usuarios.

    Se abre la página Usuarios de administración.

    Inicialmente tras la instalación el nombre y usuario del administrador introducidos durante la instalación aparecen en un archivo llamado admin-keyfile. Por defecto este usuario pertenece al grupo asadmin, que da derecho a modificar GlassFish Server. Asigne usuarios a este grupo únicamente si les quiere otorgar privilegios de administrador para la instancia o clúster de GlassFish Server seleccionada.

    Si agrega usuarios al dominio admin-realm, pero los asigna a un grupo diferente de asadmin, la información de los usuarios permanecerá escrita en el archivo admin-keyfile, pero el usuario no tendrá acceso a las herramientas administrativas ni a las aplicaciones del dominio file.

  6. En la página Usuarios de administración, haga clic en Nuevo para añadir un nuevo usuario al dominio admin-realm.

    Se abrirá la página Nuevo usuario de dominio file.

  7. En el campo Id. de usuario, introduzca el nombre del nuevo usuario.
  8. En el campo Lista de grupos, introduzca los nombres de los grupos a los que pertenecerá el usuario.

    Para autorizar a un usuario la realización de cambios a GlassFish Server, incluya el grupo asadmin en la Lista de grupos.

  9. En los campos Nueva contraseña y Confirmar nueva contraseña, escriba la contraseña del nuevo usuario.
  10. Haga clic en Aceptar para agregar este usuario al dominio admin-realm o haga clic en Cancelar para salir sin guardar.
