Cómo crear un módulo de auditoría

La auditoría es el modo en que Servidor GlassFish registra ciertos sucesos, como errores o violaciones de seguridad. Servidor GlassFish registra todos los sucesos de autenticación en archivos de registro. Un registro de acceso completo ofrece un rastro secuencial de los sucesos de acceso de Servidor GlassFish.

Servidor GlassFish ofrece un sencillo módulo de auditoría predeterminado; para obtener más información, consulte Cómo utilizar el módulo de auditoría predeterminado.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, expanda el nodo Seguridad.
  3. En el nodo Seguridad, seleccione el nodo Módulos de auditoría.

    Se abrirá la página Módulos de auditoría.

  4. En la página Módulos de auditoría haga clic en Nuevo.

    Se abrirá la página Nuevo módulo de auditoría.

  5. En la página Nuevo módulo de auditoría, en el campo Nombre, escriba el nombre del módulo de auditoría.
  6. En el campo Nombre de clase, escriba el nombre completo de la clase que implementa este módulo.

    El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.

  7. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Para añadir una propiedad, haga clic en el botón Agregar propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.

    Para ver una descripción de las propiedades adicionales disponibles para los módulos de auditoría, consulte Propiedades específicas de los módulos de auditoría.

  8. Haga clic en Aceptar.
