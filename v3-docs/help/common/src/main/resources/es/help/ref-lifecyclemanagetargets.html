Administrar destinos

Utilice la página Administrar destinos de recursos para cambiar los clústeres de destino y las instancias de servidor independientes en los que se puede habilitar un módulo de ciclo de vida. El módulo de ciclo de vida solo se puede habilitar en destinos que aparezcan en la columna Destinos seleccionados.

La página Administrar destinos de recursos contiene la siguiente información.

Destinos disponibles

Los clústeres e instancias en los que no se puede habilitar el módulo de ciclo de vida.

Destinos seleccionados

Los clústeres e instancias en los que se puede habilitar el módulo de ciclo de vida.

Agregar

Botón para mover el destino seleccionado de Destinos disponibles a Destinos seleccionados.

Añadir todos

Botón para mover todos los Destinos disponibles a Destinos seleccionados.

Borrar

Botón para mover el destino seleccionado de Destinos seleccionados a Destinos disponibles.

Eliminar todos

Botón para mover todos los Destinos seleccionados a Destinos disponibles.
