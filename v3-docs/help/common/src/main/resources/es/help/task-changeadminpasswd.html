Cambio de la contraseña de un usuario en el dominio de administrador

Para cambiar la contraseña de la cuenta de administrador actual, consulte Cambio de contraseña del administrador. Si admin-realm tiene más de un usuario, puede cambiar también la contraseña de los demás usuarios.

El cifrado de la contraseña del administrador mejora la seguridad de su aplicación. Consulte el Capítulo 1, Administering System Security de Oracle GlassFish Server 3.1 Security Guide, para obtener más información sobre procedimientos de seguridad.

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para el que desee modificar una contraseña de administración.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.
  4. Seleccione el nodo admin-realm.

    Aparecerá la página Editar dominio.

  5. En la página Editar dominio, haga clic en el botón Administrar usuarios.

    Se abre la página Usuarios de administración.

  6. En la página Usuarios de administración, haga clic en el usuario cuya contraseña desee añadir o cambiar.

    Se abrirá la página Edición de usuario del dominio file.

  7. En los campos Nueva contraseña y Confirmar nueva contraseña, escriba la nueva contraseña.
  8. Haga clic en Guardar.
