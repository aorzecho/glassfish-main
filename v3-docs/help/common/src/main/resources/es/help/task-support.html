Cómo ver la página de noticias de GlassFish para GlassFish Server

La página Noticias de GlassFish proporciona una recopilación de vínculos de información adicional sobre Oracle GlassFish Server, como, por ejemplo, hojas informativas, artículos, consejos técnicos, foros de discusión y blogs relevantes.

  1. En el árbol de navegación, seleccione el nodo Tareas comunes.

    Se abrirá la página Tareas comunes.

  2. En Noticias de GlassFish, haga clic en el vínculo Noticias de GlassFish.

    Se abrirá la página web Noticias de GlassFish en una ficha o ventana del explorador independiente.
