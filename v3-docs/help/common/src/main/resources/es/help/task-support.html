Ver la Página Noticias de GlassFish de Servidor GlassFish

La página Noticias de GlassFish proporciona una recopilación de enlaces a información adicional sobre Oracle GlassFish Server, como, por ejemplo, hojas informativas, artículos, consejos técnicos, foros de discusión y blogs relevantes.

  1. En el árbol de navegación, seleccione el nodo Tareas Comunes.

    Se abrirá la página Tareas Comunes.

  2. En Noticias de GlassFish, haga clic en el enlace Noticias de GlassFish.

    Se abrirá la página Noticias de GlassFish en una ficha o ventana del explorador independiente.
