Eliminación de la configuración de seguridad de mensajes

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. Expanda el nodo Seguridad.
  3. Seleccione el nodo Seguridad de mensajes

    Se abrirá la página Configuraciones de seguridad de mensajes.

  4. En la página Configuraciones de seguridad de mensajes, seleccione la casilla que se encuentra a la izquierda de la Configuración de seguridad de mensajes que está eliminando.

    Nota - No es posible crear o suprimir proveedores de seguridad de mensajes para la configuración predeterminada server-config. Los únicos proveedores de seguridad de mensajes compatibles con la configuración predeterminada server-config son SOAP y HttpServlet. Dado que estos dos proveedores de mensajes se crean de forma predeterminada al instalar Servidor GlassFish, el botón Suprimir no aparece en la página Configuraciones de seguridad de mensajes para la configuración predeterminada server-config.

  5. Haga clic en Eliminar.
