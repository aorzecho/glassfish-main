Suprimir una Configuración de Seguridad de Mensajes

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Amplíe el nodo Seguridad.

  3. Seleccione el nodo Seguridad de Mensajes.

    Se abrirá la página Configuraciones de Seguridad de Mensajes.

  4. En la página Configuraciones de Seguridad de Mensajes, active la casilla de control que se encuentra a la izquierda de la configuración de seguridad de mensajes que está suprimiendo.

    Nota:

    No es posible crear o suprimir proveedores de seguridad de mensajes para la configuración por defecto server-config. Los únicos proveedores de seguridad de mensajes soportados con la configuración por defecto server-config son SOAP y HttpServlet. Dado que estos dos proveedores de mensajes se crean por defecto al instalar Servidor GlassFish, el botón Suprimir no aparece en la página Configuraciones de Seguridad de Mensajes para la configuración por defecto server-config.

  5. Haga clic en Suprimir.
