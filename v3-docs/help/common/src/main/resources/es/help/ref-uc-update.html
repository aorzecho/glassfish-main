Actualizaciones disponibles

La página Actualizaciones disponibles ofrece información sobre las últimas versiones de los componentes de la imagen de instalación que están instalados.

Para instalar las actualizaciones disponibles, primero debe detener el dominio de GlassFish Server. A continuación, utilice uno de estos comandos:

  • glassfishv3/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • glassfishv3/bin/pkg, una versión de línea de comandos de la Herramienta de actualización

Para obtener información sobre el uso del comando pkg, consulte el Capítulo 11, Extending and Updating GlassFish Server de Oracle GlassFish Server 3.1 Administration Guide. La utilidad Herramienta de actualización tiene su propia ayuda en línea.

Reinicie el servidor después de que haya acabado de instalar las actualizaciones.

La página Actualizaciones disponibles contiene las siguientes opciones.

Ruta de instalación

Ruta del sistema de archivos de la raíz o directorio superior de la imagen de instalación.

Actualizaciones disponibles

Para cada componente para el que haya disponible una versión más reciente, en esta tabla se proporciona la siguiente información.

Componente

Nombre del componente.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas de sistemas y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión instalada

El número de versión del paquete en el que se proporciona el componente actual.

Nueva versión

El número de versión de la actualización disponible.

Cambiar

Diferencia de tamaño entre la versión instalada y la actualización disponible.

Origen

El nombre del Depósito de red del que se obtendría el componente.
