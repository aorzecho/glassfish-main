Cómo configurar funciones de GlassFish Server para que las utilicen los proveedores de seguridad de mensajes

  1. Si utiliza un token de nombre de usuario, configure una base de datos de usuarios en caso que sea necesario.

    Si utiliza un token de nombre de usuario/contraseña, primero tiene que configurar un dominio apropiado y una base de datos de usuarios apropiada para el dominio.

    La configuración de una base de datos de usuario se trata en Cómo editar un dominio.

  2. Administre certificados y claves privadas en caso necesario.
Pasos siguientes

Una vez ha configurado las funciones de GlassFish Server para que las utilicen los proveedores de seguridad de mensajes, los proveedores instalados con GlassFish Server pueden habilitarse tal como se describe en Edición de la configuración de seguridad de mensajes..
