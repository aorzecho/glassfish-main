Contrato de licencia

La página Contrato de licencia muestra los contratos de licencia de todos los componentes seleccionados para instalarse o actualizarse. El contrato de licencia de un componente lo proporciona el autor del componente con el mismo. Si existen varios componentes en el Depósito de red, deben publicarse con licencias diferentes, por ejemplo:

  • Licencia de desarrollo y distribución común (CDDL) (http://www.sun.com/cddl/)

  • Licencia pública general GNU (GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Licencia Apache (http://www.apache.org/licenses/)

  • Licencia de biblioteca wxWindows (http://www.opensource.org/licenses/wxwindows.php)

La página Seguridad contiene las siguientes opciones.

Aceptar

Este botón se utiliza para aceptar los términos de las licencias de todos los componentes seleccionados y seguir con la instalación o actualización.

Rechazar

Este botón se utiliza para rechazar los términos de las licencias de todos los componentes seleccionados y le devuelve a la página en la que había seleccionado los componentes.
