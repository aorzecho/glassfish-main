Edición del conector JMX

Utilice el servicio de administración para configurar un conector JMX remoto compatible con JSR 160.

El conector JMX remoto compatible con JSR 169 procesa la comunicación entre clientes JMX, como JConsole y el servicio JMX con fines de administración del dominio.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Servicio de administración.

    Se abrirá la página Editar conector JMX.

  3. Seleccione la casilla de verificación Seguridad habilitada para que la comunicación JMX esté cifrada.

    Esta opción se encuentra deshabilitada de forma predeterminada.

  4. En el campo Dirección, introduzca la dirección IP del servicio de asignación de nombres donde se encuentra registrado el código auxiliar del servidor del conector JMX.

    Éste no es el puerto del socket del servidor que posibilita la comunicación JMX. Ésta es la dirección de la interfaz de red donde se ha iniciado el registro RMI. Si su sistema dispone de varias interfaces de red, modifique este valor para que sólo se seleccione una interfaz determinada. El valor predeterminado es 0.0.0.0.

  5. En el campo Puerto, introduzca el Número de puerto en el que el servicio de asignación de nombres (registro RMI) escucha las conexiones del cliente RMI.

    La única función de este servicio de asignación de nombres es descargar los códigos auxiliares de RMI. Si el puerto predeterminado está ocupado, se utilizará otro puerto que esté libre. Los valores válidos se encuentran entre 1 y 65535. En sistemas UNIX, la creación de sockets de escucha para puertos 1-1024 requiere privilegios de superusuario. El valor predeterminado es 8686.

  6. En el campo Nombre de dominio, escriba el nombre que representa el dominio administrativo especial.

    Este dominio administrará toda la autenticación. El valor predeterminado es admin-realm.

  7. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Servidor GlassFish especifica varias propiedades de manera predeterminada.

    • Para añadir una propiedad, haga clic en el botón Agregar propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.
    • Para modificar una propiedad, edite el campo Valor de la propiedad.
    • Para eliminar una propiedad, marque la casilla de verificación situada a la izquierda del campo Nombre de la propiedad que quiera eliminar y, a continuación, haga clic en el botón Eliminar propiedades.
  8. Haga clic en Guardar.
  9. Si marca la casilla de verificación Seguridad habilitada, siga los pasos descritos en Edición de configuración SSL para el conector JMX.
