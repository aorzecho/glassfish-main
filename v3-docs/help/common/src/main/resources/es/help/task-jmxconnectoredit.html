Editar el Conector JMX

Utilice el servicio de administración para configurar un conector JMX remoto compatible con JSR 160.

El conector JMX remoto compatible con JSR 160 maneja la comunicación entre clientes JMX, como JConsole y el servicio JMX con fines de gestión del dominio.

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, seleccione el nodo Servicio de Administración.

    Se abrirá la página Editar Conector JMX.

  3. Active la casilla de control Seguridad Activada para que la comunicación JMX esté cifrada.

    Esta opción está desactivada por defecto.

  4. En el campo Dirección, introduzca la dirección IP del servicio de asignación de nombres donde se encuentra registrado el stub del servidor del conector JMX.

    Éste no es el puerto del socket del servidor que posibilita la comunicación JMX. Ésta es la dirección de la interfaz de red donde se ha iniciado el registro RMI. Si su sistema dispone de varias interfaces de red, modifique este valor para que sólo se seleccione una interfaz determinada. El valor por defecto es 0.0.0.0.

  5. En el campo Puerto, introduzca el número de puerto en el que el servicio de asignación de nombres (registro RMI) recibe las conexiones del cliente RMI.

    La única función de este servicio de asignación de nombres es descargar los stubs de RMI. Si el puerto por defecto está ocupado, se utilizará otro puerto que esté libre. Los valores válidos se encuentran entre 1 y 65535. En sistemas UNIX, la creación de sockets de escucha para los puertos 1-1024 requiere privilegios de superusuario. El valor por defecto es 8686.

  6. En el campo Nombre de Dominio, escriba el nombre que representa el dominio administrativo especial.

    Este dominio maneja toda la autenticación. El valor por defecto es admin-realm.

  7. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    Servidor GlassFish especifica varias propiedades por defecto.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  8. Haga clic en Guardar.

  9. Si activa la casilla de control Seguridad Activada, siga los pasos descritos en Editar Configuración de SSL para el Conector JMX.
