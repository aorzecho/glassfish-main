Editar aplicación

Utilice la página Editar aplicación para modificar una aplicación existente.

La página Editar aplicación contiene algunas o todas las opciones siguientes, en función del tipo de aplicación.

Nombre

Campo de solo lectura que muestra el nombre de la aplicación que está editando.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte Module and Application Versions de Oracle GlassFish Server 3.1 Application Deployment Guide.

Estado

Si se selecciona la casilla de verificación Habilitado, se habilita la aplicación. Esta opción está seleccionada por defecto.

Raíz de contexto

Para una aplicación web, se especifica la ruta de la aplicación. En la URL de la aplicación web, la raíz de contexto sigue inmediatamente al número de puerto (http://host:puerto/ raíz de contexto/...). La raíz de contexto debe comenzar por una barra diagonal, por ejemplo, /hello.

Disponibilidad

Si se selecciona la casilla de verificación Habilitado, se habilita la alta disponibilidad para la aplicación de puntos de control de SFSB y posiblemente para la pasivación. Si se configura como False (el valor predeterminado), se inhabilita el guardado de sesiones web y puntos de control de SFSB para la aplicación, aplicación web o módulo EJB especificado. Si se define como true, la aplicación o módulo especificados se habilita para alta disponibilidad. Establezca esta opción como true sólo si se ha configurado y habilitado la alta disponibilidad en niveles superiores, como, por ejemplo, los niveles de servidor y contenedor.

Esta opción aparece si existen clústeres o instancias independientes del servidor aparte de la instancia de servidor predeterminada (server).

Servidores virtuales

Servidores virtuales asociados a esta aplicación.

La opción Servidores virtuales aparece si solo existe la instancia de servidor predeterminada, server. Si existen clústeres u otras instancias de servidor independientes, puede seleccionar los servidores virtuales después de la implementación. Vaya a la página Editar aplicación, seleccione la ficha Destino y seleccione Administrar servidores virtuales para el destino que desee.

Java Web Start

(Para algunos tipos de aplicaciones) Si se selecciona la casilla de verificación Habilitado, la aplicación empleará el software Java Web Start. Java Web Start proporciona una forma independiente de los navegadores de implementar aplicaciones Java para que se ejecuten en una máquina virtual de Java específica.

Descripción

Breve descripción de la aplicación.

Ubicación

La ubicación de la aplicación implementada. Éste es un campo de sólo lectura.

Bibliotecas

Los archivos JAR de biblioteca que necesitará la aplicación. Éste es un campo de sólo lectura.

Módulos y componentes

Tabla en la que figuran los nombres de los módulos de la aplicación y sus motores. Para un punto final de servicio web, puede seleccionar Consultar punto final para visualizar la página Información de punto final de servicio web. Para un cliente de aplicaciones, puede seleccionar Iniciar para acceder a la página Iniciar cliente de aplicaciones.
