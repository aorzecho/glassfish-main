Configuraciones

Utilice la página Configuraciones para gestionar configuraciones y ver las instancias de servidor independientes y los clústeres que utilizan cada configuración.

La configuración default-config viene predeterminada por Servidor GlassFish. La configuración server-config se crea automáticamente para el servidor de administración de dominios (DAS) al crear el dominio. Para obtener más información, consulte About Named Configurations de Oracle GlassFish Server 3.1 High Availability Administration Guide.

Para cada configuración, se proporciona la siguiente información.

Nombre

Nombre de la configuración.

Instancia

Lista de instancias independientes y clústeres que utilizan la configuración.

La tabla Configuraciones también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración.

Borrar

Botón para eliminar una o varias configuraciones seleccionadas.
