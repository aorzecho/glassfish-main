Configuraciones

Utilice la página Configuraciones para gestionar configuraciones y ver las instancias de servidor independientes y los clusters que usan las configuraciones.

La configuración default-config es predefinida por Servidor GlassFish. La configuración server-config se crea automáticamente para el servidor de administración de dominio (DAS) cuando se crea el dominio. Para obtener más información consulte la sección acerca de las configuraciones con nombre en Oracle GlassFish Server High Availability Administration Guide.

Para cada configuración, se proporciona la siguiente información.

Nombre

Nombre de la configuración.

Instancia

Lista de instancias independientes y clusters que utilizan la configuración.

La tabla Configuraciones también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración.

Suprimir

Botón para suprimir una o varias configuraciones seleccionadas.
