Cómo visualizar archivos de registro para el DAS

  1. En el árbol de navegación, seleccione el nodo del servidor (Admin Server).

    Aparecerá la página de Información general.

  2. En la página de Información general, haga clic en el botón Ver archivos de registro.

    Se abre una ventana con el Visualizador de registros.
