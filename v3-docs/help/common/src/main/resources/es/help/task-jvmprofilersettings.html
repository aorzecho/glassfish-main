Definir la Configuración de Analizador de Perfiles de JVM

Una herramienta de analizador de perfiles genera datos que se utilizan para analizar el rendimiento e identificar cuellos de botella potenciales.

  1. En el árbol de navegación, seleccione el nodo Configuración.

  2. Select the configuration that you are changing.

  3. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración General de JVM.

  4. En la página Configuración General de JVM, haga clic en el separador Analizador de Perfiles.

    Se abrirá la página Configuración de Analizador de Perfiles de JVM.

  5. Especifique la configuración del analizador de perfiles.

    La información que especifica depende del producto de analizador de perfiles que esté utilizando. Para obtener ejemplos e instrucciones, consulte el Capítulo 3, sobre la depuración de aplicaciones, en Guía de Desarrollo de la Aplicación de Oracle GlassFish Server.

  6. Haga clic en Guardar.

  7. Reinicie el servidor.
