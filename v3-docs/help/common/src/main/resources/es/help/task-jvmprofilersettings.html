Para ajustar la configuración del analizador de JVM

Una herramienta de analizador genera datos que se utilizan para analizar el rendimiento e identificar cuellos de botella potenciales.

  1. En el árbol de navegación, seleccione el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración general de JVM.

  3. En la página Configuración general de JVM, haga clic en la ficha Profiler.

    Se abrirá la página Valores de JVM Profiler.

  4. Especifique la configuración del analizador.

    La información que especifica depende del producto de analizador que esté utilizando. Para obtener ejemplos e instrucciones, consulte el Capítulo 3, Debugging Applications de Oracle GlassFish Server 3.1 Application Development Guide.

  5. Haga clic en Guardar.
  6. Reinicie el servidor.
