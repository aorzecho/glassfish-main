Destinos de Servidor Virtuales

Utilice la página Destinos de Servidor Virtuales para cambiar los servidores virtuales de destino a los que está asociada una aplicación o un módulo.

La pagina Destinos de Servidor Virtuales contiene la siguiente información.

Destinos Disponibles

Servidores virtuales a los que no está asociada una aplicación o un módulo.

Destinos Seleccionados

Servidores virtuales a los que está asociada una aplicación o un módulo.

Agregar

Botón para mover el destino seleccionado de Destinos Disponibles a Destinos Seleccionados.

Agregar Todos

Botón para mover todos los destinos disponibles a Destinos Seleccionados.

Eliminar

Botón para mover el destino seleccionado de Destinos Seleccionados a Destinos Disponibles.

Eliminar Todos

Botón para mover todos los destinos seleccionados a Destinos Disponibles.
