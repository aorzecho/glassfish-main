Destinos de servidor virtuales

Utilice la página Destinos de servidores virtuales para cambiar los destinos de servidores virtuales a los que está asociada una aplicación o un módulo.

La pagina Destinos de servidores virtuales contiene la siguiente información.

Destinos disponibles

Los servidores virtuales con los que no está asociada una aplicación o un módulo.

Destinos seleccionados

Los servidores virtuales con los que está asociada una aplicación o un módulo.

Agregar

Botón para mover el destino seleccionado de Destinos disponibles a Destinos seleccionados.

Añadir todos

Botón para mover todos los Destinos disponibles a Destinos seleccionados.

Borrar

Botón para mover el destino seleccionado de Destinos seleccionados a Destinos disponibles.

Eliminar todos

Botón para mover todos los Destinos seleccionados a Destinos disponibles.
