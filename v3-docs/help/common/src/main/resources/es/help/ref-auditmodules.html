Módulos de auditoría

Utilice la página Módulos de auditoría para configurar los módulos de auditoría.

La auditoría es el modo en que GlassFish Server registra ciertos sucesos, como errores o violaciones de seguridad. GlassFish Server registra todos los sucesos de autenticación en archivos de registro. Un registro de acceso completo ofrece un rastro secuencial de los sucesos de acceso de GlassFish Server.

En cada módulo de auditoría se proporciona la siguiente información.

Nombre

El nombre del módulo de auditoría.

Nombre de clase

El nombre completo de la clase que implementa este módulo. El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.

La tabla Módulos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo módulo de auditoría.

Borrar

Botón para eliminar uno o más módulos de auditoría seleccionados.
