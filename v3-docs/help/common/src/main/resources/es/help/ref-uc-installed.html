Componentes instalados

La página Componentes instalados ofrece información sobre los componentes de la imagen de instalación que están instalados.

Para desinstalar los componentes instalados en primer lugar detenga el dominio de Servidor GlassFish. A continuación, utilice uno de estos comandos:

  • glassfishv3/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • glassfishv3/bin/pkg, una versión de línea de comandos de la Herramienta de actualización

Para obtener información sobre el uso del comando pkg, consulte el Capítulo 11, Extending and Updating GlassFish Server de Oracle GlassFish Server 3.1 Administration Guide. La utilidad Herramienta de actualización tiene su propia ayuda en línea.

Reinicie el servidor después de que haya terminado de desinstalar los componentes.

La página Componentes instalados contiene las opciones siguientes.

Ruta de instalación

Ruta del sistema de archivos de la raíz o directorio superior de la imagen de instalación.

Installed Components

Para cada componente instalado, esta tabla proporciona la siguiente información.

Componente

Nombre del componente.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas de sistemas y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión

Número de versión del paquete en el que se proporciona el componente.

Tamaño instalado

Cantidad de espacio en disco que consume la instalación del componente.

Origen

El nombre del Depósito de red desde el que se obtuvo el componente.
