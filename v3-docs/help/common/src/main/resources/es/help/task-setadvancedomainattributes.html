Definir los Atributos de Dominio

  1. En el árbol de navegación, seleccione el nodo Dominio.

    Se abrirá la página Atributos de Dominio.

  2. En el campo Raíz de Aplicación, introduzca la ruta de acceso completa del directorio en el que residen las aplicaciones desplegadas.

    El valor por defecto es domain-dir/applications.

  3. En el campo Raíz de Log, introduzca la ruta de acceso completa del directorio en el que se guardan los archivos log del servidor.

    El valor por defecto es domain-dir/logs.

  4. Deje el campo Configuración Regional en blanco para utilizar la configuración regional por defecto del host.

    La configuración regional es un identificador que especifica una combinación concreta de idioma y región. Por ejemplo, la configuración regional para inglés de Estados Unidos es en_US y para japonés es ja_JP. Para utilizar una configuración regional que no sea del idioma inglés, Servidor GlassFish tiene que estar localizado, un proceso que incluye traducir el inglés a otro idioma.

  5. Haga clic en Guardar.

  6. Reinicie el servidor.
