Cómo configurar los atributos de dominio

  1. En el árbol de navegación, seleccione el nodo GlassFish Server.

    Aparecerá la página de Información general.

  2. En la página Información general, haga clic en la ficha Avanzado.

    Se abrirá la página Configuración de aplicaciones.

  3. En la página Configuración de aplicaciones, haga clic en la ficha Atributos de dominio.

    Se abrirá la página Atributos de dominio.

  4. En el campo Raíz de aplicación, introduzca la ruta completa del directorio en el que residen las aplicaciones implementadas.

    El valor predeterminado es domain-dir/applications .

  5. En el campo Raíz de registro, introduzca la ruta completa del directorio en el que se guardan los archivos de registro del servidor.

    El valor predeterminado es domain-dir/logs .

  6. Deje el campo Configuración regional en blanco para utilizar la configuración regional predeterminada del host.

    La configuración regional es un identificador que especifica una combinación concreta de idioma y región. Por ejemplo, la configuración regional para inglés de Estados Unidos es en_US y para japonés es ja_JP. Para utilizar una configuración regional que no sea del idioma inglés, GlassFish Server tiene que estar localizado, un proceso que incluye traducir el inglés a otro idioma.

  7. Haga clic en Guardar.
  8. Reinicie el servidor.
