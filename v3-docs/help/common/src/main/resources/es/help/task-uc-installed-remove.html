Ver Componentes Instalados

La página Componentes instalados ofrece información sobre los componentes de la imagen de instalación que están instalados.

La desinstalación de un componente incluye la eliminación de los elementos siguientes:

  • Todos los archivos y directorios introducidos con la instalación original del componente

  • Todos los archivos y directorios introducidos con las actualizaciones del componente tras la instalación original

  1. En el árbol de navegación, seleccione el nodo Herramienta de Actualización.

    Se abrirá la página Complementos Disponibles.

  2. En la página Complementos Disponibles, haga clic en el separador Componentes Instalados.

    Se abrirá la página Componentes Instalados.

Pasos Siguientes

Para desinstalar los componentes instalados en primer lugar pare el dominio de Servidor GlassFish. A continuación, utilice uno de estos comandos:

  • as-install-parent/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • as-install-parent/bin/pkg, una versión de línea de comandos de la herramienta de actualización

Para obtener información sobre el uso del comando pkg, consulte el Capítulo 11, Ampliación y actualización de GlassFish Server, en Oracle GlassFish Server Administration Guide. La utilidad Herramienta de Actualización tiene su propia ayuda en pantalla.

Reinicie el servidor después de que haya terminado de desinstalar los componentes.
