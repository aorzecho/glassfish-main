Para registrar Oracle GlassFish Server

Como usuario registrado de Oracle GlassFish Server, obtendrá las siguientes ventajas:

  • Información sobre parches y actualizaciones de corrección de errores

  • Vídeos de procedimientos en pantalla y tutoriales

  • Noticias y eventos

  • Ofertas de asistencia y formación

Para registrar su instalación de Oracle GlassFish Server, debe disponer de una Cuenta en línea de Sun. Si no tiene una cuenta Sun Online, GlassFish Server le permite crear una cuenta al registrar este producto.

Nota - Si crea una cuenta Sun Online al registrar GlassFish Server, se le pedirá sólo la información necesaria. Tras registrar GlassFish Server, puede obtener información adicional opcional actualizando su cuenta en línea de Sun.

Antes de empezar

Si el equipo en el que se está ejecutando GlassFish Server se conecta a Internet a través de un proxy, obtenga el nombre de host y el número de puerto del proxy. Si se está ejecutando un navegador web en el equipo en el que se está ejecutando GlassFish Server, podrá obtener esta información del navegador web. De lo contrario, póngase en contacto con el administrador del sistema para obtener más información.

  1. En el árbol de navegación, seleccione el nodo Registro.

    Se abrirá la página Registro de productos.

  2. En la página de registro del producto introduzca la información necesaria.
    • Si ya dispone de una cuenta Sun Online, introduzca la información necesaria siguiente:
      1. Seleccione la opción Ya tengo una cuenta en línea de Sun o una cuenta de Sun Developer Network (SDN).
      2. En el campo de Nombre de usuario, escriba el nombre de usuario que especificó al registrar su cuenta Sun Online.
      3. En el campo de Contraseña, escriba la contraseña que especificó al registrar su cuenta Sun Online.
    • Si no dispone de una cuenta Sun Online, introduzca la información necesaria siguiente:
      1. Seleccione la opción No tengo una cuenta de Sun Online.
      2. En el campo Dirección de correo electrónico introduzca su dirección de correo electrónico.

        Su dirección de correo electrónico será su nombre de usuario de cuenta Sun Online.

      3. En el campo Contraseña escriba la contraseña que utiliza para su cuenta Sun Online.

        La contraseña distingue entre mayúsculas y minúsculas y debe tener 6–10 caracteres.

      4. En el campo Vuelva a escribir su contraseña, vuélvala a escribir.
      5. En el campo Nombre escriba su nombre.
      6. En el campo Apellido escriba su apellido.
      7. En el campo Empresa, escriba el nombre de su empresa, si procede.
      8. De la lista desplegable seleccione el país o región donde vive.
  3. Si el equipo en el que se está ejecutando GlassFish Server se conecta a Internet a través de un proxy, obtenga el nombre de host y el número de puerto del proxy.

    Si el equipo en el que se está ejecutando GlassFish Server se conecta directamente a Internet, omita este paso.

    Un navegador web debe estar ejecutándose en el equipo en el que se está ejecutando GlassFish Server. En esta situación, puede obtener el nombre de host y el número de puerto del proxy del navegador web. De lo contrario, póngase en contacto con el administrador del sistema para obtener más información.

  4. Haga clic en Registrarse.

    Para confirmar que se ha completado el registro, se abrirá la página de asistencia técnica.
