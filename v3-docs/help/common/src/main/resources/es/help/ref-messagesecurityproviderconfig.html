Configuraciones del proveedor

Consulte la página Configuraciones del proveedor para configurar el proveedor de seguridad de mensajes en una capa de autenticación SOAP o HttpServlet.

De manera predeterminada, cuatro proveedores están configurados en la capa de autenticación SOAP.

Para cada configuración de proveedor, se proporciona la siguiente información.

ID de Proveedor

Nombre de la configuración del proveedor.

Tipo de proveedor

Tipo de proveedor, que puede ser cualquiera de los siguientes:

client

Especifica que el proveedor es el proveedor de autenticación de clientes.

server

Especifica que el proveedor es el proveedor de autenticación del servidor.

client-server

Especifica que el proveedor es tanto el proveedor de autenticación del servidor como del cliente.

Proveedor predeterminado

Si se configura en true, el proveedor será el proveedor predeterminado.

Nombre de clase

Nombre de la clase de implementación de Java del proveedor.

La tabla Configuraciones de proveedor también contiene las siguientes opciones.

Nueva

Botón para crear una nueva configuración de proveedor.

Borrar

Botón para eliminar una o más configuraciones de proveedor seleccionadas.
