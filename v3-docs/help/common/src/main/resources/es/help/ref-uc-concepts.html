Conceptos clave de Herramienta de Actualización

Herramienta de Actualización gestiona la instalación de componentes de software de Servidor GlassFish que se encuentran en un único directorio de sistema de archivos en su sistema.

La versión Consola de Administración de Herramienta de Actualización permite realizar las siguientes acciones:

  • Instalar complementos

  • Ver actualizaciones para los componentes instalados

  • Ver los componentes instalados

Para actualizar o eliminar los componentes instalados, deberá detener el dominio de Servidor GlassFish y, a continuación, utilizar uno de los siguientes comandos:

  • as-install-parent/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • as-install-parent/bin/pkg, una versión de línea de comandos de la herramienta de actualización

Los conceptos clave de Herramienta de Actualización se describen en las secciones que vienen a continuación.

Componente

Un componente es la unidad instalable que gestiona Herramienta de Actualización. Un componente consta de varios archivos instalados como una unidad en el sistema dentro de una imagen de instalación.

Imagen de instalación

Una imagen de instalación es el directorio que contiene los componentes del software, incluyendo el contenido del directorio. La imagen de instalación se agrega a Herramienta de Actualización como resultado de la descarga e instalación del software de Servidor GlassFish. Para habilitar la adición y actualización de componentes del software de Servidor GlassFish, cada imagen de instalación contiene información sobre los componentes de instalación.

Depósito de red

Un repositorio de red contiene los elementos siguientes:

  • Componentes que pueden agregarse a una imagen de instalación

  • Actualizaciones disponibles para los componentes existentes en una imagen de instalación

Una imagen de instalación se asocia con uno o más depósitos de red. Cada uno de ellos está representado por un nombre y una URL asociada. Cada imagen de instalación tiene un nombre de depósito único que representa tanto a la organización o al proyecto que gestiona el depósito, como a un ámbito de contenidos bien definido. Para obtener información sobre los repositorios de red preconfigurados para Servidor GlassFish, consulte Repositorios preconfigurados para GlassFish Server en Oracle GlassFish Server Administration Guide.

Componente de complementos

Un componente de complementos es un componente de la imagen de instalación que no se ha instalado todavía.
