Conceptos clave de Herramienta de actualización

Herramienta de actualización gestiona la instalación de componentes de software de GlassFish Server que se encuentran en un único directorio de sistema de archivos en su sistema.

La versión Consola de administración de Herramienta de actualización permite realizar las siguientes acciones:

  • Instalar complementos

  • Ver actualizaciones para los componentes instalados

  • Ver los componentes instalados

Para actualizar o eliminar los componentes instalados, deberá detener el dominio de GlassFish Server y, a continuación, utilizar uno de los siguientes comandos:

  • as-install-parent/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • as-install-parent/bin/pkg, una versión de línea de comandos de la herramienta de actualización

Los conceptos clave de Herramienta de actualización se describen en las secciones que vienen a continuación.

Componente

Un componente es la unidad instalable que gestiona Herramienta de actualización. Un componente consta de varios archivos instalados como una unidad en el sistema dentro de una imagen de instalación.

Imagen de instalación

Una imagen de instalación es el directorio que contiene los componentes del software, incluyendo el contenido del directorio. La imagen de instalación se añade a Herramienta de actualización como resultado de la descarga e instalación del software de GlassFish Server. Para habilitar la adición y actualización de componentes del software de GlassFish Server, cada imagen de instalación contiene información sobre los componentes de instalación.

Depósito de red

Un depósito de red contiene los elementos siguientes:

  • Componentes que pueden agregarse a una imagen de instalación

  • Actualizaciones disponibles para los componentes existentes en una imagen de instalación

Una imagen de instalación se asocia con uno o más depósitos de red. Cada uno de ellos está representado por un nombre y una URL asociada. Cada imagen de instalación tiene un nombre de depósito único que representa tanto a la organización o al proyecto que gestiona el depósito, como a un ámbito de contenidos bien definido. Para obtener información sobre los depósitos de red preconfigurados para GlassFish Server, consulte Preconfigured Repositories for GlassFish Server de Oracle GlassFish Server 3.1 Administration Guide.

Componente de complementos

Un componente de complementos es un componente de la imagen de instalación que no se ha instalado todavía.
