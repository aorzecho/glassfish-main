Editar un Módulo de Ciclo de Vida

  1. En el árbol de navegación, seleccione el nodo Módulos de Ciclo de Vida.

    Se abrirá la página Módulos de Ciclo de Vida.

  2. En la página Módulos de Ciclo de Vida, haga clic en el nombre del módulo de ciclo de vida que está editando.

    Se abrirá la página Editar Módulo de Ciclo de Vida.

  3. En el campo Nombre de Clase, escriba el nombre completo del archivo de clase del módulo de ciclo de vida.

    La clase debe implantar la interfaz com.sun.appserv.server.LifecycleListener.

  4. En el campo Classpath, introduzca la classpath para el módulo de ciclo de vida.

    La classpath especifica la ubicación del módulo de ciclo de vida. El valor por defecto es domain-dir/applications. Si el módulo ya se encuentra en este directorio (es decir, en la Classpath del servidor), este campo puede dejarse en blanco.

  5. En el campo Orden de Carga, introduzca un valor que especifique el orden en que este módulo de ciclo de vida se cargará al inicio.

    Los módulos con valores de orden de carga enteros inferiores se cargarán antes. El valor puede estar entre 101 y el MAXINT del sistema operativo. Los valores de 0 a 100 están reservados.

  6. En el campo Descripción, introduzca una descripción del módulo de ciclo de vida.

  7. Active la casilla de control Estado Activado para activar el módulo de ciclo de vida.

    Esta opción está activada por defecto.

  8. Active la casilla de control Ante Fallo de Carga: Evitar Inicio de Instancia para especificar que el servidor debe apagarse si se produce un fallo en la carga del módulo de ciclo de vida.

    Esta opción está desactivada por defecto.

  9. El separador Destinos aparece si existen clusters o instancias de servidor independientes aparte de la instancia de servidor por defecto (server). Seleccione el separador Destinos para ver los destinos en los que se ha activado el módulo de ciclo de vida. Consulte Ver los Destinos de Aplicaciones para obtener instrucciones sobre cómo activar o desactivar un módulo de ciclo de vida en uno o varios destinos.

  10. Haga clic en Guardar.
