Cómo instalar componentes de complementos

La instalación de un componente de complementos incluye las operaciones siguientes:

  • Transferir el contenido del componente y de los depósitos del componente desde un depósito de red

  • Instalar el componente dentro de la imagen de instalación de GlassFish Server en su sistema

Si el componente requiere otros componentes que no estén instalados, estos componentes también se instalarán.

  1. En el árbol de navegación, seleccione el nodo Herramienta de actualización.

    Se abrirá la página Complementos disponibles.

  2. En la página Complementos disponibles, en la tabla que muestra los componentes disponibles, seleccione los que desee instalar.

    Si no hay componentes de complementos disponibles, la tabla aparecerá vacía.

    • Para seleccionar un componente individual, marque la casilla de verificación situada al lado del nombre del componente.
    • Para seleccionar todos los componentes, haga clic en el icono Seleccionar todo situado en el encabezado de la tabla.
    • Para anular la selección de todos los componentes, haga clic en el icono Anular toda la selección situado en el encabezado de la tabla.
  3. Haga clic en "Instalar".

    Aparecerá un cuadro de diálogo para que confirme que los componentes seleccionados y los componentes requeridos por estos deben instalarse.

  4. En el cuadro de diálogo, haga clic en Aceptar.

    También tendrá que confirmar que acepta los contratos de licencia de los componentes que va a instalar.

  5. Acepte los acuerdos de licencia de los componentes que se van a instalar.

    GlassFish Server confirma que la instalación ha finalizado. Los componentes desaparecerán de la tabla de componentes de complementos y se agregarán a la tabla de componentes instalados de la página Componentes instalados.

  6. Reinicie el servidor.

    Para obtener instrucciones detalladas, consulte la documentación siguiente:

    • To Stop a Domain de Oracle GlassFish Server 3.1 Administration Guide

    • To Start a Domain de Oracle GlassFish Server 3.1 Administration Guide
