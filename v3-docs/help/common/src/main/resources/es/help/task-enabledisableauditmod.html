Cómo habilitar o deshabilitar el registro de auditoría

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  3. En la página Seguridad, seleccione la casilla de verificación Registro de auditoría habilitado para habilitar el registro de auditoría.

    Para activar el registro de auditoría, debe además establecer la propiedad auditOn en true para que el módulo de auditoría esté activo.

  4. Haga clic en Guardar.
