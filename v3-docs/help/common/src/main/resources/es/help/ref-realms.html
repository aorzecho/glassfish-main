Dominios

Servidor GlassFish viene preconfigurado con tres dominios: file, certificate y admin-realm. También se pueden crear dominios ldap, JDBC, solaris y personalizados. Generalmente, tendrá un dominio de cada tipo en un servidor, pero en Servidor GlassFish hay dos dominios de archivo: file y admin-realm. Hay dos dominios del mismo tipo utilizados para dos propósitos diferentes. También se puede tener una base de datos de certificado distinta para cada servidor virtual en el sistema.

Utilice la página Dominios para configurar los dominios.

Para cada dominio, se proporciona la siguiente información.

Nombre

Nombre del dominio.

Nombre de clase

El nombre de clase del dominio.

La tabla Dominios también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo dominio.

Borrar

Botón para eliminar uno o varios dominios seleccionados.
