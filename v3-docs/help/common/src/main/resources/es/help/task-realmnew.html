Crear un Dominio

Servidor GlassFish viene preconfigurado con tres dominios: file, certificate y admin-realm. También se pueden crear dominios ldap, JDBC, solaris, PAM y dominios personalizados. Normalmente se tiene un dominio de cada tipo en un servidor, pero en Servidor GlassFish existen dos dominios de archivo: file y admin-realm. Se trata de dos dominios del mismo tipo utilizados para dos fines diferentes. También se puede tener una base de datos de certificado distinta para cada servidor virtual en el sistema.

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee configurar los dominios.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

    Se abrirá la página Dominios.

  4. En la página Dominios haga clic en Nuevo.

    Se abrirá la página Nuevo Dominio.

  5. En el campo Nombre, introduzca un nombre para el dominio.

  6. Para especificar el nombre de clase del dominio, realice una de las siguientes acciones:

    • En la lista desplegable Nombre de Clase, seleccione el nombre de clase para el dominio creado.

      Las opciones válidas se muestran en la siguiente tabla.

      Nombre de Dominio Nombre de Clase
      file com.sun.enterprise.security.auth.realm.file.FileRealm
      certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • Active el botón de radio inferior y escriba el nombre de clase del dominio personalizado creado.

  7. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    Las propiedades válidas dependen del tipo de dominio.

    • Propiedades específicas de la clase FileRealm

    • Propiedades específicas de la clase CertificateRealm

    • Propiedades específicas de la clase JDBCRealm

    • Propiedades específicas de la clase LDAPRealm

    • Propiedades específicas de la clase SolarisRealm

    • Propiedades específicas de la clase PAMRealm

  8. Haga clic en Aceptar.
