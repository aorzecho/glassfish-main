Para añadir un usuario al dominio de administración

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para la que desee configurar la Configuración de registro.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.
  4. Seleccione el nodo admin-realm.

    Aparecerá la página Editar dominio.

  5. En la página Editar dominio, haga clic en el botón Administrar usuarios.

    Se abre la página Usuarios de administración.

  6. En la página Usuarios de administración, haga clic en Nuevo.

    Se abrirá la página Nuevo usuario de dominio file.

  7. En el campo Id. de usuario, introduzca el nombre del usuario.
  8. En el campo Lista de grupos, introduzca la lista de los grupos a los que pertenece el usuario.

    Utilice comas para separar los nombres de grupo. Estos grupos no hace falta definirlos en otro lugar.

  9. En el campo Contraseña, introduzca una contraseña exclusiva.
  10. En el campo Confirmar nueva contraseña, escriba la contraseña de usuarios de nuevo.
  11. Haga clic en Aceptar.
