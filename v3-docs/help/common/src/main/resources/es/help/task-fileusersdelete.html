Cómo eliminar a un usuario del dominio de archivos

  1. En el nodo Configuraciones, amplíe la configuración de instancia o clúster del servidor para la que desee configurar la Configuración de registro.
  2. Expanda el nodo Seguridad.
  3. Expanda el nodo Dominios.
  4. Seleccione el nodo file.

    Aparecerá la página Editar dominio.

  5. En la página Editar dominio, haga clic en el botón Administrar usuarios.

    Se abrirá la página Usuarios de file.

  6. En la página Usuarios de archivo, marque la casilla de verificación situada a la izquierda del nombre del usuario que desee eliminar.
  7. Haga clic en Eliminar.

    Aparecerá un cuadro de diálogo para que confirme que debe eliminarse el usuario.

  8. En el cuadro de diálogo, haga clic en Aceptar.
