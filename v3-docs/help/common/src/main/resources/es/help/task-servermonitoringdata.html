Ver Datos de Supervisión de Servidor

Antes de Empezar

Antes de poder ver datos de supervisión, es necesario que configure la supervisión. Consulte Configurar el Servicio de Supervisión para obtener detalles.

  1. En el árbol de navegación, seleccione el nodo Datos de Supervisión.

    Se abrirá la página Supervisión.

  2. En la columna Ver Datos de Supervisión, haga clic en el enlace Servidor que corresponda a la instancia de servidor que vaya a supervisar.

    Se abrirá la página Supervisión de Servidor.

  3. En la lista desplegable Ver, seleccione el componente para el cual desea ver los datos de supervisión.

  4. Haga clic en el botón Refrescar para actualizar los datos de supervisión que aparecen.

Pasos alternativos:

  1. En el árbol de navegación, seleccione el nodo del servidor (servidor de administración).

    Se abrirá la página Información General.

  2. En la página Información General, haga clic en el separador Supervisar.

    Se abrirá la página Supervisión de Aplicación.

  3. Haga clic en el separador Servidor.

    Se abrirá la página Supervisión de Servidor.

  4. En la lista desplegable Ver, seleccione el componente para el cual desea ver los datos de supervisión.

  5. Haga clic en el botón Refrescar para actualizar los datos de supervisión que aparecen.
