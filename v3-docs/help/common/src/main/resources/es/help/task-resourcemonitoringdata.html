Para ver los datos de supervisión de recursos

Antes de empezar

Antes de poder ver datos de supervisión, es necesario que configure la supervisión. Consulte Cómo configurar el servicio de supervisión para obtener más información.

  1. En el árbol de navegación, seleccione el nodo del servidor (Admin Server).

    Aparecerá la página de Información general.

  2. En la página Información general, haga clic en la ficha Supervisar.

    Se abrirá la página Supervisión de aplicación.

  3. Haga clic en la ficha Recursos.

    Se abre la página Supervisión de recursos.

  4. En la lista desplegable Ver, seleccione el recurso cuyos datos de supervisión desea ver.
  5. (Opcional) Haga clic en el botón Actualización para renovar los datos de supervisión que aparecen.
