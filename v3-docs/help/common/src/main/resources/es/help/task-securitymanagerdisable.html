Desactivar el Gestor de Seguridad

Si desactiva el gestor de seguridad de Servidor GlassFish, puede ver mejorado el rendimiento de algunos tipos de aplicaciones. La funciones de autorización y autenticación de la plataforma Java seguirán funcionando, aunque se desactive el gestor de seguridad. Puede desactivar el gestor de seguridad en un entorno de desarrollo; sin embargo, no recomendamos desactivarlo en un entorno de producción.

  1. En el árbol de navegación, seleccione el nodo Configuración.

  2. Select the configuration that you are changing.

  3. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración General de JVM.

  4. En la página Configuración General de JVM, haga clic en el separador Opciones de JVM.

    Se abrirá la página Opciones de JVM.

  5. Active la casilla de control que aparece a la izquierda del campo Valor que contiene la opción -Djava.security.policy.

  6. Haga clic en Suprimir.

  7. Haga clic en Guardar.

  8. Reinicie el servidor.
