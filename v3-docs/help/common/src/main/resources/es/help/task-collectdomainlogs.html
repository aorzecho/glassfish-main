Recopilar y Descargar Logs de Dominio

  1. En el árbol de navegación, seleccione el nodo Dominio.

    Se abrirá la página Atributos de Dominio.

  2. Seleccione el separador Logs de Dominio.

    Se abrirá la página Logs de Dominio.

    Si existen clusters, se muestran las listas desplegables para los clusters y las instancias. Si no existen clusters, sólo aparece una lista desplegable para las instancias.

  3. Seleccione Cluster o Instancia.

  4. Seleccione el cluster o la instancia para el que vaya a recopilar y descargar archivos log.

  5. Haga clic en Recopilar Logs.

    El mecanismo de descargas de su explorador le indica que descargue el archivo log-files-name.zip, dondename es el nombre del cluster o de la instancia.
