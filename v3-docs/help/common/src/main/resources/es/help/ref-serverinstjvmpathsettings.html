Configuración de ruta JVM

Utilice la página Configuración de ruta JVM para configurar la ruta de biblioteca nativa. La ruta de biblioteca nativa consiste en una concatenación de las rutas siguientes:

  • El prefijo de la ruta de biblioteca nativa

  • La ruta del servidor para sus bibliotecas nativas (es decir, as-install/lib)

  • Los directorios de biblioteca nativa de 64 bits, si procede

  • El valor de la variable java.library.path para la JVM utilizada para iniciar GlassFish Server (este valor es distinto en diferentes sistemas operativos)

  • Las rutas que se especifican en la página Configuración de analizador JVM, si el analizador está habilitado

  • El sufijo de la ruta de biblioteca nativa

Otros parámetros de configuración se muestran como valores de sólo lectura para los dominios que se han actualizado desde una versión anterior de GlassFish Server. No son compatibles con la versión v3.

La página Configuración de ruta JVM contiene las siguientes opciones.

Nombre de configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Ruta de clase de entorno

Si se activa la casilla de verificación Ignorar, se omitirá la variable de entorno CLASSPATH. La variable de entorno CLASSPATH resulta cómoda para tutoriales básicos de programación, pero no se recomienda en entornos de empresa. Éste es un campo de sólo lectura.

Ruta de clase del sistema

La ruta a las clases del sistema. Éste es un campo de sólo lectura.

Prefijo de ruta de clase

Rutas completas a los archivos JAR que se van a anteponer a la ruta de clase del servidor, una por línea. Éste es un campo de sólo lectura.

Sufijo de ruta de clase

Rutas completas a los archivos JAR que se van a anexar a la ruta de clase del servidor, una por línea. Éste es un campo de sólo lectura.

Prefijo de ruta de biblioteca nativa

Rutas completas a los archivos JAR que se van a anteponer a la ruta de biblioteca nativa, una por línea.

Sufijo de ruta de biblioteca nativa

Rutas completas a los archivos JAR que se van a anexar a la ruta de biblioteca nativa, una por línea.
