Detalles de los componentes

La página Detalles de los componentes ofrece información detallada sobre un componente.

La página Detalles de los componentes incluye las opciones siguientes.

Atrás

Este botón se utiliza para regresar a la página donde ha seleccionado el componente del cual estás visualizando los detalles.

Nombre del paquete

Nombre del paquete en el que se proporciona el componente .

Identificador único

Identificador completo de este componente tal y como existe en el depósito.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas de sistemas y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión

Número de versión del paquete en el que se proporciona el componente.

Fecha publicada

Fecha y hora en la que esta versión del componente se publicó en el depósito. Esta fecha no es la fecha y hora en la que el componente se instaló en la imagen de instalación.

Tamaño instalado

Cantidad de espacio en disco que consume la instalación del componente.

Origen

Para un componente instalado: nombre del Depósito de red desde el que se consiguió el componente.

Para un componente de complementos o actualización: nombre del depósito de red desde el que se obtendría el componente.

Descripción

Ofrece una descripción general del componente.
