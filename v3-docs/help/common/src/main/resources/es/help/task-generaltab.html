Ver Información General para DAS

  1. En el árbol de navegación, seleccione el nodo del servidor (servidor de administración).

    Se abrirá la página Información General.

  2. En la página Información General, realice uno de los siguientes pasos.

    • Mire la pantalla de estado de sólo lectura para ver si se está ejecutando el servidor.

    • Mire la pantalla Tiempo de Actividad de sólo lectura para ver el número de horas y minutos en que el servidor se ha ejecutado de forma continua.

    • Haga clic en Parar para parar Servidor GlassFish.

    • Haga clic en el botón Reiniciar para parar y reiniciar Servidor GlassFish.

    • Haga clic en el botón Ver Archivos Log para ver los archivos log de las instancias o clusters de Servidor GlassFish.

      Consulte Ver Archivos Log para DAS para obtener detalles.

    • Haga clic en el botón Ver Log No Procesado para ver archivos log no procesados para DAS.

      Consulte Ver Archivos Log No Procesados para DAS para obtener detalles.

    • Haga clic en Informe de JVM para abrir una ventana independiente que le permitirá ver informes en Java Virtual Machine, incluyendo un informe resumido, un informe sobre gestión de la memoria, otro informe sobre carga de clases y un volcado de threads actuales.
