Cómo ver información general para el DAS

  1. En el árbol de navegación, seleccione el nodo del servidor (Admin Server).

    Aparecerá la página de Información general.

  2. En la página Información general, realice uno de los siguientes pasos.
    • Haga clic en el botón Detener para detener GlassFish Server.
    • Haga clic en el botón Reiniciar para detener y reiniciar GlassFish Server.
    • Haga clic en el botón Ver archivos de registro para ver los archivos de registro de las instancias o clústeres de GlassFish Server.

      Para más detalles, consulte Cómo visualizar archivos de registro para el DAS.

    • Haga clic en Informe de JVM para abrir una ventana independiente que le permitirá ver informes en la máquina virtual de Java, incluyendo un informe resumido, un informe sobre administración de la memoria, otro informe sobre carga de clases y un volcado de subprocesos actuales.
