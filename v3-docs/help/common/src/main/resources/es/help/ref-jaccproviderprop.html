Propiedades específicas de los proveedores JACC

La siguiente opción está disponible para un proveedor JACC.

repository

El directorio que contiene el archivo de política JACC . Para el proveedor JACC de Servidor GlassFish default, el directorio predeterminado es ${com.sun.aas.instanceRoot}/generated/policy. Esta propiedad no está definida de manera predeterminada para el proveedor JACC de Servidor GlassFish simple.
