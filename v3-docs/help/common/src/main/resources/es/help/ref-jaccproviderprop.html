Propiedades específicas de los proveedores JACC

La siguiente opción está disponible para un proveedor JACC.

repository

El directorio que contiene el archivo de política JACC . Para el proveedor JACC de Servidor GlassFish por defecto, el directorio por defecto es ${com.sun.aas.instanceRoot}/generated/policy. Esta propiedad no está definida por defecto para el proveedor JACC Servidor GlassFish simple.
