Como establecer el dominio predeterminado

El dominio predeterminado es el dominio que GlassFish Server utiliza para la autenticación y autorización en caso de que el descriptor de implementación de una aplicación no especifique un dominio.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  3. En la lista desplegable Dominio predeterminado, seleccione el dominio activo que el servidor utilizará para la autenticación.

    El valor predeterminado es file.

  4. Haga clic en Guardar.
