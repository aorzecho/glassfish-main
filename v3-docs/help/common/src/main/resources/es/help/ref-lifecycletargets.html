Destinos del módulo de ciclo de vida

Utilice la página Destinos del módulo de ciclo de vida para ver los clústeres de destino y las instancias de servidor independientes en los que se puede habilitar el módulo de ciclo de vida.

La pagina Destinos del módulo de ciclo de vida contiene la siguiente información.

Nombre de destino

El nombre del clúster o instancia de servidor independiente.

Habilitado

Muestra true si se ha habilitado el módulo de ciclo de vida en el destino, o false si el módulo de ciclo de vida está inhabilitado.

La tabla Destinos también contiene las siguientes opciones.

Administrar destinos

Botón para administrar los destinos del módulo de ciclo de vida.

Más acciones

Lista desplegable con las siguientes acciones.

Habilitar

Acción para habilitar el módulo de ciclo de vida en el destino seleccionado.

Deshabilitar

Acción para habilitar el módulo de ciclo de vida en el destino seleccionado.
