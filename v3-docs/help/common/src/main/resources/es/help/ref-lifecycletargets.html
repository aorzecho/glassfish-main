Destinos del Módulo de Ciclo de Vida

Utilice la página Destinos del Módulo de Ciclo de Vida para ver los clusters de destino y las instancias de servidor independientes en los que se puede activar el módulo de ciclo de vida.

La pagina Destinos del Módulo de Ciclo de Vida contiene la siguiente información.

Nombre de Destino

Nombre del cluster o instancia de servidor independiente.

Activado

Muestra true si se ha activado el módulo de ciclo de vida en el destino o false si el módulo de ciclo de vida está desactivado.

La tabla Destinos también contiene las siguientes opciones.

Gestionar Destinos

Botón para gestionar los destinos del módulo de ciclo de vida.

Más Acciones

Lista desplegable con las siguientes acciones.

Activar

Acción para activar el módulo de ciclo de vida en el destino seleccionado.

Desactivar

Acción para desactivar el módulo de ciclo de vida en el destino seleccionado.
