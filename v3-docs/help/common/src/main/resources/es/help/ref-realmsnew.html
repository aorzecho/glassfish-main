Nuevo dominio

Utilice la página Nuevo dominio para crear un dominio.

La página Nuevo dominio contiene las siguientes opciones.

Nombre de configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

El nombre del nuevo dominio.

Nombre de clase

El nombre de clase del nuevo dominio.

Las opciones válidas se muestran en la siguiente tabla.

Nombre de dominio

Nombre de clase

file

com.sun.enterprise.security.auth.realm.file.FileRealm

certificate

com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

ldap

com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

JDBC

com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

solaris

com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

PAM

com.sun.enterprise.security.auth.realm.solaris.PamRealm

Para especificar un dominio personalizado, seleccione el botón de opción inferior y escriba el nombre de clase para el dominio personalizado que se está creando.

Propiedades adicionales

Propiedades adicionales del dominio. Las propiedades válidas dependen del tipo de dominio.

  • Propiedades específicas de la clase FileRealm

  • Propiedades específicas de la clase CertificateRealm

  • Propiedades específicas de la clase JDBCRealm

  • Propiedades específicas de la clase LDAPRealm

  • Propiedades específicas de la clase SolarisRealm

  • Propiedades específicas de la clase PamRealm
