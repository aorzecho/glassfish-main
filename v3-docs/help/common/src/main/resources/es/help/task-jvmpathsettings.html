Para configurar la configuración de ruta de JVM

  1. En el árbol de navegación, seleccione el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración general de JVM.

  3. En la página Configuración general de JVM, haga clic en la ficha Configuración de ruta.

    Se abre la página Configuración de ruta JVM.

  4. En el campo Prefijo de ruta de biblioteca nativa, introduzca las rutas completas de los archivos JAR que se van a anteponer a la ruta de biblioteca nativa.

    Para anteponer varios archivos, escriba una ruta por línea.

  5. En el campo Sufijo de ruta de biblioteca nativa, introduzca las rutas completas de los archivos JAR que se van a anexar a la ruta de biblioteca nativa.

    Para anexar varios archivos, escriba una ruta por línea.

  6. Haga clic en Guardar.
  7. Reinicie el servidor.
