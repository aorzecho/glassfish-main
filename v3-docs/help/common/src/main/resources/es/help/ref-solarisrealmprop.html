Propiedades específicas de la clase SolarisRealm

Un dominio Solaris requiere la siguiente propiedad.

Contexto JAAS

El contexto JAAS (Servicio de autenticación y autorización de Java) (mecanismo identificador para que el módulo de entrada utilice este dominio). El único valor válido es solarisRealm.

La siguiente propiedad adicional está disponible para un dominio Solaris.

Asignar grupos

Una lista separada por comas de nombres de grupos. Todos los clientes que presenten certificados válidos se asignan a estos grupos, por ejemplo, employee,manager , donde estos son los nombres de grupos de usuarios.
