Información de punto final del servicio web

Utilice la página Información de punto final del servicio web para ver información sobre el punto final de un servicio web.

La página Información de punto final del servicio web contiene la siguiente información.

Nombre de la aplicación

El nombre de la aplicación implementada. Haga clic en este enlace o en el botón Atrás para volver a la página Editar aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte Module and Application Versions de Oracle GlassFish Server 3.1 Application Deployment Guide.

Probador

El vínculo Probador sólo está visible si Tipo de implementación es distinto de RI (por ejemplo, 109). Haga clic en este vínculo para ver una página que permita proporcionar parámetros para probar los métodos del servicio web.

WSDL

El idioma de definición de los servicios web (WSDL) para el punto final. Haga clic en este vínculo para ver el archivo WSDL en una nueva pestaña o ventana del explorador.

Tenga en cuenta que algunos navegadores tratan de interpretar el código fuente WSDL al hacer clic en el vínculo WSDL, de modo que muestran una página en blanco. Para estos navegadores, utilice el comando Ver código fuente del navegador en la página en blanco para ver el código fuente WSDL.

Nombre del punto final

El nombre del punto final.

Nombre del servicio

El nombre del servicio web.

Nombre del puerto

El nombre del puerto del servicio web.

Tipo de implementación

El tipo de implementación del servicio web (por ejemplo, RI o 109).

Tipo de implementación

El tipo de implementación del servicio web (por ejemplo, SERVLET o EJB).

Nombre de clase de implementación

El nombre de la clase de implementación del servicio web.

URI de dirección de punto final

El URI de la dirección del punto final.

Espacio de nombre

El espacio de nombre de destino asociado con el servicio web.

Descripción

Una descripción del punto final, si existe.
