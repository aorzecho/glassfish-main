Propiedades específicas de los módulos de auditoría

La siguientes propiedades están disponibles para un módulo de auditoría.

auditOn

Un valor true origina la carga del módulo de auditoría y asegura que sea llamado por la biblioteca de auditoría de Servidor GlassFish en los puntos de auditoría. Un valor false deshabilita el registro de auditorías para el módulo.
