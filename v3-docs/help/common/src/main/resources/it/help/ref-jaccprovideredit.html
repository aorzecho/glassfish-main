Edit JACC Provider

Nella pagina Edit JACC Provider è possibile modificare un provider JACC per l'istanza o il cluster GlassFish Server selezionato.

La pagina Edit JACC Provider contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Name

Nome del provider JACC. Il campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo provider JACC.

Policy Configuration

Nome di classe completo per javax.security.jacc.PolicyConfigurationFactory.provider che implementa la classe javax.security.jacc.PolicyConfigurationFactory specifica del provider.

Policy Provider

Nome di classe completo per javax.security.jacc.policy.provider che implementa java.security.Policy.

Additional Properties

Proprietà aggiuntive per il provider JACC. Per la descrizione delle proprietà aggiuntive disponibili per i provider JACC, vedere Proprietà specifiche dei provider JACC.
