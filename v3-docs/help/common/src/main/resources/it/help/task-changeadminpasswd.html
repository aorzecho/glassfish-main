Modificare la password per un utente nel realm di amministrazione

Per modificare la password per l'account dell'amministratore corrente, vedere Modificare la password dell'amministratore. Se admin-realm include più utenti è possibile modificare anche le password degli altri utenti.

La cifratura della password dell'amministratore consente di ottenere maggiore sicurezza per l'applicazione. Per informazioni più dettagliate sulle procedure di sicurezza, vedere il Chapter 1, Administering System Security, in Oracle GlassFish Server Security Guide.

  1. Nel nodo Configurazioni, espandere la configurazione dell'istanza server o del cluster per cui si desidera modificare una password di amministrazione.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Modifica realm.

  5. Nella pagina Modifica realm, fare clic sul pulsante Gestisci utenti.

    Viene visualizzata la pagina Utenti amministratori.

  6. Nella pagina Utenti amministratori fare clic sull'utente per il quale si desidera aggiungere o modificare la password.

    Viene visualizzata la pagina Modifica utente realm.

  7. Nei campi Nuova password e Conferma nuova password digitare la nuova password.

  8. Fare clic su Salva.
