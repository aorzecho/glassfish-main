Modificare la password di un utente nel settore Admin

Per cambiare la password per l'account dell'amministratore corrente, vedere Modificare la password dell'amministratore. Se admin-realm include più utenti è possibile cambiare anche le password degli altri utenti.

La crittografia della password dell'amministratore consente di ottenere maggiore sicurezza per l'applicazione. Per informazioni più dettagliate sulle procedure di sicurezza, vedere il Capitolo 1, Administering System Security in Oracle GlassFish Server 3.1 Security Guide.

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera modificare una password di amministrazione.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.
  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Edit Realm.

  5. Nella pagina Edit Realm, fare clic sul pulsante Manage Users.

    Viene visualizzata la pagina Admin Users.

  6. Nella pagina Admin Users fare clic sull'utente per il quale si desidera aggiungere o modificare la password.

    Viene visualizzata la pagina Edit File Realm User.

  7. Nei campi New Password e Confirm New Password digitare la nuova password.
  8. Fare clic su Save.
