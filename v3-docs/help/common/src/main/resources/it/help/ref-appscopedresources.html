Risorse con ambito applicazione

Nella pagina Risorse con ambito applicazione è possibile visualizzare le risorse disponibili solo per una determinata applicazione.

La pagina Risorse con ambito applicazione contiene una tabella con le informazioni riportate di seguito.

Nome applicazione

Nome dell'applicazione.

Nome modulo

Nome del modulo all'interno dell'applicazione.

Nome risorsa

Nome del risorsa con ambito a livello di applicazione per l'applicazione o il modulo. Fare clic sul nome per modificare le impostazioni della risorsa.

Tipo di risorsa

Il tipo di risorsa, ad esempio Connection pool JDBC o Risorse connettore.
