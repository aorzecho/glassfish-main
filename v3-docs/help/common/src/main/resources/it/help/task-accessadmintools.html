Consentire l'accesso agli strumenti di amministrazione

Soltanto gli utenti del gruppo asadmin possono accedere a Administration Console e all'utility della riga di comando asadmin. Per garantire agli utenti l'accesso a questi strumenti di amministrazione, aggiungerli al gruppo asadmin nel settore admin-realm.

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i privilegi di accesso per l'amministrazione.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.
  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Edit Realm.

  5. Nella pagina Edit Realm, fare clic sul pulsante Manage Users.

    Viene visualizzata la pagina Admin Users.

    Subito dopo l'installazione, il nome utente e la password di amministratore immessi durante l'installazione vengono elencati in un file denominato admin-keyfile. Per impostazione predefinita tale utente fa parte del gruppo asadmin, ossia dispone dei diritti per la modifica di GlassFish Server. Assegnare utenti a questo gruppo solo se si desidera concedere loro privilegi di amministratore per la specifica istanza o cluster GlassFish Server.

    Se si aggiungono utenti al settore admin-realm assegnandoli a un gruppo diverso da asadmin, le informazioni sugli utenti verranno comunque salvate nel file admin-keyfile, ma gli utenti non avranno accesso agli strumenti o alle applicazioni di amministrazione nel settore file.

  6. Nella pagina Admin Users, fare clic su New per aggiungere un nuovo utente al settore admin-realm.

    Viene visualizzata la pagina New File Realm User.

  7. Nel campo User ID digitare il nuovo nome utente.
  8. Nel campo Group List digitare i nomi dei gruppi di appartenenza dell'utente.

    Per autorizzare un utente ad apportare modifiche a GlassFish Server, includere asadmin nell'elenco Group List.

  9. Nei campi New Password e Confirm New Password digitare la password per il nuovo utente.
  10. Fare clic su OK per aggiungere questo utente al settore admin-realm, oppure su Cancel per uscire senza salvare.
