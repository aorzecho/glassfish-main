Creare un modulo del ciclo di vita

  1. Nella struttura ad albero, selezionare il nodo Lifecycle Modules.

    Viene visualizzata la pagina Lifecycle Modules.

  2. Nella pagina Lifecycle Modules, fare clic su New.

    Viene visualizzata la pagina New Lifecycle Module.

  3. Nel campo Name, digitare un nome per il modulo del ciclo di vita.
  4. Nel campo Class Name digitare il nome completo del file di classe del modulo.

    La classe deve implementare l'interfaccia com.sun.appserv.server.LifecycleListener.

  5. Nel campo Classpath digitare il classpath per il modulo del ciclo di vita.

    Il classpath specifica la posizione del modulo del ciclo di vita. Il valore predefinito è domain-dir/applications. Se il modulo si trova già in questa directory (ossia nel classpath del server) il campo può essere lasciato vuoto.

  6. Nel campo Load Order, digitare un valore per specificare l'ordine di caricamento del modulo del ciclo di vita all'avvio.

    I moduli con i numeri di ordine di caricamento più bassi verranno caricati per primi. Il valore può essere compreso tra 101 e il valore MAXINT del sistema operativo. I valori compresi tra 1 e 100 sono riservati.

  7. Nel campo Description, digitare una descrizione del modulo del ciclo di vita.
  8. Selezionare la casella di controllo Status Enabled per abilitare il modulo del ciclo di vita.

    Questa opzione è abilitata per impostazione predefinita.

  9. Selezionare la casella di controllo On Load Failure Prevent Instance Startup per impostare l'arresto del server in caso di mancato caricamento del modulo del ciclo di vita.

    Questa opzione è disabilitata per impostazione predefinita.

  10. Per aggiungere istanze server e cluster di destinazione in cui abilitare il modulo del ciclo di vita, selezionarli dall'elenco Available Targets e utilizzare il pulsante Add per spostarli nell'elenco Selected Targets.

    L'opzione Target viene visualizzata se esistono cluster o istanze server standalone oltre all'istanza server predefinita (server). Per impostazione predefinita, il modulo del ciclo di vita viene abilitato nell'istanza server predefinita, denominata server. Utilizzare il pulsante Add All per eseguire la distribuzione in tutte le destinazioni. Utilizzare i pulsanti Remove e Remove All per rimuovere destinazioni dall'elenco Selected Targets.

  11. Fare clic su OK.
