Creare un modulo del ciclo di vita

  1. Nell'albero di navigazione selezionare il nodo Moduli del ciclo di vita.

    Viene visualizzata la pagina Moduli del ciclo di vita.

  2. Nella pagina Moduli del ciclo di vita, fare clic su Nuovo.

    Viene visualizzata la pagina Nuovo modulo del ciclo di vita.

  3. Nel campo Nome, digitare un nome per il modulo del ciclo di vita.

  4. Nel campo Nome classe digitare il nome completo del file di classe del modulo del ciclo di vita.

    La classe deve implementare l'interfaccia com.sun.appserv.server.LifecycleListener.

  5. Nel campo Classpath digitare il classpath per il modulo del ciclo di vita.

    Il classpath specifica la posizione del modulo del ciclo di vita. Il valore predefinito è domain-dir/applications. Se il modulo si trova già in questa directory (ossia nel classpath del server) il campo può essere lasciato vuoto.

  6. Nel campo Ordine di caricamento, digitare un valore per specificare l'ordine di caricamento del modulo del ciclo di vita all'avvio.

    I moduli con i numeri di ordine di caricamento più bassi verranno caricati per primi. Il valore può essere compreso tra 101 e il valore MAXINT del sistema operativo. I valori compresi tra 1 e 100 sono riservati.

  7. Nel campo Descrizione, digitare una descrizione del modulo del ciclo di vita.

  8. Selezionare la casella di controllo Stato abilitato per abilitare il modulo del ciclo di vita.

    Questa opzione è abilitata per impostazione predefinita.

  9. Selezionare la casella di controllo In caso di errore di caricamento: Impedisci avvio istanza per impostare l'arresto del server in caso di mancato caricamento del modulo del ciclo di vita.

    Questa opzione è disabilitata per impostazione predefinita.

  10. Per aggiungere istanze server e cluster di destinazione in cui abilitare il modulo del ciclo di vita, selezionarli dalla lista Destinazioni disponibili e utilizzare il pulsante Aggiungi per spostarli nella lista Destinazioni selezionate.

    L'opzione Destinazione viene visualizzata se esistono cluster o istanze server standalone oltre all'istanza server predefinita (server). Per impostazione predefinita, il modulo del ciclo di vita viene abilitato nell'istanza server predefinita, denominata server. Utilizzare il pulsante Aggiungi tutto per eseguire la distribuzione in tutte le destinazioni. Utilizzare i pulsanti Rimuovi e Rimuovi tutto per rimuovere destinazioni dalla lista Destinazioni selezionate.

  11. Fare clic su OK.
