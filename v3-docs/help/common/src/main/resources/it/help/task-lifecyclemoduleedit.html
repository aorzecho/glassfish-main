Modificare un modulo del ciclo di vita

  1. Nella struttura ad albero, selezionare il nodo Lifecycle Modules.

    Viene visualizzata la pagina Lifecycle Modules.

  2. Nella pagina Lifecycle Modules fare clic sul nome del modulo del ciclo di vita modificare.

    Viene visualizzata la pagina Edit Lifecycle Module.

  3. Nel campo Class Name digitare il nome completo del file di classe del modulo.

    La classe deve implementare l'interfaccia com.sun.appserv.server.LifecycleListener.

  4. Nel campo Classpath digitare il classpath per il modulo del ciclo di vita.

    Il classpath specifica la posizione del modulo del ciclo di vita. Il valore predefinito è domain-dir/applications. Se il modulo si trova già in questa directory (ossia nel classpath del server) il campo può essere lasciato vuoto.

  5. Nel campo Load Order, digitare un valore per specificare l'ordine di caricamento del modulo del ciclo di vita all'avvio.

    I moduli con i numeri di ordine di caricamento più bassi verranno caricati per primi. Il valore può essere compreso tra 101 e il valore MAXINT del sistema operativo. I valori compresi tra 1 e 100 sono riservati.

  6. Nel campo Description, digitare una descrizione del modulo del ciclo di vita.
  7. Selezionare la casella di controllo Status Enabled per abilitare il modulo del ciclo di vita.

    Questa opzione è abilitata per impostazione predefinita.

  8. Selezionare la casella di controllo On Load Failure Prevent Instance Startup per impostare l'arresto del server in caso di mancato caricamento del modulo del ciclo di vita.

    Questa opzione è disabilitata per impostazione predefinita.

  9. Viene visualizzata la scheda Targets se esistono cluster o istanze server standalone oltre all'istanza server predefinita (server). Selezionare la scheda Targets per visualizzare le destinazioni in cui è stato abilitato il modulo del ciclo di vita. Per ulteriori informazioni su come abilitare o disabilitare un modulo del ciclo di vita in una o più destinazioni, vedere Visualizzare le destinazioni di un'applicazione.
  10. Fare clic su Save.
