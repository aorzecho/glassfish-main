Product Registration

Per registrare l'installazione di Oracle GlassFish Server è necessario disporre di un Sun Online Account. Se non si dispone di un account Sun Online, è possibile crearne uno durante la registrazione di GlassFish Server.

La pagina Product Registration contiene le seguenti opzioni.

I already have a Sun Online Account or a Sun Developer Network (SDN) Account

Selezionare questa opzione se si dispone già di un account. Se si seleziona questa opzione, saranno visibili i seguenti campi.

User Name

Nome utente specificato durante la registrazione dell'account Sun Online.

Password

Password specificata durante la registrazione dell'account Sun Online.

Proxy Host

Nome host del server proxy per il computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Proxy Port

Numero della porta del server proxy per i computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

I don't have a Sun Online Account. Sign me up.

Selezionare questa opzione se non si dispone di un account e si desidera registrare l'installazione di GlassFish Server. Se si seleziona questa opzione, saranno visibili i seguenti campi.

Email Address

Indirizzo di posta elettronica personale da utilizzare come nome utente dell'account Sun Online.

Password

La password da utilizzare per l'account Sun Online. Alla password viene applicata la distinzione tra maiuscole e minuscole; la sua lunghezza deve essere compresa tra 6 e 10 caratteri.

Retype Password

È necessario ridigitare la password specificata nel campo Password.

First Name

Nome.

Last Name

Cognome.

Company

Società per cui si lavora, se applicabile.

Country/Territory

Paese o regione in cui si risiede.

Proxy Host

Nome host del server proxy per il computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Proxy Port

Numero della porta del server proxy per i computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.
