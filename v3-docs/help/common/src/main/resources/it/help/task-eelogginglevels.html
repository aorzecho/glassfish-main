Configurare i livelli di log

  1. Nella struttura ad albero, espandere il nodo Configurations.
  2. Nel nodo Configuration, fare clic sulla configurazione dell'istanza server o del cluster per cui si desidera configurare le impostazioni del programma di log.

    Viene visualizzata la pagina Configuration.

  3. Nella pagina Configuration, fare clic su Logger Settings.

    Viene visualizzata la pagina Logger Settings per la destinazione della configurazione selezionata.

  4. Nella pagina Logger Settings fare clic sulla scheda Log Levels.

    Viene visualizzata la pagina Module Log Levels.

  5. Nella pagina Module Log Levels, modificare i livelli di log di uno o più moduli attenendosi alla seguente procedura.
    • Per modificare il livello di log di un singolo modulo, selezionare un nuovo valore dall'elenco a discesa corrispondente al modulo.
    • Per modificare il livello di log di un gruppo di moduli:
      1. Selezionare le caselle di controllo in corrispondenza dei nomi dei moduli per i quali si desidera modificare i livelli di log.
      2. Selezionare un valore dall'elenco a discesa nella parte superiore della tabella.
      3. Fare clic sul pulsante Change Level.

    Il log visualizzerà i messaggi corrispondenti al livello selezionato o ai livelli superiori. Scegliere uno dei seguenti valori (elencati in ordine decrescente):

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • OFF

    Per informazioni dettagliate sul significato dei livelli di log, vedere Livelli di log del modulo. Il livello di default è INFO: di conseguenza, per impostazione predefinita, il log conterrà messaggi di livello INFO, WARNING o SEVERE.

  6. Fare clic su Save.

    Le modifiche delle impostazioni Log Level hanno effetto immediato. Vengono inoltre salvate in modo da poter essere utilizzate al riavvio del server.
