Configurare le risorse di GlassFish Server utilizzate dai provider di sicurezza dei messaggi

  1. Se si utilizza un token nome utente, configurare un database di utenti (se necessario).

    Se si utilizza un token nome utente/password, in primo luogo configurare un settore e un database di utenti appropriato per il settore.

    La configurazione di un database di utenti è illustrata in Modificare un settore.

  2. Se necessario, gestire i certificati e le chiavi private.
Procedura successiva

Una volta configurate le risorse di GlassFish Server per l'utilizzo da parte dei provider di sicurezza dei messaggi, è possibile abilitare i provider installati con GlassFish Server, come descritto in Modificare una configurazione per la sicurezza dei messaggi.
