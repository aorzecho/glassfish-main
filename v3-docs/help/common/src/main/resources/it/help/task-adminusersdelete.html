Eliminare un utente dal settore Admin

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera gestire gli utenti del settore Admin.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.
  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Edit Realm.

  5. Nella pagina Edit Realm, fare clic sul pulsante Manage Users.

    Viene visualizzata la pagina Admin Users.

  6. Nella pagina Admin Users selezionare la casella di controllo a sinistra del nome dell'utente da eliminare.
  7. Fare clic su Delete.

    Viene richiesto di confermare l'eliminazione.

  8. Fare clic su OK per eliminare l'utente.
