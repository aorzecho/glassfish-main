Common Tasks

Nella pagina Common Tasks sono disponibili tasti di scelta rapida per attività comuni di Administration Console.

Nella pagina Common Tasks è possibile eseguire le operazioni seguenti:

  • Visualizzare la pagina GlassFish News per GlassFish Server

  • Visualizzare la pagina Subscriptions per GlassFish Server

  • Visualizzare un elenco di applicazioni distribuite

  • Distribuire un'applicazione Web

  • Distribuire un'applicazione enterprise

  • Distribuire un client dell'applicazione

  • Distribuire un modulo connettore

  • Distribuire un file JAR EJB

  • Visualizzare i componenti installati

  • Installare i componenti add-on

  • Visualizzare gli aggiornamenti disponibili per i componenti

  • To Create a JDBC Connection Pool

Nella pagina Common Tasks sono inoltre disponibili alcuni collegamenti alla documentazione di GlassFish Server.
