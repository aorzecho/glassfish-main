Visualizzare i componenti installati

Nella pagina Installed Components sono disponibili informazioni sui components dell'installation image installati.

La disinstallazione di un component implica la rimozione dei seguenti elementi:

  • Tutti i file e le directory introdotti con l'installazione originale del componente

  • Tutti i file e le directory introdotti con gli aggiornamenti del componente successivi all'installazione originale

  1. Nella struttura ad albero, selezionare il nodo Update Tool.

    Viene visualizzata la pagina Available Add-Ons.

  2. Nella pagina Available Add-Ons fare clic sulla scheda Installed Components.

    Viene visualizzata la pagina Installed Components.

Procedura successiva

Prima di disinstallare gli aggiornamenti installati, arrestare il dominio di GlassFish Server, quindi utilizzare uno dei seguenti comandi:

  • as-install-parent/bin/updatetool, che avvia l'utility grafica Update Tool

  • as-install-parent/bin/pkg, una versione a riga di comando dell'utility Update Tool

Per informazioni sull'utilizzo del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Per l'utility Update Tool è disponibile una Guida in linea specifica.

Una volta completata la disinstallazione degli aggiornamenti, riavviare il server.
