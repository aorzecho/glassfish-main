Lifecycle Modules

Nella pagina Lifecycle Modules è possibile configurare moduli del ciclo di vita.

I moduli del ciclo di vita eseguono attività quando vengono attivati da uno o più eventi nel ciclo di vita del server. Gli eventi del server in grado di attivare i moduli del ciclo di vita sono inizializzazione, avvio, richieste "ready to service" e arresti. I moduli del ciclo di vita non sono parte delle specifiche Java EE, ma costituiscono un miglioramento di GlassFish Server.

Per ogni modulo del ciclo di vita vengono fornite le seguenti informazioni.

Name

Nome del modulo del ciclo di vita.

Enabled or Status

Il segno di spunta indica che l'applicazione è abilitata, mentre la X indica che è disabilitata, se esiste solo l'istanza server predefinita server.

Se esistono cluster o altre istanze server standalone, selezionare Enabled on All Targets per visualizzare le destinazioni in cui l'applicazione è distribuita.

Load Order

Valore che specifica l'ordine relativo con cui si desidera caricare il modulo del ciclo di vita.

La tabella Lifecycle Modules contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo modulo del ciclo di vita.

Delete

Pulsante che consente di eliminare uno o più moduli del ciclo di vita selezionati.

Enable

Pulsante che consente di abilitare uno o più moduli del ciclo di vita selezionati. Presente se esiste solo l'istanza server predefinita server.

Disable

Pulsante che consente di disabilitare uno o più moduli del ciclo di vita selezionati. Presente se esiste solo l'istanza server predefinita server.
