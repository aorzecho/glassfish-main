Impostazioni profiler JVM

Uno strumento profiler genera dati utilizzati per analizzare le prestazioni e identificare potenziali punti critici.

Specificare le impostazioni del profiler. Le informazioni specificate dipendono dal prodotto profiler in uso. Per esempi e istruzioni, consultare Chapter 3, Debugging Applications in Oracle GlassFish Server Application Development Guide.
