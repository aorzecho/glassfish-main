JVM Profiler Settings

Uno strumento profiler genera dati utilizzabili per analizzare le prestazioni e identificare potenziali colli di bottiglia.

Specificare le impostazioni del profiler. Le informazioni specificate dipendono dal prodotto profiler utilizzato. Per esempi e istruzioni, vedere il Capitolo 3, Debugging Applications in Oracle GlassFish Server 3.1 Application Development Guide.
