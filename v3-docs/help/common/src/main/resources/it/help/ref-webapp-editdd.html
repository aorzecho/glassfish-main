Module Descriptors

Nella pagina Module Descriptors è possibile visualizzare i descrittori di modulo per un'applicazione.

La pagina Module Descriptors contiene le seguenti opzioni.

Deployment Descriptors

Tabella in cui sono indicati il nome dell'applicazione, il nome del componente secondario (se applicabile), e il nome del file del descrittore.
