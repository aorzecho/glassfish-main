Proprietà specifiche della classe FileRealm

Le seguenti proprietà sono obbligatorie per un settore file.

JAAS Context

Contesto di JAAS (Java Authentication and Authorization Service): l'identificatore del modulo di login da utilizzare per questo settore. L'unico valore valido è fileRealm.

Key File

Percorso completo e nome del file in cui il server memorizzerà tutte le informazioni su utente, gruppo e password relative a questo settore. Il valore predefinito è domain-dir/config/keyfile per il settore file e domain-dir/config/admin-keyfile per il settore admin-realm.

Il file di chiave per il settore file inizialmente è vuoto, pertanto è necessario aggiungere utenti alla pagina File Users prima che il settore file venga utilizzato.

Il file di chiave per il settore admin-realm inizialmente contiene il nome utente dell'amministratore, la password dell'amministratore in formato crittografato e il gruppo di appartenenza dell'utente (che per impostazione predefinita è asadmin). Per maggiori informazioni su come aggiungere utenti al settore admin-realm, vedere Consentire l'accesso agli strumenti di amministrazione.

Nota - Gli utenti del gruppo asadmin nel settore admin-realm sono autorizzati a utilizzare gli strumenti di Administration Console e asadmin. Aggiungere soltanto utenti con privilegi di amministrazione server.

Per un settore file è disponibile la seguente priorità opzionale.

Assign Groups

Elenco separato da virgole di nomi di gruppi. Tutti i client con certificati validi vengono assegnati a questi gruppi, ad esempio employee,manager, dove questi sono i nomi dei gruppi di utenti.
