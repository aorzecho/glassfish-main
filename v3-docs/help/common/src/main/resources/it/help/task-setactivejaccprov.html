Impostare il provider JACC attivo

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Security.

    Viene visualizzata la pagina Security.

  3. Nell'elenco a discesa JACC selezionare il provider JACC che verrà utilizzato dal server.

    Per impostazione predefinita, sono disponibili due provider JACC, denominati default e simple. L'impostazione predefinita è default.

  4. Fare clic su Save.
