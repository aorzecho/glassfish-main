Log di dominio

Utilizzare la pagina Log di dominio per raccogliere e scaricare i file di log per GlassFish Server.

La pagina Log di dominio contiene le opzioni riportate di seguito. È possibile selezionare un cluster o un'istanza server per la quale raccogliere i log.

Cluster

Elenco a discesa di cluster, se disponibili.

Istanza

Elenco a discesa di istanze server.

Raccogli log

Pulsante che consente di raccogliere i log in un file log-files-nome.zip file, dovenome è il nome del cluster o dell'istanza.

Il file log-files-nome.zip viene scaricato utilizzando il meccanismo di scaricamento del browser.
