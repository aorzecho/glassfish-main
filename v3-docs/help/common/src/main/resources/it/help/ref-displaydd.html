Deployment Descriptor

Nella pagina Deployment Descriptor è possibile visualizzare il descrittore di distribuzione per un modulo.

La pagina Deployment Descriptor contiene le seguenti informazioni.

Application Name

Nome dell'applicazione.

Module Name

Nome del modulo che contiene il descrittore di distribuzione.

Descriptor File Name

Nome del file del descrittore di distribuzione.
