Configurare le opzioni JVM

Nella pagina JVM Options è possibile specificare le opzioni del programma di avvio dell'applicazione Java (java) che esegue GlassFish Server. L'opzione -D indica le proprietà specifiche di GlassFish Server.

  1. Nella struttura ad albero, selezionare il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo JVM Settings.

    Viene visualizzata la pagina JVM General Settings.

  3. Nella pagina JVM General Settings, fare clic sulla scheda JVM Options.

    Viene visualizzata la pagina JVM Options.

  4. Per modificare un'opzione, è necessario cambiare il valore del campo Value dell'opzione.
  5. Per aggiungere un'opzione:
    1. Fare clic sul pulsante Add JVM Option.
    2. Nella riga vuota visualizzata, digitare nel campo Value le informazioni sull'opzione.
  6. Per eliminare un'opzione:
    1. Selezionare la casella di controllo a sinistra del campo Value corrispondente all'opzione da eliminare.
    2. Fare clic su Delete.
  7. Fare clic su Save.
  8. Riavviare il server.
