Raccogliere e scaricare i log di dominio

  1. Nell'albero di navigazione, selezionare il nodo Dominio.

    Viene visualizzata la pagina Attributi dominio.

  2. Selezionare la scheda Log di dominio.

    Viene visualizzata la pagina Log di dominio.

    Se sono presenti dei cluster, vengono visualizzati gli elenchi a discesa per cluster e istanze. Se non esistono cluster, viene visualizzato un solo elenco a discesa per le istanze.

  3. Selezionare Cluster o Istanza.

  4. Selezionare il cluster o l'istanza per cui si stanno raccogliendo e scaricando file di log.

  5. Fare clic su Raccogli log.

    Viene richiesto tramite il meccanismo di scaricamento del browser di scaricare il file log-files-name.zip, dove name è il nome del cluster o dell'istanza.
