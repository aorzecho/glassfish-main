Aggiungere risorse configurate in un file XML

  1. Nell'albero di navigazione, selezionare il nodo Risorse.

    Viene visualizzata la pagina Risorse.

  2. Fare clic sul pulsante Aggiungi risorse.

    Viene visualizzata la pagina Aggiungi risorse.

  3. Selezionare una delle opzioni riportate di seguito per specificare la posizione del file glassfish-resources.xml.

    File XML da caricare nel server

    Digitare il percorso del file se situato su un computer remoto.

    File XML locale accessibile da GlassFish Server

    Digitare il percorso del file se situato nel computer del server di amministrazione del dominio (DAS).

  4. Dall'elenco a discesa Destinazione, selezionare il cluster o l'istanza server standalone su cui creare le risorse.

  5. Fare clic su OK.
