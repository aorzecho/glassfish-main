Destinazioni server virtuali

Nella pagina Destinazioni server virtuali è possibile modificare i server virtuali di destinazione a cui è associata un'applicazione o un modulo.

La pagina Destinazioni server virtuali contiene le informazioni riportate di seguito.

Destinazioni disponibili

Server virtuali non associati a un'applicazione o un modulo.

Destinazioni selezionate

Server virtuali associati a un'applicazione o un modulo.

Aggiungi

Pulsante che consente di spostare la destinazione selezionata da Destinazioni disponibili a Destinazioni selezionate.

Aggiungi tutto

Pulsante che consente di spostare tutte le destinazioni da Destinazioni disponibili a Destinazioni selezionate.

Rimuovi

Pulsante che consente di spostare la destinazione selezionata da Destinazioni selezionate a Destinazioni disponibili.

Rimuovi tutto

Pulsante che consente di spostare tutte le destinazioni da Destinazioni selezionate a Destinazioni disponibili.
