Destinazioni server virtuale

Nella pagina Virtual Server Targets è possibile modificare i server virtuali di destinazione a cui è associata un'applicazione o un modulo.

La pagina Virtual Server Targets contiene le seguenti informazioni.

Available Targets

Server virtuali non associati a un'applicazione o un modulo.

Selected Targets

Server virtuali associati a un'applicazione o un modulo.

Add

Pulsante che consente di spostare la destinazione selezionata da Available Targets in Selected Targets.

Add All

Pulsante che consente di spostare tutte le destinazioni di Available Targets in Selected Targets.

Remove

Pulsante che consente di spostare la destinazione selezionata da Selected Targets in Available Targets.

Remove All

Pulsante che consente di spostare tutte le destinazioni di Selected Targets in Available Targets.
