Alias password

Nella pagina Alias password è possibile creare e gestire gli alias password.

Nella pagina Alias password viene visualizzata una lista di alias password del dominio. Per ogni alias vengono visualizzate le informazioni riportate di seguito.

Nome

Nome assegnato all'alias password durante la creazione. Se si fa clic sul nome, viene visualizzata la pagina Modifica alias password per l'alias.

La tabella Alias password contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un alias password.

Elimina

Pulsante che consente di eliminare uno o più alias password selezionati.
