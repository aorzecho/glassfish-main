Password Aliases

Nella pagina Password Aliases è possibile creare e gestire gli alias delle password.

Nella pagina Password Aliases viene visualizzato un elenco di alias delle password del dominio. Per ogni alias vengono visualizzate le seguenti informazioni:

Name

Nome assegnato all'alias della password al momento della creazione. Se si fa clic sul nome, viene visualizzata la pagina Edit Password Alias per l'alias.

La tabella Password Aliases contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un alias per una password.

Delete

Pulsante che consente di eliminare uno o più alias di password selezionati.
