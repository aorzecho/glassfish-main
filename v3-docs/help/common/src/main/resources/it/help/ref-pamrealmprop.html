Proprietà specifiche della classe PamRealm

La seguente proprietà è obbligatoria per un settore PAM.

JAAS Context

Contesto di JAAS (Java Authentication and Authorization Service): l'identificatore del modulo di login da utilizzare per questo settore. L'unico valore valido è pamRealm.

Per un settore PAM è disponibile la seguente proprietà opzionale.

Assign Groups

Elenco separato da virgole di nomi di gruppi. Tutti i client con certificati validi vengono assegnati a questi gruppi, ad esempio employee,manager, dove questi sono i nomi dei gruppi di utenti.
