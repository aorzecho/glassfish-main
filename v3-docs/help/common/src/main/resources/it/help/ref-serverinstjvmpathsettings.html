JVM Path Settings

Nella pagina JVM Path settings è possibile configurare il percorso della libreria nativa, costituito da una concatenazione dei percorsi seguenti:

  • Prefisso di libreria nativa

  • Percorso del server per le librerie native (ossia as-install/lib )

  • Directory della libreria nativa a 64 bit, se applicabile

  • Valore della variabile java.library.path per JVM utilizzato per avviare GlassFish Server (differisce in base al sistema operativo in uso)

  • Qualsiasi percorso specificato nella pagina JVM Profiler Settings, se è abilitato il profiler

  • Suffisso del percorso della libreria nativa

Altre impostazioni sono visualizzate come valori di sola lettura per domini aggiornati da una versione precedente di GlassFish Server. Non sono supportati nella versione v3.

La pagina JVM Path Settings contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Environment Classpath

Se è selezionata la casella di controllo Ignore, la variabile di ambiente CLASSPATH viene ignorata. La variabile di ambiente CLASSPATH è adatta a esercitazioni di base di programmazione ma non è consigliabile per gli ambienti aziendali. Il campo è di sola lettura.

System Classpath

Percorso per le classi del sistema. Il campo è di sola lettura.

Classpath Prefix

Percorsi completi dei file JAR da aggiungere come prefisso al classpath del server, uno per riga. Il campo è di sola lettura.

Classpath Suffix

Percorsi completi dei file JAR da aggiungere come suffisso al classpath del server, uno per riga. Il campo è di sola lettura.

Native Library Path Prefix

Percorsi completi dei file JAR da aggiungere come prefisso al percorso della libreria nativa, uno per riga.

Native Library Path Suffix

Percorsi completi dei file JAR da aggiungere come suffisso al percorso della libreria nativa, uno per riga.
