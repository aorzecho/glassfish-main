Impostazioni percorso JVM

Nella pagina Impostazioni percorso JVM è possibile configurare il percorso della libreria nativa, Il percorso della libreria nativa è una concatenazione dei percorsi seguenti:

  • Prefisso percorso di libreria nativa

  • Percorso del server per le librerie native, ovvero as-install/lib

  • Directory di libreria nativa a 64 bit, se applicabile

  • Valore della variabile java.library.path per JVM utilizzato per avviare GlassFish Server. Varia in base al sistema operativo in uso.

  • Qualsiasi percorso specificato nella pagina Impostazioni profiler JVM, se è abilitato il profiler.

  • Suffisso percorso di libreria nativa

Altre impostazioni vengono visualizzate come valori di sola lettura per i domini aggiornati da una release precedente di GlassFish Server. Non sono supportati nella release v3.

La pagina Impostazioni percorso JVM contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Classpath di ambiente

Se è selezionata la casella di controllo Ignora, la variabile di ambiente CLASSPATH viene ignorata. La variabile di ambiente CLASSPATH è utile per le esercitazioni di base di programmazione, ma non è consigliabile per gli ambienti enterprise. Questo campo è di sola lettura.

Classpath di sistema

Percorso delle classi del sistema. Questo campo è di sola lettura.

Prefisso classpath

Percorsi completi dei file JAR da aggiungere come prefisso al classpath del server, uno per riga. Questo campo è di sola lettura.

Suffisso classpath

Percorsi completi dei file JAR da aggiungere come suffisso al classpath del server, uno per riga. Questo campo è di sola lettura.

Prefisso percorso di libreria nativa

Percorsi completi dei file JAR da aggiungere come prefisso al percorso di libreria nativa, uno per riga.

Suffisso percorso di libreria nativa

Percorsi completi dei file JAR da aggiungere come suffisso al percorso di libreria nativa, uno per riga.
