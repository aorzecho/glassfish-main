Configurare il servizio di monitoraggio

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare il monitoraggio.
  2. Espandere il nodo Monitoring.

    Viene visualizzata la pagina Monitoring Service.

  3. Selezionare la casella di controllo Monitoring Service Enabled per abilitare il monitoraggio per GlassFish Server o deselezionarla per disabilitare il monitoraggio.

    Questa opzione è abilitata per impostazione predefinita.

  4. Selezionare la casella di controllo Monitoring MBeans Enabled per distribuire tutti gli elementi MBeans necessari per il monitoraggio o deselezionare la casella per annullare la distribuzione degli elementi MBeans.

    Questa opzione è abilitata per impostazione predefinita.

    Se la casella di controllo non è selezionata non è possibile visualizzare i dati di monitoraggio in Administration Console, anche se è selezionata la casella di controllo Monitoring Service. In tal caso per visualizzare i dati di monitoraggio è necessario utilizzare la riga di comando. Se si deseleziona la casella di controllo, l'utilizzo di memoria di GlassFish Server risulterà ridotto.

  5. Selezionare la casella di controllo DTrace Enabled per abilitare l'utilizzo della risorsa Solaris DTrace.

    Per utilizzare Solaris DTrace è necessario disporre anche di Oracle Enterprise Manager DTrace Monitoring. Questa opzione è disabilitata per impostazione predefinita.

  6. Nella tabella procedere in uno dei seguenti modi.
    • Nella colonna Monitoring Level, selezionare LOW o HIGH dall'elenco a discesa per uno o più componenti della colonna Component, in modo da abilitare il monitoraggio per i componenti corrispondenti.
    • Selezionare le caselle di controllo in corrispondenza di uno o più componenti, quindi selezionare LOW o HIGH nell'elenco a discesa Level nella parte superiore della tabella e fare clic sul pulsante Change Level per impostare il livello di monitoraggio sul valore desiderato per tutti i componenti.

    Nella release corrente i livelli di monitoraggio ottenibili con le opzioni LOW e HIGH non presentano differenze. Per disabilitare il monitoraggio per uno o più componenti, selezionare OFF. Il valore predefinito è OFF.

    Prima di configurare il monitoraggio di un'applicazione, un componente o un servizio, è necessario eseguirne la distribuzione.

  7. Fare clic su Save.
