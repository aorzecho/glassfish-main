Gestisci destinazioni

Nella pagina Manage Targets è possibile modificare i cluster e le istanze server standalone di destinazione in cui un'applicazione può essere abilitata. L'applicazione può essere abilitata solo nelle destinazioni riportate nella colonna Selected Targets.

La pagina Manage Targets contiene le seguenti informazioni.

Available Targets

Cluster e istanze in cui non è distribuita l'applicazione.

Selected Targets

Cluster e istanze in cui è distribuita l'applicazione.

Add

Pulsante che consente di spostare la destinazione selezionata da Available Targets in Selected Targets.

Add All

Pulsante che consente di spostare tutte le destinazioni di Available Targets in Selected Targets.

Remove

Pulsante che consente di spostare la destinazione selezionata da Selected Targets in Available Targets.

Remove All

Pulsante che consente di spostare tutte le destinazioni di Selected Targets in Available Targets.
