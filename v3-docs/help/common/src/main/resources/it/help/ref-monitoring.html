Monitoraggio

Nella pagina Monitoraggio è possibile configurare il monitoraggio e visualizzare i dati di monitoraggio per le istanza server in cluster e non in cluster.

Per ogni istanza server vengono fornite le informazioni riportate di seguito.

Nome istanza

Nome dell'istanza server. Fare clic sul nome istanza per andare alla pagina Informazioni generali per l'istanza.

Nome cluster

Nome del cluster a cui appartiene l'istanza server, se è un'istanza in cluster. Fare clic sul nome cluster per andare alla pagina Informazioni generali per il cluster.

Azione

Collegamenti che consentono di effettuare le operazioni riportate di seguito sulle istanza server.

Configura monitoraggio

Visualizza la pagina Servizio di monitoraggio per la configurazione del cluster o dell'istanza server.

Visualizza dati di monitoraggio

Collegamenti che consentono di visualizzare i tipi di dati di monitoraggio riportati di seguito per le istanza server.

Applicazione

Visualizza la pagina Monitoraggio applicazione per l'istanza server.

Server

Visualizza la pagina Monitoraggio server per l'istanza server.

Risorse

Visualizza la pagina Monitoraggio risorse per l'istanza server.

Task correlati

  • Configurare il servizio di monitoraggio

  • Visualizzare i dati di monitoraggio delle applicazioni

  • Visualizzare i dati di monitoraggio dei server virtuali

  • Visualizzare i dati di monitoraggio delle risorse
