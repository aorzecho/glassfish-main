Registrare Oracle GlassFish Server

Gli utenti registrati di Oracle GlassFish Server possono avvalersi dei seguenti vantaggi:

  • Informazioni su patch e bug

  • Screencast ed esercitazioni

  • Novità ed eventi

  • Offerte di supporto e formazione

Per registrare l'installazione di Oracle GlassFish Server è necessario disporre di un account Sun Online. Se non si dispone di un account Sun Online, è possibile crearne uno durante la registrazione di GlassFish Server.

Nota - In tal caso verranno richieste soltanto le informazioni obbligatorie. Dopo la registrazione di GlassFish Server è possibile fornire informazioni aggiuntive mediante l'aggiornamento dell'account Sun Online.

Prima di cominciare

Se il computer in cui viene eseguito GlassFish Server si connette a Internet mediante un proxy, recuperare il nome host e il numero di porta del proxy. Se nel computer in cui viene eseguito GlassFish Server è in esecuzione un browser Web, è possibile ottenere tali dati dal browser Web. In caso contrario richiederli all'amministratore di sistema.

  1. Nella struttura ad albero, selezionare il nodo Registration.

    Viene visualizzata la pagina Product Registration.

  2. Nella pagina Product Registration immettere le informazioni necessarie.
    • Se si dispone già di un account Sun Online, fornire le informazioni necessarie come indicato di seguito:
      1. Selezionare l'opzione I Already Have a Sun Online Account or a Sun Developer Network (SDN).
      2. Nel campo User Name digitare il nome utente specificato al momento della registrazione per l'account Sun Online.
      3. Nel campo Password digitare la password specificata durante la registrazione per l'account Sun Online.
    • Se non si dispone di un account Sun Online, fornire le seguenti informazioni obbligatorie:
      1. Selezionare l'opzione I Don't Have a Sun Online Account.
      2. Nel campo Email Address, digitare l'indirizzo di posta elettronica.

        L'indirizzo di posta elettronica viene utilizzato come nome utente per l'account Sun Online.

      3. Nel campo Password digitare la password da utilizzare per l'account Sun Online.

        Alla password viene applicata la distinzione tra maiuscole e minuscole; la sua lunghezza deve essere compresa tra 6 e 10 caratteri.

      4. Nel campo Retype Password ridigitare la password.
      5. Nel campo First Name digitare il proprio nome.
      6. Nel campo Last Name digitare il proprio cognome.
      7. Nel campo Company digitare il nome dell'azienda, se necessario.
      8. Nell'elenco a discesa selezionare il paese o la regione di residenza.
  3. Se il computer in cui viene eseguito GlassFish Server si connette a Internet mediante un proxy, specificare il nome host e il numero di porta del proxy.

    Se il computer nel quale viene eseguito GlassFish Server si connette a Internet direttamente, omettere questo passaggio.

    Se nel computer in cui viene eseguito GlassFish Server è disponibile un browser Web, è possibile usarlo per ottenere il nome host e il numero di porta del proxy. In caso contrario richiederli all'amministratore di sistema.

  4. Fare clic su Register.

    Per confermare il completamento della registrazione viene visualizzata la Support page.
