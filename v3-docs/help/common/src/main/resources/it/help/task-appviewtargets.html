Visualizzare le destinazioni di un'applicazione

  1. Nella struttura ad albero, selezionare il nodo Applications.

    Viene visualizzata la pagina Applications.

  2. Selezionare un'applicazione di cui si desidera visualizzare le destinazioni.

    Il nome dell'applicazione può includere un identificatore di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide.

    Viene visualizzata la pagina per l'applicazione.

  3. Selezionare la scheda Target.

    Viene visualizzata la pagina Application Targets con le destinazioni attualmente specificate per l'applicazione.

  4. Per abilitare o disabilitare un'applicazione in un'istanza o in un cluster di destinazione specifico, fare clic sulla casella di controllo accanto alla destinazione e selezionare Enable o Disable dal menu a discesa More Actions.
  5. Per abilitare una destinazione per il bilanciamento del carico in un'istanza o cluster di destinazione specifico, fare clic sulla casella di controllo accanto alla destinazione e selezionare Load Balancer Enable o Load Balancer Disable dal menu a discesa More Actions.
  6. Per associare un'applicazione a server virtuali specifici, selezionare Manage Virtual Servers per l'istanza o cluster di destinazione desiderato.

    Viene visualizzata la pagina Virtual Server Targets.

    Per aggiungere server virtuali, selezionarli dall'elenco Available Targets e utilizzare il pulsante Add per spostarli nell'elenco Selected Targets.

    Per impostazione predefinita, l'associazione del modulo o dell'applicazione viene eseguita con il server virtuale predefinito, denominato server. Utilizzare il pulsante Add All per aggiungere tutte le destinazioni. Utilizzare i pulsanti Remove e Remove All per rimuovere destinazioni dall'elenco Selected Targets.

    Selezionare Save per tornare alla pagina Application Targets.

  7. Per aggiungere o eliminare istanze o cluster per l'applicazione, selezionare Manage Targets.

    Viene visualizzata la pagina Manage Target.

    Per aggiungere destinazioni, selezionarle dall'elenco Available Targets e utilizzare il pulsante Add per spostarle nell'elenco Selected Targets.

    Utilizzare il pulsante Add All per aggiungere tutte le destinazioni. Utilizzare i pulsanti Remove e Remove All per rimuovere destinazioni dall'elenco Selected Targets.

    Selezionare Save per tornare alla pagina Application Targets.

  8. Fare clic su Save.

    L'applicazione è ora disponibile nell'elenco aggiornato delle destinazioni.
