Visualizzare le destinazioni di un'applicazione

  1. Nell'albero di navigazione selezionare il nodo Applicazioni.

    Viene visualizzata la pagina Applicazioni.

  2. Selezionare un'applicazione di cui si desidera visualizzare le destinazioni.

    Il nome dell'applicazione può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata la pagina per l'applicazione.

  3. Selezionare la scheda Destinazione.

    Viene visualizzata la pagina Destinazioni applicazioni con le destinazioni attualmente specificate per l'applicazione.

  4. Per abilitare o disabilitare un'applicazione in un'istanza o in un cluster di destinazione specifico, fare clic sulla casella di controllo accanto alla destinazione e selezionare Abilita o Disabilita dal menu a discesa Altre azioni.

  5. Per abilitare una destinazione per il bilanciamento del carico in un'istanza o cluster di destinazione specifico, fare clic sulla casella di controllo accanto alla destinazione e selezionare Abilitazione load balancer o Disabilitazione load balancer dal menu a discesa Altre azioni.

  6. Per associare un'applicazione a server virtuali specifici, selezionare Gestisci server virtuali per l'istanza o cluster di destinazione desiderato.

    Viene visualizzata la pagina Destinazioni server virtuali.

    Per aggiungere server virtuali, selezionarli dalla lista Destinazioni disponibili e utilizzare il pulsante Aggiungi per spostarli nella lista Destinazioni selezionate.

    Per impostazione predefinita, l'associazione del modulo o dell'applicazione viene eseguita con il server virtuale predefinito, denominato server. Utilizzare il pulsante Aggiungi tutto per aggiungere tutte le destinazioni. Utilizzare i pulsanti Rimuovi e Rimuovi tutto per rimuovere destinazioni dalla lista Destinazioni selezionate.

    Selezionare Save per tornare alla pagina Destinazioni applicazioni.

  7. Per aggiungere o eliminare istanze o cluster per l'applicazione, selezionare Gestisci destinazioni.

    Viene visualizzata la pagina Gestisci destinazione.

    Per aggiungere destinazioni, selezionarle dalla lista Destinazioni disponibili e utilizzare il pulsante Aggiungi per spostarle nella lista Destinazioni selezionate.

    Utilizzare il pulsante Aggiungi tutto per aggiungere tutte le destinazioni. Utilizzare i pulsanti Rimuovi e Rimuovi tutto per rimuovere destinazioni dalla lista Destinazioni selezionate.

    Selezionare Save per tornare alla pagina Destinazioni applicazioni.

  8. Fare clic su Salva.

    L'applicazione è ora disponibile nella lista aggiornata delle destinazioni.
