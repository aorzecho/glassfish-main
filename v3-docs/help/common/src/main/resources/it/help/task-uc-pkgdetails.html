Visualizzare informazioni dettagliate su un componente

In Update Tool sono disponibili informazioni dettagliate sui components , installati, sugli aggiornamenti disponibili per tali componenti e sui add-on components.

  1. Nella struttura ad albero, selezionare il nodo Update Tool.

    Viene visualizzata la pagina Available Add-Ons.

  2. Fare clic sulla scheda corrispondente alla pagina in cui è riportato il componente.

    Il componente è elencato in una delle seguenti pagine:

    • Available Add-Ons

    • Available Updates

    • Installed Components

  3. Nella tabella dei componenti, selezionare il componente.

    Viene visualizzata la pagina Component Details per il componente selezionato.
