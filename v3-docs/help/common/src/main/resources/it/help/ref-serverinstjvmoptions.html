JVM Options

Nella pagina JVM Options è possibile specificare le opzioni del programma di avvio dell'applicazione Java (java) che esegue GlassFish Server. L'opzione -D indica le proprietà specifiche di GlassFish Server.

  • Per modificare un'opzione, è necessario cambiare il valore del campo Value dell'opzione.

  • Per aggiungere un'opzione, fare clic sul pulsante Add JVM Option. Nella riga vuota visualizzata, digitare nel campo Value le informazioni sull'opzione.

    Se nelle informazioni sull'opzione sono presenti uno o più spazi, racchiudere l'intera stringa tra virgolette doppie.

  • Per eliminare un'opzione, selezionare la casella di controllo a sinistra del campo Value dell'opzione da eliminare, quindi fare clic su Delete.
