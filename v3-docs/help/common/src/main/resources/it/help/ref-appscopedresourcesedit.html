Modificare le risorse con ambito applicazione

  1. Nell'albero di navigazione selezionare il nodo Applicazioni.

    Viene visualizzata la pagina Applicazioni.

  2. Selezionare un'applicazione di cui si desidera visualizzare le risorse con ambito applicazione.

    Il nome dell'applicazione può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata la pagina per l'applicazione.

  3. Selezionare la scheda Risorse.

    Viene visualizzata la pagina Risorse con ambito applicazione, e le risorse attualmente specificate per l'applicazione vengono visualizzate in una tabella con le colonne riportate di seguito.

    Nome applicazione

    Nome dell'applicazione.

    Nome modulo

    Il nome del modulo all'interno dell'applicazione.

    Nome risorsa

    Il nome della risorsa con ambito applicazione per l'applicazione o il modulo.

    Tipo di risorsa

    Il tipo di risorsa, ad esempio Connection pool JDBC o Risorse connettore.

  4. Fare clic sul nome della risorsa da modificare.

    La pagina che si apre dipende dal tipo di risorsa. Ad esempio, se la risorsa è un connection pool JDBC, si apre la pagina Modifica connection pool JDBC.
