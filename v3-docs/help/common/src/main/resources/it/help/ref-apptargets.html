Destinazioni applicazione

Nella pagina Destinazioni applicazione è possibile visualizzare i cluster e le istanze server standalone di destinazione in cui l'applicazione può essere abilitata.

La pagina Destinazioni applicazione contiene le informazioni riportate di seguito.

Nome destinazione

Nome del cluster o dell'istanza server standalone.

Abilitato

Viene visualizzato true se l'applicazione è abilitata nella destinazione o false se è disabilitata.

LB abilitato

Viene visualizzato true se l'applicazione è abilitata per il bilanciamento del carico nella destinazione o false se è disabilitata per il bilanciamento del carico.

Server virtuali

Per associare l'applicazione a server virtuali specifici in una destinazione, selezionare Gestisci server virtuali per tale destinazione.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Gestisci destinazioni

Pulsante che consente di gestire le destinazioni dell'applicazione.

Altre azioni

Elenco a discesa delle azioni riportate di seguito.

Abilita

Azione che consente di abilitare l'applicazione nella destinazione selezionata.

Disabilita

Azione che consente di disabilitare l'applicazione nella destinazione selezionata.

Abilitazione load balancer

Azione che consente di abilitare l'applicazione per il bilanciamento del carico nella destinazione selezionata.

Disabilitazione load balancer

Azione che consente di disabilitare l'applicazione per il bilanciamento del carico nella destinazione selezionata.
