Configurare le impostazioni proxy di rete per l'accesso ai repository di rete

Le impostazioni del proxy di rete consentono di specificare la modalità di connessione di GlassFish Server a Internet. Per abilitare Update Tool per l'accesso ai repository di rete mediante Internet, configurare le impostazioni del proxy di rete.

  1. Nella struttura ad albero, selezionare il nodo Update Tool.

    Viene visualizzata la pagina Available Add-Ons.

  2. Nella pagina Available Add-Ons fare clic sulla scheda Configuration.

    Viene visualizzata la pagina Update Tool Configuration.

  3. Nella pagina Update Tool Configuration, specificare la modalità di connessione a Internet del computer in cui è eseguito GlassFish Server.
    • Se il computer si connette a Internet direttamente, selezionare l'opzione Direct Connection.
    • Se il computer si connette a Internet mediante un proxy, selezionare l'opzione Through a Proxy.
  4. Se il computer in cui viene eseguito GlassFish Server si connette a Internet mediante un proxy, specificare il nome del proxy in Proxy Host e il numero della porta in Proxy Port.

    Se il computer nel quale viene eseguito GlassFish Server si connette a Internet direttamente, omettere questo passaggio.

    Se nel computer in cui viene eseguito GlassFish Server è disponibile un browser Web, è possibile usarlo per ottenere il nome host e il numero di porta del proxy. In caso contrario richiederli all'amministratore di sistema.

  5. Fare clic su Save.
