Configurare le impostazioni proxy di rete per l'accesso ai repository di rete

Le impostazioni del proxy di rete consentono di specificare la modalità di connessione di GlassFish Server a Internet. Per abilitare Strumento di aggiornamento per l'accesso ai repository di rete mediante Internet, configurare le impostazioni del proxy di rete.

  1. Nell'albero di navigazione selezionare il nodo Strumento di aggiornamento.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili fare clic sulla scheda Configurazione.

    Viene visualizzata la pagina Configurazione Strumento di aggiornamento.

  3. Nella pagina Configurazione Strumento di aggiornamento, specificare la modalità di connessione a Internet del computer in cui è eseguito GlassFish Server.

    • Se il computer si connette a Internet direttamente, selezionare l'opzione Connessione diretta.

    • Se il computer si connette a Internet mediante un proxy, selezionare l'opzione Tramite un proxy.

  4. Se il computer in cui viene eseguito GlassFish Server si connette a Internet mediante un proxy, specificare il nome del proxy in Host proxy e il numero della porta in Porta proxy.

    Se il computer nel quale viene eseguito GlassFish Server si connette a Internet direttamente, omettere questo passo.

    Se nel computer in cui viene eseguito GlassFish Server è disponibile un browser Web, è possibile usarlo per ottenere il nome host e il numero di porta del proxy. In caso contrario richiederli all'amministratore di sistema.

  5. Fare clic su Salva.
