Eliminare un modulo di audit

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. Nel nodo Configurazione, espandere il nodo Sicurezza.

  4. Nel nodo Sicurezza, selezionare il nodo Moduli di audit.

    Viene visualizzata la pagina Moduli di audit.

  5. Nella pagina Moduli di audit, selezionare la casella di controllo a sinistra del nome del modulo di revisione da eliminare.

  6. Fare clic su Elimina.
