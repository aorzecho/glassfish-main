Audit Modules

Nella pagina Audit Modules è possibile configurare moduli di revisione.

Mediante la revisione, in GlassFish Server vengono registrati eventi significativi, quali errori o violazioni della sicurezza. GlassFish Server registra tutti gli eventi di autenticazione nei file di log. Un log di accesso completo fornisce una traccia sequenziale degli eventi di accesso di GlassFish Server.

Per ogni modulo di revisione vengono fornite le seguenti informazioni.

Name

Nome del modulo di revisione.

Class Name

Nome completo della classe che implementa questo modulo. Il nome della classe per il modulo di revisione predefinito è com.sun.enterprise.security.Audit.

La tabella Modules contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo modulo di revisione.

Delete

Pulsante che consente di eliminare uno o più moduli di revisione selezionati.
