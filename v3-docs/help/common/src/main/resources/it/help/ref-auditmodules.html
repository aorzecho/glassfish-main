Moduli di audit

Nella pagina Moduli di audit è possibile configurare moduli di audit.

L'audit è il processo mediante il quale GlassFish Server registra gli eventi significativi, quali errori o violazioni della sicurezza. GlassFish Server registra tutti gli eventi di autenticazione nei file di log. Un log degli accessi completo fornisce una traccia sequenziale degli eventi di accesso di GlassFish Server.

Per ogni modulo di audit vengono fornite le informazioni riportate di seguito.

Nome

Nome del modulo di audit.

Nome classe

Nome completamente qualificato della classe che implementa questo modulo. Il nome della classe per il modulo di audit predefinito è com.sun.enterprise.security.Audit.

La tabella Moduli contiene anche le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo modulo di audit.

Elimina

Pulsante che consente di eliminare uno o più moduli di audit selezionati.
