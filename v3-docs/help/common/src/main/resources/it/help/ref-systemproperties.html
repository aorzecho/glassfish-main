System Properties

Nella pagina System Properties è possibile visualizzare e gestire le proprietà di GlassFish Server.

Le proprietà di sistema consentono di definire un valore comune per un'impostazione a livello di server. È possibile fare riferimento a una proprietà di sistema in un campo di testo Admin Console racchiudendola tra parentesi graffe abbinate a un simbolo del dollaro. Ad esempio per fare riferimento a una proprietà denominata prop-name, utilizzare la sintassi ${prop-name}.

Per ogni proprietà, la pagina System Properties contiene le seguenti informazioni.

Name

Nome della proprietà.

Value

Valore della proprietà.

Description

Descrizione opzionale della proprietà.

Numerose proprietà di sistema sono predefinite e non vengono visualizzate nella pagina. Per dettagli, vedere Predefined System Properties.
