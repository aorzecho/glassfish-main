Proprietà di sistema

Nella pagina Proprietà di sistema è possibile visualizzare e gestire le proprietà di GlassFish Server.

Le proprietà di sistema consentono di definire un valore comune per un'impostazione a livello di server. È possibile fare riferimento a una proprietà di sistema in un campo di testo della console di amministrazione racchiudendola tra parentesi graffe abbinate a un simbolo del dollaro. Ad esempio, per fare riferimento a una proprietà denominata prop-name, utilizzare la sintassi${prop-name}.

La pagina Proprietà di sistema contiene le informazioni riportate di seguito per ogni proprietà.

Nome

Nome della proprietà.

Valore

Valore della proprietà.

Descrizione

Descrizione opzionale della proprietà.

Alcune proprietà di sistema sono predefinite, ma non vengono visualizzate in questa pagina. Per informazioni dettagliate, vedere Proprietà di sistema predefinite.
