Monitoraggio risorse

Nella pagina Monitoraggio risorse è possibile visualizzare i dati di monitoraggio per una risorsa selezionata.

La pagina Monitoraggio risorse contiene le informazioni riportate di seguito.

Nome istanza

Nome dell'istanza server.

Connection pool

Connection pool per il quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Chapter 8, Administering the Monitoring Service, in Oracle GlassFish Server Administration Guide.
