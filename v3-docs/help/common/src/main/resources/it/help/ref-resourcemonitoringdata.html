Resource Monitoring

Nella pagina Resource Monitoring è possibile visualizzare i dati di monitoraggio per una risorsa selezionata. Per una descrizione delle proprietà monitorate, vedere Capitolo 8, Administering the Monitoring Service in Oracle GlassFish Server 3.1 Administration Guide.

Attività correlate

  • Configurare il servizio di monitoraggio

  • Visualizzare i dati di monitoraggio delle applicazioni

  • Visualizzare i dati di monitoraggio dei server

  • Visualizzare i dati di monitoraggio delle risorse

Comandi asadmin correlati

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
