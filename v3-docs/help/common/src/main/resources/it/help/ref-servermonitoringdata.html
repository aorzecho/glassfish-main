Monitoraggio server

Nella pagina Monitoraggio server è possibile visualizzare i dati di monitoraggio per l'istanza server.

La pagina Monitoraggio server contiene le informazioni riportate di seguito.

Nome istanza

Nome dell'istanza server.

Visualizza

Server virtuale, listener HTTP, Garbage Collector o sistema di thread per il quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Chapter 8, Administering the Monitoring Service, in Oracle GlassFish Server Administration Guide.
