Componenti aggiuntivi disponibili

La pagina Componenti aggiuntivi disponibili fornisce informazioni sui componenti dell'immagine di installazione non ancora installati.

La pagina Componenti aggiuntivi disponibili contiene le opzioni riportate di seguito.

Percorso di installazione

Percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Installa

Pulsante che consente di installare o più componenti aggiuntivi selezionati.

Componenti aggiuntivi disponibili

Per ogni componente dell'immagine di installazione non ancora installato vengono fornite le informazioni riportate nella tabella seguente.

Componente

Nome del componente.

Categoria

Categoria del software a cui è associato il componente, ad esempio application server, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente quando viene creato.

Versione

Numero di versione del package in cui viene fornito il componente.

Dimensioni installazione

Quantità di spazio su disco usata per l'installazione del componente.

Origine

Nome del Repository di rete da cui è stato ottenuto il componente.

Task correlati
