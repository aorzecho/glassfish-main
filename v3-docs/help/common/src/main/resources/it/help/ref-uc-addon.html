Available Add-Ons

Nella pagina Available Add-Ons sono disponibili informazioni sui components dell'installation image che non sono ancora stati installati.

La pagina Available Add-Ons contiene le seguenti opzioni.

Installation Path

Percorso del file system in cui si trova la directory root o di livello principale dell'immagine di installazione.

Install

Pulsante che consente di installare o più componenti add-on selezionati.

Available Add-Ons

Per ogni componente dell'immagine di installazione che non è ancora stato installato, vengono fornite le informazioni riportate nella tabella seguente.

Component

Nome del componente.

Category

Categoria del software a cui è associato il componente, ad esempio server di applicazioni, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente al momento della creazione.

Version

Numero della versione del pacchetto con cui viene fornito il componente.

Install Size

Quantità di spazio su disco utilizzata per l'installazione del componente.

Source

Nome del Repository di rete da cui ottenere il componente.
