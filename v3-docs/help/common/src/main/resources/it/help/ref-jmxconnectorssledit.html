SSL

Nella pagina SSL è possibile modificare le impostazioni SSL per il connettore JMX.

Le impostazioni specificate in questa pagina sono valide solo se la sicurezza è abilitata per il connettore JMX nella pagina Modifica connettore JMX.

La pagina SSL contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

SSL3

Se questa casella di controllo è selezionata, per il connettore JMX è abilitato il protocollo SSL3. Questa opzione è abilitata per impostazione predefinita.

TLS

Se questa casella di controllo è selezionata, per il connettore JMX è abilitato il protocollo TLS. Questa opzione è abilitata per impostazione predefinita.

Autenticazione client

Se questa casella di controllo è selezionata, i client devono identificarsi a ogni richiesta del server. Questa opzione è disabilitata per impostazione predefinita.

Pseudonimo certificato

Pseudonimo del certificato del server nel database di certificati o del token PKCS#11. Nel certificato il formato del nome è nometoken:pseudonimo. L'inclusione della parte nometoken: del nome in questo attributo è facoltativa.

Keystore

Nome del file keystore, ad esempio keystore.jks .

Algoritmo di protezione

Nome dell'algoritmo di gestione della protezione, ad esempio PKIX, da usare per la convalida del percorso di certificazione.

Lunghezza massima certificato

Numero massimo di certificati intermedi non autoemessi che possono esistere in un percorso di certificazione. Questo campo viene utilizzato solo se il campo Algoritmo di protezione è impostato su PKIX. Il valore 0 indica che il percorso può contenere un solo certificato. Il valore -1 indica che la lunghezza del percorso non è vincolata, ovvero non esiste un valore massimo. Se si imposta un valore inferiore a -1, viene restituita un'eccezione.

Truststore

Nome del file truststore, ad esempio cacerts.jks.

Suite di cifratura

Area in cui è possibile aggiungere o rimuovere suite di cifratura. Se non si aggiungono suite di cifratura, verranno utilizzate tutte quelle disponibili.
