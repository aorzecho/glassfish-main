Modificare un settore

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i settori.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.

    Viene visualizzata la pagina Realms.

  4. Selezionare il nome di un settore esistente.

    Viene visualizzata la pagina Edit Realm.

  5. Per modificare il nome della classe per il settore, attenersi a una delle seguenti procedure:
    • Dall'elenco a discesa Nome classe, selezionare il nome della classe per il settore da creare.
    • Selezionare il secondo pulsante di scelta e digitare il nome della classe per il settore personalizzato da creare.
  6. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    Le proprietà valide variano in base al tipo di settore.

    • Proprietà specifiche della classe FileRealm

    • Proprietà specifiche della classe CertificateRealm

    • Proprietà specifiche della classe JDBCRealm

    • Proprietà specifiche della classe LDAPRealm

    • Proprietà specifiche della classe SolarisRealm

    • Proprietà specifiche della classe PamRealm

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  7. Fare clic su Save.
