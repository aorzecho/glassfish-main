Modificare un realm

  1. Nel nodo Configurazioni, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i realm.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

    Viene visualizzata la pagina Realm.

  4. Selezionare il nome di un realm esistente.

    Viene visualizzata la pagina Modifica realm.

  5. Per modificare il nome della classe per il realm, attenersi a una delle seguenti procedure:

    • Dall'elenco a discesa Nome classe, selezionare il nome della classe per il realm da creare.

    • Selezionare il secondo pulsante di scelta e digitare il nome della classe per il realm personalizzato da creare.

  6. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    Le proprietà valide variano in base al tipo di realm.

    • Proprietà specifiche della classe FileRealm

    • Proprietà specifiche della classe CertificateRealm

    • Proprietà specifiche della classe JDBCRealm

    • Proprietà specifiche della classe LDAPRealm

    • Proprietà specifiche della classe SolarisRealm

    • Proprietà specifiche della classe PamRealm

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  7. Fare clic su Salva.
