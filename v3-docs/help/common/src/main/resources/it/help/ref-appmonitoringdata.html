Monitoraggio applicazioni

Nella pagina Monitoring applicazioni è possibile visualizzare i dati di monitoraggio per un'applicazione selezionata.

La pagina Monitoraggio applicazioni contiene le informazioni riportate di seguito.

Nome istanza

Nome dell'istanza server.

Applicazione

Applicazione per la quale vengono visualizzate le statistiche di monitoraggio.

Componente

Componente dell'applicazione per la quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Chapter 8, Administering the Monitoring Service, in Oracle GlassFish Server Administration Guide.

  • Visualizzare i dati di monitoraggio dei server virtuali

  • Visualizzare i dati di monitoraggio delle risorse
