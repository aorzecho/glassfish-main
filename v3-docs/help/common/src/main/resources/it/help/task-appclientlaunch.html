Avviare un client dell'applicazione

Prima di cominciare

Verificare che il client dell'applicazione da avviare sia abilitato. Deve trattarsi di un client Web che è possibile eseguire con Java Web Start.

  1. Nella struttura ad albero, selezionare il nodo Applications.

    Viene visualizzata la pagina Applications.

  2. Dalla tabella Deployed Applications, fare clic sul nome del client dell'applicazione da avviare.

    Il nome può includere un identificatore di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide.

    Viene visualizzata la pagina Edit Application.

  3. Verificare che la casella di controllo Java Web Start sia selezionata.

    Java Web Start deve essere abilitato prima di avviare un client dell'applicazione. Se la casella di controllo Java Web Start non è selezionata, selezionarla, quindi fare clic su Save.

  4. Nella tabella Modules and Components, fare clic su Launch nella colonna Action.

    Viene visualizzata la pagina Launch Application Client.

  5. Nella pagina Launch Application Client, digitare gli argomenti da passare al client dell'applicazione.
  6. Fare clic su Launch.
