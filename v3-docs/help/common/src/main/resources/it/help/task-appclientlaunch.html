Avviare un client dell'applicazione

Prima di iniziare

Verificare che il client dell'applicazione da avviare sia abilitato. Deve trattarsi di un client Web che è possibile eseguire con Java Web Start.

  1. Nell'albero di navigazione selezionare il nodo Applicazioni.

    Viene visualizzata la pagina Applicazioni.

  2. Dalla tabella Applicazioni distribuite, fare clic sul nome del client dell'applicazione da avviare.

    Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata la pagina Modifica applicazione.

  3. Verificare che la casella di controllo Java Web Start sia selezionata.

    Java Web Start deve essere abilitato prima di avviare un client dell'applicazione. Se la casella di controllo Java Web Start non è selezionata, selezionarla, quindi fare clic su Save.

  4. Nella tabella Moduli e componenti, fare clic su Avvia nella colonna Azione.

    Viene visualizzata la pagina Avvia client applicazione.

  5. Nella pagina Avvia client applicazione, digitare gli argomenti da passare al client dell'applicazione.

  6. Fare clic su Avvia.
