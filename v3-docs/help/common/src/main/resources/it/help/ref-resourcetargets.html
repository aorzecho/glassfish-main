Destinazioni risorse

Nella pagina Resource Targets è possibile visualizzare i cluster e le istanze server standalone di destinazione in può essere abilitata la risorsa.

Per ogni destinazione, la pagina Resource Targets contiene le seguenti informazioni.

Target Name

Nome del cluster o dell'istanza server standalone.

Enabled

Viene visualizzato un segno di spunta se la risorsa è abilitata nella destinazione o una X se è disabilitata.

La tabella Targets contiene inoltre le seguenti opzioni.

Manage Targets

Pulsante che consente di gestire le destinazioni della risorsa.

Enable

Pulsante che consente di abilitare la risorsa nella destinazione selezionata.

Disable

Azione che consente di disabilitare la risorsa nella destinazione selezionata.
