Destinazioni risorse

Nella pagina Destinazioni risorse è possibile visualizzare i cluster e le istanze server standalone di destinazione in cui può essere abilitata la risorsa.

Per ogni destinazione la pagina Destinazioni risorse contiene le informazioni riportate di seguito.

Nome destinazione

Nome del cluster o dell'istanza server standalone.

Abilitato

Visualizza un segno di spunta se la risorsa è abilitata nella destinazione o una X se è disabilitata.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Gestisci destinazioni

Pulsante che consente di gestire le destinazioni delle risorse.

Abilita

Pulsante che consente di abilitare la risorsa nella destinazione selezionata.

Disabilita

Azione che consente di disabilitare la risorsa nella destinazione selezionata.
