Modificare un modulo di revisione

I moduli di revisione non sono attivati per impostazione predefinita. Per ulteriori informazioni sull'attivazione dei moduli di revisione, vedere Abilitare o disabilitare il log di revisione.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, espandere il nodo Security.
  3. Nel nodo Security, selezionare il nodo Audit Modules.

    Viene visualizzata la pagina Audit Modules.

  4. Nella pagina Audit Modules fare clic sul nome del modulo di revisione da modificare.

    Viene visualizzata la pagina Edit Audit Module.

  5. Nella pagina Edit Audit Module digitare un nuovo valore nel campo Class Name per modificare il nome della classe.
  6. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    Per la descrizione delle proprietà aggiuntive disponibili per i moduli di revisione, vedere Proprietà specifiche dei moduli di revisione.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  7. Fare clic su Save.
