Modificare un modulo di audit

I moduli di audit non sono attivati per impostazione predefinita. Per ulteriori informazioni su come attivare i moduli di audit, vedere Abilitare o disabilitare il log di audit.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. Nel nodo Configurazione, espandere il nodo Sicurezza.

  4. Nel nodo Sicurezza, selezionare il nodo Moduli di audit.

    Viene visualizzata la pagina Moduli di audit.

  5. Nella pagina Moduli di audit fare clic sul nome del modulo di revisione da modificare.

    Viene visualizzata la pagina Modifica modulo di audit.

  6. Nella pagina Modifica modulo di audit digitare un nuovo valore nel campo Nome classe per modificare il nome della classe.

  7. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    Per una descrizione delle proprietà aggiuntive disponibili per i moduli di audit, vedere Proprietà specifiche dei moduli di audit.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  8. Fare clic su Salva.
