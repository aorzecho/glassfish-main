Proprietà di sistema

Nella pagina Proprietà di sistema viene visualizzata una lista delle proprietà di sistema Java definite nella configurazione denominata selezionata. Le proprietà di sistema Java vengono passate al launcher dell'applicazione Java mediante l'opzione -D del launcher dell'applicazione Java all'avvio di GlassFish Server.

Queste proprietà definiscono i valori predefiniti delle impostazioni delle porte per tutte le istanze GlassFish Server che fanno riferimento alla configurazione.

Per informazioni sulle impostazioni predefinite delle porte in una configurazione, vedere Proprietà di sistema predefinite.

Per ogni proprietà vengono visualizzate le informazioni riportate di seguito.

Nome variabile istanza

Nome della proprietà di sistema.

Valore predefinito

Valore impostato per la proprietà nella configurazione denominata a cui fa riferimento l'istanza. Questo campo è di sola lettura.

Valori istanza

Collegamento alla pagina Valori istanza per la proprietà.

La tabella Proprietà aggiuntive contiene inoltre le opzioni riportate di seguito.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il valore predefinito corrispondente, oppure se non è specificato il valore predefinito, il valore resta non definito.

La pagina Proprietà di sistema contiene inoltre le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Riconfigurazione dinamica

Se questa opzione è abilitata, le modifiche apportate alla configurazione vengono applicate alle istanze che fanno riferimento ad essa senza che sia necessario riavviare le istanze.

Cluster e/o istanze che utilizzano la configurazione

Lista dei cluster e delle istanze che fanno riferimento alla configurazione con un'indicazione relativa all'eventuale esecuzione di ciascun cluster o istanza:

  • Se il cluster o l'istanza è in esecuzione, viene visualizzato un segno di spunta (✓).

  • Se il cluster o l'istanza è arrestata, viene visualizzato un punto esclamativo (!).
