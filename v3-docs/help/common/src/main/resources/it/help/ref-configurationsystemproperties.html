System Properties

Nella pagina System Properties viene visualizzato un elenco delle proprietà di sistema Java definite nella configurazione denominata selezionata. Le proprietà di sistema Java vengono passate al programma di avvio dell'applicazione Java tramite l'opzione -D del programma di avvio dell'applicazione Java quando viene avviato GlassFish Server.

Tali proprietà definiscono i valori predefiniti delle impostazioni delle porte per tutte le istanze GlassFish Server che fanno riferimento alla configurazione.

Per informazioni sulle impostazioni predefinite delle porte in una configurazione, vedere Predefined System Properties .

Per ogni proprietà vengono visualizzate le seguenti informazioni:

Instance Variable Name

Nome della proprietà di sistema.

Default Value

Valore impostato per la proprietà nella configurazione con nome a cui fa riferimento l'istanza. Campo di sola lettura.

Instance Values

Collegamento alla pagina Valori istanza per la proprietà.

La tabella Additional Properties contiene inoltre le seguenti opzioni.

Add Property

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Additional Properties.

Delete Properties

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il rispettivo valore predefinito, oppure, se non è specificato il valore predefinito, il valore resta non definito.

La pagina System Properties contiene inoltre le seguenti opzioni:

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Dynamic Reconfiguration

Se questa opzione è abilitata, le modifiche apportate alla configurazione vengono applicate alle istanze che fanno riferimento ad essa senza che sia necessario riavviare le istanze.

Clusters and/or Instances Using This Configuration

Elenco dei cluster e delle istanze che fanno riferimento alla configurazione con indicazione relativa all'eventuale esecuzione di ciascun cluster o istanza:

  • Se il cluster o l'istanza è in esecuzione, viene visualizzato un segno di spunta (✓).

  • Se il cluster o l'istanza è arrestata, viene visualizzato un punto esclamativo (!).

Attività correlate

  • Impostare i valori predefiniti per le proprietà di configurazione
