Proprietà specifiche dei moduli di revisione

Per un modulo di revisione è disponibile la seguente priorità.

auditOn

Un valore true comporta il caricamento del modulo di revisione e assicura che venga richiamato dalla libreria di revisione di GlassFish Server in corrispondenza dei punti di revisione. Un valore false disabilita la registrazione nel log di revisione per il modulo.
