Proprietà specifiche delle configurazioni di provider di sicurezza dei messaggi

Per una configurazione di provider di sicurezza dei messaggi sono disponibili le seguenti proprietà.

server.config

Directory e nome file di un file XML che contiene informazioni relative alla configurazione del server. Il valore predefinito è domain-dir/config/wss-server-config-1.0.xml.

debug

Se impostato su true, consente di eseguire il dump dei messaggi di debug del provider server nel log del server. Il valore predefinito è false.

dynamic.username.password

Se impostato su true, comunica al runtime del provider di recuperare nome utente e password da CallbackHandler per ogni richiesta. Se impostato su false, nome utente e password di wsse:UsernameToken(s) vengono recuperati una volta sola, durante l'inizializzazione del modulo. Questa proprietà è applicabile solo per un ClientAuthModule. Il valore predefinito è false.

encryption.key.alias

Specifica la chiave di codifica utilizzata dal provider. La chiave viene identificata dal rispettivo alias keystore. Il valore predefinito è s1as.

signature.key.alias

Specifica la chiave di firma utilizzata dal provider. La chiave viene identificata dal rispettivo alias keystore. Il valore predefinito è s1as.
