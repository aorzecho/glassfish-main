Configurazione di Update Tool

Nella pagina Configuration sono disponibili informazioni sulla configurazione di Update Tool e al suo interno è possibile modificare la modalità di connessione di Update Tool a Internet. Per modificare altri aspetti della configurazione di Update Tool, utilizzare l'applicazione standalone updatetool disponibile in as-install-parent/bin .

La pagina Update Tool contiene le seguenti opzioni.

Installation Path

Percorso del file system in cui si trova la directory root o di livello principale dell'immagine di installazione.

Preferred Repository

Nome del network repository preferito associato a questa immagine di installazione, ad esempio glassfish.org.

Internet Connection

Modalità di connessione di GlassFish Server a Internet:

Direct Connection

Questa opzione consente di specificare che GlassFish Server venga connesso direttamente a Internet.

Through a Proxy

Questa opzione consente di specificare l'uso di un server proxy per connettere GlassFish Server a Internet tramite. Se l'opzione è selezionata, a questa pagina vengono aggiunti i campi Proxy Host e Proxy Port.

Proxy Host

Nome host del server proxy per il computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Proxy Port

Numero della porta del server proxy per i computer in cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer in cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Repositories

Per ogni network repository associato a questa immagine di installazione, vengono fornite le informazioni riportate nella tabella seguente:

Repository

Nome del repository, ad esempio dev.glassfish.org.

Attività correlate
