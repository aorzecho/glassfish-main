Configurazione di Strumento di aggiornamento

La pagina Configurazione contiene informazioni sulla configurazione di Strumento di aggiornamento e consente di modificare la modalità di connessione di Strumento di aggiornamento a Internet. Per modificare altri aspetti della configurazione di Strumento di aggiornamento, utilizzare l'applicazione standalone updatetool disponibile in as-install-parent/bin.

La pagina Strumento di aggiornamento contiene le opzioni riportate di seguito.

Percorso di installazione

Percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Repository preferito

Nome del repository di rete preferito associato a questa immagine di installazione, ad esempio glassfish.org.

Connessione Internet

Modalità di connessione di GlassFish Server a Internet:

Connessione diretta

Questa opzione specifica che GlassFish Server viene connesso direttamente a Internet.

Tramite un proxy

Questa opzione specifica l'uso di un server proxy per la connessione di GlassFish Server a Internet. Se questa opzione è selezionata, a questa pagina vengono aggiunti i campi Host proxy e Porta proxy.

Host proxy

Nome host del server proxy per il computer su cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer su cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Porta proxy

Numero di porta del server proxy per il computer su cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer su cui è in esecuzione GlassFish Server è connesso a Internet tramite un server proxy.

Repository

Per ogni repository di rete associato a questa immagine di installazione vengono fornite le informazioni riportate nella tabella seguente.

Repository

Nome del repository, ad esempio dev.glassfish.org.
