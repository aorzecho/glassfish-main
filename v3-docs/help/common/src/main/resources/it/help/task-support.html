Visualizzare la pagina GlassFish News per GlassFish Server

Nella pagina GlassFish News è disponibile una serie di collegamenti a informazioni aggiuntive su Oracle GlassFish Server, quali newsletter, articoli, suggerimenti tecnici, forum di discussione e blog.

  1. Nella struttura ad albero, selezionare il nodo Common Tasks.

    Viene visualizzata la pagina Common Tasks.

  2. Sotto GlassFish News, fare clic sul collegamento GlassFish News.

    La pagina GlassFish News viene aperta in una scheda o finestra distinta del browser.
