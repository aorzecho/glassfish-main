Configurazioni

Nella pagina Configurations è possibile gestire le configurazioni e visualizzare le istanze server standalone e i cluster che le utilizzano.

In GlassFish Server la configurazione default-config è predefinita. La configurazione server-config viene creata automaticamente per il server di amministrazione del dominio (DAS) al momento della creazione del dominio. Per ulteriori informazioni, vedere About Named Configurations in Oracle GlassFish Server 3.1 High Availability Administration Guide .

Per ogni configurazione vengono fornite le seguenti informazioni.

Name

Nome della configurazione.

Instance

Elenco delle istanze standalone e dei cluster che utilizzano la configurazione.

La tabella Configurations contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova configurazione.

Delete

Pulsante che consente di eliminare una o più configurazioni selezionate.
