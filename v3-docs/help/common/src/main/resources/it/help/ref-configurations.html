Configurazioni

Nella pagina Configurazioni è possibile gestire le configurazioni e visualizzare le istanze server standalone e i cluster che utilizzano le configurazioni.

La configurazione default-config è predefinita in GlassFish Server. La configurazione server-config viene creata automaticamente per il server di amministrazione del dominio (DAS) quando si crea il dominio. Per ulteriori informazioni, consultare About Named Configurations in Oracle GlassFish Server High Availability Administration Guide.

Per ogni configurazione vengono fornite le informazioni riportate di seguito.

Nome

Nome della configurazione.

Istanza

Lista delle istanze standalone e dei cluster che utilizzano la configurazione.

La tabella Configurations contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione.

Elimina

Pulsante che consente di eliminare una o più configurazioni selezionate.
