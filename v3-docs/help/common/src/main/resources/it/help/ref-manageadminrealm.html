Admin Users

Nella pagina Admin Users è possibile gestire utenti admin-realm. Il settore admin-realm è un settore file. Utenti e gruppi del settore file vengono elencati nel file di chiave, la cui posizione è specificata dalla proprietà file.

Inizialmente, dopo l'installazione di GlassFish Server, l'unico utente è l'amministratore che ha eseguito l'accesso durante l'installazione. Per impostazione predefinita tale utente fa parte del gruppo asadmin, nel settore admin-realm, ossia dispone dei diritti per la modifica di GlassFish Server. Qualsiasi utente assegnato a questo gruppo disporrà dei privilegi di amministratore, ossia avrà accesso allo strumento asadmin e ad Administration Console.

Nella pagina Admin Users viene visualizzato un elenco di utenti per il settore selezionato. Per ogni utente vengono fornite le seguenti informazioni:

User ID

Nome dell'utente.

Group List

Gruppi di appartenenza dell'utente.

La tabella Admin Users contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo utente.

Delete

Pulsante che consente di eliminare uno o più utenti selezionati.

Attività correlate

  • Aggiungere un utente al settore Admin

  • Modificare le informazioni utente per il settore Admin

  • Eliminare un utente dal settore Admin
