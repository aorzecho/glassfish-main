Predefined System Properties

Le seguenti proprietà di sistema sono predefinite per GlassFish Server.

com.sun.aas.installRoot

Directory in cui è installato GlassFish Server.

com.sun.aas.instanceRoot

Directory di primo livello per un dominio GlassFish Server.

com.sun.aas.hostName

Nome del computer host.

com.sun.aas.javaRoot

Directory di installazione per il runtime Java.

com.sun.aas.imqLib

Directory della libreria per il software Oracle Message Queue.

com.sun.aas.domainName

Nome del dominio. Questa proprietà non viene utilizzata nella configurazione predefinita, ma può essere utilizzata per personalizzare la configurazione. Il valore predefinito è domain1.
