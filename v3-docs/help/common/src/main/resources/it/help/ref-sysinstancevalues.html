Valori istanza

Nella pagina Valori istanza è possibile modificare i valori di una proprietà di configurazione di sistema per le istanze di GlassFish Server che fanno riferimento alla configurazione denominata selezionata.

Nella pagina Valori istanza viene visualizzata la lista delle istanze che fanno riferimento alla configurazione selezionata, in cui viene impostata la proprietà.

Nome istanza

Nome assegnato all'istanza quando è stata creata. Se si fa clic su tale nome, viene visualizzata la pagina Informazioni generali per l'istanza.

Nome cluster

Nome del cluster a cui appartiene l'istanza, se applicabile. Questo campo è di sola lettura.

Valore predefinito

Valore impostato per la proprietà nella configurazione denominata a cui fa riferimento l'istanza. Questo campo è di sola lettura.

Valore di sostituzione

Valore della proprietà impostato per l'istanza selezionata. Questo valore sostituisce il valore predefinito.
