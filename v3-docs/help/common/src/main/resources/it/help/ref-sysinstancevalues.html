Valori istanza

Nella pagina Instance Values è possibile modificare i valori di una proprietà di sistema della configurazione per le istanze di GlassFish Server che fanno riferimento alla configurazione denominata selezionata.

Nella pagina Instance Values viene visualizzato l'elenco delle istanze che fanno riferimento alla configurazione selezionata, in cui viene impostata la proprietà.

Instance Name

Nome assegnato all'istanza quando l'istanza è stata creata. Se si fa clic su tale nome, viene visualizzata la pagina General Information per l'istanza.

Cluster Name

Nome del cluster a cui appartiene l'istanza, se applicabile. Campo di sola lettura.

Default Value

Valore impostato per la proprietà nella configurazione con nome a cui fa riferimento l'istanza. Campo di sola lettura.

Override Value

Valore della proprietà impostato per l'istanza selezionata. Questo valore sovrascrive il valore predefinito.
