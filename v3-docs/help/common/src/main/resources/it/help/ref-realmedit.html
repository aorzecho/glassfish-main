Edit Realm

Nella pagina Edit Realm è possibile creare un settore.

La pagina Edit Realm contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui vengono applicate le impostazioni in questa pagina. Il campo è di sola lettura.

Realm Name

Nome del settore. Il campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo settore.

Class Name

Nome della classe per il settore da modificare.

Additional Properties

Proprietà aggiuntive per il settore. Le proprietà valide variano in base al tipo di settore.

  • Proprietà specifiche della classe FileRealm

  • Proprietà specifiche della classe CertificateRealm

  • Proprietà specifiche della classe JDBCRealm

  • Proprietà specifiche della classe LDAPRealm

  • Proprietà specifiche della classe SolarisRealm

  • Proprietà specifiche della classe PamRealm
