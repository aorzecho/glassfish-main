Visualizzare i file di log per il DAS

  1. Nella struttura ad albero, selezionare il nodo server (Admin Server).

    Viene visualizzata la pagina General Information.

  2. Nella pagina General Information, fare clic sul pulsante View Log Files.

    Viene visualizzata la finestra Log Viewer.
