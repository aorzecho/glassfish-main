New Administrator Password

Nella pagina New Administrator Password è possibile modificare la password per l'account di amministratore corrente.

La pagina New Administrator Password contiene le seguenti opzioni.

User ID

ID utente dell'amministratore corrente. Il campo è di sola lettura.

Group List

Gruppo di appartenenza dell'amministratore. Il campo è di sola lettura. Il valore è asadmin.

New Password

Nuova password dell'amministratore.

Confirm New Password

Nuova password dell'amministratore.

Attività correlata

  • Modificare la password dell'amministratore

Comando asadmin correlato

  • change-admin-password(1)
