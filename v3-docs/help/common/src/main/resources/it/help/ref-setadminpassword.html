Nuova password amministratore

Nella pagina Nuova password amministratore è possibile modificare la password per l'account di amministratore corrente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile modificare la password in un valore vuoto.

La pagina Nuova password amministratore contiene le opzioni riportate di seguito.

ID utente

ID utente dell'amministratore corrente. Questo campo è di sola lettura.

Lista gruppi

Gruppo di appartenenza dell'amministratore. Questo campo è di sola lettura. Il valore è asadmin.

Nuova password

Nuova password dell'amministratore.

Conferma nuova password

Nuova password dell'amministratore.

Task correlato

Modificare la password dell'amministratore

Comando asadmin correlato

change-admin-password(1)
