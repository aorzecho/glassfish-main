Visualizzare un elenco di applicazioni distribuite

  1. Nella struttura ad albero, selezionare il nodo Applications.

    Viene visualizzata la pagina Applications. Nella tabella sono elencate le applicazioni distribuite.

  2. Selezionare un tipo di modulo dall'elenco Filter per filtrare l'elenco delle applicazioni in base al tipo di modulo.
