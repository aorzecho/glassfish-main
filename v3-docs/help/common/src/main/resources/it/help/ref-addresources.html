Aggiungi risorse

Nella pagina Aggiungi risorse è possibile creare una o più risorse da un file glassfish-resources.xml nelle destinazioni specificate.

La pagina Aggiungi risorse contiene le informazioni riportate di seguito.

Posizione

Posizione del file glassfish-resources.xml:

File XML da caricare nel server

Percorso del file su un computer remoto.

File XML locale accessibile da GlassFish Server

Percorso del file sul computer server di amministrazione del dominio (DAS).

Destinazione

Un elenco a discesa dei cluster e delle istanze server stand-alone.

Task correlati

Aggiungere risorse configurate in un file XML

Comandi asadmin correlati

add-resources(1)
