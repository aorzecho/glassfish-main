Configurare le impostazioni del profiler JVM

Uno strumento profiler genera dati utilizzabili per analizzare le prestazioni e identificare potenziali colli di bottiglia.

  1. Nella struttura ad albero, selezionare il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo JVM Settings.

    Viene visualizzata la pagina JVM General Settings.

  3. Nella pagina JVM General Settings, fare clic sulla scheda Profiler.

    Viene visualizzata la pagina JVM Profiler Settings.

  4. Specificare le impostazioni del profiler.

    Le informazioni specificate dipendono dal prodotto profiler utilizzato. Per esempi e istruzioni, vedere il Capitolo 3, Debugging Applications in Oracle GlassFish Server 3.1 Application Development Guide.

  5. Fare clic su Save.
  6. Riavviare il server.
