Configurare le impostazioni del profiler JVM

Uno strumento profiler genera dati utilizzabili per analizzare le prestazioni e identificare potenziali colli di bottiglia.

  1. Nell'albero di navigazione selezionare il nodo Configurazione.

  2. Select the configuration that you are changing.

  3. Nel nodo Configurazione, selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM, fare clic sulla scheda Profiler.

    Viene visualizzata la pagina Impostazioni profiler JVM.

  5. Specificare le impostazioni del profiler.

    Le informazioni specificate dipendono dal prodotto profiler utilizzato. Per esempi e istruzioni, vedere il Chapter 3, Debugging Applications, in Oracle GlassFish Server Application Development Guide.

  6. Fare clic su Salva.

  7. Riavviare il server.
