Eliminare un modulo del ciclo di vita

  1. Nella struttura ad albero, selezionare il nodo Lifecycle Modules.

    Viene visualizzata la pagina Lifecycle Modules.

  2. Nella pagina Lifecycle Modules, selezionare la casella di controllo a sinistra del nome del modulo del ciclo di vita da eliminare.
  3. Fare clic su Delete.
