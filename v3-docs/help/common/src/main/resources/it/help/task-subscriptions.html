Visualizzare la pagina Sottoscrizioni per GlassFish Server

  1. Nell'albero di navigazione selezionare il nodo Task comuni.

    Viene visualizzata la pagina Task comuni.

  2. Sotto Notizie su GlassFish, fare clic sul collegamento Sottoscrizioni.

    La pagina Web delle sottoscrizioni di GlassFish viene aperta in una scheda o finestra distinta del browser.
