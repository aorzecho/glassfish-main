Visualizzare la pagina Subscriptions per GlassFish Server

  1. Nella struttura ad albero, selezionare il nodo Common Tasks.

    Viene visualizzata la pagina Common Tasks.

  2. Sotto GlassFish News, fare clic sul collegamento Subscriptions.

    La pagina Web GlassFish Subscriptions viene aperta in una scheda o finestra distinta del browser.
