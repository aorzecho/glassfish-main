Informazioni sull'endpoint del servizio Web

Nella pagina Informazioni sull'endpoint del servizio Web è possibile visualizzare le informazioni relative a un endpoint del servizio Web.

La pagina Informazioni sull'endpoint del servizio Web contiene le informazioni riportate di seguito.

Nome applicazione

Nome dell'applicazione distribuita. Fare clic sul collegamento o sul pulsante Indietro per tornare alla pagina Modifica applicazione.

Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

Tester

Il collegamento Tester è visibile solo se il tipo di distribuzione dell'applicazione è diverso da RI, ad esempio se è 109. Fare clic su questo collegamento per visualizzare una pagina che consente di fornire i parametri per eseguire il test dei metodi del servizio Web.

WSDL

WSDL (Web Services Definition Language) per l'endpoint. Fare clic su questo collegamento per visualizzare il file WSDL in una nuova scheda o finestra del browser.

Quando si fa clic sul collegamento WSDL, alcuni browser Web tentano di interpretare l'origine WSDL visualizzando quindi una pagina vuota. Per visualizzare l'origine WSDL con tali browser, utilizzare il comando del browser per la visualizzazione dell'origine nella pagina vuota.

Nome endpoint

Nome dell'endpoint.

Nome di servizio

Nome del servizio Web.

Nome porta

Nome della porta del servizio Web.

Tipo di distribuzione

Tipo di distribuzione del servizio Web, ad esempio RI o 109.

Tipo di implementazione

Tipo di distribuzione del servizio Web, ad esempio SERVLET o EJB.

Nome classe di implementazione

Nome della classe di implementazione del servizio Web.

URI indirizzo endpoint

URI dell'indirizzo dell'endpoint.

Spazio di nomi

Spazio di nomi di destinazione associato al servizio Web.

Descrizione

Descrizione dell'endpoint, se esistente.
