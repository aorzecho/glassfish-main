Web Service Endpoint Information

Nella pagina Web Service Endpoint Information è possibile visualizzare le informazioni relative a un endpoint del servizio Web.

La pagina Web Service Endpoint Information contiene le seguenti informazioni.

Application Name

Nome dell'applicazione distribuita. Fare clic sul collegamento o sul pulsante Back per tornare alla pagina Edit Application.

Il nome può includere un identificatore di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide.

Tester

Il collegamento Tester è visibile solo se il Deployment Type dell'applicazione è diverso da RI (per esempio, se è 109). Fare clic su questo collegamento per visualizzare una pagina in cui è possibile specificare parametri per testare metodi di servizio Web.

WSDL

WSDL (Web Services Definition Language) per l'endpoint. Fare clic su questo collegamento per visualizzare il file WSDL in una nuova scheda o finestra del browser.

Quando si fa clic sul collegamento WSDL, alcuni browser Web tentano di interpretare l'origine WSDL visualizzando quindi una pagina vuota. Per visualizzare l'origine WSDL con tali browser, utilizzare il comando del browser per la visualizzazione dell'origine nella pagina vuota.

Endpoint Name

Nome dell'endpoint.

Service Name

Nome del servizio Web.

Port Name

Nome della porta del servizio Web.

Deployment Type

Tipo di distribuzione del servizio Web (ad esempio, RI o 109).

Implementation Type

Tipo di distribuzione del servizio Web (ad esempio, SERVLET o EJB).

Implementation Class Name

Nome della classe di implementazione del servizio Web.

Endpoint Address URI

URI dell'indirizzo dell'endpoint.

Namespace

Namespace di destinazione associato al servizio Web.

Description

Descrizione dell'endpoint, se esistente.
