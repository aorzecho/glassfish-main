Abilitare l'amministrazione sicura

  1. Nell'albero di navigazione selezionare il nodo server (server di amministrazione).

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali, fare clic sul pulsante Amministrazione sicura.

    Viene visualizzata la pagina Amministrazione sicura

  3. Fare clic su Abilita amministrazione sicura.

    Viene visualizzato un messaggio che indica la necessità di riavviare DAS per abilitare l'amministrazione sicura.

  4. Fare clic su OK.

    L'amministrazione sicura viene abilitata e viene visualizzata la pagina di riavvio del server.
