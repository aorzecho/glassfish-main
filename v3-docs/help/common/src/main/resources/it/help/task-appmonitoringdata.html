Visualizzare i dati di monitoraggio delle applicazioni

Prima di cominciare

Per visualizzare i dati di monitoraggio è necessario configurare il monitoraggio. Per informazioni dettagliate, vedere Configurare il servizio di monitoraggio.

  1. Nella struttura ad albero, selezionare il nodo server (Admin Server).

    Viene visualizzata la pagina General Information.

  2. Nella pagina General Information, fare clic sulla scheda Monitor.

    Viene visualizzata la pagina Application Monitoring.

  3. Nell'elenco a discesa View, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.
  4. Fare clic sul pulsante Refresh per aggiornare i dati di monitoraggio visualizzati.
