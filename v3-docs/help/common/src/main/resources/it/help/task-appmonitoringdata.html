Visualizzare i dati di monitoraggio delle applicazioni

Prima di iniziare

Per visualizzare i dati di monitoraggio è necessario configurare il monitoraggio. Per ulteriori dettagli, vedere Configurare il servizio di monitoraggio.

  1. Nell'albero di navigazione, selezionare il nodo Dati di monitoraggio.

    Viene visualizzata la pagina Monitoraggio.

  2. Nella colonna Visualizza dati di monitoraggio fare clic sul collegamento Applicazione corrispondente all'istanza del server da monitorare.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  3. Nell'elenco a discesa Applicazione, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.

  4. Nell'elenco a discesa Componente, selezionare il componente dell'applicazione per il quale si desidera visualizzare i dati di monitoraggio.

  5. Fare clic sul pulsante Aggiorna per aggiornare i dati di monitoraggio visualizzati.

Procedura alternativa:

  1. Nell'albero di navigazione selezionare il nodo server (server di amministrazione).

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali, fare clic sulla scheda Monitoraggio.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  3. Nell'elenco a discesa Applicazione, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.

  4. Nell'elenco a discesa Componente, selezionare il componente dell'applicazione per il quale si desidera visualizzare i dati di monitoraggio.

  5. Fare clic sul pulsante Aggiorna per aggiornare i dati di monitoraggio visualizzati.
