Visualizzare le informazioni generali per DAS

  1. Nell'albero di navigazione selezionare il nodo server (server di amministrazione).

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali, eseguire una delle operazioni seguenti.

    • Consultare la visualizzazione in sola lettura Stato per verificare che il server sia in esecuzione.

    • Consultare la visualizzazione in sola lettura Tempo di attività per visualizzare il numero di ore e minuti di attività continua del server.

    • Fare clic sul pulsante Arresta per arrestare GlassFish Server.

    • Fare clic sul pulsante Riavvia per arrestare e riavviare GlassFish Server.

    • Fare clic sul pulsante Visualizza file di log per visualizzare i file di log per i cluster o le istanze GlassFish Server.

      Per ulteriori dettagli, vedere Visualizzare file di log per DAS.

    • Fare clic sul pulsante Visualizza log raw per visualizzare i file di log raw per DAS.

      Per ulteriori dettagli, vedere Visualizzare i file di log raw per DAS.

    • Fare clic su JVM Report per aprire una finestra distinta in cui è possibile visualizzare i report relativi alla Java Virtual Machine, tra cui un report di riepilogo, un report sulla gestione della memoria, un report sul caricamento delle classi e un dump del thread corrente.
