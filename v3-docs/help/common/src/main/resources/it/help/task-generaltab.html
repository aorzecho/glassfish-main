Visualizzare informazioni generali per il DAS

  1. Nella struttura ad albero, selezionare il nodo server (Admin Server).

    Viene visualizzata la pagina General Information.

  2. Nella pagina General Information, eseguire una delle operazioni seguenti.
    • Fare clic sul pulsante Stop per arrestare GlassFish Server.
    • Fare clic sul pulsante Restart per arrestare e riavviare GlassFish Server.
    • Fare clic sul pulsante View Log Files per visualizzare i file di log per i cluster o le istanze GlassFish Server.

      Per i dettagli, vedere Visualizzare i file di log per il DAS.

    • Fare clic su JVM Report per aprire una finestra distinta in cui è possibile visualizzare i rapporti relativi alla macchina virtuale Java, tra cui un rapporto di riepilogo, un rapporto sulla gestione della memoria, un rapporto sul caricamento delle classi e un dump del thread corrente.
