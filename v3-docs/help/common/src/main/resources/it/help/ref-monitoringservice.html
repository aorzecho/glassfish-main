Monitoring Service

Nella pagina Monitoring Service è possibile configurare le opzioni di monitoraggio per componenti o servizi server.

Prima di configurare il monitoraggio di un'applicazione, un componente o un servizio, è necessario eseguirne la distribuzione.

La pagina Monitoring Service contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui vengono applicate le impostazioni in questa pagina. Il campo è di sola lettura.

Monitoring Service

Se la casella di controllo Monitoring Service Enabled è selezionata, il monitoraggio è abilitato per GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

Monitoring MBeans

Se la casella di controllo Monitoring MBeans Enabled è selezionata, tutti i MBeans necessari per il monitoraggio vengono distribuiti in GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

Se la casella di controllo non è selezionata non è possibile visualizzare i dati di monitoraggio in Administration Console, anche se è selezionata la casella di controllo Monitoring Service. In tal caso per visualizzare i dati di monitoraggio è necessario utilizzare la riga di comando. Se si deseleziona la casella di controllo, l'utilizzo di memoria di GlassFish Server risulterà ridotto.

DTrace

Se la casella di controllo DTrace Enabled è selezionata e si dispone di Oracle Enterprise Manager DTrace Monitoring, è possibile utilizzare la risorsa Solaris DTrace. Questa opzione è disabilitata per impostazione predefinita.

Per ogni componente disponibile, vengono fornite le seguenti informazioni.

Component

Nome del componente GlassFish Server.

Monitoring Level

Livello di monitoraggio per il componente. Per ogni componente, i livelli disponibili sono LOW, HIGH e OFF. Il livello predefinito è OFF. Nel release corrente i livelli di monitoraggio ottenibili con le opzioni LOW e HIGH non presentano differenze.

La tabella Component Level Settings contiene le seguenti opzioni.

Change Level

Pulsante che consente di cambiare livello di monitoraggio per uno o più componenti selezionati.

Level

Elenco a discesa da cui è possibile selezionare un livello di monitoraggio da applicare a tutti i componenti selezionati.

Attività correlate

  • Visualizzare i dati di monitoraggio delle applicazioni

  • Visualizzare i dati di monitoraggio dei server

  • Visualizzare i dati di monitoraggio delle risorse
