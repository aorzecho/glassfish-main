Servizio di monitoraggio

Nella pagina Servizio di monitoraggio è possibile configurare le opzioni di monitoraggio per singoli componenti o servizi server.

Prima di configurare il monitoraggio di un'applicazione, di un componente o di un servizio, è necessario eseguirne la distribuzione.

La pagina Servizio di monitoraggio contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Servizio di monitoraggio

Se la casella di controllo Servizio di monitoraggio è selezionata, il monitoraggio è abilitato per GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

MBean di monitoraggio

Se la casella di controllo MBean di monitoraggio abilitati è selezionata, tutti gli MBean necessari per il monitoraggio vengono distribuiti in GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

Se la casella di controllo non è selezionata, non è possibile visualizzare i dati di monitoraggio nella Console di amministrazione, anche se è selezionata la casella di controllo Servizio di monitoraggio. In tal caso, per visualizzare i dati di monitoraggio è necessario utilizzare la riga di comando. Se si deseleziona la casella di controllo, l'utilizzo di memoria di GlassFish Server risulterà ridotto.

DTrace

Se la casella di controllo DTrace abilitato è selezionata e si dispone di Oracle Enterprise Manager DTrace Monitoring, è possibile utilizzare la risorsa Solaris DTrace. Questa opzione è disabilitata per impostazione predefinita.

Per ogni componente disponibile vengono fornite le informazioni riportate di seguito.

Componente

Nome del componente GlassFish Server.

Livello di monitoraggio

Livello di monitoraggio impostato per il componente. Per ogni componente i livelli disponibili sono LOW, HIGH e OFF. Il livello predefinito è OFF. Nella release corrente i livelli di monitoraggio ottenibili con le opzioni LOW e HIGH non presentano differenze.

La tabella Impostazioni a livello di componente contiene le opzioni riportate di seguito.

Cambia livello

Pulsante che consente di cambiare livello di monitoraggio per uno o più componenti selezionati.

Livello

Elenco a discesa da cui è possibile selezionare un livello di monitoraggio da applicare a tutti i componenti selezionati.
