Abilitare o disabilitare il log di revisione

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Security.

    Viene visualizzata la pagina Security.

  3. Nella pagina Security selezionare la casella di controllo Audit Logging Enabled per abilitare il log di revisione.

    Per attivare il log di revisione è inoltre necessario impostare su true la proprietà auditOn relativa al modulo di revisione attivo.

  4. Fare clic su Save.
