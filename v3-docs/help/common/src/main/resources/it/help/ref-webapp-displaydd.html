Descrittore di distribuzione

Nella pagina Descrittore di distribuzione è possibile visualizzare il descrittore di distribuzione per un modulo.

La pagina Descrittore di distribuzione contiene le informazioni riportate di seguito.

Nome applicazione

Nome dell'applicazione.

Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

Nome modulo

Nome del modulo che contiene il descrittore di distribuzione.

Nome file descrittore

Nome del file descrittore di distribuzione.
