Destinazioni modulo di ciclo di vita

Nella pagina Lifecycle Module Targets è possibile visualizzare i cluster e le istanze server standalone di destinazione in cui può essere abilitato il modulo del ciclo di vita.

La pagina Lifecycle Module Targets contiene le seguenti informazioni.

Target Name

Nome del cluster o dell'istanza server standalone.

Enabled

Viene visualizzato true se il modulo del ciclo di vita è abilitato nella destinazione o false se è disabilitato.

La tabella Targets contiene inoltre le seguenti opzioni.

Manage Targets

Pulsante che consente di gestire le destinazioni del modulo del ciclo di vita.

More Actions

Elenco a discesa delle seguenti azioni.

Enable

Azione che consente di abilitare il modulo del ciclo di vita nella destinazione selezionata.

Disable

Azione che consente di disabilitare il modulo del ciclo di vita nella destinazione selezionata.
