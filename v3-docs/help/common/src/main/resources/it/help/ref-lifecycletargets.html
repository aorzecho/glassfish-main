Destinazioni moduli del ciclo di vita

Nella pagina Destinazioni moduli del ciclo di vita è possibile visualizzare i cluster e le istanze server standalone di destinazione in cui può essere abilitato il modulo del ciclo di vita.

La pagina Destinazioni moduli del ciclo di vita contiene le informazioni riportate di seguito.

Nome destinazione

Nome del cluster o dell'istanza server standalone.

Abilitato

Viene visualizzato true se il modulo del ciclo di vita è abilitato nella destinazione o false se è disabilitato.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Gestisci destinazioni

Pulsante che consente di gestire le destinazioni dei moduli del ciclo di vita.

Altre azioni

Elenco a discesa delle azioni riportate di seguito.

Abilita

Azione che consente di abilitare il modulo del ciclo di vita nella destinazione selezionata.

Disabilita

Azione che consente di disabilitare il modulo del ciclo di vita nella destinazione selezionata.
