Ricaricare un'applicazione distribuita

Quando si ricarica un'applicazione, questa viene semplicemente riavviata senza essere ridistribuita e vengono mantenute le impostazioni e altre opzioni con cui l'applicazione era stata distribuita.

  1. Nell'albero di navigazione selezionare il nodo Applicazioni.

    Viene visualizzata la pagina Applicazioni.

  2. Nella tabella Applicazioni distribuite, fare clic su Reload nella riga corrispondente all'applicazione da ridistribuire.

    Il nome dell'applicazione può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata nuovamente la pagina Applicazioni, con un messaggio che indica se il ricaricamento è stato completato.
