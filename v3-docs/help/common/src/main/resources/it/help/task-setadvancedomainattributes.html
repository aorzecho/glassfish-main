Impostare gli attributi di dominio

  1. Nella struttura ad albero, selezionare il nodo GlassFish Server.

    Viene visualizzata la pagina General Information.

  2. Nella pagina General Information, fare clic sulla scheda Advanced.

    Viene visualizzata la pagina Applications Configuration.

  3. Nella pagina Applications Configuration, fare clic sulla scheda Domain Attributes.

    Viene visualizzata la pagina Domain Attributes.

  4. Nel campo Application Root, digitare il percorso completo della directory in cui si trovano le applicazioni distribuite.

    Il valore predefinito è domain-dir/applications.

  5. Nel campo Log Root, digitare il percorso completo della directory in cui si trovano i file di log del server.

    Il valore predefinito è domain-dir/logs.

  6. Lasciare vuoto il campo Locale per utilizzare la versione locale predefinita dell'host.

    Per versione locale si intende un identificatore che specifica una combinazione particolare di lingua e regione. Ad esempio la versione locale per l'inglese americano è en_US, mentre per il giapponese è ja_JP. Per utilizzare una versione locale non inglese, è necessario che sia stata eseguita la localizzazione di GlassFish Server, un processo che prevede anche la traduzione dall'inglese a un'altra lingua.

  7. Fare clic su Save.
  8. Riavviare il server.
