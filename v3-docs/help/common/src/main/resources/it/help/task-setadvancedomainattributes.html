Impostare gli attributi del dominio

  1. Nell'albero di navigazione selezionare il nodo Dominio.

    Viene visualizzata la pagina Attributi dominio.

  2. Nel campo Radice applicazioni, digitare il percorso completo della directory in cui risiedono le applicazioni distribuite.

    Il valore predefinito è domain-dir/applications.

  3. Nel campo Radice log, digitare il percorso completo della directory in cui si trovano i file di log del server.

    Il valore predefinito è domain-dir/logs.

  4. Lasciare vuoto il campo Impostazioni nazionali per utilizzare la versione locale predefinita dell'host.

    Per versione locale si intende un identificativo che specifica una combinazione particolare di lingua e regione. Ad esempio le impostazioni nazionali per l'inglese americano sono en_US mentre per il giapponese sono ja_JP. Per utilizzare una versione locale non inglese, è necessario che sia stata eseguita la localizzazione di GlassFish Server, un processo che prevede anche la traduzione dall'inglese a un'altra lingua.

  5. Fare clic su Salva.

  6. Riavviare il server.
