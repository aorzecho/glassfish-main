Aggiornamenti disponibili

Nella pagina Aggiornamenti disponibili sono descritte informazioni sulle versioni più aggiornate dei componenti dell'immagine di installazione installati.

Per installare gli aggiornamenti disponibili, arrestare innanzitutto il dominio di GlassFish Server, quindi utilizzare uno dei comandi riportati di seguito.

  • glassfish3/bin/updatetool, che avvia la utility grafica Strumento di aggiornamento.

  • glassfish3/bin/pkg, una versione della riga di comando dello strumento di aggiornamento.

Per informazioni sull'utilizzo del comando pkg, vedere il Chapter 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. Per la utility Strumento di aggiornamento è disponibile una Guida in linea specifica.

Al termine dell'installazione degli aggiornamenti, riavviare il server.

La pagina Aggiornamenti disponibili contiene le opzioni riportate di seguito.

Percorso di installazione

Percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Aggiornamenti disponibili

Per ogni componente per cui è disponibile una versione più recente, vengono fornite le informazioni riportate nella tabella seguente.

Componente

Nome del componente.

Categoria

Categoria del software a cui è associato il componente, ad esempio application server, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente quando viene creato.

Versione installata

Numero di versione del package in cui viene fornito il componente corrente.

Nuova versione

Numero di versione dell'aggiornamento disponibile.

Modifica

Differenza tra le dimensioni della versione installata e quelle dell'aggiornamento disponibile.

Origine

Nome del Repository di rete da cui è stato ottenuto il componente.
