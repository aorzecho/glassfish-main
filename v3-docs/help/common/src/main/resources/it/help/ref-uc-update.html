Available Updates

Nella pagina Available Updates sono disponibili informazioni sulle versioni più aggiornate dei components dell'installation image installati.

Prima di installare gli aggiornamenti disponibili, arrestare il dominio di GlassFish Server, quindi utilizzare uno dei seguenti comandi:

  • glassfish3/bin/updatetool, che avvia l'utility grafica Update Tool

  • glassfish3/bin/pkg, una versione dell'utility Update Tool basata su riga di comando

Per informazioni sull'utilizzo del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Per l'utility Update Tool è disponibile una Guida in linea specifica.

Una volta completata l'installazione degli aggiornamenti, riavviare il server.

La pagina Available Updates contiene le seguenti opzioni.

Installation Path

Percorso del file system in cui si trova la directory root o di livello principale dell'immagine di installazione.

Available Updates

Per ogni componente per cui è disponibile una versione più recente, vengono fornite le informazioni riportate nella tabella seguente.

Component

Nome del componente.

Category

Categoria del software a cui è associato il componente, ad esempio server di applicazioni, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente al momento della creazione.

Installed Version

Numero della versione del pacchetto con cui viene fornito il componente corrente.

New Version

Numero di versione dell'aggiornamento disponibile.

Change

Differenza tra le dimensioni della versione installata e quelle dell'aggiornamento disponibile.

Source

Nome del Repository di rete da cui ottenere il componente.
