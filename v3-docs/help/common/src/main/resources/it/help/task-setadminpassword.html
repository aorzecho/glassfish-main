Modificare la password dell'amministratore

  1. Nella struttura ad albero, selezionare il nodo GlassFish Server.

    Viene visualizzata la pagina General Information.

  2. Selezionare la scheda Administrator Password.

    Viene visualizzata la pagina New Administrator Password.

    Nella pagina sono riportati l'ID utente e il nome gruppo dell'account dell'amministratore corrente. Per impostazione predefinita, l'ID utente è admin e il nome del gruppo è asadmin.

  3. Nel campo New Password digitare la nuova password per l'account dell'amministratore corrente.
  4. Nel campo Confirm New Password ridigitare la nuova password.
  5. Fare clic su Save.
