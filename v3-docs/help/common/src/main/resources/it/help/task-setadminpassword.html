Modificare la password dell'amministratore

  1. Nell'albero di navigazione, selezionare il nodo Dominio.

    Viene visualizzata la pagina Attributi dominio.

  2. Selezionare la scheda Password amministratore.

    Viene visualizzata la pagina Nuova password amministratore.

    Nella pagina sono riportati l'ID utente e il nome gruppo dell'account dell'amministratore corrente. Per impostazione predefinita, l'ID utente è admin e il nome del gruppo è asadmin.

  3. Nel campo Nuova password digitare la nuova password per l'account dell'amministratore corrente.

  4. Nel campo Conferma nuova password ridigitare la nuova password.

  5. Fare clic su Salva.
