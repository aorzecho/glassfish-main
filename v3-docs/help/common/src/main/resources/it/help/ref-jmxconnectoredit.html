Modifica connettore JMX

Nella pagina Modifica connettore JMX è possibile modificare il connettore JMX conforme allo standard JSR 360.

La pagina Modifica connettore JMX contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome del connettore JMX. Questo campo è di sola lettura.

Protocollo JMX

Nome del protocollo supportato dal connettore JMX. Questo campo è di sola lettura.

JMXServiceURL

URL del servizio JMX. Questo campo è di sola lettura.

Sicurezza

Se la casella di controllo Abilitato è selezionata, la comunicazione JMX viene cifrata. Questa opzione è disabilitata per impostazione predefinita.

Indirizzo

Indirizzo IP del servizio di denominazione in cui è registrato lo stub del server del connettore JMX. Non si tratta della porta del socket server che esegue la comunicazione JMX effettiva, bensì dell'indirizzo dell'interfaccia di rete in cui viene avviato il registro RMI. Se il sistema dispone di più interfacce di rete, modificare questo valore in modo che venga selezionata solo un'interfaccia specifica. Il valore predefinito è 0.0.0.0.

Porta

Numero della porta su cui il servizio di denominazione (registro RMI) ascolta le connessioni al client RMI. Questo servizio di denominazione viene utilizzato esclusivamente per lo scaricamento degli stub RMI. Se la porta predefinita è occupata, viene utilizzata una porta libera. I valori validi sono compresi tra 1 e 65535. Sui sistemi UNIX, la creazione di socket che restano in ascolto sulle porte comprese tra 1 e 1024 richiede privilegi di utente privilegiato. Il valore predefinito è 8686.

Nome realm

Nome che rappresenta il realm amministrativo speciale, in cui vengono gestite tutte le operazioni di autenticazione. Il valore predefinito è admin-realm .

Proprietà aggiuntive

Proprietà aggiuntive per il connettore JMX. Alcune proprietà sono specificate per impostazione predefinita.
