Edit JMX Connector

Nella pagina Edit JMX Connector è possibile modificare il connettore JMX conforme allo standard JSR 360.

La pagina Edit JMX Connector contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui vengono applicate le impostazioni in questa pagina. Il campo è di sola lettura.

Name

Nome del connettore JMX. Il campo è di sola lettura.

JMX Protocol

Nome del protocollo supportato dal connettore JMX. Il campo è di sola lettura.

JMXServiceURL

URL del servizio JMX. Il campo è di sola lettura.

Security

Se è selezionata la casella di controllo Enabled, la comunicazione JMX viene crittografata. Questa opzione è disabilitata per impostazione predefinita.

Address

Indirizzo IP del servizio di denominazione in cui è registrato lo stub del server del connettore JMX. Non si tratta della porta del socket server che esegue la comunicazione JMX effettiva, bensì dell'indirizzo dell'interfaccia di rete in cui viene avviato il registro RMI. Se il sistema è dotato di più interfacce di rete, è possibile modificare questo valore in modo da selezionare solo un'interfaccia specifica. Il valore predefinito è 0.0.0.0.

Port

Numero della porta su cui il servizio di denominazione (registro RMI) rileva le connessioni del client RMI. L'uso del servizio di denominazione è esclusivamente destinato al download degli stub RMI. Se la porta predefinita è occupata, viene utilizzata una porta libera. Sono consentiti valori compresi tra 1 e 65535. In sistemi UNIX, per creare socket in attesa su porte comprese tra 1 e 1024 sono necessari privilegi di superutente. Il valore predefinito è 8686.

Realm Name

Nome che rappresenta il settore amministrativo specifico in cui viene interamente gestita l'autenticazione. Il valore predefinito è admin-realm .

Additional Properties

Proprietà aggiuntive per il connettore JMX. Alcune proprietà sono specificate per impostazione predefinita.
