Provider Configurations

Nella pagina Provider Configurations è possibile specificare configurazioni di provider di sicurezza dei messaggi per il livello di autenticazione SOAP o HttpServlet.

Per impostazione predefinita vengono configurati quattro provider per il livello di autenticazione SOAP.

Per ogni configurazione di provider vengono fornite le seguenti informazioni.

Provider ID

Nome della configurazione di provider.

Provider Type

Tipo di provider; può essere uno qualsiasi dei seguenti:

client

Specifica che il provider è il provider di autenticazione client.

server

Specifica che il provider è il provider di autenticazione server.

client-server

Specifica che il provider è un provider di autenticazione sia client sia server.

Default Provider

Se impostato su true, il provider è il provider predefinito.

Class Name

Nome della classe di implementazione Java del provider.

La tabella Provider Configurations contiene le seguenti opzioni.

New

Pulsante che consente di creare una nuova configurazione di provider.

Delete

Pulsante che consente di eliminare una o più configurazioni di provider selezionate.
