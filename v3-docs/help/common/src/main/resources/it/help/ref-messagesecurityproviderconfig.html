Configurazioni provider

Nella pagina Configurazioni provider è possibile specificare configurazioni del provider di sicurezza dei messaggi per il layer di autenticazione SOAP o HttpServlet.

Per impostazione predefinita, vengono configurati quattro provider per il layer di autenticazione SOAP.

Per ogni configurazione di provider vengono fornite le informazioni riportate di seguito.

ID provider

Nome della configurazione del provider.

Tipo di provider

Il tipo di provider può essere uno dei seguenti:

client

Specifica che il provider è il provider di autenticazione client.

server

Specifica che il provider è il provider di autenticazione server.

client-server

Specifica che il provider è un provider di autenticazione sia client sia server.

Provider predefinito

Se impostato su true, il provider è il provider predefinito.

Nome classe

Nome della classe di implementazione Java del provider.

La tabella Configurazioni provider contiene le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione di provider.

Elimina

Pulsante che consente di eliminare una o più configurazioni di provider selezionate.
