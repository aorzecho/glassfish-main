Visualizzatore log raw

Nella pagina Visualizzatore log raw è possibile visualizzare i file di log raw per il server di amministrazione (denominato server).

La pagina Visualizzatore log raw contiene le opzioni riportate di seguito.

Istanza

Istanza server o cluster per cui vengono visualizzati i record del log. L'unica opzione è server.

Per ogni messaggio, nella pagina Visualizzatore log raw vengono visualizzate le informazioni riportate di seguito.

Indicatore orario

Data e ora di esecuzione del messaggio di log.

Numero record

Numero di messaggi nel log del server.

Livello di log

Livello di log del messaggio.

Versione

Versione di GlassFish Server.

Logger

Modulo del logger che ha generato il messaggio di log.

Coppie nome-valore

Nome del thread e ID del messaggio di log.

Messaggio

Breve testo del messaggio di log. Fare clic sul collegamento (dettagli) per visualizzare la versione formattata di un messaggio.
