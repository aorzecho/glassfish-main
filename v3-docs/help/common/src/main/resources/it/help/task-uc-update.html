Visualizzare gli aggiornamenti disponibili per i componenti

Nella pagina Aggiornamenti disponibili sono riportate informazioni sulle versioni più aggiornate dei componenti dell'immagine di installazione installati.

  1. Nell'albero di navigazione selezionare il nodo Strumento di aggiornamento.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili fare clic sulla scheda Aggiornamenti disponibili.

    Viene visualizzata la pagina Aggiornamenti disponibili. Se non è disponibile nessun aggiornamento, la tabella è vuota.

Passi successivi

Prima di installare gli aggiornamenti disponibili, arrestare il dominio di GlassFish Server, quindi utilizzare uno dei seguenti comandi:

  • as-install-parent/bin/updatetool, che avvia la utility grafica Strumento di aggiornamento

  • as-install-parent/bin/pkg, una versione da riga di comando dello Strumento di aggiornamento

Per informazioni sull'utilizzo del comando pkg, vedere il Chapter 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. Per la utility Strumento di aggiornamento è disponibile una Guida in linea specifica.

Una volta completata l'installazione degli aggiornamenti, riavviare il server.
