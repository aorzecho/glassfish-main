Visualizzare gli aggiornamenti disponibili per i componenti

Nella pagina Available Updates sono disponibili informazioni sulle versioni più aggiornate dei components dell'installation image installati.

  1. Nella struttura ad albero, selezionare il nodo Update Tool.

    Viene visualizzata la pagina Available Add-Ons.

  2. Nella pagina Available Add-Ons fare clic sulla scheda Available Updates.

    Viene visualizzata la pagina Available Updates. Se non è disponibile nessun aggiornamento, la tabella è vuota.

Procedura successiva

Prima di installare gli aggiornamenti disponibili, arrestare il dominio di GlassFish Server, quindi utilizzare uno dei seguenti comandi:

  • as-install-parent/bin/updatetool, che avvia l'utility grafica Update Tool

  • as-install-parent/bin/pkg, una versione a riga di comando dell'utility Update Tool

Per informazioni sull'utilizzo del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide. Per l'utility Update Tool è disponibile una Guida in linea specifica.

Una volta completata l'installazione degli aggiornamenti, riavviare il server.
