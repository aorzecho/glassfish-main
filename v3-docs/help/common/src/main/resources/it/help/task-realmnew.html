Creare un realm

GlassFish Server è preconfigurato con tre realm: file, certificate e admin-realm. È inoltre possibile creare realm ldap, JDBC, solaris, PAM e personalizzati. In genere in un server è disponibile un realm di ciascun tipo. Tuttavia in GlassFish Server sono presenti due realm file: file e admin-realm. Si tratta di due realm dello stesso tipo utilizzati per due scopi differenti. È inoltre possibile disporre di un database dei certificati distinto per ciascun server virtuale del sistema.

  1. Nel nodo Configurazioni, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i realm.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

    Viene visualizzata la pagina Realm.

  4. Nella pagina Realm fare clic su Nuovo.

    Viene visualizzata la pagina Nuovo realm.

  5. Nel campo Nome, digitare un nome per il realm.

  6. Per specificare il nome della classe per il realm, attenersi a una delle seguenti procedure:

    • Dall'elenco a discesa Nome classe, selezionare il nome della classe per il realm da creare.

      Nella tabella seguente sono riportate le opzioni valide.

      Nome realm Nome classe
      file com.sun.enterprise.security.auth.realm.file.FileRealm
      certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • Selezionare il pulsante di scelta inferiore e digitare il nome della classe per il realm personalizzato da creare.

  7. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    Le proprietà valide variano in base al tipo di realm.

    • Proprietà specifiche della classe FileRealm

    • Proprietà specifiche della classe CertificateRealm

    • Proprietà specifiche della classe JDBCRealm

    • Proprietà specifiche della classe LDAPRealm

    • Proprietà specifiche della classe SolarisRealm

    • Proprietà specifiche della classe PamRealm

  8. Fare clic su OK.
