Creare un settore

GlassFish Server è preconfigurato con tre settori: file, certificate e admin-realm. È inoltre possibile creare settori ldap, JDBC, solaris, PAM e personalizzati. In genere in un server è disponibile un settore di ciascun tipo. Tuttavia in GlassFish Server sono presenti due settori di file: file e admin-realm. Si tratta di due settori dello stesso tipo utilizzati per due scopi differenti. È inoltre possibile disporre di un database dei certificati distinto per ciascun server virtuale del sistema.

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i settori.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.

    Viene visualizzata la pagina Realms.

  4. Nella pagina Realms fare clic su New.

    Viene visualizzata la pagina New Realm.

  5. Nel campo Name, digitare un nome per il settore.
  6. Per specificare il nome della classe per il settore, attenersi a una delle seguenti procedure:
    • Dall'elenco a discesa Nome classe, selezionare il nome della classe per il settore da creare.

      Nella tabella seguente sono riportate le opzioni valide.

      Nome Realm

      Nome classe

      file

      com.sun.enterprise.security.auth.realm.file.FileRealm

      certificate

      com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

      ldap

      com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

      JDBC

      com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

      solaris

      com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

      PAM

      com.sun.enterprise.security.auth.realm.solaris.PamRealm
    • Selezionare il pulsante di scelta inferiore e digitare il nome della classe per il settore personalizzato da creare.
  7. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

    Le proprietà valide variano in base al tipo di settore.

    • Proprietà specifiche della classe FileRealm

    • Proprietà specifiche della classe CertificateRealm

    • Proprietà specifiche della classe JDBCRealm

    • Proprietà specifiche della classe LDAPRealm

    • Proprietà specifiche della classe SolarisRealm

    • Proprietà specifiche della classe PamRealm

  8. Fare clic su OK.
