Edit Audit Module

Nella pagina Edit Audit Module è possibile modificare un modulo di revisione.

La pagina Edit Audit Module contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Name

Nome del modulo di revisione. Il campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo modulo di revisione.

Class Name

Nome completo della classe che implementa questo modulo. Il nome della classe per il modulo di revisione predefinito è com.sun.enterprise.security.Audit.

Additional Properties

Proprietà aggiuntive per il modulo di revisione. Per la descrizione delle proprietà aggiuntive disponibili per i moduli di revisione, vedere Proprietà specifiche dei moduli di revisione.
