Modifica modulo di audit

Nella pagina Modifica modulo di audit è possibile modificare un modulo di audit.

La pagina Modifica modulo di audit contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome del modulo di audit. Questo campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo modulo di audit.

Nome classe

Nome completamente qualificato della classe che implementa questo modulo. Il nome della classe per il modulo di audit predefinito è com.sun.enterprise.security.Audit.

Proprietà aggiuntive

Proprietà aggiuntive per il modulo di audit. Per la descrizione delle proprietà aggiuntive disponibili per i moduli di audit, vedere Proprietà specifiche dei moduli di audit.
