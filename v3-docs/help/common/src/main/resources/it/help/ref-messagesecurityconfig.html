Configurazioni di sicurezza dei messaggi

Nella pagina Configurazioni di sicurezza dei messaggi è possibile visualizzare e modificare le configurazioni disponibili per i provider di sicurezza dei messaggi.

Per configurare la sicurezza dei messaggi è necessario avere una conoscenza di base delle nozioni fondamentali dei servizi Web. Per ulteriori informazioni, vedere Chapter 3, Administering Message Security, in Oracle GlassFish Server Security Guide e Chapter 6, Administering Web Applications, in Oracle GlassFish Server Administration Guide.

GlassFish Server implementa la sicurezza dei messaggi mediante provider di sicurezza dei messaggi integrati nel layer di elaborazione SOAP. I provider di sicurezza dei messaggi dipendono da altre risorse di protezione di GlassFish Server.

Per ogni configurazione vengono fornite le informazioni riportate di seguito.

Layer di autenticazione

Layer di messaggio in cui viene eseguita l'autenticazione. Per impostazione predefinita, viene fornita una configurazione SOAP. È inoltre possibile specificare una configurazione HttpServlet.

Provider predefinito

Provider server predefinito per questa configurazione. Per impostazione predefinita, non viene configurato alcun provider server predefinito.

Provider client predefinito

Provider client predefinito per questa configurazione. Per impostazione predefinita, non viene configurato alcun provider client predefinito.

La tabella Configurazione di sicurezza dei messaggi contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione.

Elimina

Pulsante che consente di eliminare una o più configurazioni selezionate.
