Message Security Configurations

Nella pagina Message Security Configurations è possibile visualizzare e modificare le configurazioni disponibili per i provider di sicurezza dei messaggi.

Per configurare la sicurezza dei messaggi è necessario conoscere i concetti di base relativi a sicurezza e servizi Web. Per ulteriori informazioni, vedere il Capitolo 3, Administering Message Security in Oracle GlassFish Server 3.1 Security Guide e il Capitolo 6, Administering Web Applications in Oracle GlassFish Server 3.1 Administration Guide.

GlassFish Server implementa la sicurezza dei messaggi mediante provider di sicurezza dei messaggi integrati nel livello di elaborazione SOAP. I provider di sicurezza dei messaggi dipendono da altre risorse di protezione di GlassFish Server.

Per ogni configurazione vengono fornite le seguenti informazioni.

Authentication Layer

Livello dei messaggi al quale viene eseguita l'autenticazione. Per impostazione predefinita viene fornita una configurazione SOAP. È inoltre possibile specificare una configurazione HttpServlet.

Default Provider

Provider server predefinito per questa configurazione. Per impostazione predefinita non viene configurato alcun provider server predefinito.

Default Client Provider

Provider client predefinito per questa configurazione. Per impostazione predefinita non viene configurato alcun provider client predefinito.

La tabella Message Security Configuration contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova configurazione.

Delete

Pulsante che consente di eliminare una o più configurazioni selezionate.
