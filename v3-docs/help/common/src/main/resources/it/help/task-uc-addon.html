Installare i componenti add-on

L'installazione di un add-on component richiede le seguenti operazioni:

  • Trasferimento del contenuto e delle dipendenze del componente da un network repository

  • Installazione del componente sotto l'installation image di GlassFish Server disponibile nel sistema

Se un componente richiede altri componenti che non sono installati, verranno installati anche tali componenti.

  1. Nella struttura ad albero, selezionare il nodo Update Tool.

    Viene visualizzata la pagina Available Add-Ons.

  2. Nella pagina Available Add-Ons, nella tabella dei componenti add-on disponibili, selezionare i componenti da installare.

    Se non è disponibile nessun componente add-on, la tabella è vuota.

    • Per selezionare un componente singolo, selezionare la casella di controllo in corrispondenza del nome del componente.
    • Per selezionare tutti i componenti fare clic sull'icona Select All nell'intestazione della tabella.
    • Per deselezionare tutti i componenti fare clic sull'icona Deselect All nell'intestazione della tabella.
  3. Fare clic su Install.

    Viene visualizzata una finestra di dialogo con la richiesta di conferma dell'installazione del componente e degli eventuali componenti dipendenti.

  4. Nella finestra di dialogo, fare clic su OK.

    Viene richiesto di accettare i contratti di licenza dei componenti da installare.

  5. Accettare i contratti di licenza dei componenti da installare.

    In GlassFish Server viene confermato il completamento dell'installazione. I componenti vengono rimossi dalla tabella di componenti add-on e aggiunti alla tabella dei componenti installati nella pagina Installed Components.

  6. Riavviare il server.

    Per istruzioni dettagliate, vedere la seguente documentazione:

    • To Stop a Domain in Oracle GlassFish Server 3.1 Administration Guide

    • To Start a Domain in Oracle GlassFish Server 3.1 Administration Guide
