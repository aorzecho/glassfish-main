Installare componenti aggiuntivi

L'installazione di un componente aggiuntivo richiede le operazioni riportate di seguito.

  • Trasferimento del contenuto e delle dipendenze del componente da un repository di rete

  • Installazione del componente sotto l'immagine di installazione di GlassFish Server disponibile nel sistema

Se un componente richiede altri componenti che non sono installati, verranno installati anche tali componenti.

  1. Nell'albero di navigazione selezionare il nodo Strumento di aggiornamento.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili, nella tabella dei componenti add-on disponibili, selezionare i componenti da installare.

    Se non è disponibile nessun componente add-on, la tabella è vuota.

    • Per selezionare un componente singolo, selezionare la casella di controllo in corrispondenza del nome del componente.

    • Per selezionare tutti i componenti fare clic sull'icona Seleziona tutto nell'intestazione della tabella.

    • Per deselezionare tutti i componenti fare clic sull'icona Deseleziona tutto nell'intestazione della tabella.

  3. Fare clic su Installa.

    Viene visualizzata una finestra di dialogo con la richiesta di conferma dell'installazione del componente e degli eventuali componenti dipendenti.

  4. Nella finestra di dialogo, fare clic su OK.

    Viene richiesto di accettare i contratti di licenza dei componenti da installare.

  5. Accettare i contratti di licenza dei componenti da installare.

    In GlassFish Server viene confermato il completamento dell'installazione. I componenti vengono rimossi dalla tabella di componenti add-on e aggiunti alla tabella dei componenti installati nella pagina Componenti installati.

  6. Riavviare il server.

    Per istruzioni dettagliate, vedere la seguente documentazione:

    • To Stop a Domain in Oracle GlassFish Server Administration Guide

    • To Start a Domain in Oracle GlassFish Server Administration Guide
