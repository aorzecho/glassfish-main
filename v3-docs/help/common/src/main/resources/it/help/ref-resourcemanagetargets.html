Manage Resource Targets

Nella pagina Manage Resource Targets è possibile modificare i cluster e le istanze server standalone di destinazione in cui può essere abilitata una risorsa. La risorsa può essere abilitata solo nelle destinazioni riportate nella colonna Selected Targets.

La pagina Manage Resource Targets contiene le seguenti informazioni.

Available Targets

Cluster e istanze in cui non può essere abilitata la risorsa.

Selected Targets

Cluster e istanze in cui può essere abilitata la risorsa.

Add

Pulsante che consente di spostare la destinazione selezionata da Available Targets in Selected Targets.

Add All

Pulsante che consente di spostare tutte le destinazioni di Available Targets in Selected Targets.

Remove

Pulsante che consente di spostare la destinazione selezionata da Selected Targets in Available Targets.

Remove All

Pulsante che consente di spostare tutte le destinazioni di Selected Targets in Available Targets.
