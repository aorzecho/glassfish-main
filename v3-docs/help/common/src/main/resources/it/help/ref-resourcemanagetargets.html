Gestisci destinazioni risorse

Nella pagina Gestisci destinazioni risorse è possibile modificare i cluster e le istanze server standalone di destinazione in cui è possibile abilitare una risorsa. La risorsa può essere abilitata solo nelle destinazioni riportate nella colonna Destinazioni selezionate.

La pagina Gestisci destinazioni risorse contiene le informazioni riportate di seguito.

Destinazioni disponibili

Cluster e istanze in cui non può essere abilitata la risorsa.

Destinazioni selezionate

Cluster e istanze in cui può essere abilitata la risorsa.

Aggiungi

Pulsante che consente di spostare la destinazione selezionata da Destinazioni disponibili a Destinazioni selezionate.

Aggiungi tutto

Pulsante che consente di spostare tutte le destinazioni da Destinazioni disponibili a Destinazioni selezionate.

Rimuovi

Pulsante che consente di spostare la destinazione selezionata da Destinazioni selezionate a Destinazioni disponibili.

Rimuovi tutto

Pulsante che consente di spostare tutte le destinazioni da Destinazioni selezionate a Destinazioni disponibili.
