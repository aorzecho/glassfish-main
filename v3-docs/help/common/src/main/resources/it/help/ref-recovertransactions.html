Recupera transazioni

Nella pagina Recupera transazioni è possibile recuperare transazioni per un'istanza di GlassFish Server.

Se l'istanza è in esecuzione, la pagina Recupera transazioni contiene solo il testo Il recupero viene eseguito dallo stesso server in esecuzione e il pulsante OK.

Se l'istanza non è in esecuzione, la pagina Recupera transazioni contiene le opzioni riportate di seguito.

Nome istanza

Nome assegnato all'istanza quando è stata creata. Questo campo è di sola lettura.

Server di destinazione

Altra istanza in grado di recuperare le transazioni nell'istanza corrente.

Directory log delle transazioni

La directory in cui vengono memorizzati i log delle transazioni se i log sono memorizzati in un file system condiviso. Questa impostazione non si applica ai log delle transazioni memorizzati in un database. L'impostazione predefinita è /logs/tx nella directory del dominio.

I log delle transazioni sono memorizzati nella sottodirectory tx della directory specificata nell'impostazione Posizione log delle transazioni del servizio transazioni relativo alla configurazione cluster. Il valore predefinito di Posizione log delle transazioni è la directory specificata nel campo Radice log della pagina Attributi dominio all'interno della scheda Avanzate del nodo di GlassFish Server.

Comandi secondari asadmin equivalenti
