Recover Transactions

Nella pagina Recover Transactions è possibile ripristinare transazioni per un'istanza di GlassFish Server.

Se l'istanza è in esecuzione, la pagina Recover Transactions contiene solo il testo in cui viene indicato che il ripristino viene eseguito dal server in esecuzione e il pulsante OK.

Se l'istanza non è in esecuzione, la pagina Recover Transactions contiene le seguenti opzioni.

Instance Name

Nome assegnato all'istanza quando l'istanza è stata creata. Campo di sola lettura.

Destination Server

Altra istanza in grado di ripristinare le transazioni nell'istanza corrente.

Transaction Log Directory

Directory nella quale sono memorizzati i log delle transazioni. L'impostazione predefinita è /logs/tx nella directory di dominio.

I log delle transazioni sono memorizzati nella sottodirectory tx della directory specificata nell'impostazione Transaction Log Location del servizio transazioni relativo alla configurazione cluster. Il valore predefinito Transaction Log Location è la directory specificata nel campo Log Root disponibile nella pagina Domain Attributes all'interno della scheda Advanced del nodo GlassFish Server.
