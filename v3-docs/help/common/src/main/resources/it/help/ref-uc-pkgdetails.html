Component Details

Nella pagina Component Details sono disponibili informazioni dettagliate sui componenti.

La pagina Component Details contiene le seguenti opzioni.

Back

Pulsante che consente di tornare alla pagina in cui è stato selezionato il componente di cui sono visualizzati i dettagli.

Package Name

Nome del pacchetto con cui viene fornito il componente.

Unique Identifier

Identificatore completo del componente così come è stato specificato nel repository.

Category

Categoria del software a cui è associato il componente, ad esempio server di applicazioni, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente al momento della creazione.

Version

Numero della versione del pacchetto con cui viene fornito il componente.

Date Published

Data e ora in cui la versione in uso del componente è stata pubblicata nel repository. Non corrispondono alla data e all'ora di installazione del componente nell'immagine di installazione.

Installed Size

Quantità di spazio su disco utilizzata per l'installazione del componente.

Source

Per un componente installato, nome del Repository di rete da cui è stato ottenuto il componente.

Per un componente add-on o aggiornamento, nome del repository di rete da cui ottenere il componente.

Description

Panoramica del componente.
