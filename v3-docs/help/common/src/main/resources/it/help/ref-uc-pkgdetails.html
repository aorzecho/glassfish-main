Dettagli componenti

Nella pagina Dettagli componenti sono descritte informazioni dettagliate su un componente.

La pagina Dettagli componenti contiene le opzioni riportate di seguito.

Indietro

Pulsante che consente di tornare alla pagina da cui è stato selezionato il componente di cui sono visualizzati i dettagli.

Nome package

Nome del package in cui viene fornito il componente.

Identificativo univoco

Identificativo completamente qualificato di questo componente così come è stato specificato nel repository.

Categoria

Categoria del software a cui è associato il componente, ad esempio application server, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore del componente quando viene creato.

Versione

Numero di versione del package in cui viene fornito il componente.

Data di pubblicazione

Data e ora di pubblicazione della versione del componente nel repository. Questa data non corrisponde alla data e all'ora di installazione del componente nell'immagine di installazione.

Dimensione installata

Quantità di spazio su disco usata per l'installazione del componente.

Origine

Per un componente installato indica il nome del Repository di rete da cui è stato ottenuto il componente.

Per un componente aggiuntivo o un aggiornamento indica il nome del repository di rete da cui è stato ottenuto il componente.

Descrizione

Panoramica del componente.

Task correlati
