Predefined System Properties

GlassFish Server definisce le seguenti proprietà di sistema per la configurazione:

ASADMIN_LISTENER_PORT

Questa proprietà specifica il numero della porta HTTP o della porta HTTPS per l'amministrazione. La porta è quella indicata nell'URL che viene specificato nel browser per la gestione dell'istanza, ad esempio http://localhost:4949 . I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

HTTP_LISTENER_PORT

Questa proprietà specifica il numero della porta usata per l'attesa delle richieste HTTP. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

HTTP_SSL_LISTENER_PORT

Questa proprietà specifica il numero della porta usata per l'attesa delle richieste HTTPS. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

IIOP_LISTENER_PORT

Questa proprietà specifica il numero della porta usata per le connessioni IIOP. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

IIOP_SSL_LISTENER_PORT

Questa proprietà specifica il numero della porta usata per le connessioni IIOP sicure. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

IIOP_SSL_MUTUALAUTH_PORT

Questa proprietà specifica il numero della porta usata per le connessioni IIOP sicure con autenticazione client. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

JAVA_DEBUGGER_PORT

Questa proprietà specifica il numero della porta utilizzata per le connessioni al debugger Java Platform Debugger Architecture (JDPA). I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

JMS_PROVIDER_PORT

Questa proprietà specifica il numero di porta del provider JMS (Java Message Service). I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

JMX_SYSTEM_CONNECTOR_PORT

Questa proprietà specifica il numero di porta su cui è in attesa il connettore JMX. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.

OSGI_SHELL_TELNET_PORT

Questa proprietà specifica il numero della porta utilizzata per le connessioni ad Apache Felix Remote Shell. Tale shell utilizza il servizio shell Felix per l'interazione con il sottosistema di gestione del modulo OSGi. I valori validi sono 1-65535. In UNIX, la creazione di socket che restano in attesa sulle porte 1-1024 richiede privilegi di superutente.
