Livelli di log del modulo

Nella pagina Module Log Levels è possibile configurare i livelli di log per singoli moduli.

Per ogni modulo vengono fornite le seguenti informazioni.

Logger Name

Nome del programma di log per il modulo (ad esempio, javax.enterprise.system.tools.admin ).

Logger Level

Livello di log corrente per il modulo.

La tabella contiene le seguenti opzioni.

Add Logger

Pulsante per aggiungere un modulo del programma di log. Se si fa clic sul pulsante, viene aggiunta una riga alla tabella Logger Settings. Immettere informazioni sul modulo del programma di log nella nuova riga.

Delete Logger

Pulsante per eliminare uno o più moduli selezionati.

Level

Elenco a discesa da cui è possibile selezionare un livello di log da applicare a tutti i moduli selezionati.

Change Level

Pulsante che consente di modificare il livello di log per uno o più moduli selezionati.

Sono disponibili i seguenti livelli di registro, elencati dal più elevato al più basso.

SEVERE

Eventi che interferiscono con l'esecuzione normale del programma.

WARNING

Avvisi, incluse le eccezioni.

INFO

Messaggi relativi alla configurazione del server o allo stato del server, esclusi gli errori.

CONFIG

Messaggi relativi alla configurazione del server.

FINE

Livello di dettaglio minimo.

FINER

Livello di dettaglio moderato.

FINEST

Livello di dettaglio massimo.

OFF

Nessun messaggio di log.
