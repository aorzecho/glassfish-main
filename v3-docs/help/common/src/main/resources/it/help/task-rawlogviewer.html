Visualizzare i file di log raw per DAS

  1. Nell'albero di navigazione selezionare il nodo server (server di amministrazione).

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali, fare clic sul pulsante Visualizza log raw.

    Viene visualizzata la finestra Visualizzatore log raw.
