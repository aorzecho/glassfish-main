Aggiungere un utente al settore File

  1. Nel nodo Configurations, espandere la configurazione dell'istanza server o del cluster per cui si desidera gestire gli utenti del settore Admin.
  2. Espandere il nodo Security.
  3. Espandere il nodo Realms.
  4. Selezionare il nodo file.

    Viene visualizzata la pagina Edit Realm.

  5. Nella pagina Edit Realm, fare clic sul pulsante Manage Users.

    Viene visualizzata la pagina File Users.

  6. Nella pagina File Users, fare clic su New.

    Viene visualizzata la pagina New File Realm User.

  7. Nel campo User ID digitare il nome dell'utente.
  8. Nel campo Group List digitare l'elenco dei gruppi di appartenenza dell'utente.

    Utilizzare le virgole per separare i nomi dei gruppi. Non è necessario definire altrove tali gruppi.

  9. Nel campo Password digitare una password univoca.
  10. Nel campo Confirm New Password ridigitare la password utente.
  11. Fare clic su OK.
