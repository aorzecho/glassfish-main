Disabilitare il gestore della sicurezza

La disabilitazione del gestore della sicurezza di GlassFish Server può determinare un miglioramento delle prestazioni per determinati tipi di applicazioni. Le funzioni di autorizzazione e autenticazione della piattaforma Java continuano a funzionare anche se è stato disabilitato il gestore della sicurezza. È possibile disabilitare il gestore della sicurezza in un ambiente di sviluppo, mentre si sconsiglia di disabilitarlo in un ambiente di produzione.

  1. Nella struttura ad albero, selezionare il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo JVM Settings.

    Viene visualizzata la pagina JVM General Settings.

  3. Nella pagina JVM General Settings, fare clic sulla scheda JVM Options.

    Viene visualizzata la pagina JVM Options.

  4. Selezionare la casella di controllo a sinistra del campo Value contenente l'opzione -Djava.security.policy.
  5. Fare clic su Delete.
  6. Fare clic su Save.
  7. Riavviare il server.
