Disabilitare Security Manager

La disabilitazione del Security Manager di GlassFish Server può determinare un miglioramento delle prestazioni per determinati tipi di applicazioni. Le funzioni di autorizzazione e autenticazione della piattaforma Java continuano a funzionare anche se è stato disabilitato il gestore della sicurezza. È possibile disabilitare il gestore della sicurezza in un ambiente di sviluppo, mentre si sconsiglia di disabilitarlo in un ambiente di produzione.

  1. Nell'albero di navigazione selezionare il nodo Configurazione.

  2. Select the configuration that you are changing.

  3. Nel nodo Configurazione, selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM, fare clic sulla scheda Opzioni JVM.

    Viene visualizzata la pagina Opzioni JVM.

  5. Selezionare la casella di controllo a sinistra del campo Valore contenente l'opzione -Djava.security.policy.

  6. Fare clic su Elimina.

  7. Fare clic su Salva.

  8. Riavviare il server.
