Nuovo provider JACC

Nella pagina Nuovo provider JACC è possibile creare un nuovo provider JACC (Java Authorization Contract for Containers) conforme a JSR-115/196, che può essere usato dai moduli di autorizzazione di terze parti per applicazioni in esecuzione su GlassFish Server. Il provider JACC viene creato come elemento jacc-provider all'interno dell'elemento security-service nel file domain.xml del dominio.

Nell'installazione predefinita di GlassFish Server sono inclusi due provider JACC, denominati default e simple. I nuovi provider JACC creati si aggiungono a questi due provider predefiniti. I provider JACC predefiniti di GlassFish Server implementano un motore di autorizzazione semplice, basato su file e conforme alla specifica JACC.

Tutti i provider JACC di GlassFish Server devono utilizzare le API JAAS (Java Authentication and Authorization Service). JAAS consente servizi di autenticazione e applicazione dei controlli di accesso degli utenti e implementa una versione Java della struttura PAM (Pluggable Authentication Module) standard.

JSR 196 consente di sviluppare plugin a layer diversi. È possibile definire plugin che modificano la modalità di configurazione di nuovi meccanismi di autenticazione, ad esempio AuthConfigProvider e AuthConfigFactory. È inoltre possibile definire nuovi meccanismi di autenticazione, ad esempio ServerAuthModule eClientAuthModule.

La pagina Nuovo provider JACC contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome del provider JACC. Questo nome viene utilizzato per fare riferimento all'elemento jacc-provider nel file domain.xml.

Configurazione criteri

Nome di classe completamente qualificato per javax.security.jacc.PolicyConfigurationFactory.provider che implementa la classe javax.security.jacc.PolicyConfigurationFactory specifica del provider.

Provider criteri

Nome di classe completamente qualificato per javax.security.jacc.policy.provider che implementa java.security.Policy.

Proprietà aggiuntive

Proprietà aggiuntive per il provider JACC. Per la descrizione delle proprietà aggiuntive disponibili per i provider JACC, vedere Proprietà specifiche dei provider JACC.
