Modificare il connettore JMX

Utilizzare il servizio di amministrazione per configurare il connettore JMX remoto conforme allo standard JSR 160.

Il connettore JMX remoto conforme allo standard JSR 160 gestisce le comunicazioni tra i client JMX quali JConsole e il servizio JMX per la gestione dei domini.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Nel nodo Configurazione, selezionare il nodo Servizio di amministrazione.

    Viene visualizzata la pagina Modifica connettore JMX.

  3. Selezionare la casella di controllo Sicurezza abilitata per impostare la cifratura delle comunicazioni JMX.

    Questa opzione è disabilitata per impostazione predefinita.

  4. Nel campo Indirizzo digitare l'indirizzo IP del servizio di denominazione in cui è registrato lo stub del server del connettore JMX.

    Non si tratta della porta del socket server che esegue la comunicazione JMX effettiva, bensì dell'indirizzo dell'interfaccia di rete in cui viene avviato il registro RMI. Se il sistema è dotato di più interfacce di rete, è possibile modificare questo valore in modo da selezionare solo un'interfaccia specifica. Il valore predefinito è 0.0.0.0.

  5. Nel campo Porta, digitare il numero della porta su cui il servizio di denominazione (registro RMI) resta in attesa delle connessioni client RMI.

    Questo servizio di denominazione viene utilizzato esclusivamente per lo scaricamento degli stub RMI. Se la porta predefinita è occupata, viene utilizzata una porta libera. Sono consentiti valori compresi tra 1 e 65535. Nei sistemi UNIX, per creare socket in ascolto sulle porte comprese tra 1 e 1024 sono necessari privilegi di utente privilegiato. Il valore predefinito è 8686.

  6. Nel campo Nome realm, digitare il nome che rappresenta il realm amministrativo speciale

    in cui viene interamente gestita l'autenticazione. Il valore predefinito è admin-realm .

  7. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server varie proprietà vengono specificate per impostazione predefinita.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  8. Fare clic su Salva.

  9. Se si seleziona la casella di controllo Sicurezza abilitata, seguire le indicazioni riportate in Modificare le impostazioni SSL per il connettore JMX.
