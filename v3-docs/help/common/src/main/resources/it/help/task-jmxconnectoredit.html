Modificare il connettore JMX

Utilizzare il servizio Admin per configurare il connettore JMX remoto conforme allo standard JSR 160.

Il connettore JMX remoto conforme allo standard JSR 160 gestisce le comunicazioni tra i client JMX quali JConsole e il servizio JMX per la gestione dei domini.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Admin Service.

    Viene visualizzata la pagina Edit JMX Connector.

  3. Selezionare la casella di controllo Security Enabled per impostare la crittografia delle comunicazioni JMX.

    Questa opzione è disabilitata per impostazione predefinita.

  4. Nel campo Address digitare l'indirizzo IP del servizio di denominazione in cui è registrato lo stub del server del connettore JMX.

    Non si tratta della porta del socket server che esegue la comunicazione JMX effettiva, bensì dell'indirizzo dell'interfaccia di rete in cui viene avviato il registro RMI. Se il sistema è dotato di più interfacce di rete, è possibile modificare questo valore in modo da selezionare solo un'interfaccia specifica. Il valore predefinito è 0.0.0.0.

  5. Nel campo Port, digitare il numero della porta su cui il servizio di denominazione (registro RMI) resta in attesa delle connessioni client RMI.

    L'uso del servizio di denominazione è esclusivamente destinato al download degli stub RMI. Se la porta predefinita è occupata, viene utilizzata una porta libera. Sono consentiti valori compresi tra 1 e 65535. In sistemi UNIX, per creare socket in attesa su porte comprese tra 1 e 1024 sono necessari privilegi di superutente. Il valore predefinito è 8686.

  6. Nel campo Realm Name, digitare il nome che rappresenta il settore amministrativo speciale

    in cui viene interamente gestita l'autenticazione. Il valore predefinito è admin-realm .

  7. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server varie proprietà vengono specificate per impostazione predefinita.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  8. Fare clic su Save.
  9. Se è stata selezionata la casella di controllo Security Enabled, attenersi alla procedura descritta in Modificare le impostazioni SSL del connettore JMX.
