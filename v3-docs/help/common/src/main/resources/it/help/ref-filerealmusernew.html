Nuovo utente realm

Nella pagina Nuovo utente realm è possibile concedere a un nuovo utente l'accesso al realm corrente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile creare un utente di admin-realm con una password vuota.

La pagina Nuovo utente realm contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome realm

Nome del realm attualmente selezionato. Questo campo è di sola lettura.

ID utente

ID utente per il nuovo account utente.

Lista gruppi

Gruppi di appartenenza del nuovo utente. Separare i nomi dei gruppi utilizzando una virgola.

Per gli utenti admin-realm, il campo è di sola lettura e il gruppo è asadmin.

Nuova password

Password per il nuovo utente.

Conferma nuova password

Password per il nuovo utente.
