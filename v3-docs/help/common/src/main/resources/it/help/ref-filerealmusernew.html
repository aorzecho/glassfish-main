New File Realm User

Nella pagina New File Realm User è possibile concedere a un nuovo utente l'accesso al settore corrente.

La pagina New File Realm User contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Realm Name

Nome del settore correntemente selezionato. Il campo è di sola lettura.

User ID

ID utente per il nuovo account utente.

Group List

Gruppi di appartenenza del nuovo utente. Separare i nomi dei gruppi utilizzando una virgola.

Per gli utenti admin-realm, il campo è di sola lettura e il gruppo è asadmin.

New Password

Password per il nuovo utente.

Confirm New Password

Password per il nuovo utente.

Attività correlate

  • Aggiungere un utente al settore File

  • Modificare le informazioni utente per il settore File

  • Eliminare un utente dal settore File
