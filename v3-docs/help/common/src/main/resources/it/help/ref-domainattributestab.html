Attributi dominio

Nella pagina Attributi dominio è possibile impostare attributi di dominio avanzati per GlassFish Server.

La pagina Attributi dominio contiene le opzioni riportate di seguito.

Radice applicazioni

Percorso completo della directory in cui risiedono le applicazioni distribuite. Il valore predefinito è domain-dir/applications.

Radice log

Identifica dove sono memorizzati i file di log del server. Il valore predefinito è domain-dir/logs.

Impostazioni nazionali

Se questo campo viene lasciato vuoto, vengono utilizzate le impostazioni nazionali predefinite dell'host.

Per impostazioni nazionali si intende un identificativo che specifica una determinata combinazione di lingua e regione. Ad esempio, le impostazioni nazionali per l'inglese americano sono en_US, mentre per l'italiano sono it_IT. Per utilizzare impostazioni nazionali diverse dall'inglese, è necessario che sia stata eseguita la localizzazione di GlassFish Server, ovvero un processo che prevede la traduzione dall'inglese a un'altra lingua.
