Domain Attributes

Nella pagina Domain Attributes è possibile impostare attributi di dominio avanzati per GlassFish Server.

La pagina Domain Attributes contiene le seguenti opzioni.

Application Root

Percorso completo della directory in cui si trovano le applicazioni distribuite. Il valore predefinito è domain-dir/applications.

Log Root

Identifica il log del server in cui si trovano i file. Il valore predefinito è domain-dir/logs.

Locale

Se il campo resta vuoto, viene utilizzata la versione locale predefinita dell'host.

Per versione locale si intende un identificatore che specifica una combinazione particolare di lingua e regione. Ad esempio la versione locale per l'inglese americano è en_US mentre per il giapponese è ja_JP. Per utilizzare una versione locale non inglese, è necessario che sia stata eseguita la localizzazione di GlassFish Server, un processo che prevede anche la traduzione dall'inglese a un'altra lingua.
