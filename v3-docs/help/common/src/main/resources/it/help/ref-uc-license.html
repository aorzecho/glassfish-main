License Agreement

Nella pagina License Agreement vengono visualizzati i contratti di licenza di tutti i componenti selezionati per l'installazione o l'aggiornamento. Insieme al componente, l'autore del componente specifica il rispettivo contratto di licenza. Nel Repository di rete è possibile pubblicare componenti diversi con licenze diverse, ad esempio:

  • Common Development and Distribution License (CDDL)

  • GNU General Public License (GPL)

  • Apache License

  • wxWindows Library Licence

La pagina License Agreement contiene le seguenti opzioni.

Accept

Pulsante che consente di accettare le condizioni delle licenze di tutti i componenti selezionati e procedere con l'installazione o l'aggiornamento.

Decline

Pulsante che consente di rifiutare le condizioni delle licenze di tutti i componenti selezionati e tornare alla pagina di selezione dei componenti.

Attività correlate

  • Visualizzare gli aggiornamenti disponibili per i componenti

  • Installare i componenti add-on
