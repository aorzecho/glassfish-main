Impostare il settore predefinito

Il settore predefinito è quello utilizzato da GlassFish Server per l'autenticazione e l'autorizzazione se il descrittore di distribuzione di un'applicazione non specifica un settore.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare il nodo Security.

    Viene visualizzata la pagina Security.

  3. Dall'elenco a discesa Default Realm selezionare il settore attivo che il server utilizzerà per l'autenticazione.

    Il valore predefinito è file.

  4. Fare clic su Save.
