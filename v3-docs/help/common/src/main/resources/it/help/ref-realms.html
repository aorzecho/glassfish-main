Realm

GlassFish Server è preconfigurato con tre realm: file, certificate e admin-realm. È inoltre possibile creare realm ldap, JDBC, solaris e personalizzati. In genere, in un server è disponibile un realm di ciascun tipo. Tuttavia, in GlassFish Server sono presenti due realm file: file e admin-realm. Si tratta di due realm dello stesso tipo utilizzati per due scopi differenti. È inoltre possibile disporre di un database dei certificati distinto per ciascun server virtuale del sistema.

Nella pagina Realm è possibile configurare realm.

Per ogni realm vengono fornite le informazioni riportate di seguito.

Nome

Nome del realm.

Nome classe

Nome di classe per il realm.

La tabella Realm contiene le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo realm.

Elimina

Pulsante che consente di eliminare uno o più realm selezionati.
