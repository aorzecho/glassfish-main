Realms

GlassFish Server è preconfigurato con tre settori: file, certificate e admin-realm. È inoltre possibile creare settori ldap, JDBC, solaris e personalizzati. In genere in un server è disponibile un settore di ciascun tipo. Tuttavia in GlassFish Server sono presenti due settori di file: file e admin-realm. Si tratta di due settori dello stesso tipo utilizzati per due scopi differenti. È inoltre possibile disporre di un database dei certificati distinto per ciascun server virtuale del sistema.

Nella pagina Realms è possibile configurare settori.

Per ogni settore vengono fornite le seguenti informazioni.

Name

Nome del settore.

Class Name

Nome della classe per il settore.

La tabella Realms contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo settore.

Delete

Pulsante che consente di eliminare uno o più settori selezionati.
