New Password Alias

Nella pagina New Password Alias è possibile creare un nuovo alias di password da utilizzare in un file di password o nel file di configurazione del dominio.

La pagina New Password Alias contiene le seguenti opzioni:

Alias Name

Nome dell'alias. Tale nome deve essere univoco tra tutti gli alias di password presenti nel dominio.

Password

Password che l'alias memorizza in forma crittografata.

Confirm Password

Password che l'alias memorizza in forma crittografata. I valori di Password e Confirm Password vengono confrontati per verificare che la password sia stata immessa senza errori di digitazione.
