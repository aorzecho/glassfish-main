Nuovi alias password

Nella pagina Nuovi alias password è possibile creare un nuovo alias password da utilizzare in un password file o nel file di configurazione del dominio.

La pagina Nuovo alias password contiene le opzioni riportate di seguito.

Nome alias

Nome dell'alias. Tale nome deve essere univoco tra tutti gli alias password presenti nel dominio.

Password

Password che l'alias memorizza in forma cifrata.

Conferma password

Password che l'alias memorizza in forma cifrata. I valori di Password e Conferma password vengono confrontati per assicurarsi che la password sia stata immessa senza errori di digitazione.
