General Information

Nella pagina General Information è possibile verificare le impostazioni di GlassFish Server e visualizzare i dati delle macchine virtuali Java.

La pagina General Information contiene le seguenti informazioni.

Stop Button

Fare clic sul pulsante Stop per arrestare GlassFish Server.

Restart Button

Fare clic sul pulsante Restart per riavviare GlassFish Server.

View Log Files Button

Fare clic sul pulsante View Log Files per visualizzare i file di log per un'istanza o un cluster GlassFish Server.

Name

Nome del server.

JVM

Se si fa clic su JVM Report, viene visualizzata una finestra distinta in cui sono disponibili i rapporti relativi alla macchina virtuale Java, tra cui un rapporto di riepilogo, un rapporto di gestione memoria e recupero spazio, un rapporto sul caricamento delle classi e un dump del thread corrente.

Configuration Directory

Directory del computer host in cui sono memorizzati i file di configurazione, come domain.xml.

Installed Version

Identificatore della versione di prodotto.

Debug

Indica se il debug è attivo. Se il debug è attivo, viene visualizzato anche il numero di porta.

HTTP Port(s)

Le porte correntemente configurate per richieste e risposte HTTP.

IIOP Port(s)

Le porte correntemente configurate per richieste e risposte IIOP.
