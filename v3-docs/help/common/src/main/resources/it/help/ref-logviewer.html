Log Viewer

Nella pagina Log Viewer è possibile visualizzare i file di log per le istanze e i cluster GlassFish Server selezionati e personalizzarne la visualizzazione.

La pagina Log Viewer contiene quattro aree:

  • Search Criteria

  • Advanced Search

  • Modifica ricerca

  • Search Results

L'area Search Criteria contiene le seguenti opzioni.

Advanced Search

Fare clic questo collegamenti per visualizzare un'area che consente di eseguire ulteriori miglioramenti al visualizzatore del log.

Timestamp

Il pulsante di scelta Most Recent consente di visualizzare i messaggi più recenti. Il pulsante di scelta Specific Range consente di visualizzare i campi From e To, con cui è possibile recuperare i messaggi appartenenti a periodi di tempo specifici.

Per il valore Time, la sintassi deve essere la seguente (SSS corrisponde ai millisecondi):

hh:mm:ss.SSS

Ad esempio:

17:10:00.000

Se il valore incluso in From è successivo al valore incluso in To, viene visualizzato un messaggio di errore.

Log Level

Livello del log selezionato per il filtraggio. Per impostazione predefinita la visualizzazione include tutti i messaggi visualizzati nel log del server al livello di log selezionato e ai livelli di maggior gravità. Selezionare la casella di controllo Do not include more severe messages per visualizzare soltanto i messaggi corrispondenti al livello scelto.

Per assicurare che i messaggi da visualizzare siano presenti nel log del server, impostare innanzitutto i livelli di log appropriati nella pagina Module Log Levels. Vedere Configurare i livelli di log.

Se si sceglie di filtrare messaggi di log in base al livello di log, vengono visualizzati solo i messaggi che soddisfano i criteri di filtro specificati. Il filtraggio non ha tuttavia effetto sui messaggi che sono registrati nel log del server.

L'area Advanced Search contiene le seguenti opzioni.

Logger

Namespace del programma di log da filtrare in base al modulo. Per selezionare più namespace, fare clic su di essi tenendo premuto il tasto Maiusc o Ctrl.

Se si seleziona un namespace a un livello elevato vengono selezionati tutti i namespace di livello sottostante. Ad esempio, se si seleziona javax.enterprise.system vengono selezionati anche i programmi di log relativi a tutti i moduli in tale namespace: javax.enterprise.system.core , javax.enterprise.system.tools.admin e così via.

Custom Logger

Programmi di log specifici per una applicazione particolare, uno per riga.

Se l'applicazione è costituita da diversi moduli, è possibile visualizzarne alcuni o tutti. Ad esempio, prendere in considerazione il caso in cui l'applicazione includa programmi di log con i seguenti nomi:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Per visualizzare i messaggi di tutti i moduli dell'applicazione, digitare com.mycompany.myapp . Per visualizzare solo messaggi di module2, digitare com.mycompany.myapp.module2.

Quando si specifica uno o più programmi di log personalizzati, i messaggi dei moduli GlassFish Server vengono visualizzati solo se vengono specificati esplicitamente nell'area Logger.

Display

Numero di messaggi visualizzabili in una volta. Il numero predefinito di messaggi visualizzati è 40. Altre opzioni sono 100, 250 e 1000.

Per visualizzare le tracce dello stack, deselezionare la casella di controllo Limit excessively long messages. Per impostazione predefinita, le tracce dello stack non vengono visualizzate nel visualizzatore; per visualizzarle, fare clic sul collegamento (details) per un messaggio.

L'area Modify Search consente di specificare l'istanza server o il cluster e il nome di un file di log associato da visualizzare.

Instance

Istanza server o cluster per cui saranno visualizzati i record del log. Il valore predefinito è server. Altre opzioni dipendono dai cluster e dalle istanze che possono essere definite e abilitate per il dominio.

Log File

Nome del file di log da visualizzare. Il valore predefinito è server.log. Altre opzioni dipendono dalle impostazioni di log per la destinazione selezionata.

Nella tabella Log Viewer Results vengono visualizzati i messaggi del log del server. Per scegliere l'intervallo di record da visualizzare, utilizzare i pulsanti Records before e Records after nella parte superiore della tabella.

Per ogni messaggio la tabella Log Viewer Results contiene le seguenti informazioni.

Record Number

Numero di messaggi nel log del server.

Log Level

Livello di log del messaggio.

Message

Breve testo del messaggio di log. Fare clic sul collegamento (details) per visualizzare la versione formattata di un messaggio.

Logger

Modulo del programma di log che ha generato il messaggio di log.

Timestamp

Data e ora dell'emissione del messaggio di log.

Name/Value Pairs

Nome del thread e ID del messaggio di log.
