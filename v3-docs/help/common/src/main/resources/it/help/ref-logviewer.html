Visualizzatore log

Nella pagina Visualizzatore log è possibile visualizzare i file di log per le istanze e i cluster GlassFish Server selezionati e personalizzare la visualizzazione del log.

La pagina Visualizzatore log contiene quattro aree:

  • Criteri di ricerca

  • Ricerca avanzata

  • Modifica ricerca

  • Risultati ricerca

L'area Criteri di ricerca contiene le opzioni riportate di seguito.

Ricerca avanzata

Fare clic su questo collegamento per visualizzare un'area che consente di eseguire ulteriori miglioramenti al visualizzatore log.

Indicatore orario

Il pulsante di scelta Più recente consente di visualizzare i messaggi più recenti. Il pulsante di scelta Intervallo specifico consente di visualizzare i campi Da e A che consentono di visualizzare i messaggi solo a partire da un determinato periodo di tempo.

Per il valore Ora, la sintassi deve essere la seguente (SSS corrisponde ai millisecondi):

hh:mm:ss.SSS

Ad esempio:

17:10:00.000

Se il valore del campo Da è successivo al valore del campo A, viene visualizzato un messaggio di errore.

Livello di log

Livello di log selezionato per l'applicazione dei filtri. Per impostazione predefinita, la visualizzazione include tutti i messaggi visualizzati nel log del server al livello di log scelto e ai livelli SEVERE maggiori. Selezionare la casella di controllo "Non includere messaggi di livello SEVERE maggiore" per visualizzare solo i messaggi corrispondenti al livello scelto.

Per assicurare che i messaggi da visualizzare siano presenti nel log del server, impostare innanzitutto i livelli di log appropriati nella pagina Livelli di log dei moduli. Vedere Configurare i livelli di log.

Se si sceglie di filtrare messaggi di log in base al livello di log, vengono visualizzati solo i messaggi che soddisfano i criteri di filtro specificati. L'applicazione del filtro non ha tuttavia effetto sui messaggi registrati nel log del server.

L'area Ricerca avanzata contiene le opzioni riportate di seguito.

Logger

Spazio di nomi del logger da filtrare in base al modulo. Per scegliere più spazi di nomi, fare clic su di essi tenendo premuto il tasto Maiusc o Ctrl.

Se si seleziona uno spazio di nomi a un livello superiore, vengono selezionati tutti gli spazi di nomi sottostanti. Ad esempio, se si seleziona javax.enterprise.system vengono selezionati anche i logger relativi a tutti i moduli sotto tale spazio di nomi: javax.enterprise.system.core , javax.enterprise.system.tools.admin e così via.

Logger personalizzato

Logger specifici di una determinata applicazione, uno per riga.

Se l'applicazione è costituita da diversi moduli, è possibile visualizzarne alcuni o tutti. Ad esempio, si supponga che l'applicazione includa logger con i seguenti nomi:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Per visualizzare i messaggi di tutti i moduli dell'applicazione, digitare com.mycompany.myapp . Per visualizzare solo messaggi di module2, digitare com.mycompany.myapp.module2.

Quando si specifica uno o più logger personalizzati, i messaggi dei moduli GlassFish Server vengono visualizzati solo se vengono specificati esplicitamente nell'area Logger.

Visualizza

Numero di messaggi visualizzabili in una volta. Il numero predefinito di messaggi visualizzati è 40. Altre opzioni sono 100, 250 e 1000.

Per visualizzare gli stack trace, deselezionare la casella di controllo "Limita messaggi troppo lunghi". Per impostazione predefinita, gli stack trace non vengono visualizzati nel visualizzatore; per visualizzarli, fare clic sul collegamento (dettagli) per un messaggio.

L'area Modifica ricerca consente di specificare l'istanza server o il cluster e il nome di un file di log associato da visualizzare.

Istanza

Istanza server o cluster per cui vengono visualizzati i record del log. Il valore predefinito è server. Altre opzioni dipendono dai cluster e dalle istanze che è possibile definire e abilitare per il dominio.

File di log

Nome del file di log da visualizzare. Il valore predefinito è server.log. Altre opzioni dipendono dalle impostazioni di log per la destinazione selezionata.

Nella tabella Risultati visualizzatore log vengono visualizzati i messaggi del log del server. Per scegliere l'intervallo di record da visualizzare, utilizzare i pulsanti Record prima di e Record dopo nella parte superiore della tabella.

Per ogni messaggio la tabella Risultati visualizzatore log contiene le informazioni riportate di seguito.

Numero record

Numero di messaggi nel log del server.

Livello di log

Livello di log del messaggio.

Messaggio

Breve testo del messaggio di log. Fare clic sul collegamento (dettagli) per visualizzare la versione formattata di un messaggio.

Logger

Modulo del logger che ha generato il messaggio di log.

Indicatore orario

Data e ora di esecuzione del messaggio di log.

Coppie nome-valore

Nome del thread e ID del messaggio di log.
