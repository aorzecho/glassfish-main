Modifica alias password

Nella pagina Modifica alias password è possibile modificare la password associata a un alias password esistente.

La pagina Modifica alias password contiene le opzioni riportate di seguito.

Nome alias

Nome dell'alias. Questa opzione non è modificabile.

Password

Password che l'alias memorizza in forma cifrata.

Conferma password

Password che l'alias memorizza in forma cifrata. I valori di Password e Conferma password vengono confrontati per assicurarsi che la password sia stata immessa senza errori di digitazione.
