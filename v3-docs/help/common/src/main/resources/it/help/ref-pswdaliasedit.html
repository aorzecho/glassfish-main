Edit Password Alias

Nella pagina Edit Password Alias è possibile cambiare la password associata a un alias esistente.

La pagina Edit Password Alias contiene le seguenti opzioni:

Alias Name

Nome dell'alias. Questa opzione non è modificabile.

Password

Password che l'alias memorizza in forma crittografata.

Confirm Password

Password che l'alias memorizza in forma crittografata. I valori di Password e Confirm Password vengono confrontati per verificare che la password sia stata immessa senza errori di digitazione.
