Impostare il modulo di audit attivo

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. Nel nodo Configurazione, selezionare il nodo Sicurezza.

    Viene visualizzata la pagina Sicurezza.

  4. Nell'elenco a discesa Moduli di audit, selezionare il nome del modulo di revisione da utilizzare nel server.

    Per impostazione predefinita è disponibile un solo modulo di revisione denominato default.

  5. Fare clic su Salva.
