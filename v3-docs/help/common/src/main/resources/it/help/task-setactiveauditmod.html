Impostare il modulo di revisione attivo

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Security.

    Viene visualizzata la pagina Security.

  3. Nell'elenco a discesa Audit Modules, selezionare il nome del modulo di revisione da utilizzare nel server.

    Per impostazione predefinita è disponibile un solo modulo di revisione denominato default.

  4. Fare clic su Save.
