Edit File Realm User

Nella pagina Edit File Realm User è possibile modificare una password utente per il settore corrente.

La pagina Edit File Realm User contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Realm Name

Nome del settore correntemente selezionato. Il campo è di sola lettura.

User ID

Nome dell'account da modificare. Il campo è di sola lettura.

Group List

Gruppi di appartenenza dell'utente corrente. Separare i nomi dei gruppi utilizzando una virgola.

Per gli utenti admin-realm, il campo è di sola lettura e il gruppo è asadmin.

New Password

Nuova password per l'utente corrente.

Confirm New Password

Nuova password per l'utente corrente.

Attività correlate

  • Aggiungere un utente al settore File

  • Modificare le informazioni utente per il settore File

  • Eliminare un utente dal settore File
