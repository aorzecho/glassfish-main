Modifica utente realm

Nella pagina Modifica utente realm è possibile modificare una password dell'utente per il realm corrente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile modificare la password di un utente di admin-realm in un valore vuoto.

La pagina Modifica utente realm contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome realm

Nome del realm attualmente selezionato. Questo campo è di sola lettura.

ID utente

Nome dell'account da modificare. Questo campo è di sola lettura.

Lista gruppi

Gruppi di appartenenza dell'utente corrente. Separare i nomi dei gruppi utilizzando una virgola.

Per gli utenti admin-realm, il campo è di sola lettura e il gruppo è asadmin.

Nuova password

Nuova password per l'utente corrente.

Conferma nuova password

Nuova password per l'utente corrente.
