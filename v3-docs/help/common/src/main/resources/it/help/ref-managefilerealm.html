File Users

Nella pagina File Users è possibile gestire gli utenti del settore file. Utenti e gruppi del settore file vengono elencati nel file di chiave, la cui posizione è specificata dalla proprietà file.

Un utente nel settore file può appartenere a un gruppo Java, una categoria di utenti con caratteristiche comuni. Ad esempio, gli utenti di un'applicazione di e-commerce potrebbero far parte del gruppo CUSTOMER, ma quelli che spendono di più verrebbero inclusi nel gruppo PREFERRED. La suddivisione degli utenti in categorie di gruppi semplifica il controllo dell'accesso di numerosi utenti.

Inizialmente, dopo l'installazione di GlassFish Server, l'unico utente è l'amministratore che ha eseguito l'accesso durante l'installazione. Per impostazione predefinita tale utente fa parte del gruppo asadmin, nel settore admin-realm, ossia dispone dei diritti per la modifica di GlassFish Server. Qualsiasi utente assegnato a questo gruppo disporrà dei privilegi di amministratore, ossia avrà accesso allo strumento asadmin e ad Administration Console.

Nella pagina File Users viene visualizzato un elenco di utenti per il settore selezionato. Per ogni utente vengono fornite le seguenti informazioni:

User ID

Nome dell'utente.

Group List

Gruppi di appartenenza dell'utente.

La tabella File Users contiene le seguenti opzioni.

New

Pulsante che consente di creare un nuovo utente.

Delete

Pulsante che consente di eliminare uno o più utenti selezionati.

Attività correlate

  • Modificare le informazioni utente per il settore File

  • Eliminare un utente dal settore File
