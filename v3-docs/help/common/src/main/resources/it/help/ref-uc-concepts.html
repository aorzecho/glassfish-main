Concetti chiave di Update Tool

Update Tool consente di gestire le installazioni dei componenti software di GlassFish Server inclusi in una singola directory del file system nel sistema in uso.

La versione Administration Console di Update Tool consente di effettuare le seguenti operazioni:

  • Installare i componenti add-on

  • Visualizzare gli aggiornamenti per i componenti installati

  • Visualizzare i componenti installati

Per aggiornare o rimuovere i componenti installati è necessario chiudere il dominio GlassFish Server, quindi eseguire uno dei seguenti comandi:

  • as-install-parent/bin/updatetool, che avvia l'utility grafica Update Tool

  • as-install-parent/bin/pkg, una versione a riga di comando dell'utility Update Tool

Nelle sezioni che seguono vengono descritti i concetti chiave di Update Tool.

Componente

Per componente si intende l'unità installabile gestita da Update Tool. Un componente è composto da file installati come unità di sistema in un'immagine di installazione.

Immagine di installazione

Con il termine immagine di installazione si definisce la directory in cui si trovano i componenti software, incluso il contenuto della directory. L'immagine di installazione viene aggiunta a Update Tool in seguito al download e all'installazione del software GlassFish Server. Per consentire l'aggiunta e l'aggiornamento dei componenti software di GlassFish Server, ogni immagine di installazione include informazioni sull'installazione di componenti.

Repository di rete

I repository di rete contengono i seguenti elementi:

  • Componenti che possono essere aggiunti a un'immagine di installazione

  • Aggiornamenti per i componenti esistenti di un'immagine di installazione

A uno o più repository di rete è associata un'immagine di installazione. Ogni repository di rete è rappresentato da un nome e un URL associato. I nomi di repository sono univoci per ogni immagine di installazione e rappresentano sia l'organizzazione o il progetto che gestisce il repository che un ambito di contenuti definito correttamente. Per ulteriori informazioni sui repository di rete preconfigurati per GlassFish Server, vedere Preconfigured Repositories for GlassFish Server in Oracle GlassFish Server 3.1 Administration Guide .

Componente add-on

Per componente add-on si intende un componente dell'immagine di installazione che non è ancora stato installato.
