Configurare le impostazioni del percorso JVM

  1. Nella struttura ad albero, selezionare il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo JVM Settings.

    Viene visualizzata la pagina JVM General Settings.

  3. Nella pagina JVM General Settings fare clic sulla scheda Path Settings.

    Viene visualizzata la pagina JVM Path Settings.

  4. Nel campo Native Library Path Prefix, digitare il percorso completo dei file JAR da aggiungere come prefisso al percorso della libreria nativa.

    Per inserire come prefisso il percorso di più file, digitare un percorso per riga.

  5. Nel campo Native Library Path Suffix, digitare il percorso completo dei file JAR da aggiungere come suffisso al percorso della libreria nativa.

    Per inserire come suffisso il percorso di più file, digitare un percorso per riga.

  6. Fare clic su Save.
  7. Riavviare il server.
