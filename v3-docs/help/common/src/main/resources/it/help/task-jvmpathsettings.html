Configurare le impostazioni del percorso JVM

  1. Nell'albero di navigazione selezionare il nodo Configurazione.

  2. Select the configuration that you are changing.

  3. Nel nodo Configurazione, selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM fare clic sulla scheda Impostazioni percorso.

    Viene visualizzata la pagina Impostazioni percorso JVM.

  5. Nel campo Prefisso percorso di libreria nativa, digitare il percorso completo dei file JAR da aggiungere come prefisso al percorso della libreria nativa.

    Per inserire come prefisso il percorso di più file, digitare un percorso per riga.

  6. Nel campo Suffisso percorso di libreria nativa, digitare il percorso completo dei file JAR da aggiungere come suffisso al percorso della libreria nativa.

    Per aggiungere più file, digitare un percorso per riga.

  7. Fare clic su Salva.

  8. Riavviare il server.
