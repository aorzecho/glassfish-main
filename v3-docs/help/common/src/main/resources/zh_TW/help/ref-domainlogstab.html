網域記錄

使用「網域記錄」頁面可收集與下載 GlassFish Server 的記錄檔。

「網域記錄」頁面包含下列選項。您可選取一個要收集其記錄的叢集或伺服器實例。

叢集

叢集的下拉式清單 (若叢集存在的話)。

實例

伺服器實例的下拉式清單。

收集記錄

將記錄收集到 log-files-name.zip 檔案的按鈕，其中 name 是叢集或實例的名稱。

log-files-name.zip 檔案可透過瀏覽器的下載機制下載。
