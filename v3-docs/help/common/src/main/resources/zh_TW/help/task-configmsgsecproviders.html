組態訊息安全性提供者所使用的 GlassFish Server 功能

  1. 如果使用了使用者名稱記號，請在必要時組態使用者資料庫。

    使用使用者名稱/密碼記號時，請先組態一個適當的範圍以及一個用於該範圍的適當使用者資料庫。

    編輯範圍中會討論如何組態使用者資料庫。

  2. 管理憑證和私密金鑰 (如有必要)。
