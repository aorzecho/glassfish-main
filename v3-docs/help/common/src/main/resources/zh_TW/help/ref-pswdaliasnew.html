新建密碼別名

使用 [新建密碼別名] 頁面可建立新的密碼別名，以用於密碼檔案或網域配置檔案。

[新建密碼別名] 頁面包含下列選項：

別名

別名的名稱。此名稱在網域內所有密碼別名中必須是唯一的。

密碼

以加密格式儲存密碼別名的密碼。

確認密碼

以加密格式儲存密碼別名的密碼。會比較 [密碼] 與 [確認密碼] 值，確定輸入的密碼沒有拼錯。
