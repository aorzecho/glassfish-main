模組記錄層級

使用 「模組記錄層級」 頁面可組態個別模組的記錄層級。

對於每個模組，均提供以下資訊。

記錄程式名稱

模組的記錄程式名稱 (例如 javax.enterprise.system.tools.admin)。

記錄程式層級

模組的目前記錄層級。

記錄程式設定表也包含下列選項。

增加記錄程式

增加記錄程式模組的按鈕。按一下此按鈕即可在記錄程式設定表中增加一列。在新列中輸入記錄程式模組的相關資訊。

刪除記錄程式

刪除一或多個選取的模組的按鈕。

層級

下拉式清單，您可以從中選取要套用到所有選取模組的記錄層級。

變更層級

變更一或多個選取模組之記錄層級的按鈕。

可用的記錄層級如下，以由高至低的順序列出。

嚴重

干擾正常程式執行的事件。

警告

包括異常在內的警告。

資訊

與伺服器組態或伺服器狀態相關的訊息，不包括錯誤。

組態

與伺服器組態相關的訊息。

詳細

最低詳細程度。

更詳細

中等詳細程度。

最詳細

最高詳細程度。

關閉

關閉記錄訊息功能。
