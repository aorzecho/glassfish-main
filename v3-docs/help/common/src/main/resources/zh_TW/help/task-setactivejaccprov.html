設定使用中的 JACC 提供者

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  3. 從 [JACC] 下拉式選單中，選取將由伺服器使用的 JACC 提供者。

    根據預設，有兩個 JACC 提供者可用，分別名為 default 和 simple。預設提供者為 default。

  4. 按一下 [儲存]。
