配置記錄層級

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，按一下您要配置 [記錄程式設定] 的伺服器實例或叢集配置。

    [配置] 頁面會隨即開啟。

  3. 在 [配置] 頁面上，按一下 [記錄程式設定]。

    所選取配置目標的 [記錄程式設定] 頁面會隨即開啟。

  4. 在 [記錄程式設定] 頁面上，按一下 [記錄層級] 標籤。

    [模組記錄層級] 頁面會隨即開啟。

  5. 在 [模組記錄層級] 頁面上，依下列步驟變更一個或多個模組的記錄層級。
    • 若要變更個別模組的記錄層級，請從要變更記錄層級的模組相對的下拉式清單，選取新的值。
    • 變更模組群組的記錄層級：
      1. 選取要變更記錄層級的模組名稱旁的核取方塊。
      2. 從表格頂端的下拉式清單選取值。
      3. 按一下 [變更層級] 按鈕。

    記錄中將顯示已選取層級或更高層級的訊息。選擇以下任一值 (以最高層級至最低層級的順序列出)︰

    • 嚴重

    • 警告

    • 資訊

    • 配置

    • 詳細

    • 更詳細

    • 最詳細

    • 關

    如需記錄層級意義的詳細資訊，請參閱模組記錄層級。預設層級為 資訊，所以依預設記錄將包含屬於 資訊、警告 或 嚴重 層級的訊息。

  6. 按一下 [儲存]。

    對記錄層級設定所做的變更將立即生效。這些變更也會儲存以供伺服器重新啟動時使用。
