刪除 File 範圍中的使用者

  1. 在 [配置] 節點下，展開您要管理其管理範圍使用者的伺服器實例或叢集配置。
  2. 展開 [安全性] 節點。
  3. 展開 [範圍] 節點。
  4. 選取 [file] 節點。

    [編輯範圍] 頁面會隨即開啟。

  5. 在 [編輯範圍] 頁面上，按一下 [管理使用者] 按鈕。

    [檔案使用者] 頁面會隨即開啟。

  6. 在 [檔案使用者] 頁面上，選取要刪除之使用者名稱左側的核取方塊。
  7. 按一下 [刪除]。

    對話方塊隨即出現，要求您確認是否應該刪除該使用者。

  8. 在對話方塊中，按一下 [確定]。
