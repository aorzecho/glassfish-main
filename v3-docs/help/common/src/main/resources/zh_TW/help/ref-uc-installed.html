已安裝的元件

「已安裝的元件」頁面針對已安裝之安裝影像的元件提供相關資訊。

若要解除安裝任何已安裝的元件，請先停止 GlassFish Server 網域。然後使用以下其中一個命令：

  • glassfish3/bin/updatetool，這會啟動更新工具圖形化公用程式

  • glassfish3/bin/pkg，這是命令行版本的更新工具

如需有關使用 pkg 命令的資訊，請參閱 Oracle GlassFish Server Administration Guide 中的第 11 章 Extending and Updating GlassFish Server。更新工具公用程式本身具有線上說明。

當您完成解除安裝元件後，請重新啟動伺服器。

「已安裝的元件」 頁面包含下列選項。

安裝路徑

根目錄的檔案系統路徑，或安裝影像的頂層目錄。

已安裝的元件

此表格提供有關每個已安裝元件的下列資訊。

元件

元件的名稱。

種類

與元件相關的軟體種類，例如應用程式伺服器、系統工具和資料庫。建立元件時，元件的作者會為元件指定種類。

版本

提供元件的套裝軟體版本號碼。

安裝的大小

安裝元件所使用的磁碟空間量。

來源

要從該處取得元件的網路儲存庫名稱。
