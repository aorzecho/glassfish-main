系統特性

使用 「系統特性」 頁面可檢視和管理 GlassFish Server 特性。

系統特性會在伺服器層級定義設定的一般值。若要在管理主控台文字欄位中參照系統特性，可用美元符號和大括號括住系統特性。例如，若要參照名為 prop-name 的特性，可使用語法 ${prop-name}。

「系統特性」 頁面可顯示每則訊息的下列資訊。

名稱

特性的名稱。

值

特性的值。

說明

特性的選擇性說明。

數個預先定義的系統特性不會顯示在此頁面上。如需詳細資訊，請參閱預先定義的系統特性。
