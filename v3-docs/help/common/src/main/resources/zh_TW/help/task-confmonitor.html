配置監視服務

  1. 在 [配置] 節點下，展開您要配置監視功能的伺服器實例或叢集配置。
  2. 展開 [監視] 節點。

    [監視服務] 頁面會隨即開啟。

  3. 選取 [啟用監視服務] 核取方塊，為 GlassFish Server 啟用監視，或取消選取核取方塊以停用監視。

    此選項預設為啟用。

  4. 選取 [啟用監視 MBean] 核取方塊，部署監視需要的所有 MBeans，或取消選取核取方塊以將這些 MBeans 解除部署。

    此選項預設為啟用。

    如果沒有選取此核取方塊，即使已選取 [監視服務] 核取方塊，您也無法在 管理主控台 上檢視監視資料。您必須使用指令行來檢視監視資料。如果您取消選取此核取方塊，GlassFish Server 的記憶體使用量會減少。

  5. 選取 [啟用 DTrace] 核取方塊，啟用 Solaris DTrace 功能。

    要使用 Solaris DTrace 功能，您也必須具有 Oracle Enterprise Manager DTrace Monitoring。 此選項預設為停用。

  6. 在表格中，執行下列其中一個動作。
    • 在 [監視層級] 欄中，針對 [元件] 欄中的一個或多個元件，從下拉式清單中選取 [低] 或 [高]，以啟用這些元件的監視功能。
    • 選取一個或多個元件旁的核取方塊，然後從表格頂端的 [層級] 下拉式清單中選取 [低] 或 [高]，再按一下 [變更層級] 按鈕，將所有元件的監視層級設為該值。

    在此發行版本中，[低] 與 [高] 選項所提供的監視層級並無不同。若要停用一個或多個元件的監視，請選取 [關閉]。預設值為 [OFF]。

    您必須先部署應用程式、元件或服務，才能為其配置監視。

  7. 按一下 [儲存]。
