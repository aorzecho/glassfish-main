系統特性

「系統特性」 頁面會顯示所選取已命名組態中定義的 Java 系統特性清單。GlassFish Server 啟動時，會透過 Java 應用程式啟動程式的 -D 選項，將 Java 系統特性傳送至 Java 應用程式啟動程式。

這些特性可為參照組態之所有 GlassFish Server 實例的連接埠設定定義預設值。

如需有關組態中預先定義之連接埠設定的資訊，請參閱預先定義的系統特性。

對於每個特性，都會顯示以下資訊：

實例變數名稱

系統特性的名稱。

預設值

為實例所參照之已命名組態中的特性所設定的值。本欄位是唯讀的。

實例值

連結至特性實例值頁面的連結。

其他特性表也包含下列選項。

增加特性

增加特性的按鈕。按一下此按鈕即可在其他特性表中增加一列。

刪除特性

刪除一或多個所選取特性的按鈕。刪除的任何特性都會復原成其預設值或未定義 (如果未設定任何預設值)。

「系統特性」 頁面也包含下列選項：

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

動態重新組態

若啟用此選項，不需要重新啟動實例，即可將組態變更套用至參照該組態的實例。

使用該組態的叢集和/或實例

參照組態的叢集與實例清單，指出每個叢集或實例是否處於執行中：

  • 如果叢集或實例正在執行中，便會顯示核取標記 (✓)。

  • 如果叢集或實例已停止，便會顯示驚嘆號 (!)。
