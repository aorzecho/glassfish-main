JVM 選項

使用 「JVM 選項」 頁面可指定執行 GlassFish Server 的 Java 應用程式啟動程式 (java) 的選項。-D 選項會指定專屬於 GlassFish Server 的特性。

  • 若要修改選項，請編輯選項的 「值」 欄位。

  • 若要增加選項，請按一下 「增加 JVM 選項」 按鈕。在顯示的空白列中，在 「值」 欄位中鍵入選項資訊。

    若選項資訊包含一或多個空格，請將整個字串括在雙引號中。

  • 若要刪除選項，請選取要刪除之選項 「值」 欄位左側的核取方塊，然後按一下 「刪除」。
