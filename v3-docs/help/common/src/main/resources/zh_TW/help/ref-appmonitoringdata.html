應用程式監視

使用 「應用程式監視」頁面可檢視所選應用程式的監視資料。

「應用程式監視」頁面包含以下資訊。

實例名稱

伺服器實例的名稱。

應用程式

顯示之監視統計資料所屬的應用程式。

元件

顯示之監視統計資料所屬的應用程式元件。

如需受監視特性的說明，請參閱 Oracle GlassFish Server Administration Guide 中的第 8 章 Administering the Monitoring Service。
