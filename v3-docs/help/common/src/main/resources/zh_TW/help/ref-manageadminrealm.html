管理使用者

使用 [管理使用者] 頁面可管理 admin-realm 使用者。 admin-realm 是 File 範圍。file 範圍中的使用者和群組會列於金鑰檔案，此檔案的位置則由 file 特性指定。

剛安裝完 GlassFish Server 之後，唯一的使用者是在安裝期間輸入的管理員。依預設，該使用者屬於 admin-realm 範圍中的 asadmin 群組，該群組有權修改 GlassFish Server。任何指定到該群組的使用者都將具有管理特權，亦即可以存取 asadmin 工具和管理主控台。

[管理使用者] 頁面會顯示所選取範圍的使用者清單。對於每個使用者，均提供以下資訊：

使用者 ID

使用者名稱。

群組清單

使用者所屬的群組。

管理使用者表也包含下列選項。

新建

建立新使用者的按鈕。

刪除

刪除一或多個選取使用者的按鈕。

相關作業

  • 增加使用者到管理範圍

  • 編輯管理範圍使用者資訊

  • 從管理範圍刪除使用者
