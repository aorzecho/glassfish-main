檢視 GlassFish Server 的 GlassFish 新聞頁面

「GlassFish 新聞」頁面提供 Oracle GlassFish Server 的其他資訊連結集合 (例如電子報、文章、技術秘訣、論壇和相關部落格)。

  1. 在瀏覽樹狀結構中，選取 「常用工作」 節點。

    「常用工作」 頁面會隨即開啟。

  2. 在 「GlassFish 新聞」 下，按一下 「GlassFish 新聞」 連結。

    「GlassFish 新聞」 頁面會在另外一個瀏覽器標籤或視窗中開啟。
