新建稽核模組

使用 「新建稽核模組」 頁面可定義新的稽核模組。

「新建稽核模組」 頁面包含下列選項。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

名稱

新稽核模組的名稱。

類別名稱

實作此模組的類別之完全合格的名稱。預設稽核模組的類別名稱為 com.sun.enterprise.security.Audit。

其他特性

稽核模組的其他特性。如需適用於稽核模組之其他特性的說明，請參閱稽核模組特定的特性。
