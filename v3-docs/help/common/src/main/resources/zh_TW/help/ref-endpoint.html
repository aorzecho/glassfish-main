Web 服務端點資訊

使用 [Web 服務端點資訊] 頁面檢視有關 Web 服務端點的資訊。

[Web 服務端點資訊] 頁面包含下列資訊。

應用程式名稱

所部署的應用程式的名稱。按一下此連結或 [返回] 按鈕，可以回到 [編輯應用程式] 頁面。

名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

測試器

只有在應用程式部署類型並非 RI (例如，類型為 109) 的情況下，[測試] 連結才會顯示。按一下此連結，可以檢視能讓您提供測試 Web 服務方法之參數的頁面。

WSDL

端點的 Web 服務定義語言 (WSDL)。 按一下此連結，可以在新的瀏覽器標籤或視窗中檢視 WSDL 檔案。

請注意，當您按一下 WSDL 連結時，有些 Web 瀏覽器會嘗試解譯 WSDL 來源，因而顯示空白頁面。對於此類瀏覽器，可在空白頁面上使用瀏覽器的 [檢視來源] 指令，檢視 WSDL 來源。

端點名稱

端點的名稱。

服務名稱

Web 服務的名稱。

連接埠名稱

Web 服務連接埠的名稱。

部署類型

Web 服務的部署類型 (例如，RI 或 109)。

實作類型

Web 服務的實作類型 (例如，SERVLET 或 EJB)。

實作類別名稱

Web 服務的實作類別名稱。

端點位址 URI

端點位址的 URI。

名稱空間

與 Web 服務關聯的目標名稱空間。

說明

端點 (如果存在) 的說明
