回復作業事件

使用 [回復作業事件] 頁面可回復 GlassFish Server 實例的作業事件。

如果實例在執行中，[回復作業事件] 頁面只包含文字「回復是由正在執行的伺服器自行執行」與 [確定] 按鈕。

如果實例不在執行中，[回復作業事件] 頁面會包含下列選項。

實例名稱

建立實例時指定給實例的名稱。本欄位是唯讀的。

目標伺服器

可在目前實例上回復作業事件的另一個實例。

作業事件記錄目錄

儲存作業事件記錄的目錄。預設是位於網域目錄下的 /logs/tx。

作業事件記錄保存在 tx 中，它是叢集配置之作業事件服務 [作業事件記錄位置] 設定所指定目錄的子目錄。[作業事件記錄位置] 預設值為 [網域屬性] 頁面上的 [記錄根目錄] 欄位所指定的目錄，該頁面位於 [GlassFish Server] 節點的 [進階] 標籤之下。
