檢視伺服器監視資料

開始之前

在檢視監視資料之前，您必須配置監視。如需詳細資訊，請參閱配置監視服務。

  1. 在瀏覽樹狀結構中，選取伺服器 (管理伺服器) 節點。

    此時將開啟 [一般資訊] 頁面。

  2. 在 [一般資訊] 頁面上，按一下 [監視] 標籤。

    此時將開啟 [應用程式監視] 頁面。

  3. 按一下 [伺服器] 標籤。

    此時將開啟 [伺服器監視] 頁面。

  4. 從 [檢視] 下拉式清單中，選取您要檢視其監視資料的元件。
  5. 按一下 [重新整理] 按鈕可更新顯示的監視資料。
