實例值

使用 「實例值」 頁面可為參照所選取已命名組態之 GlassFish Server 實例編輯組態系統特性值。

「實例值」 頁面會顯示參照所選取組態之實例的清單，其中已設定特性。

實例名稱

建立實例時指定給實例的名稱。按一下此名稱，隨即會開啟實例的 「一般資訊」 頁面。

叢集名稱

實例所屬的叢集 (如果存在) 名稱。本欄位是唯讀的。

預設值

為實例所參照之已命名組態中的特性所設定的值。本欄位是唯讀的。

置換值

為所選取實例所設定的特性值。此值會置換預設值。
