組態 JVM 路徑設定

  1. 在瀏覽樹狀結構中，選取 「組態」 節點。

  2. Select the configuration that you are changing.

  3. 在 「組態」 節點下，選取 「JVM 設定」 節點。

    「JVM 一般設定」 頁面會隨即開啟。

  4. 在 「JVM 一般設定」 頁面上，按一下 「路徑設定」 標籤。

    此時會開啟 「JVM 路徑設定」 頁面。

  5. 在 「本機程式庫路徑前綴」 欄位中，鍵入前置於本機程式庫路徑之 JAR 檔案的完整路徑。

    若要前置多個檔案，請一行鍵入一個路徑。

  6. 在 「本機程式庫路徑後綴」 欄位中，鍵入附加於本機程式庫路徑之 JAR 檔案的完整路徑。

    若要附加多個檔案，請一行鍵入一個路徑。

  7. 按一下 「儲存」。

  8. 重新啟動伺服器。
