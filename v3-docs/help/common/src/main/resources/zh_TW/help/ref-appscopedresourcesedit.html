修改應用程式範圍資源

  1. 在瀏覽樹狀結構中，選取 「應用程式」 節點。

    「應用程式」 頁面會隨即開啟。

  2. 選取您要檢視其應用程式範圍資源的應用程式。

    應用程式名稱中可以包含一個選擇性的版本識別碼，位於名稱之後，兩者間用冒號 (:) 隔開。如需有關模組與應用程式版本的更多資訊，請參閱 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

    該應用程式的頁面會隨即開啟。

  3. 選取「資源」標籤。

    「應用程式範圍資源」頁面會隨即開啟，而目前指定給該應用程式的資源會出現在含有下列各欄的表格中：

    應用程式名稱

    應用程式的名稱。

    模組名稱

    應用程式內的模組名稱。

    資源名稱

    應用程式或模組的應用程式範圍資源名稱。

    資源類型

    資源類型，例如「JDBC 連線集區」或「連接器資源」。

  4. 按一下您想要修改的資源名稱。

    開啟的頁面取決於資源類型。例如，如果資源是 JDBC 連線池，則「編輯 JDBC 連線池」頁面會隨即開啟。
