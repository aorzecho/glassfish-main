建立稽核模組

稽核是指 GlassFish Server 如何記錄重要事件，例如錯誤或安全性漏洞。GlassFish Server 在記錄檔中記錄所有認證事件。完整的存取記錄提供了 GlassFish Server 存取事件的循序追蹤。

GlassFish Server 提供一個簡單的預設稽核模組；如需更多資訊，請參閱使用預設稽核模組。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [安全性] 節點。
  3. 在 [安全性] 節點下，選取 [稽核模組] 節點。

    [稽核模組] 頁面會隨即開啟。

  4. 在 [稽核模組] 頁面上，按一下 [新建]。

    [新建稽核模組] 頁面會隨即開啟。

  5. 在 [新建稽核模組] 頁面的 [名稱] 欄位中，鍵入稽核模組的名稱。
  6. 在 [類別名稱] 欄位中，鍵入實作此模組之類別的完全合格名稱。

    預設稽核模組的類別名稱為 com.sun.enterprise.security.Audit。

  7. (可選擇) 在 [其他特性] 區段中，指定其他特性。

    若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。

    如需適用於稽核模組之其他特性的說明，請參閱稽核模組專屬的特性。

  8. 按一下 [確定]。
