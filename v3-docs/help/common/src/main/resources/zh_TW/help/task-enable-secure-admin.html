啟用安全管理

  1. 在瀏覽樹狀結構中，選取伺服器 (管理伺服器) 節點。

    此時將開啟 「一般資訊」 頁面。

  2. 在「一般資訊」頁面上，按一下「安全管理」按鈕。

    「安全管理」頁面會隨即開啟。

  3. 按一下「啟用安全管理」。

    會出現一則訊息，指出啟用安全管理會立即重新啟動 DAS。

  4. 按一下 「確定」。

    會啟用安全管理，而伺服器重新啟動頁面會隨即開啟。
