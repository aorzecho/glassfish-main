編輯範圍

使用 「編輯範圍」 頁面可修改範圍。

「編輯範圍」 頁面包含下列選項。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

範圍名稱

範圍的名稱。此為唯讀欄位。您僅能在建立新的範圍時指定名稱。

類別名稱

正在編輯之範圍的類別名稱。

其他特性

範圍的其他特性。有效的特性取決於範圍的類型。

  • FileRealm 類別特定的特性

  • CertificateRealm 類別特定的特性

  • JDBCRealm 類別特定的特性

  • LDAPRealm 類別特定的特性

  • SolarisRealm 類別特定的特性

  • PamRealm 類別特定的特性
