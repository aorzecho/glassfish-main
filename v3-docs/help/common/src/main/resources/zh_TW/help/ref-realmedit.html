編輯範圍

使用 [編輯範圍] 頁面可修改範圍。

[編輯範圍] 頁面包含下列選項。

配置名稱

此頁面上的設定要套用到的配置名稱。此為唯讀欄位。

範圍名稱

範圍的名稱。此為唯讀欄位。您僅能在建立新的範圍時指定名稱。

類別名稱

正在編輯之範圍的類別名稱。

其他特性

範圍的其他特性。有效的特性取決於範圍的類型。

  • 專屬於 FileRealm 類別的特性

  • 專屬於 CertificateRealm 類別的特性

  • 專屬於 JDBCRealm 類別的特性

  • LDAPRealm 類別特有的特性

  • SolarisRealm 類別特有的特性

  • 專屬於 PamRealm 類別的特性
