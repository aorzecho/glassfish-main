使用預設稽核模組

default 稽核模組將認證和授權請求記錄到伺服器記錄檔。如需有關變更記錄檔位置的資訊，請參閱配置一般記錄設定。

認證記錄項目包括以下資訊︰

  • 已嘗試進行認證的使用者名稱

  • 處理存取請求的範圍

  • 請求的 Web 模組 URI 或 EJB 元件

  • 請求成功或失敗

無論是否啟用了稽核記錄功能，GlassFish Server 都會記錄所有被拒絕的認證事件。

授權記錄項目包括以下資訊︰

  • 認證使用者 (如果有) 的名稱

  • 請求的 Web URI 或 EJB 元件

  • 請求成功或失敗

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  3. 從 [稽核模組] 下拉式清單中，選取名為 default 的稽核模組。
  4. 按一下 [儲存]。
