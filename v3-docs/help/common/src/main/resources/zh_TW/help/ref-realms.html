範圍

GlassFish Server 預先組態了三個範圍： file、certificate 以及 admin-realm。您也可以建立 ldap、JDBC、solaris 以及自訂範圍。伺服器上的每種類型通常只有一個範圍，但是在 GlassFish Server 上有以下兩個 file 範圍︰ file 和 admin-realm。同一類型有兩個範圍，用於兩種不同目的。系統上的每個虛擬伺服器可以有不同的憑證資料庫。

使用 「範圍」 頁面可組態範圍。

對於每個範圍，都將提供以下資訊。

名稱

範圍的名稱。

類別名稱

範圍的類別名稱。

範圍表也包含下列選項。

新建

建立新範圍的按鈕。

刪除

刪除一或多個選取範圍的按鈕。
