刪除管理範圍中的使用者

  1. 在 「組態」 節點下，展開您要管理其管理範圍使用者的伺服器實例或叢集組態。

  2. 展開 「安全性」 節點。

  3. 展開 「範圍」 節點。

  4. 選取 「admin-realm」 節點。

    「編輯範圍」 頁面會隨即開啟。

  5. 在 「編輯範圍」 頁面上，按一下 「管理使用者」 按鈕。

    「管理使用者」 頁面會隨即開啟。

  6. 在 「管理使用者」 頁面上，選取要刪除之使用者名稱左側的核取方塊。

  7. 按一下 「刪除」。

    系統會提示您確認刪除。

  8. 按一下 「確定」 以刪除使用者。
