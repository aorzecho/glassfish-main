增加使用者到管理範圍

  1. 在 [配置] 節點下，展開您要管理其管理範圍使用者的伺服器實例或叢集配置。
  2. 展開 [安全性] 節點。
  3. 展開 [範圍] 節點。
  4. 選取 [admin-realm] 節點。

    [編輯範圍] 頁面會隨即開啟。

  5. 在 [編輯範圍] 頁面上，按一下 [管理使用者] 按鈕。

    [管理使用者] 頁面會隨即開啟。

  6. 在 [管理使用者] 頁面上，按一下 [新建]。

    [新建 File 範圍使用者] 頁面會隨即開啟。

  7. 在 [使用者 ID] 欄位中，鍵入使用者的名稱。
  8. 在 [群組清單] 欄位中，鍵入使用者所屬的群組清單。

    請使用逗號分隔群組名稱。這些群組無需在其他地方定義。

  9. 在 [密碼] 欄位中，鍵入唯一的密碼。
  10. 在 [確認新密碼] 欄位中，再次鍵入使用者密碼。
  11. 按一下 [確定]。
