可用的附加元件

「可用的附加元件」頁面針對尚未安裝之安裝影像的元件提供相關資訊。

「可用的附加元件」 頁面包含下列選項。

安裝路徑

根目錄的檔案系統路徑，或安裝影像的頂層目錄。

安裝

安裝一或多個選取附加元件的按鈕。

可用的附加元件

此表格提供有關未安裝之安裝影像的每個元件的下列資訊。

元件

元件的名稱。

種類

與元件相關的軟體種類，例如應用程式伺服器、系統工具和資料庫。建立元件時，元件的作者會為元件指定種類。

版本

提供元件的套裝軟體版本號碼。

安裝大小

安裝元件所使用的磁碟空間量。

來源

要從該處取得元件的網路儲存庫名稱。
