密碼別名

使用 [密碼別名] 頁面可建立和管理密碼別名。

[密碼別名] 頁面會顯示網域中的密碼別名清單。對於每個別名，都會顯示以下資訊：

名稱

在密碼別名建立時指定的名稱。按一下該名稱，隨即會開啟該別名的 [編輯密碼別名] 頁面。

[密碼別名] 表格也包含下列選項。

新建

建立密碼別名的按鈕。

刪除

刪除一或多個所選取密碼別名的按鈕。

等效 asadmin 子指令

  • list-password-aliases(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • update-password-alias(1)
