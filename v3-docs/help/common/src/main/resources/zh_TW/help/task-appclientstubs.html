下載應用程式用戶端 Stub

  1. 在瀏覽樹狀結構中，選取 「應用程式」 節點。

    「應用程式」 頁面會隨即開啟。

  2. 從「已建置的應用程式」表格中，按一下您要下載 Stub 的應用程式用戶端名稱。

    名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

    「編輯應用程式」 頁面會隨即開啟。

  3. 在「模組和元件」表格中，按一下「動作」欄下的「下載用戶端 Stub」。

    您可以使用瀏覽器的下載機制下載 client-stubs.zip 檔案。
