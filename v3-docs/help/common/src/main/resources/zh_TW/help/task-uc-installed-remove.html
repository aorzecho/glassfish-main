檢視已安裝的元件

[已安裝的元件] 頁面針對已安裝之安裝影像的元件提供相關資訊。

解除安裝元件包括移除下列項目：

  • 最初安裝元件時加入的所有檔案和目錄

  • 最初安裝元件後更新元件時加入的所有檔案和目錄

  1. 在瀏覽樹狀結構中，選取 [更新工具] 節點。

    [可用的附加元件] 頁面會隨即開啟。

  2. 在 [可用的附加元件] 頁面上，按一下 [已安裝的元件] 標籤。

    [已安裝的元件] 頁面會隨即開啟。

接下來的步驟

若要解除安裝任何已安裝的元件，請先停止 GlassFish Server 網域。然後使用以下其中一個指令：

  • as-install-parent/bin/updatetool，這會啟動更新工具圖形化公用程式

  • as-install-parent/bin/pkg，這是指令行版本的更新工具

如需有關使用 pkg 指令的資訊，請參閱「Oracle GlassFish Server 3.1 Administration Guide」中的第 11 章「Extending and Updating GlassFish Server」。更新工具公用程式本身具有線上說明。

當您完成解除安裝元件後，請重新啟動伺服器。
