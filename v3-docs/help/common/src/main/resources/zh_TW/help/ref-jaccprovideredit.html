編輯 JACC 提供者

使用 [編輯 JACC 提供者] 頁面可修改所選取 GlassFish Server 實例或叢集的 JACC 提供者。

[編輯 JACC 提供者] 頁面包含下列選項。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

名稱

JACC 提供者的名稱。此為唯讀欄位。您僅能在建立新的 JACC 提供者時指定名稱。

策略配置

javax.security.jacc.PolicyConfigurationFactory.provider 的完全合格類別名稱，會實作提供者特定的 javax.security.jacc.PolicyConfigurationFactory。

策略提供者

javax.security.jacc.policy.provider 的完全合格類別名稱，會實作 java.security.Policy。

其他特性

JACC 提供者的其他特性。如需適用於 JACC 提供者之其他特性的說明，請參閱JACC 提供者專屬的特性。
