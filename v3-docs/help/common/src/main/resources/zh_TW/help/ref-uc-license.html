授權合約

「授權合約」 頁面會顯示為安裝或更新所選取之所有元件的授權合約。元件的授權合約由元件作者透過元件提供。網路儲存庫中的不同元件可能會根據不同的授權發行，例如：

  • 通用開發與散佈授權 (Common Development and Distribution License，CDDL) (http://www.sun.com/cddl/)

  • GNU 通用公共授權 (GNU General Public License，GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Apache 授權 (http://www.apache.org/licenses/)

  • wxWindows Library 授權 (http://www.opensource.org/licenses/wxwindows.php)

「授權合約」 頁面包含下列選項。

接受

此按鈕表示接受所有選取元件的授權條款，並繼續安裝或升級。

拒絕

此按鈕表示拒絕所有選取元件的授權條款，並返回選取元件的頁面。
