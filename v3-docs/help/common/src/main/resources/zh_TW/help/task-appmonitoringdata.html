檢視應用程式監視資料

開始前

在檢視監視資料之前，您必須組態監視。如需詳細資訊，請參閱組態監視服務。

  1. 在瀏覽樹狀結構中，選取「監視資料」節點。

    「監視」頁面會隨即開啟。

  2. 在「檢視監視資料」欄中，按一下與所監視伺服器實例對應的「應用程式」連結。

    此時將開啟 「應用程式監視」 頁面。

  3. 從「應用程式」下拉式清單中，選取您要檢視其監視資料的應用程式。

  4. 從「元件」下拉式清單中，選取您要檢視其監視資料的應用程式元件。

  5. 按一下 「重新整理」 按鈕可更新顯示的監視資料。

替代步驟：

  1. 在瀏覽樹狀結構中，選取伺服器 (管理伺服器) 節點。

    此時將開啟 「一般資訊」 頁面。

  2. 在 「一般資訊」 頁面上，按一下 「監視」 標籤。

    此時將開啟 「應用程式監視」 頁面。

  3. 從「應用程式」下拉式清單中，選取您要檢視其監視資料的應用程式。

  4. 從「元件」下拉式清單中，選取您要檢視其監視資料的應用程式元件。

  5. 按一下 「重新整理」 按鈕可更新顯示的監視資料。
