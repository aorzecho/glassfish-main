安全管理

使用「安全管理」頁面，可啟用或停用安全管理。

當安全管理啟用時：

  • 網域中的 DAS 與實例會使用 SSL 憑證加密彼此間傳送的訊息。

  • DAS 可接受來自遠端管理用戶端 (例如 asadmin 公用程式與 IDE) 的管理訊息。

「安全管理」頁面包含以下資訊。

啟用安全管理

啟用安全管理的按鈕。只有在未啟用安全管理時，才會顯示此按鈕。

注意:

如果網域中的任一管理使用者密碼空白，則嘗試啟用安全管理將會失敗。
停用安全管理

停用安全管理的按鈕。只有在啟用安全管理時，才會顯示此按鈕。

管理別名

DAS 上的 SSL/TLS 憑證的別名。DAS 會使用此別名向實例表明其身分。預設值為 s1as。只有在未啟用安全管理時，才會顯示此選項。

實例別名

實例上 SSL/TLS 憑證的別名。實例會使用此別名向 DAS 表明其身分。預設值為 glassfish-instance。只有在未啟用安全管理時，才會顯示此選項。
