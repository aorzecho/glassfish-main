訊息安全性配置

使用 [訊息安全性配置] 頁面可針對訊息安全性提供者檢視和修改可用的配置。

若要配置訊息安全性，您需要對安全性和 Web 服務概念有基本瞭解。如需更多資訊，請參閱「Oracle GlassFish Server 3.1 Security Guide」中的第 3 章「Administering Message Security」，以及「Oracle GlassFish Server 3.1 Administration Guide」中的第 6 章「Administering Web Applications」。

&ProductName 使用 SOAP 處理層中整合的訊息安全性提供者來實作訊息安全性。訊息安全性提供者取決於 GlassFish Server 的其他安全性功能。

對於每個配置，都將提供以下資訊。

認證層

執行認證的訊息層。根據預設，會提供 SOAP 配置。您也可以提供 HttpServlet 配置。

預設提供者

此配置的預設伺服器提供者。根據預設，不會配置預設伺服器提供者。

預設用戶端提供者

此配置的預設用戶端提供者。根據預設，不會配置預設用戶端提供者。

訊息安全性配置表也包含下列選項。

新建

建立新配置的按鈕。

刪除

刪除一或多個選取配置的按鈕。
