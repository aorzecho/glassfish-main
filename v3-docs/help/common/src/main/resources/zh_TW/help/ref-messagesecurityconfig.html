訊息安全性組態

使用 「訊息安全性組態」 頁面可針對訊息安全性提供者檢視和修改可用的組態。

若要組態訊息安全性，您需要對安全性和 Web 服務概念有基本瞭解。如需詳細資訊，請參閱 Oracle GlassFish Server Security Guide 中的第 3 章 Administering Message Security 與 Oracle GlassFish Server Administration Guide 中的第 6 章 Administering Web Applications。

&ProductName 使用 SOAP 處理層中整合的訊息安全性提供者來實作訊息安全性。訊息安全性提供者取決於 GlassFish Server 的其他安全性功能。

對於每個組態，都將提供以下資訊。

認證層

執行認證的訊息層。根據預設，會提供 SOAP 組態。您也可以提供 HttpServlet 組態。

預設提供者

此組態的預設伺服器提供者。根據預設，不會組態預設伺服器提供者。

預設用戶端提供者

此組態的預設用戶端提供者。根據預設，不會組態預設用戶端提供者。

訊息安全性組態表也包含下列選項。

新建

建立新組態的按鈕。

刪除

刪除一或多個選取組態的按鈕。
