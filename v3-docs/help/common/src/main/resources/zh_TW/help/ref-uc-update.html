可用的更新

[可用的更新] 頁面針對已安裝之安裝影像的更新版本元件提供相關資訊。

若要安裝可用的更新，請先停止 GlassFish Server 網域。然後使用以下其中一個指令：

  • glassfish3/bin/updatetool，這會啟動更新工具圖形公用程式

  • glassfish3/bin/pkg，這是指令行版本的更新工具

如需有關使用 pkg 指令的資訊，請參閱「Oracle GlassFish Server 3.1 Administration Guide」中的第 11 章「Extending and Updating GlassFish Server」。更新工具公用程式本身具有線上說明。

當您完成安裝更新後，請重新啟動伺服器。

[可用的更新] 頁面包含下列選項。

安裝路徑

根目錄的檔案系統路徑，或安裝影像的頂層目錄。

可用的更新

此表格提供有關每個具有可用更新版本元件的下列資訊。

元件

元件的名稱。

種類

與元件相關的軟體種類，例如應用程式伺服器、系統工具和資料庫。建立元件時，元件的作者會為元件指定種類。

安裝的版本

提供目前元件的套裝軟體版本號碼。

新版本

可用更新的版本號碼。

變更

安裝的版本和可用的更新之間的大小差異。

來源

要從該處取得元件的網路儲存庫名稱。
