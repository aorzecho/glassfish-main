編輯生命週期模組

使用 「編輯生命週期模組」 頁面可修改生命週期模組的設定。

「編輯生命週期模組」 頁面包含下列選項。

名稱

生命週期模組的名稱。名稱是唯讀欄位。您僅能在建立新的生命週期模組時指定名稱。

類別名稱

生命週期模組之類別檔案的完全合格名稱。此類別必須實作 com.sun.appserv.server.LifecycleListener 介面。

類別路徑

生命週期模組的類別路徑。指定生命週期模組所在的類別路徑。預設值為 domain-dir/applications。如果模組已在此目錄中 (也就是在伺服器類別路徑中)，此欄位可保留空白。

載入順序

此生命週期模組在電腦啟動時的載入順序。模組的載入順序整數值越小，越快載入。此值的範圍可介於 101 到作業系統的 MAXINT。1 到 100 是保留值。

說明

生命週期模組的說明。

狀態

若選取此選項，則會啟用生命週期模組。此選項預設為啟用。

載入失敗時

若選取此選項，如果生命週期模組無法載入，伺服器將關閉。此選項預設為停用。
