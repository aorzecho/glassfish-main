生命週期模組

使用 「生命週期模組」 頁面可組態生命週期模組。

生命週期模組在伺服器生命週期中有一個或多個事件觸發時會執行工作。可能觸發該模組的伺服器事件包括：初始化、啟動、準備處理請求以及關機。生命週期模組不屬於 Java EE 規格，而是 GlassFish Server 的增強功能。

對於每個生命週期模組，均提供以下資訊。

名稱

生命週期模組的名稱。

已啟用或狀態

核取標記 (如果已啟用應用程式) 或 X (如果已停用應用程式)，唯有預設伺服器實例 server 存在時才提供此資訊。

如果有叢集或其他獨立伺服器實例存在，選取 「已在所有目標上啟用」 可以檢視建置應用程式的目標。

載入順序

指定載入生命週期模組時，應採用的相對順序值。

生命週期模組表也包含下列選項。

新建

建立新生命週期模組的按鈕。

刪除

刪除一或多個選取的生命週期模組的按鈕。

啟用

啟用一或多個選取的生命週期模組的按鈕。唯有預設伺服器實例 server 存在時才會顯示。

停用

停用一或多個選取的生命週期模組的按鈕。唯有預設伺服器實例 server 存在時才會顯示。
