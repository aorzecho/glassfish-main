啟用或停用稽核記錄

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  3. 在 [安全性] 頁面上，選取 [稽核記錄已啟用] 核取方塊來啟用稽核記錄。

    若要啟用稽核記錄，您也必須將使用中稽核模組的 auditOn 特性設為 true。

  4. 按一下 [儲存]。
