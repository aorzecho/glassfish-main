原始記錄檢視器

使用「原始記錄檢視器」頁面，可檢視管理伺服器 (名為 server) 的原始記錄檔。

「原始記錄檢視器」頁面包含下列選項。

實例

將為其顯示記錄的伺服器實例或叢集。唯一的選擇是 server。

「原始記錄檢視器」針對每個訊息會顯示下列資訊。

時間戳記

發出記錄訊息的日期和時間。

條目號碼

訊息在伺服器記錄中的號碼。

記錄層級

訊息的記錄層級。

版本

GlassFish Server 的版本。

記錄程式

產生記錄訊息的記錄程式模組。

名稱/值組

記錄訊息的執行緒名稱和 ID。

訊息

記錄訊息的簡短文字。按一下 「(details)」 連結可檢視訊息的格式化版本。
