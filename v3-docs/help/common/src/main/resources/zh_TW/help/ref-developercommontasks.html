常用工作

「常用工作」 頁面提供常用管理主控台作業的捷徑。

您可以使用 「常用工作」 頁面執行以下作業︰

  • 檢視 GlassFish Server 的 GlassFish 新聞頁面

  • 檢視 GlassFish Server 的訂閱頁面

  • 檢視已建置應用程式的清單

  • 建置 Web 應用程式

  • 建置企業應用程式

  • 建置應用程式用戶端

  • 建置連接器模組

  • 建置 EJB Jar

  • 組態監視服務

  • 檢視應用程式監視資料

  • 檢視伺服器監視資料

  • 檢視資源監視資料

  • 檢視已安裝的元件

  • 安裝附加元件

  • 檢視元件的可用更新

  • 建立 JDBC 連線集區

「常用工作」 頁面還提供部分 GlassFish Server 文件的連結。
