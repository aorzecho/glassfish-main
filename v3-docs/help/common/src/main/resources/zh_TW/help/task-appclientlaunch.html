啟動應用程式用戶端

開始之前

請確定您要啟動的應用程式用戶端已啟用。用戶端必須是能夠使用 Java Web Start 執行的 Web 用戶端。

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 從 [已部署的應用程式] 表格中，按一下您要啟動的應用程式用戶端名稱。

    名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

    [編輯應用程式] 頁面會隨即開啟。

  3. 確認已選取 [Java Web Start] 核取方塊。

    在啟動應用程式用戶端之前，必須先啟用 Java Web Start。 如果沒有選取 [Java Web Start] 核取方塊，請選取它然後按一下 [儲存]。

  4. 在 [模組和元件] 表中，按一下 [動作] 欄下的 [啟動]。

    [啟動應用程式用戶端] 頁面隨即開啟。

  5. 在 [啟動應用程式用戶端] 頁面中，鍵入要傳送給應用程式用戶端的引數。
  6. 按一下 [啟動]。
