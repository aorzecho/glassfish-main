增加資源

使用「增加資源」頁面，可從指定目標上的 glassfish-resources.xml 檔案建立一或多個資源。

「增加資源」頁面包含以下資訊。

位置

glassfish-resources.xml 檔案的位置：

要上傳至伺服器的 XML 檔案

遠端機器上之檔案的路徑。

可從 GlassFish Server 存取的本機 XML 檔案

網域管理伺服器 (DAS) 機器上的檔案路徑。

目標

叢集與獨立伺服器實例的下拉式清單。

相關作業

增加 XML 檔案中組態的資源

相關的 asadmin 命令

add-resources(1)
