檢視 Web 服務的端點資訊

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 在 [應用程式] 頁面上，按一下您要檢視其端點資訊的 Web 服務名稱。

    名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

    [編輯應用程式] 頁面會隨即開啟。

  3. 在 [模組和元件] 表中，按一下子元件的 [檢視端點] 連結。

    會開啟 [Web 服務端點資訊] 頁面。

  4. 如果 [測試器] 連結有顯示，按一下此連結，可以檢視能讓您提供測試 Web 服務方法之參數的頁面。

    只有在應用程式部署類型並非 RI (例如，類型為 109) 的情況下，[測試] 連結才會顯示。將在新的瀏覽器標籤或視窗中開啟 [測試器] 頁面。

  5. 按一下 [WSDL] 連結，可以檢視端點的 Web 服務定義語言 (WSDL) 檔案。

    將在新的瀏覽器標籤或視窗中開啟 WSDL 檔案。

    請注意，當您按一下 WSDL 連結時，有些 Web 瀏覽器會嘗試解譯 WSDL 來源，因而顯示空白頁面。對於此類瀏覽器，可在空白頁面上使用瀏覽器的 [檢視來源] 指令，檢視 WSDL 來源。

  6. 按一下 [應用程式名稱] 連結或 [返回] 按鈕，可以回到 [編輯應用程式] 頁面。
