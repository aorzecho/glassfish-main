刪除訊息安全性提供者配置

開始之前

無法建立或刪除預設 server-config 配置的訊息安全性提供者。預設 server-config 所支援的訊息安全性提供者只有 SOAP 與 HttpServlet。因為安裝 GlassFish Server 時，預設已經建立這兩個訊息提供者，所以在預設 server-config 的 [訊息安全性配置] 頁面上，不會顯示 [刪除] 按鈕。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 展開 [安全性] 節點。
  3. 選取 [訊息安全性] 節點。

    [訊息安全性配置] 頁面會隨即開啟。

  4. 在 [訊息安全性配置] 頁面上，選取 [SOAP]。

    [編輯訊息安全性配置] 頁面會隨即開啟。

  5. 在 [編輯訊息安全性配置] 頁面上，按一下 [提供者] 標籤。

    [編輯提供者配置] 頁面會隨即開啟。

  6. 在 [編輯提供者配置] 頁面上，選取您要刪除的提供者配置左側的核取方塊。
  7. 按一下 [刪除]。
