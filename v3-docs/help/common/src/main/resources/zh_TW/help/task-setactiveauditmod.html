設定使用中的稽核模組

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  3. 從 [稽核模組] 下拉式清單中，選取要由伺服器使用的稽核模組之名稱。

    根據預設，只有一個名為 default 的稽核模組可用。

  4. 按一下 [儲存]。
