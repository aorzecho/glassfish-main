編輯檔案範圍使用者

使用 「編輯 File 範圍使用者」 頁面可變更目前範圍的使用者密碼。

注意:

如果啟用安全管理，則不能將 admin-realm 使用者的密碼變更為空白值。

「編輯 File 範圍使用者」 頁面包含下列選項。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

範圍名稱

目前所選範圍的名稱。此為唯讀欄位。

使用者 ID

正在編輯之帳號的名稱。此為唯讀欄位。

群組清單

此使用者所屬的群組。以逗號分隔群組名稱。

針對 admin-realm 使用者，此為唯讀欄位，群組為 asadmin。

新密碼

此使用者的新密碼。

確認新密碼

此使用者的新密碼。

  • 刪除檔案範圍中的使用者
