預先定義的系統特性

GlassFish Server 會定義下列配置系統特性：

ASADMIN_LISTENER_PORT

此特性指定 HTTP 連接埠或 HTTPS 連接埠的連接埠號碼，DAS 透過該連接埠連線至實例以管理實例。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

HTTP_LISTENER_PORT

此特性指定要用來偵聽 HTTP 請求之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

HTTP_SSL_LISTENER_PORT

此特性指定要用來偵聽 HTTPS 請求之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

IIOP_LISTENER_PORT

此特性指定用於 IIOP 連線之連接埠的連接埠號碼。此特性指定用於 IIOP 連線之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

IIOP_SSL_LISTENER_PORT

此特性指定用於安全 IIOP 連線之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

IIOP_SSL_MUTUALAUTH_PORT

此特性指定用於與用戶端認證之安全 IIOP 連線之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

JAVA_DEBUGGER_PORT

此特性可指定用來連線至 Java 平台除錯程式架構 (JPDA) 除錯程式之連接埠的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

JMS_PROVIDER_PORT

此特性會指定 Java Message Service 提供者的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

JMX_SYSTEM_CONNECTOR_PORT

此特性會指定 JMX 連接器進行偵聽的連接埠號碼。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。

OSGI_SHELL_TELNET_PORT

此特性會指定用來連線至 Apache Felix 遠端 Shell 之連接埠的連接埠號碼。此 Shell 會使用 Felix Shell 服務來與 OSGi 模組管理子系統互動。有效值為 1–65535。在 UNIX 上，建立偵聽連接埠 1–1024 的通訊端需要超級使用者權限。
