收集和下載網域記錄

  1. 在瀏覽樹狀結構中，選取「網域」節點。

    「網域屬性」 頁面會隨即開啟。

  2. 選取「網域記錄」標籤。

    「網域記錄」頁面會隨即開啟。

    如果叢集存在，則會顯示叢集和實例的下拉式清單。如果叢集不存在，則只會顯示實例的下拉式清單。

  3. 選取「叢集」或「實例」。

  4. 選取要收集和下載記錄檔的叢集或實例。

  5. 按一下「收集記錄」。

    您瀏覽器的下載機制會提示您下載 log-files-name.zip 檔案，其中 name 是叢集或實例的名稱。
