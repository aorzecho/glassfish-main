預先定義的系統特性

GlassFish Server 會預先定義下列系統特性。

com.sun.aas.installRoot

安裝 GlassFish Server 的目錄。

com.sun.aas.instanceRoot

GlassFish Server 網域的頂層目錄。

com.sun.aas.hostName

主機 (電腦) 的名稱。

com.sun.aas.javaRoot

Java 執行階段的安裝目錄。

com.sun.aas.imqLib

Oracle Message Queue 軟體的程式庫目錄。

com.sun.aas.domainName

網域的名稱。此特性不會用於預設配置，但可用來自訂配置。預設值為 domain1。
