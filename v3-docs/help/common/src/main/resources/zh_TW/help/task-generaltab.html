檢視 DAS 的一般資訊

  1. 在瀏覽樹狀結構中，選取伺服器 (管理伺服器) 節點。

    此時將開啟 「一般資訊」 頁面。

  2. 在 「一般資訊」 頁面上，執行下列任何步驟。

    • 查看唯讀「狀態」顯示，以確認伺服器是否執行中。

    • 查看唯讀「正常執行時間」顯示，以確認伺服器持續執行的小時數和分鐘數。

    • 按一下 「停止」 按鈕以停止 GlassFish Server。

    • 按一下 「重新啟動」 按鈕以停止和重新啟動 GlassFish Server。

    • 按一下「檢視記錄檔」按鈕，可檢視 GlassFish Server 實例或叢集的記錄檔。

      如需詳細資訊，請參閱檢視 DAS 的記錄檔。

    • 按一下「檢視原始記錄」按鈕，可檢視 DAS 的原始記錄檔。

      如需詳細資訊，請參閱檢視 DAS 的原始記錄檔。

    • 按一下 「JVM 報告」 以開啟另外一個視窗，其中您可檢視 Java 虛擬機器的相關報告，包括摘要報告、記憶體管理報告、類別載入報告以及目前執行緒傾印。
