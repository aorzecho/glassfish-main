檢視 DAS 的記錄檔

  1. 在瀏覽樹狀結構中，選取伺服器 (管理伺服器) 節點。

    此時將開啟 [一般資訊] 頁面。

  2. 在 [一般資訊] 頁面上，按一下 [檢視記錄檔] 按鈕。

    此時將開啟 [記錄檢視器] 視窗。
