稽核模組

使用 [稽核模組] 頁面可配置稽核模組。

稽核是指 GlassFish Server 如何記錄重要事件，例如錯誤或安全性漏洞。GlassFish Server 在記錄檔中記錄所有認證事件。完整的存取記錄提供了 GlassFish Server 存取事件的循序追蹤。

對於每個稽核模組，均提供以下資訊。

名稱

稽核模組的名稱。

類別名稱

實作此模組的類別之完全合格的名稱。預設稽核模組的類別名稱為 com.sun.enterprise.security.Audit。

模組表也包含下列選項。

新建

建立新稽核模組的按鈕。

刪除

刪除一或多個選取的稽核模組的按鈕。
