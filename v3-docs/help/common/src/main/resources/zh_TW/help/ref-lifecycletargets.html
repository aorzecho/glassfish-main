生命週期模組目標

使用 [生命週期模組目標] 頁面可檢視能在其上啟用生命週期模組的目標叢集與獨立伺服器實例。

[生命週期模組目標] 頁面包含以下資訊。

目標名稱

叢集或獨立伺服器實例的名稱。

已啟用

啟用目標上的生命週期模組時顯示 true；停用生命週期模組時顯示 false。

[目標] 表格也包含下列選項。

管理目標

管理生命週期模組目標的按鈕。

更多動作

下列動作的下拉式清單。

啟用

在所選取目標上啟用生命週期模組的動作。

停用

在所選取目標上停用生命週期模組的動作。
