編輯 File 範圍使用者資訊

  1. 在 [配置] 節點下，展開您要管理其管理範圍使用者的伺服器實例或叢集配置。
  2. 展開 [安全性] 節點。
  3. 展開 [範圍] 節點。
  4. 選取 [file] 節點。

    [編輯範圍] 頁面會隨即開啟。

  5. 在 [編輯範圍] 頁面上，按一下 [管理使用者] 按鈕。

    [檔案使用者] 頁面會隨即開啟。

  6. 在 [檔案使用者] 頁面上表格的 [使用者 ID] 欄中，按一下要修改的使用者名稱。

    [編輯 File 範圍使用者] 頁面會隨即開啟。

  7. 編輯 [群組清單] 欄位以修改使用者所屬的群組。

    請使用逗號分隔群組名稱。這些群組無需在其他地方定義。

  8. 在 [新密碼] 和 [確認新密碼] 欄位中鍵入新密碼，以變更使用者的密碼。
  9. 按一下 [儲存]。
