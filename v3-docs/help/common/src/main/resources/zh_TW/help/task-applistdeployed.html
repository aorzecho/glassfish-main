檢視已部署應用程式清單

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。表格會列出已部署的應用程式。

  2. 從 [篩選] 下拉式清單選取模組類型，依模組類型篩選應用程式清單。
