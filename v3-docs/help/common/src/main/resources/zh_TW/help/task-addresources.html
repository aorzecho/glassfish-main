增加 XML 檔案中組態的資源

  1. 在瀏覽樹狀結構中，選取「資源」節點。

    「資源」頁面會隨即開啟。

  2. 按一下「增加資源」按鈕。

    「增加資源」頁面會隨即開啟。

  3. 選取下列其中一個選項，可指定 glassfish-resources.xml 檔案的位置：

    要上傳至伺服器的 XML 檔案

    如果檔案位於遠端機器上，請鍵入檔案的路徑。

    可從 GlassFish Server 存取的本機 XML 檔案

    如果檔案位於網域管理伺服器 (DAS) 機器上，請鍵入檔案的路徑。

  4. 從「目標」下拉式清單中，選取在其上建立資源的叢集或獨立伺服器實例。

  5. 按一下 「確定」。
