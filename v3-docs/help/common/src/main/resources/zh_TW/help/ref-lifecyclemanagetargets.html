管理目標

使用 [管理資源目標] 頁面可變更能在其上啟用生命週期模組的目標叢集與獨立伺服器實例。您只能在 [選取的目標] 欄中，啟用目標上的生命週期模組。

[管理資源目標] 頁面包含以下資訊。

可用的目標

無法啟用生命週期模組的叢集與實例。

選取的目標

可以啟用生命週期模組的叢集與實例。

增加

可將選取的目標從 [可用的目標] 移至 [選取的目標] 的按鈕。

全部加入

可將所有 [可用的目標] 移至 [選取的目標] 的按鈕。

移除

可將選取的目標從 [選取的目標] 移至 [可用的目標] 的按鈕。

全部移除

可將所有 [選取的目標] 移至 [可用的目標] 的按鈕。
