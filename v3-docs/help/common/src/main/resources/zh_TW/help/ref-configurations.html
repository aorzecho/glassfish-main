組態

使用「配置」頁面可管理配置，並檢視使用配置的獨立伺服器實例與叢集。

default-config 是 GlassFish Server 預先定義的配置。網域建立時，會為網域管理伺服器 (DAS) 自動建立 server-config 配置。如需更多資訊，請參閱 Oracle GlassFish Server High Availability Administration Guide 中的 About Named Configurations。

對於每個組態，都將提供以下資訊。

名稱

組態的名稱。

實例

使用配置之獨立實例與叢集的清單。

「配置」表格也包含下列選項。

新建

建立新組態的按鈕。

刪除

刪除一或多個選取組態的按鈕。
