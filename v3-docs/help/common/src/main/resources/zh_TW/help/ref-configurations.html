配置

使用 [配置] 頁面可管理配置，並使用配置來檢視獨立伺服器實例與叢集。

default-config 配置是由 GlassFish Server 預先定義的。server-config 配置是在建立網域時自動為 Domain Administration Server (DAS) 建立的。如需更多資訊，請參閱「Oracle GlassFish Server 3.1 High Availability Administration Guide」中的「About Named Configurations」。

對於每個配置，都將提供以下資訊。

名稱

配置的名稱。

實例

使用配置之獨立實例與叢集的清單。

[配置] 表格也包含下列選項。

新建

建立新配置的按鈕。

刪除

刪除一或多個選取配置的按鈕。
