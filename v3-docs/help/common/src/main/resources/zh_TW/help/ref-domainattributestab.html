網域屬性

使用 「網域屬性」 頁面可設定 GlassFish Server 的進階網域屬性。

「網域屬性」 頁面包含下列選項：

應用程式根

已建置應用程式所在的目錄完整路徑。預設值為 domain-dir/applications。

記錄檔根

指定儲存伺服器記錄檔的位置。預設值為 domain-dir/logs。

語言環境

若此欄位保留為空白，則會使用主機的預設語言環境。

語言環境是指定特定的語言和區域組合的識別碼。例如，美式英文的語言環境為 en_US，日文的語言環境為 ja_JP。若要使用非英文的語言環境，必須對 GlassFish Server 進行本土化 ，此程序包括將英文翻譯為其他語言。
