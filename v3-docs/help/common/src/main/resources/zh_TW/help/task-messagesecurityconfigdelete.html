刪除訊息安全性配置

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 展開 [安全性] 節點。
  3. 選取 [訊息安全性] 節點。

    [訊息安全性配置] 頁面會隨即開啟。

  4. 在 [訊息安全性配置] 頁面上，選取要刪除的訊息安全性配置左側的核取方塊。

    備註 - 無法建立或刪除預設 server-config 配置的訊息安全性提供者。預設 server-config 所支援的訊息安全性提供者只有 SOAP 與 HttpServlet。因為安裝 GlassFish Server 時，預設已經建立這兩個訊息提供者，所以在預設 server-config 的 [訊息安全性配置] 頁面上，不會顯示 [刪除] 按鈕。

  5. 按一下 [刪除]。
