更新工具配置

[配置] 頁面可提供有關 更新工具 的配置資訊，還可以讓您變更 更新工具 連線至網際網路的方式。若要變更 更新工具 配置的其他設定，請使用位於 as-install-parent/bin 中的獨立 updatetool 應用程式。

[更新工具配置] 頁面包含下列選項。

安裝路徑

根目錄的檔案系統路徑，或安裝影像的頂層目錄。

喜好的儲存庫

與此安裝影像相關之喜好的網路儲存庫名稱，例如 glassfish.org。

網際網路連線

GlassFish Server 連線至網際網路的方式：

直接連線

此選項指定 GlassFish Server 直接連線至網際網路。

透過代理

此選項指定 GlassFish Server 透過代理伺服器連線至網際網路。選取此選項時，即會將 [代理主機] 欄位和 [代理連接埠] 欄位增加至此頁面。

代理主機

GlassFish Server 執行所在機器的代理伺服器主機名稱。

此欄位僅在 GlassFish Server 執行所在機器透過代理伺服器連線至網際網路時才適用。

代理連接埠

GlassFish Server 執行所在機器的代理伺服器連接埠號碼。

此欄位僅在 GlassFish Server 執行所在機器透過代理伺服器連線至網際網路時才適用。

儲存庫

此表格提供有關與此安裝影像相關之每個網路儲存庫的下列資訊：

儲存庫

儲存庫的名稱，例如 dev.glassfish.org。

相關作業

配置網路代理伺服器設定以存取網路儲存庫

相關的 asadmin 指令
