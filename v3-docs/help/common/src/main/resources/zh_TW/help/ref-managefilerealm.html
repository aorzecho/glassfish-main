檔案使用者

使用 [檔案使用者] 頁面可管理 File 範圍使用者。 file 範圍中的使用者和群組會列於金鑰檔案，此檔案的位置則由 file 特性指定。

file 範圍中的使用者可以屬於 Java 群組，該群組是依照一般特性分類的使用者種類。例如，電子商務應用程式的用戶可能屬於 CUSTOMER 群組，但是大用戶可以屬於 PREFERRED 群組。將使用者進行群組分類可以簡化有大量使用者時的存取控制。

剛安裝完 GlassFish Server 之後，唯一的使用者是在安裝期間輸入的管理員。依預設，該使用者屬於 admin-realm 範圍中的 asadmin 群組，該群組有權修改 GlassFish Server。任何指定到該群組的使用者都將具有管理特權，亦即可以存取 asadmin 工具和管理主控台。

[檔案使用者] 頁面會顯示所選取範圍的使用者清單。對於每個使用者，均提供以下資訊：

使用者 ID

使用者名稱。

群組清單

使用者所屬的群組。

檔案使用者表也包含下列選項。

新建

建立新使用者的按鈕。

刪除

刪除一或多個選取使用者的按鈕。
