檢視應用程式目標

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 選取您要檢視目標的應用程式。

    應用程式名稱中可以包含一個選擇性的版本識別碼，位於名稱之後，兩者間用冒號 (:) 隔開。如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

    該應用程式的頁面會隨即開啟。

  3. 選取 [目標] 標籤。

    [應用程式目標] 頁面隨即會開啟，顯示目前指定給應用程式的目標。

  4. 若要啟用或停用特定目標實例或叢集上的應用程式，請按一下該目標旁的核取方塊，然後從 [更多動作] 下拉式功能表中選取 [啟用] 或 [停用]。
  5. 若要讓目標在特定目標實例或叢集上進行負載平衡，請按一下該目標旁的核取方塊，然後從 [更多動作] 下拉式功能表中選取 [負載平衡程式啟用] 或 [負載平衡程式停用]。
  6. 若要讓應用程式與特定虛擬伺服器建立關聯，請為所需的目標實例或叢集選取 [管理 VirtualServer]。

    [虛擬伺服器目標] 頁面會隨即開啟。

    若要增加目標虛擬伺服器，請從 [可用的目標] 清單中加以選取，然後使用 [增加] 按鈕，將其移至 [選取的目標] 清單。

    依預設，模組或應用程式會與名為 server 的預設虛擬伺服器產生關聯。使用 [全部加入] 按鈕可加入所有目標。使用 [移除] 與 [全部移除] 按鈕可從 [選取的目標] 清單中移除目標。

    選取 [儲存] 可以回到 [應用程式目標] 頁面。

  7. 若要增加或刪除應用程式的目標實例或叢集，請選取 [管理目標]。

    [管理目標] 頁面會隨即開啟。

    若要增加目標，請從 [可用的目標] 清單中加以選取，然後使用 [增加] 按鈕，將其移至 [選取的目標] 清單。

    使用 [全部加入] 按鈕可加入所有目標。使用 [移除] 與 [全部移除] 按鈕可從 [選取的目標] 清單中移除目標。

    選取 [儲存] 可以回到 [應用程式目標] 頁面。

  8. 按一下 [儲存]。

    修訂的目標清單中現在可提供該應用程式。
