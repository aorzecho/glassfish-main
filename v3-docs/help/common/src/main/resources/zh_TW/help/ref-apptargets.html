應用程式目標

使用 「應用程式目標」 頁面可檢視能在其上啟用應用程式的目標叢集與獨立伺服器實例。

「應用程式目標」 頁面包含以下資訊。

目標名稱

叢集或獨立伺服器實例的名稱。

已啟用

目標上的應用程式啟用時顯示 true；應用程式停用時顯示 false。

負載平衡已啟用

在目標上啟用應用程式負載平衡時顯示 true；停用應用程式負載平衡時顯示 false。

虛擬伺服器

若要讓應用程式與目標上的特定虛擬伺服器建立關聯，請選取該目標的 「管理 VirtualServer」。

「目標」 表格也包含下列選項。

管理目標

管理應用程式目標的按鈕。

更多動作

下列動作的下拉式清單。

啟用

在所選取目標上啟用應用程式的動作。

停用

在所選取目標上停用應用程式的動作。

負載平衡程式啟用

在所選取目標上啟用應用程式負載平衡的動作。

負載平衡程式停用

在所選取目標上停用應用程式負載平衡的動作。
