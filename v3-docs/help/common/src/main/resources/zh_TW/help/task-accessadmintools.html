授與管理工具的存取權

只有 asadmin 群組中的使用者才能存取管理主控台和 asadmin 命令行公用程式。若要授予使用者對這些管理工具的存取權限，請將他們增加到 admin-realm 中的 asadmin 群組。

  1. 在 「組態」 節點下，展開您要組態管理存取權限的伺服器實例或叢集組態。

  2. 展開 「安全性」 節點。

  3. 展開 「範圍」 節點。

  4. 選取 「admin-realm」 節點。

    「編輯範圍」 頁面會隨即開啟。

  5. 在 「編輯範圍」 頁面上，按一下 「管理使用者」 按鈕。

    「管理使用者」 頁面會隨即開啟。

    剛安裝完之後，admin-keyfile 檔案中會列出安裝期間所輸入的管理員使用者名稱和密碼。依預設，該使用者屬於 asadmin 群組，有權修改 GlassFish Server。請僅在要授予使用者所選取 GlassFish Server 實例或叢集的管理員權限時，才將使用者指定到該群組。

    如果將使用者增加到 admin-realm 範圍，但是將某個使用者指定到 asadmin 以外的群組，則該使用者的資訊仍會寫入名為 admin-keyfile 的檔案，但是該使用者無權存取管理工具或 file 範圍中的應用程式。

  6. 在 「管理使用者」 頁面上，按一下 「新建」，以便將新使用者增加到 admin-realm 範圍。

    「新建 File 範圍使用者」 頁面會隨即開啟。

  7. 在 「使用者 ID」 欄位中，鍵入新使用者名稱。

  8. 在 「群組清單」 欄位中，鍵入使用者所屬群組的名稱。

    若要授予使用者對 GlassFish Server 的修改權限，請將 asadmin 群組納入 「群組清單」 中。

  9. 在 「新密碼」 和 「確認新密碼」 欄位中，鍵入新使用者的密碼。

  10. 按一下 「確定」 以將此使用者增加到 admin-realm 範圍，或者按一下 「取消」 直接退出而不進行儲存。
