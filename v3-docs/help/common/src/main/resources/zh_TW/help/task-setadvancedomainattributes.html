設定網域屬性

  1. 在瀏覽樹狀結構中，選取 [GlassFish Server] 節點。

    此時將開啟 [一般資訊] 頁面。

  2. 在 [一般資訊] 頁面上，按一下 [進階] 標籤。

    [應用程式配置] 頁面會隨即開啟。

  3. 在 [應用程式配置] 頁面上，按一下 [網域屬性] 標籤。

    [網域屬性] 頁面會隨即開啟。

  4. 在 [應用程式根] 欄位中，鍵入已部署應用程式所在的目錄完整路徑。

    預設值為 domain-dir/applications。

  5. 在 [記錄檔根] 欄位中，鍵入保存伺服器記錄檔之目錄的完整路徑。

    預設值為 domain-dir/logs。

  6. 將 [語言環境] 欄位保留為空白，會使用主機的預設語言環境。

    語言環境是指定特定的語言和區域組合的識別碼。例如，美式英文的語言環境為 en_US，日文的語言環境為 ja_JP。若要使用非英文的語言環境，必須對 GlassFish Server 進行本土化 ，此程序包括將英文翻譯為其他語言。

  7. 按一下 [儲存]。
  8. 重新啟動伺服器。
