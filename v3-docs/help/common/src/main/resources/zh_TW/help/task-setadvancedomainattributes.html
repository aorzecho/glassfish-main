設定網域屬性

  1. 在瀏覽樹狀結構中，選取「網域」節點。

    「網域屬性」 頁面會隨即開啟。

  2. 在 「應用程式根」 欄位中，鍵入已建置應用程式所在的目錄完整路徑。

    預設值為 domain-dir/applications。

  3. 在 「記錄檔根」 欄位中，鍵入保存伺服器記錄檔之目錄的完整路徑。

    預設值為 domain-dir/logs。

  4. 將 「語言環境」 欄位保留為空白，會使用主機的預設語言環境。

    語言環境是指定特定的語言和區域組合的識別碼。例如，美式英文的語言環境為 en_US，日文的語言環境為 ja_JP。若要使用非英文的語言環境，必須對 GlassFish Server 進行本土化 ，此程序包括將英文翻譯為其他語言。

  5. 按一下 「儲存」。

  6. 重新啟動伺服器。
