編輯密碼別名

使用 「編輯密碼別名」 頁面可變更與現有密碼別名相關聯的密碼。

「編輯密碼別名」 頁面包含下列選項：

別名

別名的名稱。此選項無法編輯。

密碼

以加密格式儲存密碼別名的密碼。

確認密碼

以加密格式儲存密碼別名的密碼。會比較 「密碼」 與 「確認密碼」 值，確定輸入的密碼沒有拼錯。

等效 asadmin 子命令

  • update-password-alias(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • list-password-aliases(1)
