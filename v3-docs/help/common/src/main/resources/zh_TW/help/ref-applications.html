應用程式

[應用程式] 頁面會顯示 GlassFish Server 上部署的應用程式清單。您可以檢視和管理已部署的應用程式，還可以部署更多應用程式。

對於每個應用程式，均提供以下資訊：

名稱

應用程式名稱。

名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

已啟用或狀態

核取標記 (如果已啟用應用程式) 或 X (如果已停用應用程式)，僅存在預設伺服器實例 server 時才提供此資訊。

如果存在叢集或其他獨立伺服器實例，選取 [在目標上啟用] 可以檢視部署應用程式的目標。

引擎

應用程式所使用的容器類型。容器類型可以是下列任何一項：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld (用於 Java EE 平台應用程式之環境與相依性注入的容器)

動作

在部署後的元件上，您可以執行之動作的連結：[重新部署] 和 [重新載入] 適用於所有元件，而 [啟動] 則適用於 Web 應用程式。

已部署的應用程式表也包含下列選項。

部署

部署應用程式的按鈕。

解除部署

解除部署一或多個所選取應用程式的按鈕。

啟用

啟用一或多個所選取應用程式的按鈕，僅存在預設伺服器實例 server 時才提供此資訊。

停用

停用一或多個所選取應用程式的按鈕，僅存在預設伺服器實例 server 時才提供此資訊。

篩選

依引擎篩選應用程式的下拉式清單。
