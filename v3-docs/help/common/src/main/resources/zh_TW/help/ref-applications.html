應用程式

「應用程式」 頁面會顯示 GlassFish Server 上建置的應用程式清單。您可以檢視和管理已建置的應用程式，還可以建置更多應用程式。

對於每個應用程式，均提供以下資訊：

名稱

應用程式名稱。

名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

已啟用或狀態

核取標記 (如果已啟用應用程式) 或 X (如果已停用應用程式)，唯有預設伺服器實例 server 存在時才提供此資訊。

如果有叢集或其他獨立伺服器實例存在，選取 「在目標上啟用」 可以檢視建置應用程式的目標。

引擎

應用程式所使用的容器類型。容器類型可以是下列任何一項：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld (用於 Java EE 平台應用程式之環境與相依性注入的容器)

動作

在建置元件後，可對元件執行之動作的連結：重新建置與重新載入所有元件、啟動 Web 應用程式與應用程式用戶端，以及下載應用程式用戶端的用戶端 Stub。

已建置的應用程式表也包含下列選項。

建置

建置應用程式的按鈕。

解除建置

解除建置一或多個所選取應用程式的按鈕。

啟用

啟用一或多個所選取應用程式的按鈕，唯有預設伺服器實例 server 存在時才提供此資訊。

停用

停用一或多個所選取應用程式的按鈕，唯有預設伺服器實例 server 存在時才提供此資訊。

篩選

依引擎篩選應用程式的下拉式清單。
