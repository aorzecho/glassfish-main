檢視元件的詳細資訊

更新工具提供已安裝元件、已安裝元件的可用更新和附加元件的詳細資訊。

  1. 在瀏覽樹狀結構中，選取 「更新工具」 節點。

    「可用的附加元件」 頁面會隨即顯示。

  2. 按一下列出元件之頁面的標籤。

    下列其中一個頁面會列出元件：

    • 可用的附加元件

    • 可用的更新

    • 已安裝的元件

  3. 在元件表格中，選取元件。

    所選取元件的元件詳細資訊頁面會隨即顯示。
