停用安全性管理員

停用 GlassFish Server 安全性管理員可以提昇某些類型應用程式的效能。即使已停用安全性管理員，Java 平台授權和認證功能仍將發揮作用。您可以在開發環境中停用安全性管理員，但是請勿在生產環境中停用它。

  1. 在瀏覽樹狀結構中，選取 [配置] 節點。
  2. 在 [配置] 節點下，選取 [JVM 設定] 節點。

    [JVM 一般設定] 頁面會隨即開啟。

  3. 在 [JVM 一般設定] 頁面上，按一下 [JVM 選項] 標籤。

    [JVM 選項] 頁面會隨即開啟。

  4. 選取包含 -Djava.security.policy 選項的 [值] 欄位左側的核取方塊。
  5. 按一下 [刪除]。
  6. 按一下 [儲存]。
  7. 重新啟動伺服器。
