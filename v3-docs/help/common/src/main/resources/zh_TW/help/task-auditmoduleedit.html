編輯稽核模組

依預設，稽核模組未開啟。如需有關如何啟用稽核模組的更多資訊，請參閱啟用或停用稽核模組。

  1. 在瀏覽樹狀結構中，展開 「組態」 節點。

  2. Select the instance to configure:

    • To configure a particular instance, expand the instance's config node. For example, the default instance, server, expand the server-config node.

    • To configure the default settings for all instances, expand the default-config node.

  3. 在 「組態」 節點下，展開 「安全性」 節點。

  4. 在 「安全性」 節點下，選取 「稽核模組」 節點。

    「稽核模組」 頁面會隨即開啟。

  5. 在 「稽核模組」 頁面上，按一下您要編輯的稽核模組名稱。

    「編輯稽核模組」 頁面會隨即開啟。

  6. 在 「編輯稽核模組」 頁面的 「類別名稱」 欄位中，鍵入新值以修改類別名稱。

  7. 在 「其他特性」 區段中，指定其他特性。

    如需適用於稽核模組之其他特性的說明，請參閱稽核模組特定的特性。

    • 若要增加特性，請按一下 「增加特性」 按鈕。在出現的空白列中，於 「名稱」 欄位中鍵入特性名稱，然後在 「值」 欄位中鍵入特性值。

    • 若要修改特性，請編輯特性的 「值」 欄位。

    • 若要刪除特性，請選取您要刪除之特性 「名稱」 欄位左側的核取方塊，然後按一下 「刪除特性」 按鈕。

  8. 按一下 「儲存」。
