元件詳細資訊

[元件詳細資訊] 頁面提供有關元件的詳細資訊。

[元件詳細資訊] 頁面包含下列選項。

上一步

此按鈕讓您回到可選取元件以檢視詳細資訊的頁面。

套裝軟體名稱

提供元件的套裝軟體名稱。

唯一識別碼

此元件在儲存庫中顯示的完全合格識別碼。

種類

與元件相關的軟體種類，例如應用程式伺服器、系統工具和資料庫。建立元件時，元件的作者會為元件指定種類。

版本

提供元件的套裝軟體版本號碼。

發佈日期

此元件版本發佈至儲存庫的日期與時間。此日期不是安裝影像內元件安裝的日期與時間。

安裝的大小

安裝元件所使用的磁碟空間量。

來源

對於已安裝的元件，會是先前取得之元件的網路儲存庫名稱。

對於附加元件或更新，會是即將取得之元件的網路儲存庫名稱。

說明

元件的簡介。

相關作業

檢視有關元件的詳細資訊
