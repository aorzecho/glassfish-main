安裝附加元件

安裝附加元件包括下列作業：

  • 從網路儲存庫傳送元件的內容和元件的相依性

  • 在系統上安裝 GlassFish Server 安裝影像下的元件

如果元件需要任何其他尚未安裝的元件，也會一併安裝所需的元件。

  1. 在瀏覽樹狀結構中，選取 「更新工具」 節點。

    「可用的附加元件」 頁面會隨即開啟。

  2. 在 「可用的附加元件」 頁面上，於可用附加元件的表格中，選取您要安裝的元件。

    如果沒有可用的附加元件，表格會是空的。

    • 若要選取個別元件，請選取元件名稱旁的核取方塊。

    • 若要選取所有元件，請按一下表格標頭中的 「全選」 圖示。

    • 若要取消選取所有元件，請按一下表格標頭中的 「全不選」 圖示。

  3. 按一下 「安裝」。

    會出現對話方塊請您確認所選應安裝的元件及其所需的元件。

  4. 在對話方塊中，按一下 「確定」。

    系統會詢問您是否接受正在安裝之元件的授權合約。

  5. 接受正在安裝之元件的授權合約。

    GlassFish Server 會確認安裝已完成。再從附加元件表格中移除元件，並在 「已安裝的元件」 頁面上，將這些元件增加至安裝的元件表格中。

  6. 重新啟動伺服器。

    如需詳細說明，請參閱下列文件：

    • Oracle GlassFish Server Administration Guide中的To Stop a Domain

    • Oracle GlassFish Server Administration Guide中的To Start a Domain
