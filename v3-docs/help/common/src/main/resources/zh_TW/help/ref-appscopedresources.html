應用程式範圍資源

使用「應用程式範圍資源」頁面，可檢視僅供特定應用程式使用的資源。

「應用程式範圍資源」頁面的表格中含有下列資訊。

應用程式名稱

應用程式的名稱。

模組名稱

應用程式內的模組名稱。

資源名稱

應用程式或模組的應用程式範圍資源名稱。按一下名稱可修改資源設定。

資源類型

資源類型，例如「JDBC 連線集區」或「連接器資源」。
