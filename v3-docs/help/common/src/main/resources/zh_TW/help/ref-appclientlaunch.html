應用程式用戶端啟動

使用 [應用程式用戶端啟動] 頁面可以啟動部署的應用程式用戶端。用戶端必須是能夠使用 Java Web Start 執行的 Web 用戶端。

[啟動應用程式用戶端] 頁面包含下列選項。

應用程式

包含應用程式用戶端之應用程式的名稱。

名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱「Oracle GlassFish Server 3.1 Application Deployment Guide」中的「Module and Application Versions」。

模組

包含應用程式用戶端之模組的名稱。

連結

所部署用戶端的 URL。

引數

要傳送給應用程式的引數。
