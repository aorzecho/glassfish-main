監視

使用「監視」頁面可對叢集和非叢集伺服器實例進行組態監視並檢視監視資料。

每個伺服器實例均提供以下資訊。

實例名稱

伺服器實例的名稱。按一下實例名稱，隨即會開啟實例的「一般資訊」頁面。

叢集名稱

伺服器實例 (若為叢集實例) 所屬叢集的名稱。按一下叢集名稱，隨即會開啟叢集的「一般資訊」頁面。

動作

可讓您對伺服器實例執行以下作業的連結。

配置監視

顯示叢集或獨立實例組態的「監視服務」頁面。

檢視監視資料

可讓您檢視以下類型之伺服器實例監視資料的連結。

應用程式

顯示伺服器實例的「應用程式監視」頁面。

伺服器

顯示伺服器實例的「伺服器監視」頁面。

資源

顯示伺服器實例的「資源監視」頁面。
