JVMオプション

「JVMオプション」ページを使用して、GlassFish Serverを実行するJavaアプリケーション・ランチャ(java)のオプションを指定します。-Dオプションは、GlassFish Serverに固有のプロパティを指定します。

  • オプションを変更するには、そのオプションの「値」フィールドを編集します。

  • オプションを追加するには、「JVMオプションを追加」ボタンをクリックします。表示される空白行に、「値」フィールドのオプション情報を入力します。

    オプション情報にスペースが1つ以上含まれている場合は、文字列全体を二重引用符で囲みます。

  • オプションを削除するには、削除するオプションの「値」フィールドの左側にあるチェックボックスを選択し、「削除」をクリックします。
