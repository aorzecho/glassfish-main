管理者パスワードを変更する

  1. ナビゲーション・ツリーで、「ドメイン」ノードを選択します。

    「ドメイン属性」ページが開きます。

  2. 「管理者パスワード」タブを選択します。

    「新しい管理者パスワード」ページが開きます。

    このページには、現在の管理者アカウントのユーザーIDとグループ名が表示されます。デフォルトでは、ユーザーIDはadmin、グループ名はasadminです。

  3. 「新しいパスワード」フィールドに、現在の管理者アカウントの新しいパスワードを入力します。

  4. 「新しいパスワードを確認」フィールドに、新しいパスワードを再度入力します。

  5. 「保存」をクリックします。

  • 管理ツールにアクセス権を付与する

  • ユーザーをファイル・レルムに追加する

  • ファイル・レルムからユーザーを削除する

  • DASの一般情報を表示する

  • DASのログ・ファイルを表示する

  • 表示されたログ・レコードをフィルタリングする

  • アプリケーション構成を設定する

  • ドメイン属性を設定する

  • ドメイン・ログを収集してダウンロードする

  • システム・プロパティを管理する

  • アプリケーションの監視データを表示する

  • サーバー監視データを表示する

  • リソース監視データを表示する
