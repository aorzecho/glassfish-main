監査ログを有効または無効にする

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「セキュリティー」ノードを選択します。

    「セキュリティ」ページが開きます。

  3. 「セキュリティー」ページで、「監査ログが有効」チェックボックスを選択して、監査ログを有効にします。

    監査ログを有効にするには、アクティブな監査モジュールの auditOn プロパティーも true に設定する必要があります。

  4. 「保存」をクリックします。
