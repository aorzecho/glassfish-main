rawログ・ビューア

「rawログ・ビューア」ページを使用し、管理サーバー(serverという名前)のrawログ・ファイルを表示します。

「rawログ・ビューア」ページには、次のオプションが含まれます。

インスタンス

ログ・レコードを表示するサーバー・インスタンスまたはクラスタ。選択できるのはserverのみです。

「rawログ・ビューア」には、メッセージごとに次の情報が表示されます。

タイムスタンプ

ログ・メッセージが発行された日時。

レコード番号

サーバー・ログのメッセージ番号。

ログ・レベル

メッセージのログ・レベル。

バージョン

GlassFish Serverのバージョン。

ロガー

ログ・メッセージを生成したロガー・モジュール。

名前と値のペア

スレッド名とログ・メッセージのID。

メッセージ

ログ・メッセージの簡単な説明。(details)のリンクをクリックすると、書式設定されたメッセージが表示されます。
