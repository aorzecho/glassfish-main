ドメイン・ログを収集してダウンロードする

  1. ナビゲーション・ツリーで、「ドメイン」ノードを選択します。

    「ドメイン属性」ページが開きます。

  2. 「ドメイン・ログ」タブを選択します。

    「ドメイン・ログ」ページが開きます。

    クラスタが存在する場合は、そのクラスタおよびインスタンスのドロップダウン・リストが表示されます。クラスタが存在しない場合は、インスタンスのドロップダウン・リストのみが表示されます。

  3. クラスタまたはインスタンスを選択します。

  4. ログ・ファイルを収集してダウンロードするクラスタまたはインスタンスを選択します。

  5. 「ログの収集」をクリックします。

    ブラウザのダウンロード・メカニズムにより、log-files-name.zipファイル(nameはクラスタまたはインスタンスの名前)のダウンロードを求めるプロンプトが表示されます。

関連項目

  • 管理ツールにアクセス権を付与する

  • ユーザーをファイル・レルムに追加する

  • ファイル・レルムのユーザー情報を編集する

  • ファイル・レルムからユーザーを削除する

  • DASの一般情報を表示する

  • DASのログ・ファイルを表示する

  • 表示されたログ・レコードをフィルタリングする

  • アプリケーション構成を設定する

  • ドメイン属性を設定する

  • システム・プロパティを管理する

  • アプリケーションの監視データを表示する

  • サーバー監視データを表示する

  • リソース監視データを表示する
