ライフサイクルモジュールを削除する

  1. ナビゲーションツリーで、「ライフサイクルモジュール」ノードを選択します。

    「ライフサイクルモジュール」ページが開きます。

  2. 「ライフサイクルモジュール」ページで、削除するライフサイクルモジュールの左側にあるチェックボックスを選択します。
  3. 「削除」をクリックします。
