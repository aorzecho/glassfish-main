新しいパスワードエイリアス

「新しいパスワードエイリアス」ページを使用し、パスワードファイルまたはドメイン設定ファイルで使用する新しいパスワードエイリアスを作成します。

「新しいパスワードエイリアス」ページには、次のオプションが含まれます。

エイリアス名

エイリアスの名前。この名前は、ドメイン内のすべてのパスワードエイリアスの中で一意である必要があります。

パスワード

パスワードエイリアスによって暗号化された形式で格納されるパスワード。

パスワードの確認

パスワードエイリアスによって暗号化された形式で格納されるパスワード。「パスワード」と「パスワードの確認」の値が比較され、タイプミスのないパスワードが入力されたか確認されます。
