システム・プロパティを管理する

  1. ナビゲーション・ツリーで、GlassFish Serverノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「システム・プロパティ」タブをクリックします。

    「システム・プロパティ」ページが開きます。

  3. 「追加プロパティ」セクションで、構成のシステム・プロパティを指定します。

    多数のシステム・プロパティが事前定義されていますが、これらはこのページに表示されません。詳細は、定義済のシステム・プロパティを参照してください。

    • プロパティを追加するには、「プロパティを追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティ名を入力し、「値」フィールドにプロパティ値を入力します。

    • プロパティを変更するには、そのプロパティの「値」フィールドを編集します。

    • プロパティを削除するには、削除するプロパティの「名前」フィールドの左側にあるチェックボックスを選択し、「プロパティを削除」ボタンをクリックします。

  4. 「保存」をクリックします。

関連項目

  • DASの一般情報を表示する

  • DASのログ・ファイルを表示する

  • 表示されたログ・レコードをフィルタリングする

  • アプリケーション構成を設定する

  • ドメイン属性を設定する

  • 管理者パスワードを変更する

  • アプリケーションの監視データを表示する

  • サーバー監視データを表示する

  • リソース監視データを表示する
