新しいファイルレルムユーザー

「新しいファイルレルムユーザー」ページを使用して、新しいユーザーに現在のレルムに対するアクセス権を付与します。

「新しいファイルレルムユーザー」ページには、次のオプションが含まれます。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

レルム名

現在選択しているレルムの名前。このフィールドは読み取り専用です。

ユーザー ID

新しいユーザーアカウントのユーザー ID。

グループリスト

新しいユーザが属するグループ。複数のグループ名をコンマで区切ります。

admin-realm ユーザーの場合、このフィールドは読み取り専用です。グループは asadmin になります。

新しいパスワード

新しいユーザーのパスワード。

新しいパスワードを確認

新しいユーザーのパスワード。
