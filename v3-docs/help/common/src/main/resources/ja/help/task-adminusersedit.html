管理レルムユーザー情報を編集する

  1. 「設定」ノードの下で、管理レルムユーザーを管理するサーバーインスタンスまたはクラスタの設定を展開します。
  2. 「セキュリティー」ノードを開きます。
  3. 「レルム」ノードを開きます。
  4. 「admin-realm」ノードを選択します。

    「レルムを編集」ページが開きます。

  5. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

    「管理ユーザー」ページが開きます。

  6. 「管理ユーザー」ページの表の「ユーザー ID」列で、変更するユーザーの名前をクリックします。

    「file レルムユーザーを編集」ページが開きます。

  7. 「グループリスト」フィールドを編集し、ユーザーが属すグループを変更します。

    複数のグループ名はコンマで区切ります。これらのグループは、ほかの場所で定義される必要はありません。

  8. 「新規パスワード」フィールドと「新しいパスワードを確認」フィールドに新しいパスワードを入力し、ユーザーのパスワードを変更します。
  9. 「保存」をクリックします。
