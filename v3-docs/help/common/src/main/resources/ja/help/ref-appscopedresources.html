アプリケーションを対象範囲とするリソース

「アプリケーションを対象範囲とするリソース」を使用して、特定のアプリケーションでのみ使用可能なリソースを表示します。

「アプリケーションを対象範囲とするリソース」ページには、次の情報が表示されます。

アプリケーション名

アプリケーションの名前。

モジュール名

アプリケーション内のモジュールの名前。

リソース名

アプリケーションまたはモジュールのアプリケーションを対象範囲とするリソースの名前。リソース設定を変更するには、名前をクリックします。

リソース・タイプ

リソース・タイプ(例: 「JDBC接続プール」、「コネクタ・リソース」)。
