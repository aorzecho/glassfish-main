ファイルレルムユーザーの編集

「ファイルレルムユーザーの編集」ページを使用して、現在のレルムに対するユーザーのパスワードを変更します。

「ファイルレルムユーザーの編集」ページには、次のオプションが含まれます。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

レルム名

現在選択しているレルムの名前。このフィールドは読み取り専用です。

ユーザー ID

編集中のアカウントの名前。このフィールドは読み取り専用です。

グループリスト

このユーザーが所属しているグループ。グループ名をコンマで区切ってください。

admin-realm ユーザーの場合、このフィールドは読み取り専用です。グループは asadmin になります。

新しいパスワード

このユーザーの新しいパスワード。

新しいパスワードを確認

このユーザーの新しいパスワード。
