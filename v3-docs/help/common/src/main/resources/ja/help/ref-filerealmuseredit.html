ファイル・レルム・ユーザーの編集

「ファイル・レルム・ユーザーの編集」ページを使用して、現在のレルムに対するユーザーのパスワードを変更します。

注意:

セキュリティ保護された管理が有効な場合は、admin-realmユーザーのパスワードを空の値に変更できません。

「ファイル・レルム・ユーザーの編集」ページには、次のオプションが含まれます。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

レルム名

現在選択しているレルムの名前。このフィールドは読取り専用です。

ユーザーID

編集中のアカウントの名前。このフィールドは読取り専用です。

グループ・リスト

このユーザーが所属しているグループ。グループ名をカンマで区切ってください。

admin-realmユーザーの場合、このフィールドは読取り専用です。グループはasadminになります。

新しいパスワード

このユーザーの新しいパスワード。

新しいパスワードを確認

このユーザーの新しいパスワード。
