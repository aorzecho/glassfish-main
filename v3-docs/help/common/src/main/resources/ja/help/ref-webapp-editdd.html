モジュール記述子

アプリケーションのモジュール記述子を表示するには、「モジュール記述子」ページを使用します。

「モジュール記述子」ページには、次のオプションが含まれます。

配備記述子

アプリケーションの名前、サブコンポーネントの名前 (ある場合)、記述子ファイルの名前を表示する表。

関連操作

  • 「アプリケーションの記述子を表示する」

  • 「配備されているアプリケーションのリストを表示する」

  • 「Web アプリケーションを配備する」

  • 「エンタープライズアプリケーションを配備する」

  • 「アプリケーションクライアントを配備する」

  • 「コネクタモジュールを配備する」

  • 「EJB JAR を配備する」

  • 「配備済みアプリケーションを編集する」

  • 「アプリケーションターゲットを表示する」

  • 「Web サービスのエンドポイント情報を表示する」

  • 「アプリケーションを起動する」

  • 「アプリケーションクライアントを起動する」

  • 「配備済みアプリケーションを再配備する」

  • 「配備済みアプリケーションを再読み込みする」

関連する asadmin コマンド

  • list-applications(1)
