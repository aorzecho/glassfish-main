レルムを編集

「レルムを編集」ページを使用して、レルムを変更します。

「レルムを編集」ページには、次のオプションが含まれます。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

レルム名

レルムの名前。このフィールドは読取り専用です。名前を指定できるのは、新しいレルムの作成時のみです。

クラス名

編集するレルムのクラス名。

追加プロパティ

レルムの追加プロパティ。有効なプロパティはレルムのタイプによって異なります。

  • FileRealmクラスに固有のプロパティ

  • CertificateRealmクラスに固有のプロパティ

  • JDBCRealmクラスに固有のプロパティ

  • LDAPRealmクラスに固有のプロパティ

  • SolarisRealmクラスに固有のプロパティ

  • PamRealmクラスに固有のプロパティ
