レルムを削除する

  1. 「設定」ノードの下で、レルムを設定するサーバーインスタンスまたはクラスタの設定を展開します。
  2. 「セキュリティー」ノードを開きます。
  3. 「レルム」ノードを開きます。

    「レルム」ページが開きます。

  4. 「レルム」ページで、削除するレルムの名前の左側にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
