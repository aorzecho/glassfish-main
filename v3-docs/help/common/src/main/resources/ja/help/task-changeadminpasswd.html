管理レルムのユーザーのパスワードを変更する

現在の管理者アカウントのパスワードを変更する方法は、管理者パスワードを変更するを参照してください。admin-realmに複数のユーザーがある場合は、他のユーザーのパスワードも変更できます。

管理者パスワードを暗号化すると、アプリケーションのセキュリティが向上します。セキュリティ設定手順の詳細は、Oracle GlassFish Serverセキュリティ・ガイドの第1章、Administering System Securityを参照してください。

  1. 「構成」ノードの下で、管理パスワードを変更するサーバー・インスタンスまたはクラスタの構成を展開します。

  2. 「セキュリティ」ノードを展開します。

  3. 「レルム」ノードを展開します。

  4. 「admin-realm」ノードを選択します。

    「レルムを編集」ページが開きます。

  5. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

    「管理ユーザー」ページが開きます。

  6. 「管理ユーザー」ページで、パスワードを追加または変更するユーザーをクリックします。

    「ファイル・レルム・ユーザーの編集」ページが開きます。

  7. 「新しいパスワード」および「新しいパスワードを確認」フィールドに、新しいパスワードを入力します。

  8. 「保存」をクリックします。
