パスワード・エイリアス

「パスワード・エイリアス」ページを使用し、パスワード・エイリアスの作成と管理を行います。

「パスワード・エイリアス」ページには、ドメイン内のパスワード・エイリアスのリストが表示されます。エイリアスごとに、次の情報が表示されます。

名前

作成されたときにそのパスワード・エイリアスに割り当てられた名前。名前をクリックすると、そのエイリアスの「パスワード・エイリアスの編集」ページが開きます。

「パスワード・エイリアス」の表には、次のオプションも表示されます。

新規

パスワード・エイリアスを作成するボタン。

削除

選択した1つまたは複数のパスワード・エイリアスを削除するボタン。
