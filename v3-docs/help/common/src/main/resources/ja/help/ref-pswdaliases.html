パスワードエイリアス

「パスワードエイリアス」ページを使用し、パスワードエイリアスの作成と管理を行います。

「パスワードエイリアス」ページには、ドメイン内のパスワードエイリアスの一覧が表示されます。エイリアスごとに、次の情報が表示されます。

名前

作成されたときにそのパスワードエイリアスに割り当てられた名前。名前をクリックすると、そのエイリアスの「「パスワードエイリアスの編集」」ページが表示されます。

「パスワードエイリアス」の表には、次のオプションも表示されます。

新規

パスワードエイリアスを作成するボタン。

削除

選択した 1 つまたは複数のパスワードエイリアスを削除するボタン。
