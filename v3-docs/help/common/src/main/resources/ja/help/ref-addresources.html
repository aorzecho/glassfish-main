リソースの追加

「リソースの追加」ページを使用して、指定したターゲットのglassfish-resources.xmlファイルから1つまたは複数のリソースを作成します。.

「リソースの追加」には、次の情報が表示されます。

位置

glassfish-resources.xmlファイルの位置。

サーバーにアップロードされるXMLファイル

リモート・マシン上のファイルへのパス。

GlassFish Serverからアクセス可能なローカルXMLファイル

ドメイン管理サーバー(DAS)マシン上のファイルへのパス。

ターゲット

クラスタおよびスタンドアロン・サーバー・インスタンスのドロップダウン・リスト。

関連タスク

XMLファイルに構成されたリソースを追加する

関連するasadminコマンド

add-resources(1)
