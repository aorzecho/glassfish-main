JACC プロバイダを削除する

  1. ナビゲーションツリーの「設定」ノードの下で、JACC プロバイダを作成する GlassFish Server インスタンスまたはクラスタを展開します。

    すべてのインスタンスおよびクラスタで使用できるプロバイダを作成するには、default-config ノードを選択します。

  2. 展開されたインスタンスまたはクラスタの「設定」ノードの下で、「セキュリティー」ノードを展開します。
  3. 「セキュリティー」ノードの下で、「JACC プロバイダ」ノードを選択します。

    「JACC プロバイダ」ページが開きます。

  4. 削除する JACC プロバイダの左側にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
