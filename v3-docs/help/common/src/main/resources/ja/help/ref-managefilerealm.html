ファイルユーザー

file レルムのユーザーを管理するには、「ファイルユーザー」ページを使用します。file レルムのユーザーおよびグループはキーファイルに表示され、その場所は file プロパティーで指定されます。

file レルムのユーザーは、共通の特性で分類されるユーザーのカテゴリである 「Java グループ」に属することができます。たとえば、E コマースアプリケーションの顧客は CUSTOMER グループに属しますが、お得意様は PREFERRED グループに属します。ユーザーをグループに分類すると、大量のユーザーによるアクセスを容易に制御できます。

GlassFish サーバー のインストール後の当初は、ユーザーはインストールの際に入力した管理者だけです。このユーザーは、デフォルトで GlassFish サーバー を変更する権限を付与される、admin-realm レルムの asadmin グループに属します。このグループに割り当てられるすべてのユーザーは、管理者権限が付与されます。つまり、asadmin ツールおよび管理コンソールへのアクセス権があります。

「ファイルユーザー」ページには、選択したレルムのユーザーのリストが表示されます。ユーザーごとに、次の情報が表示されます。

ユーザー ID

ユーザー名。

グループリスト

ユーザーが属しているグループ。

「ファイルユーザー」の表には、次のオプションも表示されます。

新規

新しいユーザーを作成するボタン。

削除

選択した 1 人または複数のユーザーを削除するボタン。
