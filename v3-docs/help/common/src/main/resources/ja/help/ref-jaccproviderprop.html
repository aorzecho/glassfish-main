JACCプロバイダに固有のプロパティ

JACCプロバイダでは、次のプロパティを使用できます。

repository

JACCポリシー・ファイルが含まれるディレクトリ。default GlassFish Server JACCプロバイダの場合、デフォルトのディレクトリは${com.sun.aas.instanceRoot}/generated/policyです。このプロパティは、デフォルトではsimple GlassFish Server JACCプロバイダに定義されません。
