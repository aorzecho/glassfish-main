JACC プロバイダに固有のプロパティー

JACC プロバイダでは、次のプロパティーを使用できます。

repository

JACC ポリシーファイルが含まれるディレクトリ。default GlassFish サーバー JACC プロバイダの場合、デフォルトディレクトリーは ${com.sun.aas.instanceRoot}/generated/policy です。simpleGlassFish サーバー JACC プロバイダの場合、デフォルトではこのプロパティーは定義されません。
