定義済みのシステムプロパティー

GlassFish サーバー では、次の設定システムプロパティーが定義されます。

ASADMIN_LISTENER_PORT

このプロパティーは、管理用の HTTP ポートまたは HTTPS ポートのポート番号を指定します。このポートは、インスタンスを管理するために Web ブラウザで指定する URL 内のポートです (例: http://localhost:4949)。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

HTTP_LISTENER_PORT

This property specifies the port number of the port that is used to listen for HTTP requests. 有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

HTTP_SSL_LISTENER_PORT

This property specifies the port number of the port that is used to listen for HTTPS requests. 有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

IIOP_LISTENER_PORT

このプロパティーは、IIOP 接続に使用するポートのポート番号を指定します。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

IIOP_SSL_LISTENER_PORT

このプロパティーは、安全な IIOP 接続に使用するポートのポート番号を指定します。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

IIOP_SSL_MUTUALAUTH_PORT

This property specifies the port number of the port that is used for secure IIOP connections with client authentication.有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

JAVA_DEBUGGER_PORT

このプロパティーは、JPDA (Java Platform Debugger Architecture) デバッガへの接続に使用されるポートのポート番号を指定します。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

JMS_PROVIDER_PORT

このプロパティーは、Java Message Service プロバイダのポート番号を指定します。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

JMX_SYSTEM_CONNECTOR_PORT

This property specifies the port number on which the JMX connector listens. 有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。

OSGI_SHELL_TELNET_PORT

このプロパティーは、Apache Felix Remote Shell への接続に使用されるポートのポート番号を指定します。このシェルは、Felix シェルサービスを使用して OSGi モジュール管理サブシステムとの対話を行います。有効な値は 1 から 65535 です。UNIX では、ポート 1 から 1024 で待機するソケットを作成するには 、スーパーユーザー権限が必要です。
