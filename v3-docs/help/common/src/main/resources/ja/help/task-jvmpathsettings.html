JVMパス設定を構成する

  1. ナビゲーション・ツリーで、「構成」ノードを選択します。

  2. Select the configuration that you are changing.

  3. 「構成」ノードの下で、「JVM設定」ノードを選択します。

    「JVMの全般的な設定」ページが開きます。

  4. 「JVMの全般的な設定」ページで、「パス設定」タブをクリックします。

    「JVMパス設定」ページが開きます。

  5. 「ネイティブ・ライブラリ・パスの接頭辞」フィールドに、ネイティブ・ライブラリ・パスの前に追加するJARファイルのフルパスを入力します。

    複数のファイルを追加するには、1行ごとに1つのパスを入力します。

  6. 「ネイティブ・ライブラリ・パスの接尾辞」フィールドに、ネイティブ・ライブラリ・パスの後に追加するJARファイルのフルパスを入力します。

    複数のファイルを追加するには、1行ごとに1つのパスを入力します。

  7. 「保存」をクリックします。

  8. サーバーを再起動します。
