JVM パスの設定を行う

  1. ナビゲーションツリーで、「設定」ノードを選択します。
  2. 「設定」ノードの下で、「JVM 設定」ノードを選択します。

    「JVM 一般設定」ページが開きます。

  3. 「JVM 一般設定」ページで、「パス設定」タブをクリックします。

    「JVM パス設定」ページが開きます。

  4. 「ネイティブライブラリパスのプレフィックス」フィールドに、ネイティブライブラリパスの前に追加する JAR ファイルの完全パスを入力します。

    複数のファイルを追加するには、1 行ごとに 1 つのパスを入力します。

  5. 「ネイティブライブラリパスのサフィックス」フィールドに、ネイティブライブラリパスの後に追加する JAR ファイルの完全パスを入力します。

    複数のファイルを追加するには、1 行ごとに 1 つのパスを入力します。

  6. 「保存」をクリックします。
  7. サーバーを再起動します。
