デフォルトレルムを設定する

デフォルトレルムとは、アプリケーションの配備記述子がレルムを指定しない場合に、GlassFish サーバー が認証と承認に使用するレルムです。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「セキュリティー」ノードを選択します。

    「セキュリティ」ページが開きます。

  3. 「デフォルトレルム」ドロップダウンリストから、サーバーが認証に使用するアクティブなレルムを選択します。

    デフォルト値は file です。

  4. 「保存」をクリックします。
