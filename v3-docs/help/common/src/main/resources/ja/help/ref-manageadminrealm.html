管理ユーザー

admin-realmのユーザーを管理するには、「管理ユーザー」ページを使用します。admin-realmはファイル・レルムです。fileレルムのユーザーおよびグループは鍵ファイルに一覧表示され、その場所はfileプロパティで指定されます。

GlassFish Serverのインストール後の当初は、ユーザーはインストールの際に入力した管理者のみです。このユーザーは、デフォルトでGlassFish Serverを変更する権限を付与される、admin-realmレルムのasadminグループに属します。このグループに割り当てられるすべてのユーザーは、管理者権限が付与されます。つまり、asadminツールおよび管理コンソールへのアクセス権があります。

「管理ユーザー」ページには、選択したレルムのユーザーのリストが表示されます。ユーザーごとに、次の情報が表示されます。

ユーザーID

ユーザー名。

グループ・リスト

ユーザーが属しているグループ。

「管理ユーザー」の表には、次のオプションも含まれます。

新規

新しいユーザーを作成するボタン。

削除

選択した1人または複数のユーザーを削除するボタン。
