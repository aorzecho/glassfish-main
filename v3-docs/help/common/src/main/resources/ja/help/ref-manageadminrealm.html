管理ユーザー

admin-realm のユーザーを管理するには、「管理ユーザー」ページを使用します。 admin-realm はファイルレルムです。file レルムのユーザーおよびグループはキーファイルに表示され、その場所は file プロパティーで指定されます。

GlassFish Server のインストール後の当初は、ユーザーはインストールの際に入力した管理者だけです。このユーザーは、デフォルトで GlassFish Server を変更する権限を付与される、admin-realm レルムの asadmin グループに属します。このグループに割り当てられるすべてのユーザーは、管理者権限が付与されます。つまり、asadmin ツールおよび管理コンソールへのアクセス権があります。

「管理ユーザー」ページには、選択したレルムのユーザーのリストが表示されます。ユーザーごとに、次の情報が表示されます。

ユーザー ID

ユーザー名。

グループリスト

ユーザーが属しているグループ。

「管理ユーザー」の表には、次のオプションも含まれます。

新規

新しいユーザーを作成するボタン。

削除

選択した 1 人または複数のユーザーを削除するボタン。

関連操作

  • 「管理レルムにユーザーを追加する」

  • 「管理レルムユーザー情報を編集する」

  • 「管理レルムからユーザーを削除する」
