共通タスク

「共通タスク」ページには、管理コンソールの一般的なタスクのためのショートカットが用意されています。

「共通タスク」ページを使用して次の操作を実行できます。

  • GlassFish Serverの「GlassFishニュース」ページを表示する

  • GlassFish Serverのサブスクリプション・ページを表示する

  • デプロイされたアプリケーションの一覧を表示する

  • Webアプリケーションをデプロイする

  • エンタープライズ・アプリケーションをデプロイする

  • アプリケーション・クライアントをデプロイする

  • コネクタ・モジュールをデプロイする

  • EJB Jarをデプロイする

  • 監視サービスを構成する

  • アプリケーションの監視データを表示する

  • サーバー監視データを表示する

  • リソース監視データを表示する

  • インストールされたコンポーネントを表示する

  • アドオン・コンポーネントをインストールする

  • コンポーネントに使用可能な更新を表示する

  • JDBC接続プールを作成する

「共通タスク」ページには、一部のGlassFish Serverドキュメントへのリンクも表示されます。
