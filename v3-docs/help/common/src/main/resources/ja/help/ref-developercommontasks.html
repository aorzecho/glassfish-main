共通操作

「共通操作」ページには、管理コンソールの一般的な操作のためのショートカットが用意されています。

「共通操作」ページを使用して次の操作を実行できます。

  • 「GlassFish サーバー の「GlassFish ニュース」ページを表示する」

  • 「GlassFish サーバー の「サブスクリプション」ページを表示する」

  • 「配備されているアプリケーションのリストを表示する」

  • 「Web アプリケーションを配備する」

  • 「エンタープライズアプリケーションを配備する」

  • 「アプリケーションクライアントを配備する」

  • 「コネクタモジュールを配備する」

  • 「EJB JAR を配備する」

  • 「インストールされたコンポーネントを表示する」

  • 「アドオンコンポーネントをインストールする」

  • 「コンポーネントの利用可能な更新を表示する」

  • JDBC 接続プールを作成する

「共通操作」ページには、一部の GlassFish サーバー マニュアルへのリンクも表示されます。
