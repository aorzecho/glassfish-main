リソース・ターゲットの管理

「リソース・ターゲットの管理」ページを使用し、リソースを有効にできるターゲット・クラスタとスタンドアロン・サーバー・インスタンスを変更します。リソースを有効にできるのは、「選択したターゲット」列内のターゲットのみです。

「リソース・ターゲットの管理」ページには、次の情報が表示されます。

使用可能なターゲット

リソースを有効にできないクラスタとインスタンス。

選択したターゲット

リソースを有効にできるクラスタとインスタンス。

追加

選択したターゲットを「使用可能なターゲット」から「選択したターゲット」に移動するボタン。

すべて追加

すべての「使用可能なターゲット」を「選択したターゲット」に移動するボタン。

削除

選択したターゲットを「選択したターゲット」から「使用可能なターゲット」に移動するボタン。

すべて削除

すべての「選択したターゲット」を「使用可能なターゲット」に移動するボタン。
