システム設定

「システム設定」ページには、選択した名前付き設定で定義されている Java システムプロパティーの一覧が表示されます。Java システムプロパティーは、GlassFish サーバー の起動時に Java アプリケーション起動用ウィンドウの -D オプションによって、Java アプリケーション起動用ウィンドウに渡されます。

これらのプロパティーにより、設定を参照するすべての GlassFish サーバー インスタンスのデフォルトのポート設定値が定義されます。

設定内の定義済みポート設定については、「定義済みのシステムプロパティー 」を参照してください。

プロパティーごとに、次の情報が表示されます。

インスタンス変数名

システムプロパティーの名前。

デフォルト値

インスタンスが参照する名前付き設定のプロパティーに対して設定されている値。このフィールドは読み取り専用です。

インスタンス値

プロパティーの「「インスタンス値」」ページに対するリンク。

「追加のプロパティー」の表には、次のオプションも表示されます。

プロパティーを追加

プロパティーを追加するボタン。このボタンをクリックすると、「追加のプロパティー」の表に行が追加されます。

プロパティーを削除

選択した 1 つまたは複数のプロパティーを削除するボタン。削除されたプロパティーはデフォルト値に戻ります。デフォルト値が設定されていない場合は未定義になります。

「システム設定」ページには、次のオプションも表示されます。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

動的再設定

このオプションを有効にすると、インスタンスを再起動しなくても、設定を参照するインスタンスに設定の変更が適用されます。

この設定を使用するクラスタ、インスタンス、または両方

設定を参照するクラスタとインスタンスの一覧。各クラスタまたはインスタンスが実行されているかどうかも示されます。

  • クラスタまたはインスタンスが実行されている場合は、チェックマーク (✓) が表示されます。

  • クラスタまたはインスタンスが停止している場合は、感嘆符 (!) が表示されます。
