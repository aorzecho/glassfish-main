ライフサイクルモジュールターゲット

「ライフサイクルモジュールターゲット」ページを使用し、ライフサイクルモジュールを有効にできるターゲットクラスタとスタンドアロンサーバーインスタンスを表示します。

「ライフサイクルモジュールターゲット」ページには、次の情報が表示されます。

ターゲット名

クラスタまたはスタンドアロンサーバーインスタンスの名前。

使用可能

ターゲットでライフサイクルモジュールが有効に設定されていると true が表示され、ライフサイクルモジュールが無効に設定されていると false が表示されます。

「ターゲット」の表には、次のオプションも表示されます。

ターゲットの管理

ライフサイクルモジュールターゲットを管理するボタン。

追加の操作

次のアクションが入ったドロップダウンリスト。

有効

選択したターゲットでライフサイクルモジュールを有効にするアクション。

無効

選択したターゲットでライフサイクルモジュールを無効にするアクション。
