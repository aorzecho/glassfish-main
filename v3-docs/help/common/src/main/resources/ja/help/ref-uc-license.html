ライセンス契約書

「ライセンス契約書」ページには、インストールまたは更新のために選択したすべてのコンポーネントに関するライセンス契約が表示されます。コンポーネントのライセンス契約は、コンポーネントの作成者によってコンポーネントとともに提供されます。ネットワーク・リポジトリの各種コンポーネントは、それぞれのライセンスに従って公開されます。例:

  • Common Development and Distribution License(CDDL)(http://www.sun.com/cddl/)

  • GNU General Public License(GPL)(http://www.gnu.org/copyleft/gpl.html)

  • Apache License(http://www.apache.org/licenses/)

  • wxWindows Library Licence(http://www.opensource.org/licenses/wxwindows.php)

「ライセンス契約書」ページには、次のオプションが含まれます。

受諾する

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件を受諾し、インストールまたはアップグレードを実行します。

同意しない

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件に同意せず、コンポーネントを選択したページに戻ります。
