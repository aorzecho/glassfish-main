ライセンス契約書

「ライセンス契約書」ページには、インストールまたは更新のために選択したすべてのコンポーネントに関するライセンス契約が表示されます。コンポーネントのライセンス契約は、コンポーネントの作成者によってコンポーネントとともに提供されます。「ネットワークリポジトリ」の各種コンポーネントは、それぞれのライセンスに従って公開されます。たとえば、次のようなものです。

  • Common Development and Distribution License (CDDL)

  • GNU General Public License (GPL)

  • Apache License

  • wxWindows Library Licence

「ライセンス契約書」ページには、次のオプションが含まれます。

同意する

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件に同意し、インストールまたはアップグレードを実行します。

同意しない

このボタンは、選択したすべてのコンポーネントのライセンス契約の条件に同意せず、コンポーネントを選択したページに戻ります。

関連操作

  • 「アドオンコンポーネントをインストールする」
