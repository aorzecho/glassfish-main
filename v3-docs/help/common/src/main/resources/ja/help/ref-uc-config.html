更新ツールの設定

「設定」ページには、更新ツール についての設定情報が表示されます。このページを使用し、更新ツール をインターネットに接続する方法を変更できます。更新ツール 設定のほかの内容を変更するには、as-install-parent/bin にあるスタンドアロンアプリケーション updatetool を使用してください。

「更新ツール の設定」ページには、次のトピックが含まれます。

インストールパス

インストールイメージのルートディレクトリまたは最上位ディレクトリのファイルシステムパス。

優先リポジトリ

インストールイメージに関連付けられている優先ネットワークリポジトリの名前。たとえば、glassfish.org などです。

インターネット接続

GlassFish Server のインターネットへの接続方法は次のとおりです。

直接接続

このオプションは、GlassFish Server がインターネットに直接接続するように指定します。

プロキシ経由

このオプションは、GlassFish Server がプロキシサーバー経由でインターネットに接続するように指定します。このオプションを選択する場合、「プロキシホスト」フィールドと「プロキシポート」フィールドがこのページに追加されます。

プロキシホスト

GlassFish Server を実行しているマシンのプロキシサーバーのホスト名。

このフィールドは、GlassFish Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

プロキシポート

GlassFish Server を実行しているマシンのプロキシサーバーのポート番号。

このフィールドは、GlassFish Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

リポジトリ

この表には、このインストールイメージに関連付けられている各ネットワークリポジトリに関する次の情報が含まれます。

リポジトリ

リポジトリの名前。たとえば、dev.glassfish.org などです。
