GlassFish Serverの「GlassFishニュース」ページを表示する

「GlassFishニュース」ページには、ニュースレター、記事、技術的なヒント、ディスカッション・フォーラム、関連ブログなどの、Oracle GlassFish Serverに関する追加情報へのリンクが表示されます。

  1. ナビゲーション・ツリーで、「共通タスク」ノードを選択します。

    「共通タスク」ページが開きます。

  2. 「GlassFishニュース」の下で、「GlassFishニュース」のリンクをクリックします。

    「GlassFishニュース」ページが、新しいブラウザ・タブまたはウィンドウで開きます。
