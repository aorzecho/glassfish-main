GlassFish サーバー の「GlassFish ニュース」ページを表示する

「GlassFish ニュース」ページには、ニュースレター、記事、技術的なヒント、ディスカッションフォーラム、関連ブログなどの、Oracle GlassFish サーバー に関する追加情報へのリンクが表示されます。

  1. ナビゲーションツリーで、「共通操作」ノードを選択します。

    「共通操作」ページが開きます。

  2. 「GlassFish ニュース」の下で、「GlassFish ニュース」のリンクをクリックします。

    「GlassFish ニュース」ページが、新しいブラウザタブまたはウィンドウで開きます。
