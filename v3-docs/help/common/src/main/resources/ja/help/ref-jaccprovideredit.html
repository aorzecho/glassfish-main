JACCプロバイダを編集

「JACCプロバイダを編集」ページを使用して、選択したGlassFish ServerインスタンスまたはクラスタのJACCプロバイダを変更します。

「JACCプロバイダを編集」ページには、次のオプションが含まれます。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

名前

JACCプロバイダの名前。このフィールドは読取り専用です。名前を指定できるのは、新しいJACCプロバイダの作成時のみです。

ポリシーの構成

プロバイダ固有のjavax.security.jacc.PolicyConfigurationFactoryを実装するjavax.security.jacc.PolicyConfigurationFactory.providerの完全修飾クラス名。

ポリシー・プロバイダ

java.security.Policyを実装するjavax.security.jacc.policy.providerの完全修飾クラス名。

追加プロパティ

JACCプロバイダの追加プロパティ。JACCプロバイダで使用可能な追加プロパティについては、JACCプロバイダに固有のプロパティを参照してください。
