ドメイン属性

GlassFish Server に関するドメインの詳細属性を設定するには、「ドメイン属性」ページを使用します。

「ドメイン属性」ページには、次のトピックが含まれます。

アプリケーションルート

配備されたアプリケーションが存在するディレクトリのフルパス。デフォルト値は domain-dir/applications です。

ログルート

サーバーのログファイルを保持する場所を指定します。デフォルト値は domain-dir/logs です。

ロケール

このフィールドを空白のままにすると、ホストのデフォルトのロケールが使用されます。

ロケールは、言語と地域の特定の組み合わせを指定する識別子です。たとえば、米国英語のロケールは en_US で、日本語のロケールは ja_JP です。英語以外のロケールを使用するには、英語を目的の言語に翻訳して、GlassFish Server をローカライズする必要があります。
