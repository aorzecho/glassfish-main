新しい監査モジュール

「新しい監査モジュール」ページを使用して、新しい監査モジュールを定義します。

「新しい監査モジュール」ページには、次のオプションが含まれます。

設定名

このページの設定を適用する設定の名前。 このフィールドは読み取り専用です。

名前

新しい監査モジュールの名前。

クラス名

このモジュールを実装するクラスの完全修飾名。デフォルトの監査モジュールのクラス名は、com.sun.enterprise.security.Audit です。

追加プロパティー

監査モジュールの追加プロパティー。監査モジュールで使用可能な追加プロパティーについては、「監査モジュールに固有のプロパティー」を参照してください。

関連操作

  • 「監査モジュールを作成する」

  • 「監査モジュールを編集する」

  • 「監査モジュールを削除する」

  • 「監査ログを有効または無効にする」

  • 「有効な監査モジュールを設定する」

  • 「デフォルト監査モジュールを使用する」

関連する asadmin コマンド

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
