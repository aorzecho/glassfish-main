監査モジュールを編集

「監査モジュールを編集」ページを使用して、監査モジュールを変更します。

「監査モジュールを編集」ページには、次のオプションが含まれます。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

名前

監査モジュールの名前。このフィールドは読み取り専用です。名前を指定できるのは、新しい監査モジュールの作成時のみです。

クラス名

このモジュールを実装するクラスの完全修飾名。デフォルトの監査モジュールのクラス名は、com.sun.enterprise.security.Audit です。

追加プロパティー

監査モジュールの追加プロパティー。監査モジュールで使用可能な追加プロパティーについては、「監査モジュールに固有のプロパティー」を参照してください。
