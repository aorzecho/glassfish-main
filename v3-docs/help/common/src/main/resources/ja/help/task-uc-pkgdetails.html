コンポーネントに関する詳細情報を表示する

更新ツール では、インストール済みコンポーネント、インストール済みコンポーネントに対して利用可能な更新、およびアドオンコンポーネントに関する詳細な情報を提供しています。

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「利用可能なアドオン」ページが表示されます。

  2. コンポーネントが一覧表示されているページのタブをクリックします。

    コンポーネントは次のページのいずれかに一覧表示されます。

    • 「利用可能なアドオン」

    • 「利用可能な更新」

    • 「インストール済みコンポーネント」

  3. コンポーネントの表で、コンポーネントを選択します。

    選択したコンポーネントの「コンポーネントの詳細」 ページが表示されます。
