コンポーネントに関する詳細情報を表示する

更新ツールでは、インストール済コンポーネント、インストール済コンポーネントに対して使用可能な更新、およびアドオン・コンポーネントに関する詳細な情報を提供しています。

  1. ナビゲーション・ツリーで、更新ツールノードを選択します。

    「使用可能なアドオン」ページが表示されます。

  2. コンポーネントが一覧表示されているページのタブをクリックします。

    コンポーネントは次のページのいずれかに一覧表示されます。

    • 使用可能なアドオン

    • 使用可能な更新

    • インストール済コンポーネント

  3. コンポーネントの表で、コンポーネントを選択します。

    選択したコンポーネントのコンポーネントの詳細ページが表示されます。

関連項目

  • コンポーネントに使用可能な更新を表示する

  • アドオン・コンポーネントをインストールする

  • インストールされたコンポーネントを表示する

  • ネットワーク・リポジトリにアクセスするためのネットワーク・プロキシ設定を構成する
