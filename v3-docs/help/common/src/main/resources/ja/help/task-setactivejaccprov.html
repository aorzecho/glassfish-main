有効な JACC プロバイダを設定する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「セキュリティー」ノードを選択します。

    「セキュリティ」ページが開きます。

  3. 「JACC」ドロップダウンリストから、サーバーで使用する JACC プロバイダを選択します。

    デフォルトでは、default および simple という名前の 2 つの JACC プロバイダを使用できます。デフォルトプロバイダは default です。

  4. 「保存」をクリックします。
