定義済のシステム・プロパティ

GlassFish Serverでは、次のシステム・プロパティが事前定義されています。

com.sun.aas.installRoot

GlassFish Serverがインストールされるディレクトリ。

com.sun.aas.instanceRoot

GlassFish Serverドメインの最上位ディレクトリ。

com.sun.aas.hostName

ホスト(マシン)の名前。

com.sun.aas.javaRoot

Javaランタイムのインストール・ディレクトリ。

com.sun.aas.imqLib

OracleMessage Queueソフトウェアのライブラリ・ディレクトリ。

com.sun.aas.domainName

ドメインの名前。このプロパティはデフォルトの構成では使用されませんが、構成のカスタマイズに使用できます。デフォルト値はdomain1です。
