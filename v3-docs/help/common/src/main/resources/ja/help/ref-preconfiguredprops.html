定義済みのシステムプロパティー

GlassFish サーバー では、次のシステムプロパティーがあらかじめ定義されています。

com.sun.aas.installRoot

GlassFish サーバー がインストールされるディレクトリ。

com.sun.aas.instanceRoot

GlassFish サーバー ドメインの最上位ディレクトリ。

com.sun.aas.hostName

ホスト (マシン) の名前。

com.sun.aas.javaRoot

Java ランタイムのインストールディレクトリ。

com.sun.aas.imqLib

Oracle メッセージキュー ソフトウェアのライブラリディレクトリ。

com.sun.aas.domainName

ドメインの名前。このプロパティーはデフォルトの設定では使用されませんが、設定のカスタマイズに使用できます。デフォルト値は domain1 です。
