監視

「監視」ページを使用して、監視を構成し、クラスタ化されたサーバー・インスタンスおよびクラスタ化されていないサーバー・インスタンスの監視データを表示します。

サーバー・インスタンスごとに、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。インスタンス名をクリックすると、インスタンスの「一般情報」ページに移動します。

クラスタ名

サーバー・インスタンスが属するクラスタの名前(クラスタ化されたインスタンスの場合)。クラスタ名をクリックすると、クラスタの「一般情報」ページに移動します。

アクション

サーバー・インスタンス上で次の操作を実行できるリンク。

監視を構成

クラスタまたはスタンドアロン・インスタンス構成の「監視を構成」ページが表示されます。

監視データを表示

サーバー・インスタンスの次のタイプの監視データを表示できるリンク。

アプリケーション

サーバー・インスタンスの「アプリケーションの監視」ページが表示されます。

サーバー

サーバー・インスタンスの「サーバーの監視」ページが表示されます。

リソース

サーバー・インスタンスの「リソースの監視」ページが表示されます。
