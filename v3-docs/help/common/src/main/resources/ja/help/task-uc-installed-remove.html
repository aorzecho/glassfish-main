インストールされたコンポーネントを表示する

「インストール済みコンポーネント」ページには、インストールされるインストールイメージのコンポーネントに関する情報が含まれます。

コンポーネントのアンインストールには次の項目の削除が含まれます。

  • コンポーネントの最初のインストールによって導入されたすべてのファイルとディレクトリ

  • 最初のインストール後に、コンポーネントの更新によって導入されたすべてのファイルとディレクトリ

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「利用可能なアドオン」ページが開きます。

  2. 「利用可能なアドオン」ページで、「インストール済みコンポーネント」タブをクリックします。

    「インストール済みコンポーネント」ページが開きます。

次の手順

インストール済みコンポーネントをアンインストールするには、まず GlassFish サーバー ドメインを停止します。続いて、次のいずれかのコマンドを使用します。

  • as-install-parent/bin/updatetool (更新ツールグラフィカルユーティリティーを起動)

  • as-install-parent/bin/pkg (コマンド行バージョンの更新ツール)

pkg コマンドの使用法については、『Oracle GlassFish Server 3.1 Administration Guide』の第 11 章「Extending and Updating GlassFish Server」を参照してください。更新ツールユーティリティーにはオンラインヘルプが用意されています。

コンポーネントのアンインストールが完了したら、サーバーを再起動します。
