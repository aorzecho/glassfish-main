メッセージセキュリティープロバイダが使用する GlassFish Server の機能を設定する

  1. ユーザー名トークンを使用する場合は、必要に応じてユーザーデータベースを設定します。

    ユーザー名およびパスワードトークンを使用する場合は、最初に適切なレルムを設定し、このレルムに適切なユーザーデータベースを設定します。

    ユーザーデータベースの設定方法については、「レルムを編集する」を参照してください。

  2. 必要に応じて証明書と非公開鍵を管理します。

次の手順

GlassFish Server の機能の設定が完了し、メッセージセキュリティープロバイダがそれらの機能を使用できるようになると、GlassFish Server とともにインストールされたプロバイダを有効にできます。手順については、「メッセージセキュリティー設定を編集する」を参照してください。
