監視サービス

「監視サービス」ページを使用して、個々のサーバーコンポーネントまたはサービスの監視オプションを設定します。

監視を設定する前に、監視対象のアプリケーション、コンポーネント、またはサービスを配備する必要があります。

「監視サービス」ページには、次のオプションが含まれます。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

Monitoring Service

「監視サービス」の「有効」チェックボックスをオンにすると、GlassFish サーバー で監視が有効になります。このオプションはデフォルトで有効になっています。

監視 MBean

「監視 MBean」の「有効」チェックボックスをオンにすると、監視に必要なすべての MBean が GlassFish サーバー に配備されます。このオプションはデフォルトで有効になっています。

チェックボックスがオンでない場合、「監視サービス」チェックボックスがオンになっている場合でも、管理コンソール で監視データを表示することはできません。この場合は、コマンド行を使用して監視データを表示する必要があります。チェックボックスのチェックマークを外すと、GlassFish サーバー のメモリーフットプリントが減少します。

動的トレース

Oracle Enterprise Manager DTrace Monitoring を使用している場合は、「DTrace」の「有効」チェックボックスをオンにすると、Solaris の動的トレース機能を使用できます。このオプションは、デフォルトでは無効になっています。

利用可能なコンポーネントごとに、次の情報が表示されます。

コンポーネント

GlassFish サーバー コンポーネントの名前。

監視レベル

コンポーネントに設定された監視のレベル。コンポーネントごとに、「LOW」、「HIGH」、および「OFF」のレベルを指定できます。デフォルト値は「OFF」です。現在のリリースでは、「LOW」オプションと「HIGH」オプションによる監視レベルに差はありません。

「コンポーネントレベルの設定」の表には、次のオプションも表示されます。

レベルの変更

選択した 1 つまたは複数のコンポーネントの監視レベルを変更するボタン。

レベル

選択したすべてのコンポーネントに適用する監視レベルを選択するドロップダウンリスト。
