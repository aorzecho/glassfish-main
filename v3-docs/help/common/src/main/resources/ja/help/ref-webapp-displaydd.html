配備記述子

「Deployment Descriptor」ページを使用して、モジュールの配備記述子を表示します。

「Deployment Descriptor」ページには、次の情報が表示されます。

アプリケーション名

アプリケーションの名前。

名前には、オプションのバージョン ID を含めることができます。このバージョン ID は名前のあとに続き、コロン (:) で名前と区切ります。モジュールとアプリケーションのバージョンについては、『Oracle GlassFish Server 3.1 Application Deployment Guide』の「Module and Application Versions」を参照してください。

モジュール名

配備記述子を含むモジュール名。

記述子ファイル名

配備記述子ファイル名。

  • 「アプリケーションクライアントを配備する」

  • 「コネクタモジュールを配備する」

  • 「EJB JAR を配備する」

  • 「配備済みアプリケーションを編集する」

  • 「アプリケーションを起動する」

  • 「アプリケーションクライアントを起動する」

  • 「配備済みアプリケーションを再配備する」

  • 「配備済みアプリケーションを再読み込みする」
