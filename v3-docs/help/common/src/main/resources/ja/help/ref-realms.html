レルム

GlassFish サーバー には、file、certificate、および admin-realm の 3 つのレルムが事前に設定されています。さらに、ldap、JDBC、solaris、またはカスタムレルムも作成できます。通常、サーバーにはタイプによってレルムが 1 つずつありますが、GlassFish サーバー の場合は、file および admin-realm の 2 つの file レルムがあります。これらは、異なる 2 つの目的に使用される同じタイプの 2 つのレルムです。また、システムの各仮想サーバーで異なる証明書データベースを持つこともできます。

レルムを設定するには、「レルム」ページを使用します。

レルムごとに、次の情報が表示されます。

名前

レルムの名前。

クラス名

レルムのクラス名。

「レルム」の表には、次のオプションも表示されます。

新規

新しいレルムを作成するボタン。

削除

選択した 1 つまたは複数のレルムを削除するボタン。

関連操作

  • 「レルムを作成する」

  • 「レルムを編集する」

  • 「レルムを削除する」

  • 「カスタムレルムを作成する」

関連する asadmin コマンド

  • create-auth-realm(1)

  • delete-auth-realm(1)

  • list-auth-realms(1)
