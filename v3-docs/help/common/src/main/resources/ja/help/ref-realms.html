レルム

GlassFish Serverには、file、certificate、およびadmin-realmの3つのレルムが事前に構成されています。さらに、ldap、JDBC、solaris、またはカスタム・レルムも作成できます。通常、サーバーにはタイプによってレルムが1つずつありますが、GlassFish Serverの場合は、fileおよびadmin-realmの2つのfileレルムがあります。これらは、異なる2つの目的に使用される同じタイプの2つのレルムです。また、システムの各仮想サーバーで異なる証明書データベースを持つこともできます。

レルムを構成するには、「レルム」ページを使用します。

レルムごとに、次の情報が表示されます。

名前

レルムの名前。

クラス名

レルムのクラス名。

「レルム」の表には、次のオプションも表示されます。

新規

新しいレルムを作成するボタン。

削除

選択した1つまたは複数のレルムを削除するボタン。
