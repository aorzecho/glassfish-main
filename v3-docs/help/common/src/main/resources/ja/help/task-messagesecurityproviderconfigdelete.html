メッセージセキュリティープロバイダ設定を削除する

始める前に

デフォルトの server-config 設定のメッセージセキュリティープロバイダの作成と削除は行えません。デフォルトの server-config でサポートされるメッセージセキュリティープロバイダは、SOAP と HttpServlet だけです。これらの 2 つのメッセージプロバイダは GlassFish Server のインストール時にデフォルトですでに作成されているため、デフォルトの server-config の「メッセージセキュリティー設定」ページに「削除」ボタンは表示されません。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「セキュリティー」ノードを開きます。
  3. 「メッセージセキュリティー」ノードを選択します。

    「メッセージセキュリティー設定」ページが開きます。

  4. 「メッセージセキュリティー設定」ページで、「SOAP」を選択します。

    「メッセージセキュリティ設定の編集」ページが開きます。

  5. 「メッセージセキュリティ設定の編集」ページで、「プロバイダ」タブをクリックします。

    「プロバイダ設定の編集」ページが開きます。

  6. 「プロバイダ設定の編集」ページで、削除するプロバイダ設定の左側にあるチェックボックスを選択します。
  7. 「削除」をクリックします。
