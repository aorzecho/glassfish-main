メッセージ・セキュリティ・プロバイダ構成を削除する

開始する前の準備

デフォルトのserver-config構成のメッセージ・セキュリティ・プロバイダの作成と削除は実行できません。デフォルトのserver-configでサポートされるメッセージ・セキュリティ・プロバイダは、SOAPとHttpServletのみです。これらの2つのメッセージ・プロバイダはGlassFish Serverのインストール時にデフォルトですでに作成されているため、デフォルトのserver-configの「メッセージ・セキュリティ構成」ページに「削除」ボタンは表示されません。

  1. ナビゲーション・ツリーで、「構成」ノードを展開します。

  2. 「セキュリティ」ノードを展開します。

  3. 「メッセージ・セキュリティ」ノードを選択します。

    「メッセージ・セキュリティ構成」ページが開きます。

  4. 「メッセージ・セキュリティ構成」ページで、「SOAP」を選択します。

    「メッセージ・セキュリティ構成の編集」ページが開きます。

  5. 「メッセージ・セキュリティ構成の編集」ページで、「プロバイダ」タブをクリックします。

    「プロバイダ構成の編集」ページが開きます。

  6. 「プロバイダ構成の編集」ページで、削除するプロバイダ構成の左側にあるチェックボックスを選択します。

  7. 「削除」をクリックします。
