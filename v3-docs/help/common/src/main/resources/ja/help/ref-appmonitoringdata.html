アプリケーションの監視

「アプリケーションの監視」ページを使用して、選択したアプリケーションの監視データを表示します。監視対象のプロパティーの説明については、『Oracle GlassFish Server 3.1 Administration Guide』の第 8 章「Administering the Monitoring Service」を参照してください。

関連操作

  • 「監視サービスを設定する」

  • 「アプリケーション監視データを表示する」

  • 「サーバーの監視データを表示する」

  • 「リソースの監視データを表示する」

関連する asadmin コマンド

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
