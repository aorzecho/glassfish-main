アプリケーションの監視

「アプリケーションの監視」ページを使用して、選択したアプリケーションの監視データを表示します。

「アプリケーションの監視」ページには、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。

アプリケーション

監視統計が表示されるアプリケーション。

コンポーネント

監視統計が表示されるアプリケーション・コンポーネント。

監視対象のプロパティについては、Oracle GlassFish Server管理ガイドの第8章、Administering the Monitoring Serviceを参照してください。
