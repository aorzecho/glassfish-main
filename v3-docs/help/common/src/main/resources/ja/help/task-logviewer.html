DAS のログファイルを表示する

  1. ナビゲーションツリーで、サーバー (管理サーバー) ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「ログファイルを表示」ボタンをクリックします。

    「ログビューア」ウィンドウが表示されます。
