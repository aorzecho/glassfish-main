新しい管理者パスワード

現在の管理者アカウントのパスワードを変更するには、「新しい管理者パスワード」ページを使用します。

「新しい管理者パスワード」ページには、次のオプションが含まれます。

ユーザー ID

現在の管理者のユーザー ID。このフィールドは読み取り専用です。

グループリスト

管理者が所属するグループ。このフィールドは読み取り専用です。値は asadmin です。

新しいパスワード

管理者の新しいパスワード。

新しいパスワードを確認

管理者の新しいパスワード。
