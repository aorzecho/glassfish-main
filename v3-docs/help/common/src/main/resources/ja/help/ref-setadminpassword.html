新しい管理者パスワード

現在の管理者アカウントのパスワードを変更するには、「新しい管理者パスワード」ページを使用します。

注意:

セキュリティ保護された管理が有効な場合は、パスワードを空の値に変更できません。

「新しい管理者パスワード」ページには、次のオプションが含まれます。

ユーザーID

現在の管理者のユーザーID。このフィールドは読取り専用です。

グループ・リスト

管理者が所属するグループ。このフィールドは読取り専用です。値はasadminです。

新しいパスワード

管理者の新しいパスワード。

新しいパスワードを確認

管理者の新しいパスワード。
