インスタンス値

「インスタンス値」ページを使用し、選択した名前付き設定を参照する GlassFish サーバー インスタンスの設定システムプロパティーの値を編集します。

「インスタンス値」ページには、プロパティーが設定されている選択した設定を参照するインスタンスの一覧が表示されます。

インスタンス名

インスタンスが作成されたときにインスタンスに割り当てられた名前。この名前をクリックすると、インスタンスの「一般情報」ページが表示されます。

クラスタ名

インスタンスがクラスタのメンバーである場合、そのクラスタの名前。このフィールドは読み取り専用です。

デフォルト値

インスタンスが参照する名前付き設定のプロパティーに対して設定されている値。このフィールドは読み取り専用です。

オーバーライド値

選択したインスタンスに対して設定されているプロパティーの値。この値はデフォルト値をオーバーライドします。
