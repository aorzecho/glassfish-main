仮想サーバーターゲット

「仮想サーバーターゲット」ページを使用し、アプリケーションまたはモジュールが関連付けられているターゲット仮想サーバーを変更します。

「仮想サーバーターゲット」ページには、次の情報が表示されます。

利用可能なターゲット

アプリケーションまたはモジュールが関連付けられていない仮想サーバー。

選択したターゲット

アプリケーションまたはモジュールが関連付けられている仮想サーバー。

追加

選択したターゲットを「利用可能なターゲット」から「選択したターゲット」に移動するボタン。

すべて追加

すべての「利用可能なターゲット」を「選択したターゲット」に移動するボタン。

削除

選択したターゲットを「選択したターゲット」から「利用可能なターゲット」に移動するボタン。

すべて削除

すべての「選択したターゲット」を「利用可能なターゲット」に移動するボタン。
