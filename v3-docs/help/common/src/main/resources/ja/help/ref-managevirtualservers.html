仮想サーバー・ターゲット

「仮想サーバー・ターゲット」ページを使用し、アプリケーションまたはモジュールが関連付けられているターゲット仮想サーバーを変更します。

「仮想サーバー・ターゲット」ページには、次の情報が表示されます。

使用可能なターゲット

アプリケーションまたはモジュールが関連付けられていない仮想サーバー。

選択したターゲット

アプリケーションまたはモジュールが関連付けられている仮想サーバー。

追加

選択したターゲットを「使用可能なターゲット」から「選択したターゲット」に移動するボタン。

すべて追加

すべての「使用可能なターゲット」を「選択したターゲット」に移動するボタン。

削除

選択したターゲットを「選択したターゲット」から「使用可能なターゲット」に移動するボタン。

すべて削除

すべての「選択したターゲット」を「使用可能なターゲット」に移動するボタン。
