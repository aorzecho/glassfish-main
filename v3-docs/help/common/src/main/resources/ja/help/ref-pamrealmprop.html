PamRealmクラスに固有のプロパティ

PAMレルムに必要なプロパティは次のとおりです。

JAASコンテキスト

JAAS(Java Authentication and Authorization Service)コンテキスト(このレルムに使用するログイン・モジュールの識別子)。有効な値はpamRealmのみです。

PAMレルムで使用可能なオプションのプロパティは次のとおりです。

グループ割当て

グループ名のカンマ区切りリスト。有効な証明書を提示するすべてのクライアントがこれらのグループに割り当てられます。たとえば、employee,managerなどで、これらはユーザー・グループの名前です。
