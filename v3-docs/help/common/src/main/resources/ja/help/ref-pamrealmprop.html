PamRealm クラスに固有のプロパティー

PAM レルムに必要なプロパティーは次のとおりです。

JAAS Context

JAAS (Java Authentication and Authorization Service) コンテキスト (このレルムに使用するログインモジュールの識別子)。有効な値は pamRealm のみです。

PAM レルムで使用可能なオプションのプロパティーは次のとおりです。

Assign Groups

コンマで区切られたグループ名のリスト。有効な証明書を提示するすべてのクライアントがこれらのグループに割り当てられます。たとえば、employee,manager などで、これらはユーザーグループの名前です。
