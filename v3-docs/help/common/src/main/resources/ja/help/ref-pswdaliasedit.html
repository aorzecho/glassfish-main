パスワードエイリアスの編集

「パスワードエイリアスの編集」ページを使用し、既存のパスワードエイリアスに関連付けるパスワードを変更します。

「パスワードエイリアスの編集」ページには、次のオプションが含まれます。

エイリアス名

エイリアスの名前。このオプションは編集できません。

パスワード

パスワードエイリアスによって暗号化された形式で格納されるパスワード。

パスワードの確認

パスワードエイリアスによって暗号化された形式で格納されるパスワード。「パスワード」と「パスワードの確認」の値が比較され、タイプミスのないパスワードが入力されたか確認されます。
