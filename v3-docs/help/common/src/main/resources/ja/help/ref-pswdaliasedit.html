パスワード・エイリアスの編集

「パスワード・エイリアスの編集」ページを使用し、既存のパスワード・エイリアスに関連付けるパスワードを変更します。

「パスワード・エイリアスの編集」ページには、次のオプションが含まれます。

エイリアス名

エイリアスの名前。このオプションは編集できません。

パスワード

パスワード・エイリアスによって暗号化された形式で格納されるパスワード。

パスワードの確認

パスワード・エイリアスによって暗号化された形式で格納されるパスワード。「パスワード」と「パスワードの確認」の値が比較され、タイプミスのないパスワードが入力されたか確認されます。
