ファイルレルムにユーザーを追加する

  1. 「設定」ノードの下で、管理レルムユーザーを管理するサーバーインスタンスまたはクラスタの設定を展開します。
  2. 「セキュリティー」ノードを開きます。
  3. 「レルム」ノードを開きます。
  4. 「file」ノードを選択します。

    「レルムを編集」ページが開きます。

  5. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

    「ファイルユーザー」ページが開きます。

  6. 「ファイルユーザー」ページの「新規」をクリックします。

    「新しいファイルレルムユーザー」ページが開きます。

  7. 「ユーザー ID 」フィールドに、ユーザーの名前を入力します。
  8. 「グループリスト」フィールドに、ユーザーが属するグループのリストを入力します。

    複数のグループ名はコンマで区切ります。これらのグループは、ほかの場所で定義される必要はありません。

  9. 「パスワード」フィールドに、一意のパスワードを入力します。
  10. 「新しいパスワードを確認」フィールドに、ユーザーパスワードを再度入力します。
  11. 「了解 (OK)」をクリックします。
