JVM プロファイラ設定

プロファイラツールは、パフォーマンスを分析し、潜在的な障害を特定するための情報を生成します。

プロファイラ設定を指定します。指定する情報は、使用するプロファイラ製品によって異なります。具体例と手順については、『Oracle GlassFish Server 3.1 Application Development Guide』の第 3 章「Debugging Applications」を参照してください。
