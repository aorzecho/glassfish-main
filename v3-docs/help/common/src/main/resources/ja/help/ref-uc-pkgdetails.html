コンポーネントの詳細

「コンポーネントの詳細」ページには、コンポーネントに関する詳細情報が含まれます。

「コンポーネントの詳細」ページには、次のオプションが含まれます。

戻る

このボタンをクリックすると、詳細が表示されているコンポーネントを選択したページに戻ります。

パッケージ名

コンポーネントが提供されるパッケージの名前。

一意の識別子

リポジトリに存在する、このコンポーネントの完全修飾識別子。

カテゴリ

コンポーネントが関連付けられているソフトウェアのカテゴリ。たとえば、アプリケーション・サーバー、システム・ツール、データベースなどです。カテゴリは、コンポーネントの作成時にコンポーネントの作成者によってコンポーネントに関連付けられます。

バージョン

コンポーネントが提供されるパッケージのバージョン番号。

公開日

このバージョンのコンポーネントがリポジトリに公開された日時。この日付は、インストール・イメージ内にコンポーネントがインストールされた日時ではありません。

インストール済サイズ

コンポーネントのインストールで消費されるディスク容量。

ソース

インストール済コンポーネントでは、コンポーネントが取得されたネットワーク・リポジトリの名前です。

アドオン・コンポーネントや更新では、コンポーネントが取得されるネットワーク・リポジトリの名前です。

説明

コンポーネントの概要。

関連タスク

コンポーネントに関する詳細情報を表示する
