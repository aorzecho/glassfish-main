アプリケーション監視データを表示する

始める前に

監視データを表示する前に、監視を設定する必要があります。詳細は、「監視サービスを設定する」を参照してください。

  1. ナビゲーションツリーで、サーバー (管理サーバー) ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  3. 「表示」ドロップダウンリストから、監視データを表示するアプリケーションを選択します。
  4. 表示されている監視データを更新するには、「更新」ボタンをクリックします。
