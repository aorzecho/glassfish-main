アプリケーションの監視データを表示する

開始する前の準備

監視データを表示する前に、監視を構成する必要があります。詳細は、監視サービスを構成するを参照してください。

  1. ナビゲーション・ツリーで、「監視データを表示」ノードを選択します。

    「監視」ページが開きます。

  2. 「監視データを表示」列で、監視しているサーバー・インスタンスに対応する「アプリケーション」リンクをクリックします。

    「アプリケーションの監視」ページが開きます。

  3. 「アプリケーション」ドロップダウン・リストから、監視データを表示するアプリケーションを選択します。

  4. 「コンポーネント」ドロップダウン・リストから、監視データを表示するアプリケーション・コンポーネントを選択します。

  5. 表示されている監視データを更新するには、「リフレッシュ」ボタンをクリックします。

代替手順:

  1. ナビゲーション・ツリーで、サーバー(管理サーバー)ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  3. 「アプリケーション」ドロップダウン・リストから、監視データを表示するアプリケーションを選択します。

  4. 「コンポーネント」ドロップダウン・リストから、監視データを表示するアプリケーション・コンポーネントを選択します。

  5. 表示されている監視データを更新するには、「リフレッシュ」ボタンをクリックします。
