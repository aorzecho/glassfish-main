設定

「設定」ページを使用し、設定の管理と、それらの設定を使用しているスタンドアロンサーバーインスタンスとクラスタの表示を行います。

default-config 設定は、GlassFish Server によってあらかじめ定義されています。server-config 設定は、ドメインが作成されると、ドメイン管理サーバー (DAS) 用に自動的に作成されます。詳細は、『Oracle GlassFish Server 3.1 High Availability Administration Guide』の「About Named Configurations」を参照してください。

設定ごとに、次の情報が表示されます。

名前

設定の名前。

インスタンス

設定を使用しているスタンドアロンインスタンスとクラスタの一覧。

「設定」の表には、次のオプションも表示されます。

新規

新しい設定を作成するボタン。

削除

選択した 1 つまたは複数の設定を削除するボタン。
