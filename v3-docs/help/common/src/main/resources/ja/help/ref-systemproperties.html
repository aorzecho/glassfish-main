システム・プロパティ

「システム・プロパティ」ページを使用して、GlassFish Serverのプロパティを表示および管理します。

システム・プロパティは、サーバー・レベルの設定に対して共通の値を定義します。管理コンソールのテキスト・フィールドでは、システム・プロパティはドル記号と中括弧で囲んで表します。たとえば、prop-nameという名前のプロパティを表すには、${prop-name}の構文を使用します。

「システム・プロパティ」ページには、プロパティごとに次の情報が表示されます。

名前

プロパティの名前。

値

プロパティの値

説明

プロパティのオプションの説明。

多数のシステム・プロパティが事前定義されていますが、これらはこのページに表示されません。詳細は、定義済のシステム・プロパティを参照してください。
