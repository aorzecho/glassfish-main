Listeners IIOP

Utilice la página Escuchas IIOP para configurar las escuchas IIOP.

Es posible configurar una serie de escuchas IIOP para ORB, cada una de las cuales puede aceptar conexiones en diferentes direcciones de red y combinaciones de puertos. Las escuchas se pueden configurar de forma diferente para las comunicaciones seguras y la autenticación de clientes.

Por defecto, hay tres listeners IIOP configurados: orb-listener-1, SSL y SSL_MUTUALAUTH.

Para cada escucha IIOP, la página Escuchas IIOP contiene la siguiente información.

Nombre

El nombre de la escucha IIOP.

Habilitado

"True" si la escucha IIOP está habilitada, o "false" si no lo está.

Dirección de red

La dirección IP (Protocolo de Internet) o el nombre de host con solución DNS de la escucha IIOP.

Puerto de escucha

El número de puerto de la escucha IIOP.
