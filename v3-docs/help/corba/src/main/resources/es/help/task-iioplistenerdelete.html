Cómo eliminar una escucha IIOP

  1. En el componente de árbol, expanda el nodo Configuración.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia predeterminada, server, seleccione el nodo server-config.
    • Para configurar los ajustes predeterminados para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.
  3. En el nodo Configuración, expanda el nodo ORB.
  4. Seleccione el nodo Escuchas IIOP.

    Se abrirá la página Escuchas IIOP.

  5. En la tabla, seleccione la casilla de verificación situada junto al nombre de la escucha que se va a eliminar.
  6. Haga clic en Eliminar.
