Configurar ORB

  1. En el componente de árbol, expanda el nodo Configuración.

  2. Seleccione la instancia que va a configurar:

    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia por defecto, server, seleccione el nodo server-config.

    • Para definir la configuración por defecto para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.

  3. Seleccione el nodo ORB.

    Se abrirá la página ORB.

  4. Seleccione el conjunto de subprocesos que utilizará ORB de la lista desplegable Identificador de conjunto de subprocesos.

    ORB utiliza conjuntos de subprocesos para responder a solicitudes de clientes remotos de beans empresariales y otros clientes que se comunican por medio de RMI-IIOP. El pool de threads por defecto para ORB es thread-pool-1, que utiliza valores por defecto apropiados para las solicitudes de RMI/IIOP.

    Consulte Conjuntos de subprocesos para obtener más información.

  5. En la lista desplegable Tamaño máximo de fragmento de mensaje, seleccione el tamaño máximo para los fragmentos de mensajes IIOP.

    Los mensajes más grandes que este tamaño quedarán fragmentados. El valor predeterminado es de 1024 bytes.

  6. En el campo Número total de conexiones, establezca el número máximo de conexiones entrantes para las escuchas IIOP.

    El valor predeterminado es 1024.

  7. Seleccione la casilla de verificación de Autenticación necesaria del cliente IIOP para especificar que se requiere la autenticación del cliente IIOP.

    Si la autenticación de cliente IIOP está habilitada, el servidor rechazará las solicitudes no autenticadas e insertará un bit de autenticación requerida en las referencias de objetos remotos interoperables (IORs) enviadas a los clientes. Esta opción se deshabilita por defecto.

  8. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para el ORB.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  9. Haga clic en Guardar.

    Sugerencia:

    Para volver a los valores de configuración predeterminados, haga clic en el botón Cargar predeterminados y, a continuación, en Guardar.

  10. Reinicie GlassFish Server.
