Nuevo Listener IIOP

Utilice la página Nueva escucha IIOP para crear una escucha IIOP nueva.

La página Nueva escucha IIOP contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

Un nombre exclusivo que identifica la escucha IIOP.

Dirección de red

La dirección de red de la escucha IIOP. La dirección de red puede ser una dirección IP o un nombre de host con resolución DNS.

Puerto de escucha

El número de puerto en el que la escucha IIOP va a recibir las conexiones.

Escucha

El estado de la escucha. Si la casilla de verificación Habilitado está seleccionada, estarán habilitadas las conexiones entrantes ORB IIOP para GlassFish Server. Esta opción se habilita por defecto.

Seguridad

Si la casilla de verificación Habilitado está seleccionada, SSL está habilitado para la escucha IIOP. Esta opción se deshabilita por defecto.

Propiedades adicionales

Propiedades adicionales de la escucha IIOP. GlassFish Server no define ninguna propiedad adicional de las escuchas IIOP.
