Cómo crear una escucha IIOP

  1. En el componente de árbol, expanda el nodo Configuración.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia predeterminada, server, seleccione el nodo server-config.
    • Para configurar los ajustes predeterminados para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.
  3. En el nodo Configuración, expanda el nodo ORB.
  4. Seleccione Escuchas IIOP.

    Se abrirá la página Escuchas IIOP.

  5. Haga clic en Nuevo.
  6. En el campo Nombre, escriba un nombre para identificar la escucha.
  7. En el campo Dirección de red, introduzca la dirección de red de la escucha.

    Puede introducir una dirección IP o un nombre de host que se pueda resolver por medio de un DNS.

  8. En el campo Puerto de la escucha, escriba el número del puerto en el que se tiene que realizar la escucha IIOP.
  9. Seleccione la casilla de verificación Escucha habilitada para habilitar las conexiones entrantes ORB IIOP para Servidor GlassFish.

    Esta opción se habilita por defecto.

  10. Seleccione la casilla de verificación Seguridad habilitada para habilitar SSL para la escucha IIOP.

    Esta opción se deshabilita por defecto.

  11. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Servidor GlassFish no define ninguna propiedad adicional de las escuchas IIOP.

    Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.

  12. Haga clic en Aceptar.
  13. Si ha marcado la casilla de verificación Seguridad habilitada, siga los pasos descritos en Cómo editar los parámetros de configuración de SSL de una escucha IIOP segura.
