Crear un Listener IIOP

  1. En el componente de árbol, expanda el nodo Configuración.

  2. Seleccione la instancia que va a configurar:

    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia por defecto, server, seleccione el nodo server-config.

    • Para definir la configuración por defecto para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.

  3. En el nodo Configuración, expanda el nodo ORB.

  4. Seleccione Escuchas IIOP.

    Se abrirá la página Escuchas IIOP.

  5. Haga clic en Nuevo.

  6. En el campo Nombre, escriba un nombre para identificar la escucha.

  7. En el campo Dirección de red, introduzca la dirección de red de la escucha.

    Puede introducir una dirección IP o un nombre de host que se pueda resolver por medio de un DNS.

  8. En el campo Puerto de la escucha, escriba el número del puerto en el que se tiene que realizar la escucha IIOP.

  9. Seleccione la casilla de verificación Escucha habilitada para habilitar las conexiones entrantes ORB IIOP para GlassFish Server.

    Esta opción se habilita por defecto.

  10. Seleccione la casilla de verificación Seguridad habilitada para habilitar SSL para la escucha IIOP.

    Esta opción se deshabilita por defecto.

  11. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional de las escuchas IIOP.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  12. Haga clic en Aceptar.

  13. Si activa la casilla de control Seguridad Activada, siga los pasos descritos en Editar Configuración SSL para un Listener IIOP Seguro.
