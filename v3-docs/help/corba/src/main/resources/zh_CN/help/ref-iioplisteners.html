IIOP 监听程序

使用“IIOP 监听程序”页可以配置 IIOP 监听程序。

可以为 ORB 配置多个 IIOP 监听程序，每个 IIOP 监听程序接受不同的网络地址和端口组合上的连接。可以为安全通信和客户机验证配置不同的监听程序。

默认情况下，已配置了三个 IIOP 监听程序：orb-listener-1、SSL 和 SSL_MUTUALAUTH。

对于每个 IIOP 监听程序，“IIOP 监听程序”页包含以下信息。

名称

IIOP 监听程序的名称。

已启用

如果 IIOP 监听程序已启用，则为“真”；如果 IIOP 监听程序未启用，则为“假”。

网络地址

IIOP 监听程序的 Internet 协议 (IP) 地址或 DNS 可解析的主机名。

监听程序端口

IIOP 监听程序的端口号。
