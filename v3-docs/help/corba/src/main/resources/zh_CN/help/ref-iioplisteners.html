IIOP 侦听器

使用“IIOP 侦听器”页面可以配置 IIOP 侦听器。

可以为 ORB 配置多个 IIOP 侦听器，每个 IIOP 侦听器接受不同的网络地址和端口组合上的连接。可以为安全通信和客户机验证配置不同的侦听器。

默认情况下，已配置了三个 IIOP 侦听器：orb-listener-1、SSL 和 SSL_MUTUALAUTH。

对于每个 IIOP 侦听器，“IIOP 侦听器”页面包含以下信息。

名称

IIOP 侦听器的名称。

已启用

如果 IIOP 侦听器已启用，则为 true；如果 IIOP 侦听器未启用，则为 false。

网络地址

IIOP 侦听器的 Internet 协议 (Internet Protocol, IP) 地址或 DNS 可解析的主机名。

侦听器端口

IIOP 侦听器的端口号。
