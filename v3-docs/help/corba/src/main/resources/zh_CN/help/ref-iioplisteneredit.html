编辑 IIOP 侦听器

使用“编辑 IIOP 侦听器”页面可以修改 IIOP 侦听器的设置。

“编辑 IIOP 侦听器”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用此页面上的设置的配置名称。此字段为只读字段。

名称

标识 IIOP 侦听器的唯一名称。“名称”字段是只读字段。只能在创建 IIOP 侦听器时指定名称。

网络地址

IIOP 侦听器的网络地址。网络地址可以是 IP 地址或 DNS 可解析的主机名。

侦听器端口

IIOP 侦听器要侦听的端口号。

侦听器

侦听器的状态。如果选中“已启用”复选框，则将启用至 GlassFish Server 的 ORB IIOP 入站连接。默认情况下启用此选项。

Security

如果选中“已启用”复选框，则将为 IIOP 侦听器启用 SSL。默认情况下禁用此选项。

其他属性

IIOP 侦听器的其他属性。GlassFish Server 不定义 IIOP 侦听器的任何其他属性。
