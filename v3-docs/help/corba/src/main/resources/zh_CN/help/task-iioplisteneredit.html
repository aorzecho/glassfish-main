编辑 IIOP 侦听器

  1. 在树组件中，展开“配置”节点。
  2. 选择要配置的实例：
    • 要配置特定的实例，请选择该实例的配置节点。例如，对于默认实例 server，请选择 server-config 节点。
    • 要为以后使用 default-config 副本的实例配置默认设置，请选择 default-config 节点。
  3. 在“配置”节点下，展开 "ORB" 节点。
  4. 选择“IIOP 侦听程序”节点。

    此时将打开“IIOP 侦听器”页面。

  5. 在表中，单击要编辑的侦听器的名称。

    此时将打开“编辑 IIOP 侦听器”页面。

  6. 在“网络地址”字段中，键入侦听器的网络地址。

    可以键入 IP 地址，也可以键入 DNS 可解析的主机名。

  7. 在“侦听器端口”字段中，键入 IIOP 侦听器要侦听的端口号。
  8. 选中“启用侦听器”复选框以启用至 GlassFish Server 的 ORB IIOP 入站连接。

    默认情况下启用此选项。

  9. 选中“启用安全性”复选框，为 IIOP 侦听器启用 SSL。

    默认情况下禁用此选项。

  10. （可选的）在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义 IIOP 侦听器的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。
    • 要修改属性，请编辑该属性的“值”字段。
    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。
  11. 单击“保存”。

    提示 - 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。

  12. 如果选中“启用安全性”复选框，请单击 "SSL" 制表符窗格，以修改此 IIOP 侦听器的安全套接字层设置。

    有关详细信息，请参见编辑安全 IIOP 侦听器的 SSL 设置。

  13. 如果更改了侦听器的端口号，请重新启动 GlassFish Server。
