IIOP-Listener

Verwenden Sie die Seite "IIOP-Listener", um IIOP-Listener zu konfigurieren.

Es kann eine bestimmte Anzahl von IIOP-Zielgeräten für einen ORB konfiguriert werden, von denen jedes Verbindungen an verschiedenen Netzwerkadressen und Portkombinationen annimmt. Listener können für sichere Kommunikationen und Clientauthentifizierungen unterschiedlich konfiguriert werden.

Standardmäßig werden drei IIOP-Listener konfiguriert: orb-listener-1, SSL und SSL_MUTUALAUTH.

Die Seite "IIOP-Listener" bietet für jeden IIOP-Listener die folgenden Informationen.

Name

Der Name des IIOP-Listeners.

Enabled (Aktiviert)

Der Wert "true", wenn der IIOP-Listener aktiviert ist, oder "false", wenn der Listener nicht aktiviert ist.

Netzwerkadresse

Die IP-Adresse (Internet Protocol) oder der von DNS auflösbare Hostname für den IIOP-Listener.

Listening-Port

Die Portnummer für den IIOP-Listener.

Verwandte Aufgaben

  • So bearbeiten Sie einen IIOP-Listener

  • So bearbeiten Sie die SSL-Einstellungen für einen sicheren IIOP-Listener

  • So löschen Sie einen IIOP-Listener
