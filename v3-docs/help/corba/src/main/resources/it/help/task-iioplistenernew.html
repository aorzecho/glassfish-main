Creare un listener IIOP

  1. Nella struttura ad albero di navigazione, espandere il nodo Configurazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza specifica, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite di istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.

  3. Nel nodo Configurazione, espandere il nodo ORB.

  4. Selezionare Listener IIOP.

    Viene visualizzata la pagina Listener IIOP.

  5. Fare clic su Nuovo.

  6. Nel campo Nome, immettere un nome per identificare il listener.

  7. Nel campo Indirizzo di rete, immettere l'indirizzo di rete del listener.

    È possibile immettere un indirizzo IP o un nome host DNS risolvibile.

  8. Nel campo Porta listener, immettere il numero della porta in cui il listener IIOP è in ascolto.

  9. Selezionare la casella di controllo Listener abilitato per abilitare le connessioni in entrata ORB IIOP per GlassFish Server.

    Questa opzione è abilitata per impostazione predefinita.

  10. Selezionare la casella di controllo Sicurezza abilitata per abilitare SSL per il listener IIOP.

    Questa opzione è disabilitata per impostazione predefinita.

  11. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per i listener IIOP.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  12. Fare clic su OK.

  13. Se si seleziona la casella di controllo Sicurezza abilitata, seguire la procedura descritta in Modificare le impostazioni SSL per un listener IIOP sicuro.
