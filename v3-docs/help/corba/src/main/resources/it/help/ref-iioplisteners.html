Listener IIOP

Nella pagina Listener IIOP è possibile configurare i listener IIOP.

Per un ORB possono essere configurati più listener IIOP in modo che ciascuno di essi accetti connessioni su diverse combinazioni di porta e indirizzo di rete. I listener possono essere configurati in modo diverso per le comunicazioni sicure e per l'autenticazione dei client.

Per impostazione predefinita, sono configurati tre listener IIOP: orb-listener-1, SSL e SSL_MUTUALAUTH.

Per ciascun listener IIOP la pagina Listener IIOP contiene le informazioni riportate di seguito.

Nome

Nome del listener IIOP.

Abilitato

Opzione attivata se il listener IIOP è abilitato; disattivata se il listener è disabilitato.

Indirizzo di rete

L'indirizzo IP (Internet Protocol) o il nome host risolvibile da DNS per il listener IIOP.

Porta listener

Numero di porta del listener IIOP.
