IIOP Listeners

Nella pagina IIOP Listeners è possibile configurare i listener IIOP.

Per un ORB è possibile configurare vari listener IIOP. Ognuno di essi accetta connessioni su combinazioni diverse indirizzo di rete/porta. È possibile configurare i listener in modo diverso per le comunicazioni sicure e per l'autenticazione client.

Per impostazione predefinita, sono configurati tre listener IIOP: orb-listener-1, SSL e SSL_MUTUALAUTH.

Per ciascuno di essi la pagina IIOP Listeners contiene le seguenti informazioni.

Name

Nome del listener IIOP.

Enabled

Opzione attivata se il listener IIOP è abilitato e disattivata se il listener è disabilitato.

Network Address

L'indirizzo IP (Internet Protocol) o il nome host DNS risolvibile per il listener IIOP.

Listener Port

Numero di porta del listener IIOP.
