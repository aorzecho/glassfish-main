Nuovo listener IIOP

Nella pagina Nuovo listener IIOP è possibile creare un nuovo listener IIOP.

La pagina Nuovo listener IIOP contiene le opzioni riportate di seguito.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome univoco che identifica il listener IIOP.

Indirizzo di rete

Indirizzo di rete del listener IIOP. L'indirizzo di rete può essere un indirizzo IP o un nome host risolvibile da DNS.

Porta listener

Numero di porta in cui il listener IIOP è in ascolto.

Listener

Stato del listener. Se la casella di controllo Abilitato è selezionata, le connessioni IIOP ORB in entrata per GlassFish Server sono abilitate. Questa opzione è abilitata per impostazione predefinita.

Sicurezza

Se la casella di controllo Abilitato è selezionata, SSL è abilitato per il listener IIOP. Questa opzione è disabilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per il listener IIOP. In GlassFish Server non è definita alcuna proprietà aggiuntiva per i listener IIOP.
