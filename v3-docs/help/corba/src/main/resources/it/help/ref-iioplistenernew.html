New IIOP Listener

Nella pagina New IIOP Listener è possibile creare un nuovo listener IIOP.

La pagina New IIOP Listener contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Name

Nome univoco che identifica il listener IIOP.

Network Address

Indirizzo di rete del listener IIOP. L'indirizzo di rete può essere un indirizzo IP o un nome host DNS risolvibile.

Listener Port

Numero di porta in cui il listener IIOP è in ascolto.

Listener

Stato del listener. Se è stata selezionata la casella di controllo Enabled, le connessioni in entrata ORB IIOP per GlassFish Server sono abilitate. Questa opzione è abilitata per impostazione predefinita.

Security

Se la casella di controllo Enabled è selezionata, SSL è abilitato per il listener IIOP. Questa opzione è disabilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per il listener IIOP. In GlassFish Server non è definita alcuna proprietà aggiuntiva per i listener IIOP.
