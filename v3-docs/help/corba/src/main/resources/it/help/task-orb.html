Configurazione dell'ORB

  1. Nella struttura ad albero di navigazione, espandere il nodo Configurazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza specifica, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite per le istanze future che utilizzino una copia di default-config, selezionare il nodo default-config.

  3. Selezionare il nodo ORB.

    Viene visualizzata la pagina ORB.

  4. Nell'elenco a discesa ID pool di thread selezionare il pool del thread utilizzato in ORB.

    ORB utilizza i pool del thread per rispondere alle richieste di client remoti di bean enterprise e di altri client che comunicano mediante RMI-IIOP. Il pool del thread predefinito per l'ORB è thread-pool-1, che utilizza impostazioni predefinite appropriate per RMI/IIOP.

    Per ulteriori informazioni, vedere Thread Pools.

  5. Nell'elenco a discesa Max Message Fragment Size, selezionare la dimensione massima del frammento per messaggi IIOP.

    Se i messaggi hanno dimensioni superiori a quella specificata, vengono frammentati. Il valore predefinito è pari a 1024 byte.

  6. Nel campo Connessioni totali, impostare il numero massimo di connessioni in entrata per tutti i listener IIOP.

    Il valore predefinito è pari a 1024.

  7. Selezionare la casella di controllo Autenticazione client IIOP necessaria per specificare che è necessaria l'autorizzazione client IIOP.

    Se l'autenticazione client IIOP è abilitata, il server rifiuta le richieste non autenticate e inserisce un bit richiesto per l'autenticazione nei riferimenti IOR (Interoperable Object References) inviati ai client. Questa opzione è disabilitata per impostazione predefinita.

  8. Nell'area Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per ORB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  10. Riavviare GlassFish Server.
