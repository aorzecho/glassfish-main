Configurare ORB

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare l'istanza da configurare:
    • Per configurare una particolare istanza, selezionare il nodo config dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.
    • Per configurare le impostazioni predefinite di istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.
  3. Selezionare il nodo ORB.

    Viene visualizzata la pagina ORB.

  4. Nell'elenco a discesa Thread Pool ID selezionare il pool del thread utilizzato in ORB.

    ORB utilizza i pool del thread per rispondere alle richieste di client remoti di bean enterprise e di altri client che comunicano mediante RMI-IIOP. Il pool del thread predefinito per ORB è thread-pool-1, che utilizza impostazioni predefinite appropriate per RMI/IIOP.

    Per ulteriori informazioni, vedere Thread Pools.

  5. Nell'elenco a discesa Max Message Fragment Size, selezionare la dimensione massima del frammento per messaggi IIOP.

    Se i messaggi hanno dimensioni superiori a quella specificata, vengono frammentati. Il valore predefinito è pari a 1024 byte.

  6. Nel campo Total Connections, impostare il numero massimo di connessioni in entrata per tutti i listener IIOP.

    Il valore predefinito è pari a 1024.

  7. Selezionare la casella di controllo IIOP Client Authentication Required per specificare che è necessaria l'autorizzazione client IIOP.

    Se l'autenticazione client IIOP è abilitata, il server rifiuta le richieste non autenticate e inserisce un bit richiesto per l'autenticazione nei riferimenti IOR (Interoperable Object References) inviati ai client. Questa opzione è disabilitata per impostazione predefinita.

  8. (Opzionale) Nell'area Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per ORB.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario modificarne il campo Value.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  9. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.

  10. Riavviare GlassFish Server.
