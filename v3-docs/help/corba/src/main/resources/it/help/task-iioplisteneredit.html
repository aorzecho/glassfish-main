Modificare un listener IIOP

  1. Nella struttura ad albero di navigazione, espandere il nodo Configurazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza specifica, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite di istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.

  3. Nel nodo Configurazione, espandere il nodo ORB.

  4. Selezionare il nodo Listener IIOP.

    Viene visualizzata la pagina Listener IIOP.

  5. Nella tabella fare clic sul nome del listener da modificare.

    Viene visualizzata la pagina Modifica listener IIOP.

  6. Nel campo Indirizzo di rete, immettere l'indirizzo di rete del listener.

    È possibile immettere un indirizzo IP o un nome host DNS risolvibile.

  7. Nel campo Porta listener, immettere il numero della porta in cui il listener IIOP è in ascolto.

  8. Selezionare la casella di controllo Listener abilitato per abilitare le connessioni in entrata ORB IIOP per GlassFish Server.

    Questa opzione è abilitata per impostazione predefinita.

  9. Selezionare la casella di controllo Sicurezza abilitata per abilitare SSL per il listener IIOP.

    Questa opzione è disabilitata per impostazione predefinita.

  10. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per i listener IIOP.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  11. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  12. Se la casella di controllo Sicurezza abilitata è selezionata, fare clic sul riquadro a schede SSL per modificare le impostazioni Secure Socket Layer relative a tale listener IIOP.

    Per ulteriori dettagli, vedereModificare le impostazioni SSL per un listener IIOP sicuro.

  13. Se è stato modificato il numero di porta del listener, riavviare il GlassFish Server.
