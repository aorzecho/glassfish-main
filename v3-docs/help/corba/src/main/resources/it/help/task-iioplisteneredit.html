Modificare un listener IIOP

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare l'istanza da configurare:
    • Per configurare una particolare istanza, selezionare il nodo config dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.
    • Per configurare le impostazioni predefinite di istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.
  3. Nel nodo Configuration, espandere il nodo ORB.
  4. Selezionare il nodo IIOP Listeners.

    Viene visualizzata la pagina IIOP Listeners.

  5. Nella tabella fare clic sul nome del listener da modificare.

    Viene visualizzata la pagina Edit IIOP Listener.

  6. Nel campo Network Address, immettere l'indirizzo di rete del listener.

    È possibile immettere un indirizzo IP o un nome host DNS risolvibile.

  7. Nel campo Listener Port, immettere il numero della porta in cui il listener IIOP è in ascolto.
  8. Selezionare la casella di controllo Listener Enabled per abilitare le connessioni in entrata ORB IIOP per GlassFish Server.

    Questa opzione è abilitata per impostazione predefinita.

  9. Selezionare la casella di controllo Security Enabled per abilitare SSL per il listener IIOP.

    Questa opzione è disabilitata per impostazione predefinita.

  10. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per i listener IIOP.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario modificarne il campo Value.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  11. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.

  12. Se la casella di controllo Security Enabled è selezionata, fare clic sul riquadro a schede SSL per modificare le impostazioni Secure Socket Layer relative a tale listener IIOP.

    Per dettagli, vedere Modificare impostazioni SSL per un listener IIOP sicuro.

  13. Se è stato modificato il numero di porta del listener, riavviare il GlassFish Server.
