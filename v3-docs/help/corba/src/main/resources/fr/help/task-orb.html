Configurer le programme ORB

  1. Dans l'arborescence, développez le noeud Configuration.

  2. Sélectionnez l'instance à configurer :

    • Pour configurer une instance spécifique, sélectionnez le noeud de configuration de l'instance. Par exemple, pour l'instance par défaut, server, sélectionnez le noeud server-config.

    • Pour configurer les paramètres par défaut des futures instances utilisant une copie de default-config, sélectionnez le noeud default-config.

  3. Sélectionnez le noeud ORB.

    La page ORB apparaît.

  4. Choisissez le pool de threads utilisé par le programme ORB dans la liste déroulante ID du pool de threads.

    L'ORB utilise des pools de threads pour répondre aux demandes des clients distants d'Enterprise Beans et d'autres clients qui communiquent via le protocole RMI-IIOP. Le pool de threads par défaut de l'ORB est thread-pool-1 qui utilise les valeurs par défaut appropriées aux demandes RMI/IIOP.

    Pour plus d'informations, reportez-vous à Pools de threads.

  5. Dans la liste déroulante Taille max. du fragment de message, sélectionnez la taille maximale du fragment des messages IIOP.

    Les messages dépassant cette taille sont fragmentés. La valeur par défaut est 1 024 octets.

  6. Dans le champ Nombre total de connexions, définissez le nombre maximal de connexions entrantes pour l'ensemble des processus d'écoute IIOP.

    La valeur par défaut est 1 024.

  7. Cochez la case Requis du champ Authentification client IIOP pour indiquer que l'authentification client IIOP est obligatoire.

    Si l'authentification client IIOP est activée, le serveur rejette les demandes non authentifiées et insère un bit de demande d'authentification dans les références IOR (Interoperable Object Reference) envoyées aux clients. Cette option est désactivée par défaut.

  8. Dans la zone Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server ne définit aucune autre propriété pour l'ORB.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  9. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.

  10. Redémarrez GlassFish Server.
