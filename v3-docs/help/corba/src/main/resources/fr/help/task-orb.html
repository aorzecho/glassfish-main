Pour configurer l’ORB

  1. Dans l’arborescence, développez le nœud Configuration.
  2. Sélectionnez l’instance à configurer :
    • Pour configurer une instance en particulier, sélectionnez le nœud de configuration de l’instance. Par exemple, pour l’instance par défaut, server, sélectionnez le nœud server-config.
    • Pour configurer les paramètres par défaut des futures instances utilisant une copie de default-config, sélectionnez le nœud default-config.
  3. Sélectionnez le nœud ORB.

    La page ORB s’ouvre.

  4. Choisissez le pool de threads utilisé par ORB dans la liste déroulante ID du pool de threads.

    L’ORB utilise des pools de threads pour répondre aux requêtes des clients distants de beans entreprise et d’autres clients qui communiquent par RMI-IIOP. Le pool de threads par défaut de l’ORB est thread-pool-1 qui utilise les valeurs par défaut appropriées aux requêtes RMI/IIOP.

    Voir Thread Pools pour de plus amples informations.

  5. Dans la liste déroulante Taille max. du fragment de message, sélectionnez la taille maximale de fragment des messages IIOP.

    Les messages dépassant cette taille sont fragmentés. La valeur par défaut est 1024 octets.

  6. Dans le champ Nombre total de connexions, définissez le nombre maximum de connexions entrantes pour tous les listeners IIOP.

    La valeur par défaut est 1024.

  7. Cochez la case Authentification client IIOP pour indiquer que l’authentification client IIOP est requise.

    Si l’authentification client IIOP est activée, le serveur refuse les requêtes non authentifiées et insère un bit de demande d’authentification dans les IOR (Interoperable Object Reference) envoyées aux clients. Cette option est désactivée par défaut.

  8. (Facultatif) Dans la zone Autres propriétés, spécifiez d’autres propriétés.

    GlassFish Server ne définit aucune autre propriété pour l’ORB.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  9. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.

  10. Redémarrez GlassFish Server.
