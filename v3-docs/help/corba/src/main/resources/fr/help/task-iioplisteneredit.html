Modifier un processus d'écoute IIOP

  1. Dans l'arborescence, développez le noeud Configuration.

  2. Sélectionnez l'instance à configurer :

    • Pour configurer une instance spécifique, sélectionnez le noeud de configuration de l'instance. Par exemple, pour l'instance par défaut, server, sélectionnez le noeud server-config.

    • Pour configurer les paramètres par défaut des futures instances utilisant une copie de default-config, sélectionnez le noeud default-config.

  3. Sous le noeud Configuration, développez le noeud ORB.

  4. Sélectionnez le noeud Processus d'écoute IIOP.

    La page Processus d'écoute IIOP apparaît.

  5. Dans la table, cliquez sur le nom du processus d'écoute à modifier.

    La page Modifier un processus d'écoute IIOP apparaît.

  6. Dans le champ Adresse réseau, entrez l'adresse réseau du processus d'écoute.

    Vous pouvez indiquer une adresse IP ou un nom d'hôte DNS pouvant être résolu.

  7. Dans le champ Port du processus d'écoute, entrez le numéro de port d'écoute que le processus d'écoute IIOP doit utiliser.

  8. Cochez la case Activé du champ Processus d'écoute pour activer les connexions entrantes IIOP de l'ORB à GlassFish Server.

    Cette option est activée par défaut.

  9. Cochez la case Sécurité activée pour activer le protocole SSL pour le processus d'écoute IIOP.

    Cette option est désactivée par défaut.

  10. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server ne définit aucune autre propriété pour les processus d'écoute IIOP.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  11. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.

  12. Si la case Sécurité activée est cochée, cliquez sur le volet à onglets SSL pour modifier les paramètres SSL (Secure Socket Layer) du processus d'écoute IIOP.

    Pour plus d'informations, reportez-vous à Modifier les paramètres SSL d'un processus d'écoute IIOP sécurisé.

  13. Si vous avez modifié le numéro de port du processus d'écoute, redémarrez GlassFish Server.
