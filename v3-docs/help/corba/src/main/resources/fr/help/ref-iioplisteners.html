Modules d’écoute IIOP

La page Modules d’écoute IIOP vous permet de configurer les listeners.

Certains listeners IIOP peuvent être configurés pour un ORB, chacun acceptant des connexions sur différentes adresses réseau et combinaisons de port. Les listeners peuvent être configurés différemment pour des communications sécurisées et l’authentification client.

Par défaut, trois listeners IIOP sont configurés : orb-listener-1, SSL et SSL_MUTUALAUTH.

Pour chaque listener IIOP, la page Modules d’écoute IIOP contient les informations suivantes :

Nom

Nom du listener IIOP.

Activée

Vrai si le listener IIOP est activé ou faux si le listener est désactivé.

Adresse réseau

Adresse IP (Internet Protocol) ou nom d’hôte DNS pouvant être résolu par le listener IIOP.

Port du module d’écoute

Numéro de port du listener IIOP.
