Modification d’un listener IIOP

Utilisez la page Modification d’un module d’écoute IIOP pour modifier les paramètres d’un listener IIOP.

La page Modification d’un module d’écoute IIOP contient les options suivantes :

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom

Nom unique qui identifie le listener IIOP. Le champ Nom est en lecture seule. Vous ne pouvez spécifier le nom qu’à la création d’un listener IIOP.

Adresse réseau

Adresse réseau du listener IIOP. L’adresse réseau peut être une adresse IP ou un nom d’hôte DNS pouvant être résolu.

Port du module d’écoute

Numéro du port d’écoute du listener IIOP.

Module d’écoute

Statut du listener. Si la case Activé est cochée, les connexions entrantes IIOP de l’ORB à GlassFish Server sont activées. Cette option est activée par défaut.

Securité

Si la case Activé est cochée, la SSL est activée pour le listener IIOP. Cette option est désactivée par défaut.

Autres propriétés

Propriétés supplémentaires du listener IIOP. GlassFish Server ne définit aucune autre propriété pour les listeners IIOP.
