刪除 IIOP 偵聽程式

  1. 在樹形元件中，展開 [配置] 節點。
  2. 選取要配置的實例：
    • 若要配置特定的實例，請選取該實例的配置節點。例如，對於預設實例 server，請選取 server-config 節點。
    • 若要配置使用 default-config 副本的未來實例的預設設定，請選取 default-config 節點。
  3. 在 [配置] 節點下，展開 [ORB] 節點。
  4. 選取 [IIOP 偵聽程式] 節點。

    [IIOP 偵聽程式] 頁面會隨即開啟。

  5. 在表中，選取要刪除的偵聽程式名稱旁的核取方塊。
  6. 按一下 [刪除]。
