編輯 IIOP 偵聽程式

  1. 在樹形元件中，展開 「組態」 節點。

  2. 選取要組態的實例：

    • 若要組態特定的實例，請選取該實例的組態節點。例如，對於預設實例 server，請選取 server-config 節點。

    • 若要組態使用 default-config 副本的未來實例的預設設定，請選取 default-config 節點。

  3. 在 「組態」 節點下，展開 「ORB」 節點。

  4. 選取 「IIOP 偵聽程式」 節點。

    「IIOP 偵聽程式」 頁面會隨即開啟。

  5. 在表中，按一下要編輯的偵聽程式名稱。

    「編輯 IIOP 偵聽程式」 頁面會隨即開啟。

  6. 在 「網路位址」 欄位中，鍵入偵聽程式的網路位址。

    您可以鍵入 IP 位址或 DNS 可解析主機名稱。

  7. 在 「偵聽程式連接埠」 欄位中，鍵入 IIOP 偵聽程式要偵聽的連接埠號碼。

  8. 選取「偵聽程式」的「已啟用」核取方塊可啟用通往 GlassFish Server 的 ORB IIOP 輸入連線。

    此選項預設為啟用。

  9. 選取「安全性」的「已啟用」核取方塊可為 IIOP 偵聽程式啟用 SSL。

    此選項預設為停用。

  10. 在 「其他特性」 區段中，指定其他特性。

    GlassFish Server 未定義 IIOP 偵聽程式的其他特性。

    • 若要增加特性，請按一下 「增加特性」 按鈕。在出現的空白列中，於 「名稱」 欄位中鍵入特性名稱，然後在 「值」 欄位中鍵入特性值。

    • 若要修改特性，請編輯特性的 「值」 欄位。

    • 若要刪除特性，請選取您要刪除之特性 「名稱」 欄位左側的核取方塊，然後按一下 「刪除特性」 按鈕。

  11. 按一下 「儲存」。

    秘訣:

    若要回到預設設定，請按一下 「載入預設值」 按鈕，然後按一下 「儲存」。

  12. 若選取了「安全性」的「已啟用」核取方塊，請按一下 SSL 標籤式窗格以修改此 IIOP 偵聽程式的安全通訊端層設定。

    如需詳細資訊，請參閱編輯安全 IIOP 偵聽程式的 SSL 設定。

  13. 如果變更了偵聽程式的連接埠號碼，請重新啟動 GlassFish Server。
