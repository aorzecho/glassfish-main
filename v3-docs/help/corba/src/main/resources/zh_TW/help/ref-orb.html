ORB

使用 [ORB] 頁面可配置物件請求代理程式。

ORB 提供所需的基礎架構來識別並尋找物件、處理連線管理、傳送資料並請求通訊。企業 Bean (EJB 模組) 的遠端用戶端使用 ORB 與 GlassFish Server 進行通訊。

[ORB] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

執行緒池 ID

ORB 所使用的執行緒池。ORB 使用執行緒池回應來自使用 RMI-IIOP 進行通訊的企業 Bean 的遠端用戶端和其他用戶端的請求。ORB 的預設執行緒池為 thread-pool-1，它會使用適合 RMI/IIOP 請求的預設值。

最大訊息片段大小

IIOP 訊息的最大片段大小。大於該大小的訊息將被分段。預設值為 1024 位元組。

連線總數

所有 IIOP 偵聽程式的最大內送連線數目。預設值為 1024。

IIOP 用戶端認證

若選取了 [啟用] 核取方塊，伺服器會拒絕未經認證的請求，並在傳送給用戶端的可互通物件參照 (IOR) 中插入需要認證的位元。您可以啟用或停用 IIOP 用戶端認證。此選項預設為停用。

其他特性

ORB 的其他特性。GlassFish Server 未定義 ORB 的其他特性。
