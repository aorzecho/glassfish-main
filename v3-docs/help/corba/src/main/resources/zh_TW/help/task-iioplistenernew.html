建立 IIOP 偵聽程式

  1. 在樹形元件中，展開 [配置] 節點。
  2. 選取要配置的實例：
    • 若要配置特定的實例，請選取該實例的配置節點。例如，對於預設實例 server，請選取 server-config 節點。
    • 若要配置使用 default-config 副本的未來實例的預設設定，請選取 default-config 節點。
  3. 在 [配置] 節點下，展開 [ORB] 節點。
  4. 選取 [IIOP 偵聽程式]。

    [IIOP 偵聽程式] 頁面會隨即開啟。

  5. 按一下 [新建]。
  6. 在 [名稱] 欄位中，鍵入識別偵聽程式的名稱。
  7. 在 [網路位址] 欄位中，鍵入偵聽程式的網路位址。

    您可以鍵入 IP 位址或 DNS 可解析主機名稱。

  8. 在 [偵聽程式連接埠] 欄位中，鍵入 IIOP 偵聽程式要偵聽的連接埠號碼。
  9. 選取 [啟用偵聽程式] 核取方塊可啟用通往 GlassFish Server 的 ORB IIOP 輸入連線。

    此選項預設為啟用。

  10. 選取 [啟用安全性] 核取方塊可為 IIOP 偵聽程式啟用 SSL。

    此選項預設為停用。

  11. (可選擇) 在 [其他特性] 區段中，指定其他特性。

    GlassFish Server 未定義 IIOP 偵聽程式的其他特性。

    若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。

  12. 按一下 [確定]。
  13. 若您選取了 [啟用安全性] 核取方塊，請依照為安全 IIOP 偵聽程式編輯 SSL 設定中的步驟操作。
