編輯 IIOP 偵聽程式

使用 [編輯 IIOP 偵聽程式] 頁面可修改 IIOP 偵聽程式的設定。

[編輯 IIOP 偵聽程式] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

名稱

可識別 IIOP 偵聽程式的唯一名稱。[名稱] 欄位是唯讀欄位。您僅能在建立 IIOP 偵聽程式時指定名稱。

網路位址

IIOP 偵聽程式的網路位址。網路位址可以是 IP 位址或 DNS 可解析主機名稱。

偵聽程式連接埠

IIOP 偵聽程式進行偵聽的連接埠號碼。

偵聽程式

偵聽程式的狀態。若選取了 [啟用] 核取方塊，會啟用通往 GlassFish Server 的 ORB IIOP 輸入連線。此選項預設為啟用。

Security

若選取了 [啟用] 核取方塊，會為 IIOP 偵聽程式啟用 SSL。此選項預設為停用。

其他特性

IIOP 偵聽程式的其他特性。GlassFish Server 未定義 IIOP 偵聽程式的其他特性。
