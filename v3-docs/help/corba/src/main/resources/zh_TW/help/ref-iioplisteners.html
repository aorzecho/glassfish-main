IIOP 偵聽程式

使用 [IIOP 偵聽程式] 頁面可配置 IIOP 偵聽程式。

可以針對 ORB 配置一些 IIOP 偵聽程式，其中的每個偵聽程式均接受不同網路位址和連接埠組合上的連線。您可以為安全通訊和用戶端認證進行不同的偵聽程式配置。

依預設，會配置三個 IIOP 偵聽程式：orb-listener-1、SSL 和 SSL_MUTUALAUTH。

對於每個 IIOP 偵聽程式，[IIOP 偵聽程式] 頁面均包含以下資訊。

名稱

IIOP 偵聽程式的名稱。

已啟用

IIOP 偵聽程式啟用時為 True，偵聽程式未啟用時為 False。

網路位址

IIOP 偵聽程式的網際網路協定 (IP) 位址或 DNS 可解析主機名稱。

偵聽程式連接埠

IIOP 偵聽程式的連接埠號碼。
