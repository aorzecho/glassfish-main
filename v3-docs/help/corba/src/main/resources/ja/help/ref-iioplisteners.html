IIOP リスナー

IIOP リスナーを設定するには、「IIOP リスナー」ページを使用します。

ORB に複数の IIOP リスナーを設定することで、ネットワークアドレスとポートの組み合わせが異なる接続を受け付けることができます。リスナーは、セキュリティー保護された通信とクライアント認証ごとに設定できます。

デフォルトでは、orb-listener-1、SSL、および SSL_MUTUALAUTH の 3 つの IIOP リスナーが設定されます。

「IIOP リスナー」ページには、IIOP リスナーごとに次の情報が含まれます。

名前

IIOP リスナーの名前。

使用可能

IIOP リスナーが有効な場合は true、リスナーが無効の場合は false です。

ネットワークアドレス

IIOP リスナーのインターネットプロトコル (IP) アドレス、または DNS で解決可能なホスト名。

リスナーポート

IIOP リスナーのポート番号。
