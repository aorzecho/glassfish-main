IIOPリスナー

IIOPリスナーを構成するには、「IIOPリスナー」ページを使用します。

ORBに複数のIIOPリスナーを構成することで、ネットワーク・アドレスとポートの組合せが異なる接続を受け入れることができます。リスナーは、セキュリティ保護された通信とクライアント認証ごとに構成できます。

デフォルトでは、orb-listener-1、SSL、およびSSL_MUTUALAUTHの3つのIIOPリスナーが構成されます。

「IIOPリスナー」ページには、IIOPリスナーごとに次の情報が含まれます。

名前

IIOPリスナーの名前。

有効

IIOPリスナーが有効な場合はtrue、リスナーが無効の場合はfalseです。

ネットワーク・アドレス

IIOPリスナーのインターネット・プロトコル(IP)アドレス、またはDNSで解決可能なホスト名。

リスナー・ポート

IIOPリスナーのポート番号。
