IIOP 수신기 편집

IIOP 수신기 편집 페이지에서는 IIOP 수신기에 대한 설정을 수정할 수 있습니다.

IIOP 수신기 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

IIOP 수신기를 식별하는 고유한 이름입니다. 이름 필드는 읽기 전용 필드입니다. IIOP 수신기를 만들 때만 이름을 지정할 수 있습니다.

네트워크 주소

IIOP 수신기의 네트워크 주소입니다. 네트워크 주소는 IP 주소 또는 DNS에서 확인 가능한 호스트 이름입니다.

수신기 포트

IIOP 수신기의 수신 위치에 해당하는 포트 번호입니다.

수신기

수신기의 상태입니다. 사용 가능 확인란이 선택되면 GlassFish Server에 대한 ORB IIOP 인바운드 연결이 활성화됩니다. 이 옵션은 기본적으로 활성화됩니다.

보안

사용 가능 확인란이 선택되면 IIOP 수신기에 대해 SSL이 활성화됩니다. 이 옵션은 기본적으로 비활성화됩니다.

추가 등록 정보

IIOP 수신기에 대한 추가 등록 정보입니다. GlassFish Server에서는 IIOP 수신기에 대한 추가 등록 정보를 정의하지 않습니다.
