ORB

ORB 페이지에서는 ORB(Object Request Broker)를 구성할 수 있습니다.

ORB는 객체를 식별하여 찾고 연결 관리를 처리하며 데이터를 전달하고 통신을 요청하는 데 필요한 인프라를 제공합니다. Enterprise Bean(EJB 모듈)의 원격 클라이언트는 ORB를 사용하여 GlassFish 서버와 통신합니다.

ORB 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 데 사용하는 버튼입니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

스레드 풀 ID

ORB에서 사용하는 스레드 풀입니다. ORB는 스레드 풀을 사용하여 RMI-IIOP를 사용하여 통신하는 다른 클라이언트와 Enterprise Bean의 원격 클라이언트에서 들어오는 요청에 응답합니다. ORB의 기본 스레드 풀은 thread-pool-1로서 이것은 RMI/IIOP 요청에 해당하는 기본값을 사용합니다.

최대 메시지 단편 크기

IIOP 메시지에 대한 최대 단편 크기입니다. 이 크기보다 큰 메시지는 단편화됩니다. 기본값은 1024바이트입니다.

총 연결 수

모든 IIOP 수신기에 들어오는 최대 연결 수입니다. 기본값은 1024입니다.

IIOP 클라이언트 인증

사용 가능 확인란이 선택되면 서버가 인증되지 않은 요청을 거부하고 클라이언트에게 전송된 IOR(Interoperable Object Reference)에 인증 요청 비트를 삽입합니다. IIOP 클라이언트 인증은 활성화 또는 비활성화할 수 있습니다. 이 옵션은 기본적으로 비활성화됩니다.

추가 등록 정보

ORB의 추가 등록 정보입니다. GlassFish 서버에서는 ORB에 추가 등록 정보를 정의하지 않습니다.
