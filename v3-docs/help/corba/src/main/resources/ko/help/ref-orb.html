ORB

ORB 페이지에서는 ORB(Object Request Broker)를 구성할 수 있습니다.

ORB는 객체를 식별하여 찾고 접속 관리를 처리하며 데이터를 전달하고 통신을 요청하는 데 필요한 기반 구조를 제공합니다. 엔터프라이즈 Bean(EJB 모듈)의 원격 클라이언트는 ORB를 사용하여 GlassFish Server과(와) 통신합니다.

ORB 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 단추입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

스레드 풀 ID

ORB에서 사용하는 스레드 풀입니다. ORB는 스레드 풀을 사용하여 RMI-IIOP를 사용하여 통신하는 다른 클라이언트와 엔터프라이즈 Bean의 원격 클라이언트에서 들어오는 요청에 응답합니다. ORB의 기본 스레드 풀은 thread-pool-1이며, 이 풀은 RMI/IIOP 요청에 해당하는 기본값을 사용합니다.

최대 메시지 단편 크기

IIOP 메시지에 대한 최대 단편 크기입니다. 이 크기보다 큰 메시지는 단편화됩니다. 기본값은 1024바이트입니다.

총 접속 수

모든 IIOP 리스너에 들어오는 최대 접속 수입니다. 기본값은 1024입니다.

IIOP 클라이언트 인증

[사용] 체크 박스를 선택하면 서버가 인증되지 않은 요청을 거부하고 클라이언트에게 전송된 IOR(Interoperable Object Reference)에 인증 요청 비트를 삽입합니다. IIOP 클라이언트 인증은 사용 또는 사용 안함으로 설정할 수 있습니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

추가 속성

ORB의 추가 속성입니다. GlassFish Server에서는 ORB에 대한 추가 속성을 정의하지 않습니다.
