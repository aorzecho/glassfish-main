IIOP 수신기

IIOP 수신기 페이지에서는 IIOP 수신기를 구성할 수 있습니다.

한 개의 ORB에 대해 각각 다른 네트워크 주소와 포트 조합에서 연결을 허용하는 많은 IIOP 수신기를 구성할 수 있습니다. 보안 통신과 클라이언트 인증을 위해 수신기를 다르게 구성할 수 있습니다.

기본적으로 세 개의 IIOP 수신기(orb-listener-1, SSL 및 SSL_MUTUALAUTH)가 구성됩니다.

IIOP 수신기 페이지에는 각 IIOP 수신기에 대한 다음 정보가 포함되어 있습니다.

이름

IIOP 수신기의 이름입니다.

사용 가능

IIOP 수신기를 사용할 수 있는 경우 True이고 수신기를 사용할 수 없는 경우 False입니다.

네트워크 주소

IIOP 수신기에 대한 IP(Internet Protocol) 주소 또는 DNS에서 확인 가능한 호스트 이름입니다.

수신기 포트

IIOP 수신기의 포트 번호입니다.
