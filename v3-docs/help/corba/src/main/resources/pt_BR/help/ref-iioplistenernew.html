Nova escuta de IIOP

Utilize a página Nova escuta de IIOP para criar uma nova escuta de IIP.

A página Nova escuta de IIOP contém as opções a seguir.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Nome

Um nome exclusivo que identifica a escuta de IIOP.

Endereço de Rede

O endereço de rede da escuta de IIOP. O endereço de rede pode ser um endereço IP ou um nome de host resolúvel de DNS.

Porta de escuta

O número da porta onde a escuta de IIOP irá escutar.

Escuta

O status da escuta. Se a caixa de seleção Ativo estiver marcada, as conexões de entrada de IIOP do ORB com o Servidor GlassFish são ativadas. A opção está ativada por padrão.

Segurança

Se a caixa de seleção Ativo estiver marcada, SSL é ativado para a escuta de IIOP. A opção está desativada por padrão.

Propriedades Adicionais

Propriedades adicionais para a escuta de IIOP. O Servidor GlassFish não define propriedades adicionais para escutas de IIOP.
