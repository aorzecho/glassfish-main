Para criar uma escuta de IIOP

  1. No componente da árvore, expanda o nó Configuração.
  2. Selecione a instância que será configurada:
    • Para configurar uma instância em particular, selecione o nó de configuração da instância. Por exemplo, na instância padrão, server, selecione o nó server-config.
    • Para configura as definições padrão das futuras instâncias que utilizam uam cópia de default-config, selecione o nó default-config.
  3. No nó Configuração, expanda o nó ORB.
  4. Selecione Escutas de IIOP.

    A página Escutas de IIOP é aberta.

  5. Clique em Novo.
  6. No campo Nome, digite um nome para identificar a escuta.
  7. No campo Endereço de Rede, digite o endereço de rede da escuta.

    Você pode digitar uma endereço IP ou um nome de host resolúvel de DNS.

  8. No campo Porta de escuta, digite o número da porta em que a escuta de IIOP será efetuada.
  9. Marque a caixa de seleção Escuta ativa para ativar conexões de entrada de IIOP do ORB com o Servidor GlassFish.

    A opção está ativada por padrão.

  10. Marque a caixa de seleção Segurança ativa para ativar SSL para a escuta de IIOP.

    A opção está desativada por padrão.

  11. (Opcional) Na seção Propriedades adicionais, especifique as propriedades.

    O Servidor GlassFish não define propriedades adicionais para escutas de IIOP.

    Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  12. Clique em OK.
  13. Se você marcou a caixa de seleção Segurança ativa, siga as etapas encontradas em Para editar configurações de SSL para uma escuta de IIOP segura.
