Escutas de IIOP

Utilize a página Escutas de IIOP para configurar escutas de IIOP.

Uma quantidade de escutas de IIOP pode ser configurada para um ORB, cada uma delas aceitando conexões em diferentes combinações de endereço e porta de rede. Escutas podem ser configuradas de maneiras diferentes para comunicações seguras e autenticação de clientes.

Por padrão, três escutas de IIOP são configuradas: orb-listener-1, SSL e SSL_MUTUALAUTH.

Para cada escuta de IIOP, a página Escutas de IIOP contém as informações a seguir.

Nome

O nome da escuta de IIOP.

Ativo

Verdadeiro se a escuta de IIOP estiver ativa ou falso se a escuta não estiver ativa.

Endereço de Rede

O endereço do Protocolo Internet (IP) ou o nome do resolúvel DNS para a escuta de IIOP.

Porta de escuta

O número da porta para a escuta de IIOP.
