Editar Listener de IIOP

Utilize a página Editar Listener IIOP para modificar as cdefinições de um listener IIOP.

A página Editar Listener IIOP contém as opções a seguir.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

Um nome exclusivo que identifica o listener IIOP. O campo Nome é somente para leitura. Você pode especificar o nome somente quando cria uma novo listener IIOP.

Endereço de Rede

O endereço de rede do listener IIOP. O endereço de rede pode ser um endereço IP ou um nome de host resolúvel de DNS.

Porta de Listener

O número da porta onde o listener IIOP fará listening.

Listener

O status do listener. Se a caixa de seleção Ativo estiver marcada, as conexões de entrada IIOP do ORB com o GlassFish Server são ativadas. Por default, a opção está ativada.

Segurança

Se a caixa de seleção Ativado estiver marcada, a SSL será ativada para o listener IIOP. Por default, essa opção está desativada.

Propriedades Adicionais

Propriedades adicionais para o listener IIOP. O GlassFish Server não define propriedades adicionais para listeners IIOP.
