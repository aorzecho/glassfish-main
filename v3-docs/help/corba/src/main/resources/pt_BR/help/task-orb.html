Para configurar o ORB

  1. No componente da árvore, expanda o nó Configuração.
  2. Selecione a instância que será configurada:
    • Para configurar uma instância em particular, selecione o nó de configuração da instância. Por exemplo, na instância padrão, server, selecione o nó server-config.
    • Para configura as definições padrão das futuras instâncias que utilizam uma cópia de default-config, selecione o nó default-config.
  3. Selecione o nó ORB.

    A página Segurança é aberta.

  4. Na lista suspensa ID de Grupo de segmentos, escolha o grupo de segmentos que o ORB utiliza.

    O ORB utiliza grupos de segmentos para responder a solicitações de clientes remotos de beans empresariais e outros clientes que se comunicam utilizando RMI-IIOP. O grupo de segmentos padrão do ORB é thread-pool-1, que utiliza padrões apropriados para solicitações RMI/IIOP.

    Consulte Grupos de Segmentos para obter mais informações.

  5. Na lista suspensa Tamanho máx. do fragmento de mensagem, selecione o tamanho máximo de fragmento para mensagens IIOP.

    As mensagens com tamanho maior do que o máximo permitido são fragmentadas. O valor padrão é 1024 bytes.

  6. No campo Total de conexões, defina o número máximo de conexões de entrada para todas as escutas de IIOP.

    O valor padrão é 1024.

  7. Marque a caixa de seleção Autenticação de cliente IIOP necessária para especificar que a autenticação de cliente IIOP é necessária.

    Se a autenticação de IIOP estiver ativa, o servidor rejeita solicitações não autenticadas e insere um bit de autenticação-necessária em Referências de Objeto Interoperáveis (IORs) enviadas para clientes. A opção está desativada por padrão.

  8. (Opcional) Na seção Propriedades Adicionais, especifique propriedades adicionais.

    O Servidor GlassFish não define propriedades adicionais para o ORB.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  9. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e, em seguida, em Salvar.

  10. Reinicie o Servidor GlassFish.
