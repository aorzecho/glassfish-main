Para Configurar o ORB

  1. No componente da árvore, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância em particular, selecione o nó de configuração da instância. Por exemplo, na instância default, server, selecione o nó server-config.

    • Para configura as definições default das futuras instâncias que utilizam uma cópia de default-config, selecione o nó default-config.

  3. Selecione o nó ORB.

    A página Segurança é aberta.

  4. Na lista drop-down ID do Pool de Threads, escolha o pool de threads que o ORB utiliza.

    O ORB utiliza pools de threads para responder a solicitações de clientes remotos de enterprise beans e outros clientes que se comunicam utilizando RMI-IIOP. O pool de thread default do ORB é thread-pool-1, que usa defaults adequados para solicitações de RMI/IIOP.

    Consulte Pools de Threads para obter mais informações.

  5. Na lista drop-down Tamanho máx. do fragmento de mensagem, selecione o tamanho máximo de fragmento para mensagens IIOP.

    As mensagens com tamanho maior do que o máximo permitido são fragmentadas. O valor default é 1024 bytes.

  6. No campo Total de conexões, defina o número máximo de conexões de entrada para todas as listeners IIOP.

    O valor default é 1024.

  7. Marque a caixa de seleção Autenticação de cliente IIOP necessária para especificar que a autenticação de cliente IIOP é necessária.

    Se a autenticação IIOP estiver ativa, o servidor rejeita solicitações não autenticadas e insere um bit de autenticação-necessária em Referências de Objeto Interoperáveis (IORs) enviadas para clientes. Por default, essa opção está desativada.

  8. Na seção Propriedades Adicionais, especifique propriedades adicionais.

    O GlassFish Server não define propriedades adicionais para o ORB.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  9. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Default e depois em Salvar.

  10. Reinicie o GlassFish Server.
