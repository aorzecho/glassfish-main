編輯連線集區特性

使用 「編輯連線集區特性」 頁面可修改現有 JDBC 連線集區的其他特性。

其他特性

JDBC 連線集區的其他特性。

如需部份可用特性的說明，請參閱JDBC 連線集區特定的特性。

若您未針對特性指定值，連線集區編輯完成時，會從表中移除該特性。若您不想立即為特性設定值，但希望將該特性保留在表中，待稍後為其設定值，請在 「值」 欄位中鍵入一組空的圓括號：

()
