編輯 JDBC 資源目標

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 JDBC 節點。
  3. 展開 [JDBC 資源] 節點。
  4. 選取要編輯的 JDBC 資源。
  5. 選取 [編輯 JDBC 資源] 頁面中的 [目標] 標籤。
  6. 若要將新目標指定給資源，請選取 [管理目標] 按鈕。
  7. 在 [管理目標] 頁面中，選取在 [選取的目標] 清單中要增加或移除的 [可用的目標]。
  8. 按一下 [確定] 以套用您的編輯。
