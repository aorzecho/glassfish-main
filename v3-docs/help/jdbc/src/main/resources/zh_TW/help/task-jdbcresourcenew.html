建立 JDBC 資源

Java 資料庫連結 (JDBC) 資源 (資料來源) 提供應用程式連線至資料庫的方法。

開始之前

您必須先建立 JDBC 連線池，才可建立 JDBC 資源。請參閱建立 JDBC 連線池。

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 在 [資源] 節點下，展開 [JDBC] 節點。
  3. 在 [JDBC] 節點下，選取 [JDBC 資源] 節點。

    [JDBC 資源] 頁面會隨即開啟。

  4. 在 [JDBC 資源] 頁面上，按一下 [新增]。

    [新 JDBC 資源] 頁面會隨即開啟。

  5. 在 [JNDI 名稱] 欄位中，鍵入唯一的資源名稱。

    JNDI 名稱可用來為分散式運算環境中的元件加以組織與定位，就如同圖書館裡的卡片目錄，可用來將書籍分門別類並表示擺放位置。因此，JNDI 名稱便成為存取 JDBC 資源的重要方法。依慣例，會是以 jdbc/ 字串開始的名稱，例如 jdbc/payrolldb。請勿遺漏正斜線。

    名稱必須僅包含字母數字、底線、破折號或點字符。

  6. 從 [池名稱] 下拉式清單中，選取要與新 JDBC 資源產生關聯的連線池。
  7. 在 [說明] 欄位中，鍵入新 JDBC 資源的簡短說明。
  8. 選取 [啟用狀態] 核取方塊能讓資源可在執行階段使用。

    此選項預設為啟用。

  9. (可選擇) 在 [其他特性] 區段中，指定其他特性。

    GlassFish Server 未定義 JDBC 資源的任何其他特性。

    若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。

  10. 選取在 [選取的目標] 清單中要增加或移除的 [可用的目標]。
  11. 按一下 [確定]。
