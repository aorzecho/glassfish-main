新建 JDBC 連線集區 (步驟 1，共 2 步)

使用 「新建 JDBC 連線集區 (步驟 1，共 2 步)」 頁面可建立新的 JDBC 連線集區。

「新建 JDBC 連線集區 (步驟 1，共 2 步)」 頁面包含下列選項。

集區名稱

連線集區的名稱。

資源類型

可用的資源類型包括 javax.sql.XADataSource (全域作業事件)、java.sql.ConnectionPoolDataSource (本機作業事件，可能可提昇效能)、javax.sql.DataSource (僅限於本機作業事件) 以及 java.sql.Driver。

資料庫驅動程式供應商

資料庫產品的供應商名稱。

自我檢查

若啟用此選項，資料庫驅動程式會依據資料來源或驅動程式實作類別名稱進行自我檢查。
