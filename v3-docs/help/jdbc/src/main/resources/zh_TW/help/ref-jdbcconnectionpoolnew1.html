新建 JDBC 連線池 (步驟 2 之 1)

使用 [新建 JDBC 連線池 (步驟 1，共 2 步)] 頁面可建立新的 JDBC 連線池。

[新建 JDBC 連線池 (步驟 1，共 2 步)] 頁面包含下列選項。

池名稱

連線池的名稱。

資源類型

可用的資源類型包括 javax.sql.XADataSource (全域作業事件)、java.sql.ConnectionPoolDataSource (本機作業事件，可能可提昇效能)、javax.sql.DataSource (僅限於本機作業事件) 以及 java.sql.Driver。

資料庫供應商

資料庫產品的供應商名稱。

相關作業

  • 設定資料庫存取

  • 整合 JDBC 驅動程式

  • 建立 JDBC 連線池

  • 編輯 JDBC 連線池一般設定

  • 驗證連線池設定

  • 編輯 JDBC 連線池進階屬性

  • 編輯 JDBC 連線池特性

  • 刪除 JDBC 連線池
