JDBC 資源

Java 資料庫連結 (JDBC) 資源 (資料來源) 提供應用程式連線至資料庫的方法。管理員一般會為每個資料庫 (由部署於網域的應用程式所存取) 建立一個 JDBC 資源，但是您可為一個資料庫建立多個 JDBC 資源。

應用程式會在 Java Naming and Directory Interface (JNDI) API 樹狀結構上查找資料來源並請求連線，以便從連線池取得資料庫連線。與資料來源相關聯的連線池可提供與應用程式的連線。

使用 [JDBC 資源] 頁面可配置 JDBC 資源。

對於每個資源，都將提供以下資訊。

JNDI 名稱

可識別 JDBC 資源的唯一名稱。

已啟用

如果資源已啟用，則為 True；如果資源已停用，則為 False。

連線池

與資源關聯的 JDBC 連線池。

說明

資源的簡短說明。

資源表也包含下列選項。

新建

建立新資源的按鈕。

刪除

刪除一或多個所選取資源的按鈕。

啟用

啟用一或多個所選取資源的按鈕。

停用

停用一或多個所選取資源的按鈕。
