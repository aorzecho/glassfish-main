驗證連線池設定

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 在 [資源] 節點下，展開 [JDBC] 節點。
  3. 在 [JDBC] 節點下，選取 [連線池] 節點。

    [連線池] 頁面會隨即開啟。

  4. 在 [連線池] 頁面上，按一下您要驗證的連線池名稱。

    [編輯連線池] 頁面會隨即開啟。

  5. 在 [編輯連線池] 頁面上，按一下 [Ping] 按鈕。

    管理主控台將嘗試連線至資料庫。如果顯示錯誤訊息，請檢查資料庫伺服器是否已啟動。
