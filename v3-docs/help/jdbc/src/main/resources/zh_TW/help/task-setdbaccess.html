設定資料庫存取

為了儲存、組織和擷取資料，大多數應用程式均使用關聯式資料庫。

GlassFish Server 包括一個 Java DB 資料庫，且已預先配置了存取此資料庫的功能。

  1. 安裝支援的資料庫產品。

    如需支援之資料庫的清單，請參閱「Oracle GlassFish Server 3.1 Release Notes」。

  2. 為資料庫安裝支援 JDBC API 的驅動程式 (JDBC 驅動程式)。

    請確保已在所有要部署資料來源的伺服器上，安裝了要用於建立資料庫連線的驅動程式。

  3. 請確保網域的伺服器實例可以存取該驅動程式的 JAR 檔案。

    請參閱整合 JDBC 驅動程式。

  4. 建立資料庫。

    通常，應用程式供應程式提供了用於建立和填入資料庫的程序檔。

  5. 為資料庫建立連線池。

    請參閱建立 JDBC 連線池。

  6. 建立指向連線池的 JDBC 資源。

    請參閱建立 JDBC 資源。
