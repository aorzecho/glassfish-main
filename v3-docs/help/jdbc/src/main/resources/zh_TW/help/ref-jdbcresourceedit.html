編輯 JDBC 資源

使用 [編輯 JDBC 資源] 頁面可修改現有的 JDBC 資源。

[編輯 JDBC 資源] 包含下列選項。

JNDI 名稱

可識別 JDBC 資源的唯一名稱。[JNDI 名稱] 是唯讀欄位。您僅能在建立新的 JDBC 資源時指定 JNDI 名稱。

池名稱

與 JDBC 資源相關聯的連線池。

說明

JDBC 資源的簡短說明。

狀態

若啟用此選項，則資源可在執行階段使用。此選項預設為啟用。

其他特性

JDBC 資源的其他特性。GlassFish Server 未定義 JDBC 資源的任何其他特性。
