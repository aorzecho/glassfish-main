Para Deletar um Pool de Conexões JDBC

Quando você deleta um pool de conexões JDBC, quaisquer recursos JDBC que dependem daquele pool de conexões também são deletados.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JDBC.

  3. No nó JDBC, selecione o nó Pools de Conexões.

    A página Pools de Conexões é exibida.

  4. Na página Pools de Conexões, marque a caixa de seleção à esquerda do pool de conexões a ser deletado.

  5. Clique em Deletar.
