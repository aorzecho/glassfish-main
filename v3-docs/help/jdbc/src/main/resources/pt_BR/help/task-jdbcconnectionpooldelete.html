Para excluir um conjunto de conexões do JDBC

Quando você exclui um conjunto de conexões JDBC, quaisquer recursos JDBC que dependem daquele conjunto de conexões também são excluídos.

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó JDBC.
  3. No nó JDBC, selecione o nó Conjuntos de conexões.

    A página Conjuntos de conexões é exibida.

  4. Na página Conjuntos de conexões, marque a caixa de seleção à esquerda do conjunto de conexões a ser excluído.
  5. Clique em Excluir.
