Para editar um recurso JDBC

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó JDBC.
  3. No nó JDBC, selecione o nó Recursos do JDBC.

    A página Recursos do JDBC é exibida.

  4. Na página Recursos do JDBC, clique no nome do recurso a ser editado.

    A página Editar recurso do JDBC é exibida.

  5. Na lista suspensa Nome do conjunto, selecione um conjunto de conexões a ser associado ao recurso do JDBC.
  6. No campo Descrição, digite uma descrição breve do recurso do JDBC.
  7. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    O Servidor GlassFish não define nenhuma propriedade adicional para os recursos JDBC.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em brando que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  8. Clique na guia Destino para gerenciar os destinos do recurso.
    1. Ative ou desative os destinos definidos para o recurso, conforme desejado.
    2. Clique no botão Gerenciar destinos para adicionar ou remover destinos do recurso.

    Consulte Para editar destinos do recurso do JDBC para obter mais informações.

  9. Clique na guia Geral para voltar à página Editar recurso do JDBC.
  10. Clique em Salvar.
