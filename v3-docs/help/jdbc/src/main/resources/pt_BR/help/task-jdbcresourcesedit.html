Para Editar um Recurso JDBC

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JDBC.

  3. No nó JDBC, selecione o nó Recursos JDBC.

    A página Recursos JDBC é exibida.

  4. Na página Recursos JDBC, clique no nome do recurso a ser editado.

    A página Editar Recurso JDBC é exibida.

  5. Na lista drop-down Nome do Pool, selecione um pool de conexões a ser associado ao recurso JDBC.

  6. No campo Descrição, digite uma descrição breve do recurso JDBC.

  7. Na seção Propriedades adicionais, especifique essas propriedades.

    O GlassFish Server não define nenhuma propriedade adicional para os recursos JDBC.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  8. Clique na guia Destino para gerenciar os destinos do recurso.

    1. Ative ou desative os destinos definidos para o recurso, conforme desejado.

    2. Clique no botão Gerenciar destinos para adicionar ou remover destinos do recurso.

    Consulte Para editar destinos do recurso JDBC para obter mais informações.

  9. Clique na guia Geral para voltar à página Editar recurso JDBC.

  10. Clique em Salvar.
