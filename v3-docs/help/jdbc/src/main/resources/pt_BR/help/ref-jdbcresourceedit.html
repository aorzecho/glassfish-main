Editar recurso do JDBC

Use a página Editar recurso do JDBC para modificar um recurso existente do JDBC.

A página Editar recurso do JDBC contém estas opções.

Nome JNDI

Um nome único que identifica o recurso JDBC. O nome JNDI é um campo somente leitura. Você somente pode especificar o nome JNDI ao criar um novo recurso JDBC.

Nome do conjunto

O conjunto de conexões associado ao recurso do JDBC.

Descrição

Uma breve descrição do recurso do JDBC.

Status

Se essa opção for ativada, o recurso estará disponível no momento da execução. Por padrão, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso JDBC. O Servidor GlassFish não define nenhuma propriedade adicional para os recursos JDBC.
