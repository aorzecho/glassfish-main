Editar Recurso do JBDC

Use a página Editar Recurso JDBC para modificar um recurso existente JDBC.

A página Editar Recurso JDBC contém estas opções.

Nome JNDI

Um nome único que identifica o recurso JDBC. O nome JNDI é um campo somente para leitura. Você somente pode especificar o nome JNDI ao criar um novo recurso JDBC.

Nome do conjunto

O pool de conexões associado ao recurso JDBC.

Descrição

Uma breve descrição do recurso JDBC.

Status

Se essa opção for ativada, o recurso estará disponível no momento da execução. Por default, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso JDBC. O GlassFish Server não define nenhuma propriedade adicional para os recursos JDBC.
