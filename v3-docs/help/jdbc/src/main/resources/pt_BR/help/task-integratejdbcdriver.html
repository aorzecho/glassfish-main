Para integrar um driver JDBC

Um driver compatível com a API JDBC (driver JDBC) converte as chamadas do JDBC de um aplicativo no protocolo do servidor de banco de dados.

  • Para integrar o driver JDBC a um domínio do Servidor GlassFish, execute qualquer um desses procedimentos:
    • Torne o driver acessível ao carregador de classe comum.
      1. Copie os arquivos JAR e ZIP do driver no diretório domain-dir /lib.

        Observação - Há uma exceção a essa regra. Se você estiver usando o driver JDBC da Oracle e deseja tirar vantagem das extensões EclipseLink que estão disponíveis para uso com bancos de dados da Oracle, é preciso colocar os arquivos JAR para o driver JDBC para o JDBC da Oracle JDBC no diretório domain-dir/lib/ext.

      2. Reinicie o domínio.
    • Torne o driver acessível ao carregador de classe do sistema.
      1. Na árvore de navegação, expanda o nó Configuração.
      2. No nó Configuração, selecione o nó Configurações de JVM.

        A página Configurações Gerais de JVM é exibida.

      3. Na página Configurações Gerais de JVM, clique na guia Configurações de Caminho.
      4. No campo Sufixo de Caminho de Classe, digite o nome de caminho totalmente qualificado para o arquivo JAR do driver.
      5. Clique em Salvar.
      6. Reinicie o servidor.
