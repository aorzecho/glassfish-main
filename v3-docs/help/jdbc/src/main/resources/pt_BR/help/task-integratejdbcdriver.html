Para Integrar um Driver JDBC

Um driver que suporte a API da JDBC (driver JDBC) converte as chamadas JDBC de uma aplicação no protocolo do servidor de bancos de dados.

Para integrar o driver JDBC a um domínio do GlassFish Server, execute qualquer um desses procedimentos:

  • Torne o driver acessível ao carregador de classe comum.

    1. Copie os arquivos JAR e ZIP do driver para o diretório domain-dir/lib.

      Observação:

      Há uma exceção a essa regra. Se você estiver usando o driver Oracle JDBC e quiser tirar proveito das extensões EclipseLink que estão disponíveis para uso com bancos de dados Oracle, você deverá colocar os arquivos JAR do driver Oracle JDBC no diretório domain-dir/lib/ext.

    2. Reinicie o domínio.

  • Torne o driver acessível ao carregador de classe do sistema.

    1. Na árvore de navegação, expanda o nó Configuração.

    2. No nó Configuração, selecione o nó Configurações de JVM.

      A página Definições Gerais de JVM é exibida.

    3. Na página Definições Gerais de JVM, clique na guia Definições de Caminho.

    4. No campo Sufixo de Caminho de Classe, digite o nome de caminho totalmente qualificado para o arquivo JAR do driver.

    5. Clique em Salvar.

    6. Reinicie o servidor.
