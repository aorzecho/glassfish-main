Para editar destinos do recurso JDBC

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó JDBC.

  3. Expanda o nó Recursos JDBC.

  4. Selecione o recurso JDBC que deseja editar.

  5. Na página Editar recurso JDBC, selecione a guia Destinos.

  6. Para designar um novo destino ao recurso, selecione o botão Gerenciar Destinos.

  7. Na página Gerenciar destino, selecione um destino disponível para adicioná-lo ou removê-lo da lista Destinos selecionados.

  8. Clique em OK para aplicar as edições.
