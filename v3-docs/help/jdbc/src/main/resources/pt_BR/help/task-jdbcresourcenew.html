Para criar um recurso do JDBC

Um recurso (fonte de dados) do Java DataBase Connectivity (JDBC) fornece aplicativos com os meios de conexão a um banco de dados.

Antes de começar

Para criar um recurso do JDBC, crie primeiro um conjunto de conexões do JDBC. Consulte Para criar um conjunto de conexões do JDBC.

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó JDBC.
  3. No nó JDBC, selecione o nó Recursos do JDBC.

    A página Recursos do JDBC é exibida.

  4. Na página Recursos do JDBC, clique em Novo.

    A página Novo Recurso do JDBC é exibida.

  5. No campo Nome JNDI, digite um nome de recurso exclusivo.

    O nome JNDI organiza e localiza os componentes em um ambiente de computação distribuído, de maneira semelhante ao catálogo de cartão que organiza e representa os locais de livros em uma biblioteca. Consequentemente, o nome JNDI se torna um método importante de acesso ao recurso JDBC. Por convenção, o nome começa com a sequência jdbc/; por exemplo, jdbc/payrolldb. Não se esqueça da barra.

    O nome deve conter somente caracteres alfanuméricos, sublinhado, traço ou ponto.

  6. Na lista suspensa Nome do conjunto, selecione um conjunto de conexões a ser associado ao novo recurso do JDBC.
  7. No campo Descrição, digite uma descrição breve do novo recurso do JDBC.
  8. Selecione a caixa de seleção Status ativo para tornar o recurso disponível no tempo de execução.

    Por padrão, a opção está ativada.

  9. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    O Servidor GlassFish não define nenhuma propriedade adicional para os recursos JDBC.

    Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  10. Selecione um destino disponível para adicioná-lo ou removê-lo da lista Destinos selecionados.
  11. Clique em OK.
