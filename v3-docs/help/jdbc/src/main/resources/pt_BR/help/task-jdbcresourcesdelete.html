Para excluir um recurso do JDBC

Um recurso JDBC é automaticamente excluído se você exclui o conjunto de conexões JDBC associado ao recurso.

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó JDBC.
  3. No nó JDBC, selecione o nó Recursos do JDBC.

    A página Recursos do JDBC é exibida.

  4. Na página Recursos do JDBC, marque a caixa de seleção à esquerda do recurso a ser excluído.
  5. Clique em Excluir.
