Para Deletar um Recurso JDBC

Um recurso JDBC será automaticamente deletado se você deletar o pool de conexões JDBC associado ao recurso.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JDBC.

  3. No nó JDBC, selecione o nó Recursos JDBC.

    A página Recursos JDBC é exibida.

  4. Na página Recursos JDBC, marque a caixa de seleção à esquerda do recurso a ser deletado.

  5. Clique em Deletar.
