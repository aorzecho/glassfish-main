A tecnologia Java DataBase Connectivity (JDBC) fornece conectividade para bancos de dados relacionais. No Servidor GlassFish, para configurar a conectividade de banco de dados, adicione os recursos do JDBC (fontes de dados) ao domínio. Uma fonte de dados é um método padrão da plataforma Java, Enterprise Edition (plataforma Java EE) para configurar conectividade com um banco de dados. Cada fonte de dados contém um conjunto de conexões de bancos de dados. Os aplicativos pesquisam a origem de dados na árvore da API Java Naming and Directory Interface (JNDI) ou no contexto do aplicativo local e a seguir recuperam a conexão de um banco de dados com o método getConnection. As fontes de dados e seus respectivos conjuntos de conexões fornecem processos de gerenciamento de conexão, os quais ajudam a manter o sistema em execução.

O Servidor GlassFish inclui uma cópia do banco de dados Java DB e é pré-configurado com a habilidade de acessar esse banco de dados.
