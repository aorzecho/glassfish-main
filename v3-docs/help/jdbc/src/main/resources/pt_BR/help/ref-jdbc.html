A tecnologia Java DataBase Connectivity (JDBC) oferece conectividade aos bancos de dados relacionais. No GlassFish Server, para configurar a conectividade de banco de dados, adicione os recursos JDBC (fontes de dados) ao domínio. Uma origem de dados é um método padrão da plataforma Java, Enterprise Edition (plataforma Java EE) para configurar conectividade com um banco de dados. Cada fonte de dados contém um pool de conexões de bancos de dados. As aplicações pesquisam a origem de dados na árvore API do Java Naming and Directory Interface (JNDI) ou no contexto da aplicação local e recuperam uma conexão do banco de dados com o métodogetConnection. As origens de dados e seus respectivos pools de conexões oferecem processos de gerenciamento de conexões que ajudam a manter o sistema em execução.

O GlassFish Server inclui uma cópia do banco de dados Java DB e é pré-configurado com a habilidade de acessar esse banco de dados.
