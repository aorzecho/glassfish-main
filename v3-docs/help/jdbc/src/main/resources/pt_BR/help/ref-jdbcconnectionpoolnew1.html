Novo Pool de Conexões JDBC (Etapa 1 de 2)

Use a página Novo Pool de Conexões JDBC (Etapa 1 de 2) para criar um novo pool de conexões.

A página Novo Pool de Conexões JDBC (Etapa 1 de 2) contém as opções a seguir.

Nome do Pool

O nome do pool de conexões.

Tipo de recurso

Os tipos de recursos disponíveis são javax.sql.XADataSource (transações globais), java.sql.ConnectionPoolDataSource (transações locais, possíveis melhorias de desempenho) e javax.sql.DataSource (somente transações locais) e java.sql.Driver.

Fornecedor do Driver do Banco de Dados

O nome do fornecedor do produto de banco de dados.

Introspecção

Se essa opção estiver ativada, o driver do banco de dados é submetido a introspecção com base na origem de dadou ou nomes da classe de implantação do driver.
