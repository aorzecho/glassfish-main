Novo conjunto de conexões do JDBC (Etapa 1 de 2)

Use a página Novo conjunto de conexões do JDBC (Etapa 1 de 2) para criar um novo conjunto.

A página Novo conjunto de conexões do JDBC (Etapa 1 de 2) contém as opções a seguir.

Nome do conjunto

O nome do conjunto de conexões.

Tipo de recurso

Os tipos de recursos disponíveis são javax.sql.XADataSource (transações globais), java.sql.ConnectionPoolDataSource (transações locais, possíveis melhorias de desempenho) e javax.sql.DataSource (somente transações locais) e java.sql.Driver.

Fornecedor do banco de dados

O nome do fornecedor do produto de banco de dados.
