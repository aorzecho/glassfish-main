Recursos do JDBC

Um recurso (fonte de dados) do Java DataBase Connectivity (JDBC) fornece aplicativos com os meios de conexão a um banco de dados. Em geral, o administrador cria um recurso do JDBC para cada banco de dados acessado pelos aplicativos implantados em um domínio; entretanto, é possível criar mais de um recurso do JDBC para um banco de dados.

Os aplicativos obtêm uma conexão de banco de dados no conjunto de conexões, pesquisando uma origem de dados na árvore da API Java Naming and Directory Interface (J.N.D.I.) e solicitando uma conexão. O conjunto de conexões associado à fonte de dados fornece a conexão ao aplicativo.

Use a página Recursos JDBC para configurar os recursos JDBC.

Para cada recurso, as seguintes informações são fornecidas.

Nome JNDI

Um nome único que identifica o recurso JDBC.

Ativado

Verdadeiro se o recurso estiver ativado ou falso se o recurso estiver desativado.

Conjunto de Conexões

O conjunto de conexões do JDBC associado ao recurso.

Descrição

Uma breve descrição do recurso.

A tabela Recursos também contém as seguintes opções.

Novo

Botão utilizado para criar um novo recurso.

Excluir

Botão utilizado para excluir um ou mais recursos selecionados.

Ativar

Botão utilizado para ativar um ou mais recursos selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos selecionados.
