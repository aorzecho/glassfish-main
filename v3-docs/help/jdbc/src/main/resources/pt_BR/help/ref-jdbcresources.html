Recursos do JDBC

Um recurso de Java DataBase Connectivity (origem de dados) fornece aplicações com todos os meios para conectar-se a um banco de dados. Em geral, o administrador cria um recurso JDBC para cada banco de dados acessado pelas aplicações implantadas em um domínio; entretanto, é possível criar mais de um recurso JDBC para um banco de dados.

As aplicações obtêm uma conexão do banco de dados de um pool de conexões, pesquisando por uma origem de dados na árvore API de Java Naming and Directory Interface (JNDI) e solicitando uma conexão. O pool de conexões associado à origem de dados provê a conexão com a aplicação.

Use a página Recursos JDBC para configurar os recursos JDBC.

Para cada recurso, as seguintes informações são fornecidas.

Nome JNDI

Um nome único que identifica o recurso JDBC.

Ativado

Verdadeiro se o recurso estiver ativado ou falso se o recurso estiver desativado.

Pool de Conexões

O pool de conexões JDBC associado ao recurso.

Descrição

Uma breve descrição do recurso.

A tabela Recursos também contém as seguintes opções.

Novo

Botão utilizado para criar um novo recurso.

Deletar

Botão utilizado para deletar um ou mais recursos selecionados.

Ativar

Botão utilizado para ativar um ou mais recursos selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos selecionados.
