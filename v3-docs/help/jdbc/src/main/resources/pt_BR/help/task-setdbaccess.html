Para configurar o acesso ao banco de dados

Para armazenar, organizar e recuperar dados, a maioria dos aplicativos usa bancos de dados relacionais.

O Servidor GlassFish inclui uma cópia do banco de dados Java DB e é pré-configurado com a habilidade de acessar esse banco de dados.

  1. Instale um produto de banco de dados compatível.

    Para obter uma lista dos bancos de dados que possuem suporte, consulte Oracle GlassFish Server 3.1 Release Notes .

  2. Instale um driver que suporte a API JDBC (driver JDBC) para o banco de dados.

    Verifique se o driver a ser utilizado para criar as conexões com bancos de dados está instalado em todos os servidores em que a fonte de dados será instalada.

  3. Verifique se o arquivo JAR do driver é acessado na instância de servidor do domínio.

    Consulte Para integrar um driver JDBC.

  4. Crie o banco de dados.

    Em geral, o fornecedor do aplicativo disponibiliza scripts para criação e preenchimento do banco de dados.

  5. Crie um conjunto de conexões para o banco de dados.

    Consulte Para criar um conjunto de conexões do JDBC.

  6. Crie um recurso do JDBC que aponte para o conjunto de conexões.

    Consulte Para criar um recurso do JDBC.
