So bearbeiten Sie Ziele für JDBC-Ressourcen

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "JDBC".
  3. Erweitern Sie den Knoten "JDBC-Ressourcen".
  4. Wählen Sie die JDBC-Ressource, die Sie bearbeiten möchten.
  5. Wählen Sie auf der Seite "JDBC-Ressource bearbeiten" die Registerkarte "Ziele".
  6. Um der Ressource ein neues Ziel zuzuweisen, klicken Sie auf die Schaltfläche "Ziele verwalten".
  7. Wählen Sie auf der Seite "Ziele verwalten" ein verfügbares Ziel, um es hinzuzufügen oder aus der Liste "Ausgewählte Ziele" zu entfernen.
  8. Klicken Sie auf "OK", um die Änderungen zu übernehmen.
