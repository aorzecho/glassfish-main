So wird eine JDBC-Ressource gelöscht

Eine JDBC-Ressource wird automatisch gelöscht, wenn Sie den mit der Ressource verknüpften JDBC-Connection Pool löschen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JDBC-Knoten ein.

  3. Wählen Sie unter dem JDBC-Knoten den JDBC-Ressourcenknoten.

    Die Seite "JDBC-Ressourcen" wird geöffnet.

  4. Aktivieren Sie auf der Seite "JDBC-Ressourcen" das Kontrollkästchen links neben der zu löschenden Ressource.

  5. Klicken Sie auf "Löschen".
