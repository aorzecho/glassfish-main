So löschen Sie eine JDBC-Ressource

Eine JDBC-Ressource wird automatisch gelöscht, wenn Sie den mit der Ressource verknüpften JDBC-Verbindungspool löschen.

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie unter dem Ressourcenknoten den JDBC-Knoten.
  3. Wählen Sie unter dem JDBC-Knoten den JDBC-Ressourcenknoten aus.

    Die Seite "JDBC-Ressourcen" wird geöffnet.

  4. Aktivieren Sie auf der Seite "JDBC-Ressourcen" das Kontrollkästchen links neben der zu löschenden Ressource.
  5. Klicken Sie auf "Löschen".
