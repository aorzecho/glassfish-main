Neuer JDBC-Verbindungspool (Schritt 1 von 2)

Sie verwenden die Seite New JDBC Connection Pool (Step 1 of 2) Neuer JDBC-Verbindungspool (Schritt 1 von 2) zum Erstellen eines neuen JDBC-Verbindungspools.

Die Seite New JDBC Connection Pool (Step 1 of 2) Neuer JDBC-Verbindungspool (Schritt 1 von 2) enthält folgende Optionen.

Pool Name (Poolname)

Der Name des Verbindungspools.

Resource Type (Ressourcen-Typ)

Verfügbare Ressourcentypen sind javax.sql.XADataSource (globale Transaktionen), java.sql.ConnectionPoolDataSource (lokale Transaktionen, mögliche Leistungsverbesserungen), javax.sql.DataSource (nur lokale Transaktionen) und java.sql.Driver.

Database Vendor (Datenbankhersteller)

Der Name des Herstellers Ihres Datenbankproduktes.

Verwandte Aufgaben

  • So richten Sie den Datenbankzugriff ein

  • So integrieren Sie einen JDBC-Treiber

  • So erstellen Sie einen JDBC-Verbindungspool

  • So bearbeiten Sie die allgemeinen JDBC-Verbindungspooleinstellungen

  • So überprüfen Sie die Verbindungspooleinstellungen

  • So bearbeiten Sie die erweiterten Attribute eines JDBC-Verbindungspools

  • So bearbeiten Sie Eigenschaften von JDBC-Verbindungspools

  • So löschen Sie einen JDBC-Verbindungspool
