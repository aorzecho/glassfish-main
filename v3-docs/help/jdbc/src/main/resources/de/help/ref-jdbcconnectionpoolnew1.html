Neuer JDBC-Connection Pool (Schritt 1 von 2)

Sie verwenden die Seite "Neuer JDBC-Connection Pool (Schritt 1 von 2)" zum Erstellen eines neuen JDBC-Connection Pools.

Die Seite "Neuer JDBC-Connection Pool (Schritt 1 von 2)" enthält folgende Optionen.

Pool Name (Poolname)

Der Name des Connection Pools.

Resource Type (Ressourcen-Typ)

Verfügbare Ressourcentypen sind javax.sql.XADataSource (globale Transaktionen), java.sql.ConnectionPoolDataSource (lokale Transaktionen, mögliche Performanceverbesserungen), javax.sql.DataSource (nur lokale Transaktionen) und java.sql.Driver.

Datenbanktreiberhersteller

Der Name des Herstellers Ihres Datenbankproduktes.

Introspektieren

Wenn diese Option aktiviert wird, wird der Datenbanktreiber basierend auf dem Namen der Datenquelle oder der Treiberimplementierungsklasse introspektiert.

Verwandte Aufgaben

  • So wird der Datenbankzugriff eingerichtet

  • So wird ein JDBC-Treiber integriert

  • So wird ein JDBC-Connection Pool erstellt

  • So werden allgemeine Eigenschaften eines JDBC-Connection Pools bearbeitet

  • So werden Connection Pool-Einstellungen geprüft

  • So werden erweiterte Attribute des JDBC-Connection Pools bearbeitet

  • So werden JDBC-Connection Pool-Eigenschaften bearbeitet

  • So wird ein JDBC-Connection Pool gelöscht
