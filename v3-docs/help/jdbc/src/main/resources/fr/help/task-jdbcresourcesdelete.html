Supprimer une ressource JDBC

Une ressource JDBC est automatiquement supprimée si vous supprimez le pool de connexions JDBC associé à la ressource.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, développez le nœud JDBC.
  3. Sous le nœud JDBC, sélectionnez le nœud Ressources JDBC.

    La page Ressources JDBC s’affiche.

  4. Dans la page Ressources JDBC, cochez la case située à gauche de la ressource à supprimer.
  5. Cliquez sur Supprimer.
