Créer une ressource JDBC

Une ressource JDBC (Java DataBase Connectivity) (source de données) fournit des applications via une connexion à une base de données.

Avant de commencer

Pour créer une ressource JDBC, vous devez commencer par créer un pool de connexions JDBC. Voir Créer un pool de connexions JDBC.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, développez le nœud JDBC.
  3. Sous le nœud JDBC, sélectionnez le nœud Ressources JDBC.

    La page Ressources JDBC s’affiche.

  4. Dans la page Ressources JDBC, cliquez sur Nouveau.

    La page Nouvelle ressource JDBC s’ouvre.

  5. Dans le champ Nom JNDI, saisissez un nom de ressource unique.

    Le nom JNDI organise et recherche des composants dans un environnement informatique distribué de la même manière que des catalogues sur fiches organisent et représentent des emplacements de livres dans une bibliothèque. Par conséquent, le nom JNDI devient une méthode importante d’accès à la ressource JDBC. Par convention, le nom commence par la chaîne jdbc/, comme par exemple dans jdbc/payrolldb. N’oubliez pas la barre oblique.

    Le nom ne peut contenir que des caractères alphanumériques, des caractères de soulignement, des tirets ou des points.

  6. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la nouvelle ressource JDBC.
  7. Dans le champ Description, entrez une brève description de la nouvelle ressource JDBC.
  8. Cochez la case État activé pour rendre la ressource disponible au moment de l’exécution.

    Cette option est activée par défaut.

  9. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    GlassFish Server ne définit aucune autre propriété pour les ressources JDBC.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  10. Sélectionnez une cible disponible pour l'ajouter ou la supprimer de la liste Cibles sélectionnées.
  11. Cliquez sur OK.
