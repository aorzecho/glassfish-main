JDBC

La page JDBC permet de configurer les types de ressource suivants :

  • Ressources JDBC

  • Pools de connexions

La technologie JDBC (Java DataBase Connectivity) assure la connectivité des bases de données relationnelles. Dans GlassFish Server, vous pouvez configurer la connectivité des bases de données en ajoutant des ressources JDBC (sources de données) dans le domaine. Une source de données est une méthode standard Java Platform, Enterprise Edition (plate-forme Java EE) de configuration de la connectivité à une base de données. Chaque source de données contient un pool de connexions à une base de données. Les applications examinent la source de données dans l'arborescence de l'API JNDI (Java Naming and Directory Interface) ou dans le contexte de l'application locale, puis extraient une connexion de base de données avec la méthode getConnection. Les sources de données et leurs pools de connexions incluent des processus de gestion de connexions qui permettent de garantir le fonctionnement continu de votre système.

GlassFish Server inclut une copie de la base de données Java DB et est préconfiguré pour accéder à cette base de données.
