Pour modifier les cibles de ressources JDBC

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le nœud JDBC.
  3. Développez le nœud Ressources JDBC.
  4. Sélectionnez la ressource JDBC à modifier.
  5. À partir de la page Modification d'une ressource JDBC, sélectionnez l'onglet Cibles.
  6. Pour affecter une nouvelle cible à la ressource, sélectionnez le bouton Gestion des cibles.
  7. Sur la page Gestion des cibles, sélectionnez une cible disponible pour l'ajouter ou la supprimer de la liste Cibles sélectionnées.
  8. Cliquez sur OK pour appliquer vos modifications.
