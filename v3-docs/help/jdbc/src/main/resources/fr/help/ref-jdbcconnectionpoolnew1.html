Nouveau pool de connexions JDBC (étape 1 sur 2)

La page Nouveau pool de connexions JDBC (étape 1 sur 2) vous permet de créer un nouveau pool de connexions JDBC.

La page Nouveau pool de connexions JDBC (étape 1 sur 2) contient les options suivantes :

Nom du pool

Nom du pool de connexions.

Type de ressource

Les types de ressources disponibles incluent : javax.sql.XADataSource (transactions globales), java.sql.ConnectionPoolDataSource (transactions locales, améliorations possibles des performances), javax.sql.DataSource (transactions locales uniquement) et java.sql.Driver.

Fournisseur de base de données

Nom du fournisseur de votre base de données.
