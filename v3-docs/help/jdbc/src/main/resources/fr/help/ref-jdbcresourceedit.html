Modifier une ressource JDBC

La page Modifier une ressource JDBC permet de modifier une ressource JDBC existante.

Elle contient les options suivantes.

Nom JNDI

Nom unique identifiant la ressource JDBC. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource JDBC.

Nom du pool

Pool de connexions associé à la ressource JDBC.

Description

Brève description de la ressource JDBC.

Statut

Si cette option est activée, la ressource est disponible lors de l'exécution. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource JDBC. GlassFish Server ne définit aucune autre propriété pour les ressources JDBC.
