Supprimer un pool de connexions JDBC

Lorsque vous supprimez un pool de connexions JDBC, toutes les ressources JDBC qui dépendent de ce pool de connexions sont également supprimées.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, développez le nœud JDBC.
  3. Sous le nœud JDBC, sélectionnez le nœud Pools de connexions.

    La page Pools de connexions s’ouvre.

  4. Dans la page Pools de connexions, cochez la case située à gauche du pool de connexions à supprimer.
  5. Cliquez sur Delete.
