Modifier une ressource JDBC

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JDBC.

  3. Sous le noeud JDBC, sélectionnez le noeud Ressources JDBC.

    La page Ressources JDBC apparaît.

  4. Sur la page Ressources JDBC, cliquez sur le nom de la ressource à modifier.

    La page Modifier une ressource JDBC apparaît.

  5. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la ressource JDBC.

  6. Dans le champ Description, entrez une brève description de la ressource JDBC.

  7. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server ne définit aucune autre propriété pour les ressources JDBC.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  8. Cliquez sur l'onglet Cible pour gérer les cibles de ressources.

    1. Activez ou désactivez les cibles définies pour la ressource, comme vous le souhaitez.

    2. Cliquez sur le bouton Gérer les cibles pour ajouter des cibles à la ressource ou en enlever.

    Pour plus d'informations, reportez-vous à Modifier les cibles de ressources JDBC.

  9. Cliquez sur l'onglet Général pour revenir à la page Modifier une ressource JDBC.

  10. Cliquez sur Enregistrer.
