Modifier une ressource JDBC

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, développez le nœud JDBC.
  3. Sous le nœud JDBC, sélectionnez le nœud Ressources JDBC.

    La page Ressources JDBC s’affiche.

  4. Sur la page Ressources JDBC, cliquez sur le nom de la ressource à modifier.

    La page Modification d’une ressource JDBC s’ouvre.

  5. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la ressource JDBC.
  6. Dans le champ Description, entrez une brève description de la ressource JDBC.
  7. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    GlassFish Server ne définit aucune autre propriété pour les ressources JDBC.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  8. Cliquez sur l'onglet Cible pour gérer les cibles de ressources.
    1. Activez ou désactivez les cibles définies pour la ressource, comme vous le souhaitez.
    2. Cliquez sur le bouton Gestion des cibles pour ajouter ou supprimer des cibles de la ressource.

    Voir Pour modifier les cibles de ressources JDBC pour plus d'informations.

  9. Cliquez sur l'onglet Général pour revenir à la page Modification d'une ressource JDBC.
  10. Cliquez sur Enregistrer.
