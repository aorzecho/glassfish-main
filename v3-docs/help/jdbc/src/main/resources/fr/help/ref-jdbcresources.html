Ressources JDBC

Une ressource JDBC (Java DataBase Connectivity) (source de données) fournit des applications via une connexion à une base de données. En général, l'administrateur crée une ressource JDBC pour chaque base de données consultée par les applications déployées dans un domaine. Plusieurs ressources JDBC peuvent cependant être créées pour une même base de données.

Les applications obtiennent une connexion à la base de données à partir d'un pool de connexions en recherchant une source de données dans l'arborescence de l'API JNDI (Java Naming and Directory Interface), puis en demandant une connexion. Le pool de connexions associé à la source de données fournit la connexion à l'application.

La page Ressources JDBC permet de configurer les ressources JDBC.

Pour chaque ressource, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource JDBC.

Activé

True si la ressource est activée ou False si la ressource est désactivée.

Pool de connexions

Pool de connexions JDBC associé à la ressource.

Description

Brève description de la ressource.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource.

Supprimer

Bouton permettant de supprimer les ressources sélectionnées.

Activer

Bouton permettant d'activer les ressources sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources sélectionnées.
