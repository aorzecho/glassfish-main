JDBCリソースを編集する

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「JDBC」ノードを展開します。

  3. 「JDBC」ノードで、「JDBCリソース」ノードを選択します。

    「JDBCリソース」ページが開きます。

  4. 「JDBCリソース」ページで、編集するリソースの名前をクリックします。

    「JDBCリソースを編集」ページが開きます。

  5. 「プール名」ドロップダウン・リストから、このJDBCリソースに関連付ける接続プールを選択します。

  6. 「説明」フィールドで、このJDBCリソースの簡単な説明を入力します。

  7. 「追加プロパティ」セクションで、追加プロパティを指定します。

    GlassFish ServerはJDBCリソースの追加プロパティを定義しません。

    • プロパティを追加するには、「プロパティを追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティ名を入力し、「値」フィールドにプロパティ値を入力します。

    • プロパティを変更するには、そのプロパティの「値」フィールドを編集します。

    • プロパティを削除するには、削除するプロパティの「名前」フィールドの左側にあるチェックボックスを選択し、「プロパティを削除」ボタンをクリックします。

  8. 「ターゲット」タブをクリックして「リソース・ターゲット」を管理します。

    1. 必要に応じて、リソースの定義済ターゲットを有効または無効にします。

    2. 「ターゲットの管理」ボタンをクリックして、リソースのターゲットを追加または削除します。

    詳細は、JDBCリソースのターゲットを編集するを参照してください。

  9. 「一般」タブをクリックして、「JDBCリソースを編集」ページに戻ります。

  10. 「保存」をクリックします。
