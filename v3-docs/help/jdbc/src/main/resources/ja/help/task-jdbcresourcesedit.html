JDBC リソースを編集する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JDBC」ノードを展開します。
  3. 「JDBC」ノードで、「JDBC リソース」ノードを選択します。

    「JDBC リソース」ページが開きます。

  4. 「JDBC リソース」ページで、編集するリソースの名前をクリックします。

    「JDBC リソースを編集」ページが開きます。

  5. 「プール名」ドロップダウンリストから、この JDBC リソースに関連付ける接続プールを選択します。
  6. 「説明」フィールドで、この JDBC リソースの簡単な説明を入力します。
  7. (省略可能) 「追加プロパティー」セクションで、追加プロパティーを指定します。

    GlassFish Server は JDBC リソースの追加プロパティーを定義しません。

    • プロパティーを追加するには、「プロパティーを追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティー名を入力し、「値」フィールドにプロパティー値を入力します。
    • プロパティーを変更するには、そのプロパティーの「値」フィールドを編集します。
    • プロパティーを削除するには、削除するプロパティーの「名前」フィールドの左側にあるチェックボックスを選択し、「プロパティを削除」ボタンをクリックします。
  8. 「ターゲット」タブをクリックして「リソースターゲット」を管理します。
    1. 必要に応じて、リソースの定義済みターゲットを有効または無効にします。
    2. 「ターゲットの管理」ボタンをクリックして、リソースのターゲットを追加または削除します。

    詳細は、「JDBC リソースのターゲットを編集する」を参照してください。

  9. 「一般」タブをクリックして、「JDBC リソースを編集」ページに戻ります。
  10. 「保存」をクリックします。
