JDBC リソースのターゲットを編集する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「JDBC」ノードを展開します。
  3. 「JDBC リソース」ノードを展開します。
  4. 編集する JDBC リソースを選択します。
  5. 「JDBC リソースを編集」ページで、「ターゲット」タブを選択します。
  6. リソースに新しいターゲットを割り当てるには、「ターゲットの管理」ボタンを選択します。
  7. 「ターゲットの管理」ページで、利用可能なターゲットを選択してターゲットを追加するか、「選択したターゲット」リストからターゲットを削除します。
  8. 「了解」をクリックして編集を適用します。
