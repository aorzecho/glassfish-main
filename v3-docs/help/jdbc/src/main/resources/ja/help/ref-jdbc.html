JDBC

「JDBC」ページを使用して、次のタイプのリソースを設定します。

  • 「JDBC リソース」

  • 「接続プール」

JDBC (Java Database Connectivity) テクノロジは、リレーショナルデータベースへの接続を提供します。GlassFish サーバー では、JDBC リソース (データソース) をドメインに追加することで、データベースの接続を設定します。データソースは、データベースへの接続を設定する Java Platform, Enterprise Edition (Java EE プラットフォーム) の標準メソッドです。各データソースには、データベース接続のプールが格納されます。アプリケーションは、Java Naming and Directory Interface (JNDI) API ツリー上またはローカルのアプリケーションコンテキスト内でデータソースを検索し、getConnection メソッドを使用してデータベース接続を取得します。データソースと接続プールによって、システムの稼働を維持する接続管理プロセスが提供されます。

GlassFish サーバー には Java DB データベースのコピーが含まれ、このデータベースにアクセスする機能があらかじめ設定されています。
