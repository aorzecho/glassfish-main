接続プールのプロパティを編集

「接続プールのプロパティを編集」ページを使用して、既存のJDBC接続プールの追加プロパティを変更します。

追加プロパティ

JDBC接続プールの追加プロパティ。

使用可能なプロパティの説明については、JDBC接続プールに固有のプロパティを参照してください。

プロパティの値を指定しない場合は、接続プールの編集が終了したときにプロパティが表から削除されます。プロパティの値をすぐに設定せず、後で設定できるように表内に残す場合は、「値」フィールドに空の括弧を入力します。
