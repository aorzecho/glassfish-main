JDBC リソースを削除する

リソースに関連付けられている JDBC 接続プールを削除すると、JDBC リソースが自動的に削除されます。

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JDBC」ノードを展開します。
  3. 「JDBC」ノードで、「JDBC リソース」ノードを選択します。

    「JDBC リソース」ページが開きます。

  4. 「JDBC リソース」ページで、削除するリソースの左側にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
