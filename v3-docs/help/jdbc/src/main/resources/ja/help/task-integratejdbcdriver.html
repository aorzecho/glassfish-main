JDBC ドライバを統合する

JDBC API をサポートするドライバ (JDBC ドライバ) は、アプリケーションの JDBC 呼び出しを、データベースサーバーのプロトコルに変換します。

  • JDBC ドライバを GlassFish サーバー ドメインに統合するには、次のいずれかの操作を実行します。
    • 共通クラスローダーにアクセスできるドライバを作成します。
      1. ドライバの JAR ファイルおよび ZIP ファイルを domain-dir/lib ディレクトリにコピーする

        注 - この手順では、例外が 1 つあります。Oracle JDBC ドライバを使用していて、Oracle データベースで使用可能な EclipseLink 拡張機能を利用する場合は、Oracle JDBC ドライバの JAR ファイルを domain-dir/lib/ext ディレクトリに配置する必要があります。

      2. ドメインを再起動します。
    • システムクラスローダーにアクセスできるドライバを作成する
      1. ナビゲーションツリーで、「設定」ノードを展開します。
      2. 「設定」ノードの下で、「JVM 設定」ノードを選択します。

        「JVM 一般設定」ページが開きます。

      3. 「JVM 一般設定」ページで、「パス設定」タブをクリックします。
      4. 「クラスパスのサフィックス」フィールドで、ドライバの JAR ファイルの完全修飾パス名を入力します。
      5. 「保存」をクリックします。
      6. サーバーを再起動します。
