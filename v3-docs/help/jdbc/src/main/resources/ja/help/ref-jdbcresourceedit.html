JDBC リソースを編集

「JDBC リソースを編集」ページを使用して、既存の JDBC リソースを変更します。

「JDBC リソースを編集」ページには、次のオプションが含まれます。

JNDI 名

JDBC リソースを識別する一意の名前。JNDI 名は読み取り専用フィールドです。JNDI 名を指定できるのは、新しい JDBC リソースの作成時のみです。

プール名

JDBC リソースに関連付けられた接続プール。

説明

JDBC リソースの簡単な説明。

ステータス

このオプションが有効な場合、リソースは実行時に利用可能です。このオプションはデフォルトで有効になっています。

追加プロパティー

JDBC リソースの追加プロパティー。GlassFish Server は JDBC リソースの追加プロパティーを定義しません。
