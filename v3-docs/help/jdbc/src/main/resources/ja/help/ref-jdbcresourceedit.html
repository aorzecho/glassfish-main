JDBCリソースを編集

「JDBCリソースを編集」ページを使用して、既存のJDBCリソースを変更します。

「JDBCリソースを編集」ページには、次のオプションが含まれます。

JNDI名

JDBCリソースを識別する一意の名前。JNDI名は読取り専用フィールドです。JNDI名を指定できるのは、新しいJDBCリソースの作成時のみです。

プール名

JDBCリソースに関連付けられた接続プール。

説明

JDBCリソースの簡単な説明。

ステータス

このオプションが有効な場合、リソースは実行時に使用可能です。このオプションはデフォルトで有効になっています。

追加プロパティ

JDBCリソースの追加プロパティ。GlassFish ServerはJDBCリソースの追加プロパティを定義しません。
