Modificare una risorsa JDBC

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JDBC.

  3. Nel nodo JDBC, selezionare il nodo Risorse JDBC.

    Viene visualizzata la pagina Risorse JDBC.

  4. Nella pagina Risorse JDBC, fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica risorsa JDBC.

  5. Dall'elenco a discesa Nome pool, selezionare il connection pool da associare alla risorsa JDBC.

  6. Nel campo Descrizione, digitare una breve descrizione per la risorsa JDBC.

  7. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  8. Fare clic sulla scheda Destinazione per gestire le destinazioni della risorsa.

    1. Abilitare o disabilitare le destinazioni definite per la risorsa, come desiderato.

    2. Fare clic sul pulsante Gestisci destinazioni per aggiungere o rimuovere destinazioni per la risorsa.

    Per ulteriori informazioni, vedere Modificare le destinazioni di una risorsa JDBC.

  9. Fare clic sulla scheda Generale per tornare alla pagina Modifica risorsa JDBC.

  10. Fare clic su Salva.
