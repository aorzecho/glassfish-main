Modificare una risorsa JDBC

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JDBC.
  3. Nel nodo JDBC, selezionare il nodo JDBC Resources.

    Viene visualizzata la pagina JDBC Resources.

  4. Nella pagina JDBC Resources, fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Edit JDBC Resource.

  5. Dall'elenco a discesa Pool Name, selezionare il pool di connessione da associare alla risorsa JDBC.
  6. Nel campo Description, digitare una breve descrizione per la risorsa JDBC.
  7. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  8. Fare clic sulla scheda Target per gestire le destinazioni della risorsa.
    1. Abilitare o disabilitare le destinazioni definite per la risorsa, come desiderato.
    2. Fare clic sul pulsante Manage Targets per aggiungere o rimuovere destinazioni per la risorsa.

    Per ulteriori informazioni, vedere Modificare le destinazioni di una risorsa JDBC.

  9. Fare clic sulla scheda General per tornare alla pagina JDBC Resource.
  10. Fare clic su Save.
