Impostare l'accesso al database

Per l'archiviazione, l'organizzazione e il recupero di dati, la maggior parte delle applicazioni si serve di database relazionali.

GlassFish Server include una copia del database Java DB ed è preconfigurato con l'accesso a tale database.

  1. Installare un database supportato.

    Per una lista dei database supportati, vedere Oracle GlassFish Server Release Notes.

  2. Installare un driver che supporti l'API JDBC (driver JDBC) per il database.

    Verificare che il driver che si desidera utilizzare per creare connessioni al database sia installato in tutti i server nei quali si desidera distribuire l'origine dati.

  3. Verificare che il file JAR del driver sia accessibile per l'istanza server del dominio.

    Vedere Integrare un driver JDBC.

  4. Creare il database.

    In genere il provider dell'applicazione fornisce script per la creazione e l'inserimento di dati nel database.

  5. Creare un connection pool per il database.

    Vedere Creare un connection pool JDBC.

  6. Creare una risorsa JDBC che faccia riferimento al connection pool.

    Vedere Creare una risorsa JDBC.
