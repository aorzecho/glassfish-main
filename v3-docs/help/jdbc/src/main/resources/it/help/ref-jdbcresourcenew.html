New JDBC Resource

Nella pagina New JDBC Resource è possibile creare una nuova risorsa JDBC.

La pagina New JDBC Resource contiene le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa JDBC. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza, il nome JNDI rappresenta un metodo importante per accedere alla risorsa JDBC. Per convenzione, il nome inizia con la stringa jdbc/, ad esempio jdbc/payrolldb. Ricordare di includere la barra (/). Il nome può contenere solo caratteri alfanumerici, caratteri di sottolineatura, trattini e punti.

Pool Name

Pool di connessione associato alla risorsa JDBC.

Description

Breve descrizione della risorsa JDBC.

Status

Se questa opzione è abilitata, la risorsa è disponibile in fase di runtime. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa JDBC. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

Targets

Cluster e istanze standalone per la risorsa JDBC. Per spostare le destinazioni desiderate nella colonna Selected Targets, utilizzare i pulsanti Add e Add All. Per spostare le destinazioni non necessarie nella colonna Available Targets, utilizzare i pulsanti Remove e Remove All.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
