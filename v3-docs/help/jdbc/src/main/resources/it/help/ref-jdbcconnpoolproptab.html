Edit Connection Pool Properties

Nella pagina Edit Connection Pool Properties è possibile modificare proprietà aggiuntive per un pool di connessione JDBC esistente.

Additional Properties

Proprietà aggiuntive per il pool di connessione JDBC.

Per una descrizione di alcune delle proprietà disponibili, vedere Proprietà specifiche per i pool di connessione JDBC.

Se non si specifica un valore per una proprietà, questa viene rimossa dalla tabella al termine della modifica del pool di connessione. Se anziché impostare il valore di una proprietà in questa fase si preferisce conservare la proprietà nella tabella in modo da impostare il valore in un secondo momento, digitare due parentesi vuote nel campo Value:
