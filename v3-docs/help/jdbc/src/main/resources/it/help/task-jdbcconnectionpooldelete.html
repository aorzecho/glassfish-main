Eliminare un pool di connessione JDBC

Quando si elimina un pool di connessione JDBC, vengono eliminate anche tutte le risorse JDBC che dipendono da tale pool.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JDBC.
  3. Nel nodo JDBC, selezionare il nodo Connection Pools.

    Viene visualizzata la pagina Connection Pools.

  4. Nella pagina Connection Pools, selezionare la casella di controllo a sinistra del nome del pool di connessione da eliminare.
  5. Fare clic su Delete.
