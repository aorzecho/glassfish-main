Modificare le destinazioni di una risorsa JDBC

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo JDBC.

  3. Espandere il nodo Risorse JDBC.

  4. Selezionare la risorsa JDBC da modificare.

  5. Nella pagina Modifica risorsa JDBC, selezionare la scheda Destinazioni.

  6. Per assegnare una nuova destinazione alla risorsa, scegliere il pulsante Gestisci destinazioni.

  7. Nella pagina Gestisci destinazioni, selezionare una destinazione disponibile per aggiungerla o rimuoverla dalla lista Destinazioni selezionate.

  8. Fare clic su OK per applicare le modifiche.
