Modificare le destinazioni di una risorsa JDBC

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo JDBC.
  3. Espandere il nodo JDBC Resources.
  4. Selezionare la risorsa JDBC da modificare.
  5. Nella pagina Edit JDBC Resource, selezionare la scheda Targets.
  6. Per assegnare una nuova destinazione alla risorsa, scegliere il pulsante Manage Targets.
  7. Nella pagina Manage Targets, selezionare una destinazione disponibile per aggiungerla o rimuoverla dall'elenco Selected Targets.
  8. Fare clic su OK per applicare le modifiche.
