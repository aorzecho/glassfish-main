New JDBC Connection Pool (Step 1 of 2)

Nella pagina New JDBC Connection Pool (Step 1 of 2) è possibile creare un nuovo pool di connessione JDBC.

La pagina New JDBC Connection Pool (Step 1 of 2) contiene le seguenti opzioni.

Pool Name

Nome del pool di connessione.

Resource Type

I tipi di risorse disponibili sono javax.sql.XADataSource (transazioni globali), java.sql.ConnectionPoolDataSource (transazioni locali, possibili miglioramenti delle prestazioni), javax.sql.DataSource (solo transazioni locali) e java.sql.Driver.

Database Vendor

Nome del produttore del database utilizzato.
