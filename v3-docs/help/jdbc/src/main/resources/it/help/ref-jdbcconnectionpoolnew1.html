Nuovo connection pool JDBC (passo 1 di 2)

Nella pagina Nuovo connection pool JDBC (passo 1 di 2) è possibile creare un nuovo connection pool JDBC.

La pagina Nuovo connection pool JDBC (passo 1 di 2) contiene le opzioni riportate di seguito.

Nome pool

Nome del connection pool.

Tipo di risorsa

I tipi di risorse disponibili sono javax.sql.XADataSource (transazioni globali), java.sql.ConnectionPoolDataSource (transazioni locali, possibili miglioramenti delle prestazioni), javax.sql.DataSource (solo transazioni locali) e java.sql.Driver.

Fornitore driver database

Nome del fornitore del database in uso.

Analisi

Se questa opzione è abilitata, il driver di database viene analizzato in base ai nomi di classe di implementazione dell'origine dati o del driver.
