Integrare un driver JDBC

I driver che supportano l'API JDBC (driver JDBC) traducono le chiamate JDBC di un'applicazione nel protocollo del database server.

Per integrare un driver JDBC in un dominio GlassFish Server, attenersi a una delle procedure seguenti:

  • Rendere accessibile il driver al classloader comune.

    1. Copiare i file JAR e ZIP del driver nella directory domain-dir/lib.

      Nota:

      Esiste un'eccezione a questa regola: se si usa il driver Oracle JDBC e si desidera utilizzare le estensioni EclipseLink disponibili per i database Oracle, è necessario posizionare i file JAR del driver Oracle JDBC nella directory domain-dir/lib/ext.

    2. Riavviare il dominio.

  • Rendere il driver accessibile al classloader di sistema.

    1. Nell'albero di navigazione espandere il nodo Configurazione.

    2. Nel nodo Configurazione, selezionare il nodo Impostazioni JVM.

      Viene visualizzata la pagina Impostazioni generali JVM.

    3. Nella pagina Impostazioni generali JVM, fare clic sulla scheda Impostazioni percorso.

    4. Nel campo Suffisso classpath, digitare il percorso completo del file JAR del driver.

    5. Fare clic su Salva.

    6. Riavviare il server.
