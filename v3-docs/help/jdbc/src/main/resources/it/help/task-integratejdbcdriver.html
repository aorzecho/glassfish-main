Integrare un driver JDBC

I driver che supportano l'API JDBC (driver JDBC) traducono le chiamate JDBC di un'applicazione per il protocollo del server database.

  • Per integrare un driver JDBC in un dominio GlassFish Server, attenersi a una delle procedure seguenti:
    • Rendere accessibile il driver al classloader comune.
      1. Copiare i file JAR e ZIP del driver nella directory domain-dir/lib.

        Nota - Esiste un'eccezione a questa regola: se si usa il driver Oracle JDBC e si desidera utilizzare le estensioni EclipseLink disponibili per i database Oracle, è necessario copiare i file JAR del driver Oracle JDBC nella directory domain-dir/lib/ext.

      2. Riavviare il dominio.
    • Rendere il driver accessibile al classloader di sistema.
      1. Nella struttura ad albero, espandere il nodo Configuration.
      2. Nel nodo Configuration, selezionare il nodo JVM Settings.

        Viene visualizzata la pagina JVM General Settings.

      3. Nella pagina JVM General Settings, fare clic sulla scheda Path Settings.
      4. Nel campo Classpath Suffix, digitare il percorso completo del file JAR del driver.
      5. Fare clic su Save.
      6. Riavviare il server.
