Verificare le impostazioni del connection pool

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JDBC.

  3. Nel nodo JDBC, selezionare il nodo Connection pool.

    Viene visualizzata la pagina Connection pool.

  4. Nella pagina Connection pool, fare clic sul nome del connection pool da verificare.

    Viene visualizzata la pagina Modifica connection pool.

  5. Nella pagina Modifica connection pool, fare clic sul pulsante Ping.

    Console di amministrazione tenta di eseguire la connessione al database. Se viene visualizzato un messaggio di errore, verificare che il server database sia stato avviato.
