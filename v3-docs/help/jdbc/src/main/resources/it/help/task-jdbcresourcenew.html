Creare una risorsa JDBC

Le risorse (origini dati) Java DataBase Connectivity (JDBC) consentono alle applicazioni di connettersi a un database.

Prima di cominciare

Per poter creare una risorsa JDBC, è necessario creare un pool di connessione JDBC. Vedere Creare un pool di connessione JDBC.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JDBC.
  3. Nel nodo JDBC, selezionare il nodo JDBC Resources.

    Viene visualizzata la pagina JDBC Resources.

  4. Nella pagina JDBC Resources, fare clic su New.

    Viene visualizzata la pagina New JDBC Resource.

  5. Nel campo JNDI Name, digitare un nome di risorsa univoco.

    Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Di conseguenza, il nome JNDI rappresenta un metodo importante per accedere alla risorsa JDBC. Per convenzione, il nome inizia con la stringa jdbc/, ad esempio jdbc/payrolldb. Ricordare di includere la barra (/).

    Il nome può contenere solo caratteri alfanumerici, caratteri di sottolineatura, trattini e punti.

  6. Dall'elenco a discesa Pool Name, selezionare il pool di connessione da associare alla nuova risorsa JDBC.
  7. Nel campo Description, digitare una breve descrizione per la nuova risorsa JDBC.
  8. Selezionare la casella di controllo Status Enabled per rendere disponibile la risorsa in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  9. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  10. Selezionare una destinazione disponibile per aggiungerla o rimuoverla dall'elenco Selected Targets.
  11. Fare clic su OK.
