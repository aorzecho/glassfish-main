Edit JDBC Resource

Nella pagina Edit JDBC Resource è possibile modificare una risorsa JDBC esistente.

La pagina Edit JDBC Resource contiene le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa JDBC. Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa JDBC.

Pool Name

Pool di connessione associato alla risorsa JDBC.

Description

Breve descrizione della risorsa JDBC.

Status

Se questa opzione è abilitata, la risorsa è disponibile in fase di runtime. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa JDBC. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.
