Modifica risorsa JDBC

Nella pagina Modifica risorsa JDBC è possibile modificare una risorsa JDBC esistente.

La pagina Modifica risorsa JDBC contiene le opzioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa JDBC. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa JDBC.

Nome pool

Connection pool associato alla risorsa JDBC.

Descrizione

Breve descrizione della risorsa JDBC.

Stato

Se questa opzione è abilitata, la risorsa è disponibile in fase di esecuzione. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa JDBC. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.
