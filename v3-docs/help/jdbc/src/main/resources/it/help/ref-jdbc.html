JDBC

Nella pagina JDBC è possibile configurare i seguenti tipi di risorse:

  • Risorse JDBC

  • Pool di connessione

La tecnologia Java DataBase Connectivity (JDBC) consente di eseguire la connettività ai database relazionali. In GlassFish Server è possibile configurare la connettività ai database aggiungendo risorse JDBC (origini dati) al dominio. Un'origine dati è un metodo standard utilizzato da Java Platform, Enterprise Edition (Java EE) per la configurazione della connettività a un database. Ogni origine dati contiene un pool di connessioni database. Le applicazioni cercano l'origine dati nella struttura ad albero dell'API Java Naming and Directory Interface (JNDI) o nel contesto dell'applicazione locale, quindi recuperano una connessione database con il metodo getConnection. Le origini dati e i relativi pool di connessione rendono disponibili processi di gestione delle connessioni che garantiscono il funzionamento del sistema.

GlassFish Server include una copia del database Java DB ed è preconfigurato con l'accesso a tale database.
