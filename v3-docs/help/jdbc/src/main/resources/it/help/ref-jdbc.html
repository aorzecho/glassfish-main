La tecnologia Java DataBase Connectivity (JDBC) consente di eseguire la connettività ai database relazionali. In GlassFish Server è possibile configurare la connettività ai database aggiungendo risorse JDBC (origini dati) al dominio. Un'origine dati è un metodo standard utilizzato dalla piattaforma Java EE (Java Platform Enterprise Edition) per la configurazione della connettività a un database. Ogni origine dati contiene un pool di connessioni al database. Le applicazioni cercano l'origine dati nella struttura dell'API JNDI (Java Naming and Directory Interface)o nel contesto dell'applicazione locale, quindi recuperano una connessione al database con il metodo getConnection. Le origini dati e i connection pool rendono disponibili processi di gestione delle connessioni che garantiscono il funzionamento del sistema.

GlassFish Server include una copia del database Java DB ed è preconfigurato con l'accesso a tale database.
