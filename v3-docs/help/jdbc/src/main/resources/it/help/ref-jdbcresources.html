Risorse JDBC

Le risorse (origini dati) Java DataBase Connectivity (JDBC) consentono alle applicazioni di connettersi a un database. In genere, l'amministratore crea una risorsa JDBC per ogni database al quale accedono le applicazioni distribuite in un dominio. È tuttavia possibile creare più risorse JDBC per un singolo database.

Per ottenere una connessione al database da un connection pool, le applicazioni cercano un'origine dati nella struttura dell'API Java Naming and Directory Interface (JNDI) e quindi richiedono la connessione. Il connection pool associato all'origine dati fornisce la connessione all'applicazione.

Nella pagina Risorse JDBC è possibile configurare le risorse JDBC.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa JDBC.

Abilitato

Opzione attivata se la risorsa è abilitata; disattivata se la risorsa è disabilitata.

Connection pool

Connection pool JDBC associato alla risorsa.

Descrizione

Breve descrizione della risorsa.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa.

Elimina

Pulsante che consente di eliminare una o più risorse selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse selezionate.
