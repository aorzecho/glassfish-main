Risorse JDBC

Le risorse (origini dati) Java DataBase Connectivity (JDBC) consentono alle applicazioni di connettersi a un database. In genere l'amministratore crea una risorsa JDBC per ogni database al quale accedono le applicazioni distribuite in un dominio. È tuttavia possibile creare più risorse JDBC per un singolo database.

Per ottenere una connessione database da un pool di connessione, le applicazioni cercano un'origine dati nella struttura ad albero dell'API Java Naming and Directory Interface (JNDI) e quindi richiedono la connessione. Il pool di connessione associato all'origine dati fornisce la connessione all'applicazione.

Nella pagina JDBC Resources è possibile configurare le risorse JDBC.

Per ogni risorsa vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa JDBC.

Enabled

Opzione attivata se la risorsa è abilitata e disattivata se la risorsa è disabilitata.

Connection Pool

Pool di connessione JDBC associato alla risorsa.

Description

Breve descrizione della risorsa.

La tabella Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa.

Delete

Pulsante che consente di eliminare una o più risorse selezionate.

Enable

Pulsante che consente di abilitare una o più risorse selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse selezionate.
