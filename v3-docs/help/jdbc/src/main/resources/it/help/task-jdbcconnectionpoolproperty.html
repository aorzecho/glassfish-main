Modificare le proprietà del connection pool JDBC

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JDBC.

  3. Nel nodo JDBC, selezionare il nodo Connection pool.

    Viene visualizzata la pagina Connection pool.

  4. Fare clic sul nome del connection pool da modificare.

    Viene visualizzata la pagina Modifica connection pool.

  5. Nella pagina Modifica connection pool, fare clic sulla scheda Proprietà aggiuntive.

    Viene visualizzata la pagina Modifica proprietà connection pool.

  6. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    Alcuni esempi di proprietà obbligatorie sono il nome del database (URL), il nome utente e la password. Per informazioni sulle proprietà disponibili, vedere Proprietà specifiche dei connection pool JDBC.

    Se non si specifica un valore per una proprietà, questa viene rimossa dalla tabella al termine della modifica del connection pool. Se anziché impostare il valore di una proprietà in questa fase si preferisce conservare la proprietà nella tabella in modo da impostare il valore in un secondo momento, digitare due parentesi vuote nel campo Valore:

    ()

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  7. Fare clic su Salva.
