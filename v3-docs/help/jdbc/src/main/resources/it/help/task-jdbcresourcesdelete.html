Eliminare una risorsa JDBC

Una risorsa JDBC viene eliminata automaticamente quando si elimina il pool di connessione JDBC associato alla risorsa.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo JDBC.
  3. Nel nodo JDBC, selezionare il nodo JDBC Resources.

    Viene visualizzata la pagina JDBC Resources.

  4. Nella pagina JDBC Resources, selezionare la casella di controllo accanto al nome della risorsa da eliminare.
  5. Fare clic su Delete.
