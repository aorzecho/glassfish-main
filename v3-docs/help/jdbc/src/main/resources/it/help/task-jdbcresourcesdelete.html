Eliminare una risorsa JDBC

Una risorsa JDBC viene eliminata automaticamente quando si elimina il connection pool JDBC associato alla risorsa.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo JDBC.

  3. Nel nodo JDBC, selezionare il nodo Risorse JDBC.

    Viene visualizzata la pagina Risorse JDBC.

  4. Nella pagina Risorse JDBC, selezionare la casella di controllo accanto al nome della risorsa da eliminare.

  5. Fare clic su Elimina.
