JDBC 자원 편집

JDBC 자원 편집 페이지에서는 기존 JDBC 자원을 수정할 수 있습니다.

JDBC 자원 편집 페이지에는 다음 옵션이 포함되어 있습니다.

JNDI 이름

JDBC 자원을 식별하는 고유 이름입니다. JNDI 이름은 읽기 전용 필드입니다. 새 JDBC 자원을 만들 때 JNDI 이름만 지정할 수 있습니다.

풀 이름

JDBC 자원과 연결된 연결 풀입니다.

설명

JDBC 자원에 대한 간단한 설명입니다.

상태

이 옵션을 활성화하면 런타임 시 자원을 사용할 수 있습니다. 이 옵션은 기본적으로 활성화됩니다.

추가 등록 정보

JDBC 자원의 추가 등록 정보입니다. GlassFish Server는 JDBC 자원의 추가 등록 정보를 정의하지 않습니다.

관련 작업

  • 데이터베이스 액세스를 설정하는 방법

  • JDBC 드라이버를 통합하는 방법

  • JDBC 자원을 만드는 방법

  • JDBC 자원을 편집하는 방법

  • JDBC 자원을 삭제하는 방법

관련 asadmin 명령

  • add-resources(1)

  • create-jdbc-resource(1)

  • delete-jdbc-resource(1)

  • list-jdbc-resources(1)

  • start-database(1)

  • stop-database(1)
