JDBC 자원 대상을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. JDBC 노드를 확장합니다.
  3. JDBC 자원 노드를 확장합니다.
  4. 편집할 JDBC 자원을 선택합니다.
  5. JDBC 자원 편집 페이지에서 대상 탭을 선택합니다.
  6. 새 대상을 자원에 할당하려면 대상 관리 버튼을 선택합니다.
  7. 대상 관리 페이지에서 사용 가능한 대상을 선택하여 추가하거나 선택한 대상 목록에서 제거합니다.
  8. 편집 내용을 적용하려면 확인을 누릅니다.
