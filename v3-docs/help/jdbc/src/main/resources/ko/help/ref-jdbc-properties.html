JDBC 연결 풀과 관련된 등록 정보

대부분의 JDBC 드라이버에서는 표준 등록 정보 목록을 사용하여 사용자, 암호 및 기타 자원 구성 정보를 지정할 수 있습니다. GlassFish Server에서는 등록 정보가 선택적이지만 일부 등록 정보는 대부분의 데이터베이스에서 필요할 수 있습니다. 자세한 내용은 JDBC 4.0 Standard Extension API를 참조하십시오.

JDBC 드라이버 등록 정보를 변경할 경우 서버를 다시 시작해야 합니다.

다음 표준 등록 정보는 여러 JDBC 공급업체에서 일반적으로 사용됩니다. JDBC 드라이버에서 사용할 수 있는 등록 정보에 대한 자세한 내용은 공급업체 설명서를 참조하십시오.

user

데이터베이스 연결에 사용되는 사용자 이름을 지정합니다.

password

데이터베이스 연결에 사용되는 암호를 지정합니다.

databaseName

이 연결 풀의 데이터베이스를 지정합니다.

serverName

이 연결 풀의 데이터베이스 서버를 지정합니다.

port

데이터베이스 서버에서 요청을 수신하는 포트를 지정합니다.

networkProtocol

통신 프로토콜을 지정합니다.

roleName

초기 SQL 역할 이름을 지정합니다.

datasourceName

기본 XADataSource 또는 연결 풀링이 수행된 경우 ConnectionPoolDataSource를 지정합니다.

description

텍스트 설명을 지정합니다.

url

이 연결 풀의 URL을 지정합니다. 표준 등록 정보는 아니지만 일반적으로 사용됩니다.
