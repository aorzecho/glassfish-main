데이터베이스 액세스를 설정하는 방법

데이터 저장과 구성 및 검색을 위해 대부분의 응용 프로그램은 관계형 데이터베이스를 사용합니다.

GlassFish Server에는 Java DB 데이터베이스의 복사본이 포함되고 이 데이터베이스에 액세스할 수 있는 기능이 사전 구성되어 있습니다.

  1. 지원되는 데이터베이스 제품을 설치합니다.

    지원되는 데이터베이스 목록은 Oracle GlassFish Server 릴리스 노트을(를) 참조하십시오.

  2. JDBC API를 지원하는 드라이버(JDBC 드라이버)는 응용 프로그램에 대한 JDBC 호출을 데이터베이스 서버의 프로토콜로 변환합니다.

    데이터베이스 접속을 생성하는 데 사용하려는 드라이버가 데이터 소스를 배치하려는 모든 서버에 설치되어 있는지 확인합니다.

  3. 도메인의 서버 인스턴스가 드라이버의 JAR 파일에 액세스할 수 있는지 확인합니다.

    JDBC 드라이버를 통합하는 방법을(를) 참조하십시오.

  4. 데이터베이스를 생성합니다.

    대개, 응용 프로그램 제공자가 데이터베이스를 생성하고 채우는 스크립트를 제공합니다.

  5. 데이터베이스의 접속 풀을 생성합니다.

    JDBC 접속 풀을 생성하는 방법을(를) 참조하십시오.

  6. 접속 풀을 가리키는 JDBC 리소스를 생성합니다.

    JDBC 리소스를 생성하는 방법을(를) 참조하십시오.
