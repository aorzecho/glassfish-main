JDBC 연결 풀 등록 정보를 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 연결 풀 노드를 선택합니다.

    연결 풀 페이지가 열립니다.

  4. 편집할 연결 풀의 이름을 누릅니다.

    연결 풀 편집 페이지가 열립니다.

  5. 연결 풀 편집 페이지에서 추가 등록 정보 탭을 누릅니다.

    연결 풀 등록 정보 편집 페이지가 열립니다.

  6. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    필요한 등록 정보의 예로는 데이터베이스 이름(URL), 사용자 이름, 암호 등이 있습니다. 사용할 수 있는 등록 정보에 대한 설명은 JDBC 연결 풀과 관련된 등록 정보를 참조하십시오.

    등록 정보의 값을 지정하지 않으면 연결 풀 편집을 마칠 때 표에서 제거됩니다. 현재는 등록 정보 값을 설정하지 않지만 나중에 설정할 수 있도록 등록 정보를 테이블에 유지하려면 값 필드에 일련의 빈 괄호를 입력합니다.

    ()
    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  7. 저장을 누릅니다.
