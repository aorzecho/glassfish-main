JDBC 자원을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 JDBC 자원 노드를 선택합니다.

    JDBC 자원 페이지가 열립니다.

  4. JDBC 자원 페이지에서 편집할 자원의 이름을 누릅니다.

    JDBC 자원 편집 페이지가 열립니다.

  5. 풀 이름 드롭다운 목록에서 이 JDBC 자원과 연결할 연결 풀을 선택합니다.
  6. 설명 필드에 이 JDBC 자원에 대한 간단한 설명을 입력합니다.
  7. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish 서버는 JDBC 자원의 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  8. 자원 대상을 관리하려면 대상 탭을 누릅니다.
    1. 자원에 대해 정의된 대상을 원하는 대로 활성화하거나 비활성화합니다.
    2. 자원에 대한 대상을 추가하거나 제거하려면 대상 관리 버튼을 누릅니다.

    자세한 내용은 JDBC 자원 대상을 편집하는 방법을 참조하십시오.

  9. JDBC 자원 편집 페이지로 돌아가려면 일반 탭을 누릅니다.
  10. 저장을 누릅니다.
