JDBC 리소스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 JDBC 노드를 확장합니다.

  3. JDBC 노드에서 [JDBC 리소스] 노드를 선택합니다.

    [JDBC 리소스] 페이지가 열립니다.

  4. [JDBC 리소스] 페이지에서 편집할 리소스의 이름을 누릅니다.

    JDBC 리소스 편집 페이지가 열립니다.

  5. 풀 이름 드롭다운 목록에서 이 JDBC 리소스와 접속할 접속 풀을 선택합니다.

  6. [설명] 필드에 이 JDBC 리소스에 대한 간단한 설명을 입력합니다.

  7. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    GlassFish Server에서는 JDBC 리소스에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  8. 리소스 대상을 관리하려면 [대상] 탭을 누릅니다.

    1. 리소스에 대해 정의된 대상을 원하는 대로 사용 또는 사용 안함으로 설정합니다.

    2. 리소스에 대한 대상을 추가하거나 제거하려면 대상 관리 단추를 누릅니다.

    자세한 내용은 JDBC 리소스 대상을 편집하는 방법을(를) 참조하십시오.

  9. JDBC 리소스 편집 페이지로 돌아가려면 일반 탭을 누릅니다.

  10. [저장]을 누릅니다.
