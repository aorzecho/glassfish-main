연결 풀

JDBC 연결 풀에는 연결 풀 등록 시 생성된 JDBC 연결의 그룹이 포함됩니다. 연결 풀은 JDBC 드라이버를 사용하여 물리적 데이터베이스 연결을 만듭니다. 응용 프로그램은 풀에서 연결을 빌려서 사용한 다음, 연결을 닫아 풀로 반환합니다. JDBC 자원을 만들려면 연관된 연결 풀을 지정합니다. 여러 JDBC 자원이 단일 연결 풀을 지정할 수 있습니다.

JDBC 연결 풀은 특정 데이터베이스에 대해 재사용 가능한 연결 그룹입니다. 새로운 물리적 연결을 만드는 데 시간이 많이 소모되기 때문에 서버에서는 사용 가능한 연결 풀을 유지 관리하여 성능을 늘립니다. 응용 프로그램은 연결을 요청할 때 풀에서 연결을 가져옵니다. 응용 프로그램에서 연결을 닫으면 연결이 풀로 반환됩니다.

관리 콘솔에서 연결 풀을 만들 때 특정 데이터베이스에 대한 연결 부분을 정의합니다.
