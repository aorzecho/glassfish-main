접속 풀

JDBC 접속 풀에는 접속 풀 등록 시 생성된 JDBC 접속의 그룹이 포함됩니다. 접속 풀은 JDBC 드라이버를 사용하여 물리적 데이터베이스 접속을 생성합니다. 응용 프로그램은 풀에서 접속을 빌려서 사용한 다음, 접속을 닫아 풀로 반환합니다. JDBC 리소스를 생성하려면 연관된 접속 풀을 지정합니다. 여러 JDBC 리소스가 단일 접속 풀을 지정할 수 있습니다.

JDBC 접속 풀은 특정 데이터베이스에 대해 재사용 가능한 접속 그룹입니다. 새로운 물리적 접속을 생성하는 데 시간이 많이 소모되기 때문에 서버에서는 사용 가능한 접속 풀을 유지 관리하여 성능을 늘립니다. 응용 프로그램은 접속을 요청할 때 풀에서 접속을 가져옵니다. 응용 프로그램에서 접속을 닫으면 접속이 풀로 반환됩니다.

관리 콘솔에서 접속 풀을 생성할 때 특정 데이터베이스에 대한 접속 부분을 정의합니다.
