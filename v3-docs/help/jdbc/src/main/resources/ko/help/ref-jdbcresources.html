JDBC 자원

JDBC(Java DataBase Connectivity) 자원(데이터 소스)은 응용 프로그램을 데이터베이스에 연결하는 방법을 제공합니다 일반적으로 관리자는 도메인에 배포된 응용 프로그램에서 액세스하는 각 데이터베이스에 대해 JDBC 자원을 만들지만 데이터베이스에 대해 둘 이상의 JDBC 자원을 만들 수 있습니다.

응용 프로그램은 JNDI(Java Naming and Directory Interface) API 트리에서 데이터 소스를 찾은 다음 연결을 요청하여 연결 풀에서 데이터베이스 연결을 가져옵니다. 데이터 소스와 연결된 연결 풀은 응용 프로그램에 대한 연결을 제공합니다.

JDBC 자원 페이지를 사용하여 JDBC 자원을 구성합니다.

각 자원에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

JDBC 자원을 식별하는 고유 이름입니다.

사용 가능

자원이 사용 가능한 경우 true, 사용 가능하지 않은 경우 false입니다.

연결 풀

자원과 연관된 JDBC 연결 풀입니다.

설명

자원에 대한 간단한 설명입니다.

자원 테이블에는 다음 옵션도 포함되어 있습니다.
