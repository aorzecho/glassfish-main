JDBC 리소스

JDBC(Java DataBase Connectivity) 리소스(데이터 소스)는 응용 프로그램을 데이터베이스에 접속하는 방법을 제공합니다. 일반적으로 관리자는 도메인에 배치된 응용 프로그램에서 액세스하는 각 데이터베이스에 대해 JDBC 리소스를 생성하지만 데이터베이스에 대해 둘 이상의 JDBC 리소스를 생성할 수 있습니다.

응용 프로그램은 JNDI(Java Naming and Directory Interface) API 트리에서 데이터 소스를 찾은 다음 연결을 요청하여 접속 풀에서 데이터베이스 접속을 가져옵니다. 데이터 소스와 연관된 접속 풀은 응용 프로그램에 대한 접속을 제공합니다.

[JDBC 리소스] 페이지를 사용하여 JDBC 리소스를 구성합니다.

각 리소스에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

JDBC 리소스를 식별하는 고유 이름입니다.

사용

리소스가 사용으로 설정된 경우 true, 사용 안함으로 설정된 경우 false입니다.

접속 풀

리소스와 연관된 JDBC 접속 풀입니다.

설명

리소스에 대한 간단한 설명입니다.

리소스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 리소스를 생성하는 단추입니다.

삭제

선택한 리소스를 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 리소스를 사용 안함으로 설정하는 단추입니다.
