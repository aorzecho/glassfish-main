JDBC 드라이버를 통합하는 방법

JDBC API를 지원하는 드라이버(JDBC 드라이버)는 응용 프로그램에 대한 JDBC 호출을 데이터베이스 서버의 프로토콜로 변환합니다.

  • JDBC 드라이버를 GlassFish 서버 도메인에 통합하려면 다음 중 하나를 수행합니다.
    • 드라이버가 공통 클래스 로더에 액세스할 수 있게 합니다.
      1. 드라이버의 JAR 및 ZIP 파일을 domain-dir/lib 디렉토리로 복사합니다.

        주 - 이 규칙에는 한 가지 예외가 있습니다. Oracle JDBC 드라이버를 사용 중이고 Oracle 데이터베이스에서 사용할 수 있는 EclipseLink 확장명을 이용하려는 경우 domain-dir/lib/ext 디렉토리에 Oracle JDBC 드라이버용 JAR 파일을 두어야 합니다.

      2. 도메인을 다시 시작합니다.
    • 드라이버가 시스템 클래스 로더에 액세스할 수 있게 합니다.
      1. 탐색 트리에서 구성 노드를 선택합니다.
      2. 구성 노드에서 JVM 설정 노드를 선택합니다.

        JVM 일반 설정 페이지가 열립니다.

      3. JVM 일반 설정 페이지에서 경로 설정 탭을 누릅니다.
      4. 클래스 경로 접미어 필드에 드라이버의 JAR 파일에 대한 정규화된 경로 이름을 입력합니다.
      5. 저장을 누릅니다.
      6. 서버를 다시 시작합니다.
