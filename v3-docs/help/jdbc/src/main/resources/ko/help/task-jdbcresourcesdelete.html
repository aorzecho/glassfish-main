JDBC 자원을 삭제하는 방법

JDBC 자원은 자원과 관련된 JDBC 연결 풀을 삭제하는 경우 자동으로 삭제됩니다.

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 JDBC 자원 노드를 선택합니다.

    JDBC 자원 페이지가 열립니다.

  4. JDBC 자원 페이지에서 삭제할 자원 왼쪽의 확인란을 선택합니다.
  5. 삭제를 누릅니다.
