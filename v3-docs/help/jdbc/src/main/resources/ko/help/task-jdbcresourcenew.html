JDBC 자원을 만드는 방법

JDBC(Java DataBase Connectivity) 자원(데이터 소스)은 응용 프로그램을 데이터베이스에 연결하는 방법을 제공합니다

시작하기 전에

JDBC 자원을 만들려면 먼저 JDBC 연결 풀을 만들어야 합니다. JDBC 연결 풀을 만드는 방법을 참조하십시오.

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 JDBC 자원 노드를 선택합니다.

    JDBC 자원 페이지가 열립니다.

  4. JDBC 자원 페이지에서 새로 만들기를 누릅니다.

    새 JDBC 자원 페이지가 열립니다.

  5. JNDI 이름 필드에 고유한 자원 이름을 입력합니다.

    JNDI 이름은 카드 카탈로그가 도서관 내 책의 위치를 조직화하여 나타내는 것과 유사한 방식으로 분산 컴퓨팅 환경 내에서 구성 요소를 조직화하고 구성 요소의 위치를 찾아냅니다. 이에 따라 JNDI 이름은 JDBC 자원에 액세스하는 중요한 방법이 됩니다. 일반적으로 이름은 jdbc/ 문자열로 시작합니다. 예를 들어 jdbc/payrolldb입니다. 슬래시를 반드시 입력합니다.

    이름에는 영숫자, 밑줄, 대시 또는 점 문자만 사용해야 합니다.

  6. 풀 이름 드롭다운 목록에서 새 JDBC 자원과 연결할 연결 풀을 선택합니다.
  7. 설명 필드에 새 JDBC 자원에 대한 간단한 설명을 입력합니다.
  8. 런타임 시 자원을 사용 가능하게 하려면 사용 가능 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  9. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish Server는 JDBC 자원의 추가 등록 정보를 정의하지 않습니다.

    등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.

  10. 사용 가능한 대상을 선택하여 선택한 대상 목록에 추가하거나 선택한 대상 목록에서 제거합니다.
  11. 확인을 누릅니다.
