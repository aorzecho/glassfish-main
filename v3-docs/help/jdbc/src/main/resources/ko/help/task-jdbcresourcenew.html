JDBC 리소스를 생성하는 방법

JDBC(Java DataBase Connectivity) 리소스(데이터 소스)는 응용 프로그램을 데이터베이스에 접속하는 방법을 제공합니다.

시작하기 전에

JDBC 리소스를 생성하려면 먼저 JDBC 접속 풀을 생성되어야 합니다. JDBC 접속 풀을 생성하는 방법을(를) 참조하십시오.

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 JDBC 노드를 확장합니다.

  3. JDBC 노드에서 [JDBC 리소스] 노드를 선택합니다.

    [JDBC 리소스] 페이지가 열립니다.

  4. [JDBC 리소스] 페이지에서 [새로 만들기]를 누릅니다.

    [새 JDBC 리소스] 페이지가 열립니다.

  5. [JNDI 이름] 필드에 고유한 리소스 이름을 입력합니다.

    JNDI 이름은 카드 카탈로그가 도서관 내 책의 위치를 조직화하여 나타내는 것과 유사한 방식으로 분산 컴퓨팅 환경 내에서 구성 요소를 조직화하고 구성 요소의 위치를 찾아냅니다. 이에 따라 JNDI 이름은 JDBC 리소스에 액세스하는 중요한 방법이 됩니다. 일반적으로 이름은 jdbc/ 문자열로 시작합니다(예: jdbc/payrolldb). 슬래시를 반드시 입력합니다.

    이름에는 영숫자, 밑줄, 대시 또는 점 문자만 사용해야 합니다.

  6. 풀 이름 드롭다운 목록에서 새 JDBC 리소스와 접속할 접속 풀을 선택합니다.

  7. [설명] 필드에 새 JDBC 리소스에 대한 간단한 설명을 입력합니다.

  8. 런타임 시 리소스를 사용 가능하게 하려면 사용 체크 박스를선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    GlassFish Server에서는 JDBC 리소스에 대한 추가 속성을 정의하지 않습니다.

    속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

  10. 사용 가능한 대상을 선택하여 선택한 대상 목록에 추가하거나 선택한 대상 목록에서 제거합니다.

  11. [확인]을 누릅니다.
