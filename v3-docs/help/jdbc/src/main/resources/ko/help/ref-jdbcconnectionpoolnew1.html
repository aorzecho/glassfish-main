새 JDBC 접속 풀(단계 1/2)

새 JDBC 접속 풀(단계 1/2) 페이지에서는 새 JDBC 접속 풀을 생성할 수 있습니다.

새 JDBC 접속 풀(단계 1/2) 페이지에는 다음 옵션이 포함되어 있습니다.

풀 이름

접속 풀의 이름입니다.

리소스 유형

사용 가능한 리소스 유형은 javax.sql.XADataSource(전역 트랜잭션), java.sql.ConnectionPoolDataSource(로컬 트랜잭션, 성능 향상 가능), javax.sql.DataSource (로컬 트랜잭션만) 및 java.sql.Driver입니다.

데이터베이스 드라이버 공급업체

데이터베이스 제품의 공급업체 이름입니다.

검토

이 옵션을 사용으로 설정하면 데이터 소스 또는 드라이버 구현 클래스 이름에 따라 데이터베이스 드라이버가 검토됩니다.

관련 작업

  • 데이터베이스 액세스를 설정하는 방법

  • JDBC 드라이버를 통합하는 방법

  • JDBC 접속 풀을 생성하는 방법

  • JDBC 접속 풀 일반 설정을 편집하는 방법

  • 접속 풀 설정을 확인하는 방법

  • JDBC 접속 풀 고급 속성을 편집하는 방법

  • JDBC 접속 풀 속성을 편집하는 방법

  • JDBC 접속 풀을 삭제하는 방법
