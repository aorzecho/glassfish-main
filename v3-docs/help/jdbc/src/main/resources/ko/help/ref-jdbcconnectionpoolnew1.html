새 JDBC 연결 풀(단계 1/2)

새 JDBC 연결 풀(단계 1/2) 페이지에서는 새 JDBC 연결 풀을 만들 수 있습니다.

새 JDBC 연결 풀(단계 1/2) 페이지에는 다음 옵션이 포함되어 있습니다.

풀 이름

연결 풀의 이름입니다.

자원 유형

사용 가능한 자원 유형은 javax.sql.XADataSource(전역 트랜잭션), java.sql.ConnectionPoolDataSource(로컬 트랜잭션, 성능 향상 가능), javax.sql.DataSource(로컬 트랜잭션만) 및 java.sql.Driver입니다.

데이터베이스 공급업체

데이터베이스 제품의 공급업체 이름입니다.
