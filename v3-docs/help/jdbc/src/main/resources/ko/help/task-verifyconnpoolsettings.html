연결 풀 설정을 확인하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 연결 풀 노드를 선택합니다.

    연결 풀 페이지가 열립니다.

  4. 연결 풀 페이지에서 검증할 연결 풀의 이름을 누릅니다.

    연결 풀 편집 페이지가 열립니다.

  5. 연결 풀 편집 페이지에서 핑 버튼을 누릅니다.

    관리 콘솔이 데이터베이스 연결을 시도합니다. 오류 메시지가 표시되면 데이터베이스 서버가 시작되었는지 확인합니다.

참조

  • 데이터베이스 액세스를 설정하는 방법

  • JDBC 드라이버를 통합하는 방법

  • JDBC 자원을 만드는 방법

  • JDBC 자원을 편집하는 방법

  • JDBC 자원 대상을 편집하는 방법

  • JDBC 자원을 삭제하는 방법

  • JDBC 연결 풀을 만드는 방법

  • JDBC 연결 풀 일반 설정을 편집하는 방법

  • JDBC 연결 풀 고급 속성을 편집하는 방법

  • JDBC 연결 풀 등록 정보를 편집하는 방법

  • JDBC 연결 풀을 삭제하는 방법
