접속 풀 속성 편집

접속 풀 속성 편집 페이지에서는 기존 JDBC 접속 풀의 추가 속성을 수정할 수 있습니다.

추가 속성

JDBC 접속 풀의 추가 속성입니다.

사용 가능한 일부 속성에 대한 설명은 JDBC 접속 풀 관련 속성을(를) 참조하십시오.

속성의 값을 지정하지 않으면 접속 풀 편집을 마칠 때 표에서 제거됩니다. 지금은 속성값을 설정하지 않지만 나중에 설정할 수 있도록 속성을 테이블에 유지하려면 [값] 필드에 일련의 빈 괄호를 입력합니다.
