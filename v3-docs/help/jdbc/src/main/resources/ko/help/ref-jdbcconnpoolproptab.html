연결 풀 등록 정보 편집

연결 풀 등록 정보 편집 페이지에서는 기존 JDBC 연결 풀의 추가 등록 정보를 수정할 수 있습니다.

추가 등록 정보

JDBC 연결 풀의 추가 등록 정보입니다.

사용 가능한 일부 등록 정보에 대한 설명은 JDBC 연결 풀과 관련된 등록 정보를 참조하십시오.

등록 정보의 값을 지정하지 않으면 연결 풀 편집을 마칠 때 표에서 제거됩니다. 현재는 등록 정보 값을 설정하지 않지만 나중에 설정할 수 있도록 등록 정보를 테이블에 유지하려면 값 필드에 일련의 빈 괄호를 입력합니다.
