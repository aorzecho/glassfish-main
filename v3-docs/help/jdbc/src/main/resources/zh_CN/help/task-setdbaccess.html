设置数据库访问权限

为了存储、组织和检索数据，大多数应用程序都采用了关系数据库。

GlassFish Server 包括 Java DB 数据库的副本，并且预配置了访问此数据库的功能。

  1. 安装支持的数据库产品。

    有关受支持的数据库列表，请参阅 Oracle GlassFish Server 发行说明。

  2. 为数据库安装支持 JDBC API (JDBC 驱动程序) 的驱动程序。

    确保在要部署数据源的所有服务器上均安装了要用于创建数据库连接的驱动程序。

  3. 确保域的服务器实例可以访问此驱动程序的 JAR 文件。

    请参阅集成 JDBC 驱动程序。

  4. 创建数据库。

    通常，应用程序提供方提供了用于创建和填充数据库的脚本。

  5. 为数据库创建连接池。

    请参阅创建 JDBC 连接池。

  6. 创建指向连接池的 JDBC 资源。

    请参阅创建 JDBC 资源。
