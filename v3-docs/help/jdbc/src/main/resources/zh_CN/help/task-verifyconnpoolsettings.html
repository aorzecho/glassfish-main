验证连接池设置

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开 "JDBC" 节点。
  3. 在 "JDBC" 节点下，选择“连接池”节点。

    此时将打开“连接池”页面。

  4. 在“连接池”页面上，单击您要验证的连接池的名称。

    此时将打开“编辑连接池”页面。

  5. 在“编辑连接池”页面上，单击 "Ping" 按钮。

    管理控制台将尝试连接到数据库。如果出现错误消息，请检查数据库服务器是否已启动。
