创建 JDBC 资源

Java 数据库连接 (JDBC) 资源 (数据源) 为应用程序提供连接到数据库的方法。

开始之前

为了创建 JDBC 资源，您必须先创建一个 JDBC 连接池。请参阅创建 JDBC 连接池。

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JDBC”节点。

  3. 在“JDBC”节点下，选择“JDBC 资源”节点。

    此时将打开“JDBC 资源”页。

  4. 在“JDBC 资源”页上，单击“新建”。

    此时将打开“新建 JDBC 资源”页。

  5. 在“JNDI 名称”字段中，键入一个唯一的资源名称。

    JNDI 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，JNDI 名称是访问 JDBC 资源的重要方法之一。按照约定，名称以 jdbc/ 字符串开头，例如 jdbc/payrolldb。请不要忘记正斜杠。

    名称只能包含字母数字、下划线、短划线或点字符。

  6. 从“池名称”下拉列表中，选择要与新的 JDBC 资源关联的连接池。

  7. 在“说明”字段中，键入新 JDBC 资源的简短说明。

  8. 选择“启用状态”复选框，使资源在运行时可用。

    默认情况下启用此选项。

  9. 在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义 JDBC 资源的任何其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  10. 选择一个可用目标以从“所选目标”列表中添加或删除该目标。

  11. 单击“确定”。
