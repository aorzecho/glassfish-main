集成 JDBC 驱动程序

支持 JDBC API 的驱动程序（JDBC 驱动程序）将应用程序的 JDBC 调用转换为数据库服务器的协议。

  • 要将 JDBC 驱动程序集成到 GlassFish Server 域中，请执行以下操作之一：
    • 使通用类加载器可以访问该驱动程序。
      1. 将驱动程序的 JAR 和 ZIP 文件复制到 domain-dir/lib 目录中。

        注 - 此规则有一个例外。如果您使用 Oracle JDBC 驱动程序，并希望利用可用来与 Oracle 数据库搭配使用的 EclipseLink 扩展，则必须将 Oracle JDBC 驱动程序的 JAR 文件放在 domain-dir/lib/ext 目录中。

      2. 重新启动域。
    • 使系统类加载器可以访问驱动程序。
      1. 在导航树中，展开“配置”节点。
      2. 在“配置”节点下，选择“JVM 设置”节点。

        此时将打开“JVM 常规设置”页面。

      3. 在“JVM 常规设置”页面上，单击“路径设置”选项卡。
      4. 在“类路径后缀”字段中，键入驱动程序 JAR 文件的全限定路径名。
      5. 单击“保存”。
      6. 重新启动服务器。
