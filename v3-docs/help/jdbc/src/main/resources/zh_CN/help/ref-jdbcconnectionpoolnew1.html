新建 JDBC 连接池 (第 1 步，共 2 步)

使用“新建 JDBC 连接池 (第 1 步，共 2 步)”页可以创建新的 JDBC 连接池。

“新建 JDBC 连接池 (第 1 步，共 2 步)”页包含以下选项：

池名称

连接池的名称。

资源类型

可用的资源类型包括 javax.sql.XADataSource (全局事务处理)、java.sql.ConnectionPoolDataSource (本地事务处理，性能可能会提高)、javax.sql.DataSource (仅本地事务处理) 和 java.sql.Driver。

数据库驱动程序供应商

数据库产品的提供方的名称。

自测

如果启用此选项，数据库驱动程序将基于数据源或驱动程序实现类名进行自测。
