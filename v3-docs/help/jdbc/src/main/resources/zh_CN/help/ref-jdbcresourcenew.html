新建 JDBC 资源

使用“新建 JDBC 资源”页可以创建新的 JDBC 资源。

“新建 JDBC 资源”页包含以下选项：

JNDI 名称

标识 JDBC 资源的唯一名称。JNDI 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，JNDI 名称是访问 JDBC 资源的重要方法之一。按照约定，名称以 jdbc/ 字符串开头，例如 jdbc/payrolldb。请不要忘记正斜杠。名称只能包含字母数字、下划线、短划线或点字符。

池名称

与 JDBC 资源关联的连接池。

说明

JDBC 资源的简短说明。

状态

如果启用此选项，则资源将在运行时可用。默认情况下启用此选项。

其他属性

JDBC 资源的其他属性。GlassFish Server 不定义 JDBC 资源的任何其他属性。

目标

JDBC 资源的集群和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“所选目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。

仅当在域中已创建集群或独立实例时才会显示此选项。
