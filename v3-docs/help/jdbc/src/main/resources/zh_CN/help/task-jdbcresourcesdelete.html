删除 JDBC 资源

如果删除与资源关联的 JDBC 连接池，则将自动删除 JDBC 资源。

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开 "JDBC" 节点。
  3. 在 "JDBC" 节点下，选择“JDBC 资源”节点。

    此时将打开“JDBC 资源”页面。

  4. 在“JDBC 资源”页面上，选中要删除的资源左侧的复选框。
  5. 单击“删除”。
