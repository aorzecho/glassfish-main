删除 JDBC 连接池

当您删除某个 JDBC 连接池时，任何依赖于该连接池的 JDBC 资源也将被删除。

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JDBC”节点。

  3. 在“JDBC”节点下，选择“连接池”节点。

    此时将打开“连接池”页。

  4. 在“连接池”页中，选中要删除的连接池左侧的复选框。

  5. 单击“删除”。
