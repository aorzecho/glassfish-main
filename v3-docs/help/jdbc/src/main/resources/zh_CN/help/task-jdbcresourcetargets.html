编辑 JDBC 资源目标

  1. 在导航树中，展开“资源”节点。
  2. 展开 "JDBC" 节点。
  3. 展开“JDBC 资源”节点。
  4. 选择要编辑的 JDBC 资源。
  5. 从“编辑 JDBC 资源”页面中，选择“目标”选项卡。
  6. 要为资源指定一个新的目标，请选择“管理目标”按钮。
  7. 在“管理目标”页面中，选择一个可用目标以从“选定目标”列表中添加或删除该目标。
  8. 单击“确定”应用所做编辑。
