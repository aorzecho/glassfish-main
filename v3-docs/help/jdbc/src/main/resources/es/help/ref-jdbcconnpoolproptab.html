Editar Propiedades de Pools de Conexiones

Utilice la página Editar propiedades de conjunto de conexiones para modificar las propiedades adicionales de un conjunto de conexiones JDBC existente.

Propiedades adicionales

Propiedades adicionales del conjunto de conexiones JDBC.

Para obtener una descripción de algunas de las propiedades disponibles, consulte Propiedades Específicas de Pools de Conexiones JDBC.

Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del conjunto de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, escriba un conjunto de paréntesis vacío en el campo Valor:

()
