Editar Recurso JDBC

Utilice la página Editar recurso JDBC para modificar un recurso JDBC existente.

La página Editar recurso JDBC contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso JDBC. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI. al crear un nuevo recurso JDBC.

Nombre de conjunto

Conjunto de conexiones relacionado con el recurso JDBC.

Descripción

Breve descripción del recurso JDBC.

Estado

Si esta opción está habilitada, el recurso está disponible en el tiempo de ejecución. Esta opción se habilita por defecto.

Propiedades adicionales

Propiedades adicionales para el recurso JDBC. GlassFish Server no define ninguna propiedad adicional para recursos JDBC.
