Para integrar un controlador JDBC

Un controlador compatible con la API JDBC (controlador JDBC) convierte las llamadas JDBC de una aplicación en el protocolo del servidor de base de datos.

  • Para integrar el controlador JDBC en un dominio de GlassFish Server, haga una de las dos cosas siguientes:
    • Haga accesible el controlador al cargador de clases comunes.
      1. Copie los archivos JAR y ZIP del controlador en el directorio domain-dir/lib.

        Nota - Hay una excepción a esta regla. Si está utilizando el controlador JDBC de Oracle y desea aproveche las extensiones EclipseLink disponibles con las bases de datos de Oracle, debe colocar los archivos JAR para el controlador JDBC de Oracle en el directorio domain-dir/lib/ext.

      2. Reinicie el dominio.
    • Haga accesible el controlador al cargador de clases del sistema.
      1. En el árbol de navegación, expanda el nodo Configuración.
      2. En el nodo Configuración, seleccione el nodo Configuración de JVM.

        Se abrirá la página Configuración general de JVM.

      3. En la página Configuración general de JVM, haga clic en la ficha Configuración de ruta.
      4. En el campo Sufijo de ruta de clase, introduzca el nombre de ruta completo para el archivo JAR del controlador.
      5. Haga clic en Guardar.
      6. Reinicie el servidor.
