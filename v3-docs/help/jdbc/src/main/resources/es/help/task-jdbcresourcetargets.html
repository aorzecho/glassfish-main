Para editar destinos de recurso JDBC

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Expanda el nodo JDBC.
  3. Expanda el nodo Recursos JDBC.
  4. Seleccione el recurso JDBC que quiera editar.
  5. Desde la página Editar recurso JDBC, seleccione la ficha Destinos.
  6. Para asignar un nuevo destino al recurso, seleccione el botón Administrar destinos.
  7. En la página Administrar destinos, seleccione un destino disponible para agregarlo o quitarlo de la lista Destinos seleccionados.
  8. Haga clic en Aceptar para aplicar las ediciones.
