Editar un Recurso JDBC

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Bajo el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Recursos JDBC.

    Se abre la página Recursos de JDBC.

  4. En la página Recursos JDBC, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar recurso JDBC.

  5. En la lista desplegable Nombre de conjunto, seleccione un conjunto de conexiones para asociarlo al recurso JDBC.

  6. En el campo Descripción, introduzca una breve descripción del recurso JDBC.

  7. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para recursos JDBC.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  8. Haga clic en la ficha Destino para administrar los destinos de recurso.

    1. Habilite o deshabilite los destinos definidos para el recurso, según desee.

    2. Haga clic en el botón Administrar destinos para agregar o quitar destinos para el recurso.

    Consulte Para editar destinos de recurso JDBC para obtener más información.

  9. Haga clic en la ficha General para volver a la página Editar recurso JDBC.

  10. Haga clic en Guardar.
