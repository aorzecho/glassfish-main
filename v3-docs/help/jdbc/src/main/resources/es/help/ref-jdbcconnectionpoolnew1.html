Nuevo Pool de Conexiones JDBC (Paso 1 de 2)

Utilice la página Nuevo conjunto de conexiones JDBC (Paso 1 de 2) para crear un nuevo conjunto de conexiones JDBC.

La página Nuevo conjunto de conexiones JDBC (Paso 1 de 2) contiene las siguientes opciones.

Nombre de conjunto

El nombre del conjunto de conexiones.

Tipo de recurso

Los tipos de recursos disponibles son javax.sql.XADataSource (transacciones globales), java.sql.ConnectionPoolDataSource (transacciones locales, posibles mejoras de rendimiento), javax.sql.DataSource (sólo transacciones locales) y java.sql.Driver.

Proveedor de Controladores de la Base de Datos

Nombre del proveedor del producto de la base de datos.

Introspección

Si se activa esta opción, se realiza una introspección del controlador de la base de datos en función del origen de datos o de los nombres de clase de la implantación del controlador.
