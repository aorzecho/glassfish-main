Para verificar la configuración del conjunto de conexiones

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Bajo el nodo Recursos, amplíe el nodo JDBC.
  3. En el nodo JDBC, seleccione el nodo Conjuntos de conexiones.

    Se abrirá la página Conjuntos de conexiones.

  4. En la página Conjunto de conexiones, haga clic en el nombre del conjunto de conexiones que desea verificar.

    Se abrirá la página Editar conjunto de conexiones.

  5. En la página Editar conjunto de conexiones, haga clic en el botón Ping.

    La Consola de administración intenta la conexión a la base de datos. Si aparece un mensaje de error, compruebe si el servidor de la base de datos se ha reiniciado.
