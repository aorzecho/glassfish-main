Editar las Propiedades de los Pools de Conexiones JDBC

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Bajo el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Conjuntos de conexiones.

    Se abrirá la página Conjuntos de conexiones.

  4. Haga clic en el nombre del conjunto de conexiones que desea verificar.

    Se abrirá la página Editar conjunto de conexiones.

  5. En la página Editar Pool de Conexiones, haga clic en el separador Propiedades Adicionales.

    Se abrirá la página Editar propiedades del conjunto de conexiones.

  6. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Entre los ejemplos de propiedades necesarias, se incluyen el nombre de la base de datos (URL), el nombre de usuario y la contraseña. Para obtener información sobre las propiedades disponibles, consulte Propiedades Específicas de Pools de Conexiones JDBC.

    Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del conjunto de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, escriba un conjunto de paréntesis vacío en el campo Valor:

    ()

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  7. Haga clic en Guardar.
