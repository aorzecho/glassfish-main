Para editar las propiedades del conjunto de conexiones JDBC

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Bajo el nodo Recursos, amplíe el nodo JDBC.
  3. En el nodo JDBC, seleccione el nodo Conjuntos de conexiones.

    Se abrirá la página Conjuntos de conexiones.

  4. Haga clic en el nombre del conjunto de conexiones que desea verificar.

    Se abrirá la página Editar conjunto de conexiones.

  5. En la página Editar conjunto de conexiones, haga clic en la ficha Propiedades adicionales.

    Se abrirá la página Editar propiedades del conjunto de conexiones.

  6. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Entre los ejemplos de propiedades necesarias, se incluyen el nombre de la base de datos (URL), el nombre de usuario y la contraseña. Para obtener información sobre las propiedades disponibles, consulte Propiedades específicas de los conjuntos de conexiones JDBC.

    Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del conjunto de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, escriba un conjunto de paréntesis vacío en el campo Valor: 

    ()
    • Para añadir una propiedad, haga clic en el botón Añadir propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.
    • Para modificar una propiedad, edite su campo Valor.
    • Para eliminar una propiedad, marque la casilla de verificación situada a la izquierda del campo Nombre de la propiedad que quiera eliminar y, a continuación, haga clic en el botón Eliminar propiedades.
  7. Haga clic en Guardar.
