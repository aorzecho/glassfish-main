Para eliminar un conjunto de conexiones JDBC

Cuando se elimina un conjunto de conexiones JDBC, los recursos JDBC que dependen de ese conjunto de conexiones también se eliminan.

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Bajo el nodo Recursos, amplíe el nodo JDBC.
  3. En el nodo JDBC, seleccione el nodo Conjuntos de conexiones.

    Se abrirá la página Conjuntos de conexiones.

  4. En la página Conjunto de conexiones, marque la casilla de verificación que aparece a la izquierda del conjunto de conexiones que desee eliminar.
  5. Haga clic en Eliminar.
