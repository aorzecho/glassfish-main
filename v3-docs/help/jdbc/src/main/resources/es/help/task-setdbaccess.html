Configurar el Acceso a la Base de Datos

Para almacenar, organizar y recuperar datos, la mayoría de las aplicaciones utilizan bases de datos relacionales.

GlassFish Server incluye una copia de la base de datos Java DB y se ha configurado previamente con la capacidad para acceder a esta base de datos.

  1. Instale un producto de base de datos compatible.

    Para obtener una lista de bases de datos soportadas, consulte Notas Técnicas sobre la Versión de Oracle GlassFish Server.

  2. Instale un controlador que soporte la API JDBC (controlador JDBC) de la base de datos.

    Asegúrese de que el controlador que desea utilizar para crear conexiones de bases de datos está instalado en todos los servidores en los que va a desplegar el origen de datos.

  3. Asegúrese de que desde la instancia del servidor de dominio se pueda acceder al archivo JAR del controlador.

    Consulte Integrar un Controlador JDBC.

  4. Cree la base de datos.

    Generalmente, el proveedor de la aplicación suministra scripts para crear y rellenar la base de datos.

  5. Cree un conjunto de conexiones para la base de datos.

    Consulte Crear un Pool de Conexiones JDBC.

  6. Cree un recurso JDBC que apunte al pool de conexiones.

    Consulte Crear un Recurso JDBC.
