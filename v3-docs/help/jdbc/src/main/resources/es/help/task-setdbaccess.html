Para configurar el acceso a la base de datos

Para almacenar, organizar y recuperar datos, la mayoría de las aplicaciones utilizan bases de datos relacionales.

Servidor GlassFish incluye una copia de la base de datos Java DB y se ha configurado previamente con la capacidad para acceder a esta base de datos.

  1. Instale un producto de base de datos compatible.

    Para obtener una lista de las bases de datos admitidas, consulte las Oracle GlassFish Server 3.1-3.1.1 Release Notes .

  2. Instale un controlador compatible con la API JDBC (controlador JDBC) de la base de datos.

    Asegúrese de que el controlador que desea utilizar para crear conexiones de bases de datos está instalado en todos los servidores en los que va a implementar la fuente de datos.

  3. Asegúrese de que desde la instancia del servidor de dominio se puede acceder al archivo JAR del controlador.

    Consulte Para integrar un controlador JDBC.

  4. Cree la base de datos.

    Generalmente, el proveedor de la aplicación suministra scripts para crear y rellenar la base de datos.

  5. Cree un conjunto de conexiones para la base de datos.

    Consulte Para crear un conjunto de conexiones JDBC.

  6. Cree un recurso JDBC que señale el conjunto de conexiones.

    Consulte Para crear un recurso JDBC.
