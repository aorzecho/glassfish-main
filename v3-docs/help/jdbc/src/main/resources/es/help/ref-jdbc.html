La tecnología Java DataBase Connectivity (JDBC) ofrece conectividad a bases de datos relacionales. En Servidor GlassFish, configura la conectividad a una base de datos agregando recursos JDBC (fuentes de datos) al dominio. Una fuente de datos es un método estándar de plataforma Java, Enterprise Edition (plataforma Java EE) de configuración de la conectividad a una base de datos. Cada fuente de datos contiene un conjunto de conexiones de bases de datos. Las aplicaciones buscan la fuente de datos en el árbol API de Java Naming and Directory Interface (JNDI) o en el contexto de la aplicación local y, a continuación, recuperan la conexión con la base de datos con el método getConnection. Las fuentes de datos y sus conjuntos de conexiones ofrecen procesos de administración de conexión que ayudan a mantener su sistema en funcionamiento.

Servidor GlassFish incluye una copia de la base de datos Java DB y se ha configurado previamente con la capacidad para acceder a esta base de datos.
