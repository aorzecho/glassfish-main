Recursos JDBC

Un recurso (fuente de datos) Java DataBase Connectivity (JDBC) proporciona a las aplicaciones los medios de conexión a una base de datos. Normalmente, el administrador crea un recurso JDBC para cada base de datos a las que acceden las aplicaciones implementadas en un dominio; sin embargo, se puede crear más de un recurso JDBC para una base de datos.

Las aplicaciones obtienen una conexión con la base de datos a partir de un conjunto de conexiones buscando una fuente de datos en el árbol API de Java Naming and Directory Interface (JNDI) y, posteriormente, solicitando una conexión. El conjunto de conexiones asociado a la fuente de datos proporciona la conexión con la aplicación.

Utilice la página Recursos JDBC para configurar los recursos de JDBC.

En cada recurso, aparece la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso JDBC.

Habilitado

True si el recurso está habilitado o false, si no lo está.

Conjunto de conexiones

El conjunto de conexiones JDBC asociado con el recurso.

Descripción

Una breve descripción del recurso.

La tabla de recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso.

Borrar

Botón para eliminar uno o varios recursos seleccionados.

Habilitar

Botón para habilitar uno o varios recursos seleccionados.

Deshabilitar

Botón para deshabilitar uno o varios recursos seleccionados.
