Suprimir un Recurso JDBC

Los recursos JDBC se eliminan automáticamente si se borra el conjunto de conexiones JDBC asociado al recurso.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Bajo el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Recursos JDBC.

    Se abre la página Recursos de JDBC.

  4. En la página Recursos JDBC, marque la casilla de verificación situada a la izquierda del recurso que se va a eliminar.

  5. Haga clic en Eliminar.
