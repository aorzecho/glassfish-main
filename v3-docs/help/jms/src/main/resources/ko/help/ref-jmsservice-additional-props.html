JMS 서비스 관련 속성

JMS 서비스에 대해 다음과 같은 Oracle Message Queue 브로커 구성 속성을 사용할 수 있습니다.

instance-name

전체 Message Queue 브로커 인스턴스 이름을 지정합니다. 기본값은 imqbroker입니다.

instance-name-suffix

전체 Message Queue 브로커 인스턴스 이름에 추가할 접미어를 지정합니다. 접미어는 밑줄 문자(_)로 인스턴스 이름과 구분됩니다. 예를 들어 인스턴스 이름이 imqbroker인 경우 접미어 xyz를 추가하면 인스턴스 이름이 imqbroker_xyz로 변경됩니다.

append-version

true인 경우 전체 Message Queue 브로커 인스턴스 이름에 밑줄 문자(_)와 주 버전 및 부 버전 번호가 추가됩니다. 예를 들어 인스턴스 이름이 imqbroker인 경우 버전 번호를 추가하면 인스턴스 이름이 imqbroker_8_0으로 변경됩니다. 기본값은 false입니다.

user-name

JMS 접속을 생성하기 위한 사용자 이름입니다. 브로커에서 guest/guest의 기본 사용자 이름/비밀번호를 사용할 수 없는 경우에만 필요합니다. 기본값은 guest입니다.

password

JMS 접속을 생성하기 위한 비밀번호입니다. 브로커에서 guest/guest의 기본 사용자 이름/비밀번호를 사용할 수 없는 경우에만 필요합니다. 기본값은 guest입니다.
