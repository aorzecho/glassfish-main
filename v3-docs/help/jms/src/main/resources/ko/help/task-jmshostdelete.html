JMS 호스트를 삭제하는 방법

기본 JMS 호스트의 삭제는 권장하지 않습니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 Java Message Service 노드를 확장합니다.
  3. Java Message Service 노드에서 JMS 호스트 노드를 선택합니다.

    JMS 호스트 페이지가 열립니다.

  4. JMS 호스트 페이지에서 삭제할 호스트 이름 옆의 확인란을 선택합니다.

    주의 - 기본 JMS 호스트는 삭제하지 마십시오. 삭제하면 예기치 않은 오류가 발생할 수 있습니다.

  5. 삭제를 누릅니다.
