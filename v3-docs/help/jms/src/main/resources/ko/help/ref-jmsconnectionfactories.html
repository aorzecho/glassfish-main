JMS 연결 팩토리

JMS 연결 팩토리 페이지에서는 JMS 연결 팩토리 자원을 구성할 수 있습니다.

각 자원에 대해 제공되는 정보는 다음과 같습니다.

풀 이름

연결 팩토리를 식별하는 고유한 이름입니다.

JNDI 이름

연결 팩토리를 식별하는 JNDI 이름입니다.

사용 가능

연결 팩토리를 사용할 수 있으면 true이고, 연결 팩토리를 사용할 수 없으면 false입니다.

자원 유형

연결 팩토리의 자원 유형입니다.

설명

연결 팩토리에 대한 설명입니다.

연결 팩토리 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 연결 팩토리를 만드는 버튼입니다.

삭제

하나 이상의 선택된 연결 팩토리를 삭제하는 버튼입니다.

활성화

하나 이상의 선택된 연결 팩토리를 활성화하는 버튼입니다.

비활성화

하나 이상의 선택된 연결 팩토리를 비활성화하는 버튼입니다.

관련 작업

  • JMS 연결 팩토리 자원을 만드는 방법

  • JMS 연결 팩토리 자원을 편집하는 방법

  • JMS 연결 팩토리 자원을 삭제하는 방법

관련 asadmin 명령

  • create-jms-resource(1)

  • list-jms-resources(1)

  • delete-jms-resource(1)
