JMS 대상 리소스 편집

JMS 대상 리소스 편집 페이지에서는 JMS 대상 리소스에 대한 설정을 수정할 수 있습니다.

JMS 대상 리소스 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 단추입니다. 기본값이 없는 설정은 변경되지 않습니다.

JNDI 이름

대상 리소스를 식별하는 고유한 이름입니다. JNDI 이름은 읽기 전용 필드입니다. 새 JMS 대상 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

물리적 대상 이름

이 리소스와 연관된 물리적 대상의 이름입니다.

GlassFish Server은(는) 물리적 대상이 필요할 때 자동으로 물리적 대상을 생성하고 사용자가 대상 리소스를 삭제할 때 물리적 대상을 삭제합니다.

리소스 유형

대상 리소스의 유형입니다. 사용 가능한 선택 항목은 javax.jms.Topic 및 javax.jms.Queue입니다.

설명

대상 리소스에 대한 설명입니다.

상태

대상 리소스의 상태입니다. 대상 리소스는 사용 또는 사용 안함으로 설정할 수 있습니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.

추가 속성

대상 리소스의 추가 속성. GlassFish Server에서는 JMS 대상 리소스에 대한 추가 속성을 정의하지 않습니다.

관련 작업

  • JMS 대상 리소스를 생성하는 방법

  • JMS 대상 리소스를 편집하는 방법

  • JMS 대상 리소스를 삭제하는 방법

관련 asadmin 명령

  • create-jms-resource(1)

  • list-jms-resources(1)

  • delete-jms-resource(1)
