JMS 대상 자원을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JMS 자원 노드를 확장합니다.
  3. JMS 자원 노드에서 JMS 대상 자원 노드를 선택합니다.

    JMS 대상 자원 페이지가 열립니다.

  4. JMS 대상 자원 페이지에서 편집할 대상 자원의 이름을 누릅니다.

    JMS 대상 자원 편집 페이지가 열립니다.

  5. 물리적 대상 이름 필드에 이 자원과 연결할 물리적 대상의 이름을 입력합니다.

    GlassFish 서버는 물리적 대상이 필요할 때 자동으로 물리적 대상을 만들고 사용자가 대상 자원을 삭제할 때 물리적 대상을 삭제합니다.

  6. 자원 유형 드롭다운 목록에서 javax.jms.Topic 또는 javax.jms.Queue를 선택합니다.
  7. 설명 필드에 대상 자원에 대한 설명을 입력합니다.
  8. 런타임에 자원을 활성화하려면 상태 사용 가능 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  9. (옵션) 추가 등록 정보 영역에서 대상 자원에 대한 추가 등록 정보를 지정합니다.

    GlassFish 서버에서는 JMS 대상 자원에 대한 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  10. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.
