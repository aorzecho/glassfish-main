JMS 대상 리소스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [JMS 리소스] 노드를 확장합니다.

  3. [JMS 리소스] 노드에서 [JMS 대상 리소스] 노드를 선택합니다.

    [JMS 대상 리소스] 페이지가 열립니다.

  4. [JMS 대상 리소스] 페이지에서 편집할 대상 리소스의 이름을 누릅니다.

    [JMS 대상 리소스 편집] 페이지가 열립니다.

  5. [물리적 대상 이름] 필드에 이 리소스와 연결할 물리적 대상의 이름을 입력합니다.

    GlassFish Server은(는) 물리적 대상이 필요할 때 자동으로 물리적 대상을 생성하고 사용자가 대상 리소스를 삭제할 때 물리적 대상을 삭제합니다.

  6. 리소스 유형 드롭다운 목록에서 javax.jms.Topic 또는 javax.jms.Queue를 선택합니다.

  7. [설명] 필드에 대상 리소스에 대한 설명을 입력합니다.

  8. 런타임에 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. [추가 속성] 영역에서 대상 리소스에 대한 추가 속성을 지정합니다.

    GlassFish Server에서는 JMS 대상 리소스에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  10. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.

  11. Optionally, click the Targets tab to go to the JMS Destination Resource Targets page. On this page, do the following:

    1. Click Manage Targets to open the Manage Resource Targets page.

      On this page, move targets between the Available column and the Selected column. Make sure to place in the Selected column the target or targets to which applications that use the resource will be deployed. The available targets include the available clusters and server instances, as well as the default server instance, server. Click OK to save the changes.

    2. Select the checkbox for a target, then click Enable or Disable to enable or disable the resource for that target.
