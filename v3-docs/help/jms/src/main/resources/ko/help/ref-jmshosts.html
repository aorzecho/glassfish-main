JMS 호스트

JMS 호스트 페이지에서는 JMS 서비스 호스트를 구성할 수 있습니다.

각 호스트에 대해 제공되는 정보는 다음과 같습니다.

이름

JMS 호스트의 이름입니다.

호스트

JMS 호스트가 실행되고 있는 시스템의 이름 또는 IP(인터넷 프로토콜) 주소입니다.

포트

호스트에 있는 JMS 서비스의 포트 번호입니다.

호스트 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 JMS 호스트를 만드는 버튼입니다.

삭제

하나 이상의 선택된 JMS 호스트를 삭제하는 버튼입니다.

관련 작업

  • JMS 서비스를 구성하는 방법

  • JMS 호스트를 만드는 방법

  • JMS 호스트를 편집하는 방법

  • JMS 호스트를 삭제하는 방법

관련 asadmin 명령

  • create-jms-host(1)

  • list-jms-hosts(1)

  • delete-jms-host(1)
