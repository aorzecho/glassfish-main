JMS 물리적 대상

JMS(Java Message Service) 물리적 대상 객체는 Message Queue 브로커에서 유지 관리합니다. mq.sys.dmq라는 대기열은 만료된 메시지와 전달할 수 없는 메시지가 리디렉션되는 시스템 대상입니다.

JMS 물리적 대상 페이지에서는 Message Queue 물리적 대상을 구성할 수 있습니다.

각 물리적 대상에 대해 제공되는 정보는 다음과 같습니다.

이름

물리적 대상의 이름입니다.

유형

물리적 대상의 유형입니다.

통계

물리적 대상의 통계입니다.

대상 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 물리적 대상을 만드는 버튼입니다.

삭제

하나 이상의 선택된 물리적 대상을 삭제하는 버튼입니다.

플러시

하나 이상의 선택된 물리적 대상에서 메시지를 제거하는 버튼입니다.

관련 작업

  • JMS 서비스를 구성하는 방법

  • JMS 물리적 대상을 만드는 방법

  • JMS 물리적 대상을 편집하는 방법

  • JMS 물리적 대상을 삭제하는 방법

  • JMS 물리적 대상에서 메시지를 제거하는 방법

  • JMS 물리적 대상의 통계를 보는 방법
