를JMS 물리적 대상 통계를 보는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 Java Message Service 노드를 확장합니다.

  3. Java Message Service 노드에서 [물리적 대상] 노드를 선택합니다.

    [물리적 대상] 페이지가 열립니다.

  4. [물리적 대상] 페이지에서 통계를 보려는 대상의 보기 링크를 누릅니다.

    JMS 물리적 대상 통계 보기 페이지가 열립니다.

  5. 통계 보기를 마치면 뒤로 단추를 누릅니다.
