JMS 호스트를 편집하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. [구성] 노드에서 Java Message Service 노드를 확장합니다.

  4. Java Message Service 노드에서 [JMS 호스트] 노드를 선택합니다.

    [JMS 호스트] 페이지가 열립니다.

  5. [JMS 호스트] 페이지에서 편집할 호스트를 선택합니다.

    JMS 호스트 편집 페이지가 열립니다.

  6. [호스트] 필드에 JMS 호스트가 실행될 시스템의 이름이나 IP 주소를 입력합니다(localhost 또는 로컬이나 원격 시스템의 이름).

  7. [포트] 필드에 JMS 서비스의 포트 번호를 입력합니다.

    사용될 JMS 서비스가 기본값이 아닌 포트에서 실행 중인 경우에만 이 필드를 변경합니다 기본 포트는 7676입니다.

  8. [관리 사용자 이름] 및 [관리 비밀번호] 필드에 Message Queue 브로커 사용자 이름 및 비밀번호를 입력합니다.

    이는 GlassFish Server 사용자 이름 및 비밀번호와는 다릅니다. 이 필드는 Message Queue 브로커 값이 as-install-parent/mq/bin/imqusermgr 명령을 사용하여 변경된 경우에만 편집하십시오. 기본값은 admin 및 admin입니다.

  9. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.
