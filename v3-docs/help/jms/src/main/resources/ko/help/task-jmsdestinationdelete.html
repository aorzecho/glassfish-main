JMS 대상 자원을 삭제하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JMS 자원 노드를 확장합니다.
  3. JMS 자원 노드에서 JMS 대상 자원 노드를 선택합니다.

    JMS 대상 자원 페이지가 열립니다.

  4. JMS 대상 자원 페이지에서 삭제할 대상 자원 이름 옆의 확인란을 선택합니다.
  5. 삭제를 누릅니다.
참조

  • JMS 연결 팩토리 자원을 만드는 방법

  • JMS 연결 팩토리 자원을 편집하는 방법

  • JMS 연결 팩토리 자원을 삭제하는 방법
