JMS 가용성

JMS 가용성 페이지를 사용하여 JMS(Java Message Service)에 대한 가용성 설정을 구성합니다.

JMS 가용성 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 데 사용하는 버튼입니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

가용성 서비스

사용 가능 확인란을 선택하면 JMS에 대해 가용성이 활성화됩니다.

MQ 저장소 풀 이름

GlassFish Message Queue 브로커 클러스터가 지속성 JMS 메시지 및 기타 브로커 구성 정보를 저장할 때 사용하는 JDBC 자원의 JNDI 이름입니다.

풀 이름에는 문자, 숫자 또는 밑줄(_), 대시(-)나 마침표(.)만 사용할 수 있습니다.

추가 등록 정보

JMS 가용성에 대한 추가 등록 정보입니다.

GlassFish 서버에서는 추가 JMS 가용성 등록 정보를 미리 정의하지 않습니다.
