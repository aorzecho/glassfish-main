JMS 호스트 편집

JMS 호스트 편집 페이지에서는 JMS 호스트에 대한 설정을 수정할 수 있습니다.

JMS 호스트 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 데 사용하는 버튼입니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

호스트의 이름입니다. 이름 필드는 읽기 전용 필드입니다. 새 JMS 호스트를 만들 때만 이름 필드를 지정할 수 있습니다.

호스트

JMS 호스트가 실행될 시스템의 이름 또는 IP(인터넷 프로토콜) 주소(localhost 또는 로컬이나 원격 시스템의 이름)입니다.

포트

JMS 서비스의 포트 번호입니다. 사용될 JMS 서비스가 기본값이 아닌 포트에서 실행 중인 경우에만 이 필드를 변경합니다 기본 포트는 7676입니다.

관리 사용자 이름 및 관리 비밀번호

Message Queue 브로커 사용자 이름 및 비밀번호입니다. 이는 GlassFish 서버 사용자 이름 및 비밀번호와는 다릅니다. 이 필드는 Message Queue 브로커 값이 as-install-parent/mq/bin/imqusermgr? 명령을 통해 변경된 경우에만 편집하십시오. 기본값은 admin 및 admin입니다.

관련 작업

  • JMS 서비스를 구성하는 방법

  • JMS 호스트를 만드는 방법

  • JMS 호스트를 편집하는 방법

  • JMS 호스트를 삭제하는 방법
