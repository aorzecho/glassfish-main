删除 JMS 主机

建议不要删除默认 JMS 主机。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“Java 消息服务”节点。
  3. 在“Java 消息服务”节点下，选择“JMS 主机”节点。

    此时将打开“JMS 主机”页面。

  4. 在“JMS 主机”页面上，选中您要删除的主机名称旁边的复选框。

    注意 - 请勿删除默认的 JMS 主机。这样做可能会导致意外错误。

  5. 单击“删除”。
