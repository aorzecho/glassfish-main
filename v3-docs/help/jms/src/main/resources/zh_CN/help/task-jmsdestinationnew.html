创建 JMS 目的地资源

创建 JMS 目的地资源时，GlassFish Server 会为该目的地资源创建“管理对象资源”。您可以在“连接器”节点下查看此资源。

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开“JMS 资源”节点。
  3. 在“JMS 资源”节点下，选择“JMS 目的地资源”节点。

    此时将打开“JMS 目的地资源”页面。

  4. 在“JMS 目的地资源”页面中，单击“新建”。

    此时将打开“新建 JMS 目的地资源”页面。

  5. 在“JNDI 名称”字段中，键入资源的 JNDI 名称。

    建议的做法是使用 JMS 资源的命名子上下文前缀 jms/。例如：jms/Queue。

  6. 在“物理目的地名称”字段中，键入要与此资源关联的物理目的地的名称。

    GlassFish Server 在需要时自动创建物理目的地，并在您删除目的地资源时删除物理目的地。

  7. 从“资源类型”下拉式列表中，选择 javax.jms.Topic? 或 javax.jms.Queue。
  8. 在“说明”字段中，键入目的地资源的说明。
  9. 选择“启用状态”复选框以在运行时启用该资源。

    默认情况下启用此选项。

  10. （可选的）在“其他属性”区域中，指定目的地资源的其他属性。

    GlassFish Server 不定义 JMS 目的地资源的任何其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  11. 单击“确定”。
