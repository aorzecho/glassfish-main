JMS 主机

使用“JMS 主机”页面可以配置 JMS 服务主机。

对于每台主机，将提供以下信息。

名称

JMS 主机的名称。

主机

运行 JMS 主机的系统的名称或 Internet 协议 (Internet Protocol, IP) 地址。

端口

主机上 JMS 服务的端口号。

“主机”表还包含以下选项。

新建

用于创建新 JMS 主机的按钮。

删除

用于删除一个或多个选定 JMS 主机的按钮。
