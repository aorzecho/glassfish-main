新建 JMS 主机

使用“新建 JMS 主机”页可以创建新的 JMS 主机。

“新建 JMS 主机”页包含以下选项：

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

名称

主机的名称 (例如 NewJmsHost)。

主机

将运行 JMS 主机的系统的名称或 Internet 协议 (IP) 地址 (localhost 或者本地或远程系统的名称)。

端口

JMS 服务的端口号。仅当使用的 JMS 服务在非默认端口上运行时，才需要更改此字段。默认端口为 7676。

管理员用户名和管理员口令

消息队列中介用户名和口令。该用户名和口令与 GlassFish Server 的用户名和口令不同。仅当已使用 as-install-parent/mq/bin/imqusermgr 命令更改了消息队列中介值时，才需要编辑这些字段。默认值为 admin 和 admin。
