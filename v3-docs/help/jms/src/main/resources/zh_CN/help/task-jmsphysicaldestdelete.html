删除 JMS 物理目的地

建议不要删除系统目的地 mq.sys.dmq。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“Java 消息服务”节点。
  3. 在“Java 消息服务”节点下，选择“物理目的地”节点。

    此时将打开“物理目的地”页面。

  4. 在“物理目的地”页面上，选中您要删除的目的地名称旁边的复选框。
  5. 单击“删除”。
