查看 JMS 物理目的地统计信息

“查看 JMS 物理目的地统计信息”页面显示有关 JMS 物理目的地使用情况的信息。

除了目的地的名称和类型以外，该页面还提供了有关以下内容的详细信息：

  • 生成方和使用方数量

  • 目的地处理的消息的数量和长度

  • 磁盘利用率
