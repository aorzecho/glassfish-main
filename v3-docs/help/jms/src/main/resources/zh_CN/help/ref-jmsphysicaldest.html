JMS 物理目标

Java 消息服务 (JMS) 物理目标对象由消息队列中介维护。名为 mq.sys.dmq 的队列是系统目标，过期和无法传送的消息将被重定向至该系统目标。

使用“JMS 物理目标”页可以配置消息队列物理目标。

对于每个物理目标，将提供以下信息：

名称

物理目标的名称。

类型

物理目标的类型。

统计信息

物理目标的统计信息。

“目标”表还包含以下选项：

新建

用于创建新物理目标的按钮。

删除

用于删除一个或多个所选物理目标的按钮。

删除所有消息

用于从一个或多个所选物理目标删除消息的按钮。
