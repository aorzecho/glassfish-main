JMS 物理目的地

Java 消息服务 (JMS) 物理目的地对象由 Message Queue 代理维护。名为 mq.sys.dmq 的队列是系统目的地，过期和无法传送的消息将被重定向至该系统目的地。

使用“JMS 物理目的地”页面可以配置 Message Queue 物理目的地。

对于每个物理目的地，将提供以下信息。

名称

物理目的地的名称。

类型

物理目的地的类型。

统计信息

物理目的地的统计信息。

“目的地”表还包含以下选项。

新建

用于创建新物理目的地的按钮。

删除

用于删除一个或多个选定物理目的地的按钮。

刷新

用于从一个或多个选定物理目的地删除消息的按钮。
