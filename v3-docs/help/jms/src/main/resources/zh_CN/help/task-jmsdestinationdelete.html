删除 JMS 目的地资源

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开“JMS 资源”节点。
  3. 在“JMS 资源”节点下，选择“JMS 目的地资源”节点。

    此时将打开“JMS 目的地资源”页面。

  4. 在“JMS 目的地资源”页面上，选中您要删除的目的地资源名称旁边的复选框。
  5. 单击“删除”。
