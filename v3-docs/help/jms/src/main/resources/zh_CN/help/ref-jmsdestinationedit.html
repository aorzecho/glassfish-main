编辑 JMS 目标资源

使用“编辑 JMS 目标资源”页可以修改 JMS 目标资源的设置。

“编辑 JMS 目标资源”页包含以下选项：

加载默认值

使具有默认值的设置还原到其默认值的按钮。不具有默认值的设置不会发生更改。

JNDI 名称

标识目标资源的唯一名称。JNDI 名称是只读字段。只能在创建新的 JMS 目标资源时指定 JNDI 名称。

物理目标名称

与此资源关联的物理目标的名称。

GlassFish Server 在需要时自动创建物理目标，并在您删除目标资源时删除物理目标。

资源类型

目标资源的类型。可用选项为 javax.jms.Topic 和 javax.jms.Queue。

说明

目标资源的说明。

状态

目标资源的状态。可以启用或禁用目标资源。默认情况下启用此选项。

其他属性

目标资源的其他属性。GlassFish Server 不定义 JMS 目标资源的任何其他属性。
