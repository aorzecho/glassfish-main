编辑 JMS 目的地资源

使用“编辑 JMS 目的地资源”页面可以修改 JMS 目的地资源的设置。

“编辑 JMS 目的地资源”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

JNDI 名称

标识目的地资源的唯一名称。JNDI 名称是只读字段。只能在创建新的 JMS 目的地资源时指定 JNDI 名称。

物理目的地名称

与此资源关联的物理目的地的名称。

GlassFish Server 在需要时自动创建物理目的地，并在您删除目的地资源时删除物理目的地。

资源类型

目的地资源的类型。可用选项为 javax.jms.Topic 和 javax.jms.Queue。

说明

目的地资源的说明。

状态

目的地资源的状态。可以启用或禁用目的地资源。默认情况下启用此选项。

其他属性

目的地资源的其他属性。GlassFish Server 不定义 JMS 目的地资源的任何其他属性。
