JMS 可用性

使用“JMS 可用性”页面可配置 Java 消息服务 (Java Message Service, JMS) 的可用性设置。

“JMS 可用性”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

可用性服务

如果选中“已启用”复选框，将为 JMS 启用可用性。

MQ 存储池名称

在保存持久性 JMS 消息和其他代理配置信息时 GlassFish Message Queue 代理群集要使用的 JDBC 资源的 JNDI 名称。

池名称只能包含字母、数字或下划线 (_)、短划线 (-) 或句点 (.) 字符。

其他属性

JMS 可用性的其他属性。

&GlassFish Server 不预定义任何其他 JMS 可用性属性。
