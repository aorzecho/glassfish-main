编辑 JMS 主机

使用“编辑 JMS 主机”页面可以修改 JMS 主机的设置。

“编辑 JMS 主机”页面包含以下选项。

装入默认值

使具有默认值的设置恢复其默认值的按钮。不具有默认值的设置不会发生更改。

配置名称

要应用该页面上设置的配置名称。此字段为只读。

名称

主机的名称。“名称”字段是只读字段。只能在创建新 JMS 主机时指定“名称”字段。

主机

将运行 JMS 主机的系统的名称或 Internet 协议 (Internet Protocol, IP) 地址（localhost 或者本地或远程系统的名称）。

端口

JMS 服务的端口号。仅当使用的 JMS 服务在非默认端口上运行时，才需要更改此字段。默认端口为 7676。

管理用户名和管理密码

Message Queue 代理用户名和密码。该用户名和密码与 GlassFish Server 的用户名和密码不同。仅当已使用 as-install-parent/mq/bin/imqusermgr 命令更改了 Message Queue 代理值时，才需要编辑这些字段。默认值为 admin 和 admin。
