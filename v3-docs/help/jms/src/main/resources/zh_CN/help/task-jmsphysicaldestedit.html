编辑 JMS 物理目的地

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“Java 消息服务”节点。
  3. 在“Java 消息服务”节点下，选择“物理目的地”节点。

    此时将打开“物理目的地”页面。

  4. 在“物理目的地”页面上，单击您要编辑的物理目的地的名称。

    此时将打开“编辑物理目的地”页面。
