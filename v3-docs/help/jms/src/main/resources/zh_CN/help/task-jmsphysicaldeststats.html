查看 JMS 物理目标的统计信息

  1. 在导航树中，展开“配置”节点。

  2. 在“配置”节点下，展开“Java 消息服务”节点。

  3. 在“Java 消息服务”节点下，选择“物理目标”节点。

    此时将打开“物理目标”页。

  4. 在“物理目标”页上，单击您要查看其统计信息的目标的“查看”链接。

    此时将打开“查看 JMS 物理目标统计信息”页。

  5. 查看完统计信息后，单击“上一页”按钮。
