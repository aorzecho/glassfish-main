JMS 连接工厂

使用“JMS 连接工厂”页面可以配置 JMS 连接工厂资源。

对于每个资源，将提供以下信息。

池名称

标识连接工厂的唯一名称。

JNDI 名称

标识连接工厂的 JNDI 名称。

已启用

如果连接工厂已启用，则为 true；如果连接工厂已禁用，则为 false。

资源类型

连接工厂的资源类型。

说明

连接工厂的说明。

“连接工厂”表还包含以下选项。

新建

用于创建新连接工厂的按钮。

删除

用于删除一个或多个选定连接工厂的按钮。

启用

用于启用一个或多个选定连接工厂的按钮。

禁用

用于禁用一个或多个选定连接工厂的按钮。
