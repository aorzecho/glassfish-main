编辑 JMS 主机

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“Java 消息服务”节点。
  3. 在“Java 消息服务”节点下，选择“JMS 主机”节点。

    此时将打开“JMS 主机”页面。

  4. 在“JMS 主机”页面上，选择您要编辑的主机。

    此时将打开“编辑 JMS 主机”页面。

  5. 在“主机”字段中，键入将运行 JMS 主机的系统的名称或 Internet 协议 (Internet Protocol, IP) 地址（localhost 或者本地或远程系统的名称）。
  6. 在“端口”字段中，键入 JMS 服务的端口号。

    仅当使用的 JMS 服务在非默认端口上运行时，才需要更改此字段。默认端口为 7676。

  7. 在“管理用户名”和“管理密码”字段中，键入 Message Queue 代理用户名和密码。

    该用户名和密码与 GlassFish Server 的用户名和密码不同。仅当已使用 as-install-parent/mq/bin/imqusermgr 命令更改了 Message Queue 代理值时，才需要编辑这些字段。默认值为 admin 和 admin。

  8. 单击“保存”。

    提示 - 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。
