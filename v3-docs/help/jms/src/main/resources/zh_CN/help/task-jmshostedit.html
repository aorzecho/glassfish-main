编辑 JMS 主机

  1. 在导航树中，展开“配置”节点。

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. 在“配置”节点下，展开“Java 消息服务”节点。

  4. 在“Java 消息服务”节点下，选择“JMS 主机”节点。

    此时将打开“JMS 主机”页。

  5. 在“JMS 主机”页上，选择您要编辑的主机。

    此时将打开“编辑 JMS 主机”页。

  6. 在“主机”字段中，键入将运行 JMS 主机的系统的名称或 Internet 协议 (IP) 地址 (localhost 或者本地或远程系统的名称)。

  7. 在“端口”字段中，键入 JMS 服务的端口号。

    仅当使用的 JMS 服务在非默认端口上运行时，才需要更改此字段。默认端口为 7676。

  8. 在“管理员用户名”和“管理员口令”字段中，键入消息队列中介用户名和口令。

    该用户名和口令与 GlassFish Server 的用户名和口令不同。仅当已使用 as-install-parent/mq/bin/imqusermgr 命令更改了消息队列中介值时，才需要编辑这些字段。默认值为 admin 和 admin。

  9. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。
