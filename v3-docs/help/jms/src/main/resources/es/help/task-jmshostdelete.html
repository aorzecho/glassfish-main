Para eliminar un host JMS

No se recomienda eliminar el host JMS predeterminado.

  1. En el árbol de navegación, amplie el nodo Configuración.
  2. En el nodo Configuración, seleccione y expanda el nodo Java Message Service.
  3. En el nodo Java Message Service, seleccione el nodo Hosts JMS.

    Se abrirá la página Hosts JMS.

  4. En la página Hosts JMS, seleccione la casilla de verificación situada junto al nombre del host que va a eliminar.

    Precaución - No elimine el host JMS predeterminado. Si lo hace, se pueden producir errores imprevistos.

  5. Haga clic en Eliminar.
