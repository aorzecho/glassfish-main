Suprimir un Recurso de Destino JMS

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos de JMS.

  3. En el nodo Recursos de JMS, seleccione el nodo Recursos de Destino JMS.

    Se abrirá la página Recursos de Destino JMS.

  4. En la página Recursos de Destino JMS, seleccione la casilla de verificación situada junto al nombre del recurso de destino que va a suprimir.

  5. Haga clic en Suprimir.
