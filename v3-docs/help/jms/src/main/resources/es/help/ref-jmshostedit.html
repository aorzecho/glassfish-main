Editar host JMS

Utilice la página Editar host JMS para modificar los parámetros de configuración de un host JMS.

La página Editar host JMS contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer las opciones que tienen valores por defecto a estos. Las opciones que no tienen valores por defecto no se modifican.

Configuration Name

El nombre de la configuración a la que aplicar los valores de esta página. Este campo es de sólo lectura.

Nombre

El nombre del host. El campo Nombre es un campo de sólo lectura. Sólo puede especificar el campo Nombre cuando crea un nuevo host JMS.

Host

El nombre o la dirección IP (Internet Protocol) del sistema donde se ejecutará el host JMS (localhost o el nombre del sistema remoto o local).

Puerto

El número de puerto del servicio JMS. Modifique este campo sólo si el servicio JMS que va a utilizar se ejecuta en un puerto que no es el predeterminado. El puerto predeterminado es 7676.

Nombre de usuario de administrador (Admin) y contraseña de administrador

El nombre de usuario y la contraseña del agente de Message Queue. Éstos son diferentes del nombre de usuario y la contraseña de Servidor GlassFish. Edite estos campos solamente si los valores del agente intermediario de Message Queue se han modificado mediante el comando as-install-parent/mq/bin/imqusermgr. Los valores predeterminados son admin y admin.
