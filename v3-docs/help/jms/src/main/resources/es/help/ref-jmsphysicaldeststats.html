Ver Estadísticas de Destino Físico JMS

La página Ver Estadísticas de Destino Físico JMS muestra información acerca del uso de un destino físico JMS.

Además del nombre y tipo del destino, la página proporciona información detallada sobre los siguientes:

  • Números de los productores y los consumidores

  • Números y tamaños de mensajes gestionados por el destino

  • Utilización del disco
