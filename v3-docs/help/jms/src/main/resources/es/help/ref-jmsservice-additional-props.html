Propiedades Específicas del Servicio JMS

Las siguientes propiedades de configuración del broker de Oracle Message Queue están disponibles para el servicio JMS.

instance-name

Especifica el nombre completo de la instancia del broker de Message Queue. El valor por defecto es imqbroker.

instance-name-suffix

Especifica un sufijo para agregar al nombre completo de la instancia del broker de Message Queue. El sufijo está separado del nombre de la instancia por un guión de subrayado (_). Por ejemplo, si el nombre de la instancia es imqbroker, al agregar el sufijo xyz, el nombre de la instancia cambia a imqbroker_xyz.

append-version

El valor true agrega los números de versión mayor y menor, precedidos por caracteres de subrayado (_) al nombre completo de la instancia del broker de Message Queue. Por ejemplo, si el nombre de la instancia es imqbroker, al agregar los números de versión, el nombre de la instancia cambia a imqbroker_8_0. El valor por defecto es false.

user-name

El nombre de usuario para crear la conexión JMS. Solamente es necesario si el nombre de usuario o la contraseña de guest/guest por defecto no están disponibles en el broker. El valor por defecto es guest.

password

La contraseña para crear la conexión JMS. Solamente es necesario si el nombre de usuario o la contraseña de guest/guest por defecto no están disponibles en el broker. El valor por defecto es guest.
