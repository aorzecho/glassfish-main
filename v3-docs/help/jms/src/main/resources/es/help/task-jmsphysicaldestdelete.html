Para eliminar un destino físico JMS

No se recomienda eliminar el destino del sistema, mq.sys.dmq.

  1. En el árbol de navegación, amplie el nodo Configuración.
  2. En el nodo Configuración, seleccione y expanda el nodo Java Message Service.
  3. En el nodo Java Message Service, seleccione el nodo Destinos físicos.

    Se abrirá la página Destinos físicos.

  4. En la página Destinos físicos, seleccione la casilla de verificación situada junto al nombre del destino que va a eliminar.
  5. Haga clic en Eliminar.
