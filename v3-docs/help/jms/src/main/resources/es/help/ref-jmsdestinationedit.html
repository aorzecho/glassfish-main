Editar recurso de destino JMS

Utilice la página Editar recurso de destino JMS para modificar los parámetros de configuración de un recurso de destino JMS.

La página Editar recurso de destino JMS contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de destino. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un recurso de destino JMS nuevo.

Nombre de destino físico

El nombre del destino físico asociado a este recurso.

GlassFish Server creará el destino físico automáticamente cuando sea necesario y lo eliminará cuando se elimine el recurso de destino.

Tipo de recurso

El tipo de recurso de destino. Las opciones disponibles son javax.jms.Topic y javax.jms.Queue.

Descripción

Una descripción del recurso de destino.

Estado

El estado del recurso de destino. El recurso de destino se puede habilitar o deshabilitar. Esta opción se habilita por defecto.

Propiedades adicionales

Propiedades adicionales para el recurso de destino. ProductName; no define ninguna propiedad adicional para los recursos de destino JMS.
