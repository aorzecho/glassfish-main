Editar Recurso de Destino JMS

Utilice la página Editar Recurso de Destino JMS para modificar la configuración de un recurso de destino JMS.

La página Editar Recurso de Destino JMS contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre JNDI

Un nombre único que identifica el recurso de destino. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un nuevo recurso de destino JMS.

Nombre de Destino Físico

El nombre del destino físico asociado a este recurso.

Servidor GlassFish creará el destino físico automáticamente cuando sea necesario y lo suprimirá cuando se suprima el recurso de destino.

Tipo de Recurso

El tipo de recurso de destino. Las opciones disponibles son javax.jms.Topic y javax.jms.Queue.

Descripción

Una descripción del recurso de destino.

Estado

El estado del recurso de destino. El recurso de destino se puede activar o desactivar. Esta opción se encuentra activada por defecto.

Propiedades Adicionales

Propiedades adicionales para el recurso de destino. ProductName; no define ninguna propiedad adicional para los recursos de destino JMS.
