Recursos de destino JMS

Utilice la página Recursos de destino JMS para configurar los recursos de destino JMS.

Para cada recurso de destino, aparecerá la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de destino.

Habilitado

True si el recurso está habilitado, o false si no lo está.

Tipo de recurso

El tipo de recurso del recurso de destino.

Descripción

Una descripción del recurso de destino.

La tabla Recursos de destino también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de destino.

Borrar

Botón para eliminar uno o más recursos de destino seleccionados.

Habilitar

Botón para habilitar uno o más recursos de destino seleccionados.

Deshabilitar

Botón para deshabilitar uno o más recursos de destino seleccionados.
