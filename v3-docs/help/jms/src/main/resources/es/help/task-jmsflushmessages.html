Eliminar Mensajes de un Destino Físico JMS

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, amplíe el nodo Java Message Service.

  3. En el nodo Java Message Service, seleccione el nodo Destinos Físicos.

    Se abrirá la página Destinos Físicos.

  4. En la página Destinos Físicos, marque la casilla de verificación situada junto al nombre del destino del que desea eliminar los mensajes.

  5. Haga clic en el botón Suprimir Todos los Mensajes.
