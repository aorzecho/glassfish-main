Para ver las estadísticas para un destino físico JMS

  1. En el árbol de navegación, amplie el nodo Configuración.
  2. En el nodo Configuración, seleccione y expanda el nodo Java Message Service.
  3. En el nodo Java Message Service, seleccione el nodo Destinos físicos.

    Se abrirá la página Destinos físicos.

  4. En la página Destinos físicos, haga clic en el vínculo de vista para el destino cuyas estadísticas desee ver.

    Se abrirá la página Ver estadísticas de destino físico JMS.

  5. Cuando haya terminado de ver las estadísticas, haga clic en el botón Atrás.
