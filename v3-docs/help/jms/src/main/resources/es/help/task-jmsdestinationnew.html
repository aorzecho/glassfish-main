Crear un Recurso de Destino JMS

Cuando se crea un recurso de destino JMS, Servidor GlassFish crea un Recurso de Objeto de Administración para el recurso de destino. Puede visualizar este recurso bajo el nodo Conectores.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos de JMS.

  3. En el nodo Recursos de JMS, seleccione el nodo Recursos de Destino JMS.

    Se abrirá la página Recursos de Destino JMS.

  4. En la página Recursos de Destino JMS, haga clic en Nuevo.

    Se abrirá la página Nuevo Recurso de Destino JMS.

  5. En el campo Nombre JNDI, introduzca el nombre JNDI del recurso.

    Es una práctica recomendada utilizar el prefijo de subcontexto de nomenclatura jms/ para recursos de JMS. Por ejemplo: jms/Queue.

  6. En el campo Nombre de Destino Físico, introduzca el nombre del destino físico que se asociará a este recurso.

    Servidor GlassFish creará el destino físico automáticamente cuando sea necesario y lo suprimirá cuando se suprima el recurso de destino.

  7. En la lista desplegable Tipo de Recurso, seleccione javax.jms.Topic o javax.jms.Queue.

  8. En el campo Descripción, escriba una descripción del recurso de destino.

  9. Seleccione la casilla de verificación de activación de Estado para activar el recurso en tiempo de ejecución.

    Esta opción se encuentra activada por defecto.

  10. En el área de Propiedades Adicionales, indique las propiedades adicionales para el recurso de destino.

    ProductName; no define ninguna propiedad adicional para los recursos de destino JMS.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila vacía, escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.

  11. In the Targets area, do the following:

    1. From the Available column, select the target or targets to which applications that use the resource will be deployed.

      The available targets include the available clusters and server instances, as well as the default server instance, server.

    2. Click Add to move the target to the Selected column.

  12. Haga clic en Aceptar.
