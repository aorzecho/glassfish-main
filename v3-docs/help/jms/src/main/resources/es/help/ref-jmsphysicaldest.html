Destinos físicos JMS

Los agentes de Message Queue conservan los objetos de destino físico de JMS (Java Message Service). La cola llamada mq.sys.dmq es el destino del sistema al que se redireccionan los mensajes caducados y no enviados.

Utilice la página Destinos físicos JMS para configurar los destinos físicos de Message Queue.

Para cada destino físicos, aparecerá la siguiente información.

Nombre

El nombre del destino físico.

Tipo

El tipo del destino físico.

Estadísticas

Las estadísticas del destino físico.

La tabla de destinos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo destino físico.

Borrar

Botón para eliminar uno o más destinos físicos seleccionados.

Vaciar

Botón para retirar mensajes de uno o más destinos físicos seleccionados.
