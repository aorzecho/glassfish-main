Fábricas de Conexiones JMS

Utilice la página Fábricas de Conexiones JMS para configurar recursos de la fábrica de conexiones JMS.

Para cada recurso, se proporciona la siguiente información.

Nombre de Pool

Un nombre único que identifica la fábrica de conexiones.

Nombre JNDI

El nombre JNDI que identifica la fábrica de conexiones.

Activada

"True" si la fábrica de conexiones está activada, y "False" si la fábrica de conexiones está desactivada.

Tipo de Recurso

El tipo de recurso de la fábrica de conexiones.

Descripción

Una descripción de la fábrica de conexiones.

La tabla Fábricas de Conexiones también contiene las siguientes opciones.

Nueva

Botón para crear una nueva fábrica de conexiones.

Suprimir

Botón para suprimir una o varias fábricas de conexiones seleccionadas.

Activar

Botón para activar una o varias fábricas de conexiones seleccionadas.

Desactivar

Botón para desactivar una o varias fábricas de conexiones seleccionadas.
