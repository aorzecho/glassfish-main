Fábricas de conexión JMS

Utilice la página de fábricas de conexión JMS para configurar recursos de fábrica de conexión JMS.

Para cada recurso, aparece la siguiente información.

Nombre de conjunto

Un nombre exclusivo que identifica la fábrica de conexión.

Nombre JNDI

El nombre JNDI que identifica la fábrica de conexión.

Habilitado

"True" si la fábrica de conexión está habilitada, y "False" si la fábrica de conexión está deshabilitada.

Tipo de recurso

El tipo de recurso de la fábrica de conexión.

Descripción

Una descripción de la fábrica de conexión.

La tabla Fábricas de conexión también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva fábrica de conexión.

Borrar

Botón para eliminar una o varias fábricas de conexión seleccionadas.

Habilitar

Botón para habilitar una o varias fábricas de conexión seleccionadas.

Deshabilitar

Botón para deshabilitar una o varias fábricas de conexión seleccionadas.
