Hosts JMS

Utilice la página Hosts JMS para configurar los hosts del servicio JMS.

Para cada host, aparece la siguiente información.

Nombre

El nombre del host JMS.

Host

El nombre o la dirección IP (Internet Protocol) del sistema donde el host JMS se está ejecutando.

Puerto

El número de puerto del servicio JMS en el host.

La tabla de hosts también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo host JMS.

Borrar

Botón para eliminar uno o más hosts JMS seleccionados.
