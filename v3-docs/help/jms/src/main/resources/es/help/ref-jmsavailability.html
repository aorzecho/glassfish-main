Disponibilidad de JMS

Utilice la página Disponibilidad de JMS para configurar las opciones de disponibilidad para el Java Message Service (JMS).

La página Disponibilidad de JMS contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar la configuración que tiene valores por defecto a sus valores por defecto. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Servicio de Disponibilidad

El tipo de cluster de Message Queue utilizado para proporcionar los servicios JMS a un cluster GlassFish. Si se selecciona la casilla de verificación Activado, se utiliza un cluster Message Queue mejorado. De lo contrario, se utiliza el cluster Message Queue convencional.

Si se selecciona la casilla de verificación Activado, se omiten las opciones de Tipo de Almacén de Configuración y Tipo de Almacén de Mensajes.

Tipo de Almacén de Configuración

El tipo de almacén de datos para datos de configuración de un cluster Message Queue convencional. El valor masterbroker especifica el uso de un broker maestro para almacenar y gestionar los datos de configuración. El valor shareddb especifica el uso de una base de datos compartida para almacenar los datos de configuración.

Esta opción se omite si se activa el Servicio de Disponibilidad.

Tipo de Almacén de Mensajes

El tipo de almacén de datos para datos de los mensajes en los brokers de un cluster convencional. El valor file especifica un almacén de archivos. El valor jdbc especifica un almacén JDBC.

Esta opción se omite si se activa el Servicio de Disponibilidad.

URL de Acceso a Base de Datos

El proveedor de la base de datos, el usuario, la contraseña y la dirección url de acceso de la base de datos JDBC a utilizar en cualquiera de estas situaciones:

  • Cuando el Servicio de Disponibilidad está activado

  • Cuando el Tipo de Almacén de Configuración se define como shareddb

  • Cuando Tipo de Almacén de Mensajes se define como jdbc

Para obtener información sobre proveedores compatibles y los formatos de direcciones url de acceso de cada proveedor, consulte la sección sobre persistencia basada en JDBC en la guía Oracle GlassFish Server Message Queue Administration Guide.

Propiedades Adicionales

Propiedades adicionales para la disponibilidad de JMS.

Servidor GlassFish no predefine las propiedades adicionales de disponibilidad de JMS.
