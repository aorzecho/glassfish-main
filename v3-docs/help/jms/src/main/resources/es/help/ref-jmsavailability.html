Disponibilidad de JMS

Utilice la página Disponibilidad de JMS para configurar las opciones de disponibilidad para el Java Message Service (JMS).

La página Disponibilidad JMS contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre de la configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Servicio de disponibilidad

Si se selecciona la casilla de verificación Habilitado, la disponibilidad para el JMS está habilitada.

Nombre de conjunto del almacén de MQ

El nombre JNDI del recurso JDBC que el clúster de agente de Cola de mensajes de GlassFish debe utilizar al guardar los mensajes de JMS persistentes y otra información de la configuración del agente.

El nombre de conjunto puede contener sólo letras, dígitos o caracteres de subrayado (_), guión (-) o punto (.) .

Propiedades adicionales

Propiedades adicionales de la disponibilidad de JMS.

GlassFish Server no predefine ninguna propiedad de disponibilidad de JMS adicional.
