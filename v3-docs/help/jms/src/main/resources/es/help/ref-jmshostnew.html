Nuevo host JMS

Utilice la página Nuevo host JMS para crear para crear un nuevo host JMS.

La página Nuevo host JMS contiene las siguientes opciones.

Nombre de la configuración

El nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

El nombre del host (por ejemplo, NewJmsHost).

Host

El nombre o la dirección IP (Internet Protocol) del sistema donde se ejecutará el host JMS (localhost o el nombre del sistema remoto o local).

Puerto

El número de puerto del servicio JMS. Modifique este campo sólo si el servicio JMS que va a utilizar se ejecuta en un puerto que no es el predeterminado. El puerto predeterminado es 7676.

Nombre de usuario de administrador (Admin) y contraseña de administrador

El nombre de usuario y la contraseña del agente de Message Queue. Éstos son diferentes del nombre de usuario y la contraseña de GlassFish Server. Edite estos campos solamente si los valores del agente intermediario de Message Queue se han modificado mediante el comando as-install-parent/mq/bin/imqusermgr. Los valores predeterminados son admin y admin.
