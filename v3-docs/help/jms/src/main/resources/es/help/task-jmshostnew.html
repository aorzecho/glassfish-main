Para crear un host JMS

  1. En el árbol de navegación, amplie el nodo Configuración.
  2. En el nodo Configuración, seleccione y expanda el nodo Java Message Service.
  3. En el nodo Java Message Service, seleccione el nodo Hosts JMS.

    Se abrirá la página Hosts JMS.

  4. En la página Host JMS, haga clic en Nuevo.

    Se mostrará la página Nuevo host JMS.

  5. En el campo Nombre, introduzca el nombre del host.

    Por ejemplo:

    NewJmsHost
  6. En el campo Host, introduzca el nombre o la dirección IP (Internet Protocol) del sistema donde se ejecutará el host JMS (localhost o el nombre del sistema local o remoto).
  7. En el campo Puerto, introduzca el número de puerto del servicio JMS.

    Modifique este campo sólo si el servicio JMS que va a utilizar se ejecuta en un puerto que no es el predeterminado. El puerto predeterminado es 7676.

  8. En los campos Nombre de usuario de administración y Contraseña de administración, introduzca el nombre de usuario y la contraseña del agente de Message Queue.

    Éstos son diferentes del nombre de usuario y la contraseña de GlassFish Server. Edite estos campos solamente si los valores del agente intermediario de Message Queue se han modificado mediante el comando as-install-parent/mq/bin/imqusermgr. Los valores predeterminados son admin y admin.

  9. Haga clic en Aceptar.
