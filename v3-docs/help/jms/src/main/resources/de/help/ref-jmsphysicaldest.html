Physische JMS-Ziele

Physische Java Message Service (JMS)-Zielobjekte werden von Message Queue-Brokern verwaltet. Die Queue mit dem Namen mq.sys.dmq ist das Systemziel, zu dem abgelaufene und unzustellbare Meldungen umgeleitet werden.

Auf der Seite "Physische JMS-Ziele" können Sie physische Message Queue-Ziele konfigurieren.

Für jedes physische Ziel werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des physischen Ziels.

Typ

Der Typ des physischen Ziels.

Statistiken

Statistische Daten für das physische Ziel.

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen physischen Ziels.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter physischer Ziele.

Alle Meldungen löschen

Schaltfläche zum Entfernen von Meldungen von einem oder mehreren ausgewählten physischen Zielen.

Verwandte Aufgaben

  • So wird der JMS-Service konfiguriert

  • So wird ein physisches JMS-Ziel erstellt

  • So wird ein physisches JMS-Ziel bearbeitet

  • So wird ein physisches JMS-Ziel gelöscht

  • So werden Nachrichten aus einem physischen JMS-Ziel entfernt

  • So werden Statistiken für ein physisches JMS-Ziel angezeigt
