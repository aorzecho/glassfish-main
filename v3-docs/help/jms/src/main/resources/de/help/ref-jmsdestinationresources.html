JMS-Zielressourcen

Auf der Seite "JMS-Zielressourcen" können Sie JMS-Zielressourcen konfigurieren.

Für jede Zielressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die Zielressource identifiziert.

Enabled (Aktiviert)

Wahr, wenn die Ressource aktiviert ist. Falsch, wenn sie deaktiviert ist.

Resource Type (Ressourcen-Typ)

Der Ressourcentyp der Zielressource.

Description (Beschreibung)

Eine Beschreibung der Zielressource.

Die Tabelle "Zielressourcen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Zielressource.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Zielressourcen.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter Zielressourcen.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter Zielressourcen.
