So löschen Sie einen JMS-Host

Es wird nicht empfohlen, den standardmäßigen JMS-Host zu löschen.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "Java Message Service".
  3. Wählen Sie unter dem Knoten "Java Message Service" den Knoten "JMS-Hosts" aus.

    Die Seite "JMS-Hosts" wird geöffnet.

  4. Aktivieren Sie auf der Seite "JMS-Hosts" das Kontrollkästchen neben dem Namen des zu löschenden Hosts.

    Achtung - Löschen Sie nicht den standardmäßigen JMS-Host. Dies kann zu unerwarteten Fehlern führen.

  5. Klicken Sie auf "Löschen".
