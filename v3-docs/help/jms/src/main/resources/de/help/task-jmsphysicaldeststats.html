So zeigen Sie Statistiken für ein physisches JMS-Ziel an

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "Java Message Service".
  3. Wählen Sie unter dem Knoten "Java Message Service" den Knoten "Physikalische Ziele" aus.

    Die Seite "Physikalische Ziele" wird geöffnet.

  4. Klicken Sie auf der Seite "Physikalische Ziele" auf den Link "Anzeigen" für das Ziel, dessen statistische Daten Sie anzeigen möchten.

    Die Seite "Statistik für physische JMS-Ziele anzeigen" wird geöffnet.

  5. Wenn Sie die Statistiken durchgesehen haben, klicken Sie auf die Schaltfläche "Zurück".
