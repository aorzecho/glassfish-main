JMS-Hosts

Auf der Seite "JMS-Hosts" können Sie JMS-Diensthosts konfigurieren.

Für jeden Host werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des JMS-Hosts.

Host

Der Name oder die IP-Adresse (Internet Protocol) des Systems, auf dem der JMS-Host ausgeführt wird.

Port

Die Portnummer des JMS-Diensts auf dem Host.

Die Tabelle "Hosts" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen JMS-Hosts.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter JMS-Hosts.

Verwandte Aufgaben

  • So konfigurieren Sie den JMS-Dienst

  • So erstellen Sie einen JMS-Host

  • So bearbeiten Sie einen JMS-Host

  • So löschen Sie einen JMS-Host

Verwandte asadmin-Befehle

  • create-jms-host(1)

  • list-jms-hosts(1)

  • delete-jms-host(1)
