JMS-Verfügbarkeit

Auf der Seite "JMS-Verfügbarkeit" können Sie Verfügbarkeitseinstellungen für den Java Message Service (JMS) konfigurieren.

Die Seite "JMS-Verfügbarkeit" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Verfügbarkeitsdienst

Wenn das Kontrollkästchen "Aktiviert" aktiviert ist, wird die Verfügbarkeit für den JMS-Dienst aktiviert.

MQ-Speicherpoolname

Der JNDI-Name der JDBC-Ressource, die vom MQ-Broker-Cluster beim Speichern von persistenten JMS-Meldungen und anderen Informationen zur Broker-Konfiguration verwendet wird.

Der Poolname darf nur Buchstaben, Ziffern oder Unterstriche (_), Bindestriche (-) oder Punkte (.) enthalten .

Weitere Eigenschaften

Weitere Eigenschaften für die JMS-Verfügbarkeit.

Bei GlassFish-Server sind keine weiteren Eigenschaften für die JMS-Verfügbarkeit vordefiniert.
