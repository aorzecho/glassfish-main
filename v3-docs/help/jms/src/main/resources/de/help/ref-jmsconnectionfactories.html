JMS-Verbindungsfactories

Auf der Seite "JMS-Verbindungsfactories" können Sie JMS-Verbindungsfactory-Ressourcen konfigurieren.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

Pool Name (Poolname)

Ein eindeutiger Name, der die Verbindungsfactory identifiziert.

JNDI Name (JNDI-Name)

Der JNDI-Name, der die Verbindungsfactory identifiziert.

Aktiviert

"True", wenn die Verbindungsfactory aktiviert ist, bzw. "false", wenn sie nicht aktiviert ist.

Resource Type (Ressourcen-Typ)

Der Ressourcentyp der Verbindungsfactory.

Description (Beschreibung)

Eine Beschreibung der Verbindungsfactory.

Die Tabelle "Verbindungs-Factories" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Verbindungsfactory.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Verbindungsfactories.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter Verbindungsfactories.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter Verbindungsfactories.

Verwandte Aufgaben

  • So erstellen Sie eine JMS-Verbindungsfactory-Ressource

  • So bearbeiten Sie eine JMS-Verbindungsfactory-Ressource

  • So löschen Sie eine JMS-Verbindungsfactory-Ressource

Verwandte asadmin-Befehle

  • create-jms-resource(1)

  • list-jms-resources(1)

  • delete-jms-resource(1)
