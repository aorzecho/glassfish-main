JMS-Connection Factorys

Auf der Seite "JMS-Connection Factorys" können Sie JMS-Connection Factory-Ressourcen konfigurieren.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

Pool Name (Poolname)

Ein eindeutiger Name, der die Connection Factory identifiziert.

JNDI Name (JNDI-Name)

Der JNDI-Name, der die Connection Factory identifiziert.

Aktiviert

"True", wenn die Connection Factory aktiviert ist, bzw. "False", wenn sie nicht aktiviert ist.

Resource Type (Ressourcen-Typ)

Der Ressourcentyp der Connection Factory.

Description (Beschreibung)

Eine Beschreibung der Connection Factory.

Die Tabelle "Connection Factorys" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Connection Factory.

Löschen

Schaltfläche zum Löschen von mindestens einer Connection Factory.

Enable (Aktivieren)

Schaltfläche zum Aktivieren von mindestens einer Connection Factory.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren von mindestens einer Connection Factory.

Verwandte Aufgaben

  • So wird eine JMS-Connection Factory-Ressource erstellt

  • So wird eine JMS-Connection Factory-Ressource bearbeitet

  • So wird eine JMS-Connection Factory-Ressource gelöscht

Zugehörige asadmin-Befehle

  • create-jms-resource(1)

  • list-jms-resources(1)

  • delete-jms-resource(1)
