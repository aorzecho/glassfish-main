So bearbeiten Sie einen JMS-Host

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "Java Message Service".
  3. Wählen Sie unter dem Knoten "Java Message Service" den Knoten "JMS-Hosts" aus.

    Die Seite "JMS-Hosts" wird geöffnet.

  4. Wählen Sie auf der Seite "JMS-Hosts" den zu bearbeitenden Host aus.

    Die Seite "JMS-Host bearbeiten" wird geöffnet.

  5. Geben Sie im Feld "Host" den Namen oder die IP-Adresse (Internet Protocol) des Systems ein, auf dem der JMS-Host ausgeführt wird (localhost oder der Name des lokalen oder Remotesystems).
  6. Geben Sie im Feld "Port" die Portnummer des JMS-Dienstes ein.

    Ändern Sie dieses Feld nur, wenn der JMS-Dienst nicht über den Standardport ausgeführt werden soll. Der Standardport lautet 7676.

  7. Geben Sie in den Feldern "Admin-Benutzername" und "Admin-Passwort" den Benutzernamen und das Passwort für den Message Queue-Broker ein.

    Diese Eingaben sind nicht mit dem GlassFish-Server-Benutzernamen und -Passwort identisch. Bearbeiten Sie diese Felder nur, wenn die Werte des Message Queue-Brokers mit dem Befehl as-install-parent/mq/bin/imqusermgr? geändert wurden. Die Standardwerte sind admin und admin.

  8. Klicken Sie auf "Speichern".

    Tipp - Um zu den Standardeinstellungen zurückzukehren, klicken Sie auf die Schaltfläche "Standardwerte laden" und anschließend auf "Speichern".
