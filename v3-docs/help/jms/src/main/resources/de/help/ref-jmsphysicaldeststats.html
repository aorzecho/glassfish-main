Statistik für physische JMS-Ziele anzeigen

Auf der Seite "Statistik für physische JMS-Ziele anzeigen" werden Informationen über die Verwendung eines physischen JMS-Ziels angezeigt.

Neben Name und Typ des Ziels enthält die Seite detaillierte Informationen zu folgenden Aspekten:

  • Anzahl an Produzenten und Konsumenten

  • Anzahl und Größe der vom Ziel verwalteten Meldungen

  • Datenträgernutzung

Verwandte Aufgaben

  • So konfigurieren Sie den JMS-Dienst

  • So erstellen Sie ein physisches JMS-Ziel

  • So bearbeiten Sie ein physisches JMS-Ziel

  • So löschen Sie ein physisches JMS-Ziel

  • So entfernen Sie Meldungen von einem physischen JMS-Ziel

  • So zeigen Sie Statistiken für ein physisches JMS-Ziel an

Verwandte asadmin-Befehle

  • create-jmsdest(1)

  • list-jmsdest(1)

  • delete-jmsdest(1)

  • flush-jmsdest(1)

  • jms-ping(1)
