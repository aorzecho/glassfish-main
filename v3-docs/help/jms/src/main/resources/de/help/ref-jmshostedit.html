JMS-Host bearbeiten

Auf der Seite "JMS-Host bearbeiten" können Sie die Einstellungen für einen JMS-Host ändern.

Die Seite "JMS-Host bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Name

Der Name des Hosts. Das Feld Name ist ein schreibgeschütztes Feld. Sie können den Namen nur angeben, wenn Sie einen neuen JMS-Host erstellen.

Host

Der Name oder die IP-Adresse (Internet Protocol) des Systems, auf dem der JMS-Host ausgeführt wird (localhost oder der Name des lokalen oder Remotesystems).

Port

Die Portnummer des JMS-Diensts. Ändern Sie dieses Feld nur, wenn der JMS-Dienst nicht über den Standardport ausgeführt werden soll. Der Standardport lautet 7676.

Admin-Benutzername und Admin-Passwort

Der Benutzername und das Passwort für den Message Queue-Broker. Diese Eingaben sind nicht mit dem GlassFish-Server-Benutzernamen und -Passwort identisch. Bearbeiten Sie diese Felder nur, wenn die Werte des Message Queue-Brokers mit dem Befehl as-install-parent/mq/bin/imqusermgr? geändert wurden. Die Standardwerte sind admin und admin.

Verwandte Aufgaben
