JMS-Host bearbeiten

Auf der Seite "JMS-Host bearbeiten" können Sie die Einstellungen für einen JMS-Host ändern.

Die Seite "JMS-Host bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Name

Der Name des Hosts. Das Feld Name ist ein schreibgeschütztes Feld. Sie können den Namen nur angeben, wenn Sie einen neuen JMS-Host erstellen.

Host

Der Name oder die IP-Adresse (Internet Protocol) des Systems, auf dem der JMS-Host ausgeführt wird (localhost oder der Name des lokalen oder Remotesystems).

Port

Die Portnummer des JMS-Service. Ändern Sie dieses Feld nur, wenn der JMS-Service nicht über den Standardport ausgeführt werden soll. Der Standardport lautet 7676.

Admin-Benutzername und Admin-Kennwort

Der Benutzername und das Kennwort für den Message Queue-Broker. Diese Eingaben sind nicht mit dem GlassFish Server-Benutzernamen und -Kennwort identisch. Bearbeiten Sie diese Felder nur, wenn die Werte des Message Queue Brokers mit dem folgenden Befehl geändert wurden. as-install-parent/mq/bin/imqusermgr-Befehl. Die Standardwerte sind admin und admin.

Verwandte Aufgaben

  • So wird der JMS-Service konfiguriert

  • So wird ein JMS-Host erstellt

  • So wird ein JMS-Host bearbeitet

  • So wird ein JMS-Host gelöscht
