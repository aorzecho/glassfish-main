Spezifische Eigenschaften des JMS-Diensts

Für den JMS-Dienst stehen die folgenden Oracle Message Queue-Broker-Konfigurationseigenschaften zur Verfügung.

instance-name

Gibt den vollständigen Namen der Message Queue-Brokerinstanz an. Der Standardname lautet imqbroker.

instance-name-suffix

Gibt ein Suffix an, das zum vollständigen Namen der Message Queue-Brokerinstanz hinzugefügt wird. Das Suffix und der Instanzname werden durch einen Unterstrich (_) getrennt. Wenn der Instanzname z. B. imqbroker lautet, ändert sich dieser Name durch das Hinzufügen des Suffix xyz in imqbroker_xyz.

append-version

Wenn diese Eigenschaft auf true gesetzt ist, werden die Versionsnummern zum vollständigen Namen der Message Queue-Brokerinstanz hinzugefügt; zur Trennung wird ein Unterstrich (_) verwendet. Wenn der Instanzname z. B. imqbroker lautet, ändert sich der Name durch das Hinzufügen der Versionsnummern in imqbroker_8_0. Der Standardwert ist false.

user-name

Der Benutzername zum Herstellen der JMS-Verbindung. Wird nur benötigt, wenn der Standardbenutzername/das Standardpasswort (guest/guest) im Broker nicht verfügbar ist. Der Standardwert ist guest.

password (Passwort)

Das Passwort zum Herstellen der JMS-Verbindung. Wird nur benötigt, wenn der Standardbenutzername/das Standardpasswort (guest/guest) im Broker nicht verfügbar ist. Der Standardwert ist guest.
