Ressources destination JMS

Utilisez la page Ressources destination JMS pour configurer les ressources de destination JMS .

Pour chaque ressource de destination, les informations suivantes sont fournies.

Nom JNDI

Un nom unique qui identifie la ressource de destination.

Activée

True si la ressource est activée ou False si la ressource est désactivée.

Type de ressource

Le type de ressource de la ressource de destination.

Description

Une description de la ressource de destination.

Le tableau Ressources de destination contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle ressource de destination.

Supprimer

Bouton pour supprimer une ou plusieurs ressources de destination sélectionnées.

Activer

Bouton pour activer une ou plusieurs ressources de destination sélectionnées.

Désactiver

Bouton pour désactiver une ou plusieurs ressources de destination sélectionnées.
