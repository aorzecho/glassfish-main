Disponibilité JMS

Utilisez la page Disponibilité JMS pour configurer les paramètres de disponibilité du service JMS (Java Message Service).

La page Disponibilité JMS contient les options suivantes.

Restaurer les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut de tous les paramètres.

Nom de configuration

Le nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Service de disponibilité

Si la case Activé est cochée, la disponibilité est activée pour JMS.

Nom du pool de stockage MQ

Nom JNDI de la ressource JDBC que le cluster du courtier GlassFish Message Queue doit utiliser lors de l'enregistrement des messages JMS persistants et d'autres informations de configuration du courtier.

Le nom du pool ne peut contenir que des lettres, des chiffres, des traits de soulignement (_), des tirets (-) ou des points (.).

Autres propriétés

Propriétés supplémentaires de disponibilité JMS.

GlassFish Server ne prédéfinit aucune autre propriété de disponibilité JMS.
