Modifier une ressource de destination JMS

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le noeud Connecteurs, développez le noeud Ressources JMS.
  3. Sous le noeud Ressources JMS, sélectionnez le noeud Ressources de destination JMS.

    La page Ressources de destination JMS s’ouvre.

  4. Sur la page Ressources de destination JMS, cliquez sur le nom de la ressource de destination que vous modifiez.

    La page Modifier la ressource de destination JMS s’ouvre.

  5. Dans le champ Nom de la destination physique, saisissez le nom de la destination physique à associer à cette ressource.

    GlassFish Server crée la destination physique automatiquement lorsque c’est nécessaire et la supprime lorsque vous supprimez la ressource de destination.

  6. Dans la liste déroulante Type de ressource, sélectionnez javax.jms.Topic ou javax.jms.Queue.
  7. Dans le champ Description, saisissez une description de la ressource de destination.
  8. Sélectionnez la case Activé de Statut pour activer la ressource au moment de l’exécution.

    Cette option est activée par défaut.

  9. (Facultatif) Dans la zone Autres propriétés, spécifiez les autres propriétés pour la ressource de destination.

    GlassFish Server ne définit pas d’autres propriétés pour les ressources de destination JMS.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  10. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
