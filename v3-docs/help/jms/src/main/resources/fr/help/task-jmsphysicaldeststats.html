Affichage des statistiques d’une destination physique JMS

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Java Message Service.
  3. Sous le noeud Java Message Service, sélectionnez le noeud Destinations physiques.

    La page Destinations physiques s’ouvre.

  4. Sur la page Destinations physiques, cliquez sur le lien Afficher de la destination dont vous souhaitez consulter les statistiques.

    La page Affichage des statistiques de la destination physique JMS s’ouvre.

  5. Une fois que vous avez consulté les statistiques, cliquez sur le bouton Précédent.
