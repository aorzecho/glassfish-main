Afficher les statistiques d'une destination physique JMS

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Java Message Service.

  3. Sous le noeud Java Message Service, sélectionnez le noeud Destinations physiques.

    La page Destinations physiques apparaît.

  4. Sur la page Destinations physiques, cliquez sur le lien Afficher de la destination dont vous souhaitez consulter les statistiques.

    La page Afficher les statistiques de la destination physique JMS apparaît.

  5. Une fois que vous avez consulté les statistiques, cliquez sur le bouton Précédent.
