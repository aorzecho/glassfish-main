Création d’un hôte JMS

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Java Message Service.
  3. Sous le noeud Java Message Service, sélectionnez le noeud Hôtes JMS.

    La page Hôtes JMS s’ouvre.

  4. Sur la page Hôtes JMS, cliquez sur Nouveau.

    La page Nouvel hôte JMS apparaît.

  5. Dans le champ Nom, saisissez le nom de l’hôte.

    Exemple :

    NewJmsHost
  6. Dans le champ Hôte, saisissez le nom ou l’adresse IP du système sous lequel l’hôte JMS fonctionnera (localhost ou le nom du système local ou distant).
  7. Dans le champ Port, indiquez le numéro de port du service JMS.

    Ne modifiez ce champ que si le service JMS à utiliser n’utilise pas le port par défaut. Le port par défaut est 7676.

  8. Dans les champs Nom d’utilisateur Admin et Mot de passe Admin, saisissez le nom d’utilisateur et le mot de passe du courtier Message Queue.

    Ceux-ci diffèrent du nom d’utilisateur et du mot de passe du GlassFish Server. Modifiez ces champs uniquement si les valeurs du courtier Message Queue ont été modifiées à l’aide de la commande as-install-parent/mq/bin/imqusermgr. Les valeurs par défaut sont admin et admin.

  9. Cliquez sur OK.
