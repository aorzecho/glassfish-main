Destinations physiques JMS

Les objets de destination physique JMS (Java Message Service) sont gérés par les courtiers Message Queue. La file d’attente nommée mq.sys.dmq est la destination système vers laquelle les messages parvenus à expiration et non livrables sont réacheminés.

Utilisez la page Destinations physiques JMS pour configurer des destinations physiques Message Queue.

Pour chaque ressource physique, les informations suivantes sont fournies.

Nom

Le nom de la destination physique.

Type

Le type de la destination physique.

Statistiques

Les statistiques pour la destination physique.

Le tableau Destinations contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle destination physique.

Supprimer

Bouton pour supprimer une ou plusieurs destinations physiques sélectionnées.

Vider

Bouton pour supprimer les messages d’une ou de plusieurs destinations physiques sélectionnées.
