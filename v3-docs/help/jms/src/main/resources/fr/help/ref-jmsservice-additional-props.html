Propriétés spécifiques au service JMS

Les propriétés suivantes de configuration du courtier Message Queue de Oracle sont disponibles pour le service JMS.

instance-name

Indique le nom complet de l’instance du courtier Message Queue. La valeur par défaut est imqbroker.

instance-name-suffix

Spécifie un suffixe à ajouter au nom complet de l’instance du courtier Message Queue. Le suffixe est séparé du nom de l’instance par un tiret (_). Par exemple, si le nom de l’instance est imqbroker, l’ajout du suffixe xyz donne le nom imqbroker_xyz.

append-version

Si la valeur est true, ajoute au nom d’instance du courtier Message Queue les numéros de version minimum et maximum, précédés par des tirets (_). Par exemple, si le nom de l’instance est imqbroker, l’ajout des numéros de version donne le nom imqbroker_8_0 . La valeur par défaut est faux.

user-name

Le nom d’utilisateur pour créer la connexion JMS. Requis uniquement si le nom d’utilisateur ou le mot de passe par défaut de guest/guest n’est pas disponible dans le courtier. La valeur par défaut est guest.

password

Le mot de passe pour créer la connexion JMS. Requis uniquement si le nom d’utilisateur ou le mot de passe par défaut de guest/guest n’est pas disponible dans le courtier. La valeur par défaut est guest.
