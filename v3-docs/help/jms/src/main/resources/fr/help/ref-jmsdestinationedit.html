Modifier une ressource de destination JMS

La page Modifier une ressource de destination JMS permet de modifier les paramètres d'une ressource de destination JMS.

Elle contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom JNDI

Nom unique identifiant la ressource de destination. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource de destination JMS.

Nom de la destination physique

Nom de la destination physique associée à cette ressource.

GlassFish Server crée la destination physique automatiquement lorsque cela est nécessaire et la supprime lorsque vous supprimez la ressource de destination.

Type de ressource

Type de la ressource de destination. Les options disponibles sont javax.jms.Topic et javax.jms.Queue.

Description

Description de la ressource de destination.

Statut

Statut de la ressource de destination. La ressource de destination peut être activée ou désactivée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource de destination. GlassFish Server ne définit aucune autre propriété pour les ressources de destination JMS.
