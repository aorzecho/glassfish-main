Modification d’une ressource destination JMS

Utilisez la page Modification d’une ressource destination JMS pour modifier les paramètres pour une ressource de destination JMS.

La page Modification d’une ressource destination JMS contient les options suivantes.

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom JNDI

Un nom unique qui identifie la ressource de destination. Le champ Nom JNDI est en lecture seule. Vous pouvez spécifier le nom JNDI uniquement lorsque vous créez une nouvelle ressource de destination JMS.

Nom de la destination physique

Le nom de la destination physique associée à cette ressource.

GlassFish Server crée la destination physique automatiquement lorsque c’est nécessaire et la supprime lorsque vous supprimez la ressource de destination.

Type de ressource

Le type de la ressource de destination. Les choix disponibles sont javax.jms.Topic et javax.jms.Queue.

Description

Une description de la ressource de destination.

Statut

Le statut de la ressource de destination. La ressource de destination peut être activée ou désactivée. Cette option est activée par défaut.

Autres propriétés

Autres propriétés pour la ressource de destination. GlassFish Server ne définit aucune autre propriété pour les ressources de destination JMS.
