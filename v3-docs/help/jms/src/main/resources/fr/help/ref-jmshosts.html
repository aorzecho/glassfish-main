Hôtes JMS

Utilisez la page Hôtes JMS pour configurer les hôtes du service JMS.

Pour chaque hôte, les informations suivantes sont fournies.

Nom

Le nom de l’hôte JMS.

Hôte

Le nom ou l’adresse IP du système sur lequel l’hôte JMS est en cours d’exécution.

Port

Le numéro de port du service JMS sur l’hôte.

Le tableau Hôtes contient également les options suivantes.

Nouveau

Bouton pour créer un nouvel hôte JMS.

Supprimer

Bouton pour supprimer un ou plusieurs hôtes JMS sélectionnés.
