Hôtes JMS

La page Hôtes JMS permet de configurer les hôtes du service JMS.

Pour chaque hôte, les informations suivantes sont fournies.

Nom

Nom de l'hôte JMS.

Hôte

Nom ou adresse IP du système sur lequel l'hôte JMS est exécuté.

Port

Numéro de port du service JMS sur l'hôte.

La table Hôtes contient également les options suivantes.

Nouveau

Bouton permettant de créer un hôte JMS.

Supprimer

Bouton permettant de supprimer les hôtes JMS sélectionnés.
