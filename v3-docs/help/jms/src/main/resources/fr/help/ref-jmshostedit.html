Modifier un hôte JMS

La page Modifier un hôte JMS permet de modifier les paramètres d'un hôte JMS.

Elle contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom

Nom de l'hôte. Le champ Nom est en lecture seule. Vous ne pouvez spécifier le nom qu'à la création d'un hôte JMS.

Hôte

Nom ou adresse IP du système sur lequel l'hôte JMS sera exécuté (localhost ou le nom du système local ou distant).

Port

Numéro de port du service JMS. Ne modifiez ce champ que si le service JMS à utiliser n'est pas exécuté sur le port par défaut. Le port par défaut est 7676.

Nom utilisateur de l'administrateur et Mot de passe de l'administrateur

Nom utilisateur et mot de passe du broker Message Queue. Ceux-ci diffèrent du nom utilisateur et du mot de passe GlassFish Server. Ne modifiez ces champs que si les valeurs du broker Message Queue ont été modifiées à l'aide de la commande as-install-parent/mq/bin/imqusermgr. Les valeurs par défaut sont admin et admin.
