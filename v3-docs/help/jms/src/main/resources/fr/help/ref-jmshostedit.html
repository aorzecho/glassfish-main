Modification d’un hôte JMS

Utilisez la page Modification d’un hôte JMS pour modifier les paramètres d’un hôte JMS.

La page Modification d’un hôte JMS contient les options suivantes.

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Nom

Le nom de l’hôte. Le champ Nom est en lecture seule. Vous pouvez spécifier le champ Nom uniquement lorsque vous créez un nouvel hôte JMS.

Hôte

Le nom ou l’adresse IP du système sur lequel l’hôte JMS s’exécutera (localhost ou le nom du système local ou distant).

Port

Le numéro de port du service JMS. Ne modifiez ce champ que si le service JMS à utiliser n’utilise pas le port par défaut. Le port par défaut est 7676.

Nom d’utilisateur admin et Mot de passe Admin

Le nom d’utilisateur et le mot de passe du courtier Message Queue. Ceux-ci diffèrent du nom d’utilisateur et du mot de passe du GlassFish Server. Modifiez ces champs uniquement si les valeurs du courtier Message Queue ont été modifiées à l’aide de la commande as-install-parent/mq/bin/imqusermgr. Les valeurs par défaut sont admin et admin.
