Fabriques de connexion JMS

Utilisez la page Usines de connexion JMS pour configurer les ressources de l’usine de connexion JMS.

Pour chaque ressource, les informations suivantes sont fournies.

Nom du pool

Un nom unique qui identifie l’usine de connexion.

Nom JNDI

Nom JNDI unique identifiant la fabrique de connexion.

Activée

True si l’usine de connexion est activée ou false si l’usine de connexion est désactivée.

Type de ressource

Le type de ressource de l’usine de connexion.

Description

Une description de l’usine de connexion.

Le tableau Usines de connexion contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle usine de connexion.

Supprimer

Bouton pour supprimer une ou plusieurs usines de connexion sélectionnées.

Activer

Bouton pour activer une ou plusieurs usines de connexion sélectionnées.

Désactiver

Bouton pour désactiver une ou plusieurs usines de connexion sélectionnées.
