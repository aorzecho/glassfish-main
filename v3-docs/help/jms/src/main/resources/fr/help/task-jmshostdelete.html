Suppression d’un hôte JMS

Il est recommandé de ne pas supprimer l’hôte JMS par défaut.

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Java Message Service.
  3. Sous le noeud Java Message Service, sélectionnez le noeud Hôtes JMS.

    La page Hôtes JMS s’ouvre.

  4. Sur la page Hôtes JMS, sélectionnez la case en regard du nom de l’hôte que vous souhaitez supprimer.

    Attention - Attention à ne pas supprimer l’hôte JMS par défaut. Cela pourrait entraîner des erreurs inattendues.

  5. Cliquez sur Supprimer.
