Supprimer un hôte JMS

Il est recommandé de ne pas supprimer l'hôte JMS par défaut.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. Sous le noeud Configuration, développez le noeud Java Message Service.

  4. Sous le noeud Java Message Service, sélectionnez le noeud Hôtes JMS.

    La page Hôtes JMS apparaît.

  5. Sur la page Hôtes JMS, sélectionnez la case en regard du nom de l'hôte à supprimer.

    Attention:

    Ne supprimez pas l'hôte JMS par défaut. Cela pourrait entraîner des erreurs inattendues.

  6. Cliquez sur Supprimer.
