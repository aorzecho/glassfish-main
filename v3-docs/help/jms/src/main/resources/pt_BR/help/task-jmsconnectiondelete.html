Para Deletar um Recurso da Fábrica de Conexões JMS

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó Recursos JMS.

  3. No nó Recursos JMS, selecione o nó Fábricas de Conexões JMS.

    A página Fábricas de Conexões JMS é aberta.

  4. Na página Fábricas de Conexões JMS, marque a caixa de seleção ao lado do nome da fábrica de conexões que está sendo deletada.

  5. Clique em Deletar.
