Hosts de JMS

Utilize a página Hosts JMS para configurar hosts do serviço JMS.

Para cada aplicativo, as informações a seguir são fornecidas.

Nome

O nome do host do JMS.

Host

O nome ou o endereço do Protocolo Internet (IP) do sistema que o host do JMS está sendo executado.

Porta

O número da porta do serviço JMS no host.

O quadro Hosts também contém as opções a seguir.

Novo

Botão utilizado para criar um novo host do JMS.

Excluir

Botão utilizado para excluir um ou mais hosts do JMS selecionados.
