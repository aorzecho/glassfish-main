Para editar um host do JMS

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó do Java Message Service.
  3. No nó do Java Message Service, selecione o nó Hosts do JMS.

    A página Hosts do JMS é aberta.

  4. Na página Hosts do JMS, selecione o host que está sendo editado.

    A página Editar host do JMS é aberta.

  5. No campo Host, digite o nome ou o endereço do Protocolo Internet (IP) do sistema em que o host do JMS será executado (localhost ou o nome do sistema local ou remoto).
  6. No campo Porta, digite o número da porta do serviço JMS.

    Altere esse campo somente se o serviço JMS a ser utilizado estiver sendo executado em uma porta não padrão. A porta padrão é 7676.

  7. Nos campos Nome de usuário e Senha Admin, digite o nome de usuário e a senha do broker da Fila de mensagens.

    São diferentes do nome de usuário e da senha do Servidor GlassFish. Edite esses campos somente se os valores do broker da Fila de mensagens foram alterados com o comando as-install-parent/mq/bin/imqusermgr. Os valores padrão são admin e admin.

  8. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e depois em Salvar.
