Disponibilidade do JMS

Use a página de Disponibilidade do JMS para configurar definições de disponibilidade para o Java Message Service (JMS).

A página de Disponibilidade do JMS contém as seguintes opções.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Serviço de Disponibilidade

O tipo de cluster do Message Queue usado para oferecer serviços JMS a um cluster do GlassFish. Se a caixa de seleção Ativado for marcada, um cluster Message Queue será utilizado. Caso contrário, um cluster Message Queue será utilizado.

Se a caixa de seleção Ativado for marcada, as opções Tipo de Armazenamento de Configurações e Tipo de Armazenamento de Mensagens serão ignoradas.

Tipo de Armazenamento de Configurações

O tipo de armazenamento de dados para dados de configuração em um cluster Message Queue convencional. O valor masterbroker especifica o uso de um broker master para armazenar e gerenciar os dados da configuração. O valor shareddb especifica o uso de um banco de dados compartilhado para armazenar os dados de configuração.

Essa opção será ignorada se o Serviço de Disponibilidade for desativado.

Tipo de Armazenamento de Mensagens

O tipo de armazenador de dados para os dados de mensagens em um cluster convencional. O valor file especifica um armazenador de arquivos. O valor jdbc especifica um armazenador JDBC.

Essa opção será ignorada se o Serviço de Disponibilidade for desativado.

Fornecedor do Banco de Dados
Nome de Usuário do Banco de Dados
Senha do Banco de Dados
URL de Acesso do Banco de Dados

O fornecedor do banco de dados, o usuário, a senha e o url de acesso do banco de dados JDBC para usar em uma dessas situações:

  • Quando o Serviço de Disponibilidade é ativado.

  • Quando o Tipo de Armazenamento de Configurações é definido para shareddb

  • Quando o Tipo de Armazenamento de Mensagens é definido para jdbc

Para obter informações sobre fornecedores e os formatos dos urls de acesso para cada fornecedor, consulte Persistência baseada em JDBC no Oracle GlassFish Server Message Queue Administration Guide.

Propriedades adicionais para disponibilidade do JMS.
