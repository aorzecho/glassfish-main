Disponibilidade de JMS

Utilize a página Disponibilidade de JMS para definir as configurações de disponibilidade do Java Message Service (JMS).

A página Disponibilidade de JMS contém as opções a seguir.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Serviço de disponibilidade

Se a caixa de seleção Ativo estiver marcada, a disponibilidade está ativada para o JMS.

Nome do grupo de armazenamento de MQ

Nome JNDI do recurso JDBC que o cluster do broker de Fila de mensagens do GlassFish deve utilizar ao salvar mensagens JMS persistentes e outras informações de configuração do broker.

O nome do conjunto pode conter somente, números ou os caracteres sublinhado (_), hífen (-) ou ponto (.).

Propriedades adicionais

Propriedades adicionais da disponibilidade do JMS.

O Servidor GlassFish não define propriedades adicionais da disponibilidade do JMS.
