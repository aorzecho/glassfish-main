Editar host do JMS

Utilize a página Editar host do JMS para modificar as configurações de um host do JMS.

A página Editar host do JMS contém as opções a seguir.

Carregar padrão

Botão para restaurar os valores padrão de todas as configurações.

Nome da configuração

Nome da configuração ao qual as configurações desta página serão enviadas. Este campo é de somente leitura.

Nome

O nome do host. O campo Nome é de somente leitura. Você pode especificar somente o campo Nome ao criar um novo host do JMS.

Host

O nome ou o endereço do Protocolo Internet (IP) do sistema em que o host do JMS executará (localhost ou o nome do sistema local ou remoto).

Porta

O número da porta do serviço JMS. Altere esse campo somente se o serviço JMS a ser utilizado estiver sendo executado em uma porta não padrão. A porta padrão é 7676.

Nome de usuário e senha Admin

O nome de usuário e a senha do broker da Fila de mensagens. São diferentes do nome de usuário e da senha do Servidor GlassFish. Edite esses campos somente se os valores do broker da Fila de mensagens foram alterados com o comando as-install-parent/mq/bin/imqusermgr. Os valores padrão são admin e admin.
