Recursos de Destinação do JMS

Utilize a página Recursos de destino JMS para configurar os recursos de destino JMS.

Para cada recurso de destino, as seguintes informações são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso de destino.

Ativado

Verdadeiro se o recurso estiver ativo ou falso se o recurso estiver desativado.

Tipo de recurso

O tipo de recurso do recurso de destino.

Descrição

Uma descrição do recurso de destino.

O quadro Recursos de destino também contém as opções a seguir.

Novo

Botão utilizado para criar um novo recurso de destino.

Deletar

Botão utilizado para deletar um ou mais recursos de destino selecionados.

Ativar

Botão utilizado para ativar um ou mais recursos de destino selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos de destino selecionados.
