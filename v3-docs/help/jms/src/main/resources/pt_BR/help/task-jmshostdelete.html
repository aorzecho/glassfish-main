Para Deletar um Host de JMS

Não é recomendável a deleção do host default JMS.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. No nó Configuração, expanda o nó do Java Message Service.

  4. No nó do Java Message Service, selecione o nó Hosts JMS.

    A página Hosts JMS é aberta.

  5. Na página Hosts JMS, marque a caixa de seleção ao lado do nome do host que está sendo deletado.

    Cuidado:

    Não delete o host default JMS. Isso causaria erros inesperados.

  6. Clique em Deletar.
