Para excluir um host do JMS

Não é recomendável a exclusão do host padrão do JMS.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó do Java Message Service.
  3. No nó do Java Message Service, selecione o nó Hosts do JMS.

    A página Hosts do JMS é aberta.

  4. Na página Hosts do JMS, marque a caixa de seleção ao lado do nome do host que está sendo excluído.

    Cuidado - Não exclua o host padrão do JMS. Isso causaria erros inesperados.

  5. Clique em Excluir.
