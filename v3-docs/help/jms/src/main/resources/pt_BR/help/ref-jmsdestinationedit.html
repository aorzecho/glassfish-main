Editar recurso de destino JMS

Utilize a página Editar recurso de destino JMS para modificar as configurações de um recurso de destino JMS.

A página Editar recurso de destino JMS contém as opções a seguir.

Carregar padrões

Botão para restaurar os valores padrão de todas as configurações.

Nome JNDI

Um nome exclusivo que identifica o recurso de destino. O nome JNDI é um campo de somente leitura. Você pode especificar o nome JNDI somente quando cria um novo recurso de destino JMS.

Nome de destino físico

O nome do destino físico associado a esse recurso.

O Servidor GlassFish cria um destino físico automaticamente quando necessário e o exclui quando você exclui o recurso de destino.

Tipo de recurso

O tipo do recurso de destino. As escolhas disponíveis são javax.jms.Topic e javax.jms.Queue.

Descrição

Uma descrição do recurso de destino.

Status

O status do recurso de destino. O recurso de destino pode ser ativado ou desativado. Por padrão, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso de destino. O Servidor GlassFish não define quaisquer propriedades adicionais para recursos de destino JMS.
