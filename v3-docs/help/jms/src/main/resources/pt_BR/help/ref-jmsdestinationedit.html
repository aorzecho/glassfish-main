Editar Recurso de Destinação do JMS

Utilize a página Editar recurso de destino JMS para modificar as definições de um recurso de destino JMS.

A página Editar recurso de destino JMS contém as opções a seguir.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome JNDI

Um nome exclusivo que identifica o recurso de destino. O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI somente quando cria um novo recurso de destino JMS.

Nome de destino físico

O nome do destino físico associado a esse recurso.

O GlassFish Server cria um destino físico automaticamente quando necessário e o deleta quando você deleta o recurso de destino.

Tipo de recurso

O tipo do recurso de destino. As escolhas disponíveis são javax.jms.Topic e javax.jms.Queue.

Descrição

Uma descrição do recurso de destino.

Status

O status do recurso de destino. O recurso de destino pode ser ativado ou desativado. Por default, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso de destino. O GlassFish Server não define quaisquer propriedades adicionais para recursos de destino JMS.
