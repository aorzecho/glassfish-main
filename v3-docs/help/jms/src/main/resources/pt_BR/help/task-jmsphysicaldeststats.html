Para Exibir Estatísticas de um Destino Físico do JMS

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó do Java Message Service.

  3. No nó Java Message Service, selecione o nó Destinos físicos.

    A página Destinos físicos é aberta.

  4. Na página Destinos físicos, clique no link Exibir para o destino para o qual você está visualizando a estatística.

    A página Visualizar estatísticas de destino físico JMS é aberta.

  5. Quando finalizar a exibição da estatística, clique no botão Voltar.
