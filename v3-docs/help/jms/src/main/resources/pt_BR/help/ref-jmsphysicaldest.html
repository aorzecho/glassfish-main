Destinos físicos do JMS

Os objetos de destino físico JMS (Java Message Service) são mantidos pelos brokers da Fila de mensagens. A fila nomeada mq.sys.dmq é o destino do sistema para o qual mensagens expiradas ou não entregues são redirecionadas.

Utilize a página Destinos físicos do JMS para configurar destinos físicos da Fila de mensagens.

Para cada destino físico, as informações a seguir são fornecidas.

Nome

O nome do destino físico.

Tipo

O tipo do destino físico.

Estatística

Estatística do destino físico.

O quadro Destinos também contêm as opções a seguir.

Novo

Botão utilizado para criar um novo destino físico.

Excluir

Botão para excluir um ou mais destinos físicos selecionados.

Esvaziamento

Botão para remover mensagens de um ou mais destinos físicos selecionados.
