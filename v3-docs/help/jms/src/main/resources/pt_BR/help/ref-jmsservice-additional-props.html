Propriedades específicas do serviço JMS

As propriedades de configuração do broker da Fila de mensagens do Oracle a seguir estão disponíveis para o serviço JMS.

instance-name

Especifica o nome da instância do broker da Fila de mensagens. O padrão é imqbroker.

instance-name-suffix

Especifica um sufixo a ser adicionado ao nome completo da instância do broker da Fila de mensagens. O sufixo é separado do nome da instância por um caractere de sublinhado (_). Por exemplo, se a instância do nome for imqbroker, colocar o sufixo xyz altera o nome da instância para imqbroker_xyz.

append-version

Se verdadeiro, adiciona os números de versão maiores e menores, precedidos de caracteres de sublinhado (_), ao nome completo da instância do broker da Fila de mensagens. Por exemplo, se o nome da instância for imqbroker, a adição dos números de versão altera o nome da instância para imqbroker_8_0. O padrão é falso.

user-name

O nome de usuário para a criação da conexão JMS. Necessário somente se nome de usuário/senha de guest/guest não estiver disponível no broker. O valor padrão é guest.

password

A senha para a criação da conexão JMS. Necessário somente se nome de usuário/senha deguest/guest não estiver disponível no broker. O valor padrão é guest.
