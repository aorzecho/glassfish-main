Para remover mensagens de um destino físico do JMS

  1. Na árvore de navegação, expanda o nó Configuração.
  2. No nó Configuração, expanda o nó do Java Message Service.
  3. No nó Java Message Service, selecione o nó Destinos físicos.

    A página Destinos físicos é aberta.

  4. Na página Destinos físicos, marque a caixa de seleção ao lado do nome do destino do qual as mensagens devem ser excluídas.
  5. Clique no botão Excluir todas as mensagens.
