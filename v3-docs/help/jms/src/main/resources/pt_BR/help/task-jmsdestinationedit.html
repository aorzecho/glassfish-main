Para editar um recurso de destino do JMS

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó Recursos do JMS.
  3. No nó Recursos do JMS, selecione o nó Recursos de destino do JMS.

    A página Recursos de destino do JMS é aberta.

  4. Na página Recursos de destino do JMS, clique no nome do recurso de destino que está sendo editado.

    A página página Editar recursos de destino do JMS é aberta.

  5. No campo Nome do destino físico, digite o nome do destino físico associado a esse recurso.

    O Servidor GlassFish cria um destino físico automaticamente quando necessário e o exclui quando você exclui o recurso de destino.

  6. Na lista suspensa Tipo de recurso, selecione javax.jms.Topic ou javax.jms.Queue.
  7. No campo Descrição, digite uma descrição do recurso de destino.
  8. Marque a caixa de seleção Status ativo para ativar o recurso no tempo de execução.

    Por padrão, a opção está ativada.

  9. (Opcional) Na seção Propriedades adicionais, especifique propriedades adicionais do servidor de destino.

    O Servidor GlassFish não define propriedades adicionais para recursos de destino do JMS.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  10. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e depois em Salvar.
