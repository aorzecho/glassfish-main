Fábricas de conexões JMS

Utilize a página Fábricas de conexões JMS para configurar recursos da fábrica de conexões JMS.

Para cada recurso, as informações abaixo são fornecidas.

Nome do conjunto

Um nome exclusivo que identifica a fábrica de conexões.

Nome JNDI

Um nome JNDI que identifica a fábrica de conexões.

Ativo

Verdadeiro se a fábrica de conexões estiver ativada, ou falso se a fábrica de conexões estiver desativada.

Tipo de recurso

O tipo de recurso da fábrica de conexões.

Descrição

Uma descrição da fábrica de conexões.

O quadro Fábricas de conexões também contém as opções a seguir.

Novo

Botão para criar uma nova fábrica de conexões.

Excluir

Botão para excluir uma ou mais fábricas de conexões selecionadas.

Ativar

Botão para ativar uma ou mais fábricas de conexões selecionadas.

Desativar

Botão para desativar uma ou mais fábricas de conexões selecionadas.
