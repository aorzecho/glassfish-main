Para excluir um recurso de destino do JMS

  1. Na árvore de navegação, expanda o nó Recursos.
  2. No nó Recursos, expanda o nó Recursos do JMS.
  3. No nó Recursos do JMS, selecione o nó Recursos de destino do JMS.

    A página Recursos de destino do JMS é aberta.

  4. Na página Recursos de destino do JMS, marque a caixa de seleção ao lado nome do recurso de destino que está sendo excluído.
  5. Clique em Excluir.
