JMS 物理送信先の統計情報の表示

「JMS 物理送信先の統計情報の表示」ページには JMS 物理送信先の使用状況に関する情報が表示されます。

送信先の名前とタイプのほかに、次の詳細な情報が表示されます。

  • プロデューサとコンシューマの数

  • 送信先で処理されたメッセージの数とサイズ

  • ディスクの使用率
