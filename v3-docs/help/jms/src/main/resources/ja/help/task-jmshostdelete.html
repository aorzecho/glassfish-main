JMS ホストを削除する

デフォルトの JMS ホストを削除することはお勧めできません。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードで、「Java メッセージサービス」ノードを展開します。
  3. 「Java メッセージサービス」ノードで、「JMS ホスト」ノードを選択します。

    「JMS ホスト」ページが開きます。

  4. 「JMS ホスト」ページで、削除するホストの名前の隣にあるチェックボックスを選択します。

    注意 - デフォルトの JMS ホストは削除しないでください。削除すると、予期しないエラーが起きることがあります。

  5. 「削除」をクリックします。
