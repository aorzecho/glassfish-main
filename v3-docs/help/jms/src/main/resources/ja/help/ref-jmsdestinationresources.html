JMS 送信先リソース

JMS 送信先リソースを設定するには、「JMS 送信先リソース」ページを使用します。

送信先リソースごとに、次の情報が表示されます。

JNDI 名

送信先リソースを識別する一意の名前。

使用可能

リソースが使用可能な場合は true、使用不可の場合は false です。

リソースタイプ

送信先リソースのリソースタイプ。

説明

送信先リソースの説明。

「送信先リソース」の表には、次のオプションも表示されます。

新規

新しい送信先リソースを作成するボタン。

削除

選択した 1 つまたは複数の送信先リソースを削除するボタン。

有効

選択した 1 つまたは複数の送信先リソースを有効にするボタン。

無効

選択した 1 つまたは複数の送信先リソースを無効にするボタン。

関連操作

  • 「JMS 送信先リソースを作成する」

  • 「JMS 送信先リソースを編集する」

  • 「JMS 送信先リソースを削除する」

関連する asadmin コマンド

  • create-jms-resource(1)

  • list-jms-resources(1)

  • delete-jms-resource(1)
