JMS宛先リソース

JMS宛先リソースを構成するには、「JMS宛先リソース」ページを使用します。

宛先リソースごとに、次の情報が表示されます。

JNDI名

宛先リソースを識別する一意の名前。

有効

リソースが使用可能な場合はtrue、使用不可の場合はfalseです。

リソース・タイプ

宛先リソースのリソース・タイプ。

説明

宛先リソースの説明。

「宛先リソース」の表には、次のオプションも表示されます。

新規

新しい宛先リソースを作成するボタン。

削除

選択した1つまたは複数の宛先リソースを削除するボタン。

有効

選択した1つまたは複数の宛先リソースを有効にするボタン。

無効

選択した1つまたは複数の宛先リソースを無効にするボタン。
