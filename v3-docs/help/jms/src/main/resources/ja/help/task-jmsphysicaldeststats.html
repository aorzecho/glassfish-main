JMS 物理送信先の統計情報を表示する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードで、「Java メッセージサービス」ノードを展開します。
  3. 「Java メッセージサービス」ノードで、「物理送信先」ノードを選択します。

    「物理送信先」ページが開きます。

  4. 「物理送信先」ページで、統計情報を表示する送信先の「表示」のリンクをクリックします。

    「JMS 物理送信先の統計情報の表示」ページが開きます。

  5. 統計情報の確認が終わったら、「戻る」ボタンをクリックします。
