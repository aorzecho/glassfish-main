JMS 送信先リソースを編集

JMS 送信先リソースの設定を変更するには、「JMS 送信先リソースを編集」ページを使用します。

「JMS 送信先リソースを編集」ページには、次のオプションが含まれます。

デフォルトを読込み

すべての設定の値をデフォルトに戻すボタン。

JNDI 名

送信先リソースを識別する一意の名前。JNDI 名は読み取り専用フィールドです。JNDI 名を指定できるのは、新しい JMS 送信先リソースの作成時のみです。

物理送信先名

このリソースに関連付けられた物理送信先の名前。

GlassFish サーバー は、物理送信先を必要なときに自動的に作成し、送信先リソースを削除するときに自動的に削除します。

リソースタイプ

送信先リソースのタイプ。指定できるタイプは、javax.jms.Topic と javax.jms.Queue です。

説明

送信先リソースの説明。

ステータス

送信先リソースのステータス。送信先リソースは、有効または無効の状態になります。このオプションはデフォルトで有効になっています。

追加プロパティー

送信先リソースの追加プロパティー。GlassFish サーバー は JMS 送信先リソースの追加プロパティーを定義しません。
