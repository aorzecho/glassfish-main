JMS の可用性

「JMS の可用性」ページを使用し、JMS (Java メッセージサービス) の可用性を設定します。

「JMS の可用性」ページには、次のオプションが含まれます。

デフォルトを読込み

すべての設定の値をデフォルトに戻すボタン。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

可用性サービス

「有効」チェックボックスを選択すると、JMS で可用性が有効になります。

MQ ストアプール名

持続 JMS メッセージとそのほかのブローカの設定情報の保存時に、GlassFish メッセージキュー ブローカクラスタで使用される JDBC リソースの JNDI 名。

プール名に使用できるのは、文字、数字、下線 (_)、ダッシュ (-)、ピリオド (.) だけです。

追加プロパティー

JMS の可用性の追加プロパティー。

GlassFish サーバー では、JMS の可用性の追加プロパティーは事前定義されません。
