JMS物理宛先

JMS(Java Message Service)物理宛先オブジェクトは、Message Queueブローカによって管理されます。mq.sys.dmqという名前のキューはシステム宛先であり、期限切れメッセージおよび配信不能メッセージがリダイレクトされます。

Message Queue物理宛先を構成するには、「JMS物理宛先」ページを使用します。

物理宛先ごとに、次の情報が表示されます。

名前

物理宛先の名前。

タイプ

物理宛先のタイプ。

統計

物理宛先の統計。

「宛先」の表には、次のオプションも表示されます。

新規

新しい物理宛先を作成するボタン。

削除

選択した1つまたは複数の物理宛先を削除するボタン。

すべてのメッセージの削除

選択した1つまたは複数の物理宛先からメッセージを削除するボタン。
