JMS 物理送信先

JMS (Java Message Service) 物理送信先オブジェクトは、Message Queue ブローカによって管理されます。mq.sys.dmq という名前のキューはシステム送信先であり、期限切れメッセージおよび配信不能メッセージがリダイレクトされます。

Message Queue 物理送信先を設定するには、「JMS 物理送信先」ページを使用します。

物理送信先ごとに、次の情報が表示されます。

名前

物理送信先の名前。

タイプ

物理送信先のタイプ。

統計情報

物理送信先の統計情報。

「送信先」の表には、次のオプションも表示されます。

新規

新しい物理送信先を作成するボタン。

削除

選択した 1 つまたは複数の物理送信先を削除するボタン。

すべてのメッセージの削除

選択した 1 つまたは複数の物理送信先からメッセージを削除するボタン。
