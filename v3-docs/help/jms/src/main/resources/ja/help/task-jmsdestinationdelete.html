JMS宛先リソースを削除する

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「JMSリソース」ノードを展開します。

  3. 「JMSリソース」ノードで、「JMS宛先リソース」ノードを選択します。

    「JMS宛先リソース」ページが開きます。

  4. 「JMS宛先リソース」ページで、削除する宛先リソースの名前の隣にあるチェックボックスを選択します。

  5. 「削除」をクリックします。
