JMS 送信先リソースを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JMS リソース」ノードを展開します。
  3. 「JMS リソース」ノードで、「JMS 送信先リソース」ノードを選択します。

    「JMS 送信先リソース」ページが開きます。

  4. 「JMS 送信先リソース」ページで、削除する送信先リソースの名前の隣にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
