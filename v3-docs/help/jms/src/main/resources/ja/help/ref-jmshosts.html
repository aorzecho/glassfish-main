JMS ホスト

JMS サービスのホストを設定するには、「JMS ホスト」ページを使用します。

ホストごとに、次の情報が表示されます。

名前

JMS ホストの名前。

ホスト

JMS ホストを実行しているシステムの名前または IP (Internet Protocol) アドレス。

ポート

ホスト上の JMS サービスのポート番号。

「ホスト」の表には、次のオプションも表示されます。

新規

新しい JMS ホストを作成するボタン。

削除

選択した 1 つまたは複数の JMS ホストを削除するボタン。

関連操作

  • 「JMS サービスを設定する」

  • 「JMS ホストを作成する」

  • 「JMS ホストを編集する」

  • 「JMS ホストを削除する」

関連する asadmin コマンド

  • create-jms-host(1)

  • list-jms-hosts(1)

  • delete-jms-host(1)
