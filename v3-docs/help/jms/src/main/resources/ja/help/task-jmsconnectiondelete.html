JMS 接続ファクトリリソースを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JMS リソース」ノードを展開します。
  3. 「JMS リソース」ノードで、「JMS 接続ファクトリ」ノードを選択します。

    「JMS 接続ファクトリ」ページが開きます。

  4. 「JMS 接続ファクトリ」ページで、削除する接続ファクトリの名前の隣にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
