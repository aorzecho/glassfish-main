JMS 接続ファクトリ

JMS 接続ファクトリリソースを設定するには、「JMS 接続ファクトリ」ページを使用します。

リソースごとに、次の情報が表示されます。

プール名

接続ファクトリを識別する一意の名前。

JNDI 名

接続ファクトリを識別する JNDI 名。

使用可能

接続ファクトリが有効な場合は true、接続ファクトリが無効の場合は false です。

リソースタイプ

接続ファクトリのリソースタイプ。

説明

接続ファクトリの説明。

「接続ファクトリ」の表には、次のオプションも表示されます。

新規

新しい接続ファクトリを作成するボタン。

削除

選択した 1 つまたは複数の接続ファクトリを削除するボタン。

有効

選択した 1 つまたは複数の接続ファクトリを有効にするボタン。

無効

選択した 1 つまたは複数の接続ファクトリを無効にするボタン。

関連操作

  • 「JMS 接続ファクトリリソースを作成する」

  • 「JMS 接続ファクトリリソースを編集する」

  • 「JMS 接続ファクトリリソースを削除する」
