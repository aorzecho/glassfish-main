JMS物理宛先からメッセージを削除する

  1. ナビゲーション・ツリーで、「構成」ノードを展開します。

  2. 「構成」ノードで、「Javaメッセージ・サービス」ノードを展開します。

  3. 「Javaメッセージ・サービス」ノードで、「物理宛先」ノードを選択します。

    「物理宛先」ページが開きます。

  4. 「物理宛先」ページで、メッセージを削除する宛先の名前の隣にあるチェックボックスを選択します。

  5. 「すべてのメッセージの削除」ボタンをクリックします。
