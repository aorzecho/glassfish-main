JMS 物理送信先からメッセージを削除する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードで、「Java メッセージサービス」ノードを展開します。
  3. 「Java メッセージサービス」ノードで、「物理送信先」ノードを選択します。

    「物理送信先」ページが開きます。

  4. 「物理送信先」ページで、メッセージをフラッシュする送信先の名前の隣にあるチェックボックスを選択します。
  5. 「フラッシュ」ボタンをクリックします。
