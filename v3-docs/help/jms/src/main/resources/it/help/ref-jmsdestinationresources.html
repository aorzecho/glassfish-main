JMS Destination Resources

Nella pagina JMS Destination Resources è possibile configurare risorse di destinazione JMS.

Per ogni risorsa esterna vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa di destinazione.

Enabled

Opzione attivata se la risorsa è abilitata e disattivata se la risorsa è disabilitata.

Resource Type

Tipo della risorsa di destinazione.

Description

Descrizione della risorsa di destinazione.

La tabella Destination Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa di destinazione.

Delete

Pulsante che consente di eliminare una o più risorse di destinazione esterne selezionate.

Enable

Pulsante che consente di abilitare una o più risorse di destinazione selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse di destinazione selezionate.

  • Eliminare una risorsa di destinazione JMS
