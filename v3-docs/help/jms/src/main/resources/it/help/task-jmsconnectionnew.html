Creare una risorsa factory di connessione JMS

Quando si crea una factory di connessione JMS, vengono creati un pool di connessione del connettore e una risorsa connettore per la factory di connessione. È possibile visualizzare questi oggetti nel nodo Connectors.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources espandere il nodo JMS Resources.
  3. Nel nodo JMS Resources selezionare il nodo JMS Connection Factories.

    Viene visualizzata la pagina JMS Connection Factories.

  4. Nella pagina JMS Connection Factories fare clic su New.

    Viene visualizzata la pagina New JMS Connection Factory.

  5. Nel campo Pool Name immettere il nome univoco della factory di connessione.

    Il nome JNDI della factory di connessione viene impostato su tale valore da GlassFish Server.

  6. Dall'elenco a discesa Resource Type, selezionare javax.jms.ConnectionFactory , javax.jms.QueueConnectionFactory o javax.jms.TopicConnectionFactory.
  7. Nel campo Description, digitare una descrizione della factory di connessione.
  8. Selezionare la casella di controllo Status Enabled per abilitare la risorsa in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  9. Nel campo Initial and Minimum Pool Size, specificare il numero minimo di connessioni per il pool.

    Il valore predefinito è 8.

  10. Nel campo Maximum Pool Size, specificare il numero massimo di connessioni per il pool.

    Il valore predefinito è 32.

  11. Nel campo Pool Resize Quantity, specificare il numero di connessioni da rimuovere allo scadere del timeout di inattività del pool.

    Il valore predefinito è 2.

  12. Nel campo Idle Timeout, specificare il numero massimo di secondi durante i quali una connessione può rimanere inattiva nel pool.

    Il valore predefinito è 300.

  13. Nel campo Max Wait Time, specificare il tempo di attesa per la richiesta di connessione di un'applicazione, prima che si verifichi il timeout della connessione.

    Il valore predefinito è 60000.

  14. Se la casella di controllo On Any Failure Close All Connections è selezionata, in GlassFish Server tutte le connessioni del pool vengono chiuse e quindi ristabilite in caso di errore di una singola connessione.

    Se non si seleziona la casella di controllo, le singole connessioni vengono ristabilite solo quando vengono utilizzate. Questa opzione è disabilitata per impostazione predefinita.

  15. Per una risorsa factory di connessione JMS, selezionare il livello di Transaction Support.

    Il valore selezionato sovrascrive l'attributo di supporto della transazione in Resource Adapter compatibilmente con gli elementi derivati.

    • Selezionare XATransaction (valore predefinito per JMS Resource Adapter) per una risorsa utilizzabile per transazioni che implicano l'uso di più di una risorsa nell'ambito della transazione.

      Ad esempio è possibile che in una transazione sia necessario utilizzare questa risorsa in abbinamento a una risorsa JDBC, una risorsa del connettore o un'altra risorsa della factory di connessione JMS. Questo valore offre la massima flessibilità. Una risorsa configurata come XATransaction verrà utilizzata in operazioni di commit a due fasi.

    • Selezionare LocalTransaction per una risorsa utilizzabile in transazioni che prevedono l'uso di una sola risorsa o come ultimo agente in una transazione distribuita che prevede l'uso di più risorse XA.

      Il valore offre prestazioni decisamente migliori. Le risorse configurate come LocalTransaction non vengono utilizzate in operazioni di commit a due fasi.

    • Selezionare NoTransaction per una risorsa che non può mai partecipare a una transazione.

      L'impostazione è di uso limitato in applicazioni JMS.

  16. Selezionare la casella di controllo Connection Validation Required per abilitare la convalida della connessione.

    Se l'opzione è selezionata, le connessioni vengono convalidate prima di essere fornite all'applicazione. Se la convalida non viene eseguita, la risorsa viene eliminata. Viene creata e restituita una risorsa nuova. Questa opzione è disabilitata per impostazione predefinita.

  17. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la factory di connessione.

    Per informazioni sulle proprietà disponibili, vedere Proprietà specifiche delle factory di connessione JMS.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  18. Fare clic su OK.
