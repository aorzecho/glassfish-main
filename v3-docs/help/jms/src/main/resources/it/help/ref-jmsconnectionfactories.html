JMS Connection Factories

Nella pagina JMS Connection Factories è possibile configurare le risorse della factory di connessione del connettore.

Per ogni risorsa vengono fornite le seguenti informazioni.

Pool Name

Nome univoco che identifica la factory di connessione.

JNDI Name

Nome JNDI che identifica la factory di connessione.

Enabled

Opzione attivata se la factory di connessione è abilitata e disattivata se la factory di connessione è disabilitata.

Resource Type

Tipo di risorsa della factory di connessione.

Description

Descrizione della factory di connettore.

La tabella Connection Factories contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova factory di connessione.

Delete

Pulsante che consente di eliminare una o più factory di connessione selezionate.

Enable

Pulsante che consente di abilitare una o più factory di connessione selezionate.

Disable

Pulsante che consente di disabilitare una o più factory di connessione selezionate.
