Connection Factory JMS

Nella pagina Connection Factory JMS è possibile configurare le risorse Connection Factory JMS.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome pool

Nome univoco che identifica la Connection Factory.

Nome JNDI

Nome JNDI che identifica la Connection Factory.

Abilitato

Opzione attivata se la Connection Factory è abilitata; disattivata se la Connection Factory è disabilitata.

Tipo di risorsa

Tipo di risorsa della Connection Factory.

Descrizione

Descrizione della Connection Factory.

La tabella Connection Factory contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova Connection Factory.

Elimina

Pulsante che consente di eliminare una o più Connection Factory selezionate.

Abilita

Pulsante che consente di abilitare una o più Connection Factory selezionate.

Disabilita

Pulsante che consente di disabilitare una o più Connection Factory selezionate.
