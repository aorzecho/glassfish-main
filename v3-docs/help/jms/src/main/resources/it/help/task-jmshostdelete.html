Eliminare un host JMS

L'eliminazione dell'host JMS predefinito è sconsigliata.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, espandere il nodo Java Message Service.
  3. Nel nodo Java Message Service, selezionare il nodo JMS Hosts.

    Viene visualizzata la pagina JMS Hosts.

  4. Nella pagina JMS Hosts selezionare la casella di controllo in corrispondenza del nome dell'host da eliminare.

    Avvertenza - Non eliminare l'host JMS predefinito: potrebbero verificarsi errori inattesi.

  5. Fare clic su Delete.
