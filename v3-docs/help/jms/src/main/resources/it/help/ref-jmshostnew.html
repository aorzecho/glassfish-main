New JMS Host

Nella pagina New JMS Host è possibile creare un nuovo host JMS.

La pagina New JMS Host contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui vengono applicate le impostazioni in questa pagina. Il campo è di sola lettura.

Name

Nome dell'host (ad esempio NewJmsHost).

Host

Il nome o l'indirizzo IP (Internet Protocol) del sistema in cui l'host JMS viene eseguito (localhost o il nome del sistema locale o remoto).

Port

Numero della porta del servizio JMS. Modificare il valore di questo campo solo se il servizio JMS da utilizzare è in esecuzione su una porta non predefinita. La porta predefinita è 7676.

Admin Username e Admin Password

Nome utente e password del broker Message Queue. Non coincidono con il nome utente e la password di GlassFish Server. Modificare i valori di questi campi solo se i valori del broker di Message Queue sono stati modificati tramite il comando as-install-parent/mq/bin/imqusermgr. I valori predefiniti sono admin e admin.
