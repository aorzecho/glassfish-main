Proprietà specifiche del servizio JMS

Per il servizio JMS sono disponibili le seguenti proprietà di configurazione del broker Oracle Message Queue.

instance-name

Specifica il nome di istanza del broker Message Queue completo. Il valore predefinito è imqbroker.

instance-name-suffix

Specifica un suffisso da aggiungere al nome completo dell'istanza del broker Message Queue. Il suffisso è separato dal nome di istanza tramite un carattere di sottolineatura (_). Ad esempio, se il nome dell'istanza è imqbroker, con l'aggiunta del suffisso xyz viene modificato in imqbroker_xyz.

append-version

Se l'impostazione è true, aggiunge al nome completo dell'istanza del broker Message Queue i numeri per la parte intera e decimale della versione, preceduti dai caratteri di sottolineatura (_). Ad esempio, se il nome dell'istanza è imqbroker, con l'aggiunta dei numeri di versione viene modificato in imqbroker_8_0. Il valore predefinito è false.

user-name

Nome utente per la creazione della connessione JMS. Necessario solo se il nome utente e/o la password predefiniti guest/guest non è disponibile nel broker. Il valore predefinito è guest.

password

Password per la creazione della connessione JMS. Necessario solo se il nome utente e/o la password predefiniti guest/guest non è disponibile nel broker. Il valore predefinito è guest.
