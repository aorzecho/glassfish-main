Edit JMS Destination Resource

Nella pagina Edit JMS Destination Resource è possibile modificare le impostazioni per una risorsa di destinazione JMS.

La pagina Edit JMS Destination Resource include le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

JNDI Name

Nome univoco che identifica la risorsa di destinazione. Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa di destinazione JMS.

Physical Destination Name

Nome della destinazione fisica associata alla risorsa.

In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

Resource Type

Tipo di risorsa di destinazione. Le opzioni disponibili sono javax.jms.Topic e javax.jms.Queue.

Description

Descrizione della risorsa di destinazione.

Status

Stato della risorsa di destinazione. È possibile abilitare o disabilitare la risorsa di destinazione. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa di destinazione. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.
