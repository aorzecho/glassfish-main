Modifica risorsa di destinazione JMS

Nella pagina Modifica risorsa di destinazione JMS è possibile modificare le impostazioni per una risorsa di destinazione JMS.

La pagina Modifica risorsa di destinazione JMS contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome JNDI

Nome univoco che identifica la risorsa di destinazione. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa di destinazione JMS.

Nome destinazione fisica

Nome della destinazione fisica associata a questa risorsa.

In caso di necessità, GlassFish Server crea la destinazione fisica automaticamente e la elimina al momento dell'eliminazione della risorsa di destinazione.

Tipo di risorsa

Tipo di risorsa di destinazione. Le opzioni disponibili sono javax.jms.Topic e javax.jms.Queue.

Descrizione

Descrizione della risorsa di destinazione.

Stato

Stato della risorsa di destinazione. È possibile abilitare o disabilitare la risorsa di destinazione. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa di destinazione. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.
