New JMS Destination Resource

Nella pagina JMS Destination Resource è possibile creare una nuova risorsa di destinazione JMS.

La pagina New JMS Destination Resource include le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa di destinazione.

Si consiglia di utilizzare il prefisso di sottocontesto di denominazione jms/ per risorse JMS, ad esempio jms/Queue.

Physical Destination Name

Nome della destinazione fisica da associare alla risorsa.

In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

Resource Type

Tipo di risorsa di destinazione. Le opzioni disponibili sono javax.jms.Topic e javax.jms.Queue.

Description

Descrizione della risorsa di destinazione.

Status

Stato della risorsa di destinazione. È possibile abilitare o disabilitare la risorsa di destinazione. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa di destinazione. In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

Targets

Cluster e istanze standalone per la risorsa di destinazione. Per spostare le destinazioni desiderate nella colonna Selected Targets, utilizzare i pulsanti Add e Add All. Per spostare le destinazioni non necessarie nella colonna Available Targets, utilizzare i pulsanti Remove e Remove.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
