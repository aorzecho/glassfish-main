Eliminare una destinazione fisica JMS

L'eliminazione della destinazione di sistema, mq.sys.dmq, è sconsigliata.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, espandere il nodo Java Message Service.
  3. Nel nodo Java Message Service, selezionare il nodo Physical Destinations.

    Viene visualizzata la pagina Physical Destinations.

  4. Nella pagina Physical Destinations selezionare la casella di controllo in corrispondenza del nome della destinazione da eliminare.
  5. Fare clic su Delete.
