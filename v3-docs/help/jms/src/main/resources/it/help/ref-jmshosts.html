Host JMS

Nella pagina Host JMS è possibile configurare gli host di servizio JMS.

Per ogni host vengono fornite le informazioni riportate di seguito.

Nome

Nome dell'host JMS.

Host

Nome o indirizzo IP (Internet Protocol) del sistema su cui è in esecuzione l'host JMS.

Porta

Numero di porta del servizio JMS sull'host.

La tabella Host contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo host JMS.

Elimina

Pulsante che consente di eliminare uno o più host JMS selezionati.
