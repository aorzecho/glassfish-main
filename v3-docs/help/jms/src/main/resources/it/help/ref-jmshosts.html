JMS Hosts

Nella pagina JMS Hosts è possibile configurare gli host di servizio JMS.

Per ogni host vengono fornite le seguenti informazioni.

Name

Nome dell'host JMS.

Host

Nome o indirizzo IP (Internet Protocol) del sistema in cui è in esecuzione l'host JMS.

Port

Numero della porta del servizio JMS nell'host.

La tabella Hosts contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un nuovo host JMS.

Delete

Pulsante che consente di eliminare uno o più host JMS selezionati.

Attività correlate

  • Modificare un host JMS

  • Eliminare un host JMS
