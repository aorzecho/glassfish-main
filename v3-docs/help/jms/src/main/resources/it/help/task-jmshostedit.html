Modificare un host JMS

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. Nel nodo Configurazione, espandere il nodo Java Message Service.

  4. Nel nodo Java Message Service, selezionare il nodo Host JMS.

    Viene visualizzata la pagina Host JMS.

  5. Nella pagina Host JMS, selezionare l'host da modificare.

    Viene visualizzata la pagina Modifica host JMS.

  6. Nel campo Host, digitare il nome o l'indirizzo IP (Internet Protocol) del sistema in cui l'host viene eseguito (localhost o il nome del sistema locale o remoto).

  7. Nel campo Porta, digitare il numero di porta del servizio JMS.

    Modificare il valore di questo campo solo se il servizio JMS da utilizzare è in esecuzione su una porta non predefinita. La porta predefinita è 7676.

  8. Nei campi Nome utente amministratore e Password amministratore, digitare il nome utente e la password del broker Message Queue.

    Non coincidono con il nome utente e la password di GlassFish Server. Modificare i valori di questi campi solo se i valori del broker di Message Queue sono stati modificati tramite il comando as-install-parent/mq/bin/imqusermgr. I valori predefiniti sono admin e admin.

  9. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
