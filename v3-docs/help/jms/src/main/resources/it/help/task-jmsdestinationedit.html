Creare una risorsa di destinazione JMS

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse espandere il nodo Risorse JMS.

  3. Nel nodo Risorse JMS selezionare il nodo Risorse di destinazione JMS.

    Viene visualizzata la pagina Risorse di destinazione JMS.

  4. Nella pagina Risorse di destinazione JMS fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica risorsa di destinazione JMS.

  5. Nel campo Nome destinazione fisica digitare il nome della destinazione fisica da associare alla risorsa.

    In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

  6. Dall'elenco a discesa Tipo di risorsa, selezionare javax.jms.Topic o javax.jms.Queue.

  7. Nel campo Descrizione, digitare una descrizione della risorsa di destinazione.

  8. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  9. Nell'area Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa di destinazione.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  11. Optionally, click the Targets tab to go to the JMS Destination Resource Targets page. On this page, do the following:

    1. Click Manage Targets to open the Manage Resource Targets page.

      On this page, move targets between the Available column and the Selected column. Make sure to place in the Selected column the target or targets to which applications that use the resource will be deployed. The available targets include the available clusters and server instances, as well as the default server instance, server. Click OK to save the changes.

    2. Select the checkbox for a target, then click Enable or Disable to enable or disable the resource for that target.
