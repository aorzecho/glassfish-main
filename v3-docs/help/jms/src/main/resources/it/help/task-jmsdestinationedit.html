Modificare una risorsa di destinazione JMS

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources espandere il nodo JMS Resources.
  3. Nel nodo JMS Resources selezionare il nodo JMS Destination Resources.

    Viene visualizzata la pagina JMS Destination Resources.

  4. Nella pagina JMS Destination Resources fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Edit JMS Destination Resource.

  5. Nel campo Physical Destination Name digitare il nome della destinazione fisica da associare alla risorsa.

    In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

  6. Dall'elenco a discesa Resource Type, selezionare javax.jms.Topic o javax.jms.Queue.
  7. Nel campo Description, digitare una descrizione della risorsa di destinazione.
  8. Selezionare la casella di controllo Status Enabled per abilitare la risorsa in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  9. (Opzionale) Nell'area Additional Properties, specificare le proprietà aggiuntive per la risorsa di destinazione.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  10. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.
