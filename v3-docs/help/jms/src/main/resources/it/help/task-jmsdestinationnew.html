Creare una risorsa di destinazione JMS

Quando si crea una risorsa di destinazione JMS, viene creata una risorsa oggetto amministrato per la risorsa di destinazione. È possibile visualizzare questa risorsa nel nodo Connectors.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources espandere il nodo JMS Resources.
  3. Nel nodo JMS Resources selezionare il nodo JMS Destination Resources.

    Viene visualizzata la pagina JMS Destination Resources.

  4. Nella pagina JMS Destination Resources fare clic su New.

    Viene visualizzata la pagina New JMS Destination Resource.

  5. Nel campo JNDI Name digitare il nome JNDI della risorsa.

    Si consiglia di utilizzare il prefisso di sottocontesto di denominazione jms/ per risorse JMS, ad esempio jms/Queue.

  6. Nel campo Physical Destination Name digitare il nome della destinazione fisica da associare alla risorsa.

    In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

  7. Dall'elenco a discesa Resource Type, selezionare javax.jms.Topic o javax.jms.Queue.
  8. Nel campo Description, digitare una descrizione della risorsa di destinazione.
  9. Selezionare la casella di controllo Status Enabled per abilitare la risorsa in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  10. (Opzionale) Nell'area Additional Properties, specificare le proprietà aggiuntive per la risorsa di destinazione.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  11. Fare clic su OK.
