Creare una risorsa di destinazione JMS

Quando si crea una risorsa di destinazione JMS, viene creata una risorsa oggetto amministrato per la risorsa di destinazione. È possibile visualizzare questa risorsa nel nodo Connettori.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse espandere il nodo Risorse JMS.

  3. Nel nodo Risorse JMS selezionare il nodo Risorse di destinazione JMS.

    Viene visualizzata la pagina Risorse di destinazione JMS.

  4. Nella pagina Risorse di destinazione JMS fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa di destinazione JMS.

  5. Nel campo Nome JNDI digitare il nome JNDI della risorsa.

    Si consiglia di utilizzare il prefisso del contesto secondario di denominazione jms/ per risorse JMS. Ad esempio: jms/Queue.

  6. Nel campo Nome destinazione fisica digitare il nome della destinazione fisica da associare alla risorsa.

    In caso di necessità, la destinazione fisica viene creata automaticamente ed eliminata al momento dell'eliminazione della risorsa di destinazione.

  7. Dall'elenco a discesa Tipo di risorsa, selezionare javax.jms.Topic o javax.jms.Queue.

  8. Nel campo Descrizione, digitare una descrizione della risorsa di destinazione.

  9. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  10. Nell'area Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa di destinazione.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  11. In the Targets area, do the following:

    1. From the Available column, select the target or targets to which applications that use the resource will be deployed.

      The available targets include the available clusters and server instances, as well as the default server instance, server.

    2. Click Add to move the target to the Selected column.

  12. Fare clic su OK.
