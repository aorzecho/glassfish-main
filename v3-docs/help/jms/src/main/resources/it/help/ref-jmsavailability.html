Disponibilità JMS

Nella pagina JMS Availability è possibile configurare le impostazioni di disponibilità per il servizio JMS (Java Message Service).

La pagina JMS Availability contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Configuration Name

Nome della configurazione a cui vengono applicate le impostazioni in questa pagina. Il campo è di sola lettura.

Availability Service

Tipo di cluster Message Queue utilizzato per fornire servizi JMS a un cluster GlassFish. Se è selezionata la casella di controllo Enabled, viene utilizzato un cluster Message Queue ottimizzato. In caso contrario, viene utilizzato un cluster Message Queue tradizionale.

Se è selezionata la casella di controllo Enabled, le opzioni Configuration Store Type e Message Store Type vengono ignorate.

Configuration Store Type

Tipo di archivio dati per i dati di configurazione in un cluster Message Queue tradizionale. Il valore masterbroker specifica l'utilizzo di un broker principale per archiviare e gestire i dati di configurazione. Il valore shareddb specifica l'utilizzo di un database condiviso per archiviare i dati di configurazione.

Se il servizio di disponibilità è abilitato, l'opzione viene ignorata.

Message Store Type

Tipo di archivio dati per i dati del messaggio nei broker in un cluster tradizionale. Il valore file specifica un archivio file. Il valore jdbc specifica un archivio JDBC.

Se il servizio di disponibilità è abilitato, l'opzione viene ignorata.

Database Vendor
Database User Name
Database Password
Database Access URL

Produttore, utente, password e URL di accesso del database JDBC da utilizzare in ognuna delle situazioni seguenti:

  • Quando il servizio di disponibilità è abilitato

  • Quando Configuration Store Type è impostato su shareddb

  • Quando Message Store Type è impostato su jdbc

Per informazioni sui produttori supportati e sui formati degli URL di accesso per ciascun produttore, vedere JDBC-Based Persistence in Oracle GlassFish Server Message Queue 4.5 Administration Guide.

Additional Properties

Proprietà aggiuntive per la disponibilità JMS.

In GlassFish Server non è predefinita alcuna proprietà di disponibilità JMS aggiuntiva.
