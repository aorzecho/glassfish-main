Disponibilità JMS

Nella pagina JMS Availability è possibile configurare le impostazioni di disponibilità per il servizio JMS (Java Message Service).

La pagina JMS Availability contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Availability Service

Se è selezionata la casella di controllo Enabled, la disponibilità è abilitata per JMS.

MQ Store Pool Name

Nome JNDI della risorsa JDBC che il cluster del broker GlassFish Message Queue deve utilizzare per il salvataggio di messaggi JMS persistenti e di altre informazioni sulla configurazione del broker.

Il nome del pool può contenere solo lettere, numeri, caratteri di sottolineatura (_), trattini (-) e punti (.) .

Additional Properties

Proprietà aggiuntive per la disponibilità JMS.

In GlassFish Server non è predefinita alcuna proprietà di disponibilità JMS aggiuntiva.
