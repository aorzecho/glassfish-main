JMS Physical Destinations

Gli oggetti di destinazione fisica JMS (Java Message Service) vengono mantenuti dai broker Message Queue. La coda denominata mq.sys.dmq è la destinazione del sistema, alla quale vengono reindirizzati i messaggi scaduti e non inviabili.

Nella pagina JMS Physical Destinations è possibile configurare le destinazioni fisiche Message Queue.

Per ogni destinazione fisica vengono fornite le seguenti informazioni.

Name

Nome della destinazione fisica.

Type

Tipo della destinazione fisica.

Statistics

Statistiche relative alla destinazione fisica.

La tabella Destinations contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova destinazione fisica.

Delete

Pulsante che consente di eliminare una o più destinazioni fisiche selezionate.

Flush

Pulsante che consente di rimuovere una o più destinazioni fisiche selezionate.
