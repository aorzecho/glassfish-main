Destinazioni fisiche JMS

Gli oggetti di destinazione fisica di JMS (Java Message Service) vengono mantenuti dai broker Message Queue. La coda denominata mq.sys.dmq è la destinazione di sistema a cui vengono reindirizzati i messaggi scaduti e non recapitabili.

Nella pagina Destinazioni fisiche JMS è possibile configurare le destinazioni fisiche di Message Queue.

Per ogni destinazione fisica vengono fornite le informazioni riportate di seguito.

Nome

Nome della destinazione fisica.

Tipo

Tipo di destinazione fisica.

Statistiche

Statistiche relative alla destinazione fisica.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova destinazione fisica.

Elimina

Pulsante che consente di eliminare una o più destinazioni fisiche selezionate.

Elimina tutti i messaggi

Pulsante che consente di rimuovere una o più destinazioni fisiche selezionate.
