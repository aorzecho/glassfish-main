Modifica host JMS

Nella pagina Modifica host JMS è possibile modificare le impostazioni per un host JMS.

La pagina Modifica host JMS contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Nome

Nome dell'host. Il campo Nome è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo host JMS.

Host

Nome o indirizzo IP (Internet Protocol) del sistema su cui verrà eseguito l'host JMS (localhost o il nome del sistema locale o remoto).

Porta

Numero della porta del servizio JMS. Modificare il valore di questo campo solo se il servizio JMS da utilizzare è in esecuzione su una porta non predefinita. La porta predefinita è 7676.

Nome utente amministratore e Password amministratore

Nome utente e password del broker Message Queue. Non coincidono con il nome utente e la password di GlassFish Server. Modificare i valori di questi campi solo se i valori del broker Message Queue sono stati modificati con il comando as-install-parent/mq/bin/imqusermgr. I valori predefiniti sono admin e admin.
