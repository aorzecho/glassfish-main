Eliminare una risorsa Connection Factory JMS

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse espandere il nodo Risorse JMS.

  3. Nel nodo Risorse JMS selezionare il nodo Connection Factory JMS.

    Viene visualizzata la pagina Connection Factory JMS.

  4. Nella pagina Connection Factory JMS, selezionare la casella di controllo in corrispondenza del nome della Connection Factory da eliminare.

  5. Fare clic su Elimina.
