Eliminare una risorsa factory di connessione JMS

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources espandere il nodo JMS Resources.
  3. Nel nodo JMS Resources selezionare il nodo JMS Connection Factories.

    Viene visualizzata la pagina JMS Connection Factories.

  4. Nella pagina JMS Connection Factories, selezionare la casella di controllo in corrispondenza del nome della factory di connessione da eliminare.
  5. Fare clic su Delete.
