Visualizzare statistiche per una destinazione fisica JMS

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, espandere il nodo Java Message Service.
  3. Nel nodo Java Message Service, selezionare il nodo Physical Destinations.

    Viene visualizzata la pagina Physical Destinations.

  4. Nella pagina Physical Destinations, fare clic sul collegamento View relativo alla destinazione di cui si desidera visualizzare le statistiche.

    Viene visualizzata la pagina View JMS Physical Destination Statistics.

  5. Al termine della visualizzazione delle statistiche, fare clic sul pulsante Back.
