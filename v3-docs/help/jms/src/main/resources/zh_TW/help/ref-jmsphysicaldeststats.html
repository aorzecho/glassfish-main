檢視 JMS 實體目標統計

[檢視 JMS 實體目標統計] 頁面會顯示有關使用 JMS 實體目標的資訊。

除了目標的名稱和類型之外，頁面會提供有關以下方面的詳細資訊：

  • 產生器和用戶數目

  • 目標所處理的訊息數目和大小

  • 磁碟使用率
