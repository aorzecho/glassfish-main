刪除 JMS 實體目標

不建議刪除系統目標 mq.sys.dmq。

  1. 在瀏覽樹狀結構中，展開 「組態」 節點。

  2. 在 「組態」 節點下，展開 「Java 訊息服務」 節點。

  3. 在 「Java 訊息服務」 節點下，選取 「實體目標」 節點。

    將開啟 「實體目標」 頁面。

  4. 在 「實體目標」 頁面上，選取您要刪除的目標名稱旁的核取方塊。

  5. 按一下 「刪除」。
