JMS 連線工廠

使用 [JMS 連線工廠] 頁面可配置 JMS 連線工廠資源。

對於每個資源，都將提供以下資訊。

池名稱

可識別連線工廠的唯一名稱。

JNDI 名稱

可識別連線工廠的 JNDI 名稱。

已啟用

若連線工廠啟用則為 True，若連線工廠停用則為 False。

資源類型

連線工廠的資源類型。

說明

連線工廠的說明。

連線工廠表也包含下列選項。

新建

建立新連線工廠的按鈕。

刪除

刪除一或多個選取連線工廠的按鈕。

啟用

啟用一或多個選取連線工廠的按鈕。

停用

停用一或多個選取連線工廠的按鈕。

  • 編輯 JMS 連線工廠資源

  • 刪除 JMS 連線工廠資源
