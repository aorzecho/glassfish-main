編輯 JMS 目標資源

使用 [編輯 JMS 目標資源] 頁面可修改 JMS 目標資源的設定。

[編輯 JMS 目標資源] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

JNDI 名稱

可識別目標資源的唯一名稱。[JNDI 名稱] 是唯讀欄位。您僅能在建立新的 JMS 目標資源時指定 JNDI 名稱。

實體目標名稱

與此資源關聯的實體目標名稱。

GlassFish Server 會在需要時自動建立實體目標，並在您刪除目標資源時將其刪除。

資源類型

目標資源的類型。可用的選項包括 javax.jms.Topic 和 javax.jms.Queue。

說明

目標資源的說明。

狀態

目標資源的狀態。您可啟用或停用目標資源。此選項預設為啟用。

其他特性

目標資源的其他特性。GlassFish Server 未定義 JMS 目標資源的任何其他特性。
