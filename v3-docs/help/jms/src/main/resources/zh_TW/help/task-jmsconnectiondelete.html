刪除 JMS 連線工廠資源

  1. 在瀏覽樹狀結構中，展開 「資源」 節點。

  2. 在 「資源」 節點下，展開 「JMS 資源」 節點。

  3. 在 「JMS 資源」 節點下，選取 「JMS 連線工廠」 節點。

    將開啟 「JMS 連線工廠」 頁面。

  4. 在 「JMS 連線工廠」 頁面上，選取要刪除的連線工廠名稱旁邊的核取方塊。

  5. 按一下 「刪除」。
