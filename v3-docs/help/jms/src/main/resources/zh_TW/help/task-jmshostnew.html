建立 JMS 主機

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [Java 訊息服務] 節點。
  3. 在 [Java 訊息服務] 節點下，選取 [JMS 主機] 節點。

    將開啟 [JMS 主機] 頁面。

  4. 在 [JMS 主機] 頁面上，按一下 [新建]。

    將顯示 [新建 JMS 主機] 頁面。

  5. 在 [名稱] 欄位中，鍵入主機的名稱。

    例如︰

    NewJmsHost
  6. 在 [主機] 欄位中，鍵入執行 JMS 主機的系統名稱或網際網路通訊協定 (IP) 位址 (localhost 或者本機或遠端系統的名稱)。
  7. 在 [連接埠] 欄位中，鍵入 JMS 服務的連接埠號。

    僅當要使用的 JMS 服務在非預設連接埠上執行時，才需要變更該欄位。預設連接埠為 7676。

  8. 在 [管理使用者名稱] 和 [管理員密碼] 欄位中，鍵入 Message Queue 代理程式的使用者名稱和密碼。

    該使用者名稱和密碼與 GlassFish Server 的使用者名稱和密碼不同。僅當已使用 as-install-parent/mq/bin/imqusermgr 指令變更了 Message Queue 代理程式值時，才需要編輯這些欄位。預設值為 admin 和 admin。

  9. 按一下 [確定]。
