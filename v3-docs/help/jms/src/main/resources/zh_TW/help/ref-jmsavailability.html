JMS 可用性

使用 [JMS 可用性] 頁面可配置 Java 訊息服務 (JMS) 的可用性設定。

[JMS 可用性] 頁面包含下列選項。

載入預設值

將所有設定復原為預設值的按鈕。

配置名稱

套用此頁面上之設定的配置名稱。此欄位是唯讀欄位。

可用性服務

如果選取 [已啟用] 核取方塊，則會為 JMS 啟用可用性。

MQ 儲存池名稱

當儲存永久性 JMS 訊息和其他代理程式配置資訊時，GlassFish Message Queue 代理程式叢集要使用之 JDBC 資源的 JNDI 名稱。

池名稱只能包含字母、數字或底線 (_)、破折號 (-) 或句點 (.) 字元。

其他特性

JMS 可用性的其他特性。

GlassFish Server 未預先定義任何其他的 JMS 可用性特性。
