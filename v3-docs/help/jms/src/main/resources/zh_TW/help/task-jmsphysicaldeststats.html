檢視 JMS 實體目標的統計

  1. 在瀏覽樹狀結構中，展開 「組態」 節點。

  2. 在 「組態」 節點下，展開 「Java 訊息服務」 節點。

  3. 在 「Java 訊息服務」 節點下，選取 「實體目標」 節點。

    將開啟 「實體目標」 頁面。

  4. 在 「實體目標」 頁面上，按一下您要檢視統計的目標之 「檢視」 連結。

    將開啟 「檢視 JMS 實體目標統計」 頁面。

  5. 當您檢視完統計後，請按一下 「返回」 按鈕。
