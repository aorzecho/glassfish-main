JMS 主機

使用 [JMS 主機] 頁面可配置 JMS 服務主機。

對於每個主機，都將提供以下資訊。

名稱

JMS 主機的名稱。

主機

執行 JMS 主機的系統名稱或網際網路協定 (IP) 位址。

連接埠

主機上 JMS 服務的連接埠號碼。

主機表也包含下列選項。

新建

建立新 JMS 主機的按鈕。

刪除

刪除一或多個所選取 JMS 主機的按鈕。
