編輯 JMS 主機

使用 「編輯 JMS 主機」 頁面可修改 JMS 主機的設定。

「編輯 JMS 主機」 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

名稱

主機的名稱。「名稱」 欄位是唯讀欄位。您僅能在建立新的 JMS 主機時指定 「名稱」 欄位。

主機

執行 JMS 主機的系統名稱或網際網路協定 (IP) 位址 (localhost 或者本機或遠端系統的名稱)。

連接埠

JMS 服務的連接埠號碼。僅當要使用的 JMS 服務在非預設連接埠上執行時，才需要變更該欄位。預設連接埠為 7676。

管理使用者名稱和管理密碼

Message Queue 代理程式使用者名稱和密碼。該使用者名稱和密碼與 GlassFish Server 的使用者名稱和密碼不同。僅當已使用 as-install-parent/mq/bin/imqusermgr 命令變更了 Message Queue 代理程式值時，才需要編輯這些欄位。預設值為 admin 和 admin。
