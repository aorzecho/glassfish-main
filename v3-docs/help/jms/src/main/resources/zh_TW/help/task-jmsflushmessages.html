從 JMS 實體目標移除訊息

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [Java 訊息服務] 節點。
  3. 在 [Java 訊息服務] 節點下，選取 [實體目標] 節點。

    將開啟 [實體目標] 頁面。

  4. 在 [實體目標] 頁面上，選取要清除訊息的目標名稱旁邊的核取方塊。
  5. 按一下 [清除] 按鈕。
