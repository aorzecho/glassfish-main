JMS 服務特定的特性

以下的 Oracle Message Queue 代理程式組態特性適用於 JMS 服務。

instance-name

指定完整的 Message Queue 代理程式實例名稱。預設值為 imqbroker。

instance-name-suffix

指定要增加到完整的 Message Queue 代理程式實例名稱的後綴。該後綴與實例名稱之間以底線字元 (_) 分隔。例如，如果實例名稱為 imqbroker，則附加後綴 xyz 之後，實例名稱將變更為 imqbroker_xyz。

append-version

如果為 true，則應在完整的 Message Queue 代理程式實例名稱後附加主要和次要版本編號，這兩個版本編號之前都有一個底線字元 (_)。例如，如果實例名稱為 imqbroker，則附加版本編號之後，實例名稱將變更為 imqbroker_8_0。預設值為 false。

user-name

建立 JMS 連線的使用者名稱。只有當代理程式中沒有 guest/guest 的預設使用者名稱/密碼時才需要。預設值為 guest。

password

建立 JMS 連線的密碼。只有當代理程式中沒有 guest/guest 的預設使用者名稱/密碼時才需要。預設值為 guest。
