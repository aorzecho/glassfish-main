Resources

Nella pagina Resources è possibile eseguire le attività seguenti nell'istanza standalone di GlassFish Server selezionata:

  • Visualizzazione e gestione delle risorse esistenti dell'istanza

  • Creazione di risorse aggiuntive per l'istanza

Il campo Instance Name è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Nella pagina Resources viene visualizzato un elenco delle risorse esistenti delle istanze. Per ogni risorsa vengono fornite le seguenti informazioni.

Resource Name

Nome assegnato alla risorsa al momento della creazione. Se si fa clic sul nome, viene aperta una delle seguenti pagine per la modifica della risorsa, in base al tipo di risorsa:

  • Edit JDBC Resource

  • Edit JavaMail Session

  • Edit Custom Resource

  • New Custom Resource

  • Edit External Resource

  • Edit Connector Resource

  • Edit Admin Object Resource

Enabled

Indicazione relativa all'eventuale abilitazione della risorsa:

  • Se la risorsa è abilitata, viene visualizzato un segno di spunta (✓).

  • Se la risorsa è disabilitata, viene visualizzato un simbolo a forma di croce (✗).

Type

Tipo della risorsa.

La tabella Resources contiene inoltre le seguenti opzioni.

Enable

Pulsante che consente di abilitare una o più risorse selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse selezionate.

New

Elenco a discesa che consente di creare una risorsa del tipo selezionato. Se si seleziona un tipo di risorsa dall'elenco, viene visualizzata una delle seguenti pagine per creare una risorsa del tipo selezionato:

  • New JDBC Resource

  • New JMS Connection Factory

  • New JMS Destination Resource

  • New JavaMail Session

  • New Custom Resource

  • New External Resource

  • New Connector Resource

  • New Admin Object Resource

Filter

Elenco a discesa che consente di filtrare l'elenco delle risorse per tipo.
