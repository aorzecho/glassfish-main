Creare una risorsa per un'istanza standalone di ProductName;

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Nella pagina Istanze server standalone selezionare l'istanza.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Risorsa per l'istanza.

  4. Nel menu a discesa Nuovo selezionare il tipo di risorsa da creare.

    Viene visualizzata la pagina per la creazione di una risorsa del tipo selezionato.

  5. Nella pagina per la creazione di una risorsa del tipo selezionato, apportare le modifiche necessarie e salvarle.

    Per dettagli, vedere la Guida in linea relativa alla pagina per la creazione di una risorsa del tipo selezionato.

    Nota:

    Quando si crea la risorsa, assicurarsi di specificare l'istanza come destinazione.
