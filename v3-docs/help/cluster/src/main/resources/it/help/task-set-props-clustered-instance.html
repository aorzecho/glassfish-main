Impostare proprietà specifiche dell'istanza per un'istanza di GlassFish Server in cluster

Le proprietà di un'istanza di GlassFish Server aggiungono informazioni di configurazione opzionali relative all'istanza.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Selezionare l'istanza per la quale modificare le proprietà specifiche.

    Viene visualizzata la pagina General Information per l'istanza.

  5. Nella pagina General Information, fare clic sulla scheda Properties.
  6. Se necessario, fare clic sulla scheda Instance Properties.

    Viene visualizzata la pagina Instance Properties.

    Nella pagina viene visualizzato un elenco delle proprietà impostate per l'istanza GlassFish Server.

  7. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Value relativo alla proprietà.
  8. Impostare i valori per eventuali proprietà aggiuntive necessarie.
    1. Nella tabella Additional Properties fare clic su Add Property.

      Viene aggiunta una riga alla tabella Additional Properties.

    2. Nel campo Name, digitare il nome della proprietà.
    3. Nel campo Value digitare il valore.
    4. Nel campo Description digitare una breve descrizione della proprietà.
  9. Fare clic su Save.
