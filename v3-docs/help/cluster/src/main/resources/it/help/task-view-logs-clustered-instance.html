Visualizzare i file di log per un'istanza di GlassFish Server in cluster

Prima di cominciare

È necessario che l'istanza per la quale si visualizzano i file di log sia in esecuzione.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Selezionare l'istanza per la quale visualizzare i file di log.

    Viene visualizzata la pagina General Information per l'istanza.

  5. Nella pagina General Information, fare clic su View Log Files.

    Viene visualizzata la finestra Log Viewer.

    Nota - L'istanza deve essere già in esecuzione. Se l'istanza non è in esecuzione, il pulsante View Log Files è disattivato.
