Visualizzare i file di log per un'istanza standalone di GlassFish Server

Prima di cominciare

È necessario che l'istanza per la quale si visualizzano i file di log sia in esecuzione.

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Selezionare l'istanza per la quale visualizzare i file di log.

    Viene visualizzata la pagina General Information per l'istanza.

  3. Nella pagina General Information, fare clic su View Log Files.

    Viene visualizzata la finestra Log Viewer.

    Nota - L'istanza deve essere già in esecuzione. Se l'istanza non è in esecuzione, il pulsante View Log Files è disattivato.
