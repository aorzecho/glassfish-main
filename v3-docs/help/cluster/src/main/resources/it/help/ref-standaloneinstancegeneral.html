General Information

La pagina General Information contiene le seguenti opzioni.

Start

Pulsante che consente di avviare l'istanza. Se l'istanza è in esecuzione, il pulsante è disattivato.

Stop

Pulsante che consente di arrestare l'istanza. Se l'istanza viene arrestata, il pulsante è disattivato.

Restart

Pulsante che consente di riavviare l'istanza. Se l'istanza viene arrestata, il pulsante è disattivato.

View Log Files

Pulsante che consente di visualizzare i file di log per l'istanza.

Rotate Log

Pulsante che consente di ruotare il file del log per l'istanza. Quando il file di log viene ruotato, il file viene rinominato con un nome timestamp con il formato server.log_ date-and-time, e viene creato un file di log vuoto per nuovi messaggi di log. Le modifiche vengono applicate dinamicamente. Non è necessario riavviare il server.

Recover Transactions

Pulsante che consente di ripristinare transazioni per l'istanza.

Name

Nome assegnato all'istanza quando l'istanza è stata creata. Campo di sola lettura.

JVM

Se si fa clic su JVM Report, viene visualizzata una finestra distinta in cui sono disponibili i rapporti relativi alla macchina virtuale Java, tra cui un rapporto di riepilogo, un rapporto di gestione memoria e recupero spazio, un rapporto sul caricamento delle classi e un dump del thread corrente.

Status

Indicazione relativa all'eventuale esecuzione dell'istanza: Campo di sola lettura.

Node

Nome del nodo nel quale si trova l'istanza. Se si fa clic sul nome, viene visualizzata la pagina Edit Node per il nodo.

Configuration

Configurazione a cui l'istanza fa riferimento. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per l'istanza.

Debug

Indica se per l'istanza è abilitato il debug Java Platform Debugger Architecture (JPDA). Campo di sola lettura.

Se un'istanza viene avviata da Administration Console, il debug JPDA non viene abilitato per l'istanza. Per avviare un'istanza con debug JPDA abilitato, utilizzare il comando secondario start-instance(1).

HTTP Ports

Elenco, separato da virgole, dei numeri di porta su cui l'istanza resta in attesa per i seguenti tipi di richieste:

  • Richieste di amministrazione

  • Richieste HTTP

  • Richieste HTTPS

Campo di sola lettura.

IIOP Ports

Elenco, separato da virgole, dei numeri delle porte che l'istanza utilizza per i seguenti tipi di connessioni:

  • Connessioni IIOP sicure

  • Connessioni IIOP sicure con autenticazione del client

  • Connessioni IIOP

Campo di sola lettura.
