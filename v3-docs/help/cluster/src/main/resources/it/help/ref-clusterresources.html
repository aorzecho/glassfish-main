Risorse

Nella pagina Risorse è possibile eseguire i task riportati di seguito nel cluster selezionato.

  • Visualizzazione e gestione delle risorse esistenti del cluster

  • Creazione di risorse aggiuntive per il cluster

Il campo Nome cluster è un campo di sola lettura in cui viene visualizzato il nome del cluster corrente.

Nella pagina Risorse viene visualizzata una lista di risorse dei cluster esistenti. Per ogni risorsa vengono fornite le seguenti informazioni.

Nome risorsa

Nome assegnato alla risorsa al momento della creazione. Se si fa clic sul nome, viene aperta una delle seguenti pagine per la modifica della risorsa, in base al tipo di risorsa:

  • Edit JDBC Resource

  • Edit JavaMail Session

  • Edit Custom Resource

  • New Custom Resource

  • Edit External Resource

  • Edit Connector Resource

  • Edit Admin Object Resource

Abilitato

Indicazione relativa all'eventuale abilitazione della risorsa:

  • Se la risorsa è abilitata, viene visualizzato un segno di spunta (✓).

  • Se la risorsa è disabilitata, viene visualizzato un simbolo a forma di croce (✗).

Tipo

Tipo della risorsa.

La tabella Risorse contiene inoltre le seguenti opzioni.

Abilita

Pulsante che consente di abilitare una o più risorse selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse selezionate.

Nuovo

Elenco a discesa che consente di creare una risorsa del tipo selezionato. Se si seleziona un tipo di risorsa dalla lista, viene visualizzata una delle seguenti pagine per creare una risorsa del tipo selezionato:

  • New JDBC Resource

  • New JMS Connection Factory

  • New JMS Destination Resource

  • New JavaMail Session

  • New Custom Resource

  • New External Resource

  • New Connector Resource

  • New Admin Object Resource

Filtro

Elenco a discesa che consente di filtrare la lista delle risorse per tipo.
