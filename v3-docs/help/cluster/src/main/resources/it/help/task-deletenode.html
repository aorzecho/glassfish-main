Eliminare i nodi

Se si elimina un nodo, viene rimosso dalla configurazione del server di amministrazione del dominio (DAS, Domain Administration Server). Le directory e i file del nodo vengono eliminati quando viene eliminata l'ultima istanza di GlassFish Server che si trova nel nodo.

È inoltre possibile scegliere di disinstallare il software GlassFish Server dall'host del nodo quando viene eliminato il nodo.

Prima di iniziare

Verificare che nel nodo da eliminare non risiedano istanze di GlassFish Server.

  1. Nell'albero di navigazione selezionare il nodo Nodi.

    Viene visualizzata la pagina Nodi.

  2. Per ogni istanza eliminata, selezionare la casella di controllo in corrispondenza del nodo nella tabella Nodi.

    Nota:

    Non è possibile eliminare il nodo predefinito localhost-domain.

  3. Fare clic sul pulsante per eliminare i nodi o eliminare i nodi e disinstallare il software GlassFish Server dagli host del nodo.

    • Per eliminare i nodi senza disinstallare il software GlassFish Server, fare clic su Elimina.

      Viene richiesto di confermare l'eliminazione dei nodi.

    • Per eliminare i nodi disinstallando il software GlassFish Server, fare clic su Elimina e disinstalla.

      Viene richiesto di confermare l'eliminazione dei nodi e la disinstallazione del software GlassFish Server dagli host del nodo.

  4. Confermare che si desidera eliminare i nodi o eliminare i nodi e disinstallare il software GlassFish Server dagli host del nodo.

    I nodi vengono eliminati e rimossi dalla tabella di nodi della pagina Nodi. Se si fa clic su Elimina e disinstalla, viene eseguita anche la disinstallazione del software GlassFish Server dagli host del nodo.
