Eliminare nodi

Se si elimina un nodo, viene rimosso dalla configurazione del server di amministrazione del dominio (DAS, Domain Administration Server). Le directory e i file del nodo vengono eliminati quando viene eliminata l'ultima istanza di GlassFish Server che si trova nel nodo.

Prima di cominciare

Verificare che non siano presenti istanze di GlassFish Server nel nodo da eliminare.

  1. Nella struttura ad albero, selezionare il nodo Nodes.

    Viene visualizzata la pagina Nodes.

  2. Per ogni istanza eliminata, selezionare la casella di controllo in corrispondenza del nodo nella tabella Nodes.

    Nota - Non è possibile eliminare il nodo predefinito localhost-domain.

  3. Fare clic su Delete.

    Viene richiesto di confermare l'eliminazione del nodo.

  4. Confermare l'eliminazione del nodo.

    Il nodo viene eliminato e rimosso dalla tabella di nodi nella pagina Nodes.
