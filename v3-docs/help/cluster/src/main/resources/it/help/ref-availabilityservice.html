Availability Service

Nella pagina Availability Service è possibile modificare le impostazioni per un cluster.

La pagina Availability Service contiene le seguenti opzioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Availability Service

Se è selezionata la casella di controllo Enabled, la disponibilità è abilitata per il cluster. Questa opzione è selezionata per impostazione predefinita.

Additional Properties

Tabella di proprietà aggiuntive per il servizio di disponibilità.

In GlassFish Server non vengono definite proprietà aggiuntive per il servizio di disponibilità.

Per ogni proprietà vengono visualizzate le seguenti informazioni:

Name

Nome della proprietà.

Value

Il valore della proprietà è impostato per l'istanza selezionata.

Description

Descrizione testuale in cui sono riportate ulteriori informazioni sulla proprietà.

La tabella Additional Properties contiene inoltre le seguenti opzioni.

Add Property

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Additional Properties.

Delete Properties

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il rispettivo valore predefinito oppure, se non è specificato il valore predefinito, il valore resta non definito.
