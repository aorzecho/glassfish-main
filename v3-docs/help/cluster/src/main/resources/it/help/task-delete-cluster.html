Eliminare un cluster

L'eliminazione di un cluster causa l'eliminazione del cluster e di tutte le istanze di GlassFish Server in esso presenti. Prima di eliminare un cluster, non è necessario eliminare tutte le istanze che esso contiene.

Se la configurazione utilizzata nel cluster non è condivisa, viene eliminata.

Prima di iniziare

Verificare che il cluster da eliminare sia arrestato. Per istruzioni, vedere Arrestare un cluster.

  1. Nell'albero di navigazione selezionare il nodo Cluster.

    Viene visualizzata la pagina Cluster.

  2. Nella pagina Cluster selezionare la casella di controllo in corrispondenza del nome del cluster.

  3. Fare clic su Elimina.

    Viene richiesto di confermare l'eliminazione del cluster e di tutte le istanze nel cluster.

  4. Confermare l'eliminazione del cluster e di tutte le istanze nel cluster.

    Il cluster viene eliminato e rimosso dalla tabella di cluster della pagina Cluster.
