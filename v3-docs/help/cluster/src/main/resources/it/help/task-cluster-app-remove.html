Rimuovere applicazioni da un cluster

Se si rimuove un'applicazione, essa viene rimossa solo dalla destinazione corrente. Quando viene rimossa da una destinazione, l'applicazione continua a essere elencata nella pagina Applications, nella quale vengono visualizzate tutte le applicazioni distribuite in GlassFish Server. L'applicazione rimane inoltre distribuita su eventuali altre destinazioni sulle quali è stata effettuata la distribuzione.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Applications.

    Viene visualizzata la pagina Applications per il cluster.

  4. Per ogni applicazione da rimuovere, selezionare la casella di controllo in corrispondenza dell'applicazione nella tabella Deployed Applications.
  5. Fare clic su Remove.

    Viene richiesto di confermare la rimozione delle applicazioni.

  6. Confermare la rimozione delle applicazioni.

    Le applicazioni vengono rimosse dalla tabella delle applicazioni distribuite visualizzata nella pagina.
