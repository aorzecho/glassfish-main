Impostare le proprietà di sola istanza per un'istanza standalone di GlassFish Server

Le proprietà di un'istanza di GlassFish Server aggiungono informazioni di configurazione opzionali relative all'istanza.

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale modificare le proprietà specifiche.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Properties.

  4. Se necessario, fare clic sulla scheda Proprietà istanza.

    Viene visualizzata la pagina Proprietà istanza.

    Nella pagina viene visualizzata una lista delle proprietà impostate per l'istanza GlassFish Server selezionata.

  5. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore relativo alla proprietà.

  6. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome, digitare il nome della proprietà.

    3. Nel campo Valore digitare il valore.

    4. Nel campo Descrizione digitare una breve descrizione della proprietà.

  7. Fare clic su Salva.
