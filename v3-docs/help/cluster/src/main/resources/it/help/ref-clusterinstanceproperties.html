Proprietà istanza

Nella pagina Proprietà istanza viene visualizzata una lista delle proprietà impostate per l'istanza di GlassFish Server corrente. Queste proprietà aggiungono informazioni di configurazione opzionali relative all'istanza.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Per ogni proprietà vengono visualizzate le seguenti informazioni:

Nome

Nome della proprietà.

Valore

Il valore della proprietà è impostato per l'istanza selezionata.

Descrizione

Descrizione testuale in cui sono riportate ulteriori informazioni sulla proprietà.

La tabella Proprietà aggiuntive contiene inoltre le seguenti opzioni.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il rispettivo valore predefinito oppure, se non è specificato il valore predefinito, il valore resta non definito.

GlassFish Server definisce le seguenti proprietà dell'istanza:

rendezvousOccurred

Specifica se l'istanza ha contattato il server di amministrazione del dominio (DAS, Domain Administration Server). I possibili valori sono i seguenti:

true

L'istanza ha contattato il DAS.

false

L'istanza non ha contattato il DAS.
