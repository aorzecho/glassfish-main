Modificare un'applicazione distribuita a un cluster

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Applications.

    Viene visualizzata la pagina Applications per il cluster.

  4. Nella tabella Deployed Applications della pagina Applications, fare clic sul nome dell'applicazione.

    Viene visualizzata la pagina Edit Application.

  5. Nella pagina Edit Application fornire le informazioni necessarie relative all'applicazione e salvare le modifiche.

    Per dettagli, vedere l'argomento Edit Application della guida in linea.
