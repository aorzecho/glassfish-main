Modificare un'applicazione distribuita in un cluster

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Dopo aver espanso il nodo Cluster, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per il cluster.

  4. Nella tabella Applicazioni distribuite della pagina Applicazioni, fare clic sul nome dell'applicazione.

    Viene visualizzata la pagina Modifica applicazione.

  5. Nella pagina Modifica applicazione fornire le informazioni necessarie relative all'applicazione e salvare le modifiche.

    Per dettagli, vedere l'argomento Edit Application della guida in linea.
