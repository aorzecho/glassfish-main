Visualizzare dati di monitoraggio dell'applicazione per un'istanza standalone di GlassFish Server

Prima di cominciare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio dell'applicazione.

    Viene visualizzata la pagina General Information per l'istanza.

  3. Nella pagina General Information, fare clic sulla scheda Monitor.

    Viene visualizzata la pagina Application Monitoring.

  4. Nell'elenco a discesa View, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.
  5. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Refresh.
