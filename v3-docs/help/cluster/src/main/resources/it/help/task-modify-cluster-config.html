Modificare la configurazione di un cluster

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Selezionare il nodo per il cluster.

  3. Fare clic sul nome della configurazione.

    Viene visualizzata la pagina Configuration per il cluster.

  4. Nella pagina Configurazione selezionare il componente della configurazione da modificare.

    Viene visualizzata la pagina per la modifica del componente selezionato della configurazione.

  5. Nella pagina per la modifica del componente selezionato, eseguire le modifiche necessarie e salvarle.

    Per dettagli, vedere la Guida in linea relativa alla pagina per la modifica del componente selezionato.
