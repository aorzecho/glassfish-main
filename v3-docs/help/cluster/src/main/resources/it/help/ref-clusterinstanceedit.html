Informazioni generali

La pagina Informazioni generali contiene le seguenti opzioni.

Avvia

Pulsante che consente di avviare l'istanza. Se l'istanza è in esecuzione, il pulsante è disattivato.

Arresta

Pulsante che consente di arrestare l'istanza. Se l'istanza viene arrestata, il pulsante è disattivato.

Riavvia

Pulsante che consente di riavviare l'istanza. Se l'istanza viene arrestata, il pulsante è disattivato.

Visualizza file di log

Pulsante che consente di visualizzare i file di log per l'istanza.

Recupera transazioni

Pulsante che consente di ripristinare transazioni per l'istanza.

Nome cluster

Nome assegnato al cluster al momento della creazione del cluster. Se si fa clic sul nome, viene visualizzata la pagina Informazioni generali per il cluster.

Nome istanza

Nome assegnato all'istanza al momento dell'aggiunta dell'istanza al cluster. Campo di sola lettura.

Stato

Indica se l'istanza è in esecuzione, se richiede il riavvio per l'aggiornamento con modifiche alla configurazione o se non è in esecuzione. Campo di sola lettura.

JVM

Se si fa clic su Report JVM, viene visualizzata una finestra distinta in cui sono disponibili i report relativi alla Java Virtual Machine, tra cui un report di riepilogo, un report di gestione memoria e garbage collection, un report sul caricamento delle classi e un dump del thread corrente.

Nodo

Nome del nodo nel quale si trova l'istanza. Se si fa clic sul nome, viene visualizzata la pagina Modifica nodo per il nodo.

Configurazione

Configurazione a cui l'istanza fa riferimento. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per l'istanza.

Porte HTTP

Lista separata da virgole dei numeri di porta su cui l'istanza resta ascolta i tipi di richiesta riportati di seguito.

  • Richieste di amministrazione

  • Richieste HTTP

  • Richieste HTTPS

Campo di sola lettura.

Porte IIOP

Lista separata da virgole dei numeri delle porte che l'istanza utilizza per i seguenti tipi di connessione riportati di seguito.

  • Connessioni IIOP sicure

  • Connessioni IIOP sicure con autenticazione del client

  • Connessioni IIOP

Campo di sola lettura.
