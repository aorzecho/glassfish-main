Nodi

Nella pagina Nodi viene visualizzata una lista di nodi del dominio. Per ogni nodo vengono visualizzate le seguenti informazioni:

Nome

Nome assegnato al nodo al momento della creazione. Se si fa clic sul nome, viene visualizzata la pagina Modifica nodo per il nodo.

Host nodo

Nome dell'host che il nodo rappresenta.

Tipo

Tipo del nodo, che può essere uno dei seguenti:

CONFIG

Il nodo non supporta la comunicazione remota.

DCOM

Il nodo supporta comunicazioni sul protocollo remoto DCOM (Distributed Component Object Model). Il protocollo DCOM è disponibile solo sui sistemi Windows.

SSH

Il nodo supporta la comunicazione su SSH (Secure Shell).

Istanze

Lista delle istanze che risiedono nel nodo con indicazione relativa all'eventuale esecuzione di ciascuna istanza:

  • Se l'istanza è in esecuzione, viene visualizzato un segno di spunta (✓).

  • Se l'istanza viene arrestata, viene visualizzato un punto esclamativo (!).

Se si fa clic sul nome di un'istanza, viene visualizzata la pagina Informazioni generali per tale istanza.

Azione

Nodi esclusivamente di tipo DCOM e SSH: un collegamento all'azione Ping per il nodo. Se si fa clic sul collegamento Ping viene eseguito il test della raggiungibilità del nodo sul protocollo di comunicazione supportato dal nodo.

Non sono possibili azioni per i nodi CONFIG.

La tabella Nodi contiene inoltre le seguenti opzioni.

Nuovo

Pulsante che consente di creare un nodo.

Elimina

Pulsante che consente di eliminare uno o più nodi selezionati.

Elimina e disinstalla

Pulsante per eliminare uno o più nodi selezionati e disinstallare il software GlassFish Server dall'host rappresentato da ciascun nodo.
