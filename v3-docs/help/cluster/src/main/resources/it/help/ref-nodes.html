Nodes

Nella pagina Nodes viene visualizzato un elenco di nodi del dominio. Per ogni nodo vengono visualizzate le seguenti informazioni:

Name

Nome assegnato al nodo al momento della creazione. Se si fa clic sul nome, viene visualizzata la pagina Edit Node per il nodo.

Node Host

Nome dell'host che il nodo rappresenta.

Type

Tipo del nodo, che può essere uno dei seguenti:

CONFIG

Il nodo non supporta la comunicazione remota.

SSH

Il nodo supporta la comunicazione su SSH (Secure Shell).

Instances

Elenco delle istanze presenti nel nodo con indicazione relativa all'eventuale esecuzione di ciascuna istanza:

  • Se l'istanza è in esecuzione, viene visualizzato un segno di spunta (✓).

  • Se l'istanza viene arrestata, viene visualizzato un punto esclamativo (!).

Se si fa clic sul nome di un'istanza, viene visualizzata la pagina General Information per tale istanza.

La tabella Nodes contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un nodo.

Delete

Pulsante che consente di eliminare uno o più nodi selezionati.
