Abilitare o disabilitare il bilanciamento del carico per un'applicazione distribuita in un cluster

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Dopo aver espanso il nodo Cluster, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per il cluster.

  4. Selezionare la casella di controllo in corrispondenza dell'applicazione.

  5. Nel menu a discesa Altre azioni, scegliere il comando appropriato.

    • Per abilitare il bilanciamento del carico per l'applicazione, scegliere Abilitazione load balancer.

    • Per disabilitare il bilanciamento del carico per l'applicazione, scegliere Disabilitazione load balancer.
