Abilitare o disabilitare una risorsa per un'istanza standalone di GlassFish Server

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Nella pagina Standalone Server Instances selezionare l'istanza.

    Viene visualizzata la pagina General Information per l'istanza.

  3. Nella pagina General Information, fare clic sulla scheda Resources.

    Viene visualizzata la pagina Resource per l'istanza.

  4. Selezionare la casella di controllo in corrispondenza della risorsa.
  5. Fare clic sul pulsante appropriato.
    • Per abilitare la risorsa, scegliere Enable.
    • Per disabilitare la risorsa, scegliere Disable.

      La disabilitazione di una risorsa non ne determina la rimozione.
