Rimuovere le applicazioni da un'istanza standalone di &ProductName

Se si rimuove un'applicazione, essa viene rimossa solo dalla destinazione corrente. Quando viene rimossa da una destinazione, l'applicazione continua a essere elencata nella pagina Applications, nella quale vengono visualizzate tutte le applicazioni distribuite in GlassFish Server. L'applicazione rimane inoltre distribuita su eventuali altre destinazioni sulle quali è stata effettuata la distribuzione.

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Nella pagina Istanze server standalone selezionare l'istanza.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali relativa all'istanza fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per l'istanza.

  4. Per ogni applicazione da rimuovere, selezionare la casella di controllo in corrispondenza dell'applicazione nella tabella Applicazioni distribuite.

  5. Fare clic su Rimuovi.

    Viene richiesto di confermare la rimozione delle applicazioni.

  6. Confermare la rimozione delle applicazioni.

    Le applicazioni vengono rimosse dalla tabella delle applicazioni distribuite visualizzata nella pagina.
