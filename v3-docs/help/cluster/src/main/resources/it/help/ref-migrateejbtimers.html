Esegui migrazione timer EJB

La pagina Esegui migrazione timer EJB contiene le seguenti opzioni.

Origine

Elenco a discesa di istanze di GlassFish Server arrestate nel cluster. È possibile eseguire la migrazione dei timer EJB solo dalle istanze che sono state arrestate.

Destinazione

Elenco a discesa di istanze di GlassFish Server in esecuzione nel cluster. È possibile eseguire la migrazione dei timer EJB solo in istanze in esecuzione.

Nota:

Se tutte le istanze nel cluster vengono arrestate, nella pagina viene visualizzato solo un messaggio di avvertenza che notifica che non vi sono istanze in esecuzione disponibili per la migrazione.

La pagina Esegui migrazione timer EJB contiene una tabella di timer EJB con le informazioni riportate di seguito.

Nome server

Lista delle istanze di GlassFish Server nel cluster.

Conteggio timer

Numero di timer persistenti in ciascuna istanza.
