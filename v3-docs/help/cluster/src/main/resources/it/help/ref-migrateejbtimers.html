Migrate EJB Timers

La pagina Migrate EJB Timers contiene le seguenti opzioni.

Source

Elenco a discesa di istanze di GlassFish Server arrestate nel cluster. È possibile eseguire la migrazione dei timer EJB solo dalle istanze che sono state arrestate.

Destination

Elenco a discesa di istanze di GlassFish Server in esecuzione nel cluster. È possibile eseguire la migrazione dei timer EJB solo in istanze in esecuzione.

Nota - Se tutte le istanze nel cluster vengono arrestate, nella pagina viene visualizzato solo un messaggio che notifica che sono disponibili istanze in esecuzione disponibili per la migrazione.

Attività correlate

  • Eseguire la migrazione dei timer EJB

Comandi secondari asadmin equivalenti

  • migrate-timers(1)
