Visualizzare i dati di monitoraggio delle risorse per un'istanza standalone di GlassFish Server

Prima di iniziare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio della risorsa.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Monitoraggio.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  4. Fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Monitoraggio risorse.

  5. Nell'elenco a discesa Visualizza, selezionare la risorsa per la quale si desidera visualizzare i dati di monitoraggio per le risorse.

  6. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Aggiorna.
