Arrestare istanze di GlassFish Server in cluster

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Per ogni istanza da arrestare, selezionare la casella di controllo corrispondente nella tabella Server Instances.
  5. Fare clic su Stop.

    Viene richiesto di confermare l'arresto delle istanze.

  6. Confermare l'arresto delle istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Server Instances quando l'istanza viene arrestata.
