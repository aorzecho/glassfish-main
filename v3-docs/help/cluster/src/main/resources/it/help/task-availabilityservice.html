Configurare le impostazioni di disponibilità per un cluster

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare la configurazione per il cluster da modificare.

  3. Nel nodo Configurazione, selezionare il nodo Servizio disponibilità.

    Viene visualizzata la pagina Servizio disponibilità.

  4. Nel campo Servizio disponibilità selezionare la casella Abilitato per abilitare la disponibilità per il cluster con la configurazione selezionata.

    Per disabilitare la disponibilità, rimuovere il segno di spunta dalla casella Servizio disponibilità abilitato.

  5. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà aggiuntiva per la disponibilità del cluster.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  6. Fare clic su Salva.
