Visualizzare l'elenco di risorse per un cluster

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Resources.

    Viene visualizzata la pagina Resources per il cluster.

  4. (Opzionale) Per visualizzare solo risorse di un tipo specifico, nell'elenco a discesa Filter selezionare il tipo di risorsa da visualizzare.
