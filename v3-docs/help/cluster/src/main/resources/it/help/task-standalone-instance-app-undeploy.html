Annullare la distribuzione delle applicazioni in un'istanza standalone di GlassFish Server

Se si annulla la distribuzione di un'applicazione, l'applicazione viene rimossa da tutte le destinazioni e dal dominio. Quando la distribuzione di un'applicazione viene annullata, l'applicazione non viene più elencata nella pagina Applications, nella quale sono visualizzate tutte le applicazioni distribuite in GlassFish Server.

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Nella pagina Istanze server standalone selezionare l'istanza.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali relativa all'istanza fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per l'istanza.

  4. Per ogni applicazione di cui annullare la distribuzione, selezionare la casella di controllo in corrispondenza dell'applicazione nella tabella Applicazioni distribuite.

  5. Fare clic su Annulla distribuzione.

    Viene richiesto di confermare l'annullamento della distribuzione delle applicazioni.

  6. Confermare l'annullamento della distribuzione delle applicazioni.

    La distribuzione delle applicazioni viene annullata e le applicazioni vengono rimosse dalla tabella delle applicazioni distribuite visualizzata nella pagina.
