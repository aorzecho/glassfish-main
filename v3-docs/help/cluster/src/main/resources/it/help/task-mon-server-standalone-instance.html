Visualizzare dati di monitoraggio del server virtuale per un'istanza standalone di GlassFish Server

Prima di cominciare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio per i server virtuali.

    Viene visualizzata la pagina General Information per l'istanza.

  3. Nella pagina General Information, fare clic sulla scheda Monitor.

    Viene visualizzata la pagina Application Monitoring.

  4. Fare clic sulla scheda Server.

    Viene visualizzata la pagina Server Monitoring.

  5. Nell'elenco a discesa View, selezionare il componente per il quale si desidera visualizzare i dati di monitoraggio.
  6. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Refresh.
