Istanze server standalone

Nella pagina Istanze server standalone è possibile creare e gestire istanze standalone di GlassFish Server.

Nella pagina Istanze server standalone viene visualizzata una lista di tutte le istanze standalone del dominio. Per ogni istanza vengono fornite le seguenti informazioni.

Nome

Nome assegnato all'istanza quando l'istanza è stata creata. Se si fa clic sul nome, viene visualizzata la pagina Informazioni generali per l'istanza.

Fattore ponderale LB

Numero intero che rappresenta il fattore ponderale per il bilanciamento del carico dell'istanza.

Il fattore ponderale per il bilanciamento del carico determina la proporzione di tutte le richieste inviate al cluster che l'istanza dovrebbe elaborare.

Ad esempio, in un cluster a due istanze può essere necessaria un'istanza per elaborare una di quattro richieste e un'altra istanza per elaborarne tre di quattro. In questa situazione, impostare su 1 il fattore ponderale dell'istanza che elabora una delle quattro richieste e su 3 il fattore ponderale dell'altra istanza. Se si preferisce utilizzare valori percentuali, impostare i fattori ponderali delle istanze rispettivamente su 25 e 75.

Il fattore ponderale predefinito è 100.

Configurazione

Configurazione selezionata per l'istanza al momento della creazione. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per l'istanza.

Nodo

Nome del nodo nel quale si trova l'istanza. Se si fa clic sul nome, viene visualizzata la pagina Modifica nodo per il nodo.

Stato

Indicazione relativa all'eventuale esecuzione dell'istanza:

La tabella Istanze server contiene inoltre le seguenti opzioni.

Nuovo

Pulsante che consente di creare un'istanza standalone.

Elimina

Pulsante che consente di eliminare una o più istanze standalone selezionate.

Avvia

Pulsante che consente di avviare una o più istanze standalone selezionate.

Arresta

Pulsante che consente di arrestare una o più istanze standalone selezionate.
