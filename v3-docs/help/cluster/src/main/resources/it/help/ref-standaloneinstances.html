Standalone Server Instances

Nella pagina Standalone Server Instances è possibile creare e gestire istanze standalone di GlassFish Server.

Nella pagina Standalone Server Instances viene visualizzato un elenco di tutte le istanze standalone del dominio. Per ogni istanza vengono fornite le seguenti informazioni.

Name

Nome assegnato all'istanza quando l'istanza è stata creata. Se si fa clic sul nome, viene visualizzata la pagina General Information per l'istanza.

LB Weight

Numero intero che rappresenta il peso per il bilanciamento del carico dell'istanza.

Il peso per il bilanciamento del carico determina la proporzione di tutte le richieste inviate al cluster che l'istanza dovrebbe elaborare.

Ad esempio, in un cluster a due istanze può essere necessaria un'istanza per elaborare una di quattro richieste e un'altra istanza per elaborarne tre di quattro. In questa situazione, impostare su 1 il peso dell'istanza che elabora una delle quattro richieste e su 3 il peso dell'altra istanza. Se si preferisce utilizzare valori percentuali, impostare i pesi delle istanze rispettivamente su 25 e 75.

Il peso predefinito è 100.

Configuration

Configurazione selezionata per l'istanza al momento della creazione. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per l'istanza.

Node

Nome del nodo nel quale si trova l'istanza. Se si fa clic sul nome, viene visualizzata la pagina Edit Node per il nodo.

Status

Indicazione relativa all'eventuale esecuzione dell'istanza:

La tabella Server Instances contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un'istanza standalone.

Delete

Pulsante che consente di eliminare una o più istanze standalone selezionate.

Start

Pulsante che consente di avviare una o più istanze standalone selezionate.

Stop

Pulsante che consente di arrestare una o più istanze standalone selezionate.
