Eliminare istanze standalone di GlassFish Server

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Per ogni istanza da eliminare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Server Instances.
  3. Fare clic su Delete.

    Viene richiesto di confermare l'eliminazione delle istanze.

  4. Confermare l'eliminazione delle istanze.

    Le istanze vengono eliminate e rimosse dalla tabella di istanze della pagina.
