Eliminare le istanze standalone di GlassFish Server

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Per ogni istanza da eliminare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Istanze server.

  3. Fare clic su Elimina.

    Viene richiesto di confermare l'eliminazione delle istanze.

  4. Confermare l'eliminazione delle istanze.

    Le istanze vengono eliminate e rimosse dalla tabella di istanze della pagina.
