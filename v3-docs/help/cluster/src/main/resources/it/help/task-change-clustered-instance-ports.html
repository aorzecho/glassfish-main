Modificare i numeri di porta di un'istanza di GlassFish Server in cluster

I numeri di porta di un'istanza di GlassFish Server in cluster sono impostati inizialmente nella configurazione che l'istanza eredita dal cluster padre. È necessario modificare questi numeri di porta se, ad esempio, due istanze cluster risiedono nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze condividono la stessa configurazione.

I numeri di porta di un'istanza sono archiviati come proprietà di sistema Java. Tali proprietà vengono passate al launcher dell'applicazione Java tramite l'opzione -D del launcher stesso all'avvio di &ProductName.

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Dopo aver espanso il nodo Cluster, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic sulla scheda Properties.

    Viene visualizzata la pagina Proprietà di sistema.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per l'istanza:

    • Port numbers that are predefined in the instance's configuration

    • Proprietà personalizzate definite per l'istanza, se presenti

  6. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore di sostituzione relativo alla proprietà.

  7. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile digitare il nome della proprietà.

    3. Nel campo Valore di sostituzione digitare il valore.

  8. Fare clic su Salva.
