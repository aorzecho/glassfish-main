Modificare i numeri di porta di un'istanza di GlassFish Server in cluster

I numeri di porta di un'istanza di GlassFish Server in cluster sono impostati inizialmente nella configurazione che l'istanza eredita dal cluster padre. È necessario modificare questi numeri di porta se, ad esempio, due istanze cluster si trovano nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze condividono la stessa configurazione.

I numeri di porta di un'istanza sono archiviati come proprietà di sistema Java. Tali proprietà vengono passate al programma di avvio dell'applicazione Java tramite l'opzione -D del programma di avvio dell'applicazione Java quando viene avviato GlassFish Server.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina General Information per l'istanza.

  5. Nella pagina General Information, fare clic sulla scheda Properties.

    Viene visualizzata la pagina System Properties.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per l'istanza:

    • Port numbers that are predefined in the instance's configuration

    • Proprietà personalizzate definite per l'istanza, se presenti

  6. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Override Value relativo alla proprietà.
  7. Impostare i valori per eventuali proprietà aggiuntive necessarie.
    1. Nella tabella Additional Properties fare clic su Add Property.

      Viene aggiunta una riga alla tabella Additional Properties.

    2. Nel campo Variable Name digitare il nome della proprietà.
    3. Nel campo Override Value digitare il valore.
  8. Fare clic su Save.
