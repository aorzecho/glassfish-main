Arrestare le istanze standalone di &ProductName

  1. Nella struttura ad albero di navigazione selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Per ogni istanza da arrestare, selezionare la casella di controllo corrispondente nella tabella Istanze server.

  3. Fare clic su Arresta.

    Viene richiesto di confermare l'arresto delle istanze.

  4. Confermare l'arresto delle istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Istanze server quando l'istanza viene arrestata.
