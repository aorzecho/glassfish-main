Arrestare istanze standalone di GlassFish Server

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Per ogni istanza da arrestare, selezionare la casella di controllo corrispondente nella tabella Server Instances.
  3. Fare clic su Stop.

    Viene richiesto di confermare l'arresto delle istanze.

  4. Confermare l'arresto delle istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Server Instances quando l'istanza viene arrestata.
