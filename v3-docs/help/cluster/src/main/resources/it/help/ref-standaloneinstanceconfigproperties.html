Proprietà di sistema dell'istanza

Nella pagina Proprietà di sistema dell'istanza viene visualizzata una lista delle proprietà di sistema Java impostate per l'istanza di GlassFish Server corrente. Le proprietà di sistema Java vengono passate al launcher dell'applicazione Java tramite l'opzione -D del launcher stesso all'avvio di GlassFish Server.

Queste proprietà prevalgono sulle definizioni delle proprietà relative alle impostazioni delle porte, contenute nella configurazione dell'istanza. Le impostazioni predefinite per le porte devono essere ignorate se, ad esempio, due istanze in cluster risiedono nello stesso host. In questa situazione è necessario che le impostazioni delle porte per una delle istanze siano ignorate, perché le due istanze condividono la stessa configurazione.

Per informazioni sulle impostazioni predefinite della porta in una configurazione, vedere Predefined Configuration System Properties.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Per ogni proprietà vengono visualizzate le seguenti informazioni:

Nome variabile istanza

Nome della proprietà di sistema.

Valore corrente

Valore attualmente impostato per la proprietà. Campo di sola lettura.

Valore di sostituzione

Valore su cui verrà impostata la proprietà dopo il salvataggio delle modifiche.

La tabella Proprietà aggiuntive contiene inoltre le seguenti opzioni.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il rispettivo valore predefinito oppure, se non è specificato il valore predefinito, il valore resta non definito.
