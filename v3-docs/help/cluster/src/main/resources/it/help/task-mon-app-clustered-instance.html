Visualizzare i dati di monitoraggio dell'applicazione per un'istanza di GlassFish Server in cluster

Prima di cominciare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio dell'applicazione.

    Viene visualizzata la pagina General Information per l'istanza.

  5. Nella pagina General Information, fare clic sulla scheda Monitor.

    Viene visualizzata la pagina Application Monitoring.

  6. Nell'elenco a discesa View, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.
  7. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Refresh.
