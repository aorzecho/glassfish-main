Visualizzare i dati di monitoraggio delle applicazioni per un'istanza di GlassFish Server in cluster

Prima di iniziare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Dopo aver espanso il nodo Cluster, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio dell'applicazione.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic sulla scheda Monitoraggio.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  6. Nell'elenco a discesa Visualizza, selezionare l'applicazione per la quale si desidera visualizzare i dati di monitoraggio.

  7. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Aggiorna.
