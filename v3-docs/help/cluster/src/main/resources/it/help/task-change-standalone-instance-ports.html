Modificare i numeri di porta di un'istanza standalone di GlassFish Server

I numeri di porta di un'istanza standalone di GlassFish Server sono impostati inizialmente nella configurazione a cui l'istanza fa riferimento. È necessario modificare questi numeri di porta se, ad esempio, due istanze che fanno riferimento alla stessa configurazione si trovano nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze fanno riferimento alla stessa configurazione.

I numeri di porta di un'istanza sono archiviati come proprietà di sistema Java. Tali proprietà vengono passate al programma di avvio dell'applicazione Java tramite l'opzione -D del programma di avvio dell'applicazione Java quando viene avviato GlassFish Server.

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina General Information per l'istanza.

  3. Nella pagina General Information, fare clic sulla scheda Properties.

    Viene visualizzata la pagina System Properties.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per l'istanza:

    • Port numbers that are predefined in the instance's configuration

    • Proprietà personalizzate definite per l'istanza, se presenti

  4. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Override Value relativo alla proprietà.
  5. Impostare i valori per eventuali proprietà aggiuntive necessarie.
    1. Nella tabella Additional Properties fare clic su Add Property.

      Viene aggiunta una riga alla tabella Additional Properties.

    2. Nel campo Variable Name digitare il nome della proprietà.
    3. Nel campo Override Value digitare il valore.
  6. Fare clic su Save.
