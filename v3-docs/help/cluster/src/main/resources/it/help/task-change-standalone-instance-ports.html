Modificare i numeri di porta di un'istanza standalone di GlassFish Server

I numeri di porta di un'istanza standalone di GlassFish Server sono impostati inizialmente nella configurazione a cui l'istanza fa riferimento. È necessario modificare questi numeri di porta se, ad esempio, due istanze che fanno riferimento alla stessa configurazione risiedono nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze fanno riferimento alla stessa configurazione.

I numeri di porta di un'istanza sono archiviati come proprietà di sistema Java. Tali proprietà vengono passate al launcher dell'applicazione Java tramite l'opzione -D del launcher stesso all'avvio di &ProductName.

  1. Nella struttura ad albero selezionare il nodo Istanze standalone.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Properties.

    Viene visualizzata la pagina Proprietà di sistema.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per l'istanza:

    • Port numbers that are predefined in the instance's configuration

    • Proprietà personalizzate definite per l'istanza, se presenti

  4. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore di sostituzione relativo alla proprietà.

  5. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile digitare il nome della proprietà.

    3. Nel campo Valore di sostituzione digitare il valore.

  6. Fare clic su Salva.
