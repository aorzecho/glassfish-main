Impostare i valori predefiniti per le proprietà di sistema della configurazione

I valori predefiniti delle proprietà di configurazione sono definiti nella configurazione alla quale fa riferimento il cluster o l'istanza.

  1. Nella struttura ad albero, espandere il nodo per la configurazione.
  2. Nell'elenco dei tipi di proprietà nel nodo espanso, selezionare System Properties.

    Viene visualizzata la pagina System Properties per la configurazione selezionata.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per la configurazione:

    • Port numbers that are predefined when the configuration was created

    • Proprietà personalizzate aggiunte alla configurazione, se presenti.

  3. Per assicurare che le modifiche vengano applicate senza necessità di riavviare il cluster o l'istanza, selezionare l'opzione Dynamic Reconfiguration.
  4. Per ogni proprietà per la quale si imposta un valore predefinito, digitare il nuovo valore nel campo Override Value relativo alla proprietà.
  5. Impostare i valori predefiniti per eventuali proprietà aggiuntive necessarie.
    1. Nella tabella Additional Properties fare clic su Add Property.

      Viene aggiunta una riga alla tabella Additional Properties.

    2. Nel campo Variable Name digitare il nome della proprietà.
    3. Nel campo Default Value, digitare il valore.
  6. Fare clic su Save.
