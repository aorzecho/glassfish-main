Impostare i valori predefiniti per le proprietà di sistema della configurazione

I valori predefiniti delle proprietà di configurazione sono definiti nella configurazione alla quale fa riferimento il cluster o l'istanza.

  1. Nell'albero di navigazione espandere il nodo per la configurazione.

  2. Nella lista dei tipi di proprietà nel nodo espanso, selezionare Proprietà di sistema.

    Viene visualizzata la pagina Proprietà di sistema per la configurazione selezionata.

    Nella pagina vengono visualizzati i valori correnti delle seguenti proprietà di sistema per la configurazione:

    • Port numbers that are predefined when the configuration was created

    • Proprietà personalizzate aggiunte alla configurazione, se presenti.

  3. Per assicurare che le modifiche vengano applicate senza necessità di riavviare il cluster o l'istanza, selezionare l'opzione Riconfigurazione dinamica.

  4. Per ogni proprietà per la quale si imposta un valore predefinito, digitare il nuovo valore nel campo Valore di sostituzione relativo alla proprietà.

  5. Impostare i valori predefiniti per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile digitare il nome della proprietà.

    3. Nel campo Valore predefinito, digitare il valore.

  6. Fare clic su Salva.
