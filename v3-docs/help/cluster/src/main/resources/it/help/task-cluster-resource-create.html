Creare una risorsa per un cluster

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Resources.

    Viene visualizzata la pagina Resources per il cluster.

  4. Nel menu a tendina New selezionare il tipo di risorsa da creare.

    Viene visualizzata la pagina per la creazione di una risorsa del tipo selezionato.

  5. Nella pagina per la creazione di una risorsa del tipo selezionato, apportare le modifiche necessarie e salvarle.

    Per dettagli, vedere la Guida in linea relativa alla pagina per la creazione di una risorsa del tipo selezionato.

    Nota - Quando si crea la risorsa, assicurarsi di specificare il cluster come destinazione.
