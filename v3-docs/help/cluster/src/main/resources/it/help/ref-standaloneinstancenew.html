New Standalone Server Instance

Nella pagina New Standalone Server Instance è possibile creare una istanza standalone di GlassFish Server.

La pagina New Standalone Server Instance contiene le seguenti opzioni.

Instance Name

Nome dell'istanza che si sta creando.

Il nome deve soddisfare i seguenti requisiti:

  • Il nome deve iniziare con una lettera, un numero o un carattere di sottolineatura.

  • Il nome può contenere solo i seguenti caratteri:

    • Lettere minuscole

    • Lettere maiuscole

    • Numeri

    • Trattino

    • Punto

    • Carattere di sottolineatura

  • Il nome deve essere univoco nel dominio e non deve essere il nome di un'altra istanza di GlassFish Server, di un cluster, di una configurazione con nome o di un nodo.

  • Il nome non deve essere domain, server né nessun'altra parola chiave riservata per GlassFish Server.

Node

Elenco discesa di nodi esistenti che definiscono gli host in cui può essere posizionata l'istanza. È necessario selezionare un nodo dall'elenco. Se l'istanza deve essere creata nell'host in cui è in esecuzione il server di amministrazione del dominio (DAS, Domain Administration Server), selezionare il nodo predefinito localhost-domain.

Configuration

Elenco a discesa di configurazioni con nome esistenti utilizzabili dall'istanza. È necessario selezionare una configurazione dall'elenco. L'istanza utilizzerà la configurazione selezionata al momento della creazione dell'istanza stessa.

Make a Copy of the Selected Configuration

Se l'opzione è selezionata, la configurazione selezionata viene copiata al momento della creazione dell'istanza. Alla copia della configurazione viene assegnato il nome instance-name-config, in cui instance-name è il nome dell'istanza digitato nel campo Instance Name.

È necessario selezionare tale opzione se nell'elenco a discesa Configuration è selezionato default-config. La configurazione default-config può essere copiata ma non utilizzata come riferimento.

Reference the Selected Configuration

Se l'opzione è selezionata, l'istanza utilizza la configurazione con nome esistente specificata.

L'opzione non deve essere selezionata se nell'elenco a discesa Configuration è selezionato default-config. La configurazione default-config non può essere utilizzata come riferimento, ma solo copiata.
