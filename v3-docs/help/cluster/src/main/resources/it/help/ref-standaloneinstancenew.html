Nuova istanza server standalone

Nella pagina Nuova istanza server standalone è possibile creare una istanza standalone di GlassFish Server.

La pagina Nuova istanza server standalone contiene le seguenti opzioni.

Nome istanza

Nome dell'istanza che si sta creando.

Il nome deve soddisfare i seguenti requisiti:

  • Il nome può contenere solo caratteri ASCII.

  • Il nome deve iniziare con una lettera, un numero o un carattere di sottolineatura.

  • Il nome può contenere solo i seguenti caratteri:

    • Lettere minuscole

    • Lettere maiuscole

    • Numeri

    • Trattino

    • Punto

    • Carattere di sottolineatura

  • Il nome deve essere univoco nel dominio e non deve essere il nome di un'altra istanza di GlassFish Server, di un cluster, di una configurazione denominata o di un nodo.

  • Il nome non deve essere domain, server né nessun'altra parola chiave riservata per GlassFish Server.

Nodo

Elenco a discesa di nodi esistenti che definiscono gli host in cui può risiedere l'istanza. È necessario selezionare un nodo dalla lista. Se l'istanza deve essere creata nell'host in cui è in esecuzione il server di amministrazione del dominio (DAS, Domain Administration Server), selezionare il nodo predefinito localhost-domain.

Configurazione

Elenco a discesa di configurazioni con nome esistenti utilizzabili dall'istanza. È necessario selezionare una configurazione dalla lista. L'istanza utilizzerà la configurazione selezionata al momento della creazione dell'istanza stessa.

Crea una copia della configurazione selezionata

Se l'opzione è selezionata, la configurazione selezionata viene copiata al momento della creazione dell'istanza. Alla copia della configurazione viene assegnato il nome instance-name-config, in cui instance-name è il nome dell'istanza digitato nel campo Nome istanza.

È necessario selezionare tale opzione se nell'elenco a discesa Configurazione è selezionato default-config. La configurazione default-config può essere copiata ma non utilizzata come riferimento.

Crea riferimento alla configurazione selezionata

Se l'opzione è selezionata, l'istanza utilizza la configurazione con nome esistente specificata.

L'opzione non deve essere selezionata se nell'elenco a discesa Configurazione è selezionato default-config. La configurazione default-config non può essere utilizzata come riferimento, ma solo copiata.
