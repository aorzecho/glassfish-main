Configurare GMS

Il servizio GMS (Group Management Service) è un servizio in-process che offre monitoraggio del cluster, appartenenza del cluster e servizi di comunicazione di gruppo. Esegue le funzioni seguenti:

  • Notifica ai moduli registrati in un'istanza di GlassFish Server l'aggiunta o la rimozione di una o più istanze membro da un cluster oppure un errore sospetto o confermato di tali istanze.

  • Consente di inviare e ricevere messaggi tra processi di un gruppo.

  1. Nell'albero di navigazione espandere il nodo Configurazione.

  2. Selezionare la configurazione per il cluster da modificare.

  3. Nel nodo della configurazione, selezionare il nodo Group Management Service.

    Viene visualizzata la pagina Group Management Service.

  4. Nel campo Numero massimo heartbeat mancati immettere il numero massimo di tentativi di monitoraggio precedenti alla conferma, da parte del servizio GMS, di un probabile errore.

    Il valore predefinito è 3.

  5. Nel campo Frequenza heartbeat immettere il periodo di tempo tra i tentativi di monitoraggio.

    Il valore predefinito è 2000 millisecondi.

  6. Nel campo Timeout di individuazione gruppo immettere il tempo di attesa del servizio GMS per l'individuazione di altri membri del cluster.

    Il valore predefinito è 5000 millisecondi.

  7. Nel campo Tempo di attesa verifica errore immettere il tempo al termine del quale contrassegnare un errore sospetto come verificato.

    Il valore predefinito è 1500 millisecondi.

  8. Per ripristinare valori predefiniti, fare clic sul pulsante Carica impostazioni predefinite.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.
