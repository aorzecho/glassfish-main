Configurare il servizio GMS

Il servizio GMS (Group Management Service) è un servizio in-process che offre monitoraggio del cluster, appartenenza del cluster e servizi di comunicazione di gruppo. Esegue le funzioni seguenti:

  • Notifica ai moduli registrati in un'istanza di GlassFish Server l'aggiunta o la rimozione di una o più istanze membro da un cluster oppure un errore sospetto o confermato di tali istanze.

  • Consente di inviare e ricevere messaggi tra processi di un gruppo.

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Selezionare la configurazione per il cluster da modificare.
  3. Nel nodo della configurazione, selezionare il nodo Group Management Service.

    Viene visualizzata la pagina Group Management Service.

  4. Nel campo Maximum Missed Heartbeats immettere il numero massimo di tentativi di monitoraggio precedenti alla conferma, da parte del servizio GMS, di un probabile errore.

    Il valore predefinito è 3.

  5. Nel campo Heartbeat Frequency immettere il periodo di tempo tra i tentativi di monitoraggio.

    Il valore predefinito è 2000 millisecondi.

  6. Nel campo Group Discovery Timeout immettere il tempo di attesa del servizio GMS per l'individuazione di altri membri del cluster.

    Il valore predefinito è 5000 millisecondi.

  7. Nel campo Failure Verification Wait Time immettere il tempo al termine del quale contrassegnare un errore sospetto come verificato.

    Il valore predefinito è 1500 millisecondi.

  8. Per ripristinare valori predefiniti, fare clic sul pulsante Load Defaults.
  9. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive.

    In GlassFish Server non è definita alcuna proprietà GMS aggiuntiva.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  10. Fare clic su Save.
