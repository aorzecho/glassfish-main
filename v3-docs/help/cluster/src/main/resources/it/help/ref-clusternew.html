Nuovo cluster

Nella pagina New Cluster è possibile creare un cluster.

La pagina New Cluster contiene le seguenti opzioni.

Cluster Name

Nome del cluster.

Il nome deve soddisfare i seguenti requisiti:

  • Il nome deve iniziare con una lettera, un numero o un carattere di sottolineatura.

  • Il nome può contenere solo i seguenti caratteri:

    • Lettere minuscole

    • Lettere maiuscole

    • Numeri

    • Trattino

    • Punto

    • Carattere di sottolineatura

  • Il nome deve essere univoco nel dominio e non deve essere il nome di un altro cluster, di una configurazione con nome, di un'istanza di GlassFish Server o di un nodo.

  • Il nome non deve essere domain, server né nessun'altra parola chiave riservata per GlassFish Server.

Configuration

Elenco discesa di configurazioni con nome esistenti utilizzabili dal cluster. Un cluster richiede una configurazione con nome per la definizione della configurazione di tutte le istanze che vengono aggiunte al cluster. È necessario selezionare una configurazione dall'elenco. Il cluster utilizzerà la configurazione selezionata al momento della creazione del cluster.

Make a Copy of the Selected Configuration

Se l'opzione è selezionata, la configurazione selezionata viene copiata al momento della creazione del cluster. Alla copia della configurazione viene assegnato il nome cluster-name-config, in cui cluster-name è il nome del cluster digitato nel campo Cluster Name.

È necessario selezionare tale opzione se nell'elenco a discesa Configuration è selezionato default-config. La configurazione default-config può essere copiata ma non utilizzata come riferimento.

Reference the Selected Configuration

Se l'opzione è selezionata, il cluster utilizza la configurazione con nome esistente specificata.

L'opzione non deve essere selezionata se nell'elenco a discesa Configuration è selezionato default-config. La configurazione default-config non può essere utilizzata come riferimento, ma solo copiata.

Server Instances to be Created

Elenco di istanze di &ProductName da creare al momento della creazione del cluster. Per ogni istanza vengono fornite le seguenti informazioni.

Instance Name

Nome dell'istanza.

Weight

Numero intero che rappresenta il peso per il bilanciamento del carico dell'istanza.

Il peso per il bilanciamento del carico determina la proporzione di tutte le richieste inviate al cluster che l'istanza dovrebbe elaborare.

Ad esempio, in un cluster a due istanze può essere necessaria un'istanza per elaborare una di quattro richieste e un'altra istanza per elaborarne tre di quattro. In questa situazione, impostare su 1 il peso dell'istanza che elabora una delle quattro richieste e su 3 il peso dell'altra istanza. Se si preferisce utilizzare valori percentuali, impostare i pesi delle istanze rispettivamente su 25 e 75.

Il peso predefinito è 100.

Node

Elenco discesa di nodi esistenti in cui può essere posizionata l'istanza. È necessario selezionare un nodo dall'elenco. L'istanza verrà posizionata nel nodo selezionato al momento della creazione del cluster.

La tabella Server Instances to be Created contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un'istanza. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Server Instances to be Created.

Delete

Pulsante che consente di eliminare un'istanza.
