Abilitare o disabilitare una risorsa a livello globale

  1. Nell'albero di navigazione espandere il nodo Cluster.

  2. Dopo aver espanso il nodo Cluster, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Risorse per il cluster.

  4. Selezionare la casella di controllo in corrispondenza della risorsa.

  5. Fare clic sul pulsante appropriato.

    • Per abilitare la risorsa, scegliere Abilita.

    • Per disabilitare la risorsa, scegliere Disabilita.

      La disabilitazione di una risorsa non ne determina la rimozione.
