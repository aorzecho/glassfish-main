Avviare istanze standalone di GlassFish Server

Prima di cominciare

Verificare che siano soddisfatti i seguenti prerequisiti:

  • Tutti i nodi in cui si trovano le istanze sono attivati per la comunicazione remota.

  • L'utente del server di amministrazione del dominio (DAS, Domain Administration Server) può utilizzare SSH (Secure Shell) per registrarsi sull'host per qualsiasi nodo in cui si trovano le istanze.

Se uno o più prerequisiti non sono soddisfatti, avviare ogni istanza singolarmente dalla riga di comando utilizzando il comando secondario start-local-instance(1).

  1. Nella struttura ad albero selezionare il nodo Standalone Instances.

    Viene visualizzata la pagina Standalone Server Instances.

  2. Per ogni istanza da avviare, selezionare la casella di controllo corrispondente nella tabella Server Instances.
  3. Fare clic su Start.

    Viene richiesto di confermare l'avvio delle istanze.

  4. Confermare l'avvio delle istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Server Instances quando l'istanza viene avviata.
