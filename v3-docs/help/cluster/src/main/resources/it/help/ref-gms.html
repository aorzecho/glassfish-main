Modifica Group Management Service

Nella pagina Group Management Service è possibile modificare le impostazioni GMS.

Il servizio GMS (Group Management Service) è un servizio in-process che offre monitoraggio del cluster, appartenenza del cluster e servizi di comunicazione di gruppo. Il servizio GMS è stato creato all'inizio di uno stack configurabile di protocolli di monitoraggio dell'integrità e di individuazione di appartenenza al gruppo. Esegue le funzioni seguenti:

  • Notifica ai moduli registrati in un'istanza di GlassFish Server l'aggiunta o la rimozione di una o più istanze membro da un cluster oppure un errore sospetto o confermato di tali istanze.

  • Consente di inviare e ricevere messaggi tra processi di un gruppo.

La pagina Group Management Service contiene le seguenti opzioni.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Campo di sola lettura.

Numero massimo heartbeat mancati

Numero di tentativi di monitoraggio dopo i quali è probabile un errore dell'istanza del server. Il valore predefinito è 3.

Frequenza heartbeat

Intervallo di tempo che intercorre tra i tentativi di monitoraggio. Il valore predefinito è 2000 millisecondi.

Timeout di individuazione gruppo

Tempo di attesa del servizio GMS per l'individuazione di altri membri del cluster. Il valore predefinito è 5000 millisecondi.

Tempo di attesa verifica errore

Tempo dopo il quale un errore sospetto viene contrassegnato come verificato. Il valore predefinito è 1500 millisecondi.

Proprietà aggiuntive

Tabella di proprietà aggiuntive per il servizio GMS.

In GlassFish Server non è definita alcuna proprietà GMS aggiuntiva.

Per ogni proprietà vengono visualizzate le seguenti informazioni:

Nome

Nome della proprietà.

Valore

Il valore della proprietà è impostato per l'istanza selezionata.

Descrizione

Descrizione testuale in cui sono riportate ulteriori informazioni sulla proprietà.

La tabella Proprietà aggiuntive contiene inoltre le seguenti opzioni.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. Quando una proprietà viene eliminata, viene ripristinato il rispettivo valore predefinito oppure, se non è specificato il valore predefinito, il valore resta non definito.
