Distribuire un'applicazione in un cluster

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Applications.

    Viene visualizzata la pagina Applications per il cluster.

  4. Nella pagina Applications, fare clic su Deploy.

    Viene visualizzata la pagina Deploy Applications or Modules.

  5. Nella pagina Deploy Applications or Modules, specificare i dati necessari sull'applicazione, quindi fare clic su OK.

    Per dettagli, vedere l'argomento Deploy Applications or Modules della Guida in linea.
