Arrestare un cluster

Se si arresta un cluster, vengono arrestate tutte le istanze di GlassFish Server in esecuzione nel cluster.

  1. Nella struttura ad albero, selezionare il nodo Clusters.

    Viene visualizzata la pagina Clusters.

  2. Nella pagina Clusters selezionare la casella di controllo in corrispondenza del nome del cluster.
  3. Fare clic su Stop Cluster.

    Viene richiesto di confermare che si desidera arrestare il cluster.

  4. Confermare che si desidera arrestare il cluster.

    Il cluster viene arrestato. Lo stato di ogni istanza del cluster viene aggiornato nella pagina Clusters quando l'istanza viene arrestata.
