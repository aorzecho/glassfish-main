Arrestare un cluster

Se si arresta un cluster, vengono arrestate tutte le istanze di GlassFish Server in esecuzione nel cluster.

  1. Nell'albero di navigazione selezionare il nodo Cluster.

    Viene visualizzata la pagina Cluster.

  2. Nella pagina Cluster selezionare la casella di controllo in corrispondenza del nome del cluster.

  3. Fare clic su Arresta cluster.

    Viene richiesto di confermare che si desidera arrestare il cluster.

  4. Confermare che si desidera arrestare il cluster.

    Il cluster viene arrestato. Lo stato di ogni istanza del cluster viene aggiornato nella pagina Cluster quando l'istanza viene arrestata.
