Cluster

Nella pagina Cluster viene visualizzato una lista di cluster nel dominio. Per ciascun cluster vengono visualizzate le seguenti informazioni:

Nome

Nome assegnato al cluster al momento della creazione del cluster. Se si fa clic sul nome, viene visualizzata la pagina Informazioni generali per il cluster.

Configurazione

Configurazione selezionata per il cluster al momento della creazione del cluster. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per il cluster.

Istanze

Nome e stato di ogni istanza del cluster. Se si fa clic sul nome dell'istanza, viene visualizzata la pagina Informazioni generali per l'istanza.

La tabella Cluster contiene inoltre le seguenti opzioni.

Nuovo

Pulsante che consente di creare un cluster.

Elimina

Pulsante che consente di eliminare un cluster.

Avvia cluster

Pulsante che consente di avviare un cluster.

Arresta cluster

Pulsante che consente di arrestare un cluster.
