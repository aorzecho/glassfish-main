Cluster

Nella pagina Clusters viene visualizzato un elenco di cluster nel dominio. Per ciascun cluster vengono visualizzate le seguenti informazioni:

Name

Nome assegnato al cluster al momento della creazione del cluster. Se si fa clic sul nome, viene visualizzata la pagina General Information per il cluster.

Configuration

Configurazione selezionata per il cluster al momento della creazione del cluster. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurations per il cluster.

Instances

Nome e stato di ogni istanza del cluster. Se si fa clic sul nome dell'istanza, viene visualizzata la pagina General Information per l'istanza.

La tabella Clusters contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un cluster.

Delete

Pulsante che consente di eliminare un cluster.

Start Cluster

Pulsante che consente di avviare un cluster.

Stop Cluster

Pulsante che consente di arrestare un cluster.
