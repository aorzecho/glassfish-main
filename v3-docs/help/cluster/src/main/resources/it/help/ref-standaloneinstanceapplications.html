Applicazioni

Nella pagina Applicazioni è possibile eseguire i task riportati di seguito nell'istanza standalone di GlassFish Server selezionata.

  • Visualizzazione e gestione delle applicazioni già distribuite all'istanza

  • Distribuzione di più applicazioni all'istanza

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Nella pagina Applicazioni viene visualizzata la lista delle applicazioni distribuite all'istanza selezionata. Per ogni applicazione vengono fornite le seguenti informazioni.

Nome

Nome dell'applicazione.

Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo da due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

Abilitato

Indicazione relativa all'eventuale abilitazione dell'applicazione:

  • Se l'applicazione è abilitata, viene visualizzato un segno di spunta (✓).

  • Se l'applicazione è disabilitata, viene visualizzato un simbolo a forma di croce (✗).

LB abilitato

Indicazione relativa all'eventuale abilitazione dell'applicazione per il bilanciamento del carico:

  • Se l'applicazione è abilitata per il bilanciamento del carico, viene visualizzato un segno di spunta (✓).

  • Se l'applicazione è disabilitata per il bilanciamento del carico, viene visualizzato un simbolo a forma di croce (✗).

Motori

Tipi di contenitori utilizzati nell'applicazione. I tipi di contenitori possono essere di uno dei seguenti tipi:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld: contenitore per Contexts and Dependency Injection per le applicazioni Java EE Platform

La tabella Applicazioni distribuite contiene inoltre le seguenti opzioni.

Distribuisci

Pulsante che consente di distribuire un'applicazione.

Annulla distribuzione

Pulsante che consente di annullare la distribuzione di una o più applicazioni selezionate.

Se si annulla la distribuzione di un'applicazione, l'applicazione viene rimossa da tutte le destinazioni e dal dominio. Quando la distribuzione di un'applicazione viene annullata, l'applicazione non viene più elencata nella pagina Applications, nella quale sono visualizzate tutte le applicazioni distribuite in GlassFish Server.

Per rimuovere applicazioni solo dalla destinazione corrente, utilizzare il pulsante Rimuovi di questa pagina.

Rimuovi

Pulsante che consente di rimuovere una o più applicazioni selezionate.

Se si rimuove un'applicazione, essa viene rimossa solo dalla destinazione corrente. Quando viene rimossa da una destinazione, l'applicazione continua a essere elencata nella pagina Applications, nella quale vengono visualizzate tutte le applicazioni distribuite in GlassFish Server. L'applicazione rimane inoltre distribuita su eventuali altre destinazioni sulle quali è stata effettuata la distribuzione.

Per rimuovere applicazioni da tutte le destinazioni e dal dominio, utilizzare il pulsante Annulla distribuzione di questa pagina.

Altre azioni

Elenco a discesa che consente di eseguire azioni aggiuntive in una o più applicazioni selezionate.

Abilita

Abilita una o più applicazioni selezionate.

Disabilita

Disabilita una o più applicazioni selezionate.

Abilitazione load balancer

Abilita una o più applicazioni selezionate per il bilanciamento del carico.

Disbilitazione load balancer

Disabilita una o più applicazioni selezionate per il bilanciamento del carico.
