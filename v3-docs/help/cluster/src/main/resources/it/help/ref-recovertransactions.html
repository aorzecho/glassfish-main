Recupero transazioni

Nella pagina Recupera transazioni è possibile ripristinare transazioni per un'istanza di GlassFish Server.

Se l'istanza è in esecuzione, la pagina Recupera transazioni contiene solo il testo Il recupero viene eseguito dallo stesso server in esecuzione e il pulsante OK.

Se l'istanza non è in esecuzione, la pagina Recupera transazioni contiene le seguenti opzioni.

Nome istanza

Nome assegnato all'istanza quando l'istanza è stata creata. Campo di sola lettura.

Server di destinazione

Altra istanza in grado di ripristinare le transazioni nell'istanza corrente.

Directory log delle transazioni

Directory nella quale sono memorizzati i log delle transazioni. L'impostazione predefinita è /logs/tx nella directory di dominio.

I log delle transazioni sono memorizzati nella sottodirectory tx della directory specificata nell'impostazione Posizione log delle transazioni del servizio transazioni relativo alla configurazione cluster. Il valore predefinito Posizione log delle transazioni è la directory specificata nel campo Radice log disponibile nella pagina Attributi dominio all'interno della scheda Avanzate del nodo GlassFish Server.

Task correlati

  • Recuperare le transazioni per un'istanza standalone di GlassFish Server
