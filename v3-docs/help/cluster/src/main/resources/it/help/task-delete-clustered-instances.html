Eliminare istanze di GlassFish Server in cluster

Avvertenza - Se si utilizza un cluster JMS (Java Message Service) con un broker principale, non eliminare l'istanza associata al broker principale. Se è necessario eliminare tale istanza, utilizzare il comando secondario change-master-broker(1) per assegnare il broker principale a un'altra istanza.

  1. Nella struttura ad albero, espandere il nodo Clusters.
  2. Dopo aver espanso il nodo Clusters, selezionare il nodo per tale cluster.

    Viene visualizzata la pagina General Information per il cluster.

  3. Nella pagina General Information, fare clic sulla scheda Instances.

    Viene visualizzata la pagina Clustered Server Instances per il cluster.

  4. Per ogni istanza da eliminare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Server Instances.
  5. Fare clic su Delete.

    Viene richiesto di confermare l'eliminazione delle istanze.

  6. Confermare l'eliminazione delle istanze.

    Le istanze vengono eliminate e rimosse dalla tabella di istanze della pagina.
