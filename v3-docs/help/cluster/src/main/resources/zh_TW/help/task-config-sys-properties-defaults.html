為組態系統特性設定預設值

組態特性的預設值定義於叢集或實例所參照的組態中。

  1. 在瀏覽樹狀結構中，展開組態的節點。

  2. 從展開的節點下的特性類型清單中，選取 「系統特性」。

    此時將開啟選定組態的 「系統特性」 頁面。

    此頁面會顯示組態之下列系統特性目前的值：

    • 建立組態時預先定義的連接埠號碼

    • 增加至組態的自訂特性 (如果有)

  3. 為了確保您不需要重新啟動叢集或實例就能套用變更，請選取 「動態重新組態」 選項。

  4. 針對您要為其設定預設值的每個特性，請在特性的 「預設值」 欄位中鍵入新值。

  5. 為所需的任何其他特性設定預設值。

    1. 在 「其他特性」 表格中，按一下 「增加特性」。

      隨即會在其他特性表中增加一列。

    2. 在 「實例變數名稱」 欄位中，鍵入特性名稱。

    3. 在 「預設值」 欄位中鍵入值。

  6. 按一下 「儲存」。
