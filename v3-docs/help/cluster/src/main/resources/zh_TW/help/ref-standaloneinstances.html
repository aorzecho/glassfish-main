獨立伺服器實例

使用 「獨立伺服器實例」 頁面可建立和管理獨立 GlassFish Server 實例。

「獨立伺服器實例」 頁面會顯示網域中所有獨立實例的清單。對於每個實例，均提供以下資訊。

名稱

建立實例時指定給實例的名稱。按一下該名稱，隨即會開啟實例的一般資訊頁面。

負載平衡權數

一個整數，代表實例的負載平衡權數。

負載平衡權數可決定送至叢集供實例處理之所有請求的比例。

例如，在有兩個實例的叢集中，您可能需要讓其中一個實例處理每四個請求中的一個請求，而讓另一個實例處理每四個請求中的三個請求。在這種情況下，可將處理每四個請求中之一個請求的實例權數設定為 1，將另一個實例權數設定為 3。如果您要使用百分比，請將這兩個實例的權數分別設定為 25 與 75。

預設權數為 100。

組態

建立實例時為實例選取的組態。按一下組態，隨即會開啟實例的 「組態」 頁面。

節點

實例所在之節點的名稱。按一下該名稱，隨即會開啟節點的編輯節點頁面。

狀態

指示實例是否正在執行中。

伺服器實例表也包含下列選項。

新建

建立獨立實例的按鈕。

刪除

刪除一或多個所選取獨立實例的按鈕。

啟動

啟動一或多個所選取獨立實例的按鈕。

停止

停止一或多個所選取獨立實例的按鈕。
