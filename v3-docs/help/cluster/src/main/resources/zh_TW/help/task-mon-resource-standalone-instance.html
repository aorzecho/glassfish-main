檢視獨立 GlassFish Server 實例的資源監視資料

開始之前

在檢視監視資料之前，您必須配置監視服務。

  1. 在瀏覽樹狀結構中，選取 [獨立實例] 節點。

    此時將開啟 [獨立伺服器實例] 頁面。

  2. 選取要檢視其資源監視資料的實例。

    此時將開啟實例的 [一般資訊] 頁面。

  3. 在 [一般資訊] 頁面上，按一下 [監視] 標籤。

    此時將開啟 [應用程式監視] 頁面。

  4. 按一下 [資源] 標籤。

    此時將開啟 [資源監視] 頁面。

  5. 從 [檢視] 下拉式清單中，選取您要檢視其資源監視資料的資源。
  6. 若要更新顯示的監視資料，請按一下 [重新整理]。
