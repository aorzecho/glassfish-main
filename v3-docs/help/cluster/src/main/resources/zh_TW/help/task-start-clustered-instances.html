啟動叢集 GlassFish Server 實例

開始前

請確保符合以下先決條件︰

  • 實例所在的所有節點都已啟用，以進行遠端通訊。

  • 網域管理伺服器 (DAS) 的使用者可以使用安全 Shell (SSH) 登入實例所在之任何節點的主機。

如果不符合上述任一個先決條件，請使用 start-local-instance(1) 子命令，從命令行個別啟動各個實例。

  1. 在瀏覽樹狀結構中，展開 「叢集」 節點。

  2. 在展開的 「叢集」 節點下，選取該叢集的節點。

    此時將開啟叢集的 「一般資訊」 頁面。

  3. 在 「一般資訊」 頁面上，按一下 「實例」 標籤。

    此時將開啟該叢集的 「叢集伺服器實例」 頁面。

  4. 針對您要啟動的每個實例，請在伺服器實例表中選取該實例旁邊的核取方塊。

  5. 按一下 「啟動」。

    系統會要求您確認是否要啟動這些實例。

  6. 請確認您要啟動這些實例。

    實例啟動後，便會更新伺服器實例表中各個實例的狀態。
