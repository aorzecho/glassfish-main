新建獨立伺服器實例

使用 「新建獨立伺服器實例」 頁面可建立獨立 GlassFish Server 實例。

「新建獨立伺服器實例」 包含下列選項。

實例名稱

要建立之實例的名稱。

名稱必須符合下列需求：

  • 名稱只能包含 ASCII 字元。

  • 名稱必須以字母、數字或底線為開頭。

  • 名稱只能包含下列字元：

    • 小寫字母

    • 大寫字母

    • 數字

    • 連字符

    • 小數點號

    • 底線

  • 名稱在網域中必須是唯一的，且不得為另一個 GlassFish Server 實例、叢集、已命名組態或節點的名稱。

  • 名稱不得為 domain、server 或 GlassFish Server 所保留的任何其他關鍵字。

節點

現有節點的下拉式清單，能定義可存在實例的主機。您必須從清單中選取一個節點。如果要在執行網域管理伺服器 (DAS) 的主機上建立實例，請選取預先定義的節點 localhost-domain。

組態

實例可使用之現有已命名組態的下拉式清單。您必須從清單中選取一個組態。建立實例時，實例將使用選取的組態。

複製選取的組態

如果選取此選項，就會在建立實例時複製選取的組態。為組態副本指定的名稱是 instance-name-config，其中 instance-name 是您在「實例名稱」欄位中鍵入的實例名稱。

如果已從 「組態」 下拉式清單中選取 default-config，則必須選取此選項。default-config 組態僅可複製，無法參照。

參照選取的組態

如果選取此選項，實例將會使用指定的現有已命名組態。

如果已從 「組態」 下拉式清單中選取 default-config，則不能選取此選項。default-config 組態無法參照，僅可複製。
