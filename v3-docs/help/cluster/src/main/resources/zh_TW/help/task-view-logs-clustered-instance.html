檢視叢集 GlassFish Server 實例的記錄檔

開始前

您要檢視其記錄檔的實例必須正在執行中。

  1. 在瀏覽樹狀結構中，展開 「叢集」 節點。

  2. 在展開的 「叢集」 節點下，選取該叢集的節點。

    此時將開啟叢集的 「一般資訊」 頁面。

  3. 在 「一般資訊」 頁面上，按一下 「實例」 標籤。

    此時將開啟該叢集的 「叢集伺服器實例」 頁面。

  4. 選取要檢視其記錄檔的實例。

    此時將開啟實例的 「一般資訊」 頁面。

  5. 在 「一般資訊」 頁面上，按一下 「檢視記錄檔」。

    此時將開啟 「記錄檢視器」 視窗。

    注意:

    實例必須已在執行中。如果實例未在執行中，便會停用 「檢視記錄檔」 按鈕。
