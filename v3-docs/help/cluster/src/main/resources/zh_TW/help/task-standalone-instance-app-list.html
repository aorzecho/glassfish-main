列出已部署至獨立 GlassFish Server 實例的應用程式

  1. 在瀏覽樹狀結構中，選取 [獨立實例] 節點。

    此時將開啟 [獨立伺服器實例] 頁面。

  2. 在 [獨立伺服器實例] 頁面中，選取實例。

    此時將開啟實例的 [一般資訊] 頁面。

  3. 在實例的 [一般資訊] 頁面上，按一下 [應用程式] 標籤。

    此時將開啟實例的 [應用程式] 頁面。
