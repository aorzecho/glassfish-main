取消部署叢集中的應用程式

取消部署應用程式會從所有目標與網域中移除應用程式。取消部署應用程式後，在顯示 GlassFish Server 中已部署之所有應用程式的 [應用程式] 頁面上，便不再列出該應用程式。

  1. 在瀏覽樹狀結構中，展開 [叢集] 節點。
  2. 在展開的 [叢集] 節點下，選取該叢集的節點。

    此時將開啟叢集的 [一般資訊] 頁面。

  3. 在 [一般資訊] 頁面上，按一下 [應用程式] 標籤。

    此時將開啟叢集的 [應用程式] 頁面。

  4. 對於您要取消部署的每個應用程式，請在 [已部署的應用程式] 表格中，選取該應用程式旁邊的核取方塊。
  5. 按一下 [取消部署]。

    系統會要求您確認是否要取消部署應用程式。

  6. 請確認您要取消部署應用程式。

    此時便會從此頁面上的已部署應用程式表格中取消部署並移除應用程式。
