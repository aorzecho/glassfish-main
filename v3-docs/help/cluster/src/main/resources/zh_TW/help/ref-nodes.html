節點

「節點」 頁面會顯示網域中的節點清單。對於每個節點，都會顯示以下資訊：

名稱

建立節點時指定給節點的名稱。按一下該名稱，隨即會開啟節點的編輯節點頁面。

節點主機

節點代表的主機名稱。

類型

節點的類型，是下列類型之一：

組態

節點不支援遠端通訊。

DCOM

節點支援透過分散式元件物件模型 (DCOM) 遠端協定的通訊。DCOM 協定只適用於 Windows 系統。

SSH

節點支援透過安全 shell (SSH) 進行通訊。

實例

位於節點上的實例清單，指出各個實例是否正在執行中：

  • 如果實例正在執行中，便會顯示核取記號 (✓)。

  • 如果實例已停止，便會顯示驚嘆號 (!)。

按一下實例名稱，隨即會開啟實例的 「一般資訊」 頁面。

動作

僅限節點類型 DCOM 和 SSH：節點之 Ping 動作的連結。按一下 Ping 連結，測試是否可以透過節點所支援的通訊協定連線節點。

無法對 CONFIG 節點執行任何動作。

「節點」 表也包含下列選項。

新建

建立節點的按鈕。

刪除

刪除一或多個所選取節點的按鈕。

刪除並解除安裝

刪除一個或多個所選節點並解除安裝每個所刪除節點代表之主機中的 GlassFish Server 軟體的按鈕。
