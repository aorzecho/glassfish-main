節點

[節點] 頁面會顯示網域中的節點清單。對於每個節點，都會顯示以下資訊：

名稱

建立節點時指定給節點的名稱。按一下該名稱，隨即會開啟節點的 編輯節點 頁面。

節點主機

節點代表的主機名稱。

類型

節點的類型，是下列類型之一：

CONFIG

節點不支援遠端通訊。

SSH

節點支援透過安全 shell (SSH) 進行通訊。

實例

位於節點上的實例清單，指出各個實例是否正在執行中：

  • 如果實例正在執行中，便會顯示核取記號 (✓)。

  • 如果實例已停止，便會顯示驚嘆號 (!)。

按一下實例名稱，隨即會開啟實例的 [一般資訊] 頁面。

[節點] 表也包含下列選項。

新建

建立節點的按鈕。

刪除

刪除一或多個所選取節點的按鈕。
