新建叢集伺服器實例

使用 [新建叢集伺服器實例] 頁面，可將 GlassFish Server 實例增加至現有叢集。

[新建叢集伺服器實例] 包含下列選項。

叢集名稱

建立叢集時指定給叢集的名稱。本欄位是唯讀的。

實例名稱

要建立之實例的名稱。

名稱必須符合下列需求：

  • 名稱只能包含 ASCII 字元。

  • 名稱必須以字母、數字或底線為開頭。

  • 名稱只能包含下列字元：

    • 小寫字母

    • 大寫字母

    • 數字

    • 連字符

    • 小數點號

    • 底線

  • 名稱在網域中必須是唯一的，且不得為另一個 GlassFish Server 實例、叢集、已命名配置或節點的名稱。

  • 名稱不得為 domain、server 或 GlassFish Server 所保留的任何其他關鍵字。

節點

現有節點的下拉式清單，能定義可存在實例的主機。您必須從清單中選取一個節點。如果要在執行 Domain Administration Server (DAS) 的主機上建立實例，請選取預先定義的節點 localhost-domain。
