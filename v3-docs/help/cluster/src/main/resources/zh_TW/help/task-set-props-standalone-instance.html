為獨立 GlassFish Server 實例設定僅限實例的特性

GlassFish Server 實例的特性可增加有關實例的選擇性配置資訊。

  1. 在瀏覽樹狀結構中，選取 [獨立實例] 節點。

    此時將開啟 [獨立伺服器實例] 頁面。

  2. 選取只要修改其實例特性的實例。

    此時將開啟實例的 [一般資訊] 頁面。

  3. 在 [一般資訊] 頁面上，按一下 [特性] 標籤。
  4. 如果需要，請按一下 [實例特性] 標籤。

    此時將開啟 [實例特性] 頁面。

    此頁面會顯示為所選取 GlassFish Server 實例設定的特性清單。

  5. 對於您要修改的每個現有特性，請在該特性的 [值] 欄位中鍵入新值。
  6. 為所需的任何其他特性設定值。
    1. 在 [其他特性] 表格中，按一下 [增加特性]。

      隨即會在其他特性表中增加一列。

    2. 在 [名稱] 欄位中，鍵入特性名稱。
    3. 在 [值] 欄位中，鍵入值。
    4. 在 [說明] 欄位中，鍵入特性的簡短說明。
  7. 按一下 [儲存]。
