停止叢集

停止叢集會停止叢集內所有執行中的 GlassFish Server 實例。

  1. 在瀏覽樹狀結構中，選取 「叢集」 節點。

    此時將開啟 「叢集」 頁面。

  2. 在 「叢集」 頁面上，選取叢集名稱旁邊的核取方塊。

  3. 按一下 「停止叢集」。

    系統會要求您確認是否要停止叢集。

  4. 請確認您要停止叢集。

    叢集即會停止。實例停止後，便會更新 「叢集」 頁面上叢集中各個實例的狀態。
