叢集

「叢集」 頁面會顯示網域中的叢集清單。對於每個叢集，將會顯示以下資訊：

名稱

建立叢集時指定給叢集的名稱。按一下該名稱，隨即會開啟叢集的一般資訊頁面。

組態

建立叢集時為叢集選取的組態。按一下該組態，即會開啟叢集的 「組態」 頁面。

實例

叢集中每個實例的名稱和狀態。按一下實例名稱，隨即會開啟該實例的一般資訊頁面。

叢集表也包含下列選項。

新建

建立叢集的按鈕。

刪除

刪除叢集的按鈕。

啟動叢集

啟動叢集的按鈕。

停止叢集

停止叢集的按鈕。
