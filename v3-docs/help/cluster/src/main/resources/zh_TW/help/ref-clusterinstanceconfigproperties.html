實例系統特性

[實例系統特性] 頁面會顯示為目前 GlassFish Server 實例設定的 Java 系統特性清單。GlassFish Server 啟動時，Java 系統特性會透過 Java 應用程式啟動程式的 -D 選項，傳遞至 Java 應用程式啟動程式。

這些特性會置換實例配置中連接埠設定的特性定義。例如，如果出現兩個叢集實例位於相同主機的情況，則必須置換預先定義的連接埠設定。在此情況下，必須置換其中一個實例的連接埠設定，因為兩個實例會共用相同的配置。

如需有關配置中預先定義之連接埠設定的資訊，請參閱「預先定義的配置系統特性」。

[實例名稱] 欄位是唯讀欄位，會顯示目前實例的名稱。

對於每個特性，都會顯示以下資訊：

實例變數名稱

系統特性的名稱。

目前值

目前為特性所設定的值。本欄位是唯讀的。

置換值

儲存變更之後，特性即將設定的值。

其他特性表也包含下列選項。

增加特性

增加特性的按鈕。按一下此按鈕即可在其他特性表中增加一列。

刪除特性

刪除一或多個所選取特性的按鈕。刪除的任何特性都會復原成其預設值或未定義 (如果未設定任何預設值)。

相關作業

  • 為配置系統特性設定預設值
