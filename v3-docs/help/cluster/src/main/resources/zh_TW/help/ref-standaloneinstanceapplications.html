應用程式

使用 「應用程式」 頁面，可對選取的獨立 GlassFish Server 實例執行下列作業：

  • 檢視和管理已建置至實例的應用程式

  • 將更多應用程式建置至實例

「實例名稱」 欄位是唯讀欄位，會顯示目前實例的名稱。

「應用程式」 頁面會顯示建置至所選取實例的應用程式清單。對於每個應用程式，均提供以下資訊：

名稱

應用程式名稱。

名稱中可以包含一個選擇性的版本識別碼，後面接著名稱，兩者間用冒號 (:) 隔開。 如需有關模組與應用程式版本的更多資訊，請參閱 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

已啟用

指示應用程式是否已啟用：

  • 如果應用程式已啟用，便會顯示核取記號 (✓)。

  • 如果應用程式已停用，便會顯示叉號 (✗)。

負載平衡已啟用

指示應用程式是否已啟用負載平衡：

  • 如果應用程式已啟用負載平衡，便會顯示核取記號 (✓)。

  • 如果應用程式已停用負載平衡，便會顯示叉號 (✗)。

引擎

應用程式使用的容器類型。容器類型可以是下列任何類型：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — 用於 Java EE 平台應用程式之環境與相依性注入的容器

已建置的應用程式表也包含下列選項。

建置

建置應用程式的按鈕。

解除建置

解除建置一或多個所選取應用程式的按鈕。

取消建置應用程式會從所有目標與網域中移除應用程式。取消建置應用程式後，在顯示 GlassFish Server 中已建置之所有應用程式的應用程式頁面上，便不再列出該應用程式。

若要只從目前的目標移除應用程式，請使用此頁面上的 「移除」 按鈕。

移除

移除一或多個所選取應用程式的按鈕。

移除應用程式只會從目前目標中移除應用程式。從目標中移除應用程式時，在顯示 GlassFish Server 中已建置之所有應用程式的 「應用程式」 頁面上，仍會繼續列出該應用程式。在已建置該應用程式的任何其他目標上，該應用程式仍然保持已建置狀態。

若要從所有目標與網域中移除應用程式，請使用此頁面上的 「未建置」 按鈕。

更多動作

可對一或多個所選取應用程式執行其他動作的下拉式清單。

啟用

啟用一或多個所選取的應用程式。

停用

停用一或多個所選取的應用程式。

負載平衡程式啟用

啟用一或多個所選取應用程式的負載平衡。

負載平衡程式停用

停用一或多個所選取應用程式的負載平衡。

相關作業

  • 將應用程式建置至獨立 GlassFish Server 實例

  • 列示建置至獨立 GlassFish Server 實例的應用程式

  • 從獨立 GlassFish Server 實例解除建置應用程式

  • 從獨立 GlassFish Server 實例移除應用程式

  • 編輯建置至獨立 GlassFish Server 實例的應用程式

  • 啟用或停用建置至獨立 GlassFish Server 實例的應用程式

  • 啟用或停用建置至獨立 GlassFish Server 實例之應用程式的負載平衡
