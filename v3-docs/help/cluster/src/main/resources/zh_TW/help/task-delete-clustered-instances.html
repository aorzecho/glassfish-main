刪除叢集 GlassFish Server 實例

注意 - 如果透過主代理程式使用 Java Message Service (JMS) 叢集，請勿刪除與主代理程式相關聯的實例。如果必須刪除此實例，請使用 change-master-broker(1) 子指令，將主代理程式指定給不同的實例。

  1. 在瀏覽樹狀結構中，展開 [叢集] 節點。
  2. 在展開的 [叢集] 節點下，選取該叢集的節點。

    此時將開啟叢集的 [一般資訊] 頁面。

  3. 在 [一般資訊] 頁面上，按一下 [實例] 標籤。

    此時將開啟該叢集的 [叢集伺服器實例] 頁面。

  4. 針對您要刪除的每個實例，請在伺服器實例表中選取該實例旁邊的核取方塊。
  5. 按一下 [刪除]。

    系統會要求您確認是否要刪除這些實例。

  6. 請確認您要刪除這些實例。

    這些實例會刪除，並從頁面上的實例表中移除。
