測試是否可以連線節點

若要在代表遠端主機的節點上建立或啟動 GlassFish Server 實例，則 DAS 必須可以連線節點。嘗試在這類節點上建立或啟動實例之前，請測試 DAS 是否可以連線節點。

您所測試節點的類型必須是 DCOM 或 SSH。因為類型為 CONFIG 的節點未啟用遠端通訊，所以您無法測試類型為 CONFIG 的節點。

開始前

請確定節點所代表的主機上已組態節點類型適用的協定。透過此協定，可啟用節點的遠端通訊功能。

  • 如果節點類型為 DCOM，則必須組態分散式元件物件模型 (DCOM) 協定。

  • 如果節點類型為 SSH，則必須組態安全 Shell (SSH) 協定。

    1. 在瀏覽樹狀結構中，選取 「節點」 節點。

      節點頁面會隨即開啟。此頁面會列出目前網域中的所有節點。

    2. 在節點表格中，請按一下所測試節點的 Ping 連結。

      「節點」頁面中會加入指出 DAS 是否可以連線節點的訊息。
