列示獨立 GlassFish Server 實例的資源

  1. 在瀏覽樹狀結構中，選取 「獨立實例」 節點。

    此時將開啟 「獨立伺服器實例」 頁面。

  2. 在 「獨立伺服器實例」 頁面中，選取實例。

    此時將開啟實例的 「一般資訊」 頁面。

  3. 在 「一般資訊」 頁面上，按一下 「資源」 標籤。

    此時將開啟實例的 「資源」 頁面。

  4. 若要僅顯示特定類型的資源，請從 「篩選器」 下拉式清單中選取要顯示的資源類型。
