列出已部署至叢集的應用程式

  1. 在瀏覽樹狀結構中，展開 [叢集] 節點。
  2. 在展開的 [叢集] 節點下，選取該叢集的節點。

    此時將開啟叢集的 [一般資訊] 頁面。

  3. 在 [一般資訊] 頁面上，按一下 [應用程式] 標籤。

    此時將開啟叢集的 [應用程式] 頁面。
