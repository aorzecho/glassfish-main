刪除叢集

刪除叢集會刪除叢集與叢集中的所有 GlassFish Server 實例。刪除叢集之前，不需要刪除叢集中的實例。

如果沒有共用叢集所使用的配置，則配置也會一併刪除。

開始之前

確定您要刪除的叢集已停止。

  1. 在瀏覽樹狀結構中，選取 [叢集] 節點。

    此時將開啟 [叢集] 頁面。

  2. 在 [叢集] 頁面上，選取叢集名稱旁邊的核取方塊。
  3. 按一下 [刪除]。

    系統會要求您確認是否要刪除叢集與叢集中的所有實例。

  4. 請確認要刪除叢集與叢集中的所有實例。

    該叢集便會刪除，並從 [叢集] 頁面上的叢集表中移除。
