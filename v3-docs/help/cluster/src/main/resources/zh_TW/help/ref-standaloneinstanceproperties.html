實例特性

[實例特性] 頁面會顯示為目前 GlassFish Server 實例設定的特性清單。這些特性可增加有關實例的選擇性配置資訊。

[實例名稱] 欄位是唯讀欄位，會顯示目前實例的名稱。

對於每個特性，都會顯示以下資訊：

名稱

特性的名稱。

值

為所選取實例設定的特性值。

說明

提供有關特性之更多資訊的文字說明。

其他特性表也包含下列選項。

增加特性

增加特性的按鈕。按一下此按鈕即可在其他特性表中增加一列。

刪除特性

刪除一或多個所選取特性的按鈕。刪除的任何特性都會復原成其預設值或未定義 (如果未設定任何預設值)。

GlassFish Server 會定義下列實例特性：

rendezvousOccurred

指定實例是否已與 Domain Administration Server (DAS) 取得連絡。可能值如下：

true

實例已與 DAS 取得連絡。

false

實例尚未與 DAS 取得連絡。
