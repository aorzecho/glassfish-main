將應用程式建置至叢集

  1. 在瀏覽樹狀結構中，展開 「叢集」 節點。

  2. 在展開的 「叢集」 節點下，選取該叢集的節點。

    此時將開啟叢集的 「一般資訊」 頁面。

  3. 在 「一般資訊」 頁面上，按一下 「應用程式」 標籤。

    此時將開啟叢集的 「應用程式」 頁面。

  4. 在 「應用程式」 頁面上，按一下 「建置」。

    「建置應用程式或模組」 頁面會隨即開啟。

  5. 在 「建置應用程式或模組」 頁面上，提供有關應用程式的必要資訊，然後按一下 「確定」。

    如需詳細資訊，請參閱 Deploy Applications or Modules 線上說明。
