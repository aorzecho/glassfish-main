啟動叢集

啟動叢集會啟動叢集中所有尚未執行的 GlassFish Server 實例。

開始之前

請確保符合以下先決條件︰

  • 叢集中的實例所在的所有節點都已啟用，以進行遠端通訊。

  • Domain Administration Server (DAS) 使用者可以使用安全 shell (SSH) 登入叢集中的實例所在之任何節點的主機。

如果不符合上述任一個先決條件，請使用 start-local-instance(1)子指令，從指令行個別啟動各個實例，以啟動叢集。

  1. 在瀏覽樹狀結構中，選取 [叢集] 節點。

    此時將開啟 [叢集] 頁面。

  2. 在 [叢集] 頁面上，選取叢集名稱旁邊的核取方塊。
  3. 按一下 [啟動叢集]。

    系統會要求您確認是否要啟動叢集。

  4. 請確認您要啟動叢集。

    叢集即會啟動。實例啟動後，便會更新 [叢集] 頁面上叢集中各個實例的狀態。
