停止叢集 GlassFish Server 實例

  1. 在瀏覽樹狀結構中，展開 「叢集」 節點。

  2. 在展開的 「叢集」 節點下，選取該叢集的節點。

    此時將開啟叢集的 「一般資訊」 頁面。

  3. 在 「一般資訊」 頁面上，按一下 「實例」 標籤。

    此時將開啟該叢集的 「叢集伺服器實例」 頁面。

  4. 針對您要停止的每個實例，請在伺服器實例表中選取該實例旁邊的核取方塊。

  5. 按一下 「停止」。

    系統會要求您確認是否要停止這些實例。

  6. 請確認您要停止這些實例。

    實例停止後，便會更新伺服器實例表中各個實例的狀態。
