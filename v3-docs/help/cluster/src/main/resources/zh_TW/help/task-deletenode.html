刪除節點

刪除節點會將該節點從 Domain Administration Server (DAS) 的配置中移除。刪除節點上存在的最後一個 GlassFish Server 實例時，會刪除節點的目錄和檔案。

開始之前

請確認要刪除的節點上沒有任何 GlassFish Server 實例。

  1. 在瀏覽樹狀結構中，選取 [節點] 節點。

    此時將開啟 [節點] 頁面。

  2. 對於您要刪除的每個節點，請在節點表中選取該節點旁邊的核取方塊。

    備註 - 無法刪除預先定義的節點 localhost-domain。

  3. 按一下 [刪除]。

    系統會要求您確認是否要刪除該節點。

  4. 請確認您要刪除節點。

    該節點便會刪除，並從 [節點] 頁面上的節點表中移除。
