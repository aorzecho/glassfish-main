一般資訊

[一般資訊] 頁面包含下列選項。

啟動

啟動實例的按鈕。如果實例正在執行中，便會關閉此按鈕。

停止

停止實例的按鈕。如果實例已停止，便會關閉此按鈕。

重新啟動

重新啟動實例的按鈕。如果實例已停止，便會關閉此按鈕。

檢視記錄檔

檢視實例記錄檔的按鈕。

回復作業事件

回復實例的作業事件的按鈕。

叢集名稱

建立叢集時指定給叢集的名稱。按一下該名稱，隨即會開啟叢集的 [一般資訊] 頁面。

實例名稱

將實例增加至叢集時指定給實例的名稱。本欄位是唯讀的。

狀態

指示實例是在執行中 (需要重新啟動才能更新配置變更) 還是不在執行中。本欄位是唯讀的。

JVM

如果按一下 [JVM 報告]，會開啟另外一個視窗並顯示 Java 虛擬機器的相關報告，包括摘要報告、記憶體管理和資源回收報告、類別載入報告以及目前執行緒傾印。

節點

實例所在之節點的名稱。按一下該名稱，隨即會開啟節點的 編輯節點 頁面。

配置

實例參照的配置。按一下配置，隨即會開啟實例的 [配置] 頁面。

HTTP 連接埠

實例偵聽下列請求類型所用連接埠的連接埠號碼清單 (以逗號分隔)：

  • 管理請求

  • HTTP 請求

  • HTTPS 請求

本欄位是唯讀的。

IIOP 連接埠

實例針對下列連線類型所用連接埠的連接埠號碼清單 (以逗號分隔)：

  • 安全 IIOP 連線

  • 使用用戶端認證的安全 IIOP 連線

  • IIOP 連線

本欄位是唯讀的。
