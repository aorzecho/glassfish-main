遷移 EJB 計時器

「遷移 EJB 計時器」 頁面包含下列選項。

來源

叢集中已停止之 GlassFish Server 實例的下拉式清單。EJB 計時器只能從已停止的實例進行遷移。

目標

叢集內執行中 GlassFish Server 實例的下拉式清單。EJB 計時器只能遷移到執行中的實例。

注意:

如果叢集中的所有實例均已停止，此頁面只會顯示一個警告，說明沒有執行中的實例可供遷移 EJB 計時器。

「遷移 EJB 計時器」頁面包含 EJB 計時器表格與下列資訊。

伺服器名稱

叢集中的 GlassFish Server 實例清單。

計時器計數

每個實例上的永久性計時器數目。

相關作業
