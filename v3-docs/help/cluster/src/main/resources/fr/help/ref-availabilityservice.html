Service de disponibilité

La page Service de disponibilité vous permet de modifier les paramètres de disponibilité d'un cluster.

Elle contient les options suivantes.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Service de disponibilité

Si la case Activé est cochée, la disponibilité est activée pour le cluster. Cette option est sélectionnée par défaut.

Autres propriétés

Tableau des propriétés supplémentaires pour le Service de disponibilité.

GlassFish Server ne définit aucune autre propriété de Service de disponibilité.

Pour chaque propriété, les informations suivantes sont affichées :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété définie pour l'instance sélectionnée.

Description

Texte de description qui fournit de plus amples informations sur la propriété.

Le tableau Propriétés supplémentaires contient également les options suivantes.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Le fait de cliquer sur ce bouton permet d'ajouter une ligne au tableau Propriétés supplémentaires.

Supprimer des propriétés

Bouton permettant de supprimer une ou plusieurs propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est définie, n'est pas définie.
