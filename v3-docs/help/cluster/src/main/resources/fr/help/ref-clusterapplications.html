Applications

Utilisez la page Applications pour effectuer les tâches suivantes sur le cluster en cours :

  • Afficher et gérer les applications déjà déployées sur le cluster

  • Déployer davantage d'applications sur le cluster

Le champ Nom du cluster est un champ en lecture seule qui affiche le nom du cluster en cours.

La page Applications présente la liste des applications déployées sur le cluster en cours. Pour chaque application, les informations suivantes sont fournies :

Nom

Nom de l'application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à "Module and Application Versions" dans le manuel Oracle GlassFish Server Application Deployment Guide.

Activé

Mention indiquant si l'application est activée :

  • Si l'application est activée, une coche (✓) apparaît.

  • Si l'application est désactivée, une croix (✗) apparaît.

EC activé

Mention indiquant si l'application est activée pour l'équilibrage de charge :

  • Si l'application est activée pour l'équilibrage de charge, une coche (✓) apparaît.

  • Si l'application est désactivée pour l'équilibrage de charge, une croix (✗) apparaît.

Moteurs

Types de conteneur utilisé par l'application. Voici les différents types de conteneur :

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld (conteneur Contextes et Injection de dépendance des applications de la plate-forme Java EE)

La table Applications déployées contient également les options suivantes.

Déployer

Bouton permettant de déployer une application.

Annuler le déploiement

Bouton permettant d'annuler le déploiement des applications sélectionnées.

L'annulation du déploiement d'une application enlève l'application de toutes les cibles et du domaine. Lorsque le déploiement d'une application est annulé, l'application n'est plus répertoriée sur la page Applications qui affiche toutes les applications déployées dans GlassFish Server.

Pour enlever des applications de la cible en cours uniquement, utilisez le bouton Enlever de cette page.

Enlever

Bouton permettant d'enlever les applications sélectionnées.

La suppression d'une application enlève l'application de la cible en cours uniquement. Lorsqu'une application est enlevée d'une cible, elle reste répertoriée sur la page Applications, qui affiche toutes les applications déployées dans GlassFish Server. L'application reste également déployée sur les autres cibles sur lesquelles elle a été déployée.

Pour enlever des applications de toutes les cibles et du domaine, utilisez le bouton Annuler le déploiement sur cette page.

Plus d'actions

Liste déroulante qui procède à d'autres actions sur les applications sélectionnées.

Activer

Active les applications sélectionnées.

Désactiver

Désactive les applications sélectionnées.

Equilibreur de charge activé

Active les applications sélectionnées pour l'équilibrage de charge.

Equilibreur de charge désactivé

Désactive les applications sélectionnées pour l'équilibrage de charge.
