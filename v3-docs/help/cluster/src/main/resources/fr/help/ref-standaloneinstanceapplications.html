Applications

La page Applications permet d'effectuer les tâches suivantes sur l'instance de GlassFish Server autonome sélectionnée :

  • Afficher et gérer les applications déjà déployées sur l'instance

  • Déployer davantage d'applications sur l'instance

Le champ Nom de l'instance est un champ en lecture seule qui affiche le nom de l'instance active.

La page Applications présente la liste des applications déployées sur l'instance sélectionnée. Pour chaque application, les informations suivantes sont fournies :

Nom

Nom de l’application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur le module et les versions d'application, reportez-vous à la section Module and Application Versions du Oracle GlassFish Server 3.1 Application Deployment Guide.

Activée

Mention indiquant si l'application est activée :

  • Si l'application est activée, une coche (✓) s'affiche.

  • Si l'application est désactivée, une croix (✗) s'affiche.

LB activé

Mention indiquant si l'application est activée pour l'équilibrage de charge :

  • Si l'application est activée pour l'équilibrage de charge, une coche (✓) s'affiche.

  • Si l'application est désactivée pour l'équilibrage de charge, une croix (✗) s'affiche.

Moteurs

Types de conteneurs utilisés par l'application. Voici les différents types de conteneurs :

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld - conteneur Contextes et Injection de dépendance pour des applications de la plate-forme Java EE)

Le tableau Applications déployées contient également les options suivantes.

Déployer

Bouton permettant de déployer une application.

Annuler le déploiement

Bouton permettant d’annuler le déploiement d’une ou de plusieurs applications sélectionnées.

L'annulation du déploiement d'une application supprime l'application de toutes les cibles et du domaine. Lorsque le déploiement d'une application est annulé, l'application n'est plus répertoriée sur la page Applications qui affiche toutes les applications déployées dans GlassFish Server.

Pour supprimer des applications de la cible active uniquement, utilisez le bouton Supprimer sur cette page.

Supprimer

Bouton permettant de supprimer une ou plusieurs applications sélectionnées.

La suppression d'une application supprime l'application de la cible active uniquement. Lorsqu'une application est supprimée d'une cible, elle reste répertoriée sur la page Applications, qui affiche toutes les applications déployées dans GlassFish Server. L'application reste également déployée sur les autres cibles sur lesquelles elle a été déployée.

Pour supprimer des applications de toutes les cibles et du domaine, utilisez le bouton Annuler le déploiement sur cette page.

Plus d’actions

Liste déroulante qui procède à d'autres actions sur une ou plusieurs applications sélectionnées.

Activer

Active une ou plusieurs applications sélectionnées.

Désactiver

Désactive une ou plusieurs applications sélectionnées.

Équilibreur de charge activé

Active une ou plusieurs applications sélectionnées pour l'équilibrage de charge.

Équilibreur de charge désactivé

Désactive une ou plusieurs applications sélectionnées pour l'équilibrage de charge.
