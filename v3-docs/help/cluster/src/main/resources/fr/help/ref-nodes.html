Nœuds

La page Nœuds affiche une liste des nœuds du domaine. Pour chaque nœud, les informations suivantes sont affichées :

Nom

Nom assigné au nœud lors de sa création. Le fait de cliquer sur le nom ouvre la page Modifier le nud pour le nœud.

Hôte du nœud

Nom de l'hôte que le noeud représente.

Type

Type de nœud parmi les suivants :

CONFIG

Le nœud ne prend pas en charge la communication à distance.

SSH

Le nœud prend en charge la communication via le protocole SSH (shell sécurisé).

Instances

Liste des instances résidant sur le nœud avec une indication de si chaque instance est en cours d'exécution :

  • Si l'instance est en cours d'exécution, une coche (✓) s'affiche.

  • Si l'instance est arrêtée, un point d'exclamation (!) s'affiche.

Le fait de cliquer sur le nom d'une instance ouvre la page Informations générales de l'instance.

Le tableau Nœuds contient également les options suivantes.

Nouveau

Bouton permettant de créer un nœud.

Supprimer

Bouton permettant de supprimer un ou plusieurs nœuds sélectionnés.
