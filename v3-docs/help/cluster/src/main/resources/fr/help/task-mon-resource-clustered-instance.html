Afficher les données de contrôle des ressources pour une instance de GlassFish Server clusterisée

Avant de commencer

Avant d'afficher les données de contrôle, vous devez configurer le service de contrôle.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Sélectionnez l'instance pour laquelle vous souhaitez afficher les données de contrôle des ressources.

    La page Informations générales de l'instance s'ouvre.

  5. Dans la page Informations générales, cliquez sur l’onglet Contrôle.

    La page Contrôle des applications s’ouvre.

  6. Cliquez sur l’onglet Ressources.

    La page Contrôle des ressources s’ouvre.

  7. Dans la liste déroulante Afficher, sélectionnez la ressource pour laquelle vous souhaitez afficher les données de contrôle des ressources.
  8. Pour mettre à jour les données de contrôle affichées, cliquez sur Actualiser.
