Activer ou désactiver l'équilibrage de charge pour une application déployée dans un cluster

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Applications.

    La page Applications du cluster s'ouvre.

  4. Sélectionnez la case à cocher située en regard de l'application.
  5. Dans le menu déroulant Autres actions, sélectionnez la commande appropriée.
    • Pour activer l'équilibrage de charge pour l'application, sélectionnez Activer l'équilibreur de charge.
    • Pour désactiver l'équilibrage de charge pour l'application, choisissez Désactiver l'équilibreur de charge.
