Modifier une ressource d'un cluster

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Ressources.

    La page Ressources du cluster s'ouvre.

  4. Cliquez sur le nom de la ressource.

    La page de modification de la ressource s'ouvre.

  5. Sur la page de modification de la ressource, apportez les modifications requises et enregistrez vos modifications.

    Pour en savoir plus, reportez-vous à la section sur la modification de la ressource de l'aide en ligne de la page.
