Modifier une ressource d'un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Ressources.

    La page Ressources du cluster apparaît.

  4. Cliquez sur le nom de la ressource.

    La page de modification de la ressource apparaît.

  5. Sur la page de modification de la ressource, apportez les modifications nécessaires et enregistrez-les.

    Pour plus d'informations, reportez-vous à l'aide en ligne de la page de modification de la ressource.
