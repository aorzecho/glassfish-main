Arrêter des instances de GlassFish Server autonomes

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Pour chaque instance à arrêter, cochez la case en regard de l'instance dans la table Instances de serveur.

  3. Cliquez sur Arrêter.

    Vous êtes invité à confirmer que vous souhaitez arrêter les instances.

  4. Confirmez l'arrêt des instances.

    Le statut de chaque instance est mis à jour dans la table Instances de serveur à l'arrêt de l'instance.
