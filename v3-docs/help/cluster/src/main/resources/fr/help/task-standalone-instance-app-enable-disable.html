Activer ou désactiver une application déployée vers une instance de GlassFish Server autonome

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sur la page Instances de serveur autonomes, sélectionnez l'instance.

    La page Informations générales de l'instance apparaît.

  3. Sur la page Informations générales de l'instance, cliquez sur l'onglet Applications.

    La page Applications de l'instance apparaît.

  4. Sélectionnez la case située en regard de l'application.

  5. Dans le menu déroulant Plus d'actions, sélectionnez la commande appropriée.

    • Pour activer l'application, choisissez Activer.

    • Pour désactiver l'application, choisissez Désactiver.
