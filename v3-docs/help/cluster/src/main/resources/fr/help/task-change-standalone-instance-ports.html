Modifier les numéros de port d'une instance de GlassFish Server autonome

Les numéros de port d'une instance de GlassFish Server autonome sont initialement définis dans la configuration à laquelle l'instance fait référence. Vous devez modifier ces numéros de port si, par exemple, deux instances qui font référence à la même configuration résident sur le même hôte. Dans ce cas, les numéros de port d'une instance doivent être modifiés car les deux instances font référence à la même configuration.

Les numéros de port d'une instance sont stockés en tant que propriétés système Java. Ces propriétés sont transmises au lanceur d'application Java par l'intermédiaire de l'option -D du lanceur d'application Javaau démarrage de &ProductName ;.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Instances autonomes.

    La page Instances de serveur autonomes s’ouvre.

  2. Sélectionnez l'instance pour laquelle vous souhaitez modifier les numéros de port.

    La page Informations générales de l'instance s'ouvre.

  3. Sur la page Informations générales, cliquez sur l’onglet Propriétés.

    La page Propriétés système de l'instance s’affiche.

    Cette page affiche les valeurs actives des propriétés système suivantes pour l'instance :

    • Port numbers that are predefined in the instance's configuration

    • Propriétés personnalisées définies pour l'instance, le cas échéant

  4. Entrez la nouvelle valeur dans le champ Remplacer la valeur de chaque propriété à modifier.
  5. Définissez les valeurs des propriétés supplémentaires requises.
    1. Dans le tableau Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée au tableau Autres propriétés.

    2. Dans le champ Nom de la variable d'instance, entrez le nom de la propriété.
    3. Dans le champ Remplacer la valeur, entrez la valeur.
  6. Cliquez sur Enregistrer.
