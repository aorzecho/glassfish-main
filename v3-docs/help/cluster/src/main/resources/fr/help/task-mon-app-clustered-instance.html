Visualiser les données de surveillance des applications pour une instance de GlassFish Server incluse dans un cluster

Avant de commencer

Avant de visualiser les données de surveillance, vous devez configurer le service de surveillance.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Sélectionnez l'instance pour laquelle vous souhaitez afficher les données de surveillance des applications.

    La page Informations générales de l'instance apparaît.

  5. Dans la page Informations générales, cliquez sur l'onglet Surveiller.

    La page Surveillance d'application apparaît.

  6. Dans la liste déroulante Afficher, sélectionnez l'application pour laquelle vous souhaitez visualiser les données de surveillance.

  7. Pour mettre à jour les données de surveillance affichées, cliquez sur Actualiser.
