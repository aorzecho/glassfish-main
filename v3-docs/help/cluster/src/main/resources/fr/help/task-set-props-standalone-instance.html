Configurer des propriétés spécifiques à l'instance d'une instance de GlassFish Server autonome

Les propriétés d'une instance de GlassFish Server ajoutent des informations de configuration facultatives sur l'instance.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Instances autonomes.

    La page Instances de serveur autonomes s’ouvre.

  2. Sélectionnez l'instance dont vous souhaitez modifier les propriétés.

    La page Informations générales de l'instance s'ouvre.

  3. Sur la page Informations générales, cliquez sur l’onglet Propriétés.
  4. Si nécessaire, cliquez sur l’onglet Propriétés de l’instance.

    La page Propriétés de l'instance s’affiche.

    Cette page affiche la liste des propriétés définies pour l'instance de GlassFish Server sélectionnée.

  5. Entrez la nouvelle valeur dans le champ Valeur de chaque propriété à modifier.
  6. Définissez les valeurs des propriétés supplémentaires requises.
    1. Dans le tableau Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée au tableau Autres propriétés.

    2. Dans le champ Nom, entrez le nom de la propriété.
    3. Dans le champ Valeur, entrez la valeur.
    4. Dans le champ Description, entrez une brève description de la propriété.
  7. Cliquez sur Enregistrer.
