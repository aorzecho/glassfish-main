Définir les propriétés propres à une instance de GlassFish Server autonome

Les propriétés d'une instance de GlassFish Server ajoutent des informations de configuration facultatives sur l'instance.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sélectionnez l'instance dont vous souhaitez modifier les propriétés.

    La page Informations générales de l'instance apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Propriétés.

  4. Si nécessaire, cliquez sur l'onglet Propriétés d'instance.

    La page Propriétés d'instance apparaît.

    Cette page affiche la liste des propriétés définies pour l'instance de GlassFish Server sélectionnée.

  5. Indiquez la nouvelle valeur de chaque propriété existante à modifier dans le champ Valeur.

  6. Définissez les valeurs des propriétés supplémentaires requises.

    1. Dans la table Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée à la table Autres propriétés.

    2. Dans le champ Nom, entrez le nom de la propriété.

    3. Dans le champ Valeur, indiquez la valeur.

    4. Dans le champ Description, entrez une brève description de la propriété.

  7. Cliquez sur Enregistrer.
