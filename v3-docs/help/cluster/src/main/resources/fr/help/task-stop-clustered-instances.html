Arrêter des instances de GlassFish Server incluses dans un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Pour chaque instance à arrêter, cochez la case en regard de l'instance dans la table Instances de serveur.

  5. Cliquez sur Arrêter.

    Vous êtes invité à confirmer que vous souhaitez arrêter les instances.

  6. Confirmez l'arrêt des instances.

    Le statut de chaque instance est mis à jour dans la table Instances de serveur à l'arrêt de l'instance.
