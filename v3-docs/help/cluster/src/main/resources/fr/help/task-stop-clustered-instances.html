Arrêter les instances de GlassFish Server clusterisées

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Pour chaque instance arrêtée, cochez la case en regard de l'instance dans le tableau Instances de serveur.
  5. Cliquez sur Arrêter.

    Vous êtes invité à confirmer que vous souhaitez arrêter les instances.

  6. Confirmez l'arrêt des instances.

    Le statut de chaque instance est mis à jour dans le tableau Instances de serveur à l'arrêt de l'instance.
