Définir les valeurs par défaut des propriétés système de configuration

Les valeurs par défaut des propriétés de configuration sont définies dans la configuration à laquelle le cluster ou l'instance fait référence.

  1. Dans l'arborescence de navigation, développez le noeud de la configuration.

  2. Dans la liste des types de propriété sous le noeud développé, sélectionnez Propriétés système.

    La page Propriétés système pour la configuration sélectionnée apparaît.

    Cette page affiche les valeurs en cours des propriétés système suivantes pour la configuration :

    • Numéros de port prédéfinis lors de la création de la configuration

    • Propriétés personnalisées ajoutées à la configuration, le cas échéant

  3. Pour vous assurer que vos modifications sont appliquées sans redémarrer le cluster ou l'instance, sélectionnez l'option Reconfiguration dynamique.

  4. Indiquez la nouvelle valeur de chaque propriété pour laquelle vous définissez une valeur par défaut dans le champ Valeur par défaut.

  5. Définissez les valeurs par défaut des propriétés supplémentaires requises.

    1. Dans la table Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée à la table Autres propriétés.

    2. Dans le champ Nom de la variable d'instance, entrez le nom de la propriété.

    3. Dans le champ Valeur par défaut, entrez la valeur.

  6. Cliquez sur Enregistrer.
