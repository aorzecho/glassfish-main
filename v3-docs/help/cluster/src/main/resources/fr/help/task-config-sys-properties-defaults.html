Définir les valeurs par défaut des propriétés système de configuration

Les valeurs par défaut des propriétés de configuration sont définies dans la configuration à laquelle le cluster ou l'instance fait référence.

  1. Dans l'arborescence de navigation, développez le nœud de la configuration.
  2. À partir de la liste des types de propriétés sous le nœud développé, sélectionnez Propriétés système.

    La page Propriétés système pour la configuration sélectionnée s'ouvre.

    Cette page affiche les valeurs actuelles des propriétés système suivantes pour la configuration :

    • Port numbers that are predefined when the configuration was created

    • Propriétés personnalisées ajoutées à la configuration, le cas échéant

  3. Pour vous assurer que vos modifications sont appliquées sans redémarrer le cluster ou l'instance, sélectionnez l'option Reconfiguration dynamique.
  4. Pour chaque propriété pour laquelle vous définissez une valeur par défaut, entrez la nouvelle valeur dans le champ Valeur par défaut de la propriété.
  5. Définissez les valeurs par défaut des propriétés supplémentaires requises.
    1. Dans le tableau Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée au tableau Autres propriétés.

    2. Dans le champ Nom de la variable d'instance, entrez le nom de la propriété.
    3. Dans le champ Valeur par défaut, entrez la valeur.
  6. Cliquez sur Enregistrer.
