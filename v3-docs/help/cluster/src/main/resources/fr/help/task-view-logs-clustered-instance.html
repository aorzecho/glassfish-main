Afficher les journaux d'une instance de GlassFish Server clusterisée

Avant de commencer

L'instance pour laquelle vous affichez les journaux doit être en cours d'exécution.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Sélectionnez l'instance pour laquelle vous souhaitez afficher les fichiers journaux.

    La page Informations générales de l'instance s'ouvre.

  5. Sur la page Informations générales, cliquez sur Afficher les journaux.

    Une fenêtre de visionneur de journaux s’ouvre.

    Remarque - L'instance doit déjà être en cours d'exécution. Si l'instance n'est pas en cours d'exécution, le bouton Afficher les journaux est désactivé.
