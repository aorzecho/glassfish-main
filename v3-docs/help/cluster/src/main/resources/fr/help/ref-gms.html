Modifier le service de gestion de groupes

La page Service de gestion de groupes permet de modifier les paramètres GMS.

Le service de gestion de groupes (GMS) est un service inclus dans un processus offrant des services de surveillance de clusters, d'appartenance au cluster et de communication de groupe. Le GMS est construit sur une pile configurable de repérage d'appartenance à des groupes et de protocoles de surveillance de l'intégrité. Il exécute les fonctions suivantes :

  • Il signale les modules inscrits dans une instance GlassFish Server lorsqu'une ou plusieurs instances membres sont ajoutées à un cluster et enlevées d'un cluster, ou lorsque leur échec est suspecté ou a été confirmé.

  • Il permet d'envoyer et recevoir des messages dans un groupe de processus.

La page Service de gestion de groupes contient les options suivantes :

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nb maximal de signaux d'activité manqués

Nombre de tentatives de surveillance après lequel l'échec d'une instance de serveur est suspecté. La valeur par défaut est 3.

Fréquence de signal

Temps entre les tentatives de surveillance. La valeur par défaut est 2 000 millisecondes.

Temporisation de détection de groupe

Durée d'attente du GMS pour la détection d'autres membres du cluster. La valeur par défaut est 5 000 millisecondes.

Temps d'attente de vérification d'échec

Durée après laquelle un échec suspecté est marqué comme étant vérifié. La valeur par défaut est 1 500 millisecondes.

Autres propriétés

Table des propriétés supplémentaires pour le service de gestion de groupes.

GlassFish Server ne définit pas d'autre propriété GMS.

Pour chaque propriété, les informations suivantes sont affichées :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété définie pour l'instance sélectionnée.

Description

Texte de description qui fournit de plus amples informations sur la propriété.

La table Autres propriétés contient également les options suivantes :

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Le fait de cliquer sur ce bouton permet d'ajouter une ligne à la table Autres propriétés.

Supprimer des propriétés

Bouton permettant de supprimer les propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est configurée, n'est pas définie.
