Modifier le service de gestion de groupes

La page Service de gestion de groupes vous permet de modifier les paramètres GMS.

Le service de gestion de groupes (GMS) est un service inclus dans un processus offrant des services de contrôle de clusters, d'appartenance au cluster et de communication de groupe. Le GMS est construit sur une pile configurable de découverte d'appartenance au groupe et de protocoles de contrôle de l'intégrité. Il exécute les fonctions suivantes :

  • Signaler les modules enregistrés dans une instance de GlassFish Server lorsqu'une ou plusieurs instances membres sont ajoutées ou supprimées d'un cluster ou lorsque leur échec est suspecté ou a été confirmé.

  • Offrir la possibilité d'envoyer et recevoir des messages dans un groupe de processus.

La page Service de gestion de groupes contient les options suivantes.

Restaurer les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut de tous les paramètres.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Maximum Missed Heartbeats (Nombre maximal de pulsations manquées)

Nombre de tentatives de contrôle après lequel l'échec d'une instance de serveur est suspecté. La valeur par défaut est 3.

Fréquence de pulsation

Durée entre les tentatives de contrôle. La valeur par défaut est 2000 millisecondes.

Group Discovery Timeout (Délai d'attente de détection de groupe)

Durée d'attente du GMS pour la détection d'autres membres du cluster. La valeur par défaut est 5000 millisecondes.

Failure Verification Wait Time (Temps d'attente de vérification d'échec)

Durée après laquelle un échec suspecté est marqué comme étant vérifié. La valeur par défaut est 1500 millisecondes.

Autres propriétés

Tableau des propriétés supplémentaires pour le Service de gestion de groupes.

GlassFish Server ne définit pas d'autres propriétés GMS.

Pour chaque propriété, les informations suivantes sont affichées :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété définie pour l'instance sélectionnée.

Description

Texte de description qui fournit de plus amples informations sur la propriété.

Le tableau Propriétés supplémentaires contient également les options suivantes.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Le fait de cliquer sur ce bouton permet d'ajouter une ligne au tableau Propriétés supplémentaires.

Supprimer des propriétés

Bouton permettant de supprimer une ou plusieurs propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est définie, n'est pas définie.
