Configurer des propriétés spécifiques à l'instance d'une instance de GlassFish Server clusterisée

Les propriétés d'une instance de GlassFish Server ajoutent des informations de configuration facultatives sur l'instance.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Sélectionnez l'instance dont vous souhaitez modifier les propriétés.

    La page Informations générales de l'instance s'ouvre.

  5. Sur la page Informations générales, cliquez sur l’onglet Propriétés.
  6. Si nécessaire, cliquez sur l’onglet Propriétés de l’instance.

    La page Propriétés de l'instance s’affiche.

    Cette page affiche la liste des propriétés définies pour l'instance de GlassFish Server sélectionnée.

  7. Entrez la nouvelle valeur dans le champ Valeur de chaque propriété à modifier.
  8. Définissez les valeurs des propriétés supplémentaires requises.
    1. Dans le tableau Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée au tableau Autres propriétés.

    2. Dans le champ Nom, entrez le nom de la propriété.
    3. Dans le champ Valeur, entrez la valeur.
    4. Dans le champ Description, entrez une brève description de la propriété.
  9. Cliquez sur Enregistrer.
