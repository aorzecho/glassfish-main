Nouvelle instance de serveur autonome

La page Nouvelle instance de serveur autonome permet de créer une instance GlassFish Server autonome.

Elle contient les options suivantes :

Nom d'instance

Nom de l'instance en cours de création.

Le nom doit respecter les exigences suivantes :

  • Il ne peut contenir que des caractères ASCII.

  • Il doit commencer par une lettre, un chiffre ou un trait de soulignement.

  • Il ne peut contenir que les caractères suivants :

    • Lettres minuscules

    • Lettres majuscules

    • Chiffres

    • Trait d'union

    • Point

    • Trait de soulignement

  • Le nom du noeud doit être unique dans le domaine ; il ne doit pas appartenir à une autre instance de GlassFish Server, à une configuration nommée, à un cluster ou à un noeud.

  • Il ne doit pas être domain, server ou tout autre mot-clé réservé par GlassFish Server.

Noeud

Liste déroulante des noeuds existants définissant les hôtes sur lesquels l'instance peut résider. Vous devez sélectionner un noeud dans la liste. Si vous devez créer l'instance sur l'hôte hébergeant le serveur d'administration de domaine (DAS), utilisez le noeud prédéfini localhost-domain.

Configuration

Liste déroulante des configurations nommées existantes pouvant être utilisées par l'instance. Vous devez sélectionner une configuration dans la liste. L'instance utilise la configuration sélectionnée lors de sa création.

Créer une copie de la configuration sélectionnée

Si cette option est sélectionnée, la configuration choisie est copiée lors de la création de l'instance. Le nom instance-name-config est affecté à la copie de la configuration, instance-name étant le nom de l'instance tel que vous l'avez saisi dans le champ Nom d'instance.

Si default-config est sélectionné dans la liste déroulante Configuration, vous devez choisir cette option. La configuration default-config peut être copiée, mais pas utilisée comme référence.

Utiliser la configuration sélectionnée comme référence

Si cette option est sélectionnée, l'instance utilise la configuration nommée existante indiquée.

Si default-config est sélectionné dans la liste déroulante Configuration, vous ne devez pas choisir cette option. La configuration default-config peut être copiée, mais pas utilisée comme référence.
