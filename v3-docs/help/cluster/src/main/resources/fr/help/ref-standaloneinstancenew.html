Nouvelle instance de serveur autonome

La page Nouvelle instance de serveur autonome permet de créer une instance de GlassFish Server autonome.

Elle contient les options suivantes.

Nom d’instance

Nom de l'instance en cours de création.

Le nom doit respecter les exigences suivantes :

  • Le nom doit commencer par une lettre, un chiffre ou un caractère de soulignement.

  • Le nom ne peut contenir que les caractères suivants :

    • Lettres minuscules

    • Lettres majuscules

    • Chiffres

    • Trait d'union

    • Point

    • Caractère de soulignement

  • Le nom doit être unique dans le domaine et ne doit pas être le nom d'une autre instance de GlassFish Server, d'un cluster, d'une configuration nommée ou d'un nœud.

  • Le nom ne doit pas être domain, server ou tout autre mot-clé réservé par GlassFish Server.

Nœud

Liste déroulante des nœuds existants définissant les hôtes sur lesquels l'instance peut résider. Un nœud doit être sélectionné dans la liste. Si l'instance doit être créée sur l'hôte hébergeant le serveur d'administration de domaine (DAS), utilisez le nœud prédéfini localhost-domain.

Configuration

Liste déroulante des configurations nommées existantes pouvant être utilisées par l'instance. Une configuration doit être sélectionnée dans la liste. L'instance utilise la configuration sélectionnée lors de sa création.

Créer une copie de la configuration sélectionnée

Si cette option est sélectionnée, la configuration sélectionnée est copiée lors de la création de l'instance. Le nom instance-name-config est affecté à la copie de la configuration, instance-name étant le nom de l'instance tel que vous l'avez saisi dans le champ Nom d'instance.

Si default-config est sélectionné dans la liste déroulante Configuration, vous devez sélectionner cette option. La configuration default-config peut être copiée, mais ne peut pas être utilisée comme référence.

Utiliser la configuration sélectionnée comme référence

Si cette option est sélectionnée, l'instance utilise la configuration nommée existante indiquée.

Si default-config est sélectionné dans la liste déroulante Configuration, vous ne devez pas sélectionner cette option. La configuration default-config peut être copiée, mais ne peut pas être utilisée comme référence.
