Supprimer des nœuds

Lorsque vous supprimez un nœud, celui-ci est supprimé de la configuration du serveur d'administration de domaine (DAS). Les répertoires et fichiers du nœud sont supprimés lorsque la dernière instance de GlassFish Server résidant sur le nœud est supprimée.

Avant de commencer

Assurez-vous qu'aucune instance de GlassFish Server ne réside sur le nœud en cours de suppression.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Nœuds.

    La page Nœuds s'ouvre.

  2. Pour chaque nœud supprimé, sélectionnez la case à cocher en regard du nœud dans le tableau Nœuds.

    Remarque - Le nœud prédéfini localhost-domain ne peut pas être supprimé.

  3. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer le nœud.

  4. Confirmez la suppression du nœud.

    Le nœud est supprimé et effacé du tableau des nœuds sur la page Nœuds.
