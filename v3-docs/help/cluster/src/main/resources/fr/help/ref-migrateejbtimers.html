Migrer les horloges EJB

La page Migrer les horloges EJB contient les options suivantes :

Source

Liste déroulante des instances GlassFish Server arrêtées dans le cluster. Vous ne pouvez migrer les horloges EJB qu'à partir d'instances arrêtées.

Destination

Liste déroulante des instances GlassFish Server en cours d'exécution dans le cluster. Vous ne pouvez migrer les horloges EJB que vers des instances en cours d'exécution.

Remarque:

Si toutes les instances du cluster sont arrêtées, cette page affiche uniquement un avertissement indiquant l'indisponibilité de toute instance en cours d'exécution vers laquelle les horloges EJB peuvent être migrées.

La page Migrer les horloges EJB contient une table des horloges EJB présentant les informations suivantes.

Nom de serveur

Liste des instances GlassFish Server dans le cluster.

Nombre d'horloges

Nombre d'horloges persistantes sur chaque instance.
