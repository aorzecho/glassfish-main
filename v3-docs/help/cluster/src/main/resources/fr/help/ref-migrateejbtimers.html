Migrer les services d’horloge EJB

La page Migrer les services d'horloge EJB contient les options suivantes.

Source

Liste déroulante des instances GlassFish Server arrêtées dans le cluster. Les services d'horloges EJB peuvent uniquement être migrés à partir d'instances arrêtées.

Destination

Liste déroulante des instances GlassFish Server en cours d'exécution dans le cluster. Les services d'horloge EJB peuvent uniquement être migrés vers des instances en cours d'exécution.

Remarque - Si toutes les instances du cluster sont arrêtées, cette page affiche uniquement un avertissement indiquant qu'aucune instance en cours d'exécution vers laquelle les services d'horloge EJB peuvent être migrés n'est disponible.
