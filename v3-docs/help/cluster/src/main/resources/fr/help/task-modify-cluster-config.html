Modifier la configuration d'un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sélectionnez le noeud correspondant au cluster.

  3. Cliquez sur le nom de la configuration.

    La page Configuration du cluster apparaît.

  4. Sur la page Configuration, sélectionnez le composant de la configuration à modifier.

    La page de modification du composant sélectionné de la configuration apparaît.

  5. Sur la page de modification du composant sélectionné, apportez les modifications nécessaires et enregistrez-les.

    Pour plus d'informations, reportez-vous à l'aide en ligne de la page de modification du composant sélectionné.
