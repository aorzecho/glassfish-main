Clusters

La page Clusters affiche la liste des clusters du domaine. Pour chaque cluster, les informations suivantes sont affichées :

Nom

Nom affecté au cluster lors de sa création. Le fait de cliquer sur le nom ouvre la page Informations générales du cluster.

Configuration

Configuration sélectionnée pour le cluster lors de sa création. Le fait de cliquer sur la configuration ouvre la page Configurations du cluster.

Instances

Nom et statut de chaque instance du cluster. Le fait de cliquer sur le nom de l'instance ouvre la page Informations générales de l'instance.

La table Clusters contient également les options suivantes.

Nouveau

Bouton permettant de créer un cluster.

Supprimer

Bouton permettant de supprimer un cluster.

Démarrer le cluster

Bouton permettant de démarrer un cluster.

Arrêter le cluster

Bouton permettant d'arrêter un cluster.
