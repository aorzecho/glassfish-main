Clusters

La page Clusters affiche la liste des clusters dans le domaine. Pour chaque cluster, les informations suivantes sont affichées :

Nom

Nom qui a été assigné au cluster lorsque le cluster a été créé. Le fait de cliquer sur le nom ouvre la page General Information du cluster.

Configuration

Configuration qui a été sélectionnée pour le cluster lorsque le cluster a été créé. Le fait de cliquer sur la configuration ouvre la page Configurations du cluster.

Instances

Nom et statut de chaque instance dans le cluster. Le fait de cliquer sur le nom de l'instance ouvre la page General Information de l'instance.

Le tableau Clusters contient également les options suivantes.

Nouveau

Bouton permettant de créer un cluster.

Supprimer

Bouton permettant de supprimer un cluster.

Démarrer le cluster

Bouton permettant de démarrer un cluster.

Arrêter le cluster

Bouton permettant d'arrêter un cluster.
