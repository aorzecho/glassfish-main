Propriétés système de l'instance

La page Instance System Properties (Propriétés système de l'instance) affiche une liste des propriétés système Java définies pour l'instance de GlassFish Server active. Les propriétés système Java sont transmises au lanceur d'application Java par l'intermédiaire de l'option -D du lanceur d'application Javaau démarrage de &ProductName ;.

Ces propriétés remplacent les définitions des propriétés pour les paramètres de port dans la configuration de l'instance. Les paramètres de port prédéfinis doivent être remplacés si, par exemple, deux instances clusterisées résident sur le même hôte. Dans cette situation, les paramètres de port pour une seule instance doivent être remplacés, car les deux instances partagent la même configuration.

Pour plus d'informations sur les paramètres de port prédéfinis dans une configuration, reportez-vous à la section Predefined Configuration System Properties.

Le champ Nom de l'instance est un champ en lecture seule qui affiche le nom de l'instance active.

Pour chaque propriété, les informations suivantes sont affichées :

Nom de la variable d’instance

Nom de la propriété système.

Valeur actuelle

Valeur actuellement définie pour la propriété. Ce champ est en lecture seule.

Remplacer la valeur

Valeur sur laquelle la propriété sera définie après l'enregistrement des modifications.

Le tableau Propriétés supplémentaires contient également les options suivantes.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Le fait de cliquer sur ce bouton permet d'ajouter une ligne au tableau Propriétés supplémentaires.

Supprimer des propriétés

Bouton permettant de supprimer une ou plusieurs propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est définie, n'est pas définie.
