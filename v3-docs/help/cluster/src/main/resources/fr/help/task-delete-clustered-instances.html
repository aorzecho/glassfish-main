Supprimer les instances de GlassFish Server clusterisées

Attention - Si vous utilisez un cluster Java Message Service (JMS) avec un courtier principal, ne supprimez pas l'instance associée au courtier principal. Si cette instance doit être supprimée, utilisez la sous-commande change-master-broker(1) pour affecter le courtier principal à une autre instance.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Pour chaque instance que vous supprimez, cochez la case en regard de l'instance dans le tableau Instances de serveur.
  5. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer les instances.

  6. Confirmez la suppression des instances.

    Les instances sont supprimées et effacées du tableau d'instances sur la page.
