Supprimer des instances de GlassFish Server incluses dans un cluster

Attention:

Si vous utilisez un cluster JMS (Java Message Service) avec un broker maître, ne supprimez pas l'instance associée au broker maître. Si cette instance doit être supprimée, utilisez la sous-commande change-master-broker(1) pour affecter le broker maître à une autre instance.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Pour chaque instance à supprimer, cochez la case en regard de l'instance dans la table Instances de serveur.

  5. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer les instances.

  6. Confirmez la suppression des instances.

    Les instances sont supprimées et enlevées de la table des instances sur la page.
