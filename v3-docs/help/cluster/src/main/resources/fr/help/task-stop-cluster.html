Arrêter un cluster

L'arrêt d'un cluster arrête toutes les instances de GlassFish Server du cluster.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Clusters.

    La page Clusters s’ouvre.

  2. Sur la page Clusters, cochez la case située en regard du nom du cluster.
  3. Cliquez sur Arrêter le cluster.

    Vous êtes invité à confirmer que vous souhaitez arrêter le cluster.

  4. Confirmez l'arrêt du cluster.

    Le cluster est arrêté. Le statut de chaque instance du cluster est mis à jour sur la page Clusters lorsque l'instance est arrêtée.
