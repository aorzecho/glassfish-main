Créer une ressource pour un cluster

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Ressources.

    La page Ressources du cluster s'ouvre.

  4. Dans le menu déroulant Nouveau, sélectionnez le type de la ressource à créer.

    La page de création d'une ressource du type sélectionné s'ouvre.

  5. Sur la page de création d'une ressource du type sélectionné, apportez les modifications requises et enregistrez vos modifications.

    Pour plus d'informations, reportez-vous à l'aide en ligne de la page de création d'une ressource du type sélectionné.

    Remarque - Lorsque vous créez la ressource, assurez-vous de spécifier le cluster en tant que cible.
