Créer une ressource pour un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Ressources.

    La page Ressources du cluster apparaît.

  4. Dans le menu déroulant Nouveau, sélectionnez le type de la ressource à créer.

    La page de création d'une ressource du type sélectionné apparaît.

  5. Sur la page de création d'une ressource du type sélectionné, apportez les modifications nécessaires et enregistrez-les.

    Pour plus d'informations, reportez-vous à l'aide en ligne de la page de création d'une ressource du type sélectionné.

    Remarque:

    Lorsque vous créez la ressource, assurez-vous de spécifier le cluster en tant que cible.
