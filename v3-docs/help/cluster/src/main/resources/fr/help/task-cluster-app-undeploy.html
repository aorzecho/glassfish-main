Annuler le déploiement d'applications d'un cluster

L'annulation du déploiement d'une application supprime l'application de toutes les cibles et du domaine. Lorsque le déploiement d'une application est annulé, l'application n'est plus répertoriée sur la page Applications qui affiche toutes les applications déployées dans GlassFish Server.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Applications.

    La page Applications du cluster s'ouvre.

  4. Pour chaque application dont vous annulez le déploiement, cochez la case en regard de l'application dans le tableau Applications déployées.
  5. Cliquez sur Annuler le déploiement.

    Vous êtes invité à confirmer que vous souhaitez annuler le déploiement des applications.

  6. Confirmez l'annulation du déploiement des applications.

    Le déploiement des applications est annulé et les applications sont supprimées du tableau des applications déployées sur la page.
