Visualiser les données de surveillance des serveurs virtuels pour une instance de GlassFish Server autonome

Avant de commencer

Avant de visualiser les données de surveillance, vous devez configurer le service de surveillance.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sélectionnez l'instance pour laquelle vous souhaitez visualiser les données de surveillance des serveurs virtuels.

    La page Informations générales de l'instance apparaît.

  3. Dans la page Informations générales, cliquez sur l'onglet Surveiller.

    La page Surveillance d'application apparaît.

  4. Cliquez sur l'onglet Serveur.

    La page Surveillance de serveur apparaît.

  5. Dans la liste déroulante Afficher, sélectionnez le composant pour lequel vous souhaitez visualiser les données de surveillance.

  6. Pour mettre à jour les données de surveillance affichées, cliquez sur Actualiser.
