Afficher les fichiers journaux d'une instance de GlassFish Server autonome

Avant de commencer

L'instance pour laquelle vous souhaitez afficher les journaux doit être en cours d'exécution.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sélectionnez l'instance pour laquelle vous souhaitez afficher les fichiers journaux.

    La page Informations générales de l'instance apparaît.

  3. Sur la page Informations générales, cliquez sur Afficher les fichiers journaux.

    La fenêtre du visualiseur de journaux apparaît.

    Remarque:

    L'instance doit déjà être en cours d'exécution. Si elle ne l'est pas, le bouton Afficher les fichiers journaux est désactivé.
