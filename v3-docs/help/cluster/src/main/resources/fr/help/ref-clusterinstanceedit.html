Informations générales

La page Informations générales contient les options suivantes :

Démarrer

Bouton permettant de démarrer l'instance. Si l'instance est en cours d'exécution, ce bouton est désactivé.

Arrêter

Bouton permettant d'arrêter l'instance. Si l'instance est arrêtée, ce bouton est désactivé.

Redémarrer

Bouton permettant de redémarrer l'instance. Si l'instance est arrêtée, ce bouton est désactivé.

Afficher les fichiers journaux

Bouton permettant d'afficher les fichiers journaux de l'instance.

Récupérer les transactions

Bouton permettant de récupérer les transactions de l'instance.

Nom du cluster

Nom affecté au cluster lors de sa création. Le fait de cliquer sur le nom ouvre la page Informations générales du cluster.

Nom d'instance

Nom affecté à l'instance lors de son ajout au cluster. Ce champ est en lecture seule.

Statut

Mention indiquant si l'instance est en cours d'exécution, requiert un redémarrage pour être mise à jour avec les modifications de la configuration ou n'est pas en cours d'exécution. Ce champ est en lecture seule.

JVM

Si vous cliquez sur Rapport JVM, une fenêtre distincte s'ouvre et affiche des rapports sur la Java Virtual Machine, notamment un rapport de synthèse, un rapport sur la gestion de la mémoire et les opérations Garbage Collector, un rapport de chargement de classes et le vidage des threads en cours.

Noeud

Nom du noeud sur lequel se trouve l'instance. Le fait de cliquer sur le nom ouvre la page Modifier le noeud du noeud.

Configuration

Configuration à laquelle l'instance fait référence. Le fait de cliquer sur la configuration ouvre la page Configurations de l'instance.

Ports HTTP

Liste séparée par des virgules des numéros des ports sur lesquels l'instance écoute les types de demande suivants :

  • Demandes d'administration

  • Demandes HTTP

  • Demandes HTTPS

Ce champ est en lecture seule.

Ports IIOP

Liste séparée par des virgules des numéros des ports utilisés par l'instance pour les types de connexion suivants :

  • Connexion IIOP sécurisée

  • Connexions IIOP sécurisées avec authentification client

  • Connexions IIOP

Ce champ est en lecture seule.
