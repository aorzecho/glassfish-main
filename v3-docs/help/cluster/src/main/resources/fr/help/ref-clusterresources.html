Ressources

La page Ressources vous permet d'effectuer les tâches suivantes sur le cluster sélectionné :

  • Afficher et gérer les ressources existantes du cluster

  • Créer d'autres ressources pour le cluster

Le champ Nom du cluster est un champ en lecture seule qui affiche le nom du cluster actif.

La page Ressources affiche une liste des ressources existantes sur le cluster. Pour chaque ressource, les informations suivantes sont fournies.

Nom de ressource

Nom assigné à la ressource lors de sa création. Le fait de cliquer sur le nom ouvre l'une des pages suivantes pour modifier la ressource, selon le type de la ressource :

  • Edit JDBC Resource

  • Edit JavaMail Session

  • Edit Custom Resource

  • New Custom Resource

  • Edit External Resource

  • Edit Connector Resource

  • Edit Admin Object Resource

Activée

Mention indiquant si la ressource est activée :

  • Si la ressource est activée, une coche (✓) s'affiche.

  • Si la ressource est désactivée, une croix (✗) s'affiche.

Type

Le type de la ressource de

Le tableau Ressources contient également les options suivantes :

Activer

Bouton permettant d’activer une ou plusieurs ressources sélectionnées.

Désactiver

Bouton permettant de désactiver une ou plusieurs ressources sélectionnées.

Nouveau

Liste déroulante qui crée une ressource du type sélectionné. La sélection d'un type de ressource dans la liste ouvre l'une des pages suivantes, qui permet de créer une ressource du type sélectionné :

  • New JDBC Resource

  • New JMS Connection Factory

  • New JMS Destination Resource

  • New JavaMail Session

  • New Custom Resource

  • New External Resource

  • New Connector Resource

  • New Admin Object Resource

Filtre

Liste déroulante qui filtre la liste des ressources affichées par type.
