Supprimer des applications d'une instance autonome GlassFish Server

La suppression d'une application supprime l'application de la cible active uniquement. Lorsqu'une application est supprimée d'une cible, elle reste répertoriée sur la page Applications, qui affiche toutes les applications déployées dans GlassFish Server. L'application reste également déployée sur les autres cibles sur lesquelles elle a été déployée.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Instances autonomes.

    La page Instances de serveur autonomes s’ouvre.

  2. Sur la page Instances de serveur autonomes, sélectionnez l'instance.

    La page Informations générales de l'instance s'ouvre.

  3. Sur la page Informations générales de l'instance, cliquez sur l'onglet Applications.

    La page Applications de l'instance s'ouvre.

  4. Pour chaque application supprimée, cochez la case en regard de l'application dans le tableau Applications déployées.
  5. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer les applications.

  6. Confirmez la suppression des applications.

    Les applications sont supprimées du tableau des applications déployées sur la page.
