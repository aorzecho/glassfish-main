Déployer une application sur une instance de GlassFish Server autonome

  1. Dans l’arborescence de navigation, sélectionnez le nœud Instances autonomes.

    La page Instances de serveur autonomes s’ouvre.

  2. Sur la page Instances de serveur autonomes, sélectionnez l'instance.

    La page Informations générales de l'instance s'ouvre.

  3. Sur la page Informations générales de l'instance, cliquez sur l'onglet Applications.

    La page Applications de l'instance s'ouvre.

  4. Sur la page Applications, cliquez sur Déployer.

    La page Déploiement d’applications ou de modules s’affiche.

  5. Sur la page Déploiement d’applications ou de modules, fournissez les informations requises sur l'application et cliquez sur OK.

    Pour plus de détails, reportez-vous à la section la section Deploy Applications or Modules de l'aide en ligne.
