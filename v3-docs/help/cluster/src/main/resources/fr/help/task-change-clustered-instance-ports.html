Modifier les numéros de port d'une instance de GlassFish Server clusterisée

Les numéros de port d'une instance de GlassFish Server clusterisée sont initialement définis dans la configuration que l'instance hérite de son cluster parent. Vous devez modifier ces numéros de port si, par exemple, deux instances clusterisées résident sur le même hôte. Dans ce cas, les numéros de port d'une instance doivent être modifiés car les deux instances partagent la même configuration.

Les numéros de port d'une instance sont stockés en tant que propriétés système Java. Ces propriétés sont transmises au lanceur d'application Java par l'intermédiaire de l'option -D du lanceur d'application Javaau démarrage de &ProductName ;.

  1. Dans l’arborescence de navigation, développez le nœud Clusters.
  2. Sous le nœud Clusters développé, sélectionnez le nœud pour le cluster.

    La page Informations générales du cluster s'ouvre.

  3. Dans la page Informations générales, cliquez sur l’onglet Instances.

    La page Instances de serveur clusterisées du cluster s'ouvre.

  4. Sélectionnez l'instance pour laquelle vous souhaitez modifier les numéros de port.

    La page Informations générales de l'instance s'ouvre.

  5. Sur la page Informations générales, cliquez sur l’onglet Propriétés.

    La page Propriétés système de l'instance s’affiche.

    Cette page affiche les valeurs actives des propriétés système suivantes pour l'instance :

    • Port numbers that are predefined in the instance's configuration

    • Propriétés personnalisées définies pour l'instance, le cas échéant

  6. Entrez la nouvelle valeur dans le champ Remplacer la valeur de chaque propriété à modifier.
  7. Définissez les valeurs des propriétés supplémentaires requises.
    1. Dans le tableau Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée au tableau Autres propriétés.

    2. Dans le champ Nom de la variable d'instance, entrez le nom de la propriété.
    3. Dans le champ Remplacer la valeur, entrez la valeur.
  8. Cliquez sur Enregistrer.
