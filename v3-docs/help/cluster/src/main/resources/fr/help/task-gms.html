Configurer le service de gestion de groupes

Le service de gestion de groupes (GMS) est un service inclus dans un processus offrant des services de surveillance de clusters, d'appartenance au cluster et de communication de groupe. Il exécute les fonctions suivantes :

  • Il signale les modules inscrits dans une instance GlassFish Server lorsqu'une ou plusieurs instances membres sont ajoutées à un cluster et enlevées d'un cluster, ou lorsque leur échec est suspecté ou a été confirmé.

  • Il permet d'envoyer et recevoir des messages dans un groupe de processus.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sélectionnez la configuration à modifier pour le cluster.

  3. Sous le noeud Configuration, sélectionnez le noeud Service de gestion de groupes.

    La page Service de gestion de groupes apparaît.

  4. Dans le champ Nb maximal de signaux d'activité manqués, entrez le nombre maximal de tentatives de surveillance effectuées avant que le service GMS ne confirme qu'une défaillance est suspectée.

    La valeur par défaut est 3.

  5. Dans le champ Fréquence de signal, définissez le délai entre les tentatives de surveillance.

    La valeur par défaut est 2 000 millisecondes.

  6. Dans le champ Temporisation de détection de groupe, indiquez le délai au-delà duquel le service GMS lance la détection d'autres membres du cluster.

    La valeur par défaut est 5 000 millisecondes.

  7. Dans le champ Temps d'attente de vérification d'échec, indiquez le délai au-delà duquel une défaillance suspectée est marquée comme étant vérifiée.

    La valeur par défaut est 1 500 millisecondes.

  8. Pour restaurer les valeurs par défaut, cliquez sur le bouton Charger les valeurs par défaut.

  9. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  10. Cliquez sur Enregistrer.
