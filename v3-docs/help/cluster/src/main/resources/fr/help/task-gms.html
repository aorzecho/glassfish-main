Configurer le service de gestion de groupes

Le service de gestion de groupes (GMS) est un service inclus dans un processus offrant des services de contrôle de clusters, d'appartenance au cluster et de communication de groupe. Il exécute les fonctions suivantes :

  • Signaler les modules enregistrés dans une instance de GlassFish Server lorsqu'une ou plusieurs instances membres sont ajoutées ou supprimées d'un cluster ou lorsque leur échec est suspecté ou a été confirmé.

  • Offrir la possibilité d'envoyer et recevoir des messages dans un groupe de processus.

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez la configuration à modifier pour le cluster.
  3. Sous le nœud Configuration, sélectionnez le nœud Service de gestion de groupes.

    La page Service de gestion de groupes s'ouvre.

  4. Dans le champ Maximum Missed Heartbeats, entrez le nombre maximal de tentatives de contrôle avant que le GMS ne confirme qu'une défaillance est suspectée.

    La valeur par défaut est 3.

  5. Dans le champ Fréquence de pulsation, entrez la durée entre les tentatives de contrôle.

    La valeur par défaut est 2000 millisecondes.

  6. Dans le champ Group Discovery Timeout field, entrez la durée d'attente du GMS pour la détection d'autres membres du cluster.

    La valeur par défaut est 5000 millisecondes.

  7. Dans le champ Failure Verification Wait Time, entrez la durée au bout de laquelle une défaillance suspectée est marquée comme étant vérifiée.

    La valeur par défaut est 1500 millisecondes.

  8. Pour restaurer les valeurs par défaut, cliquez sur le bouton Charger les valeurs par défaut.
  9. (Facultatif) Dans la section Autres propriétés, spécifiez d’autres propriétés.

    GlassFish Server ne définit pas d'autres propriétés GMS.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  10. Cliquez sur Enregistrer.
