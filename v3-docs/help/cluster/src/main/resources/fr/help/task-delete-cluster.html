Supprimer un cluster

La suppression d'un cluster supprime le cluster et toutes les instances GlassFish Server qu'il contient. Il n'est pas obligatoire de supprimer les instances du cluster avant de supprimer le cluster.

Si la configuration que le cluster utilise n'est pas partagée, elle est également supprimée.

Avant de commencer

Assurez-vous que le cluster que vous supprimez est arrêté.

  1. Dans l’arborescence de navigation, sélectionnez le nœud Clusters.

    La page Clusters s’ouvre.

  2. Sur la page Clusters, cochez la case située en regard du nom du cluster.
  3. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer le cluster et toutes les instances qu'il contient.

  4. Confirmez la suppression du cluster et de toutes les instances qu'il contient.

    Le cluster est supprimé et effacé du tableau des clusters sur la page Clusters.
