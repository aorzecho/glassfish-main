Supprimer un cluster

La suppression d'un cluster supprime le cluster et toutes les instances de GlassFish Server qu'il contient. Il n'est pas obligatoire de supprimer les instances du cluster avant de supprimer ce dernier.

Si la configuration que le cluster utilise n'est pas partagée, elle est également supprimée.

Avant de commencer

Assurez-vous que le cluster que vous souhaitez supprimer est arrêté. Pour plus d'informations, reportez-vous à Arrêter un cluster.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Clusters.

    La page Clusters apparaît.

  2. Sur la page Clusters, cochez la case située en regard du nom du cluster.

  3. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer le cluster et toutes les instances qu'il contient.

  4. Confirmez la suppression du cluster et de toutes les instances qu'il contient.

    Le cluster est supprimé et enlevé de la table des clusters sur la page Clusters.
