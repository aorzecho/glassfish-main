设置独立 GlassFish Server 实例的仅实例属性

GlassFish Server 实例的属性可添加有关实例的可选配置信息。

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 选择要修改仅实例属性的实例。

    此时将打开实例的“常规信息”页面。

  3. 在“常规信息”页面中，单击“属性”选项卡。
  4. 如有必要，单击“实例属性”选项卡。

    此时将打开“实例属性”页面。

    此页面会显示为选定的 GlassFish Server 实例设置的属性列表。

  5. 对于要修改的每个现有属性，在该属性的“值”字段中键入新的值。
  6. 对于其他属性，可根据需要设置相应的值。
    1. 在“其他属性”表中，单击“添加属性”。

      会向“其他属性”表中添加一行。

    2. 在“名称”字段中，键入属性名称。
    3. 在“值”字段中，键入值。
    4. 在“说明”字段中，键入属性的简要说明。
  7. 单击“保存”。
