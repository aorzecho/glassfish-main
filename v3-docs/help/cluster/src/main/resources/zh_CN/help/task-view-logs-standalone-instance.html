查看独立 GlassFish Server 实例的日志文件

开始之前

您要查看其日志文件的实例必须正在运行。

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 选择要查看日志文件的实例。

    此时将打开实例的“常规信息”页面。

  3. 在“常规信息”页面中，单击“查看日志文件”。

    此时将打开“日志查看器”窗口。

    注 - 实例必须已在运行。如果实例没有运行，将取消激活“查看日志文件”按钮。
