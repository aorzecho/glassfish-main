从群集中删除应用程序

删除应用程序只会从当前目标中删除该应用程序。将应用程序从目标中删除时，该应用程序会继续列在应用程序页面上，该页面显示所有已部署到 GlassFish Server 中的应用程序。在曾经部署了该应用程序的任何其他目标中，该应用程序也仍保持处于部署状态。

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“应用程序”选项卡。

    此时将打开群集的“应用程序”页面。

  4. 对于要删除的每个应用程序，选中与“已部署的应用程序”表中的应用程序相邻的复选框。
  5. 单击“删除”。

    系统将要求您确认是否要删除这些应用程序。

  6. 确认您要删除这些应用程序。

    将从该页面上的已部署应用程序表中删除这些应用程序。
