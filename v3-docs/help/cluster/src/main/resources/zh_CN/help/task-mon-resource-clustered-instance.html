查看群集 GlassFish Server 实例的资源监视数据

开始之前

必须先配置监视服务，才可以查看监视数据。

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“实例”选项卡。

    此时将打开群集的“群集服务器实例”页面。

  4. 选择要查看资源监视数据的实例。

    此时将打开实例的“常规信息”页面。

  5. 在“常规信息”页面中，单击“监视”选项卡。

    此时将打开“应用程序监视”页面。

  6. 单击“资源”选项卡。

    此时将打开“资源监视”页面。

  7. 在“查看”下拉式列表中，选择要查看资源监视数据的资源。
  8. 要更新显示的监视数据，可单击“刷新”。
