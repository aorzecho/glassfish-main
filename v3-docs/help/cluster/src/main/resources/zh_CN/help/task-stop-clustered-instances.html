停止群集 GlassFish Server 实例

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“实例”选项卡。

    此时将打开群集的“群集服务器实例”页面。

  4. 对于要停止的每个实例，选择“服务器实例”表中该实例旁边的复选框。
  5. 单击“停止”。

    系统会要求您确认是否要停止实例。

  6. 确认您要停止实例。

    停止实例后，将会更新“服务器实例”表中每个实例的状态。
