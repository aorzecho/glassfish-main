创建独立 GlassFish Server 实例的资源

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 在“独立服务器实例”页面上，选择实例。

    此时将打开实例的“常规信息”页面。

  3. 在“常规信息”页面中，单击“资源”选项卡。

    将打开实例的“资源”页面。

  4. 从“新建”下拉式菜单中，选择要创建的资源类型。

    此时将打开用于创建选定类型资源的页面。

  5. 在用于创建选定类型资源的页面上，进行所需更改并保存更改。

    有关详细信息，请参见用于创建选定类型资源的页面的联机帮助。

    注 - 创建资源时，请确保将实例指定为目标。
