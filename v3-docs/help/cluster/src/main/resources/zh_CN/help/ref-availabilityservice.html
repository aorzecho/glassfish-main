可用性服务

使用“可用性服务”页面可更改群集的可用性设置。

“可用性服务”页面包含以下选项。

配置名称

要应用该页面上的设置的配置的名称。此字段为只读。

装入默认值

用于恢复所有设置的默认值的按钮。

可用性服务

如果选中“已启用”复选框，将为群集启用可用性。默认情况下，此选项已选中。

其他属性

“可用性服务”的其他属性表。

GlassFish Server 不定义任何其他“可用性服务”属性。

对于每个属性，都将显示以下信息：

名称

属性名称。

值

为选定的实例设置的属性的值。

说明

提供有关属性的详细信息的文本说明。

“其他属性”表还包含以下选项。

添加属性

用于添加属性的按钮。单击此按钮可向“其他属性”表中添加一行。

删除属性

用于删除一个或多个选定属性的按钮。删除的任何属性会恢复其默认值，或者如果没有设置默认值，则未定义。
