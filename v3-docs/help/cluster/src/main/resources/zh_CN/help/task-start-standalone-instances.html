启动独立 GlassFish Server 实例

开始之前

应确保满足以下先决条件：

  • 为远程通信启用实例驻留的所有节点。

  • 域管理服务器 (domain administration server, DAS) 的用户可以使用安全 shell (secure shell, SSH) 登录实例驻留的所有节点的主机。

如果不满足其中任一先决条件，可通过使用 start-local-instance(1) 子命令从命令行单独启动每个实例。

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 对于要启动的每个实例，选择“服务器实例”表中该实例旁边的复选框。
  3. 单击“启动”。

    系统会要求您确认是否要启动实例。

  4. 确认您要启动实例。

    启动实例后，将会更新“服务器实例”表中每个实例的状态。
