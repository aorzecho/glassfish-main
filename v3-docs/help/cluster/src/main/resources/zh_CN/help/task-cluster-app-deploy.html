将应用程序部署到群集

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“应用程序”选项卡。

    此时将打开群集的“应用程序”页面。

  4. 在“应用程序”页面中，单击“部署”。

    此时将打开“部署应用程序或模块”页面。

  5. 在“部署应用程序或模块”页面中，提供有关应用程序的所需信息并单击“确定”。

    有关详细信息，请参见部署应用程序或模块联机帮助。
