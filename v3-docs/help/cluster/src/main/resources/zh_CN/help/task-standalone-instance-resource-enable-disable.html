启用或禁用独立 GlassFish Server 实例的资源

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 在“独立服务器实例”页面上，选择实例。

    此时将打开实例的“常规信息”页面。

  3. 在“常规信息”页面中，单击“资源”选项卡。

    将打开实例的“资源”页面。

  4. 选择资源旁边的复选框。
  5. 单击相应的按钮。
    • 要启用资源，可选择“启用”。
    • 要禁用资源，可选择“禁用”。

      禁用某个资源不会删除该资源。
