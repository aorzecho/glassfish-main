测试是否可以访问节点

要在代表远程主机的节点上创建或启动 GlassFish Server 实例，DAS 必须能够连接到该节点。尝试在这样的节点上创建或启动实例之前，请测试 DAS 是否能够连接到该节点。

所测试的节点类型必须为 DCOM 或 SSH。您无法测试类型为 CONFIG 的节点，因为未对类型为 CONFIG 的节点启用远程通信。

开始之前

请确保已在节点代表的主机上为节点类型配置了适当的协议。已对节点启用了通过此协议远程通信。

  • 如果节点类型为 DCOM，必须配置分布式组件对象模型 (DCOM) 协议。

  • 如果节点类型为 SSH，必须配置安全 shell (SSH) 协议。

    1. 在导航树中，选择“节点”节点。

      此时将打开“节点”页。此页会列出当前域中的所有节点。

    2. 在节点表中，访问您所测试的节点的试通链接。

      此时会显示一条消息，指示 DAS 是否能够连接添加到“节点”页的节点。
