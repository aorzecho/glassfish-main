资源

使用“资源”页面可对所选群集执行以下任务：

  • 查看和管理群集的现有资源

  • 创建群集的其他资源

“群集名称”字段是只读字段，其中显示当前群集的名称。

“资源”页面显示群集的现有资源的列表。对于每个资源，将提供以下信息。

资源名称

创建资源时为其指定的名称。单击该名称将打开用于编辑资源的以下页面之一，具体取决于资源的类型：

  • 编辑 JDBC 资源

  • 编辑 JavaMail 会话

  • 编辑自定义资源

  • 新建自定义资源

  • 编辑外部资源

  • 编辑连接器资源

  • 编辑受管理对象资源

已启用

指示资源是否已启用：

  • 如果资源已启用，将会显示复选标记 (✓)。

  • 如果资源已禁用，将会显示十字标记 (✗)。

类型

资源的类型。

“资源”表还包含以下选项。

启用

用于启用一个或多个选定资源的按钮。

禁用

用于禁用一个或多个选定资源的按钮。

新建

用于创建选定类型资源的下拉式列表。从列表中选择资源类型将打开用于创建选定类型资源的以下页面之一：

  • 新建 JDBC 资源

  • 新建 JMS 连接工厂

  • 新建 JMS 目的地资源

  • 新建 JavaMail 会话

  • 新建自定义资源

  • 新建外部资源

  • 新建连接器资源

  • 新建受管理对象资源

过滤

按类型过滤所显示资源列表的下拉式列表。
