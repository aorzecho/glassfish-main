删除独立 GlassFish Server 实例

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 对于要删除的每个实例，选择“服务器实例”表中该实例旁边的复选框。
  3. 单击“删除”。

    系统会要求您确认是否要删除实例。

  4. 确认您要删除实例。

    将删除该实例，并将其从该页的实例表中删除。
