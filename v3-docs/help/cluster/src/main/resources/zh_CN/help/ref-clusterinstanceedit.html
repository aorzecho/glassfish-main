常规信息

“常规信息”页面包含以下选项。

开始

用于启动实例的按钮。如果实例正在运行，则会取消激活此按钮。

停止

用于停止实例的按钮。如果实例已停止，则会取消激活此按钮。

重新启动

用于重新启动实例的按钮。如果实例已停止，则会取消激活此按钮。

查看日志文件

用于查看实例的日志文件的按钮。

恢复事务

用于恢复实例事务的按钮。

群集名称

创建群集时为其指定的名称。单击此名称可打开群集的常规信息页面。

实例名称

将实例添加到群集时为其指定的名称。此字段为只读。

状态

指示实例是在运行、需要重新启动以根据配置更改进行更新，还是未运行。此字段为只读。

JVM

如果单击“JVM 报告”，则会打开一个单独的窗口，此窗口显示有关 Java 虚拟机的报告，包括摘要报告、内存管理和垃圾收集报告、类装入报告和当前线程转储。

节点

实例驻留的节点的名称。单击名称打开节点的编辑节点页面。

配置

实例引用的配置。单击配置可打开实例的配置页面。

HTTP 端口

以逗号分隔的端口号列表，实例在这些端口上侦听以下请求类型：

  • 管理请求

  • HTTP 请求

  • HTTPS 请求

此字段为只读。

IIOP 端口

以逗号分隔的端口号列表，实例将这些端口用于以下连接类型：

  • 安全 IIOP 连接

  • 带有客户机认证的安全 IIOP 连接

  • IIOP 连接

此字段为只读。
