群集

“群集”页面显示域中群集的列表。对于每个群集，都将显示以下信息：

名称

创建群集时为其指定的名称。单击此名称可打开群集的常规信息页面。

配置

创建群集时为其选择的配置。单击配置将打开群集的配置页面。

实例

群集中每个实例的名称和状态。单击实例的名称可打开该实例的常规信息页面。

“群集”表还包含以下选项。

新建

用于创建群集的按钮。

删除

用于删除群集的按钮。

启动群集

用于启动群集的按钮。

停止群集

用于停止群集的按钮。
