启动集群

启动某个集群将启动该集群中所有尚未运行的 GlassFish Server 实例。

开始之前

应确保满足以下先决条件：

  • 已为集群中实例驻留的所有节点启用了远程通信。

  • 域管理服务器 (DAS) 的用户可使用分布式组件对象模型 (DCOM) 远程协议或安全 shell (SSH) 登录到集群中实例驻留的任何节点的主机。

如果不能满足上述任一先决条件，则通过使用 start-local-instance(1) 子命令从命令行单独启动每个实例来启动集群。

  1. 在导航树中，选择“集群”节点。

    此时将打开“集群”页。

  2. 在“集群”页中，选中集群名称旁边的复选框。

  3. 单击“启动集群”。

    系统会要求您确认是否要启动集群。

  4. 确认您要启动集群。

    将启动集群。启动集群中的每个实例后，“集群”页中会更新每个实例的状态。
