移植 EJB 计时器

“移植 EJB 计时器”页包含以下选项：

源

集群中已停止的 GlassFish Server 实例的下拉列表。EJB 计时器只能从已停止的实例移植。

目标

集群中正在运行的 GlassFish Server 实例的下拉列表。EJB 计时器只能移植到正在运行的实例中。

注:

如果集群中的所有实例都已停止，则此页仅显示一条警告：任何可以将 EJB 计时器移植到的运行实例均不可用。

“移植 EJB 计时器”页包含一个提供以下信息的 EJB 计时器表：

服务器名称

集群中 GlassFish Server 实例的列表。

计时器计数

每个实例上持久性计时器的数量。
