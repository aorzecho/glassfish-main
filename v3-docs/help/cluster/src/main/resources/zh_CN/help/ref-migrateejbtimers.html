迁移 EJB 计时器

“迁移 EJB 计时器”页面包含以下选项。

源

群集中已停止的 GlassFish Server 实例的下拉式列表。EJB 计时器只能从已停止的实例迁移。

目的地

群集中正在运行的 GlassFish Server 实例的下拉式列表。EJB 计时器只能迁移到正在运行的实例中。

注 - 如果群集中的所有实例都已停止，则此页面仅显示一则警告：无任何可用的能够向其迁移 EJB 计时器的运行中实例。
