启用或禁用部署到独立 GlassFish Server 实例的应用程序的负载平衡

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页面。

  2. 在“独立服务器实例”页面上，选择实例。

    此时将打开实例的“常规信息”页面。

  3. 在实例的“常规信息”页面上，单击“应用程序”选项卡。

    将打开实例的“应用程序”页面。

  4. 选中应用程序旁边的复选框。
  5. 在“更多操作”下拉式菜单中选择相应的命令。
    • 要启用应用程序的负载平衡，可选择“负载平衡器启用”。
    • 要禁用应用程序的负载平衡，可选择“负载平衡器禁用”。
