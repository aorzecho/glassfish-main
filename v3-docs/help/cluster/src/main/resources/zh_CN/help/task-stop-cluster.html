停止群集

停止某个群集将停止该群集中所有正在运行的 GlassFish Server 实例。

  1. 在导航树中，选择“群集”节点。

    此时将打开“群集”页面。

  2. 在“群集”页面中，选中群集名称旁边的复选框。
  3. 单击“停止群集”。

    系统会要求您确认是否要停止群集。

  4. 确认您要停止群集。

    将停止群集。停止群集中的每个实例后，“群集”页面中会更新每个实例的状态。
