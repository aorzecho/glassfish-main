停止集群

停止某个集群将停止该集群中所有正在运行的 GlassFish Server 实例。

  1. 在导航树中，选择“集群”节点。

    此时将打开“集群”页。

  2. 在“集群”页中，选中集群名称旁边的复选框。

  3. 单击“停止集群”。

    系统会要求您确认是否要停止集群。

  4. 确认您要停止集群。

    将停止集群。停止集群中的每个实例后，“集群”页中会更新每个实例的状态。
