修改群集的配置

  1. 在导航树中，展开“群集”节点。
  2. 选择群集的节点。
  3. 单击配置的名称。

    此时将打开群集的“配置”页面。

  4. 在“配置”页面上，选择所要修改的配置部分。

    此时将打开用于编辑所选配置部分的页面。

  5. 在用于编辑所选配置部分的页面上，进行所需更改并保存更改。

    有关详细信息，请参见用于编辑所选配置部分的页面的联机帮助。
