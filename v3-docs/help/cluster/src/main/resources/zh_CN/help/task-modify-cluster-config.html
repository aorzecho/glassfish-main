修改集群的配置

  1. 在导航树中，展开“集群”节点。

  2. 选择集群的节点。

  3. 单击配置的名称。

    此时将打开集群的“配置”页。

  4. 在“配置”页上，选择所要修改的配置部分。

    此时将打开用于编辑所选配置部分的页。

  5. 在用于编辑所选配置部分的页上，进行所需更改并保存更改。

    有关详细信息，请参阅用于编辑所选配置部分的页的联机帮助。
