删除集群

删除某个集群将删除该集群和其中所有 GlassFish Server 实例。删除集群前不必删除其中的实例。

如果未共享集群使用的配置，该配置也会被删除。

开始之前

请确保已停止您要删除的集群。有关说明，请参阅停止集群。

  1. 在导航树中，选择“集群”节点。

    此时将打开“集群”页。

  2. 在“集群”页中，选中集群名称旁边的复选框。

  3. 单击“删除”。

    系统会要求您确认是否要删除集群和其中的所有实例。

  4. 确认您要删除集群和其中的所有实例。

    将删除该集群，并将其从“集群”页的集群表中删除。
