删除群集

删除某个群集将删除该群集和其中所有 GlassFish Server 实例。删除群集前不必删除其中的实例。

如果未共享群集使用的配置，该配置也会被删除。

开始之前

请确保已停止您要删除的群集。

  1. 在导航树中，选择“群集”节点。

    此时将打开“群集”页面。

  2. 在“群集”页面中，选中群集名称旁边的复选框。
  3. 单击“删除”。

    系统会要求您确认是否要删除群集和其中的所有实例。

  4. 确认您要删除群集和其中的所有实例。

    将删除该群集，并将其从“群集”页面的群集表中删除。
