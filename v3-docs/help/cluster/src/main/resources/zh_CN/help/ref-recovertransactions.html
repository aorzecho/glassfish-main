恢复事务

使用“恢复事务”页面可恢复 GlassFish Server 实例的事务。

如果实例正在运行，“恢复事务”页面仅包含文本恢复由运行的服务器本身来执行以及“确定”按钮。

如果实例未在运行，“恢复事务”页面包含以下选项。

实例名称

创建实例时为其指定的名称。此字段为只读。

目标服务器

可恢复当前实例的事务的另一个实例。

事务日志目录

存储事务日志的目录。默认目录是域目录下的 /logs/tx。

事务日志保存在“事务日志位置”设置（位于群集配置的“事务服务”中）指定的目录的 tx 子目录中。“事务日志位置”的默认值为“域属性”页面（位于 GlassFish Server 节点的“高级”选项卡下）的“日志根目录”字段指定的目录。
