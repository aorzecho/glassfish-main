查看群集 GlassFish Server 实例的日志文件

开始之前

您要查看其日志文件的实例必须正在运行。

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“实例”选项卡。

    此时将打开群集的“群集服务器实例”页面。

  4. 选择要查看日志文件的实例。

    此时将打开实例的“常规信息”页面。

  5. 在“常规信息”页面中，单击“查看日志文件”。

    此时将打开“日志查看器”窗口。

    注 - 实例必须已在运行。如果实例没有运行，将取消激活“查看日志文件”按钮。
