停止独立 GlassFish Server 实例

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页。

  2. 对于要停止的每个实例，选择“服务器实例”表中该实例旁边的复选框。

  3. 单击“停止”。

    系统会要求您确认是否要停止实例。

  4. 确认您要停止实例。

    停止实例后，将会更新“服务器实例”表中每个实例的状态。
