查看集群 GlassFish Server 实例的应用程序监视数据

开始之前

必须先配置监视服务，才可以查看监视数据。

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“实例”选项卡。

    此时将打开集群的“集群服务器实例”页。

  4. 选择要查看应用程序监视数据的实例。

    此时将打开实例的“一般信息”页。

  5. 在“一般信息”页中，单击“监视”选项卡。

    此时将打开“应用程序监视”页。

  6. 在“查看”下拉列表中，选择要查看监视数据的应用程序。

  7. 要更新显示的监视数据，可单击“刷新”。
