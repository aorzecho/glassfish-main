应用程序

使用“应用程序”页可对所选的独立 GlassFish Server 实例执行以下任务：

  • 查看和管理已部署到实例的应用程序

  • 将更多应用程序部署到实例

“实例名称”字段是只读字段，其中显示当前实例的名称。

“应用程序”页显示已部署到所选实例的应用程序的列表。对于每个应用程序，均提供以下信息：

名称

应用程序名称。

名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的详细信息，请参阅 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

已启用

指示应用程序是否已启用：

  • 如果应用程序已启用，将会显示复选标记 (✓)。

  • 如果应用程序已禁用，将会显示十字标记 (✗)。

已启用负载平衡器

指示用于负载平衡的应用程序是否已启用：

  • 如果用于负载平衡的应用程序已启用，将会显示复选标记 (✓)。

  • 如果用于负载平衡的应用程序已禁用，将会显示十字标记 (✗)。

引擎

应用程序所用的容器类型。容器类型可为以下类型的任意一种：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld - Java EE 平台应用程序的 Contexts 和 DependencyInjection 的容器

已部署的应用程序表还包含以下选项：

部署

用于部署应用程序的按钮。

取消部署

用于取消部署一个或多个所选应用程序的按钮。

取消部署应用程序会从所有目标和域中删除该应用程序。取消部署应用程序时，该应用程序将不再列在应用程序页上，该页显示所有已部署到 GlassFish Server 中的应用程序。

要仅从当前目标中删除应用程序，请使用该页上的“删除”按钮。

删除

用于删除一个或多个所选应用程序的按钮。

删除应用程序只会从当前目标中删除该应用程序。将应用程序从目标中删除时，该应用程序会继续列在应用程序页上，该页显示所有已部署到 GlassFish Server 中的应用程序。在曾经部署了该应用程序的任何其他目标中，该应用程序也仍保持处于部署状态。

要从所有目标和域中删除应用程序，请使用该页上的“取消部署”按钮。

更多操作

在一个或多个所选应用程序上执行其他操作的下拉列表。

启用

启用一个或多个所选应用程序。

禁用

禁用一个或多个所选应用程序。

负载平衡器启用

启用一个或多个用于负载平衡的所选应用程序。

负载平衡器禁用

禁用一个或多个用于负载平衡的所选应用程序。

相关任务

  • 将应用程序部署到独立 GlassFish Server 实例

  • 列出已部署到独立 GlassFish Server 实例的应用程序

  • 从独立 GlassFish Server 实例取消部署应用程序

  • 从独立 GlassFish Server 实例中删除应用程序

  • 编辑已部署到独立 GlassFish Server 实例的应用程序

  • 启用或禁用已部署到独立 GlassFish Server 实例的应用程序

  • 为已部署到独立 GlassFish Server 实例的应用程序启用或禁用负载平衡
