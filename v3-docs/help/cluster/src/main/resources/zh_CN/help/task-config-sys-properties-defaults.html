设置配置系统属性的默认值

配置属性的默认值是在群集或实例引用的配置中定义的。

  1. 在导航树中，展开配置节点。
  2. 从展开的节点下的属性类型列表中，选择“系统属性”。

    此时将打开选定配置的“系统属性”页面。

    此页面会显示配置的以下系统属性的当前值：

    • 创建配置时预定义的端口号

    • 添加到配置中的自定义属性（如果有）

  3. 要确保无需重新启动群集或实例即可应用所做更改，请选择“动态重新配置”选项。
  4. 对于要为其设置默认值的每个属性，请在属性的“默认值”字段中键入新值。
  5. 为所需的任何其他属性设置默认值。
    1. 在“其他属性”表中，单击“添加属性”。

      会向“其他属性”表中添加一行。

    2. 在“实例变量名称”字段中，键入属性名称。
    3. 在“默认值”字段中，键入值。
  6. 单击“保存”。
