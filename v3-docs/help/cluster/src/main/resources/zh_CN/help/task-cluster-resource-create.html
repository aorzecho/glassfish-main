为群集创建资源

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“资源”选项卡。

    此时将打开群集的“资源”页面。

  4. 从“新建”下拉式菜单中，选择要创建的资源类型。

    此时将打开用于创建选定类型资源的页面。

  5. 在用于创建选定类型资源的页面上，进行所需更改并保存更改。

    有关详细信息，请参见用于创建选定类型资源的页面的联机帮助。

    注 - 创建资源时，请确保将群集指定为目标。
