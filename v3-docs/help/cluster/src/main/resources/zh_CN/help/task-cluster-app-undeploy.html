从集群中取消部署应用程序

取消部署应用程序会从所有目标和域中删除该应用程序。取消部署应用程序时，该应用程序将不再列在应用程序页上，该页显示所有已部署到 GlassFish Server 中的应用程序。

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“应用程序”选项卡。

    此时将打开集群的“应用程序”页。

  4. 对于要取消部署的每个应用程序，选中与“已部署的应用程序”表中的应用程序相邻的复选框。

  5. 单击“取消部署”。

    系统将要求您确认是否要取消部署这些应用程序。

  6. 确认您要取消部署这些应用程序。

    将取消部署这些应用程序，并将其从该页上的已部署的应用程序表中删除。
