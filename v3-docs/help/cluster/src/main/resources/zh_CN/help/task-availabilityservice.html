配置群集的可用性设置

  1. 在导航树中，展开“配置”节点。
  2. 选择要更改的群集配置。
  3. 在配置的节点下，选择“可用性服务”节点。

    此时将打开“可用性服务”页面。

  4. 在“可用性服务”字段中，选中“已启用”框为具有选定配置的群集启用可用性。

    要禁用可用性，请清除“可用性服务”的“已启用”框中的复选标记。

  5. （可选的）在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义任何其他群集可用性属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。
    • 要修改属性，请编辑该属性的“值”字段。
    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。
  6. 单击“保存”。
