全局启用或禁用资源

  1. 在导航树中，展开“群集”节点。
  2. 在展开的“群集”节点下，选择某个群集的节点。

    此时将打开群集的“常规信息”页面。

  3. 在“常规信息”页面中，单击“资源”选项卡。

    此时将打开群集的“资源”页面。

  4. 选择资源旁边的复选框。
  5. 单击相应的按钮。
    • 要启用资源，可选择“启用”。
    • 要禁用资源，可选择“禁用”。

      禁用某个资源不会删除该资源。
