节点

“节点”页面包含域中节点的列表。对于每个节点，都将显示以下信息：

名称

创建节点时为其指定的名称。单击名称打开节点的编辑节点页面。

节点主机

节点代表的主机的名称。

类型

节点的类型，可以为以下类型之一：

CONFIG

节点不支持远程通信。

SSH

节点支持通过安全 shell (secure shell, SSH) 进行通信。

实例

驻留在节点上的实例的列表，其中标明了每个实例是否正在运行：

  • 如果实例正在运行，将显示一个复选标记 (✓)。

  • 如果实例已停止，将显示一个感叹号 (!)。

单击某个实例的名称可打开该实例的“常规信息”页面。

“节点”表还包含以下选项。

新建

用于创建节点的按钮。

删除

用于删除一个或多个选定节点的按钮。
