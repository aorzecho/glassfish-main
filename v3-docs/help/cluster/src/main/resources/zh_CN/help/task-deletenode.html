删除节点

删除某个节点即从域管理服务器 (DAS) 的配置中删除该节点。删除驻留在节点上的最后一个 GlassFish Server 实例时，会删除节点的目录和文件。

删除节点时，还可以选择从节点主机卸载 GlassFish Server 软件。

开始之前

确保没有任何 GlassFish Server 实例驻留在即将删除的节点上。

  1. 在导航树中，选择“节点”节点。

    此时将打开“节点”页。

  2. 对于要删除的每个节点，选中与“节点”表中的节点相邻的复选框。

    注:

    无法删除预定义的 localhost-domain 节点。

  3. 单击该按钮将删除节点，或删除节点并从节点主机中卸载 GlassFish Server 软件。

    • 要删除节点而不卸载 GlassFish Server 软件，请单击“删除”。

      系统将要求您确认是否要删除节点。

    • 要删除节点并从节点主机中卸载 GlassFish Server 软件，请单击“删除和卸载”。

      系统将要求您确认是否要删除节点并从节点主机中卸载 GlassFish Server 软件。

  4. 确认是要删除节点，还是删除节点并从节点主机中卸载 GlassFish Server 软件。

    将删除节点，并将其从“节点”页的节点表中删除。如果单击“删除和卸载”，还会从节点主机中卸载 GlassFish Server 软件。
