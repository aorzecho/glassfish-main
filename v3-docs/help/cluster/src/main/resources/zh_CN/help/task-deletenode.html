删除节点

删除某个节点即从域管理服务器 (domain administration server, DAS) 的配置中删除该节点。删除驻留在节点上的最后一个 GlassFish Server 实例时，会删除节点的目录和文件。

开始之前

确保没有任何 GlassFish Server 实例驻留在即将删除的节点上。

  1. 在导航树中，选择“节点”节点。

    此时将打开“节点”页面。

  2. 对于要删除的每个节点，选中与“节点”表中的节点相邻的复选框。

    注 - 无法删除预定义的 localhost-domain 节点。

  3. 单击“删除”。

    系统将要求您确认要删除该节点。

  4. 确认您要删除该节点。

    将删除该节点，并将其从“节点”页面上的节点表中删除。
