编辑独立 GlassFish Server 实例的资源

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页。

  2. 在“独立服务器实例”页上，选择实例。

    此时将打开实例的“一般信息”页。

  3. 在“一般信息”页中，单击“资源”选项卡。

    将打开实例的“资源”页。

  4. 单击资源名称。

    此时将打开用于编辑该资源的页。

  5. 在用于编辑该资源的页中，执行所需更改并保存这些更改。

    有关详细信息，请参阅用于编辑资源的页的联机帮助。
