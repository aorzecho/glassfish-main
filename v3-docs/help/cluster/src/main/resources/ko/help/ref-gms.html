그룹 관리 서비스 편집

GMS 설정을 변경하려면 그룹 관리 서비스 페이지를 사용합니다.

그룹 관리 서비스(Group Management Service, GMS)는 클러스터 모니터링, 클러스터 멤버쉽 및 그룹 통신 서비스를 제공하는 in-process 서비스입니다. GMS는 그룹 멤버쉽 검색 및 건전성 모니터링 프로토콜의 구성 가능한 스택 위에 구축됩니다. GMS는 다음 기능을 수행합니다.

  • 하나 이상의 멤버 인스턴스가 클러스터에 추가되거나 클러스터에서 제거되거나 장애가 발생한 것으로 의심되거나 확인되면 GlassFish Server 인스턴스의 등록된 모듈에 알립니다.

  • 프로세스 그룹 간에 메시지를 주고 받을 수 있는 기능을 제공합니다.

그룹 관리 서비스 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 단추입니다. 기본값이 없는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

누락된 최대 하트비트 수

서버 인스턴스의 장애 발생이 의심되면 수행되는 모니터링 시도 횟수입니다. 기본값은 3입니다.

하트비트 빈도

모니터링 시도 간의 시간입니다. 기본값은 2000밀리초입니다.

그룹 검색 시간 초과

GMS가 다른 클러스터 멤버를 검색할 때까지 대기하는 시간입니다. 기본값은 5000밀리초입니다.

장애 발생 확인 대기 시간

의심되는 장애 발생이 확인됨으로 표시된 후의 시간입니다. 기본값은 1500밀리초입니다.

추가 속성

그룹 관리 서비스에 대한 추가 속성 테이블입니다.

GlassFish Server에서는 추가 GMS 속성을 정의하지 않습니다.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

이름

속성의 이름입니다.

값

선택된 인스턴스에 대해 설정된 속성의 값입니다.

설명

속성에 대한 자세한 정보를 제공하는 텍스트 설명입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.
