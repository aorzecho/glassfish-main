독립 실행형 GlassFish Server 인스턴스를 시작하려면

시작하기 전에

다음 필수 조건이 충족되는지 확인합니다.

  • 인스턴스가 있는 모든 노드가 원격 통신에 대해 활성화되어 있습니다.

  • DAS(도메인 관리 서버) 사용자가 보안 쉘(secure shell, SSH)을 사용하여 인스턴스가 있는 노드의 호스트에 로그인할 수 있습니다.

이러한 필수 조건 중 일부가 충족되지 않는 경우 start-local-instance(1) 하위 명령을 사용하여 명령줄에서 개별적으로 각 인스턴스를 시작합니다.

  1. 탐색 트리에서 독립 실행형 인스턴스 노드를 선택합니다.

    독립 실행형 서버 인스턴스 페이지가 열립니다.

  2. 시작할 각 인스턴스에 대해 서버 인스턴스 테이블에서 인스턴스 옆에 있는 확인란을 선택합니다.
  3. 시작을 누릅니다.

    인스턴스를 시작할 것인지 확인하는 메시지가 표시됩니다.

  4. 인스턴스를 시작할 것임을 확인합니다.

    인스턴스가 시작되면 서버 인스턴스 테이블에서 각 인스턴스의 상태가 업데이트됩니다.
