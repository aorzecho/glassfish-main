독립형 GlassFish Server 인스턴스에 대한 로그 파일을 보는 방법

시작하기 전에

로그 파일을 볼 인스턴스가 실행되고 있어야 합니다.

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. 로그 파일을 표시할 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [로그 파일 보기]를 누릅니다.

    [로그 뷰어] 창이 열립니다.

    주:

    인스턴스가 이미 실행되고 있어야 합니다. 인스턴스가 실행되고 있지 않으면 [로그 파일 보기] 단추가 비활성화됩니다
