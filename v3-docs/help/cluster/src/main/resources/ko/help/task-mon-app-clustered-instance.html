클러스터링된 GlassFish Server 인스턴스에 대한 응용 프로그램 모니터링 데이터를 보는 방법

시작하기 전에

모니터링 데이터를 보려면 모니터링 서비스를 구성해야 합니다.

  1. 탐색 트리에서 [클러스터] 노드를 확장합니다.

  2. 확장된 [클러스터] 노드 아래에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [인스턴스] 탭을 누릅니다.

    클러스터에 대한 [클러스터링된 서버 인스턴스] 페이지가 열립니다.

  4. 응용 프로그램 모니터링 데이터를 표시할 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  5. [일반 정보] 페이지에서 [모니터] 탭을 누릅니다.

    [응용 프로그램 모니터링] 페이지가 열립니다.

  6. 보기 드롭다운 목록에서 모니터링 데이터를 표시할 응용 프로그램을 선택합니다.

  7. 표시된 모니터링 데이터를 업데이트하려면 [새로 고침]을 누릅니다.

관련 항목

  • 모니터링 서비스를 구성하는 방법

  • 클러스터에 GlassFish Server 인스턴스를 추가하는 방법

  • 클러스터링된 GlassFish Server 인스턴스를 삭제하는 방법

  • 클러스터링된 GlassFish Server 인스턴스를 시작하는 방법

  • 클러스터링된 GlassFish Server 인스턴스를 정지하는 방법

  • 클러스터링된 GlassFish Server 인스턴스에 대한 로그 파일을 보는 방법

  • 클러스터링된 GlassFish Server 인스턴스에 대한 트랜잭션을 복구하는 방법

  • 클러스터링된 GlassFish Server 인스턴스의 포트 번호를 수정하는 방법

  • 클러스터링된 GlassFish Server 인스턴스의 읽기 전용 속성을 설정하는 방법

  • 클러스터링된 GlassFish Server 인스턴스에 대한 가상 서버 모니터링 데이터를 보는 방법

  • 클러스터링된 GlassFish Server 인스턴스에 대한 리소스 모니터링 데이터를 보는 방법
