자원

선택한 클러스터에 대해 다음 작업을 수행하려면 자원 페이지를 사용합니다.

  • 클러스터의 기존 자원 보기 및 관리

  • 클러스터에 대한 추가 자원 만들기

클러스터 이름 필드는 현재 클러스터의 이름을 표시하는 읽기 전용 필드입니다.

자원 페이지에는 클러스터의 기존 자원 목록이 표시됩니다. 각 자원에 대해 제공되는 정보는 다음과 같습니다.

자원 이름

자원이 만들어질 때 자원에 지정된 이름입니다. 이름을 누르면 자원 유형에 따라 자원을 편집할 수 있는 다음 페이지 중 하나가 열립니다.

  • JDBC 자원 편집

  • JavaMail 세션 편집

  • 사용자 정의 자원 편집

  • 새 사용자 정의 자원

  • 외부 자원 편집

  • 커넥터 자원 편집

  • 관리 객체 자원 편집

사용 가능

자원이 활성화되었는 지의 여부를 나타냅니다.

  • 자원이 활성화된 경우 체크 표시 기호(✓)가 표시됩니다.

  • 자원이 비활성화된 경우 십자(✗)가 표시됩니다.

유형

자원의 유형입니다.

자원 테이블에는 다음 옵션도 포함되어 있습니다.

활성화

선택한 자원을 하나 이상 활성화하는 버튼입니다.

비활성화

선택한 자원을 하나 이상 비활성화하는 버튼입니다.

새로 만들기

선택한 유형의 자원을 만드는 드롭다운 목록입니다. 목록에서 자원 유형을 선택하면 선택한 유형의 자원을 만들 수 있는 다음 페이지 중 하나가 열립니다.

  • 새 JDBC 자원

  • 새 JMS 연결 팩토리

  • 새 JMS 대상 자원

  • 새 JavaMail 세션

  • 새 사용자 정의 자원

  • 새 외부 자원

  • 새 커넥터 자원

  • 새 관리 객체 자원

필터

유형별로 표시된 자원 목록을 필터링하는 드롭다운 목록입니다.
