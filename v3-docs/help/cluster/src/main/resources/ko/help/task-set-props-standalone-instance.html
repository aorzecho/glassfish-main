독립 실행형 GlassFish Server 인스턴스에 대한 인스턴스 전용 등록 정보를 설정하려면

GlassFish Server 인스턴스의 등록 정보는 인스턴스에 대한 선택적 구성 정보를 추가합니다.

  1. 탐색 트리에서 독립 실행형 인스턴스 노드를 선택합니다.

    독립 실행형 서버 인스턴스 페이지가 열립니다.

  2. 인스턴스 전용 등록 정보를 수정할 인스턴스를 선택합니다.

    인스턴스에 대한 일반 정보 페이지가 열립니다.

  3. 일반 정보 페이지에서 등록 정보 탭을 누릅니다.
  4. 필요한 경우 인스턴스 등록 정보 탭을 누릅니다.

    인스턴스 등록 정보 페이지가 열립니다.

    이 페이지에는 선택된 GlassFish Server 인스턴스에 대해 설정된 등록 정보 목록이 표시됩니다.

  5. 수정할 각 기존 등록 정보에 대해 등록 정보의 값 필드에 새 값을 입력합니다.
  6. 필요한 추가 등록 정보 값을 설정합니다.
    1. 추가 등록 정보 테이블에서 등록 정보 추가를 누릅니다.

      추가 등록 정보 테이블에 행이 추가됩니다.

    2. 이름 필터에서 등록 정보 이름을 입력합니다.
    3. 값 필드에서 값을 입력합니다.
    4. 설명 필드에 등록 정보에 대한 간단한 설명을 입력합니다.
  7. 저장을 누릅니다.
