리소스를 전역적으로 사용 또는 사용 안함으로 설정하는 방법

  1. 탐색 트리에서 [클러스터] 노드를 확장합니다.

  2. 확장된 [클러스터] 노드 아래에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [리소스] 탭을 누릅니다.

    클러스터에 대한 [리소스] 페이지가 열립니다.

  4. 리소스 옆에 있는 체크 박스를 선택합니다.

  5. 적절한 단추를 누릅니다.

    • 리소스를 사용으로 설정하려면 [사용]을 선택합니다.

    • 리소스를 사용 안함으로 설정하려면 [사용 안함]을 선택합니다.

      리소스를 사용 안함으로 설정한다고 해서 리소스가 제거되지는 않습니다.
