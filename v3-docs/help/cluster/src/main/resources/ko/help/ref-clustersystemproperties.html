클러스터 시스템 등록 정보

클러스터 시스템 등록 정보 페이지에는 선택된 클러스터에 대해 설정된 Java 시스템 등록 정보 목록이 표시됩니다. Java 시스템 등록 정보는 GlassFish 서버 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다.

클러스터 이름 필드는 선택한 클러스터의 이름을 표시하는 읽기 전용 필드입니다.

각 등록 정보에 대해 표시되는 정보는 다음과 같습니다.

인스턴스 변수 이름

시스템 등록 정보의 이름입니다.

현재 값

현재 등록 정보에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

대체 값

변경 사항이 저장되면 등록 정보에 설정되는 값입니다.

또한 추가 등록 정보 테이블에는 다음 옵션이 포함되어 있습니다.

등록 정보 추가

등록 정보를 추가하는 버튼입니다. 이 버튼을 누르면 추가 등록 정보 테이블에 행이 추가됩니다.

등록 정보 삭제

하나 이상의 선택된 등록 정보를 삭제하는 버튼입니다. 삭제된 등록 정보는 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

관련 작업

  • 구성 시스템 등록 정보의 기본값을 설정하는 방법

동일한 asadmin 하위 명령

  • create-system-properties(1)

  • delete-system-property(1)

  • list-system-properties(1)
