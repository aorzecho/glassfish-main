리소스

선택한 독립형 GlassFish Server 인스턴스에 대해 다음 작업을 수행하려면 [리소스] 페이지를 사용합니다.

  • 인스턴스의 기존 리소스 보기 및 관리

  • 인스턴스에 대한 추가 리소스 생성

[인스턴스 이름] 필드는 현재 인스턴스의 이름을 표시하는 읽기 전용 필드입니다.

[리소스] 페이지에는 인스턴스의 기존 리소스 목록이 표시됩니다. 각 리소스에 대해 제공되는 정보는 다음과 같습니다.

리소스 이름

리소스가 생성될 때 리소스에 지정된 이름입니다. 이름을 누르면 리소스 유형에 따라 다음 페이지 중 하나가 열려 리소스를 편집할 수 있습니다.

  • JDBC 리소스 편집

  • JavaMail 세션 편집

  • 사용자 정의 리소스 편집

  • 새 사용자 정의 리소스

  • 외부 리소스 편집

  • 커넥터 리소스 편집

  • 관리 객체 리소스 편집

사용

리소스가 사용으로 설정되었는지 여부를 나타냅니다.

  • 리소스가 사용으로 설정된 경우 체크 표시 기호(✓)가 표시됩니다.

  • 리소스가 사용 안함으로 설정된 경우 십자(✗)가 표시됩니다.

유형

리소스의 유형입니다.

리소스 테이블에는 다음 옵션도 포함되어 있습니다.

사용

선택한 하나 이상의 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 리소스를 사용 안함으로 설정하는 단추입니다.

새로 만들기

선택한 유형의 리소스를 생성하는 드롭다운 목록입니다. 목록에서 리소스 유형을 선택하면 선택한 유형의 리소스를 생성할 수 있는 다음 페이지 중 하나가 열립니다.

  • 새 JDBC 리소스

  • 새 JMS 접속 팩토리

  • 새 JMS 대상 리소스

  • 새 JavaMail 세션

  • 새 사용자 정의 리소스

  • 새 외부 리소스

  • 새 커넥터 리소스

  • 새 관리 객체 리소스

필터

유형별로 표시된 리소스 목록을 필터링하는 드롭다운 목록입니다.

관련 작업

  • 독립형 GlassFish Server 인스턴스에 대한 리소스를 생성하는 방법

  • 독립형 GlassFish Server 인스턴스에 대한 리소스를 나열하는 방법

  • 독립형 GlassFish Server 인스턴스에 대한 리소스를 사용 또는 사용 안함으로 설정하는 방법

  • 독립형 GlassFish Server 인스턴스에 대한 리소스를 편집하는 방법
