독립형 서버 인스턴스

독립형 GlassFish Server 인스턴스를 생성하고 관리하려면 독립형 서버 인스턴스 페이지를 사용합니다.

독립형 서버 인스턴스 페이지에는 도메인의 모든 독립형 인스턴스 목록이 표시됩니다. 각 인스턴스에 대해 제공되는 정보는 다음과 같습니다.

이름

인스턴스가 생성될 때 인스턴스에 지정된 이름입니다. 이름을 누르면 해당 인스턴스에 대한 일반 정보 페이지가 열립니다.

LB 가중치

인스턴스의 로드 균형 조정 가중치를 나타내는 정수입니다.

로드 균형 조정 가중치는 인스턴스가 처리해야 하는 클러스터에 대한 모든 요청의 비율을 결정합니다.

예를 들어 2개의 인스턴스 클러스터에서 한 인스턴스가 네 개의 요청 중에서 하나를 처리하고 다른 인스턴스가 네 개의 요청 중에서 세 개를 처리하도록 하고자 할 수 있습니다. 이 경우 네 개의 요청 중에서 하나를 처리할 인스턴스의 가중치를 1로 설정하고 다른 인스턴스의 가중치를 3으로 설정합니다. 백분율을 사용하려는 경우 인스턴스의 가중치를 각각 25와 75로 설정합니다.

기본 가중치는 100입니다.

구성

인스턴스가 생성될 때 인스턴스에 대해 선택된 구성입니다. 구성을 누르면 인스턴스에 대한 구성 페이지가 열립니다.

노드

인스턴스가 있는 노드의 이름입니다. 이름을 누르면 해당 노드에 대한 노드 편집 페이지가 열립니다.

상태

인스턴스가 실행되고 있는지 또는 실행되고 있지 않은지를 나타냅니다.

서버 인스턴스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

독립형 인스턴스를 생성하는 단추입니다.

삭제

선택한 독립형 인스턴스를 하나 이상 삭제하는 단추입니다.

시작

선택한 하나 이상의 독립형 인스턴스를 시작하는 단추입니다.

정지

선택한 하나 이상의 독립형 인스턴스를 정지하는 단추입니다.

관련 작업

  • 독립형 GlassFish Server 인스턴스를 생성하는 방법

  • 독립형 GlassFish Server 인스턴스를 삭제하는 방법

  • 독립형 GlassFish Server 인스턴스를 시작하는 방법

  • 독립형 GlassFish Server 인스턴스를 정지하는 방법
