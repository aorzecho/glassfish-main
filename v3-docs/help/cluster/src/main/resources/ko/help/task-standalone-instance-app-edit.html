독립형 GlassFish Server 인스턴스에 배치된 응용 프로그램을 편집하는 방법

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. [독립형 서버 인스턴스] 페이지에서 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  3. 인스턴스에 대한 [일반 정보] 페이지에서 [응용 프로그램] 탭을 누릅니다.

    인스턴스에 대한 [응용 프로그램] 페이지가 열립니다.

  4. [응용 프로그램] 페이지의 배치된 응용 프로그램 테이블에서 응용 프로그램 이름을 누릅니다.

    [응용 프로그램 편집] 페이지가 열립니다.

  5. [응용 프로그램 편집] 페이지에서 응용 프로그램에 대한 필수 정보를 입력하고 변경 사항을 저장합니다.

    자세한 내용은 응용 프로그램 편집 온라인 도움말을 참조하십시오.
