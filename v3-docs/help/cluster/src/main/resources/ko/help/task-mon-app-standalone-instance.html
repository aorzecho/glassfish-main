독립형 GlassFish Server 인스턴스에 대한 응용 프로그램 모니터링 데이터를 보는 방법

시작하기 전에

모니터링 데이터를 보려면 모니터링 서비스를 구성해야 합니다.

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. 응용 프로그램 모니터링 데이터를 표시할 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [모니터] 탭을 누릅니다.

    [응용 프로그램 모니터링] 페이지가 열립니다.

  4. 보기 드롭다운 목록에서 모니터링 데이터를 표시할 응용 프로그램을 선택합니다.

  5. 표시된 모니터링 데이터를 업데이트하려면 [새로 고침]을 누릅니다.
