클러스터

클러스터 페이지에는 도메인의 클러스터 목록이 표시됩니다. 각 클러스터에 대해 표시되는 정보는 다음과 같습니다.

이름

클러스터가 만들어질 때 클러스터에 지정된 이름입니다. 이름을 누르면 클러스터에 대한 일반 정보 페이지가 열립니다.

구성

클러스터가 만들어질 때 클러스터에 대해 선택된 구성입니다. 구성을 누르면 클러스터에 대한 구성 페이지가 열립니다.

인스턴스

클러스터에 있는 각 인스턴스의 이름 및 상태입니다. 인스턴스의 이름을 누르면 인스턴스에 대한 일반 정보 페이지가 열립니다.

클러스터 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

클러스터를 만드는 버튼입니다.

삭제

클러스터를 삭제하는 버튼입니다.

클러스터 시작

클러스터를 시작하는 버튼입니다.

클러스터 중지

클러스터를 중지하는 버튼입니다.

  • 클러스터를 삭제하는 방법

  • 클러스터를 시작하는 방법

  • 클러스터를 중지하는 방법
