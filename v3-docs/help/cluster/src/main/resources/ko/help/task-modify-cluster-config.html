클러스터의 구성을 수정하는 방법

  1. 탐색 트리에서 클러스터 노드를 확장합니다.
  2. 원하는 클러스터 노드를 선택합니다.
  3. 구성의 이름을 누릅니다.

    클러스터에 대한 구성 페이지가 열립니다.

  4. 구성 페이지에서 수정할 구성의 구성 요소를 선택합니다.

    선택한 구성의 구성 요소 편집을 위한 페이지가 열립니다.

  5. 선택한 구성 요소 편집을 위한 페이지에서 필요한 변경 사항을 수행하고 변경 사항을 저장합니다.

    자세한 내용은 선택한 구성 요소 편집을 위한 페이지의 온라인 도움말을 참조하십시오.

참조

  • 클러스터를 만드는 방법

  • 클러스터를 삭제하는 방법

  • 클러스터에 대한 가용성 설정 구성하는 방법

  • 클러스터를 시작하는 방법

  • 클러스터를 중지하는 방법

  • 클러스터에 대한 응용 프로그램을 배포하는 방법

  • 클러스터에 배포된 응용 프로그램을 나열하는 방법

  • 클러스터에서 응용 프로그램 배포를 해제하는 방법

  • 클러스터에서 응용 프로그램을 제거하는 방법

  • 클러스터에 배포된 응용 프로그램을 편집하는 방법

  • 클러스터에 배포된 응용 프로그램을 활성화 또는 비활성화하는 방법

  • 클러스터에 배포된 응용 프로그램을 로드 균형 조정 활성화 또는 비활성화하는 방법

  • 클러스터에 대한 자원을 만드는 방법

  • 클러스터에 대한 자원을 나열하는 방법

  • 전역으로 자원 활성화 또는 비활성화

  • 클러스터에 대한 자원을 편집하는 방법

  • 서비스 손실 없이 구성 요소를 업그레이드하는 방법
