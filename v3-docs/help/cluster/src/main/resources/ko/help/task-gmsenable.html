그룹 관리 서비스를 사용으로 설정하는 방법

그룹 관리 서비스(Group Management Service, GMS)는 클러스터 모니터링, 클러스터 멤버쉽 및 그룹 통신 서비스를 제공하는 in-process 서비스입니다. GMS는 다음 기능을 수행합니다.

  • 하나 이상의 멤버 인스턴스가 클러스터에 추가되거나 클러스터에서 제거되거나 장애가 발생한 것으로 의심되거나 확인되면 GlassFish Server 인스턴스의 등록된 모듈에 알립니다.

  • 프로세스 그룹 간에 메시지를 주고 받을 수 있는 기능을 제공합니다.

  1. 탐색 트리에서 [클러스터] 노드를 확장합니다.

  2. 그룹 관리 서비스(Group Management Service, GMS)를 사용으로 설정할 클러스터를 선택합니다.

    클러스터에 대한 [일반 정보] 페이지가 열립니다.

  3. [GMS 사용] 체크 박스를 선택하여 클러스터에 대해 GMS를 사용으로 설정합니다.

    이 옵션은 기본적으로 선택됩니다. 클러스터에 대해 GMS를 사용 안함으로 설정하려면 [GMS 사용] 체크 박스에서 선택 표시를 제거합니다.

    사용으로 설정된 경우 그룹 관리 서비스가 클러스터 및 DAS(도메인 관리 서버)의 각 서버 인스턴스에서 시작됩니다. GMS가 사용으로 설정된 각 클러스터에 DAS가 참여합니다.

  4. [저장]을 누릅니다.
