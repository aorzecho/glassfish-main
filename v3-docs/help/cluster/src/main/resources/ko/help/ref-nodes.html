노드

노드 페이지에는 도메인의 노드 목록이 표시됩니다. 각 노드에 대해 표시되는 정보는 다음과 같습니다.

이름

노드가 만들어질 때 노드에 지정된 이름입니다. 이름을 누르면 노드에 대한 노드 편집 페이지가 열립니다.

노드 호스트

노드가 나타내는 호스트의 이름입니다.

유형

노드 유형으로, 다음 유형 중 하나입니다.

CONFIG

노드가 원격 통신을 지원하지 않습니다.

SSH

노드가 SSH(보안 쉘)를 통한 통신을 지원합니다.

인스턴스

노드에 있는 인스턴스와 각 인스턴스가 실행되고 있는지가 표시된 목록입니다.

  • 인스턴스가 실행되고 있는 경우 체크 표시 기호(✓)가 표시됩니다.

  • 인스턴스가 중지된 경우 느낌표(!)가 표시됩니다.

인스턴스의 이름을 누르면 인스턴스에 대한 일반 정보 페이지가 열립니다.

노드 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

노드를 만드는 버튼입니다.

삭제

하나 이상의 선택된 노드를 삭제하는 버튼입니다.

관련 작업

  • 노드를 만드는 방법

  • 노드를 삭제하는 방법

  • 현재 도메인의 모든 노드를 나열하는 방법

  • 노드를 편집하는 방법
