독립 실행형 GlassFish 서버 인스턴스에서 응용 프로그램을 제거하는 방법

응용 프로그램을 제거하면 응용 프로그램이 현재 대상에서만 제거됩니다. 응용 프로그램이 대상에서 제거되면 해당 응용 프로그램은 GlassFish 서버에 배포된 응용 프로그램을 모두 표시하는 응용 프로그램 페이지에 계속 표시됩니다. 또한 응용 프로그램이 배포된 다른 대상에는 배포된 상태로 남아 있습니다.

  1. 탐색 트리에서 독립 실행형 인스턴스 노드를 선택합니다.

    독립 실행형 서버 인스턴스 페이지가 열립니다.

  2. 독립 실행형 서버 인스턴스 페이지에서 인스턴스를 선택합니다.

    인스턴스에 대한 일반 정보 페이지가 열립니다.

  3. 인스턴스에 대한 일반 정보 페이지에서 응용 프로그램 탭을 누릅니다.

    인스턴스에 대한 응용 프로그램 페이지가 열립니다.

  4. 제거할 각 응용 프로그램에 대해 배포된 응용 프로그램 테이블에서 응용 프로그램 옆에 있는 확인란을 선택합니다.
  5. 제거를 누릅니다.

    응용 프로그램을 제거할 것인지 확인하는 메시지가 표시됩니다.

  6. 응용 프로그램을 제거할 것임을 확인합니다.

    응용 프로그램이 페이지에 있는 배포된 응용 프로그램 테이블에서 제거됩니다.
