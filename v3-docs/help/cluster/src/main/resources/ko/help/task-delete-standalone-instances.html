    독립 실행형 서버 인스턴스 페이지가 열립니다.

    인스턴스를 삭제할 것인지 확인하는 메시지가 표시됩니다.

  4. 인스턴스를 삭제할 것임을 확인합니다.

    인스턴스가 삭제되고 페이지의 인스턴스 테이블에서 제거됩니다.
