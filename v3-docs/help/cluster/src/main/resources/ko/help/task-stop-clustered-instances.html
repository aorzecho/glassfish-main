클러스터링된 GlassFish 서버 인스턴스를 중지하려면

  1. 탐색 트리에서 클러스터 노드를 확장합니다.
  2. 확장된 클러스터 노드 아래에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 일반 정보 페이지가 열립니다.

  3. 일반 정보 페이지에서 인스턴스 탭을 누릅니다.

    클러스터에 대한 클러스터링된 서버 인스턴스 페이지가 열립니다.

  4. 중지할 각 인스턴스에 대해 서버 인스턴스 테이블에서 인스턴스 옆에 있는 확인란을 선택합니다.
  5. 중지를 누릅니다.

    인스턴스를 중지할 것인지 확인하는 메시지가 표시됩니다.

  6. 인스턴스를 중지할 것임을 확인합니다.

    인스턴스가 중지되면 서버 인스턴스 테이블에서 각 인스턴스의 상태가 업데이트됩니다.

참조

  • 클러스터에 GlassFish 서버 인스턴스를 추가하려면

  • 클러스터링된 GlassFish 서버 인스턴스를 삭제하려면

  • 클러스터링된 GlassFish 서버 인스턴스를 시작하려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 로그 파일을 보려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 트랜잭션을 복구하려면

  • 클러스터링된 GlassFish 서버 인스턴스의 포트 번호를 수정하려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 인스턴스 전용 등록 정보를 설정하려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 응용 프로그램 모니터링 데이터를 보려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 가상 서버 모니터링 데이터를 보려면

  • 클러스터링된 GlassFish 서버 인스턴스에 대한 자원 모니터링 데이터를 보려면
