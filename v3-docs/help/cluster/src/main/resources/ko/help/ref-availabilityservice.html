가용성 서비스

클러스터에 대한 가용성 설정을 변경하려면 가용성 서비스 페이지를 사용합니다.

가용성 서비스 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정을 적용할 구성의 이름입니다. 이 필드는 읽기 전용입니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

가용성 서비스

사용 가능 확인란을 선택하면 클러스터에 대해 가용성이 활성화됩니다. 이 옵션은 기본적으로 선택됩니다.

추가 등록 정보

가용성 서비스에 대한 추가 등록 정보 테이블입니다.

GlassFish Server에서는 추가 가용성 서비스 등록 정보를 정의하지 않습니다.

각 등록 정보에 대해 표시되는 정보는 다음과 같습니다.

이름

등록 정보의 이름입니다.

값

선택된 인스턴스에 대해 설정된 등록 정보의 값입니다.

설명

등록 정보에 대한 자세한 정보를 제공하는 텍스트 설명입니다.

또한 추가 등록 정보 테이블에는 다음 옵션이 포함되어 있습니다.

등록 정보 추가

등록 정보를 추가하는 버튼입니다. 이 버튼을 누르면 추가 등록 정보 테이블에 행이 추가됩니다.

등록 정보 삭제

하나 이상의 선택된 등록 정보를 삭제하는 버튼입니다. 삭제된 등록 정보는 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.
