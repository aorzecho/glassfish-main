클러스터에 대한 가용성 설정 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 변경할 클러스터에 대한 구성을 선택합니다.
  3. 구성 노드 아래에서 가용성 서비스 노드를 선택합니다.

    가용성 서비스 페이지가 열립니다.

  4. 가용성 서비스 필드에서 사용 가능 상자를 선택하여 선택한 구성이 포함된 클러스터에 대한 가용성을 활성화합니다.

    가용성을 비활성화하려면 가용성 서비스 사용 가능 상자에서 선택 표시를 제거합니다.

  5. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    GlassFish Server에서는 클러스터 가용성에 대한 추가 등록 정보를 정의하지 않습니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  6. 저장을 누릅니다.

  • 클러스터에 배포된 응용 프로그램을 로드 균형 조정 활성화 또는 비활성화하는 방법

  • 클러스터에 대한 자원을 나열하는 방법
