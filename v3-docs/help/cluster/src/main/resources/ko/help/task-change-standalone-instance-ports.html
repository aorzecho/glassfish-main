독립형 GlassFish Server 인스턴스의 포트 번호를 수정하는 방법

독립형 GlassFish Server 인스턴스의 포트 번호는 처음에 인스턴스가 참조하는 구성에 설정되어 있습니다. 예를 들어 동일한 구성을 참조하는 두 개의 인스턴스가 동일한 호스트에 있는 경우 이러한 포트 번호를 수정해야 합니다. 이 경우 두 인스턴스에서 동일한 구성을 참조하므로 한 인스턴스의 포트 번호를 수정해야 합니다.

인스턴스의 포트 번호는 Java 시스템 속성으로 저장됩니다. 이러한 속성은 GlassFish Server 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. 포트 번호를 수정할 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [속성] 탭을 누릅니다.

    [인스턴스 시스템 속성] 페이지가 열립니다.

    이 페이지에는 인스턴스에 대한 다음 시스템 속성의 현재 값이 표시됩니다.

    • 인스턴스의 구성에 미리 정의된 포트 번호

    • 인스턴스에 대해 정의된 사용자 정의 속성(있는 경우)

  4. 수정할 각 기존 속성에 대해 속성의 [대체 값] 필드에 새 값을 입력합니다.

  5. 필요한 추가 속성값을 설정합니다.

    1. 추가 속성 테이블에서 [속성 추가]를 누릅니다.

      추가 속성 테이블에 행이 추가됩니다.

    2. [인스턴스 변수 이름] 필드에 속성 이름을 입력합니다.

    3. [대체 값] 필드에 값을 입력합니다.

  6. [저장]을 누릅니다.
