독립 실행형 GlassFish 서버 인스턴스에 대한 자원을 활성화 또는 비활성화하려면

  1. 탐색 트리에서 독립 실행형 인스턴스 노드를 선택합니다.

    독립 실행형 서버 인스턴스 페이지가 열립니다.

  2. 독립 실행형 서버 인스턴스 페이지에서 인스턴스를 선택합니다.

    인스턴스에 대한 일반 정보 페이지가 열립니다.

  3. 일반 정보 페이지에서 자원 탭을 누릅니다.

    인스턴스에 대한 자원 페이지가 열립니다.

  4. 자원 옆에 있는 확인란을 선택합니다.
  5. 적절한 버튼을 누릅니다.
    • 자원을 활성화하려면 활성화를 선택합니다.
    • 자원을 비활성화하려면 비활성화를 선택합니다.

      자원을 비활성화한다고 해서 자원이 제거되지는 않습니다.
