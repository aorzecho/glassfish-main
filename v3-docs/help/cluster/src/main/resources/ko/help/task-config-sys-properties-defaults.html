구성 시스템 등록 정보의 기본값을 설정하는 방법

구성 등록 정보의 기본값은 클러스터 또는 인스턴스가 참조하는 구성에 정의되어 있습니다.

  1. 탐색 트리에서 구성 노드를 확장합니다.
  2. 확장된 노드 아래의 등록 정보 유형 목록에서 시스템 등록 정보를 선택합니다.

    선택한 구성에 대한 시스템 등록 정보 페이지가 열립니다.

    이 페이지에는 구성에 대한 다음 시스템 등록 정보의 현재 값이 표시됩니다.

    • 구성이 작성되었을 때 포트 번호가 미리 정의됨

    • 구성에 추가된 사용자 정의 등록 정보(있는 경우)

  3. 클러스터 또는 인스턴스를 다시 시작할 필요 없이 변경 사항이 적용되도록 하려면 동적 재구성 옵션을 선택합니다.
  4. 기본값을 설정할 각 등록 정보에 대해 등록 정보의 기본값 필드에 새 값을 입력합니다.
  5. 필요한 추가 등록 정보의 기본값을 설정합니다.
    1. 추가 등록 정보 테이블에서 등록 정보 추가를 누릅니다.

      추가 등록 정보 테이블에 행이 추가됩니다.

    2. 인스턴스 변수 이름 필드에 등록 정보 이름을 입력합니다.
    3. 기본값 필드에 값을 입력합니다.
  6. 저장을 누릅니다.
