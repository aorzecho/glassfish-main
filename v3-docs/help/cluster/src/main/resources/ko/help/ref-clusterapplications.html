응용 프로그램

현재 클러스터에 대해 다음 작업을 수행하려면 응용 프로그램 페이지를 사용합니다.

  • 클러스터에 이미 배치된 응용 프로그램 보기 및 관리

  • 클러스터에 응용 프로그램 추가 배치

[클러스터 이름] 필드는 현재 클러스터의 이름을 표시하는 읽기 전용 필드입니다.

응용 프로그램 페이지에는 현재 클러스터에 배치된 응용 프로그램의 목록이 표시됩니다. 각 응용 프로그램에 대해 제공되는 정보는 다음과 같습니다.

이름

응용 프로그램 이름입니다.

이름에는 버전 식별자(옵션)가 포함될 수 있으며, 이 식별자는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

사용

응용 프로그램이 사용으로 설정되었는지 여부를 나타냅니다.

  • 응용 프로그램이 사용으로 설정된 경우 체크 표시 기호(✓)가 표시됩니다.

  • 응용 프로그램이 사용 안함으로 설정된 경우 십자(✗)가 표시됩니다.

LB 사용

응용 프로그램이 로드 균형 조정에 대해 사용으로 설정되었는지 여부를 나타냅니다.

  • 응용 프로그램이 로드 균형 조정에 대해 사용으로 설정된 경우 체크 표시 기호(✓)가 표시됩니다.

  • 응용 프로그램이 로드 균형 조정에 대해 사용 안함으로 설정된 경우 십자(✗)가 표시됩니다.

엔진

응용 프로그램에서 사용하는 컨테이너 유형입니다. 컨테이너 유형은 다음과 같은 유형일 수 있습니다.

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — Java EE 플랫폼 응용 프로그램에 대한 Contexts and Dependency Injection 컨테이너

배치된 응용 프로그램 테이블에는 다음 옵션도 포함되어 있습니다.

배치

응용 프로그램을 배치하는 단추입니다.

배치 해제

선택한 하나 이상의 응용 프로그램을 배치 해제하는 단추입니다.

응용 프로그램을 배치 해제하면 모든 대상과 도메인에서 응용 프로그램이 제거됩니다. 응용 프로그램이 배치 해제되면 해당 응용 프로그램은 GlassFish Server에 배치되는 모든 응용 프로그램을 표시하는 응용 프로그램 페이지에 더 이상 표시되지 않습니다.

현재 대상에서만 응용 프로그램을 제거하려면 이 페이지에 있는 [제거] 단추를 사용합니다.

제거

선택한 하나 이상의 응용 프로그램을 제거하는 단추입니다.

응용 프로그램을 제거하면 응용 프로그램이 현재 대상에서만 제거됩니다. 응용 프로그램이 대상에서 제거되면 해당 응용 프로그램은 GlassFish Server에 배치된 응용 프로그램을 모두 표시하는 응용 프로그램 페이지에 계속 표시됩니다. 또한 응용 프로그램이 배치된 다른 대상에는 배치된 상태로 남아 있습니다.

모든 대상 및 도메인에서 응용 프로그램을 제거하려면 이 페이지에 있는 [배치 해제] 단추를 사용합니다.

추가 작업

선택한 하나 이상의 응용 프로그램에 대해 추가 작업을 수행하는 드롭다운 목록입니다.

사용

선택한 하나 이상의 응용 프로그램을 사용으로 설정합니다.

사용 안함

선택한 하나 이상의 응용 프로그램을 사용 안함으로 설정합니다.

로드 밸런서 사용

선택한 하나 이상의 응용 프로그램을 로드 균형 조정에 대해 사용으로 설정합니다.

로드 밸런서 사용 안함

선택한 하나 이상의 응용 프로그램을 로드 균형 조정에 대해 사용 안함으로 설정합니다.
