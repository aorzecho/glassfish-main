인스턴스 시스템 속성

[인스턴스 시스템 속성] 페이지에는 현재 GlassFish Server 인스턴스에 대해 설정된 Java 시스템 속성 목록이 표시됩니다. Java 시스템 속성은 GlassFish Server 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다

이러한 속성은 인스턴스 구성의 포트 설정에 대한 속성 정의를 무시합니다. 예를 들어 두 개의 클러스터링된 인스턴스가 동일한 호스트에 있는 경우 사전 정의된 포트 설정을 무시해야 합니다. 이런 경우 두 인스턴스에서 동일한 구성을 공유하므로 한 인스턴스의 포트 설정을 무시해야 합니다.

구성의 사전 정의된 포트 설정에 대한 자세한 내용은 미리 정의된 구성 시스템 속성을 참조하십시오.

[인스턴스 이름] 필드는 현재 인스턴스의 이름을 표시하는 읽기 전용 필드입니다.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

인스턴스 변수 이름

시스템 속성의 이름입니다.

현재 값

현재 속성에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

대체 값

변경 사항이 저장되면 속성에 설정되는 값입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

관련 작업

  • 독립형 GlassFish Server 인스턴스의 포트 번호를 수정하는 방법

  • 구성 시스템 속성의 기본값을 설정하는 방법

동일한 asadmin 하위 명령

  • create-system-properties(1)

  • delete-system-property(1)

  • list-system-properties(1)
