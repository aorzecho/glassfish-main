일반 정보

일반 정보 페이지에는 다음 옵션이 포함되어 있습니다.

시작

인스턴스를 시작하는 버튼입니다. 인스턴스가 실행되고 있는 경우 이 버튼이 비활성화됩니다.

중지

인스턴스를 중지하는 버튼입니다. 인스턴스가 중지된 경우 이 버튼이 비활성화됩니다.

다시 시작

인스턴스를 다시 시작하는 버튼입니다. 인스턴스가 중지된 경우 이 버튼이 비활성화됩니다.

로그 파일 보기

인스턴스에 대한 로그 파일을 보는 버튼입니다.

트랜잭션 복구

인스턴스에 대한 트랜잭션을 복구하는 버튼입니다.

클러스터 이름

클러스터가 만들어질 때 클러스터에 지정된 이름입니다. 이름을 누르면 클러스터에 대한 일반 정보 페이지가 열립니다.

인스턴스 이름

인스턴스가 클러스터에 추가될 때 인스턴스에 지정된 이름입니다. 이 필드는 읽기 전용입니다.

상태

인스턴스가 실행 중인지, 구성 변경 사항으로 업데이트하기 위해 다시 시작해야 하는지 또는 실행되고 있지 않은지를 나타냅니다. 이 필드는 읽기 전용입니다.

JVM

JVM 보고서를 누르면 별도의 창이 열리고 요약 보고서, 메모리 관리 및 가비지 컬렉션 보고서, 클래스 로딩 보고서 및 현재 스레드 덤프를 포함한 Java Virtual Machine에 대한 보고서가 표시됩니다.

노드

인스턴스가 있는 노드의 이름입니다. 이름을 누르면 노드에 대한 노드 편집 페이지가 열립니다.

구성

인스턴스가 참조하는 구성입니다. 구성을 누르면 인스턴스에 대한 구성 페이지가 열립니다.

HTTP 포트

인스턴스가 다음 요청 유형을 수신하는 포트의 쉼표로 구분된 포트 번호 목록입니다.

  • 관리 요청

  • HTTP 요청

  • HTTPS 요청

이 필드는 읽기 전용입니다.

IIOP 포트

인스턴스가 다음 연결 유형을 사용하는 포트의 쉼표로 구분된 포트 번호 목록입니다.

  • 보안 IIOP 연결

  • 보안 IIOP 연결(클라이언트 인증 포함)

  • IIOP 연결

이 필드는 읽기 전용입니다.
