독립형 GlassFish Server 인스턴스에 배치된 응용 프로그램에 대한 로드 균형 조정을 사용 또는 사용 안함으로 설정하는 방법

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. [독립형 서버 인스턴스] 페이지에서 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  3. 인스턴스에 대한 [일반 정보] 페이지에서 [응용 프로그램] 탭을 누릅니다.

    인스턴스에 대한 [응용 프로그램] 페이지가 열립니다.

  4. 응용 프로그램 옆에 있는 체크 박스를 선택합니다.

  5. 추가 작업 풀다운 메뉴에서 적절한 명령을 선택합니다.

    • 응용 프로그램에 대해 로드 균형 조정을 사용으로 설정하려면 [로드 밸런서 사용]을 선택합니다.

    • 응용 프로그램에 대해 로드 균형 조정을 사용 안함으로 설정하려면 [로드 밸런서 사용 안함]을 선택합니다.
