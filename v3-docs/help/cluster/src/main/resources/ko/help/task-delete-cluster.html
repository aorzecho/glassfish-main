클러스터를 삭제하는 방법

클러스터를 삭제하면 클러스터와 클러스터에 있는 모든 GlassFish Server 인스턴스가 삭제됩니다. 클러스터를 삭제하기 전에 클러스터에 있는 인스턴스를 삭제할 필요는 없습니다.

클러스터가 사용하는 구성이 공유되어 있지 않으면 구성도 삭제됩니다.

시작하기 전에

삭제할 클러스터가 중지되었는지 확인합니다.

  1. 탐색 트리에서 클러스터 노드를 선택합니다.

    클러스터 페이지가 열립니다.

  2. 클러스터 페이지에서 클러스터 이름 옆에 있는 확인란을 선택합니다.
  3. 삭제를 누릅니다.

    클러스터와 클러스터에 있는 모든 인스턴스를 삭제할 것인지 확인하는 메시지가 표시됩니다.

  4. 클러스터와 클러스터에 있는 모든 인스턴스를 삭제할 것임을 확인합니다.

    클러스터가 삭제되고 클러스터 페이지의 클러스터 테이블에서 제거됩니다.

참조

  • 클러스터를 만드는 방법

  • 클러스터에 대한 가용성 설정 구성하는 방법

  • 클러스터를 시작하는 방법

  • 클러스터를 중지하는 방법

  • 클러스터의 구성을 수정하는 방법

  • 클러스터에 대한 응용 프로그램을 배포하는 방법

  • 클러스터에 배포된 응용 프로그램을 나열하는 방법

  • 클러스터에서 응용 프로그램 배포를 해제하는 방법

  • 클러스터에서 응용 프로그램을 제거하는 방법

  • 클러스터에 배포된 응용 프로그램을 편집하는 방법

  • 클러스터에 배포된 응용 프로그램을 활성화 또는 비활성화하는 방법

  • 클러스터에 배포된 응용 프로그램을 로드 균형 조정 활성화 또는 비활성화하는 방법

  • 클러스터에 대한 자원을 만드는 방법

  • 클러스터에 대한 자원을 나열하는 방법

  • 전역으로 자원 활성화 또는 비활성화

  • 클러스터에 대한 자원을 편집하는 방법

  • 서비스 손실 없이 구성 요소를 업그레이드하는 방법
