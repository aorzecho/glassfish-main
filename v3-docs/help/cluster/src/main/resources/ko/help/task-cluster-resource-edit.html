클러스터 리소스를 편집하는 방법

  1. 탐색 트리에서 [클러스터] 노드를 확장합니다.

  2. 확장된 [클러스터] 노드 아래에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [리소스] 탭을 누릅니다.

    클러스터에 대한 [리소스] 페이지가 열립니다.

  4. 리소스 이름을 누릅니다.

    리소스 편집을 위한 페이지가 열립니다.

  5. 리소스 편집을 위한 페이지에서 필요한 변경 사항을 수행하고 변경 사항을 저장합니다.

    자세한 내용은 리소스 편집을 위한 페이지의 온라인 도움말을 참조하십시오.
