클러스터에 대한 자원을 편집하는 방법

  1. 탐색 트리에서 클러스터 노드를 확장합니다.
  2. 확장된 클러스터 노드 아래에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 일반 정보 페이지가 열립니다.

  3. 일반 정보 페이지에서 자원 탭을 누릅니다.

    클러스터에 대한 자원 페이지가 열립니다.

  4. 자원 이름을 누릅니다.

    자원 편집을 위한 페이지가 열립니다.

  5. 자원 편집을 위한 페이지에서 필요한 변경 사항을 수행하고 변경 사항을 저장합니다.

    자세한 내용은 자원 편집을 위한 페이지의 온라인 도움말을 참조하십시오.

  • 클러스터를 삭제하는 방법

  • 클러스터를 시작하는 방법

  • 클러스터를 중지하는 방법

  • 클러스터의 구성을 수정하는 방법

  • 클러스터에 대한 응용 프로그램을 배포하는 방법

  • 클러스터에 배포된 응용 프로그램을 나열하는 방법

  • 클러스터에서 응용 프로그램 배포를 해제하는 방법

  • 클러스터에서 응용 프로그램을 제거하는 방법

  • 클러스터에 배포된 응용 프로그램을 활성화 또는 비활성화하는 방법

  • 클러스터에 배포된 응용 프로그램을 로드 균형 조정 활성화 또는 비활성화하는 방법
