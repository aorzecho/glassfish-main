EJB 타이머 마이그레이션

EJB 타이머 마이그레이션 페이지에는 다음 옵션이 포함되어 있습니다.

소스

클러스터에 있는 정지된 GlassFish Server 인스턴스의 드롭다운 목록입니다. EJB 타이머는 정지된 인스턴스에서만 마이그레이션할 수 있습니다.

대상

클러스터에 있는 실행 중인 GlassFish Server 인스턴스의 드롭다운 목록입니다. EJB 타이머는 실행되고 있는 인스턴스로만 마이그레이션할 수 있습니다.

주:

클러스터의 모든 인스턴스가 정지된 경우 이 페이지에는 EJB 타이머를 마이그레이션할 수 있는 실행되고 있는 인스턴스가 없다는 경고만 표시됩니다.

EJB 타이머 마이그레이션 페이지는 다음 정보를 포함하는 EJB 타이머 테이블로 구성됩니다.

서버 이름

클러스터에 있는 GlassFish Server 인스턴스의 목록입니다.

타이머 수

각 인스턴스에 있는 영구 타이머의 수입니다.

관련 작업

EJB 타이머를 마이그레이션하는 방법

동일한 asadmin 하위 명령

  • list-timers(1)

  • migrate-timers(1)
