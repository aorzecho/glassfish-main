노드를 삭제하는 방법

노드를 삭제하면 DAS(도메인 관리 서버)의 구성에서 노드가 제거됩니다. 노드에 있는 마지막 GlassFish Server 인스턴스가 삭제되면 노드의 디렉토리 및 파일이 삭제됩니다.

시작하기 전에

삭제할 노드에 GlassFish Server 인스턴스가 없는지 확인합니다.

  1. 탐색 트리에서 노드 노드를 선택합니다.

    노드 페이지가 열립니다.

  2. 삭제할 각 노드에 대해 노드 테이블에서 노드 옆에 있는 확인란을 선택합니다.

    주 - 미리 정의된 노드 localhost-domain은 삭제할 수 없습니다.

  3. 삭제를 누릅니다.

    노드를 삭제할 것인지 확인하는 메시지가 표시됩니다.

  4. 노드를 삭제할 것임을 확인합니다.

    노드가 삭제되고 노드 페이지의 노드 테이블에서 제거됩니다.
