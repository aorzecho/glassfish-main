Para ver archivos de registro para una instancia de Servidor GlassFish agrupada en clúster

Antes de empezar

La instancia cuyos archivos de registro desea ver debe estar en ejecución.

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Seleccione la instancia para la cual desea ver los archivos de registro.

    Se abrirá la página Información general para la instancia.

  5. En la página Información general, haga clic en Visualizar archivos de registro.

    Se abre una ventana con el Visualizador de registros.

    Nota - La instancia ya debe estar en ejecución. Si la instancia no está ejecutándose, el botón Visualizar archivos de registro estará desactivado.
