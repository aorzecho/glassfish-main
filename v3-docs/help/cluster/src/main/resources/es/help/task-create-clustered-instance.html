Para agregar una instancia de Servidor GlassFish a un clúster

Las instancias de Servidor GlassFish forman la base de la implementación de aplicaciones. Una instancia agrupada en clúster es un miembro del clúster. Cada instancia pertenece a un solo dominio y tiene su propia estructura de directorios, configuración y aplicaciones implementadas. Cada instancia incluye también el contenedor web de la plataforma Java, Enterprise Edition (Java EE) y el contenedor de la tecnología Enterprise JavaBeans (tecnología EJB). Cada instancia nueva debe contener una referencia a un nodo que defina el host donde residirá la instancia.

Nota - Para crear una instancia en un nodo de tipo CONFIG que representa un host remoto deberá utilizar el subcomando create-local-instance(1). No se puede utilizar el Consola de administración para este fin.

Antes de empezar

Asegúrese de que existan los elementos siguientes:

  • El clúster al que se agregará la instancia

  • El nodo en el que residirá la instancia

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. En la página Instancias de servidor agrupadas, haga clic en Nuevo.

    Se abrirá la página Nueva instancia del servidor agrupada.

  5. En el campo Nombre, escriba un nombre para la instancia.

    El nombre debe cumplir los siguientes requisitos:

    • El nombre sólo puede contener caracteres ASCII.

    • El nombre debe empezar por una letra, un número o un carácter de subrayado.

    • El nombre sólo puede contener los siguientes caracteres:

      • Letras en minúscula

      • Letras en mayúscula

      • Números

      • Guión

      • Punto

      • Subrayado

    • El nombre debe ser exclusivo en el dominio y no debe ser el nombre de otra instancia de Servidor GlassFish, de un clúster, de una configuración con nombre ni de un nodo.

    • El nombre non debe ser domain, server ni otra palabra clave reservada para Servidor GlassFish.

  6. En la lista desplegable Nodo, seleccione el nodo donde residirá la instancia.
  7. Haga clic en Aceptar.
