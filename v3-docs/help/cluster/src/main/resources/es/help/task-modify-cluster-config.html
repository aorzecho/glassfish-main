Para modificar la configuración de un clúster

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. Seleccione el nodo para el clúster.
  3. Haga clic en el nombre de la configuración.

    Se abrirá la página Configuración para el clúster.

  4. En la página Configuración, seleccione el componente de la configuración que va a modificar.

    Se abrirá la página para editar el componente seleccionado de la configuración.

  5. En la página para editar el componente seleccionado, realice los cambios oportunos y guárdelos.

    Para obtener más información, consulte la ayuda en línea para la página para editar el componente seleccionado.
