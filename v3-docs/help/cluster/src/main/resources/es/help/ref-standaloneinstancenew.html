New Standalone Server Instance (Nueva instancia del servidor independiente)

Utilice la página New Standalone Server Instance (Nueva instancia del servidor independiente) para crear una instancia de GlassFish Server independiente.

La página New Standalone Server Instance (Nueva instancia del servidor independiente) contiene las opciones siguientes.

Nombre de instancia

El nombre de la instancia que se va a crear.

El nombre debe cumplir los siguientes requisitos:

  • El nombre debe empezar por una letra, un número o un carácter de subrayado.

  • El nombre sólo puede contener los siguientes caracteres:

    • Letras en minúscula

    • Letras en mayúscula

    • Números

    • Guión

    • Punto

    • Subrayado

  • El nombre debe ser exclusivo en el dominio y no debe ser el nombre de otra instancia de GlassFish Server, de un clúster, de una configuración con nombre ni de un nodo.

  • El nombre non debe ser domain, server ni otra palabra clave reservada para GlassFish Server.

Nodo

Lista desplegable de los nodos existentes que definen los hosts en los que puede residir la instancia. Debe seleccionarse un nodo en la lista. Si la instancia se va a crear en el host en el que se está ejecutando el servidor de administración de dominio (DAS), seleccione el nodo predefinido localhost-dominio.

Configuración

Lista desplegable de las configuraciones con nombre existentes que la instancia puede utilizar. Debe seleccionarse una configuración en la lista. La instancia utilizará la configuración seleccionada cuando se cree.

Realizar una copia de la configuración seleccionada

Si se selecciona esta opción, se copia la configuración seleccionada cuando se crea la instancia. Se asigna el nombre nombre_instancia-config a la copia de la configuración, donde nombre_instancia es el nombre de la instancia del campo Nombre de instancia.

Si se selecciona default-config en la lista desplegable Configuración, debe elegir esta opción. La configuración default-config sólo se puede copiar, no tomar como referencia.

Hacer referencia a la configuración seleccionada

Si esta opción está seleccionada, la instancia usará la configuración con nombre especificada.

Si se selecciona default-config en la lista desplegable Configuración, no debe seleccionar esta opción. La configuración default-config no puede tomarse como referencia, sino sólo copiarse.
