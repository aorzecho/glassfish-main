Para iniciar instancias de GlassFish Server independientes

Antes de empezar

Asegúrese de que se cumplan los siguientes requisitos previos:

  • Todos los nodos en los que residen las instancias están habilitados para la comunicación remota.

  • El usuario del servidor de administración de dominios (DAS) puede utilizar Secure Shell (SSH) para iniciar sesión en el host para cualquier nodo en el que residan las instancias.

Si no se cumplen algunos de estos requisitos previos, inicie cada instancia de forma individual desde la línea de comandos mediante el subcomando start-local-instance(1).

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. Para cada instancia que vaya a iniciar, seleccione la casilla de verificación adyacente a la instancia en la tabla Instancias del servidor.
  3. Haga clic en Iniciar.

    Se le pedirá que confirme que desea iniciar las instancias.

  4. Confirme que desea iniciar las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias del servidor cuando se inicia la instancia.
