Para enumerar los recursos para un clúster

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Recursos.

    Se abrirá la página Recursos para el clúster.

  4. (Opcional) Para mostrar sólo los recursos de un determinado tipo, en la lista desplegable Filtro, seleccione el tipo de recurso que mostrar.
