Propiedades del Sistema de Instancias

La página Propiedades del Sistema de Instancias muestra una lista de las propiedades del sistema Java definidas para la instancia actual de GlassFish Server. Las propiedades del sistema Java se transfieren al iniciador de aplicaciones Java mediante la opción -D del iniciador de aplicaciones Java al iniciar GlassFish Server.

Estas propiedades sustituyen las definiciones de propiedades de configuración de puertos en la configuración de la instancia. La configuración de puertos predefinida se debe sustituir si, por ejemplo, dos instancias en cluster residen en el mismo host. En esta situación, la configuración de puertos para una instancia debe sustituirse porque las dos instancias comparten la misma configuración.

Para obtener información sobre la configuración de puertos predefinida, consulte Propiedades Predefinidas del Sistema de Configuración.

El campo Nombre de Instancia es un campo de sólo lectura que muestra el nombre de la instancia actual.

Para cada propiedad se muestra la información siguiente:

Nombre de Variable de Instancia

Nombre de la propiedad del sistema.

Valor Actual

Valor actualmente definido para la propiedad. Este campo es de sólo lectura.

Valor de Sustitución

Valor en el que se definirá la propiedad después de guardar los cambios.

La tabla Propiedades Adicionales también contiene las opciones siguientes.

Agregar Propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se agrega una fila a la tabla Propiedades Adicionales.

Suprimir Propiedades

Botón para suprimir una o varias propiedades seleccionadas. Cualquier propiedad que se suprima vuelve a su valor por defecto o, si no se ha definido ningún valor por defecto, se queda sin definir.
