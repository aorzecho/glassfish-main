Configurar Opciones de Disponibilidad para un Cluster

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración para el cluster que vaya a cambiar.

  3. En el nodo de configuración, seleccione el nodo Servicio de Disponibilidad.

    Se abre la página Servicio de Disponibilidad.

  4. En el campo Servicio de Disponibilidad, marque la casilla Activada para activar la disponibilidad para el cluster con la configuración que ha seleccionado.

    Para desactivar la disponibilidad, quite la marca de selección de la casilla de activación de Servicio de Disponibilidad.

  5. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para la disponibilidad del cluster.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite su campo Valor.

    • Para suprimir una propiedad, seleccione la casilla de verificación a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  6. Haga clic en Guardar.
