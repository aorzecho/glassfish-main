Recursos

Utilice la página Recursos para realizar las tareas siguientes en la instancia independiente seleccionada de GlassFish Server:

  • Ver y gestionar los recursos existentes de la instancia

  • Crear recursos adicionales para la instancia

El campo Nombre de Instancia es un campo de sólo lectura que muestra el nombre de la instancia actual.

La página Recursos muestra una lista de los recursos existentes de la instancia. Para cada recurso, aparece la siguiente información.

Nombre de Recurso

Nombre que se asignó al recurso en el momento de su creación. Al hacer clic en el nombre se abrirá una de las páginas siguientes para editar el recurso, en función del tipo de recurso:

  • Editar Recurso JDBC

  • Editar Sesión JavaMail

  • Editar Recurso Personalizado

  • Nuevo Recurso Personalizado

  • Editar Recurso Externo

  • Editar Recurso de Conector

  • Editar Recurso de Objeto Administrado

Activado

Indica si se ha activado el recurso:

  • Si el recurso se ha activado, se muestra una marca de control (✓).

  • Si el recurso está desactivado, se muestra una cruz (✗).

Tipo

Tipo de recurso.

La tabla Recursos también contiene las siguientes opciones.

Activar

Botón para activar uno o varios recursos seleccionados.

Desactivar

Botón para desactivar uno o varios recursos seleccionados.

Nuevo

Lista desplegable que crea un recurso del tipo seleccionado. Al seleccionar un tipo de recurso en la lista se abrirá una de las páginas siguientes para crear un recurso del tipo seleccionado:

  • Nuevo Recurso JDBC

  • Nueva Fábrica de Conexiones JMS

  • Nuevo Recurso de Destino JMS

  • Nueva Sesión de JavaMail

  • Nuevo Recurso Personalizado

  • Nuevo Recurso Externo

  • Nuevo Recurso de Conector

  • Nuevo Recurso de Objeto Administrado

Filtro

Lista desplegable que filtra la lista de recursos mostrados por tipo.
