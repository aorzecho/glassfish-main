Aplicaciones

Utilice la página Aplicaciones para llevar a cabo las tareas siguientes en el clúster actual:

  • Ver y administrar las aplicaciones que ya estén implementadas en el clúster

  • Implementar más aplicaciones en el clúster

El campo Nombre de clúster es un campo de sólo lectura que muestra el nombre del clúster actual.

La página Aplicaciones muestra una lista de las aplicaciones que se han implementado en el clúster actual. En cada aplicación aparece la siguiente información.

Nombre

El nombre de la aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte Module and Application Versions de Oracle GlassFish Server 3.1 Application Deployment Guide.

Habilitado

Indica si se ha habilitado la aplicación:

  • Si se ha habilitado la aplicación, se muestra una marca de verificación (✓).

  • Si la aplicación está deshabilitada, se muestra una cruz (✗).

Equilibrador de carga activado

Indica si se ha habilitado la aplicación para el equilibrio de carga:

  • Si se ha habilitado la aplicación para el equilibrio de carga, se muestra una marca de verificación (✓).

  • Si el equilibrio de carga de la aplicación está deshabilitado, se muestra una cruz (✗).

Motores

Tipos de contenedores que emplea la aplicación. Los tipos de contenedores pueden ser:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld (el contenedor para los contextos y la inyección de dependencias para las aplicaciones de la plataforma Java EE)

La tabla Aplicaciones implementadas también contiene las siguientes opciones.

Implementar

Botón para implementar una aplicación.

Anular implementación

Botón para anular la implementación de una o más aplicaciones seleccionadas.

Al anular la implementación de una aplicación se elimina la aplicación de todos los destinos y del dominio. Cuando se anula la implementación de una aplicación, la aplicación ya no aparece en la página Aplicaciones que enumera todas las aplicaciones implementadas en Servidor GlassFish.

Para eliminar aplicaciones solo del destino actual, utilice el botón Eliminar en esta página.

Borrar

Botón para eliminar una o más aplicaciones seleccionadas.

Al eliminar una aplicación solo se elimina del destino actual. Cuando una aplicación se elimina de un destino, la aplicación sigue apareciendo en la página Aplicaciones que enumera todas las aplicaciones implementadas en Servidor GlassFish. La aplicación también seguirá implementada en otros destinos en los que se hubiera implementado.

Para eliminar aplicaciones de todos los destinos y del dominio, utilice el botón Anular la implementación en esta página.

Más acciones

Lista desplegable que realiza acciones adicionales en una o más aplicaciones seleccionadas.

Habilitar

Habilita una o más aplicaciones seleccionadas.

Deshabilitar

Desactiva una o más aplicaciones seleccionadas.

Habilitar Equilibrador de carga

Habilita una o más aplicaciones seleccionadas para el equilibrio de carga.

Deshabilitar Equilibrador de carga

Deshabilita una o más aplicaciones seleccionadas para el equilibrio de carga.
