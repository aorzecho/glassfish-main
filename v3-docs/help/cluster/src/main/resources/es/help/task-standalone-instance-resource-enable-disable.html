Activar o Desactivar un Recurso que se Despliega en una Instancia de GlassFish Server Independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abre la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Recursos.

    Se abre la página Recursos para la instancia.

  4. Seleccione la casilla de verificación que aparece junto al recurso.

  5. Haga clic en el botón correspondiente.

    • Para activar el recurso, elija Activar.

    • Para desactivar el recurso, seleccione Desactivar.

      Al desactivar un recurso no se elimina.
