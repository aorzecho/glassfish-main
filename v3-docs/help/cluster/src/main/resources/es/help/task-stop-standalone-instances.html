Para detener instancias de Servidor GlassFish independientes

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. Para cada instancia que vaya a detener, seleccione la casilla de verificación adyacente a la instancia en la tabla Instancias del servidor.
  3. Haga clic en Detener.

    Se le pedirá que confirme que desea detener las instancias.

  4. Confirme que desea detener las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias del servidor cuando se detiene la instancia.
