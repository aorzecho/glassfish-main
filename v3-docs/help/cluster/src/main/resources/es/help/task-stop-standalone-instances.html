Parar Instancias de GlassFish Server Independientes

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. Para cada instancia que vaya a parar, seleccione la casilla de verificación que aparece junto a la instancia en la tabla Instancias de Servidor.

  3. Haga clic en Parar.

    Se le pedirá que confirme que desea parar las instancias.

  4. Confirme que desea parar las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias de Servidor cuando se para la instancia.
