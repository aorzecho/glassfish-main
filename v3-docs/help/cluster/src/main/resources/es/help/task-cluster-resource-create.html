Crear un Recurso para un Cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Recursos.

    Se abre la página Recursos para el cluster.

  4. En el menú desplegable Nuevo, seleccione el tipo de recurso que vaya a crear.

    Se abre la página para crear un recurso del tipo seleccionado.

  5. En la página para crear un recurso del tipo seleccionado, realice los cambios oportunos y guárdelos.

    Para obtener más información, consulte la ayuda en pantalla de la página para crear un recurso del tipo seleccionado.

    Nota:

    Cuando cree el recurso, asegúrese de especificar el cluster como destino.
