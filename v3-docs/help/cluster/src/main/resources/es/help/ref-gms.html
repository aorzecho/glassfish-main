Editar servicio de administración de grupos:

Utilice la página Servicios de administración de grupos para cambiar la configuración de los GMS.

Los Servicios de administración de grupos (GMS) son servicios en curso que proporcionan supervisión del clúster, pertenencia al clúster y servicios de comunicación de grupos. Los GMS se basan en una pila configurable de protocolos de descubrimiento de pertenencia a grupos y de supervisión de estado. Los GMS realizan las funciones siguientes:

  • Notifican los módulos registrados en una instancia de GlassFish Server cuando se añaden o eliminan una o varias instancias en un clúster, o cuando se sospecha o se confirma que han fallado.

  • Permiten enviar y recibir mensajes entre un grupo de procesos.

La página Servicios de administración de grupos contiene las opciones siguientes.

Cargar valores predeterminados

Botón para restablecer los valores predeterminados de todos los parámetros de configuración.

Nombre de configuración

El nombre de la configuración a la que se aplican los parámetros de esta página. Este campo es de sólo lectura.

Maximum Missed Heartbeats (Máximo de latidos descartados)

Número de intentos de supervisión después de los cuales se sospecha que se producirá un error en la instancia de servidor. El valor predeterminado es 3.

Heartbeat Frequency (Frecuencia de latido)

Tiempo que transcurre entre los intentos de supervisión. El valor predeterminado es 2.000 milisegundos.

Group Discovery Timeout (Tiempo de espera de detección de grupos)

Tiempo que esperan los GMS para detectar otros miembros del clúster. El valor predeterminado es 5.000 milisegundos.

Failure Verification Wait Time (Tiempo de espera para la verificación de errores)

Tiempo tras el cual un presunto error se considera comprobado. El valor predeterminado es 1.500 milisegundos.

Propiedades adicionales

Una tabla de propiedades adicionales para el Servicio de administración de grupos.

GlassFish Server no define propiedades adicionales para el Servicio de administración de grupos.

Para cada propiedad se muestra la información siguiente:

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad que se define para la instancia seleccionada.

Descripción

Descripción textual que ofrece más información sobre la propiedad.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se añade una fila a la tabla Propiedades adicionales.

Eliminar propiedades

Botón para eliminar una o varias propiedades seleccionadas. Cualquier propiedad que se elimine vuelve a su valor predeterminado o, si no se ha definido ningún valor predeterminado, se queda sin definir.
