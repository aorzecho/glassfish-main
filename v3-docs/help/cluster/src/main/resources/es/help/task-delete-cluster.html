Para eliminar un clúster

Al eliminar un clúster se eliminan el clúster y todas las instancias de GlassFish Server que contenga. No es necesario eliminar las instancias que contiene el clúster antes de eliminarlo.

Si en la configuración se indica que el clúster no es compartido, la configuración también se eliminará.

Antes de empezar

Asegúrese de que el clúster que vaya a eliminar está detenido.

  1. En el árbol de navegación, seleccione el nodo Clústeres.

    Se abrirá la página Clústeres.

  2. En la página Clústeres, seleccione la casilla de verificación adyacente al nombre del clúster.
  3. Haga clic en Eliminar.

    Se le pedirá que confirme que desea eliminar el clúster y todas las instancias que contenga.

  4. Confirme que desea eliminar el clúster y todas las instancias que contenga.

    Se eliminará el clúster y se quitará de la tabla de clústeres de la página Clústeres.
