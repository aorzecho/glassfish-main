Información general

La página Información general contiene las opciones siguientes.

Inicio

Botón para iniciar la instancia. Si la instancia se está ejecutando, este botón está desactivado.

Detener

Botón para detener la instancia. Si la instancia se detiene, este botón se desactiva.

Reiniciar

Botón para reiniciar la instancia. Si la instancia se detiene, este botón se desactiva.

Visualizar archivos de registro

Botón para ver archivos de registro para la instancia.

Recuperar transacciones

Botón para recuperar las transacciones para la instancia.

Nombre de clúster

El nombre que se asignó al clúster en el momento de crearlo. Al hacer clic en el nombre se abre la página Información general del clúster.

Nombre de instancia

Nombre asignado a la instancia cuando se agregó al clúster. Este campo es de sólo lectura.

Estado

Una indicación de si la instancia se está ejecutando, si es necesario reiniciar para que se actualice con cambios de configuración, o si no se está ejecutando. Este campo es de sólo lectura.

JVM

Si hace clic sobre el Informe de JVM, se abrirá una ventana independiente que muestra informes en la máquina virtual de Java, incluyendo un informe resumido, un informe sobre administración de la memoria y recolección de elementos no utilizados, otro informe sobre carga de clases y un volcado de subprocesos actuales.

Nodo

Nombre del nodo en el que reside la instancia. Al hacer clic en el nombre se abre la página Editar nodo del nodo.

Configuración

la configuración a la que hace referencia la instancia. Al hacer clic en la configuración se abre la página Configuraciones para la instancia.

Puertos HTTP

Lista separada por comas de los números de puerto en los que la instancia escucha los siguientes tipos de solicitudes:

  • Solicitudes de administración

  • Solicitudes HTTP

  • Solicitudes HTTPS

Este campo es de sólo lectura.

Puertos IIOP

Lista separada por comas de los números de puerto que utiliza la instancia para los siguientes tipos de conexiones:

  • Conexión IIOP segura

  • Conexiones IIOP seguras con autenticación de cliente

  • Conexiones IIOP

Este campo es de sólo lectura.
