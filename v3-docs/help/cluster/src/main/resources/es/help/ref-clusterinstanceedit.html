Información General

La página Información General contiene las opciones siguientes.

Iniciar

Botón para iniciar la instancia. Si la instancia se está ejecutando, este botón está desactivado.

Parar

Botón para parar la instancia. Si la instancia está parada, este botón está desactivado.

Reiniciar

Botón para reiniciar la instancia. Si la instancia está parada, este botón está desactivado.

Ver Archivos Log

Botón para ver archivos log para la instancia.

Recuperar Transacciones

Botón para recuperar las transacciones para la instancia.

Nombre de Cluster

Nombre que se asignó al cluster en el momento de su creación. Al hacer clic en el nombre se abrirá la página Información General del clúster.

Nombre de Instancia

Nombre que se asignó a la instancia cuando se agregó al cluster. Este campo es de sólo lectura.

Estado

Indicación de si la instancia se está ejecutando, si es necesario reiniciar para que se actualice con cambios de configuración o si no se está ejecutando. Este campo es de sólo lectura.

JVM

Si hace clic en Informe de JVM, se abrirá una ventana independiente que muestra informes sobre Java Virtual Machine, incluyendo un informe resumido, un informe sobre gestión de la memoria y recolección de basura, otro informe sobre carga de clase y un volcado de thread actual.

Nodo

Nombre del nodo en el que reside la instancia. Al hacer clic en el nombre se abrirá la página Editar Nodo del nodo.

Configuración

Configuración a la que hace referencia la instancia. Al hacer clic en la configuración se abrirá la página Configuraciones para la instancia.

Puertos HTTP

Lista separada por comas de los números de puerto en los que la instancia recibe los siguientes tipos de solicitudes:

  • Solicitudes de administración

  • Solicitudes HTTP

  • Solicitudes HTTPS

Este campo es de sólo lectura.

Puertos IIOP

Lista separada por comas de los números de puerto que utiliza la instancia para los siguientes tipos de conexiones:

  • Conexión IIOP segura

  • Conexiones IIOP seguras con autenticación de cliente

  • Conexiones IIOP

Este campo es de sólo lectura.
