Instance System Properties (Propiedades de sistema de la instancia)

La página Propiedades de sistema de la instancia muestra una lista de las propiedades de sistema Java configuradas para la instancia actual de Servidor GlassFish. Las propiedades del sistema Java se transfieren al lanzador de aplicaciones Java mediante la opción -D del lanzador de aplicaciones Java al iniciar Servidor GlassFish.

Estas propiedades anulan las definiciones de propiedades de configuración de puertos en la configuración de la instancia. La configuración de puertos predefinida se debe anular si, por ejemplo, dos instancias agrupadas en clúster residen en el mismo equipo. En esta situación, la configuración de puertos para una instancia debe anularse porque las dos instancias comparten la misma configuración.

Para obtener información sobre la configuración de puertos predefinida, consulte Propiedades predefinidas del sistema de configuración.

El campo Nombre de instancia es un campo de sólo lectura que muestra el nombre de la instancia actual.

Para cada propiedad se muestra la información siguiente:

Nombre de variable de instancia

Nombre de la propiedad del sistema.

Valor Actual

El valor actualmente asignado a la propiedad. Este campo es de sólo lectura.

Reemplazar valor

El valor que se asignará a la propiedad después de guardar los cambios.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se añade una fila a la tabla Propiedades adicionales.

Eliminar propiedades

Botón para eliminar una o varias propiedades seleccionadas. Cualquier propiedad que se elimine vuelve a su valor predeterminado o, si no se ha definido ningún valor predeterminado, se queda sin definir.
