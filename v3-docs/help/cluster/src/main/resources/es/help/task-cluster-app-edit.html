Editar una Aplicación que se Despliega en un Cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para el cluster.

  4. En la tabla Aplicaciones Desplegadas de la página Aplicaciones, haga clic en el nombre de la aplicación.

    Se abre la página Editar Aplicación.

  5. En la página Editar Aplicación, indique la información necesaria sobre la aplicación y guarde los cambios.

    Para obtener más información, consulte la ayuda en pantalla de Editar Aplicación.
