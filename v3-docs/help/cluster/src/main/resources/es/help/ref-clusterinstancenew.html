Nueva instancia del servidor agrupada

Utilice la página Nueva instancia del servidor agrupada para agregar una instancia de Servidor GlassFish a un clúster.

La página Nueva instancia del servidor agrupada contiene las opciones siguientes.

Nombre de clúster

El nombre que se asignó al clúster en el momento de crearlo. Este campo es de sólo lectura.

Nombre de instancia

El nombre de la instancia que se va a crear.

El nombre debe cumplir los siguientes requisitos:

  • El nombre sólo puede contener caracteres ASCII.

  • El nombre debe empezar por una letra, un número o un carácter de subrayado.

  • El nombre sólo puede contener los siguientes caracteres:

    • Letras en minúscula

    • Letras en mayúscula

    • Números

    • Guión

    • Punto

    • Subrayado

  • El nombre debe ser exclusivo en el dominio y no debe ser el nombre de otra instancia de Servidor GlassFish, de un clúster, de una configuración con nombre ni de un nodo.

  • El nombre non debe ser domain, server ni otra palabra clave reservada para Servidor GlassFish.

Nodo

Lista desplegable de los nodos existentes que definen los hosts en los que puede residir la instancia. Debe seleccionarse un nodo en la lista. Si la instancia se va a crear en el host en el que se está ejecutando el servidor de administración de dominio (DAS), seleccione el nodo predefinido localhost-dominio.
