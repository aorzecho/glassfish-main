Nueva Instancia del Servidor en Cluster

Utilice la página Nueva Instancia del Servidor en Cluster para agregar una instancia de GlassFish Server a un cluster existente.

La página Nueva Instancia del Servidor en Cluster contiene las opciones siguientes.

Nombre de Cluster

Nombre que se asignó al cluster en el momento de su creación. Este campo es de sólo lectura.

Nombre de Instancia

Nombre de la instancia que se va a crear.

El nombre debe cumplir los siguientes requisitos:

  • El nombre sólo puede contener caracteres ASCII.

  • El nombre debe empezar por una letra, un número o un carácter de subrayado.

  • El nombre sólo puede contener los siguientes caracteres:

    • Letras en minúscula

    • Letras en mayúscula

    • Números

    • Guión

    • Punto

    • Carácter de subrayado

  • El nombre debe ser exclusivo en el dominio y no debe ser el nombre de otra instancia de GlassFish Server, un cluster, una configuración con nombre ni un nodo.

  • El nombre no debe ser domain, server ni otra palabra clave reservada por GlassFish Server.

Nodo

Lista desplegable de los nodos existentes que definen los hosts en los que puede residir la instancia. Debe seleccionarse un nodo en la lista. Si la instancia se va a crear en el host en el que se está ejecutando el servidor de administración de dominio (DAS), seleccione el nodo predefinido localhost-domain.
