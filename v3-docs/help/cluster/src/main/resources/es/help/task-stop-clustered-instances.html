Para detener instancias de Servidor GlassFish agrupadas en clúster

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Para cada instancia que vaya a detener, seleccione la casilla de verificación adyacente a la instancia en la tabla Instancias del servidor.
  5. Haga clic en Detener.

    Se le pedirá que confirme que desea detener las instancias.

  6. Confirme que desea detener las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias del servidor cuando se detiene la instancia.
