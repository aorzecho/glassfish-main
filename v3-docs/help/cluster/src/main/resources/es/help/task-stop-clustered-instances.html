Parar las Instancias de GlassFish Server en Cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abre la página Instancias de Servidor en Cluster para el cluster.

  4. Para cada instancia que vaya a parar, seleccione la casilla de verificación que aparece junto a la instancia en la tabla Instancias de Servidor.

  5. Haga clic en Parar.

    Se le pedirá que confirme que desea parar las instancias.

  6. Confirme que desea parar las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias de Servidor cuando se para la instancia.
