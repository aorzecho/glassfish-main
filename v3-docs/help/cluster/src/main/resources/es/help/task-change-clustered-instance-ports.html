Modificar los Números de Puerto de una Instancia de GlassFish Server en Cluster

Los números de puerto de una instancia de GlassFish Server en cluster se establecen inicialmente en la configuración que hereda la instancia de su cluster principal. Debe modificar estos números de puerto si, por ejemplo, dos instancias en cluster residen en el mismo host. En este caso, es necesario modificar los números de puerto de una instancia porque ambas instancias comparten la misma configuración.

Los números de puerto de una instancia se guardan como propiedades del sistema Java. Estas propiedades se transfieren al iniciador de aplicaciones Java mediante la opción -D del iniciador de aplicaciones Java cuando se inicia GlassFish Server.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abre la página Instancias de Servidor en Cluster para el cluster.

  4. Seleccione la instancia para la cual se modificarán los números de puerto.

    Se abre la página Información General para la instancia.

  5. En la página Información General, haga clic en el separador Propiedades.

    Se abre la página Propiedades del Sistema de Instancias.

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la instancia:

    • Números de puerto predefinidos en la configuración de la instancia

    • Propiedades personalizadas que se definen para la instancia, si las hay

  6. Para cada propiedad que vaya a modificar, escriba el nuevo valor en el campo Valor de Sustitución para la propiedad.

  7. Establezca los valores que necesite en las demás propiedades.

    1. En la tabla Propiedades Adicionales, haga clic en Agregar Propiedad.

      Se agrega una fila a la tabla Propiedades Adicionales.

    2. En el campo Nombre de Variable de Instancia, escriba el nombre de la propiedad.

    3. En el campo Valor de Sustitución, escriba el valor.

  8. Haga clic en Guardar.
