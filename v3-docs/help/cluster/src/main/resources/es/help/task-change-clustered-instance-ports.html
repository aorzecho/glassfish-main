Para modificar los números de puerto de una instancia de Servidor GlassFish agrupada en clúster

Los números de puerto de una instancia de Servidor GlassFish agrupada en clúster se establecen inicialmente en la configuración que hereda la instancia de su clúster principal. Debe modificar estos números de puerto si, por ejemplo, hay dos instancias en clúster en el mismo host. En este caso, es necesario modificar los números de puerto de una instancia porque ambos comparten la misma configuración.

Los números de puerto de una instancia se guardan como propiedades del sistema Java. Estas propiedades se transfieren al lanzador de aplicaciones Java mediante la opción -D del lanzador de aplicaciones Java cuando se inicia Servidor GlassFish.

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Seleccione la instancia para la cual se modificarán los números de puerto.

    Se abrirá la página Información general para la instancia.

  5. En la página Información general, haga clic en la ficha Propiedades.

    Se abrirá la página Instance System Properties (Propiedades de sistema de la instancia).

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la instancia:

    • Números de puerto predefinidos en la configuración de la instancia

    • Propiedades personalizadas que se definen para la instancia, si las hay

  6. Para cada propiedad que vaya a modificar, escriba el nuevo valor en el campo Reemplazar valor para la propiedad.
  7. Establezca los valores que necesite en las demás propiedades.
    1. En la tabla Propiedades adicionales, haga clic en Agregar propiedad.

      Se añade una fila a la tabla Propiedades adicionales.

    2. En el campo Nombre de variable de instancia, escriba el nombre de la propiedad.
    3. En el campo Reemplazar valor, escriba el valor.
  8. Haga clic en Guardar.
