Para ver archivos de registro para una instancia de Servidor GlassFish independiente

Antes de empezar

La instancia cuyos archivos de registro desea ver debe estar en ejecución.

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. Seleccione la instancia para la cual desea ver los archivos de registro.

    Se abrirá la página Información general para la instancia.

  3. En la página Información general, haga clic en Visualizar archivos de registro.

    Se abre una ventana con el Visualizador de registros.

    Nota - La instancia ya debe estar en ejecución. Si la instancia no está ejecutándose, el botón Visualizar archivos de registro estará desactivado.
