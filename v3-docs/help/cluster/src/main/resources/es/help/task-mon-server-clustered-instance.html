Para ver datos de supervisión de servidor virtual para una instancia de Servidor GlassFish agrupada en clúster

Antes de empezar

Para poder ver datos de supervisión, debe configurar el servicio de supervisión.

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Seleccione la instancia para la cual desea ver los datos de supervisión de los servidores virtuales.

    Se abrirá la página Información general para la instancia.

  5. En la página Información general, haga clic en la ficha Supervisar.

    Se abrirá la página Supervisión de aplicación.

  6. Haga clic en la ficha Servidor.

    Se abrirá la página Supervisión del servidor.

  7. En la lista desplegable Ver, seleccione el componente para el cual desea ver los datos de supervisión.
  8. Para actualizar los datos de supervisión mostrados, haga clic en Actualizar.
