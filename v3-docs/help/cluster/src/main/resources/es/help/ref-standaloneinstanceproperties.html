Propiedades de la instancia

La página Propiedades de instancia muestra una lista de las propiedades que se establecen para la instancia actual de GlassFish Server. Estas propiedades añaden información de configuración opcional sobre la instancia.

El campo Nombre de instancia es un campo de sólo lectura que muestra el nombre de la instancia seleccionada.

Para cada propiedad se muestra la información siguiente:

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad que se define para la instancia seleccionada.

Descripción

Descripción textual que ofrece más información sobre la propiedad.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se añade una fila a la tabla Propiedades adicionales.

Eliminar propiedades

Botón para eliminar una o varias propiedades seleccionadas. Cualquier propiedad que se elimine vuelve a su valor predeterminado o, si no se ha definido ningún valor predeterminado, se queda sin definir.

GlassFish Server define las siguientes propiedades de la instancia:

rendezvousOccurred

Especifica si la instancia ha contactado con el servidor de administración de dominios (DAS). Los valores posibles son los siguientes:

true

La instancia ha contactado con el DAS.

false

La instancia no ha contactado con el DAS.
