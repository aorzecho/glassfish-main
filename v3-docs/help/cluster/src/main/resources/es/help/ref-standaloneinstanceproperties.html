Propiedades de la Instancia

La página Propiedades de la Instancia muestra una lista de las propiedades que se han definido para la instancia actual de GlassFish Server. Estas propiedades agregan información de configuración opcional sobre la instancia.

El campo Nombre de Instancia es un campo de sólo lectura que muestra el nombre de la instancia actual.

Para cada propiedad se muestra la información siguiente:

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad que se ha definido para la instancia seleccionada.

Descripción

Descripción textual que ofrece más información sobre la propiedad.

La tabla Propiedades Adicionales también contiene las opciones siguientes.

Agregar Propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se agrega una fila a la tabla Propiedades Adicionales.

Suprimir Propiedades

Botón para suprimir una o varias propiedades seleccionadas. Cualquier propiedad que se suprima vuelve a su valor por defecto o, si no se ha definido ningún valor por defecto, se queda sin definir.

GlassFish Server define las siguientes propiedades de la instancia:

rendezvousOccurred

Especifica si la instancia ha contactado con el servidor de administración de dominio (DAS). Los valores posibles son los siguientes:

true

La instancia ha contactado con DAS.

false

La instancia no ha contactado con DAS.
