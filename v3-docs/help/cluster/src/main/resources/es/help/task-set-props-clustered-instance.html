Para definir las propiedades de sólo instancia para una instancia de Servidor GlassFish agrupada en clúster

Las propiedades de una instancia de Servidor GlassFish incluyen información de configuración opcional acerca de la instancia.

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Seleccione la instancia para la cual se modificarán las propiedades exclusivas de la instancia.

    Se abrirá la página Información general para la instancia.

  5. En la página Información general, haga clic en la ficha Propiedades.
  6. Si es necesario, haga clic en la ficha Propiedades de instancia.

    Se abrirá la página Propiedades de la instancia.

    Esta página muestra una lista de las propiedades que se establecen para la instancia seleccionada de Servidor GlassFish.

  7. Para cada propiedad que esté modificando, escriba el nuevo valor en el campo Valor de la propiedad.
  8. Establezca los valores que necesite en las demás propiedades.
    1. En la tabla Propiedades adicionales, haga clic en Agregar propiedad.

      Se añade una fila a la tabla Propiedades adicionales.

    2. En el campo Nombre, escriba un nombre para la propiedad.
    3. En el campo Valor, escriba el valor.
    4. En el campo Descripción, escriba una breve descripción de la propiedad.
  9. Haga clic en Guardar.
