Anular el Despliegue de Aplicaciones desde un Cluster

Al anular el despliegue de una aplicación, se elimina la aplicación de todos los objetivos y del dominio. Cuando se anula el despliegue de una aplicación, ésta deja de aparecer en la página Aplicaciones que enumera todas las aplicaciones que se despliegan en GlassFish Server.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para el cluster.

  4. Para cada aplicación cuyo despliegue vaya a anular, seleccione la casilla de verificación que aparece junto a ella en la tabla Aplicaciones Desplegadas.

  5. Haga clic en Anular Despliegue.

    Se le pedirá que confirme que desea anular el despliegue de las aplicaciones.

  6. Confirme que desea anular el despliegue de las aplicaciones.

    Se anula el despliegue de las aplicaciones y éstas se eliminan de la tabla de aplicaciones desplegadas de la página.
