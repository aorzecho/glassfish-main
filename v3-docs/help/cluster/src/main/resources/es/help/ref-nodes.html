Nodos

La página Nodos muestra una lista de los nodos del dominio. Para cada nodo, se muestra la información siguiente:

Nombre

Nombre que se asignó al nodo durante su creación. Al hacer clic en el nombre se abre la página Editar nodo del nodo.

Host de nodo

El nombre del host que el nodo representa.

Tipo

El tipo de nodo, que es uno de los siguientes:

CONFIG

El nodo no admite comunicación remota.

SSH

El modo admite comunicación sobre Secure Shell (SSH).

Instancias

Lista de las instancias que residen en el nodo con una indicación de si se están ejecutando:

  • Si la instancia se está ejecutando, se muestra una marca de verificación (✓).

  • Si la instancia se detiene, se muestra un signo de exclamación (!).

Al hacer clic en el nombre de una instancia se abre la página Información general de la instancia.

La tabla Nodos también contiene las opciones siguientes.

Nuevo

Botón para crear un nodo.

Borrar

Botón para eliminar uno o varios nodos seleccionados.
