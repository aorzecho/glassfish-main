Para modificar los números de puerto de una instancia de GlassFish Server independiente

Los números de puerto de una instancia de GlassFish Server independiente se establecen inicialmente en la configuración a la que hace referencia la instancia. Debe modificar estos números de puerto si, por ejemplo, dos instancias que hacen referencia a la misma configuración residen en el mismo host. En este caso, los números de puerto de una instancia deben modificarse porque ambas instancias hacen referencia a la misma configuración.

Los números de puerto de una instancia se guardan como propiedades del sistema Java. Estas propiedades se transfieren al lanzador de aplicaciones Java mediante la opción -D del lanzador de aplicaciones Java cuando se inicia GlassFish Server.

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. Seleccione la instancia para la cual se modificarán los números de puerto.

    Se abrirá la página Información general para la instancia.

  3. En la página Información general, haga clic en la ficha Propiedades.

    Se abrirá la página Instance System Properties (Propiedades de sistema de la instancia).

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la instancia:

    • Números de puerto predefinidos en la configuración de la instancia

    • Propiedades personalizadas que se definen para la instancia, si las hay

  4. Para cada propiedad que vaya a modificar, escriba el nuevo valor en el campo Reemplazar valor para la propiedad.
  5. Establezca los valores que necesite en las demás propiedades.
    1. En la tabla Propiedades adicionales, haga clic en Agregar propiedad.

      Se añade una fila a la tabla Propiedades adicionales.

    2. En el campo Nombre de variable de instancia, escriba el nombre de la propiedad.
    3. En el campo Reemplazar valor, escriba el valor.
  6. Haga clic en Guardar.
