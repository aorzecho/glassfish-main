Parar un Cluster

Al parar un cluster se paran todas las instancias de GlassFish Server que estén en ejecución en dicho cluster.

  1. En el árbol de navegación, seleccione el nodo Clusters.

    Se abre la página Clusters.

  2. En la página Clusters, seleccione la casilla de verificación que aparece junto al nombre del cluster.

  3. Haga clic en Parar Cluster.

    Se le pedirá que confirme que desea parar el cluster.

  4. Confirme que desea parar el cluster.

    Se para el cluster. El estado de cada instancia del cluster se actualiza en la página Clusters cuando se para la instancia.
