Para detener un clúster

Al detener un clúster se detienen todas las instancias de GlassFish Server que estén en ejecución en dicho clúster.

  1. En el árbol de navegación, seleccione el nodo Clústeres.

    Se abrirá la página Clústeres.

  2. En la página Clústeres, seleccione la casilla de verificación adyacente al nombre del clúster.
  3. Haga clic en Detener clúster.

    Se le pedirá que confirme que desea detener el clúster.

  4. Confirme que quiere detener el clúster.

    Se detendrá el clúster. El estado de cada instancia del clúster se actualiza en la página Clústeres cuando se detiene la instancia.
