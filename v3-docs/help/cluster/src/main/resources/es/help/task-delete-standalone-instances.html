Suprimir Instancias de GlassFish Server Independientes

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. Para cada instancia que vaya a suprimir, seleccione la casilla de verificación que aparece junto a la instancia en la tabla Instancias de Servidor.

  3. Haga clic en Suprimir.

    Se le pedirá que confirme que desea suprimir las instancias.

  4. Confirme que desea suprimir las instancias.

    Las instancias se suprimen y se eliminan de la tabla de instancias de la página.
