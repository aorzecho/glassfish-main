Establecer Propiedades sólo de Instancia para una Instancia de GlassFish Server Independiente

Las propiedades de una instancia de GlassFish Server incluyen información de configuración opcional acerca de la instancia.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. Seleccione la instancia para la cual se modificarán las propiedades exclusivas de la instancia.

    Se abre la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Propiedades.

  4. Si es necesario, haga clic en el separador Propiedades de la Instancia.

    Se abre la página Propiedades de la Instancia.

    Esta página muestra una lista de las propiedades que se establecen para la instancia seleccionada de GlassFish Server.

  5. Para cada propiedad que esté modificando, escriba el nuevo valor en el campo Valor de la propiedad.

  6. Establezca los valores que necesite en las demás propiedades.

    1. En la tabla Propiedades Adicionales, haga clic en Agregar Propiedad.

      Se agrega una fila a la tabla Propiedades Adicionales.

    2. En el campo Nombre, escriba un nombre para la propiedad.

    3. En el campo Valor, escriba el valor.

    4. En el campo Descripción, escriba una breve descripción de la propiedad.

  7. Haga clic en Guardar.
