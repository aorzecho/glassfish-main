Para establecer las propiedades de sólo instancia de una instancia de Servidor GlassFish independiente

Las propiedades de una instancia de Servidor GlassFish incluyen información de configuración opcional acerca de la instancia.

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. Seleccione la instancia para la cual se modificarán las propiedades exclusivas de la instancia.

    Se abrirá la página Información general para la instancia.

  3. En la página Información general, haga clic en la ficha Propiedades.
  4. Si es necesario, haga clic en la ficha Propiedades de instancia.

    Se abrirá la página Propiedades de la instancia.

    Esta página muestra una lista de las propiedades que se establecen para la instancia seleccionada de Servidor GlassFish.

  5. Para cada propiedad que esté modificando, escriba el nuevo valor en el campo Valor de la propiedad.
  6. Establezca los valores que necesite en las demás propiedades.
    1. En la tabla Propiedades adicionales, haga clic en Agregar propiedad.

      Se añade una fila a la tabla Propiedades adicionales.

    2. En el campo Nombre, escriba un nombre para la propiedad.
    3. En el campo Valor, escriba el valor.
    4. En el campo Descripción, escriba una breve descripción de la propiedad.
  7. Haga clic en Guardar.
