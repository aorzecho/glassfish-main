Para suprimir instancias de Servidor GlassFish agrupadas en clúster

Precaución - Si utiliza un clúster Java Message Service (JMS) con un agente maestro, no suprima la instancia asociada al agente maestro. Si esta instancia se debe suprimir, utilice el subcomando change-master-broker(1) para asignar el agente maestro a una instancia diferente.

  1. En el árbol de navegación, expanda el nodo Clústeres.
  2. En el nodo Clústeres expandido, seleccione el nodo para el clúster.

    Se abrirá la página Información general para el clúster.

  3. En la página Información general, haga clic en la ficha Instancias.

    Se abrirá la página Instancias de servidor agrupadas para el clúster.

  4. Para cada instancia que vaya a eliminar, seleccione la casilla de verificación adyacente a la instancia en la tabla Instancias del servidor.
  5. Haga clic en Eliminar.

    Se le pedirá que confirme que desea eliminar las instancias.

  6. Confirme que desea eliminar las instancias.

    Las instancias se suprimen y se eliminan de la tabla de instancias de la página.
