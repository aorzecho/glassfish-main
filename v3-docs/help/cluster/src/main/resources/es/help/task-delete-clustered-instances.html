Suprimir Instancias de GlassFish Server en Cluster

Atención:

Si utiliza un cluster Java Message Service (JMS) con un agente maestro, no suprima la instancia asociada al agente maestro. Si esta instancia se debe suprimir, utilice el subcomando change-master-broker(1) para asignar el broker maestro a una instancia diferente.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abre la página Instancias de Servidor en Cluster para el cluster.

  4. Para cada instancia que vaya a suprimir, seleccione la casilla de verificación que aparece junto a la instancia en la tabla Instancias de Servidor.

  5. Haga clic en Suprimir.

    Se le pedirá que confirme que desea suprimir las instancias.

  6. Confirme que desea suprimir las instancias.

    Las instancias se suprimen y se eliminan de la tabla de instancias de la página.
