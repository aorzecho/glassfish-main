Aplicaciones

Utilice la página Aplicaciones para llevar a cabo las tareas siguientes en la instancia independiente seleccionada de GlassFish Server:

  • Ver y gestionar las aplicaciones que ya están desplegadas en la instancia

  • Desplegar más aplicaciones en la instancia

El campo Nombre de Instancia es un campo de sólo lectura que muestra el nombre de la instancia actual.

La página Aplicaciones muestra una lista de las aplicaciones que se han desplegado en la instancia seleccionada. Para cada aplicación aparece la siguiente información.

Nombre

Nombre de la aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

Activado

Indica si se ha activado la aplicación:

  • Si se ha activado la aplicación, se muestra una marca de control (✓).

  • Si la aplicación está desactivada, se muestra una cruz (✗).

Equilibrador de Carga Activado

Indica si se ha activado la aplicación para el equilibrio de carga:

  • Si se ha activado la aplicación para el equilibrio de carga, se muestra una marca de control (✓).

  • Si se ha desactivado la aplicación para el equilibrio de carga, se muestra una cruz (✗).

Motores

Tipos de contenedores que emplea la aplicación. Los tipos de contenedores pueden ser:

  • web

  • webservices

  • ejb

  • conector

  • appclient

  • weld: contenedor para los contextos y la inyección de dependencia para las aplicaciones de la plataforma Java EE.

La tabla Aplicaciones Desplegadas también contiene las siguientes opciones.

Desplegar

Botón para desplegar una aplicación.

Anular Despliegue

Botón para anular el despliegue de una o más aplicaciones seleccionadas.

Al anular el despliegue de una aplicación se elimina la aplicación de todos los destinos y del dominio. Cuando se anula el despliegue de una aplicación, la aplicación ya no aparece en la página Aplicaciones que muestra todas las aplicaciones desplegadas en GlassFish Server.

Para eliminar aplicaciones solo del destino actual, utilice el botón Eliminar en esta página.

Eliminar

Botón para eliminar una o más aplicaciones seleccionadas.

Al eliminar una aplicación solo se elimina del destino actual. Cuando una aplicación se elimina de un destino, la aplicación sigue apareciendo en la página Aplicaciones que muestra todas las aplicaciones despelgadas en GlassFish Server. La aplicación también seguirá desplegada en otros destinos en los que se hubiera desplegado.

Para eliminar aplicaciones de todos los destinos y del dominio, utilice el botón Anular Despliegue en esta página.

Más Acciones

Lista desplegable que realiza acciones adicionales en una o más aplicaciones seleccionadas.

Activar

Activa una o más aplicaciones seleccionadas.

Desactivar

Desactiva una o más aplicaciones seleccionadas.

Activación de Equilibrador de Carga

Activa una o más aplicaciones seleccionadas para el equilibrio de carga.

Desactivación de Equilibrador de Carga

Desactiva una o más aplicaciones seleccionadas para el equilibrio de carga.
