Ver Datos de Supervisión de Aplicación para una Instancia de GlassFish Server Independiente

Antes de Empezar

Para poder ver datos de supervisión, debe configurar el servicio de supervisión.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. Seleccione la instancia para la cual desea ver los datos de supervisión de aplicaciones.

    Se abre la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Supervisar.

    Se abre la página Supervisión de Aplicación.

  4. En la lista desplegable Ver, seleccione la aplicación para la cual desea ver los datos de supervisión.

  5. Para actualizar los datos de supervisión mostrados, haga clic en Refrescar.
