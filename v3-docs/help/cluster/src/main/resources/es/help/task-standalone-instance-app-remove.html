Eliminar Aplicaciones desde una Instancia de GlassFish Server Independiente

Al eliminar una aplicación, sólo se elimina del destino actual. Cuando una aplicación se elimina de un destino, la aplicación sigue apareciendo en la página Aplicaciones que enumera todas las aplicaciones desplegadas en GlassFish Server. La aplicación también seguirá desplegada en otros objetivos en los que se hubiera desplegado.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abre la página Información General para la instancia.

  3. En la página Información General para la instancia, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para la instancia.

  4. Para cada aplicación que vaya a eliminar, seleccione la casilla de verificación que aparece junto a ella en la tabla Aplicaciones Desplegadas.

  5. Haga clic en Eliminar.

    Se le pedirá que confirme que desea eliminar las aplicaciones.

  6. Confirme que desea eliminar las aplicaciones.

    Las aplicaciones se eliminan de la tabla de aplicaciones desplegadas de la página.
