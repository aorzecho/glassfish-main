Migrar temporizadores EJB

La página Migrar temporizadores EJB contiene las opciones siguientes.

Origen

Lista desplegable de instancias de GlassFish Server detenidas en el clúster. Los temporizadores EJB sólo se pueden migrar desde instancias que se detienen.

Destino

Lista desplegable de instancias de GlassFish Server en ejecución en el clúster. Los temporizadores EJB sólo se pueden migrar a instancias que están en ejecución.

Nota - Si se detienen todas las instancias del clúster, esta página sólo muestra un aviso que indica que no hay disponibles instancias en ejecución a las que se pueda migrar los temporizadores EJB.
