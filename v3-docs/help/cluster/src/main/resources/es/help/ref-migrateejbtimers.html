Migrar Temporizadores EJB

La página Migrar Temporizadores EJB contiene las opciones siguientes.

Origen

Lista desplegable de instancias de GlassFish Server paradas en el cluster. Los temporizadores EJB sólo se pueden migrar desde instancias que están paradas.

Destino

Lista desplegable de instancias de GlassFish Server en ejecución en el cluster. Los temporizadores EJB sólo se pueden migrar a instancias que están en ejecución.

Nota:

si todas las instancias del clúster están paradas, esta página sólo muestra una advertencia de que de que no hay disponibles instancias en ejecución a las que se puedan migrar los temporizadores EJB.

La página Migrar Temporizadores EJB contiene una tabla de temporizadores EJB con la información siguiente.

Nombre de Servidor

Lista de instancias de GlassFish Server en el cluster.

Recuento de Temporizadores

Número de temporizadores persistentes en cada instancia.
