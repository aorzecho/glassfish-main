Suprimir Nodos

Al suprimir un nodo se elimina el nodo de la configuración del Servidor de Administración de Dominio (DAS). Los archivos y directorios del nodo se suprimirán cuando se suprima la última instancia de GlassFish Server que resida en el nodo.

También puede elegir desinstalar el software de GlassFish Server desde el host del nodo cuando se suprime el nodo.

Antes de Empezar

Asegúrese de que no resida ninguna instancia de GlassFish Server en el nodo que se va a suprimir.

  1. En el árbol de navegación, seleccione el nodo Nodos.

    Se abre la página Nodos.

  2. Para cada nodo que vaya a suprimir, seleccione la casilla de verificación que aparece junto al nodo en la tabla Nodos.

    Nota:

    El nodo predefinido localhost-dominio no se puede suprimir.

  3. Haga clic en el botón para suprimir los nodos o suprimir los nodos y desinstalar el software de GlassFish Server desde los hosts del nodo.

    • Para suprimir los nodos sin desinstalar el software de GlassFish Server, haga clic en Suprimir.

      Se le pedirá que confirme que desea suprimir los nodos.

    • Para suprimir los nodos y desinstalar el software de GlassFish Server desde los hosts del nodo, haga clic en Suprimir y Desinstalar.

      Se le pedirá que confirme que desea suprimir los nodos y desinstalar el software de GlassFish Server desde los hosts del nodo.

  4. Confirme que desea suprimir los nodos o suprimir los nodos y desinstalar el software de GlassFish Server desde los hosts del nodo.

    Los nodos se suprimen y se eliminan de la tabla de nodos en la página Nodos. Si ha hecho clic en Suprimir y Desinstalar, el software de GlassFish Server se desinstala también de los hosts del nodo.
