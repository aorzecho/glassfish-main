Para suprimir nodos

Al eliminar un nodo se elimina el nodo de la configuración del servidor de administración de dominios (DAS). Los archivos y directorios del nodo se eliminarán cuando se borre la última instancia de Servidor GlassFish que resida en el nodo.

Antes de empezar

Asegúrese de que no resida ninguna instancia de Servidor GlassFish en el nodo que se va a suprimir.

  1. En el árbol de navegación, seleccione el nodo Nodos.

    Se abrirá la página Nodos.

  2. Para cada nodo que vaya a suprimir, seleccione la casilla de verificación adyacente al nodo en la tabla Nodos.

    Nota - El nodo predefinido localhostdominio no se puede eliminar.

  3. Haga clic en Eliminar.

    Se le pedirá que confirme que desea eliminar el nodo.

  4. Confirme que quiere eliminar el nodo.

    Se eliminará el nodo y se quitará de la tabla de nodos de la página Nodos.
