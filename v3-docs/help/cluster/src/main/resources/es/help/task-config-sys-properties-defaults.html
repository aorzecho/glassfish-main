Para definir los valores predeterminados de las propiedades del sistema de configuración

Los valores predeterminados de las propiedades de configuración se definen en la configuración a la que hace referencia el clúster o la instancia.

  1. En el árbol de navegación, expanda el nodo relativo a la configuración.
  2. En la lista de tipos de propiedades bajo el nodo expandido, seleccione Propiedades del sistema.

    Se abrirá la página Propiedades del sistema para la configuración seleccionada.

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la configuración:

    • Números de puerto predefinidos cuando se creó la configuración

    • Propiedades personalizadas que se añaden a la configuración, si la hay

  3. Para asegurarse de que se apliquen los cambios sin necesidad de reiniciar el clúster o la instancia, seleccione la opción Reconfiguración dinámica.
  4. Para cada propiedad para la que configure un valor predeterminado, escriba el nuevo valor en el campo Valor predeterminado de la propiedad.
  5. Defina los valores predeterminados para las propiedades adicionales que necesite.
    1. En la tabla Propiedades adicionales, haga clic en Agregar propiedad.

      Se añade una fila a la tabla Propiedades adicionales.

    2. En el campo Nombre de variable de instancia, escriba el nombre de la propiedad.
    3. En el campo Valor predeterminado, escriba un valor.
  6. Haga clic en Guardar.
