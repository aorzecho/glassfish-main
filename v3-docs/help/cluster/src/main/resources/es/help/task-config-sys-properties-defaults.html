Definir los Valores por Defecto de las Propiedades del Sistema de Configuración

Los valores por defecto de las propiedades de configuración se definen en la configuración a la que hace referencia el cluster o la instancia.

  1. En el árbol de navegación, amplíe el nodo relativo a la configuración.

  2. En la lista de tipos de propiedades bajo el nodo ampliado, seleccione Propiedades del Sistema.

    Se abre la página Propiedades del Sistema para la configuración seleccionada.

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la configuración:

    • Números de puerto predefinidos cuando se creó la configuración

    • Propiedades personalizadas que se agregan a la configuración, si la hay

  3. Para asegurarse de que se apliquen los cambios sin necesidad de reiniciar el cluster o la instancia, seleccione la opción Nueva Configuración Dinámica.

  4. Para cada propiedad para la que configure un valor por defecto, escriba el nuevo valor en el campo Valor por Defecto de la propiedad.

  5. Defina los valores por defecto para las propiedades adicionales que necesite.

    1. En la tabla Propiedades Adicionales, haga clic en Agregar Propiedad.

      Se agrega una fila a la tabla Propiedades Adicionales.

    2. En el campo Nombre de Variable de Instancia, escriba el nombre de la propiedad.

    3. En el campo Valor por Defecto, escriba el valor.

  6. Haga clic en Guardar.
