Activar o Desactivar el Equilibrio de Carga para una Aplicación que se Despliega en una Instancia de GlassFish Server Independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abre la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abre la página Información General para la instancia.

  3. En la página Información General para la instancia, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para la instancia.

  4. Seleccione la casilla de verificación que aparece junto a la aplicación.

  5. En el menú desplegable Más Acciones, seleccione el comando pertinente.

    • Para activar el equilibrio de carga para la aplicación, elija Activación de Equilibrador de Carga.

    • Para desactivar el equilibrio de carga para la aplicación, elija Desactivación de Equilibrador de Carga.
