Para habilitar o inhabilitar el equilibrio de carga para una aplicación implementada en una instancia de Servidor GlassFish independiente

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. En la página Instancias de servidor independientes, seleccione la instancia.

    Se abrirá la página Información general para la instancia.

  3. En la página Información general de la instancia, haga clic en la ficha Aplicaciones.

    Se abrirá la página Aplicaciones para la instancia.

  4. Seleccione la casilla de verificación adyacente a la aplicación.
  5. En el menú desplegable Más acciones, seleccione el comando pertinente.
    • Para habilitar el equilibrio de carga para la aplicación, elija Equilibrador de carga habilitado.
    • Para desactivar el equilibrio de carga para la aplicación, elija Equilibrador de carga deshabilitado.
