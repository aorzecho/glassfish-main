Recursos

Utilice la página Recursos para realizar las tareas siguientes en el clúster seleccionado:

  • Visualizar y administrar los recursos del clúster

  • Crear recursos adicionales para el clúster

El campo Nombre de clúster es un campo de sólo lectura que muestra el nombre del clúster actual.

La página Recursos muestra una lista de los recursos del clúster. Para cada recurso, aparece la siguiente información.

Nombre de recurso

Nombre que se asignó al recurso en el momento de crearlo. Al hacer clic en el nombre se abre una de las páginas siguientes para editar el recurso, en función del tipo de recurso:

  • Editar recurso JDBC

  • Editar sesiones JavaMail

  • Editar recurso personalizado

  • Nuevo recurso personalizado

  • Editar recurso externo

  • Editar recurso del conector

  • Editar recurso de objeto de administración

Habilitado

Indica si se ha habilitado el recurso:

  • Si el recurso se ha habilitado, se muestra una marca de verificación (✓).

  • Si el recurso está deshabilitado, se muestra una cruz (✗).

Tipo

El tipo de recurso.

La tabla de recursos también contiene las siguientes opciones.

Habilitar

Botón para habilitar uno o varios recursos seleccionados.

Deshabilitar

Botón para deshabilitar uno o varios recursos seleccionados.

Nuevo

Lista desplegable que crea un recurso del tipo seleccionado. Al seleccionar un tipo de recurso en la lista se abre una de las páginas siguientes para crear un recurso del tipo seleccionado:

  • Nuevo recurso JDBC

  • Nueva fábrica de conexión JMS

  • Nuevo recurso de destino JMS

  • Nueva sesión JavaMail

  • Nuevo recurso personalizado

  • Nuevo recurso externo

  • Nuevo recurso de conector

  • Nuevo recurso de objeto de administración

Filtrar

Lista desplegable que filtra la lista de recursos mostrados por tipo.
