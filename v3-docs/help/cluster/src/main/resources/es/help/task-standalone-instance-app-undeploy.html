Para anular la implementación de aplicaciones de una instancia independiente de GlassFish Server

Al anular la implementación de una aplicación se elimina la aplicación de todos los destinos y del dominio. Cuando se anula la implementación de una aplicación, la aplicación ya no aparece en la página Aplicaciones que enumera todas las aplicaciones implementadas en GlassFish Server.

  1. En el árbol de navegación, seleccione el nodo Instancias independientes.

    Se abrirá la página Instancias de servidor independientes.

  2. En la página Instancias de servidor independientes, seleccione la instancia.

    Se abrirá la página Información general para la instancia.

  3. En la página Información general de la instancia, haga clic en la ficha Aplicaciones.

    Se abrirá la página Aplicaciones para la instancia.

  4. Para cada aplicación cuya implementación vaya a anular, seleccione la casilla de verificación que aparece junto a ella en la tabla Aplicaciones implementadas.
  5. Haga clic en Anular implementación.

    Se le pedirá que confirme que desea anular la implementación de las aplicaciones.

  6. Confirme que desea anular la implementación de las aplicaciones.

    Se anula la implementación de las aplicaciones y estas se eliminan de la tabla de aplicaciones implementadas de la página.
