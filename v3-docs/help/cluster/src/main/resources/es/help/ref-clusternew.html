Cluster Nuevo

Utilice la página Cluster Nuevo para crear un cluster.

La página Cluster Nuevo contiene las opciones siguientes.

Nombre de Cluster

Nombre del cluster.

El nombre debe cumplir los siguientes requisitos:

  • El nombre sólo puede contener caracteres ASCII.

  • El nombre debe empezar por una letra, un número o un carácter de subrayado.

  • El nombre sólo puede contener los siguientes caracteres:

    • Letras en minúscula

    • Letras en mayúscula

    • Números

    • Guión

    • Punto

    • Carácter de subrayado

  • El nombre debe ser exclusivo en el dominio y no debe ser el nombre de otro cluster, una configuración con nombre, una instancia de GlassFish Server ni un nodo.

  • El nombre no debe ser domain, server ni otra palabra clave reservada por GlassFish Server.

Configuración

Lista desplegable de las configuraciones con nombre existentes que puede utilizar el cluster. Un cluster necesita una configuración con nombre que defina la configuración de todas las instancias que se agregan al cluster. Debe seleccionarse una configuración en la lista. El cluster utilizará la configuración seleccionada cuando se cree.

Realizar una Copia de la Configuración Seleccionada

Si se selecciona esta opción, se copia la configuración seleccionada cuando se crea el cluster. Se asigna el nombre cluster-name-config a la copia de la configuración, donde cluster-name es el nombre del cluster del campo Nombre de Cluster.

Si se selecciona default-config en la lista desplegable Configuración, debe seleccionar esta opción. La configuración default-config sólo se puede copiar, no tomar como referencia.

Hacer Referencia a la Configuración Seleccionada

Si se selecciona esta opción, el cluster utilizará la configuración con nombre existente especificada.

Si se selecciona default-config en la lista desplegable Configuración, no debe seleccionar esta opción. La configuración default-config no puede tomarse como referencia, sino sólo copiarse.

Instancias del Servidor Que Se Van a Crear

Lista de instancias de GlassFish Server que se crearán cuando se cree el cluster. Para cada instancia, se proporciona la información siguiente:

Nombre de Instancia

Nombre de la instancia.

Peso

Entero que representa el peso de equilibrio de carga de la instancia.

El peso de equilibrio de carga determina la proporción de todas las solicitudes con respecto al cluster que debe procesar la instancia.

Por ejemplo, en un cluster de dos instancias, puede necesitar que una instancia procese una solicitud de cuatro, y que otra procese las otras tres. En este caso, defina el peso de la instancia que debe procesar una de las cuatro solicitudes en 1 y el peso de la otra instancia en 3. Si prefiere usar porcentajes, defina los pesos de las instancias en 25 y 75, respectivamente.

El peso por defecto es 100.

Nodo

Lista desplegable de los nodos existentes en los que puede residir la instancia. Debe seleccionarse un nodo en la lista. La instancia residirá en el nodo seleccionado cuando se cree el cluster.

La tabla Instancias del Servidor Que Se Van a Crear también contiene las opciones siguientes.

Nuevo

Botón para crear una instancia. Al hacer clic en este botón se agrega una fila a la tabla Instancias del Servidor Que Se Van a Crear.

Suprimir

Botón para suprimir una instancia.
