Clústeres

La página Clústeres muestra una lista de los clústeres del dominio. Para cada clúster, se muestra la información siguiente:

Nombre

El nombre que se asignó al clúster en el momento de crearlo. Al hacer clic en el nombre se abre la página Información general del clúster.

Configuración

La configuración que se seleccionó para el clúster en el momento de crearlo. Al hacer clic en la configuración se abre la página Configuración para el clúster.

Instancias

Nombre y estado de cada instancia en el clúster. Al hacer clic en el nombre de la instancia se abre la página Información general para la instancia.

La tabla Clústeres también contiene las opciones siguientes.

Nuevo

Botón para crear un clúster.

Borrar

Botón para eliminar un clúster.

Iniciar clúster

Botón para iniciar un clúster.

Detener clúster

Botón para detener un clúster.
