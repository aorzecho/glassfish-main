Para ativar ou desativar o balanceamento de carga de um aplicativo que está implantado em uma instância do Servidor GlassFish independente

  1. Na árvore de navegação, selecione o nó Instâncias independentes.

    A página Instância de servidor independente é exibida.

  2. Na página Instância de servidor independente, selecione a instância.

    A página Informações gerais da instância se abre.

  3. Na página Informações gerais da instância, clique na guia Aplicativos.

    A página Aplicativos da instância se abre.

  4. Selecione a caixa de seleção ao lado do aplicativo.
  5. No menu suspenso Mais ações, escolha o comando apropriado.
    • Para ativar o balanceamento de carga do aplicativo, escolha Balanceador de carga ativado.
    • Para desativar o balanceamento de carga do aplicativo, escolha Balanceador de carga desativado.
