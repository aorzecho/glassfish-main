Para Exibir Dados de Monitoramento de Aplicações de uma Instância Clusterizada do GlassFish Server

Antes de Começar

Antes de exibir os dados de monitoramento, é preciso configurar o serviço de monitoramento.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias Clusterizadas do Servidor relativa ao cluster se abre.

  4. Selecione a instância da qual visualizar os dados de monitoramento de aplicação.

    A página Informações gerais da instância se abre.

  5. Na página Informações gerais, clique na guia Monitor.

    A página Monitoramento da Aplicação é exibida.

  6. Na lista drop-down Exibir, selecione a aplicação cujos dados de monitoramento você deseja exibir.

  7. Para atualizar os dados de monitoramento exibidos, clique em Atualizar.
