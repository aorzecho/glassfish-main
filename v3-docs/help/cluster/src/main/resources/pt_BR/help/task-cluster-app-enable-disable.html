Para Ativar ou Desativar uma Aplicação Que Está Implantanda em um Cluster

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações Gerais, clique na guia Aplicações.

    A página Aplicações do cluster se abre.

  4. Marque a caixa de seleção ao lado da aplicação.

  5. No menu pull-down Mais Ações, escolha o comando apropriado.

    • Para ativar a aplicação, escolha Ativar.

    • Para desativar a aplicação, escolha Desativar.
