Para interromper um cluster

Ao interromper um cluster, todas as instâncias do Servidor GlassFish no cluster são interrompidas.

  1. Na árvore de navegação, selecione o nó Clusters.

    A página Clusters é exibida.

  2. Na página Clusters, marque a caixa de seleção ao lado do nome do cluster.
  3. Clique em Interromper cluster.

    Você é solicitado a confirmar que deseja interromper o cluster.

  4. Confirme que deseja interromper o cluster.

    O cluster é interrompido. O status de cada instância do cluster é atualizado na página Clusters quando a instância for interrompida.
