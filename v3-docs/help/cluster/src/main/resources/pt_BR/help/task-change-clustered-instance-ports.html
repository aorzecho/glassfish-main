Para Modificar os Números de Porta de uma Instância Clusterizada do GlassFish Server

Os números de porta de uma instância clusterizada do GlassFish Server são definidos inicialmente na configuração que a instância herda do cluster pai. É necessário modificar estes números de porta se, por exemplo, duas instâncias clusterizadas residirem no mesmo host. Nesse caso, os números de porta de uma instância devem ser modificados porque ambas as instâncias compartilham a mesma configuração.

Os números de porta de uma instância são armazenados como propriedades de sistema Java. Essas propriedades são passadas para o acionador da aplicação Java por meio da opção -D do acionador da aplicação Java quando o GlassFish Server é iniciado.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias Clusterizadas do Servidor relativa ao cluster se abre.

  4. Selecione a instância cujos números de porta você deseja modificar.

    A página Informações gerais da instância se abre.

  5. Na página Informações gerais, clique na guia Propriedades.

    A página Propriedades do sistema é exibida.

    Esta página exibe os valores atuais das seguintes propriedades do sistema para a instância:

    • Números de porta predefinidos na configuração da instância

    • Propriedades personalizadas definidas para a instância, se houver

  6. Digite o novo valor de cada propriedade que estiver sendo modificada no campo Valor Substituto.

  7. Defina os valores de quaisquer propriedades adicionais que forem necessárias.

    1. Na tabela Propriedades adicionais, clique em Adicionar propriedade.

      Uma linha é adicionada à tabela Propriedades adicionais.

    2. No campo Nome da variável, digite o nome da propriedade.

    3. No campo Valor Substituto, digite o valor.

  8. Clique em Salvar.
