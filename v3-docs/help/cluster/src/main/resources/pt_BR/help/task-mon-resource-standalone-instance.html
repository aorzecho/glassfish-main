Para Exibir Dados de Monitoramento de Recursos de uma Instância Standalone do GlassFish Server

Antes de Começar

Antes de exibir os dados de monitoramento, é preciso configurar o serviço de monitoramento.

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Selecione a instância da qual visualizar os dados de monitoramento do recurso.

    A página Informações gerais da instância se abre.

  3. Na página Informações gerais, clique na guia Monitor.

    A página Monitoramento da Aplicação é exibida.

  4. Clique na guia Recursos.

    A página Monitoramento do recurso é exibida.

  5. Na lista drop-down Exibir, selecione o recurso cujos dados de monitoramento deseja visualizar.

  6. Para atualizar os dados de monitoramento exibidos, clique em Atualizar.
