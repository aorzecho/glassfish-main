Para definir os valores padrão para configuração das propriedades do sistema

Os valores padrão das propriedades de configuração são definidos na configuração que o cluster ou a instância fazem referência.

  1. Na árvore de navegação, expanda o nó da configuração.
  2. Na lista dos tipos de propriedades do nó expandido, selecione Propriedades do sistema.

    A página Propriedades do sistema da configuração selecionada se abre.

    Esta página exibe os valores atuais das seguintes propriedades do sistema para a configuração:

    • Números de porta predefinidos quando a configuração foi criada

    • Propriedades personalizadas adicionadas à configuração, se houver

  3. Para certificar-se de que as alterações sejam aplicadas sem ter que reiniciar o cluster ou a instância, selecione a opção Reconfiguração dinâmica.
  4. Digite o novo valor da propriedade no campo Valor padrão de cada propriedade cujo valor padrão estiver sendo definido.
  5. Defina os valores padrão de quaisquer propriedades adicionais que forem necessárias.
    1. Na tabela Propriedades adicionais, clique em Adicionar propriedade.

      Uma linha é adicionada à tabela Propriedades adicionais.

    2. No campo Nome da variável, digite o nome da propriedade.
    3. No campo Valor padrão, digite o valor.
  6. Clique em Salvar.
