Para Deletar um Cluster

Ao deletar um cluster, o cluster e todas as instâncias do GlassFish Server no cluster são deletados. Não é necessário deletar as instâncias do cluster antes de deletá-lo.

Se a configuração que o cluster utilizar não for compartilhada, a configuração também é deletada.

Antes de Começar

Certifique-se de que o cluster que você está deletando esteja interrompido. Consulte Para Interromper um Cluster para obter instruções.

  1. Na árvore de navegação, selecione o nó Clusters.

    A página Clusters é exibida.

  2. Na página Clusters, marque a caixa de seleção ao lado do nome do cluster.

  3. Clique em Deletar.

    Você é solicitado a confirmar que deseja deletar o cluster e todas as instâncias do cluster.

  4. Confirme que deseja deletar o cluster e todas as instâncias do cluster.

    O cluster é deletado e removido da tabela de clusters na página Clusters.
