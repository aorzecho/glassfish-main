Para Cancelar a Implantação de Aplicações de uma Instância Standalone do GlassFish Server

O cancelamento da implantação de uma aplicação remove a aplicação de todos os alvos e do domínio. Quando a implantação de uma aplicação é cancelada, a aplicação deixa de ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server.

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Na página Instância de servidor standalone, selecione a instância.

    A página Informações gerais da instância se abre.

  3. Na página Informações gerais da instância, clique na guia Aplicações.

    A página Aplicações da instância se abre.

  4. Para cada nó cuja implantação estiver sendo cancelada, marque a caixa de seleção ao lado da aplicação na tabela Aplicações Implantadas.

  5. Clique em Cancelar Implantação.

    Você é solicitado a confirmar que deseja cancelar a implantação das aplicações.

  6. Confirme que deseja cancelar a implantação das aplicações.

    A implantação das aplicações é cancelada e as aplicações são removidas da tabela de aplicações implantadas da página.
