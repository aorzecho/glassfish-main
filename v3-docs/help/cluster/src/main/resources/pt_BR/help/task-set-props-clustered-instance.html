Para Definir Propriedades Somente da Instância de uma Instância Clusterizada do GlassFish Server

As propriedades de uma instância do GlassFish Server acrescentam informações opcionais de configuração sobre a instância.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias Clusterizadas do Servidor relativa ao cluster se abre.

  4. Selecione a instância cujas propriedades deseja modificar.

    A página Informações gerais da instância se abre.

  5. Na página Informações gerais, clique na guia Propriedades.

  6. Se necessário, clique na guia Propriedades da instância.

    A página Propriedades da instância é exibida.

    Esta página exibe uma lista das propriedades que estão definidas para a instância selecionada do GlassFish Server.

  7. Digite o novo valor de cada propriedade que estiver sendo modificada no campo Valor de tal propriedade.

  8. Defina os valores de quaisquer propriedades adicionais que forem necessárias.

    1. Na tabela Propriedades adicionais, clique em Adicionar propriedade.

      Uma linha é adicionada à tabela Propriedades adicionais.

    2. No campo Nome, digite o nome da propriedade.

    3. No campo Valor, digite o valor.

    4. No campo Descrição, digite uma descrição breve da propriedade.

  9. Clique em Salvar.
