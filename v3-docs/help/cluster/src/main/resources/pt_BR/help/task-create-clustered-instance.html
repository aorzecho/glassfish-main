Para adicionar uma instância do Servidor GlassFish a um cluster

As instâncias do Servidor GlassFish formam a base da implantação de um aplicativo. A instância em cluster é um membro de um cluster. Cada instância pertence a um único domínio e tem sua própria estrutura de diretório, configuração e aplicativos implantados. Cada instância também inclui o contêiner Web da plataforma Java, Enterprise Edition (Java EE) e o contêiner da tecnologia Enterprise JavaBeans (tecnologia EJB). Cada instância nova deve conter uma referência a um nó que defina o host no qual a instância residirá.

Observação - Para criar uma instância em um nó de tipo CONFIG que represente um host remoto, você deve utilizar o subcomando create-local-instance(1). É possível utilizar o Console de Administração para este propósito.

Antes de começar

Certifique-se de que os seguintes itens existam:

  • O cluster ao qual a instância está sendo adicionada

  • O nó no qual a instância residirá

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias do servidor em clusters do cluster se abre.

  4. Na página Instâncias do servidor em clusters, clique em Novo.

    A página Instâncias do servidor em clusters é exibida.

  5. No campo Nome, digite um nome para a instância.

    O nome deve cumprir os seguintes requisitos:

    • O nome deve começar com uma letra, um número ou um sublinhado.

    • O nome pode conter somente os seguintes caracteres:

      • Letras minúsculas

      • Letras maiúsculas

      • Números

      • Hífen

      • Período

      • Sublinhado

    • O nome deve ser único no domínio e não deve ser o nome de outra instância, um cluster, uma configuração nomeada ou um nó do Servidor GlassFish.

    • O nome não deve ser domain, server, nem nenhuma outra palavra-chave que esteja reservada pelo Servidor GlassFish.

  6. Na lista suspensa Nó, selecione o nó no qual a instância residirá.
  7. Clique em OK.
