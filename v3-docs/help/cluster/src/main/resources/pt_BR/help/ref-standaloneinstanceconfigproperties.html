Propriedades do sistema da instância

A página Propriedades do sistema exibe uma lista das propriedades do sistema Java que estão definidas para a instância atual do Servidor GlassFish. As propriedades do sistema Java são transmitidas para o iniciador de aplicativo Java através da opção -D do iniciador de aplicativo Java quando o Servidor GlassFish é iniciado.

Essas propriedades substituem as definições de propriedade para configurações de porta na configuração da instância. As configurações de porta predefinidas devem ser substituídas se, por exemplo, duas instâncias em cluster residirem no mesmo host. Nessa situação, as configurações de porta para uma instância devem ser substituídas porque ambas as instâncias compartilham a mesma configuração.

Para obter informações sobre as configurações de porta predefinidas, consulte Predefined Configuration System Properties.

O campo Nome da instância é um campo de somente leitura que exibe o nome da instância atual.

São exibidas as seguintes informações para cada propriedade:

Nome da variável da instância

Nome da propriedade do sistema.

Valor atual

Valor atualmente definido para a propriedade. Este campo é somente leitura.

Novo valor

Valor com o qual a propriedade será definida depois que as alterações forem salvas.

A tabela Propriedades adicionais também contém as seguintes opções.

Adicionar propriedade

Botão para adicionar uma propriedade. Se clicar neste botão, uma linha é adicionada à tabela Propriedades adicionais.

Excluir propriedades

Botão para excluir uma ou mais propriedades selecionadas. Qualquer propriedade excluída volta ao seu valor padrão ou, se nenhum valor tiver sido definido, fica com valor indefinido.
