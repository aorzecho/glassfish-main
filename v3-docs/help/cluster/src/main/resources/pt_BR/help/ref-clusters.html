Clusters

A página Clusters exibe uma lista de clusters no domínio. São exibidas as seguintes informações para cada cluster:

Nome

Nome atribuído ao cluster quando o cluster é criado. Ao clicar no nome do cluster, a página Informações gerais se abre.

Configuração

Nome selecionado para o cluster quando o cluster foi criado. Ao clicar na configuração, a página Configurações do cluster se abre.

Instâncias

O nome e o status de cada instância do cluster. Ao clicar no nome da instância, a página Informações gerais se abre.

A tabela Clusters também contém as opções a seguir.

Novo

Botão para criar um cluster.

Excluir

Botão para excluir um cluster.

Iniciar cluster

Botão para iniciar um cluster.

Interromper cluster

Botão para interromper um cluster.
