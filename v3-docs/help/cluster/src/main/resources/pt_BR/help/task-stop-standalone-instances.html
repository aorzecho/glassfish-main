Para Interromper Instâncias Standalone do GlassFish Server

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Para cada instância que estiver sendo parada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  3. Clique em Interromper.

    Você é solicitado a confirmar que deseja interromper as instâncias.

  4. Confirme que deseja interromper as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do servidor quando a instância for interrompida.
