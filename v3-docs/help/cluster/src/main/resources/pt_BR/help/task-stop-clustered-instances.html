Para interromper instâncias do Servidor GlassFish em cluster

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias do servidor em clusters do cluster se abre.

  4. Para cada instância que estiver sendo parada, selecione a caixa de seleção ao lado da instância na tabela Instâncias do servidor.
  5. Clique em Interromper.

    Você é solicitado a confirmar que deseja interromper as instâncias.

  6. Confirme que deseja interromper as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do servidor quando a instância for interrompida.
