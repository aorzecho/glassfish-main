Para Iniciar Instâncias Standalone do GlassFish Server

Antes de Começar

Certifique-se de que os pré-requisitos seguintes sejam cumpridos:

  • Todos os nós nos quais se encontram as instâncias estão ativados para comunicação remota.

  • O usuário do servidor de administração de domínios (DAS) pode usar o protocolo remoto Distributed Component Object Model (DCOM) ou secure shell (SSH) para efetuar log-in no host para qualquer nó em que residam as instâncias no cluster.

Se algum desses pré-requisitos não for atendido, inicie cada instância individualmente pela linha de comandos, usando o subcomando start-local-instance(1).

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Para cada instância que estiver sendo iniciada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  3. Clique em Iniciar.

    Você é solicitado a confirmar que deseja iniciar as instâncias.

  4. Confirme que deseja iniciar as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do servidor quando a instância for iniciada.
