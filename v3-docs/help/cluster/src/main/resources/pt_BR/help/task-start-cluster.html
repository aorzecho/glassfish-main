Para iniciar um cluster

Ao iniciar um cluster, todas as instâncias do Servidor GlassFish do cluster que ainda não estão em execução são iniciadas.

Antes de começar

Certifique-se de que os pré-requisitos seguintes sejam cumpridos:

  • Todos os nós nos quais se encontram as instâncias do cluster estão ativados para comunicação remota.

  • O usuário do servidor de administração de domínio (DAS) pode usar shell seguro (SSH) para efetuar logon no host de qualquer nó em que se encontrem as instâncias do cluster.

Se nenhum destes pré-requisitos for cumprido, inicie o cluster iniciando cada instância individualmente na linha de comando utilizando o subcomando start-local-instance(1).

  1. Na árvore de navegação, selecione o nó Clusters.

    A página Clusters é exibida.

  2. Na página Clusters, marque a caixa de seleção ao lado do nome do cluster.
  3. Clique em Iniciar cluster.

    Você é solicitado a confirmar que deseja iniciar o cluster.

  4. Confirme que deseja iniciar o cluster.

    O cluster é iniciado. O status de cada instância do cluster é atualizado na página Clusters quando a instância for iniciada.
