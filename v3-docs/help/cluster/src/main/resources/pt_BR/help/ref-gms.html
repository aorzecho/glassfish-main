Editar serviço de gerenciamento de grupos

Utilize a página Serviço de gerenciamento de grupos para alterar as configurações do GMS.

O serviço de gerenciamento de grupos (GMS) é um serviço em processo que oferece monitoramento de cluster, associação de cluster e serviços de comunicação de grupos. O GMS está construído sobre uma pilha configurável de protocolos de monitoramento da integridade e de descoberta da associação do grupo. O GMS realiza as seguintes funções:

  • Notifica os módulos registrados em uma instância do Servidor GlassFish quando uma ou mais instâncias do membro são adicionadas ou removidas de um cluster, ou quando tais instâncias sejam suspeitas de ter falhas ou realmente tenham falhas.

  • Oferece a capacidade de enviar e receber mensagens entre um grupo de processos.

A página Serviço de gerenciamento de grupos contém as seguintes opções.

Carregar padrões

Botão para restaurar com os valores padrão as configurações que apresentam valores padrão. As configurações que não apresentam valores padrão não são alteradas.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Maximum Missed Heartbeats (Máximo de pulsações perdidas)

Número de tentativas de monitoramento após o qual a instância do servidor é suspeita de ter falha. O padrão é 3.

Heartbeat Frequency (Frequência das pulsações)

Período de tempo entre as tentativas de monitoramento. O padrão é 2000 milissegundos.

Group Discovery Timeout (Tempo limite da descoberta de grupo)

Quantidade de tempo que o GMS espera a descoberta de outros membros do grupo. O padrão é 5000 milissegundos.

Failure Verification Wait Time (Tempo de espera para seleção da falha)

Período de tempo após o qual uma suspeita de falha é marcada como verificada. O padrão é 1500 milissegundos.

Propriedades adicionais

Tabela de propriedades adicionais do Serviço de gerenciamento de grupos.

O Servidor GlassFish não define propriedades adicionais do GMS.

São exibidas as seguintes informações para cada propriedade:

Nome

O nome da propriedade.

Valor

Valor da propriedade definida para a instância selecionada.

Descrição

Descrição textual que oferece mais informações sobre a propriedade.

A tabela Propriedades adicionais também contém as seguintes opções.

Adicionar propriedade

Botão para adicionar uma propriedade. Se clicar neste botão, uma linha é adicionada à tabela Propriedades adicionais.

Excluir propriedades

Botão para excluir uma ou mais propriedades selecionadas. Qualquer propriedade excluída volta ao seu valor padrão ou, se nenhum valor tiver sido definido, fica com valor indefinido.
