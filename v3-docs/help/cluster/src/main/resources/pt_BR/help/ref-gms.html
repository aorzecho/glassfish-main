Editar Serviço de Gerenciamento de Grupo

Utilize a página Serviço de gerenciamento de grupos para alterar as definições do GMS.

O serviço de gerenciamento de grupos (GMS) é um serviço em processo que oferece monitoramento de cluster, associação de cluster e serviços de comunicação de grupos. O GMS está construído sobre uma pilha configurável de protocolos de monitoramento da integridade e de descoberta da associação do grupo. O GMS realiza as seguintes funções:

  • Notifica os módulos registrados em uma instância do GlassFish Server quando uma ou mais instâncias do membro são adicionadas ou removidas de um cluster, ou quando tais instâncias sejam suspeitas de ter falhas ou realmente tenham falhas.

  • Oferece a capacidade de enviar e receber mensagens entre um grupo de processos.

A página Serviço de gerenciamento de grupos contém as seguintes opções.

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Maximum Missed Heartbeats (Máximo de pulsações perdidas)

Número de tentativas de monitoramento após o qual a instância do servidor é suspeita de ter falha. O default é 3.

Heartbeat Frequency (Frequência das pulsações)

Período de tempo entre as tentativas de monitoramento. O default é 2000 milissegundos.

Group Discovery Timeout (Time-out para descoberta de grupo)

Quantidade de tempo que o GMS espera a descoberta de outros membros do grupo. O default é 5000 milissegundos.

Tempo de Espera para Verificação de Falha)

Período de tempo após o qual uma suspeita de falha é marcada como verificada. O default é 1500 milissegundos.

Propriedades adicionais

Tabela de propriedades adicionais do Serviço de gerenciamento de grupos.

O GlassFish Server não define propriedades adicionais do GMS.

São exibidas as seguintes informações para cada propriedade:

Nome

O nome da propriedade.

Valor

Valor da propriedade definida para a instância selecionada.

Descrição

Descrição textual que oferece mais informações sobre a propriedade.

A tabela Propriedades adicionais também contém as seguintes opções.

Adicionar propriedade

Botão para adicionar uma propriedade. Se clicar neste botão, uma linha é adicionada à tabela Propriedades adicionais.

Deletar propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada volta ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.
