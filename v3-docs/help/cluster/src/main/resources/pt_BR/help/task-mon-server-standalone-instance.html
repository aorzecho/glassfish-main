Para exibir dados de monitoramento de servidor virtual de uma instância do Servidor GlassFish independente

Antes de começar

Antes de exibir os dados de monitoramento, é preciso configurar o serviço de monitoramento.

  1. Na árvore de navegação, selecione o nó Instâncias independentes.

    A página Instância de servidor independente é exibida.

  2. Selecione a instância cujos dados de monitoramento dos servidores virtuais deseja visualizar.

    A página Informações gerais da instância se abre.

  3. Na página Informações gerais, clique na guia Monitor.

    A página Monitoramento do aplicativo é exibida.

  4. Clique na guia Servidor.

    A página Monitoramento do servidor é exibida.

  5. Na lista suspensa Exibir, selecione o componente cujos dados de monitoramento deseja visualizar.
  6. Para atualizar os dados de monitoramento exibidos, clique em Atualizar.
