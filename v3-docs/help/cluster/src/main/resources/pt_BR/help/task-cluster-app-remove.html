Para Remover Aplicações de um Cluster

A remoção de uma aplicação remove a aplicação somente do destino atual. Quando uma aplicação é removida de um destino, a aplicação continua a ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server. A aplicação também permanece implantada nos outros alvos nos quais estava implantada.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações Gerais, clique na guia Aplicações.

    A página Aplicações do cluster se abre.

  4. Para cada nó que estiver sendo removido, marque a caixa de seleção ao lado da aplicação na tabela Aplicações Implantadas.

  5. Clique em Remover.

    Você é solicitado a confirmar que deseja remover as aplicações.

  6. Confirme que deseja remover as aplicações.

    As aplicações são removidas da tabela de aplicações implantadas da página.
