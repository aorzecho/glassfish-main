Para remover aplicativos de um cluster

A remoção de um aplicativo remove o aplicativo somente do destino atual. Quando um aplicativo é removido de um destino, a aplicação continua estando na lista na página Aplicativos que exibe todos os aplicativos implantados no Servidor GlassFish. O aplicativo também permanece implantado nos outros destinos nos quais estava implantado.

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Aplicativos.

    A página Aplicativos do cluster se abre.

  4. Para cada nó que estiver sendo excluído, selecione a caixa de seleção ao lado do aplicativo na tabela Aplicativos implantados.
  5. Clique em Remover.

    Você é solicitado a confirmar que deseja remover os aplicativos.

  6. Confirme que deseja remover os aplicativos.

    Os aplicativos são removidos da tabela de aplicativos implantados da página.
