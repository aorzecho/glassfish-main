Para Editar um Recurso de uma Instância Standalone do GlassFish Server

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Na página Instância de servidor standalone, selecione a instância.

    A página Informações gerais da instância se abre.

  3. Na página Informações gerais, clique na guia Recursos.

    A página Recursos da instância se abre.

  4. Clique no nome do recurso.

    A página para a edição do recurso se abre.

  5. Na página para edição do recurso, faça as alterações necessárias e salve-as.

    Para obter detalhes, consulte a ajuda on-line da página de edição de recursos.
