Para excluir nós

A exclusão de um nó remove o nó da configuração do servidor de administração do domínio (DAS). Os diretórios e arquivos do nó são excluídos quando a última instância do Servidor GlassFish que reside no nó é excluída.

Antes de começar

Certifique-se de que nenhuma instância do Servidor GlassFish resida no nó que está sendo excluído.

  1. Na árvore de navegação, selecione o nó Nós.

    A página Nós é exibida.

  2. Para cada nó que estiver sendo excluído, selecione a caixa de seleção ao lado do nó na tabela Nós.

    Observação - O nó predefinido localhost-domínio não pode ser excluído.

  3. Clique em Excluir.

    Você é solicitado a confirmar que deseja excluir o nó.

  4. Confirme que deseja excluir o nó.

    O nó é excluído e removido da tabela de nós na página Nós.
