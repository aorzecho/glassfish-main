Para Deletar Nós

A deleção de um nó remove o nó da configuração do servidor de administração de domínios (DAS). Os diretórios e arquivos do nó são deletados quando a última instância do GlassFish Server que reside no nó é deletada.

Você também pode optar por desinstalar o software GlassFish Server do host do nó quando o nó for deletado.

Antes de Começar

Certifique-se de que nenhuma instância do GlassFish Server resida no nó que está sendo deletado.

  1. Na árvore de navegação, selecione o nó Nós.

    A página Nós é exibida.

  2. Para cada nó que estiver sendo deletado, marque a caixa de seleção ao lado do nó na tabela Nós.

    Observação:

    O nó predefinido localhost-domain não pode ser deletado.

  3. Clique no botão para deletar os nós ou deletar os nós e desinstalar o software GlassFish Server dos hosts do nó.

    • Para deletar os nós sem desinstalar o software GlassFish Server, clique em Deletar.

      Você será solicitado a confirmar que deseja deletar os nós.

    • Para deletar os nós e desinstalar o software GlassFish Server dos hosts do nó, clique em Deletar e Desinstalar.

      Você será solicitado a confirmar que deseja deletar os nós e desinstalar o software GlassFish Server dos hosts do nó.

  4. Confirme se deseja deletar os nós ou deletar os nós e desinstalar o software GlassFish Server dos hosts do nó.

    Os nós são deletados e removidos da tabela de nós na página Nós. Se você tiver clicado em Deletar e Desinstalar, o software GlassFish Server também será desinstalado dos hosts do nó.
