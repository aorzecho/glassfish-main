Migrar Timers de EJB

A página Migrar Timers EJB contém as seguintes opções.

Origem

Lista drop-down das instâncias interrompidas do GlassFish Server no cluster. Os timers EJB podem ser migrados somente das instâncias que estiverem interrompidas.

Destino

Lista drop-down das instâncias em execução do GlassFish Server no cluster. Os timers EJB podem ser migrados somente para as instâncias que estiverem em execução.

Observação:

Se todas as instâncias do cluster estiverem interrompidas, esta página exibirá somente uma advertência de que não há nenhuma instância em execução disponível para a qual os timers EJB possam ser migrados.

A página Migrar Timers de EJB contém uma tabela de timers de EJB com as informações a seguir.

Nome do servidor

Lista de instâncias de GlassFish Server no cluster.

Contagem do Timer

Número de timers persistências em cada instância.
