Migrar temporizadores EJB

A página Migrar temporizadores EJB contém as seguintes opções.

Origem

Lista suspensa das instâncias interrompidas do Servidor GlassFish no cluster. Os temporizadores EJB podem ser migrados somente das instâncias que estiverem interrompidas.

Destino

Lista suspensa das instâncias em execução do Servidor GlassFish no cluster. Os temporizadores EJB podem ser migrados somente para as instâncias que estiverem em execução.

Observação - Se todas as instâncias do cluster estiverem interrompidas, esta página exibe somente um aviso de que não há nenhuma instância em execução disponível para a qual os temporizadores EJB possam ser migrados.
