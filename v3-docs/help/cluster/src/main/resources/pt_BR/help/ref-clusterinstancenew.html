Nova instância do servidor em clusters

Utilize a página Nova instância do servidor em cluster para adicionar uma instância do Servidor GlassFish a um cluster existente.

A página Nova instância do servidor em cluster contém as seguintes opções.

Nome do cluster

Nome atribuído ao cluster quando o cluster é criado. Este campo é somente leitura.

Nome da instância

O nome da instância que está sendo criada.

O nome deve cumprir os seguintes requisitos:

  • O nome deve começar com uma letra, um número ou um sublinhado.

  • O nome pode conter somente os seguintes caracteres:

    • Letras minúsculas

    • Letras maiúsculas

    • Números

    • Hífen

    • Período

    • Sublinhado

  • O nome deve ser único no domínio e não deve ser o nome de outra instância, um cluster, uma configuração nomeada ou um nó do Servidor GlassFish.

  • O nome não deve ser domain, server, nem nenhuma outra palavra-chave que esteja reservada pelo Servidor GlassFish.

Nó

Lista suspensa dos nós existentes que definem os hosts nos quais a instância pode estar. É necessário selecionar um nó na lista. Caso a instância deva ser criada no host onde o servidor de administração do domínio (DAS) estiver sendo executado, selecione o nó predefinido localhost-domínio.
