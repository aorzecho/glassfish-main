Para Iniciar Instâncias Clusterizadas do GlassFish Server

Antes de Começar

Certifique-se de que os pré-requisitos seguintes sejam cumpridos:

  • Todos os nós nos quais se encontram as instâncias estão ativados para comunicação remota.

  • O usuário do servidor de administração de domínios (DAS) pode usar shell seguro (SSH) para efetuar log-in no host de qualquer nó em que se encontrem as instâncias.

Se algum desses pré-requisitos não for atendido, inicie cada instância individualmente pela linha de comandos, usando o subcomando start-local-instance(1).

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias Clusterizadas do Servidor relativas ao cluster se abre.

  4. Para cada instância que estiver sendo iniciada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  5. Clique em Iniciar.

    Você é solicitado a confirmar que deseja iniciar as instâncias.

  6. Confirme que deseja iniciar as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do servidor quando a instância for iniciada.
