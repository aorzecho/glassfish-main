Recuperar transações

Utilize a página Recuperar transações para recuperar transações de uma instância do Servidor GlassFish.

Se a instância estiver em execução, a página Recuperar transações contém somente o texto A recuperação é realizada pelo próprio servidor em execução e o botão OK.

Se a instância não estiver em execução, a página Recuperar transações contém as seguintes opções.

Nome da instância

Nome atribuído à instância quando a instância foi criada. Este campo é somente leitura.

Servidor de destino

Outra instância que pode recuperar as transações na instância atual.

Diretório do log de transações

Diretório no qual os logs de transação são armazenados. O padrão é /logs/tx no diretório do domínio.

Os logs de transação são mantidos no subdiretório tx do diretório especificado pela configuração do Local do log de transação do Serviço de transação na configuração do cluster. O valor padrão do Local do log de transação é o diretório especificado pelo campo Raiz de log da página Atributos do domínio, que está localizado na guia Avançado do nó do Servidor GlassFish.

Tarefas relacionadas

  • Para recuperar transações de uma instância do Servidor GlassFish em cluster

  • Para recuperar transações de uma instância independente do Servidor GlassFish

Subcomandos asadmin equivalentes
