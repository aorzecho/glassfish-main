Para excluir instâncias independentes do Servidor GlassFish

  1. Na árvore de navegação, selecione o nó Instâncias independentes.

    A página Instância de servidor independente é exibida.

  2. Para cada instância que estiver sendo excluída, selecione a caixa de seleção ao lado da instância na tabela Instâncias do servidor.
  3. Clique em Excluir.

    Você é solicitado a confirmar que deseja excluir as instâncias.

  4. Confirme que deseja excluir as instâncias.

    As instâncias são excluídas e removidas da tabela de instâncias da página.
