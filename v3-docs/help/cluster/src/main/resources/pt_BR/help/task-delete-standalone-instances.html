Para Deletar Instâncias Standalone do GlassFish Server

  1. Na árvore de navegação, selecione o nó Instâncias standalone.

    A página Instância de servidor standalone é exibida.

  2. Para cada instância que estiver sendo deletada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  3. Clique em Deletar.

    Você é solicitado a confirmar que deseja deletar as instâncias.

  4. Confirme que deseja deletar as instâncias.

    As instâncias são deletadas e removidas da tabela de instâncias da página.
