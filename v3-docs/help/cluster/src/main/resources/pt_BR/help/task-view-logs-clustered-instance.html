Para exibir arquivos de log de uma instância do Servidor GlassFish em cluster

Antes de começar

A instância para a qual você está exibindo os arquivos de log deve estar em execução.

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias do servidor em clusters do cluster se abre.

  4. Selecione a instância cujos arquivos de log deseja visualizar.

    A página Informações gerais da instância se abre.

  5. Na página Informações gerais, clique Exibir arquivos de log.

    A janela Visualizador de logs é exibida.

    Observação - A instância já deve estar em execução. Se a instância não estiver em execução, o botão Exibir arquivos de log está desativado.
