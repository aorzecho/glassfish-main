Para configurar o serviço de gerenciamento de grupos

O serviço de gerenciamento de grupos (GMS) é um serviço em processo que oferece monitoramento de cluster, associação de cluster e serviços de comunicação de grupos. O GMS realiza as seguintes funções:

  • Notifica os módulos registrados em uma instância do Servidor GlassFish quando uma ou mais instâncias do membro são adicionadas ou removidas de um cluster, ou quando tais instâncias sejam suspeitas de ter falhas ou realmente tenham falhas.

  • Oferece a capacidade de enviar e receber mensagens entre um grupo de processos.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Selecione a configuração do cluster que está sendo alterado.
  3. No nó da configuração, selecione o nó Serviço de gerenciamento de grupos.

    A página Serviço de gerenciamento de grupos é exibida.

  4. No campo Maximum Missed Heartbeats (Máximo de pulsações perdidas), insira o número máximo de tentativas de monitoramento antes que o GMS confirme que há uma suspeita de falha.

    O padrão é 3.

  5. No campo Heartbeat Frequency (Frequência das pulsações), insira o período de tempo entre as tentativas de monitoramento.

    O padrão é 2000 milissegundos.

  6. No campo Group Discovery Timeout (Tempo limite da descoberta de grupo), insira a quantidade de tempo que o GMS espera a descoberta de outros membros do grupo.

    O padrão é 5000 milissegundos.

  7. No campo Failure Verification Wait Time (Tempo de espera para a seleção da falha), insira o período de tempo após o qual uma suspeita de falha é marcada como verificada.

    O padrão é 1500 milissegundos.

  8. Para restaurar os valores padrão, clique no botão Carregar padrões.
  9. (Opcional) Na seção Propriedades adicionais, especifique essas propriedades.

    O Servidor GlassFish não define propriedades adicionais do GMS.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  10. Clique em Salvar.
