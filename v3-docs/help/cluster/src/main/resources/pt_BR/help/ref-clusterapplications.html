Aplicativos

Utilize a página Aplicativos para realizar as seguintes tarefas no cluster atual:

  • Exibindo e gerenciando aplicativos que já estão implantados no cluster

  • Implantando mais aplicativos no cluster

O campo Nome do cluster é um campo de somente leitura que exibe o nome do cluster atual.

A página Aplicativos exibe uma lista dos aplicativos que estão implantados no cluster atual. Para cada aplicativo, as seguintes informações são fornecidas.

Nome

O nome do aplicativo.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois pontos (:). Para obter mais informações sobre versões de aplicativos e módulos, consulte Module and Application Versions no Oracle GlassFish Server 3.1 Application Deployment Guide .

Ativo

Indicação de que o aplicativo está ativo:

  • Se o aplicativo estiver ativado, uma marca de seleção (✓) é exibida.

  • Se o aplicativo estiver desativado, um xis (✓) é exibido.

Balanceador de carga ativado

Indicação de que o aplicativo está ativado para balanceamento de carga:

  • Se o aplicativo estiver ativado para balanceamento de carga, uma marca de seleção (✓) é exibida.

  • Se o aplicativo estiver desativado para balanceamento de carga, um xis (✓) é exibido.

Mecanismos

Os tipos de contêineres que o aplicativo utiliza. Os tipos de contêineres podem ser:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — o contêiner para Contextos e Injeção de dependência para aplicativos da plataforma Java EE

A tabela Aplicativos implementados também contém as seguintes opções.

Implementar

Botão para implementar um aplicativo.

Cancelar implementação

Botão para cancelar a implementação de um ou mais aplicativos selecionados.

A desimplantação de um aplicativo remove o aplicativo de todos os destinos e do domínio. Quando um aplicativo é desimplantado, a aplicativo não estará mais na lista na página Aplicativos que exibe todos os aplicativos implantados no Servidor GlassFish.

Para remover aplicativos somente do destino atual, utilize o botão Remover desta página.

Remover

Botão para remover um ou mais aplicativos selecionados.

A remoção de um aplicativo remove o aplicativo somente do destino atual. Quando um aplicativo é removido de um destino, a aplicação continua estando na lista na página Aplicativos que exibe todos os aplicativos implantados no Servidor GlassFish. O aplicativo também permanece implantado nos outros destinos nos quais estava implantado.

Para remover aplicativos de todos os destino e do domínio, utilize o botão Desimplantar desta página.

Mais ações

Lista suspensa que realiza ações adicionais em um ou mais aplicativos selecionados.

Ativar

Ativa um ou mais aplicativos selecionados.

Desativar

Desativa um ou mais aplicativos selecionados.

Ativar balanceador de carga

Ativa um ou mais aplicativos selecionados para balanceamento de carga (LB).

Desativar balanceador de carga

Desativa um ou mais aplicativos selecionados para balanceamento de carga (LB).
