Recursos

Utilize a página Recursos para realizar as seguintes tarefas na instância independente selecionada do Servidor GlassFish:

  • Exibindo e gerenciando os recursos existentes da instância

  • Criando recursos adicionais para a instância

O campo Nome da instância é um campo de somente leitura que exibe o nome da instância atual.

A página Recursos exibe uma lista de recursos existentes da instância. Para cada recurso, as informações abaixo são fornecidas.

Nome do recurso

Nome atribuído ao recurso quando o recurso é criado. Ao clicar no nome, abre-se uma das seguintes páginas para editar o recurso, dependendo do tipo do recurso:

  • Edit JDBC Resource

  • Editar sessão JavaMail

  • Editar recurso personalizado

  • Novo recurso personalizado

  • Editar recurso externo

  • Editar recurso do conector

  • Editar recurso do objeto de admin

Ativo

Indicação de que o recurso está ativado:

  • Se o recurso estiver ativado, uma marca de seleção (✓) é exibida.

  • Se o recurso estiver desativado, um xis (✓) é exibido.

Tipo

O tipo do recurso de destino.

A tabela Recurso também contém as opções a seguir.

Ativar

Botão utilizado para ativar um ou mais recursos selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos selecionados.

Novo

Lista suspensa que cria um recurso do tipo selecionado. Ao selecionar um tipo de recurso na lista, uma das páginas seguintes se abre para a criação de um recurso do tipo selecionado:

  • New JDBC Resource

  • Nova fábrica de conexões JMS

  • Novo recurso de destino JMS

  • Nova sessão JavaMail

  • Novo recurso personalizado

  • Novo recurso externo

  • Novo recurso do conector

  • Novo recurso do objeto de admin

Filtro

Lista suspensa que filtra por tipo a lista de recursos exibidos.
