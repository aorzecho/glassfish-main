Aplicações

Utilize a página Aplicações para realizar as seguintes tarefas na instância standalone selecionada do GlassFish Server:

  • Exibindo e gerenciando aplicações que já estão implantadas na instância

  • Implantando mais aplicações na instância

O campo Nome da instância é um campo somente para leitura que exibe o nome da instância atual.

A página Aplicações exibe uma lista das aplicações que estão implantadas na instância selecionada. Para cada aplicação, as seguintes informações são fornecidas.

Nome

O nome da aplicação.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões de módulo e da aplicação, consulte Versões de Módulo e Aplicação em Oracle GlassFish Server Application Deployment Guide.

Ativo

Indicação de que a aplicação está ativada:

  • Se aplicação estiver ativada, uma marca de seleção (✓) será exibida.

  • Se a aplicação estiver desativada, um xis (✓) será exibido.

Balanceador de carga ativado

Indicação de que a aplicação está ativada para balanceamento de carga:

  • Se a aplicação estiver ativada para balanceamento de carga, uma marca de seleção (✓) será exibida.

  • Se a aplicação estiver desativada para balanceamento de carga, um xis (✓) será exibido.

Mecanismos

Os tipos de contêineres que a aplicação utiliza. Os tipos de contêineres podem ser:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — o contêiner para Contextos e Injeção de Dependência para Aplicações da Plataforma Java EE

A tabela Aplicações Implantadas também contém as opções a seguir.

Implantar

Botão para implantar uma aplicação.

Cancelar Implantação

Botão para cancelar a implantação de um ou mais aplicações selecionadas.

O cancelamento da implantação de uma aplicação remove a aplicação de todos os alvos e do domínio. Quando a implantação de uma aplicação é cancelada, a aplicação deixa de ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server.

Para remover aplicações somente do destino atual, utilize o botão Remover desta página.

Remover

Botão para remover um ou mais aplicações selecionadas.

A remoção de uma aplicação remove a aplicação somente do destino atual. Quando uma aplicação é removida de um alvo, a aplicação continua a ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server. A aplicação também permanece implantada nos outros alvos nos quais foi implantada.

Para remover aplicações de todos os destino e do domínio, utilize o botão Cancelar Implantação desta página.

Mais ações

Lista drop-down que realiza ações adicionais em um ou mais aplicações selecionadas.

Ativar

Ativa uma ou mais aplicações selecionadas.

Desativar

Desativa um ou mais aplicações selecionadas.

Ativar balanceador de carga

Ativa um ou mais aplicações selecionadas para balanceamento de carga (LB).

Desativar balanceador de carga

Desativa um ou mais aplicações selecionadas para balanceamento de carga (LB).
