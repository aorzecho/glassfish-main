Informações gerais

A página Informações gerais contém as seguintes opções.

Iniciar

Botão para iniciar a instância. Se a instância estiver em execução, este botão está desativado.

Interromper

Botão para interromper a instância. Se a instância estiver interrompida, este botão está desativado.

Reiniciar

Botão para reiniciar a instância. Se a instância estiver interrompida, este botão está desativado.

Exibir arquivos de log

Botão para exibir os arquivos de log da instância.

Recuperar transações

Botão para recuperar transações da instância.

Nome do cluster

Nome atribuído ao cluster quando o cluster é criado. Ao clicar no nome do cluster, a página Informações gerais se abre.

Nome da instância

Nome atribuído à instância quando a instância foi adicionada ao cluster. Este campo é somente leitura.

Status

Indica se a instância está em execução, se requer reinicialização para que seja atualizada com as alterações na configuração ou se não está em execução. Este campo é somente leitura.

JVM

Se você clicar em JVM Report, será aberta uma janela separada que exibe relatórios sobre o Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória e de coleta de erros,um relatório de carregamento de classe e um despejo de segmentos atual.

Nó

O nome do nó no qual a instância reside. Ao clicar no nome do nó, a página Editar nó se abre.

Configuração

Configuração nomeada que a instância faz referência. Ao clicar na configuração, a página Configurações da instância se abre.

HTTP Ports (Portas HTTP)

Lista separada por vírgula dos números das portas na qual a instância escuta os seguintes tipos de solicitações:

  • Solicitações de administração

  • Solicitações HTTP

  • Solicitações HTTPS

Este campo é somente leitura.

IIOP Ports (Portas IIOP)

Lista separada por vírgula dos números das portas que a instância utiliza nos seguintes tipos de conexões:

  • Conexão IIOP segura

  • Conexões IIOP seguras com autenticação do cliente

  • Conexões IIOP

Este campo é somente leitura.

Tarefas relacionadas

  • Para iniciar instâncias do Servidor GlassFish em cluster

  • Para interromper instâncias do Servidor GlassFish em cluster

  • Para exibir arquivos de log de uma instância do Servidor GlassFish em cluster

  • Para recuperar transações de uma instância do Servidor GlassFish em cluster
