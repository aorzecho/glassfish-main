Nova instância de servidor independente

Utilize a página Nova instância de servidor independente para criar uma instância independente do Servidor GlassFish.

A página Nova instância de servidor independente contém as seguintes opções.

Nome da instância

O nome da instância que está sendo criada.

O nome deve cumprir os seguintes requisitos:

  • O nome deve começar com uma letra, um número ou um sublinhado.

  • O nome pode conter somente os seguintes caracteres:

    • Letras minúsculas

    • Letras maiúsculas

    • Números

    • Hífen

    • Período

    • Sublinhado

  • O nome deve ser único no domínio e não deve ser o nome de outra instância, um cluster, uma configuração nomeada ou um nó do Servidor GlassFish.

  • O nome não deve ser domain, server, nem nenhuma outra palavra-chave que esteja reservada pelo Servidor GlassFish.

Nó

Lista suspensa dos nós existentes que definem os hosts nos quais a instância pode estar. É necessário selecionar um nó na lista. Caso a instância deva ser criada no host onde o servidor de administração do domínio (DAS) estiver sendo executado, selecione o nó predefinido localhost-domínio.

Configuração

Lista suspensa das configurações nomeadas existentes que a instância pode utilizar. É necessário selecionar uma configuração na lista. A instância utilizará a configuração selecionada quando a instância for criada.

Criar cópia da configuração selecionada

Se esta opção for selecionada, a configuração selecionada será copiada quando a instância for criada. À cópia da configuração é atribuído o nome nome-da-instância-config, em que nome-da-instância corresponde ao nome da instância conforme digitado no campo Nome da instância.

Se default-config estiver selecionado na lista suspensa Configuração, é necessário selecionar esta opção. A configuração default-config pode ser apenas copiada, não referenciada.

Referenciar a configuração selecionada

Se esta opção for selecionada, a instância utilizará a configuração nomeada existente especificada.

Se default-config estiver selecionado na lista suspensa Configuração, esta opção não deve ser selecionada. A configuração default-config não pode ser referenciada, somente copiada.
