Para Modificar a Configuração de um Cluster

  1. Na árvore de navegação, expanda o nó Clusters.

  2. Selecione o nó do cluster.

  3. Clique no nome da configuração.

    A página Configuração do cluster se abre.

  4. Na página Configuração, selecione o componente da configuração a ser modificado.

    É exibida a página para edição do componente selecionado da configuração.

  5. Na página para edição do componente selecionado, faça as alterações necessárias e salve-as.

    Para obter detalhes, consulte a ajuda on-line da página de edição de componentes selecionados.
