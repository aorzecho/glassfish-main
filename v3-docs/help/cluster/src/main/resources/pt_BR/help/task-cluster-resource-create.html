Para criar um recurso para um cluster

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Recursos.

    A página Recursos do cluster se abre.

  4. No menu suspenso Novo, selecione o tipo do recurso a ser criado.

    É exibida a página para a criação de um recurso do tipo selecionado.

  5. Na página para edição de um recurso do tipo selecionado, faça as alterações necessárias e salve-as.

    Para obter detalhes, consulte a ajuda on-line da página de criação de recursos do tipo selecionado.

    Observação - Ao criar um recurso, certifique-se de ter especificado o cluster como destino.
