Para editar um recurso de um cluster

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Recursos.

    A página Recursos do cluster se abre.

  4. Clique no nome do recurso.

    A página para a edição do recurso se abre.

  5. Na página para edição do recurso, faça as alterações necessárias e salve-as.

    Para obter detalhes, consulte a ajuda on-line da página de edição de recursos.
