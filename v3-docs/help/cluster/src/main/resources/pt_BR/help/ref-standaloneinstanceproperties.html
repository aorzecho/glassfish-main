Propriedades da instância

A página Propriedades da instância exibe uma lista das propriedades que estão definidas para a instância atual do Servidor GlassFish. Estas propriedades acrescentam informações opcionais de configuração sobre a instância.

O campo Nome da instância é um campo de somente leitura que exibe o nome da instância atual.

São exibidas as seguintes informações para cada propriedade:

Nome

O nome da propriedade.

Valor

Valor da propriedade definida para a instância selecionada.

Descrição

Descrição textual que oferece mais informações sobre a propriedade.

A tabela Propriedades adicionais também contém as seguintes opções.

Adicionar propriedade

Botão para adicionar uma propriedade. Se clicar neste botão, uma linha é adicionada à tabela Propriedades adicionais.

Excluir propriedades

Botão para excluir uma ou mais propriedades selecionadas. Qualquer propriedade excluída volta ao seu valor padrão ou, se nenhum valor tiver sido definido, fica com valor indefinido.

O Servidor GlassFish define as seguintes propriedades da instância:

rendezvousOccurred

Especifica se a instância contatou o servidor de administração de domínio (DAS). Os valores possíveis são os seguintes:

verdadeiro

A instância contatou o DAS.

falso

A instância não contatou o DAS.
