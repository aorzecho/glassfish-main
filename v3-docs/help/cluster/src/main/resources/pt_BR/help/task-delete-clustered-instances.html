Para excluir instâncias do Servidor GlassFish em cluster

Cuidado - Se estiver utilizando um cluster Java Message Service (JMS) com um broker mestre, não exclua a instância que está associada ao broker mestre. Se esta instância deve ser excluída, utilize o subcomando change-master-broker(1) para atribuir o broker mestre a outra instância.

  1. Na árvore de navegação, expanda o nó Clusters.
  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias do servidor em clusters do cluster se abre.

  4. Para cada instância que estiver sendo excluída, selecione a caixa de seleção ao lado da instância na tabela Instâncias do servidor.
  5. Clique em Excluir.

    Você é solicitado a confirmar que deseja excluir as instâncias.

  6. Confirme que deseja excluir as instâncias.

    As instâncias são excluídas e removidas da tabela de instâncias da página.
