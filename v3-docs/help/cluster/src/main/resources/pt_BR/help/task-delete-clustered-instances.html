Para Deletar Instâncias Clusterizadas do GlassFish Server

Cuidado:

Se estiver utilizando um cluster Java Message Service (JMS) com um broker mestre, não delete a instância que está associada ao broker mestre. Se esta instância precisar ser deletada, use o subcomando change-master-broker(1) para designar o broker mestre a outra instância.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandido, selecione o nó do cluster.

    A página Informações gerais do cluster se abre.

  3. Na página Informações gerais, clique na guia Instâncias.

    A página Instâncias Clusterizadas do Servidor relativa ao cluster se abre.

  4. Para cada instância que estiver sendo deletada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  5. Clique em Deletar.

    Você é solicitado a confirmar que deseja deletar as instâncias.

  6. Confirme que deseja deletar as instâncias.

    As instâncias são deletadas e removidas da tabela de instâncias da página.
