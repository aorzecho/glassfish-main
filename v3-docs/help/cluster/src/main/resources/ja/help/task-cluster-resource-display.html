クラスタのリソースを一覧表示する

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「リソース」タブをクリックします。

    クラスタの「リソース」ページが開きます。

  4. (省略可能) 特定のタイプのリソースのみを表示するには、「フィルタ」プルダウンリストから、表示するリソースのタイプを選択します。
