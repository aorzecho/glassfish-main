クラスタのリソースを編集する

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「リソース」タブをクリックします。

    クラスタの「リソース」ページが開きます。

  4. リソース名をクリックします。

    リソースを編集するページが開きます。

  5. リソースを編集するページで、必要な変更を加え、変更を保存します。

    詳細は、リソースを編集するページのオンラインヘルプを参照してください。
