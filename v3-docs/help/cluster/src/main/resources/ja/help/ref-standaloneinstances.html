スタンドアロンサーバーインスタンス

「スタンドアロンサーバーインスタンス」ページを使用して、スタンドアロン GlassFish Server インスタンスを作成および管理します。

「スタンドアロンサーバーインスタンス」ページには、ドメイン内のすべてのスタンドアロンインスタンスの一覧が表示されます。インスタンスごとに、次の情報が表示されます。

名前

インスタンスが作成されたときにインスタンスに割り当てられた名前。名前をクリックすると、インスタンスの「一般情報」ページが表示されます。

LB の重み

インスタンスの負荷分散の重みを表す整数。

負荷分散の重みは、クラスタに対するすべての要求のうちインスタンスが処理する必要がある割合を決定します。

たとえば、2 つのインスタンスで構成されるクラスタで、一方のインスタンスで 4 つの要求のうちの 1 つを処理し、もう一方のインスタンスで残りの 3 つの要求を処理しなければならない場合があります。この場合、4 つの要求のうちの 1 つを処理するインスタンスの重みを 1 に設定し、もう一方のインスタンスの重みを 3 に設定します。パーセントを使用する場合は、インスタンスの重みをそれぞれ 25 と 75 に設定します。

デフォルトの重みは 100 です。

設定

インスタンスが作成されたときにインスタンスに対して選択された設定。設定をクリックすると、インスタンスの「設定」ページが表示されます。

ノード

インスタンスが配置されているノードの名前。名前をクリックすると、ノードの「ノードの編集」ページが開きます。

ステータス

インスタンスが実行中かどうかが表示されます。

「サーバーインスタンス」の表には、次のオプションも表示されます。

新規

スタンドアロンインスタンスを作成するボタン。

削除

選択した 1 つまたは複数のスタンドアロンインスタンスを削除するボタン。

起動

選択した 1 つまたは複数のスタンドアロンインスタンスを起動するボタン。

停止

選択した 1 つまたは複数のスタンドアロンインスタンスを停止するボタン。
