クラスタシステムプロパティー

「クラスタシステムプロパティー」ページには、選択したクラスタに対して設定されている Java システムプロパティーの一覧が表示されます。Java システムプロパティーは、GlassFish サーバー の起動時に Java アプリケーション起動用ウィンドウの -D オプションによって、Java アプリケーション起動用ウィンドウに渡されます。

「クラスタ名」フィールドは、選択したクラスタの名前が表示される読み取り専用のフィールドです。

プロパティーごとに、次の情報が表示されます。

インスタンス変数名

システムプロパティーの名前。

現在の値

プロパティーに対して現在設定されている値。このフィールドは読み取り専用です。

オーバーライド値

変更の保存後にプロパティーに対して設定される値。

「追加のプロパティー」の表には、次のオプションも表示されます。

プロパティーを追加

プロパティーを追加するボタン。このボタンをクリックすると、「追加のプロパティー」の表に行が追加されます。

プロパティーを削除

選択した 1 つまたは複数のプロパティーを削除するボタン。削除されたプロパティーはデフォルト値に戻ります。デフォルト値が設定されていない場合は未定義になります。
