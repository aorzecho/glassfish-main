ノードを削除する

ノードを削除すると、ノードはドメイン管理サーバー (Domain Administration Server、DAS) の設定から削除されます。ノードのディレクトリとファイルは、ノードに存在する最後の GlassFish サーバー インスタンスが削除されるときに削除されます。

始める前に

GlassFish サーバー インスタンスが削除されるノードに存在しないことを確認します。

  1. ナビゲーションツリーで、「ノード」ノードを選択します。

    「ノード」ページが開きます。

  2. 削除するノードごとに、「ノード」の表のノードの横にあるチェックボックスを選択します。

    注 - 定義済みノード localhost-domain は削除できません。

  3. 「削除」をクリックします。

    ノードを削除するかどうかを確認するメッセージが表示されます。

  4. ノードを削除することを確認します。

    ノードは削除されて、「ノード」ページのノードの表からも削除されます。
