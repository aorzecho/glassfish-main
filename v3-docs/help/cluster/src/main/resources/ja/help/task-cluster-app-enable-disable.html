クラスタに配備されているアプリケーションを有効または無効にする

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「アプリケーション」タブをクリックします。

    クラスタの「アプリケーション」ページが開きます。

  4. アプリケーションの横にあるチェックボックスを選択します。
  5. 「追加の操作」プルダウンメニューから、該当するコマンドを選択します。
    • アプリケーションを有効にする場合は、「有効」を選択します。
    • アプリケーションを無効にする場合は、「無効」を選択します。
