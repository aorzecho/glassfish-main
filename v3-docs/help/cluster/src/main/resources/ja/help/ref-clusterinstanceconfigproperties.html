インスタンスシステムプロパティー

「インスタンスシステムプロパティー」ページには、現在の GlassFish サーバー インスタンスに対して設定されている Java システムプロパティーの一覧が表示されます。Java システムプロパティーは、GlassFish サーバー の起動時に Java アプリケーション起動用ウィンドウの -D オプションによって、Java アプリケーション起動用ウィンドウに渡されます。

これらのプロパティーは、インスタンスの設定にあるポート設定のプロパティー定義をオ?バーライドします。2 つのクラスタ化されたインスタンスが同じホスト上に存在する場合などでは、定義済みのポート設定はオーバーライドされます。このような状況では、両方のインスタンスが同じ設定を共有するため、一方のインスタンスのポート設定はオーバーライドされる必要があります。

設定内の定義済みポート設定については、「定義済みの設定システムプロパティー」を参照してください。

「インスタンス名」フィールドは、現在のインスタンスの名前が表示される読み取り専用のフィールドです。

プロパティーごとに、次の情報が表示されます。

インスタンス変数名

システムプロパティーの名前。

現在の値

プロパティーに対して現在設定されている値。このフィールドは読み取り専用です。

オーバーライド値

変更の保存後にプロパティーに対して設定される値。

「追加のプロパティー」の表には、次のオプションも表示されます。

プロパティーを追加

プロパティーを追加するボタン。このボタンをクリックすると、「追加のプロパティー」の表に行が追加されます。

プロパティーを削除

選択した 1 つまたは複数のプロパティーを削除するボタン。削除されたプロパティーはデフォルト値に戻ります。デフォルト値が設定されていない場合は未定義になります。
