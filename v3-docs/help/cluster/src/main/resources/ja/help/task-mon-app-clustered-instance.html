クラスタ化されたGlassFish Serverインスタンスのアプリケーション監視データを表示する

開始する前の準備

監視データを表示する前に、監視サービスを構成する必要があります。

  1. ナビゲーション・ツリーで、「クラスタ」ノードを展開します。

  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「インスタンス」タブをクリックします。

    クラスタの「クラスタ化されたサーバー・インスタンス」ページが開きます。

  4. アプリケーション監視データを表示するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  5. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  6. 「表示」ドロップダウン・リストから、監視データを表示するアプリケーションを選択します。

  7. 表示される監視データを更新するには、「リフレッシュ」をクリックします。
