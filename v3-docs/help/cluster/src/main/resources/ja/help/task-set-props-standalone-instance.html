スタンドアロン GlassFish サーバー インスタンスの Instance-Only プロパティーを設定する

GlassFish サーバー インスタンスのプロパティーは、インスタンスに関するオプションの設定情報を追加します。

  1. ナビゲーションツリーで、「スタンドアロンインスタンス」ノードを選択します。

    「スタンドアロンサーバーインスタンス」ページが開きます。

  2. Instance-Only プロパティーを変更するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「プロパティー」タブをクリックします。
  4. 必要な場合は、「インスタンスプロパティー」タブをクリックします。

    「インスタンスプロパティー」ページが開きます。

    このページには、選択した GlassFish サーバー インスタンスに対して設定されているプロパティーの一覧が表示されます。

  5. 変更する既存のプロパティーごとに、プロパティーの「値」フィールドに新しい値を入力します。
  6. 必要な追加プロパティーの値を設定します。
    1. 「追加プロパティー」表で、「プロパティーを追加」をクリックします。

      「追加のプロパティー」の表に行が追加されます。

    2. 「名前」フィールドにプロパティー名を入力します。
    3. 「値」フィールドに値を入力します。
    4. 「説明」フィールドに、プロパティーの簡単な説明を入力します。
  7. 「保存」をクリックします。
