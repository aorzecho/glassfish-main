新しいスタンドアロンサーバーインスタンス

「新しいスタンドアロンサーバーインスタンス」ページを使用して、スタンドアロン GlassFish サーバー インスタンスを作成します。

「新しいスタンドアロンサーバーインスタンス」ページには、次のオプションが含まれます。

インスタンス名

作成されるインスタンスの名前。

名前は次の要件を満たしてください。

  • 名前は英数字または下線で始めます。

  • 名前に含めることができるのは次の文字だけです。

    • 小文字

    • 大文字

    • 数値

    • ハイフン

    • ピリオド

    • 下線

  • 名前はドメイン内で一意にします。また、別の GlassFish サーバー インスタンス、クラスタ、名前付き設定、またはノードの名前にすることはできません。

  • ドメイン、サーバー、または GlassFish サーバー で予約されているほかのキーワードを名前に使用してはいけません。

ノード

インスタンスを配置できるホストを定義する既存のノードのドロップダウンリスト。リストからいずれかのノードを選択する必要があります。ドメイン管理サーバー (Domain Administration Server、DAS) が実行されているホストにインスタンスを作成する場合は、定義済みノード localhost-domain を選択します。

設定

インスタンスで使用できる既存の名前付き設定のドロップダウンリスト。ドロップダウンリストからいずれかの設定を選択する必要があります。インスタンスでは、インスタンスの作成時に選択した設定が使用されます。

選択している設定のコピーの作成

このオプションをオンにすると、インスタンスの作成時に選択した設定がコピーされます。設定のコピーには、instance-name-config という名前が割り当てられます。instance-name は、「インスタンス名」フィールドに入力されているインスタンスの名前です。

「設定」ドロップダウンリストから「デフォルト設定」を選択した場合は、このオプションをオンにする必要があります。「デフォルト設定」はコピーだけできます。参照することはできません。

選択している設定への参照

このオプションをオンにすると、指定した既存の名前付き設定が使用されます。

「設定」ドロップダウンリストから「デフォルト設定」を選択した場合は、このオプションをオンにしないでください。「デフォルト設定」はコピーだけできます。参照することはできません。
