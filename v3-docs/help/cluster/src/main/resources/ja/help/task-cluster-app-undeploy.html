クラスタからアプリケーションの配備を取り消す

アプリケーションの配備取消しでは、すべてのターゲットおよびドメインからアプリケーションを削除します。アプリケーションの配備が取り消された場合、そのアプリケーションは GlassFish サーバー に配備されているすべてのアプリケーションが表示される「アプリケーション」ページに表示されなくなります。

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「アプリケーション」タブをクリックします。

    クラスタの「アプリケーション」ページが開きます。

  4. 配備を取り消すアプリケーションごとに、「配備されたアプリケーション」の表のアプリケーションの横にあるチェックボックスを選択します。
  5. 「配備取消し」をクリックします。

    アプリケーションの配備を取り消すかどうかを確認するメッセージが表示されます。

  6. アプリケーションの配備を取り消すことを確認します。

    アプリケーションの配備が取り消され、ページの配備されたアプリケーションの表からアプリケーションが削除されます。
