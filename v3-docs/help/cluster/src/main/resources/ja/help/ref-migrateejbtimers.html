EJB タイマーを移行

「EJB タイマーを移行」ページには、次のオプションが含まれます。

ソース

クラスタ内の停止している GlassFish サーバー インスタンスのドロップダウンリスト。EJB タイマーは、停止しているインスタンスからしか移行できません。

送信先

クラスタ内の実行中の GlassFish サーバー インスタンスのドロップダウンリスト。EJB タイマーは、実行中のインスタンスにしか移行できません。

注 - クラスタ内のインスタンスがすべて停止している場合は、EJB タイマーを移行できる実行中のインスタンスがないという警告のみが表示されます。
