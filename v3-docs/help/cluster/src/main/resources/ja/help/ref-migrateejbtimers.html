EJBタイマーを移行

「EJBタイマーを移行」ページには、次のオプションが含まれます。

ソース

クラスタ内の停止しているGlassFish Serverインスタンスのドロップダウン・リスト。EJBタイマーは、停止しているインスタンスからしか移行できません。

宛先

クラスタ内の実行中のGlassFish Serverインスタンスのドロップダウン・リスト。EJBタイマーは、実行中のインスタンスにしか移行できません。

注意:

クラスタ内のインスタンスがすべて停止している場合は、EJBタイマーを移行できる実行中のインスタンスがないという警告のみが表示されます。

「EJBタイマーを移行」ページには、EJBタイマーの表が含まれ、次の情報が表示されます。

サーバー名

クラスタ内のGlassFish Serverインスタンスのリスト。

タイマー・カウント

各インスタンスの永続タイマーの数。
