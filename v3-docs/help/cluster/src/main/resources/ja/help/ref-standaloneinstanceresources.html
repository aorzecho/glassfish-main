リソース

「リソース」ページを使用して、選択したスタンドアロンGlassFish Serverインスタンスで次のタスクを実行します。

  • インスタンスの既存のリソースの表示と管理

  • インスタンスの追加のリソースの作成

「インスタンス名」フィールドは、現在のインスタンスの名前が表示される読取り専用のフィールドです。

「リソース」ページには、インスタンスの既存のリソースのリストが表示されます。リソースごとに、次の情報が表示されます。

リソース名

リソースが作成されたときにリソースに割り当てられた名前。名前をクリックすると、リソースのタイプに応じて、次のいずれかのページが開き、該当するタイプのリソースを編集できます。

  • JDBCリソースを編集

  • JavaMailセッションを編集

  • カスタム・リソースを編集

  • 新しいカスタム・リソース

  • 外部リソースを編集

  • コネクタ・リソースを編集

  • 管理オブジェクト・リソースを編集

有効

リソースが有効かどうかが表示されます。

  • リソースが有効な場合は、チェックマーク(✓)が表示されます。

  • リソースが無効な場合は、十字記号(✗)が表示されます。

タイプ

リソースのタイプです。

「リソース」の表には、次のオプションも表示されます。

有効

選択した1つまたは複数のリソースを有効にするボタン。

無効

選択した1つまたは複数のリソースを無効にするボタン。

新規

選択したタイプのリソースを作成するドロップダウン・リスト。リストからリソース・タイプを選択すると、次のいずれかのページが開き、選択したタイプのリソースを作成できます。

  • 新しいJDBCリソース

  • 新しいJMS接続ファクトリ

  • 新しいJMS宛先リソース

  • 新しいJavaMailセッション

  • 新しいカスタム・リソース

  • 新しい外部リソース

  • 新しいコネクタ・リソース

  • 新しい管理オブジェクト・リソース

フィルタ

表示されるリソースのリストをタイプでフィルタするドロップダウン・リスト。
