リソース

「リソース」ページを使用して、選択したスタンドアロン GlassFish サーバー インスタンスで次のタスクを実行します。

  • インスタンスの既存のリソースの表示と管理

  • インスタンスの追加のリソースの作成

「インスタンス名」フィールドは、現在のインスタンスの名前が表示される読み取り専用のフィールドです。

「リソース」ページには、インスタンスの既存のリソースの一覧が表示されます。リソースごとに、次の情報が表示されます。

リソース名

リソースが作成されたときにリソースに割り当てられた名前。名前をクリックすると、リソースのタイプに応じて、次のいずれかのページが表示され、該当するタイプのリソースを編集できます。

  • JDBC リソースを編集

  • JavaMail セッションを編集

  • カスタムリソースを編集

  • 新しいカスタムリソース

  • 外部リソースを編集

  • コネクタリソースを編集

  • 管理オブジェクトリソースを編集

使用可能

リソースが有効かどうかが表示されます。

  • リソースが有効な場合は、チェックマーク (✓) が表示されます。

  • リソースが無効な場合は、十字記号 (✗) が表示されます。

タイプ

リソースの型です。

「リソース」の表には、次のオプションも表示されます。

有効

選択した 1 つまたは複数のリソースを有効にするボタン。

無効

選択した 1 つまたは複数のリソースを無効にするボタン。

新規

選択したタイプのリソースを作成するドロップダウンリスト。リストからリソースタイプを選択すると、次のいずれかのページが開き、選択したタイプのリソースを作成できます。

  • 新しい JDBC リソース

  • 新しい JMS 接続ファクトリ

  • 新しい JMS 送信先リソース

  • 新しい JavaMail セッション

  • 新しいカスタムリソース

  • 新しい外部リソース

  • 新しいコネクタリソース

  • 新しい管理オブジェクトリソース

フィルタ

表示されるリソースの一覧をタイプでフィルタするドロップダウンリスト。

関連操作

  • 「スタンドアロン GlassFish サーバー インスタンスのリソースを作成する」

  • 「スタンドアロン GlassFish サーバー インスタンスのリソースを一覧表示する」

  • 「スタンドアロン GlassFish サーバー インスタンスのリソースを有効または無効にする」

  • 「スタンドアロン GlassFish サーバー インスタンスのリソースを編集する」
