スタンドアロン GlassFish Server インスタンスの仮想サーバー監視データを表示する

始める前に

監視データを表示する前に、監視サービスを設定する必要があります。

  1. ナビゲーションツリーで、「スタンドアロンインスタンス」ノードを選択します。

    「スタンドアロンサーバーインスタンス」ページが開きます。

  2. 仮想サーバー監視データを表示するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  4. 「サーバー」タブをクリックします。

    「サーバーの監視」ページが開きます。

  5. 「表示」ドロップダウンリストから、監視データを表示するコンポーネントを選択します。
  6. 表示される監視データを更新するには、「再表示」をクリックします。
