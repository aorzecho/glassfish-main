スタンドアロンGlassFish Serverインスタンスの仮想サーバー監視データを表示する

開始する前の準備

監視データを表示する前に、監視サービスを構成する必要があります。

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 仮想サーバー監視データを表示するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  4. 「サーバー」タブをクリックします。

    「サーバーの監視」ページが開きます。

  5. 「表示」ドロップダウン・リストから、監視データを表示するコンポーネントを選択します。

  6. 表示される監視データを更新するには、「リフレッシュ」をクリックします。
