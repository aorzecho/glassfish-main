クラスタを停止する

クラスタを停止すると、クラスタ内で実行中のすべての GlassFish Server インスタンスが停止します。

  1. ナビゲーションツリーで、「クラスタ」ノードを選択します。

    「クラスタ」ページが開きます。

  2. 「クラスタ」ページで、クラスタ名の横にあるチェックボックスを選択します。
  3. 「クラスタの停止」をクリックします。

    クラスタを停止するかどうかを確認するメッセージが表示されます。

  4. クラスタを停止することを確認します。

    クラスタが停止します。インスタンスが停止すると、「クラスタ」ページでクラスタ内の各インスタンスの状態が更新されます。
