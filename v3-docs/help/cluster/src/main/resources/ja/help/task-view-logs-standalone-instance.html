スタンドアロンGlassFish Serverインスタンスのログ・ファイルを表示する

開始する前の準備

ログ・ファイルを表示するインスタンスが実行されている必要があります。

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. ログ・ファイルを表示するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「ログ・ファイルを表示」をクリックします。

    「ログ・ビューア」ウィンドウが開きます。

    注意:

    インスタンスは実行しておいてください。インスタンスが実行されていない場合、「ログ・ファイルを表示」ボタンは非アクティブになっています。
