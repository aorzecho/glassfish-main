スタンドアロンGlassFish Serverインスタンスのリソースを作成する

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 「スタンドアロン・サーバー・インスタンス」ページで、インスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「リソース」タブをクリックします。

    インスタンスの「リソース」ページが開きます。

  4. 「新規作成」プルダウン・メニューから、作成するリソースのタイプを選択します。

    選択したタイプのリソースを作成するページが開きます。

  5. 選択したタイプのリソースを作成するページで、必要な変更を加え、変更を保存します。

    詳細は、選択したタイプのリソースを作成するページのオンライン・ヘルプを参照してください。

    注意:

    リソースを作成するときに、インスタンスをターゲットとして指定していることを確認します。
