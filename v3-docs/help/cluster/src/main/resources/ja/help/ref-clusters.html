クラスタ

「クラスタ」ページには、ドメイン内のクラスタのリストが表示されます。クラスタごとに、次の情報が表示されます。

名前

クラスタが作成されたときにクラスタに割り当てられた名前。名前をクリックすると、クラスタの「一般情報」ページが開きます。

構成

クラスタの作成時にクラスタに対して選択された構成。構成をクリックすると、クラスタの「構成」ページが開きます。

インスタンス

クラスタ内の各インスタンスの名前およびステータス。インスタンスの名前をクリックすると、インスタンスの「一般情報」ページが開きます。

「クラスタ」の表には、次のオプションも表示されます。

新規

クラスタを作成するボタン。

削除

クラスタを削除するボタン。

クラスタの起動

クラスタを起動するボタン。

クラスタの停止

クラスタを停止するボタン。
