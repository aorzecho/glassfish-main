スタンドアロン GlassFish サーバー インスタンスからアプリケーションを削除する

アプリケーションの削除では、現在のターゲットからのみアプリケーションを削除します。ターゲットからアプリケーションを削除しても、そのアプリケーションは GlassFish サーバー に配備されているすべてのアプリケーションが表示される「アプリケーション」ページに引き続き表示されます。配備が行われたほかのターゲットのアプリケーションも配備されたままとなります。

  1. ナビゲーションツリーで、「スタンドアロンインスタンス」ノードを選択します。

    「スタンドアロンサーバーインスタンス」ページが開きます。

  2. 「スタンドアロンサーバーインスタンス」ページで、インスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. インスタンスの「一般情報」ページで、「アプリケーション」タブをクリックします。

    インスタンスの「アプリケーション」ページが開きます。

  4. 削除するアプリケーションごとに、「配備されたアプリケーション」の表のアプリケーションの横にあるチェックボックスを選択します。
  5. 「削除」をクリックします。

    アプリケーションを削除するかどうかを確認するメッセージが表示されます。

  6. アプリケーションを削除することを確認します。

    ページの配備されたアプリケーションの表からアプリケーションが削除されます。
