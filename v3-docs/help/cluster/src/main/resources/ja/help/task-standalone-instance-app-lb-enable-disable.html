スタンドアロンGlassFish Serverインスタンスにデプロイされたアプリケーションのロード・バランシングを有効または無効にする

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 「スタンドアロン・サーバー・インスタンス」ページで、インスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. インスタンスの「一般情報」ページで、「アプリケーション」タブをクリックします。

    インスタンスの「アプリケーション」ページが開きます。

  4. アプリケーションの横にあるチェックボックスを選択します。

  5. 「追加の操作」プルダウン・メニューから、該当するコマンドを選択します。

    • アプリケーションのロード・バランシングを有効にする場合は、「ロード・バランサを有効」を選択します。

    • アプリケーションのロード・バランシングを無効にする場合は、「ロード・バランサを無効」を選択します。
