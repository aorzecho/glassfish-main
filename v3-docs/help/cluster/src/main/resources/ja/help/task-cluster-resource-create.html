クラスタのリソースを作成する

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「リソース」タブをクリックします。

    クラスタの「リソース」ページが開きます。

  4. 「新規作成」プルダウンメニューから、作成するリソースのタイプを選択します。

    選択したタイプのリソースを作成するページが開きます。

  5. 選択したタイプのリソースを作成するページで、必要な変更を加え、変更を保存します。

    詳細は、選択したタイプのリソースを作成するページのオンラインヘルプを参照してください。

    注 - リソースを作成するときに、クラスタをターゲットとして指定していることを確認します。
