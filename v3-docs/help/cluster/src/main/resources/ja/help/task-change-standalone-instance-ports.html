スタンドアロンGlassFish Serverインスタンスのポート番号を変更する

スタンドアロンGlassFish Serverインスタンスのポート番号は、インスタンスが参照する構成で初期設定されています。同じ構成を参照する2つのインスタンスが同じホスト上に存在する場合などは、これらのポート番号を変更する必要があります。このような状況では、両方のインスタンスが同じ構成を参照するため、一方のインスタンスのポート番号を変更する必要があります。

インスタンスのポート番号は、Javaシステム・プロパティとして保存されています。これらのプロパティは、GlassFish Serverの起動時にJavaアプリケーション・ランチャの-Dオプションによって、Javaアプリケーション・ランチャに渡されます。

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. ポート番号を変更するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「プロパティ」タブをクリックします。

    「インスタンス・システム・プロパティ」ページが開きます。

    このページには、インスタンスの次のシステム・プロパティの現在の値が表示されます。

    • インスタンスの構成で定義されたポート番号

    • インスタンスに対して定義されているカスタム・プロパティ(存在する場合)

  4. 変更する既存のプロパティごとに、プロパティの「オーバーライド値」フィールドに新しい値を入力します。

  5. 必要な追加プロパティの値を設定します。

    1. 「追加プロパティ」表で、「プロパティを追加」をクリックします。

      「追加のプロパティ」の表に行が追加されます。

    2. 「インスタンス変数名」フィールドにプロパティ名を入力します。

    3. 「オーバーライド値」フィールドに値を入力します。

  6. 「保存」をクリックします。
