ノード

「ノード」ページには、ドメイン内のノードの一覧が表示されます。ノードごとに、次の情報が表示されます。

名前

ノードが作成されたときにノードに割り当てられた名前。名前をクリックすると、ノードの「ノードの編集」ページが開きます。

ノードホスト

ノードが表すホストの名前。

タイプ

ノードの型。次のいずれかです。

CONFIG

このノードは、リモート通信をサポートしません。

SSH

このノードは、Secure Shell (SSH) 経由の通信をサポートします。

インスタンス

ノード上に存在するインスタンスの一覧。この一覧には、各インスタンスが実行されているかどうかが表示されます。

  • インスタンスが実行されている場合は、チェックマーク (✓) が表示されます。

  • インスタンスが停止している場合は、感嘆符 (!) が表示されます。

インスタンスの名前をクリックすると、インスタンスの「一般情報」ページが表示されます。

「ノード」の表には、次のオプションも含まれます。

新規

ノードを作成するボタン。

削除

選択した 1 つまたは複数のノードを削除するボタン。

関連操作

  • 「ノードを作成する」

  • 「ノードを削除する」

  • 「現在のドメイン内にあるすべてのノードを一覧表示する」

  • 「ノードを編集する」
