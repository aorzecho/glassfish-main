クラスタの設定を変更する

  1. ナビゲーションツリーで、「クラスタ」ノードを展開します。
  2. クラスタのノードを選択します。
  3. 設定の名前をクリックします。

    クラスタの「設定」ページが開きます。

  4. 「設定」ページで、変更する設定のコンポーネントを選択します。

    選択した設定のコンポーネントを編集するページが開きます。

  5. 選択したコンポーネントを編集するページで、必要な変更を加え、変更を保存します。

    詳細は、選択したコンポーネントを編集するページのオンラインヘルプを参照してください。
