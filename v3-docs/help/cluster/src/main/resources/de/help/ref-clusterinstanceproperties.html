Instanzeigenschaften

Auf der Seite "Instanzeigenschaften" werden die Eigenschaften aufgelistet, die für die aktuelle GlassFish-Server-Instanz festgelegt sind. Diese Eigenschaften enthalten optionale Konfigurationsinformationen zur Instanz.

Das Feld "Instanzname" ist ein schreibgeschütztes Feld, in dem der Name der aktuellen Instanz angezeigt wird.

Für jede Instanz werden folgende Informationen angezeigt:

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft, der für die ausgewählte Instanz festgelegt ist.

Beschreibung

Eine Textbeschreibung mit weiteren Informationen zur Eigenschaft.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Eigenschaften. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.

GlassFish-Server definiert die folgenden Instanzeigenschaften:

rendezvousOccurred

Gibt an, ob die Instanz eine Verbindung zum Domänenadministrationsserver (DAS, Domain Administration Server) hergestellt hat. Mögliche Werte:

wahr

Die Instanz hat eine Verbindung zum DAS hergestellt.

falsch

Die Instanz hat keine Verbindung zum DAS hergestellt.
