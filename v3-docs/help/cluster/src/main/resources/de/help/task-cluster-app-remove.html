So entfernen Sie Anwendungen aus einem Cluster

Durch Entfernen einer Anwendung wird die Anwendung nur aus dem derzeitigen Ziel entfernt. Wenn eine Anwendung aus einem Ziel entfernt wird, wird die Anwendung weiterhin auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish-Server bereitgestellten Anwendungen angezeigt werden. Die Anwendung bleibt auch weiterhin auf anderen Zielen bereitgestellt, auf denen sie bereitgestellt wurde.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für den Cluster wird geöffnet.

  4. Aktivieren Sie für jede Anwendung, die Sie entfernen, das Kontrollkästchen neben der Anwendung in der Tabelle "Bereitgestellte Anwendungen".
  5. Klicken Sie auf "Löschen".

    Sie müssen das Entfernen der Anwendungen bestätigen.

  6. Bestätigen Sie das Entfernen der Anwendungen.

    Die Anwendungen werden aus der Tabelle der bereitgestellten Anwendungen auf der Seite entfernt.
