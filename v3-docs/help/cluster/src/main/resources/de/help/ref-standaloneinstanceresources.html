Ressourcen

Auf der Seite "Ressourcen" können Sie die folgenden Aufgaben für die ausgewählte eigenständige GlassFish Server-Instanz ausführen:

  • Anzeigen und Verwalten der vorhandenen Ressourcen der Instanz

  • Erstellen weiterer Ressourcen für die Instanz

Das Feld "Instanzname" ist ein schreibgeschütztes Feld, in dem der Name der aktuellen Instanz angezeigt wird.

Auf der Seite "Ressourcen" werden die vorhandenen Ressourcen der Instanzen aufgelistet. Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

Ressourcenname

Der Name, der der Ressource bei der Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird je nach Ressourcentyp eine der folgenden Seiten zum Bearbeiten der Ressource geöffnet:

  • Edit JDBC Resource

  • Edit JavaMail Session

  • Edit Custom Resource

  • New Custom Resource

  • Edit External Resource

  • Edit Connector Resource

  • Edit Admin Object Resource

Aktiviert

Gibt an, ob die Ressource aktiviert ist:

  • Wenn die Ressource aktiviert ist, wird ein Häkchen (✓) angezeigt.

  • Wenn die Ressource deaktiviert ist, wird ein Kreuz (✗) angezeigt.

Typ

Der Typ der Ressource.

Die Tabelle "Ressourcen" enthält darüber hinaus die folgenden Optionen.

Aktivieren

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter Ressourcen.

Deaktivieren

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter Ressourcen.

Neu

Dropdown-Liste, über die eine Ressource des ausgewählten Typs erstellt wird. Durch Auswahl eines Ressourcentyps aus der Liste wird eine der folgenden Seiten zum Erstellen einer Ressource des ausgewählten Typs geöffnet:

  • New JDBC Resource

  • New JMS Connection Factory

  • New JMS Destination Resource

  • New JavaMail Session

  • New Custom Resource

  • New External Resource

  • New Connector Resource

  • New Admin Object Resource

Filter

Dropdown-Liste, über die die Liste der angezeigten Ressourcen nach Typ gefiltert wird.

Verwandte Aufgaben

  • So wird eine Ressource für eine Standalone-GlassFish Server-Instanz erstellt

  • So werden Ressourcen für eine Standalone-GlassFish Server-Instanz aufgelistet

  • So wird eine Ressource für eine Standalone-GlassFish Server-Instanz aktiviert oder deaktiviert

  • So wird eine Ressource für eine Standalone-GlassFish Server-Instanz bearbeitet
