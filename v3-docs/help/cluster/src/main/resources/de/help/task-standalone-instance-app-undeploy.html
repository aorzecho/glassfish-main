So wird das Deployment von Anwendungen für eine Standalone-GlassFish Server-Instanz aufgehoben

Durch Aufheben des Deployments einer Anwendung wird die Anwendung aus allen Zielen und der Domain entfernt. Wenn das Deployment einer Anwendung aufgehoben wird, wird die Anwendung nicht mehr auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish Server bereitgestellten Anwendungen angezeigt werden.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Eigenständige Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" für die Instanz auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für die Instanz wird geöffnet.

  4. Aktivieren Sie für jede Anwendung, deren Deployment Sie aufheben, das Kontrollkästchen neben der Anwendung in der Tabelle "Bereitgestellte Anwendungen".

  5. Klicken Sie auf "Deployment aufheben".

    Sie müssen bestätigen, dass das Deployment der Anwendungen aufgehoben werden soll.

  6. Bestätigen Sie das Aufheben des Deployments der Anwendungen.

    Das Deployment der Anwendungen wird aufgehoben und die Anwendungen werden aus der Tabelle der bereitgestellten Anwendungen auf der Seite entfernt.
