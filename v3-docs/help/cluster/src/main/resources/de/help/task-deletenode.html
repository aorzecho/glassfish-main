So löschen Sie einen Knoten

Durch Löschen eines Knotens wird der Knoten aus der Konfiguration des Domänenadministrationsservers (DAS, Domain Administration Server) entfernt. Die Verzeichnisse und Dateien des Knotens werden gelöscht, wenn die letzte GlassFish-Server-Instanz auf dem Knoten gelöscht wird.

Bevor Sie beginnen

Vergewissern Sie sich, dass sich keine GlassFish-Server-Instanzen auf dem zu löschenden Knoten befinden.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Knoten".

    Die Seite "Knoten" wird geöffnet.

  2. Aktivieren Sie für jeden zu löschenden Knoten das Kontrollkästchen neben dem Knoten in der Tabelle "Knoten".

    Hinweis - Der vordefinierte Knoten localhost-domain kann nicht gelöscht werden.

  3. Klicken Sie auf "Löschen".

    Sie müssen das Löschen des Knotens bestätigen.

  4. Bestätigen Sie das Löschen des Knotens.

    Der Knoten wird gelöscht und aus der Tabelle der Knoten auf der Seite "Knoten" entfernt.
