So halten Sie eigenständige GlassFish-Server-Instanzen an

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Eigenständige Serverinstanz" wird geöffnet.

  2. Aktivieren Sie für jede anzuhaltende Instanz das Kontrollkästchen neben der Instanz in der Tabelle "Serverinstanzen".
  3. Klicken Sie auf "Anhalten".

    Sie müssen das Anhalten der Instanzen bestätigen.

  4. Bestätigen Sie das Anhalten der Instanzen.

    Der Status der Instanzen wird in der Tabelle "Serverinstanzen" aktualisiert, wenn die Instanz angehalten wird.
