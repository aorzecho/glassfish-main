So zeigen Sie Protokolldateien für eine eigenständige GlassFish-Server-Instanz an

Bevor Sie beginnen

Die Instanz, für die Sie Protokolldateien anzeigen möchten, muss ausgeführt werden.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Eigenständige Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die Protokolldateien angezeigt werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf "Protokolldateien anzeigen".

    Ein Protokoll-Viewer-Fenster wird geöffnet.

    Hinweis - Die Instanz muss bereits ausgeführt werden. Wenn die Instanz nicht ausgeführt wird, ist die Schaltfläche "Protokolldateien anzeigen" deaktiviert.
