So werden Logdateien für eine Cluster-GlassFish Server-Instanz angezeigt

Bevor Sie beginnen

Die Instanz, für die Sie Logdateien anzeigen möchten, muss ausgeführt werden.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Cluster-Serverinstanzen" wird für das Cluster geöffnet.

  4. Wählen Sie die Instanz, für die Logdateien angezeigt werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  5. Klicken Sie auf der Seite "Allgemeine Informationen" auf "Logdateien anzeigen".

    Ein Log Viewer-Fenster wird geöffnet.

    Hinweis:

    Die Instanz muss bereits ausgeführt werden. Wenn die Instanz nicht ausgeführt wird, ist die Schaltfläche "Logdateien anzeigen" deaktiviert.
