So zeigen Sie Protokolldateien für eine GlassFish-Server-Cluster-Instanz an

Bevor Sie beginnen

Die Instanz, für die Sie Protokolldateien anzeigen möchten, muss ausgeführt werden.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Cluster-Serverinstanzen" wird für den Cluster geöffnet.

  4. Wählen Sie die Instanz, für die Protokolldateien angezeigt werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  5. Klicken Sie auf der Seite "Allgemeine Informationen" auf "Protokolldateien anzeigen".

    Ein Protokoll-Viewer-Fenster wird geöffnet.

    Hinweis - Die Instanz muss bereits ausgeführt werden. Wenn die Instanz nicht ausgeführt wird, ist die Schaltfläche "Protokolldateien anzeigen" deaktiviert.
