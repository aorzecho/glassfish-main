So geben Sie Standardwerte für Konfigurations-Systemeigenschaften an

Die Standardwerte der Konfigurationseigenschaften werden in der Konfiguration definiert, auf die der Cluster oder die Instanz verweist.

  1. Erweitern Sie in der Navigationsstruktur den Knoten für die Konfiguration.
  2. Wählen Sie in der Liste der Eigenschaftentypen unter dem erweiterten Knoten den Eintrag "Systemeigenschaften".

    Die Seite "Systemeigenschaften" für die ausgewählte Konfiguration wird geöffnet.

    Auf dieser Seite werden die aktuellen Werte der folgenden Systemeigenschaften für die Konfiguration angezeigt:

    • Portnummern, die beim Erstellen der Konfiguration vordefiniert waren

    • Benutzerdefinierte Eigenschaften, die zur Konfiguration hinzugefügt werden (falls vorhanden)

  3. Um sicherzustellen, dass die Änderungen ohne Neustart des Clusters oder der Instanz übernommen werden, wählen Sie die Option "Dynamische Neukonfiguration".
  4. Geben Sie für jede Eigenschaft, für die Sie einen Standardwert festlegen, den neuen Wert in das Feld "Standardwert" für die Eigenschaft ein.
  5. Legen Sie die Standardwerte für alle zusätzlichen Eigenschaften fest, die Sie benötigen.
    1. Klicken Sie in der Tabelle "Weitere Eigenschaften" auf "Eigenschaft hinzufügen".

      Eine Zeile wird zur Tabelle "Weitere Eigenschaften" hinzugefügt.

    2. Geben Sie im Feld "Name der Instanzenvariablen" den Eigenschaftsnamen ein.
    3. Geben Sie im Feld "Standardwert" den Wert ein.
  6. Klicken Sie auf "Speichern".
