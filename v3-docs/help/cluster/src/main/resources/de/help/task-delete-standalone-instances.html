So werden Standalone-GlassFish Server-Instanzen gelöscht

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Aktivieren Sie für jede zu löschende Instanz das Kontrollkästchen neben der Instanz in der Tabelle "Serverinstanzen".

  3. Klicken Sie auf "Löschen".

    Sie müssen das Löschen der Instanzen bestätigen.

  4. Bestätigen Sie das Löschen der Instanzen.

    Die Instanzen werden gelöscht und aus der Tabelle der Instanzen auf der Seite entfernt.
