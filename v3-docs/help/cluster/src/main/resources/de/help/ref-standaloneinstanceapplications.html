Anwendungen

Auf der Seite "Anwendungen" können Sie die folgenden Aufgaben für die ausgewählte eigenständige GlassFish-Server-Instanz ausführen:

  • Anzeigen und Verwalten der Anwendungen, die bereits für die Instanz bereitgestellt sind

  • Bereitstellen weiterer Anwendungen für die Instanz

Das Feld "Instanzname" ist ein schreibgeschütztes Feld, in dem der Name der aktuellen Instanz angezeigt wird.

Auf der Seite "Anwendungen" werden die Anwendungen aufgelistet, die auf der ausgewählten Instanz bereitgestellt sind. Für jede Anwendung werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Anwendung.

Der Name kann eine optionale Versionskennung enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide .

Aktiviert

Gibt an, ob die Anwendung aktiviert ist:

  • Wenn die Anwendung aktiviert ist, wird ein Häkchen (✓) angezeigt.

  • Wenn die Anwendung deaktiviert ist, wird ein Kreuz (✗) angezeigt.

Lastenausgleich aktiviert

Gibt an, ob die Anwendung für den Lastenausgleich aktiviert ist:

  • Wenn die Anwendung für den Lastenausgleich aktiviert ist, wird ein Häkchen (✓) angezeigt.

  • Wenn die Anwendung für den Lastenausgleich deaktiviert ist, wird ein Kreuz (✗) angezeigt.

Module

Die Art der von der Anwendung verwendeten Container. Es stehen folgende Container-Typen zur Verfügung:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld – der Container für Contexts and Dependency Injection für Anwendungen der Java EE-Plattform

Die Tabelle "Bereitgestellte Anwendungen" enthält darüber hinaus die folgenden Optionen.

Bereitstellen

Schaltfläche zum Bereitstellen einer Anwendung.

Undeploy (Bereitstellung aufheben)

Schaltfläche, um die Bereitstellung einer oder mehrerer ausgewählter Anwendungen aufzuheben.

Durch Aufheben der Bereitstellung einer Anwendung wird die Anwendung aus allen Zielen und der Domäne entfernt. Wenn die Bereitstellung einer Anwendung aufgehoben wird, wird die Anwendung nicht mehr auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish-Server bereitgestellten Anwendungen angezeigt werden.

Um Anwendungen nur aus dem aktuellen Ziel zu enfernen, verwenden Sie die Schaltfläche "Entfernen", die sich auf dieser Seite befindet.

Entfernen

Schaltfläche zum Entfernen einer oder mehrerer ausgewählter Anwendungen.

Durch Entfernen einer Anwendung wird die Anwendung nur aus dem derzeitigen Ziel entfernt. Wenn eine Anwendung aus einem Ziel entfernt wird, wird die Anwendung weiterhin auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish-Server bereitgestellten Anwendungen angezeigt werden. Die Anwendung bleibt auch weiterhin auf anderen Zielen bereitgestellt, auf denen sie bereitgestellt wurde.

Um Anwendungen aus allen Zielen und der Domäne zu enfernen, verwenden Sie die Schaltfläche "Bereitstellung aufheben", die sich auf dieser Seite befindet.

Weitere Aktionen

Dropdown-Liste, über die weitere Aktionen zu einer oder mehreren Anwendungen ausgeführt werden können.

Aktivieren

Aktiviert eine oder mehrere Anwendungen.

Deaktivieren

Deaktiviert eine oder mehrere Anwendungen.

Lastenausgleich aktivieren

Aktiviert eine oder mehrere ausgewählte Anwendungen für den Lastenausgleich.

Lastenausgleich deaktivieren

Deaktiviert eine oder mehrere ausgewählte Anwendungen für den Lastenausgleich.

Verwandte Aufgaben

  • So stellen Sie eine Anwendung für eine eigenständige GlassFish-Server-Instanz bereit

  • So listen Sie die für eine eigenständige GlassFish-Server-Instanz bereitgestellten Anwendungen auf

  • So heben Sie die Bereitstellung von Anwendungen für eine eigenständige GlassFish-Server-Instanz auf

  • So entfernen Sie Anwendungen aus einer eigenständigen GlassFish-Server-Instanz

  • So bearbeiten Sie die für eine eigenständige GlassFish-Server-Instanz bereitgestellte Anwendung

  • So aktivieren oder deaktivieren Sie die für eine eigenständige GlassFish-Server-Instanz bereitgestellte Anwendung

  • So aktivieren oder deaktivieren Sie den Lastenausgleich der für eine eigenständige GlassFish-Server-Instanz bereitgestellten Anwendung
