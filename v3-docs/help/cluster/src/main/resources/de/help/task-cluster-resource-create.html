So erstellen Sie eine Ressource für einen Cluster

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für den Cluster wird geöffnet.

  4. Wählen Sie im Dropdown-Menü "Neu" den zu erstellenden Ressourcentyp.

    Die Seite zum Erstellen des ausgewählten Ressourcentyps wird geöffnet.

  5. Nehmen Sie auf der Seite zum Erstellen des ausgewählten Ressourcentyps die erforderlichen Änderungen vor und speichern Sie die Änderungen.

    Weitere Informationen zum Erstellen des ausgewählten Ressourcentyps finden Sie in der Online-Hilfe.

    Hinweis - Stellen Sie beim Erstellen der Ressource sicher, dass Sie den Cluster als Ziel angeben.
