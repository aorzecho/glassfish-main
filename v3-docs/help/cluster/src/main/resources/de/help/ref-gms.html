Bearbeiten von Group Management Service

Auf der Seite "Group Management Service" können Sie die GMS-Einstellungen ändern.

Der Group Management Service (GMS) ist ein prozessinterner Service, der Services für Clusterüberwachung, Clustermitgliedschaft und Gruppenkommunikation bereitstellt. Der GMS baut auf einem konfigurierbaren Stapel von Protokollen zu Gruppenmitgliedschaftserkennung und Zustandsüberwachung auf. Der GMS führt folgende Funktionen aus:

  • Benachrichtigt registrierte Module in einer GlassFish Server-Instanz, wenn mindestens eine Mitgliedsinstanz zum Cluster hinzugefügt bzw. aus dem Cluster entfernt wurde oder bei denen Fehler vermutet bzw. bestätigt wurden.

  • Ermöglicht das Senden und Erhalten von Meldungen zwischen einer Gruppe von Prozessen.

Die Seite "Group Management Service" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Maximum Missed Heartbeats (Maximale Anzahl nicht ausgelöster Heartbeats)

Die Anzahl der Überwachungsversuche, nach denen ein Fehler bei der Serverinstanz vermutet wird. Der Standardwert beträgt 3.

Heartbeat Frequency (Heartbeat-Häufigkeit)

Der Zeitraum zwischen den Überwachungsversuchen. Der Standardwert beträgt 2000 Millisekunden.

Timeout für Gruppenerkennung

Der Zeitraum, den der GMS wartet, bis die Erkennung anderer Clustermitglieder fortgesetzt wird. Der Standardwert beträgt 5000 Millisekunden.

Failure Verification Wait Time (Wartezeit für Fehlerüberprüfung)

Der Zeitraum, nach dem ein vermuteter Fehler als bestätigt markiert wird. Der Standardwert beträgt 1500 Millisekunden.

Für jede Instanz werden folgende Informationen angezeigt:

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft, der für die ausgewählte Instanz festgelegt ist.

Beschreibung

Eine Textbeschreibung mit weiteren Informationen zur Eigenschaft.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Eigenschaften. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.
