So wird eine Anwendung für eine Standalone-GlassFish Server-Instanz bereitgestellt

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Eigenständige Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" für die Instanz auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für die Instanz wird geöffnet.

  4. Klicken Sie auf der Seite "Anwendungen" auf "Bereitstellen".

    Die Seite "Anwendungen oder Module bereitstellen" wird geöffnet.

  5. Geben Sie auf der Seite "Anwendungen oder Module bereitstellen" die entsprechenden Informationen zur Anwendung an und klicken Sie auf "OK".

    Weitere Informationen finden Sie in der Onlinehilfe unter Deploy Applications or Modules.
