So zeigen Sie Überwachungsdaten der Anwendung für eine eigenständige GlassFish-Server-Instanz an

Bevor Sie beginnen

Vor dem Anzeigen von Überwachungsdaten müssen Sie den Überwachungsdienst konfigurieren.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Eigenständige Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die Sie Überwachungsdaten der Anwendung anzeigen möchten.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  4. Wählen Sie aus der Dropdown-Liste "Ansicht" die Anwendung, für die Sie Überwachungsdaten anzeigen möchten.
  5. Klicken Sie auf "Aktualisieren", um die Ansicht der Überwachungsdaten zu aktualisieren.
