So bearbeiten Sie die für einen Cluster bereitgestellte Anwendung

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für den Cluster wird geöffnet.

  4. Klicken Sie auf der Seite "Anwendungen" in der Tabelle "Bereitgestellte Anwendungen" auf den Anwendungsnamen.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  5. Geben Sie auf der Seite "Anwendung bearbeiten" die erforderlichen Informationen zur Anwendung an und speichern Sie die Änderungen.

    Weitere Informationen finden Sie in der Online-Hilfe unter Edit Application.
