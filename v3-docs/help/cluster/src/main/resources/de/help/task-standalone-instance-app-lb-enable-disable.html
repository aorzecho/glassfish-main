So wird das Load Balancing für eine Anwendung aktiviert oder deaktiviert, die in einer Standalone-GlassFish Server-Instanz bereitgestellt ist

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Eigenständige Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" für die Instanz auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für die Instanz wird geöffnet.

  4. Aktivieren Sie das Kontrollkästchen neben der Anwendung.

  5. Wählen Sie im Dropdown-Menü "Weitere Aktionen" den entsprechenden Befehl.

    • Wählen Sie "Load Balancer aktivieren", um das Load Balancing für die Anwendung zu aktivieren.

    • Wählen Sie "Load Balancer deaktivieren", um das Load Balancing für die Anwendung zu deaktivieren.
