So löschen Sie einen Cluster

Durch Löschen eines Clusters werden der Cluster und alle GlassFish-Server-Instanzen im Cluster gelöscht. Sie brauchen die Instanzen im Cluster nicht zu löschen, bevor Sie den Cluster löschen.

Wenn die vom Cluster verwendete Konfiguration nicht gemeinsam genutzt wird, wird die Konfiguration ebenfalls gelöscht.

Bevor Sie beginnen

Vergewissern Sie sich, dass der zu löschende Cluster angehalten ist (weitere Informationen erhalten Sie hier).

  1. Wählen Sie in der Navigationsstruktur den Knoten "Cluster".

    Die Seite "Cluster" wird geöffnet.

  2. Aktivieren Sie auf der Seite "Cluster" das Kontrollkästchen neben dem Namen des Clusters.
  3. Klicken Sie auf "Löschen".

    Sie müssen das Löschen des Clusters und aller Instanzen im Cluster bestätigen.

  4. Bestätigen Sie das Löschen des Clusters und aller Instanzen im Cluster.

    Der Cluster wird gelöscht und aus der Tabelle der Cluster auf der Seite "Cluster" entfernt.
