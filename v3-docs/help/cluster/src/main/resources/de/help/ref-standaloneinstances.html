Eigenständige Serverinstanzen

Auf der Seite "Eigenständige Serverinstanzen" können Sie eigenständige GlassFish-Server-Instanzen erstellen und verwalten.

Auf der Seite "Eigenständige Serverinstanzen" werden alle eigenständigen Instanzen in der Domäne aufgelistet. Für jede Instanz werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name, der der Instanz bei der Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Allgemeine Informationen für die Instanz geöffnet.

Gewichtung des Lastenausgleichs

Eine Ganzzahl, die die Gewichtung des Lastenausgleichs der Instanz darstellt.

Die Gewichtung des Lastenausgleichs bestimmt den Anteil aller Anforderungen an den Cluster, die von der Instanz verarbeitet werden.

Beispielsweise muss bei einem Cluster mit zwei Instanzen eine Instanz eine von vier Anforderungen und die andere Instanz drei von vier Anforderungen verarbeiten. Definieren Sie in diesem Fall für die Gewichtung der Instanz, die eine von vier Anforderungen verarbeiten soll, den Wert 1 und für die Gewichtung der anderen Instanz den Wert 3. Wenn Sie stattdessen Prozentsätze bevorzugen, legen Sie für die Gewichtung der Instanzen jeweils den Prozentwert 25 und 75 fest.

Die Standardgewichtung ist 100.

Konfiguration

Die Konfiguration, die für die Instanz bei der Erstellung ausgewählt wurde. Durch Klicken auf die Konfiguration wird die Seite Konfigurationen für die Instanz geöffnet.

Knoten

Der Name des Knotens, auf dem sich die Instanz befindet. Wenn Sie auf den Namen klicken, wird die Seite "Knoten bearbeiten" für den Knoten geöffnet (siehe Knoten bearbeiten).

Status

Gibt an, ob die Instanz ausgeführt wird.

Die Tabelle "Serverinstanzen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer eigenständigen Instanz.

Löschen

Schaltfläche zum Löschen von einer oder mehreren ausgewählten eigenständigen Instanzen.

Starten

Schaltfläche zum Starten von einer oder mehreren ausgewählten eigenständigen Instanzen.

Anhalten

Schaltfläche zum Anhalten von einer oder mehreren ausgewählten eigenständigen Instanzen.

Verwandte Aufgaben

  • So erstellen Sie eine eigenständige GlassFish-Server-Instanz

  • So löschen Sie eigenständige GlassFish-Server-Instanzen

  • So starten Sie eigenständige GlassFish-Server-Instanzen

  • So halten Sie eigenständige GlassFish-Server-Instanzen an
