So werden Nur-Instanz-Eigenschaften für eine Standalone-GlassFish Server-Instanz festgelegt

Die Eigenschaften einer GlassFish Server-Instanz enthalten optionale Konfigurationsinformationen zur Instanz.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die instanzenbezogene Eigenschaften geändert werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Eigenschaften".

  4. Klicken Sie, falls erforderlich, auf die Registerkarte "Instanzeigenschaften".

    Die Seite "Instanzeigenschaften" wird geöffnet.

    Auf dieser Seite werden die Eigenschaften aufgelistet, die für die ausgewählte GlassFish Server-Instanz festgelegt sind.

  5. Geben Sie für jede vorhandene Eigenschaft, die Sie ändern möchten, den neuen Wert im Feld "Wert" für die Eigenschaft ein.

  6. Geben Sie Werte für alle zusätzlichen Eigenschaften ein, die Sie benötigen.

    1. Klicken Sie in der Tabelle "Weitere Eigenschaften" auf "Eigenschaft hinzufügen".

      Eine Zeile wird zur Tabelle "Weitere Eigenschaften" hinzugefügt.

    2. Geben Sie im Feld "Name" den Eigenschaftsnamen ein.

    3. Geben Sie im Feld "Wert" den Wert ein.

    4. Geben Sie im Feld "Beschreibung" eine kurze Beschreibung der Eigenschaft ein.

  7. Klicken Sie auf "Speichern".
