So wird eine Ressource für eine Standalone-GlassFish Server-Instanz aktiviert oder deaktiviert

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Eigenständige Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für die Instanz wird geöffnet.

  4. Aktivieren Sie das Kontrollkästchen neben der Ressource.

  5. Klicken Sie auf die entsprechende Schaltfläche.

    • Wählen Sie "Aktivieren", um die Ressource zu aktivieren.

    • Wählen Sie "Deaktivieren", um die Ressource zu deaktivieren.

      Durch Deaktivieren wird die Ressource nicht entfernt.
