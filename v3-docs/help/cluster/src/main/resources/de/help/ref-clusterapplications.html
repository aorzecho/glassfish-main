Anwendungen

Auf der Seite "Anwendungen" können Sie die folgenden Aufgaben auf dem aktuellen Cluster ausführen:

  • Anzeigen und Verwalten der Anwendungen, die bereits für das Cluster bereitgestellt sind

  • Bereitstellen weiterer Anwendungen für das Cluster

Das Feld "Clustername" ist ein schreibgeschütztes Feld, in dem der Name des aktuellen Clusters angezeigt wird.

Auf der Seite "Anwendungen" werden die Anwendungen aufgelistet, die für den aktuellen Cluster bereitgestellt sind. Für jede Anwendung werden die folgenden Informationen zur Verfügung gestellt.

Name (Name)

Der Name der Anwendung.

Der Name kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

Aktiviert

Gibt an, ob die Anwendung aktiviert ist:

  • Wenn die Anwendung aktiviert ist, wird ein Häkchen (✓) angezeigt.

  • Wenn die Anwendung deaktiviert ist, wird ein Kreuz (✗) angezeigt.

LB aktiviert

Gibt an, ob die Anwendung für das Load Balancing aktiviert ist:

  • Wenn die Anwendung für das Load Balancing aktiviert ist, wird ein Häkchen (✓) angezeigt.

  • Wenn die Anwendung für das Load Balancing deaktiviert ist, wird ein Kreuz (✗) angezeigt.

Engines

Die Art der von der Anwendung verwendeten Container. Es stehen folgende Container-Typen zur Verfügung:

  • web

  • webservices

  • ejb

  • Connector

  • appclient

  • weld – der Container für Contexts and Dependency Injection für Anwendungen der Java EE-Plattform

Die Tabelle "Bereitgestellte Anwendungen" enthält darüber hinaus die folgenden Optionen.

Bereitstellen

Schaltfläche zum Bereitstellen einer Anwendung.

Deployment aufheben

Schaltfläche, um das Deployment einer oder mehrerer ausgewählter Anwendungen aufzuheben.

Durch Aufheben des Deployments einer Anwendung wird die Anwendung aus allen Zielen und der Domain entfernt. Wenn das Deployment einer Anwendung aufgehoben wird, wird die Anwendung nicht mehr auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish Server bereitgestellten Anwendungen angezeigt werden.

Um Anwendungen nur aus dem aktuellen Ziel zu entfernen, verwenden Sie die Schaltfläche "Entfernen", die sich auf dieser Seite befindet.

Entfernen

Schaltfläche zum Entfernen einer oder mehrerer ausgewählter Anwendungen.

Durch Entfernen einer Anwendung wird die Anwendung nur aus dem aktuellen Ziel entfernt. Wenn eine Anwendung aus einem Ziel entfernt wird, wird die Anwendung weiterhin auf der Seite Anwendungen aufgelistet, auf der alle in GlassFish Server bereitgestellten Anwendungen angezeigt werden. Die Anwendung bleibt auch weiterhin auf anderen Zielen bereitgestellt, auf denen sie bereitgestellt wurde.

Um Anwendungen aus allen Zielen und der Domain zu entfernen, verwenden Sie die Schaltfläche "Deployment aufheben", die sich auf dieser Seite befindet.

Weitere Aktionen

Dropdown-Liste, über die weitere Aktionen zu einer oder mehreren Anwendungen ausgeführt werden können.

Aktivieren

Aktiviert eine oder mehrere Anwendungen.

Deaktivieren

Deaktiviert eine oder mehrere Anwendungen.

Load Balancer aktivieren

Aktiviert mindestens eine ausgewählte Anwendung für das Load Balancing.

Load Balancer deaktivieren

Deaktiviert mindestens eine ausgewählte Anwendung für das Load Balancing.

Verwandte Aufgaben

  • So wird eine Anwendung in einem Cluster bereitgestellt

  • So werden Anwendungen aufgelistet, die in einem Cluster bereitgestellt sind

  • So wird das Deployment von Anwendungen in einem Cluster aufgehoben

  • So werden Anwendungen aus einem Cluster entfernt

  • So wird eine Anwendung bearbeitet, die in einem Cluster bereitgestellt ist

  • So wird eine Anwendung aktiviert oder deaktiviert, die in einem Cluster bereitgestellt ist

  • So wird das Load Balancing für eine Anwendung aktiviert oder deaktiviert, die in einem Cluster bereitgestellt ist
