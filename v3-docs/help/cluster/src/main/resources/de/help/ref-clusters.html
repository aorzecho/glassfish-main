Cluster

Auf der Seite "Cluster" werden die Cluster in der Domain aufgelistet. Für jeden Cluster werden folgende Informationen angezeigt:

Name

Der Name, der dem Cluster bei der Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Allgemeine Informationen für das Cluster geöffnet.

Konfiguration

Die Konfiguration, die für das Cluster bei der Erstellung ausgewählt wurde. Durch Klicken auf die Konfiguration wird die Seite Konfigurationen für das Cluster geöffnet.

Instanzen

Der Name und Status aller Instanzen im Cluster. Wenn Sie auf den Namen der Instanz klicken, wird die Seite Allgemeine Informationen für die Instanz geöffnet.

Die Tabelle "Cluster" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines Clusters.

Löschen

Schaltfläche zum Löschen eines Clusters.

Cluster starten

Schaltfläche zum Starten eines Clusters.

Cluster anhalten

Schaltfläche zum Anhalten eines Clusters.

Verwandte Aufgaben

  • So wird ein Cluster erstellt

  • So wird ein Cluster gelöscht

  • So werden Verfügbarkeitseinstellungen für ein Cluster konfiguriert

  • So wird ein Cluster gestartet

  • So wird ein Cluster gestoppt

  • So wird die Konfiguration eines Clusters geändert
