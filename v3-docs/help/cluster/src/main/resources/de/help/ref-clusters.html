Cluster

Auf der Seite "Cluster" werden die Cluster in der Domäne aufgelistet. Für jeden Cluster werden folgende Informationen angezeigt:

Name

Der Name, der dem Cluster bei der Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Allgemeine Informationen für den Cluster geöffnet.

Konfiguration

Die Konfiguration, die für den Cluster bei der Erstellung ausgewählt wurde. Durch Klicken auf die Konfiguration wird die Seite Konfigurationen für den Cluster geöffnet.

Instanzen

Der Name und Status aller Instanzen im Cluster. Wenn Sie auf den Namen der Instanz klicken, wird die Seite Allgemeine Informationen für die Instanz geöffnet.

Die Tabelle "Cluster" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines Clusters.

Löschen

Schaltfläche zum Löschen eines Clusters.

Cluster starten

Schaltfläche zum Starten eines Clusters.

Cluster anhalten

Schaltfläche zum Anhalten eines Clusters.
