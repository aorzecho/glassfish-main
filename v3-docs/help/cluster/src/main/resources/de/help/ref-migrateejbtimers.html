EJB-Zeitgeber migrieren

Die Seite "EJB-Zeitgeber migrieren" enthält die folgenden Optionen.

Quelle

Eine Dropdown-Liste der angehaltenen GlassFish-Server-Instanzen im Cluster. EJB-Zeitgeber können nur von angehaltenen Instanzen migriert werden.

Ziel

Eine Dropdown-Liste der ausgeführten GlassFish-Server-Instanzen im Cluster. EJB-Zeitgeber können nur zu ausgeführten Instanzen migriert werden.

Hinweis - Wenn alle Instanzen im Cluster angehalten sind, wird auf dieser Seite nur eine Warnung angezeigt, dass keine ausgeführten Instanzen, zu denen EJB-Zeitgeber migriert werden können, verfügbar sind.

Verwandte Aufgaben

  • So migrieren Sie EJB-Zeitgeber

Entsprechende asadmin-Unterbefehle

  • migrate-timers(1)
