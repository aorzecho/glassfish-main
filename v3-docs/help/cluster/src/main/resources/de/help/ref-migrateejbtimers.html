EJB-Timer migrieren

Die Seite "EJB-Timer migrieren" enthält die folgenden Optionen.

Quelle

Eine Dropdown-Liste der angehaltenen GlassFish Server-Instanzen im Cluster. EJB-Timer können nur von angehaltenen Instanzen migriert werden.

Ziel

Eine Dropdown-Liste der ausgeführten GlassFish Server-Instanzen im Cluster. EJB-Timer können nur zu ausgeführten Instanzen migriert werden.

Hinweis:

Wenn alle Instanzen im Cluster gestoppt sind, wird auf dieser Seite nur eine Warnung angezeigt, dass keine ausgeführten Instanzen, zu denen EJB-Timer migriert werden können, verfügbar sind.

Die Seite "EJB-Timer migrieren" enthält eine Tabelle der EJB-Timer mit den folgenden Informationen.

Server Name (Servername)

Liste der GlassFish Server-Instanzen im Cluster.

Timerwert

Anzahl persistenter Timer auf jeder Instanz.

Verwandte Aufgaben

So werden EJB-Timer migriert

Entsprechende asadmin-Unterbefehle

  • list-timers(1)

  • migrate-timers(1)
