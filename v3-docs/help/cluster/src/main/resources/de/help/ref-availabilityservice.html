Verfügbarkeitsservice

Auf der Seite "Verfügbarkeitsservice" können Sie die Verfügbarkeitseinstellungen für einen Cluster ändern.

Die Seite "Verfügbarkeitsservice" enthält die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Verfügbarkeitsservice

Wenn das Kontrollkästchen "Aktiviert" aktiviert ist, wird die Verfügbarkeit für das Cluster aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Eine Tabelle mit weiteren Eigenschaften für den Verfügbarkeitsservice.

Bei GlassFish Server sind keine zusätzlichen Eigenschaften für den Verfügbarkeitsservice definiert.

Für jede Instanz werden folgende Informationen angezeigt:

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft, der für die ausgewählte Instanz festgelegt ist.

Beschreibung

Eine Textbeschreibung mit weiteren Informationen zur Eigenschaft.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Eigenschaften. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.

Verwandte Aufgaben
