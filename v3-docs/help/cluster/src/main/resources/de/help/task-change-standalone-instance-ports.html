So werden die Portnummern einer Standalone-GlassFish Server-Instanz geändert

Die Portnummern einer eigenständigen GlassFish Server-Instanz werden ursprünglich in der Konfiguration festgelegt, auf die die Instanz verweist. Sie müssen diese Portnummern ändern, wenn sich beispielsweise zwei Instanzen, die auf dieselbe Konfiguration verweisen, auf demselben Host befinden. In diesem Fall müssen die Portnummern für eine Instanz geändert werden, da beide Instanzen auf dieselbe Konfiguration verweisen.

Die Portnummern einer Instanz werden als Java-Systemeigenschaften gespeichert. Diese Eigenschaften werden an den Java-Anwendungs-Launcher über die Option -D des Java-Anwendungs-Launchers übergeben, wenn GlassFish Server gestartet wird.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die Portnummern geändert werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Eigenschaften".

    Die Seite "Instanzsystemeigenschaften" wird geöffnet.

    Auf dieser Seite werden die aktuellen Werte der folgenden Systemeigenschaften für die Instanz angezeigt:

    • Portnummern, die in der Konfiguration der Instanz vordefiniert sind

    • Benutzerdefinierte Eigenschaften, die für die Instanz definiert sind (falls vorhanden)

  4. Geben Sie für jede vorhandene Eigenschaft, die Sie ändern möchten, den neuen Wert im Feld "Wert übergehen" für die Eigenschaft ein.

  5. Geben Sie Werte für alle zusätzlichen Eigenschaften ein, die Sie benötigen.

    1. Klicken Sie in der Tabelle "Weitere Eigenschaften" auf "Eigenschaft hinzufügen".

      Eine Zeile wird zur Tabelle "Weitere Eigenschaften" hinzugefügt.

    2. Geben Sie im Feld "Name der Instanzenvariablen" den Eigenschaftsnamen ein.

    3. Geben Sie im Feld "Wert übergehen" den Wert ein.

  6. Klicken Sie auf "Speichern".
