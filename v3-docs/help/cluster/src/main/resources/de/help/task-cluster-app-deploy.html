So wird eine Anwendung in einem Cluster bereitgestellt

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für das Cluster wird geöffnet.

  4. Klicken Sie auf der Seite "Anwendungen" auf "Bereitstellen".

    Die Seite "Anwendungen oder Module bereitstellen" wird geöffnet.

  5. Geben Sie auf der Seite "Anwendungen oder Module bereitstellen" die entsprechenden Informationen zur Anwendung an und klicken Sie auf "OK".

    Weitere Informationen finden Sie in der Onlinehilfe unter Deploy Applications or Modules.
