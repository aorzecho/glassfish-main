So bearbeiten Sie eine Ressource für einen Cluster

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für den Cluster wird geöffnet.

  4. Klicken Sie auf den Ressourcennamen.

    Die Seite zum Bearbeiten der Ressource wird geöffnet.

  5. Nehmen Sie auf der Seite zum Bearbeiten der Ressource die erforderlichen Änderungen vor und speichern Sie die Änderungen.

    Weitere Informationen zum Bearbeiten der Ressource finden Sie in der Online-Hilfe.
