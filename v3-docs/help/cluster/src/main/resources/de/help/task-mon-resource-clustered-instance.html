So zeigen Sie Überwachungsdaten der Ressource für eine GlassFish-Server-Cluster-Instanz an

Bevor Sie beginnen

Vor dem Anzeigen von Überwachungsdaten müssen Sie den Überwachungsdienst konfigurieren.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Cluster-Serverinstanzen" wird für den Cluster geöffnet.

  4. Wählen Sie die Instanz, für die Sie Überwachungsdaten der Ressource anzeigen möchten.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  5. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  6. Klicken Sie auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcenüberwachung" wird geöffnet.

  7. Wählen Sie aus der Dropdown-Liste "Ansicht" die Ressource, für die Sie Überwachungsdaten anzeigen möchten.
  8. Klicken Sie auf "Aktualisieren", um die Ansicht der Überwachungsdaten zu aktualisieren.
