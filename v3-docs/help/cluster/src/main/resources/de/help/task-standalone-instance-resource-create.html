So wird eine Ressource für eine Standalone-GlassFish Server-Instanz erstellt

  1. Wählen Sie in der Navigationsstruktur den Knoten "Eigenständige Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Eigenständige Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für die Instanz wird geöffnet.

  4. Wählen Sie im Dropdown-Menü "Neu" den zu erstellenden Ressourcentyp.

    Die Seite zum Erstellen des ausgewählten Ressourcentyps wird geöffnet.

  5. Nehmen Sie auf der Seite zum Erstellen des ausgewählten Ressourcentyps die erforderlichen Änderungen vor und speichern Sie die Änderungen.

    Weitere Informationen zum Erstellen des ausgewählten Ressourcentyps finden Sie in der Onlinehilfe.

    Hinweis:

    Stellen Sie beim Erstellen der Ressource sicher, dass Sie die Instanz als Ziel angeben.
