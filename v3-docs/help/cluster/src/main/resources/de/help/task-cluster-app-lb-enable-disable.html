So wird das Load Balancing für eine Anwendung aktiviert oder deaktiviert, die in einem Cluster bereitgestellt ist

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für das Cluster wird geöffnet.

  4. Aktivieren Sie das Kontrollkästchen neben der Anwendung.

  5. Wählen Sie im Dropdown-Menü "Weitere Aktionen" den entsprechenden Befehl.

    • Wählen Sie "Load Balancer aktivieren", um das Load Balancing für die Anwendung zu aktivieren.

    • Wählen Sie "Load Balancer deaktivieren", um das Load Balancing für die Anwendung zu deaktivieren.
