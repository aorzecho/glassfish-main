Allgemeine Informationen

Die Seite "Allgemeine Informationen" enthält die folgenden Optionen.

Starten

Schaltfläche zum Starten der Instanz. Wenn die Instanz ausgeführt wird, ist diese Schaltfläche deaktiviert.

Anhalten

Schaltfläche zum Anhalten der Instanz. Wenn die Instanz angehalten wird, ist diese Schaltfläche deaktiviert.

Neustart

Schaltfläche zum erneuten Starten der Instanz. Wenn die Instanz angehalten wird, ist diese Schaltfläche deaktiviert.

Protokolldateien anzeigen

Schaltfläche zum Anzeigen von Protokolldateien für die Instanz.

Wiederherstellungsvorgänge

Schaltfläche zum Wiederherstellen von Vorgängen für die Instanz.

Clustername

Der Name, der dem Cluster bei der Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Allgemeine Informationen für den Cluster geöffnet.

Instanzname

Der Name, der der Instanz beim Hinzufügen zum Cluster zugewiesen wurde. Dieses Feld ist schreibgeschützt.

Status

Gibt an, ob die Instanz ausgeführt wird, einen Neustart benötigt, um mit Konfigurationsänderungen aktualisiert zu werden, oder nicht ausgeführt wird. Dieses Feld ist schreibgeschützt.

JVM

Wenn Sie auf einen JVM-Bericht klicken, wird ein separates Fenster geöffnet, das Berichte über die Java Virtual Machine anzeigt, einschließlich eines zusammenfassenden Berichts, Speichermanagement- und Aufräumbericht, Klassenladungsbericht und einem aktuellen Thread-Abbild.

Knoten

Der Name des Knotens, auf dem sich die Instanz befindet. Wenn Sie auf den Namen klicken, wird die Seite "Knoten bearbeiten" für den Knoten geöffnet (siehe Knoten bearbeiten).

Konfiguration

Die Konfiguration, auf die die Instanz verweist. Durch Klicken auf die Konfiguration wird die Seite Konfigurationen für die Instanz geöffnet.

HTTP-Ports

Eine durch Kommata getrennte Liste der Nummern von Ports, die die Instanz auf die folgenden Anforderungstypen überwacht:

  • Administrationsanforderungen

  • HTTP-Anforderungen

  • HTTPS-Anforderungen

Dieses Feld ist schreibgeschützt.

IIOP-Ports

Eine durch Kommata getrennte Liste der Nummern von Ports, die die Instanz für die folgenden Verbindungstypen verwendet:

  • Sichere IIOP-Verbindung

  • Sichere IIOP-Verbindungen mit Client-Authentifizierung

  • IIOP-Verbindungen

Dieses Feld ist schreibgeschützt.

Verwandte Aufgaben

  • So starten Sie GlassFish-Server-Cluster-Instanzen

  • So halten Sie GlassFish-Server-Cluster-Instanzen an

  • So zeigen Sie Protokolldateien für eine GlassFish-Server-Cluster-Instanz an

  • So stellen Sie Vorgänge für eine GlassFish-Server-Cluster-Instanz wieder her
