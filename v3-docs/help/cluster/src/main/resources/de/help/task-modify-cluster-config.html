So ändern Sie die Konfiguration eines Clusters

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie den Knoten für den Cluster.
  3. Klicken Sie auf den Namen der Konfiguration.

    Die Seite "Konfiguration" für den Cluster wird geöffnet.

  4. Wählen Sie auf der Seite "Konfiguration" die Komponente der zu ändernden Konfiguration.

    Die Seite zum Bearbeiten der ausgewählten Komponente der Konfiguration wird geöffnet.

  5. Nehmen Sie auf der Seite zum Bearbeiten der ausgewählten Komponente die erforderlichen Änderungen vor und speichern Sie die Änderungen.

    Weitere Informationen zum Bearbeiten der ausgewählten Komponente finden Sie in der Online-Hilfe.
