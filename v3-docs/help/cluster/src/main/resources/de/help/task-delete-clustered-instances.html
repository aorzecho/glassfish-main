So löschen Sie GlassFish-Server-Cluster-Instanzen

Achtung - Löschen Sie nicht die Instanz, die mit dem Master Broker verknüpft ist, wenn Sie einen Java Message Service (JMS)-Cluster mit einem Master Broker verwenden. Wenn diese Instanz gelöscht werden muss, verwenden Sie den Unterbefehl change-master-broker(1), um den Master Broker einer anderen Instanz zuzuweisen.

  1. Erweitern Sie in der Navigationsstruktur den Knoten "Cluster".
  2. Wählen Sie unter dem erweiterten Knoten "Cluster" den Knoten für den Cluster.

    Die Seite "Allgemeine Informationen" für den Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Cluster-Serverinstanzen" wird für den Cluster geöffnet.

  4. Aktivieren Sie für jede zu löschende Instanz das Kontrollkästchen neben der Instanz in der Tabelle "Serverinstanzen".
  5. Klicken Sie auf "Löschen".

    Sie müssen das Löschen der Instanzen bestätigen.

  6. Bestätigen Sie das Löschen der Instanzen.

    Die Instanzen werden gelöscht und aus der Tabelle der Instanzen auf der Seite entfernt.
