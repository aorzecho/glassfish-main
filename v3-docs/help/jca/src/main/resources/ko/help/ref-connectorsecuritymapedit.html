커넥터 접속 풀 보안 맵 편집

커넥터 접속 풀 보안 맵 편집 페이지에서는 커넥터 접속 풀 보안 맵을 편집할 수 있습니다.

커넥터 접속 풀 보안 맵 편집 페이지에는 다음 옵션이 포함되어 있습니다.

접속 풀 이름

이 보안 맵이 속한 커넥터 접속 풀의 이름입니다. 풀 이름은 읽기 전용 필드입니다.

보안 맵 이름

보안 맵의 이름입니다. 이름은 읽기 전용 필드입니다. 새 보안 맵을 생성할 때만 보안 맵 이름을 지정할 수 있습니다.

보안 맵 이름

보안 맵의 이름입니다.

사용자 그룹

이 라디오 단추를 선택한 경우 텍스트 필드에는 사용자 그룹의 이름이 콤마로 구분되어 포함되어야 합니다.

주체

이 라디오 단추를 선택한 경우 텍스트 필드에는 주체의 이름이 콤마로 구분되어 포함되어야 합니다.

사용자 이름

EIS에 필요한 백엔드 주체 사용자 이름입니다.

비밀번호

EIS에 필요한 백엔드 주체 비밀번호입니다.
