자원 어댑터 구성을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 자원 어댑터 구성 노드를 선택합니다.

    자원 어댑터 구성 페이지가 열립니다.

  3. 자원 어댑터 구성 페이지에서 편집할 자원 어댑터 구성의 이름을 누릅니다.

    자원 어댑터 구성 편집 페이지가 열립니다.

  4. 스레드 풀 ID 드롭다운 목록에서 자원 어댑터 구성에 대해 사용할 스레드 풀을 선택합니다.
  5. (옵션) 추가 등록 정보 섹션에서 자원 어댑터에 대한 추가 등록 정보를 지정합니다.

    사용 가능한 등록 정보는 자원 어댑터 JavaBeans 구성 요소의 클래스 이름을 정의하는 ra.xml 파일의 resourceadapter-class 요소로 참조되는 클래스의 setter 메소드의 이름입니다. 여기에 정의된 등록 정보에 대한 모든 수정 사항은 ra.xml에 있는 기본값을 대체합니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  6. 저장을 누릅니다.
