커넥터 자원 편집

커넥터 자원 편집 페이지에서는 커넥터 자원에 대한 설정을 수정할 수 있습니다.

커넥터 자원 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

모든 설정의 기본값을 복원하는 버튼입니다.

JNDI 이름

커넥터 자원을 식별하는 고유한 이름입니다. JNDI 이름은 읽기 전용 필드입니다. 새 커넥터 자원을 만들 때만 JNDI 이름을 지정할 수 있습니다.

풀 이름

이 자원과 연결된 커넥터 연결 풀입니다.

설명

커넥터 자원에 대한 설명입니다.

상태

이 옵션을 선택하면 커넥터 자원이 활성화됩니다. 이 옵션은 기본적으로 활성화됩니다.

추가 등록 정보

커넥터 자원에 대한 추가 등록 정보입니다. 사용 가능한 등록 정보는 커넥터 연결 풀의 자원 어댑터에 따라 다릅니다. 이 표에 지정된 이름-값 쌍을 사용하여 자원 어댑터 공급업체에서 정의한 등록 정보의 기본값을 대체할 수 있습니다.

관련 작업

  • 커넥터 자원을 만드는 방법

  • 커넥터 자원을 편집하는 방법

  • 커넥터 자원을 삭제하는 방법

관련 asadmin 명령

  • create-connector-resource(1)

  • list-connector-resources(1)

  • delete-connector-resource(1)
