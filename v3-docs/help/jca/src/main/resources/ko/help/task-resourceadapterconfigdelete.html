자원 어댑터 구성을 삭제하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 자원 어댑터 구성 노드를 선택합니다.

    자원 어댑터 구성 페이지가 열립니다.

  3. 자원 어댑터 구성 페이지에서 삭제할 자원 어댑터 구성의 이름 옆에 있는 확인란을 선택합니다.
  4. 삭제를 누릅니다.
