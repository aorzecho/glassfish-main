커넥터 자원을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 커넥터 자원 노드를 선택합니다.

    커넥터 자원 페이지가 열립니다.

  4. 커넥터 자원 페이지에서 편집할 자원의 이름을 누릅니다.

    커넥터 자원 편집 페이지가 열립니다.

  5. 풀 이름 드롭다운 목록에서 커넥터 자원이 속한 커넥터 연결 풀을 선택합니다.
  6. 설명 필드에 커넥터 자원에 대한 설명을 입력합니다.
  7. 대상 탭에서 필요한 대상을 선택하고 활성화 또는 비활성화를 누릅니다.

    목록에서 대상을 추가하거나 제거하려면 대상 관리를 누릅니다.

    일반 탭을 눌러 커넥터 자원 편집 페이지로 돌아갑니다.

  8. 런타임에 커넥터 자원을 활성화하려면 상태 사용 가능 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  9. (옵션) 추가 등록 정보 섹션에서 커넥터 자원에 대한 추가 등록 정보를 지정합니다.

    사용 가능한 등록 정보는 커넥터 연결 풀의 자원 어댑터에 따라 다릅니다.

    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  10. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.
