커넥터 리소스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [커넥터 리소스] 노드를 선택합니다.

    [커넥터 리소스] 페이지가 열립니다.

  4. [커넥터 리소스] 페이지에서 편집할 리소스의 이름을 누릅니다.

    [커넥터 리소스 편집] 페이지가 열립니다.

  5. 풀 이름 드롭다운 목록에서 커넥터 리소스가 속한 커넥터 접속 풀을 선택합니다.

  6. [설명] 필드에 커넥터 리소스에 대한 설명을 입력합니다.

  7. 대상 탭에서 필요한 대상을 선택하고 사용 또는 사용 안함을 누릅니다.

    목록에서 대상을 추가하거나 제거하려면 대상 관리를 누릅니다.

    일반 탭을 눌러 커넥터 리소스 편집 페이지로 돌아갑니다.

  8. 런타임에 커넥터 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. [추가 속성] 섹션에서 커넥터 리소스에 대한 추가 속성을 지정합니다.

    사용 가능한 속성은 커넥터 접속 풀의 리소스 어댑터에 따라 다릅니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  10. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.
