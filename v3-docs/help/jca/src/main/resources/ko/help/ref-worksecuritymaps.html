작업 보안 맵

작업 보안 맵 페이지에서는 자원 어댑터에 대한 작업 보안 맵을 구성할 수 있습니다.

커넥터 작업 보안 맵은 자원 어댑터 EIS 기본 또는 EIS 사용자 그룹이 제출한 작업의 호출자 ID를 GlassFish Server 보안 도메인의 적절한 기본 또는 사용자 그룹에 매핑합니다. 하나 이상의 작업 보안 맵을 자원 어댑터와 연결할 수 있습니다. 커넥터 작업 보안 맵 구성은 모든 사용자 또는 모든 사용자 그룹을 나타낼 수 있도록 와일드카드 별표(*)의 사용을 지원합니다.

작업 보안 맵은 서블릿 또는 EJB 인증 중에 받은 기본을 EIS에서 수락한 자격 증명에 매핑하는 커넥터 연결 풀에 대한 보안 맵과 다릅니다. 커넥터 연결 풀의 보안 맵에 대한 자세한 내용은 커넥터 연결 풀 보안 맵를 참조하십시오.

EIS(Enterprise Information System)는 조직의 데이터를 보관하는 시스템입니다. 이는 메인프레임, 메시징 시스템, 데이터베이스 시스템 또는 응용 프로그램일 수 있습니다.
