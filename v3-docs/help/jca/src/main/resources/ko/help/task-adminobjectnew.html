관리 객체 리소스를 생성하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [관리 객체 리소스] 노드를 선택합니다.

    [관리 객체 리소스] 페이지가 열립니다.

  4. [관리 객체 리소스] 페이지에서 [새로 만들기] 단추를 누릅니다.

    [새 관리 객체 리소스] 페이지가 열립니다.

  5. [JNDI 이름] 필드에 관리 객체 리소스를 식별하는 고유한 이름을 입력합니다.

  6. 리소스 어댑터 드롭다운 목록에서 관리 객체 리소스에 대한 리소스 어댑터를 선택합니다.

  7. 리소스 유형 드롭다운 목록에서 관리 객체 리소스에 대한 Java 유형을 선택합니다.

  8. 클래스 이름 드롭다운 목록에서 리소스 유형과 연관된 구현 클래스 이름을 선택합니다.

  9. [설명] 필드에 관리 객체 리소스에 대한 설명을 입력합니다.

  10. 관리 객체 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  11. 추가 속성 표에서 관리 객체 리소스에 대한 추가 속성을 지정합니다.

    사용 가능한 속성은 커넥터 접속 풀의 리소스 어댑터에 따라 다릅니다.

    속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

  12. 사용 가능한 대상 섹션에서 관리 객체가 있게 될 도메인, 클러스터 또는 서버 인스턴스를 선택하고 추가를 누릅니다.

    선택된 필드에 나열된 도메인, 클러스터 또는 서버 인스턴스 중 하나를 제거하려면 필드에서 원하는 대상을 선택하고 [제거]를 누릅니다.

  13. [확인]을 누릅니다.
