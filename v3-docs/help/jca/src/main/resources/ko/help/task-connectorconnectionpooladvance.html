커넥터 연결 풀 고급 속성을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 커넥터 연결 풀 노드를 선택합니다.

    커넥터 연결 풀 페이지가 열립니다.

  4. 커넥터 연결 풀 페이지에서 편집할 커넥터 연결 풀의 이름을 누릅니다.

    커넥터 연결 풀 편집 페이지가 열립니다.

  5. 커넥터 연결 풀 편집 페이지에서 고급 탭을 누릅니다.

    커넥터 연결 풀 고급 속성 편집 페이지가 열립니다.

  6. 최대 1회 검증 필드에서 연결이 최대 1회 검증되기까지의 시간(초)을 입력합니다.

    이 속성을 적절한 값으로 설정하면 연결의 검증 요청 횟수가 최소화됩니다. 기본값은 0입니다. 값이 0이면 속성이 활성화되지 않습니다.

  7. 누수 시간 초과 필드에서 연결 풀의 연결 누수를 추적할 시간(초)을 식별합니다.

    값이 0이면 연결 누수 추적을 사용하지 않음을 나타냅니다. 기본값은 0입니다.

  8. 누수 연결 추적이 완료된 후 누수된 연결을 풀로 복원하려면 누수 재생 확인란을 선택합니다.

    이 옵션은 기본적으로 비활성화됩니다.

  9. 만들기 재시도 횟수 필드에서 새 연결 만들기에 실패할 경우 재시도할 횟수를 입력합니다.

    값이 0이면 연결 만들기를 다시 시도하지 않음을 의미합니다. 기본값은 0입니다.

  10. 재시도 간격에서 연결을 만들기 위한 두 시도 간의 간격(초)을 입력합니다.

    이 속성은 만들기 재시도 횟수의 값이 0보다 큰 경우에만 사용됩니다. 기본값은 10초입니다.

  11. 풀에 대한 연결 풀링을 활성화하려면 풀링 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  12. 연결에서 작업을 수행할 때 연결이 지연되도록 하려면 느린 연결 확인란을 선택합니다.

    트랜잭션이 완료되고 구성 요소 메소드가 종료되면 연결이 해제되어 물리적 연결의 재사용을 돕습니다. 이 옵션은 기본적으로 비활성화됩니다.

  13. 실제 메소드에서 사용할 경우에만 트랜잭션에 자원을 등록하려면 지연 연결 참여 확인란을 선택합니다.

    이 옵션은 기본적으로 비활성화됩니다.

  14. 연결을 스레드와 연결하려면 스레드와 연결 확인란을 선택합니다.

    동일한 스레드에서 연결을 필요로 하는 경우 이미 연결된 연결을 다시 사용할 수 있으므로 풀에서 연결을 가져오기 위한 오버헤드가 발생하지 않습니다. 이 옵션은 기본적으로 비활성화됩니다.

  15. 풀에 대해 일치하는 연결을 활성화하려면 연결 일치 확인란을 선택합니다.

    풀의 연결이 항상 같은 종류이며 따라서 풀에서 선택된 연결을 자원 어댑터와 일치시킬 필요가 없음을 알고 있는 경우 이 옵션을 비활성화합니다. 이 옵션은 기본적으로 활성화됩니다.

  16. 연결 사용 일치 필드에 풀에서 연결을 재사용할 횟수를 입력합니다.

    연결을 지정된 횟수만큼 재사용한 다음에는 해당 연결이 닫힙니다. 값이 0이면 이 속성을 사용하지 않음을 나타냅니다. 기본값은 0입니다.

  17. 저장을 누릅니다.

    참고 - 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.
