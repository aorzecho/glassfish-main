새 커넥터 접속 풀(단계 2/2)

새 커넥터 접속 풀(단계 2/2) 페이지에서는 새 커넥터 접속 풀을 생성할 수 있습니다.

새 커넥터 접속 풀(단계 2/2) 페이지에는 다음 옵션이 포함되어 있습니다.

핑

이 옵션을 사용으로 설정하면 생성 또는 재구성 중에 접속 풀을 핑하여 속성에 대한 모든 오류 값을 식별하고 경고합니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

설명

커넥터 접속 풀에 대한 설명입니다.

초기 및 최소 풀 크기

커넥터 접속 풀의 최소 접속 수입니다. 기본값은 8입니다.

최대 풀 크기

커넥터 접속 풀의 최대 접속 수입니다. 기본값은 32입니다.

풀 크기 조정 개수

풀 유휴 시간 초과가 만료될 때 제거할 접속 수입니다. 기본값은 2입니다.

유휴 시간 초과

풀에서 접속이 유휴 상태로 있을 수 있는 최대 시간(초)입니다. 기본값은 300입니다.

최대 대기 시간

접속을 요청한 응용 프로그램이 접속 시간 초과까지 대기하는 시간입니다. 기본값은 60000입니다.

접속 검증

[필요] 체크 박스가 선택된 경우 접속을 응용 프로그램에 전달하기 전에 먼저 검증합니다. 리소스의 검증이 실패하면 리소스가 삭제되고, 새 리소스가 생성된 후 반환됩니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

실패 시

[모든 접속 닫기] 체크 박스가 선택된 경우 GlassFish Server은(는) 접속이 하나라도 실패하면 풀의 모든 접속을 닫은 후 재설정합니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

트랜잭션 지원

선택한 트랜잭션 지원은 이 접속 풀과 연관된 리소스 어댑터의 트랜잭션 지원 속성을 역호환 방식으로 대체합니다. 즉, 리소스 어댑터에 지정한 레벨보다 낮은 트랜잭션 레벨을 지원하거나 리소스 어댑터에서 지정한 것과 같은 트랜잭션 레벨을 지원할 수 있지만 더 높은 레벨은 지정할 수 없습니다.

다음과 같은 값을 사용할 수 있습니다.

XATransaction

이 리소스 어댑터는 LocalTransaction 및 XAResource 인터페이스를 구현하여 리소스 관리자 로컬 및 JTA 트랜잭션을 지원합니다. XA 트랜잭션은 리소스 관리자 외부에 있는 트랜잭션 관리자가 제어 및 조정합니다.

LocalTransaction

이 리소스 어댑터는 LocalTransaction 인터페이스를 구현하여 로컬 트랜잭션을 지원합니다. 로컬 트랜잭션은 리소스 관리자 내부로 관리되고 외부 트랜잭션 관리자를 포함시키지 않습니다.

NoTransaction

이 리소스 어댑터는 리소스 관리자 로컬 또는 JTA 트랜잭션을 지원하지 않으며 XAResource 또는 LocalTransaction 인터페이스를 구현하지 않습니다.

추가 속성

커넥터 접속 풀에 대한 추가 속성입니다. 사용 가능한 속성은 커넥터 접속 풀의 리소스 어댑터에 따라 다릅니다. 이 표에 지정된 이름-값 쌍을 사용하여 리소스 어댑터 공급업체에서 정의한 속성의 기본값을 대체할 수 있습니다.

속성의 이름은 지정하고 값은 지정하지 않으면 접속 풀 생성을 마칠 때 속성이 표에서 제거됩니다. 지금은 속성값을 설정하지 않지만 나중에 설정할 수 있도록 속성을 테이블에 유지하려면 [값] 필드에 일련의 빈 괄호를 입력합니다.
