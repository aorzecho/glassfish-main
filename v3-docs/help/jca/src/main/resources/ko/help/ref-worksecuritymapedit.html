작업 보안 맵 편집

작업 보안 맵 편집 페이지에서는 작업 보안 맵에 대한 설정을 수정할 수 있습니다.

작업 보안 맵 편집 페이지에는 다음 옵션이 포함되어 있습니다.

보안 맵 이름

작업 보안 맵의 이름입니다. 이름은 읽기 전용 필드입니다. 새 작업 보안 맵을 생성할 때만 보안 맵 이름을 지정할 수 있습니다.

그룹

[그룹] 라디오 단추가 선택된 경우 텍스트 필드는 각 백엔드 EIS 사용자 그룹과 GlassFish Server 보안 도메인에 정의된 해당 사용자 그룹 간의 매핑을 지정해야 합니다. 콤마로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

eis-group-1=server-group-1,eis-group-2=server-group-2
주체

[주체] 라디오 단추가 선택된 경우 텍스트 필드는 각 백엔드 EIS 주체와 GlassFish Server 보안 도메인에 정의된 해당 주체 간의 매핑을 지정해야 합니다. 콤마로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
리소스 어댑터

EIS에 대한 리소스 어댑터입니다.

설명

작업 보안 맵에 대한 설명입니다.

상태

이 옵션을 선택하면 작업 보안 맵이 사용으로 설정됩니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.
