커넥터 연결 풀 편집

커넥터 연결 풀 편집 페이지에서는 커넥터 연결 풀에 대한 설정을 수정할 수 있습니다.

커넥터 연결 풀 편집 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값이 있는 설정을 해당 기본값으로 복원하는 버튼입니다. 기본값이 없는 설정은 변경되지 않습니다.

플러시

연결 풀의 연결을 다시 초기화하는 버튼입니다. 이 버튼을 누르면 기존의 모든 연결이 삭제되고 풀에 설정된 연결이 다시 만들어진 후 풀이 초기 및 최소 풀 크기로 복원됩니다.

Ping

연결 풀을 사용할 수 있는지 확인하는 버튼입니다.

풀 이름

커넥터 연결 풀을 식별하는 고유한 이름입니다. 읽기 전용 필드입니다. 새 커넥터 연결 풀을 만들 때만 풀 이름을 지정할 수 있습니다.

자원 어댑터

커넥터 연결 풀과 연결된 자원 어댑터입니다.

연결 정의

커넥터 연결 풀에 대한 연결 정의입니다. 목록의 선택 항목은 선택한 자원 어댑터에 따라 다릅니다. 연결 정의는 ra.xml 파일에서 자원 어댑터의 connection-definition 요소를 식별합니다.

Ping

이 옵션을 활성화하면 만들기 또는 재구성 중에 연결 풀을 ping하여 속성에 대한 모든 오류 값을 식별하고 경고합니다. 이 옵션은 기본적으로 비활성화됩니다.

설명

커넥터 연결 풀에 대한 설명입니다.

초기 및 최소 풀 크기

커넥터 연결 풀의 최소 연결 수입니다. 기본값은 8입니다.

최대 풀 크기

커넥터 연결 풀의 최대 연결 수입니다. 기본값은 32입니다.

풀 크기 조정 개수

풀 유휴 시간 초과가 만료될 때 제거할 연결 수입니다. 기본값은 2입니다.

유휴 시간 초과

풀에서 연결이 유휴 상태로 있을 수 있는 최대 시간(초)입니다. 기본값은 300입니다.

최대 대기 시간

연결을 요청한 응용 프로그램이 연결 시간 초과까지 대기하는 시간입니다. 기본값은 60000입니다.

연결 검증

필요 확인란이 선택된 경우 연결을 응용 프로그램에 전달하기 전에 먼저 검증합니다. 자원의 검증이 실패하면 자원이 삭제되고, 새 자원이 만들어진 후 반환됩니다. 이 옵션은 기본적으로 비활성화됩니다.

실패 시

모든 연결 닫기 확인란이 선택된 경우 GlassFish Server는 연결이 하나라도 실패하면 풀의 모든 연결을 닫은 후 다시 설정합니다. 이 옵션은 기본적으로 비활성화됩니다.

트랜잭션 지원

선택한 트랜잭션 지원은 이 연결 풀과 연관된 자원 어댑터의 트랜잭션 지원 속성을 역호환 방식으로 대체합니다. 즉, 자원 어댑터에 지정한 수준보다 낮은 트랜잭션 수준을 지원하거나 자원 어댑터에서 지정한 것과 같은 트랜잭션 수준을 지원할 수 있지만 더 높은 수준은 지정할 수 없습니다.

다음과 같은 값을 사용할 수 있습니다.

XATransaction

자원 어댑터는 LocalTransaction 및 XAResource 인터페이스를 구현하여 자원 관리자 로컬 및 JTA 트랜잭션을 지원합니다. 트랜잭션은 자원 관리자 외부에 있는 트랜잭션 관리자가 제어 및 조정합니다.

LocalTransaction

자원 어댑터는 LocalTransaction 인터페이스를 구현하여 로컬 트랜잭션을 지원합니다. 로컬 트랜잭션은 자원 관리자 내부로 관리되고 외부 트랜잭션 관리자를 포함시키지 않습니다.

NoTransaction

자원 어댑터는 자원 관리자 로컬 또는 JTA 트랜잭션을 지원하지 않으며 XAResource 또는 LocalTransaction 인터페이스를 구현하지 않습니다.
