리소스 어댑터 구성

[리소스 어댑터 구성] 페이지에서는 커넥터(리소스 어댑터) 구성을 관리할 수 있습니다.

리소스 어댑터 구성은 리소스 어댑터에 대한 구성 정보를 제공합니다. 관리 콘솔을 사용하여 리소스 어댑터가 배치된 후 리소스 어댑터 구성을 생성합니다. 이 경우 리소스 어댑터는 새 구성으로 재시작됩니다. 리소스 어댑터를 배치하기 전에 리소스 어댑터 구성을 생성하려면 asadmin create-resource-adapter-config 명령을 사용합니다.

리소스 어댑터 구성을 생성하려면 먼저 사용할 리소스 어댑터에 대한 스레드 풀을 생성해야 합니다.

각 리소스 어댑터 구성에 대해 제공되는 정보는 다음과 같습니다.

이름

리소스 어댑터의 이름입니다.

구성 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 리소스 어댑터 구성을 생성하는 단추입니다.

삭제

선택한 리소스 어댑터 구성을 하나 이상 삭제하는 단추입니다.

관련 작업

  • 리소스 어댑터 구성을 생성하는 방법

  • 리소스 어댑터 구성을 편집하는 방법

  • 리소스 어댑터 구성을 삭제하는 방법

관련 asadmin 명령

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)
