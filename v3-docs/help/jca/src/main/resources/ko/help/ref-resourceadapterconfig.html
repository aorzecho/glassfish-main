자원 어댑터 구성

자원 어댑터 구성 페이지에서는 커넥터(자원 어댑터) 구성을 관리할 수 있습니다.

자원 어댑터 구성은 자원 어댑터에 대한 구성 정보를 제공합니다. 관리 콘솔을 사용하여 자원 어댑터가 배포된 후 자원 어댑터 구성을 만듭니다. 이 경우 자원 어댑터는 새 구성으로 다시 시작됩니다. 자원 어댑터를 배포하기 전에 자원 어댑터 구성을 만들려면 asadmin create-resource-adapter-config 명령을 사용합니다.

자원 어댑터 구성을 만들려면 먼저 사용할 자원 어댑터에 대한 스레드 풀을 만들어야 합니다.

각 자원 어댑터 구성에 대해 제공되는 정보는 다음과 같습니다.

이름

자원 어댑터의 이름입니다.

구성 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 자원 어댑터 구성을 만드는 버튼입니다.

삭제

하나 이상의 선택된 자원 어댑터 구성을 삭제하는 버튼입니다.

관련 작업

  • 자원 어댑터 구성을 만드는 방법

  • 자원 어댑터 구성을 편집하는 방법

  • 자원 어댑터 구성을 삭제하는 방법

관련 asadmin 명령

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)
