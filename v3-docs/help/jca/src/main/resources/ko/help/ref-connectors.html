커넥터

커넥터 페이지에서는 다음과 같은 유형의 커넥터 자원을 구성할 수 있습니다.

  • 커넥터 자원

  • 커넥터 연결 풀

  • 관리 객체 자원

  • 작업 보안 맵

Java 구성 요소인 커넥터 모듈(자원 어댑터라고도 함)을 사용하면 응용 프로그램이 EIS(Enterprise Information System) 소프트웨어와 상호 작용할 수 있습니다. EIS 소프트웨어에는 ERP(전사적 자원 관리), 메인프레임 트랜잭션 처리, 비관계형 데이터베이스 및 JMS(Java Message Service)와 같은 메시징 시스템 등의 다양한 시스템 유형이 포함됩니다.

커넥터 연결 풀은 특정 EIS에 대한 재사용 가능한 연결 그룹입니다.

커넥터 자원은 응용 프로그램과의 EIS 연결을 제공하는 프로그램 객체입니다.

관리 객체 자원은 응용 프로그램에 전문적 기능을 제공합니다.

작업 보안 맵은 들어오는 작업 인스턴스와 연결된 기본을 GlassFish 서버의 보안 도메인의 기본에 매핑합니다.
