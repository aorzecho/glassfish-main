커넥터 자원을 만드는 방법

시작하기 전에

커넥터 자원을 만들기 전에 먼저 커넥터 연결 풀을 만듭니다.

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 커넥터 자원 노드를 선택합니다.

    커넥터 자원 페이지가 열립니다.

  4. 커넥터 자원 페이지에서 새로 만들기 버튼을 누릅니다.

    새 커넥터 자원 페이지가 열립니다.

  5. JNDI 이름 필드에 커넥터 자원의 고유한 JNDI 이름을 입력합니다.

    예: eis/myERP

  6. 풀 이름 드롭다운 목록에서 새 커넥터 자원이 속한 연결 풀을 선택합니다.
  7. 설명 필드에 커넥터 자원에 대한 설명을 입력합니다.
  8. 런타임에 커넥터 자원을 활성화하려면 상태 사용 가능 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  9. 사용 가능한 대상에서 커넥터 자원이 있게 될 도메인, 클러스터 또는 서버 인스턴스를 선택하고 추가를 누릅니다.

    커넥터 자원을 선택한 대상에 나열된 도메인, 클러스터 또는 서버 인스턴스 중 하나에 배포하지 않으려면 필드에서 대상을 선택한 다음 제거를 누릅니다.

  10. (옵션) 추가 등록 정보 섹션에서 커넥터 자원에 대한 추가 등록 정보를 지정합니다.

    사용 가능한 등록 정보는 커넥터 연결 풀의 자원 어댑터에 따라 다릅니다.

    등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.

  11. 확인을 누릅니다.
