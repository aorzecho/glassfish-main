커넥터 연결 풀 등록 정보를 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 커넥터 연결 풀 노드를 선택합니다.

    커넥터 연결 풀 페이지가 열립니다.

  4. 커넥터 연결 풀 페이지에서 편집할 커넥터 연결 풀의 이름을 누릅니다.

    커넥터 연결 풀 편집 페이지가 열립니다.

  5. 커넥터 연결 풀 편집 페이지에서 추가 등록 정보 탭을 누릅니다.

    커넥터 연결 풀 등록 정보 편집 페이지가 열립니다.

  6. (옵션) 추가 등록 정보 표에서 커넥터 연결 풀에 대한 추가 등록 정보를 지정합니다.

    지정되는 등록 정보는 이 풀이 사용하는 자원 어댑터에 따라 달라집니다. 이 표를 사용하여 사용자가 지정한 이름-값 쌍을 사용하여 자원 어댑터 공급업체에서 정의한 등록 정보의 기본값을 대체할 수 있습니다.

    등록 정보의 값을 지정하지 않으면 연결 풀 편집을 마칠 때 표에서 제거됩니다. 현재는 등록 정보 값을 설정하지 않지만 나중에 설정할 수 있도록 등록 정보를 테이블에 유지하려면 값 필드에 일련의 빈 괄호를 입력합니다.

    ()
    • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.
    • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
    • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다.
  7. 저장을 누릅니다.
