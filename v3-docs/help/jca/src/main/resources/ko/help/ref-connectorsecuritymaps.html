커넥터 연결 풀 보안 맵

커넥터 연결 풀 보안 맵 페이지에서는 커넥터 연결 풀에 대한 보안 맵을 관리할 수 있습니다.

커넥터 연결 풀용 보안 맵을 사용하면 Java EE 사용자와 그룹, EIS 사용자와 그룹 간의 매핑을 만들 수 있습니다. 보안 맵은 서블릿 또는 EJB 인증 중에 받은 기본을 EIS에서 수락한 자격 증명에 매핑합니다. 여러 GlassFish Server 기본을 동일한 백엔드 기본에 매핑할 수 있습니다.

연결 풀에 대한 보안 맵은 들어오는 작업 인스턴스와 연결된 기본을 GlassFish Server 도메인의 보안 영역에 있는 기본에 매핑하는 커넥터 작업 보안 맵과 다릅니다. 커넥터 작업 보안 맵에 대한 자세한 내용은 작업 보안 맵을 참조하십시오.
