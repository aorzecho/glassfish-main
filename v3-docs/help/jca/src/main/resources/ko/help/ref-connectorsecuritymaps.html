커넥터 접속 풀 보안 맵

[커넥터 접속 풀 보안 맵] 페이지에서는 커넥터 접속 풀에 대한 보안 맵을 관리할 수 있습니다.

커넥터 접속 풀 보안 맵을 사용하면 Java EE 사용자와 그룹, EIS 사용자와 그룹 간의 매핑을 생성할 수 있습니다. 보안 맵은 서블릿 또는 EJB 인증 중에 받은 주체를 EIS에서 수락한 인증서에 매핑합니다. 여러 GlassFish Server 주체를 동일한 백엔드 주체에 매핑할 수 있습니다.

접속 풀에 대한 보안 맵은 들어오는 작업 인스턴스와 연관된 주체를 GlassFish Server 도메인의 보안 영역에 있는 주체에 매핑하는 커넥터 작업 보안 맵과 다릅니다. 커넥터 작업 보안 맵에 대한 자세한 내용은 작업 보안 맵을(를) 참조하십시오.

각 보안 맵에 대해 제공되는 정보는 다음과 같습니다.

이름

보안 맵의 이름입니다.

보안 맵 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 보안 맵을 생성하는 단추입니다.

삭제

선택한 보안 맵을 하나 이상 삭제하는 단추입니다.
