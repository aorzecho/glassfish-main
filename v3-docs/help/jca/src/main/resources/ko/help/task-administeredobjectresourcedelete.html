관리 대상 객체 자원을 삭제하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 관리 객체 자원 노드를 선택합니다.

    관리 객체 자원 페이지가 열립니다.

  4. 관리 객체 자원 페이지에서 삭제할 관리 객체 자원 이름 옆의 확인란을 선택합니다.
  5. 삭제를 누릅니다.

  • 커넥터 연결 풀에 대한 보안 맵을 편집하는 방법

  • 커넥터 연결 풀에 대한 보안 맵을 삭제하는 방법
