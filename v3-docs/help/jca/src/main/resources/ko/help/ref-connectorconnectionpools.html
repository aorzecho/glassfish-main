커넥터 연결 풀

커넥터 연결 풀 페이지에서는 커넥터 연결 풀을 구성할 수 있습니다.

커넥터 연결 풀은 특정 EIS에 대한 재사용 가능한 연결 그룹입니다.

각 자원에 대해 제공되는 정보는 다음과 같습니다.

풀 이름

커넥터 연결 풀을 식별하는 고유한 이름입니다.

자원 어댑터

커넥터 연결 풀과 연결된 자원 어댑터입니다.

연결 정의

커넥터 연결 풀에 대한 연결 정의의 클래스 이름입니다.

설명

커넥터 연결 풀에 대한 설명입니다.

풀 표에는 또한 다음 옵션이 포함되어 있습니다.

새로 만들기

새 커넥터 연결 풀을 만드는 버튼입니다.

삭제

하나 이상의 선택된 커넥터 연결 풀을 삭제하는 버튼입니다.
