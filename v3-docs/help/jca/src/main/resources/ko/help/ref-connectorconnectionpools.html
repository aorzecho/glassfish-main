커넥터 접속 풀

[커넥터 접속 풀] 페이지에서는 커넥터 접속 풀을 구성할 수 있습니다.

커넥터 접속 풀은 특정 EIS에 대한 재사용 가능한 접속 그룹입니다.

각 리소스에 대해 제공되는 정보는 다음과 같습니다.

풀 이름

커넥터 접속 풀을 식별하는 고유한 이름입니다.

리소스 어댑터

커넥터 접속 풀과 연관된 리소스 어댑터입니다.

접속 정의

커넥터 접속 풀에 대한 접속 정의의 클래스 이름입니다.

설명

커넥터 접속 풀에 대한 설명입니다.

풀 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 커넥터 접속 풀을 생성하는 단추입니다.

삭제

선택한 커넥터 접속 풀을 하나 이상 삭제하는 단추입니다.

관련 작업

  • 커넥터 접속 풀을 생성하는 방법

  • 커넥터 접속 풀을 편집하는 방법

  • 커넥터 접속 풀 고급 속성을 편집하는 방법

  • 커넥터 접속 풀 속성을 편집하는 방법

  • 커넥터 접속 풀 보안 맵을 생성하는 방법

  • 커넥터 접속 풀 보안 맵을 편집하는 방법

  • 커넥터 접속 풀 보안 맵을 삭제하는 방법

  • 커넥터 접속 풀을 삭제하는 방법
