새 커넥터 연결 풀 보안 맵

새 커넥터 연결 풀 보안 맵 페이지에서는 커넥터 연결 풀에 대한 새 보안 맵을 만들 수 있습니다.

새 커넥터 연결 풀 보안 맵 페이지에는 다음 옵션이 포함되어 있습니다.

연결 풀 이름

이 보안 맵이 속한 커넥터 연결 풀의 이름입니다. 풀 이름은 읽기 전용 필드입니다.

보안 맵 이름

보안 맵의 이름입니다.

사용자 그룹

이 라디오 버튼을 선택한 경우 텍스트 필드에는 사용자 그룹의 이름이 쉼표로 구분되어 포함되어야 합니다.

기본

이 라디오 버튼을 선택한 경우 텍스트 필드에는 기본의 이름이 쉼표로 구분되어 포함되어야 합니다.

사용자 이름

EIS에 필요한 백엔드 기본 사용자 이름입니다.

암호

EIS에 필요한 백엔드 기본 암호입니다.
