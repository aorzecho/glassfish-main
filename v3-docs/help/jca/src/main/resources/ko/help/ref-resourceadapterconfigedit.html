리소스 어댑터 구성 편집

리소스 어댑터 구성 편집 페이지에서는 리소스 어댑터 구성에 대한 설정을 수정할 수 있습니다.

리소스 어댑터 구성 편집 페이지에는 다음 옵션이 포함되어 있습니다.

리소스 어댑터 이름

구성할 리소스 어댑터의 이름입니다. 이름은 읽기 전용 필드입니다. 새 리소스 어댑터 구성을 생성할 때만 리소스 어댑터 이름을 지정할 수 있습니다.

스레드 풀 ID

리소스 어댑터 구성에 사용할 스레드 풀입니다.

추가 속성

리소스 어댑터에 대한 추가 속성입니다. 사용 가능한 속성은 리소스 어댑터 JavaBeans 구성 요소의 클래스 이름을 정의하는 ra.xml 파일의 resourceadapter-class 요소로 참조되는 클래스의 setter 메소드의 이름입니다. 여기에 정의된 속성에 대한 모든 수정 사항은 ra.xml에 있는 기본값을 대체합니다.

관련 작업

  • 리소스 어댑터 구성을 생성하는 방법

  • 리소스 어댑터 구성을 편집하는 방법

  • 리소스 어댑터 구성을 삭제하는 방법

관련 asadmin 명령

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)
