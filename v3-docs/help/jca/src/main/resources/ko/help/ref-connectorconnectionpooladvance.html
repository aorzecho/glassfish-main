커넥터 연결 풀 고급 속성 편집

커넥터 연결 풀 고급 속성 편집 페이지에서는 커넥터 연결 풀에 대한 고급 설정을 수정할 수 있습니다.

커넥터 연결 풀 고급 속성 편집 페이지에 포함된 옵션은 다음과 같습니다.

기본값 로드

모든 설정의 기본값을 복원하는 버튼입니다.

최대 1회 검증 기간

연결을 최대 한 번 검증하기 위해 연속된 요청 간의 시간 간격(초)을 지정합니다. 기본값은 0입니다. 값이 0이면 속성이 활성화되지 않습니다.

누수 시간 초과

연결 풀의 연결 누수를 추적하는 시간(초)입니다. 값이 0이면 연결 누수 추적을 사용하지 않음을 나타냅니다. 기본값은 0입니다.

누수 재생

이 옵션을 선택하면 누수된 연결이 누수 연결 추적 완료 후 풀로 복원됩니다. 이 옵션은 기본적으로 비활성화됩니다.

만들기 재시도 횟수

새 연결 만들기에 실패한 경우 재시도하는 횟수입니다. 값이 0이면 연결 만들기를 다시 시도하지 않음을 의미합니다. 기본값은 0입니다.

재시도 간격

연결을 만들기 위한 두 시도 간의 간격(초)입니다. 이 속성은 만들기 재시도 횟수의 값이 0보다 큰 경우에만 사용됩니다. 기본값은 10초입니다.

풀링

풀에 대한 연결 풀링을 활성화합니다. 이 옵션은 기본적으로 활성화됩니다.

느린 연결

이 옵션을 선택하면 연결에서 작업을 수행할 때 연결이 지연됩니다. 트랜잭션이 완료되고 구성 요소 메소드가 종료되면 연결이 해제되어 물리적 연결의 재사용을 돕습니다. 이 옵션은 기본적으로 비활성화됩니다.

지연 연결 참여

이 옵션을 선택하면 자원이 메소드에서 실제로 사용될 때만 트랜잭션에 등록됩니다. 이 옵션은 기본적으로 비활성화됩니다.

스레드와 연결

이 옵션을 선택하면 연결이 스레드와 연결됩니다. 동일한 스레드에서 연결을 필요로 하는 경우 이미 연결된 연결을 다시 사용할 수 있으므로 풀에서 연결을 가져오기 위한 오버헤드가 발생하지 않습니다. 이 옵션은 기본적으로 비활성화됩니다.

연결 일치

이 옵션을 선택하면 풀에 대해 연결 일치가 활성화됩니다. 풀의 연결이 항상 같은 종류이며 따라서 풀에서 선택된 연결을 자원 어댑터와 일치시킬 필요가 없음을 알고 있는 경우 이 옵션을 비활성화할 수 있습니다. 이 옵션은 기본적으로 활성화됩니다.

연결 사용 일치

풀에서 연결을 재사용할 횟수입니다. 연결을 지정된 횟수만큼 재사용한 다음에는 해당 연결이 닫힙니다. 값이 0이면 이 속성을 사용하지 않음을 나타냅니다. 기본값은 0입니다.
