작업 보안 맵을 만드는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 작업 보안 맵 노드를 선택합니다.

    작업 보안 맵 페이지가 열립니다.

  4. 작업 보안 맵 페이지에서 새로 만들기를 누릅니다.

    새 작업 보안 맵 페이지가 열립니다.

  5. 보안 맵 이름 필드에 작업 보안 맵의 이름을 입력합니다.
  6. 다음과 같이 그룹 필드 또는 기본값 필드를 사용합니다.
    • 그룹 라디오 버튼을 선택한 경우, 각 백엔드 EIS 사용자 그룹과 GlassFish 서버 보안 도메인에서 정의된 해당 사용자 그룹 간의 매핑을 입력합니다.

      쉼표로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • 기본값 라디오 버튼을 선택한 경우, 각 백엔드 EIS 기본과 GlassFish 서버 보안 도메인에서 정의한 해당 기본 간의 매핑을 입력합니다.

      쉼표로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  7. 자원 어댑터 드롭다운 목록에서 EIS에 대한 자원 어댑터를 선택합니다.
  8. 설명 필드에 작업 보안 맵에 대한 설명을 입력합니다.
  9. 작업 보안 맵을 활성화하려면 상태 사용 가능 확인란을 선택합니다.

    이 옵션은 기본적으로 활성화됩니다.

  10. 확인을 누릅니다.
