커넥터 서비스를 구성하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 [커넥터 서비스] 노드를 선택합니다.

    커넥터 서비스 페이지가 열립니다.

  3. [종료 시간 초과] 필드에 GlassFish Server 종료 중 커넥터 모듈 인스턴스의 ResourceAdapter.stop 메소드가 완료될 때까지 허용되는 최대 시간(초)을 입력합니다.

    종료하는 데 시간이 더 오래 걸리는 리소스 어댑터는 무시되며 GlassFish Server 종료가 계속됩니다. 기본값은 30초입니다.

  4. 커넥터 클래스 로딩 정책 드롭다운 목록에서 사용할 정책을 선택합니다.

    • 리소스 어댑터의 리소스에 대한 응용 프로그램의 참조에 따라 리소스 어댑터가 제공되도록 지정하려면 derived를 선택합니다.

      이 정책은 기본값입니다.

    • 모든 응용 프로그램에 모든 리소스 어댑터가 표시되도록 지정하려면 global을 선택합니다.

    이 정책은 항상 모든 응용 프로그램에 표시되는 사전 설치된 JMS 시스템 리소스 어댑터 jmsra에는 적용되지 않습니다.

  5. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.
