새 커넥터 연결 풀(1/2단계)

새 커넥터 연결 풀(1/2단계) 페이지에서는 새 커넥터 연결 풀을 만들 수 있습니다.

새 커넥터 연결 풀(1/2단계) 페이지에는 다음 옵션이 포함되어 있습니다.

이름

커넥터 연결 풀을 식별하는 고유한 이름입니다.

자원 어댑터

커넥터 연결 풀과 연결된 자원 어댑터입니다.

연결 정의

커넥터 연결 풀에 대한 연결 정의입니다. 목록의 선택 항목은 선택한 자원 어댑터에 따라 다릅니다. 연결 정의는 ra.xml 파일에서 자원 어댑터의 connection-definition 요소를 식별합니다.

관련 작업

  • 커넥터 연결 풀을 만드는 방법

  • 커넥터 연결 풀을 편집하는 방법

  • 커넥터 연결 풀 고급 속성을 편집하는 방법

  • 커넥터 연결 풀 등록 정보를 편집하는 방법

  • 커넥터 연결 풀에 대한 보안 맵을 만드는 방법

  • 커넥터 연결 풀에 대한 보안 맵을 편집하는 방법

  • 커넥터 연결 풀에 대한 보안 맵을 삭제하는 방법

  • 커넥터 연결 풀을 삭제하는 방법

관련 asadmin 명령

  • create-connector-connection-pool(1)

  • list-connector-connection-pools(1)

  • delete-connector-connection-pool(1)
