새 커넥터 접속 풀(단계 1/2)

새 커넥터 접속 풀(단계 1/2) 페이지에서는 새 커넥터 접속 풀을 생성할 수 있습니다.

새 커넥터 접속 풀(단계 1/2) 페이지에는 다음 옵션이 포함되어 있습니다.

이름

커넥터 접속 풀을 식별하는 고유한 이름입니다.

리소스 어댑터

커넥터 접속 풀과 연관된 리소스 어댑터입니다.

접속 정의

커넥터 접속 풀에 대한 접속 정의입니다. 목록의 선택 항목은 선택한 리소스 어댑터에 따라 다릅니다. 접속 정의는 ra.xml 파일에서 리소스 어댑터의 connection-definition 요소를 식별합니다.

관련 작업

  • 커넥터 접속 풀을 생성하는 방법

  • 커넥터 접속 풀을 편집하는 방법

  • 커넥터 접속 풀 고급 속성을 편집하는 방법

  • 커넥터 접속 풀 속성을 편집하는 방법

  • 커넥터 접속 풀 보안 맵을 생성하는 방법

  • 커넥터 접속 풀 보안 맵을 편집하는 방법

  • 커넥터 접속 풀 보안 맵을 삭제하는 방법

  • 커넥터 접속 풀을 삭제하는 방법

관련 asadmin 명령

  • create-connector-connection-pool(1)

  • list-connector-connection-pools(1)

  • delete-connector-connection-pool(1)
