커넥터 접속 풀 보안 맵을 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [커넥터 접속 풀] 노드를 선택합니다.

    [커넥터 접속 풀] 페이지가 열립니다.

  4. [커넥터 접속 풀] 페이지에서 편집할 접속 풀의 이름을 누릅니다.

    [커넥터 접속 풀 편집] 페이지가 열립니다.

  5. [커넥터 접속 풀 편집] 페이지에서 [보안 맵] 탭을 누릅니다.

    [커넥터 접속 풀 보안 맵] 페이지가 열립니다.

  6. [커넥터 접속 풀 보안 맵] 페이지에서 편집할 보안 맵의 이름을 누릅니다.

    [커넥터 접속 풀 보안 맵 편집] 페이지가 열립니다.

  7. 다음 중 하나를 수행합니다.

    • [사용자 그룹] 라디오 단추를 선택하고 사용자 그룹의 이름을 콤마로 구분하여 입력합니다.

    • [주체] 라디오 단추를 선택하고 주체의 이름을 콤마로 구분하여 입력합니다.

  8. [사용자 이름] 필드에 EIS에 필요한 백엔드 주체 사용자 이름을 지정합니다.

  9. [비밀번호] 필드에 EIS에 필요한 백엔드 주체 비밀번호를 지정합니다.

  10. [저장]을 누릅니다.
