커넥터 연결 풀 등록 정보 편집

커넥터 연결 풀 등록 정보 편집 페이지에서는 커넥터 연결 풀에 대한 등록 정보를 수정할 수 있습니다.

커넥터 연결 풀 등록 정보 편집 페이지에는 다음 옵션이 포함되어 있습니다.

추가 등록 정보

커넥터 연결 풀에 대한 추가 등록 정보입니다. 사용 가능한 등록 정보는 커넥터 연결 풀의 자원 어댑터에 따라 다릅니다. 이 표에 지정된 이름-값 쌍을 사용하여 자원 어댑터 공급업체에서 정의한 등록 정보의 기본값을 대체할 수 있습니다.

등록 정보의 값을 지정하지 않으면 연결 풀 편집을 마칠 때 표에서 제거됩니다. 현재는 등록 정보 값을 설정하지 않지만 나중에 설정할 수 있도록 등록 정보를 테이블에 유지하려면 값 필드에 일련의 빈 괄호를 입력합니다.
