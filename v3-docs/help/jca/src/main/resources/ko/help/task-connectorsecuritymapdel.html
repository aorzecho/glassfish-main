커넥터 접속 풀 보안 맵을 삭제하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [커넥터 접속 풀] 노드를 선택합니다.

    [커넥터 접속 풀] 페이지가 열립니다.

  4. [커넥터 접속 풀] 페이지에서 삭제할 보안 맵이 있는 접속 풀의 이름을 누릅니다.

    [커넥터 접속 풀 편집] 페이지가 열립니다.

  5. [보안 맵] 탭을 누릅니다.

    [커넥터 접속 풀 보안 맵] 페이지가 열립니다.

  6. [커넥터 접속 풀 보안 맵] 페이지에서 삭제할 보안 맵 이름 옆의 체크 박스를 선택합니다.

  7. [삭제]를 누릅니다.
