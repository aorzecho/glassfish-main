새 작업 보안 맵

새 작업 보안 맵 페이지에서는 새 작업 보안 맵을 만들 수 있습니다.

새 작업 보안 맵 페이지에는 다음 옵션이 포함되어 있습니다.

보안 맵 이름

작업 보안 맵의 이름입니다.

그룹

그룹 매핑 라디오 버튼이 선택된 경우 텍스트 필드는 각 백엔드 EIS 사용자 그룹과 GlassFish Server 보안 도메인에 정의된 해당 사용자 그룹 간의 매핑을 지정해야 합니다. 쉼표로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

eis-group-1=server-group-1,eis-group-2=server-group-2
기본값

기본값 매핑 라디오 버튼이 선택된 경우 텍스트 필드는 각 백엔드 EIS 기본과 GlassFish Server 보안 도메인에 정의된 해당 기본 간의 매핑을 지정해야 합니다. 쉼표로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
자원 어댑터

EIS에 대한 자원 어댑터입니다.

설명

작업 보안 맵에 대한 설명입니다.
