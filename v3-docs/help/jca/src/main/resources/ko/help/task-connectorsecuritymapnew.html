커넥터 연결 풀에 대한 보안 맵을 만드는 방법

커넥터 연결 풀용 보안 맵을 사용하면 Java EE 사용자와 그룹, EIS 사용자와 그룹 간의 매핑을 만들 수 있습니다.

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 커넥터 노드를 확장합니다.
  3. 커넥터 노드에서 커넥터 연결 풀 노드를 선택합니다.

    커넥터 연결 풀 페이지가 열립니다.

  4. 커넥터 연결 풀 페이지에서 편집할 연결 풀의 이름을 누릅니다.

    커넥터 연결 풀 편집 페이지가 열립니다.

  5. 커넥터 연결 풀 편집 페이지에서 보안 맵 탭을 누릅니다.

    커넥터 연결 풀 보안 맵 페이지가 열립니다.

  6. 커넥터 연결 풀 보안 맵 페이지에서 새로 만들기를 누릅니다.

    새 커넥터 연결 풀 보안 맵 페이지가 열립니다.

  7. 보안 맵 이름 필드에 보안 맵의 이름을 입력합니다.
  8. 다음 중 하나를 수행합니다.
    • 사용자 그룹 라디오 버튼을 선택하고 사용자 그룹의 이름을 쉼표로 구분하여 입력합니다.
    • 기본 라디오 버튼을 선택하고 기본의 이름을 쉼표로 구분하여 입력합니다.
  9. 사용자 이름 필드에 EIS에 필요한 백엔드 기본 사용자 이름을 지정합니다.
  10. 암호 필드에 EIS에 필요한 백엔드 기본 암호를 지정합니다.
  11. 확인을 누릅니다.
  12. GlassFish 서버를 다시 시작합니다.
