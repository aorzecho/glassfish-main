コネクタリソースを編集

コネクタリソースの設定を変更するには、「コネクタリソースを編集」ページを使用します。

「コネクタリソースを編集」ページには、次のオプションが含まれます。

デフォルトを読み込み

すべての設定の値をデフォルトに戻すボタン。

JNDI 名

コネクタリソースを識別する一意の名前。JNDI 名は読み取り専用フィールドです。JNDI 名を指定できるのは、新しいコネクタリソースの作成時のみです。

プール名

このリソースに関連付けられたコネクタ接続プール。

説明

コネクタリソースの説明。

ステータス

このオプションを選択すると、コネクタリソースが有効になります。このオプションはデフォルトで有効になっています。

追加プロパティー

コネクタリソースの追加プロパティー。使用可能なプロパティーは、コネクタ接続プールのリソースアダプタによって異なります。この表で指定する名前と値のペアを使用して、リソースアダプタのベンダーが定義したプロパティーのデフォルト値を上書きすることができます。
