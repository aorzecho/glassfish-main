コネクタ

次のタイプのコネクタリソースを設定するには、「コネクタ」ページを使用します。

  • 「コネクタリソース」

  • 「コネクタ接続プール」

  • 「管理オブジェクトリソース」

  • 「作業セキュリティーマップ」

コネクタモジュールは、アプリケーションと EIS (Enterprise Information System) ソフトウェアとの対話を可能にする Java コンポーネントで、リソースアダプタとも呼ばれます。EIS ソフトウェアには、統合業務ソフトウェア (ERP)、メインフレームトランザクション処理、非リレーショナルデータベース、Java Message Service (JMS) などのメッセージングシステムなど、さまざまなタイプのシステムが含まれます。

コネクタ接続プールとは、特定の EIS のための再利用可能な接続のグループです。

コネクタリソースとは、アプリケーションに EIS への接続を提供するプログラムオブジェクトです。

管理対象オブジェクトリソースは、アプリケーションに特別な機能を提供します。

作業セキュリティーマップは、受け取った作業インスタンスに関連付けられている主体を、GlassFish Server のセキュリティードメインにある主体にマッピングします。
