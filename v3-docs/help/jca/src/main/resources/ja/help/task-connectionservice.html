コネクタサービスを設定する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードで、「コネクタサービス」ノードを選択します。

    「コネクタサービス」ページが開きます。

  3. 「シャットダウンタイムアウト」フィールドに、GlassFish サーバー のシャットダウン中に、コネクタモジュールのインスタンスの ResourceAdapter.stop メソッドが完了までに待機する最大時間を秒単位で入力します。

    シャットダウンにかかる時間がこの値を超えるリソースアダプタは無視され、GlassFish サーバー はシャットダウンを継続します。デフォルト値は 30 秒です。

  4. 「コネクタ Classloading のポリシー」ドロップダウンリストから、使用するポリシーを選択します。
    • derived を選択すると、リソースアダプタのリソースを参照するアプリケーションに従って、リソースアダプタが提供されます。

      このポリシーがデフォルトです。

    • global を選択すると、すべてのリソースアダプタがすべてのアプリケーションからアクセス可能になります。

    このポリシーは、プリインストールされた JMS システムリソースアダプタの jmsra には適用されません。このリソースアダプタは、常にすべてのアプリケーションから参照可能です。

  5. 「保存」をクリックします。

    ヒント - デフォルト設定に戻すには、「デフォルトを読込み」ボタンをクリックし、「保存」を保存します。
