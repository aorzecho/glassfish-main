作業セキュリティーマップを作成する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「コネクタ」ノードを開きます。
  3. 「コネクタ」ノードで、「作業セキュリティーマップ」ノードを選択します。

    「作業セキュリティーマップ」ページが開きます。

  4. 「作業セキュリティーマップ」ページで、「新規」をクリックします。

    「New Work Security Map」ページが開きます。

  5. 「セキュリティーマップ名」フィールドに、作業セキュリティーマップの名前を入力します。
  6. 「グループ」フィールドまたは「主体」フィールドのいずれかを、次のように使用します。
    • 「グループ」ラジオボタンを選択する場合は、各バックエンド EIS のユーザーグループと、GlassFish サーバー セキュリティードメインで定義された対応するユーザーグループのマッピングを入力します。

      コンマ区切りのリストを使用して、複数のマッピングを指定します。次に例を示します。

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • 「主体」ラジオボタンを選択する場合は、各バックエンド EIS の主体と、GlassFish サーバー セキュリティードメインで定義された対応する主体のマッピングを入力します。

      コンマ区切りのリストを使用して、複数のマッピングを指定します。次に例を示します。

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  7. 「リソースアダプタ」ドロップダウンリストから、EIS のリソースアダプタを選択します。
  8. 「説明」フィールドに、作業セキュリティーマップの説明を入力します。
  9. 「状態を有効」チェックボックスを選択して、作業セキュリティーマップを有効にします。

    このオプションはデフォルトで有効になっています。

  10. 「了解」をクリックします。
