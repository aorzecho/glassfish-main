作業セキュリティーマップを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「コネクタ」ノードを展開します。
  3. 「コネクタ」ノードで、「作業セキュリティーマップ」ノードを選択します。

    「作業セキュリティーマップ」ページが開きます。

  4. 「作業セキュリティーマップ」ページで、削除する作業セキュリティーマップの名前の隣にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
