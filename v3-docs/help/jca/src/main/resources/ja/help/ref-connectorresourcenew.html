新しいコネクタリソース

新しいコネクタリソースを作成するには、「新しいコネクタリソース」ページを使用します。

「新しいコネクタリソース」ページには、次のオプションが含まれます。

JNDI 名

コネクタリソースを識別する一意の名前。

プール名

新しいコネクタリソースを作成する接続プール。

説明

コネクタリソースの説明。

ステータス

このオプションを選択すると、コネクタリソースが有効になります。このオプションはデフォルトで有効になっています。

追加プロパティー

コネクタリソースの追加プロパティー。使用可能なプロパティーは、コネクタ接続プールのリソースアダプタによって異なります。この表で指定する名前と値のペアを使用して、リソースアダプタのベンダーが定義したプロパティーのデフォルト値を上書きすることができます。

ターゲット

コネクタリソースのクラスタおよびスタンドアロンインスタンス。必要なターゲットを「選択したターゲット」列に移動するには、「追加」ボタンまたは「すべて追加」ボタンを使用します。不要なターゲットを「利用可能なターゲット」列に移動するには、「削除」ボタンまたは「すべて削除」ボタンを使用します。

このオプションは、ドメイン内にクラスタまたはスタンドアロンインスタンスが作成されている場合にのみ表示されます。
