EISアクセスを設定する

  1. コネクタをデプロイ(インストール)します。「コネクタ・モジュールをデプロイする」を参照してください。

  2. コネクタの接続プールを作成します。コネクタ接続プールを作成するを参照してください。

  3. 接続プールに関連付けるコネクタ・リソースを作成します。コネクタ・リソースの作成を参照してください。
