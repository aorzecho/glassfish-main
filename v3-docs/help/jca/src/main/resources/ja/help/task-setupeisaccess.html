EIS アクセスをセットアップする

  1. コネクタを配備 (インストール) します。コネクタモジュールを配備するを参照してください。
  2. コネクタの接続プールを作成します。「コネクタ接続プールを作成する」を参照してください。
  3. 接続プールに関連付けるコネクタリソースを作成します。「コネクタリソースを作成する」参照してください。
