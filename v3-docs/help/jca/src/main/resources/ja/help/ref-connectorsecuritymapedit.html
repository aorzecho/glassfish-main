コネクタ接続プールのセキュリティーマップを編集

コネクタ接続プールのセキュリティーマップを編集するには、「コネクタ接続プールのセキュリティーマップを編集」ページを使用します。

「コネクタ接続プールのセキュリティーマップを編集」ページには、次のオプションが含まれます。

接続プール名

このセキュリティーマップを作成するコネクタ接続プールの名前。プール名は読み取り専用フィールドです。

セキュリティーマップ名

セキュリティーマップの名前。名前は読み取り専用フィールドです。セキュリティーマップ名を指定できるのは、新しいセキュリティーマップの作成時のみです。

セキュリティーマップ名

セキュリティーマップの名前

ユーザーグループ

このラジオボタンを選択する場合は、テキストフィールドにユーザーグループの名前をコンマで区切って入力する必要があります。

主体

このラジオボタンを選択する場合は、テキストフィールドに主体の名前をコンマで区切って入力する必要があります。

ユーザー名

EIS で必要なバックエンド主体のユーザー名。

パスワード

EIS で必要なバックエンド主体のパスワード。
