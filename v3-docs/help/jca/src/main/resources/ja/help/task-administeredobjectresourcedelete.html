管理対象オブジェクトリソースを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「コネクタ」ノードを展開します。
  3. 「コネクタ」ノードで、「管理オブジェクトリソース」ノードを選択します。

    「管理オブジェクトリソース」ページが開きます。

  4. 「管理オブジェクトリソース」ページで、削除する管理対象オブジェクトリソースの名前の隣のチェックボックスを選択します。
  5. 「削除」をクリックします。
