新しいコネクタ接続プール (手順 1/2)

新しいコネクタ接続プールを作成するには、「新しいコネクタ接続プール (手順 1/2)」ページを使用します。

「新しいコネクタ接続プール (手順 1/2)」ページには、次のオプションが含まれます。

名前

コネクタ接続プールを識別する一意の名前。

リソースアダプタ

コネクタ接続プールに関連付けられたリソースアダプタ。

接続定義

コネクタ接続プールの接続定義。リストに表示される項目は、選択したリソースアダプタによって異なります。接続定義は、ra.xml ファイルに含まれるリソースアダプタの connection-definition 要素を識別します。
