リソース・アダプタ構成

コネクタ(リソース・アダプタ)の構成を管理するには、「リソース・アダプタ構成」ページを使用します。

リソース・アダプタ構成は、リソース・アダプタの構成情報を提供します。リソース・アダプタをデプロイした後、管理コンソールを使用してリソース・アダプタ構成を作成します。この場合、リソース・アダプタは新しい構成で再起動されます。リソース・アダプタをデプロイする前にリソース・アダプタ構成を作成するには、asadmin create-resource-adapter-configコマンドを使用します。

リソース・アダプタ構成を作成する前に、使用するリソース・アダプタのスレッド・プールを作成する必要があります。

リソース・アダプタ構成ごとに、次の情報が表示されます。

名前

リソース・アダプタの名前。

「構成」の表には、次のオプションも表示されます。

新規

新しいリソース・アダプタ構成を作成するボタン。

削除

選択した1つまたは複数のリソース・アダプタ構成を削除するボタン。
