リソースアダプタ設定

コネクタ (リソースアダプタ) の設定を管理するには、「リソースアダプタ設定」ページを使用します。

リソースアダプタ設定は、リソースアダプタの設定情報を提供します。リソースアダプタを配備したあと、管理コンソール を使用してリソースアダプタ設定を作成します。この場合、リソースアダプタは新しい設定で再起動されます。リソースアダプタを配備する前にリソースアダプタ設定を作成するには、asadmin create-resource-adapter-config コマンドを使用します。

リソースアダプタ設定を作成する前に、使用するリソースアダプタのスレッドプールを作成する必要があります。

リソースアダプタ設定ごとに、次の情報が表示されます。

名前

リソースアダプタの名前。

「設定」の表には、次のオプションも表示されます。

新規

新しいリソースアダプタ設定を作成するボタン。

削除

選択した 1 つまたは複数のリソースアダプタ設定を削除するボタン。

関連操作

  • 「リソースアダプタ設定を作成する」

  • 「リソースアダプタ設定を編集する」

  • 「リソースアダプタ設定を削除する」

関連する asadmin コマンド

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)
