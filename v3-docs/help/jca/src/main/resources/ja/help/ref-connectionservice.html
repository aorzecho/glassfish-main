コネクタ・サービス

リソース・アダプタを管理するコネクタ・サービスの一般設定を変更するには、「コネクタ・サービス」ページを使用します。

「コネクタ・サービス」ページには、次のオプションが含まれます。

デフォルトをロード

デフォルト値を持つ設定をデフォルト値に戻すボタン。デフォルト値を持たない設定は変更されません。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

シャットダウン・タイムアウト

GlassFish Serverのシャットダウン中に、コネクタ・モジュールのインスタンスのResourceAdapter.stopメソッドが完了するまでに待機する最大時間(秒単位)。シャットダウンにかかる時間がこの値を超えるリソース・アダプタは無視され、GlassFish Serverはシャットダウンを継続します。デフォルト値は30秒です。

コネクタClassloadingのポリシー

クラスのロードに使用するポリシー。選択可能な値は次のとおりです:

derived

リソース・アダプタのリソースを参照するアプリケーションに従って、リソース・アダプタが提供されることを示します。このポリシーがデフォルトです。

global

すべてのリソース・アダプタが、すべてのアプリケーションから参照可能であることを示します。

このポリシーは、プリインストールされたJMSシステム・リソース・アダプタのjmsraには適用されません。このリソース・アダプタは、常にすべてのアプリケーションから参照可能です。
