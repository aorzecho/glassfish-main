コネクタサービス

リソースアダプタを管理するコネクタサービスの一般設定を変更するには、「コネクタサービス」ページを使用します。

「コネクタサービス」ページには、次のオプションが含まれます。

デフォルトを読み込み

すべての設定の値をデフォルトに戻すボタン。

設定名

このページの設定が適用される設定の名前。このフィールドは読み取り専用です。

シャットダウンタイムアウト

GlassFish サーバー のシャットダウン中に、コネクタモジュールのインスタンスの ResourceAdapter.stop メソッドが完了するまでに待機する最大時間 (秒単位)。シャットダウンにかかる時間がこの値を超えるリソースアダプタは無視され、GlassFish サーバー はシャットダウンを継続します。デフォルト値は 30 秒です。

コネクタ Classloading のポリシー

クラスの読み込みに使用するポリシー。次のタイプを指定できます。

derived

リソースアダプタのリソースを参照するアプリケーションに従って、リソースアダプタを提供します。このポリシーがデフォルトです。

global

すべてのリソースアダプタが、すべてのアプリケーションから参照可能です。

このポリシーは、プリインストールされた JMS システムリソースアダプタの jmsra には適用されません。このリソースアダプタは、常にすべてのアプリケーションから参照可能です。
