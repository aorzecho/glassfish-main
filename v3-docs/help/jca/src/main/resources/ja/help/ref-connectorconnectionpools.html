コネクタ接続プール

コネクタ接続プールを設定するには、「コネクタ接続プール」ページを使用します。

コネクタ接続プールとは、特定の EIS のための再利用可能な接続のグループです。

リソースごとに、次の情報が表示されます。

プール名

コネクタ接続プールを識別する一意の名前。

リソースアダプタ

コネクタ接続プールに関連付けられたリソースアダプタ。

接続定義

コネクタ接続プールの接続定義のクラス名。

説明

コネクタ接続プールの説明。

「プール」の表には、次のオプションも表示されます。

新規

新しいコネクタ接続プールを作成するボタン。

削除

選択した 1 つまたは複数のコネクタ接続プールを削除するボタン。

関連操作

  • 「コネクタ接続プールを作成する」

  • 「コネクタ接続プールを編集する」

  • 「コネクタ接続プールの詳細属性を編集する」

  • 「コネクタ接続プールのプロパティーを編集する」

  • 「コネクタ接続プールのセキュリティーマップを作成する」

  • 「コネクタ接続プールのセキュリティーマップを編集する」

  • 「コネクタ接続プールのセキュリティーマップを削除する」

  • 「コネクタ接続プールを削除する」
