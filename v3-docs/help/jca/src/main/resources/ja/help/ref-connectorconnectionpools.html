コネクタ接続プール

コネクタ接続プールを構成するには、「コネクタ接続プール」ページを使用します。

コネクタ接続プールとは、特定のEISのための再利用可能な接続のグループです。

リソースごとに、次の情報が表示されます。

プール名

コネクタ接続プールを識別する一意の名前。

リソース・アダプタ

コネクタ接続プールに関連付けられたリソース・アダプタ。

接続定義

コネクタ接続プールの接続定義のクラス名。

説明

コネクタ接続プールの説明。

「プール」の表には、次のオプションも表示されます。

新規

新しいコネクタ接続プールを作成するボタン。

削除

選択した1つまたは複数のコネクタ接続プールを削除するボタン。
