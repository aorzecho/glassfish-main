新建連接器連線池 (步驟 1，共 2 步)

使用 [新建連接器連線池 (步驟 1，共 2 步)] 頁面可建立新的連接器連線池。

[新建連接器連線池 (步驟 1，共 2 步)] 頁面包含下列選項。

名稱

可識別連接器連線池的唯一名稱。

資源配接卡

與連接器連線池關聯的資源配接卡。

連線定義

連接器連線池的連線定義。清單中提供的選項會視您所選取的資源配接卡而定。連線定義會識別資源配接卡在 ra.xml 檔案中的 connection-definition 元素。
