新建管理物件資源

使用 [新建管理物件資源] 頁面可建立新的管理物件資源。

[新建管理物件資源] 頁面包含下列選項。

JNDI 名稱

可識別管理物件資源的唯一名稱。

資源配接卡

管理物件資源的資源配接卡。

資源類型

管理物件資源的 Java 類型 (例如 javax.jms.Topic)。

類別名稱

與資源類型關聯的實作類別名稱。

說明

管理物件資源的說明。

狀態

若選取了此選項，則會啟用管理物件資源。此選項預設為啟用。

其他特性

管理物件資源的其他特性。可用的特性會視連接器連線池的資源配接卡而定。此表中指定的「名稱/值」組合，可用於置換資源配接卡供應商定義的特性預設值。

目標

所管理物件資源的叢集與獨立實例。可使用 [增加] 與 [全部加入] 按鈕，將所需的目標移至 [選取的目標] 欄。可使用 [移除] 與 [全部移除] 按鈕，將任何不需要的目標移至 [可用的目標] 欄。

唯有網域中已建立叢集或獨立實例時，才會顯示此選項。
