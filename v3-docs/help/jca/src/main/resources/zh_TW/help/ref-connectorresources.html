連接器資源

使用 [連接器資源] 頁面可配置連接器資源。

連接器資源是為應用程式提供 EIS 連線的程式物件。

對於每個資源，都將提供以下資訊。

JNDI 名稱

可識別連接器資源的唯一名稱。

已啟用

如果連接器資源已啟用，則為 True；如果連接器資源已停用，則為 False。

連線池

與連接器資源相關聯的連接器連線池。

說明

連接器資源的說明。

資源表也包含下列選項。

新建

建立新連接器資源的按鈕。

刪除

刪除一或多個所選取連接器資源的按鈕。

啟用

啟用一或多個所選取連接器資源的按鈕。

停用

停用一或多個所選取連接器資源的按鈕。
