連接器

使用 [連接器] 頁面可配置下列類型的連接器資源：

  • 連接器資源

  • 連接器連線池

  • 管理物件資源

  • 工作安全性對映

連接器模組 (也稱為資源配接卡) 是一個 Java 元件，它可讓應用程式與企業資訊 (EIS) 軟體互動。EIS 軟體包含各種類型的系統：企業資源規劃 (ERP)、主機作業事件處理、非關聯式資料庫和訊息傳送系統 (例如 Java 訊息服務 (JMS)) 以及其他系統。

連接器連線池是針對特定 EIS 的一組可重複使用的連線。

連接器資源是為應用程式提供 EIS 連線的程式物件。

管理物件資源會為應用程式提供專用化功能。

工作安全性對映會將與內送工作實例關聯的主體對映到 GlassFish Server 安全性網域中的主體。
