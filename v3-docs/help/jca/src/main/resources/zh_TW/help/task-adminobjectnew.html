建立管理物件資源

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 [連接器] 節點。
  3. 在 [連接器] 節點下，選取 [管理物件資源] 節點。

    [管理物件資源] 頁面會隨即開啟。

  4. 在 [管理物件資源] 頁面上按一下 [新建] 按鈕。

    [新建管理物件資源] 頁面會隨即開啟。

  5. 在 [JNDI 名稱] 欄位中，鍵入識別管理物件資源的唯一名稱。
  6. 從 [資源配接卡] 下拉式清單中，選取管理物件資源的資源配接卡。
  7. 從 [資源類型] 下拉式清單中，選取管理物件資源的 Java 類型。
  8. 從 [類別名稱] 下拉式清單中，選取與資源類型關聯的實作類別名稱。
  9. 在 [說明] 欄位中，鍵入管理物件資源的說明。
  10. 選取 [啟用狀態] 核取方塊啟用管理物件資源。

    此選項預設為啟用。

  11. (可選擇) 在 [其他特性] 表中，指定管理物件資源的其他特性。

    可用的特性會視連接器連線池的資源配接卡而定。

    若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。

  12. 在 [可用的目標] 區段中，選取管理物件所在的網域、叢集或伺服器實例，然後按一下 [增加]。

    若要移除 [已選取] 欄位中所列的其中一個網域、叢集或伺服器實例，請從該欄位中予以選取，然後按一下 [移除]。

  13. 按一下 [確定]。
