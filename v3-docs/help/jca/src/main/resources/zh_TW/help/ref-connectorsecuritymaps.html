連接器連線池安全性對映

使用 [連接器連線池安全性對映] 頁面可管理連接器連線池的安全性對映。

連接器連線池的安全性對映可以在 Java EE 使用者和群組以及 EIS 使用者和群組之間建立對映。安全性對映會將在 Servlet 或 EJB 認證期間接收的主體對映到 EIS 接受的憑證。您可以將多個 GlassFish Server 主體對映到相同的後端主體。

連線池的安全性對映與連接器工作安全性對映不同，後者會將與內送工作實例關聯的主體對映到 GlassFish Server 網域之安全性範圍中的主體。請參閱工作安全性對映以取得有關連接器工作安全性對映的資訊。

對於每個安全性對映，都將提供以下資訊。

名稱

安全性對映的名稱。

安全性對映表也包含下列選項。

新建

建立新的安全性對映的按鈕。

刪除

刪除一或多個所選取安全性對映的按鈕。
