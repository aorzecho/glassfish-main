編輯管理物件資源

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 [連接器] 節點。
  3. 在 [連接器] 節點下，選取 [管理物件資源] 節點。

    [管理物件資源] 頁面會隨即開啟。

  4. 在 [管理物件資源] 頁面中，按一下要編輯的管理物件資源名稱。

    [編輯管理物件資源] 頁面會隨即開啟。

  5. 從 [資源配接卡] 下拉式清單中，選取管理物件資源的資源配接卡。
  6. 從 [資源類型] 下拉式清單中，選取管理物件資源的 Java 類型。
  7. 從 [類別名稱] 下拉式清單中，選取與資源類型關聯的實作類別名稱。
  8. 在 [說明] 欄位中，鍵入管理物件資源的說明。
  9. 選取 [啟用狀態] 核取方塊啟用管理物件資源。

    此選項預設為啟用。

  10. (可選擇) 在 [其他特性] 表中，指定管理物件資源的其他特性。

    可用的特性會視連接器連線池的資源配接卡而定。

    • 若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。
    • 若要修改特性，請編輯特性的 [值] 欄位。
    • 若要刪除特性，請選取您要刪除之特性 [名稱] 欄位左側的核取方塊，然後按一下 [刪除特性] 按鈕。
  11. 在 [目標] 標籤上，按一下 [管理目標] 編輯部署有管理物件的目標。

    按一下 [一般] 標籤，即可回到 [編輯管理物件資源] 頁面。

  12. 按一下 [儲存]。

    提示 - 若要回到預設設定，請按一下 [載入預設值] 按鈕，然後按一下 [儲存]。
