編輯工作安全性對映

使用 [編輯工作安全性對映] 頁面可修改工作安全性對映的設定。

[編輯工作安全性對映] 頁面包含下列選項。

安全性對映名稱

工作安全性對映的名稱。名稱是唯讀欄位。您僅能在建立新的工作安全性對映時指定安全性對映名稱。

群組

若選取了 [群組] 單選按鈕，文字欄位必須指定每個後端 EIS 使用者群組與 GlassFish Server 安全性網域中定義的對應使用者群組的對映。使用以逗號分隔的清單指定一個以上的對映。例如︰

eis-group-1=server-group-1,eis-group-2=server-group-2
主體

若選取了 [主體] 單選按鈕，文字欄位必須指定每個後端 EIS 主體與 GlassFish Server 安全性網域中定義的對應主體的對映。使用以逗號分隔的清單指定一個以上的對映。例如︰

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
資源配接卡

EIS 的資源配接卡。

說明

工作安全性對映的說明。

狀態

若選取了此選項，則會啟用工作安全性對映。此選項預設為啟用。
