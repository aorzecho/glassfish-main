建立連接器連線池的安全性對映

連接器連線池的安全性對映可以在 Java EE 使用者和群組以及 EIS 使用者和群組之間建立對映。

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 [連接器] 節點。
  3. 在 [連接器] 節點下，選取 [連接器連線池] 節點。

    [連接器連線池] 頁面會隨即開啟。

  4. 在 [連接器連線池] 頁面上，按一下要編輯的連線池名稱。

    [編輯連接器連線池] 頁面會隨即開啟。

  5. 在 [編輯連接器連線池] 頁面上，按一下 [安全性對映] 標籤。

    [連接器連線池安全性對映] 頁面會隨即開啟。

  6. 在 [連接器連線池安全性對映] 頁面上，按一下 [新建]。

    [新建連接器連線池安全性對映] 頁面會隨即開啟。

  7. 在 [安全性對映名稱] 欄位中，鍵入安全性對映的名稱。
  8. 進行以下其中一項作業 (不要兩者都進行)：
    • 選取 [使用者群組] 單選按鈕並鍵入使用者群組名稱 (以逗號分隔)。
    • 選取 [主體] 單選按鈕並鍵入主體名稱 (以逗號分隔)。
  9. 在 [使用者名稱] 欄位中，指定 EIS 所需的後端主體使用者名稱。
  10. 在 [密碼] 欄位中，指定 EIS 所需的後端主體密碼。
  11. 按一下 [確定]。
  12. 重新啟動 GlassFish Server。
