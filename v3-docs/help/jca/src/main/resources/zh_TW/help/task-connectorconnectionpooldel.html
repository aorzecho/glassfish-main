刪除連接器連線池

當您刪除連接器連線池時，也會刪除任何依賴該連接器連線池的連接器資源。

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 [連接器] 節點。
  3. 在 [連接器] 節點下，選取 [連接器連線池] 節點。

    [連接器連線池] 頁面會隨即開啟。

  4. 在 [連接器連線池] 頁面上，選取要刪除的連線池名稱旁邊的核取方塊。
  5. 按一下 [刪除]。
