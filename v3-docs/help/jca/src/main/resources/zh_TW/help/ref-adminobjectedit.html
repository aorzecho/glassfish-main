編輯管理物件資源

使用 「編輯管理物件資源」 頁面可修改管理物件資源的設定。

「編輯管理物件資源」 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

JNDI 名稱

可識別管理物件資源的唯一名稱。「JNDI 名稱」 是唯讀欄位。您僅能在建立新的管理物件資源時指定 JNDI 名稱。

資源配接卡

管理物件資源的資源配接卡。

資源類型

管理物件資源的 Java 類型 (例如 javax.jms.Topic)。

類別名稱

與資源類型關聯的實作類別名稱。

說明

管理物件資源的說明。

狀態

若選取了此選項，則會啟用管理物件資源。此選項預設為啟用。

其他特性

管理物件資源的其他特性。可用的特性會視連接器連線集區的資源配接卡而定。此表中指定的「名稱/值」組，可用於置換資源配接卡供應商定義的特性預設值。
