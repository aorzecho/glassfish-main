工作安全性對映

使用 [工作安全性對映] 頁面可為資源配接卡配置工作安全性對映。

連接器工作安全性對映會將資源配接卡 EIS 主體或 EIS 使用者群組所提交之工作的呼叫者身份，對映到 GlassFish Server 安全性網域中適當的主體或使用者群組。可以將一或多個工作安全性對映與一個資源配接卡建立關聯。連接器工作安全性對映配置支援使用萬用字元星號 (*) 來表示所有使用者或所有使用者群組。

工作安全性對映與連接器連線池的安全性對映不同，後者會將 Servlet 或 EJB 認證期間接收的主體對映到 EIS 接受的憑證。請參閱連接器連線池安全性對映以取得有關連接器連線池安全性對映的資訊。

企業資訊系統 (EIS) 是指控管組織資料的任何系統。它可以是主機、訊息傳送系統、資料庫系統或應用程式。

對於每個工作安全性對映，都將提供以下資訊。

名稱

工作安全性對映的名稱。

工作安全性對映表也包含下列選項。

新建

建立新的工作安全性對映的按鈕。

刪除

刪除一或多個所選取工作安全性對映的按鈕。
