編輯連接器資源

使用 [編輯連接器資源] 頁面可修改連接器資源的設定。

[編輯連接器資源] 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

JNDI 名稱

可識別連接器資源的唯一名稱。[JNDI 名稱] 是唯讀欄位。您僅能在建立新的連接器資源時指定 JNDI 名稱。

池名稱

與此資源關聯的連接器連線池。

說明

連接器資源的說明。

狀態

若選取了此選項，則會啟用連接器資源。此選項預設為啟用。

其他特性

連接器資源的其他特性。可用的特性會視連接器連線池的資源配接卡而定。此表中指定的「名稱/值」組合，可用於置換資源配接卡供應商定義的特性預設值。
