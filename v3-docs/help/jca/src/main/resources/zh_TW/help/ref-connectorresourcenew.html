新建連接器資源

使用 「新建連接器資源」 頁面可建立新的連接器資源。

「新建連接器資源」 頁面包含下列選項。

JNDI 名稱

可識別連接器資源的唯一名稱。

集區名稱

新的連接器資源所屬的連線集區。

說明

連接器資源的說明。

狀態

若選取了此選項，則會啟用連接器資源。此選項預設為啟用。

其他特性

連接器資源的其他特性。可用的特性會視連接器連線集區的資源配接卡而定。此表中指定的「名稱/值」組，可用於置換資源配接卡供應商定義的特性預設值。

目標

連接器資源的叢集與獨立實例。可使用 「加入」 與 「全部加入」 按鈕，將所需的目標移至 「選取的目標」 欄。可使用 「移除」 與 「全部移除」 按鈕，將任何不需要的目標移至 「可用的目標」 欄。

唯有網域中已建立叢集或獨立實例時，才會顯示此選項。
