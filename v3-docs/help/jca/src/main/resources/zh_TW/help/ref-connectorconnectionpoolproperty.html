編輯連接器連線池特性

使用 [編輯連接器連線池特性] 頁面可修改連接器連線池的特性。

[編輯連接器連線池特性] 頁面包含下列選項。

其他特性

連接器連線池的其他特性。可用的特性會視連接器連線池的資源配接卡而定。此表中指定的「名稱/值」組合，可用於置換資源配接卡供應商定義的特性預設值。

若您未針對特性指定值，連線池編輯完成時，會從表中移除該特性。若您不想立即為特性設定值，但希望將該特性保留在表中，待稍後為其設定值，請在 [值] 欄位中鍵入一組空的圓括號：

()
