資源配接卡組態

使用 「資源配接卡組態」 頁面可管理連接器 (資源配接卡) 組態。

「資源配接卡組態」 提供資源配接卡的組態資訊。您可以在建置資源配接卡後，使用管理主控台建立資源配接卡組態。在這個情況下，會使用新的組態重新啟動資源配接卡。若要在建置資源配接卡前建立資源配接卡組態，請使用 asadmin create-resource-adapter-config 命令。

您必須建立供資源配接卡使用的執行緒集區，才能建立資源配接卡組態。

對於每個資源配接卡組態，都將提供以下資訊。

名稱

資源配接卡的名稱。

組態表也包含下列選項。

新建

建立新的資源配接卡組態的按鈕。

刪除

刪除一或多個所選取的資源配接卡組態的按鈕。
