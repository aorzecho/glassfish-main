資源配接卡配置

使用 [資源配接卡配置] 頁面可管理連接器 (資源配接卡) 配置。

[資源配接卡配置] 提供資源配接卡的配置資訊。您可以在部署資源配接卡後，使用管理主控台建立資源配接卡配置。在這個情況下，會使用新的配置重新啟動資源配接卡。若要在部署資源配接卡前建立資源配接卡配置，請使用 asadmin create-resource-adapter-config 指令。

您必須建立供資源配接卡使用的執行緒池，才能建立資源配接卡配置。

對於每個資源配接卡配置，都將提供以下資訊。

名稱

資源配接卡的名稱。

配置表也包含下列選項。

新建

建立新的資源配接卡配置的按鈕。

刪除

刪除一或多個所選取的資源配接卡配置的按鈕。
