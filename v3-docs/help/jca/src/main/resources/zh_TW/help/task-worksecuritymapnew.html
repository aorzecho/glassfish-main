建立工作安全性對映

  1. 在瀏覽樹狀結構中，展開 「資源」 節點。

  2. 展開 「連接器」 節點。

  3. 在 「連接器」 節點下，選取 「工作安全性對映」 節點。

    「工作安全性對映」 頁面會隨即開啟。

  4. 在 「工作安全性對映」 頁面上，按一下 「新建」。

    「新建工作安全性對映」 頁面會隨即開啟。

  5. 在 「安全性對映名稱」 欄位中，鍵入工作安全性對映的名稱。

  6. 以如下方式使用 「群組」 欄位或 「主體」 欄位。

    • 如果您選取 「群組」 單選按鈕，請鍵入每個後端 EIS 使用者群組與 GlassFish Server 安全性網域中定義的對應使用者群組的對映。

      使用以逗號分隔的清單指定一個以上的對映。例如︰

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • 如果您選取 「主體」 單選按鈕，請鍵入每個後端 EIS 主體與 GlassFish Server 安全性網域中定義的對應主體的對映。

      使用以逗號分隔的清單指定一個以上的對映。例如︰

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. 從 「資源配接卡」 下拉式清單中，選取 EIS 的資源配接卡。

  8. 在 「說明」 欄位中，鍵入工作安全性對映的說明。

  9. 選取「狀態」的「已啟用」核取方塊啟用工作安全性對映。

    此選項預設為啟用。

  10. 按一下 「確定」。
