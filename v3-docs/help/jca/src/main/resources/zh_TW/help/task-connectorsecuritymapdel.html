刪除連接器連線池的安全性對映

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 展開 [連接器] 節點。
  3. 在 [連接器] 節點下，選取 [連接器連線池] 節點。

    [連接器連線池] 頁面會隨即開啟。

  4. 在 [連接器連線池] 頁面上，按一下含有所要刪除之安全性對映的連線池名稱。

    [編輯連接器連線池] 頁面會隨即開啟。

  5. 按一下 [安全性對映] 標籤。

    [連接器連線池安全性對映] 頁面會隨即開啟。

  6. 在 [連接器連線池安全性對映] 頁面上，選取要刪除的安全性對映名稱旁的核取方塊。
  7. 按一下 [刪除]。
