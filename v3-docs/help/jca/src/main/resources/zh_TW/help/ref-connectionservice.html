連接器服務

使用 「連接器服務」 頁面可修改控制資源配接卡之連接器服務的一般設定。

「連接器服務」 頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原成其預設值的按鈕。沒有預設值的設定保持不變。

組態名稱

此頁面上的設定套用的組態名稱。本欄位是唯讀的。

關機逾時

GlassFish Server 關機期間可等候連接器模組實例 ResourceAdapter.stop 方法完成的最大秒數。會忽略需要更長時間關機的資源配接卡並繼續 GlassFish Server 關機動作。預設值為 30 秒。

連接器類別載入策略

用於載入類別的策略。可用的選項包括：

derived

代表會將資源配接卡根據應用程式的參照，提供給任何資源配接卡的資源。此策略為預設值。

global

代表會對所有應用程式顯示所有資源配接卡。

此策略不會套用到預先安裝的 JMS 系統資源配接卡 jmsra，該資源配接卡會永遠對所有應用程式顯示。
