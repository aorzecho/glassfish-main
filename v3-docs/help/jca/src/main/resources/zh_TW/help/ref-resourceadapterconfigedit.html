編輯資源配接卡配置

使用 [編輯資源配接卡配置] 頁面可修改資源配接卡配置的設定。

[編輯資源配接卡配置] 頁面包含下列選項。

資源配接卡名稱

您要配置的資源配接卡名稱。名稱是唯讀欄位。您僅能在建立新的資源配接卡配置時指定資源配接卡名稱。

執行緒池 ID

要用於資源配接卡配置的執行緒池。

其他特性

資源配接卡的其他特性。可用的特性包括 ra.xml 檔案中 resourceadapter-class 元素所參照的類別之 Setter 方法名稱，該檔案定義了資源配接卡 JavaBeans 元件的類別名稱。 對此處定義的特性所做的任何修改都會置換 ra.xml 中顯示的預設值。
