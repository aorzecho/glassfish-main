編輯連接器連線集區安全性對映

使用 「編輯連接器連線集區安全性對映」 頁面可針對連接器連線集區編輯安全性對映。

「編輯連接器連線集區安全性對映」 頁面包含下列選項。

連線集區名稱

此安全性對映所屬的連接器連線集區名稱。「集區名稱」 是唯讀欄位。

安全性對映名稱

安全性對映的名稱。名稱是唯讀欄位。您僅能在建立新的安全性對映時指定安全性對映名稱。

安全性對映名稱

安全性對映的名稱。

使用者群組

若選取了此單選按鈕，則文字欄位中必須包含使用者群組的名稱 (以逗號分隔)。

主體

若選取了此單選按鈕，則文字欄位中必須包含主體的名稱 (以逗號分隔)。

使用者名稱

EIS 所需的後端主體使用者名稱。

密碼

EIS 所需的後端主體密碼。
