Eliminare una mappa di sicurezza lavoro

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Mappe di sicurezza lavoro.

    Viene visualizzata la pagina Mappe di sicurezza lavoro.

  4. Nella pagina Mappe di sicurezza lavoro, selezionare la casella di controllo in corrispondenza del nome della mappa di sicurezza operativa da eliminare.

  5. Fare clic su Elimina.
