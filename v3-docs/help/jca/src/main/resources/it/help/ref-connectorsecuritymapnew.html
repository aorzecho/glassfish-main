New Connector Connection Pool Security Map

Nella pagina New Connector Connection Pool Security Map è possibile creare una nuova mappa di sicurezza per un pool di connessione del connettore.

La pagina New Connector Connection Pool Security Map contiene le seguenti opzioni.

Connection Pool Name

Nome del pool di connessione del connettore a cui appartiene la mappa di sicurezza. Il campo relativo al nome del pool è di sola lettura.

Security Map Name

Nome della mappa di sicurezza.

User Groups

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi dei gruppi utente separati da virgole.

Principals

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi delle identità separati da virgole.

Username

Nome utente dell'identità backend richiesto dal sistema EIS.

Password

Password dell'identità backend richiesta dal sistema EIS.
