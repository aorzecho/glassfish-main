Modificare una risorsa oggetto amministrato

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Admin Object Resources.

    Viene visualizzata la pagina Admin Object Resources.

  4. Nella pagina Admin Object Resources, fare clic sul nome della risorsa oggetto amministrato da modificare.

    Viene visualizzata la pagina Edit Admin Object Resource.

  5. Dall'elenco a discesa Resource Adapter, selezionare l'adattatore di risorsa per la risorsa oggetto amministrato.
  6. Dall'elenco a discesa Resource Type, selezionare il tipo di dati Java per la risorsa oggetto amministrato.
  7. Dall'elenco a discesa Class Name, selezionare il nome della classe di implementazione associato al tipo di risorsa.
  8. Nel campo Description, digitare una descrizione della risorsa oggetto amministrato.
  9. Selezionare la casella di controllo Status Enabled per abilitare la risorsa oggetto amministrato.

    Questa opzione è abilitata per impostazione predefinita.

  10. (Opzionale) Nella tabella Additional Properties, specificare le proprietà aggiuntive per la risorsa oggetto amministrato.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  11. Nella scheda Target, fare clic su Manage Targets per modificare le destinazioni in cui è distribuito l'oggetto amministrato.

    Tornare alla pagina Edit Admin Object Resource facendo clic sulla scheda General.

  12. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.
