Modifica proprietà connection pool connettore

Nella pagina Modifica proprietà connection pool connettore è possibile modificare le proprietà di un connection pool del connettore.

La pagina Modifica proprietà connection pool connettore contiene le opzioni riportate di seguito.

Proprietà aggiuntive

Proprietà aggiuntive per il connection pool del connettore. Le proprietà disponibili dipendono dal Resource Adapter relativo al connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.

Se non si specifica un valore per una proprietà, questa viene rimossa dalla tabella al termine della modifica del connection pool. Se anziché impostare il valore di una proprietà in questa fase si preferisce conservare la proprietà nella tabella in modo da impostare il valore in un secondo momento, digitare un set di parentesi vuote nel campo Valore:
