Modifica delle proprietà del pool di connessione del connettore

Nella pagina Edit Connector Connection Pool Properties è possibile modificare le proprietà di un pool di connessione del connettore.

La pagina Edit Connector Connection Pool Properties contiene le opzioni seguenti.

Additional Properties

Proprietà aggiuntive per il pool di connessione del connettore. Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore. Le coppie valore-nome specificate in questa tabella possono essere utilizzate per sovrascrivere i valori predefiniti per le proprietà specificate dal produttore dell'adattatore di risorsa.

Se non si specifica un valore per una proprietà, questa viene rimossa dalla tabella al termine della modifica del pool di connessione. Se anziché impostare il valore di una proprietà in questa fase si preferisce conservare la proprietà nella tabella in modo da impostare il valore in un secondo momento, digitare due parentesi vuote nel campo Value:
