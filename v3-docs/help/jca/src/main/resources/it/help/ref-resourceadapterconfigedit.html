Modifica configurazione Resource Adapter

Nella pagina Modifica configurazione Resource Adapter è possibile modificare le impostazioni di una configurazione del Resource Adapter.

La pagina Modifica configurazione Resource Adapter contiene le opzioni riportate di seguito.

Nome Resource Adapter

Nome del Resource Adapter che si sta configurando. Il campo relativo al nome è di sola lettura. È possibile specificare un nome per la mappa di sicurezza solo quando si crea una nuova configurazione del Resource Adapter.

ID pool di thread

Pool di thread da usare per la configurazione del Resource Adapter.

Proprietà aggiuntive

Proprietà aggiuntive per il Resource Adapter. Le proprietà disponibili sono i nomi di metodi setter della classe cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome di classe del componente JavaBeans del Resource Adapter. Qualsiasi modifica apportata alle proprietà qui definite sostituisce i valori predefiniti presenti nel file ra.xml.
