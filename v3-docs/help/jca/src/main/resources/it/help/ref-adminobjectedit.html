Edit Admin Object Resource

Nella pagina Edit Admin Object Resource è possibile modificare le impostazioni di una risorsa oggetto amministrato.

La pagina Edit Admin Object Resource contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

JNDI Name

Nome univoco che identifica la risorsa oggetto amministrato. Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa oggetto amministrato.

Resource Adapter

Adattatore di risorsa per la risorsa oggetto amministrato.

Resource Type

Tipo di dati Java per la risorsa oggetto amministrato (ad esempio, javax.jms.Topic).

Class Name

Nome della classe di implementazione associato al tipo di dati.

Description

Descrizione della risorsa oggetto amministrato.

Status

Se l'opzione è selezionata, la risorsa oggetto amministrato è abilitata. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa oggetto amministrato. Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore. Le coppie valore-nome specificate in questa tabella possono essere utilizzate per sovrascrivere i valori predefiniti per le proprietà specificate dal produttore dell'adattatore di risorsa.
