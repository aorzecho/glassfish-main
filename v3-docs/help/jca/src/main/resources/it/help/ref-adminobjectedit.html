Modifica risorsa oggetto amministrato

Nella pagina Modifica risorsa oggetto amministrato è possibile modificare le impostazioni di una risorsa oggetto amministrato.

La pagina Modifica risorsa oggetto amministrato contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome JNDI

Nome univoco che identifica la risorsa oggetto amministrato. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa oggetto amministrato.

Resource Adapter

Resource Adapter per la risorsa oggetto amministrato.

Tipo di risorsa

Tipo Java per la risorsa oggetto amministrato, ad esempio javax.jms.Topic.

Nome classe

Nome della classe di implementazione associato al tipo di risorsa.

Descrizione

Descrizione della risorsa oggetto amministrato.

Stato

Se questa opzione è selezionata, la risorsa oggetto amministrato è abilitata. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa oggetto amministrato. Le proprietà disponibili dipendono dal Resource Adapter relativo al connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.
