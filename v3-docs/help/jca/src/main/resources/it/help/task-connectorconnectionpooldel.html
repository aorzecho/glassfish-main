Eliminare un pool di connessione del connettore

Quando si elimina un pool di connessione del connettore vengono eliminate anche tutte le risorse connettore che dipendono dal pool.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Connector Connection Pools.

    Viene visualizzata la pagina Connector Connection Pools.

  4. Nella pagina Connector Connection Pools, selezionare la casella di controllo in corrispondenza del nome del pool di connessione da eliminare.
  5. Fare clic su Delete.
