Eliminare un connection pool del connettore

Quando si elimina un connection pool del connettore vengono eliminate anche tutte le risorse connettore che dipendono dal pool.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Connection pool connettore.

    Viene visualizzata la pagina Connection pool connettore.

  4. Nella pagina Connection pool connettore, selezionare la casella di controllo in corrispondenza del nome del connection pool da eliminare.

  5. Fare clic su Elimina.
