New Connector Connection Pool (Step 1 of 2)

Nella pagina New Connector Connection Pool (Step 1 of 2) è possibile creare un nuovo pool di connessione del connettore.

La pagina New Connector Connection Pool (Step 1 of 2) contiene le seguenti opzioni.

Name

Nome univoco che identifica il pool di connessione del connettore.

Resource Adapter

Adattatore di risorsa associato al pool di connessione del connettore.

Connection Definition

Definizione della connessione per il pool di connessione del connettore. Le opzioni nell'elenco dipendono dall'adattatore di risorsa selezionato. La definizione della connessione identifica un elemento connection-definition dell'adattatore di risorsa nel file ra.xml.
