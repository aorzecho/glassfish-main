Connector Connection Pools

Utilizzare la pagina Connector Connection Pools per configurare i pool di connessione del connettore.

Per pool di connessione del connettore si intende un gruppo di connessioni riutilizzabili per un determinato EIS.

Per ogni risorsa vengono fornite le seguenti informazioni.

Pool Name

Nome univoco che identifica il pool di connessione del connettore.

Resource Adapter

Adattatore di risorsa associato al pool di connessione del connettore.

Connection Definition

Nome della classe della definizione di connessione per il pool di connessione del connettore.

Description

Descrizione del pool di connessione del connettore.

La tabella Pools contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare un nuovo pool di connessione del connettore.

Delete

Pulsante che consente di eliminare uno o più pool di connessione del connettore selezionati.
