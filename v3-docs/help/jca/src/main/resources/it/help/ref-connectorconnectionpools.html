Connection pool connettore

Utilizzare la pagina Connection pool connettore per configurare i connection pool del connettore.

Un connection pool connettore è un gruppo di connessioni riutilizzabili per un determinato sistema EIS.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome pool

Nome univoco che identifica il connection pool del connettore.

Resource Adapter

Resource Adapter associato al connection pool del connettore.

Definizione connessione

Nome di classe della definizione di connessione per il connection pool del connettore.

Descrizione

Descrizione del connection pool del connettore.

La tabella Pool contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo connection pool del connettore.

Elimina

Pulsante che consente di eliminare uno o più connection pool del connettore selezionati.
