New Resource Adapter Config

Nella pagina New Resource Adapter Config è possibile creare una nuova configurazione per un adattatore di risorsa.

La pagina New Resource Adapter Config contiene le seguenti opzioni.

Resource Adapter Name

Nome dell'adattatore di risorsa che verrà configurato.

Thread Pool ID

Pool di thread da utilizzare per la configurazione di un adattatore di risorsa.

Additional Properties

Proprietà aggiuntive per l'adattatore di risorsa. Le proprietà disponibili sono i nomi di metodi per l'impostazione della classe a cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome della classe del componente JavaBeans di un adattatore di risorsa. Qualsiasi modifica alle proprietà qui definite sovrascrive i valori predefiniti presenti nel file ra.xml.
