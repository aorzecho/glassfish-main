New Connector Resource

Nella pagina New Connector Resource è possibile creare una nuova risorsa connettore.

La pagina New Connector Resource contiene le seguenti opzioni.

JNDI Name

Nome univoco che identifica la risorsa connettore.

Pool Name

Pool di connessione a cui appartiene la nuova risorsa connettore.

Description

Descrizione della risorsa connettore.

Status

Se l'opzione è selezionata, la risorsa connettore è abilitata. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa connettore. Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore. Le coppie valore-nome specificate in questa tabella possono essere utilizzate per sovrascrivere i valori predefiniti per le proprietà specificate dal produttore dell'adattatore di risorsa.

Targets

Cluster e istanze standalone per la risorsa connettore. Per spostare le destinazioni desiderate nella colonna Selected Targets, utilizzare i pulsanti Add e Add All. Per spostare le destinazioni non necessarie nella colonna Available Targets, utilizzare i pulsanti Remove e Remove All.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
