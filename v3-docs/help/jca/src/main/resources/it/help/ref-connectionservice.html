Servizio connettore

Nella pagina Servizio connettore è possibile modificare le impostazioni generali per il servizio connettore, che gestisce i Resource Adapter.

La pagina Servizio connettore contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome configurazione

Nome della configurazione a cui vengono applicate le impostazioni definite in questa pagina. Questo campo è di sola lettura.

Timeout chiusura

Numero massimo di secondi consentito durante la chiusura di GlassFish Server per il completamento del metodo ResourceAdapter.stop di un'istanza di modulo connettore. I Resource Adapter che richiedono tempi di chiusura più lunghi vengono ignorati e si prosegue con la chiusura di GlassFish Server. Il valore predefinito è 30 secondi.

Criteri di caricamento classi del connettore

Criteri da utilizzare per il caricamento delle classi. Sono disponibili le opzioni riportate di seguito.

derived

Indica che i Resource Adapter vengono forniti in base ai riferimenti di un'applicazione a qualsiasi risorsa del Resource Adapter. Questo è il criterio predefinito.

global

Indica che tutti i Resource Adapter saranno visibili a tutte le applicazioni.

Questo criterio non è valido per il Resource Adapter di sistema JMS preinstallato, jmsra, che è sempre visibile a tutte le applicazioni.
