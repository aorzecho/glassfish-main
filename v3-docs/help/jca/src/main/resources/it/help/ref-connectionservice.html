Servizio connettore

Nella pagina Connector Service è possibile modificare le impostazioni generali del servizio connettore, che controlla gli adattatori di risorsa.

La pagina Connector Service contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti di tutte le impostazioni.

Configuration Name

Nome della configurazione a cui si applicano le impostazioni visualizzate nella pagina. Campo di sola lettura.

Shutdown Timeout

Numero massimo di secondi consentito durante l'arresto di GlassFish Server per il completamento del metodo ResourceAdapter.stop dell'istanza di un modulo connettore. Gli adattatori di risorsa che richiedono tempi di arresto più prolungati vengono ignorati e si prosegue con l'arresto di GlassFish Server. Il valore predefinito è 30 secondi.

Connector Classloading Policy

Criterio da utilizzare per il caricamento delle classi. Sono disponibili le seguenti opzioni:

derived

Indica che gli adattatori di risorsa vengono forniti in base ai riferimenti di un'applicazione a qualsiasi risorsa dell'adattatore di risorsa. Questo è il criterio predefinito.

global

Indica che tutti gli adattatori di risorsa saranno visibili a tutte le applicazioni.

Questo criterio non è valido per l'adattatore di risorsa di sistema JMS preinstallato, jmsra, che è sempre visibile a tutte le applicazioni.
