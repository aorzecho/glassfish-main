Resource Adapter Configs

Nella pagina Resource Adapter Configs è possibile gestire le configurazioni (di adattatori di risorsa) del connettore.

Al suo interno sono disponibili informazioni di configurazione per un adattatore di risorsa. Con Administration Console è possibile creare una configurazione per un adattatore di risorsa in seguito alla distribuzione dell'adattatore di risorsa. In tal caso l'adattatore di risorsa viene riavviato con la nuova configurazione. Per creare la configurazione di un adattatore di risorsa prima di distribuire l'adattatore di risorsa, utilizzare il comando asadmin create-resource-adapter-config.

Per poter creare la configurazione di un adattatore di risorsa, è necessario creare prima un pool di thread per l'adattatore di risorsa da utilizzare.

Per ogni configurazione di adattatori di risorsa vengono fornite le seguenti informazioni.

Name

Nome dell'adattatore di risorsa.

La tabella Configs contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova configurazione per adattatori di risorsa.

Delete

Pulsante che consente di eliminare una o più configurazioni di adattatori di risorsa selezionate.

Attività correlate

  • Creare la configurazione di un adattatore di risorsa

  • Modificare la configurazione di un adattatore di risorsa

  • Eliminare la configurazione di un adattatore di risorsa
