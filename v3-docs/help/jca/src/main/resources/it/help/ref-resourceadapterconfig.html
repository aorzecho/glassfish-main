Configurazioni Resource Adapter

Nella pagina Configurazioni Resource Adapter è possibile gestire le configurazioni (Resource Adapter) del connettore.

La configurazione del Resource Adapter contiene le informazioni di configurazione per un Resource Adapter. Console di amministrazione consente di creare la configurazione di un Resource Adapter dopo la distribuzione del Resource Adapter. In questo caso, il Resource Adapter viene riavviato con la nuova configurazione. Per creare la configurazione di un Resource Adapter prima di distribuirlo, usare il comando asadmin create-resource-adapter-config.

Per poter creare la configurazione di un Resource Adapter, è necessario creare un pool di thread per il Resource Adapter da usare.

Per ogni configurazione di Resource Adapter vengono fornite le informazioni riportate di seguito.

Nome

Nome del Resource Adapter.

La tabella Configurazioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione del Resource Adapter.

Elimina

Pulsante che consente di eliminare una o più configurazioni di Resource Adapter selezionate.
