Modifica risorsa connettore

Nella pagina Modifica risorsa connettore è possibile modificare le impostazioni di una risorsa connettore.

La pagina Modifica risorsa connettore contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

Nome JNDI

Nome univoco che identifica la risorsa connettore. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa connettore.

Nome pool

Connection pool del connettore associato a questa risorsa.

Descrizione

Descrizione della risorsa connettore.

Stato

Se questa opzione è selezionata, la risorsa connettore è abilitata. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa connettore. Le proprietà disponibili dipendono dal Resource Adapter relativo al connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.
