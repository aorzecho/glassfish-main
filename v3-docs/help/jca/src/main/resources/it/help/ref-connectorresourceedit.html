Edit Connector Resource

Nella pagina Edit Connector Resource è possibile modificare le impostazioni di una risorsa connettore.

La pagina Edit Connector Resource contiene le seguenti opzioni.

Load Defaults

Pulsante che consente di ripristinare i valori predefiniti delle impostazioni che dispongono di tali valori. Le impostazioni prive di valori predefiniti restano invariate.

JNDI Name

Nome univoco che identifica la risorsa connettore. Il campo JNDI Name è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa connettore.

Pool Name

Pool di connessione del connettore associato a questa risorsa.

Description

Descrizione della risorsa connettore.

Status

Se l'opzione è selezionata, la risorsa connettore è abilitata. Questa opzione è abilitata per impostazione predefinita.

Additional Properties

Proprietà aggiuntive per la risorsa connettore. Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore. Le coppie valore-nome specificate in questa tabella possono essere utilizzate per sovrascrivere i valori predefiniti per le proprietà specificate dal produttore dell'adattatore di risorsa.
