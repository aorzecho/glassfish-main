Creare una risorsa connettore

Prima di cominciare

Prima di creare una risorsa connettore, creare un pool di connessione del connettore.

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Connector Resources.

    Viene visualizzata la pagina Connector Resources.

  4. Nella pagina Connector Resources, fare clic sul pulsante New.

    Viene visualizzata la pagina New Connector Resource.

  5. Nel campo JNDI Name, digitare un nome JNDI univoco per la risorsa connettore.

    Ad esempio: eis/myERP.

  6. Dall'elenco a discesa Pool Name, selezionare il pool di connessione a cui appartiene la nuova risorsa connettore.
  7. Nel campo Description, digitare una descrizione della risorsa connettore.
  8. Selezionare la casella di controllo Status Enabled per abilitare la risorsa connettore in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  9. In Available Targets, selezionare il dominio, il cluster o le istanze server in cui collocare la risorsa connettore e fare clic su Add.

    Se non si desidera distribuire la risorsa connettore in uno dei domini, dei cluster o delle istanze server presenti in Selected Targets, selezionare la destinazione nel campo e fare clic su Remove.

  10. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa connettore.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  11. Fare clic su OK.
