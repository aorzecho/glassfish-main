Creare una risorsa del connettore

Prima di iniziare

Prima di creare una risorsa connettore, creare un connection pool del connettore.

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Risorse connettore.

    Viene visualizzata la pagina Risorse connettore.

  4. Nella pagina Risorse connettore, fare clic sul pulsante Nuovo.

    Viene visualizzata la pagina Nuova risorsa connettore.

  5. Nel campo Nome JNDI, digitare un nome JNDI univoco per la risorsa connettore.

    Ad esempio: eis/myERP.

  6. Dall'elenco a discesa Nome pool, selezionare il connection pool a cui appartiene la nuova risorsa connettore.

  7. Nel campo Descrizione, digitare una descrizione della risorsa connettore.

  8. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa connettore in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  9. In Destinazioni disponibili, selezionare il dominio, il cluster o le istanze server in cui risiederà la risorsa connettore e fare clic su Aggiungi.

    Se non si desidera distribuire la risorsa connettore in uno dei domini, dei cluster o delle istanze server presenti in Destinazioni selezionate, selezionare la destinazione nel campo e fare clic su Rimuovi.

  10. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa connettore.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al connection pool del connettore.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  11. Fare clic su OK.
