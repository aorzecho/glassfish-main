Modifica mappa di sicurezza connection pool connettore

Nella pagina Modifica mappa di sicurezza connection pool connettore è possibile modificare una mappa di sicurezza per un connection pool del connettore.

La pagina Modifica mappa di sicurezza connection pool connettore contiene le opzioni riportate di seguito.

Nome connection pool

Nome del connection pool del connettore a cui appartiene la mappa di sicurezza. Il campo relativo al nome del pool è di sola lettura.

Nome mappa di sicurezza

Nome della mappa di sicurezza. Il campo relativo al nome è di sola lettura. È possibile specificare il nome di una mappa di sicurezza solo quando si crea una nuova mappa di sicurezza.

Nome mappa di sicurezza

Nome della mappa di sicurezza.

Gruppi di utenti

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi dei gruppi di utenti, separati da virgole.

Principal

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi dei principal, separati da virgole.

Nome utente

Nome utente del principal backend richiesto dal sistema EIS.

Password

Password del principal backend richiesto dal sistema EIS.
