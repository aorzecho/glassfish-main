Creare la configurazione di un adattatore di risorsa

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, selezionare il nodo Resource Adapter Configs.

    Viene visualizzata la pagina Resource Adapter Configs.

  3. Nella pagina Resource Adapter Configs, fare clic su New.

    Viene visualizzata la pagina New Resource Adapter Config.

  4. Dall'elenco a discesa Resource Adapter Name, selezionare l'adattatore di risorsa.
  5. Dall'elenco a discesa Thread Pool ID, selezionare il pool di thread da utilizzare per la configurazione dell'adattatore di risorsa.
  6. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per l'adattatore di risorsa.

    Le proprietà disponibili sono i nomi di metodi per l'impostazione della classe a cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome della classe del componente JavaBeans di un adattatore di risorsa. Qualsiasi modifica alle proprietà qui definite sovrascrive i valori predefiniti presenti nel file ra.xml.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  7. Fare clic su OK.
