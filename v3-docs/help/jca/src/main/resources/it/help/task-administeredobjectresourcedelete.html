Eliminare una risorsa oggetto amministrato

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Admin Object Resources.

    Viene visualizzata la pagina Admin Object Resources.

  4. Nella pagina the Admin Object Resources, selezionare la casella di controllo in corrispondenza del nome della risorsa oggetto amministrato da eliminare.
  5. Fare clic su Delete.
