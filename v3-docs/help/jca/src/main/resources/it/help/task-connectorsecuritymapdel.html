Eliminare una mappa di sicurezza per un pool di connessione del connettore

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Connector Connection Pools.

    Viene visualizzata la pagina Connector Connection Pools.

  4. Nella pagina Connector Connection Pools, fare clic sul nome del pool di connessione che include la mappa di sicurezza da eliminare.

    Viene visualizzata la pagina Edit Connector Connection Pool.

  5. Fare clic sulla scheda Security Maps.

    Viene visualizzata la pagina Connector Connection Pool Security Maps.

  6. Nella pagina Connector Connection Pool Security Maps, selezionare la casella di controllo in corrispondenza del nome della mappa di sicurezza da eliminare.
  7. Fare clic su Delete.
