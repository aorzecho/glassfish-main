Creare una risorsa oggetto amministrato

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Risorse oggetto amministrato.

    Viene visualizzata la pagina Risorse oggetto amministrato.

  4. Nella pagina Risorse oggetto amministrato, fare clic sul pulsante Nuovo.

    Viene visualizzata la pagina Nuova risorsa oggetto amministrato.

  5. Nel campo Nome JNDI, digitare un nome univoco che identifica la risorsa oggetto amministrato.

  6. Dall'elenco a discesa Resource Adapter, selezionare l'adattatore di risorsa per la risorsa oggetto amministrato.

  7. Dall'elenco a discesa Tipo di risorsa, selezionare il tipo di dati Java per la risorsa oggetto amministrato.

  8. Dall'elenco a discesa Nome classe, selezionare il nome della classe di implementazione associato al tipo di risorsa.

  9. Nel campo Descrizione, digitare una descrizione della risorsa oggetto amministrato.

  10. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa oggetto amministrato.

    Questa opzione è abilitata per impostazione predefinita.

  11. Nella tabella Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa oggetto amministrato.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al connection pool del connettore.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  12. Nella sezione Destinazioni disponibili, selezionare il dominio, il cluster o le istanze server in cui risiederà l'oggetto amministrato e fare clic su Aggiungi.

    Per rimuovere uno dei domini, dei cluster o delle istanze server presenti nel campo Selected, selezionarlo nel campo e fare clic su Rimuovi.

  13. Fare clic su OK.
