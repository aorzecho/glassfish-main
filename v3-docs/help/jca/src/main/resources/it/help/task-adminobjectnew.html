Creare una risorsa oggetto amministrato

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Admin Object Resources.

    Viene visualizzata la pagina Admin Object Resources.

  4. Nella pagina Admin Object Resources, fare clic sul pulsante New.

    Viene visualizzata la pagina New Admin Object Resource.

  5. Nel campo JNDI Name, digitare un nome univoco che identifica la risorsa oggetto amministrato.
  6. Dall'elenco a discesa Resource Adapter, selezionare l'adattatore di risorsa per la risorsa oggetto amministrato.
  7. Dall'elenco a discesa Resource Type, selezionare il tipo di dati Java per la risorsa oggetto amministrato.
  8. Dall'elenco a discesa Class Name, selezionare il nome della classe di implementazione associato al tipo di risorsa.
  9. Nel campo Description, digitare una descrizione della risorsa oggetto amministrato.
  10. Selezionare la casella di controllo Status Enabled per abilitare la risorsa oggetto amministrato.

    Questa opzione è abilitata per impostazione predefinita.

  11. (Opzionale) Nella tabella Additional Properties, specificare le proprietà aggiuntive per la risorsa oggetto amministrato.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore.

    Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.

  12. Nella sezione Targets Available, selezionare il dominio, il cluster o le istanze server in cui collocare l'oggetto amministrato e fare clic su Add.

    Per rimuovere uno dei domini, dei cluster o delle istanze server presenti nel campo Selected, selezionarlo nel campo e fare clic su Remove.

  13. Fare clic su OK.
