Modificare una mappa di sicurezza lavoro

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Mappe di sicurezza lavoro.

    Viene visualizzata la pagina Mappe di sicurezza lavoro.

  4. Nella pagina Mappe di sicurezza lavoro, fare clic sul nome della mappa di sicurezza operativa che verrà modificata.

    Viene visualizzata la pagina Modifica mappa di sicurezza lavoro.

  5. Utilizzare il campo Gruppo o il campo Principal come indicato di seguito:

    • Se si seleziona il pulsante di scelta Gruppo, immettere il mapping di ogni gruppo utenti EIS backend al gruppo utenti corrispondente definito nel dominio di sicurezza di GlassFish Server.

      Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Se si seleziona il pulsante di scelta Principal, immettere il mapping di ogni identità EIS backend all'identità corrispondente definita nel dominio di sicurezza di GlassFish Server.

      Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  6. Dall'elenco a discesa Resource Adapter, selezionare l'adattatore di risorsa per il sistema EIS.

  7. Nel campo Descrizione, digitare una descrizione della mappa di sicurezza operativa.

  8. Selezionare la casella di controllo Stato abilitato per abilitare la mappa di sicurezza operativa.

    Questa opzione è abilitata per impostazione predefinita.

  9. Fare clic su OK.
