Modificare una mappa di sicurezza operativa

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Work Security Maps.

    Viene visualizzata la pagina Work Security Maps.

  4. Nella pagina Work Security Maps, fare clic sul nome della mappa di sicurezza operativa che verrà modificata.

    Viene visualizzata la pagina Edit Work Security Map.

  5. Utilizzare il campo Group o il campo Principal come indicato di seguito:
    • Se si seleziona il pulsante Group, immettere il mapping di ogni gruppo utenti EIS backend al gruppo utenti corrispondente definito nel dominio di protezione di GlassFish Server.

      Per specificare più mapping, utilizzare un elenco separato da virgole. Ad esempio:

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • Se si seleziona il pulsante Principal, immettere il mapping di ogni identità EIS backend all'identità corrispondente definita nel dominio di protezione di GlassFish Server.

      Per specificare più mapping, utilizzare un elenco separato da virgole. Ad esempio:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  6. Dall'elenco a discesa Resource Adapter, selezionare l'adattatore di risorsa per il sistema EIS.
  7. Nel campo Description, digitare una descrizione della mappa di sicurezza operativa.
  8. Selezionare la casella di controllo Status Enabled per abilitare la mappa di sicurezza operativa.

    Questa opzione è abilitata per impostazione predefinita.

  9. Fare clic su OK.
