Admin Object Resources

Nella pagina Admin Object Resources è possibile configurare risorse oggetto amministrato.

Una risorsa oggetto amministrato offre una funzionalità specializzata che viene definita dall'adattatore di risorsa relativo al modulo del connettore distribuito.

Per ogni risorsa oggetto amministrato vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa oggetto amministrato.

Enabled

Opzione attivata se la risorsa oggetto amministrato è abilitata e disattivata se la risorsa è disabilitata.

Resource Type

Tipo di dati Java della risorsa oggetto amministrato.

Description

Descrizione della risorsa oggetto amministrato.

La tabella Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa oggetto amministrato.

Delete

Pulsante che consente di eliminare una o più risorse oggetto amministrato selezionate.

Enable

Pulsante che consente di abilitare una o più risorse oggetto amministrato selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse oggetto amministrato selezionate.
