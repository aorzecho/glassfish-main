Risorse oggetto amministrato

Nella pagina Risorse oggetto amministrato è possibile configurare le risorse oggetto amministrato.

Una risorsa oggetto amministrato offre una funzionalità specializzata che viene definita dal Resource Adapter per il modulo connettore distribuito.

Per ogni risorsa oggetto amministrato vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa oggetto amministrato.

Abilitato

Opzione attivata se la risorsa oggetto amministrato è abilitata; disattivata se la risorsa è disabilitata.

Tipo di risorsa

Tipo Java della risorsa oggetto amministrato.

Descrizione

Descrizione della risorsa oggetto amministrato.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa oggetto amministrato.

Elimina

Pulsante che consente di eliminare una o più risorse oggetto amministrato selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse oggetto amministrato selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse oggetto amministrato selezionate.
