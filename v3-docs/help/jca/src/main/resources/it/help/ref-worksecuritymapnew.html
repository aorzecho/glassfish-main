New Work Security Map

Nella pagina New Work Security Map è possibile creare una nuova mappa di sicurezza operativa.

La pagina New Work Security Map contiene le seguenti opzioni.

Security Map Name

Nome della mappa di sicurezza operativa.

Group

Se il pulsante di scelta Group Mapping è selezionato, nel campo di testo deve essere specificato il mapping di ciascun gruppo utenti EIS backend sul gruppo utenti corrispondente definito nel dominio di protezione GlassFish Server. Per specificare più mapping, utilizzare un elenco separato da virgole. Ad esempio:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se il pulsante di scelta Principal Mapping è selezionato, nel campo di testo deve essere specificato il mapping di ciascuna identità EIS backend sull'identità corrispondente definita nel dominio di protezione GlassFish Server. Per specificare più mapping, utilizzare un elenco separato da virgole. Ad esempio:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Resource Adapter

Adattatore di risorsa per il sistema EIS.

Description

Descrizione della mappa di sicurezza operativa.

Status

Se l'opzione è selezionata, la mappa di sicurezza operativa viene abilitata. Questa opzione è abilitata per impostazione predefinita.
