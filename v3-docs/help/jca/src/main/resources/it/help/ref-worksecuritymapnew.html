Nuova mappa di sicurezza lavoro

Nella pagina Nuova mappa di sicurezza lavoro è possibile creare una nuova mappa di sicurezza lavoro.

La pagina Nuova mappa di sicurezza lavoro contiene le opzioni riportate di seguito.

Nome mappa di sicurezza

Nome della mappa di sicurezza lavoro.

Gruppo

Se il pulsante di scelta Mapping gruppi è selezionato, nel campo di testo deve essere specificata una mappa di ciascun gruppo di utenti EIS backend al gruppo di utenti corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, usare una lista di valori separati da virgole. Ad esempio:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se il pulsante di scelta Mapping principal è selezionato, nel campo di testo deve essere specificata la mappa di ciascun principal EIS backend al principal corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, usare una lista di valori separati da virgole. Ad esempio:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Resource Adapter

Resource Adapter per il sistema EIS.

Descrizione

Descrizione della mappa di sicurezza lavoro.

Stato

Se questa opzione è selezionata, la mappa di sicurezza lavoro è abilitata. Questa opzione è abilitata per impostazione predefinita.
