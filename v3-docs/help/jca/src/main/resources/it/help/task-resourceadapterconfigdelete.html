Eliminare la configurazione di un adattatore di risorsa

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Nel nodo Resources, selezionare il nodo Resource Adapter Configs.

    Viene visualizzata la pagina Resource Adapter Configs.

  3. Nella pagina Resource Adapter Configs, selezionare la casella di controllo in corrispondenza del nome della configurazione dell'adattatore di risorsa da eliminare.
  4. Fare clic su Delete.
