Modificare una configurazione Resource Adapter

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Nel nodo Risorse, selezionare il nodo Configurazioni Resource Adapter.

    Viene visualizzata la pagina Configurazioni Resource Adapter.

  3. Nella pagina Configurazioni Resource Adapter, selezionare la casella di controllo in corrispondenza del nome della configurazione dell'adattatore di risorsa da eliminare.

  4. Fare clic su Elimina.
