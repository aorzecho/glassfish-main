Modificare una mappa di sicurezza per un connection pool del connettore

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Connection pool connettore.

    Viene visualizzata la pagina Connection pool connettore.

  4. Nella pagina Connection pool connettore, fare clic sul nome del connection pool da modificare.

    Viene visualizzata la pagina Modifica connection pool connettore.

  5. Nella pagina Modifica connection pool connettore, fare clic sulla scheda Mappe di sicurezza.

    Viene visualizzata la pagina Mappe di sicurezza connection pool connettore.

  6. Nella pagina Mappe di sicurezza connection pool connettore, fare clic sul nome della mappa di sicurezza che verrà modificata.

    Viene visualizzata la pagina Modifica connection pool connettore Mappa di sicurezza.

  7. Attenersi a una delle seguenti procedure:

    • Selezionare il pulsante di scelta Gruppi di utenti e digitare i nomi dei gruppi utenti separati da virgole.

    • Selezionare il pulsante di scelta Principal e digitare i nomi delle identità separati da virgole.

  8. Nel campo Nome utente, specificare il nome utente del principal backend richiesto dal sistema EIS.

  9. Nel campo Password, specificare la password del principal backend richiesta dal sistema EIS.

  10. Fare clic su Salva.
