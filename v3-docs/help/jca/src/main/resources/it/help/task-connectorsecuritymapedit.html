Modificare una mappa di sicurezza per un pool di connessione del connettore

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Connector Connection Pools.

    Viene visualizzata la pagina Connector Connection Pools.

  4. Nella pagina Connector Connection Pools, fare clic sul nome del pool di connessione da modificare.

    Viene visualizzata la pagina Edit Connector Connection Pool.

  5. Nella pagina Edit Connector Connection Pool, fare clic sulla scheda Security Maps.

    Viene visualizzata la pagina Connector Connection Pool Security Maps.

  6. Nella pagina Connector Connection Pool Security Maps, fare clic sul nome della mappa di sicurezza che verrà modificata.

    Viene visualizzata la pagina Edit Connector Connection Pool Security Map.

  7. Attenersi a una delle seguenti procedure:
    • Selezionare il pulsante di scelta User Groups e digitare i nomi dei gruppi utenti separati da virgole.
    • Selezionare il pulsante di scelta Principals e digitare i nomi delle identità separati da virgole.
  8. Nel campo Username, specificare il nome utente dell'identità backend richiesto dal sistema EIS.
  9. Nel campo Password, specificare la password dell'identità backend richiesta dal sistema EIS.
  10. Fare clic su Save.
