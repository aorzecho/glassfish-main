Mappe di sicurezza connection pool connettore

Nella pagina Mappe di sicurezza connection pool connettore è possibile gestire le mappe di sicurezza per i connection pool del connettore.

Le mappe di sicurezza per i connection pool del connettore consentono la creazione di un mapping tra utenti e gruppi Java EE e utenti e gruppi EIS. Una mappa di sicurezza consente di eseguire il mapping tra il principal ricevuto durante l'autenticazione di servlet o EJB e le credenziali accettate dal sistema EIS. È possibile eseguire il mapping tra più principal di GlassFish Server e lo stesso principal backend.

Una mappa di sicurezza per un connection pool è diversa da una mappa di sicurezza lavoro del connettore, che esegue il mapping tra un principal associato a un'istanza di lavoro in entrata e un principal in un realm di sicurezza del dominio di GlassFish Server. Per informazioni sulle mappe di sicurezza lavoro del connettore, vedere .

Per ogni mappa di sicurezza vengono fornite le informazioni riportate di seguito.

Nome

Nome della mappa di sicurezza.

La tabella Mappe di sicurezza contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova mappa di sicurezza.

Elimina

Pulsante che consente di eliminare una o più mappe di sicurezza selezionate.
