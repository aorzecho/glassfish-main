Connector Connection Pool Security Maps

Nella pagina Connector Connection Pool Security Maps è possibile gestire le mappe di sicurezza per i pool di connessione del connettore.

Le mappe di sicurezza per i pool di connessione del connettore consentono la creazione di un mapping tra utenti e gruppi Java EE e utenti e gruppi EIS. La mappa di sicurezza consente di eseguire il mapping dell'identità ricevuta durante l'autenticazione di servlet o EJB sulle credenziali accettate dal sistema EIS. È possibile eseguire il mapping di più identità di GlassFish Server sulla stessa identità backend.

Una mappa di sicurezza per un pool di connessione è diversa da una mappa di sicurezza operativa per un connettore, che esegue il mapping di un'identità associata a un'istanza operativa in entrata su un'identità di un dominio di protezione di GlassFish Server. Per informazioni sulle mappe di sicurezza operativa per un connettore, vedere Work Security Maps.

Per ogni mappa di sicurezza vengono fornite le seguenti informazioni.

Name

Nome della mappa di sicurezza.

La tabella Security Maps contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova mappa di sicurezza.

Delete

Pulsante che consente di eliminare una o più mappe di sicurezza selezionate.
