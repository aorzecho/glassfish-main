Configurare il servizio connettore

  1. Nella struttura ad albero, espandere il nodo Configuration.
  2. Nel nodo Configuration, selezionare il nodo Connector Service.

    Viene visualizzata la pagina Connector Service.

  3. Nel campo Shutdown Timeout, digitare il numero massimo di secondi consentito durante l'arresto di GlassFish Server per il completamento del metodo ResourceAdapter.stop dell'istanza di un modulo connettore.

    Gli adattatori di risorsa che richiedono tempi di arresto più prolungati vengono ignorati e si prosegue con l'arresto di GlassFish Server. Il valore predefinito è 30 secondi.

  4. Dall'elenco a discesa Connector Classloading Policy, selezionare il criterio da utilizzare.
    • Selezionare derived per indicare che gli adattatori di risorsa vengono forniti in base ai riferimenti di un'applicazione a qualsiasi risorsa dell'adattatore di risorsa.

      Questo è il criterio predefinito.

    • Selezionare global per specificare che tutti gli adattatori di risorsa saranno visibili a tutte le applicazioni.

    Questo criterio non è valido per l'adattatore di risorsa di sistema JMS preinstallato, jmsra, che è sempre visibile a tutte le applicazioni.

  5. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.
