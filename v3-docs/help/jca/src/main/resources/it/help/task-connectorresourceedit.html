Modificare una risorsa connettore

  1. Nella struttura ad albero, espandere il nodo Resources.
  2. Espandere il nodo Connectors.
  3. Nel nodo Connectors, selezionare il nodo Connector Resources.

    Viene visualizzata la pagina Connector Resources.

  4. Nella pagina Connector Resources, fare clic sul nome della risorsa che verrà modificata.

    Viene visualizzata la pagina Edit Connector Resource.

  5. Dall'elenco a discesa Pool Name, selezionare il pool di connessione del connettore a cui appartiene la nuova risorsa connettore.
  6. Nel campo Description, digitare una descrizione della risorsa connettore.
  7. Nella scheda Target, selezionare le destinazioni necessarie e fare clic su Enable o Disable.

    Per aggiungere o rimuovere destinazioni dall'elenco, fare clic su Manage Targets.

    Tornare alla pagina Edit Connector Resource facendo clic sulla scheda General.

  8. Selezionare la casella di controllo Status Enabled per abilitare la risorsa connettore in fase di runtime.

    Questa opzione è abilitata per impostazione predefinita.

  9. (Opzionale) Nella sezione Additional Properties, specificare le proprietà aggiuntive per la risorsa connettore.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al pool di connessione del connettore.

    • Per aggiungere una proprietà, fare clic sul pulsante Add Property. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Name e il valore della proprietà nel campo Value.
    • Per modificare una proprietà, è necessario cambiare il valore del campo Value della proprietà.
    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Name relativo alla proprietà da eliminare, quindi fare clic sul pulsante Delete Properties.
  10. Fare clic su Save.

    Suggerimento - Per tornare alle impostazioni predefinite, fare clic sul pulsante Load Defaults, quindi su Save.
