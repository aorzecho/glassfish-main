Modificare una risorsa del connettore

  1. Nell'albero di navigazione espandere il nodo Risorse.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori, selezionare il nodo Risorse connettore.

    Viene visualizzata la pagina Risorse connettore.

  4. Nella pagina Risorse connettore, fare clic sul nome della risorsa che verrà modificata.

    Viene visualizzata la pagina Modifica risorsa connettore.

  5. Dall'elenco a discesa Nome pool, selezionare il connection pool del connettore a cui appartiene la nuova risorsa connettore.

  6. Nel campo Descrizione, digitare una descrizione della risorsa connettore.

  7. Nella scheda Destinazione, selezionare le destinazioni necessarie e fare clic su Abilita o Disabilita.

    Per aggiungere o rimuovere destinazioni dalla lista, fare clic su Gestisci destinazioni.

    Tornare alla pagina Modifica risorsa connettore facendo clic sulla scheda Generale.

  8. Selezionare la casella di controllo Stato abilitato per abilitare la risorsa connettore in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive per la risorsa connettore.

    Le proprietà disponibili dipendono dall'adattatore di risorsa relativo al connection pool del connettore.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome relativo alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.

    Suggerimento:

    Per tornare alle impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
