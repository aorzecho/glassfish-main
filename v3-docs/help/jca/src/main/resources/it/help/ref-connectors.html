Connettori

Nella pagina Connectors è possibile configurare risorse connettore dei seguenti tipi:

  • Risorse connettore

  • Connector Connection Pools

  • Admin Object Resources

  • Work Security Maps

Per modulo connettore (anche noto come adattatore di risorsa) si intende un componente Java che consente l'interazione delle applicazioni con il software EIS (Enterprise Information System). Il software EIS include vari tipi di sistemi, tra cui ERP (Enterprise Resource Planning, pianificazione delle risorse aziendali), elaborazione di transazioni mainframe, database non relazionali e servizi di messaggistica quali JMS (Java Message Service).

Per pool di connessione del connettore si intende un gruppo di connessioni riutilizzabili per un determinato EIS.

Una risorsa connettore è un oggetto di programmazione che consente di connettere un'applicazione a un EIS.

Una risorsa oggetto amministrato fornisce funzionalità specializzate per un'applicazione.

Una mappa di sicurezza operativa esegue il mapping di un'identità associata a un'istanza operativa in entrata su un'identità nel dominio di protezione di GlassFish Server.
