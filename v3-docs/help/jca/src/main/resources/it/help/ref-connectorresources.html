Risorse connettore

Nella pagina Connector Resources è possibile configurare le risorse connettore.

Una risorsa connettore è un oggetto di programmazione che consente di connettere un'applicazione a un EIS.

Per ogni risorsa vengono fornite le seguenti informazioni.

JNDI Name

Nome univoco che identifica la risorsa connettore.

Enabled

Opzione attivata se la risorsa connettore è abilitata e disattivata se la risorsa connettore è disabilitata.

Connection Pool

Pool di connessione del connettore associato alla risorsa connettore.

Description

Descrizione della risorsa connettore.

La tabella Resources contiene inoltre le seguenti opzioni.

New

Pulsante che consente di creare una nuova risorsa connettore.

Delete

Pulsante che consente di eliminare una o più risorse connettore selezionate.

Enable

Pulsante che consente di abilitare una o più risorse connettore selezionate.

Disable

Pulsante che consente di disabilitare una o più risorse connettore selezionate.
