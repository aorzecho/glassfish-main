Risorse connettore

Nella pagina Risorse connettore è possibile configurare le risorse connettore.

Una risorsa connettore è un oggetto di programma che offre a un'applicazione una connessione a un sistema EIS.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome JNDI

Nome univoco che identifica la risorsa connettore.

Abilitato

Opzione attivata se la risorsa connettore è abilitata; disattivata se la risorsa connettore è disabilitata.

Connection pool

Connection pool del connettore associato alla risorsa connettore.

Descrizione

Descrizione della risorsa connettore.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa connettore.

Elimina

Pulsante che consente di eliminare una o più risorse connettore selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse connettore selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse connettore selezionate.
