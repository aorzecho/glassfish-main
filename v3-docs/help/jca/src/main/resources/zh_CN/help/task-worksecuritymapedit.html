编辑工作安全映射

  1. 在导航树中，展开“资源”节点。
  2. 展开“连接器”节点。
  3. 在“连接器”节点下，选择“工作安全映射”节点。

    此时将打开“工作安全映射”页面。

  4. 在“工作安全映射”页面上，单击要编辑的工作安全映射的名称。

    此时将打开“编辑工作安全映射”页面。

  5. 按以下所述使用“组”字段或“主体”字段。
    • 如果选中“组”单选按钮，请键入每个后端 EIS 用户组与在 GlassFish Server 安全域中定义的相应用户组的映射。

      可使用逗号分隔列表指定多个映射。例如：

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • 如果选中“主体”单选按钮，请键入每个后端 EIS 主体与在 GlassFish Server 安全域中定义的相应主体的映射。

      可使用逗号分隔列表指定多个映射。例如：

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  6. 从“资源适配器”下拉式列表中，选择 EIS 的资源适配器。
  7. 在“说明”字段中，键入工作安全映射的说明。
  8. 选中“启用状态”复选框以启用工作安全映射。

    默认情况下启用此选项。

  9. 单击“确定”。
