编辑资源适配器配置

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，选择“资源适配器配置”节点。

    此时将打开“资源适配器配置”页面。

  3. 在“资源适配器配置”页面上，单击要编辑的资源适配器配置的名称。

    此时将打开“编辑资源适配器配置”页面。

  4. 从“线程池 ID”下拉式列表中，选择要用于资源适配器配置的线程池。
  5. （可选的）在“其他属性”部分中，指定资源适配器的其他属性。

    可用属性为 ra.xml 文件中 resourceadapter-class 元素所引用类的设置方法 (setter method) 的名称，该文件定义了资源适配器 JavaBeans 组件的类名。对此处定义的属性的任何修改都会覆盖 ra.xml 中的默认值。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。
    • 要修改属性，请编辑该属性的“值”字段。
    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。
  6. 单击“保存”。
