连接器

使用“连接器”页可以配置以下类型的连接器资源：

  • 连接器资源

  • 连接器连接池

  • 管理对象资源

  • 工作安全映射

连接器模块 (也称为资源适配器) 为 Java 组件，允许应用程序与企业信息系统 (EIS) 软件进行交互。EIS 程序包括各种类型的系统：企业资源规划 (ERP)、大型机事务处理处理、非关系数据库和消息传送系统，例如 Java 消息服务 (JMS) 等。

连接器连接池是一组用于特定 EIS 的可重复使用的连接。

连接器资源是一个程序对象，可以为应用程序提供到 EIS 的连接。

受管理对象资源可为应用程序提供专用化功能。

工作安全映射可将与接收工作实例关联的主用户映射到 GlassFish Server 安全域中的主用户。
