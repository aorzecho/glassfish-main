编辑连接器连接池属性

使用“编辑连接器连接池属性”页面可以修改连接器连接池的属性。

“编辑连接器连接池属性”页面包含以下选项：

其他属性

连接器连接池的其他属性。可用属性取决于连接器连接池的资源适配器。可以使用在此表中指定的名称-值对，覆盖由资源适配器供应商定义的属性默认值。

如果您不指定某个属性的值，则在编辑完连接池时，该属性将从表中删除。如果您现在不希望设置某个属性的值，但希望将该属性保留在表中以便在以后设置值，请在“值”字段中键入一对空括号：

()
