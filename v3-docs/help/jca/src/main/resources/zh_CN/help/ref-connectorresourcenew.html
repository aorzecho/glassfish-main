新建连接器资源

使用“新建连接器资源”页面可以创建新的连接器资源。

“新建连接器资源”页面包含以下选项。

JNDI 名称

标识连接器资源的唯一名称。

池名称

新连接器资源所属的连接池。

说明

连接器资源的说明。

状态

如果选中此选项，则将启用连接器资源。默认情况下启用此选项。

其他属性

连接器资源的其他属性。可用属性取决于连接器连接池的资源适配器。可以使用在此表中指定的名称-值对，覆盖由资源适配器供应商定义的属性默认值。

目标

连接器资源的群集和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“选定目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。

仅当已在域中创建群集或独立实例时才会显示此选项。
