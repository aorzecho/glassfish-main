编辑连接器资源

使用“编辑连接器资源”页面可以修改连接器资源的设置。

“编辑连接器资源”页面包含以下选项。

装入默认值

用于恢复所有设置的默认值的按钮。

JNDI 名称

标识连接器资源的唯一名称。JNDI 名称是只读字段。只能在创建新的连接器资源时指定 JNDI 名称。

池名称

与此资源关联的连接器连接池。

说明

连接器资源的说明。

状态

如果选中此选项，则将启用连接器资源。默认情况下启用此选项。

其他属性

连接器资源的其他属性。可用属性取决于连接器连接池的资源适配器。可以使用在此表中指定的名称-值对，覆盖由资源适配器供应商定义的属性默认值。
