编辑连接器资源

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点。

  3. 在“连接器”节点下，选择“连接器资源”节点。

    此时将打开“连接器资源”页。

  4. 在“连接器资源”页上，单击要编辑的资源的名称。

    此时将打开“编辑连接器资源”页。

  5. 从“池名称”下拉列表中，选择连接器资源所属的连接器连接池。

  6. 在“说明”字段中，键入连接器资源的说明。

  7. 在“目标”选项卡上，选择所需的目标并单击“启用”或“禁用”。

    要从列表中添加或删除目标，请单击“管理目标”。

    通过单击“常规”选项卡返回到“编辑连接器资源”页。

  8. 选择“启用状态”复选框以在运行时启用连接器资源。

    默认情况下启用此选项。

  9. 在“其他属性”部分中，指定连接器资源的其他属性。

    可用属性取决于连接器连接池的资源适配器。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  10. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。
