删除连接器连接池的安全映射

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点。

  3. 在“连接器”节点下，选择“连接器连接池”节点。

    此时将打开“连接器连接池”页。

  4. 在“连接器连接池”页上，单击具有要删除安全映射的连接池的名称。

    此时将打开“编辑连接器连接池”页。

  5. 单击“安全映射”选项卡。

    此时将打开“连接器连接池安全映射”页。

  6. 在“连接器连接池安全映射”页上，选中要删除的安全映射名称旁边的复选框。

  7. 单击“删除”。
