删除受管理对象资源

  1. 在导航树中，展开“资源”节点。
  2. 展开“连接器”节点。
  3. 在“连接器”节点下，选择“受管理对象资源”节点。

    此时将打开“受管理对象资源”页面。

  4. 在“受管理对象资源”页面上，选中要删除的受管理对象资源名称旁边的复选框。
  5. 单击“删除”。
