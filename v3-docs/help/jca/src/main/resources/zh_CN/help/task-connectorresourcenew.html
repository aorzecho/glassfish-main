创建连接器资源

开始之前

创建连接器资源之前，请先创建连接器连接池。

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开“连接器”节点。
  3. 在“连接器”节点下，选择“连接器资源”节点。

    此时将打开“连接器资源”页面。

  4. 在“连接器资源”页面上，单击“新建”按钮。

    此时将打开“新建连接器资源”页面。

  5. 在“JNDI 名称”字段中，键入连接器资源的唯一 JNDI 名称。

    例如：eis/myERP。

  6. 从“池名称”下拉式列表中，选择新连接器资源所属的连接池。
  7. 在“说明”字段中，键入连接器资源的说明。
  8. 选中“启用状态”复选框以在运行时启用连接器资源。

    默认情况下启用此选项。

  9. 在“可用目标”下，选择连接器资源将驻留的域、群集或服务器实例，然后单击“添加”。

    如果不希望将连接器资源部署到“选定目标”下列出的某个域、群集或服务器实例，请从该字段中选择该目标并单击“删除”。

  10. （可选的）在“其他属性”部分中，指定连接器资源的其他属性。

    可用属性取决于连接器连接池的资源适配器。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  11. 单击“确定”。
