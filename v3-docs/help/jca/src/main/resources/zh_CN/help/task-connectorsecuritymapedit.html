编辑连接器连接池的安全映射

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点。

  3. 在“连接器”节点下，选择“连接器连接池”节点。

    此时将打开“连接器连接池”页。

  4. 在“连接器连接池”页上，单击要编辑的连接池的名称。

    此时将打开“编辑连接器连接池”页。

  5. 在“编辑连接器连接池”页上，单击“安全映射”选项卡。

    此时将打开“连接器连接池安全映射”页。

  6. 在“连接器连接池安全映射”页上，单击要编辑的安全映射的名称。

    此时将打开“编辑连接器连接池安全映射”页。

  7. 执行以下操作之一，但不要同时执行这两项操作：

    • 选中“用户组”单选按钮，并键入用户组的名称，以逗号分隔。

    • 选中“主用户”单选按钮，并键入主用户的名称，以逗号分隔。

  8. 在“用户名”字段中，指定 EIS 所需的后端主用户用户名。

  9. 在“口令”字段中，指定 EIS 所需的后端主用户口令。

  10. 单击“保存”。
