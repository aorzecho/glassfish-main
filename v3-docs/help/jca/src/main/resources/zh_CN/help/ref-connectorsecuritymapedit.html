编辑连接器连接池安全映射

使用“编辑连接器连接池安全映射”页面可以编辑连接器连接池的安全映射。

“编辑连接器连接池安全映射”页面包含以下选项：

连接池名称

此安全映射所属的连接器连接池的名称。池名称是只读字段。

安全映射名称

安全映射的名称。名称是只读字段。只能在创建新安全映射时指定安全映射名称。

安全映射名称

安全映射的名称。

用户组

如果选中此单选按钮，则文本字段必须包含用户组的名称，以逗号分隔。

主体

如果选中此单选按钮，则文本字段必须包含主体的名称，以逗号分隔。

用户名

EIS 所需的后端主体用户名。

密码

EIS 所需的后端主体密码。
