连接器连接池安全映射

使用“连接器连接池安全映射”页可以管理连接器连接池的安全映射。

通过连接器连接池的安全映射，可在 Java EE 用户和组与 EIS 用户和组之间创建映射。安全映射可将在 Servlet 或 EJB 验证过程中接收的主用户映射到 EIS 接受的身份证明。可将多个 GlassFish Server 主用户映射到同一后端主用户。

连接池安全映射与连接器工作安全映射有所不同，后者将与接收工作实例关联的主用户映射到 GlassFish Server 域的安全领域中的主用户。有关连接器工作安全映射的信息，请参阅工作安全映射。

对于每个安全映射，将提供以下信息：

名称

安全映射的名称。

“安全映射”表还包含以下选项：

新建

用于创建新安全映射的按钮。

删除

用于删除一个或多个所选安全映射的按钮。
