Nouveau pool de connexions de connecteur (étape 1 sur 2)

Utilisez la page Nouveau pool de connexions de connecteur (étape 1 sur 2) pour créer un nouveau pool de connexions de connecteur.

La page Nouveau pool de connexions de connecteur (étape 1 sur 2) contient les options suivantes.

Nom

Un nom unique qui identifie le pool de connexions du connecteur.

Adaptateur de ressources

L’adaptateur de ressources associé au pool de connexions du connecteur.

Définition de connexion

La définition de connexion pour le pool de connexions du connecteur. Les choix dans la liste dépendent de l’adaptateur de ressource que vous avez sélectionné. La définition de connexion identifie l’élément connection-definition d’un adaptateur de ressource dans le fichier ra.xml.
