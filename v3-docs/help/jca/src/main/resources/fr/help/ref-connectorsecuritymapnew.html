Créer un mappage de sécurité pour le pool de connexions du connecteur

Utilisez la page Créer un mappage de sécurité pour le pool de connexions du connecteur pour créer un mappage de sécurité pour un pool de connexions du connecteur.

La page Créer un mappage de sécurité pour le pool de connexions du connecteur contient les options suivantes.

Nom du pool de connexions

Le nom du pool de connexions du connecteur auquel ce mappage de sécurité appartient. Le nom du pool est un champ en lecture seule.

Nom de la carte de sécurité

Le nom du mappage de sécurité.

Groupes d’utilisateurs

Si ce bouton radio est sélectionné, le champ de texte doit contenir les noms des groupes d’utilisateurs, séparés par des virgules.

Principals

Si cette case est cochée, le champ de texte doit contenir les noms des principals, séparés par des virgules.

Nom d’utilisateur

Le nom d’utilisateur du principal back-end requis par le système EIS.

Mot de passe

Le mot de passe du principal back-end requis par le système EIS.
