Nouvelle ressource de connecteur

Utilisez la page Nouvelle ressource de connecteur pour créer une nouvelle ressource du connecteur.

La page Nouvelle ressource de connecteur contient les options suivantes.

Nom JNDI

Un nom unique qui identifie la ressource du connecteur.

Nom du pool

Le pool de connexions auquel appartient la nouvelle ressource de connecteur.

Description

Une description de la ressource du connecteur.

Statut

Si cette option est sélectionnée, la ressource du connecteur est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires pour la ressource du connecteur. Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur. Les paires Nom-Valeur spécifiées dans ce tableau peuvent être utilisées pour remplacer les valeurs par défaut pour les propriétés définies par le fournisseur de l’adaptateur de ressources.

Cibles

Clusters et instances autonomes pour la ressource du connecteur. Déplacez les cibles voulues dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout Ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Supprimer et Tout Supprimer.

Cette option s'affiche uniquement si les clusters ou les instances autonomes ont été créés dans le domaine.
