Nouvelle ressource de connecteur

La page Nouvelle ressource de connecteur permet de créer une ressource de connecteur.

Elle contient les options suivantes.

Nom JNDI

Nom unique identifiant la ressource de connecteur.

Nom du pool

Pool de connexions auquel appartient la nouvelle ressource de connecteur.

Description

Description de la ressource de connecteur.

Statut

Si cette option est sélectionnée, la ressource de connecteur est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource de connecteur. Les propriétés disponibles dépendent de l'adaptateur de ressource du pool de connexions de connecteur. Les paires nom-valeur spécifiées dans cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressource.

Cibles

Clusters et instances autonomes de la ressource de connecteur. Déplacez les cibles souhaitées dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Enlever et Tout enlever.

Cette option apparaît uniquement si des clusters ou des instances autonomes ont été créés dans le domaine.
