Connecteurs

Utilisez la page Connecteurs pour configurer les ressources de connecteur des types suivants :

  • Ressources du connecteur

  • Pools de connexions de connecteur

  • Ressources d'objet d'administration

  • Mappages de sécurité de travail

Un module de connecteur (aussi appelé adaptateur de ressources) est un composant Java qui permet aux applications d’interagir avec les logiciels d’information d’entreprise (SIE). Les logiciels SIE incluent différents types de systèmes : progiciel de gestion intégré (ERP), traitement des transactions mainframe, bases de données non relationnelles, et les systèmes de messagerie tels que Java Message Service (JMS) notamment.

Un pool de connexions de connecteur est un groupe de connexions réutilisables pour un système EIS donné.

Une ressource de connecteur est un objet de programme fournissant une application et une connexion à un système EIS.

Une ressource d’objet d’administration fournit des fonctionnalités spécialisées pour une application.

Un mappage de sécurité de travail met en correspondance un principal associé à une instance de travail entrant et un principal appartenant au domaine de sécurité de GlassFish Server.
