Connecteurs

La page Connecteurs permet de configurer les types de ressource de connecteur suivants :

  • Ressources du connecteur

  • Pools de connexions du connecteur

  • Ressources d'objet d'administration

  • Mappes de sécurité de travail

Un module connecteur (également appelé adaptateur de ressource) est un composant Java qui permet aux applications d'interagir avec les systèmes d'information d'entreprise (EIS). Les systèmes EIS incluent différents types de système : progiciel de gestion intégré (ERP), traitement des transactions mainframe, bases de données non relationnelles et systèmes de messagerie tels que Java Message Service (JMS) notamment.

Un pool de connexions de connecteur est un groupe de connexions réutilisables pour un système EIS donné.

Une ressource de connecteur est un objet de programme fournissant une connexion vers un système EIS à une application.

Une ressource d'objet administré fournit des fonctionnalités spécialisées pour une application.

Une mappe de sécurité de travail met en correspondance un principal associé à une instance de travail entrante et un principal appartenant au domaine de sécurité GlassFish Server.
