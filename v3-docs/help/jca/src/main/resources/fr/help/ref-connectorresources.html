Ressources du connecteur

Utilisez la page Ressources du connecteur pour configurer les ressources du connecteur.

Une ressource de connecteur est un objet de programme fournissant une application et une connexion à un système EIS.

Pour chaque ressource, les informations suivantes sont fournies.

Nom JNDI

Un nom unique qui identifie la ressource du connecteur.

Activée

True si la ressource du connecteur est activée ou false si la ressource du connecteur est désactivée.

Pool de connexions

Le pool de connexions du connecteur associé à la ressource du connecteur.

Description

Une description de la ressource du connecteur.

Le tableau Ressources contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle ressource du connecteur.

Supprimer

Bouton pour supprimer une ou plusieurs ressources du connecteur sélectionnées.

Activer

Bouton pour activer une ou plusieurs ressources du connecteur sélectionnées.

Désactiver

Bouton pour désactiver une ou plusieurs ressources du connecteur sélectionnées.
