Créer un mappage de sécurité pour un pool de connexions du connecteur

Les mappages de sécurité des pools de connexions de connecteur permettent la création d’un Mappage entre les utilisateurs et groupes de Java EE et ceux d’EIS.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le nœud Connecteurs.
  3. Sous le nœud Connecteurs, sélectionnez le nœud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur s’ouvre.

  4. Sur la page Pools de connexions de connecteurs, cliquez sur le nom du pool de connexions que vous modifiez.

    La page Modification du pool de connexions du connecteur s’ouvre.

  5. Sur la page Modification du pool de connexions du connecteur, cliquez sur l’onglet Mappages de sécurité.

    La page Mappages de sécurité du pool de connexions du connecteur s'ouvre.

  6. Sur la page Mappages de sécurité du pool de connexions du connecteur, cliquez sur Nouveau.

    La page Créer un mappage de sécurité pour le pool de connexions du connecteur s'ouvre.

  7. Dans le champ Nom de la carte de sécurité, entrez un nom pour le mappage de sécurité.
  8. Effectuez l’une des opérations suivantes, mais pas les deux :
    • Sélectionnez le bouton radio Groupes d’utilisateurs et tapez le nom des groupes d’utilisateurs, séparées par des virgules.
    • Sélectionnez le bouton radio Principals et entrez les noms des principals, séparés par des virgules.
  9. Dans le champ Nom d’utilisateur, spécifiez le nom d’utilisateur du principal back-end requis par le système SIE.
  10. Dans le champ Mot de passe, spécifiez le mot de passe du principal back-end requis par le système SIE.
  11. Cliquez sur OK.
  12. Redémarrez GlassFish Server.
