Nouvelle configuration d’adaptateur de ressources

Utilisez la page Nouvelle configuration d’adaptateur de ressources pour créer une nouvelle configuration de l’adaptateur de ressources.

La page Nouvelle configuration d’adaptateur de ressources contient les options suivantes.

Nom de l’adaptateur de ressources

Le nom de l’adaptateur de ressources que vous configurez.

ID de pool de threads

Le pool de threads à utiliser pour la configuration de l’adaptateur de ressources.

Autres propriétés

Propriétés supplémentaires pour l’adaptateur de ressources. Les propriétés disponibles sont les noms des méthodes de mutateur de la classe référencée par l’élément resourceadapter-class dans le fichier ra.xml, qui définit le nom de la classe du composant JavaBeans de l’adaptateur de ressources. Toutes les modifications apportées aux propriétés définies ici remplacent les valeurs par défaut présentes dans le fichier ra.xml.
