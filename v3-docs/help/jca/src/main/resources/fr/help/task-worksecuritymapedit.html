Modifier une mappe de sécurité de travail

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Mappes de sécurité de travail.

    La page Mappes de sécurité de travail apparaît.

  4. Sur la page Mappes de sécurité de travail, cliquez sur le nom de la mappe de sécurité de travail à modifier.

    La page Modifier une mappe de sécurité de travail apparaît.

  5. Utilisez le champ Groupe ou le champ Principal comme suit.

    • Si vous sélectionnez le bouton radio Groupe, spécifiez une mappe pour chaque groupe d'utilisateurs du système EIS back-end avec le groupe d'utilisateurs correspondant défini dans le domaine de sécurité GlassFish Server.

      Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Si vous sélectionnez le bouton radio Principal, spécifiez une mappe pour chaque principal EIS back-end avec le principal correspondant défini dans le domaine de sécurité GlassFish Server.

      Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  6. Dans la liste déroulante Adaptateur de ressources, sélectionnez l'adaptateur de ressource pour le système EIS.

  7. Dans le champ Description, saisissez la description de la mappe de sécurité de travail.

  8. Sélectionnez la case Activé du champ Statut pour activer la mappe de sécurité de travail.

    Cette option est activée par défaut.

  9. Cliquez sur OK.
