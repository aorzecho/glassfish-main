Modifier une ressource d'objet d'administration

La page Modifier une ressource d'objet d'administration permet de modifier les paramètres d'une ressource d'objet administré.

Elle contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom JNDI

Nom unique identifiant la ressource d'objet administré. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource d'objet administré.

Adaptateur de ressources

Adaptateur de ressource pour la ressource d'objet administré.

Type de ressource

Type Java de la ressource d'objet administré (par exemple, javax.jms.Topic).

Nom de classe

Nom de la classe d'implémentation associée au type de ressource.

Description

Description de la ressource d'objet administré.

Statut

Si cette option est sélectionnée, la ressource d'objet administré est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource d'objet administré. Les propriétés disponibles dépendent de l'adaptateur de ressource du pool de connexions de connecteur. Les paires nom-valeur spécifiées dans cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressource.
