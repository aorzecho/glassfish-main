Pools de connexions de connecteur

Utilisez la page Pools de connexions du connecteur pour configurer les pools de connexions du connecteur.

Un pool de connexions de connecteur est un groupe de connexions réutilisables pour un système EIS donné.

Pour chaque ressource, les informations suivantes sont fournies.

Nom du pool

Un nom unique qui identifie le pool de connexions du connecteur.

Adaptateur de ressources

L’adaptateur de ressources associé au pool de connexions du connecteur.

Définition de connexion

Le nom de classe de la définition de connexion pour le pool de connexions du connecteur.

Description

Une description du pool de connexions du connecteur.

Le tableau Pools contient également les options suivantes.

Nouveau

Bouton pour créer un nouveau pool de connexions du connecteur.

Supprimer

Bouton pour supprimer un ou plusieurs pools de connexions du connecteur sélectionnés.
