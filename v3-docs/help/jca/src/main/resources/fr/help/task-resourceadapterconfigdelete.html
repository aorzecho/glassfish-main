Supprimer une configuration de l’adaptateur de ressources

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le noeud Ressources, sélectionnez le noeud Configurations d’adaptateurs de ressources.

    La page Configurations d’adaptateurs de ressources s’ouvre.

  3. Sur la page Configurations d’adaptateurs de ressources, sélectionnez la case en regard du nom de la configuration de l’adaptateur de ressources que vous supprimez.
  4. Cliquez sur Supprimer.
