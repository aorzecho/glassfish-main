Supprimer une configuration d'adaptateur de ressource

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez le noeud Configurations d'adaptateurs de ressources.

    La page Configurations d'adaptateurs de ressources apparaît.

  3. Sur la page Configurations d'adaptateurs de ressources, sélectionnez la case en regard du nom de la configuration de l'adaptateur de ressource à supprimer.

  4. Cliquez sur Supprimer.
