Modifier les propriétés d'un pool de connexions de connecteur

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur apparaît.

  4. Sur la page Pools de connexions du connecteur, cliquez sur le nom du pool de connexions du connecteur à modifier.

    La page Modifier le pool de connexions du connecteur apparaît.

  5. Sur la page Modifier le pool de connexions du connecteur, cliquez sur l'onglet Autres propriétés.

    La page Modifier les propriétés d'un pool de connexions de connecteur apparaît.

  6. Dans la table Autres propriétés, spécifiez les propriétés supplémentaires du pool de connexions de connecteur.

    Les propriétés spécifiées dépendent de l'adaptateur de ressource utilisé par ce pool. Les paires nom-valeur spécifiées par l'utilisateur à l'aide de cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressource.

    Si vous n'indiquez pas de valeur pour une propriété, celle-ci est enlevée de la table à l'issue de la modification du pool de connexions. Si vous ne souhaitez pas définir immédiatement la valeur d'une propriété mais souhaitez conserver cette propriété dans la table afin de définir sa valeur ultérieurement, entrez des parenthèses vides dans le champ Valeur :

    ()

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  7. Cliquez sur Enregistrer.
