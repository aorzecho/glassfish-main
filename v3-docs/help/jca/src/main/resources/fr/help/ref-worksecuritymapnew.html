Nouvelle mappe de sécurité de travail

La page Nouvelle mappe de sécurité de travail permet de créer une mappe de sécurité de travail.

Elle contient les options suivantes.

Nom de la mappe de sécurité

Nom de la mappe de sécurité de travail.

Groupe

Si le bouton radio Mise en correspondance de groupes est sélectionné, le champ de texte doit spécifier une mappe pour chaque groupe d'utilisateurs du système EIS back-end avec le groupe d'utilisateurs correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Si le bouton radio Mise en correspondance de principaux est sélectionné, le champ de texte doit spécifier une mappe pour chaque principal EIS back-end avec le principal correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptateur de ressources

Adaptateur de ressource pour le système EIS.

Description

Description de la mappe de sécurité de travail.

Statut

Si cette option est sélectionnée, la mappe de sécurité de travail est activée. Cette option est activée par défaut.
