Nouveau mappage de sécurité de travail

Utilisez la page Nouveau mappage de sécurité de travail pour créer un nouveau mappage de sécurité de travail.

La page Nouveau mappage de sécurité de travail contient les options suivantes.

Nom de la carte de sécurité

Le nom du mappage de sécurité de travail.

Groupe

Si le bouton radio Mappage de groupes est sélectionné, le champ de texte doit spécifier un mappage de chaque groupe d’utilisateurs SIE back-end avec le groupe d’utilisateurs correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Si le bouton radio Mappage de principals est sélectionné, le champ de texte doit spécifier un mappage de chaque principal SIE back-end avec le principal correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptateur de ressources

L’adaptateur de ressources pour le système EIS.

Description

Une description du mappage de sécurité de travail.

Statut

Si cette option est sélectionnée, le mappage de sécurité de travail est activé. Cette option est activée par défaut.
