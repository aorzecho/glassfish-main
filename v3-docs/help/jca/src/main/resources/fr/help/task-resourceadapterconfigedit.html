Modifier une configuration de l’adaptateur de ressources

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le noeud Ressources, sélectionnez le noeud Configurations d’adaptateurs de ressources.

    La page Configurations d’adaptateurs de ressources s’ouvre.

  3. Sur la page Configurations d’adaptateurs de ressources, cliquez sur le nom de la configuration de l’adaptateur de ressources que vous modifiez.

    La page Modifier la configuration de l’adaptateur de ressources s’ouvre.

  4. Dans la liste déroulante ID du pool de threads, sélectionnez le pool de threads à utiliser pour la configuration de l’adaptateur de ressources.
  5. (Facultatif) Dans la section Autres propriétés, spécifiez les autres propriétés pour l’adaptateur de ressources.

    Les propriétés disponibles sont les noms des méthodes de mutateur de la classe référencée par l’élément resourceadapter-class dans le fichier ra.xml, qui définit le nom de la classe du composant JavaBeans de l’adaptateur de ressources. Toutes les modifications apportées aux propriétés définies ici remplacent les valeurs par défaut présentes dans le fichier ra.xml.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  6. Cliquez sur Enregistrer.
