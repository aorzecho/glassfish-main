Modifier une configuration d'adaptateur de ressource

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez le noeud Configurations d'adaptateurs de ressources.

    La page Configurations d'adaptateurs de ressources apparaît.

  3. Sur la page Configurations d'adaptateurs de ressources, cliquez sur le nom de la configuration de l'adaptateur de ressource à modifier.

    La page Modifier la configuration de l'adaptateur de ressources apparaît.

  4. Dans la liste déroulante ID du pool de threads, sélectionnez le pool de threads à utiliser pour la configuration de l'adaptateur de ressource.

  5. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires de l'adaptateur de ressource.

    Les propriétés disponibles sont les noms des méthodes set de la classe référencée par l'élément resourceadapter-class dans le fichier ra.xml, qui définit le nom de la classe du composant JavaBeans de l'adaptateur de ressource. Toutes les modifications apportées aux propriétés définies ici remplacent les valeurs par défaut présentes dans le fichier ra.xml.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  6. Cliquez sur Enregistrer.
