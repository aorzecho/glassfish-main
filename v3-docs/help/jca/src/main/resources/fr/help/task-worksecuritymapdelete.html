Supprimer une mappe de sécurité de travail

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Mappes de sécurité de travail.

    La page Mappes de sécurité de travail apparaît.

  4. Sur la page Mappes de sécurité de travail, sélectionnez la case en regard du nom de la mappe de sécurité de travail à supprimer.

  5. Cliquez sur Supprimer.
