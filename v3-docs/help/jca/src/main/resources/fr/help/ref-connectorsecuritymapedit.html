Modifier une mappe de sécurité du pool de connexions du connecteur

La page Modifier une mappe de sécurité du pool de connexions du connecteur permet de modifier une mappe de sécurité pour un pool de connexions de connecteur.

Elle contient les options suivantes.

Nom du pool de connexions

Nom du pool de connexions de connecteur auquel cette mappe de sécurité appartient. Le nom du pool est un champ en lecture seule.

Nom de la mappe de sécurité

Nom de la mappe de sécurité. Le champ Nom est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une mappe de sécurité.

Nom de la mappe de sécurité

Nom de la mappe de sécurité.

Groupes d'utilisateurs

Si ce bouton radio est sélectionné, le champ de texte doit contenir les noms des groupes d'utilisateurs, séparés par des virgules.

Principaux

Si cette case est cochée, le champ de texte doit contenir les noms des principaux, séparés par des virgules.

Nom utilisateur

Nom utilisateur du principal back-end requis par le système EIS.

Mot de passe

Mot de passe du principal back-end requis par le système EIS.
