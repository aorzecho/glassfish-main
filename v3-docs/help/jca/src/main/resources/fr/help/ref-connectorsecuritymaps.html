Mappages de sécurité des pools de connexions du connecteur

Utilisez la page Mappages de sécurité du pool de connexions du connecteur pour gérer les mappages de sécurité des pools de connexions du connecteur.

Les mappages de sécurité des pools de connexions de connecteur permettent la création d’un Mappage entre les utilisateurs et groupes de Java EE et ceux d’EIS. Un mappage de sécurité associe le principal reçu durant l’authentification EJB ou du servlet à l’aide des informations d’authentification acceptées par le système EIS. Il est possible d’associer plusieurs principals GlassFish Server au même principal back-end.

Un mappage de sécurité pour un pool de connexions est différent d’un mappage de sécurité de travail d’un connecteur, qui fait correspondre un principal associé à une instance de travail entrante à un principal dans un domaine de sécurité du domaine GlassFish Server. Consultez Mappages de sécurité de travail pour plus d’informations sur les mappages de sécurité de travail du connecteur.

Pour chaque mappage de sécurité, les informations suivantes sont fournies.

Nom

Le nom du mappage de sécurité.

Le tableau Mappages de sécurité contient également les options suivantes.

Nouveau

Bouton pour créer un nouveau mappage de sécurité.

Supprimer

Bouton pour supprimer un ou plusieurs mappages de sécurité sélectionnés.
