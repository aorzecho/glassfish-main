Créer un mappage de sécurité de travail

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le nœud Connecteurs.
  3. Sous le noeud Connecteurs, sélectionnez le noeud Mappages de sécurité de travail.

    La page Mappages de sécurité de travail s’ouvre.

  4. Sur la page Mappages de sécurité de travail, cliquez sur Nouveau.

    La page Nouveau mappage de sécurité de travail s’ouvre.

  5. Dans le champ Nom de la carte de sécurité, entrez un nom pour le mappage de sécurité de travail.
  6. Utilisez le champ Groupe ou le champ Principal comme suit.
    • Si vous sélectionnez le bouton radio Groupe, saisissez un mappage de chaque groupe d’utilisateurs de système SIE avec le groupe d’utilisateur correspondant défini dans le domaine de sécurité GlassFish Server.

      Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • Si vous sélectionnez le bouton radio Principal, saisissez un mappage de chaque principal de système SIE back-end avec le principal correspondant défini dans le domaine de sécurité GlassFish Server.

      Utilisez une liste séparée par des virgules pour spécifier plusieurs mappes. Exemple :

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  7. Dans la liste déroulante Adaptateur de ressources, sélectionnez l’adaptateur de ressources pour le système EIS.
  8. Dans le champ Description, saisissez une description du mappage de sécurité de travail.
  9. Sélectionnez la case Activé de Statut pour activer le mappage de sécurité de travail.

    Cette option est activée par défaut.

  10. Cliquez sur OK.
