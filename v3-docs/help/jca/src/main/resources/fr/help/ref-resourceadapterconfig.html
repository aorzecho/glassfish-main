Configurations d'adaptateurs de ressources

La page Configurations d'adaptateurs de ressources permet de gérer les configurations (adaptateur de ressource) du connecteur.

La configuration d'adaptateur de ressource fournit les informations de configuration relatives à un adaptateur de ressource. Avec la Console d'administration , vous créez une configuration d'adaptateur de ressource une fois que ce dernier est déployé. Dans ce cas, l'adaptateur de ressource est redémarré avec la nouvelle configuration. Pour créer une configuration d'adaptateur de ressource avant de déployer ce dernier, utilisez la commande asadmin create-resource-adapter-config.

Avant de pouvoir créer une configuration d'adaptateur de ressource, vous devez créer un pool de threads pour l'adaptateur de ressource à utiliser.

Pour chaque configuration d'adaptateur de ressource, les informations suivantes sont fournies.

Nom

Nom de l'adaptateur de ressource.

La table Configurations contient également les options suivantes.

Nouveau

Bouton permettant de créer une configuration d'adaptateur de ressource.

Supprimer

Bouton permettant de supprimer les configurations d'adaptateur de ressource sélectionnées.
