Configurations d’adaptateurs de ressources

Utilisez la page Configurations d’adaptateurs de ressources pour gérer les configurations (adaptateur de ressources) du connecteur.

La configuration de l’adaptateur de ressources fournit les informations de configuration pour un adaptateur de ressources. Avec Console d'administration , vous créez une configuration de l’adaptateur de ressources une fois que l’adaptateur de ressources est déployé. Dans ce cas, l’adaptateur de ressources est redémarré avec la nouvelle configuration. Pour créer une configuration de l’adaptateur de ressources avant de déployer l’adaptateur de ressources, utilisez la commande asadmin create-resource-adapter-config.

Avant de pouvoir créer une configuration de l’adaptateur de ressources, vous devez créer un pool de threads pour l’adaptateur de ressources à utiliser.

Pour chaque configuration d’adaptateur de ressources, les informations suivantes sont fournies.

Nom

Le nom de l’adaptateur de ressources.

Le tableau Configurations contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle configuration de l’adaptateur de ressources.

Supprimer

Bouton pour supprimer une ou plusieurs configurations de l’adaptateur de ressources sélectionnées.

Tâches connexes

  • Modifier une configuration de l'adaptateur de ressources

  • Supprimer une configuration de l'adaptateur de ressources
