Créer une ressource de connecteur

Avant de commencer

Avant de créer une ressource de connecteur, créez d’abord un pool de connexions du connecteur.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Sous le nœud Ressources, sélectionnez le nœud Connecteurs.
  3. Sous le nœud Connecteurs, sélectionnez le nœud Ressources du connecteur.

    La page Ressources du connecteur s’ouvre.

  4. Sur la page Ressources du connecteur, cliquez sur le bouton Nouveau.

    La page Nouvelle ressource de connecteur s’ouvre.

  5. Dans le champ Nom JNDI, saisissez un nom JNDI unique pour la ressource du connecteur.

    Par exemple : eis/myERP.

  6. Dans la liste déroulante Nom du pool, sélectionnez le pool de connexions auquel la nouvelle ressource de connecteur appartient.
  7. Dans le champ Description, saisissez une description de la ressource de connecteur.
  8. Sélectionnez la case Activé de Statut pour activer la ressource de connecteur lors de l’exécution.

    Cette option est activée par défaut.

  9. Sous Cibles disponibles, sélectionnez le domaine, le cluster ou les instances de serveur où doit résider la ressource du connecteur et cliquez sur Ajouter.

    Si vous ne souhaitez pas déployer la ressource du connecteur dans l'un des domaines, clusters ou dans l'une des instances de serveur répertoriées sous Cibles sélectionnées, sélectionnez la cible dans le champ, puis cliquez sur Supprimer.

  10. (Facultatif) Dans la section Autres propriétés, spécifiez des propriétés supplémentaires pour la ressource du connecteur.

    Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  11. Cliquez sur OK.
