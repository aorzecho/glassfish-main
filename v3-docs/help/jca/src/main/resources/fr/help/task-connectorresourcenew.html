Créer une ressource de connecteur

Avant de commencer

Avant de créer une ressource de connecteur, vous devez d'abord créer un pool de connexions de connecteur.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources du connecteur.

    La page Ressources du connecteur apparaît.

  4. Sur la page Ressources du connecteur, cliquez sur le bouton Nouveau.

    La page Nouvelle ressource de connecteur apparaît.

  5. Dans le champ Nom JNDI, saisissez le nom JNDI unique de la ressource de connecteur.

    Par exemple : eis/myERP.

  6. Dans la liste déroulante Nom du pool, sélectionnez le pool de connexions auquel la nouvelle ressource de connecteur appartient.

  7. Dans le champ Description, saisissez la description de la ressource de connecteur.

  8. Sélectionnez la case Activé du champ Statut pour activer la ressource de connecteur lors de l'exécution.

    Cette option est activée par défaut.

  9. Sous Cibles disponibles, sélectionnez le domaine, le cluster ou les instances de serveur où doit résider la ressource de connecteur, puis cliquez sur Ajouter.

    Si vous ne souhaitez pas déployer la ressource de connecteur dans l'un des domaines, clusters ou dans l'une des instances de serveur répertoriés sous Cibles sélectionnées, sélectionnez la cible dans le champ, puis cliquez sur Enlever.

  10. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires de la ressource de connecteur.

    Les propriétés disponibles dépendent de l'adaptateur de ressource du pool de connexions de connecteur.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  11. Cliquez sur OK.
