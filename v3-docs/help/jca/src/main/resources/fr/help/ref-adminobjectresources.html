Ressources d’objet d’administration

Utilisez la page Ressources d’objet d’administration pour configurer les ressources de l’objet d’administration.

Une ressource d’objet gérée fournit des fonctionnalités spécialisées que l’adaptateur de ressources définit pour le module connecteur déployé.

Pour chaque ressource d’objet d’administration, les informations suivantes sont fournies.

Nom JNDI

Un nom unique qui identifie la ressource d’objet d’administration.

Activée

True si la ressource d’objet d’administration est activée ou false si la ressource est désactivée.

Type de ressource

Le type Java de la ressource d’objet d’administration.

Description

Une description de la ressource d’objet d’administration.

Le tableau Ressources contient également les options suivantes.

Nouveau

Bouton pour créer une nouvelle ressource d’objet d’administration.

Supprimer

Bouton pour supprimer une ou plusieurs ressources de l’objet d’administration sélectionnées.

Activer

Bouton pour activer une ou plusieurs ressources de l’objet d’administration sélectionnées.

Désactiver

Bouton pour désactiver une ou plusieurs ressources de l’objet d’administration sélectionnées.
