Ressources d'objet d'administration

La page Ressources d'objet d'administration permet de configurer les ressources d'objet administré.

Une ressource d'objet administré fournit des fonctionnalités spécialisées que l'adaptateur de ressource définit pour le module connecteur déployé.

Pour chaque ressource d'objet administré, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource d'objet administré.

Activé

True si la ressource d'objet administré est activée ou False si la ressource est désactivée.

Type de ressource

Type Java de la ressource d'objet administré.

Description

Description de la ressource d'objet administré.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource d'objet administré.

Supprimer

Bouton permettant de supprimer les ressources d'objet administré sélectionnées.

Activer

Bouton permettant d'activer les ressources d'objet administré sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources d'objet administré sélectionnées.

Tâches connexes

  • Créer une ressource d'objet administré

  • Modifier une ressource d'objet administré

  • Supprimer une ressource d'objet administré

Commandes asadmin connexes
