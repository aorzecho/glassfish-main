Modifier les propriétés d'un pool de connexions de connecteur

La page Modifier les propriétés d'un pool de connexions de connecteur permet de modifier les propriétés d'un pool de connexions de connecteur.

Elle contient les options suivantes.

Autres propriétés

Propriétés supplémentaires du pool de connexions de connecteur. Les propriétés disponibles dépendent de l'adaptateur de ressource du pool de connexions de connecteur. Les paires nom-valeur spécifiées dans cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressource.

Si vous n'indiquez pas de valeur pour une propriété, celle-ci est enlevée de la table à l'issue de la modification du pool de connexions. Si vous ne souhaitez pas définir immédiatement la valeur d'une propriété mais souhaitez conserver cette propriété dans la table afin de définir sa valeur ultérieurement, entrez des parenthèses vides dans le champ Valeur :

()
