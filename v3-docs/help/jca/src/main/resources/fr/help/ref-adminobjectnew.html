Nouvelle ressource d’objet d’administration

Utilisez la page Nouvelle ressource d’objet d’administration pour créer une nouvelle ressource d’objet d’administration.

La page Nouvelle ressource d’objet d’administration contient les options suivantes.

Nom JNDI

Un nom unique qui identifie la ressource d’objet d’administration.

Adaptateur de ressources

L’adaptateur de ressources pour la ressource d’objet d’administration.

Type de ressource

Le type Java de la ressource d’objet d’administration (ou par exemple javax.jms.Topic).

Nom de classe

Le nom de la classe d’implémentation associé au type de ressource.

Description

Une description de la ressource d’objet d’administration.

Statut

Si cette option est sélectionnée, la ressource d’objet d’administration est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires pour la ressource d’objet d’administration. Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur. Les paires Nom-Valeur spécifiées dans ce tableau peuvent être utilisées pour remplacer les valeurs par défaut pour les propriétés définies par le fournisseur de l’adaptateur de ressources.

Cibles

Clusters et instances autonomes pour la ressource d'objet géré. Déplacez les cibles voulues dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout Ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Supprimer et Tout Supprimer.

Cette option s'affiche uniquement si les clusters ou les instances autonomes ont été créés dans le domaine.
