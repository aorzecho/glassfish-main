Modifier une ressource du connecteur

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le nœud Connecteurs.
  3. Sous le nœud Connecteurs, sélectionnez le nœud Ressources du connecteur.

    La page Ressources du connecteur s’ouvre.

  4. Sur la page Ressources du connecteur, cliquez sur le nom de la ressource que vous modifiez.

    La page Modification d’une ressource de connecteur s’ouvre.

  5. Dans la liste déroulante Nom du pool, sélectionnez le pool de connexions de connecteur auquel la ressource du connecteur appartient.
  6. Dans le champ Description, saisissez une description de la ressource de connecteur.
  7. Dans l'onglet Cible, sélectionnez les cibles souhaitées et cliquez sur Activer ou Désactiver.

    Pour ajouter ou supprimer des cibles de la liste, cliquez sur Gestion des cibles.

    Retournez à la page Modification d'une ressource de connecteur en cliquant sur l'onglet Général.

  8. Sélectionnez la case Activé de Statut pour activer la ressource de connecteur lors de l’exécution.

    Cette option est activée par défaut.

  9. (Facultatif) Dans la section Autres propriétés, spécifiez des propriétés supplémentaires pour la ressource du connecteur.

    Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  10. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
