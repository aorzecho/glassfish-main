Modifier une ressource d’objet administré

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le noeud Connecteurs.
  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources d’objet d’administration.

    La page Ressources d’objet d’administration s’ouvre.

  4. Sur la page Ressources de l’objet d’administration, cliquez sur le nom de la ressource d’objet d’administration que vous modifiez.

    La page Modification d’une ressource d’objet d’administration s’ouvre.

  5. Dans la liste déroulante Adaptateur de ressources, sélectionnez l’adaptateur de ressources pour la ressource d’objet d’administration.
  6. Dans la liste déroulante Type de ressource, sélectionnez le type Java pour la ressource d’objet d’administration.
  7. Dans la liste déroulante Nom de classe, sélectionnez le nom de la classe d’implémentation associé au type de ressource.
  8. Dans le champ Description, saisissez une description des ressources d’objet d’administration.
  9. Sélectionnez la case Activé de Statut pour activer la ressource d’objet d’administration.

    Cette option est activée par défaut.

  10. (Facultatif) Dans le tableau Autres propriétés, spécifiez les autres propriétés pour la ressource d’objet d’administration.

    Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  11. Dans l'onglet Cible, cliquez sur Gestion des cibles pour modifier les cibles sur lesquelles l'objet géré est déployé.

    Retournez à la page Modification d'une ressource d'objet d'administration en cliquant sur l'onglet Général.

  12. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
