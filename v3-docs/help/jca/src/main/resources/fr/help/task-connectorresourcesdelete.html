Supprimer une ressource de connecteur

Une ressource de connecteur est automatiquement supprimée si vous supprimez le pool de connexions de connecteur associé à la ressource.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources du connecteur.

    La page Ressources du connecteur apparaît.

  4. Sur la page Ressources du connecteur, cochez la case en regard du nom de la ressource à supprimer.

  5. Cliquez sur Supprimer.
