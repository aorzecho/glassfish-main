Service du connecteur

Utilisez la page Service du connecteur pour modifier les paramètres généraux pour le service du connecteur, qui régit les adaptateurs de ressources.

La page Service du connecteur contient les options suivantes.

Restaurer les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de la configuration

Nom de la configuration à laquelle les paramètres de cette page s'appliquent. Ce champ est en lecture seule.

Délai d’arrêt

Le nombre maximum de secondes autorisé pendant l’arrêt de GlassFish Server pour que la méthode ResourceAdapter.stop d’une instance de module connecteur se termine. Les adaptateurs de ressources dépassant ce délai pour s’arrêter sont ignorés et l’arrêt de GlassFish Server se poursuit. La valeur par défaut est 30 secondes.

Stratégie de chargement de classe de connecteur

La stratégie à utiliser pour le chargement des classes. Les choix disponibles sont les suivants :

derived

Indique que les adaptateurs de ressources sont fournis selon les références d’une application vers l’une des ressources de l’adaptateur de ressources. Cette stratégie est la stratégie par défaut.

global

Indique que tous les adaptateurs de ressources seront visibles à toutes les applications.

Cette stratégie ne s’applique pas à l’adaptateur de ressources du système JMS préinstallé, jmsra, qui est toujours visible de toutes les applications.
