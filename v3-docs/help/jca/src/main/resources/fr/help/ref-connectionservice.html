Service du connecteur

La page Service du connecteur permet de modifier les paramètres généraux du service du connecteur, qui régit les adaptateurs de ressource.

Elle contient les options suivantes.

Charger les valeurs par défaut

Bouton de restauration des valeurs par défaut des paramètres qui ont des valeurs par défaut. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Temporisation d'arrêt

Délai maximal autorisé, en secondes, pour l'exécution de la méthode ResourceAdapter.stop d'une instance de module connecteur, au cours de l'arrêt de GlassFish Server. Les adaptateurs de ressource dépassant ce délai de fermeture sont ignorés et l'arrêt de GlassFish Server se poursuit. La valeur par défaut est de 30 secondes.

Règle de chargement de classe de connecteur

Règle à utiliser pour le chargement des classes. Les choix disponibles sont les suivants :

derived

Indique que les adaptateurs de ressource sont fournis en fonction des références d'une application à l'une des ressources de l'adaptateur de ressource. Cette règle est utilisée par défaut.

global

Indique que l'ensemble des adaptateurs de ressource sera visible par toutes les applications.

Cette règle ne s'applique pas à l'adaptateur de ressource du système JMS préinstallé, jmsra, qui est toujours visible par toutes les applications.
