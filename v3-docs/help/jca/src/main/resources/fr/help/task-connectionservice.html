Configurer le service du connecteur

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, sélectionnez le noeud Service du connecteur.

    La page Service du connecteur apparaît.

  3. Dans le champ Temporisation d'arrêt, saisissez le délai maximal autorisé, en secondes, pour l'exécution de la méthode ResourceAdapter.stop d'une instance de module connecteur, au cours de l'arrêt de GlassFish Server.

    Les adaptateurs de ressource dépassant ce délai de fermeture sont ignorés et l'arrêt de GlassFish Server se poursuit. La valeur par défaut est de 30 secondes.

  4. Dans la liste déroulante Règle de chargement de classe de connecteur, sélectionnez la règle à utiliser.

    • Sélectionnez derived pour indiquer que les adaptateurs de ressource sont fournis en fonction des références d'une application à l'une des ressources de l'adaptateur de ressource.

      Cette règle est utilisée par défaut.

    • Sélectionnez global pour indiquer que l'ensemble des adaptateurs de ressource sera visible par toutes les applications.

    Cette règle ne s'applique pas à l'adaptateur de ressource du système JMS préinstallé, jmsra, qui est toujours visible par toutes les applications.

  5. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.
