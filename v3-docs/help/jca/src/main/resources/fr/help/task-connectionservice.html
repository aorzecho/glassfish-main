Configurer le service du connecteur

  1. Dans l’arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Service du connecteur.

    La page Service du connecteur s’ouvre.

  3. Dans le champ Délai d’arrêt, saisissez le nombre maximum de secondes autorisé pendant l’arrêt de GlassFish Server pour que la méthode ResourceAdapter.stop d’une instance du module de connecteur se termine.

    Les adaptateurs de ressources dépassant ce délai pour s’arrêter sont ignorés et l’arrêt de GlassFish Server se poursuit. La valeur par défaut est 30 secondes.

  4. Dans la liste déroulante Stratégie de chargement de classe de connecteur, sélectionnez la stratégie à utiliser.
    • Sélectionnez derived pour indiquer que les adaptateurs de ressources sont fournis selon les références d’une application vers l’une des ressources de l’adaptateur de ressources.

      Cette stratégie est la stratégie par défaut.

    • Sélectionnez global pour indiquer que tous les adaptateurs de ressources seront visibles de toutes les applications.

    Cette stratégie ne s’applique pas à l’adaptateur de ressources du système JMS préinstallé, jmsra, qui est toujours visible de toutes les applications.

  5. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
