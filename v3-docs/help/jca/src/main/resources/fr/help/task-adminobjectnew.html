Créer une ressource d’objet géré

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le noeud Connecteurs.
  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources d’objet d’administration.

    La page Ressources d’objet d’administration s’ouvre.

  4. Sur la page Ressources d’objet d’administration, cliquez sur le bouton Nouveau.

    La page Ressources d’objet d’administration s’ouvre.

  5. Dans le champ Nom JNDI, saisissez un nom unique qui identifie la ressource d’objet d’administration.
  6. Dans la liste déroulante Adaptateur de ressources, sélectionnez l’adaptateur de ressources pour la ressource d’objet d’administration.
  7. Dans la liste déroulante Type de ressource, sélectionnez le type Java pour la ressource d’objet d’administration.
  8. Dans la liste déroulante Nom de classe, sélectionnez le nom de la classe d’implémentation associé au type de ressource.
  9. Dans le champ Description, saisissez une description des ressources d’objet d’administration.
  10. Sélectionnez la case Activé de Statut pour activer la ressource d’objet d’administration.

    Cette option est activée par défaut.

  11. (Facultatif) Dans le tableau Autres propriétés, spécifiez les autres propriétés pour la ressource d’objet d’administration.

    Les propriétés disponibles dépendent de l’adaptateur de ressources pour le pool de connexions du connecteur.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  12. Dans la section Cibles disponibles, sélectionnez le domaine, le cluster ou les instances de serveur où doit résider l'objet géré et cliquez sur Ajouter.

    Pour supprimer l'un des domaines, clusters ou l'une des instances de serveur répertorié(e)s dans le champ Sélectionné, sélectionnez ce dernier/cette dernière dans le champ, puis cliquez sur Supprimer.

  13. Cliquez sur OK.
