Configurer l’accès à EIS

  1. Déployez (installez) un connecteur. Reportez-vous à To Deploy a Connector Module.
  2. Créez un pool de connexions pour le connecteur. Voir Créer un pool de connexions de connecteur.
  3. Créez une ressource de connecteur associée au pool de connexions. Voir Créer une ressource de connecteur.
