Supprimer un pool de connexions de connecteur

Lorsque vous supprimez un pool de connexions de connecteur, toutes les ressources de connecteur qui dépendent de ce pool sont également supprimées.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur apparaît.

  4. Sur la page Pools de connexions du connecteur, sélectionnez la case en regard du nom du pool de connexions à supprimer.

  5. Cliquez sur Supprimer.
