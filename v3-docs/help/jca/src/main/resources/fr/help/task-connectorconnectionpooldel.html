Supprimer un pool de connexions du connecteur

Lorsque vous supprimez un pool de connexions de connecteur, toutes les ressources du connecteur qui dépendent de ce pool de connexions de connecteur sont également supprimées.

  1. Dans l’arborescence de navigation, développez le nœud Ressources.
  2. Développez le nœud Connecteurs.
  3. Sous le nœud Connecteurs, sélectionnez le nœud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur s’ouvre.

  4. Sur la page Pools de connexions de connecteurs, cochez la case en regard du nom du pool de connexions que vous supprimez.
  5. Cliquez sur Supprimer.
