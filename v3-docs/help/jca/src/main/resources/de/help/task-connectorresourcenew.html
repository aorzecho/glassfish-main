So erstellen Sie eine Verbindungsressource

Bevor Sie beginnen

Bevor Sie eine Verbindungsressource erstellen, müssen Sie zunächst einen Konnektor-Verbindungspool erstellen.

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie unter dem Ressourcenknoten den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Verbindungsressourcen" aus.

    Die Seite "Verbindungsressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "Verbindungsressourcen" auf die Schaltfläche "Neu".

    Die Seite "Neue Konnektorressource" wird geöffnet.

  5. Geben Sie im Feld "JNDI-Name" einen eindeutigen JNDI-Namen für die Verbindungsressource ein.

    Beispiel: eis/myERP.

  6. Wählen Sie in der Dropdown-Liste "Poolname" den Verbindungspool aus, dem die neue Verbindungsressource angehört.
  7. Geben Sie im Feld "Beschreibung" eine Beschreibung der Verbindungsressource ein.
  8. Aktivieren Sie das Kontrollkästchen "Status", um die Verbindungsressource zur Laufzeit zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  9. Wählen Sie unter "Verfügbare Ziele" die Domäne, Cluster oder Serverinstanzen, in denen die Verbindungsressource platziert werden soll, und klicken Sie auf "Hinzufügen".

    Wenn Sie die Verbindungsressource nicht für eine der Domänen, Cluster oder Serverinstanzen, die im Feld "Ausgewählte Ziele" aufgelistet sind, bereitstellen möchten, wählen Sie sie aus dem Feld aus, und klicken Sie auf "Entfernen".

  10. (Optional) Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für die Verbindungsressource an.

    Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Konnektor-Verbindungspool verwendet wird.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

  11. Klicken Sie auf "OK".
