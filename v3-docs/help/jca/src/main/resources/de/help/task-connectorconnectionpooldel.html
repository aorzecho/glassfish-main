So löschen Sie einen Konnektor-Verbindungspool

Wenn Sie einen Konnektor-Verbindungspool löschen, werden alle von diesem Konnektor-Verbindungspools abhängigen Verbindungsressourcen ebenfalls gelöscht.

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Konnektor-Verbindungspools" aus.

    Die Seite "Konnektor-Verbindungspool" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Konnektor-Verbindungspool" das Kontrollkästchen neben dem Namen des Verbindungspools, den Sie löschen möchten.
  5. Klicken Sie auf "Löschen".
