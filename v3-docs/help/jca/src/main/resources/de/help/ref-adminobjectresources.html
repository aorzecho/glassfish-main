Admin-Objektressourcen

Auf der Seite "Admin-Objektressourcen" können Sie verwaltete Objektressourcen konfigurieren.

Eine verwaltete Objektressource bietet spezielle Funktionen, die vom Ressourcenadapter für das bereitgestellte Verbindungsmodul definiert werden.

Für jede verwaltete Objektressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die verwaltete Objektressource identifiziert.

Enabled (Aktiviert)

"True", wenn die verwaltete Objektressource aktiviert ist, bzw. "false", wenn sie deaktiviert ist.

Resource Type (Ressourcen-Typ)

Der Java-Typ der verwalteten Objektressource.

Description (Beschreibung)

Eine Beschreibung der verwalteten Objektressource.

Die Tabelle "Ressourcen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen verwalteten Objektressource.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter verwalteter Objektressourcen.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter verwalteter Objektressourcen.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter verwalteter Objektressourcen.

Verwandte Aufgaben

  • So erstellen Sie eine verwaltete Objektressource

  • So bearbeiten Sie eine Admin-Objektressource

  • So löschen Sie eine verwaltete Objektressource

Verwandte asadmin-Befehle

  • create-admin-object(1)

  • list-admin-objects(1)

  • delete-admin-object(1)
