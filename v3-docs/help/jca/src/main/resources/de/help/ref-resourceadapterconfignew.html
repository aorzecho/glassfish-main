Neue Ressourcenadapterkonfiguration

Auf der Seite "Neue Ressourcenadapter-Konfiguration" können Sie eine neue Ressourcenadapter-Konfiguration erstellen.

Die Seite "Neue Ressourcenadapter-Konfiguration" enthält die folgenden Optionen.

Ressourcenadaptername

Der Name des Ressourcenadapters, den Sie konfigurieren.

Thread Pool ID (Threadpool-ID)

Der für die Ressourcenadapter-Konfiguration zu verwendende Threadpool.

Weitere Eigenschaften

Weitere Eigenschaften für den Ressourcenadapter. Verfügbare Eigenschaften sind die Namen von Einrichtfunktionsmethoden der Klassen, die vom Element resourceadapter-class in der Datei ra.xml referenziert werden; dieses Element definiert den Klassennamen der JavaBeans-Komponente des Ressourcenadapters. Hier definierte Eigenschaftenänderungen überschreiben die in der Datei ra.xml vorhandenen Standardwerte.

Verwandte Aufgaben

  • So wird eine Ressourcenadapterkonfiguration erstellt

  • So wird eine Ressourcenadapterkonfiguration bearbeitet

  • So wird eine Ressourcenadapterkonfiguration gelöscht
