Arbeitssicherheitszuordnungen

Auf der Seite "Arbeitssicherheitszuordnungen" können Sie Arbeitssicherheitszuordnungen für Ressourcenadapter konfigurieren.

Eine Konnektor-Arbeitssicherheitszuordnung ordnet die Identität des aufrufenden Programms der von dem Ressourcenadapter-EIS-Principal bzw. der -EIS-Benutzergruppe übergebenen Arbeit einem geeigneten Principal bzw. einer geeigneten Benutzergruppe in der GlassFish-Server-Sicherheitsdomäne zu. Mit einem Ressourcenadapter können eine oder mehrere Arbeitssicherheitszuordnungen verknüpft werden. In der Konfiguration einer Konnektor-Arbeitssicherheitszuordnung kann das Platzhalterzeichen * (Stern) verwendet werden, um alle Benutzer oder Benutzergruppen anzugeben.

Eine Arbeitssicherheitszuordnung ist nicht dasselbe wie eine Sicherheitszuordnung für einen Konnektor-Verbindungspool; diese ordnet den während einer Servlet- oder EJB-Authentifizierung empfangenen Principal den vom EIS akzeptierten Creditials zu. Weitere Informationen über Sicherheitszuordnungen für Konnektor-Verbindungspools finden Sie unter Sicherheitszuordnungen für Konnektor-Verbindungspools.

Als Enterprise Information System (EIS) wird jees System bezeichnet, das die Daten einer Organisation enthält. Dabei kann es sich um ein Mainframe-, Messaging- oder Datenbanksystem bzw. um eine Anwendung handeln.

Für jede Arbeitssicherheitszuordnung werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Arbeitssicherheitszuordnung.

Die Tabelle "Arbeitssicherheitszuordnungen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Arbeitssicherheitszuordnung.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Arbeitssicherheitszuordnungen.

Verwandte Aufgaben

  • So erstellen Sie eine Arbeitssicherheitszuordnung

  • So bearbeiten Sie eine Arbeitssicherheitszuordnung

  • So löschen Sie eine Arbeitssicherheitszuordnung
