So bearbeiten Sie eine Admin-Objektressource

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Admin-Objektressourcen" aus.

    Die Seite "Admin-Objektressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "Admin-Objektressourcen" auf den Namen der zu bearbeitenden verwalteten Objektressource.

    Die Seite "Admin-Objektressource bearbeiten" wird geöffnet.

  5. Wählen Sie in der Dropdown-Liste "Ressourcenadapter" den Ressourcenadapter für die verwaltete Objektressource aus.
  6. Wählen Sie in der Dropdown-Liste "Ressourcentyp" den Java-Typ für die verwaltete Objektressource aus.
  7. Wählen Sie in der Dropdown-Liste "Klassenname" den mit dem Ressourcentyp verknüpften Implementierungsklassennamen aus.
  8. Geben Sie im Feld "Beschreibung" eine Beschreibung der verwalteten Objektressource ein.
  9. Aktivieren Sie das Kontrollkästchen "Status", um die verwaltete Objektressource zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  10. (Optional) Geben Sie in der Tabelle "Weitere Eigenschaften" zusätzliche Eigenschaften für die verwaltete Objektressource an.

    Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Konnektor-Verbindungspool verwendet wird.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.
    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.
    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".
  11. Klicken Sie auf der Registerkarte "Ziel" auf "Ziele verwalten", um die Ziele zu bearbeiten, auf denen das verwaltete Objekt bereitgestellt wird.

    Kehren Sie zur Seite "Verwaltete Objektressource bearbeiten" zurück, indem Sie auf die Registerkarte "Allgemein" klicken.

  12. Klicken Sie auf "Speichern".

    Tipp - Um zu den Standardeinstellungen zurückzukehren, klicken Sie auf die Schaltfläche "Standardwerte laden" und anschließend auf "Speichern".
