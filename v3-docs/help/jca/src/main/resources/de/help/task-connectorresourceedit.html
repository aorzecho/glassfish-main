So wird eine Connector-Ressource bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie den Knoten "Connector" ein.

  3. Wählen Sie unter dem Knoten "Connector" den Knoten "Connector-Ressourcen".

    Die Seite "Verbindungsressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "Verbindungsressourcen" auf den Namen der zu bearbeitenden Ressource.

    Die Seite "Verbindungsressource bearbeiten" wird geöffnet.

  5. Wählen Sie in der Dropdown-Liste "Poolname" den Connector Connection Pool, dem die Connector-Ressource angehört.

  6. Geben Sie im Feld "Beschreibung" eine Beschreibung der Verbindungsressource ein.

  7. Wählen Sie auf der Registerkarte "Ziel" die benötigten Ziele aus, und klicken Sie auf "Aktivieren" oder "Deaktivieren".

    Um Ziele zur Liste hinzuzufügen oder aus der Liste zu entfernen, klicken Sie auf "Ziele verwalten".

    Kehren Sie zur Seite "Verbindungsressource bearbeiten" zurück, indem Sie auf die Registerkarte "Allgemein" klicken.

  8. Aktivieren Sie das Kontrollkästchen "Status", um die Verbindungsressource zur Laufzeit zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  9. Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für die Verbindungsressource an.

    Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection Pool verwendet wird.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".

  10. Klicken Sie auf "Speichern".

    Tipp:

    Um zu den Standardeinstellungen zurückzukehren, klicken Sie auf die Schaltfläche "Standardwerte laden" und anschließend auf "Speichern".
