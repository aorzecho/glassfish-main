So wird eine verwaltete Objektressource erstellt

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie den Knoten "Connector" ein.

  3. Wählen Sie unter dem Knoten "Connector" den Knoten "Admin-Objektressourcen".

    Die Seite "Admin-Objektressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "Admin-Objektressourcen" auf die Schaltfläche "Neu".

    Die Seite "Neue Admin-Objektressource" wird geöffnet.

  5. Geben Sie im Feld "JNDI-Name" einen eindeutigen Namen zur Identifizierung der verwalteten Objektressource ein.

  6. Wählen Sie in der Dropdown-Liste "Ressourcenadapter" den Ressourcenadapter für die verwaltete Objektressource aus.

  7. Wählen Sie in der Dropdown-Liste "Ressourcentyp" den Java-Typ für die verwaltete Objektressource aus.

  8. Wählen Sie in der Dropdown-Liste "Klassenname" den mit dem Ressourcentyp verknüpften Implementierungsklassennamen aus.

  9. Geben Sie im Feld "Beschreibung" eine Beschreibung der verwalteten Objektressource ein.

  10. Aktivieren Sie das Kontrollkästchen "Status", um die verwaltete Objektressource zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  11. Geben Sie in der Tabelle "Weitere Eigenschaften" zusätzliche Eigenschaften für die verwaltete Objektressource an.

    Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection Pool verwendet wird.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

  12. Wählen Sie im Bereich "Verfügbare Ziele" die Domains, Cluster oder Serverinstanzen, in die das verwaltete Objekt gesetzt werden soll, und klicken Sie auf "Hinzufügen".

    Um eine der Domains, eines der Cluster oder eine der Serverinstanzen zu entfernen, die im Feld "Ausgewählt" aufgelistet sind, wählen Sie die betreffende Domain, das Cluster oder die Serverinstanz aus dem Feld und klicken auf "Entfernen".

  13. Klicken Sie auf "OK".
