Neuer Konnektorverbindungspool (Schritt 1 von 2)

Sie verwenden die Seite "Neuer Konnektorverbindungspool (Schritt 1 von 2)" zum Erstellen eines neuen Konnektor-Verbindungspools.

Die Seite "Neuer Konnektorverbindungspool (Schritt 1 von 2)" enthält die folgenden Optionen.

Name

Ein eindeutiger Name, der den Konnektor-Verbindungspool identifiziert.

Ressourcenadapter

Der mit dem Konnektor-Verbindungspool verknüpfte Ressourcenadapter.

Verbindungsdefinition

Die Verbindungsdefinition für den Konnektor-Verbindungspool. Die Auswahloptionen in der Liste richten sich nach dem gewählten Ressourcenadapter. Die Verbindungsdefinition gibt das Element connection-definition eines Ressourcenadapters in der Datei ra.xml an.

  • So bearbeiten Sie die Eigenschaften eines Konnektor-Verbindungspools

  • So erstellen Sie eine Sicherheitzuordnung für einen Konnektor-Verbindungspool

  • So bearbeiten Sie eine Sicherheitzuordnung für einen Konnektor-Verbindungspool

  • So löschen Sie eine Sicherheitzuordnung für einen Konnektor-Verbindungspool

  • So löschen Sie einen Konnektor-Verbindungspool
