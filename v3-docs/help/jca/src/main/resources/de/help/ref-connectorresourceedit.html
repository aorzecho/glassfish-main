Connector-Ressource bearbeiten

Auf der Seite "Verbindungsressource bearbeiten" können Sie die Einstellungen für eine Verbindungsressource ändern.

Die Seite "Verbindungsressource bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die Verbindungsressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue Verbindungsressource erstellen.

Pool Name (Poolname)

Der mit dieser Ressource verknüpfte Connector Connection Pool.

Description (Beschreibung)

Eine Beschreibung der Verbindungsressource.

Status

Bei aktivierter Option ist die Verbindungsressource aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die Connector-Ressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection Pool verwendet wird. Mit den in dieser Tabelle angegebenen Name/Werte-Paaren können die vom Ressourcenadapterprovider definierten Standardwerte für die Eigenschaften überschrieben werden.

Verwandte Aufgaben

  • So wird eine Connector-Ressource erstellt

  • So wird eine Connector-Ressource bearbeitet

  • So wird eine Connector-Ressource gelöscht
