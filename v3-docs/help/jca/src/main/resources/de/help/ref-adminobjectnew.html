Neue Admin-Objektressource

Auf der Seite "Neue Admin-Objektressource" können Sie eine neue verwaltete Objektressource erstellen.

Die Seite "Neue Admin-Objektressource" enthält die folgenden Optionen.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die verwaltete Objektressource identifiziert.

Ressourcenadapter

Der Ressourcenadapter für die verwaltete Objektressource.

Resource Type (Ressourcen-Typ)

Der Java-Typ für die verwaltete Objektressource (z. B. javax.jms.Topic).

Klassenname

Der mit dem Ressourcentyp verknüpfte Implementierungsklassenname.

Beschreibung

Eine Beschreibung der verwalteten Objektressource.

Status

Bei aktivierter Option ist die verwaltete Objektressource aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die verwaltete Objektressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Konnektor-Verbindungspool verwendet wird. Mit den in dieser Tabelle angegebenen Namen/Werte-Paaren können die vom Ressourcenadapteranbieter definierten Standardwerte für die Eigenschaften überschrieben werden.

Ziele

Cluster und eigenständige Instanzen für die verwaltete Objektressource. Verschieben Sie die benötigten Ziele in die Spalte "Ausgewählte Ziele", indem Sie die Schaltflächen "Hinzufügen" und "Alle hinzufügen" verwenden. Verschieben Sie die nicht benötigten Ziele in die Spalte "Verfügbare Ziele", indem Sie die Schaltflächen "Entfernen" und "Alle entfernen" verwenden.

Diese Option wird nur dann angezeigt, wenn in der Domäne Cluster oder eigenständige Instanzen erstellt wurden.
