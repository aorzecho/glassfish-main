So richten Sie den EIS-Zugriff ein

  1. Stellen Sie einen Konnektor bereit (installieren Sie ihn). Siehe To Deploy a Connector Module.
  2. Erstellen Sie einen Verbindungspool für den Konnektor. Siehe So erstellen Sie einen Konnektor-Verbindungspool.
  3. Erstellen Sie eine Verbindungsressource, die mit dem Verbindungspool verknüpft ist. Siehe So erstellen Sie eine Verbindungsressource.
