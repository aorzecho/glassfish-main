Eigenschaften des Konnektor-Verbindungspools bearbeiten

Auf der Seite "Eigenschaften des Konnektor-Verbindungspools bearbeiten" können Sie die Eigenschaften eines Konnektor-Verbindungspools ändern.

Die Seite "Eigenschaften des Konnektor-Verbindungspools bearbeiten" enthält die folgenden Optionen.

Weitere Eigenschaften

Weitere Eigenschaften für den Konnektor-Verbindungspool. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Konnektor-Verbindungspool verwendet wird. Mit den in dieser Tabelle angegebenen Namen/Werte-Paaren können die vom Ressourcenadapteranbieter definierten Standardwerte für die Eigenschaften überschrieben werden.

Wenn Sie für eine Eigenschaft keinen Wert angeben, wird die Eigenschaft nach Abschluss der Bearbeitung des Verbindungspools aus der Tabelle entfernt. Falls Sie zum jetzigen Zeitpunkt keinen Wert für eine Eigenschaft festlegen möchten, die Eigenschaft jedoch in der Tabelle verbleiben soll, damit Sie den Wert später angeben können, geben Sie ein leeres Klammernpaar in das Wertefeld ein:
