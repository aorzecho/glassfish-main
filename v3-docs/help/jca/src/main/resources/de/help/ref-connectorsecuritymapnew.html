Neue Sicherheitszuordnung für Konnektor-Verbindungspools

Auf der Seite "Neue Sicherheitszuordnung für Konnektor-Verbindungspool" können Sie eine neue Sicherheitszuordnung für einen Konnektor-Verbindungspool erstellen.

Die Seite "Neue Sicherheitszuordnung für Konnektor-Verbindungspool" enthält die folgenden Optionen.

Verbindungspoolname

Der Name des Konnektor-Verbindungspools, zu dem diese Sicherheitszuordnung gehört. Der Poolname ist ein schreibgeschütztes Feld.

Sicherheitszuordnungsname

Der Name der Sicherheitszuordnung.

Benutzergruppen

Wenn diese Optionsschaltfläche aktiviert ist, muss das Textfeld die Namen der Benutzergruppen, mit Kommas getrennt, enthalten.

Principals

Wenn diese Optionsschaltfläche aktiviert ist, muss das Textfeld die Namen der Principals, mit Kommas getrennt, enthalten.

Benutzername

Der vom EIS benötigte Benutzername des Backend-Principals.

Passwort

Das vom EIS benötigte Passwort des Backend-Principals.
