So wird eine Ressourcenadapterkonfiguration bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "Ressourcenadapter-Konfigurationen" aus.

    Die Seite "Ressourcenadapter-Konfigurationen" wird geöffnet.

  3. Klicken Sie auf der Seite "Ressourcenadapter-Konfigurationen" auf den Namen der zu bearbeitenden Ressourcenadapter-Konfiguration.

    Die Seite "Ressourcenadapter-Konfiguration bearbeiten" wird geöffnet.

  4. Wählen Sie in der Dropdown-Liste "Threadpool-ID" den Threadpool aus, der für die Ressourcenadapter-Konfiguration verwendet werden soll.

  5. Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für den Ressourcenadapter an.

    Verfügbare Eigenschaften sind die Namen von Einrichtfunktionsmethoden der Klassen, die vom Element resourceadapter-class in der Datei ra.xml referenziert werden; dieses Element definiert den Klassennamen der JavaBeans-Komponente des Ressourcenadapters. Hier definierte Eigenschaftenänderungen überschreiben die in der Datei ra.xml vorhandenen Standardwerte.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen".

  6. Klicken Sie auf "Speichern".
