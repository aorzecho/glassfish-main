Neue Connector-Ressource

Auf der Seite "Neue Connector-Ressource" können Sie eine neue Connector-Ressource erstellen.

Die Seite "Neue Connector-Ressource" enthält die folgenden Optionen.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die Verbindungsressource identifiziert.

Pool Name (Poolname)

Der Connection Pool, dem die neue Verbindungsressource angehört.

Description (Beschreibung)

Eine Beschreibung der Verbindungsressource.

Status

Bei aktivierter Option ist die Verbindungsressource aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die Connector-Ressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection Pool verwendet wird. Mit den in dieser Tabelle angegebenen Name/Werte-Paaren können die vom Ressourcenadapterprovider definierten Standardwerte für die Eigenschaften überschrieben werden.

Ziele

Cluster und eigenständige Instanzen für die Verbindungsressource. Mithilfe der Schaltflächen "Hinzufügen" und "Alle hinzufügen" verschieben Sie die benötigten Ziele in die Spalte "Ausgewählte Ziele". Verschieben Sie die nicht benötigten Ziele in die Spalte "Verfügbare Ziele" mithilfe der Schaltflächen "Entfernen" und "Alle entfernen".

Diese Option wird nur dann angezeigt, wenn in der Domain Cluster oder Standalone-Instanzen erstellt wurden.

Verwandte Aufgaben

  • So wird eine Connector-Ressource erstellt

  • So wird eine Connector-Ressource bearbeitet

  • So wird eine Connector-Ressource gelöscht
