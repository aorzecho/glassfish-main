Neue Konnektorressource

Auf der Seite "Neue Konnektorressource" können Sie eine neue Verbindungsressource erstellen.

Die Seite "Neue Konnektorressource" enthält die folgenden Optionen.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die Verbindungsressource identifiziert.

Pool Name (Poolname)

Der Verbindungspool, dem die neue Verbindungsressource angehört.

Description (Beschreibung)

Eine Beschreibung der Verbindungsressource.

Status

Bei aktivierter Option ist die Verbindungsressource aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die Verbindungsressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Konnektor-Verbindungspool verwendet wird. Mit den in dieser Tabelle angegebenen Namen/Werte-Paaren können die vom Ressourcenadapteranbieter definierten Standardwerte für die Eigenschaften überschrieben werden.

Ziele

Cluster und eigenständige Instanzen für die Verbindungsressource. Verschieben Sie die benötigten Ziele in die Spalte "Ausgewählte Ziele", indem Sie die Schaltflächen "Hinzufügen" und "Alle hinzufügen" verwenden. Verschieben Sie die nicht benötigten Ziele in die Spalte "Verfügbare Ziele", indem Sie die Schaltflächen "Entfernen" und "Alle entfernen" verwenden.

Diese Option wird nur dann angezeigt, wenn in der Domäne Cluster oder eigenständige Instanzen erstellt wurden.
