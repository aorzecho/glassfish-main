Connector Connection Pool-Sicherheitszuordnung bearbeiten

Auf der Seite "Sicherheitszuordnung für Connector Connection Pools bearbeiten" können Sie eine Sicherheitszuordnung für einen Connector Connection Pool bearbeiten.

Die Seite "Sicherheitszuordnung für Connector Connection Pool bearbeiten" enthält die folgenden Optionen.

Connection Pool-Name

Der Name des Connector Connection Pools, zu dem diese Sicherheitszuordnung gehört. Der Poolname ist ein schreibgeschütztes Feld.

Sicherheitszuordnungsname

Der Name der Sicherheitszuordnung. Der Name ist ein schreibgeschütztes Feld. Sie können nur dann einen Sicherheitszuordnungsnamen festlegen, wenn Sie eine neue Sicherheitszuordnung erstellen.

Sicherheitszuordnungsname

Der Name der Sicherheitszuordnung.

Benutzergruppen

Wenn diese Optionsschaltfläche aktiviert ist, muss das Textfeld die Namen der Benutzergruppen, mit Kommas getrennt, enthalten.

Principals

Wenn diese Optionsschaltfläche aktiviert ist, muss das Textfeld die Namen der Principals, mit Kommas getrennt, enthalten.

Benutzername

Der vom EIS benötigte Benutzername des Backend-Principals.

Kennwort

Das vom EIS benötigte Kennwort des Backend-Principals.
