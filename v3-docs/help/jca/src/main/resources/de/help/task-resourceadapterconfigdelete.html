So löschen Sie eine Ressourcenadapter-Konfiguration

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Wählen Sie unter dem Ressourcenknoten den Knoten "Ressourcenadapter-Konfigurationen" aus.

    Die Seite "Ressourcenadapter-Konfigurationen" wird geöffnet.

  3. Aktivieren Sie auf der Seite "Ressourcenadapter-Konfigurationen" das Kontrollkästchen neben dem Namen der zu löschenden Ressourcenadapter-Konfiguration.
  4. Klicken Sie auf "Löschen".
