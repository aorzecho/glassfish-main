So löschen Sie eine Verbingsressource

Eine Verbindungsressource wird automatisch gelöscht, wenn Sie den mit der Ressource verknüpften Konnektor-Verbindungspool löschen.

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Verbindungsressourcen" aus.

    Die Seite "Verbindungsressourcen" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Verbindungsressourcen" das Kontrollkästchen neben dem Namen der zu löschenden Ressource.
  5. Klicken Sie auf "Löschen".
