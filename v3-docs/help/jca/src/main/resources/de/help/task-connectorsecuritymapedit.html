So bearbeiten Sie eine Sicherheitzuordnung für einen Konnektor-Verbindungspool

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Konnektor-Verbindungspools" aus.

    Die Seite "Konnektor-Verbindungspool" wird geöffnet.

  4. Klicken Sie auf der Seite "Konnektor-Verbindungspools" auf den Namen des zu bearbeitenden Verbindungspools.

    Die Seite "Konnektor-Verbindungspool bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Konnektor-Verbindungspool bearbeiten" auf die Registerkarte "Sicherheitszuordnungen".

    Die Seite "Sicherheitszuordnungen für Konnektor-Verbindungspools" wird geöffnet.

  6. Klicken Sie auf der Seite "Sicherheitszuordnungen für Konnektor-Verbindungspools" auf den Namen der zu bearbeitenden Sicherheitszuordnung.

    Die Seite "Sicherheitszuordnung für Konnektor-Verbindungspool bearbeiten" wird geöffnet.

  7. Führen Sie einen der folgenden Schritte aus, nicht jedoch beide:
    • Aktivieren Sie die Optionsschaltfläche "Benutzergruppen", und geben Sie die Namen der Benutzergruppen, mit Kommas getrennt, ein.
    • Aktivieren Sie die Optionsschaltfläche "Principals", und geben Sie die Namen der Principals, mit Kommas getrennt, ein.
  8. Geben Sie im Feld "Benutzername" den vom EIS benötigten Benutzernamen des Backend-Principals ein.
  9. Geben Sie im Feld "Passwort" das vom EIS benötigte Passwort des Backend-Principals ein.
  10. Klicken Sie auf "Speichern".
