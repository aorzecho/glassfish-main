So löschen Sie eine Sicherheitzuordnung für einen Konnektor-Verbindungspool

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Konnektor-Verbindungspools" aus.

    Die Seite "Konnektor-Verbindungspool" wird geöffnet.

  4. Klicken Sie auf der Seite "Konnektor-Verbindungspools" auf den Namen des Verbindungspools, der die zu löschende Sicherheitszuordnung enthält.

    Die Seite "Konnektor-Verbindungspool bearbeiten" wird geöffnet.

  5. Klicken Sie auf die Registerkarte "Sicherheitszuordnungen".

    Die Seite "Sicherheitszuordnungen für Konnektor-Verbindungspools" wird geöffnet.

  6. Aktivieren Sie auf der Seite "Sicherheitszuordnungen für Konnektor-Verbindungspools" das Kontrollkästchen neben dem Namen der zu löschenden Sicherheitszuordnung.
  7. Klicken Sie auf "Löschen".
