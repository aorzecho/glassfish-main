Connector-Ressourcen

Auf der Seite "Verbindungsressourcen" können Sie Verbindungsressourcen konfigurieren.

Eine Verbindungsressource ist ein Programmobjekt, das eine Anwendung mit einer Verbindung zu einem EIS versorgt.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI Name (JNDI-Name)

Ein eindeutiger Name, der die Verbindungsressource identifiziert.

Enabled (Aktiviert)

"True", wenn die Verbingungsressource aktiviert ist, bzw. "false", wenn sie deaktiviert ist.

Connection Pool

Der mit der Verbindungsressource verknüpfte Connector Connection Pool.

Beschreibung

Eine Beschreibung der Verbindungsressource.

Die Tabelle "Ressourcen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Verbindungsressource.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Verbindungsressourcen.

Enable (Aktivieren)

Schaltfläche zum Aktivieren einer oder mehrerer ausgewählter Verbindungsressourcen.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren einer oder mehrerer ausgewählter Verbindungsressourcen.

Verwandte Aufgaben

  • So wird eine Connector-Ressource erstellt

  • So wird eine Connector-Ressource bearbeitet

  • So wird eine Connector-Ressource gelöscht

Zugehörige asadmin-Befehle

  • create-connector-resource(1)

  • list-connector-resources(1)

  • delete-connector-resource(1)
