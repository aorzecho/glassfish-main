Konnektor-Verbindungspools

Auf der Seite "Konnektor-Verbindungspools" können Sie Konnektor-Verbindungspools konfigurieren.

Ein Konnektor-Verbindungspool ist eine Gruppe wiederverwendbarer Verbindungen für ein spezielles EIS.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

Pool Name (Poolname)

Ein eindeutiger Name, der den Konnektor-Verbindungspool identifiziert.

Ressourcenadapter

Der mit dem Konnektor-Verbindungspool verknüpfte Ressourcenadapter.

Verbindungsdefinition

Der Klassenname der Verbindungsdefinition für den Konnektor-Verbindungspool.

Beschreibung

Eine Beschreibung des Konnektor-Verbindungspools.

Die Tabelle "Pools" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Konnektor-Verbindungspools.

Löschen

Schaltfläche zum Löschen eines oder mehrerer ausgewählter Konnektor-Verbindungspools.
