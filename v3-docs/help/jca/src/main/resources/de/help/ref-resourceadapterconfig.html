Ressourcenadapterkonfigurationen

Auf der Seite "Ressourcenadapter-Konfigurationen" können Sie Konfigurationen von Connectorn (Ressourcenadaptern) verwalten.

Die Ressourcenadapter-Konfiguration liefert die Konfigurationsinformationen für den Ressourcenadapter. Mit der Administrationskonsole erstellen Sie eine Ressourcenadapter-Konfiguration, nachdem der Ressourcenadapter bereitgestellt wurde. In diesem Fall wird der Ressourcenadapter mit der neuen Konfiguration neu gestartet. Um eine Ressourcenadapter-Konfiguration vor der Bereitstellung des Ressourcenadapters zu erstellen, verwenden Sie den Befehl asadmin create-resource-adapter-config .

Bevor Sie eine Ressourcenadapter-Konfiguration erstellen können, müssen Sie einen Threadpool erstellen, den der Ressourcenadapter verwenden soll.

Für jede Ressourcenadapter-Konfiguration werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des Ressourcenadapters.

Die Tabelle "Konfigurationen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Ressourcenadapter-Konfiguration.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Ressourcenadapter-Konfigurationen.

Verwandte Aufgaben

  • So wird eine Ressourcenadapterkonfiguration erstellt

  • So wird eine Ressourcenadapterkonfiguration bearbeitet

  • So wird eine Ressourcenadapterkonfiguration gelöscht

Zugehörige asadmin-Befehle

  • create-resource-adapter-config(1)

  • list-resource-adapter-configs(1)

  • delete-resource-adapter-config(1)
