Sicherheitszuordnungen für Konnektor-Verbindungspools

Auf der Seite "Sicherheitszuordnungen für Konnektor-Verbindungspools" können Sie Sicherheitszuordnungen für Konnektor-Verbindungspools verwalten.

Mithilfe von Sicherheitszuordnungen für Konnektor-Verbindungspools können Sie Zuordnungen zwischen Java EE-Benutzern und -Gruppen sowie EIS-Benutzern und -Gruppen herstellen. Eine Sicherheitszuordnung ordnet den während einer Servlet- oder EJB-Authentifizierung empfangenen Principal den vom EIS akzeptierten Creditials zu. Sie können mehrere GlassFish-Server-Principals demselben Backend-Principal zuordnen.

Eine Sicherheitszuordnung für einen Verbindungspool ist nicht dasselbe wie eine Konnektor-Arbeitssicherheitszuordnung; diese ordnet einen mit einer eingehenden Arbeitsinstanz verknüpften Principal einem Principal in einem Sicherheitsbereich der GlassFish-Server-Domäne zu. Weitere Informationen zu Konnektor-Arbeitssicherheitszuordnungen finden Sie unter Arbeitssicherheitszuordnungen.

Für jede Sicherheitszuordnung werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Sicherheitszuordnung.

Die Tabelle "Sicherheitszuordnungen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Sicherheitszuordnung.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Sicherheitszuordnungen.
