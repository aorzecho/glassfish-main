Connector Connection Pool-Sicherheitszuordnungen

Auf der Seite "Sicherheitszuordnungen für Connector Connection Pools" können Sie Sicherheitszuordnungen für Connector Connection Pools verwalten.

Mit Sicherheitszuordnungen für Connector Connection Pools können Sie Zuordnungen zwischen Java EE-Benutzern und -Gruppen sowie EIS-Benutzern und -Gruppen herstellen. Eine Sicherheitszuordnung ordnet den während einer Servlet- oder EJB-Authentifizierung empfangenen Principal den vom EIS akzeptierten Zugangsdaten zu. Sie können mehrere GlassFish Server-Principals demselben Backend-Principal zuordnen.

Eine Sicherheitszuordnung für einen Connection Pool ist nicht dasselbe wie eine Connector-Arbeitssicherheitszuordnung; diese ordnet einen mit einer eingehenden Arbeitsinstanz verknüpften Principal einem Principal in einer Sicherheits-Realm der GlassFish Server-Domain zu. Weitere Informationen zu Connector-Arbeitssicherheitszuordnungen finden Sie unter Arbeitssicherheitszuordnungen.

Für jede Sicherheitszuordnung werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Sicherheitszuordnung.

Die Tabelle "Sicherheitszuordnungen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Sicherheitszuordnung.

Löschen

Schaltfläche zum Löschen einer oder mehrerer ausgewählter Sicherheitszuordnungen.
