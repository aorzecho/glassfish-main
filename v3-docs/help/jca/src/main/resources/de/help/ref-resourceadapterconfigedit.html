Ressourcenadapter-Konfiguration bearbeiten

Auf der Seite "Ressourcenadapter-Konfiguration bearbeiten" können Sie die Einstellungen einer Ressourcenadapter-Konfiguration ändern.

Die Seite "Ressourcenadapter-Konfiguration bearbeiten" enthält die folgenden Optionen.

Ressourcenadaptername

Der Name des Ressourcenadapters, den Sie konfigurieren. Der Name ist ein schreibgeschütztes Feld. Sie können nur dann einen Ressourcenadapternamen festlegen, wenn Sie eine neue Ressourcenadapter-Konfiguration erstellen.

Threadpool-ID

Der für die Ressourcenadapter-Konfiguration zu verwendende Threadpool.

Weitere Eigenschaften

Weitere Eigenschaften für den Ressourcenadapter. Verfügbare Eigenschaften sind die Namen von Einrichtfunktionsmethoden der Klassen, auf die vom Element resourceadapter-class? in der Datei ra.xml verwiesen wird; dieses Element definiert den Klassennamen der JavaBeans-Komponente des Ressourcenadapters. Hier definierte Eigenschaftenänderungen überschreiben die in der Datei ra.xml vorhandenen Standardwerte.

Verwandte Aufgaben

  • So bearbeiten Sie eine Ressourcenadapter-Konfiguration

  • So löschen Sie eine Ressourcenadapter-Konfiguration
