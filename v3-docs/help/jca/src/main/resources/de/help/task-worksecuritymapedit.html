So bearbeiten Sie eine Arbeitssicherheitszuordnung

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie den Knoten "Konnektoren".
  3. Wählen Sie unter dem Knoten "Konnektoren" den Knoten "Arbeitssicherheitszuordnungen" aus.

    Die Seite "Arbeitssicherheitszuordnungen" wird geöffnet.

  4. Klicken Sie auf der Seite "Arbeitssicherheitszuordnungen" auf den Namen der zu bearbeitenden Arbeitssicherheitszuordnung.

    Die Seite "Arbeitssicherheitszuordnung bearbeiten" wird geöffnet.

  5. Verwenden Sie das Feld "Gruppe" oder "Principal" wie folgt.
    • Wenn Sie die Optionsschaltfläche "Gruppe" aktivieren, geben Sie eine Zuordnung jeder Backend-EIS-Benutzergruppe zur entsprechenden, in der GlassFish-Server-Sicherheitsdomäne definierten Benutzergruppe ein.

      Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • Wenn Sie die Optionsschaltfläche "Principal" aktivieren, geben Sie eine Zuordnung jedes Backend-EIS-Principals zum entsprechenden, in der GlassFish-Server-Sicherheitsdomäne definierten Principal ein.

      Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  6. Wählen Sie in der Dropdown-Liste "Ressourcenadapter" den Ressourcenadapter für das EIS aus.
  7. Geben Sie im Feld "Beschreibung" eine Beschreibung der Arbeitssicherheitszuordnung ein.
  8. Aktivieren Sie das Kontrollkästchen "Status", um die Arbeitssicherheitszuordnung zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  9. Klicken Sie auf "OK".
