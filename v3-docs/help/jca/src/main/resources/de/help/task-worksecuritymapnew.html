So wird eine Arbeitssicherheitszuordnung erstellt

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie den Knoten "Connector" ein.

  3. Wählen Sie unter dem Knoten "Connector" den Knoten "Arbeitssicherheitszuordnungen" aus.

    Die Seite "Arbeitssicherheitszuordnungen" wird geöffnet.

  4. Klicken Sie auf der Seite "Arbeitssicherheitszuordnungen" auf "Neu".

    Die Seite "Neue Arbeitssicherheitszuordnung" wird geöffnet.

  5. Geben Sie im Feld "Sicherheitszuordnungsname" einen Namen für die Arbeitssicherheitszuordnung ein.

  6. Verwenden Sie das Feld "Gruppe" oder "Principal" wie folgt.

    • Wenn Sie die Optionsschaltfläche "Gruppe" aktivieren, geben Sie eine Zuordnung jeder Backend-EIS-Benutzergruppe zur entsprechenden, in der GlassFish Server-Sicherheitsdomain definierten Benutzergruppe ein.

      Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Wenn Sie die Optionsschaltfläche "Principal" aktivieren, geben Sie eine Zuordnung jedes Backend-EIS-Principals zum entsprechenden, in der GlassFish Server-Sicherheitsdomain definierten Principal ein.

      Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. Wählen Sie in der Dropdown-Liste "Ressourcenadapter" den Ressourcenadapter für das EIS aus.

  8. Geben Sie im Feld "Beschreibung" eine Beschreibung der Arbeitssicherheitszuordnung ein.

  9. Aktivieren Sie das Kontrollkästchen "Status", um die Arbeitssicherheitszuordnung zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  10. Klicken Sie auf OK.
