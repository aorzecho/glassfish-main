Connector-Service

Auf der Seite "Verbindungsservice" können Sie allgemeine Einstellungen für den Verbindungsservice ändern, mit dem Ressourcenadapter gesteuert werden.

Die Seite "Verbindungsservice" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für Einstellungen, für die es Standardwerte gibt. Einstellungen, die nicht über Standardwerte verfügen, können nicht geändert werden.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Timeout beim Herunterfahren

Die maximale Anzahl Sekunden, die während des Herunterfahrens von GlassFish Server auf die Ausführung der ResourceAdapter.stop-Methode einer Verbindungsmodulinstanz gewartet wird. Ressourcenadapter, deren Herunterfahren länger dauert, werden ignoriert, und das Herunterfahren von GlassFish Server wird fortgesetzt. Der Standardwert beträgt 30 Sekunden.

Policy zum Laden von Connector-Klassen

Die beim Laden von Klassen anzuwendende Policy. Zur Auswahl stehen:

derived

Gibt an, dass die Ressourcenadapter entsprechend den Verweisen einer Anwendung auf beliebige Ressourcen des Ressourcenadapters bereitgestellt werden. Diese Policy ist die Standardeinstellung.

global

Gibt an, dass alle Ressourcenadapter für alle Anwendungen sichtbar sind.

Diese Policy gilt nicht für den vorinstallierten JMS-Systemressourcenadapter, jmsra, der immer für alle Anwendungen sichtbar ist.
