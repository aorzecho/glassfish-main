Verbindungsdienst

Auf der Seite "Verbindungsdienst" können Sie allgemeine Einstellungen für den Verbindungsdienst ändern, mit dem Ressourcenadapter gesteuert werden.

Die Seite "Verbindungsdienst" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zum Wiederherstellen der Standardwerte für alle Einstellungen.

Konfigurationsname

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite gelten. Dieses Feld ist schreibgeschützt.

Zeitüberschreitung beim Herunterfahren

Die maximale Anzahl Sekunden, die während des Herunterfahrens von GlassFish-Server auf die Ausführung der ResourceAdapter.stop-Methode einer Verbindungsmodulinstanz gewartet wird. Ressourcenadapter, deren Herunterfahren länger dauert, werden ignoriert, und das Herunterfahren von GlassFish-Server wird fortgesetzt. Der Standardwert beträgt 30 Sekunden.

Richtlinie zum Laden von Konnektorklassen

Die beim Laden von Klassen anzuwendende Richtlinie. Zur Auswahl stehen:

derived

Gibt an, dass die Ressourcenadapter entsprechend den Verweisen einer Anwendung auf beliebige Ressourcen des Ressourcenadapters bereitgestellt werden. Diese Richtlinie ist die Standardeinstellung.

global

Gibt an, dass alle Ressourcenadapter für alle Anwendungen sichtbar sind.

Diese Richtlinie gilt nicht für den vorinstallierten JMS-Systemressourcenadapter, jmsra, der immer für alle Anwendungen sichtbar ist.
