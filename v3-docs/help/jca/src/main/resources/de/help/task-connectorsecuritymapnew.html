So wird eine Sicherheitszuordnung für einen Connector Connection Pool erstellt

Mit Sicherheitszuordnungen für Connector Connection Pools können Sie Zuordnungen zwischen Java EE-Benutzern und -Gruppen sowie EIS-Benutzern und -Gruppen herstellen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie den Knoten "Connector" ein.

  3. Wählen Sie unter dem Knoten "Connector" den Knoten "Connector Connection Pools".

    Die Seite "Connector Connection Pools" wird geöffnet.

  4. Klicken Sie auf der Seite "Connector Connection Pools" auf den Namen des zu bearbeitenden Connection Pools.

    Die Seite "Connector Connection Pool bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Connector Connection Pool bearbeiten" auf die Registerkarte "Sicherheitszuordnungen".

    Die Seite "Sicherheitszuordnungen für Connector Connection Pool" wird geöffnet.

  6. Klicken Sie auf der Seite "Sicherheitszuordnungen für Connector Connection Pools" auf "Neu".

    Die Seite "Neue Sicherheitszuordnung für Connector Connection Pool" wird geöffnet.

  7. Geben Sie im Feld "Sicherheitszuordnungsname" einen Namen für die Sicherheitszuordnung ein.

  8. Führen Sie einen der folgenden Schritte aus, nicht jedoch beide:

    • Aktivieren Sie die Optionsschaltfläche "Benutzergruppen", und geben Sie die Namen der Benutzergruppen, mit Kommas getrennt, ein.

    • Aktivieren Sie die Optionsschaltfläche "Principals", und geben Sie die Namen der Principals, mit Kommas getrennt, ein.

  9. Geben Sie im Feld "Benutzername" den vom EIS benötigten Benutzernamen des Backend-Principals ein.

  10. Geben Sie im Feld "Kennwort" das vom EIS benötigte Kennwort des Backend-Principals ein.

  11. Klicken Sie auf "OK".

  12. Starten Sie GlassFish Server neu.
