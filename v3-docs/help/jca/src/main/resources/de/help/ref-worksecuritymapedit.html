Arbeitssicherheitszuordnung bearbeiten

Auf der Seite "Arbeitssicherheitszuordnung bearbeiten" können Sie die Einstellungen für eine Arbeitssicherheitszuordnung ändern.

Die Seite "Arbeitssicherheitszuordnung bearbeiten" enthält die folgenden Optionen.

Sicherheitszuordnungsname

Der Name der Arbeitssicherheitszuordnung. Der Name ist ein schreibgeschütztes Feld. Sie können nur dann einen Sicherheitszuordnungsnamen festlegen, wenn Sie eine neue Arbeitssicherheitszuordnung erstellen.

Gruppe

Wenn die Optionsschaltfläche "Gruppe" aktiviert ist, muss im Textfeld eine Zuordnung von jeder Backend-EIS-Benutzergruppe zur entsprechenden, in der GlassFish-Server-Sicherheitsdomäne definierten Benutzergruppe angegeben werden. Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Wenn die Optionsschaltfläche "Principal" aktiviert ist, muss im Textfeld eine Zuordnung von jedem Backend-EIS-Principal zum entsprechenden, in der GlassFish-Server-Sicherheitsdomäne definierten Principal angegeben werden. Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Ressourcenadapter

Der Ressourcenadapter für das EIS.

Description (Beschreibung)

Eine Beschreibung der Arbeitssicherheitszuordnung.

Status

Bei aktivierter Option ist die Arbeitssicherheitszuordnung aktiviert. Diese Option ist standardmäßig aktiviert.
