Editar Mapa de Segurança do Trabalho

Use a página Editar Mapa de Segurança do Trabalho para modificar as definições de um mapa de segurança do trabalho.

A página Editar Mapa de Segurança do Trabalho contém as opções a seguir.

Nome do mapa de segurança

O nome do mapa de segurança do trabalho. O nome é um campo somente para leitura. Você pode apenas especificar um nome de mapa de segurança quando cria um novo mapa de segurança.

Grupo

Se o botão Grupo é selecionado, o campo de texto precisa especificar um mapa de cada grupo de usuários EIS back-end para o correspondente grupo de usuários definido no domínio de segurança do GlassFish Server. Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se o botão Principal é selecionado, o campo de texto precisa especificar um mapa para cada principal EIS back-end para o correspondente principal definido no domínio de segurança do GlassFish Server. Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptador de recurso

O adaptador de recurso para o EIS.

Descrição

Uma descrição para o mapa de segurança do trabalho.

Status

Se esta opção está selecionada, o mapa de segurança do trabalho será ativado. Por default, a opção está ativada.
