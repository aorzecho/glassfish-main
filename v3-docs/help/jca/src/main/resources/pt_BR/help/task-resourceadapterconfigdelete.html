Para Deletar uma Configuração do Adaptador de Recursos

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Sob o nó Recursos, selecione o nó Configs do adaptador de recurso.

    A página Configs do adaptador de recurso é aberta.

  3. Na página Configs do adaptador de recurso, marque a caixa de seleção junto ao nome do config do adaptador de recurso que estiver deletando.

  4. Clique em Deletar.
