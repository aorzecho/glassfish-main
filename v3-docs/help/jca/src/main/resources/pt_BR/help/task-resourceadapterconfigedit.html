Para editar um config do adaptador de recurso

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Sob o nó Recursos, selecione o nó Configs do adaptador de recurso.

    A página Configs do adaptador de recurso é aberta.

  3. Na página Configs do adaptador de recurso, clique no nome do config do adaptador de recurso a ser editado.

    A página Editar configs do adaptador de recurso é aberta.

  4. Na lista suspensa ID de conjunto de segmentos, selecione o conjunto de segmentos a ser usado para a configuração do adaptador de recurso.
  5. (Opcional) Na seção Propriedades adicionais, especifique propriedades adicionais para o adaptador de recurso.

    As propriedades disponíveis são os nomes dos métodos setter da classe referenciada pelo elemento resourceadapter-class no arquivo ra.xml, que define o nome da classe do componente JavaBeans do adaptador de recurso. Quaisquer modificações das propriedades aqui definidas sobrepõem os valores presentes no ra.xml

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de verificação à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  6. Clique em Salvar.
