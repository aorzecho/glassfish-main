Recursos do conector

Use a página Recursos do Conector para configurar os recursos do conector.

Um recurso do conector é um objeto de programa que fornece ao aplicativo uma conexão com um EIS.

Para cada recurso, as informações abaixo são fornecidas.

Nome JNDI

Um nome único que identifica o recurso do conector.

Ativado

Verdadeiro se o recurso do conector estiver ativado ou falso se o recurso do conector estiver desativado.

Conjunto de Conexões

O conjunto de conexões do conector associado ao recurso do conector.

Descrição

Uma breve descrição do recurso do conector.

A tabela Recursos também contém as seguintes opções.

Novo

Botão para criar um novo recurso do conector.

Excluir

Botão para excluir um ou mais recursos do conector selecionados.

Ativar

Botão para ativar um ou mais recursos do conector selecionados.

Desativar

Botão para desativar um ou mais recursos do conector selecionados.
