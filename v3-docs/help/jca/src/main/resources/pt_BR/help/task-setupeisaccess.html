Para Configurar o Acesso ao EIS

  1. Implantar (instalar) um conector. Para implantar um módulo conector.

  2. Crie um pool de conexões para o conector. Consulte Para Criar um Pool de Conexões do Conector.

  3. Crie um recurso do conector que está associado ao pool de conexões. Consulte Para Criar um Recurso do Conector.
