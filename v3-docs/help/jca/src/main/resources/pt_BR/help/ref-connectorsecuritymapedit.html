Editar mapa de segurança do grupo de conexões do conector

Use a página Novo mapa de segurança do conjunto de conexões do conector para editar o mapa de segurança de um conjunto de conexões do conector.

A página Editar mapa de segurança do conjunto de conexões do conector contém as seguintes opções.

Nome do conjunto de conexões

O nome do conjunto de conexões do conector ao qual pertence este mapa de segurança. O nome do conjunto é um campo somente leitura.

Nome do mapa de segurança

O nome do mapa de segurança. O nome é um campo somente leitura. Você pode apenas especificar um nome de mapa de segurança quando cria um novo mapa de segurança.

Nome do mapa de segurança

O nome do mapa de segurança.

Grupos de usuários

Se este botão é selecionado, o campo de texto precisa conter os nomes dos grupos de usuários, separados por vírgulas.

Principais

Se este botão é selecionado, o campo de texto precisa conter os nomes principais, separados por vírgulas.

Nome do usuário

O nome do usuário principal back-end requerido pelo EIS.

Senha

A senha do principal back-end requerido pelo EIS.
