Para Criar um Recurso do Conector

Antes de Começar

Antes de criar um recurso do conector, crie primeiro um pool de conexões do conector.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Sob o nó Recursos, expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Recursos do conector.

    A página Recursos do conector é aberta.

  4. Na página Recursos do conector, clique no botão Novo.

    A página Novo recurso do conector é exibida.

  5. No campo Nome JNDI, digite um nome JNDI único para o recurso do conector.

    Por exemplo: eis/myERP.

  6. Na lista drop-down Nome do Pool, selecione um pool de conexões ao qual pertence o novo recurso do conector.

  7. No campo Descrição, digite uma descrição do recurso do conector.

  8. Marque a caixa de seleção Status Ativado para ativar o recurso do conector durante o runtime.

    Por default, a opção está ativada.

  9. Em Destino disponíveis, selecione o domínio, cluster ou instâncias de servidor em que o recurso do conector residirá e clique em Adicionar.

    Se não quiser implantar o recurso do conector em um dos domínios, clusters, ou instâncias de servidor listados em Destinos selecionados, selecione o destino no campo e clique em Remover.

  10. Na seção Propriedades adicionais, especifique propriedades adicionais para o recurso do conector.

    As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector.

    Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  11. Clique em OK.
