Para Deletar um Mapa de Segurança do Trabalho

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Mapas de Segurança do Trabalho.

    A página Mapas de Segurança do Trabalho é aberta.

  4. Na página Mapas de Segurança do Trabalho, marque a caixa de seleção junto ao esquerda do mapa de segurança do trabalho que você estiver deletando.

  5. Clique em Deletar.
