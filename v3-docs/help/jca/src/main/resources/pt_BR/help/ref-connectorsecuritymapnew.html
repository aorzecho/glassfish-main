Novo Mapa de Segurança do Pool de Conexões do Conector

Use a página Novo Mapa de Segurança do Pool de Conexões do Conector para criar um novo mapa de segurança para um pool de conexões do conector.

A página Novo Mapa de Segurança do Pool de Conexões do Conector contém as opções a seguir.

Nome do Pool de Conexões

O nome do pool de conexões do conector ao qual pertence este mapa de segurança. O nome do pool é um campo somente para leitura.

Nome do mapa de segurança

O nome do mapa de segurança.

Grupos de usuários

Se este botão for selecionado, o campo de texto precisará conter os nomes dos grupos de usuários, separados por vírgulas.

Principais

Se este botão de rádio for selecionado, o campo de texto precisará conter os nomes dos principais, separados por vírgulas.

Nome do usuário

O nome do usuário principal back-end requerido pelo EIS.

Senha

A senha do principal back-end requerido pelo EIS.
