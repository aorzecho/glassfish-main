Para Deletar um Recurso de Objeto Administrado

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Recursos de administração do objeto.

    É aberta a página Recursos de administração do objeto é aberta.

  4. Na página Recursos de administração do objeto, marque a caixa de seleção junto ao do nome do recurso de objeto administrado que estiver deletando.

  5. Clique em Deletar.
