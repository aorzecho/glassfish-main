Novo recurso do conector

Use a página Novo recurso do conector para criar um novo recurso do conector.

A página Novo recurso do conector contém as seguintes opções.

Nome JNDI

Um nome único que identifica o recurso do conector.

Nome do conjunto

O conjunto de conexões ao qual pertence o recurso do conector.

Descrição

Uma breve descrição do recurso do conector.

Status

Se esta opção for selecionada, o recurso do conector será ativado. Por padrão, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso conector. As propriedades disponíveis dependem do adaptador de recurso para o conjunto de conexões do conector. Os pares de nome e valor especificados nessa tabela, podem ser usados para sobrepor os valores padrão definidos pelo fornecedor do adaptador de recurso.

Destinos

Clusters e instâncias independentes do recurso do conector. Mova os destinos desejados para a coluna Destinos selecionados utilizando os botões Adicionar e Adicionar tudo. Mova os destinos desnecessários para a coluna Destinos disponíveis utilizando os botões Remover e Remover tudo.

Esta opção é exibida somente se os clusters ou instâncias independentes tiverem sido criados no domínio.
