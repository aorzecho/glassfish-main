Para editar um mapa de segurança do trabalho

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Expanda o nó Conectores.
  3. Sob o nó Conectores, selecione o nó Mapas de segurança do trabalho.

    A página Mapas de segurança do trabalho é aberta.

  4. Na página Mapas de segurança do trabalho, clique no nome do mapa de segurança do trabalho a ser editado.

    A página Editar o mapa de segurança do trabalho é aberta.

  5. Use o campo Grupo ou o Principal, como a seguir.
    • Se o botão Grupo é selecionado, digite um mapa de cada grupo de usuários EIS back-end para o correspondente grupo de usuários definido no domínio de segurança do Servidor GlassFish.

      Use uma lista separada por vírgula para especificar mais de um mapeamento. Por exemplo:

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • Se o botão Principal é selecionado, digite um mapa de principal EIS back-end para o correspondente principal definido no domínio de segurança do Servidor GlassFish.

      Use uma lista separada por vírgula para especificar mais de um mapeamento. Por exemplo:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  6. Na lista suspensa Adaptador de recurso, selecione o adaptador de recurso para o EIS.
  7. No campo Descrição, digite uma descrição do mapa de segurança do trabalho.
  8. Selecione a caixa de verificação Status ativo, para ativar o mapa de segurança do trabalho.

    Por padrão, a opção está ativada.

  9. Clique em OK.
