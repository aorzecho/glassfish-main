Para criar um mapa de segurança para um conjunto de conexões do conector

Os mapas de segurança dos conjuntos de conexões do conector permitem a criação de um mapeamento entre usuários e grupos de Java EE e usuários e grupos de EIS.

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Expanda o nó Conectores.
  3. Sob o nó Conectores, selecione o nó Conjuntos de conexões do conector.

    A página Conjuntos de conexões do conector é aberta.

  4. Na página Conjuntos de conexões do conector, clique no nome do conjunto de conexões do conector a ser editado.

    A página Editar o conjunto de conexões do conector é aberta.

  5. Na página Editar o conjunto de conexões do conector, clique na aba Mapas de segurança.

    A página Mapas de segurança do conjunto de conexões do conector é aberta.

  6. Na página Mapas de segurança do conjunto de conexões do conector, clique em Novo.

    A página Novo mapa de segurança do conjunto de conexões do conector é aberta.

  7. No campo Nome do mapa de segurança, digite um nome para o mapa de segurança.
  8. Siga um dos procedimentos a seguir, mas não ambos:
    • Selecione o botão Grupos de usuários e digite os nomes dos grupos de usuários, separado por vírgulas.
    • Selecione o botão Principais e digite os nomes dos principais, separado por vírgulas.
  9. No campo Nome do usuário, especifique o nome do usuário principal back-end requerido pelo EIS.
  10. No campo Senha, especifique a senha do principal back-end requerido pelo EIS.
  11. Clique em OK.
  12. Reinicie o Servidor GlassFish.
