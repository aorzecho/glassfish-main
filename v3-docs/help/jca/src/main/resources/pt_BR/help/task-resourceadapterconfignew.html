Para Criar uma Configuração do Adaptador de Recursos

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Sob o nó Recursos, selecione o nó Configs do adaptador de recurso.

    A página Configs do adaptador de recurso é aberta.

  3. Na página Configs do adaptador de recurso, clique em Novo.

    A página Configs do adaptador de recurso é aberta.

  4. Na lista drop-down Nome do adaptador de recurso, selecione o adaptador de recurso.

  5. Na lista drop-down ID do Pool de Threads, selecione o Pool de Threads a ser usado para a configuração do adaptador de recurso.

  6. Na seção Propriedades adicionais, especifique propriedades adicionais para o adaptador de recurso.

    As propriedades disponíveis são os nomes dos métodos setter da classe referenciada pelo elemento resourceadapter-class no arquivo ra.xml, que define o nome da classe do componente JavaBeans do adaptador de recurso. Quaisquer modificações das propriedades aqui definidas substituem os valores presentes no ra.xml

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  7. Clique em OK.
