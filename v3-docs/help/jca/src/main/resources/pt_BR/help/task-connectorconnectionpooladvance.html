Para editar os atributos avançados do conjunto de conexões do conector

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Expanda o nó Conectores.
  3. Sob o nó Conectores, selecione o nó Conjuntos de conexões do conector.

    A página Conjuntos de conexões do conector é aberta.

  4. Na página Conjuntos de conexões do conector, clique no nome do conjunto de conexões do conector a ser editado.

    A página Editar o conjunto de conexões do conector é aberta.

  5. Na página Editar o conjunto de conexões do conector, clique na aba Avançado.

    A página Editar atributos avançados do conjunto de conexões do conector é exibida.

  6. No campo Validar no máximo uma vez, digite o período, em segundos, após o qual uma conexão será validada no máximo uma vez.

    A configuração desse atributo para um valor apropriado minimiza o número de solicitações de validação feitos por uma conexão. O valor padrão é 0, o que significa que o atributo não está ativado.

  7. No campo Tempo limite da perda, digite a quantidade de tempo, em segundos, para rastrear perdas de conexão em um conjunto de conexões.

    Um valor de 0 indica que o rastreamento do vazamento da conexão está desativado. O valor padrão é 0.

  8. Selecione a opção Reclamação de vazamento, para permitir que as conexões que vazaram sejam restauradas no conjunto após a conclusão do rastreamento de vazamento da conexão.

    Por padrão, a opção está desativada.

  9. No campo Novas tentativas de criação, digite o número de tentativas a serem feitas se houver falha ao criar uma nova conexão.

    O valor de 0 indica que nenhuma tentativa será feita para criar a conexão novamente. O valor padrão é 0.

  10. No campo Intervalo de nova tentativa, digite o intervalo, em segundos, entre duas tentativas de criar uma conexão.

    Esse atributo só será usado se o valor Novas tentativas de criação for maior que 0. O valor padrão é 10 segundos.

  11. Selecione a caixa de verificação Amostragem para ativar a amostragem de conexões para o conjunto.

    Por padrão, a opção está ativada.

  12. Selecione a opção Associação lenta ativada, para permitir que as conexões sejam lentamente associadas quando uma operação for executada na mesmas.

    As conexões serão desassociadas quando a transação for concluída e um método de componente for finalizado, o que ajuda a reutilizar as conexões físicas. Por padrão, a opção está desativada.

  13. Selecione a caixa de verificação Inscrição de conexão lenta, para inscrever um recurso para a transação somente quando ele for efetivamente utilizado em um método.

    Por padrão, a opção está desativada.

  14. Selecione a caixa de verificação Associar com segmento, para associar a conexão com um segmento.

    Quando o mesmo segmento for necessário a uma conexão, a conexão já associada ao segmento poderá ser reutilizada, não incorrendo, portanto, na sobrecarga para obtenção de outra conexão do conjunto. Por padrão, a opção está desativada.

  15. Selecione a caixa de verificação Corresponder conexões, para ativar a correspondência de conexão para o conjunto.

    Desative esta opção se você souber que as conexões no conjunto serão sempre homogêneas e, portanto, uma conexão selecionada no conjunto não precisará coincidir com o adaptador de recurso. Por padrão, a opção está ativada.

  16. No campo Corresponder uso da conexão, digite o número de vezes que uma conexão deve ser reutilizada pelo conjunto.

    Depois que uma conexão é reutilizada por um número especificado de vezes, ela é fechada. O valor de 0 indica que o atributo está desativado. O valor padrão é 0.

  17. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
