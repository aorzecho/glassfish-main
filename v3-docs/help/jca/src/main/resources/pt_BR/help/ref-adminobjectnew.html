Novo Recurso do Objeto Administrado

Use a página Novo recurso de administração dos objetos para criar um novo recurso do objeto.

A página Novo recurso de administração dos objetos contém as seguintes opções.

Nome JNDI

Um nome único que identifica o recurso do objeto administrado.

Adaptador de recurso

O adaptador de recurso para o recurso do objeto administrado.

Tipo de recurso

O tipo Java do recurso do objeto administrado (ou por exemplo,javax.jms.Topic),

Nome da classe

O nome da classe de implementação associada com o tipo de recurso.

Descrição

A descrição do recurso do objeto administrado.

Status

Se esta opção está selecionada, o recurso do objeto administrado será ativado. Por default, a opção está ativada.

Propriedades adicionais

Propriedades adicionais para o recurso do objeto administrado. As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector. Os pares de nome e valor especificados nessa tabela podem ser usados para substituir os valores default definidos pelo fornecedor do adaptador de recurso.

Destinos

Clusters e instâncias standalone do recurso do objeto administrado. Mova os destinos desejados para a coluna Destinos selecionados utilizando os botões Adicionar e Adicionar tudo. Mova os destinos desnecessários para a coluna Destinos disponíveis utilizando os botões Remover e Remover tudo.

Esta opção é exibida somente se os clusters ou instâncias standalone tiverem sido criados no domínio.
