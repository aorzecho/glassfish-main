Para editar as propriedades do conjunto de conexões do conector

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Expanda o nó Conectores.
  3. Sob o nó Conectores, selecione o nó Conjuntos de conexões do conector.

    A página Conjuntos de conexões do conector é aberta.

  4. Na página Conjuntos de conexões do conector, clique no nome do conjunto de conexões do conector a ser editado.

    A página Editar o conjunto de conexões do conector é aberta.

  5. Na página Editar o conjunto de conexões do conector, clique na aba Propriedades adicionais.

    A página Editar propriedades do conjunto de conexões do conector é aberta.

  6. (Opcional) Na tabela Propriedades adicionais, especifique propriedades adicionais para o conjunto de conexões do conector.

    As propriedades especificadas dependem do adaptador de recurso usado por este conjunto. Os pares de nome e valor especificados nessa tabela, podem ser usados para sobrepor os valores padrão definidos pelo fornecedor do adaptador de recurso.

    Se você não especifica um valor para uma propriedade, ela é removida da tabela quando termina de editar o conjunto de conexões. Caso não deseje definir agora um valor para uma propriedade, mas deseja que a propriedade permaneça na tabela, para que o valor possa ser definido em outro momento, digite um conjunto de parênteses vazios no campo Valor:

    ()
    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de verificação à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  7. Clique em Salvar.
