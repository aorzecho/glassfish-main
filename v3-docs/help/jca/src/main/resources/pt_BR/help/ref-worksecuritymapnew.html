Novo mapa de segurança do trabalho

Use a página Novo mapa de segurança do trabalho, para criar um novo mapa de segurança do trabalho.

A página Novo mapa de segurança do trabalho contém as seguintes opções.

Nome do mapa de segurança

O nome do mapa de segurança do trabalho.

Grupo

Se o botão Mapeamento de grupo é selecionado, o campo de texto precisa especificar um mapa para de cada grupo de usuários EIS back-end para o correspondente grupo de usuários definido no domínio de segurança do Servidor GlassFish. Use uma lista separada por vírgula para especificar mais de um mapeamento. Por exemplo:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se o botão Mapeamento principal é selecionado, o campo de texto precisa especificar um mapa para de cada grupo de usuários EIS back-end para o correspondente principal definido no domínio de segurança do Servidor GlassFish. Use uma lista separada por vírgula para especificar mais de um mapeamento. Por exemplo:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptador de recurso

O adaptador de recurso para o EIS.

Descrição

Uma descrição para o mapa de segurança do trabalho.

Status

Se esta opção está selecionada, o mapa de segurança do trabalho será ativado. Por padrão, a opção está ativada.
