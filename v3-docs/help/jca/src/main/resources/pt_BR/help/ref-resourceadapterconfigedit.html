Editar Configuração do Adaptador de Recursos

Use a página Editar a config do adaptador de recurso, para modificar as definições do adaptador de recurso.

A página Editar a config do adaptador de recurso contém as seguintes opções.

Nome do adaptador de recurso

O nome do adaptador de recurso que você está configurando. O nome é um campo somente para leitura. Você somente pode especificar um nome de adaptador de recurso quando cria uma nova configuração de adaptador de recurso.

ID do Pool de Threads

O nome de um pool de threads a ser usado para a configuração do adaptador de recurso.

Propriedades adicionais

Propriedades adicionais para o adaptador de recurso. As propriedades disponíveis são os nomes dos métodos setter da classe referenciada pelo elemento resourceadapter-class no arquivo ra.xml, que define o nome da classe do componente JavaBeans do adaptador de recurso. Quaisquer modificações das propriedades aqui definidas substituem os valores presentes no ra.xml
