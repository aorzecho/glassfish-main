Configs do adaptador de recurso

Use a página Configs do adaptador de recurso para gerenciar as configurações do conector (adaptador de recurso).

O config do adaptador de recurso fornece informações de configuração para um adaptador de recurso. Com o Console de Administração, você cria um config de adaptador de recurso após a implementação do adaptador de recurso. Nesse caso, o adaptador de recurso é reiniciado com a nova configuração. Para criar um config do adaptador de recurso antes de implementar o adaptador de recursos, use o comando asadmin create-resource-adapter-config.

Antes de poder criar um config do adaptador de recurso, é preciso criar um conjunto do segmento para que o adaptador de recurso o utilize.

Para cada config do adaptador de recurso, as seguintes informações são fornecidas.

Nome

O nome do adaptador de recurso.

A tabela Configs também contém as seguintes opções.

Novo

Botão para criar um novo config do adaptador de recurso.

Excluir

Botão para excluir um ou mais configs do adaptador de recurso selecionados.
