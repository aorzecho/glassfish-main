Para Deletar um Pool de Conexões do Conector

Quando você deleta um pool de conexões do conector, quaisquer recursos do conector que dependem daquele pool de conexões também são deletados.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Pools de Conexões do Conector.

    A página Pools de Conexões do Conector é aberta.

  4. Na página Pools de Conexões do Conector, marque a caixa de seleção junto ao nome do pool de conexões a ser deletado.

  5. Clique em Deletar.
