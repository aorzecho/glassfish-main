Serviço do conector

Use a página Serviço do conector para modificar as configurações gerais para o serviço do conector que governa os adaptadores de recursos.

A página Serviço do conector contém as seguintes opções:

Carregar padrões

Botão para restaurar com os valores padrão as configurações que apresentam valores padrão. As configurações que não apresentam valores padrão não são alteradas.

Configuration Name

O nome da configuração para a qual as definições nesta página se aplicam. Este campo é de somente leitura.

Tempo limite do desligamento

O número máximo de segundos permitidos durante o desligamento do Servidor GlassFish, para que o método ResourceAdapter.stop de uma instância do módulo do conector seja completada. Os adaptadores de recursos que levam mais tempo para serem desligados são ignorados e o desligamento do Servidor GlassFish continua. O valor padrão é 30 segundos.

Diretriz de carga da classe do conector

A diretriz a ser usada para a carga de classes. As opções disponíveis são:

derivado

Indica que os adaptadores de recursos são fornecidos de acordo com as referências do aplicativo para qualquer um dos recursos do adaptador de recurso. Esta diretriz é a padrão.

global

Indica que todos os adaptadores de recurso serão visíveis para todos os aplicativos.

Esta diretriz não se aplica ao adaptador de recurso JMS pré-instalado, jmsra, que sempre está visível para todos os aplicativos.
