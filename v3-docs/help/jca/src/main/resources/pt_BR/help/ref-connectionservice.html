Serviço do Conector

Use a página Serviço do conector para modificar as definições gerais para o serviço do conector que governa os adaptadores de recursos.

A página Serviço do conector contém as seguintes opções:

Carregar Defaults

Botão para restaurar com os valores default as definições que apresentam valores default. As definições que não apresentam valores default não são alteradas.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Time-out p/Shutdown

O número máximo de segundos permitidos durante o shutdown do GlassFish Server, para que o método ResourceAdapter.stop de uma instância do módulo do conector seja completada. Os adaptadores de recursos que levam mais tempo para serem submetidos a shutdown são ignorados e o shutdown do GlassFish Server continua. O valor default é 30 segundos.

Política de Carga da Classe do Conector

A política a ser usada para a carga de classes. As opções disponíveis são:

derivado

Indica que os adaptadores de recursos são fornecidos de acordo com as referências da aplicação para qualquer um dos recursos do adaptador de recurso. Essa política é a default.

global

Indica que todos os adaptadores de recurso estarão visíveis para todas as aplicações.

Esta política não se aplica ao adaptador de recursos JMS pré-instalado, jmsra, que sempre está visível para todas as aplicações.
