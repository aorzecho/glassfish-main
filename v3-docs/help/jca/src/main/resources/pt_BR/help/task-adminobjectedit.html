Para editar um recurso de objeto administrado

  1. Na árvore de navegação, expanda o nó Recursos.
  2. Expanda o nó Conectores.
  3. Sob o nó Conectores, selecione o nó Recursos de administração do objeto.

    É aberta a página Recursos de administração do objeto é aberta.

  4. Na página Recursos de administração do objeto, clique no novo recurso de objeto a ser editado.

    A página Editar o recurso de administração do objeto é aberta.

  5. Na lista suspensa Adaptador de recurso, selecione o adaptador de recurso para o recurso de objeto administrado.
  6. Na lista suspensa Tipo de recurso, selecione o tipo Java para o recurso de objeto administrado.
  7. Na lista suspensa Nome da classe, selecione o nome da classe de implementação associado com o tipo de recurso.
  8. No campo Descrição, digite uma descrição do recurso de objeto administrado.
  9. Selecione a caixa de verificação Status ativo, para ativar o recurso de objeto administrado.

    Por padrão, a opção está ativada.

  10. (Opcional) Na tabela Propriedades adicionais, especifique propriedades adicionais para o recurso de objeto administrado.

    As propriedades disponíveis dependem do adaptador de recurso para o conjunto de conexões do conector.

    • Para adicionar uma propriedade, clique no botão Adicionar propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de verificação à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir propriedades.
  11. Na guia Destino, clique em Gerenciar destinos para editar os destinos nos quais o objeto administrado está implantado.

    Volte à página Editar recurso de administração do objeto clicando na guia Geral.

  12. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
