Editar propriedades do grupo de conexões de conectores

Use a página Editar propriedades do conjunto de conexões do conector para modificar as configurações do conjunto de conexões do conector.

A página Editar propriedades do conjunto de conexões do conector contém as seguintes opções.

Propriedades adicionais

Propriedades adicionais do conjunto de conexões do conector. As propriedades disponíveis dependem do adaptador de recurso para o conjunto de conexões do conector. Os pares de nome e valor especificados nessa tabela, podem ser usados para sobrepor os valores padrão definidos pelo fornecedor do adaptador de recurso.

Se você não especifica um valor para uma propriedade, ela é removida da tabela quando termina de editar o conjunto de conexões. Caso não deseje definir agora um valor para uma propriedade, mas deseja que a propriedade permaneça na tabela, para que o valor possa ser definido em outro momento, digite um conjunto de parênteses vazios no campo Valor:

()
