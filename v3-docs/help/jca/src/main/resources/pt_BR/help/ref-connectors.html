Conectores

Use a página Conectores para configurar os recursos de conector para os seguintes tipos:

  • Recursos do Conector

  • Pools de Conexões do Conector

  • Recursos do Objeto Administrado

  • Mapas de Segurança do Trabalho

Um módulo do conector (também chamado de um adaptador de recurso) é um componente Java que permite que as aplicações interajam com o software de informações corporativas (EIS). O software EIS inclui diversos tipos de sistemas: planejamento de recursos corporativos (ERP), processamento de transação de mainframe, banco de dados relacional e sistemas de mensagem como o Java Message Service (JMS), entre outros.

Um pool de conexões do conector é um grupo de conexões reutilizáveis para um determinado EIS.

Um recurso do conector é um objeto de programa que fornece à aplicação uma conexão com um EIS.

Um recurso de objeto administrado fornece funcionalidade especializada para uma aplicação.

Um mapa de segurança do trabalho mapeia um principal associado com uma instância de trabalho de entrada para um principal no domínio de segurança do GlassFish Server.
