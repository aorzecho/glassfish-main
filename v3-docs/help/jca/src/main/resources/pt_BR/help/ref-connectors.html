Conectores

Use a página Conectores para configurar os recursos de conector para os seguintes tipos:

  • Recursos do conector

  • Conjuntos de conexões do conector

  • Recursos de administração dos objetos

  • Mapas de segurança de trabalho

Um módulo do conector (também chamado de um adaptador de recurso) é um componente Java que permite que os aplicativos façam a interação com o software de informações corporativas (EIS). O software EIS inclui diversos tipos de sistemas: planejamento de recursos corporativos (ERP), processamento de transação de mainframe, banco de dados relacional e sistemas de mensagem como o Java Message Service (JMS), entre outros.

Um conjunto de conexões do conector é um grupo de conexões reutilizáveis para um determinado EIS.

Um recurso do conector é um objeto de programa que fornece ao aplicativo uma conexão com um EIS.

Um recurso de objeto administrado fornece funcionalidade especializada para um aplicativo.

Um mapa de segurança de trabalho mapeia um principal associado com uma instância de trabalho de entrada para um principal no domínio de segurança do Servidor GlassFish.
