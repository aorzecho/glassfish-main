Mapas de Segurança do Pool de Conexões do Conector

Use a página Mapas de Segurança do Pool de Conexões do Conector para definir os mapas de segurança do pool de conexões do conector.

Os mapas de segurança dos pools de conexões do conector permitem a criação de um mapeamento entre usuários e grupos de Java EE e usuários e grupos de EIS. Um mapa de segurança mapeia o principal recuperado durante a autenticação do servlet ou EJB, para as credenciais aceitas pelo EIS. É possível mapear múltiplos principais do &ProductName: para o mesmo principal back-end.

Um mapa de segurança para um pool de conexões é diferente de um mapa de segurança do conector de trabalho, no sentido que ele mapeia um principal associado com uma instância de trabalho de entrada para um principal no realm de segurança do GlassFish Server. Consulte Mapas de Segurança do Trabalho para obter informações sobre mapas de segurança do trabalho do conector.

Para cada mapa de segurança, as seguintes informações são fornecidas.

Nome

O nome do mapa de segurança.

A tabela Mapas de segurança também contém as seguintes opções.

Novo

Botão para criar um novo mapa de segurança.

Deletar

Botão para deletar um ou mais mapas de segurança selecionados.
