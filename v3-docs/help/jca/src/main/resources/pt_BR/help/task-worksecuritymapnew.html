Para Criar um Mapa de Segurança do Trabalho

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Mapas de Segurança do Trabalho.

    A página Mapas de Segurança do Trabalho é aberta.

  4. Na página Mapas de Segurança do Trabalho, clique em Novo.

    A página Novo Mapa de Segurança do Trabalho é aberta.

  5. No campo Nome do Mapa de Segurança, digite um nome para o mapa de segurança do trabalho.

  6. Use o campo Grupo ou o Principal, como a seguir.

    • Se o botão Grupo é selecionado, digite um mapa de cada grupo de usuários EIS back-end para o correspondente grupo de usuários definido no domínio de segurança do GlassFish Server.

      Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Se o botão Principal é selecionado, digite um mapa de principal EIS back-end para o correspondente principal definido no domínio de segurança do GlassFish Server.

      Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. Na lista drop-down Adaptador de recurso, selecione o adaptador de recurso para o EIS.

  8. No campo Descrição, digite uma descrição do mapa de segurança do trabalho.

  9. Marque a caixa de seleção Status Ativado para ativar o mapa de segurança do trabalho.

    Por default, a opção está ativada.

  10. Clique em OK.
