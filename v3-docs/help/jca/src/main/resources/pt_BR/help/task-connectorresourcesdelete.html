Para Deletar um Recurso do Conector

Um recurso do conector será automaticamente deletado se você deletar o pool de conexões do conector associado ao recurso.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Recursos do conector.

    A página Recursos do conector é aberta.

  4. Na página Recursos do conector, marque a caixa de seleção à esquerda do nome do recurso que você estiver deletando.

  5. Clique em Deletar.
