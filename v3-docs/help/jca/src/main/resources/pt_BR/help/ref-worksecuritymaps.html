Mapas de segurança de trabalho

Use a página Mapas de segurança do trabalho, para definir os mapas de segurança para os adaptadores de recurso.

Um mapa de segurança do conector do trabalho, mapeia a identidade do chamador do trabalho submetido por um adaptador de recurso principal EIS ou por um grupo de usuários EIS para um principal ou grupo de usuários adequado no domínio de segurança do Servidor GlassFish. Um ou mais mapas de segurança do trabalho podem ser associados com um adaptador de recurso. A configuração do mapa de segurança do conector do trabalho suporta o uso de asterisco coringa (*) para indicar todos os usuários ou todos grupos de usuários.

Um mapa de segurança do trabalho é diferente de uma mapa de segurança para um conjunto de conexões do conector, que mapeia o principal recuperado durante a autenticação do servlet ou EJB, para as credenciais aceitas pelo EIS. Consulte Mapas de segurança do grupo de conexões do conector para obter informações sobre os mapas de segurança do conjunto de conexões do conector.

O sistema de informações corporativas (EIS) é um sistema que armazena os dados de uma empresa. Ele pode ser um mainframe, um sistema de mensagens, um sistema de banco de dados ou um aplicativo.

Para cada mapa de segurança do trabalho, as seguintes informações são fornecidas.

Nome

O nome do mapa de segurança do trabalho.

A tabela Mapas de segurança do trabalho também contém as seguintes opções.

Novo

Botão para criar um novo mapa de segurança do trabalho.

Excluir

Botão para excluir um ou mais mapas de segurança do trabalho selecionados.
