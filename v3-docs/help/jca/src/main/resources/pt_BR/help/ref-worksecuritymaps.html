Mapas de Segurança do Trabalho

Use a página Mapas de Segurança do Trabalho para definir os mapas de segurança para os adaptadores de recursos.

Um mapa de segurança do conector do trabalho, mapeia a identidade do chamador do trabalho submetido por um adaptador de recurso principal EIS ou por um grupo de usuários EIS para um principal ou grupo de usuários adequado no domínio de segurança do GlassFish Server. Um ou mais mapas de segurança do trabalho podem ser associados com um adaptador de recurso. A configuração do mapa de segurança do trabalho do conector suporta o uso do asterisco coringa (*) para indicar todos os usuários ou todos os grupos de usuários.

Um mapa de segurança do trabalho é diferente de uma mapa de segurança para um pool de conexões do conector, que mapeia o principal recuperado durante a autenticação do servlet ou EJB, para as credenciais aceitas pelo EIS. Consulte Mapas de Segurança do Pool de Conexões do Conector para obter informações sobre mapas de segurança para pools de conexões do conector.

O sistema de informações corporativas (EIS) é um sistema que armazena os dados de uma empresa. Ele pode ser um mainframe, um sistema de mensagens, um sistema de banco de dados ou uma aplicação.

Para cada mapa de segurança do trabalho, as seguintes informações são fornecidas.

Nome

O nome do mapa de segurança do trabalho.

A tabela Mapas de Segurança do Trabalho também contém as opções a seguir.

Novo

Botão para criar um novo mapa de segurança do trabalho.

Deletar

Botão para deletar um ou mais mapas de segurança do trabalho selecionados.
