Para configurar o serviço do conector

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Serviço do conector.

    É aberta a página Serviço do conector.

  3. No campo Tempo limite de desligamento, digite o número máximo de segundos permitidos durante o desligamento do Servidor GlassFish, para que o método ResourceAdapter.stop da instância do módulo do conector seja completado.

    Os adaptadores de recursos que levam mais tempo para serem desligados são ignorados e o desligamento do Servidor GlassFish continua. O valor padrão é 30 segundos.

  4. Na lista suspensa Diretriz de carga de classe de conector, selecione a diretriz a ser usada.
    • Selecione derivado, para especificar que o adaptador de recurso é fornecido de acordo com as referências de um aplicativo para qual um dos recursos do adaptador de recurso.

      Essa diretriz é a padrão.

    • Selecione global, para especificar que todos os adaptadores de recurso serão visíveis para todos os aplicativos.

    Esta diretriz não se aplica ao adaptador de recurso JMS pré-instalado, jmsra, que sempre está visível para todos os aplicativos.

  5. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar padrões e depois em Salvar.
