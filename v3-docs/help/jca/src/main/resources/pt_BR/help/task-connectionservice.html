Para Configurar o Serviço do Conector

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Sob o nó Configuração, selecione o nó Serviço do conector.

    É aberta a página Serviço do conector.

  3. No campo Time-out p/Shutdown, digite o número máximo de segundos permitidos durante o shutdwon do GlassFish Server, para que o método ResourceAdapter.stop da instância do módulo do conector seja completado.

    Os adaptadores de recursos que levam mais tempo para serem submetidos a shutdown são ignorados e o shutdown do GlassFish Server continua. O valor default é 30 segundos.

  4. Na lista drop-down Política de Carga de Classe de Conector, selecione a política a ser usada.

    • Selecione derivado, para especificar que os adaptadores de recurso são fornecidos de acordo com as referências de uma aplicação para qualquer um dos recursos do adaptador de recurso.

      Essa política é a default.

    • Selecione global, para especificar que todos os adaptadores de recurso estarão visíveis para todas as aplicações.

    Esta política não se aplica ao adaptador de recursos JMS pré-instalado, jmsra, que sempre está visível para todas as aplicações.

  5. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.
