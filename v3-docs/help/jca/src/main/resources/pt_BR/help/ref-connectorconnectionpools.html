Conjuntos de conexões do conector

Use a página Conjuntos de conexões do conector para configurar conjuntos de conexões do conector.

Um conjunto de conexões do conector é um grupo de conexões reutilizáveis para um determinado EIS.

Para cada recurso, as informações abaixo são fornecidas.

Nome do Conjunto

Um nome único que identifica o conjunto de conexões do conector.

Adaptador de recurso

O adaptador de recurso associado ao conjunto de conexões do conector.

Definição da conexão

O nome da classe da definição da conexão para o conjunto de conexões do conector.

Descrição

Uma descrição do conjunto de conexões do conector.

A tabela Conjuntos também contém as seguintes opções.

Novo

Botão para criar um novo conjunto de conexões do conector.

Excluir

Botão para excluir um ou mais conjuntos de conexões do conector selecionados.
