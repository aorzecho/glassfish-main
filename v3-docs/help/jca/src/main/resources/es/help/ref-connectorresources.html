Recursos de conectores

Utilice la página Recursos de conectores para configurar los recursos de conectores.

Un recurso de conector es un objeto de programa que proporciona una conexión a un EIS a una aplicación.

Para cada recurso, aparece la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de conector.

Habilitado

Verdadero si el recurso está habilitado, o falso si no lo está.

Conjunto de conexiones

El conjunto de conexiones de conectores relacionado con el recurso JDBC.

Descripción

Una descripción del recurso de conector.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de conector.

Borrar

Botón para eliminar uno o más recursos de conectores.

Habilitar

Botón para habilitar uno o más recursos de conectores seleccionados.

Deshabilitar

Botón para deshabilitar uno o más recursos de conectores seleccionados.
