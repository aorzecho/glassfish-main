Nuevo Recurso de Objeto Administrado

Utilice la página Nuevo recurso de objeto administrado para crear un nuevo recurso de objeto administrado.

La página Nuevo recurso de objeto administrado contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de objeto administrado.

Adaptador de recursos

El adaptador de recursos para el recurso de objeto administrado.

Tipo de recurso

El tipo Java del recurso de objeto administrado (por ejemplo, javax.jms.Topic).

Nombre de clase

El nombre de clase de implementación asociado con el tipo de recurso.

Descripción

Una descripción del recurso de objeto administrado.

Estado

Si esta opción está seleccionada, el recurso de objeto administrado está habilitado. Esta opción se habilita por defecto.

Propiedades adicionales

Propiedades adicionales para el recurso de objeto administrado. Las propiedades disponibles dependen del adaptador de recursos para el conjunto de conexiones de conectores. Los pares nombre-valor especificados en esta tabla se pueden emplear para sobrescribir los valores predeterminados de las propiedades definidas por el proveedor del adaptador de recursos.

Destinos

Clústeres e instancias independientes para el recurso de objeto administrado. Mueva los destinos deseados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos disponibles mediante los botones Eliminar y Eliminar todos.

Esta opción sólo se muestra si en el dominio se han creado instancias de clúster o independientes.
