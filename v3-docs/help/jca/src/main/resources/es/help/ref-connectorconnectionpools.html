Conjuntos de conexiones de conectores

Utilice la página Conjuntos de conexiones de conectores para configurar conjuntos de conexiones de conectores.

Un conjunto de conexiones del conector es un grupo de conexiones reutilizables para un determinado EIS.

Para cada recurso, aparece la siguiente información.

Nombre de conjunto

Un nombre exclusivo que identifica el conjunto de conexiones de conectores.

Adaptador de recursos

El adaptador de recursos asociado con el conjunto de conexiones de conectores.

Definición de conexión

El nombre de clase de la definición de conexión para el conjunto de conexiones de conectores.

Descripción

Una descripción del conjunto de conexiones de conectores.

La tabla Conjuntos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo conjunto de conexiones de conectores.

Borrar

Botón para eliminar uno o más conjuntos de conexiones de conectores seleccionados.
