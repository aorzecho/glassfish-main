Configurar el Servicio de Conector

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, seleccione el nodo Servicio de conector.

    Se abrirá la página Servicio de conector.

  3. En el campo Tiempo de espera de cierre, escriba el número máximo de segundos permitidos durante el cierre de GlassFish Server para que se complete el método ResourceAdapter.stop de una instancia del módulo del conector.

    Los adaptadores de recursos que tarden más en cerrarse se ignoran y continúa el cierre de GlassFish Server. El valor predeterminado es 30 segundos.

  4. En la lista desplegable Directiva de carga de clases del conector, seleccione la directiva que se va a utilizar.

    • Seleccione derivados para especificar que los adaptadores de recursos se ofrecen en función de las referencias de una aplicación a cualquiera de los recursos del adaptador de recursos.

      Esta directiva es la predeterminada.

    • Seleccione global para especificar que todos los adaptadores de recursos serán visibles en todas las aplicaciones.

    Esta política no se aplica al adaptador de recursos del sistema JMS preinstalado, jmsra, que es siempre visible en todas las aplicaciones.

  5. Haga clic en Guardar.

    Sugerencia:

    Para volver a la configuración por defecto, haga clic en el botón Cargar Valores por Defecto y, a continuación, en Guardar.
