Para eliminar una configuración del adaptador de recursos

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. En el nodo Recursos, seleccione el nodo Configuración de adaptadores de recursos.

    Se abre la página Configuración del adaptador de recursos.

  3. En la página Configuración del adaptador de recursos, seleccione la casilla de verificación situada junto al nombre de la configuración del adaptador de recursos que está eliminando.
  4. Haga clic en Eliminar.
