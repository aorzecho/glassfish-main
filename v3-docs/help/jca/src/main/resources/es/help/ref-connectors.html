Conectores

Utilice la página Conectores para configurar los recursos de conectores de los siguientes tipos:

  • Recursos de Conectores

  • Pools de Conexiones de Conectores

  • Recursos de Objeto de Administración

  • Asignaciones de Seguridad de Trabajo

Un módulo de conector (también llamado un adaptador de recursos) es un componente Java que permite a las aplicaciones interactuar con software de información empresarial (EIS). El software EIS incluye varios tipos de sistemas: planificación de recursos empresariales (ERP), procesamiento de transacciones mainframe, bases de datos no relacionales y sistemas de mensajería como el Java Message Service (JMS), entre otros.

Un conjunto de conexiones del conector es un grupo de conexiones reutilizables para un determinado EIS.

Un recurso de conector es un objeto de programa que proporciona una conexión a un EIS a una aplicación.

Un recurso de objeto administrado proporciona funciones especializadas para una aplicación.

Un mapa de seguridad de tareas asigna un elemento principal asociado a una instancia de tareas entrante a un elemento principal del dominio de seguridad de GlassFish Server.
