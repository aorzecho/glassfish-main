Para editar atributos avanzados del conjunto de conexiones del conector

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Expanda el nodo Conectores.
  3. En el nodo Configuración, seleccione el nodo Conjuntos de conexiones de conectores.

    Se abrirá la página Conjuntos de conexiones de conectores.

  4. En la página Conjunto de conexiones de conectores, haga clic en el nombre del conjunto de conexiones de conectores que desee editar.

    Se abrirá la página Editar conjunto de conexiones de conectores.

  5. En la página Editar conjunto de conexiones de conectores, haga clic en la ficha Avanzado.

    Se abre la página Editar atributos avanzados del conjunto de conexiones de conector.

  6. En el campo Validar al menos una vez, introduzca el tiempo, en segundos, tras el cual se valida una conexión al menos una vez.

    Si se define este atributo con un valor apropiado se minimiza el número de solicitudes de validación por conexión. El valor predeterminado es 0, lo que significa que el atributo no está habilitado.

  7. En el campo Tiempo de espera de pérdidas, introduzca el tiempo, en segundos, para realizar un seguimiento de las pérdidas de conexión de un conjunto de conexiones.

    El valor 0 indica que el seguimiento de pérdidas de conexión está deshabilitado. El valor predeterminado es 0.

  8. Seleccione la opción Reclamación de pérdidas para restaurar las conexiones perdidas en el conjunto una vez haya finalizado el seguimiento de pérdidas de conexión.

    Esta opción se deshabilita por defecto.

  9. En el campo Intentos de creación, introduzca el número de intentos que se realizarán si se produjera un error al crear una conexión nueva.

    El valor de 0 significa que no se hará ningún intento para crear la conexión de nuevo. El valor predeterminado es 0.

  10. En el campo Intervalo de reintentos, introduzca el intervalo, en segundos, que hay entre dos intentos de creación de una conexión.

    Este atributo se utiliza sólo si el valor de intentos de reintento de creación es superior a 0. El valor predeterminado es 10 segundos.

  11. Seleccione la casilla de verificación Agrupación para habilitar la agrupación de conexiones para el conjunto.

    Esta opción se habilita por defecto.

  12. Seleccione la casilla de verificación Asociación inactiva para permitir que las conexiones se asocien de forma inactiva cuando se realice una operación con ellas.

    Las conexiones se disocian cuando la transacción se completa y un método de componentes finaliza, lo que ayuda a reutilizar las conexiones físicas. Esta opción se deshabilita por defecto.

  13. Seleccione la opción Alistamiento de conexiones inactivas para alistar un recurso en la transacción sólo cuando se esté utilizando en ese instante en un método.

    Esta opción se deshabilita por defecto.

  14. Seleccione la casilla de verificación Asociar con subproceso para asociar una conexión con un subproceso.

    Cuando el mismo subproceso necesita otra conexión, puede volver a utilizar la conexión que ya está asociada con ese subproceso, sin necesidad de incurrir en la sobrecarga de coger otra conexión del conjunto. Esta opción se deshabilita por defecto.

  15. Seleccione la casilla de verificación Correspondencia de conexiones para habilitar la correspondencia de conexiones en el conjunto.

    Esta opción se puede deshabilitar si sabe que las conexiones del conjunto siempre serán homogéneas y, por tanto, para una conexión recogida del conjunto no se necesita establecer correspondencia mediante el adaptador de recursos. Esta opción se habilita por defecto.

  16. En el campo Uso de la correspondencia de conexiones, introduzca el número de veces que una conexión debe ser reutilizada por el conjunto.

    La conexión se cerrará una vez reutilizada durante un número específico de veces. El valor 0 indica que el atributo no está habilitado. El valor predeterminado es 0.

  17. Haga clic en Guardar.

    Consejo - Para volver a los valores de configuración predeterminados, haga clic en el botón Cargar predeterminados y, a continuación, en Guardar.
