Suprimir una Asignación de Seguridad de Trabajo

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Expanda el nodo Conectores

  3. En el nodo Conectores, seleccione el nodo Mapas de seguridad de tareas.

    Se abre la página Mapas de seguridad de tareas.

  4. En la página Mapas de seguridad de tareas, seleccione la casilla de verificación situada junto al nombre del mapa de seguridad de tareas que esté eliminando.

  5. Haga clic en Eliminar.
