Configuraciones de Adaptador de Recursos

Utilice la página Configuración del adaptador de recursos para gestionar los parámetros de configuración del conector (adaptador de recursos).

La configuración del adaptador de recursos proporciona información de configuración para un adaptador de recursos. Con la Consola de Administración puedes crear una configuración del adaptador de recursos después de implementar dicho adaptador. En este caso, el adaptador de recursos se reinicia con la nueva configuración. Para crear una configuración del adaptador de recursos antes de desplegar dicho adaptador, utilice el comando asadmin create-resource-adapter-config .

Antes de que pueda crear una configuración del adaptador de recursos, debe crear un conjunto de subprocesos para que lo utilice dicho adaptador.

Para la configuración de cada adaptador de recursos, se proporciona la siguiente información.

Nombre

El nombre del adaptador de recursos.

La tabla Configuración también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración del adaptador de recursos.

Borrar

Botón para eliminar una o más configuraciones seleccionadas del adaptador de recursos.
