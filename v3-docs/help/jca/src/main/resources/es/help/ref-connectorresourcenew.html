Nuevo Recurso de Conector

Utilice la página Nuevo recurso de conector para crear un nuevo recurso de conector.

La página Nuevo recurso de conector contiene las siguientes opciones.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de conector.

Nombre de conjunto

El conjunto de conexiones al que pertenece el nuevo recurso de conector.

Descripción

Una descripción del recurso de conector.

Estado

Si está opción está seleccionada, entonces el recurso de conector está activado. Esta opción se habilita por defecto.

Propiedades adicionales

Propiedades adicionales para el recurso de conector. Las propiedades disponibles dependen del adaptador de recursos para el conjunto de conexiones de conectores. Los pares nombre-valor especificados en esta tabla se pueden emplear para sobrescribir los valores predeterminados de las propiedades definidas por el proveedor del adaptador de recursos.

Destinos

Clústeres e instancias independientes para el recurso de conector. Mueva los destinos deseados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.

Esta opción sólo se muestra si se han creado clusters o instancias independientes en el dominio.
