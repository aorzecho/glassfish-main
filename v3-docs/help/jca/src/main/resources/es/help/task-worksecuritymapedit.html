Para editar un mapa de seguridad de tareas

  1. En el árbol de navegación, amplíe el nodo Configuración.
  2. Expanda el nodo Conectores.
  3. En el nodo Conectores, seleccione el nodo Mapas de seguridad de tareas.

    Se abre la página Mapas de seguridad de tareas.

  4. En la página Mapas de seguridad, haga clic en el nombre del mapa de seguridad de tareas que esté editando.

    Se abre la página Editar mapa de seguridad de tareas.

  5. Utilice el campo Grupo o el campo Elemento principal como se indica a continuación.
    • Si selecciona el botón de opción Grupo, asigne cada grupo de usuarios EIS de back-end al grupo de usuarios correspondiente definido en el dominio de seguridad de GlassFish Server.

      Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-group-1=server-group-1,eis-group-2=server-group-2
    • Si selecciona el botón de opción Elemento principal, asigne cada elemento principal EIS de back-end al elemento principal correspondiente definido en el dominio de seguridad de GlassFish Server.

      Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
  6. En la lista desplegable Adaptador de recursos, seleccione el adaptador de recursos para el EIS.
  7. En el campo Descripción, introduzca una descripción del mapa de seguridad de tareas.
  8. Seleccione la casilla de verificación Estado habilitado para habilitar el mapa de seguridad de tareas.

    Esta opción se habilita por defecto.

  9. Haga clic en Aceptar.
