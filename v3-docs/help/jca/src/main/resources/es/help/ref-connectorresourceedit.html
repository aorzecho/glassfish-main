Editar recurso de conector

Utilice la página Editar recurso de conector para modificar los parámetros de configuración para un recurso de conector.

La página Editar recurso de conector contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restablecer las opciones que tienen valores por defecto a estos. Las opciones que no tienen valores por defecto no se modifican.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de conector. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un nuevo recurso de conector.

Nombre de conjunto

El conjunto de conexiones de conectores relacionado con este recurso.

Descripción

Una descripción del recurso de conector.

Estado

Si está opción está seleccionada, entonces el recurso de conector está activado. Esta opción se habilita por defecto.

Propiedades adicionales

Propiedades adicionales para el recurso de conector. Las propiedades disponibles dependen del adaptador de recursos para el conjunto de conexiones de conectores. Los pares nombre-valor especificados en esta tabla se pueden emplear para sobrescribir los valores predeterminados de las propiedades definidas por el proveedor del adaptador de recursos.
