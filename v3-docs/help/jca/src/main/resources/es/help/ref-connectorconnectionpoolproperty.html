Editar propiedades de conjunto de conexiones de conector

Utilice la página Editar propiedades de conjunto de conexiones del conector para modificar las propiedades de un conjunto de conexiones del conector.

La página Editar propiedades de conjunto de conexiones del conector presenta las opciones siguientes:

Propiedades adicionales

Propiedades adicionales para el conjunto de conexiones de conectores. Las propiedades disponibles dependen del adaptador de recursos para el conjunto de conexiones de conectores. Los pares nombre-valor especificados en esta tabla se pueden emplear para sobrescribir los valores predeterminados de las propiedades definidas por el proveedor del adaptador de recursos.

Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del conjunto de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, escriba un conjunto de paréntesis vacío en el campo Valor: 

()
