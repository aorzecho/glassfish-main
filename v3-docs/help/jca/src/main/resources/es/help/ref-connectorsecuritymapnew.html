Nueva Asignación de Seguridad del Pool de Conexiones de Conector

Utilice la página Nueva asignación de seguridad del conjunto de conexiones de conector para crear una asignación de un conjunto de conexiones del conector.

La página Nueva asignación de seguridad del conjunto de conexiones de conector presenta las opciones siguientes:

Nombre del conjunto de conexiones

El nombre del conjunto de conexiones de conectores al que pertenece este mapa de seguridad. El nombre del conjunto es un campo sólo de lectura.

Nombre del mapa de seguridad

El nombre del mapa de seguridad.

Grupos de usuarios

Si este botón de radio está seleccionado, el campo de texto debe contener los nombres de los grupos de usuarios, separados por comas.

Principales

Si este botón de radio está seleccionado, el campo de texto debe contener los nombres de los elementos principales, separados por comas.

Nombre de usuario

El nombre de usuario del principal backend requerido por el EIS.

Contraseña

La contraseña del principal backend requerida por el EIS.
