Crear un Recurso de Objeto Administrado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Expanda el nodo Conectores

  3. En el nodo Conectores, seleccione el nodo Recursos de objeto administrado.

    La página Recursos de objeto administrado se abre.

  4. En la página Recursos de objeto administrado, haga clic en el botón Nuevo.

    Se abre la página Nuevo recurso de objeto administrado.

  5. En el campo Nombre JNDI, introduzca un nombre único para identificar el recurso de objeto administrado.

  6. En la lista desplegable Adaptador de recursos, seleccione el adaptador de recursos para el recurso de objeto administrado.

  7. En la lista desplegable Tipo de recurso, seleccione el tipo Java para el recurso de objeto administrado.

  8. En la lista desplegable Nombre de clase, seleccione el nombre de clase de implementación asociado con el tipo de recurso.

  9. En el campo Descripción, introduzca una descripción del recurso de objeto administrado.

  10. Seleccione la casilla de verificación Estado habilitado para habilitar el recurso de objeto administrado.

    Esta opción se habilita por defecto.

  11. En la tabla Propiedades adicionales, especifique las propiedades adicionales para el recurso de objeto administrado.

    Las propiedades disponibles dependen del adaptador de recursos para el conjunto de conexiones de conectores.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, escriba el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  12. En la sección Destinos Disponibles, seleccione el dominio, el cluster o las instancias de servidor donde residirá el objeto administrado y haga clic en Agregar.

    Para eliminar uno de los dominios, clústeres o instancias de servidor que se muestran en la lista del campo Seleccionado, selecciónelo y haga clic en Eliminar.

  13. Haga clic en Aceptar.
