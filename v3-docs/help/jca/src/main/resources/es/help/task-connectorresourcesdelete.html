Para borrar un recurso de conector

Un recurso de conector se elimina automáticamente si se borra el conjunto de conexiones de conector asociado al recurso.

  1. En el árbol de navegación, amplíe el nodo Recursos.
  2. Expanda el nodo Conectores.
  3. En el nodo Configuración, seleccione el nodo Recursos de conectores.

    Se abrirá la página Recursos de conectores.

  4. En la página Recursos de conectores, seleccione la casilla de verificación situada junto al nombre del recurso que está eliminando.
  5. Haga clic en Eliminar.
