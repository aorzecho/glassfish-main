Suprimir un Recurso de Conector

Un recurso de conector se suprime automáticamente si se suprime el pool de conexiones de conector asociado al recurso.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Recursos de Conector.

    Se abrirá la página Recursos de Conector.

  4. En la página Recursos de Conector, seleccione la casilla de cntrol situada junto al nombre del recurso que está suprimiendo.

  5. Haga clic en Suprimir.
