Servicio de Conector

Utilice la página Servicio de conector para modificar los parámetros de configuración generales del servicio de conector, que controla los adaptadores de recursos.

La página Servicio de conector contiene las siguientes opciones.

Cargar valores predeterminados

Botón para restaurar la configuración que tiene valores por defecto a estos valores. La configuración que no tiene valores por defecto no se modifica.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Tiempo de espera de cierre

El número máximo de segundos permitidos durante el cierre de GlassFish Server para que se complete el método ResourceAdapter.stop de una instancia del módulo del conector. Los adaptadores de recursos que tarden más en cerrarse se ignoran y continúa el cierre de GlassFish Server. El valor predeterminado es 30 segundos.

Directiva de carga de clases del conector

La directiva que se utiliza para cargar las clases. Las opciones disponibles son:

derivada

Indica que los adaptadores de recursos se proporcionan en función de las referencias de una aplicación a cualquiera de los recursos del adaptador de recursos. Esta directiva es la predeterminada.

global

Indica que todos los adaptadores de recursos serán visibles en todas las aplicaciones.

Esta política no se aplica al adaptador de recursos del sistema JMS preinstalado, jmsra, que es siempre visible en todas las aplicaciones.
