Crear una Asignación de Seguridad para Pools de Conexiones de Conector

Las asignaciones de seguridad para conjuntos de conexiones del conector permiten crear una asignación entre los usuarios y grupos de usuarios de Java EE y los de EIS.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Expanda el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Conjuntos de conexiones de conectores.

    Se abrirá la página Conjuntos de conexiones de conectores.

  4. En la página Conjunto de conexiones de conectores, haga clic en el nombre del conjunto de conexiones que desee editar.

    Se abrirá la página Editar conjunto de conexiones de conectores.

  5. En la página Editar conjunto de conexiones de conectores, haga clic en la ficha Mapas de seguridad.

    Se abre la página Asignaciones de seguridad de conjuntos de conexiones de conector.

  6. En la página Asignaciones de seguridad de conjuntos de conexiones de conector, haga clic en Nuevo.

    Se abre la página Asignaciones de seguridad de conjuntos de conexiones de conector.

  7. En el campo Nombre de mapa de seguridad, escriba un nombre para el mapa de seguridad.

  8. Realice una de las siguientes acciones, pero no ambas:

    • Active el botón de radio Grupos de Usuarios y escriba los nombres de los grupos de usuarios, separados por comas.

    • Active el botón de radio Principales y escriba los nombres del principal, separados por comas.

  9. En el campo Nombre de Usuario, especifique el nombre de usuario del principal de backend requerido por el EIS.

  10. En el campo Contraseña, especifique la contraseña del principal de backend requerida por el EIS.

  11. Haga clic en Aceptar.

  12. Reinicie GlassFish Server.
