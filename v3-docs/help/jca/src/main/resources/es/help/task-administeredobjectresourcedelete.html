Suprimir un Recurso de Objeto Administrado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Expanda el nodo Conectores

  3. En el nodo Conectores, seleccione el nodo Recursos de objeto administrado.

    La página Recursos de objeto administrado se abre.

  4. En la página Recursos de objeto administrado, seleccione la casilla de verificación situada junto al nombre del recurso de objeto administrado que está eliminando.

  5. Haga clic en Eliminar.
