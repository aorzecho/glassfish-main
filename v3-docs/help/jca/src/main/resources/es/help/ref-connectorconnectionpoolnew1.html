Nuevo Pool de Conexiones del Conector (Paso 1 de 2)

Utilice el nuevo conjunto de conexiones de conectores (paso 1 de 2) para crear un nuevo conjunto de conexiones de conectores.

La página Nuevo conjunto de conexiones de conectores (paso 1 de 2) contiene las siguientes opciones.

Nombre

Un nombre exclusivo que identifica el conjunto de conexiones de conectores.

Adaptador de recursos

El adaptador de recursos asociado con el conjunto de conexiones de conectores.

Definición de conexión

La definición de conexión para el conjunto de conexiones de conectores. Las opciones de la lista dependen del adaptador de recursos seleccionado. La definición de conexión identifica un elemento connection-definition del adaptador de recursos en el archivo ra.xml.
