Recursos de Objeto de Administración

Utilice la página Recursos de objeto administrado para configurar los recursos de objeto administrado.

Un recurso de objeto administrado proporciona funciones especializadas que se definen mediante el adaptador de recursos del módulo de conector implementado.

Para cada recurso de objeto administrado, se proporciona la siguiente información.

Nombre JNDI

Un nombre exclusivo que identifica el recurso de objeto administrado.

Habilitado

Verdadero si el recurso de objeto administrado está habilitado, o falso si el recurso está deshabilitado.

Tipo de recurso

El tipo Java del recurso de objeto administrado.

Descripción

Una descripción del recurso de objeto administrado.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de objeto administrado.

Borrar

Botón para eliminar uno o más recursos de objeto administrado seleccionados.

Habilitar

Botón para habilitar uno o más recursos de objeto administrado seleccionados.

Deshabilitar

Botón para deshabilitar uno o más recursos de objeto administrado.
