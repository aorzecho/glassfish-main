Crear una Asignación de Seguridad de Trabajo

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Expanda el nodo Conectores.

  3. En el nodo Conectores, seleccione el nodo Mapas de seguridad de tareas.

    Se abre la página Mapas de seguridad de tareas.

  4. En la página Mapas de seguridad de tareas, haga clic en Nuevo.

    Se abre la página Nuevo mapa de seguridad de tareas.

  5. En el campo Nombre de mapa de seguridad, escriba un nombre para el mapa de seguridad.

  6. Utilice el campo Grupo o el campo Elemento principal como se indica a continuación.

    • Si activa el botón de radio Grupo, asigne cada grupo de usuarios EIS de backend al grupo de usuarios correspondiente definido en el dominio de seguridad de GlassFish Server.

      Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Si activa el botón de radio Principal, asigne cada principal EIS de backend al principal correspondiente definido en el dominio de seguridad de GlassFish Server.

      Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. En la lista desplegable Adaptador de recursos, seleccione el adaptador de recursos para el EIS.

  8. En el campo Descripción, introduzca una descripción del mapa de seguridad de tareas.

  9. Seleccione la casilla de verificación Estado habilitado para habilitar el mapa de seguridad de tareas.

    Esta opción se habilita por defecto.

  10. Haga clic en Aceptar.
