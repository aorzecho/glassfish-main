Nueva Asignación de Seguridad de Trabajo

Utilice la página Nuevo mapa de seguridad de tareas para crear un nuevo mapa de seguridad de tareas.

La página Nuevo mapa de seguridad de tareas contiene las siguientes opciones.

Nombre del mapa de seguridad

El nombre del mapa de seguridad de tareas.

Grupo

Si el botón de radio Asignación de Grupo está activado, el campo de texto debe especificar una asignación de cada grupo de usuarios EIS de backend al grupo de usuarios correspondiente definido en el dominio de seguridad de GlassFish Server. Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Si el botón de radio Asignación de Principal está activado, el campo de texto debe especificar una asignación de cada principal EIS de backend al principal correspondiente definido en el dominio de seguridad de GlassFish Server. Use una lista separada por comas para especificar más de una asignación. Por ejemplo:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptador de recursos

El adaptador de recursos para el EIS.

Descripción

Una descripción del mapa de seguridad de tareas.

Estado

Si esta opción está seleccionada, el mapa de seguridad de tareas está habilitado. Esta opción se habilita por defecto.
