Editar la configuración del adaptador de recursos

Utilice la página Editar configuración del adaptador de recursos para modificar los parámetros de configuración del adaptador de recursos.

La página Editar configuración del adaptador de recursos contiene las siguientes opciones.

Nombre del adaptador de recursos

El nombre del adaptador de recursos que está configurando. Éste campo es de sólo lectura. Sólo se puede especificar un nombre de adaptador de recursos cuando cree una nueva configuración del adaptador de recursos.

Id. del conjunto de subprocesos

El conjunto de subprocesos que se va a utilizar para la configuración del adaptador de recursos.

Propiedades adicionales

Propiedades adicionales para el adaptador de recursos. Las propiedades disponibles son los nombres de métodos de establecimiento de la clase a la que hace referencia el elemento resourceadapter-class del archivo ra.xml, que define el nombre de clase del componente JavaBeans del adaptador de recursos. Las modificaciones en las propiedades definidas aquí sobreescriben los valores predeterminados presentes en ra.xml.
